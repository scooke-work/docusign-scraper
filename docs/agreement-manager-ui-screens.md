# Docusign Agreement Manager / IAM — UI Screen Inventory

A list of the UI screens (pages, panes, dashboards, views, managers, dialogs)
referenced in the Agreement Manager support documentation.

> **Scope & caveat:** This is **derived from the scraped support docs** (`data/
> agreement-manager-support`), not an official screen map — Docusign doesn't
> publish one. It's comprehensive *to what the docs name*; screens never mentioned
> in docs won't appear, and some entries are sub-views of a parent screen. Names
> match the docs (the live UI may differ slightly — e.g., the obligation Type
> list, which we corrected from screenshots).

---

## Home & navigation
- **Docusign Home** — the platform landing page.
- **Agreements page** — the Agreements area; left-nav entry into Agreement Manager.
- **Completed Documents** — the main agreement list (a.k.a. the *Agreement List
  Table*). Sub-views/tabs: **Inbox**, **Completed**, **Uploaded**; **Detail
  Listing page** for a single row.

## Agreement preview (open an agreement)
- **Agreement Preview page** — the document viewer for one agreement. Contains:
  - **Details pane** — extracted attributes/fields ("Review & edit details"),
    with the AI-suggestion spark icon and "jump-to" highlighting.
  - **Preview pane** — the rendered document with highlights.
  - **Obligations pane** — the obligation list; opens **Create Obligation pane**
    and **Edit Obligation pane**.
  - **CLM Attributes pane** — attributes sourced from CLM.
  - **AI Suggestions dialog** / **AI wizard** — review & accept AI-surfaced values.

## Insights — dashboards & reports
- **Insights hub** — the analytics/reporting area. Contains:
  - **Overview dashboard**
  - **Renewals dashboard**
  - **Obligations dashboard**
  - **Reports view** — detailed report views (incl. **Stacked Bar Chart view**),
    **Audit Logs view**, and the **Starred** tab for favorites.
  - **Save Report** / **Add Existing Report** dialog boxes.

## Parties
- **Parties page** (IAM **Party Hub**) — the parties list (**Parties view**).
  - **Hierarchies tab** — agreement hierarchies; **Create Hierarchy wizard**.
  - **Review Duplicate Parties page** — find/merge duplicate parties.

## Settings & management ("Managers")
- **Fields Manager** (Manage Fields page) — **Fields view / table**; the **Add a
  field** dialog; **Hide Fields** dialog.
- **Agreement Types Manager** (Manage Agreement Types page) — **Agreement Types
  view / table**.
- **Obligation Types** (Manage Obligation Types page) — **Obligation Types view /
  table**.
- **Notification Preferences page** — notification subscriptions.
- **Uploads & processing** — **Uploaded Agreements table**, **Jobs page**,
  **Processing tab** (upload/processing status).
- **Filters dialog box** — filtering the agreement list / dashboards.

## Account & admin
- **Users page**, **Accounts view**, **Admin page** — administration.
- **Audit Logs view** — activity logging.

## App Center
- **Docusign App Center page** — install and manage apps/extensions.
