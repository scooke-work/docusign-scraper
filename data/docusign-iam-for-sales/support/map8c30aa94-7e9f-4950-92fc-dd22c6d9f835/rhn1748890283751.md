# Customize Agreement Manager for Salesforce Columns

Learn to configure the agreement columns displayed in Agreement Manager for Salesforce.

Important: This feature is only available to IAM for
Sales customers. Contact [Docusign support](https://support.docusign.com/s/?language=en_US&_gl=1*74elk4*_gcl_au*MTY5ODQyMjA2MC4xNzY5NjM1OTA1LjE5ODM3MTQzNjguMTc2OTYzNjIwNC4xNzY5NjM2MjA0) or [Docusign
sales](https://www.docusign.com/contact-sales) for more information.

The Docusign Completed Agreements component displays Agreement Manager on Salesforce records. You can [customize the agreement filters column view](https://support.docusign.com/s/document-item?bundleId=xnz1717535025265&topicId=qrh1702945502942.html&_LANG=enus) displayed in the Docusign Completed Agreements component.

When you customize your column view, the following rules apply:

- Your changes do not affect other users.
- Your custom settings remain when you log off and on.
- Your settings apply to all records and objects that contain the Docusign Completed Agreements component.

Note: Your settings *do not* remain if you empty your browser cache, use a
new browser, or open an incognito session.

   The Customize columns dialog box opens. A number of
   fields are turned on (to the right) by default. These fields represent the
   columns in the agreement list table. Columns display in the order that the
   fields are represented in the dialog box.

   ![A screenshot of the Customize columns dialog box that shows the default selections.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Completed-documents-view_Customize-columns-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU3NDcsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.2jb2HZs5KB-TQxtjfJf2mctEmBlt8cKFLlG0sGCwNhA&_LANG=enus "A screenshot of the Customize columns dialog box that shows the default selections.")
2. Search for columns in the Find columns field or scroll
   through the list.
3. Select the toggles to turn the associated field on (to the right) or off (to
   the left).
4. (Optional) Re-order the columns: Select the dots next to a field name and drag
   it to a new location:

   ![A snippet of the Customize columns dialog box that highlights the 'drag to' feature for reordering column headers.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Completed-documents_Customize-columns-dialog-box_drag-and-drop.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU3NDcsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.2jb2HZs5KB-TQxtjfJf2mctEmBlt8cKFLlG0sGCwNhA&_LANG=enus "A snippet of the Customize columns dialog box that highlights the 'drag to' feature for reordering column headers.")
5. Select Save after configuring the columns.

   The dialog box closes.

You have completed the process. Docusign
displays your new column configuration in the Docusign Agreements
component.
