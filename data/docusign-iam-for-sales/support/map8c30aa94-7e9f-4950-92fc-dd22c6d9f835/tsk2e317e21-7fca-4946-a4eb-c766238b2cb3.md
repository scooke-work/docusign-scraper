# Step 1: Create an Agreement Desk for Salesforce External Object

Learn to create an Agreement Desk for Salesforce external object to turn on reporting.

## Before you begin

Important: This feature is only available to IAM for Sales
customers. Contact [Docusign support](https://support.docusign.com/s/?language=en_US&_gl=1*74elk4*_gcl_au*MTY5ODQyMjA2MC4xNzY5NjM1OTA1LjE5ODM3MTQzNjguMTc2OTYzNjIwNC4xNzY5NjM2MjA0) or [Docusign
sales](https://www.docusign.com/contact-sales) for more information.

[Purchase a ​ Docusign ​ Intelligent Agreement
Management (IAM) for Sales application](https://www.docusign.com/intelligent-agreement-management?utm_source=bing&utm_medium=cpc&utm_campaign=GBL_XX_PRF_UPS_2410_SEMIAM&utm_term=docusign%20iam&campaignid=569731866&adgroupid=1178678316174006&location=84406&device=c&matchtype=e&extensionid=&creative=73667602015411&keyword=docusign%20iam&placement=&network=o&msclkid=3287ee64f2c01f69bcd686665f40e7da).

Before you can view Agreement Desk reports in Salesforce, you must create an Agreement Desk
for Salesforce external object. With an external object, you can create custom
Salesforce reports that display Docusign Agreement Desk requests.

Use this procedure to create an Agreement Desk for Salesforce external object in
your Salesforce organization:

1. As a Salesforce administrator, open the New External Data
   Source page:
   1. Select the gear icon ![A screenshot displays the Salesforce Setup gear icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU3MTgsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.Tj-rc07vmhCKQgyPkzXoqdKIMwAaoS-8qsBA6NOlrGs&_LANG=enus "A screenshot displays the Salesforce Setup gear icon.") and choose Setup.
   2. Enter and select External Data Sources in the
      Quick Find field.

      The External Data Sources page displays.
   3. Select New External Data Source.

      The New External Data Source page displays.
2. Create a Docusign Agreements External Data Source:
   1. External Data Source: Enter `Docusign
      Agreements External Data Source`.
   2. Name: Enter
      `ADTickets_External_Data_Source`.
   3. Type: Select Salesforce Connect:
      Custom –dfsle.ADDataSourceProvider from the dropdown
      menu.
   4. Identity Type: Keep
      Anonymous.
   5. Authentication Protocol: Keep No
      Authentication.
   6. Select Save.

      ![A screenshot displays the External Data Sources page with settings turned on for a Docusign Agreements External Data Source.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/DAL_Agreement_Desk_Reporting_External_Data_Source_settings.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU3MTgsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.Tj-rc07vmhCKQgyPkzXoqdKIMwAaoS-8qsBA6NOlrGs&_LANG=enus "A screenshot displays the External Data Sources page with settings turned on for a Docusign Agreements External Data Source.")

   The External Data Sources page displays your new data
   source.
3. Validate and sync your data source:
   1. Select Validate and Sync.

      The Validate External Data Source page displays.
   2. Select the check box next to Docusign Agreement Desk
      Requests.
   3. Select Sync.

      ![A screenshot displays the External Data Sources page with Docusign Agreement Desk Requests and the Sync button highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/DAL_Agreement_Desk_Reporting_External_Data_Source_Validate_and_Sync.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU3MTgsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.Tj-rc07vmhCKQgyPkzXoqdKIMwAaoS-8qsBA6NOlrGs&_LANG=enus "A screenshot displays the External Data Sources page with Docusign Agreement Desk Requests and the Sync button highlighted.")

      The page refreshes.
4. Open the Docusign Agreement Desk Requests object:
   1. Enter and select External Objects in the Quick
      Find field.

      The External Objects page displays.
   2. Select Edit next to Docusign Agreement
      Desk Requests.

      The Docusign Agreement Desk Requests page
      displays.
   3. Optional Features: Select the check box next to
      Allow Reports.

      ![A screenshot displays the External Connection Detail section with the Allow Reports checkbox selected.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/DAL_Agreement_Desk_Reporting_External_Data_Source_Allow_Reports.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU3MTgsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.Tj-rc07vmhCKQgyPkzXoqdKIMwAaoS-8qsBA6NOlrGs&_LANG=enus "A screenshot displays the External Connection Detail section with the Allow Reports checkbox selected.")
   4. Select Save.

      The External Objects page displays.

You have completed the process. You have created an Agreement Desk for Salesforce external
object with reports enabled.
