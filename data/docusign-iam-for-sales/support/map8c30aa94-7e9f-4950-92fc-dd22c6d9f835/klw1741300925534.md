# Learn About the Overwrite If Field Value Exists Feature

Learn about the function of the Overwrite if field value exists feature in a
workflow.

You can use Workflow Builder to automate agreement actions from your Docusign account.

You can use a Writeback to Salesforce step to update Salesforce
records with information from your workflow. When task recipients enter data in
workflow fields, Docusign updates the information in the linked
Salesforce record fields.

The Writeback to Salesforce step contains an option: Overwrite if field value
exists.

Selecting this option replaces the Salesforce field's data with data entered by a workflow
participant.![A screenshot displays the Overwrite if field value exists option in a green highlighted box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Workflow_Builder_Writeback_to_Salesforce_overwrite_if_field_value_exists.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU0ODEsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.02cORPnaahSY4BAfFR5RL0hy5fXvuElkXVozAEkx32M&_LANG=enus "A screenshot displays the Overwrite if field value exists option in a green highlighted box.")

Example: You build a workflow with a Writeback to
Salesforce step. In the writeback step, you select Overwrite
if field value exists. for the Salesforce Amount
field.

A workflow recipient updates and signs a contract for an existing opportunity named Fontara.
The recipient updates the Fontara contract's Amount field from $20,000 to $25,000. When
the recipient completes the assignment, Docusign updates the Fontara
record's Amount field to $25,000.
