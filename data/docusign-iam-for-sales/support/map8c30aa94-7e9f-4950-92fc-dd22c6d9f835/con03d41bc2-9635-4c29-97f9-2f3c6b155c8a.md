# Getting Started With Docusign IAM for Sales: Salesforce

Learn to prepare your Salesforce organization for a Docusign IAM
for Sales implementation.

Important: This feature is only available to IAM for
Sales customers. Contact [Docusign support](https://support.docusign.com/s/?language=en_US&_gl=1*74elk4*_gcl_au*MTY5ODQyMjA2MC4xNzY5NjM1OTA1LjE5ODM3MTQzNjguMTc2OTYzNjIwNC4xNzY5NjM2MjA0) or [Docusign
sales](https://www.docusign.com/contact-sales) for more information.

Note:

Docusign has a number of products that you can use to create an
IAM for Sales experience:

Apps Launcher for Salesforce
:   Users can generate and send contracts for review, approval, and
    signature.

Workflow Builder for Salesforce
:   Users can select the action to start Workflow Builder workflows from records.

Agreement Manager for Salesforce
:   Users can view and manage Docusign requests from
    the component.

Agreement Desk for Salesforce
:   Users can view and act on Docusign agreements from
    the component.

You must prepare your Salesforce organization before using these products
together:

  Salesforce v8.4 or higher](https://login.salesforce.com/packaging/installPackage.apexp?p0=04tKg000000D3Vk).

- [Install pre-built
  workflows, templates, and fields with the IAM for Sales Solution](https://support.docusign.com/s/document-item?bundleId=map8c30aa94-7e9f-4950-92fc-dd22c6d9f835&topicId=tska61755e7-f8ba-4cfe-aad8-f02478b02dab.html&_LANG=enus "Learn to use the IAM for Sales Solution to install pre-built Docusign workflows that you can use in Salesforce.").
- [Add a Start with Docusign action to a layout](https://support.docusign.com/s/document-item?bundleId=map8c30aa94-7e9f-4950-92fc-dd22c6d9f835&topicId=tsk32e505bf-436a-47b1-b7f6-55e6ad614ec5.html&_LANG=enus "Learn to build a new Lightning web component action to start a Docusign workflow from a record.").
