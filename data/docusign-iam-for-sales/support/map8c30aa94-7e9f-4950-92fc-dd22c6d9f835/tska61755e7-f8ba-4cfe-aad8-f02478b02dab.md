# Use the IAM for Sales Solution to Install Pre-Built Docusign Workflows and Templates

Learn to use the IAM for Sales Solution to install pre-built Docusign workflows that you can use in Salesforce.

## Before you begin

Install Docusign Apps Launcher in your Salesforce organization.

You can use the Docusign IAM for Sales Solution to install pre-built Docusign resources in your Salesforce organization. These
resources quickly create the building blocks for common IAM for Sales use cases,
like creating and sending an NDA or MSA.

The solution includes the following resources:

- Document templates
- Agreement Desk statuses
- Docusign workflows
- Custom fields

These resources are functional and do not require additional
configuration.

Installing these resources gives you a quick way to implement IAM for Sales features
in your Salesforce organization. You can customize these resources to fit your exact
use cases.

Use this procedure to install the IAM for Sales Solution to your Salesforce
organization:

1. Docusign IAM for Sales Solution:
   Select Install.![A screenshot displays the Docusign Setup page with the IAM for Sales Solution Install button highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/IAM_Solutions_package_Install_button.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTUzNzYsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.p4y-PRXqR_Vy5qMtsqPIbE7yl9vmt3q8p-TlH1CMo0A&_LANG=enus "A screenshot displays the Docusign Setup page with the IAM for Sales Solution Install button highlighted.")

   The Docusign app center dialog box opens to the
   Details tab.
2. Configure settings on the Details tab:
   1. Salesforce connection: Choose the Salesforce
      connection that you set up.
   2. Select Install.![A screenshot displays the Docusign App Center Details tab with the Salesforce connection field highlighted and the Install button visible.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/IAM_Solutions_Package_Salesforce_Connection_and_Install.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTUzNzYsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.p4y-PRXqR_Vy5qMtsqPIbE7yl9vmt3q8p-TlH1CMo0A&_LANG=enus "A screenshot displays the Docusign App Center Details tab with the Salesforce connection field highlighted and the Install button visible.")

      The Install tab displays with an Installing
      progress bar. When the installation reaches 100%, the
      Publish tab displays.
3. Configure settings on the Publish tab:
   1. Select Publish All.![A screenshot displays the Docusign App Center Publish tab with the Publish All button visible.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/IAM_Solutions_Package_Publish_All_button.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTUzNzYsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.p4y-PRXqR_Vy5qMtsqPIbE7yl9vmt3q8p-TlH1CMo0A&_LANG=enus "A screenshot displays the Docusign App Center Publish tab with the Publish All button visible.")

      The Publish Workflows page displays.
   2. Select Authorize My Account.

      The Docusign Workflow Builder is requesting
      access dialog box displays.
   3. Select Allow Access.

      The dialog box closes. The Publish Workflows page
      displays.
   4. Select Publish.

      The Publish workflows page displays a spinning wheel icon. The
      Done tab displays.
4. Select Done.

   The Docusign app center dialog box
   closes.

You have completed the process. You have installed IAM Solutions
resources in your Docusign account and connected it to your Salesforce
organization. Salesforce users can start Docusign workflows from
Salesforce records.
