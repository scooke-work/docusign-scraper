# Create a Saved Search in the Docusign Agreement Requests Component

Learn how to create customized searches to retrieve an updated list of requests that
match the search criteria.

## Before you begin

Important: This feature is only available to IAM for
Sales customers. Contact [Docusign support](https://support.docusign.com/s/?language=en_US&_gl=1*74elk4*_gcl_au*MTY5ODQyMjA2MC4xNzY5NjM1OTA1LjE5ODM3MTQzNjguMTc2OTYzNjIwNC4xNzY5NjM2MjA0) or [Docusign
sales](https://www.docusign.com/contact-sales) for more information.

Agreement Desk is a central platform for streamlining your entire agreement process
within Docusign. You can use Agreement Desk to submit and manage
requests, negotiate and edit agreements, and use AI to automate contracts.

You can search and use filters to locate the requests that you want to see. You can
also save your searches to run again later.

Use this procedure to create a saved search in the Docusign
Agreement Requests component:

1. Open a record and [search for content in the Agreement Requests component](https://support.docusign.com/s/document-item?bundleId=map8c30aa94-7e9f-4950-92fc-dd22c6d9f835&topicId=tsk79fb118f-2fa3-4e60-965a-efb9edd7c2c8.html&_LANG=enus "Learn to search and filter the requests that display in the Docusign Agreement Requests component.").

   The component displays requests sent from this record and its child
   objects. The component does not display requests sent from parent objects or
   requests sent from your Docusign web account.
2. Select the Save search link.

   The New saved search dialog box displays.
3. Create a new saved search:
   1. New saved search: Enter a title for your saved search. You can enter up to 120 characters.
   2. Description: Enter text that describes the purpose of the search. You can enter up to 200 characters.
   3. Select Save.

      ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Agreement_Desk_for_Salesforce_Saved_Search_dialog_box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU2ODgsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.Czs-CLnSRKP_Wi2awEcgIsk0Th4F4OPkBVzgTtjnIRs&_LANG=enus)

      The dialog box closes. A message displays: Saved search created.

You have completed the process. You have created a saved search in the Agreement Requests
component. When you run the saved search, the component displays filtered results of all
requests sent from this record and its child objects. The results do not display
requests sent from parent objects or requests sent from your Docusign
web account.
