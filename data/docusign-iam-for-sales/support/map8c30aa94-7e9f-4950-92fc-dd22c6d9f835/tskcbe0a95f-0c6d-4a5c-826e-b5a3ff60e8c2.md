# Edit a Saved Search in the Docusign Agreement Requests Component

Learn how to edit a saved search from the Docusign Agreement Requests
component.

## Before you begin

Important: This feature is only available to IAM for Sales
customers. Contact [Docusign support](https://support.docusign.com/s/?language=en_US&_gl=1*74elk4*_gcl_au*MTY5ODQyMjA2MC4xNzY5NjM1OTA1LjE5ODM3MTQzNjguMTc2OTYzNjIwNC4xNzY5NjM2MjA0) or [Docusign
sales](https://www.docusign.com/contact-sales) for more information.

[Create a saved search in the
Docusign Agreement Requests component](https://support.docusign.com/s/document-item?topicId=tsk96d6f83e-3826-4c3f-9bba-6ac03bea79bb.html&_LANG=enus).

Agreement Desk is a central platform for streamlining your entire agreement process within Docusign. You can use Agreement Desk to submit and manage requests, negotiate and edit agreements, and use AI to automate contracts.

You can search and use filters to locate the requests that you want to see. You can also save your searches to run again later.

Use this procedure to edit a saved search in the Docusign Agreement
Requests component:

1. Open a record to the Agreement Requests component and select the
   Manage saved searches

   The Manage Saved Searches dialog box displays.
2. Select Manage saved searches.

   The Manage Saved searches dialog box displays.
3. Edit the saved search:
   1. Select the three-dot menu icon ![A screenshot displays three dots stacked vertically.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/A-screenshot-diplays-a-threedot-menu-icon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU2OTMsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.zu3WfuOqMwF-cMyxjjRC1MCKsLZKBcDb-07i4euEASQ&_LANG=enus "A screenshot displays three dots stacked vertically.") next to a saved search and choose Edit name and
      description.

      ![A screenshot displays the Manage Saved Searches dialog box with the three-dot menu displaying options to edit or delete a saved search.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Agreement_Desk_for_Salesforce_Edit_and_delete_saved_searches_menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU2OTMsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.zu3WfuOqMwF-cMyxjjRC1MCKsLZKBcDb-07i4euEASQ&_LANG=enus "A screenshot displays the Manage Saved Searches dialog box with the three-dot menu displaying options to edit or delete a saved search.")

      The Edit Saved Search dialog box displays.
   2. (Optional) New saved search: Edit the name of the saved search. You can enter up to 120 characters.
   3. (Optional) Description: Add or edit the saved search description. You can enter up to 200 characters.
   4. Select Update.

      The dialog box closes. A message displays: Search updated successfully.
   5. Select the X to close the Manage Saved Searches dialog box.

   The dialog box closes. The Docusign Agreement Requests component
   displays.

You have completed the process. You have edited a saved search from the Docusign Agreement Requests component.
