# Export and Import a Workflow Builder Workflow

Learn to export a Workflow Builder workflow and import it to either the same or another Docusign account.

You can export Docusign workflows and import them to either
the same account or another Docusign account.

Example 1: You build a workflow in your Docusign demo account. You want to add this workflow to your Docusign production account. You download the workflow from your
demo account and upload the .ZIP file to your production account. You validate that
all steps will work in production and publish the workflow.

Example 2: You build a workflow in your Docusign production account. You want to build a new workflow that
is similar to this workflow, but with a few differences. You download the workflow
and upload the .ZIP file back to your production account. You rename the imported
workflow, make some changes, and publish the workflow.

Use this procedure to export and import a workflow:

1. As a Docusign administrator, go to Agreements >  Workflow Builder Workflows.
2. Export your workflow:
   1.       Export.

      A dialog box displays: Export this
      workflow?
   2. Select Export.

   The page refreshes. The workflow configuration downloads to your device
   as a .ZIP file.
3. Import your workflow to the same Docusign account or another
   account:
   1. As a Docusign administrator, go to Agreements > Workflows.
   2. Select Import Workflow.

      The Creating your workflow dialog box displays.
   3. Select Import Workflow and select
      Upload
   4. Follow the prompts to upload the .ZIP file from your device.

   The page refreshes. The imported workflow configuration
   displays.
4. Select Review & Publish if you are ready to use the
   workflow. Select Save Draft if you are not ready to
   publish.

You have completed the process. You have exported and imported a
workflow. If you imported it into a different Docusign account, review
each step to ensure that the workflow functions.
