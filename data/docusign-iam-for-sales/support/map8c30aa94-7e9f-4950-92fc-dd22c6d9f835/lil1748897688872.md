# Learn About Agreement Manager for Salesforce User Access

Learn about granting user access to the Docusign Completed Agreements component on Salesforce layouts.

Important: This feature is only available to IAM for Sales customers. Contact [Docusign support](https://support.docusign.com/s/?language=en_US&_gl=1*74elk4*_gcl_au*MTY5ODQyMjA2MC4xNzY5NjM1OTA1LjE5ODM3MTQzNjguMTc2OTYzNjIwNC4xNzY5NjM2MjA0) or [Docusign sales](https://www.docusign.com/contact-sales) for more information.

To use Agreement Manager, Salesforce users need the following access:

- [An Apps Launcher user](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=dzl1586134206658.html&_LANG=enus).
- [An Intelligent Agreement Management (IAM) user with Agreement Manager access](https://support.docusign.com/s/document-item?bundleId=map8c30aa94-7e9f-4950-92fc-dd22c6d9f835&topicId=eut1712157553977.html&_LANG=enus "Read about user and group management tasks, such as managing agreement access and granting bulk upload permissions.").

Docusign administrators grant access to these user types. Approved users activate their access by selecting a Docusign button or component and entering their Docusign and Salesforce credentials.

## How the Docusign Completed Agreements component displays for different user types

User access dictates how the Docusign Completed Agreements component displays on a layout. Depending on the access granted, Docusign displays or hides the component on Salesforce layouts.

| Salesforce user access | How Docusign Agreements displays |
| --- | --- |
| Docusign Apps Launcher and Docusign IAM access | Users can see the Docusign Completed Agreements component and act on the agreements displayed. |
| Docusign Apps Launcher access only | Users see a blank component with a spinning wheel icon instead of the Docusign Completed Agreements component. |
| No Docusign user access | The Docusign Completed Agreements component does not display on records. |
