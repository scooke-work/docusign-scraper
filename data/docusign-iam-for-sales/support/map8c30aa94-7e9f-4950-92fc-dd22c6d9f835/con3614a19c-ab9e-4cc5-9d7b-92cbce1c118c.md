# Learn About the Docusign IAM for Sales Solution

Learn about using the IAM for Sales Solution to install pre-built Docusign workflows and templates in your
Salesforce organization.

You can install the Docusign IAM for Sales Solution in your Salesforce
organization. The solution installs a series of
pre-built Docusign resources in
your Salesforce organization. These resources
quickly create the building blocks for common IAM
for Sales use cases, like creating and sending an
NDA or MSA.
