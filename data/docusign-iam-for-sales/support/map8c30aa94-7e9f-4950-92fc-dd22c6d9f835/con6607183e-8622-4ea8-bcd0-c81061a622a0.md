# Adding Docusign Workflow Actions to Salesforce Layouts

Learn about adding Docusign workflow actions to Salesforce layouts
either manually, or with the IAM Launcher controls.

If you want to start Docusign workflows from Salesforce records, you
can add a Start with Docusign action to your layouts. The Start
with Docusign action displays Docusign workflows
that you can start from your records without logging on to Docusign.

You can use the following methods to add Start with Docusign actions
to your layout:

(Recommended) Add actions with IAM Launcher Setup controls
:   Choose layouts, select workflows, and add actions directly from the Docusign Setup page.
:   This is the simplest method and it works for almost all Salesforce objects.
:   Access the interface by going to Docusign Setup >  Manage Connections and the IAM Launcher >  Manage. ![A screenshot displays the Docusign Setup page with Manage Connections and the IAM Launcher highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/IAM_Launcher_Manage_Connections_and_the_IAM_Launcher_manage_button.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU0MDEsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.UOwvLG2mhVwmdTwGndSFxZ7dBxGiWBBh0Zb23CrF5sw&_LANG=enus "A screenshot displays the Docusign Setup page with Manage Connections and the IAM Launcher highlighted.")

[Add actions with Salesforce Object Manager](https://support.docusign.com/s/document-item?bundleId=rrf1583359212854&topicId=fji1583359148678.html&_LANG=enus)
:   Currently, you can't use IAM Launcher controls for some Salesforce objects,
    like Case and Campaign.
:   In these situations, use Object Manager to add actions to layouts.
