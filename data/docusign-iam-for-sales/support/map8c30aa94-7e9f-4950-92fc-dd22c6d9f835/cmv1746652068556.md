# Add the Docusign Completed Agreements Component to a Salesforce Layout

Learn to add the Docusign Completed Agreements component to a
Salesforce layout.

## Before you begin

Important: This feature is only available to IAM for
Sales customers. Contact [Docusign support](https://support.docusign.com/s/?language=en_US&_gl=1*74elk4*_gcl_au*MTY5ODQyMjA2MC4xNzY5NjM1OTA1LjE5ODM3MTQzNjguMTc2OTYzNjIwNC4xNzY5NjM2MjA0) or [Docusign
sales](https://www.docusign.com/contact-sales) for more information.

- [Purchase an Intelligent Agreement Management (IAM) for Sales application](https://www.docusign.com/intelligent-agreement-management?utm_source=bing&utm_medium=cpc&utm_campaign=GBL_XX_PRF_UPS_2410_SEMIAM&utm_term=docusign%20iam&campaignid=569731866&adgroupid=1178678316174006&location=84406&device=c&matchtype=e&extensionid=&creative=73667602015411&keyword=docusign%20iam&placement=&network=o&msclkid=3287ee64f2c01f69bcd686665f40e7da).
- [Install Apps
  Launcher version 8.4 or higher](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=ogj1586134254571.html&_LANG=enus).
- [Prepare Your Salesforce Organization to Use Agreement Manager](https://support.docusign.com/s/document-item?bundleId=map8c30aa94-7e9f-4950-92fc-dd22c6d9f835&topicId=vry1747859095666.html&_LANG=enus "Learn to prepare your Salesforce organization to use Docusign Agreement Manager.").

[Agreement Manager](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=adf1702945446135.html&_LANG=enus) is part of the Docusign IAM platform, powered by [Docusign
Iris](https://support.docusign.com/s/document-item?bundleId=fzd1707173174972&topicId=usy1712258503642.html). You can use Agreement Manager to store, manage, and analyze your agreements.

In Salesforce, you access Agreement Manager by adding the Docusign Agreements component to your layouts. Approved users can interact with Docusign agreements on your records without leaving Salesforce.

Use this procedure to add the Docusign Agreements component to a layout:

1. As a Salesforce administrator, open a record, select the gear icon, ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU0MjYsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.Ksb2y_--5z8Xori6-D7tWy_-UP1mJO1pyiBf3y4sS4I&_LANG=enus) and choose Edit Page from the dropdown menu.

   Note: You can add the Docusign Completed
   Agreements component to any part of your layout. The following instructions
   are recommendations, not requirements.
2. (Optional) Add a Docusign Agreements tab to your layout.
   1. Select Add Tab on the right navigation panel.

      A new Details tab displays.
   2. Select Details.

      The Tab Label dialog box opens.
   3. Tab Label: Select the Details dropdown menu and choose Custom.
   4. Custom Label: Enter a descriptive name, like Docusign Agreements.
   5. Select Done.

   ![A Salesforce layout displays a new tab with Custom as the type and Docusign Agreements as the custom label.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/A-Salesforce-layout-displays-a-new-tab-with-Custom-as-the-type-and-Docusign-Agreements-as-the-custom-label.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU0MjYsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.Ksb2y_--5z8Xori6-D7tWy_-UP1mJO1pyiBf3y4sS4I&_LANG=enus "A Salesforce layout displays a new tab with Custom as the type and Docusign Agreements as the custom label.")

   Your new tab displays in the right navigation panel.
3. Drag the Docusign Agreements component to the tab that you created and select Save.

   ![A screen capture demonstrates dragging the Docusign Completed Agreements component to a Salesforce layout and saving the action.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Agreement_Manager_for_Salesforce_add_Completed_Agreements_component_to_layout.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU0MjYsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.Ksb2y_--5z8Xori6-D7tWy_-UP1mJO1pyiBf3y4sS4I&_LANG=enus "A screen capture demonstrates dragging the Docusign Completed Agreements component to a Salesforce layout and saving the action.")
4. Activate the change to your layout:
   1. Select Activate.
   2. Select a tab to set your default options: Org Default, App Default, or App, Record Type, and Profile.
   3. Select Assign and choose options from the dialog box.
   4. Select Close.
5. Select the back arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/A-screenshot-of-the-Back-arrow-on-the-Salesforce-Edit-Page-settings-bar.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU0MjYsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.Ksb2y_--5z8Xori6-D7tWy_-UP1mJO1pyiBf3y4sS4I&_LANG=enus) to return to the record.

You have completed the process. You have added the Docusign Agreements component to a Salesforce layout. Users can interact with Agreement Manager content on Salesforce records.
