# Install the IAM for Salesforce Application (Salesforce Instructions)

Learn to install the Salesforce app in your Docusign account and connect it to your Salesforce organization.

## Before you begin

[Purchase a Docusign Intelligent Agreement Management (IAM) for Sales application](https://www.docusign.com/intelligent-agreement-management?utm_source=bing&utm_medium=cpc&utm_campaign=GBL_XX_PRF_UPS_2410_SEMIAM&utm_term=docusign%20iam&campaignid=569731866&adgroupid=1178678316174006&location=84406&device=c&matchtype=e&extensionid=&creative=73667602015411&keyword=docusign%20iam&placement=&network=o&msclkid=3287ee64f2c01f69bcd686665f40e7da).

Important: Use these instructions to install the application from your Salesforce organization. Follow this link if you want to [install the app from the Docusign App Center](https://support.docusign.com/s/document-item?bundleId=&qwr1706590641805&topicId=jxo1706295810553.html&_LANG=enus).

You can install the Salesforce application to connect your Docusign account with your Salesforce organization.

You can use that integration to build workflows that use Salesforce steps. These steps can read and write data, store documents, add merge fields, and more.

You can also add Docusign workflow actions to Salesforce layouts. Users can start Docusign workflows directly from Salesforce records.

Tip: Wait at least 30 minutes after installing the app before you configure a Workflow Builder for Salesforce step. Docusign indexes all of your Salesforce fields to make them available.

Use this procedure to install and connect the Salesforce app from the Docusign Setup page:

1. Manage Connections and the IAM Launcher: Select Manage.

   The docusign app center dialog box displays.
2. Install the Salesforce app:
   1. Select Install App.

      The Install and authorize "Salesforce" app dialog box displays.
   2. Select Install and Authorize.
   3. Select Continue

   The dialog box closes. A message displays: App successfully installed.
3. Connect your Docusign account to your Salesforce organization:

   Important: [Salesforce Government Cloud customers](https://help.salesforce.com/s/articleView?id=000386092&type=1) must use the Salesforce Production option.

   1. Select Connect Account.

      The Connect a Salesforce account dialog box displays.
   2. Select your Salesforce connection: Choose an option:

      - Salesforce Production (login.salesforce.com): Connect a Salesforce Production or Developer organization.
      - Salesforce Sandbox (test.salesforce.com): Connect to a Salesforce Sandbox organization.

      ![A screenshot displays the Connect a Salesforce account with Salesforce Production selected.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Workflow_Builder_Connect_a_Salesforce_Account.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTUzNjksInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.feA9ouqj50Z8-C9X2gMoEuU0ikJRHtiPLW7jQgMp2-w&_LANG=enus "A screenshot displays the Connect a Salesforce account with Salesforce Production selected.")
   3. Select Next.
   4. Salesforce Connection Name: Keep or change the connection name.
   5. Select Log In.

      The Salesforce login page displays.
4. Choose a Username: Select your Salesforce username from the list.

   The page refreshes. The docusign app center dialog box displays. Your connection displays with a Status of Processing.
5. Close the dialog box.

You have completed the process. You have installed the Salesforce app and connected it to Docusign. In five to ten minutes, the connection displays an Active status.
