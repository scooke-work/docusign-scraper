# Configure a Use Salesforce Files Step

Learn to add and configure a Use Salesforce Files step to add Salesforce files to
workflows.

## Before you begin

Important: This feature is only available to IAM for
Sales customers. Contact [Docusign support](https://support.docusign.com/s/?language=en_US&_gl=1*74elk4*_gcl_au*MTY5ODQyMjA2MC4xNzY5NjM1OTA1LjE5ODM3MTQzNjguMTc2OTYzNjIwNC4xNzY5NjM2MjA0) or [Docusign
sales](https://www.docusign.com/contact-sales) for more information.

Before you begin
:   - Required: Installation of the Salesforce app
      from Docusign App Center. Review the [Installing Extension
      Applications for Workflow Builder](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=bgb1697668738892.html&_LANG=enus) topic for more
      information.
    - Required: [Workflows Create
      permissions](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=gbb1696973048215.html&_LANG=enus) or be an account
      administrator.
    - Required: Create a workflow as described in [Create a Basic Workflow](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=uhm1697826856678.html&_LANG=enus). Add
      steps like [invitation](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=kfx1698167344720.html&_LANG=enus),

      more to gather recipients, documents, and merge fields for your

The Use Salesforce Files step adds Salesforce files
to workflows. Administrators open a record's Notes & Attachments and add
the record's ID to the Use Salesforce Files step. When
workflows start, Docusign adds selected files from that
record's Notes & Attachments. You can configure the following upload settings: All Files, File Name Contains, and Latest Document Only.

Example: You add a Use Salesforce Files
step to your workflow. You use the File Name Contains option
and enter the keywords `Contract Renewal`. Your workflow sends
an agreement to recipients. When recipients open their workflow task, they see a
file named Contract Renewal 2025 included in the agreement.

Use this procedure to configure a Use Salesforce Files step for
your workflow:

1. Add a Use Salesforce Files step as a Docusign administrator:
   1. Go to Agreements > Workflows and select Edit next to a workflow
      name.

      The workflow editor page displays.
   2. Select the Add a step
      ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Workflow_Builder_add_step_plus_sign.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU1MzMsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.LpPAD6khQ1QKXEOv5Ki05MVzvTqnYVQCm0SsmWvav_s&_LANG=enus) icon.

      The Add New Step dialog box displays.
   3. Search for steps: Enter and select `Use
      Salesforce Files`.

      The workflow editor page displays with the Use Salesforce Files
      step added.
2. Name your Use Salesforce Files step:
   1. Select Configure on the Use Salesforce Files
      step.

      The configuration dialog box displays.
   2. Step Name: Keep or edit the name. It is helpful
      to add the name of the Salesforce object, like `Account - Use
      Salesforce Files`.
3. Where are these files located?
   1. Select connection: Select the field and choose a
      Salesforce organization that you connected to Docusign. Multiple options display if you connect more than one Salesforce
      organization.
   2. (Optional) Add New Connection: Select this
      button if you want to [connect more than one Salesforce
      organization](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=ogk1719256147008.html&_LANG=enus).
   3. (Optional) Manage Connections: Select this
      button if you want to [add, edit, refresh, or delete a
      Salesforce connection](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=uaw1719259450391.html&_LANG=enus).
   4. Select Record ID: Select a field from the start
      step or another step in this workflow:

      - Manual entry: Add text for your new
        variable and select Save.

        Note: The name that you enter must exactly
        match an existing field. For example, the rule fails if you
        enter CompanyName for the
        Company\_Name
        variable.
      - Select a field from the start step or another step in this
        workflow:

        Note: Steps like Read
        from HubSpot and Read from
        Salesforce display fields from up to five
        records. If more than five records are present, Salesforce
        logic decides which five to display. Selecting fields from
        multiple records helps you automate processes like mapping
        multiple Salesforce contacts to multiple Docusign participants.![A screenshot displays Read from Salesforce fields highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Workflow_Builder_for_Salesforce_Read_Workflow_field_selections.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU1MzMsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.LpPAD6khQ1QKXEOv5Ki05MVzvTqnYVQCm0SsmWvav_s&_LANG=enus "A screenshot displays Read from Salesforce fields highlighted.")

        You can use the fields that you select here in
        later steps like Collect Data with
        Webforms or Send Documents for
        Signature.
   5. Select Next.

      ![Screenshot showing a connection highlighted for a Salesforce connection and a record ID path.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Workflow_Builder_for_Salesforce_Use_Salesforce_Files_Where_are_these_files_located.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU1MzMsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.LpPAD6khQ1QKXEOv5Ki05MVzvTqnYVQCm0SsmWvav_s&_LANG=enus "Screenshot showing a connection highlighted for a Salesforce connection and a record ID path.")
4. Which files do you want to import?

   Note: You can import a maximum of ten files or 100 MB.

   Filter files by: Select an option:
   - All Files: Upload all documents from the
     record.
   - File Name Contains: Upload documents that match
     file name keywords.
   - Latest Document Only: Upload the most recently
     added document from the record.
5. Select Apply.

   ![A screenshot displays file import filters for name and a notice about the ten file import limit.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Workflow_Builder_for_Salesforce_Use_Salesforce_Files_Which_files_do_you_want_to_import.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU1MzMsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.LpPAD6khQ1QKXEOv5Ki05MVzvTqnYVQCm0SsmWvav_s&_LANG=enus "A screenshot displays file import filters for name and a notice about the ten file import limit.")

   The Use Salesforce Files dialog box closes. The page displays your new step.

You have completed the process. You have built a Use Salesforce
Files step in your workflow. When a workflow starts this step, Docusign adds the most recent file from the specified record.
