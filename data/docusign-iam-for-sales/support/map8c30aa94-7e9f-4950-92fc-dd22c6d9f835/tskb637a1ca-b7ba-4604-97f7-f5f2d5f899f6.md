# Add Docusign Workflow Actions to Layouts (Manual Controls)

Learn to add a Start with Docusign action to a page layout to start
workflows.

## Before you begin

Important:

- This feature is only available to IAM for Sales customers. Contact [Docusign support](https://support.docusign.com/s/?language=en_US&_gl=1*74elk4*_gcl_au*MTY5ODQyMjA2MC4xNzY5NjM1OTA1LjE5ODM3MTQzNjguMTc2OTYzNjIwNC4xNzY5NjM2MjA0) or [Docusign
  sales](https://www.docusign.com/contact-sales) for more information.
- Docusign recommends using the [IAM Launcher Setup Controls](https://support.docusign.com/s/document-item?bundleId=map8c30aa94-7e9f-4950-92fc-dd22c6d9f835&topicId=tska9d68585-f524-49a4-8898-496b6575d036.html&_LANG=enus "Learn to use the IAM Launcher Setup interface to add Docusign workflow actions to Salesforce layouts.") to add workflow actions. Use these
  instructions if the IAM Launcher Setup controls fail.

[Build a Start with Docusign
action](https://support.docusign.com/s/document-item?bundleId=map8c30aa94-7e9f-4950-92fc-dd22c6d9f835&topicId=tsk32e505bf-436a-47b1-b7f6-55e6ad614ec5.html&_LANG=enus "Learn to build a new Lightning web component action to start a Docusign workflow from a record.").

If you want to start Docusign workflows from Salesforce records,
you can add a Start with Docusign action to your layouts. The
Start with Docusign action displays workflows that you can start
from your records. Users can start workflows from a record without logging on to
your Docusign account.

Use this procedure to add a Start with Docusign action to a
layout:

1. Access an object's page layout as a Salesforce administrator:
   1. Select the gear icon ![A screenshot displays a six-point gear with a lightning bolt inside it.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/salesforce-gear_.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU0MTYsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.GmTEyRox3CkbE0WCgfKgM7etFAQVamp4T4rgnsMt70k&_LANG=enus "A screenshot displays a six-point gear with a lightning bolt inside it.") and choose Setup.
   2. Enter and select Object Manager in the Quick Find
      field.
   3. Select a Salesforce object in the left column and select
      Page Layouts on the Details page.
   4. Select a layout from the Page Layouts
      column.

   The object's page layout displays.
2. Find the Start with Docusign action:
   1. Select Mobile & Lightning Actions in the
      upper left of the layout menu.
   2. Salesforce Mobile and Lightning Experience
      Actions: Select the override the predefined
      actions link. If the link does not display, you do not
      need to take this action.

      ![A screenshot displays the Salesforce Mobile and Lightning actions section with the override the predefined actions link highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Mobile_and_Lightning_actions_override_the_predefined_actions_link.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU0MTYsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.GmTEyRox3CkbE0WCgfKgM7etFAQVamp4T4rgnsMt70k&_LANG=enus "A screenshot displays the Salesforce Mobile and Lightning actions section with the override the predefined actions link highlighted.")
   3. Enter Start with Docusign in the Quick
      Find field.
3. Add the Start with Docusign action to the page layout:
   1. Drag the Start with Docusign action to the
      Salesforce Mobile and Lightning Experience
      Actions section.
   2. Select Save.

   ![A screen capture displays the process of dragging the Start with Docusign action to the Salesforce Mobile and Lightning Experience Actions section.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Add_Start_with_Docusign_button_to_layout.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU0MTYsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.GmTEyRox3CkbE0WCgfKgM7etFAQVamp4T4rgnsMt70k&_LANG=enus "A screen capture displays the process of dragging the Start with Docusign action to the Salesforce Mobile and Lightning Experience Actions section.")

   The Saving Layout message displays. The Page
   Layouts page opens.

You have completed the process. You have added a Start with Docusign
action to a Salesforce layout.
