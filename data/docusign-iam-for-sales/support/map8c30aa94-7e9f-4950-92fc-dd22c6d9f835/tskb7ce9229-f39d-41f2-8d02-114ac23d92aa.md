# Turn on Agreement Manager for Salesforce Notifications

Learn to set notifications for agreements displayed in the Docusign
Completed Agreements component.

## Before you begin

Important: This feature is only available to IAM for
Sales customers. Contact [Docusign support](https://support.docusign.com/s/?language=en_US&_gl=1*74elk4*_gcl_au*MTY5ODQyMjA2MC4xNzY5NjM1OTA1LjE5ODM3MTQzNjguMTc2OTYzNjIwNC4xNzY5NjM2MjA0) or [Docusign
sales](https://www.docusign.com/contact-sales) for more information.

You can turn on notifications for the agreements displayed in the Docusign Completed Agreements component. Notifications help you
stay informed about upcoming milestones for individual agreements. You can manage
notifications by interacting with agreements on the Docusign
Completed Agreements component interface. Notification settings give you control
over the email notifications that you receive.

Use this procedure to manage agreement notifications in the Docusign Completed Agreements component:

1. Open a record and look for the Docusign Agreements
   component.

   Note: A Salesforce administrator may have placed the
   component on a tab labeled Docusign Agreements or
   something similar. The component features the word
   Completed at the top.
2. Hover your cursor over the agreement that you want to track and select the
   Set a notification (bell) icon:

   Note: If a notification already exists, the
   Manage notifications (bell) icon displays. Select the bell icon to update existing notifications for the
   agreement.

   ![A screenshot of the Completed view that highlights the Set a notification option in the agreement list table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Navigator_for_Salesforce_notification_bell_icon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU3ODYsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.CYvcZ9ZKcOwnwbrgv3NXyU10w5woosaw8TJ5_CqVkdA&_LANG=enus "A screenshot of the Completed view that highlights the Set a notification option in the agreement list table.")

   The Notifications function opens. The Renewal Notice
   Date and Expiration Date notifications
   are turned on by default.

   Note: When you
   enable the notification, you are designated as the recipient and all
   frequency options are selected.
3. (Optional) + Add: Add notifications for this agreement:
   1. Select the bell icon and select + Add from the
      Notifications dialog box that displays.

      ![A screenshot of the Completed view that highlights the Add notification option in the agreement list table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Navigator_for_Salesforce_notification__Add_icon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU3ODYsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.CYvcZ9ZKcOwnwbrgv3NXyU10w5woosaw8TJ5_CqVkdA&_LANG=enus "A screenshot of the Completed view that highlights the Add notification option in the agreement list table.")

      The Add notification dialog box displays.
   2. Notification type: Select an option from the dropdown menu.
   3. Select Save.
4. (Optional) Edit: Select the link to customize the
   frequency and recipients for the notifications:
   1. Select the bell icon and select Edit next to an
      entry in the Notifications dialog box.

      The Edit notification dialog box displays.
   2. Expand the frequency dropdown, like Remind me
      prior to expiration date, and choose when you
      want to receive notifications.
   3. Add a user: Specify the users that should also receive these notifications. Type a
      name or email address in the field and
      select a name from the dropdown list.

      Note: The Self
      option is selected by default and cannot be removed.
   4. Select Save.

      ![A screenshot of the Edit notification dialog box that shows configuration options for the selected notification.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Document-level-notifications_Edit-notification-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU3ODYsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.CYvcZ9ZKcOwnwbrgv3NXyU10w5woosaw8TJ5_CqVkdA&_LANG=enus "A screenshot of the Edit notification dialog box that shows configuration options for the selected notification.")

   The Edit notification dialog box
   closes.
5. Use the Manage notifications (bell) icon to update the
   document's notifications at any time, including:

   - Adding new notifications
   - Updating existing notification settings
   - Managing the distribution list
   - Deleting the notification

You have completed the process. You have added or edited agreement notifications in the Docusign Completed Agreements component. You receive notifications
when agreement milestones occur.
