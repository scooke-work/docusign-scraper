# Open a Request from the Agreement Requests Component

Learn to open a request from the Agreement Requests component to view or act
on.

## Before you begin

Important: This feature is only available to IAM for Sales
customers. Contact [Docusign support](https://support.docusign.com/s/?language=en_US&_gl=1*74elk4*_gcl_au*MTY5ODQyMjA2MC4xNzY5NjM1OTA1LjE5ODM3MTQzNjguMTc2OTYzNjIwNC4xNzY5NjM2MjA0) or [Docusign
sales](https://www.docusign.com/contact-sales) for more information.

[Agreement Desk for Salesforce User Access](https://support.docusign.com/s/document-item?bundleId=map8c30aa94-7e9f-4950-92fc-dd22c6d9f835&topicId=ref9c5cd52b-33e2-4b25-a821-a666d4bd222e.html&_LANG=enus "Learn about granting Salesforce user access to the Docusign Agreement Requests component.")

Agreement Desk is a central platform for streamlining your entire agreement process
within Docusign. You can use Agreement Desk to submit and manage
requests, negotiate and edit agreements, and use AI to automate contracts. When you
add the Agreement Requests component to your layouts, users can review, manage,
sign, and send agreements without logging on to Docusign.

You can also open and act on requests from the Agreement Requests component.

Use this procedure to open a request:

1. Open a record and look for the Agreement Requests component.

   Note: A Salesforce administrator may have placed the
   component on a tab labeled Agreement Requests or
   something similar. The component features the word
   Requests at the top.
2. Search Request Titles: Enter search terms in the
   field.

   ![The Requests section of a Salesforce record displays a highlighted search field with Legal Request entered.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Agreement_Desk_for_Salesforce_search_results.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU3MDMsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.unSDda2GD6lpMLzm70zcgxs5bbTpqLtpLCMlf5iDcMc&_LANG=enus "The Requests section of a Salesforce record displays a highlighted search field with Legal Request entered.")
3. Press [Enter] to see all results, or select a result
   from the dropdown menu that opens beneath.

   The component displays requests sent from this record and its child
   objects. The component does not display results for parent objects, or Agreement
   Desk requests sent from your Docusign web
   account.
4. Select a request from the displayed list.

   ![A screenshot displays the Requests section of a Salesforce record with an Agreement Desk request highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Agreement_Desk_for_Salesforce_request_selected.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU3MDMsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.unSDda2GD6lpMLzm70zcgxs5bbTpqLtpLCMlf5iDcMc&_LANG=enus "A screenshot displays the Requests section of a Salesforce record with an Agreement Desk request highlighted.")

   The request opens in a new browser tab.
5. Use any control to view or act on the request:
   - Overview: View the request's activity feed. You can
     also send an email about the request.
   - Details: View the request's details, like request
     type, requestor name, owner, Salesforce account, and so on.
   - Documents: Preview, download, edit, upload a new
     version, and see the version history of all documents submitted with the
     request. You can also add content to the request by uploading or generating
     documents.
   - Approvals: View all approvals for this
     request.
   - Envelopes: View all envelopes sent for signature from
     this request.
   - Follow: Receive notifications about this request as
     it moves through the review and approval process.
   - Share: Send an email about this request to a user or
     group. You can also copy a link to share this request through another
     messaging platform.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Agreement_Desk_for_Salesforce_open_AD_request.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU3MDMsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.unSDda2GD6lpMLzm70zcgxs5bbTpqLtpLCMlf5iDcMc&_LANG=enus)

You have completed the process. You have opened a request to view or act on.
