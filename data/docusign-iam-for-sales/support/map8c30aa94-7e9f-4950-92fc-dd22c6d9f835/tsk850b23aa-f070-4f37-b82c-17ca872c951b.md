# Step 2: Create an Agreement Desk for Salesforce Report

Learn to build a custom Salesforce report that displays your Docusign
Agreement Desk requests.

## Before you begin

Important: This feature is only available to IAM for Sales
customers. Contact [Docusign support](https://support.docusign.com/s/?language=en_US&_gl=1*74elk4*_gcl_au*MTY5ODQyMjA2MC4xNzY5NjM1OTA1LjE5ODM3MTQzNjguMTc2OTYzNjIwNC4xNzY5NjM2MjA0) or [Docusign
sales](https://www.docusign.com/contact-sales) for more information.

[Create an Agreement Desk for
Salesforce external object](https://support.docusign.com/s/document-item?bundleId=map8c30aa94-7e9f-4950-92fc-dd22c6d9f835&topicId=tsk2e317e21-7fca-4946-a4eb-c766238b2cb3.html&_LANG=enus "Learn to create an Agreement Desk for Salesforce external object to turn on reporting.").

Now that you have created an Agreement Desk external object, you can build a report to
display Agreement Desk requests.

Use this procedure to build an Agreement Desk for
Salesforce report:

1. Open the Salesforce Create Report page:
   1. Select the App Launcher grid ![A screenshot displays a square icon made of nine colored dots.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/app-launcher-grid.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU3MjIsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.Q3c4SyDlHuALNr80jO6YXGNNu1GPw4rXST_8gK-s8do&_LANG=enus "A screenshot displays a square icon made of nine colored dots.").
   2. Enter and select `Reports` in the *Search apps
      and items* field.
   3. Select New Report.

      The Create Report dialog box displays.
2. Search Report Types: Enter `Docusign Agreement
   Desk Requests` in the field.
3. Select the Docusign Agreement Desk Requests link that
   displays.
4. Select Start Report.

   The Reports page displays.
5. Create an outline for your new Agreement Desk Requests report:
   1. Select the Outline tab.
   2. GROUP ROWS: Select options from the dropdown
      menu.
   3. GROUP COLUMNS: Select options from the dropdown
      menu.
   4. Columns: Select options from the dropdown
      menu.
6. Create filters for your new Agreement Desk Requests report:
   1. Select the Filters tab.
   2. Filters: Select an option from the dropdown
      menu.

      The Filter By dialog box displays.
   3. Operator: Select options from the dropdown
      menu.
   4. Complete your filter and select Apply.

      ![A screenshot displays a Salesforce report with outline and filter tabs highlighted and filters displayed.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/DAL_Agreement_Desk_Reporting_Outline_and_Filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU3MjIsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.Q3c4SyDlHuALNr80jO6YXGNNu1GPw4rXST_8gK-s8do&_LANG=enus "A screenshot displays a Salesforce report with outline and filter tabs highlighted and filters displayed.")
   5. Select Save and Run.

   The Save Report dialog box displays.
7. Enter details for your new report:
   1. Report Name: Enter a descriptive name, like
      `Docusign Agreement Desk Requests`.
   2. Report Unique Name: Keep or edit the name.
   3. (Optional) Report Description: Enter a
      description of the report's function.
   4. Folder: Keep the folder or select
      Select Folder to change locations.
   5. Select Save.

      A message displays: Report "Docusign Agreement Desk
      Requests" was saved. The page refreshes with report
      results displayed.

You have completed the process. You have completed, saved, and run a new Agreement Desk
report. You can change filters to display different data sets.
