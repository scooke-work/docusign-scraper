# Search and Filter in the Agreement Requests Component

Learn to search and filter the requests that display in the Docusign
Agreement Requests component.

## Before you begin

Important: This feature is only available to IAM for
Sales customers. Contact [Docusign support](https://support.docusign.com/s/?language=en_US&_gl=1*74elk4*_gcl_au*MTY5ODQyMjA2MC4xNzY5NjM1OTA1LjE5ODM3MTQzNjguMTc2OTYzNjIwNC4xNzY5NjM2MjA0) or [Docusign
sales](https://www.docusign.com/contact-sales) for more information.

Agreement Desk is a central platform for streamlining your entire agreement process
within Docusign. You can use Agreement Desk to submit and manage
requests, negotiate and edit agreements, and use AI to automate contracts.

You can also search and use filters to locate the requests that you want to see.

Use this procedure to search for and filter the requests that display in the Docusign Agreement Requests component:

1. Open a record and look for the Docusign Agreement
   Requests component.

   Note: A Salesforce administrator may have placed the
   component on a tab labeled Agreement Desk Requests or
   something similar. The component features the word
   Requests at the top.
2. Search Request Titles: Enter search terms in the
   field.

   ![The Requests section of a Salesforce record displays a highlighted search field with Legal Request entered.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Agreement_Desk_for_Salesforce_search_results.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU2ODMsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.VeGYDXgKgtYqR2zwvjyW67biB53lI_W0pm4a8RsmHaQ&_LANG=enus "The Requests section of a Salesforce record displays a highlighted search field with Legal Request entered.")
3. Press [Enter] to see all results, or select a result
   from the dropdown menu that opens beneath.

   The component displays requests sent from this record and its child
   objects. The component does not display results for parent objects, or Agreement
   Desk requests sent from your Docusign web
   account.
4. (Optional) Use the filters to refine the list of search results. Select a
   filter, choose a value, and select Apply.

   For example, use the Request Type filter to locate a
   New Customer Intake request:

   ![A screenshot displays the Request Type filter highlighted with the New Customer Intake check box selected.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Agreement_Desk_for_Salesforce_Filter_Request_Titles.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU2ODMsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.VeGYDXgKgtYqR2zwvjyW67biB53lI_W0pm4a8RsmHaQ&_LANG=enus "A screenshot displays the Request Type filter highlighted with the New Customer Intake check box selected.")

   New Customer Intake requests for this object and its child objects display in the
   Agreement Requests component.
5. (Optional) Continue to apply filters to refine the list of requests that are displayed.
   Select All Filters to access additional filters:
   - Created At: Select from pre-defined date ranges or
     enter a custom date range.
   - Due Date: Select from requests containing a
     specific due date.
   - Last Activity At: Select from a pre-defined date
     range or enter a custom date range.
   - Owner: Select a name from the list of request owners.
     The owner is assigned by the request submitter.
   - Status: Select from specific statuses for open and
     closed requests. The options listed include custom statuses.
   - Status Type: Select Open or
     Closed.
   - Submitter: Select a name from the list of users who
     created the requests shown in the dashboard.
   - Request Type: Select from the request types listed to
     narrow the results to only those types.

You have completed the process. You have used filters with your
search to display only the requests that you want to see.
