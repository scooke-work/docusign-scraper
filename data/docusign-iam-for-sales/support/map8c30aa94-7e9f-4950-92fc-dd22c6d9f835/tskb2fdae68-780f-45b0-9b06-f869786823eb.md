# Remove a Workflow from the Start with Docusign Action Menu

Learn to remove a Docusign workflow from a Start with Docusign action
menu.

## Before you begin

Important: This feature is only available to IAM for
Sales customers. Contact [Docusign support](https://support.docusign.com/s/?language=en_US&_gl=1*74elk4*_gcl_au*MTY5ODQyMjA2MC4xNzY5NjM1OTA1LjE5ODM3MTQzNjguMTc2OTYzNjIwNC4xNzY5NjM2MjA0) or [Docusign
sales](https://www.docusign.com/contact-sales) for more information.

[Add a Start with Docusign action to a layout](https://support.docusign.com/s/document-item?bundleId=map8c30aa94-7e9f-4950-92fc-dd22c6d9f835&topicId=tskb637a1ca-b7ba-4604-97f7-f5f2d5f899f6.html&_LANG=enus "Learn to add a Start with Docusign action to a page layout to start workflows.").

If you want to start Docusign workflows from Salesforce records,
you can add a Start with Docusign action to your layouts. The
action displays Docusign workflows that you can start from your
records. You can add and remove the workflows that display in the action menu. You
can also search for workflows to add to the action menu.

Only administrators can remove workflows from the action menu.

Use this procedure to remove a workflow from a Start with Docusign
action menu:

1. Choose your access method:
   - From the Docusign Setup page:
     1. [Access the Docusign Setup
        page as a Salesforce administrator](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=etv1586134206174.html&_LANG=enus)
     2. Manage Connections and the IAM Launcher: Select
        Manage.
     3. Select the IAM Launcher Setup tab.
     4. Select the Manage button next to a layout
        (Account, Opportunity, and so on.)

        The Start with Docusign menu displays.
   - From a Salesforce record: Select the Start with
     Docusign action in the upper right of the record.

     The Start
     with Docusign menu displays.
2.    workflow from the dropdown menu.

   ![A screenshot displays the three-dot menu and the Remove this workflow option highlighted in the Start with Docusign action menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Workflow_Builder_for_Salesforce_action_menu_Remove_this_workflow.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU1NzIsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.I5EdJ7msCd2aSbR8euYdC841vrrFm0bZQ7_-oDsPt08&_LANG=enus "A screenshot displays the three-dot menu and the Remove this workflow option highlighted in the Start with Docusign action menu.")The Remove workflow dialog box displays.
3. Select Remove

   Note: Docusign removes the workflow tile
   from the menu, but does not delete the workflow. You can re-add the workflow
   tile to the menu later.

   The Remove workflow dialog box closes. A message displays:
   (name of workflow) removed.
4. Select the X icon to close the action menu dialog
   box.

You have completed the process. You have removed a workflow from the
Start with Docusign action menu.
