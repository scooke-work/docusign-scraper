# Add the Docusign Agreement Requests Component to a Salesforce Layout

Learn to add the Docusign Agreement Requests component to a Salesforce
page layout.

## Before you begin

Important: This feature is only available to IAM for
Sales customers. Contact [Docusign support](https://support.docusign.com/s/?language=en_US&_gl=1*74elk4*_gcl_au*MTY5ODQyMjA2MC4xNzY5NjM1OTA1LjE5ODM3MTQzNjguMTc2OTYzNjIwNC4xNzY5NjM2MjA0) or [Docusign
sales](https://www.docusign.com/contact-sales) for more information.

- [Purchase a Docusign Intelligent Agreement
  Management (IAM) for Sales application](https://www.docusign.com/intelligent-agreement-management?utm_source=bing&utm_medium=cpc&utm_campaign=GBL_XX_PRF_UPS_2410_SEMIAM&utm_term=docusign%20iam&campaignid=569731866&adgroupid=1178678316174006&location=84406&device=c&matchtype=e&extensionid=&creative=73667602015411&keyword=docusign%20iam&placement=&network=o&msclkid=3287ee64f2c01f69bcd686665f40e7da).
- [Prepare Your Salesforce Organization to Use Agreement Desk](https://support.docusign.com/s/document-item?bundleId=map8c30aa94-7e9f-4950-92fc-dd22c6d9f835&topicId=tsk87e9ead6-6ace-475f-b10f-0b41ecc882e1.html&_LANG=enus "Learn to prepare your Salesforce organization to use Docusign Agreement Desk.").

Docusign Agreement Desk is part of the Docusign IAM platform, powered by [Docusign Iris](https://support.docusign.com/s/document-item?bundleId=fzd1707173174972&topicId=usy1712258503642.html).

Agreement Desk is a
central platform for streamlining your entire agreement process within Docusign. You can use Agreement Desk to submit and manage
requests, negotiate and edit agreements, and use AI to automate
contracts.

Many Salesforce users prefer to act on their agreements entirely
from Salesforce. When you add the Agreement Requests component to your layouts,
users can review, manage, sign, and send agreements without logging on to Docusign.

Use this procedure to add the component to a
layout:

1. As a Salesforce administrator, open a record, select the gear icon, ![A screenshot displays a gear icon with a lightning bolt inside.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU0MzEsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.BmnMzOnruxrsfK_28WW1ViT2eDYRa5XFoJyj6RSA3d0&_LANG=enus "A screenshot displays a gear icon with a lightning bolt inside.") and choose
   Edit Page from the dropdown menu.

   Note: You can add the Docusign Agreement
   Requests component to any part of your layout. The following
   instructions are recommendations, not requirements.
2. (Optional) Add an Agreement Requests tab to your layout.
   1. Select Add Tab on the right navigation
      panel.

      A new Details tab displays.
   2. Select Details.

      The Tab Label dialog box opens.
   3. Tab Label: Select the
      Details dropdown menu and choose
      Custom.
   4. Custom Label: Enter a descriptive name, like
      Docusign Agreement Desk.
   5. Select Done.

   ![A screenshot displays a custom tab with Docusign Agreement Desk in the label field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Agreement_Desk_Tab_Salesforce.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU0MzEsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.BmnMzOnruxrsfK_28WW1ViT2eDYRa5XFoJyj6RSA3d0&_LANG=enus "A screenshot displays a custom tab with Docusign Agreement Desk in the label field.")

   Your new tab displays in the right navigation panel.
3. Drag the Docusign Agreement Requests component to the
   tab that you created and select Save.

   ![A screen capture displays dragging the Docusign Agreement Requests component to a Salesforce layout.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/Agreement_Desk_for_Salesforce_add_Agreement_Requests_component_to_layout.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU0MzEsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.BmnMzOnruxrsfK_28WW1ViT2eDYRa5XFoJyj6RSA3d0&_LANG=enus "A screen capture displays dragging the Docusign Agreement Requests component to a Salesforce layout.")
4. Activate the change to your layout:
   1. Select Activate.
   2. Select a tab to set your default options: Org
      Default, App Default, or
      App, Record Type, and Profile.
   3. Select Assign and choose options from the dialog
      box.
   4. Select Close.
5. Select the back arrow ![A screenshot displays a blue box with a left-facing arrow icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/map8c30aa94-7e9f-4950-92fc-dd22c6d9f835/page/images/A-screenshot-of-the-Back-arrow-on-the-Salesforce-Edit-Page-settings-bar.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU0MzEsInNoZWFmIjoibWFwOGMzMGFhOTQtN2U5Zi00OTUwLTkyZmMtZGQyMmM2ZDlmODM1In0.BmnMzOnruxrsfK_28WW1ViT2eDYRa5XFoJyj6RSA3d0&_LANG=enus "A screenshot displays a blue box with a left-facing arrow icon.") to return to the record.

You have completed the process. You have added the
Docusign Agreement Requests component to a Salesforce layout.
Users can interact with Docusign Agreement Desk content on Salesforce
records.
