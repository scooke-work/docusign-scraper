# Workflow Builder for Salesforce Advanced Features

Learn about how to use advanced features in Salesforce workflow steps.

With [Workflow Builder](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=dnx1696972415150.html&_LANG=enus), you can connect Docusign workflow steps like Web Forms and eSignature templates to
third-party extensions. You can create workflows to automate your agreement process,
including reading data from and writing data to Salesforce records.

After installing and configuring the IAM for Salesforce app, you can use advanced features to
customize how the Salesforce app functions.

You can use the following advanced features when using the app:

- Update the ​Salesforce application
- Edit the name of a Salesforce organization connected to the Salesforce
  app
- Reconnect a Salesforce organization to the Salesforce app
- Refresh the connection to your Salesforce app
- Duplicate a workflow
- Export and import a workflow
- Delete the connection to a Salesforce organization from the Salesforce
  app
