# Access Required to Start Docusign Workflows from Salesforce

Learn about the permissions that you must add for users to start Docusign workflows in
Salesforce.

You can use [Workflow Builder](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=dnx1696972415150.html&_LANG=enus) to automate agreement
actions from your Docusign account.

Salesforce Flow is a workflow builder that you can use to automate agreements from
your Salesforce organization.

You can link Salesforce flows to Docusign workflows by adding a **Start Docusign
Workflow Builder** workflow step to your flow.

All users who want to start Docusign workflows from Salesforce must have proper
permissions.

Follow this link to learn how to [assign the proper
Docusign and Salesforce user permissions](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=bmk1744912007661.html&_LANG=enus "Learn to prepare your Salesforce users so that they can start Docusign workflows from Salesforce.").

## Required Docusign permissions

You must have one of the following permissions in your Docusign
account to use Docusign workflows:

| Docusign permission | Description |
| --- | --- |
| **Docusign administrator** | Authorizes applications for Workflow Builder and enables users to create and run workflows by granting required user permissions. They are granted full workflow permissions by default. |
| Workflow creator | An eSignature user who creates, edits, and publishes their own workflows. They can also use any published workflow to start an agreement process. |
| Preparer | Runs published workflows to start the agreement process and monitors the progress of running instances. They have access to use any published workflow on the account to start an agreement process. |
| Participant | Users who participate in agreement workflows. They do not require Docusign credentials. |

## Required Salesforce permissions

Users belonging to any of the following permission sets can start workflows from
Salesforce:

| Permissions | Description |
| --- | --- |
| User setting: Flow User | Provides access to Salesforce flows. |
| Permission set: **Docusign Administrator** | Provides administrative access to Docusign Apps Launcher. |
| Permission set: **Docusign User** | Provides access to eSignature in Docusign Apps Launcher. |
