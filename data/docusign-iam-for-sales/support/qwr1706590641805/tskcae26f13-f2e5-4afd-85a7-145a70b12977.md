# Start a Docusign Workflow from a Salesforce Record

Learn to start a Docusign workflow from a Start with Docusign action.

## Before you begin

Important: This feature is only available to IAM for
Sales customers. Contact [Docusign support](https://support.docusign.com/s/?language=en_US&_gl=1*74elk4*_gcl_au*MTY5ODQyMjA2MC4xNzY5NjM1OTA1LjE5ODM3MTQzNjguMTc2OTYzNjIwNC4xNzY5NjM2MjA0) or [Docusign
sales](https://www.docusign.com/contact-sales) for more information.

You can use the Start with Docusign action to start workflows
from Salesforce records.

Both administrators and standard users can start workflows from the action menu.

Use this procedure to start a workflow from a Start with Docusign
action:

Note: The action menu only displays workflows that
you can use on that record.

1. Open a record containing a Start with Docusign action.
2. Select the Start with Docusign action in the upper
   right of the record.

   The Start with Docusign action menu displays.
3. Use the action menu to search for a workflow:
   1. Enter a full or partial workflow name in the Search
      workflows field.

      ![A screenshot displays the Search workflows field highlighted in the Start with Docusign action menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qwr1706590641805/page/images/Start_Workflow_Builder_Workflow_Button_Menu_Search_Workflows.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU5MjcsInNoZWFmIjoicXdyMTcwNjU5MDY0MTgwNSJ9.X6bgKsIDECv4-4HGo_Dz3gG6uoecmgje2l5Su9rdXZc&_LANG=enus "A screenshot displays the Search workflows field highlighted in the Start with Docusign action menu.")

      Workflows matching your search query display.
   2. Select Start next to a workflow name.

      A message displays: Your workflow was
      initiated.
   3. Select the X button in the upper right to close
      the Docusign dialog box.

   The Docusign dialog box closes. The Salesforce record
   displays.

You have completed the process. You have started a workflow from a
Start with Docusign action. Docusign sends tasks to
all workflow recipients.
