# Getting Started with Workflow Builder for Salesforce

Learn about how to configure Salesforce steps in
workflows.

With [Workflow Builder](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=dnx1696972415150.html&_LANG=enus), you can connect
workflow steps like Web Forms and eSignature templates to third-party
extensions. You can create workflows to automate your agreement process,
including reading data from and writing data to Salesforce records.

Before using Salesforce features in workflows, first install, connect, and configure the Salesforce app.

Follow these steps to prepare your workflows to use the Salesforce app:

- Install the Salesforce application from the ​Docusign​ App Center
- Create a Connect configuration to add Salesforce merge fields to a Workflow Builder step
- Use ​Docusign​ Connect to link Salesforce fields to ​Docusign​ fields
- Add the Envelope Status component to a Salesforce layout
