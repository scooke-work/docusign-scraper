# Create a Connect Configuration

Connect notifies other applications when a Docusign event you want to know about
occurs. Learn how to create a Connect configuration.

## Before you begin

- Set up a webhook listener. Review URL to Publish in [Connect Configuration Settings](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=zjq1665170940873.html&_LANG=enus "Docusign Connect sends updates to other applications. Look up the settings that you use to set up Docusign Connect.") for more information.
- Connect must be enabled for the account.
- If you want to follow only selected groups of users, create the groups first.
  See [Add Groups](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=mbg1583277362677.html&_LANG=enus) for details.

Connect sends updates (event messages
or webhooks) about your Docusign activity to other
applications. This topic shows you how to create a Connect configuration so you can kick
off downstream processes in your own systems.

1. From the Accounts view, in the sidebar under INTEGRATIONS,
   select Connect.

   The Connect page displays.
2. At the top left, select Add Configuration > Custom.

   The Add Custom Configuration window
   displays.
3. Enter the settings that you want to use. For more information, see [Connect Configuration Settings](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=zjq1665170940873.html&_LANG=enus "Docusign Connect sends updates to other applications. Look up the settings that you use to set up Docusign Connect.").
4. Select Add Configuration.

The configuration saves. Consider testing the configuration
as a next step. Send Docusign transactions and observe your application's
behavior.
