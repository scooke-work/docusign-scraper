# Build a Start with Docusign Action (Manual Controls)

Learn to build a new Lightning web component action to start a Docusign
workflow from a record.

## Before you begin

Important:

- This feature is only available to IAM for Sales customers. Contact [Docusign support](https://support.docusign.com/s/?language=en_US&_gl=1*74elk4*_gcl_au*MTY5ODQyMjA2MC4xNzY5NjM1OTA1LjE5ODM3MTQzNjguMTc2OTYzNjIwNC4xNzY5NjM2MjA0) or [Docusign
  sales](https://www.docusign.com/contact-sales) for more information.
- Docusign recommends using the [IAM Launcher Setup Controls](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=tska9d68585-f524-49a4-8898-496b6575d036.html&_LANG=enus "Learn to use the IAM Launcher Setup interface to add Docusign workflow actions to Salesforce layouts.") to add workflow actions. Use these
  instructions if the IAM Launcher Setup controls fail.

[Prepare your
organization to start workflows from Salesforce](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=tsk2a2ae3f7-99c2-4deb-ab15-61fc26ade4d1.html&_LANG=enus "Learn to prepare your Salesforce organization to start Docusign Workflow Builder workflows from Salesforce.").

If you want to start workflows from Salesforce records, you can add a Start
with Docusign action to your layouts. The Start with Docusign action displays workflows that you can start from your
records. You can take the following actions from the Start with Docusign action menu:

- Start: Select the Start button to start any
  displayed workflow from a record.
- Add a Workflow: Add a new workflow to the Start with
  Docusign action.
- Remove this workflow: Remove a workflow from the
  Start with Docusign action.
- Search workflows: Find a specific workflow from the
  list.

Use this procedure to create a Start with Docusign action:

1. Access the Buttons, Links, and Actions page as a Salesforce administrator:
   1. Select the gear icon ![A screenshot displays a gear icon with a lightning bolt inside.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qwr1706590641805/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU5MDIsInNoZWFmIjoicXdyMTcwNjU5MDY0MTgwNSJ9.wAL-YSPjCAqPj-MgyoWpNF41zAuWlAcTfrPpk8SPquI&_LANG=enus "A screenshot displays a gear icon with a lightning bolt inside.") and choose Setup from the dropdown menu.
   2. Enter and select Object Manager in the Quick
      Find field.
   3. Select a Salesforce object in the left column and select
      Buttons, Links, and Actions.

   The Buttons, Links, and Actions page displays.
2. Select New Action in the upper right of the page.

   The New Action page displays.
3. Create your Start with Docusign action:
   1. Action Type: Select Lightning Web
      Component from the dropdown menu.
   2. Lightning Web Component: Select
      dfsle:embeddedWorkflowLauncher from the
      dropdown menu.
   3. Standard Label Type: Keep the default value of
      None.
   4. Label: Enter `Start with
      Docusign`
   5. Name: Keep the default value of `Start_with_Docusign`
   6. Select Save.![A screenshot displays the Action page with Start with Docusign fields and the Save button highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qwr1706590641805/page/images/Create_Start_with_Docusign_Lightning_action.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU5MDIsInNoZWFmIjoicXdyMTcwNjU5MDY0MTgwNSJ9.wAL-YSPjCAqPj-MgyoWpNF41zAuWlAcTfrPpk8SPquI&_LANG=enus "A screenshot displays the Action page with Start with Docusign fields and the Save button highlighted.")

   The page closes. The Buttons, Links, and Actions page
   displays.

You have completed the process. You have created a new Start with
Docusign Lightning web component action.
