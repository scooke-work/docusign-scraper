# Add Docusign Workflow Actions to Layouts (IAM Launcher Setup Controls)

Learn to use the IAM Launcher Setup interface to add Docusign workflow
actions to Salesforce layouts.

## Before you begin

Important: You cannot use the IAM Launcher Setup
controls to add actions to some Salesforce objects, like Case. If you can't add an action
with the IAM Launcher Setup controls, [add actions
with Salesforce Object Manager](https://support.docusign.com/s/document-item?bundleId=rrf1583359212854&topicId=fji1583359148678.html&_LANG=enus).

If you want to start
Docusign workflows from Salesforce records, you can add a
Start with Docusign action to your layouts. The Start with
Docusign action displays Docusign workflows
that users can start from records without logging on to Docusign.

Use this procedure to add a Start with Docusign action to a
layout with the IAM Launcher Setup controls:

1. Manage Connections and the IAM Launcher: Select Manage.

   The docusign app center dialog box displays.
2. Add an IAM Launcher action to a Salesforce layout:
   1. Select the IAM Launcher Setup tab.
   2. Select a connection: Choose a connection from the dropdown menu. You can only choose shared connections.

      ![A screenshot displays the IAM Launcher Setup tab and Connection menu highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qwr1706590641805/page/images/IAM_for_Sales_IAM_Launcher_Setup_tab.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU5ODEsInNoZWFmIjoicXdyMTcwNjU5MDY0MTgwNSJ9.ZQ46piEjeAWcMk5Ra0G3PmClh77HjHud45jCujUsBAM&_LANG=enus "A screenshot displays the IAM Launcher Setup tab and Connection menu highlighted.")

      The Create IAM Launcher section displays.
   3. Select Create.

      The Button Options section displays.
   4. Object: Enter an object name or select one from the dropdown menu.
   5. Account Layouts (optional): Select check boxes to add the action to specific layouts.
   6. Select Create.

      ![A screenshot displays the Button Options page with the Account object selected.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qwr1706590641805/page/images/IAM_for_Sales_IAM_Launcher_Button_Options_object.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU5ODEsInNoZWFmIjoicXdyMTcwNjU5MDY0MTgwNSJ9.ZQ46piEjeAWcMk5Ra0G3PmClh77HjHud45jCujUsBAM&_LANG=enus "A screenshot displays the Button Options page with the Account object selected.")
3. Repeat this process for each layout to which you want to add Docusign workflow actions.

You have completed the process. You have created a Start with Docusign action and added it to a layout. Users can start workflows without logging on to Docusign.
