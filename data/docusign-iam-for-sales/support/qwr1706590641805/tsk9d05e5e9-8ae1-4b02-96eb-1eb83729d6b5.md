# Add a Workflow to the Start with Docusign Action Menu

Learn to add a workflow to the Start with Docusign action menu.

## Before you begin

Important: This feature is only available to IAM for
Sales customers. Contact [Docusign support](https://support.docusign.com/s/?language=en_US&_gl=1*74elk4*_gcl_au*MTY5ODQyMjA2MC4xNzY5NjM1OTA1LjE5ODM3MTQzNjguMTc2OTYzNjIwNC4xNzY5NjM2MjA0) or [Docusign
sales](https://www.docusign.com/contact-sales) for more information.

[Add a Start with Docusign action to a layout](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=tskb637a1ca-b7ba-4604-97f7-f5f2d5f899f6.html&_LANG=enus "Learn to add a Start with Docusign action to a page layout to start workflows.").

If you want to start workflows from Salesforce records, you can add a Start
with Docusign action to your layouts. The action displays workflows
that you can start from your records. You can add and remove the workflows that
display in the action menu. You can also search for workflows in the action
menu.

Only administrators can add workflows to the action menu.

Use this procedure to add a workflow tile to the Start with Docusign action menu:

1. Choose your access method:
   - From the Docusign Setup page:
     1. [Access the Docusign Setup
        page as a Salesforce administrator](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=etv1586134206174.html&_LANG=enus)
     2. Manage Connections and the IAM Launcher: Select
        Manage.
     3. Select the IAM Launcher Setup tab.
     4. Select the Manage button next to a layout
        (Account, Opportunity, and so on.)

        The Start with Docusign menu displays.
   - From a Salesforce record: Select the Start with
     Docusign action in the upper right of the record.

     The Start
     with Docusign menu displays.
2. Select Add Workflow.

   ![A screenshot displays a list of Docusign workflows displayed in the Start with Docusign action menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qwr1706590641805/page/images/Start_Workflow_Builder_Workflow_Button_Add_a_Workflow.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU5MjIsInNoZWFmIjoicXdyMTcwNjU5MDY0MTgwNSJ9.apuVK9dmivkefC_up-Keq3gSVN7dNiBHXlRObyIweB8&_LANG=enus "A screenshot displays a list of Docusign workflows displayed in the Start with Docusign action menu.")

   The Add workflows dialog box displays.
3. Add a workflow to the action menu:
   1. Select the check box next to each workflow that you want to add.
   2. (Optional) Use the search field to find more workflows.
   3. (Optional) Filter which workflows display:

      1. Select Created by.
      2. Select Me or Not
         Me.
      3. Select Apply.
   4. Select Add Selected.

      The Add workflows dialog box closes. A message displays:
      (number of workflows) added. The workflow that
      you added displays in the Start with Docusign action
      menu.
4. Select the X icon to close the action menu dialog
   box.

You have completed the process. You have added a workflow to the Start with Docusign action menu. Users can start workflows from Salesforce
records.
