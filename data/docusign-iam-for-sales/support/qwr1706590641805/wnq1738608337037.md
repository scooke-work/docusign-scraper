# "This Template Is Not Supported" Displays When I Configure a Send Documents for Signature Step

Learn to fix the "This template is not supported" error in a Workflow Builder Send Documents for Signature
step.

I added a Salesforce field to my workflow's Send Documents for Signature
step. When I selected Next, an error displayed:
This template is not supported. I cannot save my Send
Documents for Signature step and finish building my template.

## Resolution: Delete Salesforce Fields That Are Not from Your Source Object

Cause: Your workflow
connects to a Docusign template containing fields from
multiple Salesforce objects. Workflow Builder can only use
templates containing fields from a single Salesforce source object (Opportunity,
Account, and so on.)

Use this procedure to fix the issue:

1. Log on to your Docusign account as an administrator
   and go to Templates >  My Templates.
2. Open your template's Place Fields page:
   1. Select the more actions menu next to your template ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/qwr1706590641805/page/images/A-screenshot-displays-the-use-button-with-a-three-dot-menu-next-to-it.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU5NjksInNoZWFmIjoicXdyMTcwNjU5MDY0MTgwNSJ9.rs3Yoez1JfBR_1oIXZBQyz9btwab9b3GC4RQvhEUX1M&_LANG=enus) and choose Edit from the
      dropdown menu.
   2. Select Next.

   The template's Place Fields page opens, displaying your
   template's merge fields.
3. Remove fields that are not from your Salesforce source object. For
   example, if you choose Opportunity as your source, remove fields from
   all other objects:
   1. Select a merge field.
   2. Select Salesforce in the right
      column.

      The field details display the Salesforce
      object.
   3. Select DELETE if the field is not from
      your Salesforce source object. ![A screenshot displays the Contact Full Name field with the Delete button highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qwr1706590641805/page/images/A-screenshot-displays-the-Contact-Full-Name-field-with-the-Delete-button-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU5NjksInNoZWFmIjoicXdyMTcwNjU5MDY0MTgwNSJ9.rs3Yoez1JfBR_1oIXZBQyz9btwab9b3GC4RQvhEUX1M&_LANG=enus "A screenshot displays the Contact Full Name field with the Delete button highlighted.")
4. Repeat this process for all non-source fields and select
   SAVE AND CLOSE.

   You have completed the process. You have removed all non-source Salesforce fields from
   your template. Return to your workflow and complete your Send
   Documents for Signature step. Docusign
   does not display the This template is not supported
   error.
