# Add the Docusign IAM Launcher Component to a Salesforce Layout

Learn to add the Docusign IAM Launcher component to a Salesforce
layout. [Prepare your Salesforce
organization to start Docusign workflows from Salesforce](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=tsk2a2ae3f7-99c2-4deb-ab15-61fc26ade4d1.html&_LANG=enus "Learn to prepare your Salesforce organization to start Docusign Workflow Builder workflows from Salesforce.").

Important: This feature is only available to IAM for
Sales customers. Contact [Docusign support](https://support.docusign.com/s/?language=en_US&_gl=1*74elk4*_gcl_au*MTY5ODQyMjA2MC4xNzY5NjM1OTA1LjE5ODM3MTQzNjguMTc2OTYzNjIwNC4xNzY5NjM2MjA0) or [Docusign
sales](https://www.docusign.com/contact-sales) for more information.

If you want to start workflows from Salesforce records, you can add the
Docusign IAM Launcher component to your layouts. The Docusign IAM Launcher component displays workflows that you can
start from your records. You can take the following actions from the IAM Launcher
component:

Use this procedure to add the Docusign IAM Launcher component to a
Salesforce layout:

1. As a Salesforce administrator, open a record, select the gear icon, ![A screenshot displays a gear icon with a lightning bolt inside.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qwr1706590641805/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU5MTIsInNoZWFmIjoicXdyMTcwNjU5MDY0MTgwNSJ9.oLh2dIhweQuAkJCdX7aEU4dnet-yHuN5XsRzFE0tGqQ&_LANG=enus "A screenshot displays a gear icon with a lightning bolt inside.") and choose Edit Page from the dropdown menu.

   Note: You can add the Docusign IAM
   Launcher component to any part of your layout. The following
   instructions are recommendations, not requirements.
2. (Optional) Add a Docusign IAM Launcher tab to your
   layout.
   1. Select Add Tab on the right navigation
      panel.

      A new Details tab displays.
   2. Select Details.

      The Tab Label dialog box opens.
   3. Tab Label: Select the
      Details dropdown menu and choose
      Custom.
   4. Custom Label: Enter a descriptive name, like
      Docusign IAM Launcher.
   5. Select Done.

   ![A screenshot displays a custom tab with Docusign IAM Launcher in the label field and the Done button highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qwr1706590641805/page/images/IAM_Launcher_component_tab.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU5MTIsInNoZWFmIjoicXdyMTcwNjU5MDY0MTgwNSJ9.oLh2dIhweQuAkJCdX7aEU4dnet-yHuN5XsRzFE0tGqQ&_LANG=enus "A screenshot displays a custom tab with Docusign IAM Launcher in the label field and the Done button highlighted.")

   Your new tab displays in the right navigation panel.
3. Drag the Docusign IAM Launcher component to the tab that
   you created and select Save.

   ![A screen capture displays dragging the Docusign IAM Launcher component to a Salesforce layout.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qwr1706590641805/page/images/Add_IAM_Launcher_component_to_a_layout.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU5MTIsInNoZWFmIjoicXdyMTcwNjU5MDY0MTgwNSJ9.oLh2dIhweQuAkJCdX7aEU4dnet-yHuN5XsRzFE0tGqQ&_LANG=enus "A screen capture displays dragging the Docusign IAM Launcher component to a Salesforce layout.")
4. Activate the change to your layout:
   1. Select Activate.
   2. Select a tab to set your default options: Org
      Default, App Default, or
      App, Record Type, and Profile.
   3. Select Assign and choose options from the dialog
      box.
   4. Select Close.
5. Select the back arrow ![A screenshot displays a blue box with a left-facing arrow icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qwr1706590641805/page/images/A-screenshot-of-the-Back-arrow-on-the-Salesforce-Edit-Page-settings-bar.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU5MTIsInNoZWFmIjoicXdyMTcwNjU5MDY0MTgwNSJ9.oLh2dIhweQuAkJCdX7aEU4dnet-yHuN5XsRzFE0tGqQ&_LANG=enus "A screenshot displays a blue box with a left-facing arrow icon.") to return to the record.

You have completed the process. You have added the
Docusign IAM Launcher component to a Salesforce layout. Users
can start Docusign workflows from Salesforce records.
