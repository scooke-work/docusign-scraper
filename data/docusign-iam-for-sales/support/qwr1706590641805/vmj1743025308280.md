# Use Docusign Connect to Link Salesforce Fields to Docusign Fields

Learn to map Salesforce objects to Docusign fields with Docusign Connect.

## Before you begin

[Create a Connect Configuration to Add Salesforce Merge Fields to a Workflow Step](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=xza1742958339120.html&_LANG=enus "Learn how to create a Connect configuration to add Salesforce merge fields to your workflow's Send Documents for Signature step.")

You can use Docusign Connect to link Salesforce fields to Docusign fields. When recipients complete and sign agreements, Docusign updates Salesforce fields you linked with Docusign fields.

Use this procedure to link Salesforce fields to Docusign fields
with Connect:

1. Log in to your Docusign account as an administrator and go to Admin >  INTEGRATIONS >  Connect.
2. Select the Actions button next to your Salesforce
   configuration and choose Edit from the dropdown
   menu.
3. Connect Objects: Select New Object
   and choose Salesforce Object from the dropdown
   menu.

   The Salesforce Object Connect page opens.
4. Map a Salesforce object to your Docusign account:
   1. Inactive Object: Select Activate
      Object from the dropdown menu.
   2. Object Name: Enter a name for your
      connection.
   3. (Optional) Object Description: Enter a
      description that explains details of the connection.
   4. On Completed Envelope Only: Only update
      Salesforce records when all recipients complete the signing process.
   5. Add if No Match: Add data to Salesforce if it
      does not exist.
   6. Select Salesforce Object: Choose a Salesforce
      object from the dropdown menu.

      ![A screenshot displays the General Settings section, with Inactive Object and Select Salesforce Object highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qwr1706590641805/page/images/A-screenshot-displays-the-General-Settings-section-with-Inactive-Object-and-Select-Salesforce-Object-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU4MjUsInNoZWFmIjoicXdyMTcwNjU5MDY0MTgwNSJ9.3nQCVtxau6tZlTMpAb-5jbsM4dsPUjHchMhx0l3pPbo&_LANG=enus "A screenshot displays the General Settings section, with Inactive Object and Select Salesforce Object highlighted.")
5. Select Where: Select the fields that are used to match a
   Salesforce object with Docusign information:
   1. Select Docusign Field: Select a field from your
      Docusign account to match with a Salesforce
      field.
   2. Select Salesforce Field: Select a Salesforce
      field to match with a field from your Docusign
      account.
   3. (Optional) + Add Item: Connect more Docusign and Salesforce fields.

      ![A screenshot displays the Docusign Recipient ID field and the Salesforce Owner ID field highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qwr1706590641805/page/images/A-screenshot-displays-the-Docusign-Recipient-ID-field-and-the-Salesforce-Owner-ID-field-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU4MjUsInNoZWFmIjoicXdyMTcwNjU5MDY0MTgwNSJ9.3nQCVtxau6tZlTMpAb-5jbsM4dsPUjHchMhx0l3pPbo&_LANG=enus "A screenshot displays the Docusign Recipient ID field and the Salesforce Owner ID field highlighted.")
6. Update Fields: Choose the Docusign
   fields that write back information to Salesforce fields:
   1. Select Docusign Field: Select a field from your
      Docusign account to update with a Salesforce
      field.
   2. Select Salesforce Field: Select a Salesforce field to be
      updated by your Docusign field.
   3. (Optional) + Add Item: Connect more Docusign and Salesforce fields.

      ![A screenshot displays the Update Fields section with the Docusign Recipient ID field and the Salesforce Owner ID field highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qwr1706590641805/page/images/A-screenshot-displays-the-Update-Fields-section-with-Docusign-Recipient-ID-field-and-the-Salesforce-Owner-ID-field-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU4MjUsInNoZWFmIjoicXdyMTcwNjU5MDY0MTgwNSJ9.3nQCVtxau6tZlTMpAb-5jbsM4dsPUjHchMhx0l3pPbo&_LANG=enus "A screenshot displays the Update Fields section with the Docusign Recipient ID field and the Salesforce Owner ID field highlighted.")
7. (Optional) Attachments: Choose how completed agreements
   save to Salesforce records:
   1. Attach Docusign Envelopes: Choose this option to
      save completed agreements to Salesforce records.
   2. Attach Docusign Certificate: Choose this option
      to save Docusign certificates of completion to
      Salesforce records.
8. Select Save.

You have completed the process. You have linked Salesforce fields
to Docusign fields. When recipients complete and sign agreements, Docusign updates Salesforce fields you linked with Docusign fields.
