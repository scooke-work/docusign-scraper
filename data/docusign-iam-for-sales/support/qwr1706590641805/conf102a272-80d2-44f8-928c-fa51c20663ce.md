# Learn About Starting Docusign Workflows from Salesforce

Learn about using the Docusign IAM Launcher component or a custom
button to start Docusign
workflows from Salesforce records.

Important: This feature is only available to IAM for Sales
customers. Contact [Docusign support](https://support.docusign.com/s/?language=en_US&_gl=1*74elk4*_gcl_au*MTY5ODQyMjA2MC4xNzY5NjM1OTA1LjE5ODM3MTQzNjguMTc2OTYzNjIwNC4xNzY5NjM2MjA0) or [Docusign sales](https://www.docusign.com/contact-sales) for more
information.

With [Workflow Builder](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=bqi1705961132077.html&_LANG=enus), you can
create workflows to automate your agreement
process.

You can start workflows from Salesforce records by adding a Start with Docusign action to your layouts. There are two ways to add Start with Docusign actions to layouts:

- (Recommended) [Use IAM Launcher Setup Controls](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=tska9d68585-f524-49a4-8898-496b6575d036.html&_LANG=enus "Learn to use the IAM Launcher Setup interface to add Docusign workflow actions to Salesforce layouts.")
- [Manually
  add actions with Salesforce Object Manager](https://support.docusign.com/s/document-item?bundleId=rrf1583359212854&topicId=fji1583359148678.html&_LANG=enus)

Users can start workflows by selecting the Start with Docusign action on records:![A screenshot displays a Salesforce record with a Start with Docusign custom button highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qwr1706590641805/page/images/Start_Workflow_Builder_Workflow_Custom_Button.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTU4OTgsInNoZWFmIjoicXdyMTcwNjU5MDY0MTgwNSJ9.n8l9eMK2S50ohf7WIM6UAqB-NCkvxZjYa5QNboNo80o&_LANG=enus "A screenshot displays a Salesforce record with a Start with Docusign custom button highlighted.")

You can take the following actions from a Start with Docusign action menu:

[Start](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=tskcae26f13-f2e5-4afd-85a7-145a70b12977.html&_LANG=enus "Learn to start a Docusign workflow from a Start with Docusign action.")
:   Select the Start button to start any
    displayed workflow from a record. Data from the
    record is included in the generated agreement.
    The workflow can
    read data from the record, write changes back, add Salesforce files, and
    store new agreements.

[Add a Workflow](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=tsk9d05e5e9-8ae1-4b02-96eb-1eb83729d6b5.html&_LANG=enus "Learn to add a workflow to the Start with Docusign action menu.")
:   Select this button to add a new workflow to a Start with Docusign action menu.
    Docusign only displays workflows
    related to the record's object. For example, on an Account
    record, Docusign only displays
    workflows built for accounts and related objects.

[Remove this workflow](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=tskb2fdae68-780f-45b0-9b06-f869786823eb.html&_LANG=enus "Learn to remove a Docusign workflow from a Start with Docusign action menu.")
:   Select this option to remove a workflow from the Start with Docusign action
    menu. Removing a workflow from the component view does not delete the
    workflow from your Docusign account.

[Search workflows](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=tsk5491237f-0dec-4eed-9976-7b2124f9a7df.html&_LANG=enus "Learn to search for a workflow from the Start with Docusign action menu.")
:   Use the search field to find a specific workflow. Docusign only displays
    workflows related to the record's object.
