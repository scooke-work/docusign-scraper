# Save Envelope Custom Fields (ECFs)

Read about how to use envelope custom fields (ECFs) to categorize envelopes, save form data, and customize envelope email subject lines.

Envelope custom fields (ECFs) help you categorize and identify envelopes you send for signature. You can save form data as ECFs and use it to search and report on envelopes or customize the envelope email subject line. For more information about customizing the subject line, see the [Use Merge Fields in the Template Email Subject to Customize Envelopes](https://support.docusign.com/s/document-item?bundleId=xry1643227563338&topicId=bat1578456393880.html&_LANG=enus) guide.

If you are new to envelope custom fields (ECFs), see the [Envelope Custom Fields](https://support.docusign.com/s/document-item?bundleId=gbo1643332197980&topicId=ywu1578456351241.html&_LANG=enus) guide.

Note: When you reference a form field as a merge field, you must use the Field name property, not the API reference name property.

When you create a Web Form from an eSignature template, all ECFs from the template are automatically imported to the form's sender view. For more information about the sender view, see the [Sending a Web Form](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=con56dbba74-0e1a-4356-a265-bf7ec87913e9.html&_LANG=enus) guide.

You can also designate additional form fields (except checkboxes) as ECFs by selecting the Save as envelope custom field option in the Properties tab.
