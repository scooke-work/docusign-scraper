# Diagnose the "Form Not Submitted" Error

Learn about using a debug code in the form URL to diagnose the "Form not submitted" error.

If you see a "Form not submitted" error after submitting a form, it is caused by an envelope generation error, which can result from the eSignature template configuration or account settings.

1. Add `?debug=1` to the end of the form URL.

   A notification displays the API error that caused the form submission to fail.
2. Use the error message to do one of the following:

   - Resolve the issue described in the error message. For example, if the message says, "The Tab formula is not valid", review and correct the Tab formula.
   - If you can't resolve the error, contact [Docusign Support](https://support.docusign.com/s/contactSupport) and include the error details.

After you resolve the issue or contact [Docusign Support](https://support.docusign.com/s/contactSupport) with the error details, the form submits successfully.
