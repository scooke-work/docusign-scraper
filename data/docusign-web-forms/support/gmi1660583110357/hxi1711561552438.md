# Make Web Form Fields Read-Only

Learn how to make a Web Form field read-only for recipients.

You can make a form field with default values read-only so recipients can view but not change the information.

1. Build or edit a form.

   Note: For fields without default values, such as a Name field, see the [Prefill a Web Form By URL](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=kup1721242003741.html&_LANG=enus) guide and the [Web Forms API 101](https://developers.docusign.com/docs/web-forms-api/web-forms-101/) guide.
2. Select the field you want to make read-only.
3. Toggle the Read only option in the Properties tab.

The form recipient can see the Read only field but cannot edit its value.
