# Pause and Resume a Web Form

Learn how to allow form recipients to pause and resume completing a Web Form using the Finish Later button, or require the form to be completed in a single session by turning off this option.

Form recipients can pause and resume a form by selecting the Finish Later button, which saves their progress and emails a reminder link valid for 10 days.

The Finish Later button displays when the form is activated and sent. To require completion in a single session, add `?showFinishLater=false` to the end of the form URL.

Note: The Finish Later button is not available for [standalone forms](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=pbx1716585514741.html&_LANG=enus).

1. Open the form link.
2. Select Finish Later.
3. Enter an email address and select Submit.

   A 10-day valid link to resume the form is emailed.
4. Open the emailed link and select Finish Form to resume.

The recipient can pause and resume the form without losing previously entered data. If the Finish Later option is turned off using `?showFinishLater=false`, the form must be completed in a single session.
