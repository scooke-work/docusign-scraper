# Configure Web Form Fields

Learn how to add and configure fields in a Web Form template using the Web Form Builder.

You can import template fields, such as "First name" and "Last name," and add new fields like "Middle Name" as needed. For information about each field type property, see the [Web Forms Field Type Properties](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=con145288b7-86ef-44fb-ab44-252ece5dec25.html&_LANG=enus "Learn how to use the Web Forms Builder to configure field types and their properties, including shared options such as envelope custom fields (ECFs) and sender controls.") guide.

1. Select the plus sign (+) at the top or bottom of the field where you want it to display.
2. Select the field type (Text, Number, Date, Email, and other types) that you want to add.

   The field displays in the center panel of the Web Form Builder.
3. Select the Properties or Rules tab to configure field properties or rules as needed.
4. Connect the field to a template field. For more information about connecting fields, see the [Connect Web Form Fields to Template Fields](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=vlb1680643208628.html&_LANG=enus) guide.

The field is added to the form, connected to the appropriate template field, and its properties are configured according to your selections.
