# Activate a Web Form

Learn how to activate a Web Form, authorize your account, set access, and make the form available to recipients.

After building the form, you must activate it to make it visible and usable for recipients.

1. Select Activate.
2. Select Authorize Account to verify your account as the authorized sender of the form.

   ![Authorize Account confirmation message indicating the account must be authorized before activating the Web Form](https://docusign-be-prod.zoominsoftware.io/api/bundle/gmi1660583110357/page/images/wf-activate-first-time-message.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTIyMjYsInNoZWFmIjoiZ21pMTY2MDU4MzExMDM1NyJ9.ZuEmSmTr0vxe0E_bsAHGqsZjTxOo8dxUIcxtUpMNa48&_LANG=enus "Authorize Account confirmation message indicating the account must be authorized before activating the Web Form")

   Important: If the account used to authorize the form is closed, envelope creation fails. To fix this, a user with form access (such as an account administrator) must reactivate and reauthorize the form using an active account.
3. Choose an access setting for the form; you can change it later.

   ![Access settings dropdown for activating a Web Form with Public and Private options](https://docusign-be-prod.zoominsoftware.io/api/bundle/gmi1660583110357/page/images/wf-activate-this-web-form.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTIyMjYsInNoZWFmIjoiZ21pMTY2MDU4MzExMDM1NyJ9.ZuEmSmTr0vxe0E_bsAHGqsZjTxOo8dxUIcxtUpMNa48&_LANG=enus "Access settings dropdown for activating a Web Form with Public and Private options")

   1. Select Public to make the form visible to anyone through a shareable link.

      Important: The form must be activated as "Public" to be connected to a Workflow Builder workflow.
   1. Select Private to make the form available only through API
      integration.
4. Select Activate.

   The form activates and is displayed.

The form is activated and available to recipients based on the selected access setting.
