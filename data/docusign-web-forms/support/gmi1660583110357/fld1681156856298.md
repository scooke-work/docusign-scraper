# Download the Web Form CSV Results

Learn how to download a CSV file of submitted Web Form data to review, filter, analyze, or integrate the results with other systems.

You can download a CSV file of all submitted form data to analyze it or integrate it with external systems.

Note: The CSV results may take several hours, up to 24 hours, to display after the form is completed.

1. Go to the Templates page.
2. Select My Web Forms, Shared with Me, or All Web Forms and choose the form whose CSV results you want to download.
3. Select More Actions (...) > Download Results.

The CSV file downloads to your device so you can review, filter, and analyze all submitted form data.
