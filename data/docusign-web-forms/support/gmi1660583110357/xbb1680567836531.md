# Transfer a Web Form Configuration File

Learn how to transfer a Web Form configuration file to create a new form or move an existing form between environments or users.

You can download an existing form configuration and upload it to create a new form or transfer it between environments or users, including the same user on the same account.

1. Select My Web Forms, Shared with Me, or All Web Forms to choose the form that you want to transfer.
2. Select More Actions (...) > Download Configuration.

   The JSON configuration file downloads.
3. Upload the configuration file by selecting Start > Web Forms > Upload Web Form.

The form is created or updated based on the uploaded configuration file, making it available for use in the selected environment.
