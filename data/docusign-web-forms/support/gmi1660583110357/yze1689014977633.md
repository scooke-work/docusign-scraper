# Web Forms for FedRAMP Accounts

Read about how FedRAMP account administrators can enable Web Forms for their users, similar to other accounts, without any special tasks.

Web Forms is available for government and FedRAMP plans. FedRAMP account administrators can enable Web Forms for users without any special tasks. Administrators and users can start creating or using forms immediately.

For more information about Docusign and FedRAMP, see the [FedRAMP - Docusign for Government](https://support.docusign.com/s/articles/FedRamp-FAQ?language=en_US) article.
