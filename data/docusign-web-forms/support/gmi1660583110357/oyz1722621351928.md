# Importing Attachment Fields from an eSignature Template

Learn how to import attachment fields from an eSignature template into a Web Form and add additional attachment fields manually as needed.

When you create a form from a template that includes attachment fields, those fields are imported automatically. Because each attachment field accepts only one file, you may need to add more attachment fields.

For example, if you import a template with an attachment field named "Lease Agreement", users can upload one lease agreement. If users have multiple leases, you can manually add another attachment field, such as "Lease Agreement 2", for the second lease.

You can also add attachment fields directly to a form, even if the imported template has none.

Note: You can add up to 15 attachment fields to a form.

1. Go to the Templates page.
2. Select Start > Web Forms > Create Web Form.
3. Select Start with document > Select an eSign template.
4. Select a template with multiple signers and select Next or Continue without AI.
5. If you select a template with attachments, select Import from the Import attachment
   fields? window.
6. If you select a template without attachments, select Got it from the Web Forms
   template copy created window.
7. Name the form and select
   Apply.
8. To add one or more attachment fields manually, select the plus sign (+).
9. Select Attachment.
10. Activate the form and send it to recipients so they can upload attachments.

The form is created from the selected eSignature template, and any attachment fields in the template are imported. Any attachment fields that you add manually are also available for recipients to upload files.
