# Build a Multiple-Recipient Web Form

Learn how to create a multiple-recipient Web Form from a multi-signer template so each signer sees and completes only their assigned fields.

By default, a form created from a multi-recipient template is completed only by the first recipient. Multi-recipient forms let additional recipients from the template complete the same form.

Create a multiple-recipient form from a multi-signer template so each signer sees only their assigned fields.

For example, in a two-signer template, the first signer completes and signs the form, which then emails the second signer to complete their fields. Each recipient sees only their fields, but they share the same envelope.

Keep these points in mind when building a multiple-recipient form:

- Attachment fields are not supported for additional recipients.
- The additional recipients don't see the Thank You page from the
  form configuration.
- You can add or remove recipients after you create the form. From the Outline, select the recipient dropdown and Manage Recipients.
- If you remove a recipient, their form fields are permanently deleted, and any customizations are lost.

Note: Multiple-recipient forms are available only for Advanced Web Forms. You can create multiple-recipient forms only from templates. You cannot create them as standalone forms.

1. Go to the Templates page.
2. Select Start > Web Forms > Create Web Form.
3. Select Start with document > Select an eSign template.
4. Choose a template with multiple signers, and select Next or Continue without AI.
5. From the dropdown, select the form initiator, and select Next.
6. Select the recipients who must complete the form, and select Create Form.

The form is created with the signers that you added, each with their own fields. Each recipient receives an email to complete their fields, and when all are finished, the form is ready to review or download.
