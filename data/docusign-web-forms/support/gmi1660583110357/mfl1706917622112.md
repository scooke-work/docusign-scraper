# Web Forms Accessibility

Read about how the Web Forms Recipient Experience complies with Level A and Level AA criteria of the Web Content Accessibility Guidelines 2.1.

Based on the most recent assessment, the form-filling user experience of Web Forms, referenced in the following document as the “Docusign Web Forms Recipient Experience” supports all relevant Level A and Level AA criteria of the Web Content Accessibility Guidelines 2.1. For more information about Web Forms accessibility, see the [Docusign Web Forms - Accessibility Conformance Report](https://support.docusign.com/s/document-item?language=en_US&bundleId=docusign-web-forms-accessibility-conformance-report&topicId=docusign-web-forms-accessibility-conformance-report.pdf&_LANG=enus).
