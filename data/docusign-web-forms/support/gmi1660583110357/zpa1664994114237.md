# Add a Brand to a Web Form

Read about how a Web Form inherits branding from its associated template, including logos and color schemes, affecting the appearance of elements.

A form inherits the signing branding from its associated template. If the imported template has a specific logo and color scheme, the form uses the same elements, which determine the appearance of the landing page.

You can choose a brand from the template by selecting Preview > Branding.

Brand information is stored on the Admin page. As an administrator, you can add and update the brands there. For more information about branding your form, see the [Add a Signing Brand for Recipients](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=byg1583277338658.html&_LANG=enus) guide.
