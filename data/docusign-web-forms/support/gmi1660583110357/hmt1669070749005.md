# Deactivate a Web Form

Learn how to deactivate a Web Form so it can no longer be viewed or shared.

You can deactivate an active form to prevent it from being viewed or shared.

1. Select My Web Forms, Shared with Me, or All Web Forms and open the form that you want to deactivate.
2. Select More Actions (...) > Deactivate.

   The Deactivate Form? window displays.
3. Select Deactivate.

The form is deactivated and can no longer be viewed or shared. To reactivate a form, see the [Reactivate a Web Form](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=crq1669076320407.html&_LANG=enus) guide.
