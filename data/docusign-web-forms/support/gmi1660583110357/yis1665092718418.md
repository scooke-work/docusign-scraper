# Create a Data Label on a Template

Read about how to create clear, descriptive data labels on template fields so senders can easily identify each form field.

Form fields inherit the template's data labels, which only the sender can see. When you add a field to a template, its data label displays in the Properties tab. By default, the label is a generic identifier that usually matches the field name with extra characters, such as "Text 1234".

Change this default identifier to a more descriptive name. In the Data Label field, enter a name such as "Co-Applicant Signature".

For example, a checkbox field might have the default identifier "Checkboxe88cf158-5c82-4892" in the Data Label field. Rename this label to something more descriptive, such as "Completed by Interviewer".

Text field data label
:   A text field lets recipients enter larger amounts of free-form text, such as Name, Address, or List of medications. Because a document can contain several text fields, give each one a unique name.

    - No label can be left blank.
    - Every label name must be descriptive. We strongly suggest using the template field name. For example, if the template field is "Applicant", you might name the data label "Applicant".

Dropdown field data label
:   A dropdown field lets recipients select a single option from a list you define.

    - No label can be left blank.
    - Every label must be unique.
    - Every label name must be descriptive. Docusign strongly suggests using the template field name.
    - Use only letters, numbers, and underscores. Spaces are not supported.
    - Start every label with a letter.

Checkbox field data label
:   This standard checkbox lets recipients select one or more options. You can add a single box or a group of boxes and change the data label for individual boxes or the entire group.

    - No label can be left blank.
    - Every label must be unique.
    - Every label must be descriptive. Docusign strongly suggests using the template field name. For example, if the template field is "Applicant", you might name the data label "Applicant".
    - If you create a checkbox group, you must add a descriptive Group Label name.
    - Use only letters, numbers, and underscores. Spaces are not supported.
    - Start every label with a letter.

Radio field data label
:   Radio fields let recipients select only one option from a group. You can add, remove, or reposition individual buttons.

    - No label can be left blank.
    - Every label must be unique.
    - Every label must be descriptive. We strongly suggest using the template field name.
    - If you create an option group, you must add a descriptive Group Label name.
    - Use only letters, numbers, and underscores. Spaces are not supported.
    - Start every label with a letter.
