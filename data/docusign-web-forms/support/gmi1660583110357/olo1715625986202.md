# Web Form Copy Options

Learn how to copy a published Web Form by using the Copy and Create Workflow Version options.

After you publish your form, you can either create an exact copy of the form or generate a workflow-only version without the eSignature step.

Copy
:   This option creates a duplicate of the form, labeled with "Copy" before the title. To copy a form, select More Actions (...) next to the form that you want to copy, and select Copy.

Create Workflow Version
:   This option copies an existing form, removes the eSignature step, and labels the copy with "Workflow Copy" before the title. Use this option for Docusign Workflow Builder forms, which support only forms without a signature step. To create a workflow version, select More Actions (...) next to the form, and select Create Workflow Version.
