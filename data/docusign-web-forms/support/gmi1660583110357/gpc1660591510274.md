# The Web Forms Builder

Read about the Web Forms Builder, which lets you create and configure Web Forms from templates or from scratch, customize fields and attributes, and apply conditional rules to control which fields display.

A form inherits its attributes and data labels from the template used in the Web Forms Builder. While forms may vary in appearance due to different templates, they share common building characteristics and navigation.

Standalone forms that are not based on a template can also be created and configured with their own attributes and data labels.

Each form begins with a Welcome page, where you can modify the title, subtitle, and button text.

Outline
:   Displays on the left side of the builder. This panel contains the form fields inherited from the data labels in the template. Select the fields and they display in the center panel of the builder.

Center
:   Displays the form fields. When you select a field from the Outline, it displays in the center panel. This panel reflects the final form.

    For more information about adding fields to a form, see the [Configure Web Form Fields](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=awi1670975283111.html&_LANG=enus) guide.

Properties/Rules
:   Displays on the right side of the builder. You can add or edit the properties of the form fields. For example, you can edit a field name, or add the selections to a dropdown or checkbox field.

    You can also create rules that show other fields, depending on the recipient's response. For example, you can set a rule that if a recipient responds "Yes" to the field "Are you married"? then the field "Spouse's name" displays.

    For more information about rules, see the [Add Rules to a Web Form](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=kff1668121038223.html&_LANG=enus) guide.
