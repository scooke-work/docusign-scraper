# Build a Standalone Web Form

Learn how to build a standalone Web Form to collect data for use in Docusign Workflow Builder workflows.

Build a standalone form to collect data and use it in Docusign Workflow Builder workflows. Standalone forms are not associated with a template and do not support signatures.

When you create the standalone form, it includes a Welcome page, an untitled page for collecting information, and a Thank you page.

With Advanced Web Forms, you can use APIs to populate standalone forms. For more information, see the [Web Forms API 101](https://developers.docusign.com/docs/web-forms-api/web-forms-101/) guide.

Note: Data verification is supported only for standalone forms and only for email and phone fields.

1. Go to the Templates page.
2. Select Start > Web Forms > Create Web Form.
3. Select Create blank form.
4. Name the form and select Apply.
5. Build the form by adding fields, rules, and other elements. For more information about building forms, see the [Build a Web Form](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=euk1665093155272.html&_LANG=enus) guide.

The standalone form is created with a Welcome page, an information collection page, and a Thank you page, and is ready to use in Docusign Workflow Builder workflows.
