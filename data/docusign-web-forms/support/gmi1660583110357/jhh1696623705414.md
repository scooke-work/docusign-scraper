# Format Text in Page Subtitle and Field Description Fields

Learn how to use Markdown to format plain text in the Page subtitle and Description text fields, including bold, italics, headings, lists, links, and horizontal lines.

You can format plain text in the Page subtitle or Description text fields by using Markdown to apply formatting.

Note: Markdown adds formatting to plain text. For more information, see the [Markdown](https://www.markdownguide.org/basic-syntax/) website.

- [Bold](https://www.markdownguide.org/basic-syntax/#bold)
- [Italic](https://www.markdownguide.org/basic-syntax/#italic)
- [Links](https://www.markdownguide.org/basic-syntax/#links)
- [Headings](https://www.markdownguide.org/basic-syntax/#headings)
- [Lists](https://www.markdownguide.org/basic-syntax/#lists-1)
- [Special characters](https://www.markdownguide.org/basic-syntax/#characters-you-can-escape)
- [Horizontal lines](https://www.markdownguide.org/basic-syntax/#horizontal-rules)

For more information about configuring fields, see the [Configure Web Form Fields](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=awi1670975283111.html&_LANG=enus) guide.
