# Edit Recipient Information

Learn how to review and edit recipient information on your Web Form to ensure accuracy and completeness.

You can review and edit recipient information on your form to ensure it is accurate and complete.

Note: You can update these fields, but you cannot delete them or leave them blank.

1. Select Templates > My Web Forms.
2. Select the form for which you want to edit the recipient information.
3. Select the Signature page from the
   Outline.
4. In the Template Configuration section, select the pencil icon next to Recipient Connections.
5. Select Edit Template.
6. From the Fields recipient dropdown, select Edit Recipients.
7. Edit recipient information or add a new recipient by selecting Add Recipient.
8. Select Save and Save and Close to apply the edits to your form.

Recipient information on the form is updated, and all changes are saved and applied to the selected template.
