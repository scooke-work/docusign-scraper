# The Web Forms Builder Center Panel

Read about the center panel in the Web Forms Builder, which displays fields from the Outline and allows for field additions and deletions.

Fields listed in the Outline display in the center panel, originating from data labels in the imported template or standalone forms. Edits and rules you apply also display in the center panel.

You can add or delete fields in the center panel. Select the plus sign (+) to add a field, or the trash can icon to delete one.

For more information about adding fields or pages, see the [Configure Web Form Fields](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=awi1670975283111.html&_LANG=enus) guide.
