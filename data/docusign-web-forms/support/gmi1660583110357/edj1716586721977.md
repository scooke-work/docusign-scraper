# Copy an Existing Web Form for a Workflow Builder Workflow

Learn how to copy an existing Web Form as a standalone form for use in a Docusign Workflow Builder workflow.

You can copy an existing form for a Docusign Workflow Builder workflow. Use the Create Workflow Version option to extract the signature functionality from the original form and create a standalone form.

Note: To copy a form for Docusign Workflow Builder, select Create Workflow Version, not the Copy option.

1. Go to the Templates page.
2. Select My Web
   Forms, Shared with Me, or All
   Web Forms.
3. Select the form to use in the Workflow Builder workflow.
4. Select More Actions (...) > Create Workflow Version.

   The form is copied and named "Workflow Copy - [Web Form name]", for example, "Workflow Copy - Residential Loan Application" for the "Residential Loan Application" form.

The workflow copy is created as a standalone form with the original signature functionality preserved for use in your Workflow Builder workflow.
