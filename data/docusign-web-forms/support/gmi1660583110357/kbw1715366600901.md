# Build a Workflow Builder-Compatible Web Form

Read about how to create a Workflow Builder-compatible Web Form to automate the agreement process without coding.

The Docusign Workflow Builder platform integrates workflow steps for Docusign applications like Web Forms, automating the agreement process end-to-end without coding.

For more information about using Workflow Builder, see the [Workflow Builder Overview](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=dnx1696972415150.html&_LANG=enus) guide.

You can build a Workflow Builder-compatible form in two ways:

- [Build a Standalone Web Form](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=pbx1716585514741.html&_LANG=enus)
- [Copy an Existing Web Form for a Workflow Builder Workflow](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=edj1716586721977.html&_LANG=enus)
