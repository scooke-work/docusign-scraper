# The Signature Page

Read about the Signature page in the Web Forms Builder to review recipient information.

You can use the Signature page to review the primary recipient and additional recipient information associated with a form.

![Signature page in the Web Forms Builder with options to initiate signing from email, enable document field editing, and configure template and recipient connections.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gmi1660583110357/page/images/wf-signature-page.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTIxMjUsInNoZWFmIjoiZ21pMTY2MDU4MzExMDM1NyJ9.V9wwT8wwOTSGOaBVhjL1xqMOMkdee8WwoTbF8KV1KIw&_LANG=enus "Signature page in the Web Forms Builder with options to initiate signing from email, enable document field editing, and configure template and recipient connections.")

Initiate signing session from email
:   Toggle to initiate a signing session from an email link. This action verifies the signer's email.

    After you activate the form, select Open Link > Start, and add the signing information. Select Next to display the Summary page.

    The recipient selects Review Document in the email to display the signing page. If you don't toggle this, the signing page automatically displays without going through the email flow.

Enable document field editing
:   Toggle to allow recipients to edit envelope responses before signing, enabling modifications to all fields populated from the form.

    Once the form is activated, recipients can review and modify field values before signing. Typically, template fields remain unchanged, but this toggle allows updates, such as date fields. If you don't toggle this, the field editing is disabled.

Template
:   Select Edit (pencil) to view the template.

Template Connections
:   Configure the connection between form fields and template fields. For more information about how to connect the fields, see the [Connect Web Form Fields to Template Fields](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=vlb1680643208628.html&_LANG=enus) guide.

Recipient Connections
:   Activate recipients and assign them form fields to collect their information.
