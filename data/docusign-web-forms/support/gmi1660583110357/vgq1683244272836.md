# The Thank You Page

Read about how to customize the Thank you page that confirms form submission, including modifying titles, subtitles, and adding buttons.

After submitting a Web Form, recipients see a Thank you page confirming receipt. You can use the default page or customize it.

![Example Thank You page showing a confirmation message and optional button after form submission](https://docusign-be-prod.zoominsoftware.io/api/bundle/gmi1660583110357/page/images/wf-thank-you-page.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTIxMjgsInNoZWFmIjoiZ21pMTY2MDU4MzExMDM1NyJ9.7sqm_7HaeaN8FbT9rLYQCqiUoYnUo6eM7Xa7-oO6WRA&_LANG=enus "Example Thank You page showing a confirmation message and optional button after form submission")

Page title
:   The default page title is "Thank you", which can be changed. This field must not be empty.

Page subtitle
:   A subtitle displayed under the page title. The default is "We've received your form", but you can change it.

Button
:   You can add a button by switching this toggle. The default is no button. If you add a button, provide the Button text and Button URL. These fields cannot be blank.
