# Change the eSignature Sender

Learn how to change the eSignature sender for envelopes generated from a web form by authorizing a new sender.

Although you can't assign a sender directly, you can change the eSignature sender for envelopes generated from a web form by authorizing a different user.

When a web form is activated, the user who completes the Authorize My Account step becomes the eSignature sender, even if they didn't create or own the form.

Important: If the current sender's account is deactivated or they revoke Web Forms access, envelope generation for that form fails.

1. As the form manager, open an active web form and select Edit Form.
2. Select More Actions (...) > Share to give a new user Can Edit or Can Manage access.

   Important: The new sender must authorize their account and activate the form.
3. As the new sender, select Activate to finish activation.
4. Select Authorize Account.

   If Authorize Account isn't visible, select Edit next to Authorize eSignature sender.
5. Select Activate and Replace.

Future envelopes generated from the form are sent by the new eSignature sender.
