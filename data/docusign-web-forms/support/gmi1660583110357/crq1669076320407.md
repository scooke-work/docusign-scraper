# Reactivate a Web Form

Learn how to reactivate an edited Web Form so that your changes replace the currently active version and become visible to recipients.

After editing an active form, you must reactivate it for your changes to become visible and actionable to recipients.

1. Open an activated form and select Edit Form.
2. Make the necessary edits and select Activate.

   Note: When you reactivate an edited form, it replaces the current active form.
3. Select Activate and Replace.

   The form reactivates and is displayed.

The edited form is reactivated and replaces the previously active form.
