# Connect Web Form Fields to Template Fields

Read about how template connections map Web Form fields to eSignature template fields so recipients enter information only once and see it everywhere it is used.

A template connection defines which data goes into a specific field by mapping a form field to a corresponding field in the imported eSignature template. Template fields must be mapped to form fields so recipients can see and use them.

Alternatively, you can connect fields in Document view by linking form fields directly to template fields in the document layout.

Most connections are automatically imported from the template to the form. You can manually connect a form field to a template field only when you add a new field to the form, and you can map multiple fields at once or individually.

To view all template field connections, go to the Signature page and select the Template Connections edit (pencil) icon. Or select More Actions (...) and Edit Template Connections.

You can map a form field to multiple template fields so the recipient enters the value only once and it displays everywhere that field is used in the template.

For example, map the form field "Last Name" to the template field "Last Name". If the field displays three times in the template, the recipient completes it once on the form, and the value displays in all three locations.
