# Differences Between Web Forms and PowerForms

Read about how Web Forms and PowerForms are both template-based solutions for accessing, completing, and signing forms, each offering unique features and benefits.

Web Forms and PowerForms are both template-based solutions that help users to access, complete, and sign forms. However, there are some differences between Web Forms and PowerForms.

## Web Forms

Web Forms provide an interactive, guided experience for completing forms and improve envelope completion rates by 31% compared to PowerForms, offering a superior signing experience despite requiring more setup.

## PowerForms

A PowerForm is a static PDF form that users can complete. It allows self-service PDF forms to be published on websites without requiring coding. For more information about PowerForms, see the [PowerForms](https://support.docusign.com/s/document-item?bundleId=jbx1643062255110&topicId=eke1578456649038.html&_LANG=enus) guide.

| Feature | PowerForms | Web Forms |
| --- | --- | --- |
| Generates envelopes? | The envelope is triggered when the recipient *begins* completing the form. | The envelope is triggered when the recipient *completes* the form.  Web Forms never waste an envelope because it is created *only when signing begins*. |
| Mobile-friendly user interface? | Recipients complete information on a PDF that may not be optimized for mobile devices. | Recipients complete the information on a dedicated form that is mobile-friendly. |
| Conditional logic? | You can set conditional logic in templates to hide or show fields on a PowerForm. | You can set conditional logic to hide or show fields. |
| Supports multiple users? | Multiple users can complete and sign a PowerForm. | Web Forms supports multiple users. For more information about multi-recipient Web Forms, see the [Build a Multiple-Recipient Web Form﻿](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=inc1745950777430.html&_LANG=enus) guide. |
| Shareable unique link? | Yes | Yes |
| Branded UI? | Yes | Yes |

## Convert a PowerForm to a Web Form

You can convert a PowerForm to a Web Form in two ways:

- Select Create Web Form from a PowerForm.
  - On the Agreements page, select PowerForms.
  - In the PowerForms list, find the form to convert.
  - Select More Actions (...) > Create Web Form.
- Select the template that you already use for a PowerForm and create a Web Form from it.
  - The PowerForm still exists in your account. For information about building a Web Form, see the [Build a Web Form](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=euk1665093155272.html&_LANG=enus) guide.
