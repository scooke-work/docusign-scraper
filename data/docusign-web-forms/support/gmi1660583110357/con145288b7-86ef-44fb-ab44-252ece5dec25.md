# Web Form Field Type Properties

Learn how to use the Web Forms Builder to configure field types and their properties, including shared options such as envelope custom fields (ECFs) and sender controls.

You can use Web Forms Builder field properties to control field behavior, appearance for senders and recipients, and whether values are stored for reuse across envelopes.

To make a field an envelope custom field (ECF), toggle the Save as envelope custom field option. For more information about ECFs, see the [Envelope Custom Fields](https://support.docusign.com/s/document-item?bundleId=gbo1643332197980&topicId=ywu1578456351241.html&_LANG=enus) guide.

For each field type, you can use the Sender controls to set field visibility to Hide, Display (Optional), or Required.

| Field type | Properties |
| --- | --- |
| Text | Configure the text field name and description, choose the text type and character limit, optionally add a default value, and apply response validation. |
| Email | Configure the email field name and description, optionally add a default email address, and control sender visibility. |
| Date | Configure the date field name and description, choose a date format and default value, and optionally save it as an envelope custom field (ECF). |
| Radio | Configure the radio field name and description, and edit the radio options using the Edit (pencil) icon. |
| Checkbox | Configure the checkbox field name and description, define checkbox options (including option text and descriptions), choose a response validation option and default value, and control sender visibility. |
