# Open and Edit a Web Form

Learn how to open and edit an existing Web Form in the builder, and save and activate your changes.

You can open an existing form from the Templates page to update its fields, layout, or settings.

1. Go to the Templates page.
2. Select My Web Forms, Shared with Me, or  All Web Forms.

   Only administrators can access forms from All Web Forms.
3. Select the web form that you want to open.

   The form opens in the builder.
4. If you are editing an activated form, select Edit Form.
5. Select Save.
6. [Reactivate the form](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=crq1669076320407.html&_LANG=enus) so that the edits are visible to your users.

The form opens in the Builder.
