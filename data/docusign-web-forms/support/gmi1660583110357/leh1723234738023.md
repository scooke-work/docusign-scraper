# The Hidden Data Page

Learn how to use the Hidden Data Page in Advanced Web Forms to capture and manage envelope custom field values, including populating them with default values or by the Web Forms API.

You can capture data visible only to you on the Hidden Data Page. When you create a form from a template with envelope custom fields (ECFs), they are imported automatically.

To verify that a field is an ECF, select it and the Save as envelope
custom field toggle displays in the Properties tab.

There are two ways to add values to fields on the Hidden Data Page:

- Set the Default value in the Properties tab.
- Use Web Forms APIs to automatically populate fields. For more information about Web Forms APIs, see the [Web Forms API 101](https://developers.docusign.com/docs/web-forms-api/web-forms-101/) guide.

  Note: The Hidden Data Page is an Advanced Web Forms feature for Docusign eSignature plans and is included with Docusign IAM Professional and higher plans. Advanced Web Forms is an add-on for Business Pro and higher plans.
