# Advanced Web Forms

Read about what Advanced Web Forms is, how it extends Web Forms, and which eSignature and IAM plans include this add-on.

Advanced Web Forms extends Web Forms functionality. It is available as an add-on for Business Pro or higher Docusign eSignature direct plans and is included in Docusign IAM Professional and higher plans.

With Advanced Web Forms, you can:

- [Prefill a Web Form By URL (using an API)](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=kup1721242003741.html&_LANG=enus)
- [Add a Document Rule to a Web Form](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=qvz1729637084872.html&_LANG=enus)
- [Build a Multi-Recipient Web Form](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=inc1745950777430.html&_LANG=enus)
- [Send a Web Form (using an API)](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=con56dbba74-0e1a-4356-a265-bf7ec87913e9.html&_LANG=enus)
