# Share Access to a Web Form Configuration

Learn how to share access to a Web Form by assigning view, edit, manage, or remove permissions to users and groups on your account.

You can share access to forms with users and groups on an account.

Before you begin, confirm that you have permission to manage forms and know which form and access level (view, edit, or manage) to assign.

1. Go to the Templates page.
2. Select My Web Forms, Shared with Me, or All Web Forms.
3. Select More Actions (...) next to the form that you want to share and select Share.

   The Share this web form window opens.
4. In the Add user or group field, enter the names of the individuals or groups that you want to share the form with.
5. Select the type of access that you want to grant:

   - Can View: Get the URL for an activated form by selecting Get URL next to the form on the Web Forms list.
   - Can Edit: Edit and activate the form.
   - Can Manage: Manage form access and data. Creating forms also requires a separate permission managed by account administrators.
   - Remove: Revoke access to the form.
6. Select Save.

The user access settings are updated. Users and groups you selected now have the specified level of access to the form.
