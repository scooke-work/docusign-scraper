# Build a Web Form

Learn how to build a Web Form by choosing a starting point, customizing the fields and layout, and activating the form so it is ready to share.

You can build a form by choosing a starting point, customizing the fields and layout, and then activating the form so it is ready to share with recipients.

1. Go to the Templates page.
2. Select Start > Web Forms > Create Web Form.
3. Choose how you want to start:
   1. Select an eSignature template: Select an existing eSignature template from your library to start the Template-to-Form flow, or choose an industry-standard template with Web Forms enabled from the Docusign Web Forms Library by selecting the Choose from pre-built form library link.
   2. Create blank form: Build a standalone form to collect data and download the results, or use a standalone form with Docusign [Workflow Builder](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=gua1698120920620.html&_LANG=enus) workflows.
   3. Upload document: Select a document from your device to start the document-to-form flow.
4. Review and edit the imported fields as needed.

   - [Configure Web Form Fields](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=awi1670975283111.html&_LANG=enus).
   - [Add Rules to a Web Form](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=kff1668121038223.html&_LANG=enus).
5. When you finish building the form, [activate it](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=iri1669070601520.html&_LANG=enus).

Your form is ready to email or embed on your website.
