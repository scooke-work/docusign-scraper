# Send a Web Form

Learn how to send a Web Form, including prerequisites and step-by-step instructions for
sending the web form to recipients.

## Before you begin

- Have an active form that you have created or has been shared with you.
- The form must be linked to an envelope template.
- The form must not contain attachment fields.
- To send to multiple recipients, your account must have Advanced Web Forms or an IAM
  plan.

You can send a form that you created or that an administrator or form creator shared with you directly to recipients.

1. Go to the Templates page.
2. Select My Web Forms, Shared with Me, or  All Web Forms.
3. Choose the form that you want to send and select Use.
4. In the Use Web Form window, select the
   Send tab.
5. Enter the name and email for the recipients.
6. Select Next.
7. In Pre-fill Form Details, complete the fields assigned to you.

   Note: Fields marked with a red asterisk are required and
   must be completed before you can send the form.
8. Review the information and select Send Web Form.

The system generates an envelope immediately and sends a notification email to the
recipients. You can track the progress on the Agreements page
under All Agreements.
