# Add Rules to a Web Form

Read about how conditional and document rules control which fields and documents display in a Web Form based on recipient responses.

Conditional rules control which fields display on a form based on recipient responses. They target specific information by showing additional fields when certain conditions are met.

You can use conditional rules to show or hide fields, pages, or sections based on conditions, and import rules from a template or create them directly on the form for any content type.

You create rules using an "if...then" structure. For example, if the response to "Do you take medication"? is "Yes", show the "List your medications" field. If the response to "Are you married"? is "No", hide the "Spouse" field.

For more information about adding field rules, see the [Add a Field Rule to a Web Form](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=inf1660603502823.html&_LANG=enus) guide.

You can also set up document rules for forms. Document rules control additional documents that only some recipients must review. For more information about adding document rules, see the [Add a Document Rule to a Web Form](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=qvz1729637084872.html&_LANG=enus) guide.
