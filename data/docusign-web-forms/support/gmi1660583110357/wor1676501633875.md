# The Web Forms Builder Properties and Rules Tab

Read about the Properties and Rules tabs in the Web Forms Builder, which allow configuration of form fields and control of component display based on recipient responses.

The Properties and Rules tabs display on the right side of the Web Forms Builder. Configure form fields in Properties or Rules and view the changes in the center panel.

![A field name being edited in the Properties tab of the Web Forms Builder.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gmi1660583110357/page/images/wf-properties-tab.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTIxMTYsInNoZWFmIjoiZ21pMTY2MDU4MzExMDM1NyJ9.YUxJ8n-NbZUH1hrF8ngQkP9ACn2xb6ipSa5uN4niQ9Q&_LANG=enus "A field name being edited in the Properties tab of the Web Forms Builder.")

Each field type has unique properties. For information about the field properties, see the [Configure Web Form Fields](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=awi1670975283111.html&_LANG=enus) guide.

You can also add rules to form fields to control component display based on recipient responses. For more information about field rules, see the [Add Rules to a Web Form](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=kff1668121038223.html&_LANG=enus) guide.
