# Troubleshooting

The Web Forms API is available in all developer accounts, but only in certain production account plans. Contact [Docusign Support](https://support.docusign.com/s/?language=en_US) or your account manager to find out whether the Web Forms API is available for your production account plan.

This topic lists error codes and error messages that may be returned in response to Web Forms API requests or SDK method calls.

If you are unable to resolve an issue with your application by following the troubleshooting recommendations, contact [Docusign Support](https://support.docusign.com/s/?language=en_US) for assistance.
