# Review Contracts Using the AI-Assisted Review add-in for Agreement Desk

Learn how to review contracts using the Docusign AI-Assisted Review add-in for Agreement Desk. Once a playbook has been run, users can review suggestions and make alterations to a contract based on analysis results.

## Before you begin

[Run a Playbook in the AI-Assisted Review add-in for Agreement Desk](https://support.docusign.com/s/document-item?bundleId=mapfd7284e6-b699-4527-8651-a133914111fd&topicId=tsk4aa134b9-92cc-4c36-b5dc-18d2f2c1b006.html&_LANG=enus "Learn how to run a playbook in the Docusign AI-Assisted Review add-in for Agreement Desk. Playbooks are collections of rules that can be configured to recommend markup and clauses in agreements. Once a contract has been analyzed by AI-Assisted Review, users can apply markup to the contract from suggestions made by the add-in.")

Once a playbook has been run, users can review suggestions for the selected contract. AI-Assisted Review can be used to substitute contract clauses and language according to playbook rules.

Note:

- Time to complete contract analysis using playbooks in AI-Assisted Review may vary. This is dependent on the size and complexity of the playbook and the contract being analyzed by AI-Assisted Review.
- AI-Assisted Review can only be used with DOCX files. Macros and content control are not supported in AI-Assisted Review, and may cause errors.

Agreement Desk users with the required permissions can access AI-Assisted Review in several ways.

1. From Agreement Desk:
   1. Select the Agreements tab, and choose Requests.
   2. Select a request to open it.
   3. Select the Documents tab.
   4. Select the ![Options](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Overflow-Vertical.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5MzcsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.fG2j1wxo6GP5bb3DipYgB-y9y8z_Yc64vZyeTEpfo3w&_LANG=enus "Options") menu, and choose Edit in Word. Or, open the document preview, and select Edit in Word.
   5. Open the downloaded agreement in Microsoft Word, and if prompted, log into the AI-Assisted Review add-in.
2. From a document located on your local storage:
   1. Locate the document on your local storage.
   2. Open the document.

      The document opens in Microsoft Word.
   3. Select the Docusign AI-Assisted Review add-in from the Home ribbon in Microsoft Word. If prompted, log into the AI-Assisted Review add-in.
3. After opening the agreement using one of these methods, follow these steps to review your agreement using AI-Assisted Review:
   1. Select AI-Assisted Review.

      The AI-Assisted Review window displays, and the document is ready for review by the add-in.
   2. Select the Playbooks tab, and then select a playbook to run.

      The selected playbook displays, including all rules that are part of the selected playbook.
   3. Select Run All to run the playbook.

      AI-Assisted Review analyzes the contract, and displays results once analysis is complete.
   4. Review the results of the contract analysis:

      - Needs Review indicates that a clause or section has not met the standards of a playbook rule. To resolve this result without making changes, select Resolve. To apply the provided suggestion, select Apply. To leave an inline comment, select the ![Add a Comment](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Add-Comment.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5MzcsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.fG2j1wxo6GP5bb3DipYgB-y9y8z_Yc64vZyeTEpfo3w&_LANG=enus "Add a Comment") icon.

        Note: If there are multiple sections in the agreement where AI-Assisted Review finds results, they will be displayed under the associated rule.
      - To apply all suggestions made by AI-Assisted Review, select the dropdown for Run All, and choose Apply All Suggestions. To apply all suggestions made by AI-Assisted Review and comments, select the dropdown for Run All, and choose Apply All Suggestions and Comments.
      - Passed indicates that a clause or selection has met the standards of a playbook rule. You can view how the add-in made this determination by selecting the rule. To resolve the result, select Resolve.
   5. Once all changes have been made, you are ready to save your agreement. The applicable save method will depend on how you accessed AI-Assisted Review:

      - Users who access AI-Assisted Review using Edit in Word can select Save to Request. The document is saved back to Agreement Desk as a new version.
      - Users who access AI-Assisted Review using a document from local storage should select Save in Microsoft Word to save the document locally. Once the document is saved, it can be uploaded manually to Agreement Desk.

You have successfully analyzed and made changes to a contract using AI-Assisted Review.
