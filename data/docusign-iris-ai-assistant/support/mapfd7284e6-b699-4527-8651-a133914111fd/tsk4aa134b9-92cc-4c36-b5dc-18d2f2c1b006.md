# Run a Playbook in the AI-Assisted Review add-in for Agreement Desk

Learn how to run a playbook in the Docusign AI-Assisted Review add-in for Agreement Desk. Playbooks are collections of rules that can be configured to recommend markup and clauses in agreements. Once a contract has been analyzed by AI-Assisted Review, users can apply markup to the contract from suggestions made by the add-in.

## Before you begin

[Install the AI-Assisted Review Add-In for Agreement Desk](https://support.docusign.com/s/document-item?bundleId=mapfd7284e6-b699-4527-8651-a133914111fd&topicId=con7dae9192-c00f-4290-abb2-11b9068012b3.html&_LANG=enus "Learn how to install the Docusign AI-Assisted Review add-in for Agreement Desk.")

In the Docusign AI-Assisted Review add-in for Agreement Desk, playbooks are collections of rules that can be configured to recommend markup. Users can run playbooks to analyze contracts and agreements for discrepancies or issues. When contract terms deviate from a playbook rule, AI-Assisted Review will suggest markup to bring contract language into compliance.

Once a contract has been analyzed according to the rules in a playbook, suggested markup can be applied. AI-Assisted Review can make suggestions for sections of contract language, enabling faster agreement cycles.

Agreement Desk users with the required permissions can access AI-Assisted Review in several ways.

1. From Agreement Desk:
   1. Select the Agreements tab, and choose Requests.
   2. Select a request to open it.
   3. Select the Documents tab.
   4.    5. Open the downloaded agreement in Microsoft Word, and if prompted, log into the AI-Assisted Review add-in.
2. From a document located on your local storage:
   1. Locate the document on your local storage.
   2. Open the document.

      The document opens in Microsoft Word.
   3. Select the Docusign AI-Assisted Review add-in from the Home ribbon in Microsoft Word. If prompted, log into the AI-Assisted Review add-in.
3. After opening the agreement using one of these methods, follow these steps to run a playbook using AI-Assisted Review:
   1. Select AI-Assisted Review.

      The AI-Assisted Review window displays, and the document is ready for review by the add-in.
   2. Select the Playbooks tab, and then choose a playbook to
      run.
