# Create a Playbook in the AI-Assisted Review add-in for Agreement Desk

Learn how to create a playbook in the Docusign AI-Assisted Review add-in for Agreement Desk. Playbooks are collections of rules that enable the Docusign AI-Assisted Review add-in for Agreement Desk to review and markup agreements.

Playbooks can be created to manage rules, suggest changes, and insert preferred language in AI-Assisted Review. Playbooks accelerate contract review and negotiation, empowering teams to manage lightweight review and negotiation without jeopardizing legal compliance.

Note:

- AI-Assisted Review currently supports AI-powered review of agreements in English, French, German, Spanish, and Brazilian Portuguese. The add-in will automatically detect the most commonly occurring language in a document, and return suggested edits in that language. We recommend that your document, playbook, and Chat input language match, to ensure consistent results. For example, users should review French language agreements using French language playbooks, and French language Chat input.
- AI-Assisted Review's user interface is available in all 14 languages supported in Docusign. Users can update their language settings in Docusign. For more information on user language preference, see [Update Your Regional Settings in My Profile](https://support.docusign.com/s/document-item?bundleId=jux1643235969954&topicId=xlc1717521107193.html&_LANG=enus)﻿. Docusign Administrators can set which languages are enabled for users. For more information on language settings in Docusign, see [Time Zone, Date Format, and Language](https://support.docusign.com/s/document-item?bundleId=&pik1583277475390&topicId=wyk1583277329566.html&_LANG=enus).
- Only certain types of user permission profiles have the ability to create or edit playbooks. For more information about user permission profiles for playbooks, see [Set Playbook Permissions for the Docusign AI-Assisted Review add-in for Agreement Desk](https://support.docusign.com/s/document-item?bundleId=mapfd7284e6-b699-4527-8651-a133914111fd&topicId=tsk8b028ec5-3d65-45a6-aec6-6b3540565b75.html&_LANG=enus "Read about the permissions that are required for users and groups to use playbooks and Chat in the Docusign AI-Assisted Review add-in for Agreement Desk. User and group permissions determine playbook and Chat access in the Docusign AI-Assisted Review add-in for Agreement Desk.").

Agreement Desk users with the required permissions can access AI-Assisted Review for playbook creation in several ways.

1. From Agreement Desk:
   1. Select the Agreements tab, and choose Requests.
   2. Select a request to open it.
   3. Select the Documents tab.
   4. Select the ![Options](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Overflow-Vertical.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5NDEsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.BVtWIZVnP-aEmSD5Q3qaV0LmNT_pXU4kBWNBbqtz-aM&_LANG=enus "Options") menu, and choose Edit in Word. Or, open the document preview, and select Edit in Word.
   5. Open the downloaded agreement in Microsoft Word, and if prompted, log into the AI-Assisted Review add-in.
2. From a document located on your local storage:
   1. Locate the document on your local storage.
   2. Open the document.

      The document opens in Microsoft Word.
   3. Select the Docusign AI-Assisted Review add-in from the Home ribbon in Microsoft Word. If prompted, log into the AI-Assisted Review add-in.
3. After opening the agreement using one of these methods, follow these steps to create a playbook using AI-Assisted Review:
   1. Select AI-Assisted Review.

      The AI-Assisted Review window displays, and the document is ready for review by the add-in.
   2. Select Playbooks.

      AI-Assisted Review displays all available playbooks.
   3. Select New Playbook or Generate Playbook.

      The New Playbook or Generate Playbook screen displays.
   4. To create a new playbook:

      - Enter a title and description for your new playbook.
      - Select Add Rule to add a rule to your new playbook.
      - Enter your rule for the new playbook.

      To generate a new playbook from an uploaded document:

      - Select Upload File to upload a DOCX or PDF document to act as a playbook template.
      - Choose a document from your local storage and select Upload.
      - Select Generate Playbook.

      Note:
      - Playbook rules should use plain language that describes how AI-Assisted Review should analyze your agreements. For more guidance on writing playbook rules, see [Best Practices for Playbook Rule Writing in the AI-Assisted Review add-in for Agreement Desk](https://support.docusign.com/s/document-item?bundleId=mapfd7284e6-b699-4527-8651-a133914111fd&topicId=fje1726766463553-fwcby1ke.html&_LANG=enus).
      - To refine your new playbook rule automatically, select Refine with AI. Selecting Refine with AI will suggest improvements to your rule based on these best practices for more specific, consistent results.
   5. (Optional) Select the Optional settings dropdown to add a Rule title, Internal notes, or Custom comment to your playbook rule.
   6. Select Test Rule to test your playbook rule on the contract.

      AI-Assisted Review displays the results of the rule, including reason and suggestions for markup. Contract sections that do not pass the playbook rule are shown with strikethroughs, and suggested replacement terms are bolded. If there are multiple sections in the agreement where AI-Assisted Review finds results, they will be displayed under the associated rule.
   7. Once all options and adjustments have been made, select Save.

      AI-Assisted Review returns you to your new playbook.
   8. To add more rules, select Add Rule, and follow Steps D through G.
   9. Once all rules have been submitted and tested, select Save.

You have successfully created a new playbook for AI-Assisted Review. You can select this playbook and run it against your contracts.
