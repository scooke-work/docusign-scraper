# Generate Playbooks Using the AI-Assisted Review add-in for Agreement Desk

Learn how to use the AI-Assisted Review add-in for Agreement Desk to generate playbooks. Playbooks are collections of rules that enable the Docusign AI-Assisted Review add-in for Agreement Desk to review and mark up agreements.

In AI-Assisted Review, Playbooks accelerate contract review and negotiation, empowering teams to manage lightweight review and negotiation without jeopardizing legal compliance. The AI-Assisted Review add-in can analyze your contracts and create playbooks and rules based on that analysis.

Note:

- Playbooks can only be generated from DOCX and PDF documents.
- Playbook generation in AI-Assisted review can only generate playbooks based on a single document.
- Playbook generation will use the first 100 pages of the uploaded reference document.

Agreement Desk users with the required permissions can access AI-Assisted Review for playbook generation in several ways.

1. From Agreement Desk:
   1. Select the Agreements tab, and choose Requests.
   2. Select a request to open it.
   3. Select the Documents tab.
   4. Select the ![Options](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Overflow-Vertical.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5NDQsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.pdBImVRisuKDABg8_1MQRLdk0P7S5tryZ9UKOnEK2dg&_LANG=enus "Options") menu, and choose Edit in Word. Or, open the document preview, and select Edit in Word.
   5. Open the downloaded agreement in Microsoft Word, and if prompted, log into the AI-Assisted Review add-in.
2. From a document located on your local storage:
   1. Locate the document on your local storage.
   2. Open the document.

      The document opens in Microsoft Word.
   3. Select the Docusign AI-Assisted Review add-in from the Home ribbon in Microsoft Word. If prompted, log into the AI-Assisted Review add-in.
3. After opening the agreement using one of these methods, follow these instructions to generate a new playbook in AI-Assisted Review:
   1. In the AI-Assisted Review window, log on to Docusign.

      The AI-Assisted Review window displays.
   2. Select the Playbooks tab, and choose Generate Playbook.
   3. Select Upload File to choose an agreement from your local storage. Once an agreement has been chosen, select Generate Playbook.

      The selected agreement is uploaded to Docusign AI-Assisted Review. The add-in analyzes your contract and returns a generated playbook title, description, and a set of playbook rules, based on the contents of the uploaded agreement.
   4. Select your options for your newly generated playbook:

      - To add a rule to your newly generated playbook, select Add Rule. For more information on writing playbook rules, see [Best Practices for Playbook Rule Writing in the AI-Assisted Review add-in for Agreement Desk](https://support.docusign.com/s/document-item?bundleId=mapfd7284e6-b699-4527-8651-a133914111fd&topicId=fje1726766463553-fwcby1ke.html&_LANG=enus).
      - To edit any playbook rules in your generated playbook, select the ![Options](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Overflow-Vertical.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5NDQsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.pdBImVRisuKDABg8_1MQRLdk0P7S5tryZ9UKOnEK2dg&_LANG=enus "Options")icon, and choose Edit Rule. For more information on editing playbook rules, see [Edit Playbook Rules in the Docusign AI-Assisted Review add-in for Agreement Desk](https://support.docusign.com/s/document-item?bundleId=mapfd7284e6-b699-4527-8651-a133914111fd&topicId=tsk31bc3eb4-f708-42aa-ab14-983e1ab3cb37.html&_LANG=enus).
      - To re-order rules in your playbook, select and hold the ![Order Selector](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Gripper_Bar.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5NDQsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.pdBImVRisuKDABg8_1MQRLdk0P7S5tryZ9UKOnEK2dg&_LANG=enus "Order Selector")icon, and drag the rule into the desired order.
   5. Once all changes have been made, select Save to save your new playbook.

You have successfully used the AI-Assisted Review add-in to generate a playbook based on an uploaded contract. The add-in displays an indication that your new playbook has been created. Your new playbook can be used on agreements by selecting Run All.
