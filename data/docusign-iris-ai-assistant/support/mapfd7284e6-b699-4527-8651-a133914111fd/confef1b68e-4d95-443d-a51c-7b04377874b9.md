# Docusign AI-Assisted Review for Agreement Desk

Read about the Docusign AI-Assisted Review add-in for Agreement Desk. The Docusign AI-Assisted Review add-in for Agreement Desk enables AI-powered contract review and markup suggestions.

Note: This documentation is for Docusign AI-Assisted Review for Agreement Desk. This feature is also available for Docusign CLM customers. For more information on AI-Assisted Review for Docusign CLM, see [Docusign CLM AI-Assisted Review Add-In for Microsoft Word](https://support.docusign.com/s/document-item?bundleId=alb1642965087302&topicId=pjw1724873457715.html&_LANG=enus).

The Docusign AI-Assisted Review add-in for Microsoft Word with Agreement Desk enables contract clause corrections, suggestions, and markup. The add-in uses AI to generate, substitute, explain, and add contract language for reviewers and stakeholders.

Playbooks can be created to review contracts against pre-approved rules, and get markup suggestions to align language with requirements. Draft agreements and contracts can be easily updated, accelerating contract review and negotiation.

Note:

- AI-Assisted Review is available on IAM for Enterprise and IAM for Sales plans. For more details on availability and features for your Docusign IAM plan, see [Support for AI-Assisted Review in Agreement Desk Plans](https://support.docusign.com/s/document-item?bundleId=mapfd7284e6-b699-4527-8651-a133914111fd&topicId=con74e34c35-eb00-48d8-a66b-5723cf2a8273.html&_LANG=enus "Learn about the features and permissions that are available through Agreement Desk plans for AI-Assisted Review.").
- AI-Assisted Review currently supports AI-powered review of agreements in English, French, German, Spanish, and Brazilian Portuguese. The add-in will automatically detect the most commonly occurring language in a document, and return suggested edits in that language. We recommend that your document, playbook, and Chat input language match, to ensure consistent results. For example, users should review French language agreements using French language playbooks, and French language Chat input.
- AI-Assisted Review’s user interface is available in all 14 languages supported in Docusign. Users can update their language settings in Docusign. For more information on user language preference, see [Update Your Regional Settings in My Profile](https://support.docusign.com/s/document-item?bundleId=jux1643235969954&topicId=xlc1717521107193.html&_LANG=enus)﻿. Docusign Administrators can set which languages are enabled for users. For more information on language settings in Docusign, see [Time Zone, Date Format, and Language](https://support.docusign.com/s/document-item?bundleId=&pik1583277475390&topicId=wyk1583277329566.html&_LANG=enus).
- Mobile, web, and tablet versions of Microsoft Word are not supported for this add-in. For managed installations of the Docusign AI-Assisted Contract Review add-in for Microsoft Word, a license of Office that includes Exchange and Office 365 for Desktop is required. The Docusign AI-Assisted Review add-in for Agreement Desk supports the following versions of Microsoft Office and Microsoft Word:

  - Office on Windows (build 16.0.16130+, connected to an active Microsoft 365 subscription)
  - Office 2021 or later on Windows (or versions greater than 2019 on a retail license after installing the correct WebView)
  - Office on Mac (build 16.70+, connected to an active Microsoft 365 subscription)
  - Office 2019 or later on Mac
- Users of the Docusign AI-Assisted Review for Agreement Desk will need to re-authenticate every 30 days.
- AI-Assisted Review can only be used with DOCX files. Macros and content control are not supported in AI-Assisted Review, and may cause errors.
- For this feature, there may be future limits or throttling of usage to maintain platform stability and performance.
