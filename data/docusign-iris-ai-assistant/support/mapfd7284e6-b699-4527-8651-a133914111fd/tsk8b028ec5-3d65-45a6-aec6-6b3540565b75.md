# Set Playbook Permissions for the AI-Assisted Review add-in for Agreement Desk

Read about the permissions that are required for users and groups to use playbooks and Chat in the Docusign AI-Assisted Review add-in for Agreement Desk. User and group permissions determine playbook and Chat access in the Docusign AI-Assisted Review add-in for Agreement Desk.

To use playbooks and Chat in the Docusign AI-Assisted Review add-in for Agreement Desk, an account administrator must assign specific permissions to users. Playbook permissions can be granted to individual users or groups, and permission levels define access to playbooks.

Note: To manage or view playbooks, users must have a full IAM license.

1. Log on to your Docusign account.
2. Select the Admin tab.

   The Accounts view displays.
3. Navigate to Users & Groups, and select your options:

   - To set playbook permissions for users, choose Users.
   - To set playbook permissions for groups, choose Groups.
4. Locate the user or group you want to adjust playbook permissions for, and select Actions > Manage Playbook Access.

   The Playbook Access tab for the selected user or group displays.
5. Select the playbook permissions for the selected user or group from the Set permissions dropdown menu:

   | Playbook Permissions | Features in the Docusign AI-Assisted Review add-in for Agreement Desk |
   | --- | --- |
   | Can Manage Playbooks | - Create playbooks and playbook rules - View playbooks and playbook rules - Edit playbooks and playbook rules - Run playbooks and playbook rules - Delete playbooks and playbook rules - Can use Chat capabilities |
   | Can View Playbooks | - View playbooks and playbook rules - Run playbooks and playbook rules - Can use Chat capabilities |
   | None | - Cannot use AI-Assisted Review features |
6. Select Save Changes.

You have successfully set playbook permissions for the selected user or group.
