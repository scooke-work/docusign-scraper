# Edit Playbook Rules in the Docusign AI-Assisted Review add-in for Agreement Desk

Learn how to edit existing playbook rules in the Docusign AI-Assisted Review add-in for Agreement Desk. Playbooks are collections of rules that enable the Docusign AI-Assisted Review add-in for Agreement Desk to review and markup agreements.

Playbook rules in the AI-Assisted Review can be edited anytime you need to update a rule.

Note:

- AI-Assisted Review currently supports AI-powered review of agreements in English, French, German, Spanish, and Brazilian Portuguese. The add-in will automatically detect the most commonly occurring language in a document, and return suggested edits in that language. We recommend that your document, playbook, and Chat input language match, to ensure consistent results. For example, users should review French language agreements using French language playbooks, and French language Chat input.
- AI-Assisted Review's user interface is available in all 14 languages supported in Docusign. Users can update their language settings in Docusign. For more information on user language preference, see [Update Your Regional Settings in My Profile](https://support.docusign.com/s/document-item?bundleId=jux1643235969954&topicId=xlc1717521107193.html&_LANG=enus)﻿. Docusign Administrators can set which languages are enabled for users. For more information on language settings in Docusign, see [Time Zone, Date Format, and Language](https://support.docusign.com/s/document-item?bundleId=&pik1583277475390&topicId=wyk1583277329566.html&_LANG=enus).
- Only certain types of user permission profiles have the ability to create or edit playbooks. For more information about user permission profiles for playbooks, see [Set Playbook Permissions for the Docusign AI-Assisted Review add-in for Agreement Desk](https://support.docusign.com/s/document-item?bundleId=mapfd7284e6-b699-4527-8651-a133914111fd&topicId=tsk8b028ec5-3d65-45a6-aec6-6b3540565b75.html&_LANG=enus "Read about the permissions that are required for users and groups to use playbooks and Chat in the Docusign AI-Assisted Review add-in for Agreement Desk. User and group permissions determine playbook and Chat access in the Docusign AI-Assisted Review add-in for Agreement Desk.").

Agreement Desk users with the required permissions can access AI-Assisted Review to edit playbook rules in several ways.

1. From Agreement Desk:
   1. Select the Agreements tab, and choose Requests.
   2. Select a request to open it.
   3. Select the Documents tab.
   4. Select the ![Options](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Overflow-Vertical.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5NjUsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.PHqA5ttj3LKlir3bwG2Q6bmvjSgMIutpBwmdjAwwQDQ&_LANG=enus "Options") menu, and choose Edit in Word. Or, open the document preview, and select Edit in Word.
   5. Open the downloaded agreement in Microsoft Word, and if prompted, log into the AI-Assisted Review add-in.
2. From a document located on your local storage:
   1. Locate the document on your local storage.
   2. Open the document.

      The document opens in Microsoft Word.
   3. Select the Docusign AI-Assisted Review add-in from the Home ribbon in Microsoft Word. If prompted, log into the AI-Assisted Review add-in.
3. After opening the agreement using one of these methods, follow these steps to edit playbook rules using AI-Assisted Review:
   1. Select AI-Assisted Review.

      The AI-Assisted Review window displays.
   2. Select Playbooks.

      AI-Assisted Review displays all available playbooks.
   3. Select a playbook.

      AI-Assisted Review displays all the rules for the selected playbook.
   4. Choose a playbook rule, and select the ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Pencil.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5NjUsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.PHqA5ttj3LKlir3bwG2Q6bmvjSgMIutpBwmdjAwwQDQ&_LANG=enus) icon. Or, when reviewing a playbook, choose Edit, and then select the ![Options](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Overflow-Vertical.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5NjUsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.PHqA5ttj3LKlir3bwG2Q6bmvjSgMIutpBwmdjAwwQDQ&_LANG=enus "Options") menu, and select Edit Rule.

      The Edit Rule screen displays.
   5. Make your changes to the selected rule.
   6. (Optional) Select the Optional settings dropdown to add a Rule title, Internal notes, or Custom comment to your playbook rule.
   7. Select Test Rule to test your edited playbook rule on the contract.

      Note:
      - Playbook rules should use plain language that describes how AI-Assisted Review should analyze your agreements. For more guidance on writing playbook rules, see [Best Practices for Playbook Rule Writing in the Docusign AI-Assisted Review add-in for Agreement Desk](https://support.docusign.com/s/document-item?bundleId=mapfd7284e6-b699-4527-8651-a133914111fd&topicId=fje1726766463553-fwcby1ke.html&_LANG=enus "Learn about best practices and guidance for writing playbook rules in the Docusign AI-Assisted Review add-in for Agreement Desk.").
      - To refine your new playbook rule automatically, select Refine with AI. Selecting Refine with AI will suggest improvements to your rule based on these best practices for more specific, consistent results.

      AI-Assisted Review displays the results of the rule, including reasoning, comments, and suggestions for markup. Contract terms that do not pass the playbook rule are shown with strikethroughs, and suggested replacement terms are bolded. If there are multiple sections in the agreement where AI-Assisted Review finds results, they will be displayed under the associated rule.
   8. Once all changes have been made, select Save on the Edit Rule screen.

      Note: If editing a rule through the ![Options](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Overflow-Vertical.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5NjUsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.PHqA5ttj3LKlir3bwG2Q6bmvjSgMIutpBwmdjAwwQDQ&_LANG=enus "Options") menu, select Save to return to the playbook.
   9. Repeat Steps D through H for any other playbook rules that require changes.

You have successfully edited a playbook rule for AI-Assisted Review. You can select this playbook and run it against any contracts you have downloaded from Agreement Desk for AI-Assisted Review.
