# Export and Import Playbooks in the Docusign AI-Assisted Review add-in for Agreement Desk

Learn how to export or import your organization’s playbooks in the Docusign AI-Assisted Review add-in for Agreement Desk.

Playbooks in the Docusign AI-Assisted Review add-in for Agreement Desk can be exported or imported. Users with the correct permissions can export and import playbooks to create copies or migrate playbooks between environments.

Note:

- Only certain types of user permission profiles have the ability to delete playbooks and playbook rules. For more information about user permission profiles for playbooks, see [Set Playbook Permissions for the Docusign AI-Assisted Review add-in for Agreement Desk](https://support.docusign.com/s/document-item?bundleId=mapfd7284e6-b699-4527-8651-a133914111fd&topicId=tsk8b028ec5-3d65-45a6-aec6-6b3540565b75.html&_LANG=enus "Read about the permissions that are required for users and groups to use playbooks and Chat in the Docusign AI-Assisted Review add-in for Agreement Desk. User and group permissions determine playbook and Chat access in the Docusign AI-Assisted Review add-in for Agreement Desk.").
- When exporting playbooks, all playbooks and rules will be included in the export. Users cannot export a single playbook or edit playbooks outside of the add-in before importing them.

1. From Agreement Desk:
   1. Select the Agreements tab, and choose Requests.
   2. Select a request to open it.
   3. Select the Documents tab.
   4. Select the ![Options](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Overflow-Vertical.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5NzUsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.LISy6oPS3nmZwcXC9A78n-0z-HDr6nviY9162efcmd8&_LANG=enus "Options") menu, and choose Edit in Word. Or, open the document preview, and select Edit in Word.
   5. Open the downloaded agreement in Microsoft Word, and if prompted, log into the AI-Assisted Review add-in.
2. From a document located on your local storage:
   1. Locate the document on your local storage.
   2. Open the document.

      The document opens in Microsoft Word.
   3. Select the Docusign AI-Assisted Review add-in from the Home ribbon in Microsoft Word. If prompted, log into the AI-Assisted Review add-in.
3. After opening the agreement using one of these methods, follow these steps to export or import a playbook using AI-Assisted Review:
   1. Select Playbooks.

      AI-Assisted Review displays all available playbooks.
   2. Select the ![Options](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Overflow-Vertical.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5NzUsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.LISy6oPS3nmZwcXC9A78n-0z-HDr6nviY9162efcmd8&_LANG=enus "Options") menu, and choose your options:

      - To export multiple playbooks, choose Export Playbooks. To export a single playbook, select the playbook, and choose Export Playbook from the ![Options](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Overflow-Vertical.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5NzUsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.LISy6oPS3nmZwcXC9A78n-0z-HDr6nviY9162efcmd8&_LANG=enus "Options")menu. Select Copy to copy the code for the playbook.
      - To import multiple playbooks, choose Import Playbooks, paste the code for the playbook into the field, and select Import. To import a single playbook, paste the text from a single playbook export into the field, and select Import. Once the playbook or playbooks have been imported, select Close.

      Note:
      - Once a playbook has been imported, the AI-Assisted Review add-in must be reloaded to reflect the changes. If an imported playbook has the same name as a current playbook, a versioning indicator will be added.
      - To copy playbooks within production environments, export the playbooks and import them within the same add-in environment.
      - To migrate playbooks from Demo to Production environments, you will need to log in to the target account, before importing the copied playbook code.

You have successfully exported or imported a playbook from AI-Assisted Review. For exporting playbooks, a code for the playbook is provided to the user. For importing playbooks, a message displays indicating that the playbook was successfully imported.
