# Preview an Agreement Using the Docusign AI-Assisted Review Add-In for Agreement Desk

Learn how to use AI-Assisted Review features while previewing a document in Agreement Desk.

When previewing a document in Agreement Desk, you can use AI-Assisted Review’s tools to answer questions, analyze terms, and suggest edits from within the Agreement Desk request. If further changes are needed, the selected agreement can be opened in Microsoft Word for edits.

1. In Agreement Desk, select the Agreements tab, and choose Requests.
2. Select a request to open it.
3. Select the Documents tab.
4. Open a document from the Overview tab, or choose a document and select the ![Options](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Overflow-Vertical.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5MjgsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.R7X4rwH9vRfQr8B4Nw-SIa5F0t0OZPQEjLLGgZh30Ac&_LANG=enus "Options")menu, and choose Preview.

   Agreement Desk displays a preview of the selected document, along with AI-Assisted Review. You can access AI-Assisted Review's features for chat- and playbook-based functionality.
5. (Optional) To make further changes or to apply suggestions from chat or playbooks, select Edit in Word. Open the downloaded agreement in Microsoft Word, and if prompted, log in to the AI-Assisted Review add-in.
