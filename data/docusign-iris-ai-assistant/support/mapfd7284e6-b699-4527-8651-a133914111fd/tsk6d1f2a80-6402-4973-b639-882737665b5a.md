# Open a Document from Agreement Desk in AI-Assisted Review

Learn how to open a document from an Agreement Desk request in the AI-Assisted Review add-in.

## Before you begin

[Install the AI-Assisted Review Add-In for Agreement Desk](https://support.docusign.com/s/document-item?bundleId=mapfd7284e6-b699-4527-8651-a133914111fd&topicId=con7dae9192-c00f-4290-abb2-11b9068012b3.html&_LANG=enus "Learn how to install the Docusign AI-Assisted Review add-in for Agreement Desk.")

Once the AI-Assisted Review add-in has been installed for Agreement Desk, documents can be opened in the add-in.

1. Select the Agreements tab, and choose Requests.
2. Select a request to open it.
3. Select the Documents tab.
4. Select the ![Options](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Overflow-Vertical.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5MTUsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.Q0ynnfdEftKEqnP1D1VFNQGDqtrinfS_NKAqV_3AYrc&_LANG=enus "Options") menu, and choose Edit in Word. Or, open the document preview, and select Edit in Word.
5. Open the downloaded agreement in Microsoft Word, and if prompted, log in to the AI-Assisted Review add-in.
6. Select AI-Assisted Review.

You have successfully opened a document from Agreement Desk in Microsoft Word. The AI-Assisted Review window displays, and the document is ready for review by the add-in.
