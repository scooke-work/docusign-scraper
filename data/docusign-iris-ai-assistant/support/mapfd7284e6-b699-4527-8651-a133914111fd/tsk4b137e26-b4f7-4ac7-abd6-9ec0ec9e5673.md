# Delete a Playbook or Playbook Rule in the Docusign AI-Assisted Review add-in for Agreement Desk

Learn how to delete existing playbooks or playbook rules in the Docusign AI-Assisted Review add-in for Agreement Desk. Playbooks are collections of rules that enable AI-Assisted Review to review and markup agreements.

Playbooks and playbook rules in the Docusign AI-Assisted Review add-in for Agreement Desk can be deleted. Deleting a playbook means that it can no longer be run as part of AI-Assisted Review.

Note: Only certain types of user permission profiles have the ability to delete playbooks and playbook rules. For more information about user permission profiles for playbooks, see [Set Playbook Permissions for the Docusign AI-Assisted Review add-in for Agreement Desk](https://support.docusign.com/s/document-item?bundleId=mapfd7284e6-b699-4527-8651-a133914111fd&topicId=tsk8b028ec5-3d65-45a6-aec6-6b3540565b75.html&_LANG=enus "Read about the permissions that are required for users and groups to use playbooks and Chat in the Docusign AI-Assisted Review add-in for Agreement Desk. User and group permissions determine playbook and Chat access in the Docusign AI-Assisted Review add-in for Agreement Desk.").

Agreement Desk users with the required permissions can access AI-Assisted Review to delete a playbook or playbook rules in several ways.

1. From Agreement Desk:
   1. Select the Agreements tab, and choose Requests.
   2. Select a request to open it.
   3. Select the Documents tab.
   4. Select the ![Options](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Overflow-Vertical.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5NzAsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.IiJ0sQiJ8FRYNWzCqW7H9FNtwuE4YdKHwCfe3_T5FHc&_LANG=enus "Options") menu, and choose Edit in Word. Or, open the document preview, and select Edit in Word.
   5. Open the downloaded agreement in Microsoft Word, and if prompted, log into the AI-Assisted Review add-in.
2. From a document located on your local storage:
   1. Locate the document on your local storage.
   2. Open the document.

      The document opens in Microsoft Word.
   3. Select the Docusign AI-Assisted Review add-in from the Home ribbon in Microsoft Word. If prompted, log into the AI-Assisted Review add-in.
3. After opening the agreement using one of these methods, follow these steps to delete a playbook or playbook ruleusing AI-Assisted Review:
   1. Select Playbooks.

      AI-Assisted Review displays all available playbooks.
   2. Select a playbook.

      AI-Assisted Review displays all the rules for the selected playbook.
   3. Select your options:

      - To delete an entire playbook, select the ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Overflow-Vertical.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5NzAsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.IiJ0sQiJ8FRYNWzCqW7H9FNtwuE4YdKHwCfe3_T5FHc&_LANG=enus) menu, and choose Delete.
      - To delete a playbook rule, select Edit, hover your cursor over the rule, and select the ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Trash.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5NzAsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.IiJ0sQiJ8FRYNWzCqW7H9FNtwuE4YdKHwCfe3_T5FHc&_LANG=enus) icon.

      AI-Assisted Review displays a warning asking if you want to delete the selected playbook.
   4. To confirm deletion of the selected playbook or playbook rule, select Delete.

You have successfully deleted a playbook or playbook rule. Deleted playbooks and playbook rules will no longer run as part of a playbook used in AI-Assisted Review.
