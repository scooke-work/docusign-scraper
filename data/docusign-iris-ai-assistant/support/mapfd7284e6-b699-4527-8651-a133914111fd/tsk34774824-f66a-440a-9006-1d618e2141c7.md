# Chat in the Docusign AI-Assisted Review add-in for Agreement Desk

Learn how to use chat in AI-Assisted Review to review and answer questions on contracts, analyze terms, suggest edits, and draft new language.

The chat experience in Docusign AI-Assisted Review enables users to quickly understand agreements and their terms, generate markup, modify clauses, and add language to a contract. Users can enter prompts and questions into chat to get answers, analyze terms, suggest edits, and draft new contract language. Responses can be copied from the add-in, and distributed to stakeholders as needed.

Note: AI-Assisted Review currently supports AI-powered review of agreements in English, French, German, Spanish, and Brazilian Portuguese. The add-in will automatically detect the most commonly occurring language in a document, and return suggested edits in that language. We recommend that your document, playbook, and Chat input language match, to ensure consistent results. For example, users should review French language agreements using French language playbooks, and French language Chat input.

Agreement Desk users with the required permissions can access AI-Assisted Review’s Chat in several ways.

1. From Agreement Desk:
   1. Select the Agreements tab, and choose Requests.
   2. Select a request to open it.
   3. Select the Documents tab.
   4. Or, open the document preview, and select Edit in Word.
   5. Open the downloaded agreement in Microsoft Word, and if prompted, log into the AI-Assisted Review add-in.
2. From a document located on your local storage:
   1. Locate the document on your local storage.
   2. Open the document.

      The document opens in Microsoft Word.
   3. Select the Docusign AI-Assisted Review add-in from the Home ribbon in Microsoft Word. If prompted, log into the AI-Assisted Review add-in.
3. After opening the agreement using one of these methods, follow these steps to access Chat in AI-Assisted Review:
   1. In the AI-Assisted Review window, log on to Docusign.

      The AI-Assisted Review window displays, with the Chat tab selected. An AI summary of the agreement you opened is displayed in a dropdown.

To have the add-in refer directly to specific text when responding, highlight a clause or section text of the contract. Once a section has been highlighted, select the + icon for Add source > Use current text selection.

      Note: Use plain language for queries. Query result return times will vary, depending on the size of the contract being analyzed. To leave feedback for the AI-generated results, select ![Positive feedback](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/thumbs-up.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5ODAsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.PN73N3WZ_O_glMMOkPt0QCkiNVpv-IEeqYjPBZuSy9A&_LANG=enus "Positive feedback") to leave positive feedback, or select ![Negative feedback](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/thumbs-down.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5ODAsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.PN73N3WZ_O_glMMOkPt0QCkiNVpv-IEeqYjPBZuSy9A&_LANG=enus "Negative feedback") to leave negative feedback.

      Examples:
      - "When does the contract start?"
      - "Does this agreement automatically renew?"
      - "Are there any clauses for disclosures?"

      AI-Assisted Review returns the results of your query, including links to the sections of the contract where answers were found. Results can be expanded and selected to jump to sections of the contract that match the query.
   3. Once an answer to your question has been provided by the add-in, you can enter a follow-up question, query, or prompt related to the provided answer. To copy the answers or summaries that the add-in provides for your prompt, select the ![Copy Results](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Duplicate.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5ODAsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.PN73N3WZ_O_glMMOkPt0QCkiNVpv-IEeqYjPBZuSy9A&_LANG=enus "Copy Results") icon. To apply the add-in's suggestions for your prompt, select Apply Suggestion.

      Note: Queries and prompts can be conditional and call for multiple contract language changes. If there are multiple sections in which changes are applied, you will need to select Apply Suggestion for each suggestion.

      Examples:
      - "Does this agreement auto-renew?" can be followed by "Remove the auto-renewal and also change the term length to 1 year."
      - "Are there any late fees?" can be followed by "Remove all language around late fees for contract renewal."
      - "Draft a clause granting termination for convenience."
      - "Provide a summary of this entire agreement, focusing on scope of services, liability and indemnity, and any other key risks."
      - "Generate a summary of key payment terms in list format that I can send via email."

      The add-in displays the changed sections of the agreement according to the provided prompt. Changed sections are indicated in the agreement with strikethroughs and markup formatting. The add-in indicates that the suggested changes have been applied.
   4. To generate and add language to an agreement, enter the details you want to add in the Type to ask something… field.

      Note: Use language designed as commands for queries. Return times on suggestion results may vary, depending on the query provided and length of the document.

      Examples:
      - "Generate a Choice of Law Clause for California"
      - "Provide a Force Majeure Clause"
      - "Create a definition of Confidential Information"

      AI-Assisted Review generates and displays a suggestion based on the provided query.
   5. Review the suggestion(s) generated by AI-Assisted Review. To accept one of the suggestions, select Apply Suggestion.

      The AI-Assisted Review add-in automatically suggests a location to insert the new contract language. You can also copy the language to your clipboard by selecting the ![Copy Results](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Duplicate.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5ODAsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.PN73N3WZ_O_glMMOkPt0QCkiNVpv-IEeqYjPBZuSy9A&_LANG=enus "Copy Results") icon.

      - Select the ![Options](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapfd7284e6-b699-4527-8651-a133914111fd/page/images/Overflow-Vertical.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDQ5ODAsInNoZWFmIjoibWFwZmQ3Mjg0ZTYtYjY5OS00NTI3LTg2NTEtYTEzMzkxNDExMWZkIn0.PN73N3WZ_O_glMMOkPt0QCkiNVpv-IEeqYjPBZuSy9A&_LANG=enus "Options")icon for additional options.    8. (Optional) Users can upload a total of five documents into Chat from local storage and your Docusign repository. AI-Assisted Review can make suggestions and comparisons based on the uploaded or selected agreements, and enables users to ask Chat questions about those agreements. Select the + icon, and choose your options:

      - Select Upload to upload documents from your local storage.
      - Select Choose from my repository to select documents from your Docusign repository.

      Note:
      - Supplementary documents uploaded to Chat must be either DOCX or PDF format.
      - You may choose a maximum of five documents from local storage and your Docusign repository.
   9. Once all changes have been made, you are ready to save your agreement. The applicable save method will depend on how you accessed AI-Assisted Review:

      - Users who access AI-Assisted Review using Edit in Word can select Save to Request. The document will be saved back to Agreement Desk as a new version.
      - Users who access AI-Assisted Review using a document from local storage should select Save in Microsoft Word to save the document locally. Once the document is saved, it can be uploaded manually to Agreement Desk.

You have successfully analyzed and made changes to a contract using AI-Assisted Review.
