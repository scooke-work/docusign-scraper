# IAM Toolkit overview

The IAM Toolkit provides enterprise developers and system integrators (SIs) the tools to programmatically customize, test, and scale Agreement Manager to meet specific contract management requirements.

## Places to start

[See what you can automate with the toolkit](https://developers.docusign.com/iam-toolkit/implement-at-scale/)

[Review common use cases](https://developers.docusign.com/iam-toolkit/command-line/use-cases/)

[Join the open beta](https://developers.docusign.com/iam-toolkit/open-beta/)

[Accelerate implementation with AI tools](https://developers.docusign.com/iam-toolkit/ai-tools/)

## The IAM Toolkit empowers builders to

### Customize agreement processing

Programmatically configure Agreement Manager with custom fields, agreement types, AI training sets, and other configurations.

### Refine extraction accuracy

Test extraction logic against a set of up to 400 actual agreements, enabling you to iteratively refine AI training set rules for high-quality data extraction across your entire document repository.

### Ingest agreements at scale

Programmatically ingest millions of historical agreements to Agreement Manager with metadata during testing and after go-live.
