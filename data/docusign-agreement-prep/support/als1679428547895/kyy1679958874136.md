# Docusign Template Assistant for Word

Read about how the Docusign Template Assistant for Word simplifies the creation of DOCX files and templates.

The Docusign Template Assistant for Word is an add-in for creating DOCX files and templates.

Note: The Docusign Template Assistant for
Word Add-in is available in the [Microsoft Office Store](https://appsource.microsoft.com/en-us/product/office/wa200005367?tab=overview).

## Benefits

While you can prepare your DOCX files manually, this recommended add-in provides
significant benefits:

- Automatically writes and enhances syntax, including adding validation to sender fields.
- Supports field configuration so you can specify text, date, number
  fields, and dropdowns.
- Preview the generated document in Word to identify issues before uploading to Docusign.
- Upload your document to a new or existing template.
