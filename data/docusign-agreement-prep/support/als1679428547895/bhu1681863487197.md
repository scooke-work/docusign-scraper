# Troubleshooting Syntax Errors With Document Generation Templates

Get help with common problems encountered when creating Docusign eSignature templates
for document generation. These errors pertain to syntax errors in uploaded .docx files.

When uploading a .docx file with document generation syntax to a Docusign eSignature template,
if there are any syntax errors you will get an error message. The message states the
number of syntax errors and the file will not upload. To resolve the errors, update the
.docx file to fix the syntax and re-upload. For details on the specific errors, download
the provided JSON file.

In the troubleshooting topics that follow, common syntax errors are described to help you
identify and correct them.
