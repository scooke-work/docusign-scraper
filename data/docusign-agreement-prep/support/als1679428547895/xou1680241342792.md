# Edit a Sender Field in the Template Assistant

Edit sender fields in the Docusign Template Assistant by updating configuration details or modifying JSON syntax.

Edit an existing sender field by updating the field configuration details or modifying the JSON syntax, which is automatically generated when you [add a sender field](https://support.docusign.com/s/document-item?bundleId=als1679428547895&topicId=dtw1680044374844.html&_LANG=enus).

1. Open the Docusign Template Assistant in Word and log in to your Docusign account.
2. Go to the Fields tab, and select the pencil icon next to the sender field to edit.
3. Update the configuration and select Save.
4. (Optional) To edit a field's JSON syntax, go to the Code tab, update the syntax, and select Insert.

Your sender field and JSON syntax are updated.
