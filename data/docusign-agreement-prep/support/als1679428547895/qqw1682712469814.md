# Troubleshooting Merged Data Errors

Get help with common problems encountered with merged data provided by the sender of a
document generation envelope.

If there are errors when trying to merge the data provided by the sender, they will appear on
the document as red text with a yellow highlight. An error message banner identifies the
number of merge errors and provides a link to download a .JSON errors file.
