# Agreement Template Builder for Document Generation

Read about the Agreement Template Builder and how it can help you edit your .docx files
to create document generation templates.

The Agreement Template Builder (ATB) is a feature within the Docusign platform that allows you to edit template documents directly in the application. With these tools, you have comprehensive control over your document's content, extending beyond merely adding fields for data input.

The template builder is an alternative to using the [Docusign Template Assistant for Word](https://support.docusign.com/s/document-item?bundleId=als1679428547895&topicId=kyy1679958874136.html&_LANG=enus "Read about how the Docusign Template Assistant for Word simplifies the creation of DOCX files and templates.") and offers an improved user experience for creating
document generation templates.

Note: All accounts have access to use the template builder and create
document generation templates. Docusign IAM accounts can create document templates and envelope templates and have full access to document generation features. Docusign eSignature accounts can create envelope templates only, and the Document Generation add-on for eSignature is required
to send more than five trial envelopes from a document generation template that includes
advanced features. Refer to the [Limitations and Considerations for Document Generation](https://support.docusign.com/s/document-item?bundleId=als1679428547895&topicId=cah1680213420771.html&_LANG=enus "Important information to consider before starting to develop document generation documents and templates.").

With ATB, you can prepare document generation templates quickly and
easily. The template builder offers the following key features:

- Build document templates or envelope templates.
- Add fields manually,or use AI-assisted field suggestion to quickly populate templates with the correct fields.
- Add sender fields for the preparer to insert dynamic data and generate customized, professional-looking
  agreements.
- Add traditional signer fields directly inline with text
  for recipients to complete, such as adding a signature or marking a
  checkbox.
- Edit your documents directly in Docusign; no need to use an external
  application.
- Add conditional logic rules to show or hide content based on the dynamic data entered by
  senders.
- Insert dynamic tables. Tables are populated automatically with as
  many rows as needed when the agreement is generated.

Note: In order to use the new agreement template builder, the Sending
Settings for Enable Agreement Template Builder must be enabled
for your account by your account administrator. This account setting is documented in
[this administrator guide on template usage
settings](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=edh1583277351016.html&_LANG=enus).
