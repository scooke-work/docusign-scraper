# Choosing Between Document Editors

Read about the editor options that you have for creating agreements.

When creating document generation templates in Docusign using a Word document, you have two
editor options: Agreement Template Builder (ATB) and Word Template Assistant (WTA).
These editors are a central part of Agreement Preparation and provide the ability to
edit fully the contents of your Word document. You also use them to add recipient,
sender, and third-party fields.

Understanding the differences between these editors will help you select the one that best fits your needs.

## Overview of editor options

Here's a high-level summary of the options that you have to create a template from a Word document:

Word Template Assistant (WTA)
:   WTA is designed for advanced users who need robust customization and
    flexibility. It offers a comprehensive set of features for building
    complex templates and workflows. WTA is a plug-in for Microsoft Word
    and provides the rich, native controls of that application. The WTA
    offers advanced syntax control over your content.

    Use this Microsoft Word add-in to help you build finished files for
    document generation in Docusign eSignature. It
    assists with adding sender fields, building rules to drive
    conditional content, and offers a preview option to check results.
    Upload finished files to Docusign as a document template or
    eSignature template.

Agreement Template Builder (ATB)
:   ATB provides a streamlined, user-friendly interface for quickly
    creating and editing templates. It is ideal for users who prefer
    simplicity and speed over advanced customization. ATB is an
    integrated feature within the Docusign application and provides a
    simple, streamlined method to prepare templates.

    With ATB, you edit template documents directly in Docusign to create
    and edit document generation templates. Just upload a .docx file to
    your template or, for envelope templates, create a new document
    directly in the template. Add sender and signer fields, define
    conditional logic rules to show or hide content, and preview your
    results with customized content.

Table 1. Key differences between ATB and WTA

| Feature/Capability | WTA | ATB |
| --- | --- | --- |
| Customization level | High | Moderate |
| User interface | Advanced, feature-rich | Simple, intuitive |
| Best for | Complex templates, power users | Standard templates, new user |

## How to choose the right editor

Consider the following when choosing between WTA and ATB to create your template:

- **Template Complexity:**

  - Use WTA if you need to create complex templates with advanced logic. WTA also supports inline
    conditional rules, which allow you to place conditions on text
    within a paragraph so you can show or hide content.
  - Choose ATB for straightforward templates that don’t require extensive customization.
- **User Experience:**

  - If you are new to document templates or prefer a simple interface, ATB is likely the better choice.
  - If you are comfortable with advanced features and need more control, WTA is recommended.
- **Team Needs:**

  - ATB: Ideal for teams with diverse experience levels, promoting standardized template creation.
  - WTA: Best suited for specialized teams or workflows requiring greater flexibility.
