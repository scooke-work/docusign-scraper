# Error: Incorrect Placement of SuppressParagraph

Troubleshoot a document generation syntax error that occurs when a SuppressParagraph tag
is in the wrong place.

A SuppressParagraph tag in the wrong place will cause an error when you upload a document
generation .docx file.

The SuppressParagraph tag is not positioned as part of a paragraph. Example:

```
|  |  |
| --- | --- |
|  | {"tagIdentifier": "{t:\"SuppressParagraph\", e: \"fieldName = 'value'\"}", |
|  | "errorCode": "InvalidPlacement", |
|  | "message": "Invalid placement of a Merge Tag in the template. – SuppressParagraph: This tag must be placed in a paragraph."} |
```

Template creator

Ensure that the SuppressParagraph
tag is part of a paragraph. The tag must have text next to it with no
intervening line breaks.
