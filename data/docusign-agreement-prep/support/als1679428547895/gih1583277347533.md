# Delegated Permissions

Read about how account administrators assign limited administrator capabilities through
the Delegated Administration functionality.

As a Docusign account administrator, the Permission Profiles page lets you
manage the user permission profiles for your account. The Delegated Administration
functionality allows administrators to create Permission Profiles that have limited
administrator capabilities in addition to the standard user permissions.

Note: For a more in-depth discussion of use cases,
review the [Best Practices for eSignature Delegated Administration](https://dsucustomers.docusign.com/km-guide-best-practices-for-esignature-delegated-administration/) white paper.

Note: If your account does not have Delegated
Administration enabled, the Permission Profiles dialog box only shows User Permissions. Refer
to [Permission Profiles](https://support.docusign.com/s/document-item?bundleId=als1679428547895&topicId=pof1583277362435.html&_LANG=enus "Read about how administrators use permission profiles to manage levels of access for a number of users. Permission profiles are sets of permissions that administrators assign to individual users. By managing the permission profiles, administrators can make changes to multiple users simultaneously.") for more information on User
Permissions.

Delegated Administration is available only for certain plans. If you do not
see the functionality described in this guide, contact Docusign Support to
determine if it can be enabled for your account.
