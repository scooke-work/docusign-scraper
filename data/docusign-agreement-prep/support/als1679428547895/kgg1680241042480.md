# Insert Sender Fields and Dynamic Tables Into a Document

Learn how to insert sender fields and dynamic tables into a Word document using the DocusignTemplate Assistant.

After adding sender fields or dynamic tables to the Docusign Template Assistant, insert them into a Word document.

Note: When using the Docusign Template Assistant for Word to add eSignature fields (such as Signature, Initial, and Date Signed), dynamic content can cause fields to display in the wrong position. To keep fields aligned with your content, use **Anchor Tags** to attach fields to specific text strings. For more information, see the [What are Automatic Anchor Tags?](https://support.docusign.com/s/articles/What-are-Automatic-Anchor-Tags?language=en_US) article.

1. Open Word, launch the Docusign Template Assistant add-in, and log in to your Docusign eSignature account.
2. Open the document, place the cursor where you want to insert a field or table, or highlight text to replace.
3. In the Docusign Template Assistant, go to the Fields tab, and select the Fields or Tables option.
4. Select the item that you want to insert.
5. Use Word formatting tools to apply desired formatting, such as bold, font size, or color.

   When the document is generated in Docusign eSignature, it retains your applied formatting.
6. (Optional) Review and edit column headers in the dynamic table as needed.

The sender field or dynamic table is inserted and formatted.
