# Create a Dynamic Table in the Template Assistant

Learn how to create dynamic tables using the DocusignTemplate Assistant for Word, which automatically adjusts to the data provided.

## Before you begin

Dynamic tables have these limitations:

- Formulas and inline calculations aren't supported. Only API or CSV values are accepted.
- Special characters aren't accepted in table field names.

Use the Docusign Template Assistant for Word to create dynamic tables that automatically adjust to the data provided. For more information about dynamic tables, see the [Dynamic Tables for Document Generation](https://support.docusign.com/s/document-item?bundleId=als1679428547895&topicId=zhi1724987563409.html&_LANG=enus) guide.

1. Open a Word document, launch the Template Assistant add-in, and log in to your eSignature account.
2. In the Docusign Template Assistant, go to the Fields tab.
3. Select the + button, and select Create table.

   In the Insert dynamic table window, complete the details:

   Table Name
   :   Enter a table name for identification. This name displays only in the Template Assistant and is not visible in the final document.

   FIELD NAME
   :   Each field is a table column, with the entered name as the default column heading, editable upon table insertion.

       Note: Field names cannot contain any of the following special characters

       : # % ! ? @ ^ & \* ( ) - + = [ ] { } : ; / ,
4. Finalize your table by adding and arranging field items to form columns.

   ![Screenshot of the Insert dynamic table window in the Template Assistant, showing table name and field name columns being configured before adding the table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/als1679428547895/page/images/insert-table-WTA.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgwOTUsInNoZWFmIjoiYWxzMTY3OTQyODU0Nzg5NSJ9.a6qzVMa3BgL-LBYXH_Ram1iiU3LQpJY0SnG5zqwR1jk&_LANG=enus "Screenshot of the Insert dynamic table window in the Template Assistant, showing table name and field name columns being configured before adding the table.")
5. Select Add Table to add your new dynamic table to the Template Assistant.

   ![Screenshot of an example dynamic table defined in the Template Assistant and inserted into a document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/als1679428547895/page/images/dynamictableexamples.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgwOTUsInNoZWFmIjoiYWxzMTY3OTQyODU0Nzg5NSJ9.a6qzVMa3BgL-LBYXH_Ram1iiU3LQpJY0SnG5zqwR1jk&_LANG=enus "Screenshot of an example dynamic table defined in the Template Assistant and inserted into a document.")

Your dynamic table is added to the list of tables for the selected agreement type in the Template Assistant. You can [insert the table](https://support.docusign.com/s/document-item?bundleId=als1679428547895&topicId=kgg1680241042480.html&_LANG=enus) into any Word document.
