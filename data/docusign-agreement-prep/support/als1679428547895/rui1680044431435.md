# Create Sender Fields With JSON Syntax in the Template Assistant

Learn how to quickly add and share sender fields to the Docusign Template Assistant Add-in for Word by pasting in JSON data.

Using JSON syntax in the Docusign Template Assistant offers two key benefits for managing sender fields:

- **Quick Creation**: Paste existing JSON field data into the Code tab to quickly populate the Docusign Template Assistant, streamlining document setup.
- **Easy Sharing**: Configure sender fields, copy the JSON from the Code tab, and share it externally. Others can paste this JSON into their Docusign Template Assistant to replicate the fields.

1. Open Word, launch the Docusign Template Assistant Add-in, and log in to your Docusign account.
2. Go to the Code tab.

   ![Code tab in the Template Assistant Add-in showing the JSON editor area for sender fields](https://docusign-be-prod.zoominsoftware.io/api/bundle/als1679428547895/page/images/code-tab-WTA.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgwOTAsInNoZWFmIjoiYWxzMTY3OTQyODU0Nzg5NSJ9.BAii3NxOShzyImQ8IhvvMr5sutSYyhiQKOeB_W3Vbmo&_LANG=enus "Code tab in the Template Assistant Add-in showing the JSON editor area for sender fields")
3. Copy and paste your JSON data into the Insert JSON box. Avoid overwriting existing fields unless you intend to replace them.
4. Select Insert.

The JSON-defined sender fields are added to the Fields tab in the Docusign Template Assistant. You can [select and insert them into your document](https://support.docusign.com/s/document-item?bundleId=als1679428547895&topicId=kgg1680241042480.html&_LANG=enus). Overwritten fields are removed from the sender fields list.
