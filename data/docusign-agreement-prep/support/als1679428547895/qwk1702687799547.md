# Create a Template from a Document

Learn how to prepare a DOCX file and upload it to Docusign using the Template Assistant to create a document generation template.

## Before you begin

- Open the Microsoft Word document and launch the Docusign Template Assistant in your Docusign account.
- Log in to your Docusign account with permissions to create templates.
- Recommended: Use the [Preview a Generated Document](https://support.docusign.com/s/document-item?bundleId=als1679428547895&topicId=gke1680044621975.html&_LANG=enus) feature to check and correct errors in your Word document before uploading it to an eSignature template.

Prepare your DOCX file with content, sender fields, and optional rules. Create a new template or add it to an existing one. Senders will use the template to pre-fill dynamic data and generate the final document for signing.

Important: Use the Docusign Template Assistant to upload documents with syntax. Direct uploads to a Docusign template won't recognize the syntax.

1. In the Docusign Template Assistant, select Upload document to open a new window.

   ![Upload document button in the Template Assistant window.](https://docusign-be-prod.zoominsoftware.io/api/bundle/als1679428547895/page/images/upload-document-button-WTA.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgxMDcsInNoZWFmIjoiYWxzMTY3OTQyODU0Nzg5NSJ9.LfWxcqYANrlkNcY1cgxdK2gUVkqaKsIiaTLMqr6ZBhY&_LANG=enus "Upload document button in the Template Assistant window.")
2. Choose to upload the file as a document or envelope template.
   - For a document template, create a new document template or replace an existing one.
   - For an envelope template, select Next, and create a new envelope template or add to an existing one.

   ![Screenshot of the Upload window from Template Assistant.](https://docusign-be-prod.zoominsoftware.io/api/bundle/als1679428547895/page/images/uploaddocgen.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgxMDcsInNoZWFmIjoiYWxzMTY3OTQyODU0Nzg5NSJ9.LfWxcqYANrlkNcY1cgxdK2gUVkqaKsIiaTLMqr6ZBhY&_LANG=enus "Screenshot of the Upload window from Template Assistant.")

    Your envelope template opens to the prepare step with the document labeled "Document Generation" in the documents section. Your document template displays in the Document Template library, also labeled "Document Generation".

   ![The Template prepare file with an uploaded DOCX file with document generation data fields, bearing the "Document Generation" indicator.](https://docusign-be-prod.zoominsoftware.io/api/bundle/als1679428547895/page/images/genfile.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgxMDcsInNoZWFmIjoiYWxzMTY3OTQyODU0Nzg5NSJ9.LfWxcqYANrlkNcY1cgxdK2gUVkqaKsIiaTLMqr6ZBhY&_LANG=enus "The Template prepare file with an uploaded DOCX file with document generation data fields, bearing the \"Document Generation\" indicator.")
3. Add recipient roles and message details to complete the template preparation.

   The document displays sender fields and conditional statements from the uploaded file.
4. Add additional eSignature fields as needed. Use AutoPlace to automatically position fields next to a specified character string, ensuring they remain correctly placed even as content adjusts.

Your document is now a dynamic template with all fields, dynamic tables, and logic rules applied, ready to generate and send personalized agreements. Before sending for signing, add any remaining fields and settings to finalize the template.
