# Send a Document Generation Envelope

Learn how a sender uses a template for document generation.

You start a document generation envelope by using a document generation template. You can
either start an envelope and use the configured template; or you can use the
template directly to create the envelope for document generation. As the sender, you
then pre-fill the required information to generate the final document for
signature.

Note: For document generation, you must have the appropriate user
permissions to use templates. Also, you must have access to the prepared
template. Document templates requires the **Document Template Use** permission. For an envelope template, you must either be the template owner, or the owner must share the
template with you.

1. Start your envelope in one of the following ways:
   - **Start an envelope**. From the Agreements page, select Start > Envelopes > Use a Template. In the Select a template window, select All Templates to see all envelope templates you can access and all published document templates on the account. Choose any template marked with the DOCUMENT GENERATION badge.
   - **Use a document template**. Locate the template you want to use. Select the more actions menu and select Use Template. .
   - **Use an envelope template**. Locate the template that you want to use and select
     USE. Set up the envelope details and select Next.

     Note: Document
     generation templates are marked with a DOCUMENT
     GENERATION badge for easy identification.
2. Enter field data for your document. Complete all required fields and any optional ones as needed.

   - If you used a **document template**, the Add field data form appears alongside the document, generating a dynamic preview of your agreement as you enter data.
   - If you used an **envelope template**, you first complete the Add field data form, then select Next to review your generated agreement.

   The Add field data form displays for you to complete.![Screenshot of the Add field data form for the sender to enter data for the merge fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/als1679428547895/page/images/prefillform.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgxMTQsInNoZWFmIjoiYWxzMTY3OTQyODU0Nzg5NSJ9.uactzu29QK-RDbgGhKgrPx3qZyLa8G-qPNFZ1A3-5hQ&_LANG=enus "Screenshot of the Add field data form for the sender to enter data for the merge fields.")
3. If there are multiple document generation files in the envelope the add field
   data form presents the fields for each file separately.

   If
   documents have data fields with the same field name, the sender sees the field
   listed once for each document where it appears. They can enter the same or
   different values for each.

   If your document includes one or more dynamic tables, upload the CSV file with
   the table data. For more information, see the topic [Upload Field Data for a Dynamic Table](https://support.docusign.com/s/document-item?bundleId=als1679428547895&topicId=shq1713209582306.html&_LANG=enus "Learn how to send a document generation envelope that includes a dynamic table. As the sender, you provide field data for the table by uploading a CSV file.").
4. Review the generated document, including the inserted data and any additional
   fields. If you need to edit any of the merged data, select
   BACK to return to the Add field
   data form and make your changes.
5. When you are finished, select SEND.

The envelope with your generated document is sent.
