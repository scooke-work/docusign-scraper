# Add a Rule to Show or Hide Conditional Content

Learn how to add rules in the DocusignTemplate Assistant for Word to display conditional content based on sender field values.

## Before you begin

- Your Docusign Template Assistant should already have the sender fields configured for your rules.
- Open the Word document and start the Docusign Template Assistant.
- Log in to any accessible Production or Demo account.

The Docusign Template Assistant for Word lets you add conditional content based on sender field values.

Conditional content may consist of regular text and other sender fields. Various logical operators, such as exact text matches or value comparisons, can be applied.

For example, if your offer letter includes a clause specific to a job title, you can add a rule to display the clause when the `Job Role` field is "Software Engineer".

1. Select the Rules tab, and select the Show content or Hide content option.
2. Complete the rule configuration.

   Field Name
   :   Select the sender field for the rule argument from the dropdown list of available fields in the Docusign Template Assistant.

       Note: The selected sender field is added to the pre-fill form for completion, even if it's not in your document.

   Operator
   :   Select the comparison operator for the rule. Text and Date fields support text matching (Matches, Doesn't match), while Number fields support text matching and numerical comparisons.

   Value
   :   Input the comparison value.
3. In your Word document, choose the content for the rule, and select Apply.
