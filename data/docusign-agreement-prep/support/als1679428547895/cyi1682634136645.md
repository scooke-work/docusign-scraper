# Error: Invalid Data Field Type

Troubleshoot a document generation syntax error that occurs when a data field is
formatted incorrectly and results in an unrecognized tag type error.

If a data field is incorrectly formatted it may not be recognized as a valid field type.

Example: A space before the SuppressParagraph tag`{t: “
SuppressParagraph", e:"FieldName = 'value'"}`

```
|  |  |
| --- | --- |
|  | {"tagIdentifier": "{t:\"SuppressParagraph \", e: \"FieldName = 'value'\"}", |
|  | "errorCode": "InvalidTagType", |
|  | "message": "Invalid Json tag type. -- Invalid json tag SuppressParagraph ."} |
```

Template creator

For the verbose syntax required for
SuppressParagraph, the exact name of the field type must be in quotes. Review
the tag identified in the error message and correct any formatting issues, such
as extra spaces.
