# Error Code: JsonReaderException

Troubleshoot the JsonReaderException error code.

A variety of formatting errors in your document generation syntax can lead to an error in
parsing JSON syntax.

Example: missing a comma in the SuppressParagraph tag
`{t:"SuppressParagraph" e: "FieldName = 'value'}`

```
|  |  |
| --- | --- |
|  | {"tagIdentifier":"{t:\"SuppressParagraph\" e:\"FieldName = 'value'\"}", "errorCode":"JsonReaderException","message":"Unable to parse Json object. -- There was an error parsing this merge tag as json (check spacing, quotes, json syntax): After parsing a value an unexpected character was encountered: e. Path 't', line 1, position 23."} |
```

so the solution will vary from case to case.
