# Bulk Send With Document Generation

Learn how to use bulk send with a document generation template.

Document generation supports bulk send. You can use bulk send to send many envelopes at once, each with customized data.

Starting with a document generation template, you import a bulk list with the individual envelope and recipient data as usual. The bulk list also contains the values for the document generation sender fields and dynamic tables. So for document generation with bulk send, the sender does not use the Add field data form to enter custom data before sending.

When preparing for a bulk send, consider the following requirements:

- Every sender field with the same field name must have the same value. This includes fields on different documents in the envelope.
- The maximum number of sender fields that can be used for a bulk send envelope is 100.

For more information about bulk send see [this help guide](https://support.docusign.com/s/document-item?bundleId=wtn1643071711000&topicId=kjz1578456386508.html&_LANG=enus).

1. Prepare your bulk send list CSV file to provide the data for your sender fields and dynamic tables.
   1. Sender field requirements:

      - Sender field column header format: `Document Generation::Field_Name`
      - Each required sender field must have exactly one column in the CSV
      - For each row in the CSV, there must be a value for all required sender fields
   2. Dynamic table requirements:

      - Dynamic table column header format: `Dynamic Table::{Table Name}|{Column Name}`
      - Each column in the dynamic table must have a properly formatted column in the CSV
      - Each row in the CSV includes the dynamic table data for that bulk send envelope
      - Use the pipe symbol (|) as a delimiter to create rows in the dynamic table
      - Any spacing around the delimiter will be included in the dynamic table data

      - For the example data, the bulk send envelope generated from the first row of CSV data would yield a dynamic table with two rows:![Screenshot of a dynamic table generated from a bulk send list.](https://docusign-be-prod.zoominsoftware.io/api/bundle/als1679428547895/page/images/bulksenddynamictableexample.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgxMjMsInNoZWFmIjoiYWxzMTY3OTQyODU0Nzg5NSJ9.C9-wMkmCjP_8wUFvPjQIjyxIz3pTT2130wDi5LWTy_s&_LANG=enus "Screenshot of a dynamic table generated from a bulk send list.")
2. Use your document generation template to start a new envelope.
3. In the Add recipients section, select Import Bulk List and import your prepared bulk send CSV file.
4. Review the data in the Bulk List Preview. You can complete or edit the imported column values as needed. Then select FINISH IMPORT.

   Your bulk recipient is added to the envelope. Values for the document generation sender fields and dynamic tables will be populated from the entries in the bulk send list; you will not see the usual Add field data form. These values are inserted seamlessly into the document, ensuring a well-formatted agreement and a professional look.
5. Finish preparing your envelope and send it.

   Note: Before sending the envelope, when you view your uploaded .docx file, you will see the document generation syntax. This preview does not show the final agreement with your custom data inserted. The data will be inserted when you send the envelope.

Your bulk send is sent. The document generation sender fields and dynamic tables are processed using the bulk send CSV file data.
