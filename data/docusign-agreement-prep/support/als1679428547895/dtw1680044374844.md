# Create a Sender Field in the Template Assistant

Create and configure sender fields in the Docusign Template Assistant to personalize data in templates.

## Before you begin

- Download and install the Docusign Template Assistant for Word from the [Microsoft Office Store](https://marketplace.microsoft.com/en-us/product/office/wa200005367?tab=overview).
- Open Word and launch the Template Assistant add-in.
- In the Template Assistant, log in to a Docusign eSignature account. Access any Production or Demo environment account.

  Note: Once you log in, you can see your Docusign account information by selecting the Account icon in the Template Assistant.

Sender fields are dynamic placeholders for specific data, such as start and end dates. Add and configure sender fields in the Docusign Template Assistant to personalize templates.

The data automatically populates every linked field across the template, so you define fields once and reuse them as needed.

Once fields are added, they become available in the Docusign Template Assistant for use in any document.

1. In the Docusign Template Assistant, go to the Fields tab.
2. Select the + button and Create field.

   ![Create field dialog in the Template Assistant showing options to enter the field name, optional description, and select the field type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/als1679428547895/page/images/create-field-WTA.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgwODYsInNoZWFmIjoiYWxzMTY3OTQyODU0Nzg5NSJ9.djjCH7oKTcy1Z9DKkLr__vTf5yd0Bxa-qWZ0Qy2R6Rc&_LANG=enus "Create field dialog in the Template Assistant showing options to enter the field name, optional description, and select the field type.")

   Field Name
   :   Enter a valid data field name with no spaces. Examples: "FieldName" or "Field\_Name"

   Field Description
   :   (Optional) Add a subtitle for the field name. The description is not included in the final document.

   Field Type
   :   Select the field type from the dropdown, such as text, date, dropdown, or number.
3. (Optional) Select the Required option to require the sender to enter a value for the field.
4. Select the Field Type for the field.
   - Text. A free text field.
   - Date. A date in one of these formats:

     MM/DD/YYYY

     DD/MM/YYYY

     YYYY/MM/DD
   - Number. A numeric value from a minimum of -999999999.99 to a maximum of 999999999.99 that can include:
     - An optional period as a decimal separator. Up to two digits of decimal precision are supported.
     - Optional commas as thousands separators.
     - A leading minus sign to indicate a negative value.
   - Dropdown. A predefined list of up to 250 values in Text, Date, or Number format, excluding semicolons.
5. Select Add.

The field is added to the Fields list.

Note: The list shows all fields in the Template Assistant, available for insertion into any document. For more information about inserting sender fields, see the [Insert Sender Fields and Dynamic Tables Into a Document](https://support.docusign.com/s/document-item?bundleId=als1679428547895&topicId=kgg1680241042480.html&_LANG=enus) guide.
