# Create a Document Generation Template with the Agreement Template Builder

Learn how to create a document generation template using the agreement template builder. Generate personalized and professional-looking agreements with document generation templates.

## Before you begin

- Your account administrator must enable the Enable Agreement Template
  Builder setting in your account Sending Settings.
- Have a .docx file with your basic content ready to upload to the new
  template. (For envelope templates only, you can start from a new, blank document.)
- You should already have a basic understanding of how to create document and envelope templates.

With the agreement template
builder
(ATB), you can create a document generation template directly in
Docusign. The template builder works with .docx
files that are under 1 MB.
You can upload an existing file to create either a document template or an envelope template. For an envelope template, you can also create a new document directly in the
template. See [Limitations and Considerations for Document Generation](https://support.docusign.com/s/document-item?bundleId=als1679428547895&topicId=cah1680213420771.html&_LANG=enus "Important information to consider before starting to develop document generation documents and templates.") for other
requirements.

1. From the Templates page, start a new document template or envelope template. Upload a .docx file that is 1MB or less, provide a template name and other required information, and continue to the editor view.

   Your document displays in the ATB editor view. ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/als1679428547895/page/images/editor_view_example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgwNDQsInNoZWFmIjoiYWxzMTY3OTQyODU0Nzg5NSJ9.qrRVpL6zRlZne5eBQP_VPAjFtlqdBQL2th9n-E6Edd8&_LANG=enus)
2. Add a sender field. These fields are populated by the sender and incorporated
   into your document to generate a personalized and professional-looking
   agreement.
   1. Select Sender  in the recipients list.
   2. Review the agreement type selected by the AI detection process. If the type does not match your content, select the pencil icon to change it. The agreement type classifies your document and surfaces relevant data fields. Any custom sender fields created in the account and associated with the agreement type are included in the list for you to choose from.

      For more information on agreement type detection, see [this help topic](https://support.docusign.com/s/document-item?bundleId=als1679428547895&topicId=con143e1ca0-b763-4e21-8183-d3f964492b32.html&_LANG=enus).
   3. Position your cursor where you want to insert the field and then select
      the appropriate agreement field to insert.
   4. You can also create new [custom agreement fields and insert them into your document](https://support.docusign.com/s/document-item?bundleId=als1679428547895&topicId=tskce27c962-22c4-4dfa-b799-72c9a403d8ab.html&_LANG=enus).

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/als1679428547895/page/images/add_sender_field.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgwNDQsInNoZWFmIjoiYWxzMTY3OTQyODU0Nzg5NSJ9.qrRVpL6zRlZne5eBQP_VPAjFtlqdBQL2th9n-E6Edd8&_LANG=enus)Your sender field is added to the document.
3. Add a signer field. These are the recipient fields for your signers to complete,
   such as adding a signature.
   1. Select the recipient to assign fields to from the recipients list.

      To add more recipients, select Edit Recipients.
   2. Position your cursor where you want to insert the field and then select
      the field from the Fields panel.
   3. Select the inserted field to open the configuration options and complete as needed for the different field types.
   4. Select Save.
4. Preview your template to confirm the sending experience and
   see how your final document will look when generated.
   1. Select Preview and enter data for the sender
      fields defined in your template.
   2. Use the HIghlight Data toggle to emphasize sender field data.

   A preview appears showing your generated document with the field data
   you entered and any conditional rules applied to the content. ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/als1679428547895/page/images/preview_example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgwNDQsInNoZWFmIjoiYWxzMTY3OTQyODU0Nzg5NSJ9.qrRVpL6zRlZne5eBQP_VPAjFtlqdBQL2th9n-E6Edd8&_LANG=enus)
5. Finish preparing your template by adding more fields and using the text editing controls to modify and add content, and then save your template.

Your document generation template is saved to your account.
