# Preview a Generated Document in the Template Assistant

Learn how to use the preview feature in the Template Assistant to ensure that your documents meet expectations by identifying and resolving issues.

The preview feature in the Docusign Template Assistant helps identify and resolve document issues, ensuring that final documents meet your expectations.

1. Prepare your Word document by adding sender fields, dynamic tables, and content rules.
2. In the Docusign Template Assistant, select the
   Preview icon.

   ![Preview icon in the Template Assistant toolbar](https://docusign-be-prod.zoominsoftware.io/api/bundle/als1679428547895/page/images/preview-icon-WTA.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgxMDQsInNoZWFmIjoiYWxzMTY3OTQyODU0Nzg5NSJ9.scuY-uzVm5mPPmZQ2XVELqnX0T4_uzcDg38M9FbI8Ec&_LANG=enus "Preview icon in the Template Assistant toolbar")
3. Enter test values for sender and dynamic table fields, and select Preview.

   You can switch between a standard view and a highlighted data view. The document opens in a new window, showing the final version with test values. Rules are evaluated, and values are saved for future previews.
4. Edit test values to assess different rule outcomes. Repeat the task steps to modify values.

The preview shows the final version with sender fields and content rules applied. Verify the content and adjust as needed before finalizing the template.
