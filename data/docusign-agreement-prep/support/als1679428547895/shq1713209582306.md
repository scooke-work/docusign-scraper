# Upload Field Data for a Dynamic Table

Learn how to send a document generation envelope that includes a dynamic table. As the
sender, you provide field data for the table by uploading a CSV file.

When you use a document generation template that includes a document with a dynamic table, you
provide the table data by uploading a CSV file. The CSV file must include all column
headings specified for the table.

The system validates that all the column headings in your uploaded file are present
and correct. However, there is no validation on the table data itself.

You can download an example CSV file that includes the required column headings; or you can
create your own file from scratch.

Note: With the Document Generation add-on, developers can use the
API to transfer data from external systems of record. See [this API documentation](https://developers.docusign.com/docs/esign-rest-api/esign101/concepts/documents/document-generation/#dynamic-tables) for more information.

1. Use a document generation template to start an envelope as described in [Send a Document Generation Envelope](https://support.docusign.com/s/document-item?bundleId=als1679428547895&topicId=ekg1679981354189.html&_LANG=enus "Learn how a sender uses a template for document generation.").
2. When the Add field data form displays, complete the sender
   fields as required.

   ![Screenshot of the Add field data form with a dynamic table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/als1679428547895/page/images/addfielddatedynamictable.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgxMTgsInNoZWFmIjoiYWxzMTY3OTQyODU0Nzg5NSJ9.tC_DYkNQ6rdT1AqruMDSaikmcrThrLU5vXeEDZ6jmbM&_LANG=enus "Screenshot of the Add field data form with a dynamic table.")
3. Upload a CSV file containing the data for each dynamic table.

   - Each table requires its own CSV file with the table data.
   - The CSV file must contain the column headings as specified for your
     dynamic table.
   - You can download the sample CSV, which includes the necessary
     formatting.
