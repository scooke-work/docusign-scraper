# Insert a Dynamic Table

Learn how to add a dynamic table with the agreement template builder. Dynamic tables can be populated with as many rows as necessary when the agreement is generated.

## Before you begin

- Your account administrator must enable the Enable Agreement Template Builder setting in the account Sending Settings > Templates.
- Create a template and upload your .docx file with your basic content or create a new document directly in the template.

With the agreement template builder, you can create dynamic tables to insert into your Word documents. Dynamic tables adjust to fit the data provided, adding as many rows as needed when an envelope is sent.

When a sender uses a template containing a dynamic table, they upload a CSV file with the table data. The dynamic table adjusts to fit the data inserted, adding as many table rows as needed. Dynamic tables become part of the generated document and are not editable by recipients.

For more information about dynamic tables, see the [Dynamic Tables for Document Generation](https://support.docusign.com/s/document-item?bundleId=als1679428547895&topicId=zhi1724987563409.html&_LANG=enus "Read about dynamic tables. Dynamic tables are designed for agreements that include tables with a variable number of rows, depending on the data provided.") topic.

1. In the agreement template builder, position your cursor where you want to insert the dynamic table.
2. Select + Insert Items > Table > Dynamic Table and select the number of columns you want to use.

   ![Screenshot of the Insert menu cascade to insert a dynamic table with three columns.](https://docusign-be-prod.zoominsoftware.io/api/bundle/als1679428547895/page/images/insert_dynamic_table.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgwNTAsInNoZWFmIjoiYWxzMTY3OTQyODU0Nzg5NSJ9.d2qwK6f7IEh5vmVZqMTE08xFhFsaQkuRc_Wv6T5AlMU&_LANG=enus "Screenshot of the Insert menu cascade to insert a dynamic table with three columns.")
3. In the Insert dynamic table window, define your table name and columns and select Insert Table.

   - Table name is used to identify the table to insert data. It is not part of the generated document and is not visible to recipients.
   - Field names represent the columns for your table. The table can have a maximum of 10 columns for either agreement fields or third-party data fields.

   ![Screenshot of the Insert dynamic table window.](https://docusign-be-prod.zoominsoftware.io/api/bundle/als1679428547895/page/images/definedynamictable.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgwNTAsInNoZWFmIjoiYWxzMTY3OTQyODU0Nzg5NSJ9.d2qwK6f7IEh5vmVZqMTE08xFhFsaQkuRc_Wv6T5AlMU&_LANG=enus "Screenshot of the Insert dynamic table window.")

   Restriction: Do not use the pipe delimiter ("|") in field names if you intend to do a bulk send with the document generation template. See the topic [Bulk Send With Document Generation](https://support.docusign.com/s/document-item?bundleId=als1679428547895&topicId=nan1681853631175.html&_LANG=enus "Learn how to use bulk send with a document generation template.").
4. Select your inserted table and select table elements to apply formatting.

   - **Text formatting**. Use ATB text formatting tools to apply fonts and other settings, such as bold, font size, and text alignment.
   - **Table settings**. Use table controls to configure columns (remove or reorder), add color fill to cells, or remove borders.![Screenshot of an inserted dynamic table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/als1679428547895/page/images/atbdynamictableexample.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgwNTAsInNoZWFmIjoiYWxzMTY3OTQyODU0Nzg5NSJ9.d2qwK6f7IEh5vmVZqMTE08xFhFsaQkuRc_Wv6T5AlMU&_LANG=enus "Screenshot of an inserted dynamic table.")

Your dynamic table is inserted. The table definition is added to list of tables available to account users with the selected agreement type. When a sender uses the template, they provide a CSV with the table data. See the topic [Upload Field Data for a Dynamic Table](https://support.docusign.com/s/document-item?bundleId=als1679428547895&topicId=shq1713209582306.html&_LANG=enus "Learn how to send a document generation envelope that includes a dynamic table. As the sender, you provide field data for the table by uploading a CSV file.") for details.
