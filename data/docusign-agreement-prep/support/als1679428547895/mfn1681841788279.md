# Error: Missing Field Brackets

Learn how to resolve a document generation syntax error that occurs when your syntax is
missing data field brackets.

Document generation data fields must be in double curly brackets. If a field is missing one or
more brackets, it will not be included on the pre-fill page where the sender enters
values for the data fields.

## Enclose all data fields in double curly brackets

Template creator

1. Edit your .docx file to correct the syntax errors.
2. Check each inserted data field for proper placement in double curly
   brackets.

   Correct syntax: `{{ field_name }}`
   `{{Field_Name}}`

   Incorrect syntax: `{ field_name }`
   `{{ field_name }`
3. Upload the corrected .docx file to your eSignature template.
