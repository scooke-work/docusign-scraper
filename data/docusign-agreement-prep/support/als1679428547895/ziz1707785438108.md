# Add Conditional Rules To Show or Hide Content

Learn how to show or hide content dynamically based on values entered in sender fields.

With the agreement template builder (ATB), you can use conditional rules to make a single template
work for a multitude of situations. Rather than building custom templates to cover
different agreement variables, you can add conditional rules. These rules can show
or hide content based on values entered in sender fields.

Important: Conditional rules in ATB apply to **complete paragraphs only**. You cannot apply a rule to a fragment of a clause or a specific string of text within a larger paragraph.

**Example: Vacation Policy**

In an offer letter, you might have different vacation policies that depend on where an employee lives. To handle this, you first add a **sender field** named "State" to your template. When a sender prepares the agreement, they enter the employee's state of residence into this field.

Because conditional rules rely on these values but apply to **complete paragraphs only**, you must create two distinct versions of the policy:

- **Paragraph A:** Specifies 15 days for California employees (Rule: If State = California, Show).
- **Paragraph B:** Specifies 12 days for all other states (Rule: If State ≠ California, Show).

The Rule Builder uses the "State" sender field to determine which entire block of text to include.

Note: Conditional rules are part of the advanced features in ATB. Docusign IAM customers have full access to advanced features. For eSignature customers, these features are available
with the Document Generation add-on. All eSiganture accounts have access to basic document
generation and the option to try these advanced features. The add-on is required to
send more than five trial envelopes from a document generation template that
includes advanced features.

1. Create a document generation template using ATB.
2. Highlight the complete block of text in your document to make conditional and select the
   logic rule icon to the right.

   You can also access the rule builder feature through the tool bar icon or by
   hovering over the right margin of your document. ![Screenshot of ways to access the rule builder.](https://docusign-be-prod.zoominsoftware.io/api/bundle/als1679428547895/page/images/atbrulebuilder.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgwNTQsInNoZWFmIjoiYWxzMTY3OTQyODU0Nzg5NSJ9.wVzuJWC_-x8HWb8ahLidCJa6z7YJSbNdwnqcIRzN9Mo&_LANG=enus "Screenshot of ways to access the rule builder.")
3. In the Rule Builder select whether to Apply all or Apply any conditions to satisfy the rule.
4. For the first IF statement, drag the sender field from
   your list of fields that you want to use. Then select the operator and enter a
   value.
5. To add more conditions, select And if.
6. Select whether to Show or Hide
   content and verify the text to apply.
7. Complete the Otherwise... portion of the rule. This
   determines what is shown or hidden if the rule conditions are not met.
8. Select Save.

Your conditional logic rule is configured. In your document, you
can see the conditional text and you can use the left  and right  arrows to cycle through
the possible results. This preview also provides an edit link to make changes or delete your
rule.![Screenshot of the controls for a completed rule.](https://docusign-be-prod.zoominsoftware.io/api/bundle/als1679428547895/page/images/rulepreviewatb.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgwNTQsInNoZWFmIjoiYWxzMTY3OTQyODU0Nzg5NSJ9.wVzuJWC_-x8HWb8ahLidCJa6z7YJSbNdwnqcIRzN9Mo&_LANG=enus "Screenshot of the controls for a completed rule.")
