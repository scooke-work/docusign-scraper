# Access the Gen Template Select Source Step

Learn to access the Select Source step in a Word-based Gen template.

You can view or change the Salesforce data source your Gen template is built on.

Warning: Changing your Gen template data source deletes all of
your template's Salesforce fields. Docusign fields remain
unchanged.Docusign recommends creating a new template, rather
than changing the data source of an existing template.

Use this procedure to access the Select Source step of a Gen Word template:

1. Edit the Gen template Select Source page as a Salesforce
   administrator:
   1. Select the App Launcher grid ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/app-launcher-grid.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0NzEsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.QTyfX8qGDmPrWSJnhO4nk2RHCCFqrdN-b_6rIQJcbbM&_LANG=enus) and choose View
      All.
   2. Select Docusign Apps Launcher.
   3. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0NzEsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.QTyfX8qGDmPrWSJnhO4nk2RHCCFqrdN-b_6rIQJcbbM&_LANG=enus) next to a Gen template and choose
      Edit from the dropdown menu.

   The Gen template creation page opens.
2. Select the Select Source arrow. The arrow has no text
   until selected.

You have completed the process. You have opened the
Select Source step of a Gen template.
