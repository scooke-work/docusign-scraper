# Use the Docusign Gen Tab of the CLM Document Generation Page

Learn to use the Docusign Gen tab of the CLM Document Generation
page.

When you go to Docusign Setup >  Document Generation, you see the following tabs:

- CLM: Links and information about CLM Document Generation
  features.
- Docusign Gen: Links and information about how to create,
  edit, and delete Docusign Gen templates.

This guide demonstrates how to access and use the Docusign Gen
tab:

1. Select the Docusign Gen tab:

   ![A screenshot displays the Docusign Gen tab of the CLM Document Generation page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/CLM-DocuSign-Gen-tab.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgyNzksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.2Ecz0k1XCI3DRCzrZ6ODbLCN1iwGsKTFONGvU3g2uLc&_LANG=enus "A screenshot displays the Docusign Gen tab of the CLM Document Generation page.")
2. Access any of these links or buttons:

   - Learn More: Select this button to open Docusign
     Gen guide information.
   - Create a Template Configuration: Open the Docusign Gen template configuration builder.
     - [Create a Gen template with Microsoft
       Word](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=lig1586134209279.html&_LANG=enus)
     - [Create a Gen template with the Docusign Agreement Cloud
       Editor](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=bcp1586134298132.html&_LANG=enus)
     - [Create a Gen for Billing Template to Generate
       Invoices](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=ldc1610646513281.html&_LANG=enus)
     - [Create a Gen Template using the Salesforce
       Person Account Object](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=ird1636867371186.html&_LANG=enus)
   - Quick Guide: Select this link to open Docusign
     Gen overview information.

You have completed the process. You have explored the features of
the Docusign Gen tab.
