# Change the Font Size and Style of a Word-Based Gen Template

You can change the font size and style of the Word documents used in Gen templates.

You can change the font size and style of Word
documents used in Gen templates. To make changes, open the documents in Microsoft Word. When
you are done making changes, re-upload the documents to the Gen template.

1. Select a document name.

   The document preview page opens in a new tab or window.
2. Update your Word document:
   1. Select Download.
   2. Open the document and use Microsoft Word to update the font or style.![A screenshot displays Gen anchor text fields in bold font.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/tagstylechanges.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY1NjAsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.m_8rWQISXeXK0zhmk7A6t0lz9hM8xwFRpIwX7jxhgNE&_LANG=enus "A screenshot displays Gen anchor text fields in bold font.")
   3. Save the changes in Word and return to the Gen Upload
      Documents page.
3. [Re-upload the document to your Gen
   template](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=sqp1586134339354.html&_LANG=enus "Learn to update your Gen template Word documents.").
4. Select Save and Close and select Save on
   the Do you want to save changes? dialog box.

You have completed the process. Generated agreements
display the changes that you made in Microsoft Word.
