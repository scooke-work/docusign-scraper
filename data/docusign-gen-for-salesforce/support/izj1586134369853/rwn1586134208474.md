# Clone a Gen Template

Learn to clone a Gen template to create a new template that is similar to an existing
one.

You can copy or "clone" a Docusign
Gen template. This is helpful if you want to create a new template that is similar to an
existing one.

Use this procedure to clone a Gen template:

1. Access the Docusign Gen Templates page as a Salesforce
   administrator:
   1. Select the App Launcher grid ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/app-launcher-grid.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0MzAsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.IkTnBtX1VBrwxZ5xzUFDNw2Eb7LF126SBjdVqnS0T08&_LANG=enus) and choose View
      All.
   2. Select Docusign Apps Launcher.
   3. Select Docusign Gen Templates.
2. Select the name of a Gen template.
3. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0MzAsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.IkTnBtX1VBrwxZ5xzUFDNw2Eb7LF126SBjdVqnS0T08&_LANG=enus) next to the
   Gen template and choose Clone from the dropdown menu.

   ![A screenshot displays the Clone option selected from the Edit a Gen template dropdown menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Clone-a-Gen-template.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0MzAsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.IkTnBtX1VBrwxZ5xzUFDNw2Eb7LF126SBjdVqnS0T08&_LANG=enus "A screenshot displays the Clone option selected from the Edit a Gen template dropdown menu.")

   The page refreshes. The cloned copy is now the active template on the page. The
   template displays (copy 1) at the end of the name.

You have completed the process. You have cloned a Gen
template. Follow the instructions [to edit a Gen
template](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=ywu1586134208345.html&_LANG=enus "Learn how to edit, delete, or change ownership for a Gen template or custom button.") to make any changes.
