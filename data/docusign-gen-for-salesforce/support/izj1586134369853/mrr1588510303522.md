# Learn About Using SOQL-Based Table Filtering and Sorting in Gen

Learn about inserting Salesforce Object Query Language (SOQL) code into Gen
templates.

Salesforce Object Query Language (SOQL) searches your organization’s Salesforce data
for specific information. You can insert
Salesforce SOQL code into the TableRow fields of Gen templates .

Using SOQL code in tables creates custom rules for filtering and sorting data. You
can only use sorting and filtering with child relationship fields.

Example: An Account has two contacts, each with two cases (four cases
total for that account.) A user generates an agreement for that account, setting a
limit of one for each case. Note: Limit controls the number
of results a child relationship field can add to a generated agreement.

Result: Docusign adds one Case to the generated
agreement, even though more than one contact is present.

## SOQL for Gen use cases

Note: The asterisk indicates where the option to filter/sort
exists.

| Object Relation | Example |
| --- | --- |
| 1 : Many\* | Opportunity >  Opportunity Contact Roles\* |
| 1 : Many\* : 1 | Opportunity >  Opportunity Contact Roles\* >  Contact |
| 1 : Many\* : 1 : 1 | Opportunity >  Opportunity Contact Roles\* >  Contact >  Reports To |
| 1 : Many\* : Many\* | Account  >  Contacts\* >  Cases\* >  Owner Note: This example starts from the Account object. |

## SOQL for Gen Fields

- [SOQL Filtering (WHERE)](https://developer.salesforce.com/docs/atlas.en-us.soql_sosl.meta/soql_sosl/sforce_api_calls_soql_select_conditionexpression.htm): Enter the Salesforce SOQL
  code to filter results by a certain field.
- [SOQL Sorting (ORDER BY)](https://developer.salesforce.com/docs/atlas.en-us.soql_sosl.meta/soql_sosl/sforce_api_calls_soql_select_orderby.htm): Enter the Salesforce SOQL
  code to sort results by a certain field.
- [SOQL Sorting (LIMIT)](https://developer.salesforce.com/docs/atlas.en-us.soql_sosl.meta/soql_sosl/sforce_api_calls_soql_select_limit.htm): Enter the number of results
  that you want, from 1 through 2000.
