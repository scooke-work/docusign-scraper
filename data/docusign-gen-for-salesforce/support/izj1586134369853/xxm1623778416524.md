# Understanding Docusign Permission Sets in Salesforce

Learn about the types and functions of Docusign Apps Launcher permission sets.

Permission sets are a collection of Salesforce settings and permissions that give users access to various tools and functions. Permission sets extend Salesforce users’ functional access without changing their profiles.

Docusign adds a group of permission sets when you install the Docusign Apps Launcher package. Docusign permission sets display which Salesforce users have Docusign app access.

Important:

- Administrators grant most Docusign app access with  [Docusign User Management](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=dzl1586134206658.html&_LANG=enus). Administrators can only use the Docusign Login permission set to help grant access.
- Administrators should not manually add or remove users from Docusign permissions sets except Docusign Login.

The following is a list of the Docusign permission sets and their functions:

| Permission Set Name | Permission Set Function |
| --- | --- |
| Docusign Login | Allows users to authorize Docusign Apps Launcher and auto-assign permission sets corresponding to existing Docusign user permissions |
| Docusign Administrator | Allows administrative access to Docusign Apps Launcher |
| Docusign CLM Admin | Allows administrative access to CLM in Docusign Apps Launcher |
| Docusign CLM Experience Cloud User | Allows access to CLM in Communities (Experience Cloud) |
| Docusign CLM User | Allows subscriber access to CLM in Docusign Apps Launcher |
| Docusign CLM Viewer | Allows guest access to CLM in Docusign Apps Launcher |
| Docusign Document Generator | Allows users to generate documents with Docusign Gen |
| Docusign Document Negotiator | Allows users to send files for internal approval and external review with Docusign Negotiate |
| Docusign Sender | Allows users to send envelopes as the system sender in Docusign Apps Launcher﻿ |
| Docusign User | Allows access to eSignature in Docusign Apps Launcher |
