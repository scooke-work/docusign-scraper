# Add a Table to a Word-Based Gen Template

Learn to create Gen template tables that add Salesforce data to agreements.

You can create a table in a Microsoft Word-based Gen template to import Salesforce data to
a generated agreement. You build tables by pasting tags from your Gen template into your
Word table.

Use this procedure to add a table to your Word-based Gen template:

1. Select a document name.

   The document preview page opens in a new tab or window.
2. Select Download and open the document in Microsoft Word.
3. Add a table to your Word document:
   1. Insert a table in Word, using at least two rows and one column.
   2. In the top row, enter a name for each column.
   3. Enter `<# <TableRow Select="/` in the first cell of the second
      row.![A screenshot displays a table row with <# <TableRow Select="/ under the description column.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/tablefirstrow.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY1NjMsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.WfgvgKvHR8hRLexM9zmnRIm69TIWoYA4uumMGKIgXTs&_LANG=enus "A screenshot displays a table row with <# <TableRow Select=\"/ under the description column.")
   4. On the Gen template Add Fields page, select
      Copy next to the anchor text you want in the first row of
      your table.![A screenshot displays the Gen template description field with the copy button highlighted next to the anchor text.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-displays-the-Gen-template-description-field-with-the-copy-button-highlighted-next-to-the-anchor-text.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY1NjMsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.WfgvgKvHR8hRLexM9zmnRIm69TIWoYA4uumMGKIgXTs&_LANG=enus "A screenshot displays the Gen template description field with the copy button highlighted next to the anchor text.")
   5. Paste the text directly after the Table Row text in your Word document.![A screenshot displays the Word table description field with the full line of anchor text added.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/tablerowfulltag.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY1NjMsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.WfgvgKvHR8hRLexM9zmnRIm69TIWoYA4uumMGKIgXTs&_LANG=enus "A screenshot displays the Word table description field with the full line of anchor text added.")
   6. Copy the remaining data fields from the Gen template and paste them into the
      appropriate table cells.![A screenshot displays four Word table rows with full lines of anchor text added.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/remainingtags.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY1NjMsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.WfgvgKvHR8hRLexM9zmnRIm69TIWoYA4uumMGKIgXTs&_LANG=enus "A screenshot displays four Word table rows with full lines of anchor text added.")
   7. Save the Word document and return to the Gen Upload
      Documents page.

You have completed the process. You have added a table
with XML tags to the Gen Word template. Additional rows are dynamically added to the table,
based on the number of related Salesforce fields.
