# Use Object Key Prefixes to Identify an Envelope's Salesforce Object

Learn how to find the Salesforce object from which users sent a Docusign envelope.

The Docusign Status page displays the status of Docusign envelopes sent from Salesforce records. You can find the page by opening
Docusign Apps Launcher for Salesforce and selecting the
Docusign Status tab.

However, Docusign Status does not display the name of the Salesforce object
from which you sent the envelope. You can use Salesforce object key
prefixes to identify the Salesforce object type.

Object key prefixes are the first three characters in a Docusign envelope's
Salesforce Source ID. Each three-character prefix
represents a different Salesforce object.

## Example

You want to know from which objects (Accounts, Opportunities, and so on) you sent an envelope.
You follow these steps:

1. Open Docusign Apps Launcher for Salesforce.
2. Select the Docusign Status tab.
3. Check the first three digits of the envelope IDs in the Salesforce Source
   ID column.![A screenshot of the Docusign Status page highlights the first three digits of envelopes' Salesforce Source IDs.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-of-the-Docusign-Status-page-highlights-the-first-three-digits-of-envelopes-Salesforce-Source-IDs.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYyMjAsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.-UU5oGvS4TUtJ6fLHCTwvp9L3eZVYUJ4jRMosBsSYCI&_LANG=enus)
