# Customize Agreement Cloud Editor Template Text

You can customize Agreement Cloud Editor template documents to change font size and
style, add sections of conditional content and add tables.

You can customize Agreement Cloud Editor template documents in the following ways:

- Change font size and style
- Add sections of conditional content
- Add tables

Changes made in the template display in generated documents. You can customize text by
using the toolbar displayed above the document:
