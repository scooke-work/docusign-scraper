# Gen Word Template Options: Select Source Step

Learn how to change the data source of a Gen template.

The Select Source step has only one customizable option: the Salesforce
data source. The data source is the Salesforce object that you select at the start of
Gen template creation. After you have chosen your data source, you can go back and
change it in the Select Source step.
