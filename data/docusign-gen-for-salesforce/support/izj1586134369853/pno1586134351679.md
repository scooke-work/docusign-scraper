# Add Child Objects to a Gen Template

Learn to add child objects in the Gen Data Fields step.

Use this procedure to add child objects to parent objects in the Gen Data Fields step:

1. Select + Add under a field.
2. Select the Select an Option dropdown menu.
3. Select a child object. Child objects display in the following lists:

   - Lookup Relationships: Choose fields from objects linked to
     the parent object.
   - Related Lists: Choose from fields related to the parent
     object.
4. Select + Add to add more levels of objects.![A screenshot displays the relation between child objects and parent objects on a Gen template.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-displays-the-relation-between-child-objects-and-parent-objects-on-a-Gen-template.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY1MjUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.U8vpXmC5cRYusz39eoYIUFTI-co7MuwLujyGZtqoeqE&_LANG=enus "A screenshot displays the relation between child objects and parent objects on a Gen template.")
5. Select Finish and Save when you are
   finished adding fields.

You have completed the process. You have added child
object fields with up to four levels of relation to parent objects. Paste the anchor text from
these data fields into your Microsoft Word template.
