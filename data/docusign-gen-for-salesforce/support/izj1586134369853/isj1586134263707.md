# Add the Docusign Files Component to a Salesforce Layout

Learn to add the Docusign Files component to a Salesforce page
layout.

Docusign Files is a component that displays all Docusign files associated with a record. Users can interact with Docusign
content without leaving Salesforce. You can add the Docusign Files
component to a Salesforce layout.

[Follow this how-to video](https://docusign.gallery.video/watch?videoId=6307338125112) or use this procedure to add the Docusign Files to a layout:

1. As a Salesforce administrator, open a record, select the gear icon ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/salesforce-gear_.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYxNzUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.f12QpFGKfemGiUf2vSgzuFd0NObT_uXwPx3C5Qn8SEU&_LANG=enus) and choose
   Edit Page from the dropdown menu.
2. Drag the Docusign Files component to the page layout.

   ![A screen capture displays the Docusign Files component being dragged to the Opportunity page layout.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Add-DocuSign-Files-component.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYxNzUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.f12QpFGKfemGiUf2vSgzuFd0NObT_uXwPx3C5Qn8SEU&_LANG=enus "A screen capture displays the Docusign Files component being dragged to the Opportunity page layout.")
3. Activate the change to your layout:
   1. Select Activate.
   2. Select a tab to set your default options: Org Default,
      App Default, or App, Record Type, and
      Profile.
   3. Select Assign and choose options from the dialog box.
   4. Select Close.
4. Select Save.
5. Select the back arrow ![A screenshot of the Back arrow on the Salesforce Edit Page settings bar](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-of-the-Back-arrow-on-the-Salesforce-Edit-Page-settings-bar.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYxNzUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.f12QpFGKfemGiUf2vSgzuFd0NObT_uXwPx3C5Qn8SEU&_LANG=enus "A screenshot of the Back arrow on the Salesforce Edit Page settings bar") to return to the record.

You have completed the process. You have added the Docusign Files component to a Salesforce layout. Users can interact with Docusign files on Salesforce records.
