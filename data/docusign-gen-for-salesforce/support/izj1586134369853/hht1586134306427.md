# Preview an Agreement Cloud Editor Gen Template

Learn how to see what a document generated from your template looks like.

When you add Docusign and
Salesforce fields to your Gen template, you can check your work using the Preview feature.
Selecting the Preview On button merges a Salesforce record with your
template. You see a preview of a generated document.

1. Open the Gen template Create tab as a Salesforce administrator:
   1. Select the App Launcher grid ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/app-launcher-grid.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzMzgsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.XoO_SlUPEpz09C1hv3oNnGfTdC-GHnzHOMVzWzLNfo8&_LANG=enus) and go to View All >  Docusign Apps Launcher.
   2. Select the Docusign Gen Templates tab.
   3. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzMzgsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.XoO_SlUPEpz09C1hv3oNnGfTdC-GHnzHOMVzWzLNfo8&_LANG=enus) next to a template and choose
      Edit from the dropdown menu.

   The Gen template page opens to the Create tab.
2. Select the Preview Off switch in the upper left of the page.

   The Choose a Record to Preview dialog box opens.
3. Record: Enter or select the name of a Salesforce record.
4. Select Preview.

   The template document opens, displaying the merged Salesforce
   content.
5. **(Optional)**:

   - Show smart content: Highlight all text that was imported from
     Salesforce fields
   - X: Clear this record and select a different record to
     preview![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/genacepreviewsmartcontent.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzMzgsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.XoO_SlUPEpz09C1hv3oNnGfTdC-GHnzHOMVzWzLNfo8&_LANG=enus)
6. Select Preview On to close the preview and return to the
   Template Configurations page.
7. Select Next in the upper right of the page.

   A message opens: Template (your template name)
   updated.

You have completed the process. You have previewed a
document generated from your Agreement Cloud Editor template.
