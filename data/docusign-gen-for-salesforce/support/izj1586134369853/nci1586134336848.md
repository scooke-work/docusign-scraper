# Customize a Dynamic Table in an Agreement Cloud Editor Template

Learn to customize dynamic table content in an Agreement Cloud Editor
template.

You can customize dynamic table content in an Agreement Cloud Editor template.

You can use the following actions:

- Change font size and style
- Add rows
- Add fill colors to cells
- Add, remove, or reorder Salesforce tags
- Delete rows or cells

When you create a new table with the Agreement Cloud Editor, the table toolbar displays
below the table.

|  |  |
| --- | --- |
|  | Add a row above |
|  | Add a row below |
|  | Add, remove, or reorder Salesforce tags |
|  | Add color to a cell |
|  | Delete a selected row or cell |
