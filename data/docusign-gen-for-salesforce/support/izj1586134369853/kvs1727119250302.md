# Create a Generate and Send Experience for Salesforce Flow

Learn to create a Salesforce Flow action that generates an agreement and sends it for
Docusign eSignature.

## Before you begin

Install Docusign Apps Launcher version 6.12 or higher.

[Create a Docusign
envelope template](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=uce1617376465289.html&_LANG=enus).

[Create a Gen template with Microsoft
Word](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=lig1586134209279.html&_LANG=enus "Learn to build a Gen template using a Microsoft Word document.").

[Link a Gen template to an envelope
template](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=bwo1620419687966.html&_LANG=enus).

Note: The current API limitation is 100 generations per 10 minutes.
If you want to increase your limit, [contact your Docusign representative](https://support.docusign.com/s/articles/How-do-I-contact-DocuSign-Customer-Support) about options.

You can link Docusign eSignature and Gen templates to create a
generate and send experience in Salesforce Flow. When you create a "Gen and Send"
experience, Salesforce Flow can generate an agreement and send it for signature.

Gen and Send experiences simplify your business process by automating contract
generation and signing.

Note:
Docusign does not support using the
Sign Now option when you add an In Person
Signer to a Flow.

Use this procedure to create a Gen and Send experience in Salesforce Flow:

1. As a Salesforce administrator, create or edit a Salesforce Flow.
2. Open the Add Element dialog box:
   1. Locate the point in your Flow where you want to add your action.
   2. Move the cursor to the circle icon and select the plus
      + icon that displays.![A screen capture displays the Add Element plus icon in a Salesforce Flow.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screen-capture-displays-the-Add-Element-plus-icon-in-a-Salesforce-Flow.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNzYsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.BOiljy7oQfdSlujicv2yYsqYuB8DceW3dnpbu_ckL6o&_LANG=enus "A screen capture displays the Add Element plus icon in a Salesforce Flow.")
   3. Add Element: Enter and select
      Generate agreement with Docusign
      in the search field.
3. Add a label and API name for your action:
   1. Label: Enter
       Docusign Generate & Send
      .
   2. API Name: Keep or change the default name.
4. Source Id: Choose the ID of the object your Flow is
   built on. For example, if your Flow is built on Opportunity, choose: Triggering Opportunity >  Opportunity ID.
5. Template ID: Add the ID of the Gen template used in the
   Flow.
   1. Select the Docusign Gen
      Templates
      tab.
   2. Select the Gen template that you want to use in your Flow.

      Note: You must choose a Gen template that you linked to an envelope template.
   3. Copy the alpha-numeric template ID at the end of the URL. Do not copy
      the slash marks.![A screenshot displays the Gen template ID highlighted in the envelope template URL.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-displays-the-Gen-template-ID-highlighted-in-the-envelope-template-URL-1.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNzYsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.BOiljy7oQfdSlujicv2yYsqYuB8DceW3dnpbu_ckL6o&_LANG=enus "A screenshot displays the Gen template ID highlighted in the envelope template URL.")
   4. Paste the ID (without slash marks) in the Template
      ID field.
   5. Select Done.![A screenshot displays the New Action dialog box with a Gen template ID and Account ID entered.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Generate_agreement_with_Docusign_flow_action.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNzYsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.BOiljy7oQfdSlujicv2yYsqYuB8DceW3dnpbu_ckL6o&_LANG=enus "A screenshot displays the New Action dialog box with a Gen template ID and Account ID entered.")
6. Select Save >  Activate.

   A message displays: Your automation was
   activated.

You have completed the process. You created a Gen and
Send Salesforce Flow. When conditions are met, Flow uses Docusign to generate an agreement and send it for signature.
