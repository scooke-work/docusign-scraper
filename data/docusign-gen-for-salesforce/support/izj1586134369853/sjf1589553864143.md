# Learn About Using XML to Customize a Word-Based Gen Template

Learn about using XPath/XML to customize your Word-based Gen template.

You can use XML to create advanced logic in Gen Word templates that is not available
in the Gen user interface.

The following are commonly-used XML expressions that you can add to your
template:

- Concatenate strings
- Add a comma-separated related list
- Add a comma-separated, concatenated list of fields
- Merge the current date into a document
- Merge owner information
- Insert an image

## Comma-separated related list

```
|  |  |
| --- | --- |
|  | <# <Repeat Select="//Opportunity//OpportunityLineItems" /> #> |
|  | <# <Content Select="./Name" /> #><# <Conditional Test="count(./preceding-sibling::*) |
|  | +1 < (count(//Opportunity//OpportunityLineItems) - 1)" /> #>, |
|  | <# <EndConditional /> #><# <Conditional Test="count(./preceding-sibling::*)+1 = |
|  | (count(//Opportunity//OpportunityLineItems) - 1)" /> #>, and <# <EndConditional /> #> |
|  | <# <EndRepeat /> #> |
```

## Concatenate strings

```
|  |  |
| --- | --- |
|  | <# <Content Select="concat(//Opportunity/Account.Name, ' - ', //Opportunity/Name)" /> #> |
```

## Comma-separated, concatenated list of fields

```
|  |  |
| --- | --- |
|  | <# <Repeat Select="//Field_1__c[text() != ''] | //Field_2__c[text() != ''] | |
|  | //Field_3__c[text() != '']" /> #><# <Content Select="." /> #><# <Conditional Test=" |
|  | (count(./preceding-sibling::Field_1__c[text() != ''] | ./preceding-sibling::Field_2__c[text() |
|  | != ''] | ./preceding-sibling::Field_3__c[text() != '']) + 1) < (count(//Field_1__c[text() != ''] | |
|  | //Field_2__c[text() != ''] | //Field_3__c[text() != '']) - 1)" /> #>, <# <EndConditional /> #> |
|  | <# <Conditional Test="(count(./preceding-sibling::Field_1__c[text() != ''] | ./preceding-sibling:: |
|  | Field_2__c[text() != ''] | ./preceding-sibling::Field_3__c[text() != '']) + 1) = |
|  | (count(//Field_1__c[text() != ''] | //Field_2__c[text() != ''] | //Field_3__c[text() != '']) - 1)" /> #>, |
|  | and <# <EndConditional /> #><# <EndRepeat /> #> |
```

## Merge the current date into a document

Add the TODAY() formula field to the Salesforce parent object
(Account, Opportunity, and so on.)

Note: You can hide this from page
layouts.

## Merge owner information

Add a formula field like Owner.FirstName&"
"&Owner.LastName to the Salesforce parent object (Account,
Opportunity, and so on.)

Note: You can hide this from page
layouts.

## Insert an image

Use this example to insert an image into a document:

```
|  |  |
| --- | --- |
|  | <# <Image Select="string('https://<<IMAGE ADDRESS>>.png')" Height="2" Width="6" Optional="true"/> #> |
```

## Store an image in a Salesforce field with a URL

```
|  |  |
| --- | --- |
|  | <# <Image Select="/Opportunity/MainCompetitors__c " Height="2" Width="4" |
|  | Optional="true"/> #> |
```

Note: Use inches for the Height and Width
numbers. The URL, Height, and Width specifications here are examples. Replace these
with dimensions from your own images.
