# Step 1: Add a Name, Template Editor, and Select Account as the Data Source

Learn to set the main template details: template name, description, and Salesforce data
source (Account.)

Use this procedure to decide the name, template editing
tool, and select Account as the data source.

Important:
Person
Account does not appear in the list of available objects in this
step. You must select the Account object here to use the
Person Account object later in the process.

1. Select Configure at the far right.
2. Select Create a Template Configuration.
3. Complete the following tasks:
   1. Add Template Name: Enter a name for your
      template.
   2. (Optional) Add Template Description: Add details
      about the template.
   3. Select Data Source: Select
      Account from the drop-down menu.
   4. Select Template Editor: Select
      Microsoft Word.
   5. Select Create.

You have completed step one. Move to [Step 2: Create Docusign and Salesforce Anchor Text Fields](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=irc1586134297835.html&_LANG=enus "Learn to create Docusign and Salesforce anchor text fields for your Gen template.").
