# Remove Users From the Docusign Login Permission Set

Learn to remove users from the Docusign Login permission
set.

The Docusign
 Login permission set helps you manage Docusign access for your Salesforce users. Users with Docusign Login membership can authorize Docusign
app access themselves.

Use this procedure to remove users from the Docusign Login permission set:

1. Select Docusign Login.
2. Select Manage Assignments.
3. Select the check box next to each user that you want to remove.
4. Select Remove Assignments.

   The Are you sure you want to remove these users from this
   permission set? dialog box opens.
5. Select OK.
6. Select Done.

You have completed the process. You have removed one or
more users from the Docusign Login permission set.
