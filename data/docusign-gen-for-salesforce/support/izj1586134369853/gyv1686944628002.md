# Set the Default Date and Time (Locale) Formatting of Gen Data Fields

Learn how to set the default formatting of date and time (locale) data
fields.

You can set the default formatting of date and time (locale) data fields. You can decide how
Gen formats the date and time of generated agreements by setting defaults for these
data fields.

Use this procedure to set the default date and time formatting for a data field:

1. Select Options from the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY1MTIsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.nGdKHONG2RJr4eI3K0ASJUOSOsLjo1UwnQmx1WfuagM&_LANG=enus) menu next to a date
   field:

   ![A screenshot of the dropdown menu for the Gen Created Date field displays the Options and Remove options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-of-the-dropdown-menu-for-the-Gen-Created-Date-field-displays-the-Options-and-Remove-options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY1MTIsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.nGdKHONG2RJr4eI3K0ASJUOSOsLjo1UwnQmx1WfuagM&_LANG=enus "A screenshot of the dropdown menu for the Gen Created Date field displays the Options and Remove options.")

   The Add Fields Options dialog box
   opens.
2. Date Format: Select a date format from the dropdown
   menu, or choose a custom option:

   Important: Set the Date Format to
   Use Template Format to use template-level [Gen date and currency
   formatting](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=kqf1686095608923.html&_LANG=enus).

   - Use Template Format: Use the Gen template
     Date and Currency Format settings
   - Locale Default: Use the Salesforce organization's
     locale for date formatting
   - Custom: Enter a custom date format
3. Time Format: Select a time format from the dropdown
   menu, or choose a custom option:

   - Locale Default: Use the Salesforce organization's
     locale for time formatting
   - Custom: Enter a custom time format
4. (Optional): [Add
   matches or does not match
   logic to the field](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=pwd1586134341196.html&_LANG=enus "Learn to use Salesforce record fields to trigger conditional logic that displays or hides content in your document.").
5. Select Save.
6. Select Save and Close.

You have completed the process. You have set your Gen
data field's default date and time formatting. Generated documents display the date and
time formatting that you chose.
