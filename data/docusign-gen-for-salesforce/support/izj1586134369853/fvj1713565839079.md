# Add a Generate Agreement with Docusign Action to a Salesforce Flow

Learn to add a Generate agreement with Docusign action to your
Salesforce Flow to trigger document generation.

## Before you begin

Install Docusign Apps Launcher version 6.12 or higher.

[Create a Gen template](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=lig1586134209279.html&_LANG=enus "Learn to build a Gen template using a Microsoft Word document.").

Important: The current API limitation is 100 generations per 10 minutes. We recommend running scheduled flows no more
than once every 10 minutes to avoid hitting rate limits. If you want to increase your
limit, [contact your Docusign representative](https://support.docusign.com/s/articles/How-do-I-contact-DocuSign-Customer-Support) about options.

You can build a Generate agreement with Docusign action to add document
generation capabilities to a Salesforce Flow. You can use the Generate agreement
with Docusign action to trigger a Docusign Gen
template. The Gen template generates a document from a record.

Example: You build a Record-Triggered Flow and link it to a Docusign Gen template. You configure the Flow to trigger document
generation when a record's stage changes to Negotiation/Review.
A user changes the stage of the Burlington Textiles opportunity
to Negotiation/Review. Salesforce triggers the Docusign Gen
template linked to your Flow. Docusign generates a document using
data from the record.

Use this procedure to add a Generate agreement with Docusign action to a
Salesforce Flow:

1. As a Salesforce administrator, create or edit a Salesforce Flow.
2. Add a Generate agreement with Docusign action:
   1. Locate the point in your Flow where you want to add your action.
   2. Hover over the circle icon and select the plus +
      icon that displays.![A screen capture displays the Add Element plus icon in a Salesforce Flow.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screen-capture-displays-the-Add-Element-plus-icon-in-a-Salesforce-Flow.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNzEsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.f4UMRkYzv_KPay_c3wBt8LUbOkq_pXjJICgBZcgwNvU&_LANG=enus "A screen capture displays the Add Element plus icon in a Salesforce Flow.")
   3. Choose Action from the Add
      Element dialog box.
   4. Add Element: Enter and select
      Generate agreement with Docusign in the search field.

   The Generate agreement with Docusign dialog box opens.
3. Add a label and API name for your action:
   1. Label: Enter Generate agreement with
      Docusign.
   2. API Name: Keep or change the default name.
4. Source Id: Choose the ID of the object your Flow is
   built on. For example, if your Flow is built on Opportunity, choose: Triggering Opportunity >  Opportunity ID.
5. Template ID: Add the ID of the Gen template used in the
   Flow.
   1. Select the Docusign Gen Templates tab.
   2. Select the Gen template that you want to use in the Flow.
   3. Copy the alpha-numeric template ID at the end of the URL. Do not copy
      the slash marks.![A screenshot displays the Gen template ID highlighted in the envelope template URL.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-displays-the-Gen-template-ID-highlighted-in-the-envelope-template-URL-1.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNzEsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.f4UMRkYzv_KPay_c3wBt8LUbOkq_pXjJICgBZcgwNvU&_LANG=enus "A screenshot displays the Gen template ID highlighted in the envelope template URL.")
   4. Paste the ID (without slash marks) in the Template
      ID field.
   5. Select Save.![A screenshot displays a Generate agreement with Docusign action with Source ID and Template ID highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Generate_agreement_with_Docusign_flow_action.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNzEsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.f4UMRkYzv_KPay_c3wBt8LUbOkq_pXjJICgBZcgwNvU&_LANG=enus "A screenshot displays a Generate agreement with Docusign action with Source ID and Template ID highlighted.")
6. Select Save > Activate.

   A message displays: Your automation was
   activated.

You have completed the process. You added a Generate
agreement with Docusign action to your Salesforce Flow. When conditions
are met, Flow uses Docusign to generate an agreement.
