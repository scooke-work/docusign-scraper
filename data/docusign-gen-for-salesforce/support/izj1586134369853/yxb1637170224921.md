# Step 8: Add Your Template Button to the Salesforce Person Account Layout

Learn to add a custom button to the Salesforce Person Account layout.

Use this procedure to add a custom button to the Salesforce Person
Account layout:

1. Open the Person Account Layout page as a Salesforce
   administrator:
   1. Select the gear ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/salesforce-gear_.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0MjYsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.Xtin1bIw0thkO6zAF93prpMa_by7-NSiyNhTlZB9VI0&_LANG=enus) icon and choose
      Setup.
   2. Enter Object Manager in the Quick Find
      field.
   3. Select the Person Account link in the left
      column.
   4. Select the Page Layouts link in the left
      column.
   5. Select the Person Account Layout link.
2. Select the Mobile & Lightning Actions link in the
   left column of the Person Account layout menu.
3. Drag the Person Account template button to the Salesforce Mobile and
   Lightning Experience Actions section.
4. Select the Save button in the upper left.

   ![A screen capture displays the cursor dragging the Person Account button to Mobile and Lightning Actions and selecting Save.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Add-Person-Account-custom-button.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0MjYsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.Xtin1bIw0thkO6zAF93prpMa_by7-NSiyNhTlZB9VI0&_LANG=enus "A screen capture displays the cursor dragging the Person Account button to Mobile and Lightning Actions and selecting Save.")

   The page refreshes.

You have completed the entire process. You built a template for
the Person Account object, and added it to the Person Account page layout.
