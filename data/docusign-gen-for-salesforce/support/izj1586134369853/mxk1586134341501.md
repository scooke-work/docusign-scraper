# Upload Documents, Change File Names, or Add Rules to a Gen Template

Learn to edit the generated file name, add files, or add document rules to your Gen
template.

Use this procedure to edit the generated file name, add files,
or add document rules to your Gen template:

1. Change any file name settings:

   - Generated File Name: Edit the text for the beginning of all
     generated document names.
   - File Suffix: Set the second part of all generated document
     names.
   - File Format: Choose Microsoft Word or PDF as the file type
     for all generated documents.
2. Upload Files: Add documents to the Gen template.
3. + Add Rule: [Create a Document Rule in a Word-Based Gen Template](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=djo1614894478713.html&_LANG=enus "Learn to create a document rule to enable conditional document generation in a Gen template:").
4. Bulk Document Creation: Select Generate all documents
   in this configuration with one click to merge all template documents into a
   single generated document.
5. Select Save and Close and select Save on
   the Do you want to save changes? dialog box.

You have completed the process. You have changed one or
more components of your Gen template on the Upload Documents page.
