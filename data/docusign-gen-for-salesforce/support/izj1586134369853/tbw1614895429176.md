# Create Document Rules to Add Conditional Logic to a Word-Based Gen Template

Learn about using document rules to add conditional logic to a Gen template.

Administrators can create document rules to add conditional logic to Gen templates. Conditional
logic uses document rules to customize generated agreements. You can use the following
types of logic in a document rule:

- Conditional document generation: Load different versions of a Gen
  template based on fields in a Salesforce record.
- Match: Generate content when fields are present on a Salesforce
  record.
- Does Not Match: Generate content when fields are not present on a
  Salesforce record.
