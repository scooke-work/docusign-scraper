# "Failed to Generate Lightning Component: Cannot Create a New Component with the Namespace: [Custom Object Name]" Displays When I Try to Add a Gen Button

Learn to add a Gen button to a custom object layout when the Gen Custom Button page
fails.

I used a custom object as my Data Source when building a Gen template. When I got to the
Create Button step, an error displayed:

"Failed to generate lightning component: Cannot create a new component with the
namespace: [Custom Object name]. Only components in the same namespace as the organization
can be created through the API."

I cannot add the Gen button to a layout.

## Resolution 1: Add a Custom Button to a Layout

Cause: Docusign cannot add a button to a layout
when administrators choose a custom Salesforce object as the Main Data Source. An error
displays in the Create Button step. In many cases, Docusign creates the
button, despite this error. The administrator must use Salesforce controls to add the
button to a layout.

Use this procedure to fix the issue:

1. Open the Object Manager page as an administrator:
   1. ## Resolution 2: Add a Detail Page Button to a Layout

Cause: Docusign cannot add a button to a layout
when administrators choose a custom Salesforce object as the Main Data Source. An error
displays in the Create Button step. In many cases, Docusign creates the
button, despite this error. The administrator must use Salesforce controls to add the
button to a layout.

You can use code to create a Detail Page Button instead.

Use this procedure to fix the issue:

1. Open the Buttons, Links and Actions page as an administrator:
   1. Select the gear ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg0MzMsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.OB7G3PCa3voFXutqei0ae9Beq5h73jbbb28VzMQrO20&_LANG=enus) and choose Setup.
   3. Enter and select Object Manager in the Quick Find field.
   4. Select a Salesforce object.
   5. Select Buttons, Links and Actions.

   The Buttons, Links and Actions page displays.
2. Create a new button:
   1. Select New Button or Link .
   2. Label: Enter a name for your Gen button.
   3. Name: Keep the default.
   4. Display Type: Select Detail Page
      Button.
   5. Behavior: Keep or edit the default.
   6. Content Source: Keep URL.
   7. Paste the following content in the field:
      `{!URLFOR('/apex/dfsle__gendocumentgenerator', null, [sId =
      Opportunity.Id, templateId = 'a0J6g000000nMjzEAE', recordId = Opportunity.Id,
      recordName = Opportunity.Name, title = 'Generate MSA'])}`

      Note: Replace the object name with your object, the
      templateID with your Gen template Id, and enter a title.
   8. Select Save.

      ![A screenshot displays the custom button or link page with custom Gen template text entered.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/gendetailpagebutton.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg0MzMsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.OB7G3PCa3voFXutqei0ae9Beq5h73jbbb28VzMQrO20&_LANG=enus "A screenshot displays the custom button or link page with custom Gen template text entered.")
3. A dialog box displays: The new custom button will not be displayed to users until
   you add it to the page layout. Select OK.
4. Select Page Layouts on the page that opens.
5. Add the Gen button to a layout:
   1. Page Layout Name: Select a page layout.
   2. Select Buttons.
   3. Drag the Gen button to the Custom Buttons section.
   4. Select Save.

      ![A screenshot displays a button being dragged into the Opportunity Detail section of a page layout.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/gendetailpageaddbutton.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg0MzMsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.OB7G3PCa3voFXutqei0ae9Beq5h73jbbb28VzMQrO20&_LANG=enus "A screenshot displays a button being dragged into the Opportunity Detail section of a page layout.")

   You have completed the process. You have added the Gen Detail Page Button to
   the layout.
