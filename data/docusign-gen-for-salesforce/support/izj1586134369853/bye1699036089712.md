# Connect an IL4 or MIL Salesforce Organization to Docusign

Learn to connect your IL4 or MIL Salesforce organization to your Docusign account.

## Before you begin

[Install Docusign Apps Launcher in Your Salesforce Organization](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=ogj1586134254571.html&_LANG=enus "Learn how to install the Docusign Apps Launcher package of apps in your Salesforce organization.")

To use Docusign apps in your Salesforce organization, you must first connect
Docusign and Salesforce. You create this connection by
entering Salesforce and Docusign administrator credentials on the
Docusign Setup page. The Docusign
administrator used to connect accounts is called the integration user.

Note:

- Docusign recommends that you create a generic integration
  user. For more information, see this topic: [Apps Launcher Integration User Best Practices](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=rhi1657226496755.html&_LANG=enus "Learn how best to manage the integration user that connects your Salesforce and Docusign accounts.").
- Docusign does not support connecting multiple Salesforce
  organizations to a single Docusign account.

Use this procedure to connect your IL4 or MIL Salesforce organization to a Docusign production account:

1. Update Session Settings as a Salesforce
   administrator:
   1. Select the gear ![A screenshot displays a six-point gear with a lightning bolt inside.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYxNTksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9._6UQqoOmdoySt-FH88o-YKuIRAP2wmX-H2xhFKK7ZGs&_LANG=enus "A screenshot displays a six-point gear with a lightning bolt inside.") and choose
      Setup.
   2. Enter and select `Session Settings` in the Quick
      Find field.
   3. Uncheck Lock sessions to the IP address from which they
      originated.
   4. Uncheck Lock sessions to the domain in which they were first
      used.![A screenshot displays two deselected options for Lock sessions.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-displays-deselected-options-for-Lock-sessions-to-the-IP-address-and-Lock-sessions-to-the-domain.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYxNTksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9._6UQqoOmdoySt-FH88o-YKuIRAP2wmX-H2xhFKK7ZGs&_LANG=enus "A screenshot displays two deselected options for Lock sessions.")
   5. Select Save.
2. Complete the following from the Authorize page:
   1. Select Allow Access and choose
      Allow on the dialog box that displays.
   2. Enter Salesforce administrator credentials in the dialog box that displays
      and select Log In.

   The Log in to Docusign page displays.
3. Complete the following steps on the Log in to Docusign
   page:
   1. Select Log in to Docusign. The Docusign Login dialog box displays.
   2. Select Advanced Options.
   3. Other Environment: Enter
      `USFED` in the field and select Log
      in.![A screenshot displays the Connect to Docusign page with Advanced Options and Other Environment set to USFED.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-displays-the-Other-Environment-field-with-USFED-entered-on-the-Connect-Docusign-to-Salesforce-page.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYxNTksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9._6UQqoOmdoySt-FH88o-YKuIRAP2wmX-H2xhFKK7ZGs&_LANG=enus "A screenshot displays the Connect to Docusign page with Advanced Options and Other Environment set to USFED.")
   4. Enter Docusign administrator credentials in the
      dialog box that displays and select Log in.
   5. Select Use Custom Domain in the Salesforce
      dialog box that displays.
   6. Enter your Salesforce domain in the Custom
      Domain field and select
      Continue.![A screenshot displays the Use Custom Domain field filled with an example domain.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-displays-the-Use-Custom-Domain-field-filled-with-an-example-domain.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYxNTksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9._6UQqoOmdoySt-FH88o-YKuIRAP2wmX-H2xhFKK7ZGs&_LANG=enus "A screenshot displays the Use Custom Domain field filled with an example domain.")
   7. Re-enter Salesforce administrator credentials in the dialog box that
      displays and select Log In.

   The Select an account dialog box displays.
4. Select Account: Choose a Docusign
   account and select Next.

   The Docusign Setup page displays.
5. Edit the Salesforce Connect URL in your Docusign
   account:
   1. Log on to your Docusign account as an administrator
      and select Settings.
   2. Select Connect in the left column.
   3. Docusign Apps Launcher: Select
      Actions and choose
      Edit.
   4. ULR to Publish: Change
      act.mil in the URL to
      fed-act-event.

      - [Correct example](https://act-event.docusign.mil/listeners/connect-salesforce)
      - [Incorrect example](https://fed-act-event.docusign.com/listeners/connect-salesforce)
   5. Select SAVE CONFIGURATION.

You have completed the process. You have connected your
IL4 or MIL Salesforce organization to your Docusign account. You can
set up your Docusign Apps Launcher applications.
