# Add Split and Nested Conditional Tags to a Gen Template

Learn how to add split and nested conditional tags to a Word-based Gen
template.

You can add conditional logic blocks to Gen template Anchor Text Tags. Conditional logic
creates "if-then" statements that add or exclude text from a generated agreement. For
example, you can add conditional logic that hides new customer language when generating
existing customer agreements.

You always start conditional logic blocks with a <Conditional>
tag and end with an <EndConditional> tag. You can place
<Conditional> and
<EndConditional> tags on the same line. You create split
conditionals by splitting conditional text over multiple lines of text.

You can also use nested conditional logic, which involves placing child conditional tags
inside of parent tags. For example, you have conditional text for new customers and add
child conditional text for all new California customers.

Use this procedure to add split and nested conditional tags to a Word-based Gen
template:

1. Select a document name.

   The document preview page opens in a new tab or window.
2. Add the conditional anchor text to your Word document. In this example, you split
   conditional tags over multiple lines and add a nested legal clause for California
   customers:
   1. Select Download and open the document in Microsoft
      Word.
   2. Where you want the conditional text to display, paste `<# <Conditional
      Select="Opportunity/Type" Match="New Customer"/ > #>Welcome, new
      customer.<# <Conditional Select="Opportunity/State" Match="California" />
      #>Custom California Legal clause< #<EndConditional/>
      #>`.
   3. Replace the Custom California Legal clause text with your
      California legal clause text.![A screenshot displays split and nested conditional text around California legal clause text.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-displays-split-and-nested-conditional-text-around-California-legal-clause-text.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzMDksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.o_iYB00t2aVSjhAoSsLFX37fZPuH0idsHBilwO6qqK0&_LANG=enus "A screenshot displays split and nested conditional text around California legal clause text.")
   4. Save changes and go to the Gen Upload Documents page.
3. [Re-upload the document to your Gen
   template](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=sqp1586134339354.html&_LANG=enus "Learn to update your Gen template Word documents.")
4. Select Save and Close and select Save on
   the Do you want to save changes? dialog box.

You have completed the process. You have added split and nested
conditional logic for new customers. Generated agreements for California customers display the
California legal clause. Generated agreements for all other new customers do not display the
California clause.
