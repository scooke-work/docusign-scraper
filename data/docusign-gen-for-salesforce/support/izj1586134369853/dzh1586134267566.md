# Add Docusign Buttons, Links, or Actions to a Salesforce Layout

Learn to add Docusign buttons, links, or actions to your Salesforce
layouts.

You can add Docusign buttons, links, or actions to your Salesforce
layouts. Your users can access Docusign features with these controls. For
example, when you add a Gen button to a layout, your users can select it to generate an
agreement.

Use this procedure to add Docusign buttons, links, or actions to a
layout:

1. Access a page layout as a Salesforce administrator:
   1. Select the gear icon ![Salesforce gear icon used to access Setup in the header.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/salesforce-gear_.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYxODUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.x7l4qMryLVRrPNxcHsw_g3LJpjBmkSUekKGQ9rJyZsQ&_LANG=enus "Salesforce gear icon used to access Setup in the header.") and choose Setup.
   2. Enter and select `Object Manager` in the Quick Find
      field.
2. Select a Salesforce object from the Label column.
3. Select Page Layouts.
4. Select a layout from the Page Layout Name column.
5. Drag a button, link, or action from the menu to the layout.

   ![Salesforce page layout editor showing a custom button being dragged from the menu to the page layout canvas.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/dragtocanvas.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYxODUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.x7l4qMryLVRrPNxcHsw_g3LJpjBmkSUekKGQ9rJyZsQ&_LANG=enus "Salesforce page layout editor showing a custom button being dragged from the menu to the page layout canvas.")
6. Select Save.

You have completed the process. You have added a
button, link, or action to a layout. The items you added display on every record of that
object type.
