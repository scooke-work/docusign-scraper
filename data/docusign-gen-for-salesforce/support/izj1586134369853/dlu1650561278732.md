# A Red "OK" Error Displays When I Try to Access Any Docusign Component

Learn to upgrade Docusign Apps Launcher to avoid seeing an error when
accessing components.

I can't access any Docusign components on my layouts.

A red OK error displays when I select any component on a record. This
includes any of the following components:

- Docusign Files
- Docusign Envelope Status
- Docusign CLM Files
- Docusign CLM Obligations
- Docusign CLM Tasks

## Resolution: Upgrade to the Latest Version of Docusign Apps Launcher

Cause: Your organization has an older version of Docusign Apps Launcher installed. Upgrade to the latest
version.

Use this procedure to fix the issue:

1. Check your Docusign Apps Launcher version number:
   1. Select the gear ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg0ODQsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.Y1mD8HTfhRPe5B519TWHBR_s3XfiftbQbCx99J3XmUU&_LANG=enus) and choose
      Setup.
   2. Enter and select Installed Packages in
      the Quick Find field.
   3. Check your Version Number.
2. Choose an upgrade path:

   - If your version is higher than 6.0, [upgrade to
     the latest Apps Launcher version](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=nok1698959626724.html&_LANG=enus).
   - If your version is 6.0 or lower, [use this process to upgrade
     to the latest version](https://support.docusign.com/s/articles/DocuSign-Apps-Launcher-Is-there-a-recommended-upgrade-path-for-an-older-version-of-the-package).

   You have completed the process. You have upgraded your older Docusign Apps Launcher package. You can open Docusign components on Salesforce records.
