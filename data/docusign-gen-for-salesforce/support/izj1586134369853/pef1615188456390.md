# "Failed to Update One or More Metadata Items" Displays When I Create a Gen for Billing Custom Button

Learn to fix the "Failed to update one or more metadata items" error.

I created a new Gen for Billing template and selected Save and
Close before creating the custom button.

I opened the template later and added a button name.

When I selected Update Layouts, an error displayed:
Failed to update one or more metadata items. I cannot add
the button.

## Resolution: Manually Add the Gen for Billing Custom Button to the Invoices Layout

Cause: Saving a Gen for Billing template without creating a
button causes a failure. Add the button through the Salesforce interface.

Use this procedure to fix the issue:

1. Access the Invoices Page Layout as a Salesforce
   administrator:
   1. Select the gear ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg0NjYsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.c-7EKlxCRt7JNxHO26zMDfhzM77v3yRg843GlW0vHUk&_LANG=enus) and select
      Setup.
   2. Enter and select Object Manager in the
      Quick Find field.
   3. Select Invoices from the Label
      column.
   4. Select Page Layouts.

      The Page Layouts page opens.
2. Select the Buttons menu.
3. Drag the Gen for Billing custom button to the Custom Buttons
   menu.
4.    You have completed the process. You have added the Gen for
