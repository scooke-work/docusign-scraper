# Import Custom Objects for Gen

Learn about importing custom objects to your Salesforce organization.

You use existing Salesforce data to build Docusign Gen templates. You can also import data
from other sources and add it to your Salesforce objects like Accounts and Opportunities. When
you have imported this data, you can use it to build a Gen template. You can only import data
into one Salesforce object at a time. You can import up to 50,000 records at a time.

You can take the following actions when importing data:

- Add new records to Salesforce.
- Update existing Salesforce records
- Add new records and update existing records at the same time.

Follow this link to learn more about [importing data to your Salesforce organization from other sources](https://salesforce.vidyard.com/watch/ARIjWm2qrDkJVJxEhReFug).
