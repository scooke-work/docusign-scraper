# Add Conditional Logic to an Agreement Cloud Editor Template

Learn to format Agreement Cloud Editor templates to display, hide, or change content
depending on conditions.

You can format Agreement Cloud Editor templates to display, hide, or change content depending on
conditions. The Add logic rule button ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/genaceconditionalicon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNDUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.9iiEa-l9-2dvoZFVL1IuqxQQkvHmPkO-7Cn4Kqu656E&_LANG=enus)
displays when you highlight any amount of text in the template document. You can create
logic rules to display or hide this text.

For example, you can set a logic rule to display a paragraph of New York legal text if the
Billing State of a record equals New York.

You can also chain multiple rules together to display text if the record data matches a set
of conditions.

Example:

You create a rule to display text if the record has a Billing State
of New York. You add a second rule to display text if the
Employees field value is greater than 100. You
add a third rule to display text if the Annual Revenue field value is
greater than $1,000,000.

Result: Docusign only displays the text if a record
meets each of these conditions:

- The record has a billing state of New York
- The record has more than 100 employees
- The record has annual revenue exceeding $1,000,000

Docusign does not limit the number of rules you can apply to a block of
text.

Use this procedure to create a logic rule for your Agreement Cloud Editor template:

1. Use the cursor to highlight text and select the Add logic rule
   button ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/genaceconditionalicon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNDUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.9iiEa-l9-2dvoZFVL1IuqxQQkvHmPkO-7Cn4Kqu656E&_LANG=enus) that displays to the right.

   ![A screen capture demonstrates applying an Add logic rule to a selection of text.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screen-capture-demonstrates-applying-an-Add-logic-rule-to-a-selection-of-text.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNDUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.9iiEa-l9-2dvoZFVL1IuqxQQkvHmPkO-7Cn4Kqu656E&_LANG=enus "A screen capture demonstrates applying an Add logic rule to a selection of text.")

   Rule Builder opens.
2. Create a name for your rule:
   1. Select the Edit
      ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/genaceeditlogicname.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNDUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.9iiEa-l9-2dvoZFVL1IuqxQQkvHmPkO-7Cn4Kqu656E&_LANG=enus) icon next to
      Name Rule.
   2. Enter a name for your rule.
   3. Select the check mark to save the new name.

   ![A screenshot displays a logic rule name with a green check mark highlighted in green.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/genacerenamerule.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNDUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.9iiEa-l9-2dvoZFVL1IuqxQQkvHmPkO-7Cn4Kqu656E&_LANG=enus "A screenshot displays a logic rule name with a green check mark highlighted in green.")
3. Choose the conditions for your rule:

   - Apply any of the following conditions: Trigger your rule if
     at least one condition is met.
   - Apply all of the following conditions: Trigger your rule if
     all conditions are met.
4. Create If logic. Here you create the conditions that a record must
   meet to trigger your rule:
   1. Select the Salesforce Fields icon ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/genacesalesforcefieldsicon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNDUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.9iiEa-l9-2dvoZFVL1IuqxQQkvHmPkO-7Cn4Kqu656E&_LANG=enus)
      in the left column.
   2. Drag a Salesforce field to the If box.
   3. Equals: Keep this selection or select Does not
      equal from the dropdown menu.
   4. Enter text in the field next to Equals. For example, enter
      "New York" if you want a rule of Billing State/Province >  Equals >  New York.

      ![A screen capture demonstrates applying a Salesforce field to an if logic block.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screen-capture-demonstrates-applying-a-Salesforce-field-to-an-IF-logic-block.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNDUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.9iiEa-l9-2dvoZFVL1IuqxQQkvHmPkO-7Cn4Kqu656E&_LANG=enus "A screen capture demonstrates applying a Salesforce field to an if logic block.")
   5. (Optional) OR IF: Add more conditions to your rule.
   6. (Optional): Move the cursor to a condition and select ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/genacex.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNDUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.9iiEa-l9-2dvoZFVL1IuqxQQkvHmPkO-7Cn4Kqu656E&_LANG=enus)
      to delete it.
   7. (Optional) + ADD RULE: Add another rule to this block of
      text.
5. Decide whether to display or hide text when your rule is matched:

   - Then: Choose an option from the dropdown menu:
     - Show the following content
     - Hide the following content
   - Otherwise...: Choose an option from the dropdown menu:
     - Hide the following content
     - Show the following content
6. Save and publish your rule:
   1. Select Save at the top right of the Rule Builder
      section.
   2. Select Next.
   3. Select Publish.

   A message displays: Template (your template name)
   updated.

You have completed the process. You added a logic rule to your
Agreement Cloud Editor template. When users generate documents, Docusign
displays, hides, or changes content based on record data.
