# Duplicate Salesforce Documents Display When I Send Gen Agreements for Signature

Learn to work around the duplicate files issue when sending Gen agreements for
signature.

I built a Gen template that is connected to an envelope template. When I send generated
agreements for signature, I see duplicate documents in the
Documents step.

## Resolution 1: Uncheck the Boxes of Duplicate Salesforce Files Before Sending

Cause: You selected the Attach documents from
Salesforce record check box on the envelope template
Documents page. Selecting this option in an envelope
template that is linked to a Gen template causes file duplication.

Use this procedure to fix the issue:

1. Select a Gen button on a Salesforce record.

   Gen creates a new agreement, and opens a preview page.
2. Select Send for Signature.

   Your linked envelope template displays the
   Documents page.
3. De-select the check box for any duplicate Salesforce documents that
   display.
4. Proceed with the rest of the envelope creation process.

   You have completed the process. You have removed any duplicate Salesforce
   documents and sent the envelope.
