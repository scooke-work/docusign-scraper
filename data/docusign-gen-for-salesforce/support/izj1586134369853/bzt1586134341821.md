# Delete a Gen Button

Learn to remove a Gen custom button from your Salesforce page layouts.

Use this procedure to remove a Gen custom button from
your Salesforce layouts:

1. Access the Buttons, Links, and Actions page as a Salesforce
   administrator:
   1. Select the gear ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/salesforce-gear_.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0NTksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.4N8rfJSSF4hk73r6zO5F051d2WqhXe-sbh-3DMlIOV4&_LANG=enus) icon and choose Setup.
   2. Enter and select Object Manager in the Quick Find
      field.
   3. Select an object (Account, Opportunity, and so on) in the left navigation
      bar.
   4. Select Buttons, Links, and Actions.
2. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0NTksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.4N8rfJSSF4hk73r6zO5F051d2WqhXe-sbh-3DMlIOV4&_LANG=enus) next to a custom button and
   choose Delete from the dropdown menu.

   ![A screenshot displays the Buttons, Links, and Actions page with two custom button links.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/gendeletecustombutton.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0NTksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.4N8rfJSSF4hk73r6zO5F051d2WqhXe-sbh-3DMlIOV4&_LANG=enus "A screenshot displays the Buttons, Links, and Actions page with two custom button links.")

   A dialog box displays: Are you sure?.
3. Select Delete. The page refreshes with the button no longer
   displayed.

   Note: Salesforce may display an error:
   Your request to delete this custom button or link can not be completed
   because this button or link is being used by another feature. Remove the
   button from all Salesforce layouts.

You have completed the process. You have removed the
button from Salesforce layouts.
