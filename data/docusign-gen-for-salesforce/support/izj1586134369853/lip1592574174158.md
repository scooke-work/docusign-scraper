# Add Docusign IP Ranges to Salesforce to Prevent Errors

Learn to add Docusign IP ranges to your Salesforce organization's
Trusted IP Ranges.

You can add all Docusign IP ranges to your Salesforce
organization's Trusted IP Ranges.

Adding IP ranges ensures that all Docusign apps function without
issue in your Salesforce organization.

For more information, view the
following Salesforce documentation: [Set Trusted IP Ranges for Your Organization](https://help.salesforce.com/s/articleView?id=xcloud.security_networkaccess.htm&type=5).

Use
this procedure to add Docusign IP ranges to your Salesforce
organization's Trusted IP Ranges.

Note: Also use this procedure if your organization specifies [IP ranges on user profiles](https://help.salesforce.com/s/articleView?id=platform.login_ip_ranges.htm&type=5).

1. Access the Network Access page as a Salesforce
   administrator:
   1. Select the gear icon ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/salesforce-gear_.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYyODcsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.Aq48Vr2JBHWLuE32qrRWvfgzgI4XMC2aXospE6Vk1hc&_LANG=enus)and
      choose Setup.
   2. Enter and select Network Access in the Quick
      Find field.

   ![A screenshot of the Salesforce Network Access page displays Trusted IP Ranges.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Salesforce-Network-Access-Settings.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYyODcsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.Aq48Vr2JBHWLuE32qrRWvfgzgI4XMC2aXospE6Vk1hc&_LANG=enus "A screenshot of the Salesforce Network Access page displays Trusted IP Ranges.")
2. Add IP ranges from the Docusign Trust Center to
   your Trusted IP Ranges:

   - [eSignature for Salesforce ranges](https://www.docusign.com/trust/security/esignature)
   - [Gen and CLM for Salesforce ranges](https://www.docusign.com/trust/security/clm)
3. (Optional) [Add Salesforce IP ranges](https://ip-ranges.salesforce.com/ip-ranges.json) if your Salesforce organization is built on [Hyperforce code](https://help.salesforce.com/s/articleView?id=000388902&type=1).
4. Select Save.

You have completed the process. You have added Docusign IP ranges
to your Salesforce organization's Trusted IP Ranges section.
