# Use the CLM Tab of the CLM Document Generation Page

Learn to use the CLM tab of the CLM Document Generation page.

When you go to Docusign Setup >  Document Generation, you see the following tabs:

- CLM: Links and information about CLM Document Generation
  features.
- Docusign Gen: Links and information about how to create, edit, and
  delete Docusign Gen templates.

This guide demonstrates how to access and use the CLM tab:

1. Select the CLM tab:![A screenshot displays the CLM tab of the CLM Document Generation page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/CLM-Document-Generation-tab.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgyNzQsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.4nDkz4I7KJjjE989zYonNuA2HUs5R5nqV5hTjNK1bz8&_LANG=enus "A screenshot displays the CLM tab of the CLM Document Generation page.")
2. Access any of these links or buttons:

   - Learn how to generate documents in Docusign CLM: Select this link
     to open the Components page.
     - Folder Portal: Open the Salesforce Object
       Manager page to add the folder portal component to Salesforce
       layouts.
     - CLM for Salesforce Experience Cloud
       (Communities): Set up a Docusign integration user for your Experience Cloud sites.
   - Learn how to add a button to a Salesforce Object page: Open the
     Components page.
   - Open Docusign CLM: Open your connected Docusign
     CLM account.
   - Open Object Manager: Open the Salesforce Object Manager page to add
     CLM Document Generation buttons to Salesforce layouts.
   - Go to Document Generation: Open the Document Generation page of
     your connected Docusign CLM account.

You have completed the process. You have explored the
features of the CLM tab.
