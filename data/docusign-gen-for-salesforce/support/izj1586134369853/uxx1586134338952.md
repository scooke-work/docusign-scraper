# Change the Salesforce Data Source of Your Word-Based Gen Template

Learn to change the data source of a Gen template.

Warning: Changing your Gen template data source deletes all
of your template's Salesforce fields. Docusign fields remain unchanged.Docusign recommends creating a new template, rather than changing the data
source of an existing template. If you do want to change the data source of a Gen template,
use this procedure:

1. Select the Select Source arrow. The arrow displays no text until
   selected.
2. Select Salesforce Object and select a new Salesforce object from
   the dropdown menu.
3. Select Next.

You have completed the first part of the process. You
have updated the main data source. Proceed to step two to [add Salesforce anchor text tags](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=lig1586134209279.html&_LANG=enus "Learn to build a Gen template using a Microsoft Word document.") to the Gen template.
