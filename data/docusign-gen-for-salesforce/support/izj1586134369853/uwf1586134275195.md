# Create Docusign for Salesforce Users

Learn to create Docusign Gen, CLM, Negotiate, or eSignature users on the user
management page.

Use this procedure to create Docusign
eSignature, CLM, Gen, or Negotiate users on the user management page. You can only add
permissions for Docusign products you have purchased.

Important: A new user displays a Pending
status [until activated by the user](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=ymi1586134213877.html&_LANG=enus). After activation, user status displays as Active.

1. Select Add Users.
2. Complete the following tasks in the Select Users dialog box:
   1. Value: Enter a Salesforce user name or email address.
   2. Select the check box next to each Salesforce user that you want to add.
   3. (Optional) Filtered by Filter your search by
      User, Profile, or
      Permission Set.
   4. Select Continue.![A screenshot displays the Select Users dialog box with filters added for User and Profile and a user check box selected.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Use-filters-to-add-a-user.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYyMzMsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.CfUSISMade2uC2IoMtrXRDVkFAvx3Vi2TXIIZxeMO80&_LANG=enus "A screenshot displays the Select Users dialog box with filters added for User and Profile and a user check box selected.")
3. Apply Permissions: Select permissions for your users:

   - eSignature:
     - Administrator: Administrators can create templates and
       manage all Docusign users in Salesforce.
     - User: Users can send documents for signature.
   - CLM:
     - Administrator: Administrators can access everything in
       CLM and manage connections with Salesforce. Administrators cannot manage Docusign users in Salesforce.
     - User: Users can access workflows and view, edit, and
       upload documents.
     - Viewer: Viewers can only preview and upload
       documents.
     - No Access: Users have no access to the CLM app.
   - Gen:
     - Generator: Generators can create new documents and
       templates.
     - No Access: Users have no access to the Gen app.
   - Negotiate:
     - Negotiator: Negotiators can collaborate with signers and
       viewers.
     - No Access: Users have no access to the Negotiate
       app.
   - (Optional) Update permissions for any existing users: Select
     to update a user's permissions in the connected Docusign
     account.
4. Select Apply.

   The Summary dialog box opens, displaying the message:
   (User name or number of users) was added.

You have completed the process. You have granted or updated Docusign permissions to Salesforce users.
