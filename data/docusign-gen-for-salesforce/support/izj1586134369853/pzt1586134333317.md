# Add a Dynamic Table to an Agreement Cloud Editor Template

Learn to add a dynamic table to an Agreement Cloud Editor template.

You can add a dynamic table to an Agreement Cloud Editor
template. Dynamic tables automatically add rows and columns based on the amount of data on a
Salesforce record.

Use this procedure to add a dynamic table to an Agreement Cloud Editor
Gen template:

1. Open the Gen template Create tab as a Salesforce administrator:
   1. Select the App Launcher grid ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/app-launcher-grid.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNTksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.Zn3D9L85kE9N5D79O4sIQO_V9e1npJRAjFvDy5RGLh8&_LANG=enus) and go to View All >  Docusign Apps Launcher.
   2. Select the Docusign Gen Templates tab.
   3. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNTksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.Zn3D9L85kE9N5D79O4sIQO_V9e1npJRAjFvDy5RGLh8&_LANG=enus) next to a template and choose
      Edit.

   The Gen template page opens to the Create tab.
2. Place the cursor where you want to add a table.
3. Select the Salesforce Fields tab ![A screenshot displays two interlocking blue rectangle icons.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/genacesalesforcefieldsicon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNTksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.Zn3D9L85kE9N5D79O4sIQO_V9e1npJRAjFvDy5RGLh8&_LANG=enus "A screenshot displays two interlocking blue rectangle icons.") to the left of the Docusign Fields menu.

   The Salesforce Fields menu opens.
4. Search for and select the Opportunity Line Items field.

   ![A screenshot displays the Opportunity Line Items field displayed in the Salesforce Fields section.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/genacelineitemsfield.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNTksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.Zn3D9L85kE9N5D79O4sIQO_V9e1npJRAjFvDy5RGLh8&_LANG=enus "A screenshot displays the Opportunity Line Items field displayed in the Salesforce Fields section.")
5. Create your dynamic table:
   1. Drag items from the Fields menu to the Selected
      Columns section.![A screenshot displays a Salesforce field moving to the Selected Columns section.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/genacelineitemsadd.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNTksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.Zn3D9L85kE9N5D79O4sIQO_V9e1npJRAjFvDy5RGLh8&_LANG=enus "A screenshot displays a Salesforce field moving to the Selected Columns section.")
   2. (Optional) Select X to remove a column.
   3. (Optional) Drag columns to change their order.

      ![A screenshot displays a column moving lower in the Selected Columns section.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/genacelineitemsdrag.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNTksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.Zn3D9L85kE9N5D79O4sIQO_V9e1npJRAjFvDy5RGLh8&_LANG=enus "A screenshot displays a column moving lower in the Selected Columns section.")
   4. Select Apply.

   A dynamic table with Salesforce merge fields displays in your template.
   Example:![A screenshot displays a dynamic table with Salesforce line items.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-displays-a-dynamic-table-with-Salesforce-line-items.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNTksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.Zn3D9L85kE9N5D79O4sIQO_V9e1npJRAjFvDy5RGLh8&_LANG=enus "A screenshot displays a dynamic table with Salesforce line items.")
6. Select Next and Publish.

   A message displays: Template (your template name)
   updated.

You have completed the process. You have added a
dynamic table to your Gen template. The dynamic table automatically adds rows and columns
based on the amount of data on a Salesforce record. All text in the table matches the font
size and style of the template.
