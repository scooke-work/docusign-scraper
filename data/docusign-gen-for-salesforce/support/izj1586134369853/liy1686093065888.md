# Add Gen and Send Functionality to Your Gen Template

Learn to add Gen and Send functionality to your Gen template.

## Before you begin

Install Docusign Apps Launcher version 5.3 or higher.

[Create a Docusign
envelope template](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=uce1617376465289.html&_LANG=enus).

[Create a Gen template with Microsoft
Word](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=lig1586134209279.html&_LANG=enus "Learn to build a Gen template using a Microsoft Word document.").

Important:

- Use a Word-based Gen template. You cannot link an [Agreement Cloud
  Editor](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=bcp1586134298132.html&_LANG=enus)-based Gen template to an envelope template.
- You can only link templates built on the same Salesforce object. Linking a
  Gen template and envelope template built on Opportunity
  works. Linking an Opportunity-based Gen template with
  an Account-based envelope template does not work.
- You can only link the following types of envelope templates: Default,
  Quick Send, Place Fields, and
  Send Now.

When you build a Gen and Send experience, your users can generate and send an
agreement with one click. When users generate an agreement, the linked envelope
template sending experience immediately opens. When all recipients complete the
signing process, the finished contract is saved to your Docusign
account and the Salesforce record.

Use this procedure to add Gen and Send functionality to your Gen template:

1. Add Gen and Send functionality to your Gen template:
   1. Select Enable the Send for Signature button, so generators
      can send generated documents out for signature.
   2. Select Launch automatically after
      generation.
   3. Select an Envelope Template to streamline sending for
      signature: Enter and select the name of an envelope
      template. ![A screenshot shows Launch automatically after generation selected and the Fontara NDA envelope template linked.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-shows-Launch-automatically-after-generation-selected-and-the-Fontara-Mobile-Sales-envelope-template-linked.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgyNjYsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.ltmy0KsTK2skx12WOEwjQ_KZxozTQaG0JNgTuaGEstg&_LANG=enus "A screenshot shows Launch automatically after generation selected and the Fontara NDA envelope template linked.")
2. Select Save and Finish.

   The Create Button page closes.

You have completed the process. You have linked your Gen template
to an envelope template. When users generate an agreement, the linked envelope template
sending experience opens.
