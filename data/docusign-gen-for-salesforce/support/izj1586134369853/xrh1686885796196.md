# Change the Currency Type of a Salesforce Record

Learn to change the type of currency Salesforce displays on a record.

Your Salesforce organization has a default currency format. However, your administrator can
add new currencies to your organization. Adding currencies lets you change the
default currency on a record.

Example: Your Salesforce organization uses U.S.
Dollar as the default currency, but your administrator has added
Euro. You change the Opportunity
Currency field of a French company to Euro.
The Amount field displays the value in Euros.

Note: If you want to [choose the currency format when generating documents](https://support.docusign.com/s/document-item?bundleId=bpg1586134377111&topicId=hhz1686691127617.html&_LANG=enus), you must change a record's currency first.

Use this procedure to change the currency type of a Salesforce record:

1. Select the Details tab of a record.
2. Select the pencil icon ![A screenshot of the pencil icon used to edit fields on a Salesforce record.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-of-the-pencil-icon-used-to-edit-fields-on-a-Salesforce-record.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYzMTAsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.5CBI5P8WM0rpOrCl0UFR34Gw637QwtO8YITIR8hEJao&_LANG=enus "A screenshot of the pencil icon used to edit fields on a Salesforce record.") on the (object) Currency field.
3. Select a new currency type from the dropdown menu.
4. Select Save.

   The Amount field displays the new
   currency.

You have completed the process. You have changed the default
currency of a record.
