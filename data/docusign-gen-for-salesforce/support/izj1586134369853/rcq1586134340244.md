# Edit Salesforce Data Fields in a Word-Based Gen Template

Learn to edit data fields in your Gen template configuration.

Use this procedure to edit data fields in your Gen
template configuration:

1. Select the Add Fields arrow. The arrow displays no text until
   selected.
2. Select the dropdown arrows next to a field. ![A screenshot displays the main data source page with field and anchor text fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-displays-the-main-data-source-page-with-field-and-anchor-text-fields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0OTMsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.1by7MFFQ78gAIA8CbfDbGA8Cc37SuYGjfHh9KhuePV8&_LANG=enus "A screenshot displays the main data source page with field and anchor text fields.")
3. Select a Field from the dropdown menu.
4. Select Finish when you are done making changes.

   Important: Update all
   changed tags from this template's Word document and re-upload the document.

You have completed the first step of the process. You
have changed the tags. Proceed to step two to [edit this
template's Word document](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=sqp1586134339354.html&_LANG=enus "Learn to update your Gen template Word documents.").
