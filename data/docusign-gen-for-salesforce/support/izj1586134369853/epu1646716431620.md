# Use the Docusign Logs Tab to View Docusign Errors

Learn to use the Docusign Logs tab for viewing Docusign error messages in your Salesforce organization.

The Docusign Logs tab opens a page that displays all Docusign error messages. These include all
DEBUG, INFO,
WARN, and ERROR logs.

Log entries include these details:

- Message
- Severity
- Version
- Created By
- Docusign entity ID
- Salesforce source ID
- Transaction ID
- Class
- Method

Use this procedure to view the Docusign Logs tab in your Salesforce
organization:

1. Select the Docusign Logs tab.

   The Docusign Logs page opens.
2. Select a log from the Docusign Log column.

   The Docusign log details page opens for that item.

You have completed the task. You viewed the details of an error
from your Docusign account.
