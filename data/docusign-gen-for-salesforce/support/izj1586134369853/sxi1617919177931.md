# "You Must Have Docusign Billing Admin Permissions to Perform This Operation" Displays When I Try to Create a Gen for Billing Template

Learn to fix the "You must have Docusign Billing Admin permissions to
perform this operation" error.

I am a Salesforce and Docusign administrator with a Gen for Salesforce
Billing license. An error displayed when I built a Gen template: You must
have Docusign Billing Admin permissions to perform this operation.
Why can't I build a Gen for Salesforce Billing template?

## Resolution: Assign Your Salesforce User to the Docusign Billing Admin Permission Set

Cause: Your administrator user is not a member of the **Docusign Billing Admin** permission set.

Use this procedure to fix the issue:

1. Open the **Docusign Billing Admin** permission set
   page as an administrator:
   1. Select the gear ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg0NzAsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.vS573GKfL-3A3mjSjrviEnGb9q3sEkLQzPzhAvMpaKw&_LANG=enus) and choose Setup.
   2. Enter and select Permission Sets in the
      Quick Find field.
   3. Select a Salesforce object.
   4. Select Permission Sets.
   5. Select **Docusign Billing Admin** from the
      Permission Set Label column.

   The Docusign Billing Admin permission set page
   displays.
2. Select Manage Assignments.
3. Select Add Assignment.
4. Select the check box next to your user name and select
   Next.
5. Select Assign and select
   Done.

   You have completed the process. You have added yourself to the
   **Docusign Billing Admin** permission set.
   You can create a Gen for Billing template.
