# Add Conditional Logic to a Gen Template to Show or Hide Content

Learn to add conditional logic to your Word-based Gen template.

You can add conditional logic blocks to Gen template Anchor Text Tags. Conditional
logic creates "if-then" statements that add or exclude text from a generated
agreement.

You can use conditional logic to create one Word document that can generate different
content for different customers.

The following procedure demonstrates adding conditional logic to a customer
agreement:

1.    Options. ![A screenshot displays the description field with the copy menu open to options and remove.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-displays-the-description-field-with-the-copy-menu-open-to-options-and-remove.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY1ODUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.KC8VJXWQlL8Ay4KUS4EEjThc5C7qXPL-zaQy8sLMinY&_LANG=enus "A screenshot displays the description field with the copy menu open to options and remove.")

   The Table Options dialog box opens.
2. Create match logic:
   1. Select Include other content if the conditions set for this
      field are met.
   2. matches: Keep this value.
   3. Enter New York in the text field.
   4. Select Save.![A screenshot displays the display options field with the values set to matches and New York.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Add-conditional-logic.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY1ODUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.KC8VJXWQlL8Ay4KUS4EEjThc5C7qXPL-zaQy8sLMinY&_LANG=enus "A screenshot displays the display options field with the values set to matches and New York.")

   The Billing State/Province anchor text changes to `<#
   <Conditional Select="/Opportunity/Account.BillingState" Match="New York"
   /> #> YOUR CONTENT HERE <#
   <EndConditional/> #>`
3. Select Copy next to the Billing
   State/Province anchor text.
4. Add the conditional anchor text to your Word document:
   1. Open the Gen template's Word document.
   2. Where you want the New York legal clause to display, paste
      `<# <Conditional
      Select="/Opportunity/Account.BillingState" Match="New York" />
      #>`.
   3. Replace the YOUR CONTENT HERE text with your New
      York legal clause text.
   4. Enter `<# <EndConditional/> #>` at the end
      of the New York legal clause text. ![A screenshot displays conditional logic tags surrounding legal text for New York customers.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Conditional-text.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY1ODUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.KC8VJXWQlL8Ay4KUS4EEjThc5C7qXPL-zaQy8sLMinY&_LANG=enus "A screenshot displays conditional logic tags surrounding legal text for New York customers.")
5. [Re-upload the document to your Gen
   template](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=sqp1586134339354.html&_LANG=enus "Learn to update your Gen template Word documents.").
6. Select Save and Close and select
   Save on the Do you want to save
   changes? dialog box.

You have completed the process. You have added conditional logic
for specific customers. Generated agreements from New York customers display the New
York state legal clause.
