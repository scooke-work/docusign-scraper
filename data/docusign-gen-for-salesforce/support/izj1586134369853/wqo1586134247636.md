# I Cannot Use the Gen Template Create Button Page to Remove a Button

Learn to remove Gen buttons from layouts.

I created a Gen template and want to remove the button from one of my layouts. I open the
template and go to the Create Button page. I do not see an option to remove the button.

## Resolution 1: Use the Gen Create Button Page

Cause: Docusign does not display a remove button
in the Gen template user interface.

Use this procedure to fix the issue:

1. [Access the Gen template Create Button page
   as a Salesforce administrator](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=way1621663319299.html&_LANG=enus).
2. Uncheck the box for each layout that you want to update. Uncheck all boxes if you
   want to remove the button from all layouts listed.
3. Select Update Layouts. ![A screenshot displays all page layout boxes unchecked with the Update Layouts button highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-displays-all-page-layout-boxes-unchecked-with-the-Update-Layouts-button-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg0NTYsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.JGYXy3cfauGMfF2DM-SHAtQq1Y_s_NXyIfwHMH_OFEs&_LANG=enus "A screenshot displays all page layout boxes unchecked with the Update Layouts button highlighted.")

   A message displays: Successfully modified
   layouts.
4. Select Save and Finish.

   You have completed the process. You have removed the Gen button from the
   layouts.

## Resolution 2: Remove Your Gen Button in Salesforce Setup

Cause: I added the Gen button to more layouts than the Create Button
step displays.

Use this procedure to fix the issue:

1. Open the Buttons, Links, and Actions page as an
   administrator:
   1. Select the gear ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg0NTYsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.JGYXy3cfauGMfF2DM-SHAtQq1Y_s_NXyIfwHMH_OFEs&_LANG=enus) and choose
      Setup.
   2. Enter and select Object Manager in the Quick Find field.
   3. Select a Salesforce object.
   4. Select Buttons, Links, and Actions.

   The Buttons, Links, and Actions page displays.
2. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg0NTYsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.JGYXy3cfauGMfF2DM-SHAtQq1Y_s_NXyIfwHMH_OFEs&_LANG=enus)a next to the custom button and
   choose Delete from the dropdown menu.![A screenshot displays the Buttons, Links, and Actions page with two custom button links.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/gendeletecustombutton.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg0NTYsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.JGYXy3cfauGMfF2DM-SHAtQq1Y_s_NXyIfwHMH_OFEs&_LANG=enus "A screenshot displays the Buttons, Links, and Actions page with two custom button links.")

   A dialog box displays: Are you sure?.
3. Select Delete. The page refreshes with the button no longer
   displayed.

   Note: Salesforce may display an
   error: Your request to delete this custom button or link can not be
   completed because this button or link is being used by another
   feature. Remove the button from all Salesforce layouts.

   The process is complete. You have removed the Gen button from your
   layout.
