# Step 2: Create Docusign and Salesforce Anchor Text Fields

Learn to create Docusign and Salesforce anchor text fields for your Gen
template.

Use this procedure to create Docusign and Salesforce
anchor text fields for your Gen template. You can add child objects from up to four levels of
relation to the parent Salesforce object. You can add fields from up to two levels of related
lists.

Tip:
[This video explains the step](https://bcove.video/3e1VOM3).

1. Select + Add in the Salesforce Fields section.
2. Select a Salesforce field from the Select an Option dropdown
   menu.

   ![A screen capture displays adding Salesforce fields to the Anchor Text section of the Gen Add Data Fields step.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Gen-Add-Data-Fields.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYzMzYsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9._ch8D1uCUOSj0i8lLTBp9RiLxEKv8qQQlIo0QVU18W0&_LANG=enus)
3. (Optional): Customize the Date and Currency Format settings:

   Important: To use Date and Currency
   Format, you must set [currency
   fields](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=any1686940255489.html&_LANG=enus) and [date fields](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=gyv1686944628002.html&_LANG=enus) to Use Template Format.

   1. Locale: Select a locale from the dropdown menu:

      - Generator's Locale: Use the generating user's Salesforce
        locale for date formatting
      - Org Locale: Use the Salesforce organization's locale for
        date formatting
      - All Locales: Select a country or region to set date
        formatting
   2. Currency Format: Select a currency format from the dropdown
      menu:

      - Display as symbol: Example:
        $123,456.78
      - Display as symbol (no decimals): Example:
        $123,457
      - Display as ISO code: Example: USD
        123,456.78
      - Display as ISO code (no decimals): Example:
        USD 123,457
      - Display without symbol and ISO code: Example:
        123,456.78
   3. Generator Can Edit: This check box lets users choose the
      date and currency format before generating documents.
4. Select + Add Signer at the bottom of the page.
5. Signature: Select a Docusign field from the
   dropdown menu.
6. (Optional): Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYzMzYsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9._ch8D1uCUOSj0i8lLTBp9RiLxEKv8qQQlIo0QVU18W0&_LANG=enus) to the far right of any tag:
   1. Remove: Delete a field
   2. Options: [Use conditional logic
      to show or hide text](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=nnl1586134232740.html&_LANG=enus) based on Salesforce record fields

You have completed step two. Move to [Step 3: Paste Anchor Text Fields Into Your Word Document](https://docusign-be-prod.zoominsoftware.io/bundle/izj1586134369853/page/ffk1586134297621.html "Learn to paste your Salesforce anchor text fields into a Microsoft Word document.").
