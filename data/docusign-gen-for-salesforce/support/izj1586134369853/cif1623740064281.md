# Step 3 (Optional): Remove Apps Launcher Custom Buttons from Your Layouts

Learn to remove Docusign Apps Launcher custom buttons from your
Salesforce layouts.

You can uninstall Apps Launcher without removing your Docusign custom
buttons. But consider removing these buttons to maintain a clean layout and avoid
confusing your users.

Use this procedure to remove all Apps Launcher custom buttons from your Salesforce
layouts:

1. Open the Buttons, Links, and Actions page as an
   administrator:
   1. Select the gear ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg1MjksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.E-PggmRD-RRjpgLsfJ-6O0CrWAeYzHK9fb9uIwqyERU&_LANG=enus) and select
      Setup.
   2. Enter and select Object Manager in the Quick
      Find field.
   3. Select an object (Account, Opportunity, and so on.)
   4. Select Buttons, Links, and Actions.

      The Buttons, Links, and Actions page displays.
2. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg1MjksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.E-PggmRD-RRjpgLsfJ-6O0CrWAeYzHK9fb9uIwqyERU&_LANG=enus) next to a Docusign button and choose
   Delete from the dropdown menu.![A screenshot displays the arrow menu with the delete button highlighted for a custom button.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/gendeletecustombutton.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg1MjksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.E-PggmRD-RRjpgLsfJ-6O0CrWAeYzHK9fb9uIwqyERU&_LANG=enus "A screenshot displays the arrow menu with the delete button highlighted for a custom button.")

   A dialog box displays: Are you sure?
3. Select Delete.

   The page refreshes with the button no longer displayed.

   Note: You may see:
   Your request to delete this custom button or link can not be
   completed because this button or link is being used by another
   feature. Open the object's page layout and remove the button
   there.
4. Repeat this entire procedure for each layout that contains a Docusign Apps Launcher button.

You have completed step three of the process. You have
removed all Docusign Apps Launcher custom buttons from your Salesforce layouts. Move to
[Step 4: Uninstall Apps Launcher from Your organization](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=ump1623741000284.html&_LANG=enus "Learn to uninstall the Docusign Apps Launcher package from your Salesforce organization.").
