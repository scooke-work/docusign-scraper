# Gen Template Recommended Limits

Learn the best practices of uploading files and adding Salesforce fields when building a
Gen template.

There are no definitive limitations on the documents and lists you can add to Gen templates.
But your template can fail if you add large files, too many files, or too many
lists.

Your Gen templates will not fail if you stay within the following recommended limits:

| Gen template aspect | Limit |
| --- | --- |
| Recommended maximum Word document template size | 3MB |
| Recommended maximum size of a generated Word document | 30MB |
| Recommended maximum number of Word templates used to generate a document | 10 |
| Recommended maximum number of lists connected per object | 20 |
| Maximum levels of related list per field | 2 |
| Maximum levels of parent object per field | 4 |
| Maximum number of triggered invoices (Gen for Salesforce Billing) | 1,000 per hour |
