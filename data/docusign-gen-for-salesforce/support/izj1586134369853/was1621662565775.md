# Access the Gen Template Add Fields Step

Learn to access the Add Fields step in a Word-based Gen template.

Use this procedure to access the Add Fields step of a Word-based Gen template:

1. Edit the Gen template Add Fields page as a Salesforce
   administrator:
   1. Select the App Launcher grid ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/app-launcher-grid.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0ODUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.ijsqTzJH4QqLYKahWp8XK514iloWfNbxHk_VUjN9Dsg&_LANG=enus) and choose View
      All.
   2. Select Docusign Apps Launcher.
   3. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0ODUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.ijsqTzJH4QqLYKahWp8XK514iloWfNbxHk_VUjN9Dsg&_LANG=enus) next to a Gen template and choose
      Edit from the dropdown menu.

   The Gen template creation page opens.
2. Select the Add Fields arrow. The arrow displays no text
   until selected.

You have completed the process. You have accessed the
Add Fields step of a Gen template.
