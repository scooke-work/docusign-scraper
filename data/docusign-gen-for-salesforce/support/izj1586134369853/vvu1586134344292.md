# Use Match Conditional Logic in a Word-Based Gen Template

Learn to create "match" conditional logic to add or remove text when a Salesforce field
is present.

Use this procedure to create "match" conditional logic to a
template. Match logic adds or removes text when a Salesforce field is present.

1. Select the Add Fields arrow. The arrow has no text until you
   select it.
2. Select Copy![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY1ODksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.qL5vqNSzcsR9qDo7-2fVqsZkFL8DkdkpRINzoriSe3A&_LANG=enus)
    next to a field and choose
   Options. ![A screenshot displays the description field with the copy menu open to options and remove.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/genconditionaloptionsdropdown.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY1ODksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.qL5vqNSzcsR9qDo7-2fVqsZkFL8DkdkpRINzoriSe3A&_LANG=enus "A screenshot displays the description field with the copy menu open to options and remove.")

   The Table Options dialog box opens.
3. Create match logic:
   1. Select Include other content if the conditions set for this field are
      met.
   2. Select matches.
   3. Enter keywords to match. For example, use matches >  New Customer to only send a welcome letter to your new customers.![A screenshot displays the display options field with the values set to matches and new customer.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/genconditionaltagmatch.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY1ODksInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.qL5vqNSzcsR9qDo7-2fVqsZkFL8DkdkpRINzoriSe3A&_LANG=enus "A screenshot displays the display options field with the values set to matches and new customer.")
   4. Select Save.

   The Add Fields page opens.
4. Select Copy next to the anchor text tag you updated.
5. Add the conditional logic to your Gen template:
   1. Open the Word document that you are using in your Gen template.
   2. Paste the copied anchor text into the document.
   3. Add any custom text to the tag. If you use Welcome, new
      customer! as custom text, the following tag displays: `<#
      <Conditional Select="/Opportunity/Description" Match="New Customer" /> #>
      Welcome, new customer! <# <EndConditional/> #>`
6. Select Finish when you are done editing.

   Note: You can create more advanced
   conditional logic than match and does not match. See the Conditionals topics in [Gen Advanced Features](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=pui1586134216490.html&_LANG=enus "Learn to use advanced actions when configuring your Gen templates.") for more information.

You have completed the process. You added
match conditional logic to your Gen template.
