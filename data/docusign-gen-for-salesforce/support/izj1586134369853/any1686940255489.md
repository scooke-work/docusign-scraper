# Set the Default Currency Formatting of Gen Data Fields

Learn how to set the default formatting of currency data fields.

You can set the default formatting of currency data fields. You can decide how Gen formats the
currency of generated agreements by setting defaults for these data fields.

Use this procedure to set the default currency formatting for a data field:

1. Select Options from the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY1MDcsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.vg4I-jsYP404VKinws0KtNGqPmwP3F6bpBEvpVSMJsU&_LANG=enus) menu next to a
   currency field:

   ![A screenshot of the dropdown menu for a Gen data field displays the Options and Remove options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-of-the-dropdown-menu-for-a-Gen-data-field-displays-the-Options-and-Remove-options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY1MDcsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.vg4I-jsYP404VKinws0KtNGqPmwP3F6bpBEvpVSMJsU&_LANG=enus "A screenshot of the dropdown menu for a Gen data field displays the Options and Remove options.")

   The Add Fields Options dialog box opens.
2. Currency Format: Select a currency format from the
   dropdown menu:

   Important: Set the Currency Format to
   Use Template Format to use template-level [Gen date and currency
   formatting](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=kqf1686095608923.html&_LANG=enus).

   - Use Template Format: Use the Gen template
     Date and Currency Format settings
   - Display as symbol: Example:
     $123,456.78
   - Display as symbol (no decimals): Example:
     $123,457
   - Display as ISO code: Example: USD
     123,456.78
   - Display as ISO code (no decimals): Example:
     USD 123,457
   - Display without symbol and ISO code: Example:
     123,456.78
3. (Optional): [Add
   matches or does not match
   logic to the field](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=pwd1586134341196.html&_LANG=enus "Learn to use Salesforce record fields to trigger conditional logic that displays or hides content in your document.").
4. Select Save.
5. Select Save and Close.

You have completed the process. You have set the default
currency formatting of your Gen data field. Generated documents display the currency
formatting that you chose.
