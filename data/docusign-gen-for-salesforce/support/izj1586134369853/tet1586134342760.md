# Edit a Gen Button

Learn to edit the label or details of a Gen custom button.

Use this procedure to edit the label or details of a
Gen custom button:

1. Access the Buttons, Links, and Actions page as a Salesforce
   administrator:
   1. Select the gear ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/salesforce-gear_.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0NTUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.uTxc2He8Txd72UrKsD87fvwWNhCfo5xioxmvYNoec30&_LANG=enus) icon and choose Setup.
   2. Enter and select Object Manager in the Quick Find
      field.
   3. Select an object (Account, Opportunity, and so on) in the left navigation
      bar.
   4. Select Buttons, Links, and Actions.
2. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0NTUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.uTxc2He8Txd72UrKsD87fvwWNhCfo5xioxmvYNoec30&_LANG=enus) next to a custom button and choose Edit from the
   dropdown menu.

   ![A screenshot displays the Buttons, Links, and Actions page with two custom button links.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/gendeletecustombutton.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0NTUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.uTxc2He8Txd72UrKsD87fvwWNhCfo5xioxmvYNoec30&_LANG=enus "A screenshot displays the Buttons, Links, and Actions page with two custom button links.")
3. Edit Label, Name, or other fields in the
   custom button detail page and select Save.

   ![A screenshot displays the custom button detail page with all available fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/geneditcustombutton.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0NTUsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.uTxc2He8Txd72UrKsD87fvwWNhCfo5xioxmvYNoec30&_LANG=enus "A screenshot displays the custom button detail page with all available fields.")

You have completed the process. You have edited the
custom button.
