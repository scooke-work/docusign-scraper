# Learn About Setting the Default Format of Currency, Date, and Time Data Fields

Learn about the default formatting that you can apply to data fields in a Gen
template.

When you build a Gen template, you can set the default formatting of currency, date,
and time data fields. You can set the default format of these data field types:

- Date fields: Set the date formatting of all
  documents generated from this template
- Currency fields: Set the currency formatting of
  all documents generated from this template
- Time fields: Set the time formatting of all
  documents generated from this template

Important: Set currency and date formats to Use
Template Format to use template-level [Gen date and currency formatting](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=kqf1686095608923.html&_LANG=enus).

You can use the following settings to format the currency, date, and time fields that display
in generated documents:

## Currency settings

| Setting Name | Example |
| --- | --- |
| Use Template Format | Use the Date and Currency Format settings of the Gen template |
| Display as symbol | $123,456.78 |
| Display as symbol (no decimals) | $123,457 |
| Display as ISO code | 123,456.78 |
| Display as ISO code (no decimals) | USD 123,457 |
| Display without symbol and ISO code | 123,456.78 |

## Locale (Date) Settings

| Setting Name | Description |
| --- | --- |
| Use Template Format | Use the Date and Currency Format settings of the Gen template |
| Generator's Locale | Use the generating user's Salesforce locale for date formatting |
| Locale Default | Use the Salesforce organization's locale for date formatting |
| Org Locale | Use the Salesforce organization's locale for date formatting |
| All Locales | Select a country or region to set date formatting |
| Custom | Enter a custom date format |

## Time settings

| Setting type | Example |
| --- | --- |
| Locale Default | Use the formatting from the default locale |
| 12-hour formats | 2:59 |
|  | 2:59 pm |
|  | 2:59:23 |
|  | 2:59:23 pm |
| 24-hour formats | 14:59 |
|  | 14:59:23 |
| Custom | Enter a custom time format |
