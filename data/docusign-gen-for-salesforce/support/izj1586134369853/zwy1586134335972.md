# Customize a Standard Table in an Agreement Cloud Editor Template

Learn to customize content in a standard Agreement Cloud Editor table.

You can customize content in a standard Agreement Cloud Editor table. You can use the
following actions:

- Change font size and style
- Add rows and columns
- Add fill colors to cells
- Delete rows, columns, or cells
- Merge data in multiple cells

When you create a new table with the Agreement Cloud Editor, the table toolbar displays
below your table. The merge cells button displays when you select multiple cells in the
table.![A screenshot displays the Agreement Cloud Editor table toolbar, containing 7 icons.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/genacetabletoolbar.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzNTYsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.DCH8rm__QwklrGd4lUm3uvMz6rGUrBIMvGAP1tF7c-g&_LANG=enus)
