# Use Conditional Logic in a Word-Based Gen Template

Learn to use Salesforce record fields to trigger conditional logic that displays or
hides content in your document.

You can use conditional logic in Gen data fields to display or hide language in your
generated document.

Example: You add conditional logic on a Billing
Address field to display specific legal language to California customers. You
generate a document from a record with a California address. The generated document contains
a special legal clause for California customers.

Use this procedure to create conditional logic for a Gen data field:

1. Select Options from the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0OTgsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.kyAth5C8RB6p8izQeywlfCp-nR1e47Lohdm0pKzcSp0&_LANG=enus) dropdown menu.

   ![A screenshot of the dropdown menu for a Gen data field displays the Options and Remove options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-of-the-dropdown-menu-for-a-Gen-data-field-displays-the-Options-and-Remove-options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0OTgsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.kyAth5C8RB6p8izQeywlfCp-nR1e47Lohdm0pKzcSp0&_LANG=enus "A screenshot of the dropdown menu for a Gen data field displays the Options and Remove options.")
2. Select Include other content if the conditions set for this field are
   met.
3. Choose an option from the dropdown menu:

   - matches
   - does not match
4. Enter keywords to match or not match in the text field.

   Note: See the following topic for more
   information about [match/not match
   logic](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=nnl1586134232740.html&_LANG=enus "Learn about adding conditional logic blocks to Gen template Word documents.").
5. Select Save. ![A screenshot displays the Data Field Options dialog box with the Include other content option selected.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-displays-the-Data-Field-Options-dialog-box-with-the-Include-other-content-option-selected.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0OTgsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.kyAth5C8RB6p8izQeywlfCp-nR1e47Lohdm0pKzcSp0&_LANG=enus "A screenshot displays the Data Field Options dialog box with the Include other content option selected.")
6. Select Save and Close.

You have completed the process. You have added
conditional logic to your Gen data field. Text is displayed or hidden based on keywords in the
record.
