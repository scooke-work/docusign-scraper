# Use SOQL Code to Filter and Sort Gen Data

Learn to add Salesforce Object Query Language (SOQL) code to filter and sort date in Gen
templates.

Salesforce Object Query Language (SOQL) searches your organization’s Salesforce data for
specific information. You can [insert Salesforce
SOQL code into the TableRow fields of Gen templates](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=mrr1588510303522.html&_LANG=enus "Learn about inserting Salesforce Object Query Language (SOQL) code into Gen templates.").

Using SOQL code in tables creates custom rules for filtering and sorting data. You
can only use sorting and filtering with child relationship fields.

Use this procedure to add SOQL code to Gen templates:

1. [Edit the Gen template Add Fields
   page as a Salesforce administrator](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=was1621662565775.html&_LANG=enus):
2. Select the Copy

   next to a child object and choose Options from the

   The Table Options dialog box opens.
3. Enter the Salesforce SOQL code for any of these options:
4. Select Save.

   The Table Options dialog box closes.
5. Select Finish.

   A dialog box displays: Do you want to save

6. Select Save.

You have completed the process. You have inserted Salesforce code
to sort or filter child object data in your Gen table.
