# Delete Salesforce Fields from a Word-Based Gen Template

Learn to remove data fields from a Word-based Gen template.

Use this procedure to remove fields from a Word-based
Gen template:

1. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0OTAsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.Kb3auAyI3EHPWWs62sV5Dc1NP55jKc--JPlzeGn9ox4&_LANG=enus) next to
   Copy.
2. Select Remove from the dropdown menu.![A screenshot displays the Field and Anchor Text fields with the copy, options, and remove dropdown menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/A-screenshot-displays-an-anchor-text-field-with-the-copy-menu-open-to-options-and-removepng.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY0OTAsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.Kb3auAyI3EHPWWs62sV5Dc1NP55jKc--JPlzeGn9ox4&_LANG=enus "A screenshot displays the Field and Anchor Text fields with the copy, options, and remove dropdown menu.")
3. Select Next.

   Important: Remove all
   deleted tags from this template's Word document and re-upload the document.

You have completed the first step of the process. You
have deleted the tags. [Remove the tags from this template's
Word document](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=sqp1586134339354.html&_LANG=enus "Learn to update your Gen template Word documents.")
