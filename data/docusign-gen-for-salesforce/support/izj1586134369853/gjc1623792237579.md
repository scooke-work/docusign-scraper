# Add Users to the Docusign Login Permission Set

Learn to use the Docusign Login permission set to give Docusign access to your Salesforce users.

The Docusign Login permission set helps you manage Docusign access for your Salesforce users.

Important:

- The Docusign Login permission set is available in Docusign Apps Launcher v3.5 and higher.
- When you add an existing Docusign user to this permission set, Docusign imports the user's permissions.
- Administrators can use the Docusign Login permission set with their
  single sign-on (SSO) identity provider and Docusign
  just-in-time provisioning. Just-in-time provisioning lets users create their
  own Docusign accounts in Salesforce by entering their SSO
  credentials.

Use this procedure to add users to the Docusign Login permission set:

1. Select Docusign Login.
2. Select Manage Assignments.
3. Select Add Assignments.
4. Select the check box next to each user that you want to add.
5. Select Assign.
6. Select Done.

You have completed the process. You have added one or
more users to the Docusign Login permission set.
