# Access the Gen Word Template Create Button Step

Learn to access the Create Button step of a Microsoft Word-based Gen template.

Follow this procedure to access the Create Button step of a Microsoft Word-based Gen
template:

Edit a Gen template as a Salesforce administrator:

1. Select the App Launcher grid ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/app-launcher-grid.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgyNTcsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.XDT7lsl2Rik-_BVb3aOg7E0MHK0QUtnU1uNx8P86tqA&_LANG=enus) and select View
   All from the dropdown menu.
2. Select Docusign Apps Launcher.
3. Select Docusign Gen Templates.
4. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgyNTcsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.XDT7lsl2Rik-_BVb3aOg7E0MHK0QUtnU1uNx8P86tqA&_LANG=enus) next to a Gen template and choose
   Edit from the dropdown menu. The Gen template
   creation page opens.
5. Select the Create Button arrow. The arrow has no
   text until selected.

You have completed the process. You have opened the
Create Button step of a Gen template.
