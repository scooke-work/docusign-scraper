# Download Agreement Cloud Editor Documents as Word Files

Learn how to download Agreement Document Format Gen contracts as Microsoft Word
documents.

[Agreement Cloud
Editor](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=bcp1586134298132.html&_LANG=enus "Learn to create a Gen template with the Docusign Agreement Cloud Editor.") templates create documents in ADF (Agreement Document Format.) ADF is
an HTML-based document type that you can create and edit on a Salesforce page.

You
cannot send ADF Gen documents for internal approval or external review from a
browser. To send ADF documents for review or approval, you can first download them
as Microsoft Word documents.

Use this procedure to download ADF Gen contracts
as Word documents:

1. Select an Agreement Cloud Editor Gen button on a Salesforce record.

   The generated document displays in the browser.
2. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzMzMsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.p3t6nI4TFCk5lsIjWxYjiVPuHQGC3GfANu7gku4VQF0&_LANG=enus) next to the Send for
   Signature button.
3. Select Download as Word File.![A screenshot displays the Download as Word File option under the Send for Signature menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/izj1586134369853/page/images/gendownloadasword.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTgzMzMsInNoZWFmIjoiaXpqMTU4NjEzNDM2OTg1MyJ9.p3t6nI4TFCk5lsIjWxYjiVPuHQGC3GfANu7gku4VQF0&_LANG=enus "A screenshot displays the Download as Word File option under the Send for Signature menu.")

   Docusign saves a copy of the document to your device
   in Microsoft Word format.

You have completed the process. You have downloaded a Microsoft
Word copy of the ADF contract. You can send this for internal approval, external review,
or use it to compare changes with a previous version.
