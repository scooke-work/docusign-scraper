# Rename a File in the Docusign Files Component

Learn how to rename a file in the Docusign Files component of a
Salesforce record.

Use this procedure to rename a document in the Docusign Files section
of a Salesforce record:

1. Select Rename from the dropdown menu.

   The Rename Document dialog box opens.
2. Enter a new file name in the Document Name field.
3. Select Rename.

   A message displays: (document name) has been renamed. The
   Salesforce record page displays. The document has a New Version
   status button below it.

You have completed the process. You have renamed the
file.
