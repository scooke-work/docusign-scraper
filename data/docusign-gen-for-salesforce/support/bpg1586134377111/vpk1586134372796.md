# Copy a File from Docusign Files to Notes & Attachments

Learn how to copy documents from Docusign Files to the Notes &
Attachments section of a Salesforce record.

Use this procedure to copy documents from Docusign Files to the
Notes & Attachments section of a Salesforce record.

Select Copy to Notes & Attachments from the dropdown
menu.

A dialog box displays: Document export in progress. Please check Notes
and Attachments section after some time. The dialog box closes. Another
dialog box displays: Document has been exported
successfully.

You have completed the process. You have copied a file
from Docusign Files to Salesforce Notes &
Attachments.
