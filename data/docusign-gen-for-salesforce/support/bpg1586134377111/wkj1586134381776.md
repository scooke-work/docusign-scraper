# Use Send for Approval To Send Docusign Files to Users in Your Organization

Learn how to send a document for internal approval from the Docusign Files section of a
Salesforce record.

Use this procedure to send a document for internal approval from the Docusign
Files section of a Salesforce record:

Note:

- Only customers of both Docusign Gen and Docusign Negotiate can send documents for
  internal approval
- You can only send internal approval requests to Salesforce users
- Internal approval recipients must have active Negotiator
  permissions to act on your request
- You can not send internal approval requests to Salesforce contacts or external email
  addresses

1. Select Internal Approval from the drop-down menu.

   The Select Approvers dialog box opens.
2. Complete the following in the Select Approvers dialog box:

   1. Add Approvers: Enter the name of any Salesforce user from
      your organization.
   2. (Optional) + Add Another Approver: Select this button to add
      more approvers.
   3. (Optional) Set Approval Order:
      1. Select the check box if you have more than one approver.
      2. Drag users up or down to create a new approval order.
         ![Change approval order](https://docusign-be-prod.zoominsoftware.io/api/bundle/bpg1586134377111/page/images/Change-approval-order.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYxMDMsInNoZWFmIjoiYnBnMTU4NjEzNDM3NzExMSJ9.hJRGVi-IKAhttlq-soKf_rH3ysMtsWapa52aS_g_pz0&_LANG=enus "Change approval order")
   4. Select the Next button.

   The Personalize Your Message dialog box opens.
3. Complete the following in the Personalize Your Message dialog box:

   1. Subject: Keep or change the text of the email subject.
   2. Message: Keep or change the text of the email message.
   3. Select the Send button.

   The Personalize Your Message dialog box closes. A banner
   opens: Your document has been sent for approval and copied to Docusign
   Files. The file appears in the Docusign Files section with a
   Pending Approval status.

You have completed the process. You have sent the file
to a user from your Salesforce organization. This user can view, download, and edit this
document.
