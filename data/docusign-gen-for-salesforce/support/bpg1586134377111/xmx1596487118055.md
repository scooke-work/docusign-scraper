# Step 3: Add Docusign and Salesforce Fields and Send Your Envelope (Send With Docusign Button)

The third step of the process involves placing Docusign and Salesforce fields into the
document text.

Follow this procedure to place Docusign and Salesforce fields
into the document text.

1. Add Docusign fields to the document:

   1. Select a Docusign field from the ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/bpg1586134377111/page/images/envelopetemplatestandardsfields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYxMTMsInNoZWFmIjoiYnBnMTU4NjEzNDM3NzExMSJ9.C7L17Mbq5_0yDe3pxz6Fl4wJGN-u88bJ76_g4q1gpj4&_LANG=enus)
      Standard Fields menu in the left column.
   2. Drag the field to the document.![Add Docusign fields](https://docusign-be-prod.zoominsoftware.io/api/bundle/bpg1586134377111/page/images/Add-DocuSign-fields.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYxMTMsInNoZWFmIjoiYnBnMTU4NjEzNDM3NzExMSJ9.C7L17Mbq5_0yDe3pxz6Fl4wJGN-u88bJ76_g4q1gpj4&_LANG=enus "Add Docusign fields")
2. Add Salesforce fields to the document:

   1. Select the Merge Fields button ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/bpg1586134377111/page/images/envelopetemplatemergefields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYxMTMsInNoZWFmIjoiYnBnMTU4NjEzNDM3NzExMSJ9.C7L17Mbq5_0yDe3pxz6Fl4wJGN-u88bJ76_g4q1gpj4&_LANG=enus) from the left
      column. The Salesforce Merge Fields menu opens.
   2. Select a field from the menu. You can expand arrows to find more fields, or enter text in
      the Search Fields field.
   3. Drag the field to the document text.![Add Salesforce fields](https://docusign-be-prod.zoominsoftware.io/api/bundle/bpg1586134377111/page/images/Add-Salesforce-fields.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYxMTMsInNoZWFmIjoiYnBnMTU4NjEzNDM3NzExMSJ9.C7L17Mbq5_0yDe3pxz6Fl4wJGN-u88bJ76_g4q1gpj4&_LANG=enus "Add Salesforce fields")
3. (Optional) Choose any option:

   - Preview: Select this button to see a preview of
     the document.
   - Add fields for more recipients:
     1. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/bpg1586134377111/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYxMTMsInNoZWFmIjoiYnBnMTU4NjEzNDM3NzExMSJ9.C7L17Mbq5_0yDe3pxz6Fl4wJGN-u88bJ76_g4q1gpj4&_LANG=enus) next to the recipient name in the
        upper-left of the page.
     2. Select another recipient from the drop-down menu.
     3. Drag Docusign fields for this recipient to the document
        text.

        Note: Each recipient's fields are
        represented by a separate color (yellow, blue, and so
        on.)![Switch recipients](https://docusign-be-prod.zoominsoftware.io/api/bundle/bpg1586134377111/page/images/Switch-recipients.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYxMTMsInNoZWFmIjoiYnBnMTU4NjEzNDM3NzExMSJ9.C7L17Mbq5_0yDe3pxz6Fl4wJGN-u88bJ76_g4q1gpj4&_LANG=enus "Switch recipients")
4. Select the Send button in the upper-right of the
   page.

You have completed the process. You have built and sent an
envelope.
