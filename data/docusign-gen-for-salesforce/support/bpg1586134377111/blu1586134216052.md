# Generate a Document and Add it to Docusign Files

Learn how to add files to the Docusign Files component after
generating them with Gen.

## Before you begin

Your administrator must grant Docusign
Generator and Negotiator permissions to your Docusign user.

When you generate a new document with Gen, it is automatically saved in the Notes &
Attachments section of the record. You can add the document to the Docusign
Files component to enable these options for Docusign users:

- Preview
- Download
- Send for signature
- Send for internal approval
- Send for external review

Use this procedure to generate files and add them to the Docusign
Files component:

1. Log into Salesforce as a user with Docusign
   [Generator and Negotiator](https://support.docusign.com/s/document-item?bundleId=bpg1586134377111&topicId=ymi1586134213877.html&_LANG=enus "Learn how to activate your Docusign CLM, Gen, Negotiate, or eSignature for Salesforce user.") permissions.
2. Open a Salesforce record (Account, Opportunity, and so on.)
3. Select a Gen button in the upper-right of the page.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/bpg1586134377111/page/images/Select-Gen-custom-button.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYwNDIsInNoZWFmIjoiYnBnMTU4NjEzNDM3NzExMSJ9.PY-C0oKHUnLxZ7kbZ-mhgVnlIxlhvkKnWZWo3mF3oSE&_LANG=enus)

   Note: The button may be hidden under
   the arrow drop-down menu.

   The New Documents Generated and Saved page opens. The generated
   document is saved in the Notes & Attachments section of this record.
4. Select the Done button.

   The New Documents Generated and Saved page closes. The record page
   opens.
5. Select the Import button in the Docusign
   Files section of the page.

   The Import Files dialog box opens.
6. Select Choose from Salesforce Files. Complete the following
   steps in the Select a File dialog box that opens:
   1. Select the check the box next to a document. You can only import one file at a
      time.
   2. Select Import.

      The New Document Added dialog box opens.
7. Select Done.

   The New Document Added dialog box closes. The Docusign Files
   section refreshes and the new document displays.

You have completed the process. You have generated a new
document and uploaded it to the Docusign Files component. Users can view,
download, send for signature, send for external review, or send for internal approval.
