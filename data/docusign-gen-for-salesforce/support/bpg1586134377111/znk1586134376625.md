# Delete a File from the Docusign Files Component

Learn how to delete a file in the Docusign Files component of a
Salesforce record.

Use this procedure to delete a file in the Docusign Files component of
a Salesforce record:

1. Select Delete from the dropdown menu.

   The Delete File dialog box opens.
2. Select Delete.

   A message displays: (document name) has been deleted. The
   Salesforce record page displays. The document is no longer in the Docusign Files section.

You have completed the process. You have deleted the
file.
