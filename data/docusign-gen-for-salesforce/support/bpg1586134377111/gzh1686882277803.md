# Add New Currency Types to Your Salesforce Organization

Learn to add new currency types to your Salesforce organization.

You can add new currency types to your Salesforce organization. Adding currencies lets users
select different currency types when generating documents with Docusign Gen.

Example: Your Salesforce organization uses U.S.
Dollar as the default currency. Your users want to generate
documents with Euro as the currency. You add
Euro to your organization's currency types. Your Gen
users set the currency to Euro, and Docusign generates contracts with Euro as the
currency.

Note: You must add at least one currency to your organization to use
template-level [Gen date and
currency formatting](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=kqf1686095608923.html&_LANG=enus).

Use this procedure to add a new currency to your Salesforce organization:

1. Edit the Company Information page as a Salesforce
   administrator:
   1. Select the gear ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/bpg1586134377111/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYwODQsInNoZWFmIjoiYnBnMTU4NjEzNDM3NzExMSJ9.cnJzHmD1blq3v_tTIbNrNwBXUNx0718y5qm9gXJyQ0c&_LANG=enus) icon and
      choose Setup.
   2. Enter and select Company Information in the
      Quick Find field.
   3. Select Edit.
2. Select the Activate Multiple Currencies check box and
   select Save.

   ![A screenshot displays the Activate Multiple Currencies box selected on the Salesforce Company Information page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/bpg1586134377111/page/images/A-screenshot-displays-the-Activate-Multiple-Currencies-box-selected-on-the-Salesforce-Company-Information-page.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYwODQsInNoZWFmIjoiYnBnMTU4NjEzNDM3NzExMSJ9.cnJzHmD1blq3v_tTIbNrNwBXUNx0718y5qm9gXJyQ0c&_LANG=enus "A screenshot displays the Activate Multiple Currencies box selected on the Salesforce Company Information page.")

   The Company Information page opens.
3. Select Currency Setup and select
   New.

   The New Currency page opens.
4. Add a new currency:
   1. Currency Type: Select a new currency from the
      dropdown menu.
   2. Conversion Rate: Enter the conversion rate from
      your default currency to this currency. Example:
      .092.
   3. Decimal Places: Enter the number of digits
      displayed to the right of the decimal point. This setting overrides the
      decimal places setting on all currency fields for users with this
      currency. You can choose between 0 and 6 digits.
   4. Select Save.

You have completed the process. Users can change
the Currency field on a record to the new currency. Gen users can
select the new currency before generating documents.
