# Set the Currency and Locale Before Generating a Document

Learn to set the currency and locale of a document before you generate it.

## Before you begin

- Your administrator must grant Docusign
  Generator
  permissions to your user
- Your administrator must enable the Generator Can Edit
  feature on the Gen template
- You must [set the default currency you want to
  use on the Salesforce record](https://support.docusign.com/s/document-item?bundleId=bpg1586134377111&topicId=xrh1686885796196.html&_LANG=enus "Learn to change the type of currency Salesforce displays on a record.")

You can set the currency and locale of a document before generating it from a record.
The following happens when you select a Gen button with Generator Can
Edit enabled:

- You set the date and currency formatting you want the generated document to
  display
- Gen creates a document using the formatting you selected
- Gen saves the generated document in Notes &
  Attachments

You can view or download the generated document or send it for signature.

Use this procedure to set the currency and locale of a document before generating it
from a record:

1. Select a Gen button on a Salesforce record.

   ![A screenshot of a Salesforce record displays a Gen custom button named Fresh Software Sales Contract under the arrow menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/bpg1586134377111/page/images/Select-Gen-custom-button.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYwMjksInNoZWFmIjoiYnBnMTU4NjEzNDM3NzExMSJ9.0Lu9RXrZOSh2WdDtWSwwC-sRvTzqKu7JYX-CqQxTOjg&_LANG=enus "A screenshot of a Salesforce record displays a Gen custom button named Fresh Software Sales Contract under the arrow menu.")

   The Date and Currency Format page opens.
2. (Optional): Customize the Date and Currency Format
   settings:
   1. Locale: Select a locale option from the dropdown
      menu:

      - Generator's Locale: Use the generating
        user's Salesforce locale for date formatting
      - Org Locale: Use the Salesforce
        organization's locale for date formatting
      - All Locales: Select a country or region
        to set date formatting
   2. Currency Format: Select a currency format from
      the dropdown menu:

      - **Display as symbol**: Example: **$123,456.78**
      - **Display as symbol (no decimals)**: Example:
        **$123,457**
      - **Display as ISO code**: Example: **USD 123,456.78**
      - **Display as ISO code (no decimals)**: Example: **USD
        123,457**
      - **Display without symbol and ISO code**: Example:
        **123,456.78**
3. Select Generate.

   The New Documents Generated & Saved dialog box
   opens. Salesforce saves the generated document in Notes &
   Attachments.
4. (Optional): Take action on the generated document:

   Note: Your administrator can disable the Send
   with Docusign button.

   - Preview: See a preview of the generated document
   - Download: Save a copy of the document to your
     device
   - [Send for Signature](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=gsh1675832324336.html&_LANG=enus): Send to a recipient for signature
   - Done: Take no action and close the dialog
     box

You have completed the process. You have generated
a new contract and taken any optional steps. Salesforce saves the generated document in
the Notes & Attachments section of the record.
