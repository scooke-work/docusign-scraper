# Step 1: Add Documents or Templates (Send With DocuSign Button)

Use this procedure to add documents or templates to a standard Send with DocuSign
envelope.

Use this procedure to add documents or templates to a
standard Send with DocuSign envelope.

1. Open a Salesforce record (Account, Opportunity, and so on.)
2. Select the Send with DocuSign button in the upper-right
   of the page.

   Note: It may be hidden in the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/bpg1586134377111/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYxMDgsInNoZWFmIjoiYnBnMTU4NjEzNDM3NzExMSJ9.K-Dz6zHBuVtFwPELnCIzdOdzMFJ84LPPXORrflV6woo&_LANG=enus) menu. ![Send with DocuSign button](https://docusign-be-prod.zoominsoftware.io/api/bundle/bpg1586134377111/page/images/Send-with-DocuSign-button.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYxMDgsInNoZWFmIjoiYnBnMTU4NjEzNDM3NzExMSJ9.K-Dz6zHBuVtFwPELnCIzdOdzMFJ84LPPXORrflV6woo&_LANG=enus "Send with DocuSign button")

   The Documents page opens, displaying all documents
   from the Notes & Attachments section of the record.
3. (Optional) Choose any option to upload or arrange documents for your
   envelope:

   - Select the check box next to any Notes & Attachments documents you
     want to include
   - Upload: Select this button and follow the prompts
     to add a document from your device
   - Add from Salesforce: Select this button to add
     documents from Salesforce Files
   - Drag documents to change their order in the envelope![Drag files to reorder](https://docusign-be-prod.zoominsoftware.io/api/bundle/bpg1586134377111/page/images/Drag-files-to-reorder.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYxMDgsInNoZWFmIjoiYnBnMTU4NjEzNDM3NzExMSJ9.K-Dz6zHBuVtFwPELnCIzdOdzMFJ84LPPXORrflV6woo&_LANG=enus "Drag files to reorder")
   - Select Template: Apply an existing DocuSign
     template to your envelope:
     1. Select the Select Template button. The
        Select DocuSign Template dialog box
        opens.This displays every template from your DocuSign account
        and all of your envelope templates.
     2. Select the check box next to any templates you want to use in
        this envelope.
     3. Select the Select button.![Select existing DocuSign templates](https://docusign-be-prod.zoominsoftware.io/api/bundle/bpg1586134377111/page/images/Select-existing-DocuSign-templates.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYxMDgsInNoZWFmIjoiYnBnMTU4NjEzNDM3NzExMSJ9.K-Dz6zHBuVtFwPELnCIzdOdzMFJ84LPPXORrflV6woo&_LANG=enus "Select existing DocuSign templates")The Select DocuSign Template dialog
        box closes. All documents, recipients and signature fields from
        the selected templates are added to your envelope.
4. Select the Next button. The Recipients page opens.

You have completed step one. Move to [Step 2: Add Recipients to Your Envelope (Envelope Template)](https://support.docusign.com/s/document-item?bundleId=bpg1586134377111&topicId=dfp1596477842847.html&_LANG=enus "Use this procedure to add recipients to the envelope.
