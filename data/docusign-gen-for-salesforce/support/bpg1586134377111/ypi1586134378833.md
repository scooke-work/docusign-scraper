# Create a Share Link to Send Docusign Files for Viewing, Downloading, and Editing

Learn how to send a file from Docusign Files using a share link. This link lets people
with access to your organization view, download, and edit your Docusign files.

Use this procedure to create and send a share link for a document in the Docusign
Files section of a Salesforce record:

Note: Only users with permission to access
your Docusign Agreements can use this link to view, download, and edit shared
content.

1. Select Share Link from the drop-down menu.

   The Share Link dialog box opens.
2. Select the Copy Link button.

   The Copy Link button turns into a check mark with the words Link
   Copied! underneath.
3. Select the X button above the upper-right of the Share Link
   dialog box.

   The Share Link dialog box closes.

You have completed the process. You can send the copied
link to anyone with access to your Salesforce organization. Users with this link can view,
download, and edit this document.
