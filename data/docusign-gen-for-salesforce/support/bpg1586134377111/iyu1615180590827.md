# Generate an Invoice with Gen for Billing and Email It to Recipients

You can generate an invoice from a custom button and email it to recipients.

## Before you begin

Your administrator must grant Docusign
Generator permissions to your Docusign user.

You can generate an invoice from a custom button and
email it to recipients.

Here's how to do that:

1. Find a Gen for Billing custom button:
   1. Log into Salesforce as a user with Docusign Gen
      Generator permissions.
   2. Open a Salesforce Invoice record.
   3. Select a Gen for Billing button on the page layout.

      Note: The button may be hidden under the arrow
      menu.![A screenshot displays an arrow menu opening to a button labeled Fresh Software Sales Contract.](https://docusign-be-prod.zoominsoftware.io/api/bundle/bpg1586134377111/page/images/Select-Gen-custom-button.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTYwNzcsInNoZWFmIjoiYnBnMTU4NjEzNDM3NzExMSJ9.i-RL99YGr4MW8yY7ZJFmmCyMDu9vareb5y3kU9bK1co&_LANG=enus "A screenshot displays an arrow menu opening to a button labeled Fresh Software Sales Contract.")

      The New Documents Generated and Saved page opens.
2. Select Deliver by Email.

   The Deliver By Email dialog box opens.
3. Select Add Recipient.

   The Add Recipient dialog box opens.
4. Add a recipient:
   1. Name: Enter a name for the recipient.
   2. Email: Enter an Email address for the
      recipient.
   3. (Optional) Set as Bcc: Check the box to
      hide this recipient.
   4. Add: Save the recipient and close the dialog
      box.
   5. Add & New: Save the recipient and add
      another.
5. Optional Steps:

   - Email Subject: Edit or replace text in the subject field.
   - Email Message: Edit or replace text in the message field.
   - Preview: Select this link to see a preview of the generated
     document.
   - Download: Save a copy of the contract to your device.
6. Select the Send button.

You have completed the process. You have generated an
invoice, added recipients, and sent an email.
