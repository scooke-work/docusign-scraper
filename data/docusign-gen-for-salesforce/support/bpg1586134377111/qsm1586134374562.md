# Upload a New Version of a Document

Learn how to upload a new version of a document in the Docusign Files
section of a Salesforce record.

Use this procedure to upload a new version of a document in the Docusign
Files section of a Salesforce record:

1. Select Upload a New Version from the dropdown menu.

   The Upload a New Version dialog box opens.
2. Select Upload and follow the prompts to upload a file from your
   device.

   The uploaded file displays in the dialog box.
3. Select Upload.

   Note:
   Docusign keeps
   the original file name, even if the uploaded file name is different.

   The dialog box closes. A message displays: A new version of (document
   name) has been uploaded. The Salesforce record page displays. The document
   has a New Version status button below it.

You have completed the process. You have uploaded a new
version of the document.
