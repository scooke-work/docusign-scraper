# Envelope Template Recipient Types

You can add many different recipient types to an envelope template. You can add
recipients from Salesforce, from Docusign, or manually enter them.

You can add many different recipient types to an envelope or envelope template. You can add
recipients from Salesforce, from Docusign, or manually enter them.

You can add the following recipient types:

Table 1. Recipient Types

| Recipient Type | Description |
| --- | --- |
| Lookup Fields | Choose Lookup Fields when you want to automate the adding of recipients. Example: If you choose **Owner**, the name and email address of a record owner is automatically added.  Only fields related to the template Salesforce data source (Account, Opportunity, and so on) will appear. |
| Related Lists | You can choose fields to relate to a Salesforce object. Example: Contacts related list for an Account, OpportunityContactRole for an Opportunity, and so on.  This recipient type dynamically adds recipients from a Related List. Only fields previously related to the template data source (Account, Opportunity, and so on) appear.  You can only add standard Salesforce related lists, like CaseContactRoles, OpportunityContactRoles, and so on.  Custom Related Lists do not appear in results. |
| Users/Contacts | You can add a specific user from your Salesforce Users or Contacts. This user is not added dynamically. |
| Sender | Add the sender of the envelope to the recipient list. This sender is the person who selects the envelope template custom button on a record.  To add a Sender recipient, you must select +Add Role and enter a role, like Sender 1. |
| By Role | Add a generic recipient (ex: Approver) as a placeholder, if you do not yet know the names or types of recipients to be added. The sender can then add a recipient name or type during the sending process.  You can also add a specific recipient's name and email here.  You can use custom syntax to dynamically add a By Role recipient from a Salesforce object. |
| **Signing Group** | Add a signing group from your organization's Docusign account. Any signer in the group can complete the signature process for that group. |
