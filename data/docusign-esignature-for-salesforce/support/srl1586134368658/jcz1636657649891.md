# Step 9: Add Your Template Button to the Salesforce Person Account Layout

The ninth step of the process involves adding an envelope template custom button to the
Person Account layout.

Use this procedure to add an envelope template custom button to the Person Account layout.

1. Open the Person Account Layout page as a Salesforce
   administrator:
   1. Select the gear ![A screenshot displays a six-point gear with a lightning bolt inside it.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY5MTYsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.-JJCXla0JG1GNIGxMyDtXIuikY21KVDUfIkK5d4BdwA&_LANG=enus "A screenshot displays a six-point gear with a lightning bolt inside it.") and choose Setup.
   2. Enter Object Manager in the Quick Find
      field.
   3. Select the Person Account link in the left
      column.
   4. Select the Page Layouts link in the left
      column.
   5. Select the Person Account Layout link.

   The Person Account Layout page opens.
2. Select the Mobile & Lightning Actions link in the
   left column of the Person Account Layout menu.
3. Drag the Person Account template button to the Salesforce Mobile and
   Lightning Experience Actions section.
4. Select Save.

   ![A screen capture displays the Salesforce Person Account Layout page with the Add Person Account custom button being dragged into the Salesforce Mobile and Lightning Experience Actions section.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Add-Person-Account-custom-button.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY5MTYsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.-JJCXla0JG1GNIGxMyDtXIuikY21KVDUfIkK5d4BdwA&_LANG=enus "A screen capture displays the Salesforce Person Account Layout page with the Add Person Account custom button being dragged into the Salesforce Mobile and Lightning Experience Actions section.")

   The Saving Layout message opens. The page refreshes.

You have completed the entire process. You have built a
template for the Person Account object, and added it to the Person Account page layout.
