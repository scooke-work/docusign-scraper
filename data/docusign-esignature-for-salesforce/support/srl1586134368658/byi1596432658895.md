# Change Envelope Template Recipient Actions

There are many different recipient actions that can be assigned to recipients in an
envelope template.

Use this procedure to change the action of an envelope template recipient:

1. Select from the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcwMTEsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9._UX6SOHKmNdr-UkVzVZosYxzjCncH8SeuL1xZScR9AE&_LANG=enus) dropdown menu next to a recipient
   name.
2. Select Edit.

   ![A screenshot displays an envelope template recipient with the Edit and Delete dropdown menu highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/envelopetemplateeditrecipient.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcwMTEsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9._UX6SOHKmNdr-UkVzVZosYxzjCncH8SeuL1xZScR9AE&_LANG=enus "A screenshot displays an envelope template recipient with the Edit and Delete dropdown menu highlighted.")
3. Select the Action dropdown menu.
4. Select an action from the list.
5. Select the Add button to save the action. The Add
   Recipient dialog box closes.
6. Select the Save and Close button at the top-right of the
   page.

You have completed the process. You have added or changed the recipient
action.
