# Rename a Document Uploaded to an Envelope Template

You can change the file names of uploaded documents in the envelope template by
following these steps.

You can change the file names of uploaded documents in the envelope template by following
these steps:

1. Select from the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY5ODYsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.l5hD3P9mf62Bu6wXJiDUmSNKp_wf0c_g4P5d-kn6rsk&_LANG=enus) dropdown menu next to a document.
2. Select Rename from the options.

   ![A screenshot displays the envelope template document options menu with the Rename option highlighted for an uploaded document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/envelopetemplatedocumentoptions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY5ODYsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.l5hD3P9mf62Bu6wXJiDUmSNKp_wf0c_g4P5d-kn6rsk&_LANG=enus "A screenshot displays the envelope template document options menu with the Rename option highlighted for an uploaded document.")

   The Rename Document dialog box opens.
3. Enter a new name for the document.
4. Select the Rename button.
5. Select Save and Close.

   You have completed the process. You have changed the name of the
   uploaded document.
