# Edit Server Template Options from an Envelope Template

Learn how to edit Docusign server template options by editing an
envelope template in your Salesforce organization.

Docusign server templates are signing experiences that you build in your Docusign account. You log on to your Docusign
account and select the Templates tab to build or access a
server template.

Docusign envelope templates are signing experiences that you build in your
Salesforce organization with the Docusign eSignature for
Salesforce application. You open Docusign Apps Launcher in your
Salesforce organization to build an envelope template.

Docusign offers some options in server templates that are not yet available
in envelope templates. However, you can add many of these options to envelope
templates from the server template interface.

Use this procedure to add, edit, or remove Docusign server template options
from the Docusign envelope template interface:

1. Docusign only displays the information icon for recipients
   with server template options added. ![A screenshot displays the information icon on the Recipients step of an envelope template.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Recipients-information-icon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY5NTgsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.swvJtoclQwPtxw39u7iMz0VGauioG8r6dy8HOUWu5Ps&_LANG=enus "A screenshot displays the information icon on the Recipients step of an envelope template.")

   The Add Recipient dialog box opens. A message displays some of the added
   server template features. The message does not display the following options,
   even if added:
   - Custom email and language for each recipient
   - Knowledge-based authentication (KBA)
   - Phone Call
   - Docusign ID Verification
   - Standards-based signature (SBS)
   - Don’t allow senders to edit the recipient
   - Don’t allow senders to delete the recipient
   - Draw a new signature for each signature or initial field
2. Select View Template.![A screenshot displays the View Template link in the Add Recipient dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/A-screenshot-displays-the-View-Template-link-on-the-Add-Recipient-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY5NTgsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.swvJtoclQwPtxw39u7iMz0VGauioG8r6dy8HOUWu5Ps&_LANG=enus "A screenshot displays the View Template link in the Add Recipient dialog box.")

   Your Docusign account opens to the template editing
   page.
3. [Add, edit, or remove supported server
   template options](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=nsh1649096506753.html&_LANG=enus "Learn how you can add options to an envelope template from a Docusign server template.").
4. Select SAVE AND CLOSE.

   A message displays: saving. The page refreshes.
5. Select the back arrow in your browser to return to Salesforce, and select the
   envelope template Recipients tab.
6. Select Save and Close.

   The page refreshes.

You have completed the process. You have added, edited, or deleted
server template options for a recipient. Envelopes sent from the server template and the
envelope template each contain all the options that you added.
