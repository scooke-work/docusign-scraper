# Envelope Template Field Categories and Functions

The Place Fields page in the envelope template process contains more than just standard
eSignature fields. You can add Salesforce fields, Notary fields, Seal fields, Smart Sections
or build custom fields.

The Place Fields page in the envelope template process contains
more than just standard eSignature fields. You can add Salesforce fields, Notary
fields, Seal fields, Smart Sections or build custom fields.

Here are all categories of fields that can be placed on documents:

Table 1. Field Categories and Functions

| Button | Field Type | Function |
| --- | --- | --- |
|  | Standard Fields | Add standard Docusign fields such as Signature, Initial, Date Signed, and more. |
|  | Custom Fields | Add custom to place on the agreement. |
|  | Merge Fields | Add fields that link to Salesforce data. |
|  | Notary Fields | Add fields for a signer and notary (in the same physical location) to complete signing together. |
|  | Pre-fill Tools | Add a text field, checkbox, radio button, and more. |
