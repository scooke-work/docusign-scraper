# Attach a Document From a Salesforce Record to an Envelope Template

Learn how to add Salesforce documents to an envelope template on the Documents
page.

You can add Salesforce documents to an envelope template on the
Documents page. This feature allows you to dynamically
attach documents from Salesforce Files to your envelope
template.

Use this procedure to attach documents from Salesforce Files to your envelope template

1. Select the Attach documents from Salesforce record check
   box.

   The Document Selection dropdown menu
   opens.
2. Choose an option from the Document Selection dropdown
   menu:
   - Latest Document: Uploads the latest document from the record
   - File Name Contains: Uploads documents from the record that match file name keywords you enter
   - All Documents: Uploads all documents from the record
3. Select Save and Close.

   You have completed the process. Docusign dynamically adds documents from Salesforce
   Files to your envelope template.
