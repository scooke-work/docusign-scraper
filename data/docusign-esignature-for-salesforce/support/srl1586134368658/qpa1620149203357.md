# Access the Documents Step of an Envelope Template

Learn to access the Documents step to add, edit, or delete envelope template
files.

You can add, edit, or delete envelope template files on the
Documents step.

Use this procedure to access the envelope template Documents step:

1. Access the envelope template Documents page as a
   Salesforce administrator:

   The envelope template creation page opens to the Documents
   step.
2. Select the App Launcher grid ![A screenshot of the Salesforce App Launcher grid, containing nine colored dots.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/app-launcher-grid.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY5NzEsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.iSqzMmhg-LwijJb99JYe0YJpQN0FOH7FS5rPpfwG3P0&_LANG=enus "A screenshot of the Salesforce App Launcher grid, containing nine colored dots.") and choose View All.
3. Select Docusign Apps Launcher.
4. Select the Docusign Envelope Templates tab.
5. Select the arrow ![A screenshot of a downward-facing triangle.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY5NzEsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.iSqzMmhg-LwijJb99JYe0YJpQN0FOH7FS5rPpfwG3P0&_LANG=enus "A screenshot of a downward-facing triangle.") next to a template and choose Edit.

You have completed the process. You have accessed the
Documents step of an envelope template.
