# Add a Standard Send with Docusign Button to a Salesforce Layout

Learn how to add a standard Send with Docusign button to a Salesforce
layout.

A Docusign envelope is a digital agreement that you build that contains
documents, recipients and signature fields. Recipients complete the fields assigned to them
to complete the signature process.

You can create and send envelopes from Salesforce records. Docusign can
add data from record fields to the envelope. When recipients enter or change Salesforce data
in the envelope, Docusign can update the data on the record.

When the signature process is complete, Docusign saves the signed
agreement to the Salesforce record.

When you add a Send with Docusign button to a layout, users can
send envelopes from records. When senders select this button, Docusign
opens a pre-built sending experience. Senders can add documents, recipients, Salesforce
fields, and Docusign fields to the envelope before sending.

Use this procedure to add a Send with button to a
Salesforce layout from the Docusign Setup page:

1. Open Docusign Apps Launcher as an
   administrator.
2. Go to eSignature >  Button.
3. Choose an object from the Select a Salesforce object dropdown
   menu.
4. Select the check box next to one or more layouts and select Update
