# Step 7: Create an Envelope Template Custom Button on a Salesforce Page Layout

The seventh step of the envelope template creation process involves adding a custom
button to the layout of the Salesforce object chosen in step one.

Use this procedure to add a custom button to the Salesforce page layout of the object chosen
in step one.

Tip: [This video](https://players.brightcove.net/6118377982001/default_default/index.html?videoId=6172186652001) walks you through the process.

1. Enter a descriptive name for the custom button in the Button
   Label text field.
2. Check the box for each page layout to which the custom button will be
   added.
3. Select Save and Finish.

   A message displays: Successfully modified
   layouts.

The entire process is complete. You have built a new envelope
template, and added a custom button to a Salesforce layout.
