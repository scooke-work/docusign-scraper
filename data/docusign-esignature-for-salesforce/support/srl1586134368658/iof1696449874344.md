# “This Account Lacks Sufficient Permissions to Use Merge Fields” Displays When I Add Envelope Template Merge Fields

Learn to fix the “This account lacks sufficient permissions to use Merge Fields” error
when adding envelope template merge fields.

When adding a merge field to an envelope template, an error displays: This account
lacks sufficient permissions to use Merge Fields.

Example: I start building a new envelope template. I add a merge field on
the Merge Fields page and select Save. An error displays:
This account lacks sufficient permissions to use Merge
Fields.
Docusign does not save the new merge field.

## Resolution: Contact Your Docusign Representative

Cause: Your Docusign account does not
include the Allow Merge Fields feature. Allow
Merge Fields is a paid feature and is not available on all
plans.

Use this procedure to fix the issue:

1. Contact your Docusign representative.
2. Request the Allow Merge Fields feature for your
   Docusign account.

   Note:
   - Allow Merge Fields is a paid feature
     and may incur additional charges.
   - Allow Merge Fields is not available on
     all plans. Web-only plans cannot gain access to this
     feature.

   You have completed the process. If your account allows the feature, you can add
   merge fields to envelope templates.
