# Customize eSignature for Salesforce on the Settings Page

Additional options are available for customizing the eSignature for Salesforce app in
the Docusign Setup tab.

Additional options are available for customizing the
eSignature for Salesforce app in the Docusign Setup page:

[Enable Built-In eSignature](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=kor1611642110382.html&_LANG=enus "Learn to enable Docusign as your eSignature app or disable it to use another app.")
:   When you enable this feature, users can send Docusign envelopes from
    records. Disable this feature if you want to use another app for your electronic
    signatures.

[Enable System Sender](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=jxz1611642321527.html&_LANG=enus "Learn to enable a System Sender so that non-Docusign users can send and sign envelopes.")
:   When you enable this feature, non-Docusign users can send and sign
    envelopes. You save a Docusign administrator's credentials as your
    System Sender. Docusign uses these credentials to send envelopes on
    behalf of non-Docusign users. The System Sender's username displays
    as the sender to all envelope recipients.

Important:

- The Enable System Sender option only works with the [Docusign eSignature for Salesforce Apex
  Toolkit](https://developers.docusign.com/docs/salesforce/salesforce101/apex-toolkit/).
- Add non-Docusign users to the **Docusign Sender** permission set
  to let them send envelopes through the System Sender.
