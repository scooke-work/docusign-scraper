# Resend the Activation Email for a Docusign for Salesforce User

Learn how to resend the activation email to a Docusign CLM, Gen,
Negotiate, or eSignature for Salesforce user.

When you create a Docusign user in Salesforce, Docusign sends an activation email. The user
selects a link in the email and activates their Docusign user. If the user cannot find their
original activation email, you can send them another.

Use this procedure to resend a Docusign activation email to a Salesforce user:

1. Access the Docusign
   Users page as an administrator:
   1. Log on to your Docusign account.
   2. Select Settings at the top of the page.
   3. Select Users in the left column.
2. Select Actions next to a user and choose Resend
   Invitation from the dropdown menu.
3. (Optional) Access Code: Add a security code in the dialog box
   that opens. You must send the access code to the user.
4. Select OK.

You have completed the process. You have sent a new
activation email to the inactive user.
