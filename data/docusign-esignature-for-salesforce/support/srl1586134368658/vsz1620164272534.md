# Access the Place Fields Step of an Envelope Template

Find out how to access the Place Fields step in an existing envelope template. This
includes: place fields actions (copy, paste, undo, redo, and zoom), customizing eSignature
field options, creating a new custom field, using the Actions button, and using recipient
preview actions, and more.

You can edit existing envelope templates on the Place Fields step.
This includes: place fields actions (copy, paste, undo, redo, and zoom), customizing
eSignature field options, creating a new custom field, and more.

Follow this
procedure to access the envelope template Place Fields
step:

Access the envelope template Place Fields page as a
Salesforce administrator:

1. Select the App Launcher grid ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/applaunchergrid.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcxMTYsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.Qyw042-QelaKgitGlcUOvlXpkqBvl40Nr_s2VAAWyEA&_LANG=enus) and
   select View All.
2. Select the Docusign Apps Launcher tile.
3. Select the Docusign Envelope Templates
   tab.
4. Select the arrow ![A screenshot of a downward-facing triangle.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcxMTYsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.Qyw042-QelaKgitGlcUOvlXpkqBvl40Nr_s2VAAWyEA&_LANG=enus "A screenshot of a downward-facing triangle.") next to the template to be edited.
5. Select Edit from the dropdown menu.

   The envelope template creation page opens.
6. Select the Place Fields arrow. The arrow has no
   text until selected.

You have completed the process. You have accessed the
Place Fields step of an existing envelope template.
