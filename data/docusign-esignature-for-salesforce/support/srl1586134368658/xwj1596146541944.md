# Step 1: Add a Template Name and a Salesforce Data Source

Learn to select a name and Salesforce object for your template.

Tip:
[This video demonstrates the steps in this task](https://players.brightcove.net/6118377982001/default_default/index.html?videoId=6171917923001).

In this step, you name your template and select a Salesforce data source. The data source is
the Salesforce object on which you build your template. After you select your data
source, you can add fields from that object and related objects.

Use this procedure to name the template and select the Salesforce data source:

1. Select Create Template.

   The Create Template dialog box opens.
2. Choose a name and data source:
   1. Name: Enter a descriptive template name.
   2. Select Data Source: Select a Salesforce object from the dropdown menu.
   3. Select Next.

   The Documents page opens.
