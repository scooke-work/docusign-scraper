# Add Automatic AutoPlace Fields to Documents

Learn how to add AutoPlace fields to documents used in envelope templates.

You can add automatic AutoPlace fields to envelope template documents.

These fields merge Docusign fields into your envelope.

You do not need to create, configure, or modify any Docusign fields to use automatic
AutoPlace fields.

Use automatic AutoPlace fields for these signer fields:

- Signature
- Date Signed
- Initial
- Optional Initial

Note: Use [document custom AutoPlace
fields](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=ryo1673648971228.html&_LANG=enus) or merge field AutoPlace fields for Name,
Company, Title, and other
text-based fields.

Follow this procedure to add AutoPlace fields to an envelope template document:

1. Select the name of the document you will edit.

   The document preview page opens.
2. Select the Download button in the upper-right.

   The document is saved to the downloads folder of your
   device.
3. Add AutoPlace fields to your template document:
   1. Open the document and move the cursor to where you will add AutoPlace
      fields.
   2. Enter the text for AutoPlace fields (\s1\ for
      Signer 1 signature, \n1\ for Signer 1 name, and
      so on.)![A screenshot displays a document with sample AutoPlace field codes such as \s1\ for Signer 1 signature and \n1\ for Signer 1 name entered in the text.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/AutoPlace-field-examples.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcxNTYsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.n1xm6pQfoZAUXAiO_3zThp-2NhaIrtNljuE-fGp3n54&_LANG=enus "A screenshot displays a document with sample AutoPlace field codes such as \s1\ for Signer 1 signature and \n1\ for Signer 1 name entered in the text.")
   3. Enter the text for AutoPlace fields (\s1\ for
      Signer 1 signature, \n1\ for Signer 1 name, and
      so on.)
   4. Highlight the AutoPlace text and change the font color to match the
      page background. So, if the page is white, change the AutoPlace font to
      white.

      Note: This prevents the AutoPlace text from being
      visible in your envelope. Your recipients only see the signature
      fields.

      ![A screenshot displays a document where AutoPlace field codes such as \s1\ and \n1\ are entered in the text and formatted in white font so they blend into the page background.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/AutoPlace-fields-with-white-font.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcxNTYsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.n1xm6pQfoZAUXAiO_3zThp-2NhaIrtNljuE-fGp3n54&_LANG=enus "A screenshot displays a document where AutoPlace field codes such as \s1\ and \n1\ are entered in the text and formatted in white font so they blend into the page background.")
   5. Save the changes to your template document.
4. Replace your old template document with the new version:
   1. Return to the envelope template Documents
      page.
   2. Select the arrow dropdown menu next next to the document
      you updated.
   3. Select Delete. The document disappears.

      ![A screenshot displays the envelope template Documents page with a document’s options menu open and the Delete action selected to remove the template document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Delete-template-document.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcxNTYsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.n1xm6pQfoZAUXAiO_3zThp-2NhaIrtNljuE-fGp3n54&_LANG=enus "A screenshot displays the envelope template Documents page with a document’s options menu open and the Delete action selected to remove the template document.")
   4. Select the Upload button.
   5. Select the name of the updated file from your device.
   6. Select Open. The Upload File dialog box
      opens.
   7. Select the Done button.

   The dialog box disappears. The name of your updated document
   appears.
5. Select the Save and Close button in the
   upper-right.

   The envelope template page opens.

You have completed the process. You have added automatic AutoPlace
fields to your envelope template document. When you send an envelope from this template,
Docusign or Salesforce fields are automatically placed where you added AutoPlace fields.
The AutoPlace text is not visible to your recipients, because you colored it to match
the page. Your recipients only see the fields they need to complete.
