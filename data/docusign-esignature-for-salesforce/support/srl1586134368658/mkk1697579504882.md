# Add a Send with Docusign Action to a Salesforce Flow

Learn to add a Send With Docusign action to your Salesforce Flow to
trigger envelope creation.

## Before you begin

Install Docusign Apps Launcher v6.1 or higher to
use this feature.

You can build a Send With Docusign action to add envelope-sending
capabilities to a Salesforce Flow. The Send With Docusign action
lets your Flow trigger a Docusign envelope template that creates
and sends envelopes.

Example: You build a Record-Triggered Flow and link it to a Docusign envelope template. You configure the Flow to trigger
envelope creation when a record's stage changes to
Negotiation/Review. A user changes the stage of the
Burlington Textiles opportunity to Negotiation/Review. Docusign creates an envelope using data from the record and sends
it to Burlington Textile representatives.

Note:
Docusign does not support using the Sign
Now option when you add an In Person Signer to a
Flow.

Use this procedure to add a Send With Docusign action to a Salesforce
Flow:

1. As a Salesforce administrator, create or edit a Salesforce Flow.
2. Add a Send with Docusign action:
   1. Locate the point in your Flow where you want to add your action.
   2. Hover over the circle icon and select the plus +
      icon that displays.

      ![A screen capture displays the Add Element plus icon in a Salesforce Flow.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/A-screen-capture-displays-the-Add-Element-plus-icon-in-a-Salesforce-Flow.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg2NTgsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9._1XjFBzX-1RcmidiFtG9hnfMwsFyBi-_zbFfHw1JuKI&_LANG=enus "A screen capture displays the Add Element plus icon in a Salesforce Flow.")
   3. Add Element: Enter and select
      Send with Docusign in the search field.

   The Send with Docusign dialog box opens.
3. Add a label and API name for your action:
   1. Label: Enter Send with Docusign.
   2. API Name: Keep or change the default name.
4. Envelope Template ID: Add the ID of the envelope
   template used in the Flow.
   1. Select the Docusign Envelope Templates tab.
   2. Select the envelope template that you want to use in the
      Flow.
   3. Copy the alpha-numeric template ID between the / marks at the end of the URL.![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/A-screenshot-displays-the-template-ID-highlighted-in-the-envelope-template-URL.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg2NTgsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9._1XjFBzX-1RcmidiFtG9hnfMwsFyBi-_zbFfHw1JuKI&_LANG=enus)
   4. Paste the ID (without / marks) in the Envelope
      Template ID field.
5. Source Id: Choose the ID of the object your Flow is
   built on. For example, if your Flow is built on Opportunity, choose: Triggering Opportunity >  Opportunity ID.![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Send_with_Docusign_flow_action.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg2NTgsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9._1XjFBzX-1RcmidiFtG9hnfMwsFyBi-_zbFfHw1JuKI&_LANG=enus)
6. Select Save As New Version > Save > Activate.

   A message displays: Your automation was
   activated.

You have completed the process. You added a Send With
Docusign action to your Salesforce Flow. Your Flow is connected to a Docusign envelope template. When conditions are met, your Salesforce
Flow triggers the Docusign envelope template to create an envelope.
