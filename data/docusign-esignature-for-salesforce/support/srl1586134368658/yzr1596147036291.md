# Delete an Envelope Template

Learn to delete envelope templates.

Use this procedure to delete an envelope template:

1. Access an envelope template as a Salesforce administrator:
   1. Select the App Launcher grid ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/applaunchergrid.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg2MzcsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.AaO4-jAOPmciyep8otyXDxvN44UUnjbZ540W4vmPUgE&_LANG=enus) and choose
      View All.
   2. Select the Docusign Apps Launcher tile.
   3. Select the Docusign envelope templates tab.
   4. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg2MzcsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.AaO4-jAOPmciyep8otyXDxvN44UUnjbZ540W4vmPUgE&_LANG=enus) next to the Gen template to be edited.
   5. Select Delete from the options.

      A message displays: Are you sure you want to delete this
      envelope template?
2. Select Delete.

   A message displays: The envelope template has been
   deleted.

You have completed the process. You have deleted the
template.
