# Step 3: Add Recipients to Your Template

Learn to add recipients to your envelope template.

Tip:

- [This video demonstrates the steps in this
  task](https://players.brightcove.net/6118377982001/default_default/index.html?videoId=6171919623001).
- [This topic demonstrates the
  advanced options available in this step](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=psq1595994131623.html&_LANG=enus).

In this step, you can add recipients to your template. Recipients are the people who sign the
agreement when you send an envelope.

Use this procedure to add recipients to your envelope template:

1. Select Add Recipient.
2. Select a recipient type in the Add Recipient dialog box:
   - From Salesforce
     - Lookup Fields: Enter Salesforce fields
       like Owner or Created
       By to add recipients.
     - Related Lists: Enter Salesforce fields
       related to the source object to add recipients.
     - Users/Contacts: Enter Salesforce users or
       contacts to add recipients.
     - Sender: Enter the sender of the envelope
       as a recipient. Select +Add Role and
       enter a role, like Sender 1.
   - Other Sources
     - By Role: Use a placeholder title, like
       Approver. The sender can specify a name and email address later.
       You can use custom syntax to  [add a By Role
       recipient from a Salesforce object](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=wtk1625247088017.html&_LANG=enus).
     - Signing Group: Add a group of recipients
       from the connected Docusign account. Any
       recipient can sign for the group.
3. (Optional) Delivery: This field displays for customers
   with Allow SMS Delivery and Signer can sign on
   a mobile device enabled.
   - Email: Send an envelope by email.
   - Email + SMS: Send your envelope by email and text message to as many as four recipients.

     Important: Enter a space between the country code and the phone number.
4. Action: Select a recipient option from the dropdown
   menu:
   - Needs to Sign: The recipient must sign the document.
   - Receives a Copy: The recipient gets a courtesy copy.
   - Needs to View: The recipient must view the document.
   - Specify Recipients: The recipient can add names and emails for placeholder recipients (like Approver.)
   - Allow to Edit: The recipient can make changes to the document.
   - In Person Signer: The recipients take part in a signing session with all signers in the same room.
5. Select Add to save the recipient, or Add
   & New to save and add another recipient.
6. Choose any optional steps:
   - Signing Order: Change numbers next to recipients to choose who signs first. You can also drag recipients to change order.
     ![A screenshot displays one recipient highlighted with arrows indicating that it is moving above another recipient.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/DAL_Envelope_Template_drag_recipient.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY4OTAsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.pwlcWNPbs5v894GSBxQMb3HnPP2aK3OWZUDtf515raE&_LANG=enus "A screenshot displays one recipient highlighted with arrows indicating that it is moving above another recipient.")
   - View: Select to see a visual diagram of the signing order.
   - Email Subject: Enter up to 100 characters of text.
   - Email Message: Enter up to 2,000 characters of text.
   - More Options: Select the arrow > to open the menu and choose the following options:
     - Reminders: Set the number of days between recipient reminders.
     - Expiration: Enter the number of days before the envelope expires.
7. Select Next.
