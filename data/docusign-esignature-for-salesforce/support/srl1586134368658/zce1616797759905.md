# Use Merge Field Relationships in an Envelope Template

Learn to use merge field relationships in an envelope template. Merge fields can access
up to four levels of relation to a source object.

You can create merge fields in an envelope template if they are related to the source
object.

Example:

An envelope template that uses Account as the source object sees
Opportunity fields, but not Case fields.

Merge fields can access up to four levels of relation to a source object. The following are
the possible relationship paths:

- Source object >  Field
- Source object >  Reference >  Field
- Source object >  Child >  Field
- Source object >  Reference >  Reference >  Field
- Source object >  Child >  Reference >  Field

Note: You cannot access related list fields. Use [Document Custom Fields](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=crp1583277382187.html&_LANG=enus) to access related list fields.
