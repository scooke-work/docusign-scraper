# Configure Docusign eSignature for Salesforce Using the Templates, Buttons, and Settings Tabs

Learn about configuring templates, custom buttons, and settings on the eSignature Configuration page.

The Docusign Apps Launcher package contains the Docusign eSignature for Salesforce app. You can use this app to create envelope templates, add buttons to layouts, and manage eSignature administrator settings.

You can access your eSignature for Salesforce controls by selecting eSignature on the Docusign  Setup page.![A screenshot displays the Templates, Button, and Settings tabs highlighted on the e-signature configuration page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/A-screenshot-displays-the-Templates-Button-and-Settings-tabs-highlighted-on-the-esignature-configuration-page.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY2NjksInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.72QmyqUbequXKthcnfihippd4NOHOMGM96t0m4haJdU&_LANG=enus "A screenshot displays the Templates, Button, and Settings tabs highlighted on the e-signature configuration page.")

You can access the following settings on the eSignature page:

- [Templates](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=gou1591745811909.html&_LANG=enus): Build Docusign Envelope Templates to automate your eSignature process.
- [Button](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=zsw1586134270460.html&_LANG=enus): Add a Send with Docusign button to Salesforce layouts.
- [Settings](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=kor1611642110382.html&_LANG=enus):
  - [Enable Built-in eSignature](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=kor1611642110382.html&_LANG=enus): Turn access to the eSignature for Salesforce app on or off. Turn off access if you want to use another app, like legacy Docusign eSignature for Salesforce.
  - [Enable System Sender](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=jxz1611642321527.html&_LANG=enus): Enable a user that can send envelopes on behalf of non-Docusign users.
