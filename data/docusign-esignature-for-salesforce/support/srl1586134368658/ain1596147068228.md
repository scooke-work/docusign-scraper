# Change the Owner of an Envelope Template

Learn to change the owner of envelope templates.

Use this procedure to change the owner of an envelope template:

1. Access an envelope template as a Salesforce administrator:
   1. Select the App Launcher grid ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/applaunchergrid.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg2NDEsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.MAfaTSwMLEM7lr22V7JfekhjF9jqkRUYtnZoWOxhTFw&_LANG=enus) and choose
      View All.
   2. Select the Docusign Apps Launcher tile.
   3. Select the Docusign envelope templates tab.
   4. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg2NDEsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.MAfaTSwMLEM7lr22V7JfekhjF9jqkRUYtnZoWOxhTFw&_LANG=enus) next to the Gen template to be edited.
   5. Select Change Owner from the options.

      The Change Owner dialog box displays.
2. Complete the following:
   - Search People: Enter the name of a different
     Salesforce user.
   - (Optional) Send notification email: Check this box to
     notify the new template owner.
3. Select Submit.

   ![A screenshot displays the Change Owner dialog box with Test User highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/envelopetemplatechangeowner.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg2NDEsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.MAfaTSwMLEM7lr22V7JfekhjF9jqkRUYtnZoWOxhTFw&_LANG=enus "A screenshot displays the Change Owner dialog box with Test User highlighted.")

   A message displays: (new user) now owns the record for (template
   name).

You have completed the process. You have changed the template
owner.
