# Step 7: Place Docusign and Salesforce Fields on Your Template Documents

Learn to place Docusign and Salesforce fields on the document
text.

## Before you begin

Tip:

- [This video](https://players.brightcove.net/6118377982001/default_default/index.html?videoId=6171917638001) demonstrates the
  process.
- [This topic](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=uws1596054466754.html&_LANG=enus) demonstrates the
  advanced options available.

Important:

- The Place Fields step does not display for templates with zero
  uploaded documents.
- The Place Fields step does not display in a linked envelope
  template, even with documents uploaded.

Use this procedure to place Docusign and Salesforce
fields on the document. The merge fields that you created in the previous step
display on the Salesforce Merge Fields menu.

1. Drag Docusign fields to the document.

   ![A screen capture displays a user dragging Docusign fields from the left panel to a Salesforce Envelope Template document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/DAL_envelope_template_add_Docusign_fields.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY4NjcsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.fSLRR0TE1a2-INSi--HmUl6Fsy7e9wMj51NHrBJ5XII&_LANG=enus "A screen capture displays a user dragging Docusign fields from the left panel to a Salesforce Envelope Template document.")
2. Add Salesforce fields to the document:
   1. Select the Merge Fields button
      ![A screenshot displays the Merge Fields button icon in the Salesforce Envelope Template editor toolbar.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/DAL_envelope_template_merge_fields_icon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY4NjcsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.fSLRR0TE1a2-INSi--HmUl6Fsy7e9wMj51NHrBJ5XII&_LANG=enus "A screenshot displays the Merge Fields button icon in the Salesforce Envelope Template editor toolbar.") in the left column. The Salesforce Merge Fields
      menu opens.
   2. Select a field from the menu. You can expand arrows to find
      more fields, or enter text in the Search Fields bar.
   3. Drag the field to the document.

   ![A screen capture displays a user dragging Salesforce merge fields from the left panel into the document area of a Salesforce Envelope Template.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/DAL_envelope_template_add_Salesforce_fields.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY4NjcsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.fSLRR0TE1a2-INSi--HmUl6Fsy7e9wMj51NHrBJ5XII&_LANG=enus "A screen capture displays a user dragging Salesforce merge fields from the left panel into the document area of a Salesforce Envelope Template.")
3. (Optional):
   - Recipient Preview: Select this button
     to see what the generated document will look like.
   - Switch recipient view: Follow these
     steps to add fields for more recipients:
     1. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY4NjcsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.fSLRR0TE1a2-INSi--HmUl6Fsy7e9wMj51NHrBJ5XII&_LANG=enus) next to the recipient
        name.
     2. Select another recipient from the dropdown menu.
     3. Drag fields for this recipient.

        Note: Each recipient's fields are
        represented by a separate color (yellow, blue, and
        so on.)![A screen capture displays a user switching recipients while adding Docusign fields for different recipients in an envelope template.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/DAL_envelope_template_add_secondary_recipient_fields.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY4NjcsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.fSLRR0TE1a2-INSi--HmUl6Fsy7e9wMj51NHrBJ5XII&_LANG=enus "A screen capture displays a user switching recipients while adding Docusign fields for different recipients in an envelope template.")
4. Select Save and Close when you are finished adding
   fields.

   The Docusign eSignature fields page closes. The Docusign Envelope Template page displays.
