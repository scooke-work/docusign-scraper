# Step 3: Remove Legacy eSignature Custom Buttons From Your Salesforce Layouts

Learn how to remove legacy Docusign eSignature for Salesforce custom buttons from your
Salesforce page layouts.

You need to remove legacy Docusign eSignature for Salesforce custom buttons before
you can uninstall the legacy app.

Use this procedure to remove all legacy Docusign eSignature for Salesforce custom
buttons from your Salesforce layouts:

1. Open the Buttons, Links, and Actions page as an
   administrator:

   1. Select the gear ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg4MTgsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.wZhwPPVYLZugoTXvvPNcRfIIRONF6E2hHEEmFoBfh6g&_LANG=enus) in the upper-right of the
      page.
   2. Select Setup.
   3. Enter Object Manager in the Quick Find
      field.
   4. Select the object (Account, Opportunity, and so on) link in the left
      column.
   5. Select the Buttons, Links, and Actions link.
2. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg4MTgsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.wZhwPPVYLZugoTXvvPNcRfIIRONF6E2hHEEmFoBfh6g&_LANG=enus) next to the custom button you want to remove.
3. Select Delete from the dropdown menu.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/gendeletecustombutton.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg4MTgsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.wZhwPPVYLZugoTXvvPNcRfIIRONF6E2hHEEmFoBfh6g&_LANG=enus)
4. Select the Delete button on the Are you
   sure? dialog box.

   The page refreshes with the button no longer displayed.

   Note: You may see a page that
   says: Your request to delete this custom button or link can not
   be completed because this button or link is being used by another
   feature. In this case, navigate to the Salesforce page
   layout for that object (Account, Opportunity, and so on) and remove the
   button from the page layout.
5. Repeat this entire procedure for each Salesforce layout (Account, Opportunity,
   and so on) that contains a Docusign legacy eSignature custom button.

You have completed step three of the process. You
have removed all Docusign legacy eSignature custom buttons from your Salesforce layouts.
Move to [Step 4: Uninstall the Legacy Docusign eSignature for Salesforce App from your Salesforce organization](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=the1642634056297.html&_LANG=enus "Learn how to uninstall the legacy Docusign eSignature for Salesforce app from your Salesforce organization.").
