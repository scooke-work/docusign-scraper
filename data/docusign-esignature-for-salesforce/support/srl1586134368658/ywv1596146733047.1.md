# Step 5: Place Docusign and Salesforce Fields on Your Template Documents

The fifth step of the Envelope Template creation process involves placing Docusign and
Salesforce fields on the document text.

Use this procedure to place Docusign and Salesforce fields into the document
text. The Merge Fields that you created in step four are available on the Salesforce
Merge Fields menu.

Tip:

- [This video](https://players.brightcove.net/6118377982001/default_default/index.html?videoId=6171917638001) walks you through the process.
- [This topic](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=uws1596054466754.html&_LANG=enus) shows you all of the
  advanced options available on this step.

1. Add Salesforce fields to the document:

   1. Select the Merge Fields button ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/isy1595858937605.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg5MDAsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.9qQRei1ikngvA2IzSCo4zZMnLBflKHdgAvkoGvAN9-g&_LANG=enus) from the left
      column. The Salesforce Merge Fields menu opens.
   2. Select a field from the menu. You can expand arrows to find more fields,
      or enter text in the Search Fields bar.
   3. Drag the field to the document text.
      ![Add Salesforce fields](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/ntk1617951128350.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg5MDAsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.9qQRei1ikngvA2IzSCo4zZMnLBflKHdgAvkoGvAN9-g&_LANG=enus)
      ![Add Salesforce fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/tax1611210842878.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg5MDAsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.9qQRei1ikngvA2IzSCo4zZMnLBflKHdgAvkoGvAN9-g&_LANG=enus)
2. Add Docusign fields to the document:

   1. Select the Standard Fields tab ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/jcw1595860961076.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg5MDAsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.9qQRei1ikngvA2IzSCo4zZMnLBflKHdgAvkoGvAN9-g&_LANG=enus) in the left
      column. The Docusign Fields menu opens.
   2. Select a Docusign field (Signature,
      Initial, and so on.)
   3. Drag the field to the document.
3. (Optional):

   - Recipient Preview: Select this button to see what
     the generated document will look like.
   - Switch recipient view: Follow these steps to add
     fields for more recipients:
     1. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/vga1586134207550.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg5MDAsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.9qQRei1ikngvA2IzSCo4zZMnLBflKHdgAvkoGvAN9-g&_LANG=enus) next to the recipient
        name.
     2. Select another recipient from the dropdown menu.
     3. Drag Docusign fields for this recipient to the document
        text.

        Note: Each
        recipient's fields are represented by a separate color
        (yellow, blue, and so on.)
        ![Switch recipients.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/mgl1617736353196.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg5MDAsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.9qQRei1ikngvA2IzSCo4zZMnLBflKHdgAvkoGvAN9-g&_LANG=enus)
        ![Switch recipients.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/szf1586134366923.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg5MDAsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.9qQRei1ikngvA2IzSCo4zZMnLBflKHdgAvkoGvAN9-g&_LANG=enus)
4. Select the Save and Close button when you are finished
   adding fields.

   Note: **The Docusign eSignature fields page closes. The Docusign Envelope Template page
   opens.**

You have completed step five.
