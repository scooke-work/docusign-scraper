# Learn About Using the Default Docusign for Salesforce Tabs

Learn to find and use the default Docusign Apps Launcher tabs.

A set of tabs displays when you open Docusign Apps Launcher in your Salesforce
organization. These tabs display functions commonly needed by a Docusign for Salesforce administrator.

![A screenshot displays the default Docusign tabs: Setup, Gen Templates, Envelope Templates, Logs, Envelopes, and Status.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/A-screenshot-displays-the-default-Docusign-for-Salesforce-tabs.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY2ODgsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.zhiBln6mwYKkMgrMIEHV622BbbLQ05Q2mWiVLYUwPWQ&_LANG=enus "A screenshot displays the default Docusign tabs: Setup, Gen Templates, Envelope Templates, Logs, Envelopes, and Status.")

You can access these tabs by [opening Docusign Apps Launcher](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=etv1586134206174.html&_LANG=enus).

[Docusign Setup](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=cay1654320876587.html&_LANG=enus)
:   Docusign Setup is the landing page for administering your Docusign account. You can manage Docusign
    users, components, templates, and more. Depending on the apps you purchased, you
    see a custom Docusign administrator dashboard.
    Gen customers see different administrator controls than
    CLM customers.

[Docusign Gen Templates](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=fsc1646690056619.html&_LANG=enus)
:   Create, edit, and delete Docusign Gen templates from this
    page. Users can generate agreements by selecting a Gen template button on a
    Salesforce record.

[Docusign Envelope Templates](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=scr1646715662157.html&_LANG=enus)
:   Create, edit, and delete Docusign envelope templates from this
    page. Users can send envelopes for signature by selecting an envelope template
    button on a record.

[Docusign Logs](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=epu1646716431620.html&_LANG=enus)
:   View the DEBUG, INFO, WARN, and ERROR logs of envelopes sent from Salesforce
    records.

[Docusign Envelopes](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=nbq1646756159307.html&_LANG=enus)
:   View the details of all Docusign envelopes sent from
    Salesforce records.

[Docusign Status](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=bek1646762309217.html&_LANG=enus)
:   View the status (Sent, Completed, and so on) of all Docusign
    envelopes sent from Salesforce records.

Select any tab to view the page.
