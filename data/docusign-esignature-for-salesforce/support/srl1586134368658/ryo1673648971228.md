# Use Document Custom AutoPlace Fields

Learn to use document custom auto-place fields for recipient information that does not
write back to Salesforce.

You can add document custom field AutoPlace fields to envelope template
documents.

Use document custom AutoPlace fields in templates that don’t need Docusign fields
mapped to Salesforce fields. When recipients complete their assigned fields,
Docusign does not update any fields in Salesforce.

Important:
[You must create document custom fields in your
Docusign account before you can use them in your envelope template.](https://support.docusign.com/s/document-item?language=en_US&bundleId=pik1583277475390&topicId=zus1583277381502.html&_LANG=enus)

Follow this procedure to add document custom AutoPlace fields to an envelope template
document:

1. Select the name of the document you will edit.

   The document preview page opens.
2. Select the Download button in the upper-right.

   The document is saved to the downloads folder of your
   device.
3. Add document custom AutoPlace fields to your template document:

   1. Open the document and move the cursor to where you will add document custom AutoPlace
      fields.
   2. Enter the text for AutoPlace fields (\s1\ for
      Signer 1 signature, \n1\ for Signer 1 name, and
      so on.)
      ![AutoPlace field examples](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/AutoPlace-field-examples.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg1OTAsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.zCr1H4xDSQYjxef2tBFwVczzHb7TFJfZKUY1n5MD5WE&_LANG=enus "AutoPlace field examples")
   3. Highlight the AutoPlace text and change the font color to match the page
      background. So, if the page is white, change the AutoPlace font to
      white.
      ![AutoPlace fields with white font](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/AutoPlace-fields-with-white-font.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg1OTAsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.zCr1H4xDSQYjxef2tBFwVczzHb7TFJfZKUY1n5MD5WE&_LANG=enus "AutoPlace fields with white font")

      Note: This prevents the AutoPlace text from
      being visible in your envelope. Your recipients only see the
      signature fields.
   4. Save the changes to your template document.
4. Replace your old template document with the new version:

   1. Return to the envelope template Documents
      page.
   2. Select the arrow dropdown menu next next to the document
      you updated.
   3. Select Delete. The document disappears.
      ![Delete template document](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Delete-template-document.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg1OTAsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.zCr1H4xDSQYjxef2tBFwVczzHb7TFJfZKUY1n5MD5WE&_LANG=enus "Delete template document")
   4. Select the Upload button.
   5. Select the name of the updated file from your device.
   6. Select Open. The Upload File dialog box
      opens.
   7. Select the Done button.

   The dialog box disappears. The name of your updated document
   appears.
5. Select the Save and Close button in the
   upper-right.

   The envelope template page opens.

You have completed the process. You have added standard
AutoPlace fields to your envelope template document. When you send an envelope from this
template, Docusign or Salesforce fields are automatically placed where you added
AutoPlace fields. The AutoPlace text is not visible to your recipients, because you
colored it to match the page. Your recipients only see the fields they need to complete.
