# Envelope Template Field Options

How to customize e-signature field options in a Docusign envelope template.

You can customize Docusign envelope template fields from the Place Fields
page of an existing envelope template.

Select any field to open an options panel on the far-right of the page.

Different options are shown, depending on the field type that is selected.

Follow this procedure to change a field that is placed on a page:

- Enlarge field: Drag a field anchor outward.
- Shrink field: Drag a field anchor inward.
- Delete field: Select the field and press the Delete key.

Here are the configurable options available:

Table 1.
Docusign Field Options

| Option | Action |
| --- | --- |
| Required Field | The recipient must take action. |
| Read Only | This field option cannot be changed. |
| Formatting | Change font type, size, color, etc. |
| Data Label | Link fields of the same type by using a shared label. When one field is completed, the rest are as well. |
| Tooltip | Instructive text that appears when the cursor hovers over a field. |
| Location | Use pixels from the left and top to set the exact field location on the page. |
| AutoPlace | Fields are auto-placed on document every time a keyword is matched. |
| Sender Permissions | Make a field mandatory, or prevent senders from making changes. |
| Add Text | Add a text box field to the document. |
| Validation | Force user input into a set format, like xxx-xx-xxxx for Social Security number fields, xx/xx/xxxx for Date, etc. |
| Collaboration | Allow recipients to collaborate on envelope content. |
| Conditional Logic | Fields only appear to the recipient when a specified condition is met. |
| Collect Payment | Allow recipient to pay for services. |
| Save Payment Method | Allow recipient to store payment info for future transactions. |
| Payment Gateway | Set the method for transferring funds. |
| Payment Amount | Set the amount to be paid. |
| Payment Description | Add a note about the purpose of a payment. |
| Edit Formula | Edit an existing formula built from number and date fields in an envelope. |
| Save as Custom Field | Add a customized field to the Custom Fields menu for re-use. |
| Delete | Remove a selected field from the page. |
| Resize Field | Drag field corners to enlarge or shrink field boundaries on the page. |
