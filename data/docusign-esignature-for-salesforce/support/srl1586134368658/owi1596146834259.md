# Step 9: Add a Custom Button to Salesforce Layouts

Learn to add an Experience Cloud envelope template button to Salesforce
layouts.

Use this procedure to add a custom button to Salesforce page layouts:

Tip:
[This video demonstrates the steps in this task](https://players.brightcove.net/6118377982001/default_default/index.html?videoId=6172186652001).

1. Button Label: Enter a custom button name, like
   Experience Cloud Envelope.
2. Select the check box for each layout that needs a button.
3. Select Save and Finish.

   A message displays: Successfully modified
   layouts.

You have completed the entire process. You have built a new envelope
template and added a custom button to Salesforce layouts.
