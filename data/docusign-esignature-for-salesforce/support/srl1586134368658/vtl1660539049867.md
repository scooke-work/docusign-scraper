# Step 1: Select Templates and Choose a Name and Salesforce Object

Start by naming your template, choosing a Salesforce object, and selecting the existing
templates you will use.

You start by naming your template, choosing a Salesforce object, and selecting the existing
templates you will use. You can choose as many templates as you want.

1. Select Templates >  Create Template and choose Link Template from the arrow
   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY5MjUsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.aoTP70-FZOVNBf0UNnLXni38eLKb7Wnt9QBPCAO8dEI&_LANG=enus) menu.

   The Link Docusign Template dialog box
   opens.
2. Complete the actions in the Link Docusign Template dialog
   box:
   1. Select the check box next to each template that you want to add, or use the search field
      to find a template.
   2. Select the Select button.

      The Link
      Docusign Template dialog box refreshes.
   3. Name: Enter a descriptive template name.
   4. Select Data Source: Select a Salesforce object from the dropdown
      menu.
   5. Select Next.

   The dialog box closes. The Documents page
   opens.
