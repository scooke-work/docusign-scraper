# Envelope Template Recipient Actions

Find out about the actions you can assign to the recipients of an envelope built with a
Docusign envelope template.

There are many different recipient actions that can be assigned to recipients in an envelope
or envelope template.

You can use the following Action types:

Table 1. Recipient Actions

| Action Type | Description |
| --- | --- |
| Needs to Sign | The recipient must sign the envelope. |
| Receives a Copy | The recipient receives a copy of the envelope. They do not have to take action. |
| Needs to View | The recipient must open and view the document. They do not have to sign. |
| In Person Signer | Conduct a multi-recipient signing session. The host signs, and then each additional recipient signs on the host's device.  All signers must be in the same room for the signing session. |
| Specify Recipients | The recipient can fill in the name and email address of other recipients in their stage of the signing order or later. |
| Allow to Edit | Grant another Docusign user full edit rights for an envelope. This recipient can edit the entire envelope, using the correct mode to make changes. The recipient is not assigned any fields. |
| Update Recipients | The recipient can fill in the name and email address of other recipients in their stage of the signing order or later. |
| Embedded Signer | The recipient can edit the details of any remaining recipients in the envelope. |
