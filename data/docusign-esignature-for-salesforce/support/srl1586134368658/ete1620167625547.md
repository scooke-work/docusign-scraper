# Access the Options Step of an Envelope Template

Find out how to access the Options step in an existing envelope template. This includes:
changing the sending experience, updating a field value, and using the document writeback
feature.

You can access the Options page in an existing envelope template. This
includes: changing the sending experience, updating a field value, and using the
document writeback feature.

Access the envelope template Options page as a
Salesforce administrator:

1. Select the App Launcher grid ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/applaunchergrid.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg1OTcsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.JY3eRVNLXmXv_oVpNDsh4_KhOUDfcenCB7SLksZvvI8&_LANG=enus) and
   select View All from the dropdown menu.
2. Select the Docusign Apps Launcher tile.
3. Select the Docusign Envelope Templates
   tab.
4. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg1OTcsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.JY3eRVNLXmXv_oVpNDsh4_KhOUDfcenCB7SLksZvvI8&_LANG=enus) next to the template to be
   edited.
5. Select Edit from the dropdown menu.

   The envelope template creation page opens.
6. Select the Options arrow. The arrow has no text
   until selected.

You have completed the process. You have accessed the
Options page of an existing envelope template.
