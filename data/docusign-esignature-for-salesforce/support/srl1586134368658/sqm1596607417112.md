# Envelope Template Field Types and Functions

Learn about the field types that you can use in envelope templates.

The Docusign envelope template Place Fields tab contains many different fields that can be placed on uploaded documents.

Here are the fields that can be used in an envelope template:

Table 1.  Docusign field icons and actions

| Field Icon | Field Name | Action |
| --- | --- | --- |
|  | Signature | Add a recipient's signature. |
|  | Initial | Add a recipient's initials. |
|  | Stamp | Add a stamp image instead of a standard signature. |
|  | Date Signed | Add the date of a recipient's signature. |
|  | Name | Add the full name of a recipient. |
|  | Email | Add a recipient's email address. |
|  | Company | Add recipient's company. |
|  | Title | Add recipient's title. |
|  | Automatic Renewal | Add an AI-assisted attribute to indicate that an agreement will automatically renew at the end of its term. |
|  | Text | Add a free-form field for recipient to enter text. |
|  | Number | Add a field that only accepts numbers. You can format the field as a plain number or as currency. |
|  | Checkbox | Add a check box for recipient options. |
|  | Dropdown | Add a dropdown menu for recipient options. |
|  | Radio | Add a radio button for recipient options. |
|  | Payment Item | Add a field to accept recipient payments. You must configure a payment account to use this feature. |
|  | Drawing | Add a rectangular canvas where a recipient can draw or upload an image, diagram, and more. |
|  | Formula | Add a formula to calculate a value based on number or date fields in your document. |
|  | Attachment | Add a field to request additional documentation from a recipient. |
|  | Note | Add a read-only informational field that displays text to recipients during signing. |
|  | Approve | Add an alternative to a required Signature or Initial field. Selecting Approve is the equivalent of signing. |
|  | Decline | Add a field that terminates the recipient signing session. |
