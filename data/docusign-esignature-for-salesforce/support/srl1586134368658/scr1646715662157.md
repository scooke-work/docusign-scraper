# Access the Docusign Envelope Templates Tab to Build and Edit Envelope Templates

Learn to access the Docusign Envelope Templates tab in
Salesforce.

Docusign Envelope Templates allow users to select a button on a record to
launch an administrator-built sending experience.

Use this procedure to access the Docusign Envelope Templates tab in
your Salesforce organization:

1. Select the Docusign Envelope Templates tab.
2. Make any selection:

   - [New](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=gou1591745811909.html&_LANG=enus): Create a new envelope template.
   - [Change Owner](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=ain1596147068228.html&_LANG=enus): Change the owner of an envelope template.
   - [Import](https://support.docusign.com/s/document-item?bundleId=izj1586134369853&topicId=juw1586134252760.html&_LANG=enus): Import your data into Salesforce. You can import up to 50,000 records at a time.
   - [Rename](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=bzu1596146928480.html&_LANG=enus): Change the name of an envelope template.
   - [Edit](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=xow1596147003565.html&_LANG=enus): Add or edit documents, recipients, Salesforce fields, Docusign fields, and envelope template options.
   - [Delete](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=yzr1596147036291.html&_LANG=enus): Delete an envelope template.

   You have completed the task. You accessed the Docusign Envelope
   Templates tab. From this page, you can create and edit Envelope
   Templates.
