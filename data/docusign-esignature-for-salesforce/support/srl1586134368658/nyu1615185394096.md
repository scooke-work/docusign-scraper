# Use Envelope Events To Automatically Trigger Updates to Salesforce Fields

Learn how to add envelope events to automatically update up to 25 Salesforce
fields.

Use this procedure to add envelope events to your envelope template:

1. Select the + Add Update button in the Field
   Updates section.

   ![A screenshot displays the Field Updates section with the + Add Update button and a list of envelope field events, showing that you can add up to 25 field updates.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Add-up-to-25-Field-Events.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg2MDksInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.tgY2bOnU5XVBbdQMcALR94gp-MMSvUtmNqgcuJ-X_2M&_LANG=enus "A screenshot displays the Field Updates section with the + Add Update button and a list of envelope field events, showing that you can add up to 25 field updates.")
2. Add a new Envelope Event to your template:

   1. Envelope Event: Select a Docusign envelope event
      from the dropdown menu. This is the event that triggers a Salesforce
      field update.
   2. Field Name: Select a Salesforce field from the
      dropdown menu. This is the field that is updated when the envelope
      event occurs.
   3. Field Value: Enter a value, or select a dropdown
      menu option. This is the information that updates the field that you
      selected.

   Note: You can add up to 25 Salesforce fields (five per event type.)
3. (Optional):

   - ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Delete-field-event.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg2MDksInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.tgY2bOnU5XVBbdQMcALR94gp-MMSvUtmNqgcuJ-X_2M&_LANG=enus): Select the trash icon to remove an Envelope Event
   - ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Remove-Field-Name.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg2MDksInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.tgY2bOnU5XVBbdQMcALR94gp-MMSvUtmNqgcuJ-X_2M&_LANG=enus): Select the X icon next to a field name to
     replace it
4. Select Save and Close when all options are set.

   The process is complete. Up to 25 Salesforce source fields update when triggered by
   Docusign field events.
