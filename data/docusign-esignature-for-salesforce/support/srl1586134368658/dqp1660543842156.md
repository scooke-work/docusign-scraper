# Step 3: (Optional) Add Recipients to Your Template

In the third step of creating a linked envelope template, you add recipients and a
message to recipients.

Use this procedure to add recipients to your linked envelope template. You cannot view, edit,
or remove recipients from the linked templates. You can access recipient details
when you send an envelope from this template.

Note: This step is optional. You can
add new recipients, or keep the recipients imported from your linked templates.

1. Select the Add Recipient button.

   The Add Recipient dialog box opens.
2. Select a recipient type from the left column:

   - From Salesforce:
     - Lookup Fields: Salesforce fields like
       Owner or Created
       By that are used to dynamically add recipients
       from a Salesforce record.
     - Related Lists: A Salesforce field that an
       administrator chooses to relate to a parent object (Account,
       Opportunity, and so on.)
     - Users/Contacts: Salesforce Users or
       Contacts saved in your organization.
     - Sender: Add the sender of the envelope as
       a recipient. You must select +Add Role
       and enter a role, like Sender 1.
   - Other Sources:
     - By Role: A placeholder title (like Approver) that allows the
       sender to specify a name and email address later. You can use
       custom syntax to [dynamically add a By
       Role recipient](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=wtk1625247088017.html&_LANG=enus) from a Salesforce object.
     - Signing Group: A group of recipients in
       the connected Docusign account; any signer can complete the
       signature for the group
3. (Optional) Delivery: Select an option from the dropdown
   menu:

   - Email: Send an envelope by email.
   - Email + SMS: Send an envelope by email and text
     message.

     Important: You must enter
     a space between the country code and the phone number.

   Note: The Delivery field
   only displays for customers with Allow SMS Delivery
   and Signer can sign on a mobile device enabled.
   Contact your Docusign account team to enable these
   features. You can use the email + SMS option for up to four recipients per
   envelope.
4. Action: Select a recipient option from the dropdown
   menu:

   - Needs to Sign: Must sign the document
   - Receives a Copy: Does not need to view or
     sign
   - Needs to View: Must view the document
   - Specify Recipients: Can add names and emails for
     placeholder recipients (like Approver.)
   - Allow to Edit: Can make changes to the
     document
   - In Person Signer: A multi-recipient signing
     session where all signers must be in the same room
5. Select a save option:

   - Add & New: Save the recipient and add
     another
   - Add: Save the recipient and close the dialog
     box
6. (Optional):

   - Signing Order:
     1. Select the Signing Order button. Numbers
        display next to the recipients.
     2.    - View: Select this button to see a visual diagram
     of the signing order
   - Email Subject: Enter up to 100 characters of
     text.
   - Email Message: Enter up to 2,000 characters of
     text.
   - More Options: Select the arrow
     > to open the menu and choose any
     options:
     - Reminders: Set the number of days between recipient reminders
       from the dropdown menu
     - Expiration: Enter the number of days
       before the envelope expires
7. Select Next.
