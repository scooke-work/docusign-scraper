# Change the Sending Experience of an Envelope Template

Learn to change the sending experience of an envelope template.

Use this procedure to change the sending experience of an envelope template:

1. Sending Experience: Select an option:

- Default: Senders can add documents, specify recipients, and
  place fields before sending
- Quick Send: Senders can only add non-signing
  recipients
- Place Fields: Senders start at the place fields step,
  skipping the documents and recipients steps.

  Note: Senders
  can still choose to add and edit documents, recipients, and envelope options on
  the place fields step.
- Send Now: Senders do not see any controls. The envelope
  sends immediately.

2. Select the Next button in the upper-right.
3. Select Save and Finish on the
   Custom Button page.

You have completed the process. You have switched the sender
experience of the envelope template. Senders who click the Docusign custom button see
either the default or quick send experience.
