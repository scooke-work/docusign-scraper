# Add the Envelope Status Component to a Salesforce Layout

Learn to add the Docusign Envelope Status component to a Salesforce layout.

Docusign  Envelope Status is a component that you can add to your Salesforce layouts. The component displays the progress of all Docusign envelopes sent from a record. The component displays envelopes sent using the following methods:

- A Send with Docusign action
- An Envelope Template action
- [A Start with Docusign workflow button](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=tskcae26f13-f2e5-4afd-85a7-145a70b12977.html&_LANG=enus)
- [An Agreement Desk for Salesforce request](https://support.docusign.com/s/document-item?bundleId=map8c30aa94-7e9f-4950-92fc-dd22c6d9f835&topicId=tsk151c1542-bdf4-4c8e-ba20-37affb7adc7e.html&_LANG=enus)
- [eSignature actions in Salesforce Flows](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=dzi1705696883404.html&_LANG=enus)

Salesforce users with Docusign access can view and act on envelopes without logging on to Docusign.

Tip: This procedure is explained in this [short how-to video](https://docusign.gallery.video/watch?videoId=6250567558001).

Use this procedure to add the Docusign Envelope Status component to a Salesforce layout:

1. As a Salesforce administrator, open a record, select the gear icon ![A screenshot displays the a gear icon with a lightning bolt inside.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY2NjIsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.1nuGSqRBudLuQAbvJCVwLdEIKDQfBYcuok7Lz2QIWOg&_LANG=enus "A screenshot displays the a gear icon with a lightning bolt inside.") and choose Edit Page from the dropdown menu.
2. Drag the Docusign Envelope Status component to the page layout on the right.

   ![A screen capture displays a cursor dragging the Docusign Envelope Status component to an Opportunity layout and saving the change.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Add-the-DocuSign-Envelope-Status-component.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY2NjIsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.1nuGSqRBudLuQAbvJCVwLdEIKDQfBYcuok7Lz2QIWOg&_LANG=enus "A screen capture displays a cursor dragging the Docusign Envelope Status component to an Opportunity layout and saving the change.")
3. Activate the change to your layout:
   1. Select Activation.
   2. Select a tab to set your default options: Org Default, App Default, or App, Record Type, and Profile.
   3. Select Assign and choose options from the dialog box.
   4. Select Close.
4. Select Save.
5. Select the back arrow ![A screenshot displays the Salesforce Edit Page Back arrow icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/A-screenshot-of-the-Back-arrow-on-the-Salesforce-Edit-Page-settings-bar.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY2NjIsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.1nuGSqRBudLuQAbvJCVwLdEIKDQfBYcuok7Lz2QIWOg&_LANG=enus "A screenshot displays the Salesforce Edit Page Back arrow icon.") to return to the record.

You have completed the process. You have added the Docusign Envelope Status component to the layout. Users with Docusign access can view and act on Docusign envelopes.
