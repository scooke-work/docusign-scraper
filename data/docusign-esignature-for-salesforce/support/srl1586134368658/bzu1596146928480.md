# Rename an Envelope Template

Learn to rename envelope templates.

Follow these steps to rename your envelope template:

1. Select the Docusign envelope templates tab.
2. Check the box to the left of a template name.
3. Select Edit.![A screenshot displays the template name in edit mode with the Name field active for renaming.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/A-screenshot-displays-the-template-name-in-edit-mode.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg2MjcsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.szvJDBz-MXvW_53ikIC0xBSAx1tEROCRtvhSyJYcMHQ&_LANG=enus "A screenshot displays the template name in edit mode with the Name field active for renaming.")

   The template Name opens in a text
   field.
4. Enter a new template name.
5. Select Save.

   A message displays: Your changes are saved.

You have completed the process. You have renamed the
template.
