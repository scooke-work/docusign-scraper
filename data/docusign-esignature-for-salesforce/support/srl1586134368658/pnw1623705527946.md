# Step 1: Remove Users from Apps Launcher Permission Sets

Learn to remove all users from Docusign permission sets.

You are required to remove at least one user from the following permission sets:

- Docusign Administrator
- Docusign CLM Admin

You may be required to remove users from [other Docusign Apps Launcher
permission sets](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=xxm1623778416524.html&_LANG=enus "Learn about the types and functions of Docusign Apps Launcher permission sets.").

1. Open the Permission Sets page as an administrator:
   1. Select the gear ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg4MjgsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.2I84s953OCn-UsBZP-iFCnrTaYVx3kXVTq8aKSCCw7c&_LANG=enus) and select
      Setup.
   2. Enter and select Permission Sets in the Quick
      Find field.

   The Permission Sets page opens.
2. Remove all users from the Docusign Administrator
   permission set:
   1. Select Manage Assignments.
   2. Select the Action check box to select all
      users.
   3. Select Remove Assignments.![A screenshot displays the Action check box and the remove assignments button highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/daluninstallremoveassignments.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg4MjgsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.2I84s953OCn-UsBZP-iFCnrTaYVx3kXVTq8aKSCCw7c&_LANG=enus "A screenshot displays the Action check box and the remove assignments button highlighted.")

      A dialog box displays: Are you sure you want to remove
      these users from this permission set?.
   4. Select OK.

      The dialog box closes. The Docusign
      Administrator permission set page opens.
3. Select Done.
4. Repeat this procedure for the Docusign CLM Admin
   permission set.
5. Repeat this procedure for any other Docusign permission sets
   containing users.

You have completed step one of the process. You have
removed all users from Docusign Apps Launcher permission sets. Move to
[Step 2: Remove Apps Launcher Components from Your Layouts](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=uxu1623708462949.html&_LANG=enus "Learn to remove all Docusign Apps Launcher components from Salesforce layouts.").
