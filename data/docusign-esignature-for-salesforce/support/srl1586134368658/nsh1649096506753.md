# Use Features from a Docusign Server Template in an Envelope Template

Learn how you can add options to an envelope template from a Docusign
server template.

You can build two types of Docusign eSignature templates:

## Docusign server templates

Docusign server templates are signing experiences that you build in your Docusign account. You log on to your Docusign
account and select the Templates tab to build or access a server
template.![A screenshot displays the Docusign server templates page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/A-screenshot-displays-the-Docusign-server-templates-page.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY5NTUsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.J6Ty7bN2bTgawBcrWcvFGrZrW9gFusedHKEfz2OSabk&_LANG=enus "A screenshot displays the Docusign server templates page.")

## Docusign envelope templates

Docusign envelope templates are signing experiences that you build in
your Salesforce organization with the Docusign eSignature for
Salesforce application. You open Docusign Apps Launcher in your
Salesforce organization to build an envelope template.![A screenshot displays the envelope template tab.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/A-screenshot-displays-the--envelope-template-tab.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY5NTUsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.J6Ty7bN2bTgawBcrWcvFGrZrW9gFusedHKEfz2OSabk&_LANG=enus "A screenshot displays the envelope template tab.")

## Using server template options in envelope templates

Docusign offers some options in server templates that are not yet
available in envelope templates. However, you can add many of these options to
envelope templates from the server template interface.

**Example**:

1. Create a Docusign envelope template.
2. Log in to your Docusign account. Your envelope template displays on the
   Templates page.
3. Add supported server template options to the envelope template.
4. Return to Salesforce and save your envelope template.

   The options you added to the server
   template now display in the envelope template. The previously
   unavailable server template features display when users send
   envelopes.

## Supported server template features

You can add the following server template options that are not available in the envelope
template interface:

## Advanced options

![A screenshot displays the Advanced options page of a Docusign server template.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/A-screenshot-displays-the-Advanced-Options-page-of-a-Docusign-server-template.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY5NTUsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.J6Ty7bN2bTgawBcrWcvFGrZrW9gFusedHKEfz2OSabk&_LANG=enus "A screenshot displays the Advanced options page of a Docusign server template.")

- Recipient Privileges:
  - Allow recipients to sign on paper: Give
    recipients the option to sign manually
  - Allow recipients to change signing responsibility or assign a
    delegated signer: Recipients can change their status
    (Needs to Sign, Receives a Copy) or choose another person to sign for
    them
- Password: Create a password that senders must enter

## Add documents

![A screenshot displays the three-dot menu with Set as Supplement, Replace, and other options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/A-screenshot-displays-the-threedot-menu-with-Set-as-Supplement-Replace-and-other-options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY5NTUsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.J6Ty7bN2bTgawBcrWcvFGrZrW9gFusedHKEfz2OSabk&_LANG=enus "A screenshot displays the three-dot menu with Set as Supplement, Replace, and other options.")

- Upload: Add documents from your device or Box, Dropbox,
  Google Drive, and OneDrive
- More options:
  - Set as Supplement: Set a document as an
    attachment that does not require signing
  - Replace: Replace a document with one you upload
    from your device

## Add recipients

![A screenshot displays the Add Recipients section of a Docusign server template with Advanced Settings highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Server-template-Customize--Advanced-settings.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY5NTUsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.J6Ty7bN2bTgawBcrWcvFGrZrW9gFusedHKEfz2OSabk&_LANG=enus "A screenshot displays the Add Recipients section of a Docusign server template with Advanced Settings highlighted.")

- Add From Contacts: Add contacts from your Docusign account to a template.
- Advanced Settings:
  - Draw a new signature for each signature or initial
    field: Recipients draw their signature or initials with
    a cursor
  - Don’t allow senders to edit recipient: Senders
    cannot change recipient data
  - Don’t allow senders to delete recipient: Senders
    cannot remove recipients from an envelope

## Add identity verification

![A screenshot displays the Add Recipients section of a Docusign server template with Add Identity Verification highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Server-template-Customize--Add-identity-verification.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY5NTUsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.J6Ty7bN2bTgawBcrWcvFGrZrW9gFusedHKEfz2OSabk&_LANG=enus "A screenshot displays the Add Recipients section of a Docusign server template with Add Identity Verification highlighted.")

- Phone Call: The recipient receives an access code by phone call. A
  fee is charged per usage.
- Allow recipient to provide phone number: The recipient
  can add their own phone number.
- Knowledge-based (KBA): The recipient answers a set of questions based
  on their public records. A fee is charged per usage.
- Docusign ID Verification: The signer identifies themselves with a
  valid government ID.

## Add message

![A screenshot displays a Docusign server template with custom email and language highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/A-screenshot-displays-the-Add-Message-section-of-a-Docusign-server-template.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY5NTUsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.J6Ty7bN2bTgawBcrWcvFGrZrW9gFusedHKEfz2OSabk&_LANG=enus "A screenshot displays a Docusign server template with custom email and language highlighted.")

- Custom email and language for each recipient: Send the
  email message and attached documents in the recipient's language
- Email Subject: Add a custom subject in the recipient's
  language
- Email Message: Add a custom message in the recipient's
  language

## Unsupported server template features

These options cannot be added to an envelope template from a server template:

- Template Usage: Change the options that are available to
  a sender
- Signs with Witness: A witness is present for the signing
  ceremony
- Sign with Notary: Require Notary approval for the signed document
- Send without fields: An envelope is sent with no
  signature fields
- Adding Access Code and SMS
  authentication to the same recipient
- Template Modification:
  - Don't allow senders to edit, add, or remove
    recipients
  - Don't allow senders to edit the subject, email, or private
    messages

## Unsupported envelope template features

These options cannot be added to server templates from envelope templates:

- Sending Experience: Change the options that are
  available to a sender
- Document Writeback:
  - Save completed documents back to the Salesforce
    Record: Envelope documents are added to a Salesforce
    record when the signature process is complete
  - Save certificate of completion (COC) document back to the Salesforce
    Record: Docusign adds a
    certificate of completion to a Salesforce record when the signature
    is complete
- Field Updates: Envelope events trigger updates to Salesforce
  fields
