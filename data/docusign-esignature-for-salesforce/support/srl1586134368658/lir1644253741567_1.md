# When I Upload a File to an Envelope Template, I See an Error: "This File Exceeds the Maximum Size Limit"

Learn how to work around the file size limit error in an envelope template.

When I try to upload a file to the envelope template Documents step, I
sometimes see an error. This does not happen with every file.

Note: This error displays when administrators try to upload files larger
than 3 MB to an envelope template. However, you can access the same envelope
templates directly in your connected Docusign account. Templates
in your Docusign account have a file size limit of **50
MB**.

**Steps to Reproduce**:

1. I upload a file to an envelope template at the Documents step. The document fails to
   load.
2. An error displays: This file exceeds the maximum size
   limit.

   Result: The file is not added
   to the envelope template.

## Resolution 1: Upload the file to the envelope template in your connected Docusign account

1. Select the Save and Close button for the
   envelope template in your Salesforce organization.
2. Log in to your Docusign account on another browser tab.
3. Select Templates.
4. Select the envelope template's name and choose
   Edit.

   A page with the template settings opens.
5. Select Upload and upload the file from your
   device.
6. Select SAVE AND CLOSE.

   ![A screenshot displays the Save and Close button on a Docusign template.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Save-and-close-changes-to-your-DocuSign-template.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg3NDMsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.kUUApuFYosn6RTgYAHpSH4KJUx_sAfMs3TdvADTkFIk&_LANG=enus "A screenshot displays the Save and Close button on a Docusign template.")

   You have completed the process. You have avoided the Salesforce file size
   limitation by uploading the file to your Docusign
   template. This document is included in all envelopes that are sent from
   a Salesforce record using this template.

   Important: If you later edit this envelope template in
   Salesforce, you will get an error and will be unable to make
   additional changes. You must edit the template in your Docusign account.

## Resolution 2: Upload the file to Salesforce Files Home or Notes and Attachments

1. Upload the file to your Salesforce Files Home, or
   the record's Notes and Attachments section.
2. Complete the envelope template creation process.
3. Select the envelope template custom button from a Salesforce
   record.
4. Select the Add from Salesforce button. The Add
   from Salesforce dialog box opens.
5. Check the box for the file saved in Notes and Attachments.
6. Select the Add button.
7. Complete the envelope sending process.

   The process is complete. Docusign attaches the large file from
   Notes and Attachments during the sending process.
