# Install and Configure the Docusign Connected App

Learn to install and configure the Docusign connected app for your Salesforce organization.

Important: You are not required to make any changes if you have taken the
following steps:

- Installed the Docusign connected app
- Set the OAuth Policy to Admin-approved users are
  pre-authorized

Starting in September 2025, Salesforce began enforcing [restrictions for uninstalled connected apps](https://help.salesforce.com/s/articleView?id=005132365&type=1). These
restrictions impact some organizations using Docusign Apps
Launcher for Salesforce. If you have not installed the Docusign
connected app and assigned permissions, your users cannot access Docusign apps.

Use this procedure to install the Docusign connected app and grant
access to your users:

1. Access the Connected Apps OAuth Usage page as a Salesforce
   administrator:
   1. Select the gear icon ![A screenshot displays a six-point gear with a lightning bolt inside it.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/salesforce-gear_.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY3NzgsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.Cl_DOfJW3jqgbtHnwzbXFaQ1VS3zpIX-CmlvsjpoEsU&_LANG=enus "A screenshot displays a six-point gear with a lightning bolt inside it.") and choose Setup.
   2. Enter and select Connected Apps OAuth Usage in the
      Quick Find field.

      The Connected Apps OAuth Usage page opens.
2. Install the Docusign connected app:
   1. Docusign: Select
      Install.

      The Install connected app? dialog box
      opens.
   2. Select Install.![A screenshot displays the install button highlighted on the Docusign connected app OAuth Usage page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/A_screenshot_displays_the_install_button_highlighted_on_the_Docusign_connected_app_OAuth_Usage_page.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY3NzgsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.Cl_DOfJW3jqgbtHnwzbXFaQ1VS3zpIX-CmlvsjpoEsU&_LANG=enus "A screenshot displays the install button highlighted on the Docusign connected app OAuth Usage page.")

      The Docusign Connected App Detail page
      displays.
3. Edit the connected app policies:

   Note: Salesforce and Docusign recommend
   selecting Admin-approved users are pre-authorized.
   Each user must enter credentials to gain Docusign for
   Salesforce app access if you select All users may
   self-authorize.

   1. Select Edit Policies.

      The Connected App Edit page displays.
   2. Permitted Users: Select
      Admin-approved users are
      pre-authorized.![A screenshot displays the permitted users field set to Admin approved users are pre-authorized.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/A_screenshot_displays_the_permitted_users_field_set_to_Admin_approved_users_are_pre-authorized.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY3NzgsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.Cl_DOfJW3jqgbtHnwzbXFaQ1VS3zpIX-CmlvsjpoEsU&_LANG=enus "A screenshot displays the permitted users field set to Admin approved users are pre-authorized.")

      A dialog box opens: Enabling this option will result
      in all users currently using this app being denied access. Please
      reference the Connected Apps OAuth Usage Report if you are unsure
      who is using the app.
   3. Select OK.

      The dialog box closes.
   4. Select Save.

      The Docusign Connected App Detail page
      displays.
4. Grant Apps Launcher access to Salesforce users:
   1. Select Manage Profiles or Manage
      Permission Sets at the bottom of the Connected App
      Detail page.![A screenshot displays the OAuth Usage page with the Manage Profiles and Manage Permission Sets buttons highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/A_screenshot_displays_the_OAuth_Usage_page_with_the_Manage_Profiles_and_Manage_Permission_Sets_buttons_highlighted_.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY3NzgsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.Cl_DOfJW3jqgbtHnwzbXFaQ1VS3zpIX-CmlvsjpoEsU&_LANG=enus "A screenshot displays the OAuth Usage page with the Manage Profiles and Manage Permission Sets buttons highlighted.")
   2. Select the check box for each profile or permission set you want to
      include.
   3. Select Save.

   The Docusign Connected App Detail page
   displays.
5. If your organizations has API Access Control configured, take the following actions:
   1. Enable Allow Visualforce pages to access APIs.
   2. Go to Setup > Permission Sets > your Docusign permission set > Assigned Connected Apps and assign at least one permission set to the Docusign connected app.![A screenshot displays the API Access Control Docusign permission set page with the Assigned Connected Apps link highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/API_Access_Control_Assigned_Connected_Apps.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY3NzgsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.Cl_DOfJW3jqgbtHnwzbXFaQ1VS3zpIX-CmlvsjpoEsU&_LANG=enus "A screenshot displays the API Access Control Docusign permission set page with the Assigned Connected Apps link highlighted.")

You have completed the process. You have installed the Docusign connected
app and assigned permissions to your users. Users can access Docusign
apps in your Salesforce organization. It you selected All users may
self-authorize, your users must [re-authenticate to use Docusign apps](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=ymi1586134213877.html).
