# Step 3: Add a Name, Data Source, and Experience Cloud Option

Use this procedure to add a name, data source, and enable the Experience Cloud
option.

Use this procedure to add a name, data source, and enable
the Experience Cloud option.

1. Access the Docusign eSignature Configuration page as a
   Salesforce administrator:
   1. Select the App Launcher grid ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/applaunchergrid.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY4NTUsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.uSVbb78jT764n0zKNP4nVEK0QC9LhFU5SSGZiqj_-yE&_LANG=enus) and
      select View All.
   2. Select the Docusign Apps Launcher tile.
   3. Select eSignature in the left column.
2. Select Create Template at the right of the page.

   The Create Template dialog box displays.
3. Complete all steps in the Create Template dialog box:
   1. Name: Enter a descriptive template name.
   2. Select Data Source: Select a Salesforce object from the
      dropdown menu.
   3. Select the Enable Experience Cloud Template check
      box.
   4. Select the Next button.

   ![A screenshot displays the Create Template dialog box with the Enable Community Cloud Template box checked and highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Community-Cloud-envelope-template.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY4NTUsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.uSVbb78jT764n0zKNP4nVEK0QC9LhFU5SSGZiqj_-yE&_LANG=enus "A screenshot displays the Create Template dialog box with the Enable Community Cloud Template box checked and highlighted.")

   Note: The check box is named
   Enable Community Cloud Template (Embedded Signing) in
   older versions of the Docusign eSignature for Salesforce
   app.
