# Step 7: Create a Temporary Envelope Template Custom Button on the Account Layout

The seventh step of the process involves temporarily adding a custom button to the
Salesforce Account layout.

Use this procedure to temporarily add a custom button to the Salesforce
Account layout.

Tip:
[Here is a video that walks you through the process](https://players.brightcove.net/6118377982001/default_default/index.html?videoId=6172186652001).

Note:

- You cannot add a custom button to the Person Account layout on this
  step
- You must add a custom button to at least one Account layout to save
  and finish your envelope template
- You must manually remove the button from your Account layout
- You will then add it to the Person Account layout

1. Enter a descriptive name for the custom button in the Button
   Label text field.
2. Check the box for one Account layout.

   Note: Only select one check box so that you only have to remove the button from
   one layout.
3. Select Save and Finish.

   A message displays: Successfully modified
   layouts.

You have completed step seven. Move to [Step 8: Remove the Person Account Custom Button from the Accounts Layout](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=jte1636651796525.html&_LANG=enus "The eighth step of the process involves removing the custom button from the Salesforce Account layout.").
