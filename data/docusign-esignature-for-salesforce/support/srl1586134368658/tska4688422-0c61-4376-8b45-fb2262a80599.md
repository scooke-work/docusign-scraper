# Step 2: Prepare Your Salesforce Users for Experience Cloud Envelope Templates

Learn to prepare your Salesforce users for Experience Cloud envelope template
use.

Use this procedure to prepare your Salesforce
users for Experience Cloud envelope templates:

1. Open the Docusign Sender permission set:
   1. Select the gear icon ![A screenshot displays the Salesforce Setup gear icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY4NTAsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.h1wUeSqb2LaLtQ5q978ug0AFtjDkq3wvMWgfFUz43Go&_LANG=enus "A screenshot displays the Salesforce Setup gear icon.") and choose
      Setup.
   2. Enter and select Permission Sets in the Quick Find
      field.

      The Permission Sets page displays.
   3. Permission Set Name: Select the
      Docusign Sender link.

      The Docusign Sender page
      displays.
2. Add Experience Cloud users to the Docusign Sender permission
   set:
   1. Select Manage Assignments.

      The Current Assignments page displays.
   2. Select Add Assignment.

      The Select Users to Assign page displays.
   3. Select the check box next to each user that you want to add.

      Note: You cannot add Experience Cloud
      Guest users.
   4. Select Next.

      The Select an Expiration Option For Assigned Users page
      displays.
   5. Select Assign.

      The Docusign Sender page displays with
      selected users added.
   6. Select Done.

   The Sharing Settings page refreshes, displaying your
   changes.
3. Turn on the API Enabled permission for your Experience
   Cloud user profiles:

   Note: If you are using a default
   profile, clone it first so that you can edit the
   permissions.

   1. Enter and select Profiles in the Quick Find
      field.

      The Profiles page displays.
   2. Profile Name: Select Edit
      next to the profile you want to update.

      The Profile Edit page displays.
   3. ![A screenshot displays the Administrative Permissions section of a Profile page with the API Enabled box checked and highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/DAL_Experience_Cloud_envelope_template_prepare_users.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY4NTAsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.h1wUeSqb2LaLtQ5q978ug0AFtjDkq3wvMWgfFUz43Go&_LANG=enus "A screenshot displays the Administrative Permissions section of a Profile page with the API Enabled box checked and highlighted.")Administrative Permissions: Select the check box
      next to API Enabled.
   4. Select Save.

      The Profiles page displays.
