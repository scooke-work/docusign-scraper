# Preview a Document Uploaded to an Envelope Template

You can preview uploaded documents in the envelope template by following these
steps.

You can preview uploaded documents in the envelope template by following these steps:

1. Select from the arrow ![A screenshot of a downward-facing triangle.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY5ODEsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.RADSaUBLSvkzvJbISiUnmfZa_eESIcD8hQBOjaLLENM&_LANG=enus "A screenshot of a downward-facing triangle.") dropdown menu next to a document.
2. Select Preview from the options.

   ![A screenshot displays the document options menu for an envelope template, highlighting the Preview option for an uploaded document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/envelopetemplatedocumentoptions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY5ODEsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.RADSaUBLSvkzvJbISiUnmfZa_eESIcD8hQBOjaLLENM&_LANG=enus "A screenshot displays the document options menu for an envelope template, highlighting the Preview option for an uploaded document.")

   A new tab opens with the document preview page.
3. Select the document pane to open the preview in full-screen view.
4. Select the X in the upper-right of the page to close the
   preview.

   You have completed the process. You have previewed the uploaded
   document.
