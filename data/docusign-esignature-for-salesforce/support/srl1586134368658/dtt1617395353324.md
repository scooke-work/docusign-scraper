# Step 8: Add Options to Your Experience Cloud Envelope Template

Learn to add options like Delivered by to your Experience Cloud envelope
template.

Use this procedure to enter a Delivered by user, as well as add
options to your Envelope Template:

1. Delivered by: Enter a Salesforce user who also has
   administrator status in the connected Docusign account.
   Messages sent from this template will display the Delivered by user as the
   sender.

   Important: This user must also be in the
   Salesforce System Administrator profile or permission set.

   ![A screenshot displays the Delivered By option selected in the Set Delivery Email field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Screen-Shot-20210404-at-30925-AM.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY4NzIsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.Gly4C01mCmqykY46q0q3f5ES3AbZHbqSVek3-5q0APo&_LANG=enus "A screenshot displays the Delivered By option selected in the Set Delivery Email field.")
2. (Optional): Configure any of the following
   options:
   - Reminders: Add a reminder to your recipients from
     the dropdown menu. Select Custom from the dropdown
     menu to add a custom reminder of up to 999 days.
   - Expiration: Set a date or a number of days (up to
     999) after which the envelope expires.
   - Document Writeback: Select the check box for any option:
     - Save completed documents back to the Opportunity
       Record.
     - Save certificate of completion (COC) document back to the
       Opportunity Record.

     Important: Do
     not turn on Multi-Factor Authentication for API
     Logins. Writeback for documents, merge fields, and data
     fails when you use this option.
   - Field Updates: Automatically update Salesforce
     fields when envelope events occur:
     1. Select the + Add Update button.

        A row of
        fields displays.
     2. Envelope Event: Select an event to trigger a
        Salesforce field update, like Envelope
        Completed.
     3. Field Name: Enter or select the Salesforce
        field name, like Amount, that updates when
        the envelope event occurs.
     4. Field Update: Enter or select a value that
        updates the Salesforce field name.![A screenshot displays the Field Updates page, with events created for Envelope Completed and Envelope Voided.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Field-Updates-events.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY4NzIsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.Gly4C01mCmqykY46q0q3f5ES3AbZHbqSVek3-5q0APo&_LANG=enus "A screenshot displays the Field Updates page, with events created for Envelope Completed and Envelope Voided.")

        Note:
        - You can add 5 envelope events for each type (like Envelope Completed)
          with a maximum of 25 events.
        - Envelope events of the same type cannot have the same
          field name. For example, two Envelope
          Completed events cannot use
          Stage as the field name.
3. Select Next.
