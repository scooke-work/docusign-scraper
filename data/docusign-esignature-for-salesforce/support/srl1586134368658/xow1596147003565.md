# Edit an Envelope Template

Learn to edit envelope templates.

Use this procedure to edit an envelope template:

1. Select a tab from which to edit details:

   - Documents
   - Recipients
   - Place Fields
   - Options
   - Custom Button
2. Select Save and Close when you have completed your
   changes.

You have completed the process. You have edited the
template.
