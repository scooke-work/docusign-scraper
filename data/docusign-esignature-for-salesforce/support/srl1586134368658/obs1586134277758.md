# Reset the Password for a Docusign for Salesforce User

Learn to reset the password of a Docusign CLM, Gen, Negotiate, or eSignature for
Salesforce user.

Use this procedure to reset the password of a Docusign CLM, Gen, Negotiate, or eSignature
for Salesforce user:

1. Access the Docusign
   Users page as an administrator:
   1. Log on to your Docusign account.
   2. Select Settings at the top of the page.
   3. Select Users in the left column.
2. Select Actions next to a user and choose Reset
   Password from the dropdown menu.

   A message opens: A reset password email has been sent to (user
   name).

You have completed the process. You sent a password
reset link to the user's email address.
