# Compare the Differences Between Envelope Template Types

Learn about the differences between the different envelope template types.

Sending Experience is a feature on the envelope template
Options page. You can use it to switch the options a sender
sees when creating an envelope.

You can use the following Sending Experience types:

- Default: Senders can add documents, specify
  recipients, and place fields before sending
- Quick Send: Senders can only add non-signing
  recipients
- Place Fields: Senders start at the place fields step, skipping
  the documents and recipients steps.
- Send Now: Senders do not see any controls. The envelope
  sends immediately.
