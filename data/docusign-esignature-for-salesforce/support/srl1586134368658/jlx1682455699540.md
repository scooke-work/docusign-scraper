# Learn How to Use Envelope Custom Fields in Salesforce

Learn how to use envelope custom fields in Salesforce envelopes.

## What is a Docusign envelope?

A Docusign envelope is a digital container that you build that holds
documents, recipients, and signature fields. Recipients sign the envelope by
completing the fields assigned to them. Docusign marks the
envelope as Signed when all recipients complete their
fields.

Salesforce users can create and send envelopes from Salesforce records. Users can add
recipients and documents manually, from a template, or dynamically from Salesforce
records, users, or contacts. Docusign saves the document to the
Salesforce record when all recipients finish signing.

## What is an envelope custom field?

- Envelope custom fields (ECFs) let you tag and identify the envelopes you send for signature.
  Administrators create these fields, and senders enter values when preparing an
  envelope.
- Values are not visible to envelope recipients when signing. Only the sender, users with
  shared access, and account administrators see envelope custom fields and values.
- You can build envelope custom fields such as text fields or lists of values. You can mark
  ECFs as required or optional.
- Only the sender, users with shared
  access, and account administrators see
  envelope custom fields and values in the envelope details. Recipients do not see
  envelope custom fields and values.
- When senders prepare an envelope for signature, they must enter or select field values for
  all required ECFs. Senders can only advance to the next step if they complete
  all the required fields.
- You can enter up to 100 characters in an ECF text field. The values are visible to the
  sender and any account member with shared access.
- Users can enter ECF values from the Manage page to search for envelopes. Users can also
  generate an envelope report that includes ECF values.

## How do I set default values for ECFs?

You can customize envelope custom fields in your Docusign account. One
customization you can make is assigning default values to envelope custom fields.
Default values are data that you pre-populate for senders.

Set default field values and log into your Docusign account as an
administrator. Go to Settings > Templates >  (template name) >  Edit and add values for envelope custom fields.

Adding default values makes the envelope creation process more straightforward and
less error-prone for your users.

**Example**: Most of Fontara's customers operate in the United States. Fontara
administrators set the default value of the Country field to
United States and Currency to
Dollar. When a sender creates an envelope,
United States displays in the Country field, and
Dollar displays in the Currency field. Senders can change
the Country and Currency values if needed.

In this image, the Country and Currency
fields have default values. The Account Name field has no
default value.

## How do envelope custom fields work in Salesforce?

You cannot create new envelope custom fields in Salesforce. You can only create ECFs in your
Docusign account. When you have created envelope custom
fields, Salesforce senders see ECFs on the Recipients page.

![A screenshot of the envelope Recipients page displays the Envelope Custom Fields section with four envelope custom fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Envelope-custom-fields-on-the-Recipients-page.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg4NjAsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.xwN_JHrDRRYTHl2vjfClaP9sJnlkZb5stjGbdLFvI5w&_LANG=enus "A screenshot of the envelope Recipients page displays the Envelope Custom Fields section with four envelope custom fields.")

The following rules apply to envelope custom fields in Salesforce:

- Only envelopes sent from standard Send with Docusign
  buttons display envelope custom fields. Envelopes created from custom envelope
  template buttons do not display ECFs.
- Only envelope custom fields with Show field to envelope creators
  enabled display on the Recipients page.
- The Recipients page displays all envelope custom fields
  with Show field to envelope creators enabled.
- All "shown" ECFs from your Docusign account display on the envelope
  Recipients page.
- Senders must enter values for all required fields to advance to the Prepare &
  Send page.
- Senders can add envelope templates to their envelopes on the Documents
  page. Adding templates imports template documents, recipients, and signature
  fields to the envelope.

## How do default values work in Salesforce envelopes?

When senders select multiple envelope templates, default values only display for the
last template in the list. So if a sender selects three templates, only the third
template's fields display default values. Fields from the first two templates are
blank.

In this image, default values from the Tally NDA template will display
for the sender. Envelope custom fields from the first two templates will be blank:
![A screenshot of the Select Docusign Template dialog box displays three envelope templates, each with a selected check box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/A-screenshot-of-the-Select-Docusign-Template-dialog-box-displays-three-envelope-templates-each-with-a-selected-check-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg4NjAsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.xwN_JHrDRRYTHl2vjfClaP9sJnlkZb5stjGbdLFvI5w&_LANG=enus "A screenshot of the Select Docusign Template dialog box displays three envelope templates, each with a selected check box.")

Senders not adding templates see all ECFs from the Docusign account
displayed with blank values.

**Example**: Your Docusign account has 11 envelope custom fields with
Show field to envelope creators enabled. You select a
standard Send with Docusign button on a Salesforce record.
You do not add envelope templates on the Documents page. You
advance to the Recipients page and see all 11 envelope custom
fields from your Docusign account. All 11 fields are blank. You
can enter values for any of the 11 fields.

## How do I create envelopes with envelope custom fields?

There are two ways to create Salesforce envelopes with envelope custom fields:

- Add envelope templates:
  1. Select a Send with Docusign button on a
     Salesforce record.
  2. Select the Select Template button on the
     Documents page.
  3. Select each template that you want to add from the Select Docusign
     Template dialog box.
  4. Advance to the Recipients page.
     - All ECFs marked with Show field to envelope
       creators display
     - Fields from the *last* of the selected templates display
       default values
     - Fields from all other templates are blank.
  5. Enter values for all the required fields.
- Do not add envelope templates:
  1. Select a Send with Docusign button on a
     Salesforce record.
  2. Add documents on the Documents page.
  3. Advance to the Recipients page.

     All ECFs marked
     with Show field to envelope creators display.
     All values are blank.
  4. Enter values for all the required fields.
