# Step 2: Add Documents to Your Template

Learn to add documents to your envelope template.

Tip:

- [This video demonstrates the steps in this task](https://players.brightcove.net/6118377982001/default_default/index.html?videoId=6171919055001).
- [This topic demonstrates the advanced options available
  in this step](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=dad1595995063747.html&_LANG=enus).

In this step, you can add documents to your template. Word and PDF files are most commonly
used. These documents provide the text for the agreements that your users create
with this template.

You are not required to upload a document from your device at this step. You can select
Attach documents from Salesforce record to add documents
from a Salesforce record. However, if a record does not contain matching Salesforce
documents, users cannot create envelopes.

Use this procedure to add documents to your envelope template:

1. (Optional) Upload files from your device: Choose a method to add documents. You
   can add up to ten documents.
   1. Select Upload Document.
   2. Follow the prompts to add content from your device.
   3. Select Done.
2. (Optional) Upload documents from Salesforce:
   1. Select the Attach documents from Salesforce
      record check box.
   2. Choose an option from the dropdown menu:

      - Latest Document: Upload the latest
        document from the record.
      - File Name Contains: Upload documents from
        the record that match the file name keywords you enter.
      - All Documents: Upload all documents from
        the record.
3. (Optional) Drag documents to change their order:![A screenshot displays a document moving up in the order of uploaded files.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/DAL_Envelope_Template_drag_documents.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY4ODYsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.8XKB68Z1vv3ZOfN9r9gvkyX0J_OaBOS45dJ7zs9VFSE&_LANG=enus "A screenshot displays a document moving up in the order of uploaded files.")
4. Select Next.
