# Change the Recipient Signing Order in an Envelope Template

You can change the signing order of recipients in an envelope template. This lets you
change the order in which the envelope must be signed for the signature process to complete.

You can change the signing order of recipients in an envelope template. This lets you change the order in which the envelope must be signed for the signature
process to complete. You can drag recipients to a new order, or change their recipient numbers (1, 2, 3,
etc.) Envelope templates can also be configured to have recipients sign at the same time, or in no
particular order.

Use this procedure to change the recipient signing order in an existing envelope template:

Note: You can also use this procedure on the Recipients step when sending an
envelope.

1. Select the Signing Order button.
2. Pick an option:
   - **Drag recipients**: Select a recipient and drag it up or down in the list.
     ![A screenshot displays the envelope template recipients list with drag handles, showing how to change the signing order by dragging recipients up or down.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/envelopetemplatechangesigningorder.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcwMzAsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.R9BhEdi8z3d7CR0WZRTzoPZrbO2oSft_PH7mZBnVH0c&_LANG=enus "A screenshot displays the envelope template recipients list with drag handles, showing how to change the signing order by dragging recipients up or down.")
   - **Change recipient number**: Enter a new number in the field to the left of the recipient.
     ![A screen capture displays the Signing Order options dialog with settings for recipient signing order, including options to sign in order, in parallel, or with no specific order.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Signing-Order-options.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcwMzAsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.R9BhEdi8z3d7CR0WZRTzoPZrbO2oSft_PH7mZBnVH0c&_LANG=enus "A screen capture displays the Signing Order options dialog with settings for recipient signing order, including options to sign in order, in parallel, or with no specific order.")
3. Select the Save and Close button at the top-right of the
   page.

You have completed the process. You have changed the recipient order.
