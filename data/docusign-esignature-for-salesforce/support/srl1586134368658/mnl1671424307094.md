# Use Merge Fields To Customize the Email Subject and Message of an Envelope

Learn how to use merge field syntax to dynamically fill the Email
Subject and Email Message fields of a Docusign envelope.

You can use merge field syntax to dynamically fill the Email
Subject and Email Message fields of a
Docusign envelope. Merge field syntax can add this information from standard and
custom Salesforce objects.

Using merge field syntax is helpful if you want Docusign to fill email fields with Salesforce
record names or descriptions.

**Example**: You create an envelope template for the Opportunity
object. When users create envelopes, you want Docusign to add the record's description in the
Email Subject field.

You enter `{!Opportunity.Description}` in the Email
Subject field of your envelope template.

When a user creates an envelope, the Opportunity Description of that record is added
to the Email Subject.

Use this procedure to add merge field syntax to the Email
Subject and Email Message fields of an
envelope template:

1. Add merge field syntax to any field:
   - Email Subject: Add merge field syntax to
     dynamically add content from a record, like
     `{!Opportunity.Description}`.
   - Email Message: Add merge field syntax to
     dynamically add content from a record to the email message of an envelope.
     ![A screenshot displays merge field syntax used in the Email Subject and Email Message fields](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Merge-field-syntax-used-in-the-Email-Subject-and-Email-Message-fields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcwNjAsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.qTIEtTK3W8IzUfirXWyai6euVILPnaPpN46a9Bud3ho&_LANG=enus "A screenshot displays merge field syntax used in the Email Subject and Email Message fields")
2. Select Save and Close.

You have completed the process. You have added merge field
syntax to the Email Subject or Email
Message fields of your envelope template. When users create envelopes
from this template, Docusign dynamically adds this information from the Salesforce
record.
