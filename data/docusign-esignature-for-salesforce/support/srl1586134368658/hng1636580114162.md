# Create an Envelope Template Using the Salesforce Person Account Object

Learn how to build an envelope template using the Salesforce Person Account
object.

Person accounts are Salesforce objects that store information
about people by combining certain account and contact fields into one record.

The Person Account object is available in these Salesforce
editions:

- Professional
- Enterprise
- Performance
- Unlimited
- Developer

Note: You must enable Person accounts before you can use them.
Visit the Salesforce help site to learn how to enable Person accounts.

You can build a Docusign envelope template using the Salesforce Person
Account object.

Person Account is a part of the Salesforce
Account object.

When you select a Salesforce object at the start of the envelope template creation,
Person Account is not listed.

To build a Person Account envelope template, you must select Account as the data
source.
