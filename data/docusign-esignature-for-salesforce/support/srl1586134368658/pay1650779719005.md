# Supported Devices, Operating Systems, and Browsers for Docusign Apps Launcher

Learn the Docusign Apps Launcher requirements for browsers, operating
systems, and devices.

You can use the Docusign for Salesforce apps with the following browsers,
operating systems, and devices:

## Supported browsers

Important: Salesforce and Docusign do not
support any version of Microsoft Internet Explorer.

| Browser | Supported version | Notes |
| --- | --- | --- |
| Google Chrome | Latest | n/a |
| Safari (Apple) | Latest | Desktop-only. Salesforce does not support the mobile version of Safari. Enabling third-party cookies is required. |
| Microsoft Edge | Latest | n/a |
| Mozilla Firefox | Latest | n/a |

## Supported operating systems

| Operating system | Version |
| --- | --- |
| Microsoft Windows | Windows 10 or newer |
| MacOS (Apple desktop) | 10.14 (Mojave) or newer |
| iOS (Apple mobile) | iOS 14.x or newer |
| Android OS | Android 8.0 or newer |

## Salesforce mobile app requirements

Required settings: Configure the following settings to use Docusign in the Salesforce mobile app:

- Enable third-party cookies in the [Salesforce mobile app settings](https://help.salesforce.com/s/articleView?id=sf.salesforce_app.htm&type=5).
- Turn off Include the instance name in Visualforce URLs when
  third-party cookies are blocked in Salesforce My
  Domain settings.

Approved Docusign apps: You can use the following Docusign apps with the Salesforce mobile app:

- Docusign eSignature for Salesforce
- Docusign Gen

Non-approved Docusign apps: You cannot use the following Docusign apps with the Salesforce mobile app:

- Docusign Negotiate
- Docusign CLM
- Docusign CLM Essentials

Approved envelope template types: You can only use the following
envelope template types in the Salesforce mobile app:

- [Quick send envelope template](https://support.docusign.com/s/document-item?language=en_US&bundleId=srl1586134368658&topicId=gou1591745811909.html&_LANG=enus)
- [Send now envelope template](https://support.docusign.com/s/document-item?language=en_US&bundleId=srl1586134368658&topicId=gou1591745811909.html&_LANG=enus)

Mobile app tagging requirements: Meet the following requirements
to add fields to Docusign envelopes:

- Install the Docusign mobile app on your Android or iOS
  device.
- Turn off mobile app tagging when using an iPad.

For more information about Salesforce requirements, see the [Supported Browsers and Devices](https://help.salesforce.com/s/articleView?id=sf.getstart_browser_overview.htm&type=5) page on the
Salesforce support site.
