# Delete Merge Fields From an Envelope Template

You can delete a merge field in an existing envelope template at any time.

You can delete a merge field in an envelope template at any
time.

Use this procedure to delete merge fields in an envelope
template:

1. Select from the arrow ![A screenshot of a downward-facing triangle.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcxMDcsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.Fz7nu0aZmhbQCyQekx7X3XIa8PXpiRwI3ATVUfOJ2pI&_LANG=enus "A screenshot of a downward-facing triangle.") dropdown menu next to a merge field
   name.
2. Select Delete.

   The Are you sure you want to delete this merge field?
   dialog box opens.
3. Select Delete.

   A message displays: Merge field deleted. The dialog box closes.

You have completed the process. You have deleted a merge field.
