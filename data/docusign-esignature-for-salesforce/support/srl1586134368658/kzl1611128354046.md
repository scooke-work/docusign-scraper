# Edit Merge Fields From an Envelope Template

You can edit the details of merge fields in an envelope template at any
time.

You can edit the details of merge fields in an envelope template
at any time.

Use this procedure to edit merge fields in an envelope template.

1. Select the name of a merge field under the Name header.
   The Edit Merge Field dialog box opens.
2. Edit any merge field options.
3. Select the Save button. The Edit Merge Field dialog box
   closes. A message displays: Merge field updated.
4. Select the Save and Close button at the top-right of the
   page.

   You have completed the process. You have edited a merge field.
