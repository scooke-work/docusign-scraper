# Recipient Preview Actions: Viewing As and Form Factor View

You can use the Actions button on the Place Fields page to access a menu of Docusign and
Salesforce options.

You can use the Actions button on the Place Fields page to access a menu
of Docusign and Salesforce options.

Table 1. Recipient Preview Actions

| Button | Name | Action |
| --- | --- | --- |
|  | Viewing As | Select from dropdown menu to preview as Owner or Recipient |
|  | Desktop View | Preview document as it would appear on a computer |
|  | Tablet View | Preview document as it would appear on a tablet |
|  | Mobile View | Preview document as it would appear on a mobile device |
|  | Close | Close preview screen and return to Place Fields page |
