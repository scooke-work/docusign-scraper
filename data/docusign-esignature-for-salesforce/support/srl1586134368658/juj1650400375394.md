# Add a Custom Signing Button With URLFOR Scripting to a Salesforce Layout

Learn how to user custom URLFOR scripting to add a custom button to a Salesforce layout.

You can use Docusign URLFOR scripting to create signing buttons without building an envelope
template. URLFOR buttons also let you build signing experiences not available in the
Docusign envelope template user interface. You can use [pre-built scripts provided by Docusign, or create
your own](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=nxl1649872087756.html&_LANG=enus "Learn how to use URLFOR scripting to add custom Docusign eSignature for Salesforce buttons to object layouts."), using Docusign parameters. You can then add a custom button
with Docusign URLFOR scripting to a Salesforce layout. Users who select this button
on a record are guided through your custom signing experience.

Use this procedure to build a URLFOR custom button and add it to a layout:

1. Select New Button or Link.

   The New Button or Link page opens.
2. Complete the required steps on the page:
   1. Label: Enter a name for your custom button.
   2. Name: Select the field and accept the default name that
      populates.
   3. Display Type: Select the Detail Page
      Button radio button.
   4. Behavior: Keep the default or select another option from
      the dropdown menu.
   5. Content Source: Keep the default selection of
      URL.
   6. Paste [custom Docusign
      URLFOR script](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=nxl1649872087756.html&_LANG=enus "Learn how to use URLFOR scripting to add custom Docusign eSignature for Salesforce buttons to object layouts.") in the field under the Select Field
      Type dropdown menu.![A screenshot displays configuring a URLFOR button in the New Button or Link settings.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Configure-URLFOR-New-Button-or-Link-settings.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg2MjEsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.-v55C0fsaenSbhmK7c2HOYysrfIrwwp2SFS8LIKzp0Q&_LANG=enus "A screenshot displays configuring a URLFOR button in the New Button or Link settings.")
   7. Select Save.

      A message opens: The new custom button will not be displayed
      to users until you add it to the page layout.
   8. Select OK.

   The page refreshes, showing the new button details.
3. Select the Page Layouts link in the left column.

   The Page Layouts page opens.
4. Select the link to the layout you want to update.

   The object layout page opens.
5. Add your URLFOR custom button to the layout:
   1. Select Buttons in the left column of the layout
      menu.
   2. Drag the URLFOR custom button from the layout menu to the Custom
      Buttons section of the page layout.

      ![A screenshot displays dragging a URLFOR button to the Custom Buttons section.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Drag-URLFOR-button-to-Custom-Buttons-section.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg2MjEsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.-v55C0fsaenSbhmK7c2HOYysrfIrwwp2SFS8LIKzp0Q&_LANG=enus "A screenshot displays dragging a URLFOR button to the Custom Buttons section.")
   3. Select Save.

   The Saving Layout message opens. The page
   refreshes.

You have completed the process. You have created a custom
button with Docusign URLFOR scripting and added it to a Salesforce layout. Users who
select this button on a record are guided through your custom signing experience.
