# Clone Envelope Templates in Salesforce

Learn how to clone one or more envelope templates for use in the same or another
organization.

## Before you begin

- Required: Salesforce and Docusign administrator status.
- Required: Install Docusign Apps Launcher for Salesforce v4.5 or
  higher.

You can clone a Docusign envelope template from one Salesforce organization
and use it in another organization. You can also clone templates in the same
Salesforce organization to address similar use cases. Cloning helps administrators
quickly build templates, without manually recreating them.

Important: You must take extra steps when uploading cloned
templates to Salesforce environments with different cloning envelope templates with
custom fields. Custom fields are tied to the user who created them. If the creating
user is not present in the new organization, imported custom fields fail. Add the
creating user to the new organization or manually rebuild the custom fields.

Use this procedure to clone a Docusign envelope template:

   Salesforce limitations, you cannot download templates larger than 4

3. Upload your cloned templates to an organization:

   - Current organization:
     1. Select Create
        Template ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY5NjMsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.yQp_Y6z0Pp_Ca6_el_npG82hXuUiINRlEgxLJ0UD7wE&_LANG=enus) and choose
        Upload Template from the dropdown
        menu.
     2. Follow your device's prompts to select the downloaded file.
   - Another organization:
     1. Open the eSignature Configuration page in
        your destination Salesforce organization.
     2. Select Create Template ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY5NjMsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.yQp_Y6z0Pp_Ca6_el_npG82hXuUiINRlEgxLJ0UD7wE&_LANG=enus) and choose
        Upload Template from the dropdown
        menu.
     3. Follow your device's prompts to select the downloaded file.
4. Complete the actions in the Upload Template dialog
   box:
   1. (Optional) Add custom button to matching layouts: Select this option if you want Docusign to add a template button to the same layouts as the original organization. If the template is on the Account layout in the original organization, Docusign adds it to the Account layout here. Docusign does not add custom buttons to unmatched layouts.
   2. Select Upload.

   The page refreshes.

You have completed the process. You have downloaded one or more
envelope templates and uploaded the file to create a clone. The cloned templates display
on the eSignature Configuration page of the importing Salesforce
organization.
