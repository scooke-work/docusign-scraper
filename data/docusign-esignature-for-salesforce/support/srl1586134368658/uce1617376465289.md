# Create a Default Envelope Template to Automate Your Contract Process

Learn to create a default envelope template to lead senders through a guided sending
experience.

You can create a default envelope template to lead senders through a guided sending experience.

You decide what documents, recipients, and signature fields each envelope will contain.

You also decide whether senders can customize the envelope to add documents, recipients,
and fields.

Tip: This video guides you through the process
of [creating an envelope template](https://players.brightcove.net/6118377982001/default_default/index.html?videoId=6171917923001). Each step in the
documentation also has its own video guide.

Follow this procedure to build an envelope template:

- [Add a Name and Data Source](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=xwj1596146541944.html&_LANG=enus "Learn to select a name and Salesforce object for your template.")
- [Add Documents](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=hya1596146614785.html&_LANG=enus "Learn to add documents to your envelope template.")
- [Add Recipients](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=ooo1596146674305.html&_LANG=enus "Learn to add recipients to your envelope template.")
- [Create Merge Fields](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=jdl1611042391715.1.html&_LANG=enus "Learn to create Salesforce merge fields to add to your envelope template.")
- [Place Fields](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=ywv1596146733047.html&_LANG=enus "Learn to place Docusign and Salesforce fields on the document text.")
- [Add Options](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=ktu1596146796958.1.html&_LANG=enus "How to create reminders for recipients, set an expiration date, enable the Quick Send experience, have a Salesforce status changed when a signature process is complete, and more.")
- [Create a Custom Button](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=owi1596146834259.html&_LANG=enus "Learn to add an Experience Cloud envelope template button to Salesforce layouts.")
