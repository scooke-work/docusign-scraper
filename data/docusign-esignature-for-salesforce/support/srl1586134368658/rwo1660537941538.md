# Create a Linked Envelope Template

Learn how to create an envelope template that links to one or more Docusign templates.

A Docusign envelope template is an eSignature sending experience that you
build for use in Salesforce. Your users select an envelope template button on a
Salesforce record, and are guided through the sending experience you built.

You decide whether your users can add documents, recipients, and signature fields to
those you included.

You can build envelope templates from scratch, or you can link to existing Docusign templates to build a linked envelope template.

Note: The Place Fields step does not display in a linked envelope template, even when documents are attached.
