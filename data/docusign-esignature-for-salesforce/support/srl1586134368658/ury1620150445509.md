# Access the Recipients Step of an Envelope Template

Find out how to access the Recipients step in an existing envelope template. Here, you
can add, edit and delete recipients. You can also change the email subject and message, view
a signing order diagram and more.

You can edit existing envelope templates on the Recipients step.
Here, you can add, edit and delete recipients. You can also change the email subject
and message, view a signing order diagram and more. Follow this procedure to access
the envelope template Recipients step:

1. Access the envelope template Recipientspage as a Salesforce administrator:

   The envelope template creation page opens to the Recipients
   step.
2. Select the App Launcher grid ![A screenshot of the Salesforce App Launcher grid, containing nine colored dots.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/app-launcher-grid.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcwMDAsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.sbdeJdnZjKvs0PWYTv2NnGgV3NGaL22XoR6o5FMpuxM&_LANG=enus "A screenshot of the Salesforce App Launcher grid, containing nine colored dots.") and chooseView All.
3. Select Docusign Apps Launcher.
4. Select the Docusign Envelope Templates tab.
5. Select the arrow ![A screenshot of a downward-facing triangle.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcwMDAsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.sbdeJdnZjKvs0PWYTv2NnGgV3NGaL22XoR6o5FMpuxM&_LANG=enus "A screenshot of a downward-facing triangle.") next to a template and choose Edit.
6. Select the Recipients arrow. The arrow has no text until
   selected.

You have completed the process. You have accessed the
Recipients step of an existing envelope template.
