# Step 1: Add a Template Name and Select the Account Object

The first step of the process involves naming the template and selecting the Salesforce
Data Source.

Use this procedure to name the template and select the Salesforce Data Source.

Tip:
[Step 1 is explained in full at the beginning of this
video](https://players.brightcove.net/6118377982001/default_default/index.html?videoId=6171917923001).

Important:
Person
Account does not display in the list of available objects in this
step. You must select the Account object here to use the
Person Account object later in the process.

1. Select the Create Template button at the far-right of
   the page. The Create Template dialog box opens.
2. Complete the following:
   1. Name: Enter a descriptive template name
   2. Select Data Source: Select Account from the dropdown menu.
   3. Select the Next button.
