# Create, Edit, Delete, or Download an Envelope Template from Docusign Setup

Learn how to create, edit, or delete a Docusign envelope template from
the Docusign Setup page.

When you open the Docusign Setup page, a list of controls displays. One of
these controls is the Templates tab on the eSignature
Configuration page. You can use the Templates tab to create, edit, or delete
envelope templates.

Envelope templates are pre-built sending experiences. Administrators add documents,
recipients, Docusign fields, and Salesforce fields to templates.

Users select an envelope template button on a record and launch your pre-built sending
experience.

Use this procedure to create, edit, or delete an envelope template from the Docusign Setup page.

1. Select eSignature in the left column.

   The eSignature Configuration page displays the
   Templates tab.
2. Make any selection:
   - [Create Template](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=gou1591745811909.html&_LANG=enus): Select this button to create a new envelope template.
   - [Link Template](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=rwo1660537941538.html&_LANG=enus): Select from the Create Template![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTY2NzIsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.ar68b7D8PGviSiOHlRXCymlPvfTUwywFZEe9mZSRrI8&_LANG=enus) arrow menu to create a new envelope template by linking to existing templates.
   - Edit: Select the arrow next to a template and choose Edit from the dropdown menu.

     The envelope template displays for editing.
   - Delete: Select the arrow next to a template and choose Delete > Delete.
   - Download: Select the arrow next to a template and choose Download from the dropdown menu.

   Docusign downloads the envelope template to your device. You can [upload the template to Salesforce](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=uhn1664569454728.html&_LANG=enus) and use or edit the template there.

You have completed the process. You have created, edited, deleted,
or downloaded a template from the Templates tab from the Docusign Setup page.
