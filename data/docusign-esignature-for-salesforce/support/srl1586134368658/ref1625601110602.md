# Add a By Role Recipient to an Envelope Using Merge Field Syntax

Find out how to add merge field syntax to a By Role recipient in an envelope
template.

Use this procedure to add merge field syntax to a By Role recipient in an
envelope template:

1. Select from the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcwNjksInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.a0Y39w4wkMMhb07mZ7q12uVgQpjSI0YcOxc0M0KGAvI&_LANG=enus)
   dropdown menu next to a recipient name.
2. Select Edit from the options.

   The Add Recipient dialog box opens.
3. Complete the following from the Add Recipient dialog box:
   1. Select By Role in the left column.
   2. Role: Enter a recipient role like
      Signer 1.
   3. Add Name and Email: Select the link to add merge
      field syntax.
   4. Name: Add merge field syntax for the object name,
      like `{!Contact.Name}` for a Salesforce contact.
   5. Email Address: Add merge field syntax for the
      object email address, like `{!Contact.Email}` for a
      Salesforce contact.
   6. Select the Add button to save the recipient and
      close the dialog box.

      ![A screenshot displays merge field syntax added to contact fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Screen-Shot-20210706-at-20211-PM.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcwNjksInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.a0Y39w4wkMMhb07mZ7q12uVgQpjSI0YcOxc0M0KGAvI&_LANG=enus "A screenshot displays merge field syntax added to contact fields.")

You have completed the process. You have added a By
Role recipient to an envelope template for a Salesforce user, contact, or
lead. When a user selects the envelope template button on a record, the recipient name
and email is dynamically added.
