# View the Related Objects of an Envelope

Learn to view objects related to the record from which the envelope was
sent.

A Docusign envelope is a digital container you build that holds documents,
recipients, and signature fields. Recipients sign the envelope by completing the
fields assigned to them.

Salesforce users can create and send envelopes from Salesforce records. The
Docusign Status tab lets you view the status of your
Docusign envelopes.

Docusign Status details include: Docusign envelope ID,
Sender Name, Sender Email,
Email Subject, and Related
Objects.

Use this procedure to view objects related to the record from which the envelope was
sent:

1. Select the Docusign Status tab.
2. Select an envelope from the Envelope # column.
3. Select Details.
4. Related Objects: Select a related object link.

   The related object page opens.

You have completed the task. You have opened an object
related to the record from which the envelope was sent.
