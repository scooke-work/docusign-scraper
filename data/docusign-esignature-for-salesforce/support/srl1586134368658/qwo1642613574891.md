# Step 1: Remove Users from Legacy eSignature for Salesforce Permission Sets

Learn how to remove all users from legacy eSignature for Salesforce permission sets.

To continue uninstalling the legacy app, remove all
users from the following permission sets:

- Docusign Administrator
- Docusign Sender
- Docusign User

Important: You will see permission sets with
duplicate names. Do not edit permission sets with Docusign Apps
Launcher in the Description field.

1. Open the Permission Sets page as an administrator:
   1. Select the gear ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg4MTEsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.sahLNpgTJchWVpDPI3oCnexSRnejSLZXdS91Li1xWzw&_LANG=enus) and choose
      Setup.
   2. Enter and select Permission Sets in the Quick
      Find field.

   The Permission Sets page opens.
2. Select the link for the legacy Docusign Administrator
   permission set. Do not select the entry with Docusign Apps
   Launcher in the Description field.
3. Select Manage Assignments and select all users.
4. Select Remove Assignments.![A screenshot displays check boxes selected for all users of a permission set.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/daluninstallremoveassignments.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg4MTEsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.sahLNpgTJchWVpDPI3oCnexSRnejSLZXdS91Li1xWzw&_LANG=enus "A screenshot displays check boxes selected for all users of a permission set.")
5. Select OK on the Are you sure you want to
   remove these users from this permission set? dialog box.

   The Docusign Administrator page opens.
6. Select Done.
7. Repeat this procedure for the legacy Docusign Sender and
   Docusign User permission sets.

You have completed step one of the process. You
have removed all users from legacy eSignature for Salesforce permission sets. Move to
[Step 2: Remove Legacy eSignature Components from Your Salesforce Layouts](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=dlc1642622890465.html&_LANG=enus "Learn how to remove all legacy eSignature for Salesforce components from Salesforce layouts.").
