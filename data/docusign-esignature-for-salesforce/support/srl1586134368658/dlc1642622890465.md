# Step 2: Remove Legacy eSignature Components from Your Salesforce Layouts

Learn how to remove all legacy eSignature for Salesforce components from Salesforce
layouts.

Uninstalling the legacy eSignature for Salesforce app removes all Docusign
components. However, you cannot delete components that are present on any layouts.

Use this procedure to remove Docusign components from your layouts:

1. Select the gear ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg4MTQsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.iRNNPL1pgFdestjLCzWXrA3WSSKspg0iwdq4d4x7RKY&_LANG=enus)and choose Edit Page from the
   dropdown menu.

   The Lightning App Builder page opens.
2. Remove legacy Docusign component from the canvas. Legacy
   components include: Docusign Sending, Docusign
   EnvelopeConfiguration, and Docusign
   EnvelopeConfigurationManageLayouts.
   1. Hover over a legacy component and select the trash icon ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Screen-Shot-20210615-at-123257-AM.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg4MTQsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.iRNNPL1pgFdestjLCzWXrA3WSSKspg0iwdq4d4x7RKY&_LANG=enus).![A screenshot displays the trash icon highlighted for the Docusign Files component.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Remove-DocuSign-component.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg4MTQsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.iRNNPL1pgFdestjLCzWXrA3WSSKspg0iwdq4d4x7RKY&_LANG=enus "A screenshot displays the trash icon highlighted for the Docusign Files component.")
   2. Select Save.
   3. Repeat this procedure for all legacy Docusign
      components.

   Salesforce removes the components from the canvas.
3. Select the back arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Screen-Shot-20210615-at-13251-AM.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg4MTQsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.iRNNPL1pgFdestjLCzWXrA3WSSKspg0iwdq4d4x7RKY&_LANG=enus) when you have removed all legacy
   components.
4. Repeat this procedure for each Salesforce layout (Account, Opportunity, and so
   on) that contains legacy Docusign components.

You have completed step two of the process. You have
removed all legacy Docusign components from your Salesforce layouts.
Move to [Step 3: Remove Legacy eSignature Custom Buttons From Your Salesforce Layouts](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=ddz1642632234520.html&_LANG=enus "Learn how to remove legacy Docusign eSignature for Salesforce custom buttons from your Salesforce page layouts.").
