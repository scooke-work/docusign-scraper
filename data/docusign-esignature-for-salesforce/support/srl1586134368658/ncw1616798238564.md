# Access the Merge Fields Step of an Envelope Template

Learn to add, edit, or delete merge fields in an envelope template.

Follow this procedure to add, edit, or delete merge
fields in an envelope template:

Access the envelope template Merge Fields page as a
Salesforce administrator:

1. Select the App Launcher grid ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/app-launcher-grid.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcwOTEsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.uTaWkAxlOyNNpvInAkhrsdYhcs9dBQsx66CzkN-FP14&_LANG=enus) and choose View All.
2. Select Docusign Apps Launcher.
3. Select the Docusign Envelope Templates tab.
4. Select the arrow ![A screenshot of a downward-facing triangle.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcwOTEsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.uTaWkAxlOyNNpvInAkhrsdYhcs9dBQsx66CzkN-FP14&_LANG=enus "A screenshot of a downward-facing triangle.") next to a template and choose
   Edit.
5. Select the Merge Fields arrow. The arrow has no
   text until selected.

You have completed the process. You have accessed the merge fields
step of an envelope template.
