# When I Save Changes to an Envelope Template, Multiple Headers Display on My Browser with the Message: “You Have Lightning Components Debug Mode Enabled”

Learn to fix the issue of multiple browser headers displaying the error: “You Have
Lightning Components Debug Mode Enabled”.

When I save changes to an envelope template, my browser opens multiple headers, each
displaying the message: You Have Lightning Components Debug Mode
Enabled.

![A screenshot displays three browser bars with the message “You Have Lightning Components Debug Mode Enabled”.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/A-screenshot-displays-three-browser-bars-with-the-message_-You-Have-Lightning-Components-Debug-Mode-Enabled.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg3NTYsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.932XPibl6CuKQq2PGyjwvLDKIa23f3oUjyhwOXTC4mQ&_LANG=enus "A screenshot displays three browser bars with the message “You Have Lightning Components Debug Mode Enabled”.")

## Resolution: Disable Lightning Debug Mode in your Salesforce user settings

Salesforce administrator

1. Log in to your Salesforce organization as an administrator.
2. Select Setup.
3. Enter and select Debug Mode in the Quick Find
   field.
4. Select the check box next to your user and choose
   Disable.

   You have completed the process. You have disabled Lightning Debug Mode for your
   user. The error messages no longer display when you save changes to Docusign envelope templates.
