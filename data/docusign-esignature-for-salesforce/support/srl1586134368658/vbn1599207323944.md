# Disable Docusign CLM Federated Security

Learn how to disable the Federated Security feature in your Salesforce
organization.

Federated Security can be disabled at any time. Here's how to disable Federated Security in
your Docusign CLM account:

1. Log in to your Docusign CLM account as an administrator.
2. Select the Admin tab.
3. Navigate to Integrations  >  Connected Apps.
4. Uncheck the Enable federated security box under the
   Security header.

   ![Disable Federated Security.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/Disable-Federated-Security.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg5MDQsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.FjKEKUu3v-Ed4UInN2gcYqOaBkODv8gcdUmvNUdnwpo&_LANG=enus "Disable Federated Security.")

   A modal opens, asking: Are you sure you want to disable federated
   security?
5. Select Disable federated security.
6. Select the Save button on the bottom of the page.

The process is complete. Salesforce Federated Security is disabled
on your CLM account. Salesforce and CLM security settings are separate, and
administrators must maintain security models for each.
