# Change the Docusign Template Used in a Salesforce Flow

Learn to change the Docusign template or Salesforce object connected
to your Flow.

## Before you begin

Install Docusign Apps Launcher version 6.12 or higher.

Note: The current API limitation is 100 generations per 10
minutes. If you want to increase your limit, [contact your Docusign representative](https://support.docusign.com/s/articles/How-do-I-contact-DocuSign-Customer-Support) about
options.

You can edit a Flow to change the Docusign template or Salesforce object used in your Flow.
Use this procedure to edit a Salesforce Flow:

1. Open a Flow as a Salesforce administrator:
   1. Select the gear ![A screenshot displays a gear icon with a lightning bolt in the center.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg2NjIsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.ffHnWz6AqyBrGnbfEkoQKeP1t4WV7HJA69Jh3QBZV6E&_LANG=enus "A screenshot displays a gear icon with a lightning bolt in the center.") in the upper right and choose
      Setup.
   2. Enter and select Flows in the Quick Find
      field.
   3. Select a Flow from the All Flows list.
2. Edit the Docusign action in your Flow:
   1. Select your Docusign
      Apex Action
      ![A screenshot displays the blue and white Apex Action icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/srl1586134368658/page/images/A-screenshot-displays-the-blue-and-white-Apex-Action-icon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg2NjIsInNoZWFmIjoic3JsMTU4NjEzNDM2ODY1OCJ9.ffHnWz6AqyBrGnbfEkoQKeP1t4WV7HJA69Jh3QBZV6E&_LANG=enus "A screenshot displays the blue and white Apex Action icon.").

      The action panel opens.
   2. Enter a new Docusign template ID or Salesforce
      Source Id.
   3. Select Done.
3. Select Save As New Version when you have completed editing your
   Flow.

   The Save as dialog box opens.
4. Select Save.
5. Select Activate.

   A message displays: Your automation was
   activated..

You have completed the process. You changed the template ID or
Salesforce object ID in your Flow.
