# Create New Custom Envelope Template Fields

Learn how to use the Custom Field feature on the Place Fields tab.

You can use the custom field feature on the Place Fields page to create
your own fields that are not included in the standard Docusign fields menu.

These fields can be custom radio buttons, drop-down menus, formula fields, fields requiring
users to attach a document, and many more.

Once created, these fields are added to uploaded documents in your envelope template.

Use this procedure to create a custom field:

1. Select the Custom Fields button ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/envelopetemplatecustomfields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc0NTAsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.a_Lai05HItFPbQbMP24jszO7YuS7dtUsXdz0mE1OxH0&_LANG=enus) in the left bar.
2. Select the + button to the right of the
   Custom Fields header.

   ![Select + button](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/envelopetemplatecustomfieldsadd.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc0NTAsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.a_Lai05HItFPbQbMP24jszO7YuS7dtUsXdz0mE1OxH0&_LANG=enus "Select + button")

   The Custom Field Details dialog box opens.
3. Complete the following fields:

   1. Name: Enter a descriptive name for the
      field.
   2. Type: Select an option (Checkbox, Formula,
      Attachment, and so on) from the drop-down menu.
   3. Checkbox: Select the check box for Required, Read
      Only, or Shared.
   4. Options: Depending on the Type selected, a list
      of options appear (Initial Value, Default Selection, and so on.)
   5. Formatting: Custom Fields with text show options
      for customizing Font, Font Color, Font Size, and so on.
   6. Default Advanced: Add a Tooltip or enter
      AutoPlace Text from an existing AutoPlace tag on the document.

You have completed the process. You have configured the new
custom field and can use it in the Place Fields step of an envelope template.
