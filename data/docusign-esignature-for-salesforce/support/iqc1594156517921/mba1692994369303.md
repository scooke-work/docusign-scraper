# Duplicate Envelopes Send When I Use Docusign in Salesforce Console View

Learn to prevent sending duplicate envelopes when using Docusign in Salesforce
console view.

When I send an envelope from a record in console view, Docusign sends a duplicate
envelope.

Example:

1. I visit the Burlington Textiles record on a console view
   tab.
2. I select a Docusign eSignature button.
3. A Docusign console tab opens. I send an envelope from this
   tab.
4. I take any of these actions after sending the envelope:
   - I select the Burlington Textiles tab, then return to the
     Docusign tab.
   - I refresh the browser tab.
   - I select the Return to Record button on the
     sending tab.

   Result: Docusign sends a duplicate envelope.

## Resolution: Close the eSignature console tab

Docusign envelope sender

Close the Docusign console tab that says
Your envelope was sent!

The signature process closes without sending a duplicate
envelope.
