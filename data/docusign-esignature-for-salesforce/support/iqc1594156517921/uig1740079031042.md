# Rules for Editing Envelope Template Recipients

Learn about recipient settings that you can change when sending from an envelope
template.

A Docusign envelope template is a sending experience that your
administrator builds. When you select an envelope template button on a record, a
pre-built envelope opens, containing documents, recipients, and signature fields.
You can add to or edit the contents of the envelope.

If you want to change details of an existing envelope template recipient, Docusign applies the following rules:

## "From Salesforce" Recipients

| Recipient field | Sender options |
| --- | --- |
| Action | Senders cannot change the selection. |
| Delivery | Senders cannot change the selection. |
| Delivery = Email + SMS | Senders cannot change the Delivery selection, but can edit the Phone Number field. The phone number merges with the phone number field from the new User or Contact. |
| User or Contact | Senders can delete or add users and contacts. |

## "By Name & Email" Recipients

| Recipient field | Sender options |
| --- | --- |
| Action | Senders cannot change the selection. |
| Delivery | Senders cannot change the selection. |
| Delivery = Email + SMS | Senders cannot change the Delivery selection, but can edit the Phone Number field. |
| Name | Senders can edit the field. |
| Email Address | Senders can edit the field. |

## "Signing Group" Recipients

| Recipient field | Sender options |
| --- | --- |
| Action | Senders cannot change the selection. |
| Select Group | Senders can delete or add a Signing Group. |
