# Envelope Creation Options: Recipients Step

This guide demonstrates the options that you can use in the Recipients step of the
envelope creation process.

As an envelope creator, you have many advanced options available to you. Each step of
the envelope creation process contains its own set of advanced options. In the
Recipients step, you add signers to your envelope. You can
add signers dynamically, like adding the owner of any record, or manually.

This guide demonstrates the advanced options that you can use in the
Recipients step:

- [Recipient Actions](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=byi1596432658895.html&_LANG=enus "There are many different recipient actions that can be assigned to recipients in an envelope template.")

- [Learn How to Use Envelope Custom Fields in Salesforce](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=jlx1682455699540.html&_LANG=enus "Learn how to use envelope custom fields in Salesforce envelopes.")
