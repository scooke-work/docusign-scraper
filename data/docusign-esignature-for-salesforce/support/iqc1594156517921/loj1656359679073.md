# Correct In-Flight Envelopes From the Envelope Status Component

Learn to use the correct envelope feature to add recipients or documents to an in-flight
envelope.

You can add recipients or documents to in-flight
envelopes using the correct envelope feature. You cannot use the
correct envelope feature for envelopes with a voided,
declined, or completed status.

Note: The original sender can
use Docusign Envelope Status to correct an active envelope.
Users with [shared access to the sender](https://support.docusign.com/s/document-item?bundleId=oeq1643226594604&topicId=ssa1578456277708.html) can correct the
sender's envelope using the legacy [Shared Envelope](https://support.docusign.com/s/document-item?bundleId=oeq1643226594604&topicId=xbn1578456279661.html) feature.

Use
this procedure to correct an in-flight envelope:

1. Open a Salesforce record and locate the Docusign Envelope
   Status component.
2.    ![A screenshot displays the Docusign Envelope Status drop-down menu with Send, View, Correct, and Void options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/DocuSign-Envelope-Status-dropdown-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyODksInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.HgPrHv00EQYIsVCbd3z9hciaRyLbjJlaxDvh96X7MkM&_LANG=enus "A screenshot displays the Docusign Envelope Status drop-down menu with Send, View, Correct, and Void options.")

   The envelope opens in your Docusign account.
3. (Optional) Take any action:

   - Upload: Add documents to the envelope
   - Add Recipient: Add a new recipient to the envelope
   - Edit the details of an existing recipient
   - Edit the Email Subject or Email Message
4. Select Next.
5. (Optional): Drag recipient fields to the document.
6. Select Correct.
7. Select Back to Record to return to the Salesforce
   record.

You have completed the process. You have corrected an in-flight
envelope by adding documents, recipients, or signature fields.
