# "The User Is Not the Owner of the Lock" Displays When I Edit a Gen Template or Envelope Template

Learn to fix the "User is not the Owner of the Lock" error when editing a Gen or
envelope template.

I tried to edit a Docusign Gen or envelope template. An error displayed:
The user is not the owner of the lock. The template is locked by
another user or in another application.

I cannot edit the template or save changes.

## Resolution: Re-Login to Salesforce or Empty Your Browser Cache

Cause: You closed the browser tab while editing a template
without selecting Save and Close. You opened another
browser tab and tried to edit the template you closed.

1. Select the Salesforce user icon and select Log
   Out from the dropdown menu.
2. Log back in to Salesforce.
3. Try to edit your Gen template again.
4. If the error displays, empty your browser cache and re-login.

   You have completed the process. You have removed the template
   lock. You can edit or save changes to the Gen template or envelope
   template.
