# Step 3: Add Fields and Send the Envelope (Envelope Template Button)

One or more fields are already present on the documents. These have been added to the
envelope template by your administrator. Use this procedure to view the fields or add
more.

Note: One or more fields may be present on
the documents. These have been added to the envelope template by your administrator.
Use this procedure to view the fields or add more.

1. (Optional) If your Salesforce organization has any Docusign envelope templates,
   the Select Matching Templates dialog box opens. Choose an
   option:

   - Add a template:
     1. Select the check box of each template you want to add to your
        envelope.
     2. Select the Apply button.
   - Cancel: Select the Cancel
     button if you do not want to add envelope template documents![Optional: Select Matching Templates](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Optional--Use-Envelope-Template.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyMTUsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.xUsqd-Lt_kUzm8px2xoVKluZ8et2ZaTS-HZuNQKo9EY&_LANG=enus "Optional: Select Matching Templates")

   The Select Matching Templates dialog box
   closes.
2. Add Docusign fields to the document:

   1. Select a Docusign field from the ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/envelopetemplatestandardsfields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyMTUsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.xUsqd-Lt_kUzm8px2xoVKluZ8et2ZaTS-HZuNQKo9EY&_LANG=enus)
      Standard Fields menu in the left column.
   2. Drag the field to the document.![Add Docusign fields](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Add-DocuSign-fields.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyMTUsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.xUsqd-Lt_kUzm8px2xoVKluZ8et2ZaTS-HZuNQKo9EY&_LANG=enus "Add Docusign fields")
3. Add Salesforce fields to the document:

   1. Select the Merge Fields button ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/envelopetemplatemergefields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyMTUsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.xUsqd-Lt_kUzm8px2xoVKluZ8et2ZaTS-HZuNQKo9EY&_LANG=enus) from the left
      column. The Salesforce Merge Fields menu opens.
   2. Select a field from the menu. You can expand arrows to find more fields,
      or enter text in the Search Fields bar.
   3. Drag the field to the document text.![Add Salesforce fields](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Add-Salesforce-fields.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyMTUsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.xUsqd-Lt_kUzm8px2xoVKluZ8et2ZaTS-HZuNQKo9EY&_LANG=enus "Add Salesforce fields")
4. (Optional) Choose any option:

   - Preview: Select this button to see a preview of
     the document.
   - Add fields for more recipients:
     1. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyMTUsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.xUsqd-Lt_kUzm8px2xoVKluZ8et2ZaTS-HZuNQKo9EY&_LANG=enus)
     2. Select another recipient from the drop-down menu.
     3. Drag Docusign fields for this recipient to the document
        text.

        Note: Each recipient's fields are represented by a separate color
        (yellow, blue, and so on.)![Switch recipients](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Switch-recipients.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyMTUsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.xUsqd-Lt_kUzm8px2xoVKluZ8et2ZaTS-HZuNQKo9EY&_LANG=enus "Switch recipients")
5. Select SEND.

You have completed the process. You have created an envelope and
sent it to the specified recipients.
