# Add SMS Delivery to an Envelope Recipient

Learn how to add the Email + SMS option to a recipient in your envelope.

Use this procedure to add the Email + SMS option to a
recipient in your envelope.

1. Open a Salesforce record (Account, Opportunity, etc.)
2. Select the envelope template custom button in the upper-right of the
   page.

   The button is named by your administrator. It may be hidden in the arrow
   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTczODYsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.u5UCTxBz1fieMnAhBOu3O9bpk0umqbf7q7ADK7081i0&_LANG=enus) menu.

   ![Send with envelope template button](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Send-with-Envelope-Template-button.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTczODYsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.u5UCTxBz1fieMnAhBOu3O9bpk0umqbf7q7ADK7081i0&_LANG=enus "Send with envelope template button")

   The Documents page opens, displaying the documents that have been added by
   your administrator.
3. Select the Next button in the upper-right of the
   page.

   The Recipients page opens.
4. Select the name of a recipient.

   The Add Recipient dialog box opens.
5. Delivery: Select the Email + SMS
   option from the drop-down menu.

   Note: Delivery only appears
   for Docusign customers with Allow SMS Delivery
   enabled in their account.
6. (Optional) Phone Number: If known, enter the recipient's
   phone number.

   Important: You must enter a space between the
   country code and the phone number.
7. Select Add.

   The Add Recipients dialog box closes.

You have completed the process. You have added theEmail
+ SMS option to a recipient in your envelope. When users send an
envelope from this template, the recipient receives a text message and an email. The
recipient can choose to open the envelope using either method. [Complete the sending process by adding Docusign and
Salesforce fields and sending your envelope.](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=xmx1596487118055.html&_LANG=enus)
