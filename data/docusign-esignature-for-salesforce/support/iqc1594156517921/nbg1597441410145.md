# Add Salesforce Documents to an Envelope

Learn how to add Salesforce documents to your envelope.

You can add Salesforce documents to your envelope when using the standard Send with Docusign experience, or the eSignature experience from the Docusign Negotiate action
menu.

Here are the types of Salesforce Files you can add:

Table 1. Salesforce File Types

| Name | File Type |
| --- | --- |
| Upload Documents | Add files from your device. |
| Owned by Me | Add Salesforce files that have you listed as Owner. |
| Shared with Me | Add Salesforce files that others have shared with you. |
| Recent | Add recently-viewed files from a record. |
| Following | Add all files that you are following on a record. |
| Libraries | Add all files that are grouped in a Salesforce Library. |

Use these procedures to add Salesforce files to an envelope using:

- [A standard Send with Docusign button](https://support.docusign.com/s/document-item?language=en_US&bundleId=iqc1594156517921&topicId=tco1611350907305.html&_LANG=enus)
