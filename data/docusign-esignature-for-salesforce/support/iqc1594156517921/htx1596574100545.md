# Step 2: View or Add Recipients (Envelope Template Button)

One or more recipients are already present in the envelope. These have been added to the envelope template by your administrator. Use this procedure to view the recipients or add more.

You see one or more recipients if your administrator added them to the envelope template. [Follow these rules if you want to edit envelope template recipients](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=uig1740079031042.html&_LANG=enus).

Use this procedure to add recipients or move to the next step:

1. Select the Add Recipient button.

   The Add Recipient dialog box opens, displaying the recipients that have been added by your administrator.
2. Add a recipient to your envelope:

   1. Select a recipient type from the left column:
      - From Salesforce: Enter the name or email of a Salesforce user or contact
      - By Name & Email: Enter the name and email of a recipient
      - Signing Group: Add a group of recipients from the connected Docusign account
   2. Action: Select a recipient action from the drop-down menu:
      - Needs to Sign: Must sign the document
      - Receives a Copy: Does not need to view or sign
      - Needs to View: Must view the document
      - Specify Recipients: Can add names and emails for placeholder recipients (like Approver.)
      - Allow to Edit: Can make changes to the document
   3. (Optional) Add recipient options: [Recipient Sending Options: Access Code, In-Person Signing, Sign Now, SMS and More](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=xch1620151662151.html&_LANG=enus).
   4. Select the Select button.
3. Envelope Custom Fields: Add values for all required fields. Required fields are marked with asterisks \*.

   Note: You only see Envelope Custom Fields if your administrator has created fields in the connected Docusign account. ![A screenshot displays four fields on the Envelope Custom Fields section of the Recipients page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Envelope-custom-fields-on-the-Recipients-page.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyMTIsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.GmKmwXO5PoXLqcHrq_kGS2asIiPFovXj-oftI8-d5jk&_LANG=enus "A screenshot displays four fields on the Envelope Custom Fields section of the Recipients page.")
4. (Optional):

   - View: Select this button to see a visual diagram of the signing order
   - Email Subject: Enter up to 100 characters of text.
   - Email Message: Enter up to 2,000 characters of text.
   - More Options: Choose any options:
     - Reminders: Set the number of days between recipient reminders. Select Custom to add a reminder of up to 999 days.
     - Expiration: Set a date or a number of days (up to 999) after which the envelope expires.
5. Select Next in the upper right.

   The Docusign eSignature fields page opens.

You have completed the second step of the process.
