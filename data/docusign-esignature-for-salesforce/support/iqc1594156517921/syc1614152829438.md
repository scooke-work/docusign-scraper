# Envelope and Recipient Status are Not Updated

Change Multi-Factor Authentication for API Logins settings or
Login IP Ranges to fix issues with envelope and recipient
statuses not updating.

If your Docusign Envelope Status and Docusign Recipient Status do not update, there are two
possible causes:

- The Salesforce organization has Multi-Factor Authentication for API
  Logins enabled on the user's' profile. This is incompatible with
  Docusign’s server-to-server API updates.
- Login IP Ranges are specified on the user's profile. This
  prevents API logins for the user from Docusign servers.

## Resolution 1: Disable Multi-Factor Authentication for API Logins on user's profile

1. Find the Salesforce user Profile:

   1.       the page.
   2. Select Setup.
   3. Enter Profiles in the Quick Find
      field.
   4. Select the Profiles link in the left
      navigation bar.
2. Select the name of the Profile to be edited. (Example: Standard User)
   The Profile page opens.
3. Select the Edit button.
4. Uncheck the Multi-Factor Authentication for API
   Logins box.
5. Select the Save button.

   The process is complete. Your Docusign Envelope Status and
   Docusign Recipient Status now update.

## Resolution 2: Disable Login IP Ranges on user's profile

1. Find the Salesforce user Profile:

   1. Select the gear ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc0OTUsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.cF23NRC24CIeIf2BskIh19Nte_gi_awCbB1GkIqgj8o&_LANG=enus) in the upper-right of the
      page.
   2. Select Setup.
   3. Enter Profiles in the Quick Find
      field.
   4. Select the Profiles link in the left
      navigation bar.
2. Select the name of the Profile to be edited. (Example: Standard User)
   The Profile page opens.
3. Select the Login IP Ranges link at the top of
   the Profile page.
4. Select the Del link to the left of an IP
   range.
5. Select the OK button on the Are you
   sure you want to delete the IP Login Range dialog box.
6. Repeat the Del action for all remaining
   ranges.

   The process is complete. Your Docusign Envelope Status and
   Docusign Recipient Status now update.

## Resolution 3: Add the relevant Docusign endpoint IP addresses to the allow-list

1. [Add Trusted IP Ranges for Docusign Gen for Salesforce](https://support.docusign.com/s/document-item?topicId=lip1592574174158.html&bundleId=izj1586134369853&_LANG=enus)
2. Add any other restricted IP addresses shown in the user’s login
   history.

   The process is complete. Your Docusign Envelope Status and
   Docusign Recipient Status now update.
