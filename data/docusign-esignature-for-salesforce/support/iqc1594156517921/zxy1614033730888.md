# When I Upload a PDF to an Envelope Template, I Sometimes See This Error Message: “The Validation of the PDF File Failed.”

There is an intermittent issue with adding PDF files to the Documents step of the
envelope template creation process. This does not happen with every PDF file.

I sometimes see an error message when I try to upload a PDF file to my envelope template:
The Validation of the PDF File Failed.

This does not happen with every PDF file.

**Steps to Reproduce**:

1. Upload a PDF file to an envelope template at the Documents step. The document fails to
   load.
2. An error displays: The validation of the PDF file
   failed

   **Result**: The PDF is not added to the envelope
   template.

   Note: Envelope templates currently
   support PDF file sizes of 3MB or smaller. This limit will be increased in a
   future release.

## **Resolution 1: Upload the PDF file to the envelope template in your connected Docusign account**

1. Select the Save and Close button for the
   envelope template in your Salesforce organization.
2. Log in to your Docusign account on another browser tab.
3. Select the Templates tab.

   The My Templates page opens.
4. Select the envelope template's name.

   The template's page opens.
5. Select the Edit button.

   A page with the template settings opens.
6. Select the Upload button.
7. Upload the PDF file from your device.
8. Select the SAVE AND CLOSE button.

   ![Save and close changes to your Docusign template](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Save-and-close-changes-to-your-DocuSign-template.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc0OTEsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9._9aDZaBZa53d1R4jsoR3K7DTfwBEtqW6vwyJiTsxb6Q&_LANG=enus "Save and close changes to your Docusign template")

   You have completed the process. You have added the PDF file that
   was too large to add to the template in Salesforce. This document is
   included in all envelopes that are sent from a Salesforce record using
   this template.

   Important: If you later
   edit this envelope template in Salesforce, you will get an error and
   will be unable to make additional changes.

## **Resolution 2: Upload the PDF file to Salesforce first and then attach it when creating an Envelope**

1. Upload the PDF file to Notes and Attachments in
   a Salesforce record.
2. Complete the Envelope Template creation process.
3. Select the Envelope Template custom button from a Salesforce
   record.
4. Select the Add from Salesforce button. The Add
   from Salesforce dialog box opens.
5. Check the box for the PDF saved in Notes and Attachments.
6. Select the Add button.
7. Complete the envelope sending process.

   The process is complete. The PDF file that could not be added to
   the Envelope Template is added from Notes and Attachments during the
   sending process.
