# Envelope Template Field Types and Functions

Here are the field types that can be used in an envelope template.

The Docusign envelope template Place Fields tab contains many
different fields that can be placed on uploaded documents.

Here are the fields that can be used in an envelope template:

Table 1.
Docusign field icons and actions

| Field Icon | Action |
| --- | --- |
|  | Recipient signature |
|  | Add recipient initials |
|  | Add date of recipient's signature |
|  | Add the full name of a recipient |
|  | Add a recipient's email address |
|  | Add recipient's company |
|  | Add recipient's title |
|  | Add a free-form field for recipient to enter text |
|  | Add a check box for recipient options |
|  | Add a drop-down menu for recipient options |
|  | Add a radio button for recipient options |
|  | Add a field to accept recipient payments A payment account must first be set up |
|  | Add a formula to calculate a value based on number or date fields in your document |
|  | Add a field to request additional documentation from a recipient |
|  | Add a free form text field visible only to the assigned recipient and only during the signing session |
|  | Add an alternative to a required Signature or Initial field Selecting Approve is the equivalent of signing |
|  | Add a field that terminates the recipient signing session |
