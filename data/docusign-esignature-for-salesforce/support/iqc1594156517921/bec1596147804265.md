# Place Fields Actions: Copy, Paste, Undo, Redo, and Zoom

You can use the action menu on the Place Fields page to take
actions on signature fields that are placed on the document.

Table 1. Place Field Actions

| Button | Button Name | Action |
| --- | --- | --- |
|  | Undo | Undo the last action taken on the field |
|  | Redo | Redo the last action taken on the field |
|  | Copy | Copy the selected field |
|  | Paste | Paste the selected field |
|  | Zoom | Zoom in and out on the signature page |
