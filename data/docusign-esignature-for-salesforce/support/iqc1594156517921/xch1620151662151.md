# Recipient Sending Options: Access Code, In-Person Signing, Sign Now, SMS and More

Learn how to add the role, access code, filter, private message, in person signing, SMS,
or sign now options when creating an envelope recipient.

You can add many options when creating an envelope or envelope template recipient.

Here are the option types and actions you can add to a recipient:

Table 1. Recipient Options

| Option Type | Description |
| --- | --- |
| Add Role | Enter a placeholder title (like Approver) that allows the sender to specify a name and email address later. |
| Add Filter | Add recipients based on a set of conditions being met or not met. This option only appears when you select the Related Lists recipient type. |
| Make the Sender Host | The person who sends the envelope becomes the host of the in-person signing session. The host signs, and then each additional recipient signs on the host's device.  This option only appears when you select the In Person Signer action for a recipient. |
| Host Name | Enter the name of the Docusign user who will host the In Person Signer session. This field only appears when you uncheck the Make the Sender Host box. |
| Host Email | Enter the email address of the Docusign user who will host the In Person Signer session. This field only appears when you uncheck the Make the Sender Host box. |
| Initiate Signing After Sending (Sign Now) | The sender is asked to sign their own envelope immediately after sending it. This option only appears in the following cases:   - You select Sender as the recipient   type and Needs to Sign as the action - You select any other recipient type (Users/Contacts, By   Role, and so on) and In Person Signer   as the action |
| Delivery | Select this drop-down menu to show the Email and SMS/Email options. The Delivery field only appears for customers with Allow SMS Delivery and Signer can sign on a mobile device enabled.  Contact your Docusign account manager to enable these features. |
| Email | Send an envelope to a recipient by email. |
| Email + SMS | Send an envelope to up to four recipients by email and text message. You must enter a space between the country code and the phone number. |
| WhatsApp | Send an envelope to a recipient by email and WhatsApp message. |
| Add Access Authentication | Select this link to show the SMS/Phone and Access Code options. |
| SMS/Phone | Send an access code to a recipient's mobile device to add an extra layer of security to the signing process. |
| Access Code | Create a code that a recipient must enter to start the signing process. You must send the code to the recipient separately by email or another messaging platform. |
| Add Private Message | Send a message to just this recipient or group. |
| Add Name and Email | Enter the name and email address of a recipient. This option only appears when you select the By Role recipient type. |
