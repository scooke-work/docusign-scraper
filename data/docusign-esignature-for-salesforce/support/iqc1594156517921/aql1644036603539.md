# Use a Send Now Envelope Template Button to Send an Envelope From a Salesforce Record

Learn how to use a send now envelope template button to create and send an envelope from
a Salesforce record.

You can use a send now
envelope template button to instantly create and deliver an envelope from a Salesforce
record.

Send now templates are pre-built with
documents, recipients, Docusign fields, and Salesforce fields already added. This
speeds up the envelope creation and sending process.

All you have to do is
select the send now button on a Salesforce record and the envelope is instantly
built and sent to recipients.

Use this procedure to send an envelope using
a send now envelope template:

1. Open a Salesforce record (Account, Opportunity, and so on.)
2. Select the send now envelope template button.
