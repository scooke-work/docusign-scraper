# Create and Send an Envelope with Docusign eSignature for Salesforce

Learn how to create and send an envelope with Docusign eSignature for
Salesforce.

A Docusign envelope is a digital container you build that holds documents, recipients,
and signature fields. Recipients sign the envelope by completing the fields assigned to
them. Docusign marks the envelope as Signed when all recipients
complete their fields.

Salesforce users can create and send envelopes from Salesforce records. Users can add
recipients and documents manually, from a template, or dynamically from Salesforce
records, users, or contacts. Docusign saves the document to the Salesforce record when
all recipients finish signing.

You can create and send an envelope with Docusign eSignature for Salesforce using any of
these methods:

[Standard Send with Docusign template](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=gsh1675832324336.html&_LANG=enus "Learn to send an envelope from a Salesforce record using a standard Send with Docusign template.")
:   The Send with Docusign template is a pre-built, three-stage sending experience. An
    administrator does not build this template. Senders select a Send
    with Docusign button on a record and add documents, recipients,
    and signature fields.

[Use envelope templates with a standard Send with Docusign template](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=lee1682365097112.html&_LANG=enus "Learn how to add envelope template content to a standard Send with Docusign envelope.")
:   Add existing envelope templates to your Send with Docusign envelope. Senders
    select a Send with Docusign button on a record and add
    templates from the Select Template button. The selected
    templates add documents, recipients, Envelope Custom Fields, and Salesforce and
    Docusign fields to the standard envelope.

[Default envelope template](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=zgh1596562751041.html&_LANG=enus "This guide demonstrates the process for sending a document for eSignature using an envelope template custom button.")
:   An administrator builds this three-stage sending experience. Senders may see documents,
    recipients, and signature fields in the envelope pre-added by the administrator.
    Senders can add to or edit the contents of the pre-built envelope.

[Quick Send envelope template](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=wji1611295987221.html&_LANG=enus "The Quick Send signature experience gives you a faster way to send from an envelope template. Use this procedure to send an envelope in the Quick Send experience.")
:   An administrator builds this one-stage sending experience. Senders may see documents,
    recipients, and signature fields pre-added by the administrator. Senders are
    unable to edit the pre-added documents or signature fields. Senders can edit the
    Reminders, Expiration,
    Email Subject, and Email
    Message fields. Senders can also add non-signing recipients
    (like Receives a Copy recipients) to the envelope.

[Place Fields envelope template](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=zet1644034684923.html&_LANG=enus "Learn how to use a place fields envelope template button to create and send an envelope from a Salesforce record.")
:   An administrator builds this one-stage sending experience. Senders only see
    the Place Fields step, where they can add Docusign and
    Salesforce fields to the envelope. Senders can still change envelope
    settings and add documents and recipients on the Place Fields page. The
    Place Fields envelope template is the most versatile quick-sending template
    type. Senders can place fields and send or choose to add many other
    options.

[Send Now envelope template](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=aql1644036603539.html&_LANG=enus "Learn how to use a send now envelope template button to create and send an envelope from a Salesforce record.")
:   An administrator builds this one-stage sending experience. When senders select the envelope
    template button on a record, Docusign instantly sends a pre-built envelope. The
    sender does not see the envelope's contents and can not add to or change the
    envelope.

[In Person Signer envelope](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=yyp1699651453088.html&_LANG=enus)
:   You build an envelope for a multi-recipient signing session, with all signers in
    the same room.

[Send from the Salesforce Mobile app](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=wdk1660587409378.html&_LANG=enus "Learn how to send a Docusign envelope from your mobile device using the Salesforce mobile app.")
:   Send an envelope from a Docusign template using the Salesforce
    mobile app.

[Send from a mobile browser](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=jny1660579105302.html&_LANG=enus "Learn how to send a Docusign envelope from a browser on your mobile device.")
:   Send an envelope from a Docusign template using the browser on your mobile device.

[Docusign Files action menu](https://support.docusign.com/s/document-item?bundleId=ygy1586134394050&topicId=zwk1586134361708.html&_LANG=enus)
