# Send an Envelope Using WhatsApp

Learn to use WhatsApp to deliver an envelope to a recipient.

You can use WhatsApp to deliver an envelope to a recipient. When you send an envelope by
WhatsApp, you cannot also send via SMS or email.

Note: This option does not display if your Docusign account does not have WhatsApp delivery turned on.

Use this procedure to send an envelope to a recipient by WhatsApp:

1. Open a Salesforce record and select a Docusign button.
2. [Add Documents](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=mdb1596562820671.html&_LANG=enus "One or more documents are already present in the envelope. These have been added to the envelope template by your administrator. Use this procedure to view the documents or add more.").
3. [Add recipients](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=htx1596574100545.html&_LANG=enus "One or more recipients are already present in the envelope. These have been added to the envelope template by your administrator. Use this procedure to view the recipients or add more.").
4. [Add Docusign and
   Salesforce fields](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=hke1596562836794.html&_LANG=enus "One or more fields are already present on the documents. These have been added to the envelope template by your administrator. Use this procedure to view the fields or add more.").
5. When you have added fields, select ACTIONS >  Edit Recipients.

   ![A screenshot displays the Actions menu with Edit Recipients selected.](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/A-screenshot-displays-the-Actions-menu-with-Edit-Recipients-selected.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyNTEsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9._f-z1DFXgniLC_kZMjAPvfShRjKJxu7AN_pAGnbAPKk&_LANG=enus "A screenshot displays the Actions menu with Edit Recipients selected.")
6. Select WhatsApp for any recipient and select
   DONE.

   ![A screenshot displays the WhatsApp option checked in the Edit Recipients dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/A-screenshot-displays-the-WhatsApp-option-checked-in-the-Edit-Recipients-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyNTEsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9._f-z1DFXgniLC_kZMjAPvfShRjKJxu7AN_pAGnbAPKk&_LANG=enus "A screenshot displays the WhatsApp option checked in the Edit Recipients dialog box.")
7. Select SEND.

You have completed the process. You have created an envelope
and sent it to a recipient by WhatsApp.
