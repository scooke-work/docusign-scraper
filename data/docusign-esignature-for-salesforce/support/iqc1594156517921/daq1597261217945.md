# Void Active Docusign Envelopes

Learn to use the void envelope feature to cancel the signature process of active
envelopes.

The Docusign Envelope Status
component of a Salesforce page displays the status of all envelopes sent for signature.
You can use the void envelope feature to cancel active envelopes.
Void envelope is helpful when you discover a problem with a
contract and want to stop the signing process. Recipients who have finished signing
receive an email notification with your reason for voiding. Recipients who still need to
sign cannot view or sign the enclosed documents.

Note: The
original sender can void active envelopes from the Docusign Envelope
Status component. Users with [shared access to the sender](https://support.docusign.com/s/document-item?bundleId=oeq1643226594604&topicId=ssa1578456277708.html) can void the sender's
envelope using the legacy [Shared Envelope](https://support.docusign.com/s/document-item?bundleId=oeq1643226594604&topicId=xbn1578456279661.html) feature.

Use this procedure to void an envelope from the
Docusign Envelope Status component:

1. Open a Salesforce record and locate the Docusign Envelope
   Status component.
2.    ![A screenshot of the Docusign Envelope Status component displays a drop-down menu with Send, View, Correct, and Void options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/DocuSign-Envelope-Status-dropdown-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyOTEsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.pOTWaqsiL1q_ptHpjTo6OzzjbWTD4A4yZWgpeiXd3FE&_LANG=enus "A screenshot of the Docusign Envelope Status component displays a drop-down menu with Send, View, Correct, and Void options.")
3. Enter a message to recipients in the Reason for voiding
   envelope field.

   Note: The message has a 200-character limit
   and is a required step.
4. Select Void.
5. Select Back to Record to return to the Salesforce
   record.

You have completed the process. Recipients who have finished
signing receive an email notification with your reason for voiding. Recipients who still
need to sign cannot view or sign the enclosed documents.
