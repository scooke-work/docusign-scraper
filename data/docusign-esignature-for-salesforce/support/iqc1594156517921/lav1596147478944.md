# Preview a Document Uploaded to an Envelope Template

You can preview uploaded documents in the envelope template by following these
steps.

You can preview uploaded documents in the envelope template by following these steps:

1. Select from the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTczMTksInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.0RSqF1t4ayrTAjpNOJHLdPpgRHPogWgJzEyXOV4-u5g&_LANG=enus) drop-down menu to the right of a document.
2. Select Preview from the options.

   ![Preview uploaded document](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/envelopetemplatedocumentoptions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTczMTksInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.0RSqF1t4ayrTAjpNOJHLdPpgRHPogWgJzEyXOV4-u5g&_LANG=enus "Preview uploaded document")

   A new tab opens with the document preview page.
3. Select the document pane to open the preview in full-screen view.
4. Select the X in the upper-right of the page to close the
   preview.

   You have completed the process. You have previewed the uploaded
   document.
