# Send an Envelope by Email or Text Message (SMS)

Learn how to enable the SMS + Email feature and use it to send envelopes by text
message.

You can send envelopes to recipients by text message using the Email +
SMS option.

SMS delivery is a paid feature that is activated when your Docusign account manager
enables these settings:

- Allow SMS Delivery
- Signer can sign on a mobile device

Only your Docusign account manager can activate these settings for you.

Once activated, a Delivery drop-down menu appears in the Add Recipients
step of an envelope or envelope template:![Email + SMS Delivery option](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Email--SMS-Delivery-option.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTczODAsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.X1y8FdOsMyTTcLkzdMt7b7t0v0e8IhGaw4R83p4yugo&_LANG=enus "Email + SMS Delivery option")

## SMS guidelines:

- You can use Email + SMS for up to four (4) recipients per envelope. If you attempt to add
  more than four SMS recipients, an error message opens: You're limited to 4 SMS
  recipients.
- You must enter a space between the country code and the phone number.
- You do not need to enter a phone number for dynamic recipients, like contacts, that already
  have phone numbers saved.
- You can not send an SMS message without also sending the recipient an email.
- Email + SMS can only be used for these recipient types:
  - Needs to Sign
  - Needs to View
  - Receives a Copy
- You can only choose the Access Code option in the Add
  Access Authentication field when Email + SMS is selected.
- You cannot use Email + SMS with Docusign signing groups.

## SMS number formatting:

You can use the following phone number formats:

- 9251111111 (country code defaults to 1)
- +1 9251111111
- +1 925-111-1111
- +1 (925)-111-1111

Note: If you enter a number format that is not
allowed, an error message displays. You must update the number format to save
the recipient to the envelope or template.

## SMS country code formatting:

- You must enter a space between the country code and phone number.
- Customer phone numbers are stored in Salesforce in two format types:
  - **United States and Canada**: (+1 PhoneNumber) format
  - **All other regions**: (+CountryCode PhoneNumber) format
- If you enter a phone number with the wrong country code format, 1 is
  automatically added as the country code.
- The (+CountryCode PhoneNumber) format is required when adding SMS + Email to a By
  Role recipient (not a Users/Contacts recipient).
