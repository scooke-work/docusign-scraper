# Create and Send an Envelope for an In Person Signing Session

Learn to use the In Person Signer feature to create a multi-recipient signing session with all signers in the same room.

In Person signing creates a multi-recipient signing session, where all signers must be in the same room. The host opens the envelope and each signing recipient signs on the host's device.

The following rules apply to In Person signing sessions:

- The host of the in-person signing session must be a Docusign eSignature for Salesforce user
- You can add an In Person Signer recipient anywhere in the signing order
- Other In Person Signer recipients do not need to be Docusign users
- You can add recipients to the envelope that are not in person signers (Needs to Sign, Receives a Copy, and so on)
- You can set the signer name using Lookup Fields, Related Lists, Users/Contacts, and By Role
- You cannot use SMS delivery for In Person Signer sessions
- You can only use the Sign Now option if the In Person Signer is first in the signing order (serial or parallel)
- You can use [merge field syntax](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=psq1595994131623.html&_LANG=enus) to complete the Host Name and Host Email Address fields

Example:

Your customer's president and head of finance meet in your office to review a contract. You create an envelope for an In Person Signer session. You host the session and guide your customers through the signing process. You answer questions they have as they sign. You pass your tablet to each signer until all have completed the session.

Use this procedure to create and send an envelope for an In Person signing session:

1. Select a Send with Docusign button on a record. Your administrator may choose a different name for the button.
2. Complete the actions on the Documents page.

   1. Use any option to add documents to your envelope:
      - Documents: Select the check box for up to ten documents
      - Upload: Add a document from your device
      - [Add from Salesforce](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=qva1617752033779.html&_LANG=enus): Add documents from Salesforce Files
      - [Select Template](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=fne1682396685506.html&_LANG=enus): Add documents, recipients, and signature fields from Docusign envelope templates
   2. Select Next.

   The Recipients page opens.
3. Add an In Person Signer on the Recipients page. [Follow these rules if you want to edit envelope template recipients](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=uig1740079031042.html&_LANG=enus).

   1. Select Add Recipient.
   2. Select a recipient type from the left column:
      - From Salesforce: Enter the name of a Salesforce user or contact
      - By Name & Email: Enter any name and email address
      - Signing Group: Enter the name of a signing group from your Docusign account
   3. Action: Select In Person Signer.
4. Choose a host for the signing session:

   - Make yourself the host: Leave the Make me the Host box selected
   - Make another recipient the host: Uncheck the Make me the Host box and enter a Host Name and Host Email in the fields that open

     Note: You can also [use Salesforce merge fields](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=wtk1625247088017.html&_LANG=enus) to dynamically add a host name and email from standard and custom Salesforce objects.
   - (Optional) Add recipeint options: [Recipient Sending Options: Access Code, In Person Signing, Sign Now, SMS, and More](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=xch1620151662151.html&_LANG=enus).
5. Select Select and Next.

   The Prepare & Send page opens.

   2. ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/envelopetemplatemergefields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyMzIsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.0K57oMXAUs8j0rJKl8_TB83Ruidnt__9gKA4JeA6T8Y&_LANG=enus) Merge Fields: Drag Salesforce fields to the document.
   3. (Optional) Select another recipient from the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyMzIsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.0K57oMXAUs8j0rJKl8_TB83Ruidnt__9gKA4JeA6T8Y&_LANG=enus) menu in the upper left and drag Docusign fields to the document.

You have completed the process. You created and sent an envelope for an In Person signing session.
