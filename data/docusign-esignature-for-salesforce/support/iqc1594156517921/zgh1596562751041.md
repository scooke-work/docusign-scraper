# Create an Envelope from an Envelope Template Button

This guide demonstrates the process for sending a document for eSignature using an
envelope template custom button.

This guide demonstrates the process for sending a document for eSignature using an Envelope
Template custom button. Your administrator builds envelope templates to simplify the sending
process for you. When you select an envelope template button on a record, your envelope may
contain documents, recipients and signature fields. These options were added by your
Salesforce administrator.

You can build an envelope from an envelope template by completing the following steps:

- [Step 1: View or Add Documents (Envelope Template
  Button)](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=mdb1596562820671.html&_LANG=enus)
- [Step 2: View or Add Recipients (Envelope Template
  Button)](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=htx1596574100545.html&_LANG=enus)
- [Step 3: View or Add [companyname] and Salesforce
  Fields and Send the Envelope (Envelope Template Button)](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=hke1596562836794.html&_LANG=enus)
