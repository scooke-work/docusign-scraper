# Envelope Template Error: “You Are Not Authorized to Access the Envelope”

Set account defaults to Allow sender to override account defaults
to fix this. This allows the envelope template to save on the Documents page and be edited
later.

In the Envelope Template Creation process, selecting the Save and Close
button on the Documents page results in an error: "You are not authorized to access
the envelope." This error occurs only on the Documents page. Selecting Save and
Close on the pages for Recipients, Place Fields, Options, and Custom Button do not
result in errors.

**Steps to Reproduce**:

1. Select the Docusign Envelope Templates tab.
2. Click the New button in the upper-right of the page.
3. Enter a template name and a standard object (Accounts, Opportunities, etc.)
4. Immediately click the Save and Close button on the Documents
   page.

   **Result**: An error message reads: You are not
   authorized to access the envelope. The user is unable to
   proceed with the template creation process.

## Resolution: Enable "Allow sender to override account defaults" in the connected Docusign account

1. Log in to the connected Docusign account as an Administrator.
2. Select the Admin tab.
3. Select Reminders and Expiration in the left
   column.
4. Select Allow senders to override account
   defaults.
5. Days before request expires: Enter any number
   higher than zero.

   Note: If the Days before request
   expires: field is blank or has a value of zero, you
   cannot save settings changes.

6. Select Save Changes.

   A message displays: Changes saved. You have completed the process. You have set the account defaults to Allow
   senders to override account defaults. You can save
   envelope template settings on the Documents page and move to the
   Recipients page.
