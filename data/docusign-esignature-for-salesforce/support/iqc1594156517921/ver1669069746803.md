# View In-Flight Envelopes From the Docusign Envelope Status Component

Learn to view the documents of an in-flight or completed envelope by using the correct
feature.

Using the view envelope feature, you can view the documents from an
in-flight or completed envelope.

View envelope allows you to view documents from a sent envelope.

Use this procedure to view the documents of an envelope:

1. Open a Salesforce record and locate the Docusign Envelope
   Status component.
2. Select View Envelope from the arrow ![A screenshot displays a downward-facing triangle icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyODYsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.xyIulvdc2lmpRqBUil3Cd8YtQoiTK19jXCB8UnFt0XU&_LANG=enus "A screenshot displays a downward-facing triangle icon.") drop-down menu of an in-flight envelope.

   ![A screenshot of the Docusign Envelope Status component displays a drop-down menu with Send, View, Correct, and Void options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/DocuSign-Envelope-Status-dropdown-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyODYsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.xyIulvdc2lmpRqBUil3Cd8YtQoiTK19jXCB8UnFt0XU&_LANG=enus "A screenshot of the Docusign Envelope Status component displays a drop-down menu with Send, View, Correct, and Void options.")
3. (Optional) OTHER ACTIONS: Take any action:

   - Void: Cancel the signature process for all
     recipients
   - Correct: Make changes to the documents, recipients,
     or signature fields of the envelope
   - Clone: Create a copy of the envelope
4. Select CLOSE.
5. Select Back in the upper left to return to the
   record.

You have completed the process. You viewed an in-flight envelope
from the Docusign Envelope Status component.
