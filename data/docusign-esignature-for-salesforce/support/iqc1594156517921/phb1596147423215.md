# Rename a Document Uploaded to an Envelope Template

You can change the file names of uploaded documents in the envelope template by
following these steps.

You can change the file names of uploaded documents in the envelope template by following
these steps:

1. Select from the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTczMTUsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.3mUwB6nxHTVtFjHpDQwRmjMB2cq9PzbdyTaezc6mCIk&_LANG=enus) drop-down menu to the right of a document.
2. Select Rename from the options.

   ![Rename uploaded document](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/envelopetemplatedocumentoptions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTczMTUsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.3mUwB6nxHTVtFjHpDQwRmjMB2cq9PzbdyTaezc6mCIk&_LANG=enus "Rename uploaded document")

   The Rename Document dialog box opens.
3. Enter a new name for the document.
4. Select the Rename button.
5. Select Save and Close.

   You have completed the process. You have changed the name of the
   uploaded document.
