# Users are Prompted to Reauthorize Every Hour

Change Multi-Factor Authentication for API Logins settings or
Login IP Ranges to fix issues with users being prompted to
re-enter credentials every hour.

Some users are prompted to enter their credentials every hour. This could occur for
two reasons:

- The Salesforce organization has Multi-Factor Authentication for API
  Logins enabled on the user's' profile. This is incompatible with
  Docusign’s server-to-server API updates.
- Login IP Ranges are specified on the user's profile. This
  prevents API logins for the user from Docusign servers.

## Resolution 1: Disable Multi-Factor Authentication for API Logins on user's profile

1. Find the Salesforce user Profile:

   1.       the page.
   2. Select Setup.
   3. Enter Profiles in the Quick Find
      field.
   4. Select the Profiles link in the left
      navigation bar.
2. Select the name of the Profile to be edited. (Example: Standard User)
   The Profile page opens.
3. Select the Edit button.
4. Uncheck the Multi-Factor Authentication for API
   Logins box.
5. Select the Save button.

   The process is complete. Users are no longer prompted to enter
   credentials every hour.

## Resolution 2: Disable Login IP Ranges on user's profile

1. Find the Salesforce user Profile:

   1. Select the gear ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc0OTMsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.0JzDyX4hU7x35WvVs-wqFA_Z5vOSgoINLJkXe0k9wsc&_LANG=enus) in the upper-right of the
      page.
   2. Select Setup.
   3. Enter Profiles in the Quick Find
      field.
   4. Select the Profiles link in the left
      navigation bar.
2. Select the name of the Profile to be edited. (Example: Standard User)
   The Profile page opens.
3. Select the Login IP Ranges
4. Select the Del link to the left of an IP
   range. link at the top of
   the Profile page.
5. Select the OK button on the link at the top of
   the Are you sure you want to delete the IP Login
   Range? dialog box.
6. Repeat the Del action for all remaining
   ranges.

   The process is complete. Users are no longer prompted to enter
   credentials every hour.

## Resolution 3: Add the relevant Docusign endpoint IP addresses to the allow-list

1. [Add the IP ranges from the Docusign Trust
   Center](https://www.docusign.com/trust/security/esignature).
2. (Optional): If needed, add any restricted IP addresses from the user’s
   Salesforce login history logs, including **[Salesforce IP addresses](https://help.salesforce.com/s/articleView?id=000321501&type=1)**.

   The process is complete. Users are no longer prompted to enter
   credentials every hour.
