# Send an Envelope Using the Docusign Quick Send Envelope Template Experience

The Quick Send signature experience gives you a faster way to send from an envelope
template. Use this procedure to send an envelope in the Quick Send experience.

You can use a Quick Send envelope template button to
create and deliver a Docusign envelope from Salesforce.

Documents, Recipients and
eSignature fields may already be present from the envelope template built by your
Salesforce Administrator.

Use this procedure to send an envelope using a
Quick Send envelope template:

1. Open a Salesforce record (Account, Opportunity, and so on.)
2. Select the Quick Send button. The Send with Docusign page opens.

   ![Quick send button](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Quick-Send-button.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyMTksInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.YE2bmnfCJvzX7T65eUliib44NtB1lGoE8zD_9j2qaUE&_LANG=enus "Quick send button")

   Note: Your administrator chooses the button name. It may be hidden in the arrow
   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyMTksInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.YE2bmnfCJvzX7T65eUliib44NtB1lGoE8zD_9j2qaUE&_LANG=enus)
   menu.
3. (Optional):

   - Add Recipient:
     1. Select the Add Recipient button.
     2. Add a recipient using the From
        Salesforce, By Name and Email
        or Signing Group options.

        Note: You can only use the Receives a Copy
        option when adding a recipient in Quick Send.
   - Edit recipient details:
     1. Select a recipient name. The Add
        Recipient dialog box opens.
     2. Edit any active fields. [Follow
        these rules if you want to edit envelope template
        recipients](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=uig1740079031042.html&_LANG=enus "Learn about recipient settings that you can change when sending from an envelope template.").
     3. Select the Select button. Any changes are
        saved. The dialog box closes.
   - View: See a diagram of the recipient signing
     order, from start to completion.
   - Message to Recipients:
     1. Email Subject: Enter a new subject for
        your recipient.
     2. Email Message: Enter a new message for
        your recipient.
   - More Options:
     - Reminders: Add a reminder to your recipients from the drop-down
       menu. Select Custom from the drop-down
       menu to add a custom reminder of up to 999 days.
     - Expiration: Set a date or a number of days (up to 999) after
       which the envelope expires.
4. Select the Send button at the upper-right of the
   page.

   A message opens: Sent for Signature. The record
   page refreshes.

You have completed the process. You have created and sent an
envelope using the Quick Send experience.
