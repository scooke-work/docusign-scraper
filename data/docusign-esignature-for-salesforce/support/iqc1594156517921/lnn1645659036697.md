# When Recipients Open Envelopes, They See a “Security Requests From Sender” Dialog Box

This occurs when no phone number is specified for a Salesforce Users/Contacts or By Role
recipient with SMS authentication applied.

Recipients sometimes see a Security Requests From Sender dialog box when
they open envelopes sent from my envelope template. This dialog box presents the
recipient with a long list of security questions. The recipient can not open and
complete the envelope until they have answered all of the questions.

Note: This error only occurs for envelope templates matching these
criteria:

- The template contains at least one By Role or
  Users/Contacts recipient.
- At least one By Role or Contact recipient has
  SMS/Phone authentication applied, but with no
  phone number entered.
- There is no space entered between the country code and the phone
  number.
- The envelope template has the Send Now radio button
  selected on the Options step.

## Resolution 1: Upgrade to Docusign Apps Launcher version 4.3 or higher

Docusign and Salesforce administrator

1. Log in to your Salesforce organization as an administrator.
2. [Follow these instructions to upgrade
   the Docusign Apps Launcher package to version 4.3 or
   higher.](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=ogj1586134254571.html&_LANG=enus)

   You have completed the process. The new package will not allow
   senders to add recipients with SMS/Phone security applied and no phone
   number present.

## Resolution 2: Add a phone number to all By Role or Users/Contacts recipients

Docusign and Salesforce administrator

Note: Only use this
resolution if you are unable to upgrade to the latest version of the
Docusign Apps Launcher for Salesforce package.

1. Select the name of a By Role or
   Users/Contacts recipient. The Add Recipient
   dialog box opens.
2. Enter the recipient phone number if the
   SMS/Phone field is blank.

   Important: You must enter a space
   between the country code and the phone number.
3. Select the Add button. The Add Recipient dialog
   box closes.
4. Repeat the process for any other By Role or
   Users/Contacts recipient in the
   template.
5. Select the Save and Close button in the
   upper-right. The page refreshes. The envelope template page opens.

   You have completed the process. You have added the missing phone number to any
   By Role or
   Users/Contacts recipient. By Role and
   Users/Contacts recipients can open and complete envelopes without
   answering security questions.
