# Delete a Recipient From an Envelope Template

Administrators needing to delete recipients from an existing envelope template can do so
at any time.

Administrators needing to delete recipients from an existing envelope template can do so at
any time.

Warning: If you remove a recipient that
has signature fields assigned to them, those fields will need to be removed or
re-assigned to another recipient.

**Example**: You delete **Signer 1** from
an envelope template. **Document 1** in the template contains signature
fields assigned to Signer 1. The signature process will fail unless **Signer
1**'s fields are removed from **Document 1** or reassigned to another
recipient.

Use this procedure to delete recipients from an existing envelope template:

1. Select from the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTczNzYsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.dAabmSpgt3Qp40pip7Tjw4fuXZnrnBPALr26OPmYHp8&_LANG=enus)arrowhead drop-down menu to the far-right of a recipient
   name.
2. Select Delete.

   ![A screenshot displays an envelope template recipient with the Edit and Delete dropdown menu highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/envelopetemplateeditrecipient.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTczNzYsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.dAabmSpgt3Qp40pip7Tjw4fuXZnrnBPALr26OPmYHp8&_LANG=enus "A screenshot displays an envelope template recipient with the Edit and Delete dropdown menu highlighted.")

   The recipient no longer displays on the page.
3. Select the Save and Close button at the top-right of the
   page.

   You have completed the process. You have removed the recipient from the
   template.
