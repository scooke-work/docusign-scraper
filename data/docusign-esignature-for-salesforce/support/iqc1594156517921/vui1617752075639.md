# Add Salesforce Files to a Docusign Negotiate Envelope

Learn how to access the Add from Salesforce button in the Negotiate Send for Signature
experience.

Follow this procedure to access the Add from Salesforce button in the
Negotiate Send for Signature experience:

1. Open a Salesforce record and select the Negotiate Share
   button ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/negotiateactionsarrow.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTczMDMsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.1mO-S5mg-b714AWgrMO5WWWnzcCEIAh3XvA9bV6Jj4g&_LANG=enus)
   next to a file.
2. Select Send for Signature from the drop-down menu. The
   Send with Docusign page opens.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Send-for-eSignature.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTczMDMsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.1mO-S5mg-b714AWgrMO5WWWnzcCEIAh3XvA9bV6Jj4g&_LANG=enus)
3. Select Add from Salesforce.
4. Add a file using any method. You can add up to ten documents.

   - Upload Documents: Follow the prompts to add files from your
     device.
   - Owned by Me: Select check boxes to add files with you listed as
     Owner.
   - Shared with Me: Select check boxes to add documents from Salesforce
     files that others have shared with you.
   - Recent: Select check boxes to add documents from recently-viewed
     files.
   - Following: Select check boxes to add files that you are
     following.
   - Libraries: Open Salesforce Asset Libraries
     and select check boxes to add files.
5. Select the Add button.

   You have completed the process. You have added a Salesforce document to an envelope
   in the Docusign Negotiate experience.
