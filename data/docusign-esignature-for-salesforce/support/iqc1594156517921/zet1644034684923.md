# Use a Place Fields Envelope Template Button to Send an Envelope From a Salesforce Record

Learn how to use a place fields envelope template button to create and send an envelope
from a Salesforce record.

You can use a place fields
envelope template button to create and deliver a Docusign envelope from
Salesforce.

Your administrator builds place fields envelope templates and adds
custom buttons to your Salesforce layouts.

Place fields templates are
pre-built with documents and recipients already added. This speeds up the envelope
creation and sending process.

You only have to add Docusign and Salesforce
fields to the envelope before sending.

Use this procedure to send an
envelope using a place fields envelope template:

1. Open a Salesforce record (Account, Opportunity, and so on.)
2. Select the place fields envelope template button. The place fields page
   opens.

   Note: Your administrator
   chooses the button name. It may be hidden in the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyMjMsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.uvVSzZyyj2C9141RyPyF1BGWGZlz0ujlH5zZG5Qw8UQ&_LANG=enus) menu.
3. **(Optional)** Follow these steps if you need to add Docusign fields:

   1. Select the Standard Fields tab ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/envelopetemplatestandardsfields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyMjMsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.uvVSzZyyj2C9141RyPyF1BGWGZlz0ujlH5zZG5Qw8UQ&_LANG=enus) in the left
      column. The Docusign Fields menu opens.
   2. Select a Docusign field (Signature,
      Initial, and so on.)
   3. Drag the field to the document.![Add Docusign fields](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Add-DocuSign-fields.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyMjMsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.uvVSzZyyj2C9141RyPyF1BGWGZlz0ujlH5zZG5Qw8UQ&_LANG=enus "Add Docusign fields")
4. **(Optional)** Follow these steps if you need to add Salesforce
   fields:

   1. Select the Merge Fields button ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/envelopetemplatemergefields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyMjMsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.uvVSzZyyj2C9141RyPyF1BGWGZlz0ujlH5zZG5Qw8UQ&_LANG=enus) from the left
      column. The Salesforce Merge Fields menu opens.
   2. Select a field from the menu. You can expand arrows to find more fields,
      or enter text in the Search Fields bar.
   3. Drag the field to the document.![Add Salesforce fields](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Add-Salesforce-fields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyMjMsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.uvVSzZyyj2C9141RyPyF1BGWGZlz0ujlH5zZG5Qw8UQ&_LANG=enus "Add Salesforce fields")
5. (Optional):

   - Recipient Preview: Select this button to see what
     the generated document will look like.
   - Switch recipient view: Follow these steps to add
     fields for more recipients:
     1. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyMjMsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.uvVSzZyyj2C9141RyPyF1BGWGZlz0ujlH5zZG5Qw8UQ&_LANG=enus) next to the recipient name in the upper-left of
        the page.
     2. Select another recipient from the drop-down menu.
     3. Drag Docusign fields for this recipient to the document.

        Note: Each recipient's fields are represented by a separate color
        (yellow, blue, and so on.)![Switch recipients](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Switch-recipients.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcyMjMsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.uvVSzZyyj2C9141RyPyF1BGWGZlz0ujlH5zZG5Qw8UQ&_LANG=enus "Switch recipients")
6. Select the Send button at the upper-right of the
   page.

   A message opens: Sent for Signature. The record
   page refreshes.

You have completed the process. You have created and sent an
envelope using a place fields envelope template.
