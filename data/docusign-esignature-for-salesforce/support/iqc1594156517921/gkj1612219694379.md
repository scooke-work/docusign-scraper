# Add a Filter for Related List Recipients in an Envelope Template

You can add filters to Related Lists recipients in the Add Recipient dialog box of a Docusign envelope template. This allows you to only add Related List recipients to an envelope if
certain conditions are met.

You can add filters to related lists recipients in the Add Recipient dialog box of a Docusign envelope template. This allows you to only add related list recipients to an
envelope if certain Salesforce record conditions are met.

There are two types of filter groupings:

- Any: "OR" statements that apply the filter if any one of your
  conditions is met:
  ![A screenshot displays the Related List Any dialog box options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Related-List-Any-modal-options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTczNjEsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.dmBlt-eTes6L2FfOq_ESIp4YsmDcM5rPmzPMXcTaLB8&_LANG=enus)
- All: "AND" statements that apply the filter only if all of the
  conditions are met:
  ![A screenshot displays the Related List All dialog box options](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Related-List-All-modal-options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTczNjEsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.dmBlt-eTes6L2FfOq_ESIp4YsmDcM5rPmzPMXcTaLB8&_LANG=enus)

Table 1. ADD FILTER ICONS

| Icon | Name | Action |
| --- | --- | --- |
|  | Copy Row | Copy a row of conditions and paste it below |
|  | Delete Row | Delete a row of conditions |
|  | or | Add a new blank row of "OR" conditions |
|  | and | Add a new blank row of "AND" conditions |

Table 2. ADD FILTER OPTIONS

| Name | Action |
| --- | --- |
| All | A group of conditions that add Related List recipients if all listed conditions are met |
| Any | A group of conditions that add Related List recipients if any listed conditions are met |
| equals | A Salesforce field exactly matches your condition |
| contains | Any part of a Salesforce field contains your condition |
| starts with | A Salesforce field begins with your condition |
| ends with | A Salesforce field ends with your condition |
| does not equal | A Salesforce field is not present in your condition |
| greater than | A Salesforce field (number, date, currency) is greater than your condition |
| less than | A Salesforce field (number, date, currency) is less than your condition |
| less than or equals | A Salesforce field (number, date, currency) is less than or equals your condition |
| greater than or equals | A Salesforce field (number, date, currency) is greater than or equals your condition |
| APPLY | The Any/All drop-down menu header |
| IF | The first condition you add |
| OR IF | Any additional "OR" conditions you add |
| AND IF | Any additional "IF" conditions you add |
