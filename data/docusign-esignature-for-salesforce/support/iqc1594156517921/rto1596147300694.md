# Edit the Details of an Envelope Template Recipient

You can edit the details of recipients in an existing envelope template at any time.

Use this procedure to edit the details of recipients in an existing envelope template:

1. Select from the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTczNTgsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.2jgK8HgA0sG0O3QY9gDkGWMopOhVAAQaijza92kiE9M&_LANG=enus) next to a recipient
   name and choose Edit.

   ![A screenshot displays an envelope template recipient with the Edit and Delete dropdown menu highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/envelopetemplateeditrecipient.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTczNTgsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.2jgK8HgA0sG0O3QY9gDkGWMopOhVAAQaijza92kiE9M&_LANG=enus "A screenshot displays an envelope template recipient with the Edit and Delete dropdown menu highlighted.")

   The Add Recipient dialog box opens

   Note: You can also select the name link (i.e. Owner) to edit the
   recipient.
2. Edit recipient details or actions.
3. Select the Add button. The Add Recipient dialog box
   closes.
4. Select Save and Close.

   You have completed the process. You have edited recipient details.
