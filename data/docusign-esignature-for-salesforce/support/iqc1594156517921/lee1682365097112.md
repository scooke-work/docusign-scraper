# Use Envelope Templates to Create and Send a Standard Envelope

Learn how to add envelope template content to a standard Send with Docusign
envelope.

A Docusign envelope is a digital container you build that holds
documents, recipients, and signature fields. Recipients sign the envelope by
completing the fields assigned to them. Docusign saves the document to the
Salesforce record when all recipients finish signing.

Salesforce users have two options for sending envelopes:

[Docusign envelope template](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=zgh1596562751041.html&_LANG=enus "This guide demonstrates the process for sending a document for eSignature using an envelope template custom button.")
:   An administrator builds this three-stage sending experience with documents,
    recipients, and signature fields. Senders can add to or edit the contents of
    the envelope.

[Standard Send with Docusign template](https://support.docusign.com/s/document-item?bundleId=iqc1594156517921&topicId=gsh1675832324336.html&_LANG=enus "Learn to send an envelope from a Salesforce record using a standard Send with Docusign template.")
:   This is a three-stage sending experience with no pre-built content. Senders
    select a Send with Docusign button and add documents,
    recipients, and signature fields.

You can also add content from envelope templates to a standard Send with
Docusign envelope. Adding templates lets you combine the content of
one or more envelope templates in a single envelope. Senders save time by adding
pre-built content, while keeping the flexibility that a Send with Docusign template
provides.

Use this procedure to add Docusign envelope template content to a standard Send with
Docusign envelope:

1. Select the Send with Docusign button in the upper right
   of a Salesforce record.    1. Select Select Template. The Select
      Docusign Template dialog box opens.
   2. Select the check box next to each template you want to add.
   3.    envelope](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=dad1595995063747.html&_LANG=enus).

   envelope](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=psq1595994131623.html&_LANG=enus).

You have completed the process. You have built and sent an
envelope by adding envelope templates to a standard envelope.
