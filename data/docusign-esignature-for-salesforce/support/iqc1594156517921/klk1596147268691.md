# View a Diagram of the Recipient Signing Order

You can select the View button to see a diagram of the recipient signing order, from
start to completion. This option gives administrators a quick snapshot of the signing
process.

You can select the View button to see a
diagram of the recipient signing order, from start to completion. This option gives
administrators a quick snapshot of the signing process.

Use this procedure to
view the Signing Order Diagram of an existing envelope template:

1. Select the View button.

   The Signing Order Diagram dialog box opens.
2. Select the X button in the upper-right of the dialog box to
   close the diagram.

   ![Signing Order Diagram](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/envelopetemplaterecipientsigningorder.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTczNjksInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.Lc9u4XsaAo_MDlX3B9cPrfOUp4n_9Y1LcwMrM-M1Q38&_LANG=enus "Signing Order Diagram")
3. Select the Save and Close button to exit the
   template.

   You have completed the process. You have viewed the recipient signing
   order.
