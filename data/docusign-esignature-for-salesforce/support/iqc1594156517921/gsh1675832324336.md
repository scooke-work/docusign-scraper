# Create and Send a Standard Send With Docusign Envelope

Learn to send an envelope from a Salesforce record using a standard Send with Docusign template.

A Docusign envelope is a digital container you build that holds documents, recipients, and signature fields. Recipients sign the envelope by completing the fields assigned to them. Docusign saves the document to the Salesforce record when all recipients finish signing.

You can create and send envelopes from Salesforce records. You can add recipients and documents manually, from a template, or dynamically from Salesforce records, users, or contacts. Docusign saves the signed document to the Salesforce record when all recipients complete their assigned fields, .

Use this procedure to add documents or templates to a standard Send with Docusign envelope:

1. Select the Send with Docusign button in the upper right of a Salesforce record.    The Documents page opens, displaying the latest ten documents from the record's Notes & Attachments.
2. Complete the actions on the Documents page:

   1. Choose any method to add documents to your envelope:
      - Documents: Select the check box for up to ten documents
      - Upload: Add a document from your device
      - [Add from Salesforce](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=qva1617752033779.html&_LANG=enus): Add documents from Salesforce Files
      - [Select Template](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=fne1682396685506.html&_LANG=enus): Add documents, recipients, and signature fields from Docusign envelope templates
   2. Select Next. ![A screenshot displays the Documents page with a document selected.](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Add-documents.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcxOTMsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.5gnIsFvmZBaWenpKwcpA2goV9XwjCyBa-6WuCVimWno&_LANG=enus "A screenshot displays the Documents page with a document selected.")

   The Recipients page opens.
3. Complete the actions on the Recipients page:

   1. Select Add Recipient.
   2. Select a recipient type from the left column:
      - From Salesforce: Enter the name of a Salesforce user or contact
      - By Name & Email: Enter any name and email address
      - Signing Group: Enter the name of a signing group from your Docusign account
   3. Action: Select a recipient action:
      - Needs to Sign: Recipient must sign the document
      - Receives a Copy: Recipient receives a copy of the document
      - Needs to View: Recipient must view the document
      - Specify Recipients: Recipient manages the signing process for other recipients
      - Allow to Edit: Recipient can make changes to the document
      - In Person Signer: Recipient is the host of a multi-recipient signing session where all signers are in the same room
   4. (Optional) Add recipient options: [Recipient Sending Options: Access Code, In Person Signing, Sign Now, SMS, and More](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=xch1620151662151.html&_LANG=enus).
   5. Select Select.
   6. Envelope Custom Fields: Add values for all required fields. Required fields are marked with asterisks \*.

      Note: You only see Envelope Custom Fields if your administrator has created fields in the connected Docusign account.
   7. Select Next.

   The Prepare & Send page opens.
4. Complete the actions on the Prepare & Send page:

   1. ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/envelopetemplatestandardsfields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcxOTMsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.5gnIsFvmZBaWenpKwcpA2goV9XwjCyBa-6WuCVimWno&_LANG=enus) Standard Fields: Drag Docusign fields to the document.
   2. ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/envelopetemplatemergefields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcxOTMsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.5gnIsFvmZBaWenpKwcpA2goV9XwjCyBa-6WuCVimWno&_LANG=enus) Merge Fields: Drag Salesforce fields to the document.
   3. (Optional) Select another recipient from the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcxOTMsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.5gnIsFvmZBaWenpKwcpA2goV9XwjCyBa-6WuCVimWno&_LANG=enus) menu in the upper left and drag Docusign fields to the document.

      Note: Each recipient's fields are a different color (yellow, blue, and so on.)

      ![A screen capture displays switching to a new recipient and dragging fields to the document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Switch-recipients.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTcxOTMsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.5gnIsFvmZBaWenpKwcpA2goV9XwjCyBa-6WuCVimWno&_LANG=enus "A screen capture displays switching to a new recipient and dragging fields to the document.")
5. Select SEND.

You have completed the process. You built and sent an envelope with a standard Send with Docusign template.
