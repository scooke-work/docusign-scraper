# Place Fields Actions: Thumbnails, Document Settings, and Upload a Document

You can use the Thumbnails button on the Place Fields page to see
thumbnail previews of each page of the template document. You can also access a menu of
actions for the Place Fields step.

## Upload a Document menu

The Edit Documents option opens the Upload a Document page,
where you can edit or upload content.![A screenshot displays document options, including Set as Supplement, Replace, Download, Delete, Rename, and View.](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Upload_a_doc.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc0NDUsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.Jae09GkvziHf3griynY1e9Drksadta75SXEnmLYNnh4&_LANG=enus)

Table 3. Upload a Document Page Options

| Menu Item | Action |
| --- | --- |
| **Get From Cloud** | Add documents from cloud storage (Google Drive, Box, Dropbox, OneDrive, and so on) |
| **Upload** | Add documents from your device |
| **More Options** | Menu for: Set as Supplement, Replace, Download Document, Rename Document, Delete Document, View Document |
| **Set as Supplement** | Switch documents from primary (active documents in the signing process) to supplementary (passive material) |
| **Replace** | Remove the document and upload a replacement from your device |
| **Download Document** | Download a copy of the document to your device |
| **Rename Document** | Change the document name |
| **Delete Document** | Remove the document from the template |
| **View Document** | Open a preview of the document |
