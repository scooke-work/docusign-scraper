# Place Fields Options: Actions Button

You can use the Actions button on the Place Fields page to access a menu of Docusign and
Salesforce options.

Place Fields page to access a menu of Docusign and Salesforce options.

Options include:

Table 1. Actions Button Options

| Menu Item | Action |
| --- | --- |
| **Discard Changes** | Undo all actions taken in the field placement process |
| Delete | Delete the envelope template and all attached documents |
| Edit Message | Change the email subject and email message seen by recipients |
| Edit Recipients | Edit details and actions of existing recipients or add new recipients |
| Edit Documents | View, replace, download, rename, delete, upload, or switch documents from primary (active documents in the signing process) to supplementary (passive material) |
