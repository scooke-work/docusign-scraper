# Configure Standard eSignature Fields

You can customize Docusign eSignature fields that you add in the template creation
process.

You can customize Docusign eSignature fields that you add in the template creation process.
Options include:

- Required
- Read Only
- Add additional signers
- Add tooltips
- Customize data labels
- Add validation types
- Add character limits

Here's how to access the eSignature fields page in an existing Envelope Template:

1. Access the Envelope Template Place Fields tab:
   1. Select the Docusign Envelope Templates tab.
   2. Select from the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc0MjksInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.fIKi_bY8zIjL9kOH87onAjfIPUlIOyIGA2ub2E180RY&_LANG=enus) drop-down menu to the far-right of a
      template name.
   3. Select Edit from the menu. The Docusign Envelope
      Template creation page opens.
   4. Select the Place Fields tab.![Place Fields tab](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Place-Fields-tab.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc0MjksInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.fIKi_bY8zIjL9kOH87onAjfIPUlIOyIGA2ub2E180RY&_LANG=enus "Place Fields tab")
2. Use the actions in the menu.
3. Select the Save button when changes are complete.

   **Result**:
   The process is complete. You have added or edited fields in your Envelope

Here are the standard Docusign signature fields and how to configure them for your
Envelope Template:

    A banner notifies: Template (your template name)
    updated.

**Result:** The process is complete. Each recipient gets an envelope specifically

**SIGNATURE FIELD**

![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/genacesignature.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc0MjksInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.fIKi_bY8zIjL9kOH87onAjfIPUlIOyIGA2ub2E180RY&_LANG=enus)

Add this field to the place on a document that the recipient will add their
eSignature. The field can be customized to toggle to a different recipient, be
required or optional, add a tooltip and add a custom data label.

Select the Signature field on the template document. The Signature
dialog box opens.

**Options available:**

- Signer Select the drop-down to select the signer to which
  the field will apply. (Signer 1, Signer 2, etc.)
- Required Field: Uncheck box to make the field optional
- More Details: Select button at the bottom to expand the
  list of options![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/genacemoredetails.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc0MjksInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.fIKi_bY8zIjL9kOH87onAjfIPUlIOyIGA2ub2E180RY&_LANG=enus)
- **Tooltip**: Enter custom text to show the user the purpose of the
  field.
- Data Label: Link Docusign fields. Data entered in one linked field
  populates to the others.
- Less Details: Select button to collapse the added
  options.
- X: Select the X in upper-right of
  the dialog box to close the options list.

TEXT FIELD

![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/genacetextfield.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc0MjksInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.fIKi_bY8zIjL9kOH87onAjfIPUlIOyIGA2ub2E180RY&_LANG=enus)

optional, be marked read-only, add a character limit, character type validation, add
a tooltip and add a custom data label.

  editing.

  recipient can type.
