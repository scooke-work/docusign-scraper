# Delete a Document Uploaded to an Envelope Template

You can delete documents from an envelope template at any time.

You can delete documents from an envelope template at any time.

Warning: If you delete a document that
contains signature fields for one of the template recipients, those fields will need
to be re-added to another document in the template.

**Example**: You delete a
document containing **Sender 1**'s signature fields. The signature process
will fail unless **Sender 1** is removed from the envelope template
recipients list OR **Sender 1**'s fields are added to another document in the
template

Use this procedure to delete a document from your envelope template:

1. Select from the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTczMjQsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.k7rLlEseMgGroZVZzmx2XwnxIipiYQ7gjeR6e4aNHtQ&_LANG=enus) drop-down menu to the right of a document.
2. Select Delete from the options.

   ![Delete uploaded document](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/envelopetemplatedocumentoptions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTczMjQsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.k7rLlEseMgGroZVZzmx2XwnxIipiYQ7gjeR6e4aNHtQ&_LANG=enus "Delete uploaded document")

   The document disappears from the page.
3. Select Save and Close.

   You have completed the process. You have deleted the uploaded
   document.
