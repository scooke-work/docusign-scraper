# Add Salesforce Files to a Standard Send With Docusign Envelope

Learn how to access the Add from Salesforce button in the
standard Send with Docusign signature experience or the Negotiate action menu.

Use this procedure to access the Add from Salesforce button in the
standard Send with Docusign signature experience:

1. Open a Salesforce record (Account, Opportunity, and so on.)
2. Select the Send with Docusign button in the upper-right
   of the page.

   Note: It may be hidden in
   the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTczMDEsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.k7lR_HDGuAhImDruROsBIuPISLIgmZF8tHs1uwQs37M&_LANG=enus)
   menu.

   ![Send with Docusign button](https://docusign-be-prod.zoominsoftware.io/api/bundle/iqc1594156517921/page/images/Send-with-DocuSign-button.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTczMDEsInNoZWFmIjoiaXFjMTU5NDE1NjUxNzkyMSJ9.k7lR_HDGuAhImDruROsBIuPISLIgmZF8tHs1uwQs37M&_LANG=enus "Send with Docusign button")

   The Documents page opens.
3. Select the Add from Salesforce button at the upper-right
   of the page.
4. Add a document using any method. You can add up to ten documents.

   - Upload Documents: Follow the prompts to add files
     from your device
   - Owned by Me: Select check boxes to add files with
     you listed as Owner
   - Shared with Me: Select check boxes to add
     documents from Salesforce files that others have shared with you
   - Recent: Select check boxes to add documents from
     recently-viewed files
   - Following: Select check boxes to add files that
     you are following
   - Libraries: Open Salesforce Asset
     Libraries and select check boxes to add files
5. Select the Add button.

   You have completed the process. You have added a Salesforce document to
   an envelope in the standard Send with Docusign experience.
