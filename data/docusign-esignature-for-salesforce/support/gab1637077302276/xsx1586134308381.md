# Gen Template Advanced Options: Agreement Cloud Editor

The Agreement Cloud Editor is an in-browser option for creating and editing Docusign Gen for Salesforce templates.

The Agreement Cloud Editor is an in-browser option for creating Gen templates without using
Microsoft Word. You can quickly add Docusign and Salesforce fields to a
template, without copying and pasting XML tags. You can also Download Agreement Cloud Editor
templates as Word documents.

Administrators can use these advanced features to customize their Agreement Cloud Editor Gen
templates:

- [Download Agreement Cloud Editor Documents as Word Files](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=crs1588512498923.html&_LANG=enus "Learn how to download Agreement Document Format Gen contracts as Microsoft Word documents.")
- [Customize Agreement Cloud Editor Template Text](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=iat1586134315670.html&_LANG=enus "You can customize Agreement Cloud Editor template documents to change font size and style, add sections of conditional content and add tables.")
- [Preview an Agreement Cloud Editor Gen Template](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=hht1586134306427.html&_LANG=enus "Learn how to see what a document generated from your template looks like.")
- [Add Child Objects to an Agreement Cloud Editor Template](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=qsk1586134322865.html&_LANG=enus "Learn to add up to four levels of child objects and two levels of related lists.")
- [Add Conditional Logic to an Agreement Cloud Editor Template](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=wqi1586134323385.html&_LANG=enus "Learn to format Agreement Cloud Editor templates to display, hide, or change content depending on conditions.")
- [Learn About Adding Tables to Agreement Cloud Editor Templates](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=zeh1586134329665.html&_LANG=enus "Learn to add a table to an Agreement Cloud Editor template.")
