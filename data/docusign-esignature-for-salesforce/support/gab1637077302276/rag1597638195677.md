# Fix Account-Wide Issues Using the Docusign Apps Launcher Troubleshooting Page

Docusign Apps Launcher TroubleShooting is a Salesforce page that
allows administrators to troubleshoot and resolve issues with these Docusign apps.

Docusign Apps Launcher is a package of apps that you install in your Salesforce organization.
These apps include Docusign Gen, Negotiate, CLM, CLM Essentials, and eSignature for
Salesforce.

Docusign Apps Launcher TroubleShooting is a tool that lets
administrators troubleshoot and resolve issues with Docusign apps. The page is available
in Docusign Apps Launcher version 2.5 and higher. Salesforce administrators with
Docusign administrator permissions can use it.

The TroubleShooting page helps you resolve common administration issues. You can fix complex
problems by selecting the Execute button next to a task.

Here are the functions available on the Troubleshooting page:

[Disconnect Account and Preserve User Settings](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=oks1610512362945.html&_LANG=enus "Learn how to disconnect Docusign from Salesforce while preserving user settings with the Docusign TroubleShooting tool.")
:   Disconnect your Docusign account from Salesforce without deleting your Docusign
    users.

[Disconnect Account and Remove Users](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=vli1610514116162.html&_LANG=enus "Learn how to disconnect a Docusign account and remove users with the Docusign TroubleShooting tool.")
:   Disconnect your Docusign account and remove all Docusign users and settings from
    Salesforce.

[Clear User Settings](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=ncl1610479844444.html&_LANG=enus "Learn to fix the CUSTOM_SETTINGS_LIMIT_EXCEEDED issue with the Docusign TroubleShooting tool.")
:   Delete cached user access tokens to free up Salesforce storage space.

[Folder Permissions](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=sqj1610517836441.html&_LANG=enus "Learn how to reset folder permissions for Docusign content with the Docusign TroubleShooting tool.")
:   Trigger a root folder permissions update. This update resets user permissions for all
    content.

[Refresh Products](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=agy1623993092688.html&_LANG=enus "Learn to activate and update Docusign for Salesforce apps and settings using the Refresh Products feature.")
:   Activate and update apps and settings connected to your Salesforce organization.

[Sync Domains](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=ynp1680122164220.html&_LANG=enus "Learn to fix domain integration issues using the Sync Domains feature. Sync Domains fixes issues with Enhanced Domains and any changes that you make to your Salesforce domain.")
:   Resolve errors related to Enhanced Domains or domain changes that you make.

[Manage CLM Integration User](https://support.docusign.com/s/document-item?bundleId=ysk1643667858834&topicId=yyh1698871133582.html&_LANG=enus)
:   Reset the token used by your Docusign CLM integration user. This feature is only available
    for Docusign CLM for Salesforce accounts.
