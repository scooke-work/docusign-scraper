# Clear User Settings Using the Apps Launcher TroubleShooting Tool

Learn to fix the CUSTOM\_SETTINGS\_LIMIT\_EXCEEDED issue with the Docusign TroubleShooting tool.

Administrators can use the Docusign Apps Launcher
TroubleShooting tool to fix issues with Docusign apps. The TroubleShooting tool is available in Docusign Apps Launcher version 2.5 and higher. Salesforce
administrators with Docusign administrator permissions can use it.

Salesforce Experience Cloud users sometimes see this error message when sending
envelopes: CUSTOM\_SETTINGS\_LIMIT\_EXCEEDED.

Users with too many stored access tokens see this error message. Access tokens can be stored
in custom settings without users or administrators knowing. Excessive access tokens
can violate Salesforce storage limits and prevent Experience Cloud users from
sending Docusign envelopes.

Administrators can use the Clear User Settings feature of the
TroubleShooting tool to delete access tokens for all users. Clear User
Settings deletes cached access tokens to free up storage.

Follow this procedure to use the Clear User Settings feature:

1. Access the Visualforce Pages page as a Salesforce
   administrator:

tokens. Salesforce Experience Cloud users can send envelopes again.
