# Add a Standard Table to an Agreement Cloud Editor Template

You use the table button to add a standard table to an Agreement Cloud Editor template.

You can add a standard or dynamic table to your Agreement
Cloud Editor template. Use this procedure to add a standard table to an Agreement Cloud Editor
Gen template using the table button.

Note: You cannot add Salesforce data tags to a standard table. Follow this link to [add Salesforce data tags to dynamic
tables](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=pzt1586134333317.html&_LANG=enus "Learn to add a dynamic table to an Agreement Cloud Editor template.").

1. Open the Gen template Create tab as a Salesforce administrator:
   1. Select the App Launcher grid ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/app-launcher-grid.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg5NzYsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.MHGf_1vZFzmz0APZqRp-YAL13hUCZgwMaja64IfkC3E&_LANG=enus) and go to View All >  Docusign Apps Launcher.
   2. Select the Docusign Gen Templates tab.
   3. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg5NzYsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.MHGf_1vZFzmz0APZqRp-YAL13hUCZgwMaja64IfkC3E&_LANG=enus) next to a template and choose
      Edit.

   The Gen template page opens to the Create tab.
2. Place the cursor where you want to add a table and select the
   Table button ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/genacetable.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg5NzYsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.MHGf_1vZFzmz0APZqRp-YAL13hUCZgwMaja64IfkC3E&_LANG=enus).

   A dropdown menu displays a table grid.
3. Move the cursor to the right to add columns, and down to add rows. The cells change
   from white to blue.

   Note: This example creates a table with three columns and four
   rows.![A screenshot displays a table grid with three columns and four rows of blue squares.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/genacetableadd.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg5NzYsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.MHGf_1vZFzmz0APZqRp-YAL13hUCZgwMaja64IfkC3E&_LANG=enus "A screenshot displays a table grid with three columns and four rows of blue squares.")
4. Enter text to the rows and columns of the inserted table.
5. Select Next and Publish.

   A message displays: Template (your template name)
   updated.

You have completed the process. You have added a
standard table to a Gen template.
