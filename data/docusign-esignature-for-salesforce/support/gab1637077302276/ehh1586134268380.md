# Enable Translations for Your Salesforce Organization

Learn how to use Translation Workbench to enable choosing a default Salesforce
language.

Use this procedure to enable Translation Workbench in your organization and let users
choose their default Salesforce language.

1. Access Translation Settings as a Salesforce administrator:
   1. Select the gear ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc3NDgsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.AK9QNhrxkGJYIJ8dYRmzgbXWniVxGIPVzW0-suf8YrA&_LANG=enus) icon and choose Setup.
   2. Enter and select Translation Settings in the Quick Find
      field.
2. Select Enable on the Translation Settings page.

   ![A screenshot of the Translation Settings page displays the Enable button highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/Enable-Salesforce-Translation-settings.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc3NDgsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.AK9QNhrxkGJYIJ8dYRmzgbXWniVxGIPVzW0-suf8YrA&_LANG=enus "A screenshot of the Translation Settings page displays the Enable button highlighted.")

   The Translation Workbench page opens. A message displays: The
   Translation Workbench is currently enabled for your
   organization.
3. (Optional): Select Add to add additional
   languages to the organization.

You have completed the process. Users in your
Salesforce org can switch languages and view Docusign Apps Launcher content in any of the
enabled languages.
