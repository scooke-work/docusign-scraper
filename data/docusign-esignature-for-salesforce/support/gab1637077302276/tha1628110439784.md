# Remove Docusign App Access From a Salesforce User

Learn to remove Gen, CLM, Negotiate, and eSignature access from Salesforce
users.

You can use the Remove command to remove Docusign app access from
Salesforce users.

- Remove removes a Salesforce user's access to Docusign apps. This command does not remove users from the connected Docusign account or
  delete Salesforce users.
- You can not remove and close your user while logged on.
- You can not remove and close the only eSignature administrator user.

Use this procedure to remove Gen, CLM, Negotiate, and eSignature access from
Salesforce users:

1. Select the check box next to each user you want to remove.
2. Select Remove from the Add
   Users
   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/CLM-Files-Lightning-action-arrow.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc3MTAsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.gQYy5hZNCQUqa0Sk-Cap-TU0_k8bFLHDQdo-IDReZHU&_LANG=enus) drop-down menu.
3. Select Remove.

You have completed the process. You removed a user's access to the
Gen, Negotiate, and eSignature for Salesforce apps. The user remains in the Salesforce
organization and the connected Docusign account.
