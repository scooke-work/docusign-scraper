# Reset Folder Permissions Using the Apps Launcher TroubleShooting Tool

Learn how to reset folder permissions for Docusign content with the
Docusign TroubleShooting tool.

Administrators can use the Docusign Apps Launcher
TroubleShooting tool to fix issues with Docusign apps. The TroubleShooting tool is available in Docusign Apps Launcher version 2.5 and higher. Salesforce
administrators with Docusign administrator permissions can use it.

You can reset folder permissions for Docusign content in your Salesforce
organization using the Folder Permissions feature of the Docusign TroubleShooting tool. Administrators can select one
button and quickly reset all Docusign folders to their default
permissions, removing all custom security changes.

Use this procedure to use the Folder Permissions feature:

1. Access the Visualforce Pages page as a Salesforce
   administrator:
   1.
