# Step 2: Remove Apps Launcher Components from Your Layouts

Learn to remove all Docusign Apps Launcher components from Salesforce
layouts.

Before you can uninstall Apps Launcher, you must remove all Docusign
components that are present on your layouts.

Use this procedure to remove Docusign components from your layouts:

1. Select the gear ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc4MzQsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.UUZAKHfYLLVUlWyUCKMW6bcPOmcVnbW40OfG6fFyj-o&_LANG=enus)and choose Edit Page from the dropdown
   menu.

   The Lightning App Builder page opens.
2. Remove all Docusign components from the layout:
   1. Select a Docusign component and select the trash icon
      ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/Screen-Shot-20210615-at-123257-AM.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc4MzQsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.UUZAKHfYLLVUlWyUCKMW6bcPOmcVnbW40OfG6fFyj-o&_LANG=enus) that displays. ![A screenshot displays the trash icon highlighted on the envelope status component.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/A-screenshot-displays-the-trash-icon-highlighted-for-the-Docusign-Files-component.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc4MzQsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.UUZAKHfYLLVUlWyUCKMW6bcPOmcVnbW40OfG6fFyj-o&_LANG=enus "A screenshot displays the trash icon highlighted on the envelope status component.")
   2. Select Save.
   3. Repeat this procedure for all Docusign components on
      this layout.

   Salesforce removes the components from the layout.
3. Select the back arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/Screen-Shot-20210615-at-13251-AM.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc4MzQsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.UUZAKHfYLLVUlWyUCKMW6bcPOmcVnbW40OfG6fFyj-o&_LANG=enus) when you have removed all components.
4. Repeat this procedure for each Salesforce layout that contains Docusign Apps Launcher components.

You have completed step two of the process. You have
removed all Docusign Apps Launcher components from your Salesforce
layouts. Move to [Step 3 (Optional): Remove Apps Launcher Custom Buttons from Your Layouts](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=cif1623740064281.html&_LANG=enus "Learn to remove Docusign Apps Launcher custom buttons from your Salesforce layouts.").
