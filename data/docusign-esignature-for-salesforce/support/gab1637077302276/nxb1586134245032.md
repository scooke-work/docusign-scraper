# I Want to Know More About How Gen Template Naming Conventions Work

Learn about the file name logic in the Gen template Upload Documents step.

I am building a Gen template and am confused by the File Suffix
section of the Upload Documents step.

I want to know more about the following terms:

- Generated File Name
- File Suffix
- File Format

## Resolution: Use the Following Example to Understand Gen Template File Naming Conventions

Cause: Some confusion exists about what Gen template file naming
terms mean.

Docusign uses naming conventions to make unique file names for
agreements generated by the same template.

Standard agreement names only include your organization name and the file format, like
.DOCX, or .PDF. You can use naming conventions to add the following information to your
agreement names:

- Main Data Source Name
- Date
- Main Data Source Name and Date
- Microsoft Word
- PDF

This custom information makes it easier to see when your agreements were created and
from what object.

Use this example to understand naming conventions better:

1. Build a Gen template and advance to the Upload Documents
   step.
2. Configure the following settings on your Gen template's Upload
   Documents step:
   1. Generated File Name: Enter Fresh Software
      NDA.
   2. File Suffix: Select Main Data Source Name and
      Date.
   3. File Format: Keep Microsoft
      Word.
