# Install Docusign Apps Launcher in Your Salesforce Organization

Learn how to install the Docusign Apps Launcher package of apps in your Salesforce
organization.

Docusign Apps Launcher for
Salesforce is a package that includes the following Docusign apps:

- [Docusign eSignature for Salesforce](https://appexchange.salesforce.com/appxListingDetail?listingId=a0N30000001taX4EAI&tab=e)
- [Docusign CLM for Salesforce](https://appexchange.salesforce.com/appxListingDetail?listingId=a0N30000004e3VUEAY&tab=e)
- [Docusign Gen for Salesforce](https://appexchange.salesforce.com/appxListingDetail?listingId=a0N3A00000FeGX6UAN&tab=e)

When you install any of these apps, Docusign installs the complete Docusign Apps Launcher package.

Use this procedure to install the Docusign Apps Launcher package of apps in your Salesforce
organization:

1. As a Salesforce administrator, choose a Docusign app from the
   AppExchange:
   1. Select the App Launcher grid ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/app-launcher-grid.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc2MTAsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.Ymr6banIKagVryy2ChuavyiM7WGfPc-KM27JkhACSjE&_LANG=enus) and choose View
      All.
   2. Select Visit AppExchange >  Go to AppExchange.
   3. Enter Docusign eSignature, Gen, or
      CLM in the search field.
   4. Select an app and choose Get it Now when the app page opens.
2. Log in to the AppExchange with Salesforce administrator credentials:

   Note: Enter production credentials even if installing in a sandbox
   organization. These credentials are used to access the AppExchange.

   1. Select Log In on the LET'S GET STARTED
      page.
   2. Select Salesforce on the Hello
      Trailblazer! dialog box that opens.
   3. Enter Salesforce production administrator credentials on the page that opens and
      select Log In.

      The Where do you want to install this package? dialog
      box opens.
3. Choose where to install the package:
   1. Select Install in Production or Install in
      Sandbox.
   2. Select I have read and agree to the terms and
      conditions.
   3. Select Confirm and Install.

      The page refreshes. The Salesforce login page opens.
4. Install the Apps Launcher package:
   1. Enter production or sandbox administrator credentials and select Log
      In.
   2. Select Install for All
      Users.

      Note: If you want to choose ​Install for Admins Only, or ​Install for Specific Profiles​, see this topic about [how Salesforce applies app permissions](https://help.salesforce.com/s/articleView?id=sf.distribution_installing_packages.htm&type=5).
   3. Select Install or Upgrade.
   4. Select Done when the page refreshes.

You have completed the process. You have installed the
Docusign Apps Launcher package of apps. This package includes Docusign eSignature, CLM, and Gen. You receive an email notification when
Salesforce completes the installation.
