# Use Repeat Tags in Gen to Import Data from Multiple Salesforce Objects

Learn to use XML Repeat tags in a Word-based Gen template to import data from multiple
Salesforce objects.

You can use XML Repeat tags in a Word-based Gen template to import data from multiple
Salesforce objects. Repeat tags look for Salesforce field data that exists on more
than one object.

When users generate agreements, Docusign imports field data from each
Salesforce object.

Example: You want to generate agreements that insert a Salesforce
contact that exists on multiple objects. You add Repeat tags to your Gen template to
import data from all instances of certain fields.

A user generates an agreement from your Gen template. The record contains a contact
who is the founder and CEO of a number of companies. She is listed in one parent
account and two child opportunities. Docusign generates an
agreement containing her contact information from all three businesses.

Use this procedure to add Repeat tags to your Gen template:

1. Select a document name.

   The document preview page opens in a new tab or window.
2. Add a table with custom XML to your Word template:
   1. Select Download.
   2. Open the Word document and enter the following XML code: `<#
      <Repeat Select="/Opportunity/Account/Name" /> #> Name:
      <# <Content Select="/Name" /> #><# <EndRepeat/>
      #>`
   3. Add a Repeat tag to import product information along with the name:

      `<# <Repeat Select="/Opportunity/Account/Name" />
      #>`

      Name: `<#
      <Content Select="/Name" />
      #>`

      Age: `<#
      <Content Select= “/Age” />
      #>`

      Companies:

      - `<# <Repeat Select=
        “/Opportunity/Account/Companies” /> #>`
      - Name: `<# <Content Select=
        “/Opportunity/Account/Companies/Name” />
        #>`
      - Products:
        - `<# <Repeat
          Select=”/Opportunity/Account/Companies/CompanyProducts”
          /> #>`
        - Name: `<# <Content
          Select=”/Opportunity/Account/Companies/CompanyProducts/Name”
          /> #>`
        - Founder: `<# <Content
          Select=”/Opportunity/Account/Companies/CompanyProducts/CompanyProductFounder”
          /> #>`
        - `<# <EndRepeat /> #>`
        - `<# <EndRepeat /> #>`
      - `<# <EndRepeat /> #>``<# <EndRepeat/> #>`
   4. Save the changes in Word and return to the Gen Upload
      Documents page.
