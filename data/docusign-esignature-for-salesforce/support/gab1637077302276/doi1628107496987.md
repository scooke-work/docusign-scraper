# Edit the Docusign App Permissions of Salesforce Users

Learn to edit the permissions of Gen, CLM, Negotiate, or eSignature for Salesforce
users.

You can use the Edit Permissions
command to change the Docusign app permissions of Salesforce users.

Use this
procedure to edit the Docusign app permissions of Gen, CLM Negotiate, or eSignature for
Salesforce users:

1. Select the check box next to each user you want to edit.
2. Select Edit Permissions from the Add
   Users
   ![A screenshot displays a downward-facing triangle icon in a square.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/CLM-Files-Lightning-action-arrow.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc3MDcsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.AQzDFmgELwMqjq0hWSNjUvkKzw44LUxjjz51JTGMxfM&_LANG=enus "A screenshot displays a downward-facing triangle icon in a square.") drop-down menu.
3. Change access from the Edit Permissions dialog box that
   opens:

   1. eSignature
      - Administrator: Administrators can create
        templates and manage all Docusign users in Salesforce.
      - User: Users can send documents for
        signature.
   2. CLM
      - Administrator: Administrators can access
        everything in CLM and manage connections with Salesforce. They
        can't manage Docusign users in Salesforce.
      - User: Users can access workflows and
        view, edit, and upload documents.
      - Viewer: Viewers can only preview and
        upload documents.
      - No Access: Users have no access to the
        CLM app
   3. Gen
      - Generator: Generators can create new
        documents and templates.
      - No Access: Users have no access to the
        Gen app.
   4. Negotiate
      - Negotiator: Negotiators can collaborate
        with signers and viewers.
      - No Access: Users have no access to the
        Negotiate app.
   5. (Optional) Update permissions for any existing
      users: Select to update a user's permissions in the
      connected Docusign account.
4. Select Apply.

You have completed the process. You edited the permissions of Gen,
CLM, Negotiate, or eSignature for Salesforce users.
