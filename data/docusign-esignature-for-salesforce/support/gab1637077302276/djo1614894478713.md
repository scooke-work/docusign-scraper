# Create a Document Rule in a Word-Based Gen Template

Learn to create a document rule to enable conditional document generation in a Gen
template:

Use this procedure to set up a document rule to enable conditional document generation in a
Gen template:

1. Select the Upload Documents arrow. The arrow has no text
   until you select it.
2. Select + Add Rule.

   The Document Rule dialog box opens.
3. Create a document rule:
   1. Enter rule name: Enter a descriptive name for
      the rule.
   2. APPLY: Select an option from the dropdown
      menu:

      - All: Docusign triggers
        the rule if a record matches all conditions.
      - Any: Docusign triggers
        the rule if a record matches at least one condition.
   3. IF: Add a condition to your rule:

      1. Select an Option: Choose a Salesforce
         field to start the condition:
      2. Equals: Choose
         Equals or Does Not
         Equal from the dropdown menu.
      3. Enter conditional value: Enter a value
         for the condition. Example: You start a condition with Billing Country >  Equals and enter Brazil to finish
         the condition.
   4. (Optional)
      Duplicate Row:
      Select the copy ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/Copy-Row-Button.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg5ODksInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.5yqHtASr_tgLUbj-d9SksekOTI35Wy-tD9X1EJAg3zE&_LANG=enus) button to copy a
      condition.
   5. (Optional)
      + or if: Select the
      link to add conditions.![](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/Document-Rule.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTg5ODksInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.5yqHtASr_tgLUbj-d9SksekOTI35Wy-tD9X1EJAg3zE&_LANG=enus)
4. Select Save.

You have completed the process. You built a document rule that
selects the correct template based on record conditions.
