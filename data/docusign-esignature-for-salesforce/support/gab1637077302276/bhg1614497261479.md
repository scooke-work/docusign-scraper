# Step 6: (Optional) Automatically Generate and Email Invoices

This step is optional. You can choose to generate and email invoices automatically,
instead of creating a custom button that your users select to generate
invoices.

This step is optional. Use this procedure to generate
and email invoices automatically.

If you choose to automate this task, invoices
automatically generate when an invoice record stage changes to
Posted. Invoices are saved in the Notes &
Attachments section of the record.

If you do not want to
automate invoices, select the Next button on this page to
create a custom button.

Note:

- You can only set up one automated invoice template at a time. If you try to
  set up another, a Skip button displays. Select it to
  move to step seven.
- You can generate a maximum of 1,000 invoices per hour.

Tip:
[This step is explained in
full at the beginning of this video](https://bcove.video/3uKM2F0).

1. Select the Set up automatic invoice PDF generation and save to
   parent invoice record check box.
2. (Optional)
   Send invoices by email: Select this check box to email
   the invoice to recipients. The Add Recipients section opens.

   Note: The Bill to Contact (Order) recipient is added by
   default. You can add up to four more recipients.
3. (Optional) Choose any steps from the Add Recipients
   section:

   - Add Recipient: Select this button and choose
     options from the dialog box that opens:
     - Record Fields: Dynamically add recipients
       from a record’s Bill To Contact or
       Created By field.
     - Name and Email: Add any recipient by
       entering a name and email address
       - Set as Bcc: Select the check box
         to hide this recipient from others
     - Add the recipient to the template:
       - Add: Save the recipient and close
         the dialog box
       - Add & New: Save the recipient
         and add more
     ![Add Recipient dialog box](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/Add-Recipient-modal.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc5MjksInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.hHWifuMJptUGDN8s3Bir_ZTyww4mw40qtdlMqXFmOKQ&_LANG=enus "Add Recipient dialog box")
   - Message to All Recipients:
     - Email Subject: Edit text in the subject
       field.
     - Email Message: Edit text in the message
       field. Use the HTML tag <br> to
       include line breaks in your message.
4. Select Next.

   The Custom Button page opens.

You have completed step six. Move to [Step 7: Create a Custom Button or Finish the Template](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=aup1614501330150.html&_LANG=enus "This step is optional.
