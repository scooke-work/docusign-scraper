# Edit a Gen Template

Learn to edit a Gen template.

You can edit a Gen template to add or edit documents,
Docusign fields, and Salesforce fields. If you edit the name of a
Gen template, also edit the custom button name.

Use this procedure to edit a Gen
template:

1. Select Docusign Gen Templates.
2. Select the arrow ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/Arrowhead.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc4NjIsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.WeAS-1ULXpUg-Ed8Mfi4d1gwDG2Wfj8ycWFf2HkX5g4&_LANG=enus) next to a Gen template and choose
   Edit from the dropdown menu.

   The Docusign Gen template creation page
   opens.

You have completed the task. You have opened a Gen
template for editing. You can add or edit documents, Docusign fields,
and Salesforce fields.
