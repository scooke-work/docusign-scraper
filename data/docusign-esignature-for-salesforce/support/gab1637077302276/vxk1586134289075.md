# Step 1: Choose a Name, Data Source, and Template Editor

Use this procedure to decide the name, Salesforce data source, and editing tool for the
template.

Use this procedure to decide the name, Salesforce
data source, and editing tool for the template.

1. Select Configure at the far right.
2. Select Create a Template Configuration.
3. Complete the following tasks:
   1. Add Template Name: Enter a name for your template.
   2. (Optional) Add Template Description: Add details about the
      template.
   3. Select Data Source: Select a Salesforce object to import
      data from.
   4. Select Template Editor: Select Microsoft
      Word.
   5. Select Create.

You have completed step one. Move to [Step 2: Create Docusign and Salesforce Anchor Text Fields](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=irc1586134297835.html&_LANG=enus "Learn to create Docusign and Salesforce anchor text fields for your Gen template.").
