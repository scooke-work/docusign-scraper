# Create a Gen Template With the Docusign Agreement Cloud Editor

Learn to create a Gen template with the Docusign Agreement Cloud Editor.

You can create Gen templates using the Docusign Agreement Cloud Editor instead of Microsoft
Word. Docusign uses Agreement Document Format (ADF) for generated documents. ADF documents
automatically re-size for phones, tablets, and computers.

The following are the benefits of using the Agreement Cloud Editor to build Gen templates:

- Build and edit templates in the browser instead of using Microsoft Word
- Stay within Salesforce without logging into Docusign
- Import and edit existing Word documents
- Drag Salesforce and Docusign fields to your template without pasting
  XML code
- Add conditional logic and tables to your template

Use this procedure to create a Gen template with the Agreement Cloud Editor:

1. Open the Create a Template Configuration page as a Salesforce
   administrator:
   1. Go to
      ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/salesforce-gear_.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc4NzYsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.Io3GOORt2rPMiFZWYiKCoTVdzv5t8nT_OnRG_VNnhj0&_LANG=enus) > View All >  Docusign Apps Launcher.
   2. Select Document Generation in the left column.
   3. Select Create a Template Configuration.

   The Create New Template dialog box opens.
2. Choose a name, data source, and editor for your template:
   1. Add Template Name: Enter a name for your template.
   2. Select Data Source: Select a Salesforce object to import
      data from.
   3. Select Template Editor: Select Docusign Agreement
      Cloud Editor.
   4. Select Create.

   The Create page opens.
3. Enter text or select Upload to create your template
   document.
4. Add Salesforce fields to your template document:
   1. Select the Salesforce Fields tab ![A screenshot displays an icon with two interlocking blue rectangles.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/genacesalesforcefieldsicon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc4NzYsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.Io3GOORt2rPMiFZWYiKCoTVdzv5t8nT_OnRG_VNnhj0&_LANG=enus "A screenshot displays an icon with two interlocking blue rectangles.") in the left column.
   2. Drag a Salesforce field to the document or place the cursor in the text and select
      a field.![A screen capture displays a cursor dragging Salesforce fields to a document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/ACE-SFDC.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc4NzYsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.Io3GOORt2rPMiFZWYiKCoTVdzv5t8nT_OnRG_VNnhj0&_LANG=enus "A screen capture displays a cursor dragging Salesforce fields to a document.")

   Note: The menu displays fields from your chosen data source and
   related objects. You can select up to four levels of child objects and two levels of
   Related Lists.
5. Add Docusign fields to your template document:
   1. Select the Docusign Fields tab ![A screenshot displays a blue rectangle icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/Screen-Shot-20210303-at-124311-AM.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc4NzYsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.Io3GOORt2rPMiFZWYiKCoTVdzv5t8nT_OnRG_VNnhj0&_LANG=enus "A screenshot displays a blue rectangle icon.") in the left column.
   2. Drag a Docusign field to the document or place the cursor in the text and select a
      field.
   3. Select Next.

   The Deploy page opens.
6. Set File Naming Rules: Choose the naming convention used when
   users generate files from this template:
   1. File Name: Enter the beginning text for all generated file
      names. If you enter Fontara, the file name of each generated
      document begins with the word Fontara.
   2. File Suffix: [Choose the
      end text for all generated file names](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=nxb1586134245032.html&_LANG=enus "Learn about the file name logic in the Gen template Upload Documents step."):

      - Main Data Source Name: Add your customer's name to the
        generated file name
      - Date: Add the document generation date to the generated
        file name
      - Main Data Source Name and Date: Add your customer's name
        and the generation date
7. Add Custom Button: Add a new custom button to your Salesforce
   layout:
   1. Button Label: Enter a descriptive name to demonstrate the
      purpose, like Generate NDA.
   2. Show button in: Select the check box for each layout that
      you want to update.
   3. Select Publish.

   A message displays: Template (name) buttons
   updated.

You have completed the process. You created an
Agreement Cloud Editor Gen template and added a custom button to a Salesforce layout. Approved
Gen users can generate contracts from Salesforce records.
