# Learn to Customize Docusign Fields in a Gen Template

You can customize the fields you add in the template creation process.

You can customize the Docusign fields you add in the template creation
process. You can use the following options:

- Required or read-only
- Add additional signers
- Add tooltips
- Customize data labels
- Add validation types
- Add character limits

You can use the following fields and customizations in your Gen templates:

## Add Recipient Views

Use this Docusign Fields menu to add more recipients and signature
fields. You can create a customized view for each recipient, giving each their own
signature fields.

## Signature

Place this field where the recipient adds their signature to the document. You can
configure the following options:

- Switch to a different recipient
- Required or optional
- Add a tooltip
- Add a custom data label

## Initial

Place this field where the recipient will add their initials. You can configure the
following options:

- Switch to a different recipient
- Required or optional
- Add a tooltip
- Add a custom data label

## Date Signed

Place this field where the signature date will be added. You can configure the
following options:

- Add a tooltip
- Add a custom data label

## Name

Add this field where the recipient name will be placed. You can configure the
following options:

- First name only
- Last name only
- Add a tooltip
- Add a custom data label

## Email

Add this field where the recipient email address will be placed. You can configure
the following options:

- Add a tooltip
- Add a custom data label

## Company

Add this field where the organization name will be placed. You can configure the
following options:

- Switch to a different recipient
- Required or optional
- Mark as read only
- Add a tooltip
- Add a custom data label

## Title

Add this field where the recipient title will be placed. You can configure the
following options:

- Switch to a different recipient
- Required or optional
- Mark as read only
- Add a tooltip
- Add a custom data label

## Text

Add this field where text can be added by the recipient. You can configure the
following options:

- Switch to a different recipient
- Required or optional
- Mark as read only
- Add a character limit
- Character type validation
- Add a tooltip
- Add a custom data label
