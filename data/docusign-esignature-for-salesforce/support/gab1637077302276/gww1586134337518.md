# Edit the Name of a Gen Template

Learn to change the name of a Gen template.

Use this procedure to change the name of a Gen
template:

1. Select Docusign Gen Templates.
2. Select the Name field.
3. Select the Edit
   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/Screen-Shot-20210520-at-12321-AM.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc4NTksInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.nhAS_mi6xRnRip43d8oeyio0qGR6o8_WlwQxD7RApls&_LANG=enus) button.![A screenshot displays the Gen template name next to the edit icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/Screen-Shot-20210520-at-11822-AM.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc4NTksInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.nhAS_mi6xRnRip43d8oeyio0qGR6o8_WlwQxD7RApls&_LANG=enus "A screenshot displays the Gen template name next to the edit icon.")
4. Enter a new name for the template.
5. Press Enter on your keyboard.
6. Select Save.

You have completed the process. You have changed the
name of a Gen template.
