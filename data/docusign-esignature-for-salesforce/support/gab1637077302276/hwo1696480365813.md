# A "Not Found" Error Displays When I Use the Gen Agreement Cloud Editor

Learn to fix the "Not Found" error that displays when you create an Agreement Cloud
Editor Gen template.

I cannot build a Gen template with the Agreement Cloud Editor.

A Not Found error displayed when I selected
Next on the Create page.

## Resolution: Contact Docusign Support

Cause: The Agreement Cloud Editor feature is not turned on
in your Docusign account.

Agreement Cloud Editor is a paid feature and may add cost to your Docusign plan.

Use this procedure to fix the issue:

1. [Contact Docusign support](https://support.docusign.com/s/contactSupport).
2. Add the Agreement Cloud Editor feature to your
   Docusign account.

   You have completed the process. You can build Gen templates with
   the Agreement Cloud Editor.
