# Step 1: Add a Name, Data Source, and Template Editor

Learn to set the main template details: template name, description, and Salesforce data
source (Invoice.)

Use this procedure to enter a template name and choose
Invoice as the Salesforce data source.

Important: To create a Gen for Billing
template, you must select Invoice as the Salesforce data
source.

1. Select Create a Template Configuration at the far right.
2. Complete the following tasks:
   1. Add Template Name: Enter a name for your
      template.
   2. (Optional) Add Template Description: Add details
      about the template.
   3. Select Data Source: Select
      Invoice from the drop-down menu.
   4. Select Template Editor: Select
      Microsoft Word.
   5. Select Create.

You have completed step one. Move to [Step 2: Create Docusign and Salesforce Anchor Text Fields](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=irc1586134297835.html&_LANG=enus "Learn to create Docusign and Salesforce anchor text fields for your Gen template.").
