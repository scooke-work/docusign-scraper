# Create a One-Click Generation and Sending Experience

Learn to build a one-click generation and sending experience for generating a contract
and sending it for signature.

## Before you begin

- Install Docusign Apps Launcher version 5.3 or higher
- [Create a Docusign
  envelope template](https://support.docusign.com/s/document-item?bundleId=srl1586134368658&topicId=uce1617376465289.html&_LANG=enus)
- [Create a Gen template with Microsoft
  Word](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=lig1586134209279.html&_LANG=enus "Learn to build a Gen template using a Microsoft Word document.")

You can build a combined document generation and sending experience. A
generation and sending experience lets your users generate a document and send it
for signature with one click.

Example: As an administrator, you create a Docusign envelope template. You create a Gen template and link to
your envelope template on the Gen template Create Button page. You add a Gen custom
button to a layout.

A user visits a record and selects the Gen button. Docusign
generates an agreement with that record’s data and sends it to all envelope template
recipients.

When all recipients complete the signing process, Docusign saves the agreement to your Docusign account and Salesforce
record.

Note: You can only link templates built on the same Salesforce
object. Linking a Gen template and envelope template built on
Opportunity works. Linking an
Opportunity-based Gen template with an
Account-based envelope template does not work.

Use this procedure to build a generation and sending experience:

1. Change the sending experience of your envelope template to send
   now:
   1. [Access the envelope template Options page as a
      Salesforce administrator](https://support.docusign.com/s/document-item?language=en_US&bundleId=srl1586134368658&topicId=ete1620167625547.html&_LANG=enus).
   2. Sending Experience: Select Send
      Now.
   3. Select Next.
   4. Select Save and Close.
2. Link your envelope template to your Gen template:
   1. [Open the Create Custom
      Button page as a Salesforce
      administrator](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=way1621663319299.html&_LANG=enus "Learn to access the Create Button step of a Microsoft Word-based Gen template.").
   2. Select Launch automatically after
      generation.
   3. Select an envelope template to streamline sending for
      signature: Enter and select the name of a
      send now envelope template.![A screenshot shows Launch automatically after generation selected and the Fontara Mobile NDA template linked.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/A-screenshot-shows-Launch-automatically-after-generation-selected-and-the-Fontara-Mobile-Sales-envelope-template-linked.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc4ODgsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.hv5lWugp1K8KyefxtG5OnLbrGswKYno_v8FZmvIBBCg&_LANG=enus "A screenshot shows Launch automatically after generation selected and the Fontara Mobile NDA template linked.")
   4. Select Save and Finish.

   The Create Button page closes.

You have completed the process. You have linked your Gen template
to a send now envelope template. When users select the custom
button, Docusign generates a document and sends it to the envelope
template recipients.
