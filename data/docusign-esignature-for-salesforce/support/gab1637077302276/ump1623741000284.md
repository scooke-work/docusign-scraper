# Step 4: Uninstall Apps Launcher from Your organization

Learn to uninstall the Docusign Apps Launcher package from your
Salesforce organization.

You have removed all Docusign Apps Launcher dependencies.

Use this procedure to uninstall the Apps Launcher package from your organization:

1. Open the Installed Packages page as an
   administrator:
   1. Select the gear ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/salesforcegear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc4MzgsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.0mLPwoBvqrds1euCOY4a0mAY8DYFhzv4YXpter0H7S4&_LANG=enus) and select
      Setup.
   2. Enter and select Installed Packages in the Quick
      Find field.

      The Installed Packages page opens.
2. Select Uninstall next to **Docusign
   Apps Launcher**.

   The Uninstalling a Package page opens.
3. Scroll to the bottom and select Yes, I want to uninstall this
   package and permanently delete all associated components.
4. Select Uninstall. ![A screenshot displays the uninstall button highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gab1637077302276/page/images/daluninstall.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODIxNTc4MzgsInNoZWFmIjoiZ2FiMTYzNzA3NzMwMjI3NiJ9.0mLPwoBvqrds1euCOY4a0mAY8DYFhzv4YXpter0H7S4&_LANG=enus "A screenshot displays the uninstall button highlighted.")

   Note: If an
   Unable to uninstall package message displays, you
   have remaining Docusign dependencies. Remove the listed
   dependencies and repeat the uninstall process.

You have completed the entire process. You have removed
all Docusign Apps Launcher dependencies, and have uninstalled the
package from your Salesforce organization. You can [install the
latest version from the Salesforce AppExchange](https://support.docusign.com/s/document-item?bundleId=gab1637077302276&topicId=ogj1586134254571.html&_LANG=enus "Learn how to install the Docusign Apps Launcher package of apps in your Salesforce organization.").
