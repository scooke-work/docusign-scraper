# Enter a Verification Code to Access Docusign on a New Device

Learn how to activate a new device for use with your Docusign account.

You must enter a verification code to access your Docusign account on a new device. You must verify your device again if
you empty your browser cache.

Use this procedure to activate your Docusign account on a new device:

1. Enter Docusign credentials when prompted.

   The Login from a new device dialog box opens.
2. Select Email. Docusign sends a verification code to your
   email address.
3. Open the Verify a New Device email from Docusign. Check
   your junk mail folder if you do not see the message in your Inbox.
4. Copy the verification code from the email.
5. Enter the verification code in the Login from a new device
   dialog box.
6. Select Confirm.

You have completed the process. You can access your Docusign account on this device.
