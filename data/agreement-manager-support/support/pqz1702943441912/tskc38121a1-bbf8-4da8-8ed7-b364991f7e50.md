# Save Search Criteria for Future Use

Learn how to save the search criteria for the queries that you frequently perform.

Save your search criteria, including keywords, filters, and custom columns, to find documents and
view relevant agreement data (columns) more
quickly and easily.

Important: The Saved Search feature is available at the user level. All account users have permission to save their frequently used search queries.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. The documents to which the logged-in user has access are listed in the table. The total document count is displayed the bottom right. This count is updated when you perform a search or apply filters.

   ![A screenshot of the Agreement Manager homepage that shows a list of documents to which the logged-in user has access.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-view_updated-with-document-count.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NDcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.cQatBL6EUZzenA_Rvi-XtE4XuI913HmmGcotiyoApQo&_LANG=enus "A screenshot of the Agreement Manager homepage that shows a list of documents to which the logged-in user has access.")
4. Perform a keyword search to locate the relevant agreements.
   1. Enter a keyword or phrase in the search field.

      Tip: When you click in the search field, a dropdown opens. Your search history and recent documents are displayed. You can also access your saved searches from the dropdown.
   2. Select a suggestion from the dropdown or press the Enter key.

   The agreements that contain the keyword or phrase are displayed in the table.
5. Apply filters to refine the agreement list table.
   1. Select the Filters option.
   2. Locate the filter that you want to apply.

      - Search for the filter. Enter a keyword in the Filter by field and select a value from the results list.
      - Select a filter from the list of default filters, including: Sets, Status, Parties, Agreement Type, and AI Suggestions.
      - Use the All Filters option to locate the relevant filter from a list of all available filters.
   3. Choose the relevant filter values or enter a date range.
   4. Select the Apply option.

      Tip: A number displays next to filter values to indicate the number of agreements that match the corresponding value.

   The table updates and agreements that match the applied filters are displayed.
6. Customize the agreement list table, as needed.
   1. Select the Show or Hide Fields (toggle) icon located to the right of the column headings.

      The Show or Hide Fields dialog box opens. A number of fields are turned on
      (blue) by default. These fields represent the columns in the agreement list
      table. The field order in the dialog box reflects the column order in the
      table.
   2. Select the toggles to turn the corresponding field on (blue) or off (white).
   3. Re-order the columns as needed. Hover the cursor over the Reorder (dots) icon for the field that you want to move and drag it to the appropriate location.
   4. Select the Save option.

   The dialog box closes and the new configuration is reflected in the agreement list table. Continue refining the search criteria until it achieves the expected outcome.
7. Select the Save option located to the right of the filters.

   The New saved search dialog box opens.
8. Complete the fields in the dialog box and select the Save Search option:

   ![A screenshot of the New saved search dialog box that is used to save your search criteria.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Saved-searches_New-saved-search_generic.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NDcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.cQatBL6EUZzenA_Rvi-XtE4XuI913HmmGcotiyoApQo&_LANG=enus "A screenshot of the New saved search dialog box that is used to save your search criteria.")

   1. Enter a unique name for the search in the Name field or accept the default value. The Name field is generated based on the applied filters and the time stamp.
   2. Enter a brief description for the search in the Description field.
   3. Select the Save Search option.

   The dialog box closes. Your search is available in the Saved Searches
   list. The page title and the breadcrumb navigation
   are updated to reflect the name of the saved
   search. The description is also displayed.

   ![A screenshot of a saved search that lists the agreements that matched the search criteria.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Saved-searches_search-results_force-majeure-cobranding-ex.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NDcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.cQatBL6EUZzenA_Rvi-XtE4XuI913HmmGcotiyoApQo&_LANG=enus "A screenshot of a saved search that lists the agreements that matched the search criteria.")

   Tip: Use the More actions (three dots) menu to:![A snippet of a saved search that highlights the More saved search actions dropdown.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Saved-searches_More-actions_force-majeure-cobranding-ex.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NDcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.cQatBL6EUZzenA_Rvi-XtE4XuI913HmmGcotiyoApQo&_LANG=enus "A snippet of a saved search that highlights the More saved search actions dropdown.")
   - Edit the name and description of the search
   - Delete the saved search
   - View all your saved searches
9. Return to the Agreement Manager homepage. Select the Back to Completed option.

   You are returned to the Agreement Manager homepage. The Completed Documents view is displayed.

You have successfully saved your search criteria for future use.
