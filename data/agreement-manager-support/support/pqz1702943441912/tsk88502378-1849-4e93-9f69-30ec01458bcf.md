# View the Agreement Hierarchy for a Selected Document

Learn how to view the related agreements in a hierarchy for a selected document.

The Related agreements pane, located on the Agreement preview page, displays the documents that belong in the same hierarchy as the selected agreement.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Completed Documents view opens.
4. Enter search criteria and apply filters to locate the relevant agreements. For example, expand the Filters option and apply the Parties filter to view agreements associated with specific parties or groups.

   The agreements that match the search criteria and filters are displayed in the table.
5. Select (click) the relevant agreement from the table.

   The Agreement preview page opens for the selected agreement. The Details pane is displayed by default.
6. Select the Related agreements icon from the toolbar on the left side of the page:

   ![A snippet of the Agreement preview page that highlights the Related agreements option in the toolbar on the left.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Preview-page_Related-agreements_highlighted_pointer.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MzgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.yUYPRhF0oOLS_4ZnWtC62gTR19IhwfqXIWKH0laDm88&_LANG=enus "A snippet of the Agreement preview page that highlights the Related agreements option in the toolbar on the left.")

   The Related agreements pane opens and the agreements in the hierarchy are displayed. The current document is highlighted in the list.

   ![A screenshot of the Agreement preview page that highlights the Related agreements pane, including the hierarchy for the current document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Preview-page_Related-agreements_Fontara-MSA-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MzgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.yUYPRhF0oOLS_4ZnWtC62gTR19IhwfqXIWKH0laDm88&_LANG=enus "A screenshot of the Agreement preview page that highlights the Related agreements pane, including the hierarchy for the current document.")
7. View the hierarchy from the party hub pages.
   1. Select the View Hierarchy option.

      The hierarchy details page opens in the party hub for the selected party. The Related Agreements section lists the all the agreements in the hierarchy.
   2. Update the hierarchy as needed. See the [Manage Your Existing Hierarchies](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tskf2e938fc-cb7f-4c65-a7bf-1cf1433722a2.html&_LANG=enus "Learn how to maintain your existing hierarchies, including updating the relationship structure between the related agreements.") article to learn more about updating your hierarchies.
   3. Select the Update option at the bottom of the page.

      The hierarchy is saved. The changes are reflected on the hierarchy details page.
   4. Select the browser's Back button.

   You are returned to the Agreement preview page for the selected agreement. The Details pane is displayed.
8. View the other agreements in the hierarchy.
   1. Return to the Related agreements pane. Select the Related agreements icon from the toolbar on the left.

      The Related agreements pane opens.
   2. Select the agreement that you want to view.

      The Agreement preview page opens for the selected agreement. The Details pane is displayed by default.
   3. Review the agreement as needed, including the attributes and field
      values that were surfaced.

      Tip: Continue reviewing the other agreements in the
      hierarchy. Select the Related agreements icon
      from the toolbar on the left and choose another document in the
      hierarchy. The Agreement preview page opens for
      the selected agreement.
   4. Select the browser's Back button.

   You are returned to the Agreement preview page for the selected agreement. The Details pane is displayed.
9. Select the Close (X) option from the top left.

   You are returned to the Completed Documents page. The agreement list table reflects the search criteria and filters that were applied previously.

You have successfully reviewed the hierarchy for a selected document, including viewing the other agreements in the hierarchy.
