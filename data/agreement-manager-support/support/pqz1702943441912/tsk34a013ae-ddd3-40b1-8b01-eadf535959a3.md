# Create Custom Dashboards from Templates

Learn how to build a custom dashboard by using an existing Agreement or Renewal dashboard as a template.

Dashboards are collections of standard
and custom reports that show a larger picture of a
particular focus area. When creating a custom
dashboard, you can add up to 20 reports to the
dashboard. This includes both standard and custom
reports. Once created, the dashboard is displayed
in the navigation pane under the Custom
Dashboards group.

This topic
reviews the steps to build a custom dashboard
by using an existing dashboard as a template.

Restriction: Currently, only the Agreements and Renewals dashboards can be used as templates for creating custom dashboards.

1. Log in to your **Docusign** account.

   The Docusign Home page opens.
2. Select Insights from the global header.

   The Insights hub opens and the Overview is displayed.
3. Expand the Create dropdown and select the Create Dashboard option:

4. Enter a name for the dashboard and add a description:

   ![A snippet of the Untitled dashboard page that shows the Edit Dashboard Name.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Create-custom-dashboard_Untitled_Enter-name.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwODgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.7y8l9u7t62gpoEQZTF9tjYuvlxiHmbAcX0nggUKKKUM&_LANG=enus "A snippet of the Untitled dashboard page that shows the Edit Dashboard Name.")

   1. Hover your cursor over the page title
      ("Untitled") and select the Edit
      Dashboard Name (pencil) icon or select
      (click) the page title.

      The title field opens for edits.
   2. Enter a name for the dashboard in the text-entry field.

      The text-entry field closes for edits. The entered name is displayed at the top of the page.
   3. Select the + Add description option. Enter a description in the text-entry field and press [Enter].

      The text-entry field closes. The entered description is displayed below the dashboard title.

      Tip: Update the dashboard title and description at any time:
      - Hover the cursor over the information that you want to update, either the title or the description.
      - Select the Edit (pencil) icon or select (click) the title or description
        to access the text-entry field.
      - Update the title or description by typing directly into the text-entry field and press [Enter].

      After entering a title and description, you're ready to build your custom dashboard. There are two options for creating a custom dashboard:
      - Use the Start from scratch option to get started by adding reports to this dashboard.
      - Use the Start from a template option to make a copy of an existing dashboard to get started.

        Restriction: Currently, only the Agreements and Renewals dashboards can be used as templates for creating custom dashboards.For purposes of this topic, use the Start from a template option to build a custom dashboard from an existing dashboard.
5. Choose a dashboard to use as a template:

   ![A screenshot of the Start from a template section that highlights the available dashboard templates.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Create-custom-dashboard_Start-from-a-template_Select-dropdown.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwODgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.7y8l9u7t62gpoEQZTF9tjYuvlxiHmbAcX0nggUKKKUM&_LANG=enus "A screenshot of the Start from a template section that highlights the available dashboard templates.")Use the Start from a template box to select the dashboard that you want to use as a template. Select a dashboard from the dropdown.

   Your custom dashboard is populated with the reports associated with the selected dashboard. For example, either the Agreements or Renewals dashboard reports.

   ![A snippet of a custom dashboard that shows the reports that were added from the selected dashboard template.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Create-custom-dashboard_populated-from-template.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwODgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.7y8l9u7t62gpoEQZTF9tjYuvlxiHmbAcX0nggUKKKUM&_LANG=enus "A snippet of a custom dashboard that shows the reports that were added from the selected dashboard template.")Using an existing dashboard as a template is a quick way to get started with building your custom dashboard. After initially populating the dashboard, update the reports so the dashboard reflects the insights that you want to see, including:
   - Add existing custom and standard reports to the dashboard
   - Add a new custom report to the dashboard
   - Reorganize the report charts and graphs on the dashboard

     Important: If the dashboard becomes
     cluttered, you can delete any of the reports that
     are no longer relevant. Expand the
     Report Options menu for the
     report that you want to delete and select the
     Delete option:![A snippet of a dashboard page that highlights the Delete option in the Report Options menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Create-custom-dashboard_Delete-report-from-dashboard.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwODgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.7y8l9u7t62gpoEQZTF9tjYuvlxiHmbAcX0nggUKKKUM&_LANG=enus "A snippet of a dashboard page that highlights the Delete option in the Report Options menu.")**Performing this action will delete the report, not just remove it from the dashboard.** Once deleted, it is no longer available to
     view or add to a dashboard.

**Add an existing report to the dashboard**

6. Add an existing custom or standard report to the dashboard:

   ![A snippet of a custom dashboard page that highlights options to add additional reports.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Create-custom-dashboard_template_Add-reports.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwODgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.7y8l9u7t62gpoEQZTF9tjYuvlxiHmbAcX0nggUKKKUM&_LANG=enus "A snippet of a custom dashboard page that highlights options to add additional reports.")

   1. Hover the cursor over either side of the dashboard and select the + (add) option or expand the Add dropdown from the bottom of the dashboard.
   2. Select the Existing Report option from the list.

      The Add Existing Report dialog box opens.

      ![A screenshot of the Add Existing Report dialog box that shows the reports that can be added to the dashboard.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Create-custom-dashboard_from-scratch_Add-Existing-Report-dialog-bpx.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwODgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.7y8l9u7t62gpoEQZTF9tjYuvlxiHmbAcX0nggUKKKUM&_LANG=enus "A screenshot of the Add Existing Report dialog box that shows the reports that can be added to the dashboard.")
7. Locate the report that you want to add to the dashboard and select the associated Add option.

   Tip: Enter a keyword in the Search for a report field or scroll to locate the report that you want to add.

   The dialog box closes and the selected report is displayed on the dashboard.

**Add a new custom report to the dashboard**

8. Expand the Add dropdown and select the + New Report option.

   The Untitled report page opens. Use this page to create a custom report, including choosing the data, metric, and visualization that matter to you.

   Note: See the [Create Custom Reports from Scratch](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tskfa65c658-fd7d-4ce8-b26b-08a9ec950aad.html&_LANG=enus "Learn how to create a custom report from scratch, including choosing the relevant data, metrics, and visualization options.") topic to learn more about creating a custom report.
9. Complete the sections on the Untitled report page and save the report:
   1. Complete the Data source section:

      - Expand the Data source dropdown and choose the data that you want to use to build the report:
      - Use the + Add filter option to apply filters to the agreement data.The agreement count shows the number of agreements that match the applied filters in the selected data source.
   2. Complete the Build your report section:

      - Expand the Measure dropdown and choose an attribute to start building a report.

        Important: Currently, custom reports only support numeric values for metrics or key measures.
      - Select the Group by option to choose a field by which to group or break down your metric. Select the attribute or field by which you want to group the selected metric.The pane on the right is updated to reflect the selected metrics. For example, a bar chart might show the agreements that match the applied filters grouped by the selected Group by field value.
   3. Complete the Visualization section.

      Expand the Visualization dropdown and choose how the report is displayed.
      Choices include:
      - Single metric (not
        available when the Group by
        option is used)
      - Bar chart
      - Pie chart

      The pane on the right updates to reflect the selected chart.
10. After refining the report so that it reflects
    the agreement data that you want to capture, save
    the report:
    1. Select the Save option
       from the top-right corner of the page.

       The Save report dialog
       box opens.
    2. Enter a name for the report in the Report name field. This step is required.
    3. Enter a description about the report's purpose in the Description field.
    4. Select the Save option.

       The dialog box closes and the new report is
       displayed on your custom dashboard.

       Tip: Your custom reports are available from
       the Custom tab under the
       Reports view. See the [View and Update Your Custom Reports](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=yuf1727801071281.html&_LANG=enus "This topic answers the question: How can I review and update my custom reports?") topic to learn how
       to view your custom reports.

**Organize the dashboard reports**

11. Re-organize the dashboard so it reflects the agreement insights that you want to see:

    ![A snippet of a custom dashboard page that highlights the "grab" cursor that is used to reorganize dashboard reports.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Create-custom-dashboard_from-scratch_reorganize-dashboard_Grab-cursor.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwODgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.7y8l9u7t62gpoEQZTF9tjYuvlxiHmbAcX0nggUKKKUM&_LANG=enus "A snippet of a custom dashboard page that highlights the \"grab\" cursor that is used to reorganize dashboard reports.")

    1. Hover the cursor over the report chart or graph that you want to relocate.

       A grid icon displays to the left of the report name.
    2. Place the cursor over the grid icon. The cursor turns into the "grab" cursor.
    3. Drag the report to the desired location on
       the dashboard.
    4. Continue moving reports around on the
       dashboard until it reflects the agreement insights
       that you want to see.

**Add the dashboard to Your Favorites list**

12. Add the dashboard to Your Favorites list. Select the Favorite (star) icon located in the top-right corner:

    ![A snippet of a detailed report view that highlights the Favorite (star) icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreements-dashboard_Report-details_Favorite-icon-selected_generic.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwODgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.7y8l9u7t62gpoEQZTF9tjYuvlxiHmbAcX0nggUKKKUM&_LANG=enus "A snippet of a detailed report view that highlights the Favorite (star) icon.")Your starred dashboards and reports are displayed in the Your Favorites section on the Overview dashboard.

You have successfully built a custom dashboard by using an existing dashboard as a template.
