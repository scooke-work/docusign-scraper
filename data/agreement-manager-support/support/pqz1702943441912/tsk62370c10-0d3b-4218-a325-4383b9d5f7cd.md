# Set Document-Level Notifications

Learn how to set and manage notifications to track upcoming milestones for an individual document.

Use the Notifications function to stay informed about major milestones for an individual agreement. There are two entry points for this function: the agreement list table and the Agreement preview page.

Tip: Notification settings give you control over the email notifications that you receive. By default, all the notification types related to the completed agreements in your account are selected, or enabled. Deselect a checkbox if you no longer want to receive email notifications for the corresponding notification type.

See the [Customize Notification Settings for Your Completed Agreements](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk155b1ed7-7b45-43cd-be92-2a5279630c0f.html&_LANG=enus "Learn how to customize your email notification settings from the Notification Preferences page.") article to learn more about updating your notification settings. Be sure to periodically check the Notification Preferences page as new notification types will be added in the future.

This topic reviews the steps to set a notification from the agreement list table. Begin by locating the relevant agreements.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Perform a search and apply filters to locate the relevant agreements:
   1. Enter a keyword or string in the search field and select a suggestion from the dropdown or press the Enter key.

      The agreements containing the keyword or search string that you entered are displayed in the table.
   2. Use the filters to refine the list of search results. For example, use the Parties filter to view a list of agreements for a specific party.

      1. Expand the Filters option and select the filter that you want to apply.
      2. Choose a filter value.
      3. Select the Apply option.

      The agreements that match the applied filters are displayed in the table.
5. Hover your cursor over the agreement that you want to track and select the Set a notification (bell) icon:

   ![A screenshot of the Completed view that highlights the Set a notification option in the agreement list table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-list-table_Set-a-notification_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY1MzQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.LcTzqQNvO9EsWVf86FlQlW88QzMIgXYfPrFQPmsJRIY&_LANG=enus "A screenshot of the Completed view that highlights the Set a notification option in the agreement list table.")

   Tip: If a notification already exists, the Manage notifications (bell) icon is displayed. Select the bell icon to update existing notifications for the agreement.

   The Notifications function opens. The Renewal Notice Date and Expiration Date notifications, including the reminder intervals, are enabled by default. You are designated as the recipient.

   ![A screenshot of the Notifications window that shows options for tracking agreement milestones.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Document-level-notifications_Notifications-window.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY1MzQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.LcTzqQNvO9EsWVf86FlQlW88QzMIgXYfPrFQPmsJRIY&_LANG=enus "A screenshot of the Notifications window that shows options for tracking agreement milestones.")

   Tip: Use the + Add option to configure other notification types, such as Effective Date and Execution Date notifications.
6. Update the reminder interval and recipients for the notification. Select the Edit link.

   The Edit notification dialog box opens.![A screenshot of the Edit notification dialog box that shows configuration options for the selected notification.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Document-level-notifications_Edit-notification-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY1MzQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.LcTzqQNvO9EsWVf86FlQlW88QzMIgXYfPrFQPmsJRIY&_LANG=enus "A screenshot of the Edit notification dialog box that shows configuration options for the selected notification.")
7. Update notification settings and select the Save
   option.
   1. Update the reminder interval. Expand the Remind me prior to date dropdown and choose when you want to
      receive notifications for the selected milestone. Click an interval to deselect it:

      ![A snippet of the Edit notification dialog box that shows the selected reminder intervals, 30 days and 60 days.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Set-Notification_Edit-notification_Reminder-interval-dropdown.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY1MzQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.LcTzqQNvO9EsWVf86FlQlW88QzMIgXYfPrFQPmsJRIY&_LANG=enus "A snippet of the Edit notification dialog box that shows the selected reminder intervals, 30 days and 60 days.")
   2. Specify the users that should also receive these
      notifications. Type a name or email address in the Add a
      user field and select a name from the dropdown
      list.

      Note: The Self option is selected by default and cannot be removed.
   3. Select the Save option.

   The dialog box closes and you are returned to the
   Completed view. The notification is saved and the
   Manage notifications (bell) icon is displayed when you hover
   over the document.
8. Use the Manage notifications (bell) icon to update your document-level notifications, including:

   - Add new notifications
   - Update existing notification settings
   - Manage the distribution list
   - Delete the notification

You have successfully set a document-level notification, including customizing the reminder interval and recipient list.
