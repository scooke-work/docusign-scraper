# Add and Remove Fields from Agreement Types

Learn how to add and remove standard and custom fields from agreement types.

Administrators and contract managers can add and remove fields from the agreement types in their account. This function requires Organize and manage all agreements access to Agreement Manager agreements and features.

Fields can be added to both custom (user-defined) and standard (Docusign-provided) agreement types. Likewise, fields can be removed from agreement types if they are no longer relevant or required for the type.

Important: Each Docusign-provided agreement type is pre-populated with a list of relevant standard fields. You cannot remove these default fields from the agreement type as they provide structure to the type.

Use the *Agreement Types Manager* to add and remove fields from the agreement types in your account. This task is completed from the Agreement Types option in the Settings (gear) menu.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Completed Documents page opens. A list of documents to which the
   logged-in user has access is displayed in the table.
4. Expand the Settings (gear) dropdown from the top right and select the Agreement Types option:

   ![A screenshot of the Agreement Manager homepage that highlights the Agreement Types option in the Settings (gear) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-types-manager_Settings-menu_Agreement-Types_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1NDksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.BEE4guT9h3E0PlEbQUEslcpbgc6_g0ofm1RYeNf4Kq4&_LANG=enus "A screenshot of the Agreement Manager homepage that highlights the Agreement Types option in the Settings (gear) menu.")

   The Manage Agreement Types page opens. A list of the account's standard and custom agreement types is displayed in the Agreement Types table.

**Locate the relevant agreement type**

5. Locate the agreement type that you want to update.

   ![A snippet of the Manage Agreement Types page that shows the search and filter options for locating agreement types.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Types_Agreement-Types_search-filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1NDksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.BEE4guT9h3E0PlEbQUEslcpbgc6_g0ofm1RYeNf4Kq4&_LANG=enus "A snippet of the Manage Agreement Types page that shows the search and filter options for locating agreement types.")

   - Search for the relevant agreement type. Enter a keyword in the Search types... field. As you enter the search term, the table updates and agreement types that matched the entered criteria are displayed.
   - Filter the table for the relevant agreement type. Filter options are: Created By, Category, and AI Status. Expand the filter, choose the relevant value, and select the Apply option.The table updates and the agreement types that matched the applied filters are displayed.
6. Select (click) the relevant agreement type from the table.

   The Agreement type details view opens for the selected agreement type. The fields associated with the agreement type are listed in the table.

**Add a field to the agreement type**

7. Select the + Add Fields option from the top right.

   The Add a field dialog box opens.

   ![A screenshot of the Add a field dialog box that is used to add a field to existing agreement types.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-type-management_create-type_add-a-field-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1NDksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.BEE4guT9h3E0PlEbQUEslcpbgc6_g0ofm1RYeNf4Kq4&_LANG=enus "A screenshot of the Add a field dialog box that is used to add a field to existing agreement types.")The selected agreement type is displayed in the Agreement types field by default.
8. Complete the required fields and select the Save option.
   1. Enter the name of the field in the Field name field.

      As you enter a name, a type-ahead dropdown list opens, and fields that match the entered name are displayed.
   2. Select the name from the type-ahead dropdown list or select the + Create [Field name] option to add a new custom field to the agreement type.

      For existing fields, the Field type and Category fields default based on the selected field.
   3. For custom fields, complete the Field type and Category fields:

      - Expand the Field type dropdown and select the type of custom field that you are adding. Choices are: Text, Number, Date, and Dropdown.

        Note: If you select Dropdown as the field type, you must provide a list of options for the dropdown. Enter a list of dropdown options in the Options text-entry box:![A snippet of the Add a field dialog box that shows the Options field for adding dropdown options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Add-custom-field_Options-for-dropdown-type.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1NDksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.BEE4guT9h3E0PlEbQUEslcpbgc6_g0ofm1RYeNf4Kq4&_LANG=enus "A snippet of the Add a field dialog box that shows the Options field for adding dropdown options.")**Important:** Separate each option with a semicolon.
      - Expand the Category dropdown and select a classification for the custom field.

      Choices are: General, Termination, Renewal, Payment, Legal and Compliance, and Other.
   4. Add the field to multiple agreement types if necessary. The selected type is displayed in the field automatically.

      Click in the Agreement types field to expand the dropdown. Choose the agreement types to which the field is being added and select the Apply option. You can select one or more custom and standard agreement types:

      ![A screenshot of the Agreement types dropdown that shows a selected agreement type to which the field will be added.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-agreement-types-fields_Add-a-field_Agreement-types-dropdown.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1NDksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.BEE4guT9h3E0PlEbQUEslcpbgc6_g0ofm1RYeNf4Kq4&_LANG=enus "A screenshot of the Agreement types dropdown that shows a selected agreement type to which the field will be added.")
   5. Select the Save option.

      The dialog box closes. The Agreement type details view opens for the selected agreement type. The newly added field is displayed in the Fields table.

**Remove a field from the agreement type**

This task continues with the same agreement type that was used in the previous steps.

To choose a new agreement type, return to the Agreement Types view and select a new agreement type. Select Agreement Types from the breadcrumb navigation at the top of the page. You are returned to the Agreement Types view.

9. Locate the field that you want to remove from the selected agreement type. Any fields that were previously added to an agreement type can be removed.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-type-details_Fields_search-and-filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1NDksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.BEE4guT9h3E0PlEbQUEslcpbgc6_g0ofm1RYeNf4Kq4&_LANG=enus)

   - Search for the relevant field. Enter a keyword in the Search fields... field. The table updates and the fields that matched the search criteria are displayed.
   - Filter the table for the relevant field. Filter options are: Created By, Category, Field Type, and AI Status. Expand the filter, choose the relevant value, and select the Apply option.

     The table updates and the fields that matched the applied filters are displayed.
10. Expand the More actions (three dots) menu for the field that you want to remove and select the Remove option:

    ![A snippet of the Fields table that highlights the Remove option in the More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-type-details_Fields_More-actions_Remove.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1NDksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.BEE4guT9h3E0PlEbQUEslcpbgc6_g0ofm1RYeNf4Kq4&_LANG=enus "A snippet of the Fields table that highlights the Remove option in the More actions (three dots) menu.")

    The Remove field dialog box opens.
11. Review the message and select the Remove option:

    ![A screenshot of the Remove field dialog box that prompts you to confirm that you want to remove the field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Type_Remove-field-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1NDksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.BEE4guT9h3E0PlEbQUEslcpbgc6_g0ofm1RYeNf4Kq4&_LANG=enus "A screenshot of the Remove field dialog box that prompts you to confirm that you want to remove the field.")

    The dialog box closes and the Agreement type details view is displayed. The field is no longer associated with the agreement type.
12. Return to the Agreement Types view. Select the Agreement Types option from the breadcrumb navigation in the top left.

    You are returned to the Agreement Types view. Continue updating your agreement types as needed.
13. Return to the Agreement Manager homepage. Select the Close (x) icon from the top right.

    The Agreement Manager homepage opens and the Completed Documents view is displayed.

You have successfully added and removed fields from custom (user-defined) and custom (Docusign-provided) agreement types.
