# Manage Obligations at the Agreement Level

Learn how to view and manage the obligations associated with an agreement.

Obligations, such as payments, renewal notices, and termination notices are surfaced in your
agreements. These AI-assisted insights are displayed in the
Obligations pane on the Agreement
preview page. Users can also manually capture and define obligations
from the Agreement preview page.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign
   Agreements page opens.
3. Select Completed from the
   AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. Search for the agreements
   containing the obligations that you want to manage.
4. Perform a search and apply filters to locate the relevant agreements:
   1. Enter a keyword or search string in the search field and select
      Enter.

      Agreements containing the keyword or search string that you entered
      are displayed in the table.
   2. Use the available filters to refine the list of search results. Expand the Filters option, choose the relevant filter, specify a filter value, and select
      Apply.

      The table is updated to reflect the filters that you applied.
5. Select an agreement from the list.

   The Agreement preview page opens for the selected agreement. The
   Details pane is displayed by default and shows a list
   of attributes and suggested values. Attributes, or fields, are displayed based
   on the identified agreement type.
6. Select the Obligations (flag) icon from the left-side of
   the Details pane.

   The Obligations pane opens. A list of obligations that were surfaced
   with AI or manually created is displayed in the pane.

   ![A screenshot of the Agreement preview page that highlights the Obligations pane, including surfaced obligations.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Data-review_agreement-preview_obligations-pane.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxNDYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.I6MrrCDmGResnmMUClYDQawZYUSryystssFboHxfNt8&_LANG=enus "A screenshot of the Agreement preview page that highlights the Obligations pane, including surfaced obligations.")

   The name, owner, due date, and status are displayed for each
   obligation.
7. Select (click) an obligation from the list.

   The obligation details view opens in the pane. Additional obligation fields
   are displayed. These fields were surfaced in your agreements during
   processing.

   ![A screenshot of the Agreement preview page that shows details for a selected obligation.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Navigator_Agreement-Preview_Obligation-details.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxNDYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.I6MrrCDmGResnmMUClYDQawZYUSryystssFboHxfNt8&_LANG=enus "A screenshot of the Agreement preview page that shows details for a selected obligation.")

**Complete a data review:**

8. Review and validate the obligation details:
   1. Place the pane into *edit* mode. Select either the
      Review All option or the
      Edit (pencil) icon.

      The Edit Obligation view displays and the
      attribute fields are opened for edits.
   2. Review the attribute (field) values that were surfaced in the
      agreement.

      ![An image of the spark icon used to denote the use of AI.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/AI-Spark-icon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxNDYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.I6MrrCDmGResnmMUClYDQawZYUSryystssFboHxfNt8&_LANG=enus "An image of the spark icon used to denote the use of AI.") A spark icon is displayed next to the fields that require
      review and approval.

      Compare the suggested values against the data in the agreement. Select
      (click) an attribute to view ("jump-to") the suggested value within the
      agreement.

      Tip: The "jump-to" feature is only
      available for fields with a spark icon.
   3. Select the Accept AI Suggestion (checkmark) icon
      if the value is *correct*.
   4. Enter the correct value if the suggestion is *incorrect* or
      *missing*.
   5. Repeat this step until all the attribute values are reviewed and
      approved.
   6. Select Save.

      The obligation details are saved. The pane is reverted to read-only
      mode.
9. Select the < Back option from the top of the
   pane.

   The Obligations list is displayed. Use the More actions (three dots) menu to manage obligations in the list, such
   as editing details or deleting the obligation.

**Edit an obligation**

10. Expand the More actions (three dots) menu
    for the obligation that you want to update and select
    Edit:

    ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Obligation_Actions_EditDelete-options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxNDYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.I6MrrCDmGResnmMUClYDQawZYUSryystssFboHxfNt8&_LANG=enus)

    The Edit Obligation pane is displayed. The obligation
    fields are open for updates.

    ![A screenshot of the Agreement preview page highlighting the Edit Obligation pane.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Navigator_Agreement-Preview_Edit-Obligation-pane.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxNDYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.I6MrrCDmGResnmMUClYDQawZYUSryystssFboHxfNt8&_LANG=enus "A screenshot of the Agreement preview page highlighting the Edit Obligation pane.")
11. Update the obligation's data fields, such as the name, type, status, due date,
    amount, frequency, repercussions, and description.
12. After making the necessary updates, select Save.

    The obligation details are updated, and the fields are closed to edits.
13. Select < Back from the top of the pane.

    The Obligations list is displayed.

**Delete an obligation**

14. Expand the More actions (vertical ellipsis) menu
    for the obligation that you want to update and select
    Delete:

    ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Obligation_Actions_EditDelete-options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxNDYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.I6MrrCDmGResnmMUClYDQawZYUSryystssFboHxfNt8&_LANG=enus)

    The Delete obligation dialog box opens.
15. Select Delete to confirm that you want to delete the
    obligation.

    The obligation is removed from the Obligations
    pane.
16. Return to the Agreement Manager homepage. Select the Close (x) icon from the top-left
    corner.

    You are returned to the Completed Documents view. Your
    search criteria and filters are still applied.
