# Use App Detail Pages to Learn About Docusign Apps

Use a Docusign app's detail page to view important information such as
an app summary, features, permissions, and resources.

Each Docusign app has a detail page with important information about the
app, such as a preview features, permissions, required plans, resources.![A screenshot of a sample Docusign App Detail page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/ac-use-this-app-det-page.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTgxOTUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.V7VmlxhSHrZuLvpLwUtgxuZUDC6YrJiAp1nXJw0XykI&_LANG=enus "A screenshot of a sample Docusign App Detail page.")

## App detail page information

The app detail page includes this information:

- Name
- Overview
- Use This App: Select the Docusign feature that you want to use with the app to access the page on which you use the app. For example, selecting Open in Workflow Builder opens a new workflow.
- [Docusign plan requirements](https://ecom.docusign.com/plans-and-pricing/esignature?utm_source=google&utm_medium=cpc&utm_campaign=GBL_XX_PRF_NEW_2401_SEMBranded&utm_term=docusign%20plans&campaignid=14964582893&adgroupid=127314395926&location=9198112&device=c&matchtype=e&extensionid=&creative=615714019859&keyword=docusign%20plans&placement=&network=g&gclid=Cj0KCQjwjJrCBhCXARIsAI5x66U-Tq5EeW33f-pTTNzCzGNWdogOf4m3z9WCSqCcG5BIia90eYEx7uEaAqTEEALw_wcB&gad_source=1&gad_campaignid=14964582893&gbraid=0AAAAADicDKsNWbtN78PfC2n0xdDWf3d-7)
- App Version number
- [Available Regions](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=con81e64917-4912-40c5-a4aa-69f2053f312a.html&_LANG=enus "Learn about finding the global regions where Docusign apps are available. These apps are currently deployed across five different datacenters: U.S., CA, AU, EU, and JP."):
  The global regions where Docusign apps are available.
- Certifications: [FedRAMP-certified
  apps](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=cone7f31b50-43c1-40a1-b6ed-d7ae1425ce61.html&_LANG=enus "Learn how to identify FedRAMP-certified apps in the Docusign App Center.") if available
- Developer (who builds the app)
- Resources
- Requirements: Any app prerequisites
- [Workflow
  templates](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=con8f35c78f-56b7-466c-91e6-c0ea7d03c516.html&_LANG=enus "Learn how to select and initiate a Docusign Workflow Builder template from the Docusign App Center page.") that support this app
- Works with: Docusign features where you use this app
  - Agreement Actions
  - Agreement Manager
  - CLM
  - eSignature
  - Workflow Builder
- Pricing: Option to view [available pricing plans](https://ecom.docusign.com/plans-and-pricing/iam?utm_source=google&utm_medium=cpc&utm_campaign=GBL_XX_PRF_NEW_2401_SEMBranded&utm_term=docusign%20plans&campaignid=14964582893&adgroupid=127314395926&location=9198112&device=c&matchtype=e&extensionid=&creative=615714019859&keyword=docusign%20plans&placement=&network=g&gclid=Cj0KCQjwjJrCBhCXARIsAI5x66U-Tq5EeW33f-pTTNzCzGNWdogOf4m3z9WCSqCcG5BIia90eYEx7uEaAqTEEALw_wcB&gad_source=1&gad_campaignid=14964582893&gbraid=0AAAAADicDKsNWbtN78PfC2n0xdDWf3d-7)

## App management options

Depending on the app's status, you can:

- Install an app by selecting Install as described in the [Install Apps in the App Center](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=ctg1698170340729.html&_LANG=enus "Learn how to install Docusign extension apps in the Docusign App Center.") article.
- Manage an installed app by selecting Manage as described in
  the [Manage Apps in the App Center](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=zxa1698170686717.html&_LANG=enus "Learn how to view, install, or uninstall Docusign App Center apps.") article.
