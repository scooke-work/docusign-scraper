# Delete an Agreement Set

Learn how to delete an agreement set.

Contract managers and any users who are assigned the Organize and manage all agreements permission can create and update agreement sets. This topic reviews the steps to delete an agreement set.

Note: An organization can have up to 50 agreement sets per account. If you exceed this limit, an error message is displayed.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents page is displayed.
4. Expand the Settings (gear) menu from the top right and select the Sets option:

   ![A screenshot of the Settings (gear) menu on the Agreement Manager homepage that highlights the Sets option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-sets_entry-point_Manage-sets-action-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc0MjEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.IhDFtR0J66Z1uoq5YhMzgXcpduzGVHP8rQGOoONxsyI&_LANG=enus "A screenshot of the Settings (gear) menu on the Agreement Manager homepage that highlights the Sets option.")

   The Agreement sets page opens. The All agreement sets view is displayed. The table lists the sets that were shared with or created by the logged-in user.
5. Locate the set that you want to delete.

   Tip: Select the sorting arrows in the Name column to sort the table by set name.
6. Expand the corresponding More actions (three dots) menu and select the Delete option:

   ![A snippet of the All agreement sets view that highlights the Delete option in the More actions menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Sets-dialog_All-agreement-sets_Delete.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc0MjEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.IhDFtR0J66Z1uoq5YhMzgXcpduzGVHP8rQGOoONxsyI&_LANG=enus "A snippet of the All agreement sets view that highlights the Delete option in the More actions menu.")

   The Delete this set? dialog box opens. You are prompted to confirm that you want to delete the set.

   Important: All users and groups with whom the set was shared will lose access once the set is deleted. Keep in mind, deleting the set does not delete or void any agreements.
7. Review the message and select the Delete Set option.

   The dialog box closes. The set is no longer displayed in the agreement set table.
8. Return to the Agreement Manager homepage. Select Completed Documents from the breadcrumb navigation at the top of the page.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.

You have successfully deleted a selected agreement set.
