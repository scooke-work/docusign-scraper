# Use Case: Validate Suggested Attribute Values for Your Agreements

This topic answers the question: How do I review the suggested attribute (field) values for my agreements?

Data reviews are necessary to validate the data that is surfaced during processing and to provide values for empty attributes (fields). This data review is an important part of using Agreement Manager. It ensures accurate agreement data for reporting, business planning, sales forecasting, and negotiations.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Completed Documents view opens. A list of documents to which the logged-in user has access is displayed in the table.
4. Expand the Filters option and select the View All Filters option from the dropdown.

   The Filters dialog box opens and a list of available filters is displayed.
5. Expand the AI Suggestions filter.

   The AI Suggestions dialog box opens.
6. Choose the Needs Review option and select Apply.

   The dialog box closes.
7. Select Save.

   The Filters dialog box closes and the agreements that require an AI data review are displayed in the table.

   See the [Processing Agreement Manager Agreements](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=rvw1736364838500.html&_LANG=enus "Read about agreement processing and how to opt in for additional processing to surface new insights within your agreements.") topic to learn more about processing indicators.
8. Review the list of agreements and select (click) the agreement you want to review.

   The Agreement preview page opens for the selected agreement. The Details pane is displayed by default and shows a list of attributes and suggested values. Attributes, or fields, are surfaced in the agreements based on the identified agreement type.

   Validate the suggested values for each attribute.
9. Place the Details pane into *edit* mode.
   - Select the Edit (pencil) icon.
   - Select the Review All option.

   Note: You must have Organize and manage all agreements access to perform data reviews.

   The Review & edit details pane opens for updates.

   Important: Updating the suggested attribute values does not change the content in the agreement.
10. Accept or edit the suggested values until each attribute is validated. If the attribute value does not exist within the agreement, leave the field empty.
    1. Compare the suggested attribute values against the data in the agreement.

       - Select (click) an attribute to view the suggested value within the agreement.

         The value is highlighted in the agreement and a list of suggested values is displayed.
       - Select the Accept AI Suggestion (checkmark) icon if the value is *correct*.
       - Delete the existing value and enter the correct value if the suggestion is *incorrect*.
    2. Continue reviewing and updating attribute values until all the attributes have been verified against the agreement.

       Attributes are grouped based on usage and purpose. A progress bar is displayed at the top of each group and shows the number of suggestions that require review.
11. Select Save.

    The agreement details are saved. The Details pane is reverted to read-only mode.
12. Select the Close (x) icon from the top-left corner.

    You are returned to the Completed Documents view. Your filters are still applied and agreements requiring review are displayed in the table.
