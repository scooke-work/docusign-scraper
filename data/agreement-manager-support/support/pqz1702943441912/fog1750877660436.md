# Refresh Agreements for New Data Points

Learn how to reprocess a group of selected agreements to surface new data points. For
example, to surface custom agreement types or fields within your agreements.

Additional attributes and insight become available when an updated AI model is released or when custom types and fields are created. These values are surfaced in new agreements, however, your historical documents must be reprocessed to surface the new data.

Use the Update function to reprocess selected agreements for new standard and custom data:

![A screenshot of the Update Details option and a callout that promotes this feature.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reprocessing_Completed-page_Update-Details_callout.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4ODIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Tb8qjiLN9Rm-bDoTS9Sx9N9lyqryHTeQV4zc5AHkCjs&_LANG=enus "A screenshot of the Update Details option and a callout that promotes this feature.")

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. Perform a search and apply filters
   to locate the relevant agreements.![A snippet of the Completed page that highlights the search field and available filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Navigator_Completed-documents_search-field-and-filters-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4ODIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Tb8qjiLN9Rm-bDoTS9Sx9N9lyqryHTeQV4zc5AHkCjs&_LANG=enus "A snippet of the Completed page that highlights the search field and available filters.")
4. Find the agreements that you want to process for new data points:
   1. Enter a keyword or search string in the search field and select Enter.
   2. Use the filters to refine the list of search results. Expand the Filters option, select the relevant filter, choose a value, and select the Apply option.

   The agreements that match your search criteria and applied filters are displayed in the table. The document count reflects the entered search criteria and applied filters
5. Select the agreements that you want to process:
   - Select individual agreements.
   - Select the Choose all checkbox from the header row to select all the agreements on that page of results.

     Tip: The Choose all option only selects agreements on the current page of search results. It does not select any agreements on subsequent pages of results.
   - Select the Select all that match your filters link to select all the agreements in the table that match your applied filters.

     Tip: The Select all that match your filters option is only available if filters are applied to the agreement list table.This option is not available if you enter search criteria.

   When one or more documents is selected, additional options are displayed, including the Update Details option.

   Note: A callout box opens and you are prompted to get the latest data and insights for your selected agreements: ![A screenshot of the Update Details option and a callout that promotes this feature.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reprocessing_Completed-page_Update-Details_callout.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4ODIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Tb8qjiLN9Rm-bDoTS9Sx9N9lyqryHTeQV4zc5AHkCjs&_LANG=enus "A screenshot of the Update Details option and a callout that promotes this feature.")Select the OK option to continue.
6. Expand the Update dropdown and choose to refresh the agreements for Standard Data or Custom Data:

   ![A snippet of the Completed Documents page that highlights the Update dropdown and options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reprocessing_Completed-page_Update-Details_expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4ODIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Tb8qjiLN9Rm-bDoTS9Sx9N9lyqryHTeQV4zc5AHkCjs&_LANG=enus "A snippet of the Completed Documents page that highlights the Update dropdown and options.")

   - Select the Standard Data option.

     The Update AI version dialog box opens. Select Start Update to initiate the reprocessing job.

     The dialog box closes and a confirmation message is displayed.
   - Select the Custom Data option.

     A reprocessing job is initiated and your selected agreements are processed for custom agreement types and fields. A confirmation message is displayed.

   Note: The Information marker is displayed when one or more of your selected agreements are already up to date: ![A snippet of the Completed page that shows the Update option and the expanded Information marker.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reprocessing_Completed-page_Upeate-Details_Information-marker_expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4ODIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Tb8qjiLN9Rm-bDoTS9Sx9N9lyqryHTeQV4zc5AHkCjs&_LANG=enus "A snippet of the Completed page that shows the Update option and the expanded Information marker.")This message is meant to save you time and resources by letting you know when reprocessing is not necessary. From the Information marker, select the More (arrow) option to view additional details.

   A confirmation message opens and confirms that the selected files are being processed for new data. Use the View Progress option to view
   results of the processing job.
7. Select View Progress from the message box:

   ![A screenshot of the message box that indicates the files are being processed for new data.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Refresh-custom-data_processing-message_View-Progress.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4ODIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Tb8qjiLN9Rm-bDoTS9Sx9N9lyqryHTeQV4zc5AHkCjs&_LANG=enus "A screenshot of the message box that indicates the files are being processed for new data.")The job details page opens. The Processing status and Job details are displayed for the processing job.![A screenshot of the job details page for the reprocessing job, including the Processing status and Job details.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Refresh-custom-data_processing-job-details.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4ODIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Tb8qjiLN9Rm-bDoTS9Sx9N9lyqryHTeQV4zc5AHkCjs&_LANG=enus "A screenshot of the job details page for the reprocessing job, including the Processing status and Job details.")

   Note: The Uploads and Processing Status function gives you access to job details for agreement uploads and processing jobs. This job monitoring hub provides a centralized location to view and track the status of these jobs. Access this function from the Settings (gear) menu on the Completed Documents page:![A screenshot of the Completed Documents view that shows the Uploads and Processing Status option highlighted in the Settings menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-view_Actions_Monitor-All-Jobs-option.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4ODIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Tb8qjiLN9Rm-bDoTS9Sx9N9lyqryHTeQV4zc5AHkCjs&_LANG=enus "A screenshot of the Completed Documents view that shows the Uploads and Processing Status option highlighted in the Settings menu.")See the [Job Monitoring for Agreement Uploads and Processing](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=ahg1742251814403.html&_LANG=enus "Read about the job monitoring hub that is used to track the progress for agreement uploads and processing jobs.") topic to learn more about the job monitoring hub.
8. Review the job's status and details as needed:
   - Perform a keyword search to locate processing status details for a specific file. Enter search criteria in the Search for a file name field and select a file from the type-ahead dropdown or press [Enter].
   - Apply filters to sort the list of files by Status and Error Message.
   - Select the View File option to view the corresponding file from the Agreement preview page.

     Note: The View File option takes you out of the job monitoring hub. Use these steps to return to the job monitoring hub from the Agreement preview page:
     - Select the Close (x) icon from the top-left corner of the Agreement preview page. The Agreement Manager homepage opens.
     - Expand the Settings (gear) menu and select the Uploads and Processing Status option. The job monitoring hub opens and the All jobs page is displayed.
     - Select the Processing tab. The All jobs page updates and the Processing agreements view is displayed.
     - Locate the relevant processing job and select the View details option. The job details page opens, and the Processing status and Job details are displayed.
9. Return to the Agreement Manager homepage. Select Close (x) from the top-right corner to return to the previous page.

   The Agreement Manager homepage opens and the Completed view is displayed.

You have successfully reprocessed selected agreements for new data.
