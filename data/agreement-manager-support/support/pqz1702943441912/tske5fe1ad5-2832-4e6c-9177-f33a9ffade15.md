# Define Your Custom Agreement Types

Learn how to initiate the AI detection set-up process and complete the first step to
define the custom type.

When you configure a new custom agreement type, the AI detection is inactive by default.
Complete the Set Up process to activate the custom type. Once
activated, the custom data point is surfaced in your new agreements and selected
historical documents.

Important: This task requires a user-defined custom agreement type that
is not activated for AI detection. See the [Configure New Custom Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=gke1727282238086.html&_LANG=enus "Learn how to create custom agreement types to support organizational use cases.")
topic for steps to add custom agreement types.

The definition and examples that you provide during the setup are designed to
"teach" the AI model.

Important:

Customers do not need to consent to AI training to use this feature. When customers create custom
agreement types and fields, their data is used as **input** to the AI model to help generate
results for their use. This process does not involve AI training.

If a customer has given consent to sharing data for AI
training, anonymized data may be used for training Docusign AI models. This may
include anonymized data from custom extractions.

See the [Docusign FAQs for AI](https://support.docusign.com/s/document-item?bundleId=fzd1707173174972&topicId=uss1707173279973.html) topic for more information about how
Docusign uses data for AI training.

This topic reviews the steps to launch the AI detection set-up wizard and define
your custom agreement type. This task completed by administrators and contract managers.

Remember: Administrators automatically have access to configure agreement data. For non-administrator users, agreement type management tasks require Organize and manage all agreements access to Agreement Manager agreements and functionality.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Completed Documents view opens.
4. Expand the Settings (gear) dropdown from the top right of the page and select the Agreement Types option:

   ![A snippet of the Agreement Manager homepage that highlights the Agreement Types option in the Settings (gear) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-types-manager_Settings-menu_Agreement-Types_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1OTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.tf8sVppA3b7gNpfUj5dxUoFHhVC5vbnQn-ClM35i0zs&_LANG=enus "A snippet of the Agreement Manager homepage that highlights the Agreement Types option in the Settings (gear) menu.")

   The Manage Agreement Types page opens. A list of the account's standard and custom agreement types is displayed in the Agreement Types table.
5. Locate the custom agreement type that you want to set up for AI detection and select the Set Up option:

   ![A screenshot of the Agreement Types view that highlights the Set Up option for a custom agreement type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Types-Manager_Agreement-Types-table_Set-Up-option_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1OTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.tf8sVppA3b7gNpfUj5dxUoFHhVC5vbnQn-ClM35i0zs&_LANG=enus "A screenshot of the Agreement Types view that highlights the Set Up option for a custom agreement type.")

   The AI detection set up wizard launches. Follow the prompts to find and categorize your custom extraction.

   ![A screenshot of the setup wizard that introduces the three easy steps to enable the custom agreement type extraction.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Custom-agreement-type-extractions_Setup-wizard_Get-Started.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1OTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.tf8sVppA3b7gNpfUj5dxUoFHhVC5vbnQn-ClM35i0zs&_LANG=enus "A screenshot of the setup wizard that introduces the three easy steps to enable the custom agreement type extraction.")

   This task focuses on the first step, *Define your custom extraction*.
6. Select the Get Started option.

   The Set definition dialog box opens and asks you to define the custom agreement type.

   ![A screenshot of the Set definition dialog box that prompts you to define the custom agreement type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Type_Enable-AI-detection-for-custom-agreement-type--Add-defniition.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1OTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.tf8sVppA3b7gNpfUj5dxUoFHhVC5vbnQn-ClM35i0zs&_LANG=enus "A screenshot of the Set definition dialog box that prompts you to define the custom agreement type.")
7. Enter a definition for the custom type. This field is limited to 2,000 characters. The definition describes the purpose and usage of the agreement type.

   Note: See the [Best Practice: Writing Definitions for AI Detection](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tmk1741317734856.html&_LANG=enus "Read about tips for writing good definitions for the AI detection setup process. These definitions are an important part of the process and help to \"teach\" the AI detection.") support topic to review the guidelines for writing definitions. Follow these guidelines to get the most our of the AI detection and to ensure accuracy.

   When the definition exceeds 50 characters, the Refine with AI option becomes available. Use this option to update the definition for consistency and accuracy.
8. Fine-tune the definition.

   ![A screenshot of the Set definition dialog box that shows the AI-suggested definition.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Custom-agreement-type_Set-definition_Refine-with-AI_Definition-Suggested.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1OTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.tf8sVppA3b7gNpfUj5dxUoFHhVC5vbnQn-ClM35i0zs&_LANG=enus "A screenshot of the Set definition dialog box that shows the AI-suggested definition.")

   1. Select the Refine with AI option.

      The dialog box refreshes and a suggested definition is displayed.
   2. Review and edit the suggested definition as needed. You can also select the Undo option to revert to the original version of the definition.
9. Select the Next option.

   The AI detection set-up wizard advances to the next step and you are prompted to select
   examples.

   Complete the [Add Examples of Your Custom Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tskd38465c6-c8c0-4588-bfb3-fdb6916a1926.html&_LANG=enus "Learn how to complete the second step of the AI detection set-up process, Add examples of this agreement type.") task to provide
   examples to the AI detection set-up process.

You have successfully completed the first step of the AI detection set-up process and
defined your custom agreement type.
