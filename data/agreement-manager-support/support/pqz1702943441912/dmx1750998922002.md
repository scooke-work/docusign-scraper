# Create Agreement Sets from the Agreement List Table

Learn how to create a new agreement set and apply smart filters from the agreement list table on the Completed Documents page.

Smart filters can be created from the filters that you apply to the agreement list table on the Completed Documents page. Keep in mind that not all filters are supported by the *Smart Filters* function.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. A list of documents to which the logged-in user has access is displayed in the table.
4. Apply filters to locate the documents that you want to add to the agreement set:

   ![A snippet of the Completed Documents page that shows the expanded Filters option, including suggested filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Create-agreement-sets_Completed-Documents_Apply-Filters_Filters-expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczMzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.TLtvMDQ8NasCOc74QkGEfv1h58VLrKuTz3Ret7DMEuI&_LANG=enus "A snippet of the Completed Documents page that shows the expanded Filters option, including suggested filters.")

   1. Select the Filters option. The menu expands and a list of applied and suggested filters is displayed.
   2. Locate the filter that you want to apply. Scroll through the list of suggested filters or enter a search term in the Filter by field.
   3. Expand (click) the relevant filter and select one or more values.

      Important: Not all filters can be used as a smart filter. For example, the smart filter function does not support rolling date and duration-based filters. If you select an unsupported filter, the filter name is displayed with a strike-through when you create the set. The filter is not included in the smart filtering for the set.
   4. Select the Apply option.
   5. Continue applying filters as needed.
   6. Select the Save option to save your filters.

   The agreement list table updates and agreements that match the applied filters are displayed.
5. Select one or more agreements to add to the agreement set:
   - Select individual agreements to update.
   - Select the Choose all checkbox from the header row to select all the agreements on the page.

   Tip: You can select all the agreements in the results list, not just the agreements on the first page of results. The Selected dropdown displays a counter to indicate the number of agreements that are currently selected. Expand the dropdown and select the Select all that match your filters option:![A snippet of the agreement list table that highlights the Select all that match your filters option in the Selected dropdown.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents_Selected-dropdown_Select-all-that-match-your-filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczMzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.TLtvMDQ8NasCOc74QkGEfv1h58VLrKuTz3Ret7DMEuI&_LANG=enus "A snippet of the agreement list table that highlights the Select all that match your filters option in the Selected dropdown.")All the agreements in the table are selected.

   Additional options are displayed when one or more agreements are selected from the list, including the Organize option.
6. Expand the Organize dropdown and select the + Create New Set option:

   ![A screenshot of the Organize dropdown that highlights the Create New Set option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents_Organize-expanded_Create-New-Set_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczMzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.TLtvMDQ8NasCOc74QkGEfv1h58VLrKuTz3Ret7DMEuI&_LANG=enus "A screenshot of the Organize dropdown that highlights the Create New Set option.")

   The New agreement set dialog box opens.
7. Complete the required fields and turn on smart filtering for the set:

   ![A screenshot of the New agreement set dialog box that shows the Name, Description, and smart filtering toggle.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-set-management_New-agreement-set_turn-on-smart-filtering-toggle.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczMzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.TLtvMDQ8NasCOc74QkGEfv1h58VLrKuTz3Ret7DMEuI&_LANG=enus "A screenshot of the New agreement set dialog box that shows the Name, Description, and smart filtering toggle.")

   1. Enter a name for the agreement set in the Name field.

      Tip: The Name field is limited to 60 characters.
   2. Enter a description for the agreement set in the Description field.

      Tip: The Description field is limited to 200 characters. This field is optional. The entered value is displayed in the All agreement sets table.
   3. Turn on (blue position) the toggle in the Turn on smart filtering for this set section.

      The filters that you applied to the agreement list are displayed in this section. Turn on the toggle to add agreements to the set based on your applied filters.

      Important: If you applied an unsupported filter, the filter name is displayed with a strike-through. The filter is not included in the smart filtering for the set.
8. Select Create or Create and Share:
   - Select the Create option to create the new agreement set without sharing the set with other users or groups.
   - Select the Create and Share option to create the agreement set and share it with users and groups.

     The Share with others dialog box opens. Locate the user or group to whom you want to share the set and assign an Access level. Select Share to complete the process.

     Note: See the [Share an Agreement Set](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=pvb1736186431453.html&_LANG=enus "Learn how to share an agreement set with individual users and groups. This topic answers the question: How do I share groups of agreements with the appropriate people?") topic to learn more about sharing agreement sets.

   The dialog box closes and you are returned to the Completed Documents page.

You have successfully created an agreement set from the agreement list table, including applying smart filters.
