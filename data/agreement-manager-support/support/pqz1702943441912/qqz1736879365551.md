# Filter Your Completed Documents for Custom Agreement Types

Learn how to view, sort, and filter your completed documents by a custom agreement type.

Use the Agreement Type filter to filter the agreement list table for specific agreement types, both custom and standard. This filter is located on the Completed Documents page in the Filters option.

This topic reviews the steps to filter your completed documents by custom agreement types.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. Perform a search and apply filters to locate the relevant agreements.
4. Apply the Agreement Type filter to view documents that are associated with a selected custom type:

   ![A screenshot of the Agreement Type filter that shows a selected custom agreement type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Navigator_Completed-documents_search-field-and-filters-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1MTAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.yqMhU5-GP_cDYT-CvwjCNPmhhJ4mb9j49DvKfFTLrtk&_LANG=enus "A screenshot of the Agreement Type filter that shows a selected custom agreement type.")

   1. Expand (click) the Filters option and select the Agreement Type filter.

      The Agreement Type filter expands and available types are displayed.
   2. Search for the relevant custom agreement type. Enter a keyword or search string in the Search field.

      As you enter the search criteria, a list of agreement types that matched the keyword are displayed.
   3. Select the custom agreement type by which you want to filter your documents.
   4. Select the Apply option.

   The agreement list table updates, and agreements that match the selected type are displayed.
5. After locating agreements for the selected agreement type, choose to download, share, or export the agreements:
   - See the [Download Documents from the Agreement List Table](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=ach1724363496037.html&_LANG=enus "This topic answers the question: How do I download agreements from Agreement Manager?") topic to learn more about downloading agreements.
   - See the [Share an Agreement with Others](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=vyz1722979030156.html&_LANG=enus "This topic answers the question: How do I share agreements with others? Learn how to share an agreement with other users or groups, including assigning access levels for each recipient.") topic to learn how to share a selected agreement.
   - See the [Export Data from the Agreement List Table](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=pqf1724363438241.html&_LANG=enus "Learn how to export agreement data to a CSV file for selected agreements.") topic to learn more about exporting agreement data.
