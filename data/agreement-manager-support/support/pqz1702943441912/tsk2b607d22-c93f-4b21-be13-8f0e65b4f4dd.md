# Update Your Agreements to the Latest AI Model

Learn how to reprocess the agreements in your account using the latest AI model.

When organizations are first onboarded to Agreement Manager, their historical documents are ingested and processed based on the current AI model. As new or updated models become available, the previously ingested and uploaded documents are not automatically reprocessed for the new insights.

When users interact with insights that were released since their initial onboarding, they are prompted to update their AI model:![A snippet of the Completed Documents view that highlights the banner prompt to Update Now to the latest AI version.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reprocessing_banner-messaging_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5MDIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.xuEQ225F6_C5PRhHgfyYpL4p6d7D2nzHGrY8D61O9nk&_LANG=enus "A snippet of the Completed Documents view that highlights the banner prompt to Update Now to the latest AI version.")

Note: The banner message is triggered when users apply relevant filters from the agreement list table located on the Completed Documents page. Using these filters from other locations, such as in the party hub or under the Insights menu, does *not* trigger the banner message.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. A list of documents to which the logged-in user has access is displayed in the table.

   Note: The reprocessing banner message and prompt are triggered when the certain criteria are met:

   - The user applies a filter from the Completed Documents page that is not
     available with their current AI model. For example, the banner is
     triggered if you apply the Line of Business,
     Party Role, or Status
     filters.
   - The account contains agreements with outdated extractions.

   Every user has access to initiate the reprocessing job for all the agreements in their account -
   regardless of the account size.
4. Apply filters to locate the relevant agreements. When you apply a filter that is not available with your current AI model, you are prompted to update to a new version.

   Filters that may trigger the banner include:
   - Line of Business (located in the All Filters list)
   - Party Role (located under the Parties filter)
   - Status  (displays by default but is also located in the All Filters list)

   Expand the filter that you want to apply, choose a filter value, and select
   Apply.

   The agreement list table updates and agreements that match the applied filters are displayed. A banner message prompts you to: *Update to the latest AI version to find and show all documents matching these filters*.
5. Select the Update Now option:

   ![A snippet of the Completed Documents view that highlights the Update Now option in the banner message.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reprocessing_Completed-page_Update-Now_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5MDIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.xuEQ225F6_C5PRhHgfyYpL4p6d7D2nzHGrY8D61O9nk&_LANG=enus "A snippet of the Completed Documents view that highlights the Update Now option in the banner message.")

   The Update AI version dialog box opens.
6. Expand and review the What to expect section:

   ![A screenshot of the Update AI version dialog box that highlights the What do expect section.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reprocessing_Update-AI-version_dialog-box_what-to-expect_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5MDIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.xuEQ225F6_C5PRhHgfyYpL4p6d7D2nzHGrY8D61O9nk&_LANG=enus "A screenshot of the Update AI version dialog box that highlights the What do expect section.")

   Once the reprocessing job is initiated, it cannot be canceled. These updates happen in the background and can take several minutes (small jobs) to days (1,000 files or more). See the [Agreement Manager Reprocessing Experience](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=con44590246-0f0e-43b4-9692-caadb57335ac.html&_LANG=enus "Learn about the Agreement Manager reprocessing experience, including its purpose, trigger points, and job size considerations.") topic to learn more about job size considerations.

   Note: For enterprise accounts, agreements are reprocessed in batches of 25,000 files at a time.
7. Select the Start Update option.

   The dialog box closes and the reprocessing job is initiated. A message confirms that your agreements are being processed with the latest AI model.

   Note: The reprocessing job may surface and overwrite your agreement data. However, any values that were manually entered or marked as "accepted" are not overwritten or altered. The processing job skips any documents that are up to date.
8. Track your job progress from the job
   monitoring hub. Select the View
   Progress option from the message
   box:

   ![A screenshot of the confirmation message that highlights the View Progress option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reprocessing_Completed-page_Confirmation-message_View-Progress.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5MDIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.xuEQ225F6_C5PRhHgfyYpL4p6d7D2nzHGrY8D61O9nk&_LANG=enus "A screenshot of the confirmation message that highlights the View Progress option.")

   The job details page opens. The
   Processing status and
   Job details are displayed for
   the processing job.

   Note: The Uploads and Processing Status function gives you access to job details for agreement uploads and processing jobs. This job monitoring hub provides a centralized location to view and track the status of these jobs. Access this function from the Settings (gear) menu on the Completed Documents page:![A screenshot of the Completed Documents view that shows the Uploads and Processing Status option highlighted in the Settings menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-view_Actions_Monitor-All-Jobs-option.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5MDIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.xuEQ225F6_C5PRhHgfyYpL4p6d7D2nzHGrY8D61O9nk&_LANG=enus "A screenshot of the Completed Documents view that shows the Uploads and Processing Status option highlighted in the Settings menu.")See the [Job Monitoring for Agreement Uploads and Processing](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=ahg1742251814403.html&_LANG=enus "Read about the job monitoring hub that is used to track the progress for agreement uploads and processing jobs.") topic to learn more about the job monitoring hub.
9. Review the job's status and details as needed:
   - Perform a keyword search to locate processing status details for a specific file. Enter search criteria in the Search for a file name field and select a file from the type-ahead dropdown or press [Enter].
   - Apply filters to sort the list of files by Status and Error Message.
   - Select the View File option to view the corresponding file from the Agreement preview page.
10. Select Close (X) from the top-right corner to return to the previous page.

    You are returned to the Completed Documents page.

You have successfully reprocessed your out-of-date agreements with the latest AI
model.
