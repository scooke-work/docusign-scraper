# Renewal Notifications and Subscriptions

This topic answers the question: What are my reminder options for renewals in Agreement Manager? Read about the renewals and document-level notifications, including the content and frequency of the emails.

## Renewal-related subscriptions

Use the *Subscriptions* feature to sign up to receive email report notifications. You can subscribe to receive email report notifications for any of the charts found on the Agreements, Obligations, and Renewals dashboards.

Note: The *Subscription* feature is also available for your custom reports and dashboard charts.

This feature is accessed from the detailed report view for a Renewals dashboard chart. Use the Manage Notification (bell) icon to create or manage your email reports.

Complete the Create subscription dialog box to set up one-time or recurring email reports for yourself and others:

![A screenshot of the Create subscription dialog box that shows the subscription configuration fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Renewals-dashboard_Report-view_Create-subscriptions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNjMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vXS6lli1s0FcZJABhRKq8V5bz3rkZQybny0sxsEH_1E&_LANG=enus "A screenshot of the Create subscription dialog box that shows the subscription configuration fields.")

Note: When you create a subscription, you have the option to add individual users and groups. Once created, the subscription is displayed in the Manage report subscriptions dialog box for any individual users who were added. Currently, the subscription is not visible to users if they were added as part of a group.

See the [Subscribe to Renewal Reports](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=nis1709931821078.html&_LANG=enus "Learn how to set up one-time or recurring email reports for yourself or others to receive notifications about your renewals.") article to learn more about subscribing to receive renewal-related email reports.

## Document-level notifications

Document-level email notifications help you stay informed about upcoming milestones for individual documents. This function has two entry points:

- **Agreement list table:** Hover the cursor over an agreement to display the Set a notification (bell) icon. ![A snippet of the agreement list table that shows the Set a notification (bell) icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-snippet_Document-level-notification_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNjMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vXS6lli1s0FcZJABhRKq8V5bz3rkZQybny0sxsEH_1E&_LANG=enus "A snippet of the agreement list table that shows the Set a notification (bell) icon.")
- **Agreement preview page:** Select the Set a notification (bell) icon to set a document-level notification.![A snippet of the Agreement preview page that shows the Set a notification (bell) icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-preview_Manage-Set-a-notification_snippet.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNjMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vXS6lli1s0FcZJABhRKq8V5bz3rkZQybny0sxsEH_1E&_LANG=enus "A snippet of the Agreement preview page that shows the Set a notification (bell) icon.")

When you select the bell icon, the Renewal Notice Date and Expiration Date notifications are enabled by default. Other notification types can be configured by using the + Add option.

Note: When you enable the notification, you are designated as the recipient and all frequency options are selected. Use the Edit  function to update the frequency and recipients for the notification as needed.

The Manage notifications (bell) icon is displayed once the notification is set. Select this icon to add new notifications or to edit existing ones:![A snippet of the agreement list table that shows the Manage notifications (bell) icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-list-table_Manage-notifications_snippet.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNjMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vXS6lli1s0FcZJABhRKq8V5bz3rkZQybny0sxsEH_1E&_LANG=enus "A snippet of the agreement list table that shows the Manage notifications (bell) icon.")See the [Set Document-Level Notifications](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk62370c10-0d3b-4218-a325-4383b9d5f7cd.html&_LANG=enus "Learn how to set and manage notifications to track upcoming milestones for an individual document.") article to learn more about setting document-level notifications.

## Notification preferences for completed agreements

Docusign provides email notifications for all account-related updates, including updates related to your agreements in Agreement Manager. Notification settings give you control over the email notifications that you receive.

The Notification Preferences page is accessed from the My Preferences option in the user profile menu. By default, all the notification types in the Completed Agreements section are enabled:![A screenshot of the Notification Preferences page that highlights the selected notification types for completed agreements.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/My-preferences_Notification-Preferences_For-Completed-Agreements_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNjMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vXS6lli1s0FcZJABhRKq8V5bz3rkZQybny0sxsEH_1E&_LANG=enus "A screenshot of the Notification Preferences page that highlights the selected notification types for completed agreements.")

Deselect a checkbox if you no longer want to receive email notifications for the corresponding notification type.

Tip: Periodically review your Notification Preferences to ensure that you are receiving the relevant email notifications. Additional notification types will be added in the future.

See the [Customize Notification Settings for Your Completed Agreements](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk155b1ed7-7b45-43cd-be92-2a5279630c0f.html&_LANG=enus "Learn how to customize your email notification settings from the Notification Preferences page.") topic for steps to update your notification preferences.
