# Manage Agreement-Level Access

Learn how to manage access to an agreement, including revoking access and updating access permissions.

Users can share an agreement with individuals and groups within their account if they have the appropriate permissions. When you have Can Manage access to an agreement, you can:

- Share the agreement with additional users or groups
- Assign Can Manage, Can Edit, or Can View access to an agreement for users and groups
- Update the agreement access for users or groups
- Remove agreement access for users or groups

Important: The Can Manage permission relates only to agreement access. It is not the same as having Organize and manage all agreements access to Agreement Manager.

You can have Can Manage access to an agreement while only having View all agreements in this account access to Agreement Manager.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Perform a search or use the available filters to locate the agreement you want to share.
5. Expand the More actions (three dots) menu to the right of the agreement and select the Share option:

   ![A snippet of the Completed Documents view that highlights the Share option in the More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_vertical-ellipses_Share.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2ODgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.K6-tIkmVZutb9BPAsQ5WGUXNU0201u6BhkHFf9Dy6f4&_LANG=enus "A snippet of the Completed Documents view that highlights the Share option in the More actions (three dots) menu.")

   The Share with others dialog box opens. Complete the necessary access management tasks for the selected agreement.

   ![A screenshot of the Share with others dialog box that shows the people with access to the agreement.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Share-with-others-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2ODgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.K6-tIkmVZutb9BPAsQ5WGUXNU0201u6BhkHFf9Dy6f4&_LANG=enus "A screenshot of the Share with others dialog box that shows the people with access to the agreement.")
6. Share the agreement with additional users or groups:
   1. Enter the name of the user or group in the Add user or group search field. A type-ahead dropdown displays names matching the criteria that you entered.
   2. Select a user or group from the dropdown list.

      The Add users view is displayed.

      Note: The Notify users by email option is selected by default. Leave this checkbox selected so that users are notified of shared agreements.
   3. Expand the Access dropdown and select the appropriate access level for the agreement. Choices are: Can Manage, Can Edit, and Can View.
   4. Select the Share option.

      The dialog box closes. The agreement is shared with the selected user or group and displays in their agreement list table.
7. Update the access level for users and groups with whom the agreement was previously shared.

   Open the Share with others dialog box. Expand the More actions (three dots) menu to the right of the agreement and select the Share option.

   The Share with others dialog box opens.

   ![A screenshot of the Share with others dialog box that shows pending access changes.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Share-with-others-dialog-box_Manage-user-or-group-access.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2ODgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.K6-tIkmVZutb9BPAsQ5WGUXNU0201u6BhkHFf9Dy6f4&_LANG=enus "A screenshot of the Share with others dialog box that shows pending access changes.")
8. Update the agreement access for users or groups listed in the People with access section.
   1. Locate the user or group that you want to update.
   2. Expand the Access dropdown and select the new access level for the agreement.

      Note: Select the Remove option to revoke access to the agreement for the corresponding user or group.

   Pending changes display in the People with access section.
9. Select the Save option.

   The dialog box closes. A message briefly stating the assignment was saved.

You have successfully managed user and group access for a selected agreement.
