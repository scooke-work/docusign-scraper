# Bulk Upload Documents and Set Access Controls

This topic answers the question: How do I bulk upload documents to Agreement Manager? Learn how to use the bulk upload function to upload more than five documents to Agreement Manager.

The *bulk upload* process is used when you want to upload more than five files, such as uploading a folder of files.

Important: Limitations of the bulk upload process:

- File names must contain fewer than 255 characters
- Maximum of 1,000 files per bulk upload job
- Folder structure is not maintained when uploading folders with sub-folders
- Files must be less than 100 MB each

Note: The Upload up to 1,000 files at a time permission is required for users to bulk upload files and folders to Agreement Manager. This toggle is turned on from the user's profile page by an administrator.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Expand the Add (+) menu in the top right and select the Upload option:

   The Upload documents dialog box opens.

   ![A screenshot of the Upload documents dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-view_Upload-documents-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3MzEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.GWnI5W2dVycbyBVNxK6YW1Qhr9iKe2xgOhyjkivom54&_LANG=enus "A screenshot of the Upload documents dialog box.")
5. Search for and select up to 1,000 files to bulk upload to Agreement Manager:
   - Drag the files or folders to the dialog box inside of the Drag and drop file here box.
   - Expand the Select File dropdown and select
     Browse: to search for and select files from local
     folders to upload:

     ![A screenshot of the Select File dropdown that shows the installed apps and the Browse option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Mini-App-Center_Select-File-dropdown.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3MzEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.GWnI5W2dVycbyBVNxK6YW1Qhr9iKe2xgOhyjkivom54&_LANG=enus "A screenshot of the Select File dropdown that shows the installed apps and the Browse option.")

   The Upload documents dialog box opens.

   ![A screenshot of the Upload documents dialog box that shows the number of selected files.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Upload-documents_Bulk-upload_Step-1.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3MzEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.GWnI5W2dVycbyBVNxK6YW1Qhr9iKe2xgOhyjkivom54&_LANG=enus "A screenshot of the Upload documents dialog box that shows the number of selected files.")
6. An auto-generated name is displayed in the Job name field. Accept the default value or enter a new name.

   Tip: Enter a unique job name so you can easily find it in the job monitoring hub.
7. Add more files to the upload as needed:
   1. Select the Add option.
   2. Search for and select additional files to bulk upload to Agreement Manager.

      Choices:
      - Drag the files or folders that you want to upload to the dialog box.
      - Select Upload and browse for documents to upload.
8. Select Next.

   The Allow access step is displayed in the Upload documents dialog box. You have three options:

   - Add the uploaded documents to an agreement set.
   - Share the files with others, including setting agreement-level permissions.
   - Skip this step to upload the documents without sharing or organizing into sets.

   ![A screenshot of the Upload documents dialog box that shows the Allow access step.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Upload-documents_Bulk-upload_Step-2.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3MzEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.GWnI5W2dVycbyBVNxK6YW1Qhr9iKe2xgOhyjkivom54&_LANG=enus "A screenshot of the Upload documents dialog box that shows the Allow access step.")

**Add the uploaded documents to an agreement set:**

Important: The Organize and manage all agreements
permission is required to add uploaded documents to agreement sets. See the
[Capabilities by Agreement Manager Permissions](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=vhx1743523538507.html) topic to learn
more about Agreement Manager permissions for various features.

9. Add the uploaded documents to an agreement set:

   ![A snippet of the Upload documents dialog box that shows the Add to agreement sets option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Upload-documents_Cloud-storage_Allow-access_agreement-sets-option.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3MzEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.GWnI5W2dVycbyBVNxK6YW1Qhr9iKe2xgOhyjkivom54&_LANG=enus "A snippet of the Upload documents dialog box that shows the Add to agreement sets option.")

   1. Select the Add to agreement sets option.
   2. Enter a set name in the Search sets field and select the set from the type-ahead dropdown.
   3. Select Upload and Add to Sets.
   4. Follow the prompts to start the upload process.

      A dialog box opens. The upload job is initiated and your files are being processed with AI.

   Tip: If you cannot locate the relevant set, select the + Create Set option and create a new set:
   - Complete the Name and Description fields from the New agreement set dialog box.
   - Select either Create or Create and Share.

   See the [Create an Agreement Set](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=jay1736181359265.html) support topic to learn more about creating agreement sets.

**Share the uploaded documents:**

10. Share the uploaded documents with other users and groups:

    ![A snippet of the Upload documents dialog box that shows the Share files option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Upload-documents_Cloud-storage_Allow-access_share-files-option.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3MzEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.GWnI5W2dVycbyBVNxK6YW1Qhr9iKe2xgOhyjkivom54&_LANG=enus "A snippet of the Upload documents dialog box that shows the Share files option.")

    1. Select the Share files option.
    2. Enter a name or email address in the Add user or group search field.

       A type-ahead dropdown opens, and the users and groups that match the entered search criteria are displayed.
    3. Select a user or group name from the dropdown. Continue adding users or groups as needed.
    4. Expand the Access dropdown and select an access level for each of the selected users and groups.

       Choices are: Can View, Can Edit, and Can Manage.

       Note: Select the Delete icon to remove the associated with a user or group from the list.
    5. Select Upload and Share.
    6. Follow the prompts to start the upload process.

       A dialog box opens. The upload job is initiated and your files are being processed with AI.

**Skip this step:**

11. Skip the Allow access step to continue without adding an agreement set or sharing the documents.
    1. Select the Skip this step option.
    2. Select Upload.
    3. Follow the prompts to start the upload process. Select the Upload Privately option.

       The files are added to Agreement Manager but they are not added to a set or shared with others.

    A dialog box opens. The upload job is initiated and your files are being processed with AI.

    ![A screenshot of the Upload Complete dialog box that shows the number of documents being added to Agreement Manager.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Upload-documents_Bulk-upload_Upload-Complete-modal.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3MzEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.GWnI5W2dVycbyBVNxK6YW1Qhr9iKe2xgOhyjkivom54&_LANG=enus "A screenshot of the Upload Complete dialog box that shows the number of documents being added to Agreement Manager.")
12. Select Monitor progress.

    The job monitoring hub opens and the All jobs view is displayed. The Uploaded Agreements table shows job details for each upload job.

    ![A screenshot of the Uploaded Agreements table that shows job details for all upload and sync jobs.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-view_Monitor-All-Jobs_All-JobsUploaded-tab.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3MzEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.GWnI5W2dVycbyBVNxK6YW1Qhr9iKe2xgOhyjkivom54&_LANG=enus "A screenshot of the Uploaded Agreements table that shows job details for all upload and sync jobs.")

    The created date, job name, source, number of files, and status details are displayed in the table for uploaded and ingested agreements.
13. Use the sorting arrows and filters to locate your upload job. You can filter the table by Created date, Status, and Source:
    - Expand the Created date filter, select an identifier tab, and complete the appropriate date fields.Select Apply.

      Identifier tab choices are: Between, Before, and After.

      Tip: If you choose Between, you must complete the From and To date fields.
    - Expand the Status filter, choose the preferred status, and select Apply.

      Choices are: Deleted, Upload job complete, Upload job failed, and Uploading documents.
    - Expand the Source filter, choose the relevant source, and select
      Apply.

      Choices are: Box,
      Dropbox, Email
      Ingestion, Google Drive,
      OneDrive, SharePoint,
      and Unknown.

    Upload jobs that match the applied filters are displayed in the table.
14. Review processing results, including the number of files successfully uploaded and the number of failures.
15. If any of your files failed to upload:
    1. Expand the Actions dropdown associated with your upload job and select View Details.

       The Job status page opens.
    2. Expand the Download and Fix dropdown and select the Download CSV option:

       ![A snippet of the Upload status page that shows error details and highlights the Download CSV option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Upload-documents_Bulk-upload_Upload-status_Failed-documents.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3MzEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.GWnI5W2dVycbyBVNxK6YW1Qhr9iKe2xgOhyjkivom54&_LANG=enus "A snippet of the Upload status page that shows error details and highlights the Download CSV option.")

       A .csv file containing the document name and associated error message is downloaded to your local Downloads folder.
    3. Select the Back arrow from the top-left corner.

       You are returned to the All jobs view.

    See the [View Upload Results and Exceptions](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=srq1712255787588.html&_LANG=enus "Learn how to view processing results and details for an upload job.") topic to learn more about job monitoring.
16. Select the Back arrow from the top-left corner.

    You are returned to the Completed view.

**Auto-suggestion for custom agreement types:**

When uploading agreements to Agreement Manager, if an agreement does not match an existing standard or custom type, it is identified as *Miscellaneous*. If 10 or more similar agreements are identified as *Miscellaneous*, a custom agreement type is suggested.

For example, 10 charter agreements were uploaded and identified as *Miscellaneous*. The agreements are recognized as being similar and a custom agreement type, *Charter Agreement*, is suggested:

![A snippet of the agreement list table that shows a suggested custom agreement type for a Miscellaneous document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Autosuggest_New-suggestion-available_dialog-box_charter-agreement-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3MzEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.GWnI5W2dVycbyBVNxK6YW1Qhr9iKe2xgOhyjkivom54&_LANG=enus "A snippet of the agreement list table that shows a suggested custom agreement type for a Miscellaneous document.")

17. If necessary, use the Create option to create the suggested custom agreement type. See the [Use Auto-Suggestion to Create Custom Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk2093dc24-1da8-4393-b4cd-526990bc7118.html&_LANG=enus "Learn how to create a custom agreement type from the auto-suggestion feature.") topic for steps to use this feature.
