# Access Your Completed Envelopes and Documents

Learn how to access your completed envelopes and documents in Docusign.

Docusign Intelligent Agreement Management (IAM) users can now access
their completed documents, as well as their completed eSignature envelopes, from the
same place. The Select a View dropdown makes it easy to switch
between completed envelopes and Agreement Manager documents.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Agreements page opens. The Inbox view is displayed by default.

   ![A screenshot of the Agreements page that highlights the Completed option in the navigation pane.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Access_Navigator_agreements_Agreements_Select_Completed.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYwNzMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.HBV95eEBGmDYSMm-TN0sztjiyaABIxse6FWEZjiQOdE&_LANG=enus "A screenshot of the Agreements page that highlights the Completed option in the navigation pane.")
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Completed Documents view opens. A list of documents to which the logged-in user has access is displayed in the table.

   ![A screenshot of the Agreement Manager homepage that displays the Completed Documents view.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Docusign-Navigator_Completed-documents-view.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYwNzMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.HBV95eEBGmDYSMm-TN0sztjiyaABIxse6FWEZjiQOdE&_LANG=enus "A screenshot of the Agreement Manager homepage that displays the Completed Documents view.")

   Use the Select a View dropdown to display your completed eSignature envelopes. This function is part of the page title.

   Tip: The Envelope ID column displays the envelope ID that is associated with the corresponding document. Select the Envelope ID link to view the envelope details page.
4. View your completed eSignature envelopes. Expand the Select a View dropdown and select the Envelopes option:

   ![A screenshot of the expanded Select a view dropdown that shows the Envelopes option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents_View-switcher_expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYwNzMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.HBV95eEBGmDYSMm-TN0sztjiyaABIxse6FWEZjiQOdE&_LANG=enus "A screenshot of the expanded Select a view dropdown that shows the Envelopes option.")

   The Completed Documents view refreshes and a list of completed eSignature envelopes is displayed.

   ![A screenshot of the Completed Documents view that shows a list of completed eSignature envelopes for the logged-in user.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Access-Navigator-agreements_switch-to-Completed-envelopes.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYwNzMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.HBV95eEBGmDYSMm-TN0sztjiyaABIxse6FWEZjiQOdE&_LANG=enus "A screenshot of the Completed Documents view that shows a list of completed eSignature envelopes for the logged-in user.")
