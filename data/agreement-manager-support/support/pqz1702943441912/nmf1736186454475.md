# View Agreements in a Set

Learn how to view the agreements in an agreement set.

When an agreement set is shared with a user or group, they have access to all the agreements in the set. There are two primary ways to view the agreements in a set:

- Filter the agreement list table on the Completed Documents page to display the agreements in a selected set.
- Using the Open in New Tab option from the Manage sets option.

  Note: This option is only available to users that are assigned the Organize and manage all agreements permission. For example, this option is not applicable to non-contract managers.

This topic reviews both ways to view agreements in an agreement set.

**View set agreements from the Completed Documents page:**

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Apply filters to locate the relevant set:

   ![A snippet of the Completed Documents page that displays the expanded Sets filter.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-sets_entry-point_Sets-filter.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNzQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.JQeMKCNpRUlgxADYu3FgSC9txb_Tq2VljfFPfxOeLE8&_LANG=enus "A snippet of the Completed Documents page that displays the expanded Sets filter.")

   1. Select the Filters option. The menu expands and a list of applied and suggested filters is displayed.
   2. Select the Sets filter from the list of Suggested filters.
   3. Select a set from the list.
   4. Select the Apply option.

   The filter closes and the agreements contained in the set are displayed in the table.

**View set agreements from the "Sets" option:**

5. Expand the Settings (gear) menu in the top right of the Agreement Manager homepage and select the Sets option:

   ![A screenshot of the Settings (gear) menu on the Agreement Manager homepage that highlights the Sets option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-sets_entry-point_Manage-sets-action-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNzQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.JQeMKCNpRUlgxADYu3FgSC9txb_Tq2VljfFPfxOeLE8&_LANG=enus "A screenshot of the Settings (gear) menu on the Agreement Manager homepage that highlights the Sets option.")

   The Agreement sets page opens. The All agreement sets view is displayed. The table displays sets that were shared with or created by the logged-in user.
6. Locate the set that you want to view, expand the More actions (three dots) menu, and select Open in new tab:

   ![A snippet of the All agreement sets view that highlights the Open in new tab option in the More actions menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Sets-dialog_All-agreement-sets_Open-in-New-Tab.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNzQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.JQeMKCNpRUlgxADYu3FgSC9txb_Tq2VljfFPfxOeLE8&_LANG=enus "A snippet of the All agreement sets view that highlights the Open in new tab option in the More actions menu.")

   The Completed Documents page opens in a new browser tab. The agreements contained within the selected set are displayed in the table.

   ![A snippet of the Completed Documents page that displays the agreements contained within a selected set.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_filtered-by-set_view-agreements.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNzQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.JQeMKCNpRUlgxADYu3FgSC9txb_Tq2VljfFPfxOeLE8&_LANG=enus "A snippet of the Completed Documents page that displays the agreements contained within a selected set.")

You have successfully viewed the agreements in a selected agreement set.
