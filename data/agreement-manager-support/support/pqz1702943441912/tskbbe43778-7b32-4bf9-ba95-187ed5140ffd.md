# Remove Documents from the Agreement List Table

Learn how to remove documents from the agreement list table on the Completed Documents page. This topic answers the question: How do I delete agreements from Agreement Manager?

The Remove function provides a quick way to delete unnecessary documents from the agreement list table.

Note: **Permission requirements**

The Remove function requires users to have at least one of these permissions:

- Organize and manage all agreements access to *Agreement Manager*
- Can Manage access to the *agreement*

See the [Agreement Manager Permissions, Profiles, and Roles](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=eut1712157553977.html&_LANG=enus "Read about the available Agreement Manager permissions, including how users and groups are granted access to Agreement Manager agreements and example roles for each permission.")ll support topic to learn more about Agreement Manager permissions and document-level access control.

The Remove option is available at the document-level and for a group of selected documents. Both entry points are accessed from the Agreement Manager homepage. This task reviews both entry points.

Once deleted, users and administrators no longer have access to the agreement and its data. This action does not remove the agreement from its original envelope or void the contract.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.

**Locate the relevant agreements**

4. Perform a search and apply filters to locate the agreements that you want to remove:
   1. Enter a keyword or string in the search field and select a suggestion from the dropdown or press the Enter key.

      The agreements containing the keyword or search string that you entered are displayed in the table.
   2. Use the filters to refine the list of search results. For example, use the Parties filter to view a list of agreements for a specific party.

      1. Expand the Filters option and select the filter that you want to apply.
      2. Choose a filter value.
      3. Select the Apply option.

      The agreements that match the applied filters are displayed in the table.

**Delete a single document**

5. Remove a single document from the agreement list:

   ![A snippet of the agreement list table that highlights the Remove from Account option in the More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-list-table_Remove-option_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY1MTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.a6YqD8IDCmHhYVUgOVqbfy0XggMswhsPAoZhkZ8q3Cg&_LANG=enus "A snippet of the agreement list table that highlights the Remove from Account option in the More actions (three dots) menu.")

   1. Locate the agreement that you want to remove and expand the corresponding More actions (three dots) menu.
   2. Select the Remove from Account option:

   The Permanently remove this agreement? dialog box opens.
6. Review the message and select the Remove Agreement option.

   The agreement is removed from the repository and its data is no longer available in Agreement Manager.

   Note: Removing the agreement from Agreement Manager does not remove it from its original envelope or void the contract.

   You are returned to the Agreement Manager homepage. Your search criteria and filters are still applied to the table.

**Remove a group of documents**

7. Select the agreements that you want to remove from the account.
   - Select the individual agreements.
   - Select the Choose all checkbox from the header row to select all the agreements on that page of results.

     Tip: The Choose all option only selects agreements on the current page of search results. It does not select any agreements on subsequent pages of results.
   - Expand the Selected dropdown and select the Select all that match your filters option. All the agreements in the table that matched your applied filters are selected.

     Tip: The Select all that match your filters option is only available if filters are applied to the agreement list table.

   When you select at least one document, an actions bar opens and the Remove (trash) icon is available.
8. Select the Remove (trash) icon from the actions bar:

   ![A snippet of the agreement list table that shows a group of selected documents and the Remove (trash) icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-list-table_Actions-bar_Remove-option_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY1MTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.a6YqD8IDCmHhYVUgOVqbfy0XggMswhsPAoZhkZ8q3Cg&_LANG=enus "A snippet of the agreement list table that shows a group of selected documents and the Remove (trash) icon.")

   The Remove documents from this account? dialog box opens.
9. Review the message and select the Remove from Account option.

   The agreements are removed from the repository and the associated data is no longer available in Agreement Manager.

   Note: Removing agreements from Agreement Manager does not remove them from the original envelope or void the contract.

   You are returned to the Agreement Manager

You have successfully removed documents from the Agreement Manager repository.
