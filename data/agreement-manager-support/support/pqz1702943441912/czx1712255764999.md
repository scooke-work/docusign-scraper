# Upload Documents to Agreement Manager

Learn how to use the Upload documents function to upload agreements to Agreement Manager. This topic answers the question: How do I upload documents to Agreement Manager?

You can upload up to five documents directly to the repository without being assigned Agreement Manager
permissions. Once uploaded, the documents are accessible for users and groups with
the appropriate permissions.

Important: Limitations of single document uploads include:

- File names must contain fewer than 255 characters
- Files must be less than 100 MB each

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Expand the Add (+) dropdown from the top-right and select the Upload option:

   The Upload documents dialog box opens.

   ![A screenshot of the Upload documents dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-view_Upload-documents-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3MzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.rbPC0MLHc-v66ODrOOcXb6d1fF-LItwkuGdtRbo9siE&_LANG=enus "A screenshot of the Upload documents dialog box.")
5. Search for and select up to five files to upload to Agreement Manager:

   The dialog box closes and you are returned to the Completed Documents view. A message states that the upload was successful and the files are being processed with AI.
