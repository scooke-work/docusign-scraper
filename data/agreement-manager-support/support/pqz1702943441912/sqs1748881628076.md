# Display the Envelope ID in the Agreement List Table

Learn how to configure the agreement list table to display the Envelope ID column.

The agreement list table is customizable. Update the table to display the envelope IDs for your completed documents that were ingested from eSignature envelopes.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.

**Display the Envelope iD column in the table**

4. Select the Show or Hide Fields (toggle) icon located to the right of the filters:

   ![A snippet of the Completed Documents view that highlights the Show or Hide FIelds (toggle) icon on the right-side of the page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Customize-columns-gear-icon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0ODAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.kZXsDhVP84e3BeUzCMT9MkiZ9j1ELc8S3rb2i9R8NSs&_LANG=enus "A snippet of the Completed Documents view that highlights the Show or Hide FIelds (toggle) icon on the right-side of the page.")

   The Show or Hide FIelds dialog box opens. A number of fields are turned on (blue) by default. These fields represent the columns in the agreement list table. Columns display in the order that the fields are represented in the dialog box.
5. Scroll to locate the Envelope ID column and select the toggle to turn it on (blue). You can also search for the field using the search field at the top of the dialog box.

   ![A snippet of the Show or Hide Fields dialog box that highlights the Envelope ID column.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed_view_Customize_columns-dialog_box_Envelope_ID.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0ODAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.kZXsDhVP84e3BeUzCMT9MkiZ9j1ELc8S3rb2i9R8NSs&_LANG=enus "A snippet of the Show or Hide Fields dialog box that highlights the Envelope ID column.")
6. Re-order the columns so the Envelope ID is displayed in the appropriate location. Hover the cursor over the Reorder (dots) icon and drag the Envelope ID

   ![A snippet of the Show or Hide Fields dialog box that displays the Envelope ID column in the process of being dragged to a new location.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-view_Customize-columns-dialog-box_Envelope-ID-Reorder.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0ODAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.kZXsDhVP84e3BeUzCMT9MkiZ9j1ELc8S3rb2i9R8NSs&_LANG=enus "A snippet of the Show or Hide Fields dialog box that displays the Envelope ID column in the process of being dragged to a new location.")

   For example, drag the Envelope ID up to the top five spots to display it in the agreement list table without scrolling.

   Tip: The "grabbing" cursor displays when you hover over an element that can be dragged or moved.
7. Select the Save option from the bottom of the dialog box.

   The dialog box closes. The new configuration is reflected in the agreement list table.

   ![A snippet of the Completed Documents view that highlights the Envelope ID column in the agreement list table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-view_Customize-columns_Envelope-ID-in-table.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0ODAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.kZXsDhVP84e3BeUzCMT9MkiZ9j1ELc8S3rb2i9R8NSs&_LANG=enus "A snippet of the Completed Documents view that highlights the Envelope ID column in the agreement list table.")

   Note: If an agreement was imported from a completed eSignature envelope, the envelope ID is displayed in the Envelope ID column for the agreement. Select the Envelope ID link to view the associated envelope details.

   See the [View Envelope Details from the Agreement List Table](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk15aecb4b-98ec-4b40-a739-61a3715cba02.html&_LANG=enus "Learn how to view the envelope details for an agreement that was imported from a completed eSignature envelope.") topic to learn more about viewing envelope details from the agreement list table.

You have successfully used the Show or Hide Fields (toggle) feature to display the Envelope ID column in the agreement list table.
