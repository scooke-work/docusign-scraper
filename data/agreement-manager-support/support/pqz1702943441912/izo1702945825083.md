# View and Download an Agreement Report

Learn how to view an agreement chart, drill down to the detailed report view, apply filters, and download agreement data. This topic answers the question: What are my reporting options for Agreement Manager agreements, including applying filters and downloading results?

When you apply filters to the Agreements Dashboard, the agreements that match the filters are reflected in the insights and charts.

Note: **Downloading reports to CSV files**

When downloading reports to CSV files from the Insights reporting hub, there is an agreement limit of 25,000 agreements for all IAM reports. This count includes your custom reports.

**Suggested workaround:** Adjust the report filters to run the report in batches until all the relevant agreements are downloaded. For example, run the report in batches according to varying date ranges. Once downloaded, combine the data into one CSV file.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Insights from the global header.

   The Insights hub opens and the Overview dashboard is displayed.
3. Select Agreements from the navigation pane on the left.

   The Agreements Dashboard view opens.

   Note: Select a chart segment or bar to view the agreements represented in the bar or section. The report view opens and the corresponding agreements are displayed in the table.
7. Go to the detailed report view for a selected insight or chart. Select (click) the title of the insight or chart that you want to view.

   You can also expand the More actions (three dots) menu and select the View Report option:

   ![A snippet of an Agreements Dashboard chart that shows the View Report option in the More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-dashboard_Expanded-More-actions-menu_View-Report.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5MjEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.tx_EfTBFRm5eVhqRsccpqtfY7weLSTJahZf8xMnJ6u8&_LANG=enus "A snippet of an Agreements Dashboard chart that shows the View Report option in the More actions (three dots) menu.")

   Note: For charts, you have the option to View as Bar Chart, View as Stacked Bar Chart, or Download as png.

   The detailed report view opens for the corresponding insight or chart.

   ![A screenshot of the detailed report view for the Duration of agreements chart.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Agreements_Detailed-report-example-for-download.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5MjEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.tx_EfTBFRm5eVhqRsccpqtfY7weLSTJahZf8xMnJ6u8&_LANG=enus "A screenshot of the detailed report view for the Duration of agreements chart.")
8. Verify that the report contains the appropriate agreements. Use the filters to refine the agreement list as needed.

   Tip: Select the All Filters option to choose additional filters to apply to the report. For example, filter the report by agreement status. Expand the Status filter, choose the relevant statuses, and select Apply:![A screenshot of the All Filters dialog box that shows the Status filter.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-statuses_All-Filters_Status-filter.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5MjEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.tx_EfTBFRm5eVhqRsccpqtfY7weLSTJahZf8xMnJ6u8&_LANG=enus "A screenshot of the All Filters dialog box that shows the Status filter.")
9. Customize the columns in the table:

   Important: The Customize columns function only applies to the data that is displayed in the agreement list table. When downloading or exporting the report, all fields are included in the output file. This includes both custom and standard fields.

   ![A screenshot of the Customize columns dialog box that shows the columns you can choose to display in the agreements table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Overview-dashboard_Customize-columns.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5MjEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.tx_EfTBFRm5eVhqRsccpqtfY7weLSTJahZf8xMnJ6u8&_LANG=enus "A screenshot of the Customize columns dialog box that shows the columns you can choose to display in the agreements table.")

   1. Select the Customize columns (toggle) icon.

      The Customize columns dialog box opens. These fields represent the columns in the table.
   2. Select the toggles to turn the associated field on (blue) or off (white).
   3. Re-order the columns as needed. Hover the cursor over the Reorder (dots) icon for the column that you want to move and drag it to the appropriate location.
   4. Select the Save option.
10. Download or save the report:

    ![A snippet of the report view that highlights the More actions (three dots) menu used to download or save the report.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Report-view_Dropdown-expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5MjEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.tx_EfTBFRm5eVhqRsccpqtfY7weLSTJahZf8xMnJ6u8&_LANG=enus "A snippet of the report view that highlights the More actions (three dots) menu used to download or save the report.")

    - Expand the More actions (three dots) menu in the top right and select the Download.option.

      The associated report is downloaded to your local Downloads folder.
    - Expand the More actions (three dots) menu and select the Save option. The Save Report dialog box opens.

      - Enter a unique name for the report in the Report name field. The field defaults to the name of the selected report.
      - Enter a description about the report's purpose in the Description field.
      - Select the Save option.The report is saved to the Custom tab on the Reports view.
11. Set up one-time and recurring email reports for yourself and others. Select the Manage Notification (bell) icon and complete the Create subscription dialog box:

    ![A screenshot of the Create subscription dialog box that is used to subscribe to report notifications.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Create-Subscription.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5MjEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.tx_EfTBFRm5eVhqRsccpqtfY7weLSTJahZf8xMnJ6u8&_LANG=enus "A screenshot of the Create subscription dialog box that is used to subscribe to report notifications.")

    Note: When you create a subscription, you have the option to add individual users and groups. Once created, the subscription is displayed in the Manage report subscriptions dialog box for any individual users who were added. Currently, the subscription is not visible to users if they were added as part of a group.

    See the [Subscribe to Receive Report Notifications](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=wfw1734545937270.html&_LANG=enus "Learn how to set up one-time or recurring email reports for yourself or others to receive notifications about your obligations, renewals, or agreements.") topic to learn more about this feature.

    Favorite (star) icon located in the top right:

    ![A snippet of the detailed report view for a selected report that highlights the Favorite (star) icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-dashboard_Report-details_Favorite-icon_selected.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5MjEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.tx_EfTBFRm5eVhqRsccpqtfY7weLSTJahZf8xMnJ6u8&_LANG=enus "A snippet of the detailed report view for a selected report that highlights the Favorite (star) icon.")Your starred dashboards and reports are displayed in the Your
    Favorites section on the Overview
    dashboard.
13. Return to the Agreements Dashboard. Select the Agreements Dashboard option from the breadcrumb navigation at the top of the page.

    The Agreements Dashboard opens.
14. Repeat this process to view, download, or save other agreement reports.
