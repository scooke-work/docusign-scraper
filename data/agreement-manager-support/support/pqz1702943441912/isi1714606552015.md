# Verify User and Group Changes from the Audit Logs

This topic answers the question: How do I verify the user and group permission changes that I made? Learn how to access the Audit Logs to verify permission and user management changes to your account.

Audit logs provide a record of the changes made to your account. You can select a date range and
use the Export to CSV option to export up to 1,000 records at
a time.

In Agreement Manager, audit log events are captured at the account-level. Make sure that you
are updating records for the correct account.

1. Log in to your Docusign account using administrator credentials.

   The Docusign Home page opens.
2. Select Admin from the global header.

   The Admin menu opens and the Accounts view is displayed.

**Select an account (for organizations with multiple accounts)**

Important: User and group permissions are managed at the *account*-level. Make sure that you are updating records from Accounts administration instead of Organization administration. If your organization has multiple accounts, verify that you are updating the correct account.

3. Expand the Accounts dropdown from the top of the navigation pane and select the relevant account:

   ![A snippet of the Accounts view that highlights the Accounts dropdown in the pane on the left.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Permissions_Admin-menu_Accounts-dropdown_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzMjYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.maJA__-9QCvb2u4UoTwUL05BrjNqrpeloVctopSJOpo&_LANG=enus "A snippet of the Accounts view that highlights the Accounts dropdown in the pane on the left.")

   The Accounts view refreshes and the Overview page is displayed for the selected account.

**View audit logs**

4. Select the Audit Logs option from the AUDITING section in the navigation pane.

   The Audit Logs view opens. The audit log table displays records of changes made to your account. The list is filtered to display events over the last seven days.
5. Filter the table to locate events that occurred over a specific time period.
   1. Expand the Event Date filter located above the table. The Filter Audit Logs dialog box opens.
   2. Expand the Event Date dropdown and select a time period. Choices include: Custom, Last 7 Days, Last 30 Days,
      Last 6 Months, and Last 12 Months.

      Note: If you select Custom, enter a date range in the From and To date fields. Complete the From Time and End Time fields to locate specific events.
   3. Select Apply.

      The Filter Audit Logs dialog box closes. Events for the specified time period are displayed in the Audit Logs table.
6. Review the table to verify the user and group management changes for your account.

   The table displays the Event Date, User, Description, Field, Old Value, and New Value for each change.

**Export table data**

7. Export the Audit Logs table data. Select the Export to CSV option.

   The auditLog.csv file is downloaded to your local Downloads folder.

You have successfully viewed the audit log to verify permission and user management changes for your account.
