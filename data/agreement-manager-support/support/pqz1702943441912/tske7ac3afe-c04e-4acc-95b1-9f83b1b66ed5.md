# Review Job Results from the Notification Email

Learn how to view the results of an upload job from the email notification. A system-generated email is sent when the upload process is completed.

## Before you begin

This task is part of the *Upload and sync* phase of bulk metadata editing in Agreement Manager. It relies on you completing the previous phases, including:

- [Download the Pre-Formatted Spreadsheet File](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk1b94afc6-712f-4d71-868e-0787867354d8.html&_LANG=enus "Learn how to download the pre-formatted spreadsheet file for a group of documents.")
- [Edit the Metadata Spreadsheet](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk8710c3f2-c881-4219-96b0-2b287e8b4e03.html&_LANG=enus "Learn how to edit the metadata spreadsheet file to bulk update the agreement data for the selected documents.")
- [Import Your Edited Metadata Spreadsheet](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk5ab1cd5b-00f7-42a9-bd77-728baadff903.html&_LANG=enus "Learn how to import your edited metadata spreadsheet using the Add (+) menu located on the Agreement Manager homepage.")

When you import the new data, the existing values are overwritten, including the AI-extracted values. Agreement Manager applies all valid cell values and flags formatting errors in the job monitoring hub.

A system-generated email notification is sent when the upload is completed. The email contains a brief summary of the import results, including the number of files that failed to update.

This topic begins from the system-generated email notification:

![A screenshot of the system-generated notification email that provides a brief summary of the import results.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Bulk-metadata-editing_Notification-email_View-Details.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4MTcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.elPNQUFlbAE6TbuMEgavDhW0zQPUtLV7zTVWWP4-YvY&_LANG=enus "A screenshot of the system-generated notification email that provides a brief summary of the import results.")

1. Review the notification email for a high-level summary of the import results. For example, the number of files that failed to update and possible reasons.
2. Select the View Details option.

   The Job status page opens. The files that failed to update are listed in the Error details table.

   ![A screenshot of the Job status page for the upload job, including the error details for the files that failed to update.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Bulk-metadata-editing_Notification-email_View-Job-status.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4MTcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.elPNQUFlbAE6TbuMEgavDhW0zQPUtLV7zTVWWP4-YvY&_LANG=enus "A screenshot of the Job status page for the upload job, including the error details for the files that failed to update.")

   When a data value fails validation, that field is not updated in the associated document and an error is logged.
3. Review the error details for the files that failed to update, including the reason that the data value failed:

   ![A snippet of the Job status page that shows the Error details table, including the reason the files failed to update.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Bulk-metadata-editing_Job-status_Error-details_Fontara-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4MTcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.elPNQUFlbAE6TbuMEgavDhW0zQPUtLV7zTVWWP4-YvY&_LANG=enus "A snippet of the Job status page that shows the Error details table, including the reason the files failed to update.")

   Tip: Select the View option to go to the Agreement preview page for the corresponding document. This page opens in a new tab.
4. Resolve the errors and import the spreadsheet again. Repeat this process until all the errors are resolved and the files are updated.

You have successfully viewed the results of an upload job from the email notification. Continue to the last step of the process, [Download and Fix Your Metadata Validation Errors](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk788b581e-b10b-41cc-818e-04108b62210c.html&_LANG=enus "Learn how to download and fix your metadata validation errors from the upload process.").
