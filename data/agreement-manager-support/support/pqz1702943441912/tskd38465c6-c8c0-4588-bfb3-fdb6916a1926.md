# Add Examples of Your Custom Agreement Types

Learn how to complete the second step of the AI detection set-up process, *Add examples of this agreement type*.

## Before you begin

This topic reviews the next step of the AI detection set-up process, *Add examples of this agreement type*. These steps are completed by administrators and contract managers.

Remember: Administrators automatically have access to configure agreement data. For non-administrator users, agreement type management tasks require Organize and manage all agreements access to Agreement Manager agreements and functionality.

Complete these topics before attempting this task:

- [Configure New Custom Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=gke1727282238086.html&_LANG=enus "Learn how to create custom agreement types to support organizational use cases.")
- [Define Your Custom Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tske5fe1ad5-2832-4e6c-9177-f33a9ffade15.html&_LANG=enus "Learn how to initiate the AI detection set-up process and complete the first step to define the custom type.")

When you set up custom extractions, you are prompted to provide examples, in addition to a text
definition. The definition tells the AI model what the data means and the examples
show how the data displays in your agreements.

In the previous step, you provided a definition for your custom agreement. After you selected the
Next option, the AI
detection set-up wizard advanced to the next step.
You are prompted to select examples of the custom
type.

1. Review the dialog box message for a brief
   overview of the next step. Select the
   Select Examples
   option:

   ![A screenshot of the introduction message for the next step of the AI detection set-up process.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Types-Manager_AI-detect-setup_Need-examples_prompt.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.3oI_mUYMDrZL_dZJfPlwv9aXwP7qh9SdZzvpRQP6kNg&_LANG=enus "A screenshot of the introduction message for the next step of the AI detection set-up process.")

   The Select relevant
   documents page opens. Use this page to
   select documents that are good examples of the
   custom type.
2. Locate documents that are good examples of the custom agreement type:

   ![A snippet of the Select relevant documents page that shows the search field and filter options to locate examples.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Types-Manager_AI-detect-setup_Select-examples_search-filter-options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.3oI_mUYMDrZL_dZJfPlwv9aXwP7qh9SdZzvpRQP6kNg&_LANG=enus "A snippet of the Select relevant documents page that shows the search field and filter options to locate examples.")

   1. Search for the relevant agreements. Enter a keyword or search string in the search field. The dropdown opens and search suggestions are displayed. Select a search suggestion or press the Enter key.
   2. Filter the agreement list table to refine the results list. For example, filter by Sets, Status, Parties, Agreement Type, or AI Suggestions.

      Tip: Use the All Filters option to apply additional filters to the results table.

      Expand the filter, choose the relevant filter value, and select the Apply option.

   The agreement list table updates and the documents that matched the search criteria and applied filters are displayed.
3. Choose the documents that exemplify the custom agreement type:

   ![A snippet of the Select relevant documents page that shows a number of selected examples.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Types-Manager_AI-detect-setup_Selected-documents_Save-Eamples.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.3oI_mUYMDrZL_dZJfPlwv9aXwP7qh9SdZzvpRQP6kNg&_LANG=enus "A snippet of the Select relevant documents page that shows a number of selected examples.")

   Tip: Select the Show Selected Agreements option to review your selected documents. If necessary, select the Return to Search Results option and make changes to your selections.
4. Select the Save Examples option.

   The AI detection set-up wizard advances to the next step and you are prompted to Test for accuracy.

   Complete the [Test the Extraction Accuracy for Your Custom Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=ogw1740503179389.html&_LANG=enus "Learn how to test the accuracy of the AI detection for your custom agreement types.") task to test the accuracy of your AI
   detection.

You have successfully completed the second step of the AI detection set-up process, *Add examples of this agreement type*.
