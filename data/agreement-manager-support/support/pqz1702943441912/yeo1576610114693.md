# Add Attribute Values to a Document in Docusign CLM

Learn how to add attribute values to a document in Docusign CLM.

You can add attribute values to documents in Docusign CLM. Attribute values provide content and format validation for
single and multiple entries in a document.

Note: Attribute fields that
are string fields have a limit of 400 characters. An example is the Description field or any
custom attribute that you define as a string field.

1. Log in to Docusign CLM.
2. Select Documents in the navigation header.

   The Documents page opens.
3. Navigate to the folder containing your document.
4. Select the document to open it.

   The document displays in Document Review.
5. Select the Attributes tab on the navigation panel.

   The Attributes panel opens.

   Note: Attributes that are noted with a red asterisk are required.
6.    "Document Details" section or "Groups" section to allow editing.
7. Complete each attribute field as needed according to type:

   - For dropdown menus, select the menu and then choose a value from the options.
   - For freeform entries, enter a value that matches the requested format.
   - For date selectors, manually enter the date in MM/DD/YYYY format, or use the date
     picker.
8. Select Save.

   You have successfully added attribute values to a selected document in Docusign CLM. If any values do not match the requested format, an error
   displays. If any required fields do not have values, a notification displays.
