# Create Custom Obligation Types

Learn how to create custom obligation types to support organizational and departmental use cases.

Docusign provides a set of standard obligation types, however, you can create custom types to more accurately categorize your obligations.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Expand the Settings (gear) menu from the right-side of the page and select the Obligations option:

   ![A screenshot of the Settings (gear) menu that highlights the Obligation option in the expanded](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Obligation-type-management_Manage-Obligation-Types_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxOTEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.W6GNA4Es2x1MbkGIrUaJqzYpTh9lTaj2gsiE02xobB8&_LANG=enus "A screenshot of the Settings (gear) menu that highlights the Obligation option in the expanded")

   The Manage Obligation Types page opens. The Obligation Types view is displayed. A list of custom and standard obligation types is displayed in the table.
5. Select the + New Obligation Type option from the top right.

   ![A screenshot of the Obligation Types view that shows the list of available standard and custom obligation types for the account.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Obligation-Types_New-Obligation-Type.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxOTEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.W6GNA4Es2x1MbkGIrUaJqzYpTh9lTaj2gsiE02xobB8&_LANG=enus "A screenshot of the Obligation Types view that shows the list of available standard and custom obligation types for the account.")

   The New obligation type dialog box opens.

   ![A screenshot of the New obligation type dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Obligation-Types_New-obligation-type-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxOTEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.W6GNA4Es2x1MbkGIrUaJqzYpTh9lTaj2gsiE02xobB8&_LANG=enus "A screenshot of the New obligation type dialog box.")
6. Enter a name for the obligation type in the Name field. The name cannot exceed 50 characters.
7. Select the Save option.

   The dialog box closes. The new obligation type is displayed in the Obligation Types table.

   Tip: The CREATED BY column displays "user" for custom obligation types and "Docusign" for standard types.
8. Continue to create new obligation types as needed.
9. Return to the Agreement Manager homepage. Select the Close (x) icon from the top right.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. The search criteria and filters are still applied.

You have successfully created a new obligation type.
