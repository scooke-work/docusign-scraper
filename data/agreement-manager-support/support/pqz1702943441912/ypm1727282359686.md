# Update Custom Agreement Type Details

Learn how to update the details for your custom agreement types.

Administrators and contract managers can create and maintain the custom agreement types for their account. For example, if your organization changes how a category of agreements is defined, you can edit the details accordingly.

Note: Administrators automatically have access to configure agreement data. For non-administrator users, this feature requires Organize and manage all agreements access to Agreement Manager. These tasks are primarily completed by administrators and contract managers.

Custom agreement types are managed from the *Agreement Types Manager*. Access this feature from the Settings (gear) menu on the Agreement Manager homepage.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. A list of documents to which the logged-in user has access is displayed in the table.
4. Expand the Settings (gear) dropdown from the top right and select the Agreement Types option:

   ![A screenshot of the Agreement Manager homepage that highlights the Agreement Types option in the Settings (gear) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-types-manager_Settings-menu_Agreement-Types_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3MzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.QFJzUktwa1ImFzkFMC122GRu20hJX56ZPm5zMLcfheI&_LANG=enus "A screenshot of the Agreement Manager homepage that highlights the Agreement Types option in the Settings (gear) menu.")

   The Manage Agreement Types page opens. The account's standard and custom agreement types are displayed in the Agreement Types table.
5. Locate the custom agreement type that you want to update.

   Important: Only the user-created, custom agreement types can be updated.

   ![A snippet of the Manage Agreement Types page that shows the Created By filter.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Types_Created-By-filter_User-selected.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3MzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.QFJzUktwa1ImFzkFMC122GRu20hJX56ZPm5zMLcfheI&_LANG=enus "A snippet of the Manage Agreement Types page that shows the Created By filter.")

   1. Expand (click) the Created By filter and select the User option.
   2. Select the Apply option.

      The Agreement Types table updates and all the account's custom agreement types are displayed.
   3. Refine the results list as needed. For example, apply the Category and AI Status filters or perform a keyword search.
6. Expand the More actions (three dots) menu for the relevant custom agreement type and select the Edit Details option:

   ![A snippet of the Agreement Types view that highlights the Edit Details option in the More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-type-management_Edit-details.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3MzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.QFJzUktwa1ImFzkFMC122GRu20hJX56ZPm5zMLcfheI&_LANG=enus "A snippet of the Agreement Types view that highlights the Edit Details option in the More actions (three dots) menu.")

   The Edit agreement type details dialog box opens.
7. Update the details for the custom agreement type as needed.

   ![A screenshot of the Edit agreement type details dialog box that shows editable fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-type-management_Edit-agreement-type-details-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3MzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.QFJzUktwa1ImFzkFMC122GRu20hJX56ZPm5zMLcfheI&_LANG=enus "A screenshot of the Edit agreement type details dialog box that shows editable fields.")

   1. Enter a new name in the Name field or edit the existing name.
   2. Update the Description for the type. For example, if the agreement type is being used for a different purpose, update the description to reflect the changes.
   3. Expand the Category dropdown and update the
      classification for the agreement type. For example, if the agreement
      type is classified incorrectly, select the relevant category from the
      dropdown.
   4. Select the Save option.

   The dialog box closes and the Agreement type details view is displayed for the selected agreement type. The associated standard and custom fields are displayed in the Fields table.

   ![A screenshot of the Agreement type details view that shows the standard and custom fields that are associated with the agreement type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-types_New-agreement-type_standard-contract-fields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3MzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.QFJzUktwa1ImFzkFMC122GRu20hJX56ZPm5zMLcfheI&_LANG=enus "A screenshot of the Agreement type details view that shows the standard and custom fields that are associated with the agreement type.")

   From this view, you can:

   - Update custom field details
   - Add custom and standard fields to the type
   - Remove fields from the agreement type
8. Return to the Agreement Types view. Select the Agreement Types option from the breadcrumb navigation in the top left.

   You are returned to the Agreement Types view.
9. Return to the Agreement Manager homepage. Select the Close (x) icon from the top right.

   You are returned to the Agreement Manager homepage. The Completed Documents view is displayed.

You have successfully edited the details for a custom agreement type.
