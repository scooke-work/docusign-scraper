# Create an Agreement Set

Learn how to create a new agreement set to ensure that the right people have access to relevant agreements. This topic answers the question: How can I make sure that the appropriate people have access to relevant agreements?

Contract managers use agreement sets to make sure the right people have access to relevant agreements. They add agreements into sets and share sets with the appropriate users and groups. Sets also serve organizational purposes by providing a way to group agreements.

Note: An organization can have up to 50 agreement sets per account. If you exceed this limit, an error message is displayed.

This topic shows how to create a new agreement set from the Sets option in the Settings (gear) menu.

Remember: Agreements sets can also be created from the Agreement sets pane on the Agreement preview page.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Expand the Settings (gear) menu from the top right and select the Sets option:

   ![A screenshot of the Settings (gear) menu on the Agreement Manager homepage that highlights the Sets option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-sets_entry-point_Manage-sets-action-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczMjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.LmZcInbj68rfCvIlZZvD7xM2xXjMNjU-hXdUQrogvRA&_LANG=enus "A screenshot of the Settings (gear) menu on the Agreement Manager homepage that highlights the Sets option.")

   The Agreement sets manager opens. The All agreement sets page is displayed by default. The table lists the sets that were either shared with or created by the logged-in user.

   ![A screenshot of the All agreement sets page that displays the sets that were created by or shared with the logged-in user.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Sets-dialog_All-agreement-sets_New-Agreement-Set.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczMjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.LmZcInbj68rfCvIlZZvD7xM2xXjMNjU-hXdUQrogvRA&_LANG=enus "A screenshot of the All agreement sets page that displays the sets that were created by or shared with the logged-in user.")
5. Select the + New Agreement Set option from the top right.

   The New agreement set page opens.
6. Complete the Set details step of the set creation process:

   ![A screenshot of the New agreement set page that shows the first step of the set creation process, Set details.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Sets_Add-new-agreement-set.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczMjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.LmZcInbj68rfCvIlZZvD7xM2xXjMNjU-hXdUQrogvRA&_LANG=enus "A screenshot of the New agreement set page that shows the first step of the set creation process, Set details.")

   1. Enter a name for the agreement set in the Set name field.

      Tip: The Set name field is limited to 60 characters.
   2. Enter a description for the agreement set in the Description field.

      Tip: The Description field is limited to 200 characters. This field is optional. The entered value is displayed in the All agreement sets table.
   3. Select the Next option.

   The next step of the process, *Smart filters*, is displayed.
7. Select the Choose filters option:

   ![A snippet of the New agreement set page that shows the second step in the set creation process, Smart filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-set-management_create-new-set_step-2-choose-filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczMjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.LmZcInbj68rfCvIlZZvD7xM2xXjMNjU-hXdUQrogvRA&_LANG=enus "A snippet of the New agreement set page that shows the second step in the set creation process, Smart filters.")

   Note: Select Skip for now to skip this step. You can add or update smart filters at any time.

   The Smart filters step updates and the available filters are displayed.
8. Complete the Smart filters step to add agreements to the set automatically based on the applied filters:

   ![A snippet of the Smart filters step that highlights the available filters, including the Agreement Type and Parties filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-set-management_create-new-set_step-2Apply-filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczMjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.LmZcInbj68rfCvIlZZvD7xM2xXjMNjU-hXdUQrogvRA&_LANG=enus "A snippet of the Smart filters step that highlights the available filters, including the Agreement Type and Parties filters.")

   1. Expand the filter that you want to apply.

      Tip: Select the Filters option to access additional filters, including custom fields that are enabled for AI detection. Locate the filter that you want to apply from the Available list.![A snippet of the Filters dialog box that shows the search field and a list of available filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-set-management_create-new-set_step-2All-Filters-snippet.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczMjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.LmZcInbj68rfCvIlZZvD7xM2xXjMNjU-hXdUQrogvRA&_LANG=enus "A snippet of the Filters dialog box that shows the search field and a list of available filters.") As you apply filters, they are added to the Applied section. Select the Save option to save your filters.
   2. Select or enter a value for the filter.
   3. Select Apply.

      The agreements that match the filters that you applied are listed in the table.
   4. Continue applying filters until the list is representative of the set's purpose.
   5. Select Next.

   The next step of the process, *People with access*, is displayed.
9. Complete the People with access step to share the agreement set with users and groups:

   ![A snippet of the third step that shows the users and groups with whom the set is being shared, including the assigned access level.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-set-management_create-new-set_step-3-access.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczMjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.LmZcInbj68rfCvIlZZvD7xM2xXjMNjU-hXdUQrogvRA&_LANG=enus "A snippet of the third step that shows the users and groups with whom the set is being shared, including the assigned access level.")

   1. Search for and select the user or group with whom you want to share the set.

      Enter a name in the Add user or group search field and select the name from the type-ahead dropdown list.

      Important:

      The Notify users by email option is selected by default. Leave this checkbox selected to notify users that an agreement set was shared with them.

      If you de-select this option, users are not notified of the shared agreement set.
   2. Expand the Access dropdown and select the appropriate access level for agreements in the set.

      Choices are: View, Edit, and Manage.

      Assign an access level for each of the selected users and groups.
10. Select the Create Set option.

    You are returned to the All agreement sets view. The new agreement set is displayed in the table. When new agreements match the smart filters, they are added to the set automatically.

    If an agreement's attribute values are updated and no longer match the filters, it is removed from the set dynamically. For example, if the agreement type is updated and the agreement no longer matches the applied smart filters.

    Important: **Creating a set without smart filters**

    If the agreement set was created without smart filters, the next step is to add agreements to the set. Review these topics to learn more about adding agreements to sets without smart filters:

    - [Add Agreements to Sets Based on Attribute Values](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=txb1736186243024.html&_LANG=enus "Learn how to apply filters to locate agreements to add to an agreement set. This topic answers the question: How do I control access to agreements based on attribute values?")
    - [Add Individual Agreements to Existing Agreement Sets](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=rdi1736186285124.html&_LANG=enus "Learn how to add agreements to an existing set from various places within Agreement Manager. This topic answers the question: How do I add additional agreements to an existing set?")

You have successfully created an agreement set, including applying smart filters and sharing the set with others.
