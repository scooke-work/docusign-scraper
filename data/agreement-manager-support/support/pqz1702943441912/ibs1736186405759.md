# Remove Agreements from Existing Agreement Sets

Learn how to remove agreements from an existing agreement set.

Agreements can be removed from existing sets from a number of places within Agreement Manager, such as:

- Agreement list table on the Completed Documents page
- Agreement sets pane on the Agreement preview page

This topic shows how to remove agreements from existing sets from both the agreement list table (Completed Documents page) and the Agreement preview page.

**Remove agreements from sets from the Completed Documents page:**

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. Locate the set that contains the agreements you want to remove.
4. Apply filters to locate the relevant set:

   ![A snippet of the Completed Documents page that displays the expanded Sets filter.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-sets_entry-point_Sets-filter.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNTEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.mkabalaSdMrXqrosnohCmB-UVBsah0i59ht1WdTmxl8&_LANG=enus "A snippet of the Completed Documents page that displays the expanded Sets filter.")

   1. Select the Filters option. The menu expands and a list of applied and suggested filters is displayed.
   2. Select the Sets filter from the list of Suggested filters.
   3. Select a set from the list.
   4. Select the Apply option.

   The filter closes and the agreements contained in the set are displayed in the table.
5. Select the agreements that you want to remove from the set. You can remove one or more agreements at a time.

   When one or more agreements are selected, additional actions are displayed in the bar at the top of the table.
6. Expand the Organize dropdown and select Remove from Set:

   ![A screenshot of the expanded Organize dropdown that highlights the Remove from Set option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Sets-filter_Remove-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNTEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.mkabalaSdMrXqrosnohCmB-UVBsah0i59ht1WdTmxl8&_LANG=enus "A screenshot of the expanded Organize dropdown that highlights the Remove from Set option.")

   A dialog box opens and you are prompted to confirm that you want to remove the agreements from the set.
7. Review the message and select the Remove option.

   The dialog box closes and the selected agreements are removed from the set.

**Remove an agreement from sets from the Agreement preview page:**

8. Select an agreement from the agreement list table on the Completed Documents page.

   The Agreement preview page opens for the selected agreement. The Details pane displays a list of attributes and field values that were surfaced based on the agreement type.

   ![A snippet of the Agreement preview page that highlights the Agreement sets icon located on the left-side of the page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Preview-page_Agreement-sets-icon-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNTEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.mkabalaSdMrXqrosnohCmB-UVBsah0i59ht1WdTmxl8&_LANG=enus "A snippet of the Agreement preview page that highlights the Agreement sets icon located on the left-side of the page.")
9. Select the Agreement sets icon from the tray of icons on the left-side of the page.

   The Agreement sets pane opens and a list of sets to which the agreement has been added is displayed.

   ![A snippet of the Agreement preview page that highlights the Agreement sets pane, including available actions.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Preview-page_Agreement-sets-pane.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNTEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.mkabalaSdMrXqrosnohCmB-UVBsah0i59ht1WdTmxl8&_LANG=enus "A snippet of the Agreement preview page that highlights the Agreement sets pane, including available actions.")
10. Select the Remove (x) icon associated with the set from which you want to remove the agreement.

    A message displays stating that the agreement has been removed from the set.
11. Select the Close (x) icon from the top-left corner.

    You are returned to the Agreement Manager homepage. Your search criteria and filters are still applied.

You have successfully removed agreements from an existing agreement set.
