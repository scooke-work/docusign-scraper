# Add Examples of Your Custom Fields

Learn how to complete the second step of the AI detection set-up process, *Select examples*.

## Before you begin

This topic reviews the next step of the AI detection set-up process, *Select examples*. These steps are completed by users with Organize and manage all agreements access to Agreement Manager agreements and functionality.

Note: Administrators automatically have access to configure agreement data. For non-administrator users, this feature requires Organize and manage all agreements access to Agreement Manager. These tasks are primarily completed by administrators and contract managers.

Important: Complete the [Create Custom Fields from the Agreement Preview](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk0830ea60-9b29-4efa-b73f-9eb15b2d6561.html&_LANG=enus "Learn how to create custom fields from the Agreement preview page for a selected agreement.") topic before attempting this task. When you create a field from the Agreement preview page, the Track this field with AI wizard is launched.

When you set up custom extractions, you are prompted to provide examples, in addition to a text
definition. The definition tells the AI model what the data means and the examples
show how the data displays in your agreements.

In the previous task, you added field details and provided a definition to initiate the AI setup. When you add a field from the Agreement preview page, the first step of the AI detection set-up is completed during field configuration. After saving the configuration, the Track this field with AI wizard opened:

![A screenshot of the Track this field with AI setup wizard that displays when you create custom fields from the Agreement preview page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Custom-field-extractions_Track-this-field-with-AI_setup-wizard.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.K2tsSnRuAkJjga3n1_DKzSk9Q_syUtYAHUY0altdmdA&_LANG=enus "A screenshot of the Track this field with AI setup wizard that displays when you create custom fields from the Agreement preview page.")

After you selected the Set Up option, the Track this field with AI wizard advanced to the *Select examples* step. This topic completes the *Select examples* step of the AI detection set-up process.

1. From the previous step in this process, select the Set Up option from the Track this field with AI wizard.

   The Select examples step opens. You are prompted to select two or more agreements that contain the field. These examples help the AI model locate the data point in different contexts.

   ![A screenshot of the Select examples step that prompts you to select examples of the custom field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Fields-Manager_Agreement-preview_Select-examples.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.K2tsSnRuAkJjga3n1_DKzSk9Q_syUtYAHUY0altdmdA&_LANG=enus "A screenshot of the Select examples step that prompts you to select examples of the custom field.")

   Important: Use the Finish Later option to save your progress and finish the AI detection set-up later. Using this option ends the Track this field with AI experience. See the [Enable AI Detection for Your Custom Fields](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk02bd31a2-b125-4d53-b632-20d9972163a8.html&_LANG=enus "Learn how to enable the AI detection for your custom fields. Once activated, the field is surfaced in your new agreements automatically.") topic for steps to finish the process.
2. Select the Select Documents option.

   The page refreshes and an agreement list table is displayed. Select two or more documents that include the custom field.

**Select examples of the custom field**

3. Locate the documents that contain good examples of the custom field:

   ![A snippet of the agreement list table that shows the search field and filter options to locate examples.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Fields-Manager_Agreement-preview_Select-examples_search-filter-options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.K2tsSnRuAkJjga3n1_DKzSk9Q_syUtYAHUY0altdmdA&_LANG=enus "A snippet of the agreement list table that shows the search field and filter options to locate examples.")

   1. Search for the relevant agreements. Enter a keyword or search string in the search field. The dropdown opens and search suggestions are displayed. Select a search suggestion or press the Enter key.
   2. Filter the agreement list table to refine the results list. For example, filter by Sets, Status, Parties, Agreement Type, or AI Suggestions.

      Tip: Use the All Filters option to apply additional filters to the results table.

      Expand the filter, choose the relevant filter value, and select the Apply option.

   The agreement list table updates and the documents that matched the search criteria and applied filters are displayed. The original agreement is selected by default. This is the agreement that you were viewing in the Agreement preview when you added the field.
4. Choose at least two agreements from the agreement list table:

   ![A snippet of the agreement list table that shows a number of selected examples.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Fields-Manager_Agreement-preview_Select-examples_selected-docs.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.K2tsSnRuAkJjga3n1_DKzSk9Q_syUtYAHUY0altdmdA&_LANG=enus "A snippet of the agreement list table that shows a number of selected examples.")

   Tip: Select the Show Selected Agreements option to review your selected documents. If necessary, select the Return to Search Results option and make changes to your selections.
5. Select the Start Highlighting option at the bottom of the page.

   An agreement preview view opens and the selected documents are listed in the pane on the left. For each of the examples in the list, highlight where the field is displayed in the document.

**Highlight the field in your examples**

6. Highlight the field in your example documents and confirm the field value:

   ![An animated image of the Select examples step that highlights the field value in an example document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Fields-Manager_Agreement-preview_Select-examples_animated.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.K2tsSnRuAkJjga3n1_DKzSk9Q_syUtYAHUY0altdmdA&_LANG=enus "An animated image of the Select examples step that highlights the field value in an example document.")The first example is displayed in the Preview pane.

   1. Locate the field in the document. Scroll to find the field value or use the Search (magnifying glass) icon to perform a keyword search.
   2. Highlight the field value from the Preview pane and select the Confirm selection option from the dropdown.

      You are prompted to confirm the custom field value.
   3. Enter the selected text in the Confirm [Field Name] field.

      Note: Highlight the field value in the Preview pane and select the Copy Text option from the dropdown. Paste the text into the field.
   4. Select the Confirm option:

      The next example is displayed.
   5. Continue highlighting the field value in your documents until all the examples are confirmed.

      Tip: If necessary, use the Remove icon to remove an example from the list. For example, if the agreement does not contain the custom field.
7. Select the Next option.

   You are returned to the set-up wizard. Your examples were saved and are displayed in the Add examples of list.

   ![A screenshot of the Add examples of list that displays the examples that you provided for the custom field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Fields-Manager_Agreement-preview_Confirm-examples.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.K2tsSnRuAkJjga3n1_DKzSk9Q_syUtYAHUY0altdmdA&_LANG=enus "A screenshot of the Add examples of list that displays the examples that you provided for the custom field.")
8. Review each of the examples for accuracy:
   - Use the Edit (pencil) icon to update the
     example.
   - Use the Remove icon to delete the
     example.
   - Use the + Add Example option to add more
     examples. For example, highlight the field in additional documents or paste
     your examples.

   Note: If necessary, use the +Add Example option to add
   more examples of the custom field. You have the option of highlighting
   examples in documents or pasting examples into the list.
9. Select the Next option from the bottom of the page.

   Your examples are saved. The Track this field with AI wizard advances to the next step, *Test for accuracy*.

   Complete the [Test the Extraction Accuracy for Your Custom Fields](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=wjr1740452628309.html&_LANG=enus "Learn how to test the AI detection accuracy for your custom fields and refining the results to improve accuracy.") task to test the accuracy of the AI detection. This step is required.

You have successfully completed the second step of the AI detection set-up process, *Select examples*.
