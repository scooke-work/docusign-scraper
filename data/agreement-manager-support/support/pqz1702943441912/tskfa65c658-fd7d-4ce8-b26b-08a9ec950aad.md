# Create Custom Reports from Scratch

Learn how to create a custom report from scratch, including choosing the relevant data, metrics, and visualization options.

Customizable dashboards and reports give you control over the insights that you see -
providing access to data that's tailored for your priorities. Dashboards are
collections of standard and custom reports that show a larger picture of a particular
focus area.

Reports are individual data visualizations, such as charts and graphs.
Create custom reports to focus on the agreement data that is most relevant to your
organization or industry.

1. Log in to your **Docusign** account.

   The Docusign Home page opens.
2. Select Insights from the global header.

   The Insights hub opens and the Overview is displayed.
3. Expand the Create dropdown and select the Create Report option:

4. Complete the Data source section to choose the data that you want to use to build the report:

   ![A snippet of the Untitled report page that shows the Data source dropdown and Add filter option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Create-custom-report_Untitled-report_Data-source-section.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNzQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.zRNvOPGsqg4psYI6vuB-UNsI_RmR_fRJOe0SLuhI2xg&_LANG=enus "A snippet of the Untitled report page that shows the Data source dropdown and Add filter option.")

   1. Expand the Data source dropdown and choose the data that you want to use to build the report:
   2. Select the + Add
      filter option to refine the data. For
      example, apply filters to ensure that the
      agreement data contains custom field values.

      The All Filters dialog
      box opens. A list of Available
      Filters is displayed. This list
      includes both custom and standard fields.
   3. Locate the filter that you want to apply. If necessary, perform a keyword search to locate the filter. Enter a keyword in the Search for filters field.
   4. Expand the filter, choose a value, and select
      the Apply option. Repeat
      this step to refine the report data.
   5. Select the Save option to save the applied filters.

      The dialog box closes and the applied filters are listed in the Data Source section. The agreement count shows the number of agreements that match the applied filters in the selected data source.
5. Complete the Build your report section to choose the report metrics and grouped by details:

   ![A snippet of the Untitled report page that shows the Measure dropdown and Group by option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Create-custom-report_Untitled-report_Build-your-report-section.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNzQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.zRNvOPGsqg4psYI6vuB-UNsI_RmR_fRJOe0SLuhI2xg&_LANG=enus "A snippet of the Untitled report page that shows the Measure dropdown and Group by option.")

   1. Expand the Measure dropdown and choose an attribute to start building a report.

      Important: Currently, custom reports only support numeric values for metrics or key measures.
   2. Select the Group by option to choose a field by which to group or break down your metric.

      A dropdown opens and a list of attributes is displayed.
   3. Select the attribute or field by which you want to group the selected metric. Both custom and standard fields are available for selection. If necessary, enter a keyword in the search field to locate the relevant attribute.

      The dropdown closes and the Group by section is displayed. The Group by field reflects the selected attribute.

      The pane on the right is updated to reflect the selected metrics. For example, a bar chart might show the agreements that match the applied filters grouped by the selected Group by field value.
6. Complete the Visualization section to choose how the report is displayed:

   ![A snippet of the Untitled report page that shows the Visualization dropdown.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Create-custom-report_Untitled-report_Visualization-section.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNzQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.zRNvOPGsqg4psYI6vuB-UNsI_RmR_fRJOe0SLuhI2xg&_LANG=enus "A snippet of the Untitled report page that shows the Visualization dropdown.")

   1. Expand the Visualization dropdown.
   2. Select the relevant chart-type. Choices include:

      - Single metric (not available when the Group by option is used)
      - Bar chart
      - Pie chart

   The pane on the right shows what the report chart or graph looks like after choosing the relevant data, metrics, and visualization option.

   Continue refining the report until it reflects the agreement data that you want to capture.
7. Select the Save option from the top-right corner of the page.

   The Save report dialog box opens.![A screenshot of the Save report dialog box that shows the Report name and Description fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Create-custom-report_Save-report-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNzQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.zRNvOPGsqg4psYI6vuB-UNsI_RmR_fRJOe0SLuhI2xg&_LANG=enus "A screenshot of the Save report dialog box that shows the Report name and Description fields.")
8. Enter a name for your report in the Report name field. This field is required.
9. Enter a description about the report's purpose in the Description field.
10. Select the Save option.

    The dialog box closes and the customized report is displayed.

    Tip: Your custom reports are available from the Custom tab under the Reports view. See the [View and Update Your Custom Reports](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=yuf1727801071281.html&_LANG=enus "This topic answers the question: How can I review and update my custom reports?") topic to learn how to view your custom reports.

    The custom report can now be added to a custom dashboard.
11. From the report view, you can:

    ![A snippet of the newly created custom report that highlights the various report management options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Create-custom-report_Report-view_options-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNzQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.zRNvOPGsqg4psYI6vuB-UNsI_RmR_fRJOe0SLuhI2xg&_LANG=enus "A snippet of the newly created custom report that highlights the various report management options.")

    - Apply additional filters to the report.
    - Download the report.
    - Edit the report.
    - Save updates to the report.
    - Delete the custom report.
    - Add the report to Your Favorites list.
    - Subscribe to receive email reports notifications.

You have successfully created a custom report from scratch, including choosing relevant data, metrics, and visualization options.
