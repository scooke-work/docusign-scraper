# Edit Smart Filters for an Agreement Set

Learn how to update the smart filters for an agreement set.

Contract managers and any users who are assigned the Organize and manage all agreements permission can create and update agreement sets.

Smart filters are used to add and remove agreements to sets based on filter values. This topic reviews the steps to update the smart filters for an agreement set.

Tip: During set creation, you have the choice to add smart filters to the set or skip the step. If you skipped this step during set creation, the Add filters option is displayed in the Smart filters column for the set:![A snippet of the All agreement sets view that highlights the Add filters option in the Smart filters column.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-set-management_edit-smart-filters_Add-filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc0MTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.b7zR4puYpOJglg2n8wrGFKRHHhWadwvgvVBLpHJ3A2Q&_LANG=enus "A snippet of the All agreement sets view that highlights the Add filters option in the Smart filters column.")

Select Add filters to create a recurring rule for the set by applying smart filters.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents page is displayed.
4. Expand the Settings (gear) menu located in the top right and select the Sets option:

   ![A screenshot of the Settings (gear) menu on the Agreement Manager homepage that highlights the Sets option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-sets_entry-point_Manage-sets-action-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc0MTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.b7zR4puYpOJglg2n8wrGFKRHHhWadwvgvVBLpHJ3A2Q&_LANG=enus "A screenshot of the Settings (gear) menu on the Agreement Manager homepage that highlights the Sets option.")

   The Agreement sets page opens. The All agreement sets view is displayed by default. The table lists the sets that were either shared with or created by the logged-in user.
5. Locate the set that you want to update and choose to add or edit the set's smart filters:

   ![A snippet of the All agreement sets view that highlights the Add filters button and shows the Edit details option in the More actions menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-set-management_edit-smart-filters-or-Add-filters-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc0MTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.b7zR4puYpOJglg2n8wrGFKRHHhWadwvgvVBLpHJ3A2Q&_LANG=enus "A snippet of the All agreement sets view that highlights the Add filters button and shows the Edit details option in the More actions menu.")

   - Expand the More actions (three dots) menu for the set that you want to update and select Edit smart filters.
   - Select the Add filters option for the set to which you want to add smart filters.

   The Edit smart filters page opens. The agreements that match the existing smart filters are displayed in the table.

   Tip: Select the Clear smart filters option in the top-right corner to clear existing filters and reset the page.

   ![A screenshot of the Edit smart filters page that highlights the smart filter options and the Save changes button.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-set-management_Edit-smart-filters-page.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc0MTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.b7zR4puYpOJglg2n8wrGFKRHHhWadwvgvVBLpHJ3A2Q&_LANG=enus "A screenshot of the Edit smart filters page that highlights the smart filter options and the Save changes button.")

   Agreements are added to the set if they match the fields specified in your filters. Agreements are selected if they match conditions across all filters. If you selected multiple values for a filter, the agreement must match only one of the selected values to be added in the set.
6. Update the existing filters or apply new smart filters to the agreement set:
   1. Expand the filter that you want to update or add to the set.

      Tip: Select the Filters option to access additional filters, including custom fields that are enabled for AI detection. Locate the filter that you want to apply from the Available section:![A snippet of the Filters dialog box that shows the search field and a list of available filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-set-management_create-new-set_step-2All-Filters-snippet.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc0MTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.b7zR4puYpOJglg2n8wrGFKRHHhWadwvgvVBLpHJ3A2Q&_LANG=enus "A snippet of the Filters dialog box that shows the search field and a list of available filters.")
   2. Select or enter a value for the filter.
   3. Select the Apply option.

      The agreements that match the filters that you applied are listed in the table.

      Tip: Use the Show or Hide Fields (toggle) icon to customize the table. For example, display the columns that are associated with the applied filters to verify the appropriateness of the agreements for the set.
   4. Continue applying filters until the list is representative of the set's purpose.
7. Save your changes before leaving the page. Select the Save changes option.

   The Save changes? dialog box opens.
8. Select Save changes.

   The dialog box closes. Agreements will be added and removed from the set based on the applied smart filters.

   You are returned to the Agreement sets page. The updated filters are displayed in the Smart filters column for the set.
9. Return to the Agreement Manager homepage. Select Completed from the breadcrumb navigation at the top of the page.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.

You have successfully updated the smart filters for a selected agreement set.
