# Filter the Agreement List Table to Locate Documents

Learn how to use the available filters to refine the agreement list table to locate agreements. This topic answers the question: How can I find my documents in Agreement Manager?

Instead of performing a keyword search, filter the agreement list table to locate documents. Use the Filters option to locate and apply filters to the table:

![A snippet of the Completed Documents view that highlights the Filters option, including suggested filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-view_Available-filter-pills.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NjQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.4_xghyYiJtYK1HZsk4Vk-El9C2g35fY8CUa8B9lXQSw&_LANG=enus "A snippet of the Completed Documents view that highlights the Filters option, including suggested filters.")

This topic begins from the Agreement Manager homepage. The Completed Documents view is displayed.

Remember: The Completed Documents view is located under the Agreements tab in the global header:

- Log in to your Docusign account and select the
  Agreements tab from the global header.
- Select Completed from the AGREEMENTS section in the navigation pane.

The Agreement Manager homepage opens.

A list of documents to which the logged-in user has access is displayed in the table. The total document count is displayed in the bottom-right corner of the view. This count is updated when you perform searches or apply filters.

**Filter by Sets**

1. Expand (click) the Filters option and select the Sets filter.

   The Sets filter opens and a list of sets is displayed. These sets are either created by or shared with the logged-in user.

   ![A screenshot of the Sets filter that shows the Search options and the agreement set list.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Set-filter_expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NjQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.4_xghyYiJtYK1HZsk4Vk-El9C2g35fY8CUa8B9lXQSw&_LANG=enus "A screenshot of the Sets filter that shows the Search options and the agreement set list.")

   1. Update the search operator as needed. The equals to (=) operator is selected by default. Other options are Does not exist, Exists, and Does not exist.
   2. Enter a keyword or search string in the Search field.

      As you enter search criteria, sets that match the entered keyword are displayed in the list.
   3. Select the set that contains the agreements that you want to view.
   4. Select the Apply option.

      The filter closes and the agreements that matched the applied filter value are displayed in the table.
   5. (Optional) Reset the filter. Select the Delete (x) icon in the Sets filter. The filter is cleared and the table is reset.

**Filter by Expiration Date**

2. Expand (select) the Filters option and search for the Expiration Date filter. Select the Expiration Date filter from the results list.

   The Expiration Date filter opens and a list of date ranges is displayed, including the Custom Date Range option.

   ![A screenshot of the Expiration Date filter that shows the available date range options, including the Custom Date Range option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Expiration-Date-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NjQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.4_xghyYiJtYK1HZsk4Vk-El9C2g35fY8CUa8B9lXQSw&_LANG=enus "A screenshot of the Expiration Date filter that shows the available date range options, including the Custom Date Range option.")

   1. Update the search operator as needed. The equals to (=) operator is selected by default. Other options are: Exists and Does not exist.
   2. Select the relevant date range from the list of options. Choices include: Today, Next 24 Hours, Next 7 Days, Next 14 Days, and Next 30 Days.

      Note: Select Custom Date Range to search for expiring agreements over a specific time period.

      The dialog box updates to show the From and To fields, as well as Between, Before, and After options.

      Select an option from the top of the dialog box and enter a date or date range in the From and To fields.
   3. Select the Apply option.

      The filter closes and the agreements that matched the applied date range are displayed in the table.
   4. (Optional) Reset the filter. Select the Delete (x) icon in the Expiration Date filter. The filter is cleared and the table is reset.

**Filter by Status**

3. Expand (click) the Filters option and select the Status filter.

   The Status filter opens and the available statuses are displayed.

   ![A screenshot of the Status filter that shows the available filter options, such as Active and Inactive.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-statuses_Status-filter_expanded_no-selections.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NjQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.4_xghyYiJtYK1HZsk4Vk-El9C2g35fY8CUa8B9lXQSw&_LANG=enus "A screenshot of the Status filter that shows the available filter options, such as Active and Inactive.")

   1. Select the relevant statuses. For example, choose to view agreements that are effective, expiring, or renewing soon.
   2. Select the Apply option.

      The filter closes and the agreements that matched the applied statuses are displayed in the table.
   3. (Optional) Reset the filter. Select the Delete (x) icon in the Status filter. The filter is cleared and the table is reset.

   See the [Agreement Statuses](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=con6a3bd080-da17-4d1c-a92a-b9b9340c44fd.html&_LANG=enus "Read about agreement statuses, including how they are calculated, visibility within Agreement Manager, and which types of statuses are being tracked.") topic to learn more about agreement statuses and the Status filter.

**Filter for Parties**

4. Expand (click) the Filters option and select the Parties filter.

   The Parties filter opens and a list of parties is displayed.

   ![A screenshot of the Parties filter that shows the Search field, Party Role dropdown, and the party list.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Parties-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NjQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.4_xghyYiJtYK1HZsk4Vk-El9C2g35fY8CUa8B9lXQSw&_LANG=enus "A screenshot of the Parties filter that shows the Search field, Party Role dropdown, and the party list.")

   1. Update the search operator as needed. The = ANY
      operator is selected by default. This operator means that if multiple
      parties are selected there is an *OR* matching requirement.
      Documents must contain at least one of the selected parties to be
      included in the table.

      Tip: Select the = ALL option to
      locate documents that contain all the selected parties. For example,
      to view agreements between two specific parties.
   2. Locate the relevant parties. Enter a keyword or search string in the Search field.

      As you enter search criteria, the list of parties updates to match the keyword.
   3. Select the relevant party or parties.

      Note: When multiple values are selected within the same filter, there is either an *OR* or an *AND* relationship depending on the selected search operator.

      Important: When using the Parties filter, you must also select a
      Party Role. If you do not know
      which role to select, accept the default value, All.
   4. Apply the Party Role filter to view agreements where the selected party plays a specific role. For example, to view your buy-side or sell-side agreements with a specific party.

      Select a role from the Party Role dropdown. Choose from: Buyer, Seller, and Other.

      Tip: If you do not know which role to select, accept the default value, All. See the [Filter for Agreements by Specific Party Role](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk6f2903c9-7ae7-4ad5-8acd-1c4948c98d10.html&_LANG=enus "Learn how to update the agreement list table to view agreements based on the role played by a specific party.") topic to learn more about using this filter.
   5. Select the Apply option.

      The agreements that match the applied filter are displayed in the agreement list table.
   6. (Optional) Reset the filter. Select the Delete (x) icon in the Parties filter. The filter is cleared and the table is reset.

**Filter by Agreement Type**

5. Expand (click) the Filters option and select the Agreement Type filter.

   The Agreement Type filter opens and a list of available agreement types is displayed.

   ![A screenshot of the Agreement Type filter that shows the field and the agreement type list.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Document-Type-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NjQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.4_xghyYiJtYK1HZsk4Vk-El9C2g35fY8CUa8B9lXQSw&_LANG=enus "A screenshot of the Agreement Type filter that shows the field and the agreement type list.")

   1. Locate the relevant agreement types. Enter a keyword or search string in the Search field.

      As you enter the search criteria, the agreement types that match the keyword are displayed.
   2. Select the agreement types from the list.

      Note: Select multiple options to include multiple agreement types. Multiple values within the same filter imply an *OR* relationship.
   3. Clear the search criteria. Select the Clear Search (x) icon in the Search field. The field is cleared and the filter is reset. The agreement types that you selected previously remain selected.
   4. Repeat these steps to select additional agreement types.
   5. Select the Apply option.

      The table updates and the agreements that match the applied filter are displayed.
   6. (Optional) Reset the filter. Select the Delete (x) icon in the Agreement Type filter. The filter is cleared and the table is reset.

**Additional filters: View All Filters**

6. Expand (click) the Filters option and select the View All Filters option at the bottom of the dropdown menu.

   The Filters dialog box opens.

   ![A screenshot of the Filters dialog box that shows the Applied and Available filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_All-Filters-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NjQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.4_xghyYiJtYK1HZsk4Vk-El9C2g35fY8CUa8B9lXQSw&_LANG=enus "A screenshot of the Filters dialog box that shows the Applied and Available filters.")
7. Scroll through the list to locate and apply additional filters to the agreement list table:
   1. Expand the filter that you want to apply.
   2. Enter or select filter criteria.
   3. Select the Apply option.

      The filter is highlighted and the selected criteria is displayed.
   4. Repeat this step to apply additional filters.
8. Select the Save option.

   The Filters dialog box closes. The agreements that match the applied filters are displayed in the table. The document count is updated based on your filters.

   Note: Any filters that were applied from the Filters dialog box are displayed at the top of the table. Expand these filters and select the Clear option to remove the applied filter.

You have successfully filtered the agreement list table to locate documents.
