# Add Fields to Multiple Agreement Types

Learn how to add fields to multiple agreement types from the Add a field dialog box. You can add fields to additional agreement types while you are updating a selected agreement type.

Administrators and contract managers can add fields to a selected agreement type. This includes both custom and standard fields. When completing the Add a field dialog box, the Agreement types field supports multiple selections.

The Add a field function is accessed from these locations:

- Agreement Type Manager
- Fields Manager
- Agreement Preview page

Note:

This task is completed by users with Organize and manage all agreements access to Agreement Manager and the account's agreements.

This topic reviews the steps to add fields to multiple agreement types from the *Agreement Types Manager*. Access this feature from the Agreement Manager homepage, under the Settings (gear) menu.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Completed Documents view opens. A list of documents to which the logged-in user has access is displayed in the table.
4. Expand the Settings (gear) dropdown from the top right and select the Agreement Types option:

   ![A screenshot of the Agreement Manager homepage that highlights the Agreement Types option in the Settings (gear) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-types-manager_Settings-menu_Agreement-Types_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1NTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.ZcSvHc8ssGUrnmaHwyz3v7r0o_vh6EZdkAfDBX5rtZo&_LANG=enus "A screenshot of the Agreement Manager homepage that highlights the Agreement Types option in the Settings (gear) menu.")

   The Manage Agreement Types page opens. The account's standard and custom agreement types are displayed in the Agreement Types table.
5. Locate the agreement type that you want to update. You can add fields to custom and standard agreement types.

   ![A snippet of the Manage Agreement Types page that shows the search and filter options for locating agreement types.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Types_Agreement-Types_search-filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1NTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.ZcSvHc8ssGUrnmaHwyz3v7r0o_vh6EZdkAfDBX5rtZo&_LANG=enus "A snippet of the Manage Agreement Types page that shows the search and filter options for locating agreement types.")

   - Search for the relevant agreement type. Enter a keyword in the Search types... field. As you enter the search term, the table updates and agreement types that matched the entered criteria are displayed.
   - Filter the table for the relevant agreement type. Filter options are: Created By, Category, and AI Status. Expand the filter, choose the relevant value, and select the Apply option.The table updates and the agreement types that matched the applied filters are displayed.
6. Select (click) the relevant agreement type from the table.

   The Agreement type details view opens for the selected agreement type. The Fields table displays the fields that are associated with the agreement type.
7. Select the + Add Fields option.

   The Add a field dialog box opens.

   ![A screenshot of the Add a field dialog box that is used to add a field to existing agreement types.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-type-management_create-type_add-a-field-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1NTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.ZcSvHc8ssGUrnmaHwyz3v7r0o_vh6EZdkAfDBX5rtZo&_LANG=enus "A screenshot of the Add a field dialog box that is used to add a field to existing agreement types.")The selected agreement type is displayed in the Agreement types field by default.
8. Complete the required fields and select the Save option.
   1. Enter the name of the field in the
      Field name field.

      As you enter a name, a type-ahead dropdown
      list opens, and fields that match the entered name
      are displayed.
   2. Select the name from the type-ahead dropdown
      list or select the + Create [Field
      name] option to add a custom field to
      the agreement type.

      For existing fields, the Field
      type and
      Category fields default
      based on the selected field.
   3. For custom fields, complete the
      Field type and
      Category fields:

      - Expand the Field type dropdown and select the type of field that you are adding. Choices are: Text, Number, Date, and Dropdown.

        Important: If you
        select the Dropdown field type, you must provide a list of options for
        the dropdown. Enter a list of options for the
        dropdown in the Options
        text-entry box:![A snippet of the Add a field dialog box that shows the Options field for adding dropdown options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Add-custom-field_Options-for-dropdown-type.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1NTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.ZcSvHc8ssGUrnmaHwyz3v7r0o_vh6EZdkAfDBX5rtZo&_LANG=enus "A snippet of the Add a field dialog box that shows the Options field for adding dropdown options.")**Important:** Separate each option with
        a semicolon.
      - Expand the Category dropdown and select a classification for the new custom field:

        Choices are: General, Termination, Renewal, Payment, Legal and Compliance, Human Resources, and Other.
   4. Add the field to multiple agreement types.

      ![A screenshot of the Agreement types dropdown that shows two selected agreement types to which the field will be added.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-agreement-types-fields_Add-a-field_Agreement-types-dropdown.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1NTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.ZcSvHc8ssGUrnmaHwyz3v7r0o_vh6EZdkAfDBX5rtZo&_LANG=enus "A screenshot of the Agreement types dropdown that shows two selected agreement types to which the field will be added.")

      - Click in the Agreement types field to expand the dropdown.
      - Enter search criteria or scroll to locate and select the relevant agreement types. You can select more than one agreement type.
      - Select the Apply option.The dropdown closes and the selected types are displayed in the Agreement types field.
   5. Select the Save option.

      The dialog box closes and you are returned to the
      Agreement type details view.
      The new field
      is displayed in the table.
9. Return to the Agreement Types view. Select the Agreement Types  option from the breadcrumb navigation in the top left.

   You are returned to the Manage Agreement Types page. The Agreement Types view is displayed.
10. Return to the Agreement Manager homepage. Select the Close (x)
    icon from the top right.

    The Agreement Manager homepage opens and the Completed Documents view is displayed.

You have successfully added a field to multiple agreement types at the same time, including both custom and standard types.
