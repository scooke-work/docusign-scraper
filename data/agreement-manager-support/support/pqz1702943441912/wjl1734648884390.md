# Manage Notification Subscriptions

Learn how to manage your report notification subscriptions.

Manage your subscriptions to help tailor your report notifications to fit your needs and preferences. For example, you may want to adjust the frequency of the email notifications or unsubscribe to them. This is done from the detailed report view for the report to which you are subscribed.

The Manage report subscriptions dialog box displays when you have existing subscriptions. This includes the subscriptions to which you were added as an individual user. Currently, subscriptions are not visible to users who were added as part of a group.

Tip: Notification settings give you control over the email notifications that you receive. By default, all the notification types related to the completed agreements in your account are selected, or enabled. Deselect a checkbox if you no longer want to receive email notifications for the corresponding notification type.

See the [Customize Notification Settings for Your Completed Agreements](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk155b1ed7-7b45-43cd-be92-2a5279630c0f.html&_LANG=enus "Learn how to customize your email notification settings from the Notification Preferences page.") article to learn more about updating your notification settings. Be sure to periodically check the Notification Preferences page as new notification types will be added in the future.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Insights from the global header.

   The Insights hub opens and the Overview is displayed.
3. Select the dashboard containing the subscriptions that you want to manage. Select either Agreements, Obligations, or Renewals from the navigation pane on the left.

   The selected dashboard opens.
4. Access the detailed report view for the chart with the subscription that you want to manage. Select (click) the chart heading or select the View Report option from the More actions (ellipsis) menu.

   Note: To manage a custom report subscription, locate and select the report from the Custom tab located under the Reports view. See the [View and Update Your Custom Reports](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=yuf1727801071281.html&_LANG=enus "This topic answers the question: How can I review and update my custom reports?") article to learn more about locating your custom reports.
5. Select the Manage Notification (bell) icon from the top-right corner.

   The Manage report subscriptions dialog box opens.

   Note: When you create a subscription, you have the option to add individual users and groups. Once created, the subscription is displayed in the Manage report subscriptions dialog box for any individual users who were added. Currently, the subscription is not visible to users if they were added as part of a group.
6. Expand the More actions (three dots) menu associated with the subscription that you want to update and select the action you want to perform:
   - Select Edit to update the subscription.

     The Edit subscription dialog box opens. Update the subscription as needed, including changing the name, frequency, start and end dates, and recipients. Select Save to save changes to the subscription.
   - Select Delete to delete the subscription.

     The Delete subscription dialog box opens. Select Delete to confirm that you want to unsubscribe to the report notification. This action cannot be undone.

You have successfully managed the notification subscription for a dashboard report, including how to edit and delete a subscription.
