# Use Saved Searches to Find Agreement Data

Learn how to access and run a saved search, including steps to edit and delete searches.

Run your saved searches to retrieve an updated list of documents that match the search criteria. When you run the search, the search criteria (keywords, filters, custom columns) are applied to the repository. Since new agreements are regularly added to Agreement Manager, the search results can change over time.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   TheAgreement Manager homepage opens and the Completed Documents view is displayed.

   A list of documents to which the logged-in user has access is displayed in the table. The total document count is displayed the bottom right. This count is updated when you perform a search or apply filters.
4. Run a saved search from a list of Your Recently Saved searches.

   ![A snippet of the Agreement Manager homepage that highlights the Open Saved Search (bookmark) icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Saved-searches_Access-searches_Open-Saved-Search.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NDksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.j7cv5SmqmdTjxZy9tHNbG323QCK9MYGiujaoYS76HJo&_LANG=enus "A snippet of the Agreement Manager homepage that highlights the Open Saved Search (bookmark) icon.")

   1. Select the Open Saved Search (bookmark) icon.

      A dropdown opens and Your Recently Saved searches are displayed.
   2. Select the relevant search from the list of recently saved searches.

   The search is run against the agreement list table and the agreements that matched the search criteria are displayed. The search name and description are displayed at the top of the page.

   ![A screenshot of the results table from the saved search that was run against the agreement list table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Saved-searches_search-results_distribution-agreements.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NDksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.j7cv5SmqmdTjxZy9tHNbG323QCK9MYGiujaoYS76HJo&_LANG=enus "A screenshot of the results table from the saved search that was run against the agreement list table.")

   There are a number of actions that you can take from the search results page, such as:
   - [Download Documents from the Agreement List Table](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=ach1724363496037.html&_LANG=enus "This topic answers the question: How do I download agreements from Agreement Manager?")
   - [Export Data from the Agreement List Table](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=pqf1724363438241.html&_LANG=enus "Learn how to export agreement data to a CSV file for selected agreements.")
5. Return to the Agreement Manager homepage.
   - Select the Completed Documents option from the breadcrumb navigation at the top of the page.
   - Select the Back to Completed option located to the right of the filters.

   You are returned to the Agreement Manager homepage and the Completed Documents view is displayed. Next, access your saved searches from the search field.
6. Click in the search field and select the Saved Searches option.

   ![A snippet of the Completed Documents page that highlights the Saved Searches option in the search dropdown.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Saved-searches_Access-searches_Search-dropdown.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NDksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.j7cv5SmqmdTjxZy9tHNbG323QCK9MYGiujaoYS76HJo&_LANG=enus "A snippet of the Completed Documents page that highlights the Saved Searches option in the search dropdown.")

   The Saved Searches dialog box opens and a list of your searches is displayed.

   Tip: Every saved search has a More actions (three dots) menu. Use this menu to:![A snippet of the Saved Searches dialog box that shows the More actions menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Saved-Searches-dialog-box_Manage-search-menu_expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NDksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.j7cv5SmqmdTjxZy9tHNbG323QCK9MYGiujaoYS76HJo&_LANG=enus "A snippet of the Saved Searches dialog box that shows the More actions menu.")
   - Save a new version of the search with the updated search criteria and column customizations
   - Edit the name and description of the search
   - Delete the search
7. Select (click) the search that you want to run.

   The agreements that matched the selected search are displayed in the table. The search name and description are displayed at the top of the page.
8. Adjust the search criteria for the saved search as needed.
   1. Enter a new keyword or phrase in the search field.
   2. Update the applied filters. Select an existing filter and choose new values or clear the existing selection.
   3. Use the Filters option to apply additional filters to the search. Select the relevant filter, choose the filter values, and select the Apply option.
   4. Use the Show or Hide Fields (toggle) icon to update the columns in the agreement list table. Select toggles to turn the corresponding field on (blue) or off (white). Drag the fields to re-order the columns as needed.

   Be sure to save your unsaved changes.
9. Expand the Save dropdown and choose the relevant option:

   ![A snippet of a saved search that highlights the Save options, including Save Changes and Restore Original.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Saved-search_Update-search-criteria_Save-dropdown_expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NDksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.j7cv5SmqmdTjxZy9tHNbG323QCK9MYGiujaoYS76HJo&_LANG=enus "A snippet of a saved search that highlights the Save options, including Save Changes and Restore Original.")

   - Select the Save Changes option to save
     the changes to the current search.
   - Select the Save as New Version option
     to create a new saved search with the updated search
     criteria.
   - Select the Restore Original option to
     revert to the original version of the saved search.

   The results table updates and agreements that match the updated search
   criteria are displayed.
10. Update the name and description for the search to reflect the updated search criteria.

    ![A screenshot of the Edit name and description dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Saved-search_Edit-name-and-description-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NDksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.j7cv5SmqmdTjxZy9tHNbG323QCK9MYGiujaoYS76HJo&_LANG=enus "A screenshot of the Edit name and description dialog box.")

    1. Expand (select) the More actions (three dots) menu in the top right.
    2. Select the Edit Name and Description option.

       The Edit name and description dialog box opens.
    3. Update the Name and Description of the saved search to reflect the changes to the search criteria.
    4. Select the Save option.

    The dialog box closes and the search results page is updated to reflect the new name and description.
11. Delete the search if it is no longer relevant.

    ![A screenshot of the Delete this saved search? dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Saved-search_Delete-this-saved-search-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NDksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.j7cv5SmqmdTjxZy9tHNbG323QCK9MYGiujaoYS76HJo&_LANG=enus "A screenshot of the Delete this saved search? dialog box.")

    1. Expand (select) the More actions (three dots) menu in the top right.
    2. Select the Delete option.

       The Delete this saved search? dialog box opens. A message confirms that removing the search will not delete or void any agreements.
    3. Select the Delete option.

    The dialog box closes. A message confirms that the saved search was deleted. You are returned to the Completed Documents view. The search no longer displays in your saved search list.

You have successfully used a saved search to locate agreement data, including a review of additional search actions.
