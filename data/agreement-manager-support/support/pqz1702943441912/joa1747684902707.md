# View the Account-Level Activity Log

Learn how to view the account-level Activity Log that records and logs events related to all the agreements in an account.

The account-level Activity Log captures and logs events related to all the agreements in an account. See the [Agreement Manager Activity Logs](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=pne1747684858805.html&_LANG=enus "Read about the account- and agreement-level activity logs, including the types of events that are captured in the log.") support topic to learn more about the events captured in this audit log.

Note: Access to the account-level Activity Log is dependent on your assigned Agreement Manager permission. Only users with Organize and manage all agreements permission can view this audit log.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Expand the Settings (gear) menu from the right-side of the page and select the View Activity Log option:

   ![A snippet of the Completed Documents view that shows the View Activity Log highlighted in the Settings (gear) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Account-audit_Actions_View-Activity-Log.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5MzMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Tt_brBM6awj5qIsr8RJ9eFnxuh3Yw0RqC05V1KL8P5o&_LANG=enus "A snippet of the Completed Documents view that shows the View Activity Log highlighted in the Settings (gear) menu.")

   The Activity Log opens and a list of account-level events are displayed. These events are displayed in reverse chronological order with the most recent at the top.

   ![A screenshot of the account-level Activity Log that logs events related to all the agreements in the account.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Accountlevel-Activity-Log--example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5MzMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Tt_brBM6awj5qIsr8RJ9eFnxuh3Yw0RqC05V1KL8P5o&_LANG=enus "A screenshot of the account-level Activity Log that logs events related to all the agreements in the account.")

   Note: The first 200 logged events are displayed in the table. Pagination is only available for the first 200 audit events. If the table contains more than 200 events, you are prompted to download the CSV:![A screenshot of the dialog box that prompts you to download the CSV to view all the events.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Activity-Log_Prompt-to-download-to-csv.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5MzMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Tt_brBM6awj5qIsr8RJ9eFnxuh3Yw0RqC05V1KL8P5o&_LANG=enus "A screenshot of the dialog box that prompts you to download the CSV to view all the events.")
5. Select the sorting arrows in the column headers to sort the table by DATE, RESOURCE, USER ACTION, and FIELDS.
6. Review the table data to locate the relevant event information for all the agreements in the account. Details related to these user-performed events are recorded in the log:

   - Access changes (agreements and agreement sets)
   - Agreement downloads
   - Agreement set operations
   - Attribute updates for select standard fields
   - Agreement views
   - Document uploads
7. Select the Download CSV option.

   The CSV file is downloaded to your local Downloads folder.
8. Return to the Agreement Manager homepage. Select the Close (x) icon from the top-right corner.

   You are returned to the Agreement Manager homepage. The Completed Documents view is displayed.
