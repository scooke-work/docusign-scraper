# Display the "Sets" Column in the Agreement List Table

Learn how to display the Sets column in the agreement list table on the Completed Documents page.

Use the Show or Hide Fields feature to display the Sets column in the agreement list table. The Sets column displays the sets to which the corresponding document was added.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Select the Show or Hide Fields (toggle) icon located to the right of the filters:

   ![A snippet of the Completed Documents page that highlights the Show or Hide Fields (toggle) icon on the right-side of the page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Customize-columns-gear-icon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.jJT8KX43Q7qm0cKvVJnorfV-oZZuV1aZG3k9_NiYrIo&_LANG=enus "A snippet of the Completed Documents page that highlights the Show or Hide Fields (toggle) icon on the right-side of the page.")

   The Show or Hide Fields dialog box opens. A number of fields are turned on (blue) by default. These fields represent the columns in the agreement list table. Columns display in the order that the fields are represented in the dialog box.

   Important: The Show or Hide Fields function only applies to the data that is displayed in the agreement list table. When downloading or exporting the report, all fields are included in the output file. This includes both custom and standard fields.
5. Perform a search query to locate the Sets column and select the toggle to turn it on (blue position):

   ![A screenshot of the Show or Hide Fields dialog box that shows the Sets toggle.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-set-management_Customize-columns_Sets-toggle.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.jJT8KX43Q7qm0cKvVJnorfV-oZZuV1aZG3k9_NiYrIo&_LANG=enus "A screenshot of the Show or Hide Fields dialog box that shows the Sets toggle.")
6. Select the Save option.

   The dialog box closes and the Sets column is displayed in the agreement list table.

   Tip: If necessary, re-order the columns from the Show or Hide Fields dialog box. Hover your cursor over the Reorder (dots) icon for the filter you want to move and drag it to the appropriate location. For example, drag the Sets column to the appropriate location and select the Save option.

You have successfully displayed the Sets column in the agreement list table on the Completed Documents page.
