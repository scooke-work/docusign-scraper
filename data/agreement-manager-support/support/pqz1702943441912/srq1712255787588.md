# View Upload Results and Exceptions

Learn how to view processing results and details for an upload job.

The Monitor All Jobs function gives you access to job details for agreement uploads and processing jobs. From the Completed Documents page, expand the Settings (gear) menu located in the top-right corner to access this function.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Completed Documents page opens. A list of documents to which the logged-in user has access is displayed in the table.
4. Expand the More actions menu located in the top-right corner of the view and select the Uploads and Processing Status option:

   ![A screenshot of the Completed Documents view that shows the Uploads and Processing Status option highlighted in the Settings (gear) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-view_Actions_Monitor-All-Jobs-option.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4MzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.jcPfRp4eBxfNRrTMBa4kjPkMRHINDLMhFL9raRhRj4Y&_LANG=enus "A screenshot of the Completed Documents view that shows the Uploads and Processing Status option highlighted in the Settings (gear) menu.")

   The job monitoring hub opens. The All jobs view has two tabs: Uploaded and Processing. The Uploaded tab displays job details for all agreement upload jobs.

   ![A screenshot of the Uploaded Agreements table that shows job details for all upload and sync jobs.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-view_Monitor-All-Jobs_All-JobsUploaded-tab.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4MzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.jcPfRp4eBxfNRrTMBa4kjPkMRHINDLMhFL9raRhRj4Y&_LANG=enus "A screenshot of the Uploaded Agreements table that shows job details for all upload and sync jobs.")

   The created date, job name, number of documents, and status details are displayed in the table for uploaded and synced agreements.
5. Use the filters to locate your upload job:

   ![A snippet of the Uploaded Agreements table that highlights the Created date, Status, and Source filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Monitor-uploads_All-jobs_Filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4MzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.jcPfRp4eBxfNRrTMBa4kjPkMRHINDLMhFL9raRhRj4Y&_LANG=enus "A snippet of the Uploaded Agreements table that highlights the Created date, Status, and Source filters.")

   - Expand the Created date filter, select an identifier tab, and complete the appropriate date fields. Select Apply.

     Identifier tab choices are: Between, Before, and After.

     Tip: If you choose Between, you must complete the From and To date fields.
   - Expand the Status filter, choose the preferred status, and select Apply..

     Choices are: Deleted, Upload job complete, Upload job failed, and Uploading documents.
   - Expand the Source filter, choose the relevant source, and select Apply.

     Choices are: Box, Dropbox, Email Ingestion, Google Drive, OneDrive, SharePoint, and Unknown.

   Upload jobs that match the applied filters are displayed in the table.
6. Review processing results, including the number of files successfully uploaded and the number of failures.
7. Expand the Actions dropdown associated with your upload job and choose from the available actions: Browse Files, View Details, and Delete.

   ![A snippet of the All jobs view that shows the additional actions that are available for each job.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-view_Monitor-All-Jobs_Uploaded-agreements_Actions-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4MzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.jcPfRp4eBxfNRrTMBa4kjPkMRHINDLMhFL9raRhRj4Y&_LANG=enus "A snippet of the All jobs view that shows the additional actions that are available for each job.")
8. Select Browse Files from the Actions dropdown.

   A list of agreements that were uploaded as a result of the selected job are displayed in the Search Results table. The table displays the attributes and field values that were surfaced for each file.![A snippet of the Completed Documents view that shows the agreements that were successfully added to Agreement Manager for the selected job.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Monitor-All-Jobs_Actions_Browse-Files_Agreement-list.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4MzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.jcPfRp4eBxfNRrTMBa4kjPkMRHINDLMhFL9raRhRj4Y&_LANG=enus "A snippet of the Completed Documents view that shows the agreements that were successfully added to Agreement Manager for the selected job.")

   Select the browser's Back arrow to return to the Uploaded Agreements table.
9. Select View Details from the Actions dropdown.

   The Job status view opens and processing results are displayed.

   ![A screenshot of the Job status view that shows the processing results for a selected upload job.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Monitor-All-Jobs_Actions_View-Details_Upload-status.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4MzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.jcPfRp4eBxfNRrTMBa4kjPkMRHINDLMhFL9raRhRj4Y&_LANG=enus "A screenshot of the Job status view that shows the processing results for a selected upload job.")

   1. Select Manage Files to view the files that were successfully uploaded to Agreement Manager.
   2. Select Download CSV to download the document name and error message for the files that could not be uploaded. The file downloads to your local Downloads folder.
   3. View a table of users and groups that have access to the uploaded files from the Shared access section.
   4. View a list of agreement sets to which that the uploaded files were added from the Added to agreement sets section.
   5. Select the Back arrow in the top-left corner to return to the Uploaded Agreements table.
10. Select Delete from the Actions dropdown.

    The Delete upload details dialog box opens.

    ![A screenshot of the Delete upload details dialog box that explains the implications of deleting the upload details for the selected job.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Monitor-All-Jobs_Actions_Delete_dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4MzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.jcPfRp4eBxfNRrTMBa4kjPkMRHINDLMhFL9raRhRj4Y&_LANG=enus "A screenshot of the Delete upload details dialog box that explains the implications of deleting the upload details for the selected job.")

    1. Review the message and note that you will not be able to recover any of these details later.
    2. Select Delete.

    The upload details are removed from the Uploaded Agreements table. The uploaded files are not deleted, but you cannot recover any of the upload details.

You have reviewed upload results and processing exceptions for a selected upload job.
