# Language and Regional Support

Read about language support and regional availability for Agreement Manager AI models and features.

Agreement Manager is available to customers globally, however, the features and insights vary by region and language. This page provides an overview of the supported agreement types, attributes, and data points that provide insights into your agreements. Additional information can be found in the [General Limitations and Considerations for Agreement Manager](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=jyn1704248204281.html&_LANG=enus "This topic answers the question: Where can I see a list of the Agreement Manager limitations and considerations associated with this release?") article.

CAUTION: The accuracy and appropriateness of AI-assisted insights may vary depending on the agreement language and customer use case. Customers should review all suggested data points for accuracy and to determine if a value is appropriate for its particular circumstances.

Important:

As new or updated AI models are released, your historical agreements can be processed again to surface these data points. Depending on the type of new insights that are available, users can choose to:

- Opt in for reprocessing from the New Insights Available badge (obligation-related attributes and insights)
- Reprocess all the outdated agreements for the account (new or updated AI model)
- Reprocess selected agreements (standard or custom fields)

See the [Agreement Manager Reprocessing Experience](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=con44590246-0f0e-43b4-9692-caadb57335ac.html&_LANG=enus "Learn about the Agreement Manager reprocessing experience, including its purpose, trigger points, and job size considerations.") article to learn more about this experience.

## Limitation: "Dropdown" attributes (fields) with pre-populated value sets

For attributes (fields) that use dropdowns with pre-populated value sets, only those values are surfaced in your agreements. Agreement processing extracts an attribute value only if it matches one of the values in the dropdown.

When processing cannot locate the value within the agreement, the extracted value is *Other*. For example, if an agreement's payment term is 50 days, the surfaced Payment Term value is *Other*:

## Language detection and processing

Organizations that operate globally often have agreements in both English and their local language. English is used for international and local business contracts, while the local language is used for domestic agreements. Many of these agreements are available in multiple languages.

During processing, Languages are surfaced based on the data in your agreements.

![A snippet of the Agreement preview page that highlights the LANGUAGES field in the Details pane.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-preview-page_Details-pane_Langauges-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTgxMDEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.o1pGerNKS2kYHbo0ZysjzF3TgYv1W4whmPBXj-Xx6Ak&_LANG=enus)

Note: Be sure to validate the suggested Languages in your agreements to ensure accurate reporting and analysis. See the [Validate Extracted Field Values (Complete a Data Review)](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=yqq1702945693043.html&_LANG=enus "This topic answers the question: How can I make sure that the suggested attribute (field) values are accurate? Learn how to complete a data review to verify suggested field values.") article to learn how to perform a data review.

Language detection looks for supported languages within your agreements:

- If a supported language is detected in a significant portion of an agreement, the agreement is processed with AI. The detected language and other supported attributes are surfaced in the agreement.
- If a supported language is missing or insignificant, the agreement is not processed for that language. The agreement is still processed with AI to surface other supported attributes.

See the [Supported attributes (fields)](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=xvp1729795957126.html&anchor=xvp1729795957126__section_zwp_25y_x2c) table to learn about the supported languages.

Important: All detected languages are surfaced and displayed in the Details  pane for the Languages  attribute (field). Agreements are only processed for supported languages that are detected in a significant portion of the agreement.
