# Validate Extracted Field Values (Complete a Data Review)

This topic answers the question: How can I make sure that the suggested attribute (field) values are accurate? Learn how to complete a data review to verify suggested field values.

Data reviews are necessary to validate the data that is surfaced during processing and to provide values for empty attributes (fields). This data review is an important part of using Agreement Manager. It ensures accurate agreement data for reporting, business planning, sales forecasting, and negotiations.

Tip: Use the *Bulk Metadata Editing* feature to overwrite the extracted attribute values with user-provided agreement data. With this feature, you can edit the metadata for up to 1,000 selected documents in a single processing job. See the [Bulk Metadata Editing](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=concb493b18-adf8-4fe5-8b99-dd90cbc28399.html&_LANG=enus "Read about the Bulk Metadata Editing feature, including use cases, typical roles and permissions, and a process overview.") topic to learn about more about this feature.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Use the Filters option to locate the agreements that you want to review:
   1. Select the Filters option.

      The Filters menu expands.
   2. Select the AI Suggestions filter.

      The AI Suggestions filter opens.

      ![A screenshot of the expanded AI Suggestions dialog box that shows the two filter options, Needs Review and No Suggestions.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_AI-Suggestions-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2MjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.ji2WZDNjTRDHA91rzWygwnn04pAXgBV3hgCUmEYUKFU&_LANG=enus "A screenshot of the expanded AI Suggestions dialog box that shows the two filter options, Needs Review and No Suggestions.")
   3. Choose the Needs Review option and select Apply.

   The filter closes and the agreements that match the applied filter are displayed in the agreement list table.

   Tip: ![An image of the spark icon used to denote the use of AI.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/AI-Spark-icon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2MjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.ji2WZDNjTRDHA91rzWygwnn04pAXgBV3hgCUmEYUKFU&_LANG=enus "An image of the spark icon used to denote the use of AI.") The spark icon indicates that the agreement contains suggested values. Review, edit, and approve these values to ensure accurate agreement data for reporting, forecasting, and other business purposes.
5. Select (click) the agreement that you want to review.

   The Agreement preview page opens for the selected agreement. The Details pane is displayed by default and shows a list of attributes and suggested values. These attributes and values are displayed based on the identified agreement type.

   A summary displays at the top of the Details pane. The summary includes the identified agreement type and the number of fields and suggestions to review:

   ![A snippet of the Details pane that shows the identified agreement type and the number of suggestions that require review.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Preview-page_Details-pane_summary-statement-snippet.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2MjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.ji2WZDNjTRDHA91rzWygwnn04pAXgBV3hgCUmEYUKFU&_LANG=enus "A snippet of the Details pane that shows the identified agreement type and the number of suggestions that require review.")

   Tip: Use the Find details search field to locate specific attributes or values.

   Validate the suggested values for each attribute.
6. Place the Details pane into *edit* mode.
   - Select the Edit (pencil) icon.
   - Select the Review All option.

   Note: You must have Organize and manage all agreements access to perform data reviews. For all other users, the Edit (pencil) icon and the Review All option are grayed out.

   The Review & edit details pane opens for updates.

   ![A screenshot of the Agreement preview page that shows the Review & edit details pane with editable attribute value fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Review-and-edit-details-pane_Agreement-details-open-for-edits.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2MjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.ji2WZDNjTRDHA91rzWygwnn04pAXgBV3hgCUmEYUKFU&_LANG=enus "A screenshot of the Agreement preview page that shows the Review & edit details pane with editable attribute value fields.")

   Important: When you accept or edit the suggested attribute values, the content in the agreement does not change.
7. Accept or edit the suggested values until each attribute is validated. If the attribute value does not exist within the agreement, leave the field empty.

   ![A snippet of the Review & edit details pane that shows an extracted value highlighted in the agreement.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Review-and-edit-details-pane_verify-extracted-value_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2MjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.ji2WZDNjTRDHA91rzWygwnn04pAXgBV3hgCUmEYUKFU&_LANG=enus "A snippet of the Review & edit details pane that shows an extracted value highlighted in the agreement.")

   1. Compare the suggested attribute values against the data in the agreement.

      - Select (click) an attribute to view the suggested value within the agreement.

        The value is highlighted within the agreement.
      - Select the Accept AI Suggestion (checkmark) icon if the value is *correct*.
      - Delete the existing value and enter the correct value if the suggestion is *incorrect*.
   2. Continue reviewing and updating attribute values until all the fields have been verified against the agreement.

      Attributes are grouped based on usage and purpose. A progress bar is displayed at the top of each group and shows the number of suggestions that require review.
8. Select the Save option.

   The agreement details are saved. The Details pane is reverted to read-only mode.
9. Select the Close (x) icon from the top left.

   You are returned to the Completed Documents view. Your search criteria and filters are still applied and agreements requiring review are displayed in the table.

You have successfully completed a data review for an agreement with unverified AI suggestions.
