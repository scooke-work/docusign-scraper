# Access Renewing Agreements from the "Completed Documents" Page

This topic answers the question: How can I access my renewing agreements from Agreement Manager?

A banner message displays when you have agreements with upcoming renewal notice dates in the next 30 days. This message displays at the top of the Completed Documents view and contains a link to the Renewals dashboard.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.

   The insights bar displays a message about your renewing agreements. You have agreements with renewal notice dates within the next 30 days.
4. Access your upcoming renewals:

   ![A snippet of the Completed Documents view that highlights the hover state for the Upcoming renewals card in the insights bar.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-Documents_My-Insights_Upcoming-renewals-card_View-Renewals-Dashboard.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyMzksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.37HkdlZ0iR-gt2-7C1IY5uQ4Y3IDzb7d5e-7XoGl5hk&_LANG=enus "A snippet of the Completed Documents view that highlights the hover state for the Upcoming renewals card in the insights bar.")

   1. Select the Show Insights option from the right side of the insights bar.

      The My Insights bar expands and various agreement insight cards are displayed.
   2. Hover your cursor over the Upcoming renewals card.
   3. Select the View Renewals Dashboard option. This option is displayed when you hover over the Upcoming renewals card.

   The Renewals dashboard opens in a new tab. The dashboard charts are filtered to reflect the agreements with renewal notice dates within the next 30 days.

   ![A screenshot of the Renewals dashboard that shows the dashboard charts are pre-filtered to show agreements with upcoming renewals.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Renewals_Renewals-Dashboard_Accessed-from-Completed-docs-view.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyMzksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.37HkdlZ0iR-gt2-7C1IY5uQ4Y3IDzb7d5e-7XoGl5hk&_LANG=enus "A screenshot of the Renewals dashboard that shows the dashboard charts are pre-filtered to show agreements with upcoming renewals.")
5. Access the detailed report view for the Upcoming renewals chart:
   - Select the chart title.
   - Hover the cursor over the chart, expand the More actions (three dots) menu, and select the View Report option.

   The detailed report view opens for the selected chart. The agreements represented in the chart are listed in the table.

   ![A screenshot of the Upcoming renewals report that is pre-filtered to show the agreements with upcoming renewals.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Renewals-dashboard_Upcoming-renewals-prefiltered_from-banner-message.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyMzksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.37HkdlZ0iR-gt2-7C1IY5uQ4Y3IDzb7d5e-7XoGl5hk&_LANG=enus "A screenshot of the Upcoming renewals report that is pre-filtered to show the agreements with upcoming renewals.")

   Tip: If necessary, apply additional filters or customize the table data before downloading the report.
6. Download the report. Expand the More actions (three dots) menu in the top right and select the Download option:

   ![A snippet of a detailed report view that shows the Download option in the expanded dropdown.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Renewals_Detailed-report_Download.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyMzksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.37HkdlZ0iR-gt2-7C1IY5uQ4Y3IDzb7d5e-7XoGl5hk&_LANG=enus "A snippet of a detailed report view that shows the Download option in the expanded dropdown.")

   The associated report is downloaded to your local Downloads folder.

You have successfully accessed your renewing agreements from the banner message on the Completed Documents page.
