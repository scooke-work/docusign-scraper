# View and Update Your Custom Reports

This topic answers the question: How can I review and update my custom reports?

After you customize and save a report, it is available from the Custom tab on the Reports view.

![A snippet of the Reports view that shows the custom reports found on the Custom tab.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Reports-view_Custom-tab-snippet.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwMDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.90XSO6yZMBgZQpW1P1Kx_u55qXvRFP2J2hiPgNBqmTE&_LANG=enus "A snippet of the Reports view that shows the custom reports found on the Custom tab.")

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Insights from the global header.

   The Insights hub opens and the Overview is displayed.
3. Select Reports from the navigation pane on the left.

   The Reports view opens. The Recents tab is displayed by default.
4. Select the Custom tab.

   The Custom tab opens.

   ![A snippet of the Custom view that highlights the View option for the custom reports.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_View-saved-reports.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwMDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.90XSO6yZMBgZQpW1P1Kx_u55qXvRFP2J2hiPgNBqmTE&_LANG=enus "A snippet of the Custom view that highlights the View option for the custom reports.")
5. Select the View option for the relevant report.

   The detailed report view is displayed.

   Note: See the [View and Download an Agreement Report](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=izo1702945825083.html&_LANG=enus "Learn how to view an agreement chart, drill down to the detailed report view, apply filters, and download agreement data.    ![A screenshot of the detailed report view for the selected custom report.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Reports-view_View-custom-report.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwMDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.90XSO6yZMBgZQpW1P1Kx_u55qXvRFP2J2hiPgNBqmTE&_LANG=enus "A screenshot of the detailed report view for the selected custom report.")

   Tip: Add the report to Your Favorites list. Select the Favorite (star) icon located in the top-right corner:![A snippet of a detailed report view that highlights the Favorite (star) icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreements-dashboard_Report-details_Favorite-icon-selected_generic.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwMDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.90XSO6yZMBgZQpW1P1Kx_u55qXvRFP2J2hiPgNBqmTE&_LANG=enus "A snippet of a detailed report view that highlights the Favorite (star) icon.")Your starred dashboards and reports are displayed in the Your Favorites section on the Overview dashboard.

   Custom reports can be updated or deleted at any time from the report details page.

**Edit the report name**

6. Update the name of the report, as needed:

   ![A snippet of a detailed report view for a custom report that highlights the Edit report (pencil) icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-dashboard_Edit-custom-report_Edit-icon_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwMDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.90XSO6yZMBgZQpW1P1Kx_u55qXvRFP2J2hiPgNBqmTE&_LANG=enus "A snippet of a detailed report view for a custom report that highlights the Edit report (pencil) icon.")

   1. Select the Edit (pencil) icon located next to the report name.

      The Save Report dialog box opens.
   2. Update the Report name.
   3. Select the Confirm option.

   The dialog box closes and the report name is updated.

**Update the report filters and table data**

Create custom reports by applying filters to existing dashboard charts and choosing the Save As option.

7. Update report filters and customize table data:
   1. Remove a filter. Select the Clear (x) icon associated with the filter that you want to remove.
   2. Apply new filters or update the existing filters for the report. Expand the relevant filter, choose a value or enter a date, and select the Apply option.
   3. Update the columns in the table:

      - Select the  (toggle) icon. The Customize columns dialog box opens.
      - Select the toggles to turn the associated field on (blue) or off (white).
      - Re-order the columns as needed. Hover the cursor over the Reorder (dots) icon for the column that you want to move. Drag it to the appropriate position.
      - Select the Save option.The dialog box closes and the table is updated based on your column customization.

      Tip: If the Status column is not displayed by default, use the Show or Hide Fields (toggle) option to add it to the table. From the Show or Hide Field dialog box, turn on the Status toggle and drag the field to the appropriate location. Be sure to Save your changes.
   4. Save the changes to your custom report:

      - Expand the More actions (three dots) menu in the top right and select the Save  option.

        The Save Report dialog box opens.
      - Select the Confirm option to save the changes.The dialog box closes and changes to the custom report are saved.

**Update the report configuration**

Custom reports can be created from scratch by using the *Create Report* function, located under the Create dropdown:

![A snippet of the Insights hub that shows the Create Report option in the Create dropdown.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reporting-hub_Create-menu_Create-Report_highlighted_generic.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwMDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.90XSO6yZMBgZQpW1P1Kx_u55qXvRFP2J2hiPgNBqmTE&_LANG=enus "A snippet of the Insights hub that shows the Create Report option in the Create dropdown.")

Tip: See the [Custom Dashboards and Reports](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=conac48cc6c-d92c-4187-a733-08968c218a74.html&_LANG=enus "Read about custom dashboards and reports, including use cases and limitations.") topic to learn more about configuring custom dashboards and reports.

8. Update the report configuration, including the data source, metrics, and visualization:
   1. Select the Edit report (pencil) icon from the top right:

      ![A snippet of the detailed report view that highlights the Edit report (pencil) icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Custom-report_Edit-report_highlighted_generic.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwMDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.90XSO6yZMBgZQpW1P1Kx_u55qXvRFP2J2hiPgNBqmTE&_LANG=enus "A snippet of the detailed report view that highlights the Edit report (pencil) icon.")The report configuration page opens.
   2. Update the Data source for the report, including applying filters as needed.
   3. Update the report's metrics, including the measure, aggregation, and
      currency.
   4. Change the Group by field as needed. Select the field by which you want to group or break down the metrics.
   5. Update the Visualization field to change how the report is displayed.

      As you make changes to the report, the pane on the right updates to match the new configuration.
   6. Select the Save option from the top right.

      The Save report dialog box opens.
   7. Update the Report name and Description fields as needed and select the Save option.

   You are returned to the detailed report view for the custom report. The report reflects the changes that you made to the configuration.
