# Custom Dashboards and Reports

Read about custom dashboards and reports, including use cases and limitations.

Agreement Manager provides standard reports and dashboards so customers can access essential agreement
data. However, these preconfigured reports and dashboards are limited when it comes to
reporting on specialized data, such as custom field data.

Customizable dashboards and reports give you control over the agreement insights that you see - giving you the right data to make smarter decisions. This feature is accessed from the Insights page:

![A snippet of the reporting hub that highlights the Create dropdown with options to create reports and dashboards.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Custom-report-dashboard_Create-expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNTAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.zFnrQwtMuRPhPuVQpF2DBrV8YiWh384sY-I6oz7c4s0&_LANG=enus)

- **Customizable dashboards** collections of standard and custom reports that show a larger
  picture of a particular focus area. The dashboards are:
  - Created from scratch by choosing reports for the dashboard.
  - Built from an existing template.
- **Customizable reports** individual data visualizations, such as bar charts and graphs. The
  primary ways to create custom reports:
  - **From scratch:** Create custom reports from scratch by choosing the
    relevant data, metrics, and visualization options.
  - **From existing reports:** Create custom reports by updating and
    saving standard reports.
  - **From suggested reports:** Create custom reports by updating and
    saving suggested reports.

Important: Users can create up to 100 custom
dashboards. Each dashboard can contain up to 20 custom or standard reports. By default, the user that creates a custom dashboard or report has administrator permissions for it. They can edit, rename, and delete their custom dashboards and reports.

## Example use cases

The custom dashboard and report function give organizations a way to report on specialized
agreement data across their agreement repository. Specialized agreement data could
include custom field data, organization-specific terms, or industry-specific terms.

Consider these example use cases:

**Sales operations:**

Create reports to report on custom fields. For example, to track and report on
product usage by party or region. This specialized agreement data can be used to
enhance expansion efforts and to inform the executive team.

**Legal:**

Creates reports to report on industry-specific regulations and high-risk clauses. For
example, to monitor agreements under the U.K. General Data Protection Regulation
(GDPR) or to track agreements with penalty provisions.

**Procurement:**

Creates reports to monitor custom agreement data, such as volume-discount tiers. For
example, to track agreements with volume-discount tiers to maximize cost
savings.

## Limitations

Consider these limitations for custom reports and dashboards:

| Limitation | Description |
| --- | --- |
| **Sharing** | Currently, there is no way to share custom reports and dashboards with other users or groups. |
| **Dashboard maintenance** | Users can delete reports from custom dashboards if the report is no longer relevant, however, this function deletes the report. Once deleted, it is no longer available to view or to add to a dashboard. |
| **Metrics** | Custom reports only support numeric values as metrics or key measures. The use of non-numeric fields as measures is not supported. |
| **Single stack bar charts** | Users are unable to build stacked bar charts or charts that have two levels of pivot. For example, distribution of agreements with governing law, grouped by governing law, and segmented by region or party. |
| **Dashboard templates** | Users cannot set custom dashboards as "templates" for other users. Templates help users to build custom dashboards. Currently, users only have access to Agreements and Renewals dashboards as templates. |
| **Standard report options** | Custom dashboards only include standard Agreement Manager and IAM reports. Users cannot add eSignature reports when building custom dashboards. Note: Currently, the custom report building experience is not available for eSignature reports. |
