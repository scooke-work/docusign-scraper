# Use Auto-Suggestion to Create Custom Agreement Types

Learn how to create a custom agreement type from the auto-suggestion feature.

When uploading or ingesting agreements to Agreement Manager, if an agreement does not match an existing standard or custom type, it is identified as Miscellaneous. If 10 or more similar agreements are identified as Miscellaneous, a custom agreement type is suggested:![A snippet of the agreement list table that shows a suggested custom agreement type for a Miscellaneous document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Autosuggest-custom-agreement-type_Charter-agreement-example_messaging.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.yB2EBrA5aWPu7ZOx99UtnDJv1kaRgMUfo1IpojMuB0w&_LANG=enus "A snippet of the agreement list table that shows a suggested custom agreement type for a Miscellaneous document.")Follow the steps in this topic to create a custom agreement type from the auto-suggestion feature.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Use the Upload function to add documents to Agreement Manager:

   ![A snippet of the Completed view that highlights the Upload option in the Add (+) dropdown.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Upload-agreements_Add-Documents_Upload-option_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.yB2EBrA5aWPu7ZOx99UtnDJv1kaRgMUfo1IpojMuB0w&_LANG=enus "A snippet of the Completed view that highlights the Upload option in the Add (+) dropdown.")

   1. Expand the Add (+) dropdown and select the Upload option.

      The Upload documents dialog box opens.
   2. Drag the files that you want to upload to the Drag and Drop
      files here box. You can also use the Select
      File to select the files from cloud storage locations or
      local folders.

      The Upload documents dialog box opens.

      Remember: The auto-suggestion feature is triggered when 10
      or more similar Miscellaneous agreements are
      uploaded or ingested to Agreement Manager. The Upload up to 1,000
      files at a time permission is required to upload
      more than five documents.
   3. Complete the Select Files step and select Next.
   4. Complete the Allow access step and select the relevant Upload option.
   5. Follow the prompts to complete the upload process.

   The dialog box closes and a confirmation message is displayed.

   Tip: Monitor progress and view uploaded files.
   - Select the Monitor Progress option located in the confirmation message. The All jobs page opens and the Uploaded Agreements table is displayed.
   - Locate the upload job in the Uploaded Agreements table.
   - Expand the Actions dropdown for the job and select Browse Files.

   The uploaded files are displayed in the agreement list table.

   ![A screenshot of the agreement list table that highlights the auto-suggestion messaging in the Agreement Type column.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Autosuggest_Completed-view_Agreement-Type-with-autosuggest-messaging_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.yB2EBrA5aWPu7ZOx99UtnDJv1kaRgMUfo1IpojMuB0w&_LANG=enus "A screenshot of the agreement list table that highlights the auto-suggestion messaging in the Agreement Type column.")
5. Review the suggested custom agreement type:

   ![A snippet of the agreement list table that shows a suggested custom agreement type for a Miscellaneous document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Autosuggest_New-suggestion-available_dialog-box_charter-agreement-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.yB2EBrA5aWPu7ZOx99UtnDJv1kaRgMUfo1IpojMuB0w&_LANG=enus "A snippet of the agreement list table that shows a suggested custom agreement type for a Miscellaneous document.")

   1. Select the New suggestion available message from the Agreement Type column.
   2. Review the suggested custom agreement type. The new type is suggested based on key data points within the uploaded agreements. This includes all the similar agreements in the upload job that were identified as Miscellaneous.
6. Select the Create option.

   Note: The auto-suggestion feature is also accessed from the Agreement preview page for a selected agreement. Select the New suggestion available message from the Details pane and use the Create option to create the suggested type.

   The Create new agreement type page opens. Use this page to create a custom agreement type from the auto-suggestion feature.

   ![A screenshot of the Create new agreement type page that shows the default configuration values and examples for the new type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Autosuggest_Create-new-agreement-type_full-page.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.yB2EBrA5aWPu7ZOx99UtnDJv1kaRgMUfo1IpojMuB0w&_LANG=enus "A screenshot of the Create new agreement type page that shows the default configuration values and examples for the new type.")
7. Review and update the Agreement Type, Category, and Definition values for the new type, if necessary:

   ![A snippet of the Create new agreement type page that shows the type, category, and definition details for the new type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Autosuggest_Create-new-agreement-type_default-configuration-values.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.yB2EBrA5aWPu7ZOx99UtnDJv1kaRgMUfo1IpojMuB0w&_LANG=enus "A snippet of the Create new agreement type page that shows the type, category, and definition details for the new type.")These values were generated with AI based on the agreement details from the upload job. This includes details from "similar" agreements in the upload job that were identified as Miscellaneous.
8. Complete the Provide examples of this agreement type section to enable the AI detection for the custom agreement type:

   ![A snippet of the Create new agreement type page that shows the pre-selected examples for the agreement type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Autosuggest_Create-new-agreement-type_Provide-examples-of-this-agreement-type.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.yB2EBrA5aWPu7ZOx99UtnDJv1kaRgMUfo1IpojMuB0w&_LANG=enus "A snippet of the Create new agreement type page that shows the pre-selected examples for the agreement type.")

   This section is pre-populated with a sampling of "similar" agreements from the upload job, including the current agreement.

   1. Select an example (tile) to view the agreement in the Agreement preview page.
   2. Select the Delete (x) icon to remove an example from the list.
   3. Use the + Add example option to add other examples of the agreement type.
9. Select the Save option from the top right of the
   page.

   You are returned to the **Completed Documents**
   page. The new agreement type is created and enabled for AI detection.

   Once created, the custom type is applied to all the relevant documents that were previously identified as Miscellaneous. Depending on the processing queue, it may take an hour or longer for the agreements to reflect the new type.

   You can filter the agreement list table using the new custom type:

   ![A snippet of the agreement list table that shows the new custom type in the Agreement Type filter.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Autosuggest_Agreement-Type-filter_Charter-Agreement-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.yB2EBrA5aWPu7ZOx99UtnDJv1kaRgMUfo1IpojMuB0w&_LANG=enus "A snippet of the agreement list table that shows the new custom type in the Agreement Type filter.")

   Important: The agreements contain attributes and field values that were surfaced with AI. Complete a data review to validate these values, including the Agreement Type. See the [Validate Extracted Field Values (Complete a Data Review)](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=yqq1702945693043.html&_LANG=enus "This topic answers the question: How can I make sure that the suggested attribute (field) values are accurate? Learn how to complete a data review to verify suggested field values.") article to learn more about validating AI-surfaced values.

You have successfully created a custom agreement type from the auto-suggestion feature.
