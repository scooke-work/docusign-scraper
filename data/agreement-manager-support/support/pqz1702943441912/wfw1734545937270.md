# Subscribe to Receive Report Notifications

Learn how to set up one-time or recurring email reports for yourself or others to receive notifications about your obligations, renewals, or agreements.

You can subscribe to receive one-time or recurring email reports for any of the charts on the report dashboards - Agreements, Obligations, and Renewals.

Tip: Notification settings give you control over the email notifications that you receive. By default, all the notification types related to the completed agreements in your account are selected, or enabled. Deselect a checkbox if you no longer want to receive email notifications for the corresponding notification type.

See the [Customize Notification Settings for Your Completed Agreements](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk155b1ed7-7b45-43cd-be92-2a5279630c0f.html&_LANG=enus "Learn how to customize your email notification settings from the Notification Preferences page.") article to learn more about updating your notification settings. Be sure to periodically check the Notification Preferences page as new notification types will be added in the future.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Insights from the global header.

   The Insights hub opens and the Overview is displayed.
3. Select the dashboard containing the reports to which you want to subscribe. Select either Agreements, Obligations, or Renewals from the navigation pane on the left.

   The selected dashboard opens.
4. Use the available filters to locate the relevant agreement, obligation, or renewals information.

   Tip: To subscribe to a custom report, locate and select the report from the Custom tab located under the Reports view. See the [View and Update Your Custom Reports](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=yuf1727801071281.html&_LANG=enus "This topic answers the question: How can I review and update my custom reports?") article to learn more about locating your custom reports.
5. Access the detailed report view for the chart to which you are subscribing. Select (click) the chart title or select the View Report option from the More actions (three dots) menu.

   The detailed report view opens for the selected chart. Next, apply filters, and customize the table columns until the table displays the data that you want to track.
6. Filter the table to refine the results list as needed or select Reset to clear existing filters.
7. Customize the information that is displayed in the table, if necessary:

   ![A screenshot of the Customize columns dialog box that shows the columns you can choose to display in the agreements table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Overview-dashboard_Customize-columns.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxMTksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Iy6U99DvoO8oV7VL4cWf2HZL2tncIfes_e-jcuM4JGI&_LANG=enus "A screenshot of the Customize columns dialog box that shows the columns you can choose to display in the agreements table.")

   1. Select the Customize columns (gear) icon.

      The Customize columns dialog box opens. These fields represent the columns in the table.
   2. Select the toggles to turn the associated field on (blue) or off (white).
   3. Re-order the columns as needed. Hover the cursor over the dots to the right of the field name and drag it to the appropriate location.
   4. Select Save.

   Note: If you made changes to the report, expand the dropdown and select Save As:![A snippet of an agreement dashboard report that shows the Save As option in the dropdown.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Agreements-Dashboard_Dashboard-report_Save-as.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxMTksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Iy6U99DvoO8oV7VL4cWf2HZL2tncIfes_e-jcuM4JGI&_LANG=enus "A snippet of an agreement dashboard report that shows the Save As option in the dropdown.")

   Once saved, access the report from the Custom tab located under the Reports view. Continue to the next step to subscribe to your custom report.
8. Select the Manage Notification (bell) icon from the top-right corner.

   The Create Subscription dialog box opens.

   Note: The Manage report subscriptions dialog box displays when you have existing subscriptions. This includes the subscriptions to which you were added by other users. Expand the More actions (three dots) menu and choose to Edit or Delete the corresponding subscription:![A snippet of the Manage report subscriptions dialog box that shows the available Actions for an existing subscription.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-report-subscriptions_Actions-menuP_EditDelete.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxMTksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Iy6U99DvoO8oV7VL4cWf2HZL2tncIfes_e-jcuM4JGI&_LANG=enus "A snippet of the Manage report subscriptions dialog box that shows the available Actions for an existing subscription.")If you were added to a subscription as an individual user, the subscription is displayed in the dialog box. Currently, subscriptions are not visible to users who were added to as part of a group.
9. Complete the fields in the Create subscription dialog box and select Schedule:

   ![A screenshot of the Create subscription dialog box that shows the subscription configuration fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Create-Subscription.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxMTksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Iy6U99DvoO8oV7VL4cWf2HZL2tncIfes_e-jcuM4JGI&_LANG=enus "A screenshot of the Create subscription dialog box that shows the subscription configuration fields.")

   1. Enter a descriptive name in the Subscription Name field. This field defaults to the name of the report.
   2. Indicate how often and when you want to receive the report notification. Choose from Daily, Weekly, Monthly, and Once.

      Depending on the option you selected, you may be prompted to choose a send day.
   3. If necessary, turn on the Choose start and end date toggle and enter a Start date and End date.
   4. The notification is sent to you by default. Enter additional recipients in the Add a user or group field as needed.

      Note: When you create a subscription, you have the option to add individual users and groups. Once created, the subscription is displayed in the Manage report subscriptions dialog box for any individual users who were added. Currently, the subscription is not visible to users if they were added as part of a group.
   5. Select the Attach to email as CSV option as needed.
   6. Select Schedule.

   The dialog box closes. You have successfully signed up to receive report notifications.

You have successfully subscribed to receive a report email notification for a selected dashboard report.
