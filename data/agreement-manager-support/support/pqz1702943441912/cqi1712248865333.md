# Download an Agreement

Learn how to download an agreement from Agreement Manager. This topic answers the question: How
can I download an agreement to share with others?

Use the Download feature to download and share agreements with key
stakeholders, for example during negotiations. This feature is available from the
agreement list table (Completed Documents view) and the
Agreement preview page.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign
   Agreements page opens.
3. Select Completed from the
   AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Perform a keyword search and use the filters to locate the agreement that you
   want to download.

   The agreement list table updates and agreements that match the entered search
   criteria and applied fields are displayed.
5. Expand the More actions (three dots) menu
   for the agreement that you want to download and select
   Download:

   ![A snippet of the Completed Documents view that shows the Download option in the More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed_documents_table_vertical_ellipses_Download.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2NjUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.NyrVUS3LCp65g3rJgyBC42IDJpywI9nZw61i0HtCCCY&_LANG=enus "A snippet of the Completed Documents view that shows the Download option in the More actions (three dots) menu.")

   The agreement downloads to your local Downloads
   folder.
6. You may want to view the agreement before downloading the file. Select the
   agreement that you want to review.

   The Agreement preview page opens.

   ![A screenshot of the Agreement preview page that highlights the Download button in the upper right corner.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement_Preview_page_Select_Download.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2NjUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.NyrVUS3LCp65g3rJgyBC42IDJpywI9nZw61i0HtCCCY&_LANG=enus "A screenshot of the Agreement preview page that highlights the Download button in the upper right corner.")
7. Review the agreement as needed.
8. Select Download from the top-right corner.

   The agreement is downloaded to your local Downloads folder.

   You can share the downloaded file with key stakeholders.
9. Select the Close (x) icon from the top-left
   corner.

   You are returned to the Completed Documents view. Your
   entered search criteria and applied filters are still in effect.
10. Download multiple agreements from the agreement list table:

    ![A snippet of the Completed Documents view that highlights the Download Documents option for a group of selected agreements.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Download-Documents_Highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2NjUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.NyrVUS3LCp65g3rJgyBC42IDJpywI9nZw61i0HtCCCY&_LANG=enus "A snippet of the Completed Documents view that highlights the Download Documents option for a group of selected agreements.")

    1. Select the agreements that you want to download.

       - Select individual agreements.
       - Select the Choose all checkbox from the
         header row to select all the agreements on that page of
         results.
       - Select the Select all that match your
         filters link to select all the agreements in the
         table that match your applied filters.

         Tip: The Select all
         that match your filters option is only
         available if filters are applied to the agreement list
         table.
    2. Select Download Documents.

    A zipped folder containing the selected agreements is downloaded to your local
    Downloads folder.

You have successfully downloaded an agreement.
