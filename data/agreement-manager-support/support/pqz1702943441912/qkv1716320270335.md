# Reconnect a Docusign App Connection

Learn how to use the Docusign app Reconnect a connection feature to reestablish the Docusign app’s connection to workflows.

If there is a connection disconnect, select Docusign app Reconnect a connection to reestablish the app’s connection to workflows.

For information about reconnecting Salesforce connections, see [Reconnect a Salesforce Organization to Your Workflow Builder App](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=kam1719258759323.html&_LANG=enus)

1. Select the Docusign app.
2. On the app detail page, select Manage Connections to access the
   Connections page.

   ![A screenshot of a sample App Center Connections page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/ac-newest-conn-page-ex.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTgxNjksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.t8qOwSc3PPh-EmO6mI4_yTDX5sN3xUFqh1SS_3FwryY&_LANG=enus "A screenshot of a sample App Center Connections page.")
3. For the connection you want to reconnect, select Reconnect from the overflow menu.
4. At the Reconnect this connection dialog box, type RECONNECT and select Reconnect.

The Docusign for Salesforce app connection to
workflows is available or re-established.
