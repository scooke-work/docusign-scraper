# Locate Agreements by Asking Questions

Learn how to locate data about your agreements by entering a question in the search field.

With natural language (NL) queries, you can enter questions or key phrases directly into the search field on the Agreement Manager homepage. The query returns a direct answer and a list of relevant agreements.

Important: Depending on the nature of the question posed, not all queries yield both direct answers and a list of agreements. Some queries may only provide a list of relevant agreements.

Note: The NL search function is currently available in beta. See
the *Opt-in for NL search (Join Beta)* section of the [Natural Language Search Queries](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=fas1739987568042.html&_LANG=enus "Read about how natural language search queries, including components of the Search Results view.") topic to join the beta.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the pane on the left.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. A list of documents to which the logged-in user has access is displayed in the table. The total document count is displayed the bottom right. This count is updated when you perform a search or apply filters.
4. Enter a question or key phrase in the search field and press the Enter key or select an option from the dropdown list:

   ![A snippet of the Agreement Manager homepage that shows an example search phrase to locate the value of master service agreements (MSAs) that auto renew this year.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/NL-Search_Example-key-phrase-with-dropdown.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0MjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.r1E5Hy0nWIhbXtRm_fjK98Uxbg_HEIVT_y3VsioUrMc&_LANG=enus "A snippet of the Agreement Manager homepage that shows an example search phrase to locate the value of master service agreements (MSAs) that auto renew this year.")

   Tip: Here are some sample questions that you can enter into the search field:
   - Do I have an amendment with [Party X]?
   - Do I have an active NDA with [Party X]?
   - Which MSAs will auto-renew?
   - Which agreements do not have a governing law?

   The query returns an answer to the question and a list of
   relevant agreements. The document count is updated
   based on your search query.
5. Review the generated answer:

   ![A snippet of the search results view that shows the detected filters and a direct answer to the query question.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/NL-Search_Review-answer-to-question.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0MjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.r1E5Hy0nWIhbXtRm_fjK98Uxbg_HEIVT_y3VsioUrMc&_LANG=enus "A snippet of the search results view that shows the detected filters and a direct answer to the query question.")

   1. If filters were applied while answering the question, the Filters applied field displays the relevant filters. In this example, a number of filters were applied:

      - Expiration Date: >2025-01-01 AND <2026-01-01
      - Agreement Type: Master Service Agreement
      - Renewal Type: Auto Renew
      - Currency Code: USD

      Note: Select a filter to view the selected values. For example: ![A screenshot of the Expiration Date filter that is expanded to show the selected values.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/NL-Search_Detected-filter_Selected-expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0MjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.r1E5Hy0nWIhbXtRm_fjK98Uxbg_HEIVT_y3VsioUrMc&_LANG=enus "A screenshot of the Expiration Date filter that is expanded to show the selected values.")

      Restriction: The NL search feature is restricted to agreements where the detected Language is *English* and the Currency Code is *USD*.
   2. The answer is generated based on the data points that were surfaced in your agreements and may require further review.
   3. Use the Copy this answer icon to copy the answer to your clipboard for use in another application.
   4. Provide feedback about the answer. Use the
      Rate as helpful (thumbs up) and
      Rate as unhelpful (thumbs down)
      icons to rate the quality of the generated
      answer.
   5. Select the Use keyword search instead option to perform a regular search if the query did not answer your question.
6. Review the agreement list table to see the agreements that were returned in the search.

   Tip: If you routinely search for the same type of agreements or agreement data, save your search criteria for future use. See the [Save Search Criteria for Future Use](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tskc38121a1-bbf8-4da8-8ed7-b364991f7e50.html&_LANG=enus "Learn how to save the search criteria for the queries that you frequently perform.") topic to learn more about the saved search feature.
7. Download selected agreements or export agreement data to a CSV file:

   ![A screenshot of the search results view that highlights selected agreements and the download and export options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/NL-Search_Example-key-phrase_download-export-search-results.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0MjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.r1E5Hy0nWIhbXtRm_fjK98Uxbg_HEIVT_y3VsioUrMc&_LANG=enus "A screenshot of the search results view that highlights selected agreements and the download and export options.")

   1. Select the checkboxes associated with the agreements that you want to download or export. You can also select the Choose all checkbox from the header row to select all the agreements on the page.

      The Download Documents and Export to CSV icons are displayed.
   2. Select the Download Documents (arrow) icon.

      The selected agreements are downloaded to your local Downloads folder.
   3. Select the Export to CSV icon to export agreement data for the selected files. All the available attributes and field values for the selected agreements are exported to a CSV file.

      The file is downloaded to your local Downloads folder.
8. Reset the search field to perform a new query. Select the Clear (x) icon located in the search field.

   The search field resets and all the agreements to which the logged-in user has access are displayed.

   Important: The natural language search does not support the use of filters together with questions. If you apply filters to the agreement list table, any natural language question entered in the search box is treated as a standard keyword search.

You have successfully located agreements and agreement data by asking questions.
