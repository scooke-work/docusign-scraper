# Import Your Edited Metadata Spreadsheet

Learn how to import your edited metadata spreadsheet using the Add (+) menu located on the Agreement Manager homepage.

## Before you begin

Import the updated spreadsheet file and resolve any upload errors.

Important: When you import the updated agreement data, the current data is overwritten. This action cannot be undone once saved. The user-provided data is final. The cells must contain a valid value to overwrite the current value. If the cell is empty in the spreadsheet, the current field value is retained.

This task is completed after you update the pre-formatted spreadsheet file with the relevant metadata. If you haven't updated the spreadsheet, see the [Edit the Metadata Spreadsheet](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk8710c3f2-c881-4219-96b0-2b287e8b4e03.html&_LANG=enus "Learn how to edit the metadata spreadsheet file to bulk update the agreement data for the selected documents.") article to learn more about this task.

Note: The *Bulk Metadata Editing* process is completed by users with Organize and manage all agreements access to Agreement Manager and bulk upload permissions.

Import the spreadsheet file to update the metadata for the selected agreements. Once successfully imported, the agreement data is synced and the updated values are reflected across the Intelligent Agreement Management (IAM) platform.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Expand the Add (+) dropdown and select the Import Data option:

   ![A snippet of the Completed Documents page that highlights the Import Data option in the Add (+) dropdown menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Bulk-metadata-editing_Add-menu_Import-Data_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4MTUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.I_8q6999MpteYMlP4HBkYTsVAEyPc_o3XoyeTM5Erq0&_LANG=enus "A snippet of the Completed Documents page that highlights the Import Data option in the Add (+) dropdown menu.")

   The Import data from a spreadsheet dialog box opens. You are prompted to drop or select your edited file.
5. Upload the pre-formatted spreadsheet file that you updated with metadata values:

   ![A screenshot of the Import data from a spreadsheet dialog box that highlights the selected file and the Import Data option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Bulk-metadata-editing_Import-data-from-spreadsheet_Select-file_Import-Dtata.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4MTUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.I_8q6999MpteYMlP4HBkYTsVAEyPc_o3XoyeTM5Erq0&_LANG=enus "A screenshot of the Import data from a spreadsheet dialog box that highlights the selected file and the Import Data option.")

   - Locate the metadata spreadsheet file and drop it in the dialog box.
   - Use the Select File option to search for and select the file.

   Important: Upload the same file that you downloaded previously. If the file structure does not match the original spreadsheet template, the upload is rejected.

   The dialog box updates and the selected file is displayed in a green box. This indicates that the file was accepted by the import process. If the file failed this initial validation, a red box is displayed and the messaging indicates that the file was rejected.

   Note: Keep in mind that the file can be accepted by the import process but still fail validation due to formatting errors.
6. Select the Import Data option.

   The dialog box closes and you are returned to the Agreement Manager homepage. A confirmation message indicates that the data was imported.

   Tip: Use the View Details option to view the Job status for the upload job, including files that failed to update and the associated error messages. You can access this page from the email notification too.

   When the upload is completed, a system-generated email notification is sent. The email contains a brief summary of the import results, including the number of files that failed to update.

You have successfully imported the updated metadata spreadsheet.
