# Manage Agreement Sets

Read about the agreement set management functions, such as editing smart filters and updating share settings.

Contract managers and users who are assigned the Organize and manage all agreements permission can create and update agreement sets.

Set management tasks are completed at the set-level. Locate the set that you want to update from the table on the All agreement sets view:

![A snippet of the All agreement sets view that highlights the set management capabilities in the More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-set-management_Actions_expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczOTIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.0spuCcmPWNKhSMuoIg08gQzWJ8REYtmkymrOQ5CPIPo&_LANG=enus "A snippet of the All agreement sets view that highlights the set management capabilities in the More actions (three dots) menu.")

Every set has a More actions (three dots) menu. Use this menu to perform set management tasks, such as:

- **Edit details:** used to update the set's name and description.
- **Edit smart filters:** used to add or update the set's smart filters.
- **Share:** used to share the set with additional users or groups, and update or remove set access.
- **Delete:** used to delete the set.

  Important: All users and groups with whom the set was shared will lose access once the set is deleted. Deleting the set does not delete or void any agreements.
