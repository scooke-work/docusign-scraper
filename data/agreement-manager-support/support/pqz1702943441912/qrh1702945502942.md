# Customize the Agreement List Table to Show Relevant Data

This topic answers the question: How can I see relevant agreement data in the Agreement Manager table? Learn how to configure the columns in the agreement list table to display relevant data.

The agreement list table is customizable. You can select which attributes (fields) are displayed in the table. This topic begins from the Agreement Manager homepage. The Completed Documents view is displayed.

Remember: The Completed Documents view is located under the Agreements tab in the global header:

- Select Agreements from the global header.
- Select Completed from the AGREEMENTS section in the navigation pane.

![A snippet of the Completed Documents view that highlights the Show of Hide Fields (toggle) icon on the right-side of the page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Customize-columns-gear-icon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NzUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.FTvOCeRjD3gTc0FhZ1-1yasnGabcXQKsKBo5Epa4fGE&_LANG=enus "A snippet of the Completed Documents view that highlights the Show of Hide Fields (toggle) icon on the right-side of the page.")

1. Select the Show or Hide Fields (toggle) icon located to the right of the column headings.

   The Show or Hide Fields dialog box opens. A number of fields are turned on
   (blue) by default. These fields represent the columns in the agreement list
   table. The field order in the dialog box reflects the column order in the
   table.

   Important: If you previously customized the columns in the agreement list table, you must manually display the Status column in the table. See the [Display the Status Column in the Agreement List Table](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk57b92226-d6e4-4725-99c4-79ebf42e9f8b.html&_LANG=enus "Learn how to customize the agreement list table to display agreement status information.") topic to learn about this task.

   ![A screenshot of the Show or Hide Fields dialog box that shows the default selections.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-view_Customize-columns-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NzUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.FTvOCeRjD3gTc0FhZ1-1yasnGabcXQKsKBo5Epa4fGE&_LANG=enus "A screenshot of the Show or Hide Fields dialog box that shows the default selections.")

   Note: The Show or Hide Fields function only applies to the data that is displayed in the agreement list table. When downloading or exporting the report, all fields are included in the output file. This includes both custom and standard fields.
2. Select the toggles to turn the corresponding field on (blue) or off (white).
3. Re-order the columns as needed. Hover the cursor over the Reorder (dots) icon for the field that you want to move. Drag the field to the appropriate location:

   ![A snippet of the Show or Hide Fields dialog box that highlights the 'drag to' feature for reordering column headers.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents_Customize-columns-dialog-box_drag-and-drop.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NzUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.FTvOCeRjD3gTc0FhZ1-1yasnGabcXQKsKBo5Epa4fGE&_LANG=enus "A snippet of the Show or Hide Fields dialog box that highlights the 'drag to' feature for reordering column headers.")

   Tip: The "grabbing" cursor displays when you hover over an element that can be dragged or moved.
4. Select the Save option.

   The dialog box closes. The new configuration is reflected in the agreement list table.

You have successfully updated the agreement list table to reflect the data that is relevant to you.
