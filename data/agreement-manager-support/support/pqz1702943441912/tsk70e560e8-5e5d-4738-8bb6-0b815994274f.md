# Update the Agreement Status for a Document

Learn how to update the agreement status for a document.

Agreement statuses provide a quick look at where the agreement is within its life cycle. These statuses are calculated based on an agreement's extracted Effective
Date, Expiration Date, and Renewal
Type.

Note: A Renewal Type value is not required for calculating an agreement's status. However, if an extracted value exists, it is taken into consideration when calculating the status. For example, if an agreement has an Expiration Date and the Renewal Type is Evergreen, the agreement is always Active.

When necessary, you can manually update an agreement's status. For example, update the status for a terminated agreement to *Inactive*.

Important: When you update an agreement's Status, it is no longer automatically calculated and updated. The calculation process stops, regardless of the Effective Date, Expiration Date, and Renewal Type values.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. A list of documents to which the logged-in user has access is displayed in the table.
4. Locate the agreement that you want to update.

   ![A snippet of the Completed Documents view that highlights the search field and available filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Navigator_Completed-documents_search-field-and-filters-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY1NzMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Xgs2K1cmfDpOKVBVxaDb2-lIBGNfoTmzMYY2PngQx4w&_LANG=enus "A snippet of the Completed Documents view that highlights the search field and available filters.")

   1. Enter a keyword or search string in the search field and select Enter.
   2. Use the Filters option to refine the list of search results. Locate the filter in the Filters list and choose or enter values. Select the Apply option.

      The agreements that match your search criteria and filters are displayed in the table. The document count reflects the entered search criteria and applied filters.
5. Select (click) the agreement that you want to update.

   The Agreement preview page opens for the selected agreement. The Details pane is displayed by default and shows a list of attributes and suggested values.
6. Place the Details pane into *edit* mode:

   ![A snippet of the Agreement preview page that highlights ways to place the Details pane into edit mode.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-statuses_Agreement-Preview_Details_Edit-options_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY1NzMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Xgs2K1cmfDpOKVBVxaDb2-lIBGNfoTmzMYY2PngQx4w&_LANG=enus "A snippet of the Agreement preview page that highlights ways to place the Details pane into edit mode.")

   - Select the Edit (pencil) icon.
   - Select the Review All option.

   Note: The ability to edit attribute values requires Can Manage or Can Edit access to the agreement:![A screenshot of the Access dropdown for an agreement that highlights the Can Edit and Can Manage options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-access_Can-Manage_Can-Edit_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY1NzMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Xgs2K1cmfDpOKVBVxaDb2-lIBGNfoTmzMYY2PngQx4w&_LANG=enus "A screenshot of the Access dropdown for an agreement that highlights the Can Edit and Can Manage options.")
   - Can Manage users can edit attribute values, as well as delete, share, and remove access to the agreement.
   - Can Edit users can edit attribute values, and share and download the agreement.

   The Review & edit details pane opens for updates.
7. Update the agreement's Status and select the Save option:

   ![A snippet of the Agreement preview page that highlights the Status field in the Review & edit details pane.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-statuses_Agreement-Preview_Review-edit-details_Status-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY1NzMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Xgs2K1cmfDpOKVBVxaDb2-lIBGNfoTmzMYY2PngQx4w&_LANG=enus "A snippet of the Agreement preview page that highlights the Status field in the Review & edit details pane.")

   1. Scroll to the Status field, located under the General section.
   2. Select the toggle to turn off (white position) the automatic status calculation feature. This feature is turned on by default to calculate the status based on related details.
   3. Expand the dropdown and select the relevant status, Active or Inactive.
   4. Select the Save option.

   The Details pane reverts to read-only mode and the agreement's status is updated.

   Remember: When you update an agreement's Status, it is no longer automatically calculated and updated. The calculation process stops, regardless of the Effective Date, Expiration Date, and Renewal Type values.
8. Select the Close (x) icon from the top-left corner.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. Your search criteria and filters are still applied.

You have successfully updated the status for a selected agreement.
