# View and Download an Overview Report

Learn how to view reports from the Overview dashboard, including accessing and downloading the detailed report view.

The Overview dashboard is the landing page for the reporting hub. It
is updated in actual time to provide personalized, actionable insights at a glance.
There are several ways to access the detailed report view for a selected report.

Note: **Downloading reports to CSV files**

When downloading reports to CSV files from the Insights reporting hub, there is an agreement limit of 25,000 agreements for all IAM reports. This count includes your custom reports.

**Suggested workaround:** Adjust the report filters to run the report in batches until all the relevant agreements are downloaded. For example, run the report in batches according to varying date ranges. Once downloaded, combine the data into one CSV file.

This
topic focuses on the ways to access the detailed report view directly from the
Overview dashboard, as opposed to going to another
dashboard first.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Insights from the global header.

   The Insights hub opens and the Overview dashboard is displayed.

   ![A screenshot of the Insights hub that displays the Overview dashboard.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Insights-reporting-hub_Overview_cropped.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4OTAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.AGqimN929p19iN12pNihvXz8pIZppOF2hGD8KEgn7K4&_LANG=enus "A screenshot of the Insights hub that displays the Overview dashboard.")
3. Locate and select the report that you want to view. There are a number of ways to find the report that you want to view and download.
   - Use the search bar at the top of the page to search for the report you want to view. Enter a search term in the Find reports or dashboards field and select the relevant report from the dropdown.
   - Select the report from the Your Recents section. Your five most recently viewed reports are listed. Select the View all option to view an expanded list of your recently viewed reports.
   - Select the report from the Your Favorites section. Your "starred" dashboards and reports are displayed in this section. Select the View all option to view all of your favorite dashboards and reports.

     Tip: Be sure to "star" your frequently used dashboards and reports so they are added to the Your Favorites list.
   - Select a report name from the Suggested Reports (Beta) section.
   - Select an agreement insight from the Weekly Insights section.
   - Select the report from the Review your agreements section.
     - Select (click) the report name or select View Report from the More actions (three dot) menu.
     - Select (click) a segment in the bar or pie chart to open the detailed report view for the chart.
       The report is filtered to show the agreements that are
       associated with the selected segment.

   The detailed report view opens for the selected report.

   ![A screenshot of the detailed report view for a selected Overview dashboard report.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Overview_view-detailed-report_example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4OTAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.AGqimN929p19iN12pNihvXz8pIZppOF2hGD8KEgn7K4&_LANG=enus "A screenshot of the detailed report view for a selected Overview dashboard report.")

   Note: The dashboard charts can be viewed as regular bar charts or as stacked bar charts. Expand the More actions (three dots) menu and select the View as Stacked Bar Chart option:![A screenshot of the More actions (three dots) menu for a dashboard chart.that highlights the View as Stacked Bar Chart option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Dashboard-Reports_Dashboard-charts_View-as-stacked-bar-chart_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4OTAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.AGqimN929p19iN12pNihvXz8pIZppOF2hGD8KEgn7K4&_LANG=enus "A screenshot of the More actions (three dots) menu for a dashboard chart.that highlights the View as Stacked Bar Chart option.")
4. Verify that the report contains the relevant agreements and data. Use the filters to refine the agreement list as needed:

   ![A snippet of a detailed report view that highlights the available filters, including the All Filters option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Insights-reporting-hub_Overview_Detailed-report-view_Filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4OTAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.AGqimN929p19iN12pNihvXz8pIZppOF2hGD8KEgn7K4&_LANG=enus "A snippet of a detailed report view that highlights the available filters, including the All Filters option.")

   Tip: Filter options may vary from report to report.

   1. Apply the default filters that are displayed at the top of the report. Expand a filter, choose a value, and select the Apply option.
   2. Use the All Filters option to apply additional filters to the table. Select the All Filters option and apply the relevant filters.

      - Expand the filter, choose a value or enter a date, and select the Apply option. For example, to filter the report by agreement status, expand the Status filter and choose the applicable statuses.
      - Continue applying filters as needed.
      - Save the applied filters before closing the dialog box. Select the Save option.

      The All Filters dialog box closes and the report view is updated to reflect the applied filters.
5. Customize the columns in the table.

   Important: The Show or Hide Fields function only applies to the data that is displayed in the agreement list table. When downloading or exporting the report, all fields are included in the output file. This includes both custom and standard fields.

   ![A screenshot of the Show or Hide Fields dialog box that shows the columns you can choose to display in the agreements table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Overview-dashboard_Customize-columns.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4OTAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.AGqimN929p19iN12pNihvXz8pIZppOF2hGD8KEgn7K4&_LANG=enus "A screenshot of the Show or Hide Fields dialog box that shows the columns you can choose to display in the agreements table.")

   1. Select the Show or Hide Fields (toggle) icon.

      The Show or Hide Fields dialog box opens. These fields represent the columns in the table.
   2. Select the toggles to turn the associated fields on (blue) or off (white).
   3. Re-order the columns as needed. Hover the cursor over the
      Reorder (dots) icon for the column that you
      want to move and drag it to the appropriate location.
   4. Select the Save option.

   Tip: If the Status column is not displayed by default, use the Show or Hide Fields (toggle) option to add it to the table. From the Show or Hide Field dialog box, turn on the Status toggle and drag the field to the appropriate location. Be sure to Save your changes.

   The dialog box closes and the table is updated based on the selected customizations.
6. Download or save the report.

   ![A snippet of the report view that highlights the Report Actions (vertical ellipsis) menu used to download or save the report.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Report-view_Dropdown-expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4OTAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.AGqimN929p19iN12pNihvXz8pIZppOF2hGD8KEgn7K4&_LANG=enus "A snippet of the report view that highlights the Report Actions (vertical ellipsis) menu used to download or save the report.")

   Tip: The Save As option is not available for all reports.

   - Expand the More actions (three dots) menu in the top right and select the Download option.

     The report is downloaded to your local Downloads folder.
   - Expand the dropdown in the top right and select Save As. The Save Report dialog box opens

     - Enter a unique name in the Report name field. The field defaults to the name of the selected report.
     - Enter a description about the report's purpose in the Description field.
     - Select the Save option.The report is saved to the Custom tab on the Reports view.

     .
7. Set up one-time and recurring email reports for yourself and others. Select the Manage Notification (bell) icon and complete the Create subscription dialog box:

   ![A screenshot of the Create subscription dialog box that is used to subscribe to report notifications.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Overview_View-detailed-report_Create-subscription.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4OTAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.AGqimN929p19iN12pNihvXz8pIZppOF2hGD8KEgn7K4&_LANG=enus "A screenshot of the Create subscription dialog box that is used to subscribe to report notifications.")

   Note: When you create a subscription, you have the option to add individual users and groups. Once created, the subscription is displayed in the Manage report subscriptions dialog box for any individual users who were added. Currently, the subscription is not visible to users if they were added as part of a group.

   See the [Subscribe to Receive Report Notifications](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=wfw1734545937270.html&_LANG=enus "Learn how to set up one-time or recurring email reports for yourself or others to receive notifications about your obligations, renewals, or agreements.") topic to learn more about this feature.
8. Add the report to the Your Favorites list. Select the
   Favorite (star) icon located in the top right:

   ![A snippet of a detailed report view that highlights the Favorite (star) icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreements-dashboard_Report-details_Favorite-icon-selected_generic.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4OTAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.AGqimN929p19iN12pNihvXz8pIZppOF2hGD8KEgn7K4&_LANG=enus "A snippet of a detailed report view that highlights the Favorite (star) icon.")Your starred dashboards and reports are displayed in the Your
   Favorites section on the Overview
   dashboard.
9. Return to the Overview dashboard. Select Overview from the breadcrumb navigation at the top of the page:

   ![A snippet of the detailed report view that highlights the breadcrumb navigation.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Overview_View-detailed-report_Return-to-Overview_breadcrumb.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4OTAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.AGqimN929p19iN12pNihvXz8pIZppOF2hGD8KEgn7K4&_LANG=enus "A snippet of the detailed report view that highlights the breadcrumb navigation.")

   You are returned to the Overview dashboard.
10. Repeat this process to view, download, or save other Overview dashboard reports.

You have successfully reviewed, customized, and downloaded an Overview
dashboard report.
