# Obligations Dashboard

Read about the Obligations dashboard, including available reports and filter options.

 

## The Obligations dashboard

The Obligations dashboard provides a visual summary of your agreement obligations.

![A screenshot of the Obligations dashboard that shows the available charts and filter options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Obligations-Dashboard.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwMDIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.PmgSr4Y1zeA7a91Ac3OysLlCY-aitlVSs5tcPNLKG70&_LANG=enus "A screenshot of the Obligations dashboard that shows the available charts and filter options.")

These obligations include:

- Payment, renewal, and termination obligations surfaced with AI
- Other types of obligations that were manually created

## Filter options

The data reflected in the charts can be filtered to show relevant obligations. Filters are applied at the Obligations dashboard level and once applied, both charts update based on the selected filters. Available filters include:

![A snippet of the Obligations dashboard that shows the filter options, including Due Date, Obligation Type, Parties, and Obligation Owner.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Obligations-Dashboard_Filter-options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwMDIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.PmgSr4Y1zeA7a91Ac3OysLlCY-aitlVSs5tcPNLKG70&_LANG=enus "A snippet of the Obligations dashboard that shows the filter options, including Due Date, Obligation Type, Parties, and Obligation Owner.")

- Due Date: Use this filter to specify a time period by which to filter the obligation charts. Options include: Today, Next 7 Days, Next 14 Days, Next 30 Days, Next 6 Months, and Custom Date Range.
- Obligation Type: Use this filter to view obligations based on the type of obligation, such as Compliance, Confidentiality, Indemnification, and Payment.
- Parties: Use this filter to view obligations for a specific party and party role.

  Important: When using the Parties filter, you must also select a Party Role value. If you do not know which role to select, choose the All option. If no role is selected, the search returns agreements where the party is not the buyer or the seller.
- Obligation Owner: Use this filter to view obligations by the user who created the obligation. For obligations that were surfaced with AI, the agreement owner is the obligation owner.

Note: The detailed report view offers two additional filter options: Obligation Status and Agreement Name. Use these filters to refine the report to locate relevant obligations.

## Report limitations

When downloading reports to CSV files from the Insights reporting hub, there is an agreement limit of 25,000 agreements for all IAM reports. This count includes your custom reports.

**Suggested workaround:** Adjust the report filters to run the report in batches until all the relevant agreements are downloaded. For example, run the report in batches according to varying date ranges. Once downloaded, combine the data into one CSV file.

Note: The eSignature reports have a different set of limitations. See the [Using Reports](https://support.docusign.com/s/document-item?bundleId=cnu1642812012214&topicId=wou1578456440763.html) article to learn more about eSignature reports for senders and signers.

## Limitation: Obligations processing

Obligations are surfaced in English (U.S.) agreements only. The AI model that processes obligations is run from a U.S. server and can only process English agreements in this region.

If your language and region are not supported, manually create the obligations to include this information in dashboards, reports, and other insights.
