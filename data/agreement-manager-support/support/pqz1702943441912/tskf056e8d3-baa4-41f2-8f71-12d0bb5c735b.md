# Update Custom Dropdown Fields

Learn how to update the dropdown options for custom fields with the Dropdown field type.

Administrators and contract managers can update the custom fields in their account. For example, updating the options for custom dropdown fields.

Note: The Edit Details function is available for custom fields only. You cannot update the standard, Docusign-provided fields. This function is completed by users with Organize and manage all agreements access to Agreement Manager and the account's agreements.

Use the *Fields Manager* to update the options for a custom dropdown field. Access this function from the Agreement Manager homepage under the Settings (gear) menu.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Expand the Settings (gear) dropdown from the top right and select the Fields option:

   ![A screenshot of the Agreement Manager homepage that highlights the Fields option in the Settings (gear) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-types-manager_Settings-menu_Fields_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3MzYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.LDHMG4GexASxojP65XWHMklEDYexCCGaE4ch8yPMOvs&_LANG=enus "A screenshot of the Agreement Manager homepage that highlights the Fields option in the Settings (gear) menu.")

   The Manage Fields page opens and the Fields view is displayed. All the custom and standard fields in the account are displayed in the table.
5. Locate the custom dropdown field that you want to update.
   - Search for the relevant custom dropdown field.

     Enter a keyword in the Search fields... field. As you type the keyword, the table updates and the fields that match the search term are displayed.
   - Filter the table to locate the relevant custom field. Filter options are: Created By, Category, Field Type, and AI Status.

     Use the Created By and Field Type filters to view the custom dropdown fields in your account:

     - Expand the Created By filter, choose the User value, and select the Apply option.
     - Expand the Field Type filter, choose the Dropdown option, and select the Apply option.

     The table updates and the fields that matched the applied filters are displayed.
6. Expand the More actions (three dots) menu for the relevant field and select the Edit Details option:

   ![A snippet of the Fields table that highlights the Edit Details option for a user-created custom dropdown field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Type_Update-dropdown-field_Edit-Details.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3MzYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.LDHMG4GexASxojP65XWHMklEDYexCCGaE4ch8yPMOvs&_LANG=enus "A snippet of the Fields table that highlights the Edit Details option for a user-created custom dropdown field.")The Edit field dialog box opens and a warning (!) message is displayed. Any changes that you make to the field are reflected everywhere that the field is being used.

   You can update the Name, Options, and Category details for the selected custom field.

   Tip: The Field type and Agreement types fields are closed to edits. You cannot update these fields from this location. However, you can add the field to multiple agreement types from the *Agreement Types Manager*. See the [Add Fields to Multiple Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk37e570f8-2428-4d26-a78b-b1239aca35a5.html&_LANG=enus "Learn how to add fields to multiple agreement types from the Add a field dialog box. You can add fields to additional agreement types while you are updating a selected agreement type.") article to learn about adding fields to multiple agreement types.
7. Update the Options for the custom dropdown field as needed.

   ![A screenshot of the Edit field dialog box that highlights the Options text-entry box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Type_Edit-field-details_Update-Options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3MzYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.LDHMG4GexASxojP65XWHMklEDYexCCGaE4ch8yPMOvs&_LANG=enus "A screenshot of the Edit field dialog box that highlights the Options text-entry box.")

   1. Update the options for the custom dropdown field from the Options text-entry box. Separate each option with a semicolon (;).
   2. Select the Save option.

   The dialog box closes and the Agreement type details view is displayed for the selected type. The custom field is listed in the table and reflects the updated details.

   The dialog box closes and you are returned to the Manage Fields page. The
   custom dropdown field has been updated and the changes are reflected across the
   Intelligent Agreement Management (IAM) platform.
8. Return to the Agreement Manager home page. Select the Close (x) icon from the top right.

   You are returned to the Agreement Manager homepage. The Completed Documents view is displayed.

You have successfully updated the options for a custom dropdown field.
