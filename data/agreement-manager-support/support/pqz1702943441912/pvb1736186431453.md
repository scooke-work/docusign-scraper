# Share an Agreement Set

Learn how to share an agreement set with individual users and groups. This topic answers the question: How do I share groups of agreements with the appropriate people?

Contract managers create and manage sets to make sure the right people have access to relevant agreements. After you add agreements to the set, share the set with the appropriate users and groups.

These permissions provide varying access to the agreements being shared:

- Can Manage:: Users can edit attribute values, and delete, share, and remove access to the agreement. Users with this access can also add the agreement to agreement sets.
- Can Edit:: Users can edit attribute values and share and download the agreement.
- Can View:: Users can read and download the agreement.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens.
4. Expand the Settings (gear) menu from the top right and select the Sets option.

   The Agreement sets page opens. The All agreement sets view is displayed. The table displays sets that were shared with or created by the logged-in user.

   ![A snippet of the All agreement sets view that displays sets that were created by or shared with the logged-in user.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Sets-dialog_All-agreement-sets_New-Agreement-Set.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9._U92rzZFXvNx6HOfobN3RiGtFHQJERl-f0Pkc0YNKKY&_LANG=enus "A snippet of the All agreement sets view that displays sets that were created by or shared with the logged-in user.")
5. Locate the set that you want to share, expand the More actions (three dots) menu, and select Share:

   ![A snippet of the All agreement sets view that shows the Share option in the More actions menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Sets-dialog_All-agreement-sets_Share-option.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9._U92rzZFXvNx6HOfobN3RiGtFHQJERl-f0Pkc0YNKKY&_LANG=enus "A snippet of the All agreement sets view that shows the Share option in the More actions menu.")

   The Share with others dialog box opens.

   ![A screenshot of the Share with others dialog box that shows the users and groups that currently have access to the selected set.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Sets-dialog_All-agreement-sets_Share-with-others-dialog.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9._U92rzZFXvNx6HOfobN3RiGtFHQJERl-f0Pkc0YNKKY&_LANG=enus "A screenshot of the Share with others dialog box that shows the users and groups that currently have access to the selected set.")
6. Enter the name of the user or group with whom you want to share the set in the Add user or group search field.

   A type-ahead dropdown displays names matching the criteria that you entered.
7. Select a user or group from the dropdown list.

   The Add users view is displayed.

   ![A screenshot of the Add users view that shows the notification and access-level options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-sets_share-set_Add-users-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9._U92rzZFXvNx6HOfobN3RiGtFHQJERl-f0Pkc0YNKKY&_LANG=enus "A screenshot of the Add users view that shows the notification and access-level options.")

   Important:

   The Notify users by email option is selected by default. Leave this checkbox selected to notify users that an agreement set was shared with them.

   If you de-select this option, users are not notified of the shared agreement set.
8. Expand the Access dropdown and select the appropriate access level for agreements in the set.

   Choices are: Can Manage, Can Edit, and Can View.

   Note: To remove user or group access to the set:
   - Expand the Access dropdown for the user or group that you want to remove.
   - Select Remove.
   - Select Save.

   The user or group name is removed from the People with access list.
9. Select Share.

   The dialog box closes and you are returned to the Agreement sets page. The agreement set has been shared with the selected users or groups.
10. Select Completed Documents from the breadcrumb navigation at the top of the page.

    You are returned to the Agreement Manager homepage.

You have successfully shared an agreement set with selected users or groups, including setting an acces level.
