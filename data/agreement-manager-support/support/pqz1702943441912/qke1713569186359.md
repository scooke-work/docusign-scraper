# Download the Upcoming Renewals Report

Learn how to locate upcoming renewals from the Renewals dashboard, including viewing and downloading agreements. This topic answers the question: How do I find my upcoming renewals in Agreement Manager?

The Renewals dashboard contains charts about agreements in Agreement Manager. The charts reflect the agreements that match any applied filters. Use this dashboard to locate upcoming renewals, including viewing and downloading an agreements list.

Important: The dashboard refreshes every 15 minutes to account for new agreements or agreements with manually updated renewal-related attribute values. The refresh is designed to ensure that all agreements are represented in the dashboard.

**Access Renewals dashboard:**

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Insights from the global header.

   The Insights hub opens and the Overview dashboard is displayed.
3. Select Renewals from the navigation pane on the left.

   The Renewals dashboard view opens.

**Apply filters to the dashboard:**

4. Filter the Renewals dashboard to locate upcoming renewals:

   ![A snippet of the Renewals dashboard that highlights the dashboard filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Renewals-dashboard_Filter-pills.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WaqII30O9-n6EF8KvEA5d1g3j9sBZ8RlElUUrg3FZdw&_LANG=enus "A snippet of the Renewals dashboard that highlights the dashboard filters.")

   Filters include: Parties, Renewal Notice Date, Renewal Owner, Renewal Type, and Status.

   1. Expand the filter that you want to apply.
   2. Select or enter the filter value.
   3. Select the Apply option.

   Important: When using the Parties filter, you must also select a Party Role value. If you do not know which role to select, choose the All option. If no role is selected, the search returns agreements where the party is not the buyer or the seller.

   Tip: Use the Filters option to access additional filters:![A screenshot of the Filters dialog box that shows additional filters that you can apply to the agreement list.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_All-Filters-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WaqII30O9-n6EF8KvEA5d1g3j9sBZ8RlElUUrg3FZdw&_LANG=enus "A screenshot of the Filters dialog box that shows additional filters that you can apply to the agreement list.")

   The charts are updated to reflect the filters that you applied.

**View the 'Upcoming renewals' chart:**

The Upcoming renewals chart provides an aggregate count of agreements with upcoming renewals. These agreements are grouped by Renewal Notice Date and match any applied filters, such as the Renewal Type.

5. Review the Upcoming renewals chart. For example, view the number of agreements that are represented in a particular chart segment. Hover your cursor over a segment in the chart.

   A hover message shows the break-down of the documents that are represented in the segment:

   ![A snippet of the Upcoming renewals dashboard chart that shows the hover message for a selected bar.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Renewals-dashboard_Upcoming-renewals_hover-message.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WaqII30O9-n6EF8KvEA5d1g3j9sBZ8RlElUUrg3FZdw&_LANG=enus "A snippet of the Upcoming renewals dashboard chart that shows the hover message for a selected bar.")

   Tip: Select (click) a bar to view the agreements that are represented in that part of the chart. The detailed report view opens. The chart and agreement list are filtered to show the agreements that are reflected in the selected bar.

**Access the detailed report view:**

6. Access the detailed report view for the Upcoming renewals chart:

   ![A screenshot of the Upcoming renewals chart that shows the View Report option in the More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Renewals_Renewals-Dashboard_Chart-ellipses-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WaqII30O9-n6EF8KvEA5d1g3j9sBZ8RlElUUrg3FZdw&_LANG=enus "A screenshot of the Upcoming renewals chart that shows the View Report option in the More actions (three dots) menu.")

   - Select the chart heading.
   - Hover the cursor over the chart, expand the More actions (three dots) menu and select the View Report option.

   The detailed report view opens for the Upcoming renewals chart. The agreements represented in the chart are listed in the table.

**Download an agreement:**

7. Download an agreement to your local Downloads folder:

   ![A snippet of the Renewals dashboard that highlights the Download option in the Actions menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Renewals_Renewals-Dashboard_table-ellipses-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WaqII30O9-n6EF8KvEA5d1g3j9sBZ8RlElUUrg3FZdw&_LANG=enus "A snippet of the Renewals dashboard that highlights the Download option in the Actions menu.")

   1. Expand the More actions (three dots) menu for the agreement that you want to download.
   2. Select the Download option.

   The agreement is downloaded to your Downloads folder.
8. View the Agreement preview page for a selected agreement:
   1. Select (click) an agreement from the table.

      The Agreement preview page opens for the selected agreement. The Details pane is displayed by default.
   2. Review the details, obligations, hierarchies, and agreement sets for the agreement.
   3. Return to the previous page. Select the Back arrow from the top-left corner.

   You are returned to the Upcoming renewals report.

**Customize and download the report:**

9. Customize the information that is displayed in the table:

   ![A screenshot of the Show or Hide Fields dialog box that shows the columns you can choose to display in the agreements table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Overview-dashboard_Customize-columns.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WaqII30O9-n6EF8KvEA5d1g3j9sBZ8RlElUUrg3FZdw&_LANG=enus "A screenshot of the Show or Hide Fields dialog box that shows the columns you can choose to display in the agreements table.")

   1. Select the Show or Hide Fields (toggle) icon.

      The Show or Hide Fields dialog box opens. These fields represent the columns in the table.
   2. Select the toggles to turn the associated field on (blue) or off (white).
   3. Re-order the columns as needed. Hover the cursor over the Reorder (dots) icon for the field that you want to move and drag it to the appropriate location.
   4. Select the Save option.

   Continue applying filters and customizing the table data until the report reflects relevant data.
10. Download the report. Expand the More actions (three dots) menu in the top right and select the Download option:

    ![A snippet of a detailed report view that shows the Download option in the expanded dropdown.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Renewals_Detailed-report_Download.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WaqII30O9-n6EF8KvEA5d1g3j9sBZ8RlElUUrg3FZdw&_LANG=enus "A snippet of a detailed report view that shows the Download option in the expanded dropdown.")

    The associated report is downloaded to your local Downloads folder.

**Subscribe to the report:**

11. Set up one-time and recurring email reports for yourself and others. Select the Manage Notification (bell) icon and complete the Create subscription dialog box:

    ![A screenshot of the Create subscription dialog box that is used to subscribe to report notifications.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Renewals-dashboard_Report-view_Create-subscriptions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WaqII30O9-n6EF8KvEA5d1g3j9sBZ8RlElUUrg3FZdw&_LANG=enus "A screenshot of the Create subscription dialog box that is used to subscribe to report notifications.")

    Note: When you create a subscription, you have the option to add individual users and groups. Once created, the subscription is displayed in the Manage report subscriptions dialog box for any individual users who were added. Currently, the subscription is not visible to users if they were added as part of a group.

    Review the [Subscribe to Receive Report Notifications](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=wfw1734545937270.html&_LANG=enus "Learn how to set up one-time or recurring email reports for yourself or others to receive notifications about your obligations, renewals, or agreements.") article to learn more about this feature.

**Add the report to 'Your Favorites':**

12. Add the report to Your Favorites list. Select the Favorite (star) icon located in the top-right:

    ![A snippet of the Upcoming renewals report view that highlights the Favorite (star) icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Renewals-dashboard_Upcoming-renewals-report_Mark-favorite_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WaqII30O9-n6EF8KvEA5d1g3j9sBZ8RlElUUrg3FZdw&_LANG=enus "A snippet of the Upcoming renewals report view that highlights the Favorite (star) icon.")Your starred dashboards and reports are displayed in the Your Favorites section on the Overview dashboard.

**Return to the Renewals dashboard:**

13. Return to the Renewals dashboard. Select the Renewals option from the breadcrumb navigation:

    ![A snippet of the Upcoming renewals report that highlights the breadcrumb navigation.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Renewals-dashboard_Upcoming-renewals-report_Renewals-breadcrumb_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WaqII30O9-n6EF8KvEA5d1g3j9sBZ8RlElUUrg3FZdw&_LANG=enus "A snippet of the Upcoming renewals report that highlights the breadcrumb navigation.")

    You are returned to the Renewals dashboard.

You have successfully reviewed and downloaded the Upcoming renewals report.
