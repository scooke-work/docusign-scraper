# View the Activity Log for an Agreement

Learn how to access the Activity Log to view agreement-level events for a selected agreement.

The agreement-level Activity Log records and displays user-performed events for individual agreements. Recorded events include access changes, attribute updates, and agreement uploads, downloads, and views. For each event the date, user, action, item, and details are recorded and displayed. Events can take up to 15 minutes to display in the log.

Note: Your access to the agreement-level Activity Log is dependent on your access to the associated document. You need Can Manage agreement access to view the corresponding Activity Log:![A screenshot of the document-level access permissions.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Share-an-agreement_available-access-permissions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5MzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.XCjHvogBh6WH9W7YkUE-QXpHkLC9SuHlxT1-HQgHSI0&_LANG=enus "A screenshot of the document-level access permissions.")

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Enter search criteria and apply filters to locate the agreement that you want to review.

   ![A snippet of the Completed Documents view that highlights the search field and available filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Navigator_Completed-documents_search-field-and-filters-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5MzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.XCjHvogBh6WH9W7YkUE-QXpHkLC9SuHlxT1-HQgHSI0&_LANG=enus "A snippet of the Completed Documents view that highlights the search field and available filters.")

   1. Enter a keyword or search string in the search field and select Enter.
   2. Filter the agreement list table. Expand the Filters option, select the relevant filter, choose a value, and select Apply.

   The table displays agreements matching the search criteria and filters that you applied.
5. Expand the More actions (three dots) menu for the agreement that you want to review and select Activity Log::

   ![A snippet of the Completed page that highlights the Activity Log option in the Actions menu for an agreement.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Activity-Log_access-from-Actions-menuCompleted-documents.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5MzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.XCjHvogBh6WH9W7YkUE-QXpHkLC9SuHlxT1-HQgHSI0&_LANG=enus "A snippet of the Completed page that highlights the Activity Log option in the Actions menu for an agreement.")

   The Activity Log opens for the selected agreement and a list of agreement-level events are displayed. The events are displayed in reverse chronological order with the most recent at the top.

   The table displays 200 rows of recorded events. If additional events exist, you are prompted to download the table data to a CSV to view the full list of events.

   ![A screenshot of the Activity Log that shows recorded events and data fields for the selected agreement.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Activity-Log-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5MzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.XCjHvogBh6WH9W7YkUE-QXpHkLC9SuHlxT1-HQgHSI0&_LANG=enus "A screenshot of the Activity Log that shows recorded events and data fields for the selected agreement.")
6. Select the sorting arrows in the column headers to sort the table by DATE, USER, ACTION, ITEM, and DETAILS.
7. Review the table data to locate the relevant agreement-level event information. Details related to these user-performed events are recorded in the log:

   - Access changes
   - Agreement downloads
   - Attribute updates for select standard fields
   - Agreement views
   - Document uploads

   Note: The DATE column displays the date and time that the event was performed. The time zone is localized based on the account's time zone preference.
8. Select the Download CSV option.

   The CSV file is downloaded to your local Downloads folder.
9. Return to the Agreement Manager homepage. Select the Close (x) icon from the top-right corner.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.

**View the Activity Log from the Agreement preview page:**

The agreement-level Activity Log can also be viewed from the Agreement preview page. Continue from the Completed page.

10. Select the agreement that you want to review from the agreement list.

    The Agreement preview page opens for the selected agreement.
11. Expand the dropdown in the top-right and select the Activity Log option:

    ![A snippet of the Agreement preview page that shows the Activity Log option in the dropdown.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Activity-Log_access-from-Agreement-Preview_Downloads-dropdown.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5MzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.XCjHvogBh6WH9W7YkUE-QXpHkLC9SuHlxT1-HQgHSI0&_LANG=enus "A snippet of the Agreement preview page that shows the Activity Log option in the dropdown.")

    The Activity Log opens for the selected agreement and a list of agreement-level events are displayed.
12. Review the table data to locate the relevant agreement-level event information as needed. You can also download the audit events to a CSV.
13. Select the Close (X) icon from the top-right corner.

    The Activity Log closes and you are returned to the Agreement preview page.
14. Return to the Agreement Manager homepage. Select the Close (x) icon from the top-left corner.

    You are returned to the Agreement Manager homepage. The Completed Documents view is displayed.
