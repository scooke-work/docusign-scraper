# Agreement Manager Activity Logs

Read about the account- and agreement-level activity logs, including the types of events that are captured in the log.

Audits are essential for:

- Minimizing errors
- Detecting security breaches
- Ensuring compliance
- Troubleshooting issues

Every agreement has an Activity Log that captures and logs user-performed events. Agreement Manager provides two types of activity logs:

- **Account-level:** provides a centralized location to view logged events for all agreements in an account.
- **Agreement-level:** displays logged events for an individual agreement.

Tip: **More about the Activity Log**

- The events are displayed in reverse chronological order with the most recent at the top.
- The time zone is localized based on the account's time zone preference.
- The table displays a maximum of 200 recorded events at a time.
- Events can take up to 15 minutes to display in the log.

## Recorded events

The Activity Log records and displays agreement-related events that were performed by a user, including:

- Access updates (agreement and agreement set sharing)
- Agreement downloads
- Agreement set operations (set creation and updates)
- Agreement views

  Tip: Agreement views are recorded on a per-session basis. If a user views the document multiple times, a single view is recorded in the log. If the file changes, a new session is initiated and a second "view" event is recorded.
- Attribute updates for select standard fields

  Tip: See the [Extracted Agreement Insights and Definitions](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=uwk1702945852405.html&_LANG=enus "Look up the names and definitions for the attributes and values that are surfaced in Agreement Manager agreements with AI. This topic answers the question: What data is being extracted from my Agreement Manager agreements?") topic to learn more about agreement attributes.
- Document uploads

These user-performed events are recorded the Activity Log at the account- and agreement-levels.

The account-level Activity Log provides a centralized location to view aggregate, user-performed events across all the agreements in an account.

Important: You must have the Organize and manage all agreements permission to access the account-level Activity Log. An administrator grants permissions from the user or group profile pages located under the Admin tab.

## Agreement-level activity log

The agreement-level Activity Log records and displays user-performed events for a specific agreement.

![A screenshot of the Activity Log that shows agreement-level events that were performed by a user.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Activity-Log-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5MjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.6o--FfHnwvnvgeyq4e9ghUh-H-QbK6HSrgHOcBA_-CI&_LANG=enus)

The table displays 200 rows of recorded events. If additional events exist, you are prompted to download the table data to a CSV to view the full list of events.

The columns in the table represent the data fields that are captured for each event, including:

- Date
- User
- Action
- Item
- Details

Important: The actual event time stamp is localized in the system, however, the dates are shown in the user interface (UI) in Coordinated Universal Time (UTC).

There are two entry points for the agreement-level Activity Log:

- From the More actions (three dots) menu located in the agreement list table (Completed Documents view):![A snippet of the agreement list table that shows the Activity Log option in the More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Activity-Log_access-from-Actions-menuCompleted-documents.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5MjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.6o--FfHnwvnvgeyq4e9ghUh-H-QbK6HSrgHOcBA_-CI&_LANG=enus)
- From the dropdown located in the top-right corner of the Agreement preview page:![A snippet of the Agreement preview page that shows the Activity Log option in the expanded dropdown.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Activity-Log_access-from-Agreement-Preview_Downloads-dropdown.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5MjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.6o--FfHnwvnvgeyq4e9ghUh-H-QbK6HSrgHOcBA_-CI&_LANG=enus)

## Logged events, pagination, and downloads

The Activity Log displays pagination for the first 200 logged events in the table. If the table contains more than 200 events, a message displays and you are prompted to download the CSV file to view all the events:![A screenshot of the dialog box that prompts you to download the activity log to a CSV.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Activity-Log_Prompt-to-download-to-csv.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5MjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.6o--FfHnwvnvgeyq4e9ghUh-H-QbK6HSrgHOcBA_-CI&_LANG=enus)

Note: When using the Download CSV option, there is a 5,000 row limitation for downloading logged events.
