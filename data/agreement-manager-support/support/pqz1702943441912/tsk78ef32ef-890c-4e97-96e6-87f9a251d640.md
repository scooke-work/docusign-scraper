# Star Your Favorite Reports and Dashboards

Learn how to add your frequently used dashboards and reports to the Your Favorites section on the Overview dashboard.

The Your Favorites section gives you quick access to your personalized list of starred dashboards and reports. Select the Favorite (star) icon to "star" your frequently accessed dashboards and reports. This icon is available for custom and standard dashboards and reports.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Insights from the global header.

   The Insights hub opens and the Overview dashboard is displayed.

**Star your favorite dashboards:**

3. Locate the dashboards that you want to add to the Your Favorites list:
   - Select a standard dashboard from the navigation pane on the left. The standard dashboards include: Agreements, Obligations, and Renewals.
   - Select a custom dashboard from the navigation pane on the left. Expand the Custom Dashboards group from the navigation pane and select a custom dashboard.
   - Select a standard or custom dashboard from the Overview dashboard. Perform a keyword search to locate the relevant dashboard. Enter a keyword in the search field at the top of the page and select a value from the type-ahead dropdown. Results are grouped by Reports and Dashboards.

   The selected dashboard opens and the Favorite (star) icon is displayed
   in the top right.
4. Select the Favorite (star) icon:

   ![A snippet of a selected dashboard that highlights the Favorite (star) icon in the top right corner.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Overview_star-favorite-dashboard_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4OTMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.s425S7HzdUhKyHuze7u1PhwlXs3H-UGqd4lKTFtrKlg&_LANG=enus "A snippet of a selected dashboard that highlights the Favorite (star) icon in the top right corner.")The Favorite (star) icon turns blue, indicating that the dashboard is "starred".

   Tip:

   Select the Favorite (star) icon again to remove the dashboard from the Your Favorites list.

**Star your favorite reports:**

5. Locate the reports that you want to add to the Your Favorites list:
   - Select a report from the Overview dashboard:

     - Enter a keyword in the search field at the top of the dashboard and select the relevant report from the dropdown.
     - Select a report from the Your Recents section. This section provides a list of your most recently viewed reports.
     - Select a dashboard chart to view the corresponding report.
     - Select a chart segment to view the corresponding report, filtered for the selected segment.
   - Choose a report from another dashboard. Select a dashboard from the navigation pane on the
     left. From the dashboard page, drill down to the detailed report view.
     Expand the More actions menu (ellipsis) for
     the relevant dashboard chart and select View
     Report.

   The detailed report view opens for the selected chart and the Favorite
   (star) icon is displayed in the top right.
6. Select the Favorite (star) icon:

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Overview_Star-favorite-report_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4OTMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.s425S7HzdUhKyHuze7u1PhwlXs3H-UGqd4lKTFtrKlg&_LANG=enus)

   The Favorite (star) icon turns blue, indicating that the dashboard is "starred".

   Tip:

   Select the Favorite (star) icon again to remove the report from the Your Favorites list.

You have successfully "starred" your favorite dashboards and reports.
