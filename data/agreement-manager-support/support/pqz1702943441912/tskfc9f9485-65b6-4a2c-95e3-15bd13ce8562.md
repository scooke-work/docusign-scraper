# Create Custom Reports from Suggested Reports (Beta)

Learn how to create a custom report from one of the reports in the Suggested Reports (Beta) section on the Overview dashboard.

The Suggested Reports (Beta) section on the Overview dashboard provides AI-generated custom report notifications. These suggestions are auto-generated based on the custom fields in your account. When fields (custom or standard) are located in at least five agreements, they are included when generating custom report suggestions.

This topic reviews the steps to create a custom report from one of the reports in the Suggested Reports section.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Insights from the global header.

   The Insights hub opens and the Overview dashboard is displayed.
3. Go to the Suggested Reports section and review all the custom report recommendations:

   ![A snippet of the Overview dashboard that shows the Suggested Reports (Beta) section, including the scroll bar to view all the recommendations.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Create-custom-report_Suggested-Reports_scrollbar_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNTMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.LXB6P0f6pC-7hJ8gSkqIBOj_6_GAa-yO_ZQW-FQBVcA&_LANG=enus "A snippet of the Overview dashboard that shows the Suggested Reports (Beta) section, including the scroll bar to view all the recommendations.")
4. Select the report heading for the report that you want to create.

   The detailed report view opens for the selected report.
5. Apply filters to customize the report further:

   ![A snippet of the detailed report view for a suggested custom report that highlights the filter options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Create-custom-report_from-Suggested-Reports_Apply-filters_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNTMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.LXB6P0f6pC-7hJ8gSkqIBOj_6_GAa-yO_ZQW-FQBVcA&_LANG=enus "A snippet of the detailed report view for a suggested custom report that highlights the filter options.")

   For example, sort the report by Sets, Parties, or Agreement Type. Use the All Filters option to apply additional filters to the report.

   1. Expand the filter that you wan to apply.
   2. Choose a filter value or select a date range.
   3. Select the Apply option.
6. Customize the columns in the table:

   Important: The Customize columns function only applies to the data that is displayed in the agreement list table. When downloading or exporting the report, all fields are included in the output file. This includes both custom and standard fields.

   ![A screenshot of the Customize columns dialog box that shows the columns you can choose to display in the agreements table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Overview-dashboard_Customize-columns.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNTMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.LXB6P0f6pC-7hJ8gSkqIBOj_6_GAa-yO_ZQW-FQBVcA&_LANG=enus "A screenshot of the Customize columns dialog box that shows the columns you can choose to display in the agreements table.")

   1. Select the Customize columns (toggle) icon.

      The Customize columns dialog box opens. These fields represent the columns in the table.
   2. Select the toggles to turn the associated field on (blue) or off (white).
   3. Re-order the columns as needed. Hover the cursor over the Reorder (dots) icon for the column that you want to move and drag it to the appropriate location.
   4. Select the Save option.

   Tip: If the Status column is not displayed by default, use the Show or Hide Fields (toggle) option to add it to the table. From the Show or Hide Field dialog box, turn on the Status toggle and drag the field to the appropriate location. Be sure to Save your changes.

   Continue to apply filters and customize the table until the report reflects the relevant data.
7. Save the report. Expand the More actions (three dots) menu in the top right and select the Save option:

   ![A snippet of a detailed report view for a suggested custom report that highlights the Save option in the More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Create-custom-report_from-Suggested-Reports_Save_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNTMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.LXB6P0f6pC-7hJ8gSkqIBOj_6_GAa-yO_ZQW-FQBVcA&_LANG=enus "A snippet of a detailed report view for a suggested custom report that highlights the Save option in the More actions (three dots) menu.")

   The Save report dialog box opens. The Report name
   and Description fields default based on the selected
   suggestion.

   ![A screenshot of th Save report dialog box that shows the custom report name and description.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Create-custom-report_from-Suggested-Reports_Save-report-modal.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNTMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.LXB6P0f6pC-7hJ8gSkqIBOj_6_GAa-yO_ZQW-FQBVcA&_LANG=enus "A screenshot of th Save report dialog box that shows the custom report name and description.")
8. Update the Report name field or accept the default value. This field defaults to the name of the suggested report.
9. Update the Description field or accept the default
   value.
10. Select the Save option.

    The dialog box closes and the custom report is saved. Your new report is available from the Custom tab under the Reports view:

    ![A snippet of the Reports view that shows the Custom tab, including custom reports and the View option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Create-custom-report_from-Suggested-Reports_Report-saved-to-Custom-tab.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNTMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.LXB6P0f6pC-7hJ8gSkqIBOj_6_GAa-yO_ZQW-FQBVcA&_LANG=enus "A snippet of the Reports view that shows the Custom tab, including custom reports and the View option.")

    Custom reports can be updated or deleted at any time from the report details page. See the [View and Update Your Custom Reports](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=yuf1727801071281.html&_LANG=enus "This topic answers the question: How can I review and update my custom reports?") topic to learn about editing options for your custom reports.

You have successfully created a custom report from a recommendation in the Suggested Reports section on the Overview dashboard.
