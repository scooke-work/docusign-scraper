# Job Monitoring for Agreement Uploads and Processing

Read about the job monitoring hub that is used to track the progress for agreement uploads and processing jobs.

The *job monitoring hub* gives you access to job details for agreement uploads and processing jobs. It provides a centralized location to view and track the status of these jobs.

Access this function from the Settings (gear) menu on the Agreement Manager homepage: ![A screenshot of the Settings (gear) dropdown menu that highlights the Uploads and Processing Status option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-view_Actions_Monitor-All-Jobs-option.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.02iiSnWAWIKlRX-ZL4IxzT6tw2FatYLV5AbPlhXuzho&_LANG=enus "A screenshot of the Settings (gear) dropdown menu that highlights the Uploads and Processing Status option.")

Once selected, the All Jobs page opens. The job details and status information for agreement upload and processing jobs are displayed.

Use the Uploaded and Processing tabs to switch between agreement upload jobs and processing jobs.

![A screenshot of the All jobs view that shows the job details and status information for uploaded agreements.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-view_Monitor-All-Jobs_All-JobsUploaded-tab.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.02iiSnWAWIKlRX-ZL4IxzT6tw2FatYLV5AbPlhXuzho&_LANG=enus "A screenshot of the All jobs view that shows the job details and status information for uploaded agreements.")

## Agreement upload jobs

The Uploaded tab displays the job details and status for agreement uploads and imports:

![A snippet of the All jobs view that highlights the Uploaded tab and shows job details and status information for upload jobs.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Monitor-All-Jobs_Uploaded-tab_Uploaded-agreements.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.02iiSnWAWIKlRX-ZL4IxzT6tw2FatYLV5AbPlhXuzho&_LANG=enus "A snippet of the All jobs view that highlights the Uploaded tab and shows job details and status information for upload jobs.")

The Uploaded Agreements table, located on the Uploaded tab, displays job details for all uploaded agreements, including:

- Created date
- Job name
- Source
- Files (number of documents)
- Status

Use the filters to locate job details for relevant upload jobs. You can filter the Uploaded Agreements table by:

![A snippet of the Uploaded Agreements table that highlights the Created date, Status, and Source filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Monitor-uploads_All-jobs_Filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.02iiSnWAWIKlRX-ZL4IxzT6tw2FatYLV5AbPlhXuzho&_LANG=enus "A snippet of the Uploaded Agreements table that highlights the Created date, Status, and Source filters.")

- Created date
- Status
- Source

Tip: Use the sorting arrows located in the column headers to sort the Uploaded Agreements table by the corresponding column.

After locating the relevant job, expand the Actions dropdown and choose from the available actions:

![A snippet of the All jobs view that shows the additional actions that are available for each job.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-view_Monitor-All-Jobs_Uploaded-agreements_Actions-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.02iiSnWAWIKlRX-ZL4IxzT6tw2FatYLV5AbPlhXuzho&_LANG=enus "A snippet of the All jobs view that shows the additional actions that are available for each job.")

- Browse Files: displays a list of agreements that were uploaded as a result of the selected job.
- View Details: displays upload status results, including successful uploads, upload exceptions, and assigned permissions.
- Export Agreement Data exports a pre-formatted spreadsheet to bulk update the metadata for the agreements in the selected job.
- Delete: removes the upload details for the selected job. Upload details are removed from the Uploaded Agreements table. Once deleted, the upload details cannot be recovered.

  Note: The Delete function only removes the upload details for a selected job. The uploaded files are not deleted.

## Job monitoring triggers

The job monitoring hub tracks the progress and status of agreement uploads and processing jobs. Each type of job has its own triggers that will kick off a job:

- Uploaded: jobs are initiated when agreements are added to Agreement Manager from a computer or third-party storage provider. This includes single agreement and bulk uploads.
- Processing: jobs are initiated when historical agreements are processed for new attributes or fields. For example:
  - An agreement's type is updated and processing is necessary to surface the new attributes. This applies to single and bulk agreement type changes.
  - AI detection is enabled for a new custom agreement type. Agreements are processed for the new type.
  - AI detection is enabled for a new custom fields. Agreements are processed to surface the new data points.
