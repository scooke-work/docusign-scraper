# Filter the Agreement List Table by Status

Learn how to filter the agreement list table by agreement status.

Agreement statuses provide a quick look at where an agreement is within its life cycle.
These statuses are calculated based on an agreement's extracted Effective
Date, Expiration Date, and Renewal
Type.

Note: A Renewal Type value is not required for calculating an agreement's status. However, if an extracted value exists, it is taken into consideration when calculating the status. For example, if an agreement has an Expiration Date and the Renewal Type is Evergreen, the agreement is always Active.

Agreement statuses are re-calculated and updated daily, based on the current
date.

Refine the agreement list table to view agreements based on selected
statuses. For example: Active,
Inactive, Expiring Soon,
Renewing Soon, and Effective
Soon.

The Status column and filter are associated with the agreement list table and can be found on the:

- Completed page
- Agreements page in the party hub
- Detailed report view for a selected dashboard or report

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens. The Inbox view is displayed by default.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. Perform a search and apply filters to locate the relevant agreements.
4. Apply the Status filter to the agreement list table:

   ![A snippet of the Completed view that shows the Status filter and filter options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-statuses_Completed-view_Status-filter_expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY1NzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Y2e15kI3VAdfXklAm2cNtmyNFT5LO1VZK7SfLbIMxSM&_LANG=enus "A snippet of the Completed view that shows the Status filter and filter options.")

   1. Expand (click) the Filters option and select the Status filter.

      The Status filter opens.
   2. Choose the relevant agreement statuses.

      You can select more than one status. For example, select two statuses to view agreements for either status.
   3. Select the Apply option.

   The agreement list updates and the agreements that match the selected statuses are displayed.

You have successfully filtered the agreement list table to view agreements with a specific status.
