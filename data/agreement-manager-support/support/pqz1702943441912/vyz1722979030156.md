# Share Agreements with Others

This topic answers the question: How do I share agreements with others? Learn how to
share an agreement with other users or groups, including assigning access levels for each
recipient.

The Share function provides an easy way for you to share an
agreement with other users and groups. This function also includes a step to assign
agreement access levels for each recipient.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign
   Agreements page opens.
3. Select Completed from the
   AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Perform a keyword search or use the available filters to locate the agreement
   that you want to share.
5. Expand the More actions (three dots) menu
   for the agreement that you want to share and select
   Share:

   ![A snippet of the Completed Documents view that shows the Share option in the expanded More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_vertical-ellipses_Share.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2ODUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.dxl3vDo-ElhyZhdRNyOPWKvpT210gMj9DrP_1fOAZnY&_LANG=enus "A snippet of the Completed Documents view that shows the Share option in the expanded More actions (three dots) menu.")

   The Share with others dialog box opens.

   ![A screenshot of the Share with others dialog box that shows the people with access to the agreement.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Share-with-others-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2ODUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.dxl3vDo-ElhyZhdRNyOPWKvpT210gMj9DrP_1fOAZnY&_LANG=enus "A screenshot of the Share with others dialog box that shows the people with access to the agreement.")
6. Enter the name of the user or group that you want to share the agreement with
   in the Add user or group search field.

   A type-ahead dropdown displays names matching the criteria that you
   entered.
7. Select a user or group from the dropdown list.

   The Add users view is displayed.

   ![A screenshot of the Add users view that shows the Notify users by email and Access options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Share-an-agreement_Add-users-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2ODUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.dxl3vDo-ElhyZhdRNyOPWKvpT210gMj9DrP_1fOAZnY&_LANG=enus "A screenshot of the Add users view that shows the Notify users by email and Access options.")

   Note:

   The Notify users by email option is selected by
   default. Leave this checkbox selected to notify users that an agreement
   was shared with them.

   If you de-select this option, users are not notified of the shared
   agreement.
8. Expand the Access dropdown and select the appropriate
   access level.

   Choices are: Can Manage, Can
   Edit, and Can View.
9. Select the Copy link option to obtain a link to the
   agreement that can be shared with others. The link is copied to your
   clipboard.

   This link can be pasted into a message or email and shared with other users.
   Keep in mind, the user must have access to the agreement to view the
   link.
10. Select Share.

    The dialog box closes. The agreement is shared with the selected users and displayed in
    their agreement list table.

    Note: You can also share an agreement from the Agreement
    preview page. Expand the dropdown in the top-right corner and
    select Share:![A snippet of the Agreement preview page that shows the Share option located in the dropdown in the top-right corner.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-preview-page_expanded-Download-button.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2ODUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.dxl3vDo-ElhyZhdRNyOPWKvpT210gMj9DrP_1fOAZnY&_LANG=enus "A snippet of the Agreement preview page that shows the Share option located in the dropdown in the top-right corner.")

    The Share with others dialog box opens.
    Complete the fields in the dialog box to share an agreement with other
    users or groups.

You have successfully shared an agreement with key
stakeholders.
