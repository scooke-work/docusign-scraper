# Delete Unused Custom Agreement Types

Learn how to delete custom agreement types that are no longer relevant.

Organizations can create custom agreement types to support their use cases. However, when custom
agreement types are no longer relevant, they can be deleted. Once an agreement type
is deleted, it cannot be recovered.

The documents that are currently associated with the agreement type remain
intact. However, the agreement type is no longer surfaced in new documents.

Tip: Remove unused custom agreement types if your organization is close to reaching the 200 custom agreement type limit per account. Once the custom agreement types are deleted, they no longer count towards the limit.

Delete your custom unused agreement types from the Agreement Types option in the Settings (gear) dropdown. This task requires the Organize and manage all agreements permission and is typically completed by administrators and contract managers.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Completed Documents view opens. A list of documents to which the logged-in user has access is displayed in the table.
4. Expand the Settings (gear) dropdown from the top right and select the Agreement Types option:

   ![A screenshot of the Agreement Manager homepage that highlights the Agreement Types option in the Settings (gear) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-types-manager_Settings-menu_Agreement-Types_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1MjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.xE2xH63gOy2KmKNc_a4fdsuvm3E8vUPogj-JDvomT7A&_LANG=enus "A screenshot of the Agreement Manager homepage that highlights the Agreement Types option in the Settings (gear) menu.")

   The Manage Agreement Types page opens. A list of the account's standard and custom agreement types is displayed in the Agreement Types table.
5. Locate the custom agreement type that you want to delete. Only user-created, custom agreement types can be deleted.

   ![A snippet of the Manage Agreement Types page that shows the selected Created By value, the User option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Types_Created-By-filter_User-selected.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1MjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.xE2xH63gOy2KmKNc_a4fdsuvm3E8vUPogj-JDvomT7A&_LANG=enus "A snippet of the Manage Agreement Types page that shows the selected Created By value, the User option.")

   1. Expand (click) the Created By filter and select the User option.
   2. Select the Apply option.

      The Agreement Types table updates and all the account's custom agreement types are displayed.
   3. Refine the results list as needed. For example, apply the Category and AI Status filters or perform a keyword search.
6. Expand the More actions (three dots) menu for the relevant custom agreement type and select the Delete option:

   ![A snippet of the Agreement Types view that highlights the Delete option in the More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-type-management_Delete-custom-agreement-type_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1MjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.xE2xH63gOy2KmKNc_a4fdsuvm3E8vUPogj-JDvomT7A&_LANG=enus "A snippet of the Agreement Types view that highlights the Delete option in the More actions (three dots) menu.")

   The Delete agreement type? dialog box opens. You are prompted to confirm that you want to delete the agreement type.

   ![A screenshot of the Delete agreement type? dialog box that prompts you to delete the agreement type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-type-management_Delete-agreement-type-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1MjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.xE2xH63gOy2KmKNc_a4fdsuvm3E8vUPogj-JDvomT7A&_LANG=enus "A screenshot of the Delete agreement type? dialog box that prompts you to delete the agreement type.")
7. Review the message and select the OK option.

   The message states that the agreement type won't be available for future agreements. However, the type still displays on current agreements until you change it.

   The dialog box closes and you are returned to the Manage Agreement Types page. The deleted agreement type is no longer displayed in the Agreement Types table.
8. Repeat these steps to remove other unused custom agreement types.
9. Return to the Agreement Manager homepage. Select the Close (x) icon from the top right.

   You are returned to the Completed Documents page.

You have successfully deleted an unused custom agreement type.
