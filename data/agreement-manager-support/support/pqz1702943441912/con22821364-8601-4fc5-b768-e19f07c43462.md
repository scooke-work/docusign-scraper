# IAM Agreement Types Manager

Read about the Agreement Types manager, including its purpose, use, and role within the Intelligent Agreement Management (IAM) platform.

The *Agreement Types Manager* is the central location for configuring and maintaining agreement types and their fields. These agreement types are used across the IAM platform, not just in Agreement Manager.

Tip: See the [Agreement Types Across the IAM Platform](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=con0213702d-d2af-424a-9512-1dfae7765cc5.html&_LANG=enus "Read about how agreement types are used across the Intelligent Agreement Management (IAM) platform.") topic to learn
more about how agreement types fit into the IAM platform.

## The 'Agreement Types' table

The Agreement Types table lists all the standard and custom agreement types that are associated with the account:

![A snippet of the Manage Agreement Types page that shows the Agreement Types table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Types_Agreement-Types-table_snippet.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1MDcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.sGcpbwOW9PcnIwqQp7FzZCL9er3WzjTh80rf2VFMMGM&_LANG=enus "A snippet of the Manage Agreement Types page that shows the Agreement Types table.")

- The Agreement Type column displays the name of the agreement type and the number of fields that are associated with it.

  Tip:

  Use the Sort items (arrows) icon to sort the table by agreement type. Select the field count link to view the Fields list for the agreement type.
- The Documents column shows the number of documents in your repository that are classified as each agreement type.
- The Category column indicates the classification of the agreement type. For example, Business Services, Human Resources, or Miscellaneous.
- The Created by column indicates whether the agreement type is custom (User) or standard (Docusign).
- The AI detection column displays the AI status for the agreement type. For
  example, whether the agreement type is Active or
  it must be Set Up. You can also
  Activate the agreement type from this
  column.

For custom agreement types, the More actions (three dots) menu is available. The menu options are dependent on the AI detection status for the custom aagreement type. For example, if the AI detection status is Set Up, you can Edit Details or Delete the agreement type.

Once the setup is complete, theMore actions (three dots) menu includes
actions related to the AI detection:

![A snippet of the Agreement Types table that highlights the More actions (three dots) menu, including available AI detection options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Types_More-actions-options_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1MDcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.sGcpbwOW9PcnIwqQp7FzZCL9er3WzjTh80rf2VFMMGM&_LANG=enus "A snippet of the Agreement Types table that highlights the More actions (three dots) menu, including available AI detection options.")

## Search and filter options

Use the search field and table filters to locate the relevant agreement types:

![A snippet of the Manage Agreement Types page that shows the search field and filter options for the Agreement Types table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Types_Agreement-Types_search-filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1MDcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.sGcpbwOW9PcnIwqQp7FzZCL9er3WzjTh80rf2VFMMGM&_LANG=enus "A snippet of the Manage Agreement Types page that shows the search field and filter options for the Agreement Types table.")

- Enter a search term in the Search types... field. As you type the keyword, the table updates and the types that match the search term are displayed.
- Use the Created By filter to view standard (Docusign) or custom (User) agreement types.
- Use the Category filter to view agreement types by classification. For example, Business Services, Human Resources, or Miscellaneous.
- Use the AI Status filter to view agreement types based on their AI detection status, Active or Needs Setup.

## Agreement type details - the 'Fields' table

Locate and select (click) the relevant type from the Agreement Types table. The Fields table opens for the selected type. This table lists the custom and standard fields that are associated with the selected agreement type:

![A screenshot of the Fields table that shows the custom and standard fields associated with the selected agreement type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Types_Agreement-Types_Fields-table.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1MDcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.sGcpbwOW9PcnIwqQp7FzZCL9er3WzjTh80rf2VFMMGM&_LANG=enus "A screenshot of the Fields table that shows the custom and standard fields associated with the selected agreement type.")

Review the custom and standard fields that are associated with the selected agreement type. The table provides the Field name, Category, Created By, and AI detection details for all the fields.

Use the Search fields... field to search for a specific field or use the filters to refine the Fields table. Filter options include:

![A snippet of the Fields table that shows the search field and the available filter options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Types_Agreement-Types_Fields_search-filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1MDcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.sGcpbwOW9PcnIwqQp7FzZCL9er3WzjTh80rf2VFMMGM&_LANG=enus "A snippet of the Fields table that shows the search field and the available filter options.")

- Created By: Filter the table to display custom (User) or standard (Docusign) fields.
- Category: Filter the table by field classification. Supported categories include: General, Payment, Legal and Compliance, Renewal, Termination, Other, and Human Resources.
- Field Type: Filter the table to show specific types of fields. Supported field types include: Currency, Dropdown, Contains Multiple, Number, Short Text, Date, Duration, and Text.
- AI Status: Filter the table to display fields with Active AI detection or fields that Need Setup.

Note: Administrators and contract managers can use the + Add Fields option to add custom and standard fields to the agreement type.

The More actions (three dots) menu contains options to Edit Details for custom fields or Remove fields from the type. The options in the More actions (three dots) menu are dependent on the AI detection status:

![A snippet of the Fields table that shows the More actions (three dots) menu, including available AI detection options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Types-Manager_Fields_More-actions_expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1MDcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.sGcpbwOW9PcnIwqQp7FzZCL9er3WzjTh80rf2VFMMGM&_LANG=enus "A snippet of the Fields table that shows the More actions (three dots) menu, including available AI detection options.")

Once the AI detection is set up, the menu contains actions related to the AI detection. For example, you can Refine Results or Test for Accuracy.

## Primary use cases

Administrators and contract managers can use the *Agreement Types Manager* to:

- Review custom and standard agreement types for the account
- Create custom agreement types
- Edit agreement type details
- Add or remove fields on custom and standard agreement types
- Delete or retire unused custom types
- Support AI detection for custom types
