# Upload Documents from Cloud Storage Locations

This topic answers the question: How do I add documents from cloud storage locations to Agreement Manager? Learn how to select documents from cloud storage locations to upload to Agreement Manager for processing.

Instead of uploading documents from local folders, you can choose files from your organization's cloud storage locations. Keep in mind that:

- File names must contain fewer than 255 characters
- Files must be less than 100 MB each
- Maximum of 1,000 files per bulk upload job

Note: The Upload up to 1,000 files at a time permission must be turned on for users to bulk upload files and folders to Agreement Manager. This toggle is turned on from the user's profile page by an administrator.

Important: An administrator must install third-party cloud storage apps before you can upload files from these locations. Review the [File Ingestion from Third-Party Cloud Storage Providers](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=byx1727376126901.html&_LANG=enus "This topic answers the question: How can I upload selected files to Agreement Manager from third-party cloud storage providers?") topic for information about adding apps from the Docusign App Center.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Expand the Add (+) dropdown and select the Upload option:

   ![A screenshot of the Add (+) dropdown menu that highlights the Upload option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Upload-icon-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3NjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.eka670OSS1D45PHzu62Z9q74iYccb1i4IGFx4GAbMus&_LANG=enus "A screenshot of the Add (+) dropdown menu that highlights the Upload option.")

   The Upload documents dialog box opens.

   ![A screenshot of the Upload documents dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-view_Upload-documents-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3NjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.eka670OSS1D45PHzu62Z9q74iYccb1i4IGFx4GAbMus&_LANG=enus "A screenshot of the Upload documents dialog box.")
5. Expand the Select File dropdown and select the cloud storage location for the files you want to upload:

   ![A screenshot of the Select File dropdown that shows the installed apps and options to connect to them.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Mini-App-Center_Select-File-dropdown.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3NjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.eka670OSS1D45PHzu62Z9q74iYccb1i4IGFx4GAbMus&_LANG=enus "A screenshot of the Select File dropdown that shows the installed apps and options to connect to them.")

   The available cloud storage apps are displayed in the installed apps list. Before you can upload documents from an app, you must establish a connection to your account.
6. Use the Add Connection link to connect your account to the app.
   1. Select the Add Connection link for the app that you want to use.
   2. Follow the prompts to connect your account to the app.
7. Once connected, select the app from the installed apps list and continue the upload process.

   The Select files from [Cloud Storage] dialog box opens. A list of drives is displayed.
8. Expand the Select a connection dropdown and choose the relevant connection. Select Next.
9. Expand the Select a drive dropdown and select the drive that contains the files you want to upload. Select Next.

   The drive opens. The folders and files in the drive are displayed on the Browse tab.
10. Use the search bar and sorting arrows to locate and select the files that you want to upload:

    ![A screenshot of the Select files dialog box that shows the Browse tab, including selected files.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Upload-documents_cloud-storage_Select-files-from-drive_Browse-tab.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3NjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.eka670OSS1D45PHzu62Z9q74iYccb1i4IGFx4GAbMus&_LANG=enus "A screenshot of the Select files dialog box that shows the Browse tab, including selected files.")

    1. Enter a keyword or search string in the Search field and press [Enter].
    2. Use the sorting arrows to sort the list of documents by Name, Type, or Last Modified.
    3. Select the checkboxes associated with the files that you want to upload.

       You can also choose to upload the entire contents of a folder. If you select a folder, all the files and sub-folders within it are selected.

       As each file is selected the number of Selected Items increases. The breadcrumb navigation updates to display your folder location within the drive.
11. Select the Selected Items tab.

    The list of selected files displays in the dialog box.
12. Review the list of selected files to confirm that all the relevant files are selected.

    ![A screenshot of the Select files dialog box that shows the Selected Items tab.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Upload-documents_cloud-storage_Select-files-from-drive_Selected-Items-tabpng.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3NjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.eka670OSS1D45PHzu62Z9q74iYccb1i4IGFx4GAbMus&_LANG=enus "A screenshot of the Select files dialog box that shows the Selected Items tab.")

    Note: **Regular document upload**

    If you are uploading fewer than five documents, select Submit.

    The Completed Documents view opens and a message states that the upload was successful. The files are being processed. No other actions are required.
13. Select Submit.

    The Upload documents dialog box opens.

    ![A screenshot of the Upload documents dialog box that shows the number of selected files.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Upload-documents_Cloud-storage_Upload-documents-modal_Google-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3NjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.eka670OSS1D45PHzu62Z9q74iYccb1i4IGFx4GAbMus&_LANG=enus "A screenshot of the Upload documents dialog box that shows the number of selected files.")
14. An auto-generated name is displayed in the Job name field. Accept the default value or enter a new name.

    Tip: Enter a unique job name so you can easily find it in the job monitoring hub.
15. Add more files to the upload as needed:
    1. Select Add.
    2. Search for and select additional files to bulk upload to Agreement Manager. You may be prompted to select the drive again.
    3. Select the checkboxes associated with the files that you want to upload.
    4. Select Submit.

       The Upload documents dialog box opens.
16. Select Next.

    The Allow access step is displayed in the Upload documents dialog box. You have three options:

    ![A screenshot of the Upload documents dialog box that shows the Allow access step.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Upload-documents_Bulk-upload_Step-2.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3NjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.eka670OSS1D45PHzu62Z9q74iYccb1i4IGFx4GAbMus&_LANG=enus "A screenshot of the Upload documents dialog box that shows the Allow access step.")

    - Add the uploaded documents to an agreement set.
    - Share the files with others, including setting agreement-level permissions.
    - Skip this step to upload the documents without sharing or organizing into sets.

**Add the uploaded documents to agreement sets:**

Important: The Organize and manage all agreements permission is required to add uploaded documents to agreement sets. See the [Capabilities by Agreement Manager Permissions](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=vhx1743523538507.html) topic to learn more about Agreement Manager permissions for various features.

17. Add the uploaded documents to an agreement set:

    ![A snippet of the Upload documents dialog box that shows the Add to agreement sets option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Upload-documents_Cloud-storage_Allow-access_agreement-sets-option.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3NjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.eka670OSS1D45PHzu62Z9q74iYccb1i4IGFx4GAbMus&_LANG=enus "A snippet of the Upload documents dialog box that shows the Add to agreement sets option.")

    1. Select the Add to agreement sets option.
    2. Enter a set name in the Search sets field and select the set from the type-ahead dropdown.
    3. Select Upload and Add to Sets.
    4. Follow the prompts to start the upload process.

       A dialog box opens. The upload job is initiated and your files are being processed with AI.

    Tip: If you cannot locate the relevant set, select the + Create Set option and create a new set:
    - Complete the Name and Description fields from the New agreement set dialog box.
    - Select either Create or Create and Share.

    See the [Create an Agreement Set](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=jay1736181359265.html) support topic to learn more about creating agreement sets.

**Share the uploaded documents:**

18. Share the uploaded documents with other users and groups:

    ![A snippet of the Upload documents dialog box that shows the Share files option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Upload-documents_Cloud-storage_Allow-access_share-files-option.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3NjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.eka670OSS1D45PHzu62Z9q74iYccb1i4IGFx4GAbMus&_LANG=enus "A snippet of the Upload documents dialog box that shows the Share files option.")

    1. Select the Share files option.
    2. Enter a name or email address in the Add user or group search field.

       A type-ahead dropdown opens, and the users and groups that match the entered search criteria are displayed.
    3. Select a user or group name from the dropdown. Continue adding users or groups as needed.
    4. Expand the Access dropdown and select an access level for each of the selected users and groups.

       Choices are: Can View, Can Edit, and Can Manage.

       Note: Select the Delete icon to remove the associated with a user or group from the list.
    5. Select Upload and Share.
    6. Follow the prompts to start the upload process.

       A dialog box opens. The upload job is initiated and your files are being processed with AI.

**Skip this step:**

19. Skip the Allow access step to continue without adding an agreement set or sharing the documents.
    1. Select the Skip this step option.
    2. Select Upload.
    3. Follow the prompts to start the upload process.

       The files will be added to Agreement Manager but they won't be organized or
       shared with others.

    A dialog box opens. The upload job is initiated and your files are being processed with AI.

    ![A screenshot of the Adding your files dialog box that states your files are being processing with AI and will be added to Agreement Manager once finished.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Upload-documents_Cloud-storage_Adding-files-from-Google-Drive.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3NjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.eka670OSS1D45PHzu62Z9q74iYccb1i4IGFx4GAbMus&_LANG=enus "A screenshot of the Adding your files dialog box that states your files are being processing with AI and will be added to Agreement Manager once finished.")
20. Select Monitor progress.

    The job monitoring hub opens and the All jobs view is displayed. The Uploaded Agreements table shows job details for each upload job.

    See the [View Upload Results and Exceptions](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=srq1712255787588.html&_LANG=enus "Learn how to view processing results and details for an upload job.") topic to learn more about job monitoring.
