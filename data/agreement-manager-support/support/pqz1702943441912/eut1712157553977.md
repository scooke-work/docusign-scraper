# Agreement Manager Permissions, Profiles, and Roles

Read about the available Agreement Manager permissions, including how users and groups are granted access to Agreement Manager agreements and example roles for each permission.

Administrators are responsible for safeguarding access to agreements in Agreement Manager. This includes making sure that the appropriate users and groups can view or manage agreements, and perform bulk uploads. Administrators also manage group membership for Agreement Manager groups.

This topic reviews the available Agreement Manager permissions, including how access is assigned to users and groups.

 

Tip: In some cases, you might want to restrict the ability to delete agreements or update attributes to specific users and groups.

Note: See the  [Account Administration Guide](https://support.docusign.com/s/document-item?bundleId=pik1583277475390) for more information about using the menu to manage users and groups.

## Agreement Manager access for administrators

Account administrators automatically have full secure access to all Agreement Manager features by
default. This access includes bulk ingestion and agreement set management
functionality. For example, administrators can use agreement sets to ensure that
only the appropriate users have access to sensitive documents.

Restriction: While administrators automatically have
full access to all the Agreement Manager features, they do not have access to
view the agreements. Access is explicitly assigned to administrators for them to
view any of the agreements in the account. See the [Agreement Sharing and Downloads](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=hrr1722978886188.html&_LANG=enus "This topic answers the question: What are my direct sharing and download options for Agreement Manager documents, including the required permissions?") article to learn more about sharing agreements
with others.

## IAM permission profiles

Permission profiles are groups of settings that determine the behavior and actions available for
a user. Administrators assign permission profiles to users when they are added to an
account. These profiles make it easier to manage permission settings for large
numbers of users. Administrators can update the profile settings, which are applied
to all users with that profile, instead of changing permissions for users
individually.

When creating or updating permission profiles, administrators have the option to include Intelligent Agreement Management (IAM) permissions, such as Agreement Manager Permissions:

![A screenshot of the Add Permission Profile dialog box that highlights the available Agreement Manager Permissions.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Permission-profile_add-new_Navigator-Permissions_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYyODIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.f5pIP8DOfhqOMvUSNgt4nRV1jNB4eMiGP6XVebYamKA&_LANG=enus)

## "View all agreements" versus "Organize and manage all agreements"

Important:

When organizations purchase an Intelligent Agreement Management (IAM) plan with Agreement Manager, by default, all users can access their
own shared, received, sent, and uploaded agreements. However, specific Agreement Manager access is required to view, organize, and manage all the agreements in the account.

There are two options for granting access to the agreements in an account:

![A screenshot of the Agreement Manager access section that highlights the permissions for accessing the account's agreements.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Navigator-Access_Navigator-agreement-permissions_toggles-enabled.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYyODIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.f5pIP8DOfhqOMvUSNgt4nRV1jNB4eMiGP6XVebYamKA&_LANG=enus)

- View all agreements in this account: Authorizes users or groups to view and download all agreements in the account.
- Organize and manage all agreements: Authorizes users or groups to view, edit, download, and delete all agreements in the account. This permission level also grants access to manage agreement sets and custom extractions, like agreement types and fields.

These permissions control access to view Agreement Manager agreements and to perform various management tasks. This applies to administrators as well. While administrators automatically have full access to all the Agreement Manager features, they do not have access to view the Agreement Manager agreements.

Note: Agreements are permanently stored in Agreement Manager unless they are deleted by users with the appropriate permissions:

- Users with Organize and manage all agreements access to Agreement Manager can delete any of the agreements from the repository.
- Other users can delete only the agreements to which they have Can Manage access.

eSignature retention is not applicable to Agreement Manager.

See the [Capabilities by Agreement Manager Permissions](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=vhx1743523538507.html&_LANG=enus "Look up the available capabilities for each Agreement Manager permission.") article to learn more about the specific feature access for these permission levels.

## Bulk upload permission

All users can upload up to five third-party agreements to Agreement Manager. However, to upload more than five files to Agreement Manager, an additional permission is required. The Upload up to 1,000 files at a time permission is assigned from the User page under the Admin menu. This permission can also be assigned from a permission profile.

Once this permission is turned on, the user can upload up to 1,000 files at a time to Agreement Manager:

![A screenshot of the Agreement Manager access section that highlights the bulk upload permission and shows that it is turned on.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Navigator-Access_Bulk-upload-toggle_enabled.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYyODIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.f5pIP8DOfhqOMvUSNgt4nRV1jNB4eMiGP6XVebYamKA&_LANG=enus)

Note: The *bulk upload permission* is required to upload documents from
third-party cloud storage locations.

## Agreement Manager access by roles

The primary Agreement Manager roles are: Administrators, contract managers, and contract viewers. Administrators and contract managers share similar Agreement Manager access. Both roles have full access to Agreement Manager features, however, administrators cannot view the agreements in an account.

Note: In-product prompts encourage administrators to assign permissions for contract managers:

## Access control with agreement sets

Agreement sets are named groups of agreements that can be shared with users and groups. These sets give administrators and contract managers a granular way to control access to agreements.

For example, a contract manager can organize agreements for productivity and data privacy by creating an agreement set. Once created, the contract manager now has an efficient way to grant access to the appropriate users and groups.

Consider this pain point:

- The contract manager oversees thousands of sales agreements. They need an efficient way to grant
  access to a subset of these agreements.
- The renewals team needs access to all master service agreements (MSAs) with upcoming expirations
  and no auto-renewal terms. The U.S. sales team needs access to all MSAs under
  U.S. jurisdiction.
- The contract manager creates agreement sets to grant access to the relevant agreements for each team.

See the [Agreement Set Management](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=zed1730843285476.html&_LANG=enus "Read about agreement set management including how to use sets to control access to agreements. This topic answers the question: What are agreement sets and how can I use them?") article to learn more about using this feature to control access to your agreements in Agreement Manager.

## Group membership

Administrators can manage group membership by adding and removing users from a group. Groups are managed from the Admin menu, where the administrator can:

- Add a user to an existing group
- Remove a user from an existing group
- Create a new group
- Add and update users

Note: The Bulk Add and Bulk Update functions are described in the Docusign eSignature Admin guide. See the [User Bulk Actions](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=mxm1583277431469.html) topic for more information.
