# Grant Access to Agreement Manager Agreements

This topic answers the question: How can I give access to Agreement Manager agreements and features, including bulk upload permissions?

When organizations purchase an Intelligent Agreement Management (IAM) plan with Agreement Manager, by default, all users can access their
own shared, received, sent, and uploaded agreements. However, specific Agreement Manager access is required to view, organize, and manage all the agreements in the account.

Administrators are responsible for safeguarding access to the agreements in Agreement Manager. This includes making sure that the appropriate users and groups can view or manage agreements.

Remember:

Agreement sets provide another option for sharing groups of agreements with users. These sets give contract managers and administrators a more granular way to control access to agreements. See the [Share an Agreement Set](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=pvb1736186431453.html&_LANG=enus "Learn how to share an agreement set with individual users and groups. This topic answers the question: How do I share groups of agreements with the appropriate people?") topic to learn more about granting access to a set of agreements.

Agreement Manager access is granted from the User and Group profile pages under the Admin menu. This topic reviews the steps to grant individual users and groups access to Agreement Manager agreements and features.

1. Log in to your Docusign account using administrator credentials.

   The Docusign Home page opens.
2. Select Admin from the global header.

   The Admin menu opens and the Accounts view is displayed.

**Select an account (for organizations with multiple accounts)**

Important: User and group permissions are managed at the *account*-level. Make sure that you are updating records from Accounts administration instead of Organization administration. If your organization has multiple accounts, verify that you are updating the correct account.

3. Expand the Accounts dropdown from the top of the navigation pane and select the relevant account:

   ![A snippet of the Accounts view that highlights the Accounts dropdown in the pane on the left.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Permissions_Admin-menu_Accounts-dropdown_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzMDMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.aQYD34WOFRyz1JSsiN4jc1j8b5Hw_vWRA3lPdbHyZXw&_LANG=enus "A snippet of the Accounts view that highlights the Accounts dropdown in the pane on the left.")

   The Accounts view refreshes and the Overview page is displayed for the selected account. Locate the user or group that you want to update.

**Locate the relevant user or group**

4. Select either Users or Groups from the USERS AND GROUPS section in the navigation pane.

   Depending on your selection, either the Users or Groups page opens.
5. Locate the user or group that you want to update:
   1. From the Users page:

      - Enter the name of the user in the Search Here field.
      - Select a name from the type-ahead dropdown or press the Enter key.
      - Adjust the filters as needed. Filter options include: Status, Permissions, and Groups.

      The Users page refreshes and the users who matched the search criteria are displayed in the table.
   2. From the Groups page:

      - Enter the name of the group in the Enter Keyword field.
      - Select the Search option.

      The Groups page refreshes and the groups that matched the search criteria are displayed.

**Update Agreement Manager access**

6. Expand (select) the Actions option for the relevant user or group and select the Manage Agreement Manager Access item.

   The User or Group profile page opens. The Agreement Manager Access tab is displayed by default.
7. Turn on the appropriate permissions for the selected user or group. From the Agreement Manager access section:
   - Assign the View all agreements in this account permission. Select the toggle so it is in the "on" (blue) position.

     This permission gives the user or group access to view and download all the agreements in the selected account.
   - Assign the Organize and Manage all agreements permission. Select the toggle so it is in the "on" (blue) position.

     This permission gives the user or group the highest level of Agreement Manager access. This access is typically reserved for contract managers and administrators. With this access, the user or group can create and manage:

     - Agreement sets
     - Agreement types
     - Custom fields
     - Duplicate parties and merged groups
     - Obligations and obligation types

**Assign bulk upload permissions (user-level permission)**

Important: The *bulk upload* permission is set at the user-level. This
permission is not available at the group-level. Keep in mind that user and group permissions are managed at the account-level as opposed to the organization-level.

8. If appropriate, assign bulk upload permissions to the selected user. Select the Upload up to 1,000 files at a time toggle so it is in the "on" (blue) position.

   This permission gives users access to *bulk upload* up to 1,000 files at a time. Users also have access to upload agreements to Agreement Manager from third-party cloud storage locations.
9. Select the Save Changes option.

   A message box opens to confirm that the Agreement Manager access changes were saved.

You have successfully granted users and groups Agreement Manager access to the agreements in a selected account.
