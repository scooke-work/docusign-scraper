# Update Custom Field Details

Learn how to update the details for your custom fields, such as the name and category.

Administrators and contract managers can update the custom fields in their account. For example, update the name and category details for a selected field.

Note: The Edit Details function is available for custom fields only. You cannot update the standard, Docusign-provided fields. This function is completed by users with Organize and manage all agreements access to Agreement Manager and the account's agreements.

Use the *Fields Manager* to update the details for a custom field. Access this function from the Agreement Manager homepage under the Settings (gear) menu.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. A list of documents to which the
   logged-in user has access is displayed in the table.
4. Expand the Settings (gear) dropdown from the top right and select the Fields option:

   ![A screenshot of the Agreement Manager homepage that highlights the Fields option in the Settings (gear) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-types-manager_Settings-menu_Fields_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3MzMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Szgh_xEp-74SZ_1x_N6VjMvfRS_-i1XwfanALvKO5Y8&_LANG=enus "A screenshot of the Agreement Manager homepage that highlights the Fields option in the Settings (gear) menu.")

   The Manage Fields page opens and the Fields view is displayed. All the custom and standard fields in the account are displayed in the table.
5. Locate the custom field that you want to update.

   ![A snippet of the Manage Fields page that shows the search field and filter options for the Fields table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Fields-Manager_Fields-table_search-and-filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3MzMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Szgh_xEp-74SZ_1x_N6VjMvfRS_-i1XwfanALvKO5Y8&_LANG=enus "A snippet of the Manage Fields page that shows the search field and filter options for the Fields table.")

   - Search for the relevant custom field.

     Enter a keyword in the Search fields... field. As you type the keyword, the table updates and the fields that match the search term are displayed.
   - Filter the table to locate the relevant custom field. Filter options are: Created By, Category, Field Type, and AI Status.

     For example, filter the table to view user-created fields. Expand the filter, choose a value, and select the Apply option.

     Important: The Edit Details function is only available for user-created or custom fields. You cannot update the details for Docusign-provided fields.

     The table updates and the fields that matched the applied filters are displayed.
6. Expand the More actions (three dots) menu for the relevant field and select the Edit Details option:

   ![A snippet of the agreement type details view that highlights the Edit Details option for a custom field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Fields-Manager_Fields-table_More-actions-menu_Edit-Details_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3MzMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Szgh_xEp-74SZ_1x_N6VjMvfRS_-i1XwfanALvKO5Y8&_LANG=enus "A snippet of the agreement type details view that highlights the Edit Details option for a custom field.")

   The Edit field dialog box opens and a warning (!) message is displayed. Any changes that you make to the field are reflected everywhere that the field is being used.
7. Update the details for the selected field and select the Save option:

   ![A snippet of the Edit field details dialog box that shows you can update the field's Name and Category.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Types_Agreement-type-details_Edit-field-details-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3MzMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Szgh_xEp-74SZ_1x_N6VjMvfRS_-i1XwfanALvKO5Y8&_LANG=enus "A snippet of the Edit field details dialog box that shows you can update the field's Name and Category.")

   1. Enter a new Name for the custom field as needed.
   2. Update the Category for the custom field as needed. Expand the Category dropdown and select the relevant category.
   3. Select the Save option.

   Tip: The Field type and Agreement types fields are closed to edits. You cannot update these fields from this location. However, you can add the field to multiple agreement types from the *Agreement Types Manager*. See the [Add Fields to Multiple Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk37e570f8-2428-4d26-a78b-b1239aca35a5.html&_LANG=enus "Learn how to add fields to multiple agreement types from the Add a field dialog box. You can add fields to additional agreement types while you are updating a selected agreement type.") article to learn about adding fields to multiple agreement types.
8. Select the Save option.

   The dialog box closes and you are returned to the Manage Fields page. The custom field is listed in the Fields table and it reflects the updated details.
9. Return to the Agreement Manager home page. Select the Close (x) icon from the top right.

   You are returned to the Agreement Manager homepage and the Completed Documents view is displayed.

You have successfully updated the name and category details for a custom field.
