# Export Data from the Agreement List Table

Learn how to export agreement data to a CSV file for selected agreements.

Use the Export to CSV feature to export agreement data from the agreement list table. Before exporting the data, customize the columns in the table to make sure the export captures the relevant data.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Perform a search and apply filters to locate the relevant agreements:
   1. Enter a keyword or string in the search field and select a suggestion from the dropdown or press the Enter key.

      The agreements containing the keyword or search string that you entered are displayed in the table.
   2. Use the filters to refine the list of search results. For example, use the Parties filter to view a list of agreements for a specific party.

      1. Expand the Filters option and select the filter that you want to apply.
      2. Choose a filter value.
      3. Select the Apply option.

      The agreements that match the applied filters are displayed in the table.
5. Select the agreements that you want to export:

   ![A snippet of the agreement list table that shows selected agreements and options to download documents or export agreement data.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Download-or-export-selected-agreements.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0OTAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.ZdtKjt3HXdvKphTZr2Ymn0QzEmerKXE1Oucc3k1tyz8&_LANG=enus "A snippet of the agreement list table that shows selected agreements and options to download documents or export agreement data.")

   - Select the individual agreements.
   - Select the Choose all checkbox from the header row to select all the agreements on that page of results.

     Tip: The Choose all option only selects agreements on the current page of search results. It does not select any agreements on subsequent pages of results.
   - Expand the Selected dropdown and select the Select all that match your filters option. All the agreements in the table that matched your applied filters are selected.

     Tip: The Select all that match your filters option is only available if filters are applied to the agreement list table.

   When at least one document is selected, the actions bar opens. The Download Documents and Export to CSV options are displayed. The document count indicates the number of selected agreements.
6. Select Export to CSV.

   The CSV file is downloaded to your local Downloads folder. The file contains agreement data (custom and standard fields) for the selected documents.

   Tip: Update the CSV file to further customize your information. For example, hide and reorder the columns in the downloaded file until the relevant information is displayed.

You have successfully exported the agreement data for a group of selected documents, including custom and standard fields.
