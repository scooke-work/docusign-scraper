# Enable AI Detection for Your Custom Fields

Learn how to enable the AI detection for your custom fields. Once activated, the field is surfaced in your new agreements automatically.

When you first create a custom field, the AI detection is inactive by default. Complete the Set Up process to activate the custom field. This process is designed to "teach" the AI model and enable the AI detection to "learn" by example. Once activated, the custom data point is surfaced in your new agreements and selected historical documents.

Important: This task requires a user-defined custom field that is not activated for AI detection. See the [Add Custom Fields to Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=hcn1736879099815.html&_LANG=enus "Learn how to create and add custom fields to the custom and standard agreement types in your account.") article for steps to configure new fields.

Locate the custom field that you want to enable for AI detection and complete the Set Up process. This task is completed by users with Organize and manage all agreements access to Agreement Manager agreements and functionality. For example, administrators and contract managers.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.

**Locate the custom field**

If you know the agreement type for the field, you can locate it from the *Agreement Types Manager*. If you know the name of the field, you can search for it in the *Fields Manager*.

4. Expand the Settings (gear) menu from the top right and select either Agreement Types or Fields:

   ![A snippet of the Settings dropdown that highlights the Fields and Agreement Types options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Types_More-actions-menu_expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.1vno5LRovFsIhyzJY4sZtK5fstwGHTxDAVsVnoRdEhQ&_LANG=enus "A snippet of the Settings dropdown that highlights the Fields and Agreement Types options.")Depending on which option you selected, either the Mange Fields page or the Manage Agreement Types page is displayed.
5. Search for the relevant custom field:
   - From the Manage Fields page: Enter a search term in the Search fields... field or use the filters to locate the relevant field. Filter options are: Created By, Category, Field Type, and AI Status. Expand the filter, choose a value, and select the Apply option.

     The fields that matched the entered search term and applied filters are displayed in the FIelds table.
   - From the Manage Agreement Types page: Search for and select (click) the agreement type that contains the custom field. The agreement type details page opens and the associated fields are listed in the Fields table. Perform a keyword search and apply filters to locate the relevant field.

     The fields that matched the search term and applied filters are displayed in the Fields table.
6. Select the Set Up option for the custom field that you want to enable for AI detection.

   The AI detection set-up wizard launches.

   ![A screenshot of the message box that provides high-level instructions for enabling AI detection.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-type-management_enable-AI-for-custom-fields_informational-message.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.1vno5LRovFsIhyzJY4sZtK5fstwGHTxDAVsVnoRdEhQ&_LANG=enus "A screenshot of the message box that provides high-level instructions for enabling AI detection.")

**Define the field**

7. Select the Get Started option.

   The Set definition dialog box opens and asks you to define the custom field.
8. Define the custom field.
   1. Enter a detailed definition for the custom field. This field is limited to 2,000 characters. The definition describes the purpose of the field.

      Note: See the [Best Practice: Writing Definitions for AI Detection](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tmk1741317734856.html&_LANG=enus "Read about tips for writing good definitions for the AI detection setup process. These definitions are an important part of the process and help to \"teach\" the AI detection.") article to review the
      guidelines for writing definitions. Follow these guidelines to get
      the most out of the AI detection and to ensure accuracy.

      When the definition exceeds 50 characters, the Refine with AI option becomes available. Use this option to update the definition for consistency and accuracy.
   2. Fine-tune the definition. Select the Refine with AI option. The dialog box refreshes and a suggested definition is displayed.
   3. Confirm the definition. Review and edit the suggested definition as needed. You can also select the Undo option to revert to the original version of the definition.
   4. Select the Next option.

   Note: If you are enabling a custom dropdown field, you are prompted to add details for each option. The descriptions help the AI model to identify the data points in your documents. Enter a description for each of the options in the dropdown and select the Next option:![A screenshot of the Add details for each option dialog box that shows example descriptions.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-type-management_Enable-AI_Add-details-for-options_Dropdowns.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.1vno5LRovFsIhyzJY4sZtK5fstwGHTxDAVsVnoRdEhQ&_LANG=enus "A screenshot of the Add details for each option dialog box that shows example descriptions.")

   The AI detection set-up wizard advances to the next step and you are prompted to provide contextual examples for the custom field.

**Add examples of the field**

Important:

When you provide example field values, include specific phrases or paragraphs so the AI
models can detect the data in a contextual manner. If you provide a single word, the model is unable
to detect the data due to lack of context.

9. Choose your preferred method for providing examples and select the Next option.
   - Select the Paste examples from your documents option and click Next.

     The Paste examples of the field dialog box opens.

     1. Paste examples of the field. Be sure to include whole sentences or paragraphs.
     2. Confirm the field value. Locate the field value within the provided example and enter it in the text-entry field.
     3. Repeat these steps for each example. Add at least 3–5 examples for the best results.
     4. Select Save after providing the necessary examples.
   - Select the Highlight examples directly in documents option and click Next.

     The Select relevant documents page opens.

     1. Enter search criteria and apply filters to locate documents that likely contain the custom field.
     2. Choose the documents that contain the relevant examples and select the Start Highlighting option.

        The Agreement preview page opens for the first document. You are prompted to locate the field in each document that you selected.
     3. Highlight the entire section that contains the custom field from the preview pane and select Confirm selection.

        You are prompted to confirm the field value.
     4. Locate the field value from within the agreement and enter it in the text-entry field. Select Confirm.

        Continue highlighting the example content and confirming the field value for each document that you selected.
     5. Select the Next option from the bottom of the pane.

        The Paste examples of the field dialog box opens.
     6. Review the examples that you provided, including the confirmed field values. Select Save.

   A dialog box opens and a message confirms that AI detection is now available for the custom field. You are prompted to test for accuracy or activate AI detection.

   ![A screenshot of the AI detection is now available dialog box that confirms AI detection is available for the custom field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Type_Enable-AI-detection-custom-fields_test-or-activate.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.1vno5LRovFsIhyzJY4sZtK5fstwGHTxDAVsVnoRdEhQ&_LANG=enus "A screenshot of the AI detection is now available dialog box that confirms AI detection is available for the custom field.")

**Test for accuracy**

Before you activate the AI detection for custom extractions, the setup wizard prompts you to test for accuracy. This step is optional, however, it is highly recommended. The results are a good indicator of the quality and accuracy that you can expect from the AI detection before activating it.

10. Choose the Test for accuracy option and select the Confirm option. This option is selected by default.

    The Select relevant documents page opens. Select at least one document that is likely to contain the custom field.
11. Select the documents that you want to use to test the AI detection accuracy:
    1. Search for the relevant agreements. Enter a keyword or search string in the search field. The dropdown opens and search suggestions are displayed. Select a search suggestion or press the Enter key.
    2. Filter the agreement list table to refine the results list. For example, filter by Sets, Status, Parties, Agreement Type, or AI Suggestions.

       Tip: Use the All Filters option to apply additional filters to the results table.

       Expand the filter, choose the relevant filter value, and select the Apply option.
    3. Choose the documents that you want to use in the test.
    4. Select the Start Testing option.

    The Agreement preview opens for the first document. The pane on the left shows the agreement type that the AI detection identified in the document.
12. Rate the accuracy of the findings.
    1. Review the identified field in the context of the document. Verify that the AI detection surfaced the correct value. Select Yes if the AI detection was correct or No if the field was not correctly identified.

       Note: If the field was not located in a selected document, a message is displayed. Be sure to include agreements that do not contain the field when testing the accuracy. It's important for the AI model to recognize when the field is not present.

       The Agreement preview page updates and the next document is displayed.
    2. Repeat this step until you are finished verifying the identified field in all the selected documents.
    3. Select the Done option.

    The Results dialog box opens and the AI detection test results are displayed.

    ![A screenshot of the Results dialog box that shows the AI detection test results.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Fields-Manager_Enable-AI-detection_Set-Up_Results-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.1vno5LRovFsIhyzJY4sZtK5fstwGHTxDAVsVnoRdEhQ&_LANG=enus "A screenshot of the Results dialog box that shows the AI detection test results.")

    Important: If the results do not meet your accuracy requirements, use the Refine definition and examples option to refine your definition and add more examples. Retest the AI detection. Continue refining the definition and examples until the AI detection achieves the expected results.

**Activate the AI detection**

You are prompted to activate the AI detection for the custom field.

13. Activate the AI detection for the field. The
    Activate AI detection option is selected by default. Select the Confirm option.

    The Activate AI detection dialog box opens. Use this dialog box to choose when the system should look for the custom field. The All new agreements option is selected by default. This option cannot be deselected.

    ![A screenshot of the Activate AI detection dialog box that prompts you to choose when to look for the field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Type_Enable-AI-detection-custom-fields_Activate.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.1vno5LRovFsIhyzJY4sZtK5fstwGHTxDAVsVnoRdEhQ&_LANG=enus "A screenshot of the Activate AI detection dialog box that prompts you to choose when to look for the field.")
14. Choose when to look for the custom field and select Activate.

    Tip: Select Activate to continue without selecting historical documents to process. The dialog box closes and the AI detection is active for the custom field. The definitions and examples that you provided will be used to identify the agreement type in your documents.

    In this example, accept the default option, All new agreements, and select Activate.

    Note: The Find on specific agreements option can be used at any time to process specific documents for custom fields. See the [Find Custom Fields in Specific Documents](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk79824e36-d51d-4477-a5ca-7228cde30fb2.html&_LANG=enus "Learn how to use the Find On Specific Documents function to surface custom fields in selected agreements.") article to learn more about this function.

    The dialog box closes and the custom field is published.
15. Return to the Agreement Manager homepage. Select the Close (X) icon from the top left.

    The Agreement Manager homepage opens and the Completed Documents
    view is displayed.

You have successfully enabled the AI detection for a custom field by completing the Set Up process. The definitions and examples that you provided will be used to identify the custom field across your documents.
