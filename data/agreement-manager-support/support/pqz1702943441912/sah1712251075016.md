# Validate Renewal-Related Attribute Values for an Agreement

Learn how to validate renewal-related attribute values for a selected agreement.

Data review is an important part of renewal management. If the renewal-related data points that were surfaced in your agreements are inaccurate, the data displayed in the Renewals dashboard is not accurate. This process relies on the verification of both the AI-assisted insights and the manually entered attribute values.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Use the filters to locate the agreements that you want to review:
   1. Select the Filters option.

      The Filters option expands and list of suggested filters is displayed. You can also search for the filter you want to apply.
   2. Select the AI Suggestions filter from the list of Suggested filters.

      The AI Suggestions filter opens and filter values are displayed.

      ![A screenshot of the expanded AI Suggestions filter that shows the filter options, Needs Review and No Suggestions.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_AI-Suggestions-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyMzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.5oXPRHnSx08kHvvXHLx9Ed5ypYmrqwVbR7gVAR1eSUI&_LANG=enus "A screenshot of the expanded AI Suggestions filter that shows the filter options, Needs Review and No Suggestions.")
   3. Choose the Needs Review value and select the Apply option.

   The dialog box closes and the filter is applied to the agreement list table. The agreements that matched the applied filters are displayed in the table.

   Tip: ![An image of the spark icon used to denote the use of AI.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/AI-Spark-icon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyMzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.5oXPRHnSx08kHvvXHLx9Ed5ypYmrqwVbR7gVAR1eSUI&_LANG=enus "An image of the spark icon used to denote the use of AI.") The spark icon indicates that the agreement contains suggested values. Review, edit, and approve these values to ensure accurate agreement data for reporting, forecasting, and other business purposes.
5. Select (click) the agreement that you want to review from the list.

   The Agreement preview page opens for the selected agreement. The Details pane is displayed by default and shows a list of attributes and suggested values. Attributes, or fields, are displayed based on the identified agreement type.

   A summary displays at the top of the Details pane. The summary includes the identified agreement type and the number of fields and suggestions to review:

   ![A snippet of the Details pane that shows the identified agreement type and the number of suggested field values that require review.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Preview-page_Details-pane_summary-statement-snippet.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyMzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.5oXPRHnSx08kHvvXHLx9Ed5ypYmrqwVbR7gVAR1eSUI&_LANG=enus "A snippet of the Details pane that shows the identified agreement type and the number of suggested field values that require review.")

   Validate the suggested renewal-related attribute values.
6. Place the Details pane into *Edit* mode.
   - Select the Edit (pencil) icon.
   - Select the Review All option.

   Note: Data reviews are completed by users with either of these access levels:
   - Agreement-level access: Can Manage or Can Edit access to the agreement being reviewed.
   - Account-level access: Organize and manage all agreements access to Agreement Manager.For all other users, the Edit (pencil) icon and Review All button are grayed out.

   The Review & edit details pane opens for updates. Attributes are grouped based on usage and purpose. A progress bar is displayed at the top of each group and shows the number of suggestions that require review.

   Important: Updating the suggested attribute values does not change the content in the agreement.
7. Locate the renewal-related attributes. Scroll to the bottom of the pane to the Renewal section.

   ![A screenshot of the Agreement preview page that highlights the Renewal section in the Review & edit pane.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Renewal-Management_Validate-Renewal-attributes_ReviewEdit-details-pane.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyMzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.5oXPRHnSx08kHvvXHLx9Ed5ypYmrqwVbR7gVAR1eSUI&_LANG=enus "A screenshot of the Agreement preview page that highlights the Renewal section in the Review & edit pane.")

   The Renewal section contains renewal-related attributes, such as:
   - Renewal type
   - Renewal Notice Period
   - Renewal Notice Date
   - Renewal Term
   - Extension Period
   - Renewal Owner
   - Additional Info
8. Accept or edit the suggested renewal-related values until each attribute in the Renewal section has been validated. If an attribute does not have a value, leave the field empty for now.
   1. Compare the suggested attribute values against the data in the agreement.

      - Select (click) an attribute to view the suggested value within the agreement.

        The value is highlighted within the agreement and a list of suggested values is displayed.
      - Select the Accept AI Suggestion (checkmark) icon if the value is *correct*.
      - Delete the existing value and enter the correct value if the suggestion is *incorrect*.
   2. Continue reviewing and updating values until all the attributes in the Renewal section have been verified against the agreement.
9. If a field value is missing, find the value in the agreement and update the attribute.

   ![A snippet of the Agreement preview page that shows an example search term that was found within the document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-preview-page_Search-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyMzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.5oXPRHnSx08kHvvXHLx9Ed5ypYmrqwVbR7gVAR1eSUI&_LANG=enus "A snippet of the Agreement preview page that shows an example search term that was found within the document.")

   1. Select the Search (magnifying glass) icon from the top-right corner. A dialog box opens.
   2. Enter a keyword or search string in the Find in Document field. If the value exists, the keyword or search string is highlighted in the agreement.
   3. Once you locate the value, enter it as the attribute value.

      Tip: If the value does not exist in the agreement, leave the field blank and move to the next attribute.
10. When you are finishing with the data review, save your changes. Select the Save option at the bottom of the pane.

    The agreement details are saved. The Details pane is reverted to read-only mode.
11. Return to the Agreement Manager homepage. Select the Close (x) icon in the top-left corner.

    The Agreement Manager homepage opens and the Completed Documents view is displayed. Your search criteria and filters are still applied. The agreements requiring review are displayed in the table.

You have successfully validated the renewal-related attribute values for a selected agreement.
