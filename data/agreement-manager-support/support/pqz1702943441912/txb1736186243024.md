# Add Agreements to Sets Based on Applied Filters

Learn how to apply filters to locate agreements to add to an agreement set. This topic
answers the question: How do I control access to agreements based on attribute
values?

If an agreement set was created without smart filters, agreements can be added to the set
manually. This is done by performing search queries and applying filters to locate
the agreements that you want to add to a set.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign
   Agreements page opens.
3. Select Completed from the
   AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.

   Tip: Select the Filters option to apply filters to the agreement list table::![A snippet of the Completed Documents page that highlights the Filters option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-view_Filter-pills_highlighting-All-Filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNDMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.U3tlKTKLKeTVUtPNrysMg-r1MDeSld20svyO93UE-hc&_LANG=enus "A snippet of the Completed Documents page that highlights the Filters option.")
4. Perform a search query or use the available filters to locate agreements to add
   to an agreement set.
   1. Enter a keyword or search string in the search field and select
      Enter.
   2. Apply filters to locate agreements based on attribute values:

      1. Expand the Filters option and locate the filter that you want to apply.
      2. Select the relevant filter. For example, locate agreements for a specific party, agreement type, or
         time period.
      3. Select a filter value or enter a date range if applicable.
      4. Select the Apply option.
   3. Repeat this step until the relevant agreements are displayed in the agreement list
      table.

   The agreements that matched the search criteria and filters are displayed in the table.
5. Select one or more agreements to add to the agreement set:
   - Select individual agreements to update.
   - Select the Choose all checkbox from the header row to select all the agreements on the page.

   Tip: You can select all the agreements in the results list, not just the agreements on the first page of results. The Selected dropdown displays a counter to indicate the number of agreements that are currently selected. Expand the dropdown and select the Select all that match your filters option:![A snippet of the agreement list table that highlights the Select all that match your filters option in the Selected dropdown.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents_Selected-dropdown_Select-all-that-match-your-filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNDMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.U3tlKTKLKeTVUtPNrysMg-r1MDeSld20svyO93UE-hc&_LANG=enus "A snippet of the agreement list table that highlights the Select all that match your filters option in the Selected dropdown.")All the agreements in the table are selected.

   Important: When you use the Select all that match your filters option, the processing time is dependent on the
   volume of agreements that match the applied filters.

   Restriction: The
   Select all that match your filters option only
   works when applying filters. If you performed a keyword search or if no
   active filter is applied, this option is disabled.

   Additional options are displayed when one or more agreements are selected from the list, including the Organize option.
6. Expand the Organize dropdown and select the relevant set for the agreements:

   ![A screenshot of the expanded Organize dropdown that shows a list of sets to which the agreements can be added.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents_Organize-expanded_Add-to-sets_Select-set.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNDMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.U3tlKTKLKeTVUtPNrysMg-r1MDeSld20svyO93UE-hc&_LANG=enus "A screenshot of the expanded Organize dropdown that shows a list of sets to which the agreements can be added.")

   Tip: Select + Create New Set to add
   the agreements to a new agreement set. Enter a name and description in the
   New agreement set dialog box and select
   Create. The dialog box closes and the new set is
   available for selection from the Organize
   dropdown.
7. Select the Add option.

   The dropdown closes and the selected agreements are added to the set. The Completed Documents view opens and while the agreements are no
   longer selected, the filters are still applied.

   Tip: Clear the selected filters before
   performing another search query. Select the Clear (x) icon for the filter that you want to reset.

You have successfully added agreements to a set based on the applied filters.
