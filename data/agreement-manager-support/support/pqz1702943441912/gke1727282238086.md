# Configure New Custom Agreement Types

Learn how to create custom agreement types to support organizational use cases.

Agreement Manager includes a number of standard, Docusign-provided agreement types. However, since every organization is unique, these types do not cover every type of agreement. Administrators and contract managers can create custom agreement types to support their organizational use cases.

Remember: Administrators automatically have access to configure agreement data. For non-administrator users, agreement type management tasks require Organize and manage all agreements access to Agreement Manager agreements and functionality.

This task is completed from the *Agreement Types Manager*. Access this feature from the Agreement Manager homepage under the Settings (gear) menu.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Completed Documents view opens.
4. Expand the Settings (gear) dropdown from the top right of the page and select the Agreement Types option:

   ![A snippet of the Agreement Manager homepage that highlights the Agreement Types option in the Settings (gear) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-types-manager_Settings-menu_Agreement-Types_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1OTUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WA1i8qnnUlWae-zoU-IbDLLvAEaCJ3ge6iut3c2wbgM&_LANG=enus "A snippet of the Agreement Manager homepage that highlights the Agreement Types option in the Settings (gear) menu.")

   The Manage Agreement Types page opens. A list of the account's standard and custom agreement types is displayed in the Agreement Types table.
5. Select the + Agreement Type option from the top right:

   ![A screenshot of the Agreement Types view that highlights the + Agreement Type option in the top right.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Types-Manager_Agreement-Types_Add-agreement-type-option_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1OTUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WA1i8qnnUlWae-zoU-IbDLLvAEaCJ3ge6iut3c2wbgM&_LANG=enus "A screenshot of the Agreement Types view that highlights the + Agreement Type option in the top right.")

   The New agreement type dialog box opens.

   ![A screenshot of the New agreement type dialog box that shows the Name, Description, and Category fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Types_Add-new-agreement-type.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1OTUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WA1i8qnnUlWae-zoU-IbDLLvAEaCJ3ge6iut3c2wbgM&_LANG=enus "A screenshot of the New agreement type dialog box that shows the Name, Description, and Category fields.")
6. Enter a name for the agreement type in the Name field. The name cannot exceed 50 characters.
7. Enter a description for the agreement type in the Description field. The description cannot exceed 130 characters.

   Note: This information displays on the Agreement type details view and helps others understand the purpose of the agreement type.
8. Expand the Category dropdown and select a category for the agreement type. Choose from:

   - Business Services
   - Human Resources
   - Miscellaneous

   Agreement types are grouped based on their primary use case or category.
9. Select Next.

   The Select fields for [agreement type] dialog box opens. Select the fields that you want to associate with the agreement type. These data points will be surfaced in all documents that are identified as this agreement type.

   ![A screenshot of the Select fields dialog box that shows the available and selected fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Types_Add-new-agreement-type_Select-fields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1OTUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WA1i8qnnUlWae-zoU-IbDLLvAEaCJ3ge6iut3c2wbgM&_LANG=enus "A screenshot of the Select fields dialog box that shows the available and selected fields.")

   Tip: A number of standard fields are selected by default, such as: Title, Parties, Execution Date, Effective Date, and Expiration Date. These data points are surfaced in all documents and cannot be removed from an agreement type.
10. Choose the fields that you want to associate with the agreement type:
    1. Use the Search field to locate the relevant fields.
    2. Select the fields that you want to add to the agreement type.

       The selected fields are added to the Selected Items tab.
    3. Select Save.

    The agreement type details view opens for the new custom agreement type. The selected fields are displayed in the table.

    ![A screenshot of a new custom agreement type that shows the standard fields that were added to the type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Types-Manager_Agreement-type-detail-page_Distribution-Agreement-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1OTUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WA1i8qnnUlWae-zoU-IbDLLvAEaCJ3ge6iut3c2wbgM&_LANG=enus "A screenshot of a new custom agreement type that shows the standard fields that were added to the type.")
11. Review the list of agreement fields. Once AI detection is enabled, these data points, or attributes, are surfaced in your documents for this agreement type.

    Tip: If the agreement type does not contain all the necessary fields, use the + Add Fields option to add more fields. See the [Add and Remove Fields from Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tskda96f0fc-0abb-4ac9-b138-c52e4e8ee713.html&_LANG=enus "Learn how to add and remove standard and custom fields from agreement types.") topic to learn more about adding fields to agreement types.

    When you first create a custom agreement type, the AI detection is inactive by default. Complete the AI detection setup process to surface the new data points in your agreements. See the [Custom Agreement Type Extractions](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=con6f74f3c5-e52b-4611-9d7b-8455c97725b4.html&_LANG=enus "Read about custom agreement type extractions, including their use, purpose, and entry points in Agreement Manager.") topic to learn more about activating your custom types for AI detection.
12. Return to the Agreement Types view. Select the Agreement Types option from the breadcrumb navigation in the top left.

    The Agreement Types view opens. The new custom agreement type is displayed in the table under the relevant category tab.
13. Return to the Agreement Manager homepage. Select the Close (x) icon from the top right.

    You are returned to the Completed Documents view.

You have successfully configured a new custom agreement type. The next step is to enable the AI detection for the custom extraction.
