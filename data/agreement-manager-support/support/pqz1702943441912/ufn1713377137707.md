# View Expiring Agreements for a Specific Party

This topic answers the question: How do I report on expiring agreements for a specific party? Learn how to access the Upcoming expiring agreements report and use filters to view expiring agreements for a specific party.

The Upcoming expiring agreements  chart shows an aggregated count of your expiring agreements, grouped by expiration date. By default, the chart reflects expiring agreements over the next six months. You can filter this report by time period, sets, parties, and agreement type. These filters can be applied from both the dashboard view and the report view.

Note: **Downloading reports to CSV files**

When downloading reports to CSV files from the Insights reporting hub, there is an agreement limit of 25,000 agreements for all IAM reports. This count includes your custom reports.

**Suggested workaround:** Adjust the report filters to run the report in batches until all the relevant agreements are downloaded. For example, run the report in batches according to varying date ranges. Once downloaded, combine the data into one CSV file.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Insights from the global header.

   The Insights hub opens and the Overview is displayed.
3. Select Agreements from the navigation pane on the left.

   The Agreements Dashboard view opens.

   Tip: Filters can be applied at the dashboard-level and the report-level. For example, if you are searching for generic agreement data, apply the filters at the dashboard level.

   In this example, apply filters at the report-level.
4. View the report details for the Upcoming expiring agreements chart. Select the chart title or expand the More actions (three dots) menu and select the View Report option.

   The detailed report view opens and the Upcoming expiring agreements report is displayed. By default, the report reflects agreements that are expiring over the next six months.
5. Filter the report for a specific party and date range.

   ![A snippet of the Upcoming expiring agreements report that highlights the report filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-dashboard_Upcoming-expiring-agreements_filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5NDIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Di4HfqhS5KgLrWC9vnWXeZ7sHYsdoJAcOBG72rRmboM&_LANG=enus "A snippet of the Upcoming expiring agreements report that highlights the report filters.")

   1. Expand (click) the Expiration Date filter, choose the relevant time period, and select the Apply option. For example, select the Next 12 Months option for longer range planning.

      Tip: Use the Custom Date Range option to select a specific date range.
   2. Expand (click) the Parties filter, choose the relevant party, and select the Apply option.

   The report updates automatically. The agreements that match the applied filters are reflected in the chart and displayed in the table.
6. Download or save the report:

   ![A snippet of the report view that highlights the More actions (three dots) menu used to download or save a report.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Report-view_Dropdown-expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5NDIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Di4HfqhS5KgLrWC9vnWXeZ7sHYsdoJAcOBG72rRmboM&_LANG=enus "A snippet of the report view that highlights the More actions (three dots) menu used to download or save a report.")

   - Expand the More actions (three dots) menu in the top right and select Download.

     The report is downloaded to your local Downloads folder.
   - Expand the dropdown in the top right and select the Save option. The Save Report dialog box opens.

     - Enter a unique name in the Report name field. This field defaults to the name of the selected report.
     - Enter a description about the report's purpose in the Description field.
     - Select the Save option.
7. Set up one-time and recurring email reports for yourself and others. Select the Manage Notification (bell) icon and complete the Create subscription dialog box:

   ![A screenshot of the Create subscription dialog box that is used to subscribe to report notifications.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Create-Subscription.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5NDIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Di4HfqhS5KgLrWC9vnWXeZ7sHYsdoJAcOBG72rRmboM&_LANG=enus "A screenshot of the Create subscription dialog box that is used to subscribe to report notifications.")

   Note: When you create a subscription, you have the option to add individual users and groups. Once created, the subscription is displayed in the Manage report subscriptions dialog box for any individual users who were added. Currently, the subscription is not visible to users if they were added as part of a group.

   See the [Subscribe to Receive Report Notifications](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=wfw1734545937270.html&_LANG=enus "Learn how to set up one-time or recurring email reports for yourself or others to receive notifications about your obligations, renewals, or agreements.") topic to learn more about this feature.
8. Add the report to Your Favorites list. Select the Favorite (star) icon located in the top-right corner:

   ![A snippet of a detailed report view that highlights the Favorite (star) icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreements-dashboard_Report-details_Favorite-icon-selected_generic.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5NDIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Di4HfqhS5KgLrWC9vnWXeZ7sHYsdoJAcOBG72rRmboM&_LANG=enus "A snippet of a detailed report view that highlights the Favorite (star) icon.")Your starred dashboards and reports are displayed in the Your Favorites section on the Overview dashboard.
9. Return to the Agreements Dashboard. Select the Agreements Dashboard option from the breadcrumb navigation at the top of the page.

   The Agreements Dashboard view opens.
10. Repeat this process to view, download, or save other agreement reports.
