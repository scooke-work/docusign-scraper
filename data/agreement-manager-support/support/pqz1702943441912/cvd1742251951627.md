# View and Download Processing Results

Learn how to view and download results for processing jobs.

The Monitor All Jobs function gives you access to job details for agreement processing jobs. These jobs are initiated when historical agreements are processed for new insights or attributes (fields). For example, when enabling AI detection for custom fields and agreement types.

Access the Monitor All Jobs function from the More actions menu on the Completed Documents page.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Completed Documents page opens. A list of documents to which the logged-in user has access is displayed in the table.
4. Expand the More actions menu located to the right of the filters and select the Monitor All Jobs option:

   ![A screenshot of the Completed Documents page that shows the Uploads and Processing Status option highlighted in the Settings (gear) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-view_Actions_Monitor-All-Jobs-option.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4NTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.gXAubmAMALCtSWb71Le8cfiAv_leUirzTHDzgl0rtIg&_LANG=enus "A screenshot of the Completed Documents page that shows the Uploads and Processing Status option highlighted in the Settings (gear) menu.")

   The job monitoring hub opens and the All jobs section is displayed. This section has two tabs: Uploaded and Processing. The Uploaded tab is displayed by default.
5. Select the Processing tab:

   ![A snippet of the All jobs section that highlights the Processing tab.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Monitor-All-Jobs_All-jobs_select-Processing-tab.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4NTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.gXAubmAMALCtSWb71Le8cfiAv_leUirzTHDzgl0rtIg&_LANG=enus "A snippet of the All jobs section that highlights the Processing tab.")

   The Processing tab opens. A list of processing jobs are displayed in the Processing agreements table.

   ![A screenshot of the Processing agreements table that shows the job details for each processing job.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Monitor-All-Jobs_Processing_Processing-agreements.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4NTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.gXAubmAMALCtSWb71Le8cfiAv_leUirzTHDzgl0rtIg&_LANG=enus "A screenshot of the Processing agreements table that shows the job details for each processing job.")
6. Review the list of processing jobs. The created date, job name, user, insight, number of files processed, and the status are displayed for each job.
7. Use the filters to locate details for relevant processing jobs:

   ![A snippet of the Processing agreements table that highlights the Created date and Status filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Monitor-All-Jobs_Processing_Processing-agreements_Filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4NTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.gXAubmAMALCtSWb71Le8cfiAv_leUirzTHDzgl0rtIg&_LANG=enus "A snippet of the Processing agreements table that highlights the Created date and Status filters.")

   - Expand the Created date filter, select an identifier tab, and complete the appropriate date fields.

     Identifier tab choices are: Between, Before, and After.

     Tip: If you choose Between, you must complete the From and To date fields.
   - Expand the Status filter and select the relevant status.

     Choices are: Completed, Failed, and Processing.

   The processing jobs that match the applied filters are displayed in the table.

   Tip: Use the sorting arrows located in the column headers to sort the table by the corresponding column. You can sort by Created Date, Job Name, Files, and Status.
8. Select the View details button to view processing results for the corresponding job.

   The processing details page opens for the selected job. The Processing status and Job details are displayed.

   ![A screenshot of the Processing status page that shows processing results for the corresponding job and a download option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Monitor-All-Jobs_Proessing_Processing-with-Docusign-AI_Processing-status.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4NTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.gXAubmAMALCtSWb71Le8cfiAv_leUirzTHDzgl0rtIg&_LANG=enus "A screenshot of the Processing status page that shows processing results for the corresponding job and a download option.")

   Tip: If any files could not be processed, select the Download CSV option to download the processing results:![A snippet of the processing details page that shows the Download CSV option in the Processing status section.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Job-monitoring_Processing-status_Download-CSV-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4NTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.gXAubmAMALCtSWb71Le8cfiAv_leUirzTHDzgl0rtIg&_LANG=enus "A snippet of the processing details page that shows the Download CSV option in the Processing status section.")

   The file is downloaded to your local Downloads folder.
9. A list of processed files is displayed in the Processing status details table. Use the available actions to:

   ![A snippet of the Processing status details table that lists the files that were processed in the selected job.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Job-monitoring_Processing-details_Processing-status-details-table.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4NTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.gXAubmAMALCtSWb71Le8cfiAv_leUirzTHDzgl0rtIg&_LANG=enus "A snippet of the Processing status details table that lists the files that were processed in the selected job.")

   1. Perform a keyword search to locate the file that you want to review. Enter the file name in the search field and select Enter.

      A type-ahead dropdown opens, and a list of files that match the keyword are displayed.
   2. Filter the table by Status and Error Message to refine the table. Expand the filter, select or enter a value, and select Apply.

      A list of files that match the applied filters are displayed in the table.
   3. Select the View File option associated with the file that you want to view.

      The Agreement preview page opens in a new tab. The Details pane lists the attribute values that were surfaced within the selected file.

      Close the tab to return to the processing details page.
10. Select the Close (X) icon from the top-right corner to return to the Processing agreements table.

You have successfully viewed and downloaded processing results from the job monitoring hub.
