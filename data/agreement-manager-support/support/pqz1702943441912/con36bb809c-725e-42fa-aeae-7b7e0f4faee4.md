# Overview Dashboard

Read about the Overview dashboard, including key components and available resources.

The Overview dashboard is the landing page for the Insights hub:

![A screenshot of the Insights hub that displays the Overview dashboard.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Insights-reporting-hub_Overview_cropped.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4NzksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.YljcV3G1HQgE85DGtzUzL4wK5N9NdqREs0rXSOpgzPI&_LANG=enus "A screenshot of the Insights hub that displays the Overview dashboard.")This dashboard is updated in actual time to provide personalized and actionable
insights at a glance, including:

- Recently viewed (Your Recents) reports
- Starred (Your Favorites) reports
- AI-generated custom report recommendations (Suggested Reports)
- Agreement counts (Weekly Insights) for all the agreements in the account, including newly ingested and expiring agreements
- Agreement dashboard and charts (Review your agreements)
- Additional Docusign insights (Explore Docusign Insights) for upcoming renewals, envelope metric, and custom dashboard and reports
- Help documentation and the + Suggest a feature option (Resources)

## Search bar

Use the search bar at the top of the page to Find reports or dashboards:

![A snippet of the Overview dashboard that highlights the search bar at the top of the page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Overview_Search-bar_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4NzksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.YljcV3G1HQgE85DGtzUzL4wK5N9NdqREs0rXSOpgzPI&_LANG=enus "A snippet of the Overview dashboard that highlights the search bar at the top of the page.")

As you enter a search term, the type-ahead dropdown opens:

![A snippet of the Overview dashboard that shows the reports and dashboards that match the entered search query.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Overview_Search-bar_gov-law-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4NzksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.YljcV3G1HQgE85DGtzUzL4wK5N9NdqREs0rXSOpgzPI&_LANG=enus "A snippet of the Overview dashboard that shows the reports and dashboards that match the entered search query.")

The items that match the search criteria are listed in the dropdown, grouped by Reports and Dashboards:

- Select a Report to access the detailed report view, including a data
  visualization and an agreement list.
- Select a relevant Dashboard to view a collection of related reports. Dashboards can hold up to 20 reports.

## Your Recents

The Your Recents section provides a list of your recently accessed reports, including when you last viewed them:

![A snippet of the Overview dashboard that highlights a report in the Your Recents section.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Overview_Your-Recents-section_report-higlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4NzksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.YljcV3G1HQgE85DGtzUzL4wK5N9NdqREs0rXSOpgzPI&_LANG=enus "A snippet of the Overview dashboard that highlights a report in the Your Recents section.")

The five agreements that you most recently viewed are displayed. From this section, you can:

- Access all the recently viewed reports. Select the View all option to view
  all of your recently viewed reports.
- Go to the detailed report view for a selected report. Select a report name to access the detailed
  report view, including a data visualization and an agreement list.

  Tip: See these topics to learn more about viewing,

  - [Customize and Save Reports for Future Use](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=osx1727376530640.html&_LANG=enus "Learn how to customize and save an agreement report for future use.")

## Your Favorites

The Your Favorites section gives you quick access to your personalized list of starred dashboards and reports:

![A snippet of the Overview dashboard that highlights a report in the Your Favorites section.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Overview_Your-Favorites_report-higlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4NzksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.YljcV3G1HQgE85DGtzUzL4wK5N9NdqREs0rXSOpgzPI&_LANG=enus "A snippet of the Overview dashboard that highlights a report in the Your Favorites section.")

- Access all of the starred dashboards and reports. Select the View all option to view all of your favorite dashboards and reports.
- View to a specific dashboard or report. Select (click) a dashboard or report to access the
  corresponding view, either the dashboard or the report view. Data
  visualizations and agreement lists are displayed in the report
  view.

Select the Favorite (star) icon to "star" the dashboards and reports that you frequently use:

![A snippet of a detailed report view that highlights the Favorite (star) icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreements-dashboard_Report-details_Favorite-icon-selected_generic.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4NzksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.YljcV3G1HQgE85DGtzUzL4wK5N9NdqREs0rXSOpgzPI&_LANG=enus "A snippet of a detailed report view that highlights the Favorite (star) icon.")

Your "starred" dashboards and reports are displayed in the Your Favorites section. See the [Star Your Favorite Reports and Dashboards](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk78ef32ef-890c-4e97-96e6-87f9a251d640.html&_LANG=enus "Learn how to add your frequently used dashboards and reports to the Your Favorites section on the Overview dashboard.") topic to learn more about this feature.

## Suggested Reports (Beta)

The Suggested Reports section provides AI-generated custom report recommendations based on the
custom fields in your account. These auto-generated recommendations require a minimum of five data points. When fields (custom or standard) are located in at least five agreements, they are included when generating custom report suggestions.

Use the horizontal scroll bar to view all of the auto-generated custom report recommendations:

![A snippet of the Overview dashboard that shows the Suggested Reports (Beta) section.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Insights-reporting-hub_Overview_Suggested-Reports_scrollbar-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4NzksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.YljcV3G1HQgE85DGtzUzL4wK5N9NdqREs0rXSOpgzPI&_LANG=enus "A snippet of the Overview dashboard that shows the Suggested Reports (Beta) section.")

Use these suggestions to create custom reports. Select the report heading for the report that you want to create and use the Save option from the report view:

![A snippet of the report view for an AI-generated report that highlights the Save option in the More actions menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Insights-reporting_suggested-report_save-from-report-view.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4NzksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.YljcV3G1HQgE85DGtzUzL4wK5N9NdqREs0rXSOpgzPI&_LANG=enus "A snippet of the report view for an AI-generated report that highlights the Save option in the More actions menu.")

Note: The Favorite (star) option is not available for the custom report suggestions. Save the custom report and then use the Favorite (star) icon to add the report to the Your Favorites list.

See the [Create Custom Reports from Suggested Reports (Beta)](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tskfc9f9485-65b6-4a2c-95e3-15bd13ce8562.html&_LANG=enus "Learn how to create a custom report from one of the reports in the Suggested Reports (Beta) section on the Overview dashboard.") topic to learn more about this feature.

## Weekly Insights

The Weekly Insights section displays the agreement counts for all the agreements in the account, including All agreements, Expiring agreements, and Upcoming renewals.

![A snippet of the Overview dashboard that shows the Weekly Insights section.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Insights-reporting-hub_Overview_Weekly-Insights-section.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4NzksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.YljcV3G1HQgE85DGtzUzL4wK5N9NdqREs0rXSOpgzPI&_LANG=enus "A snippet of the Overview dashboard that shows the Weekly Insights section.")

Select the report heading to view the agreements that were included in the corresponding insight. See the [View and Download an Overview Report](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tska721c00d-28c9-4949-9ff2-ad0ba7a3d0e8.html&_LANG=enus "Learn how to view reports from the Overview dashboard, including accessing and downloading the detailed report view.") topic to learn more about viewing Overview reports.

## Review your agreements

The Review your agreements section provides a view into your agreements, including Agreements by renewal date and Agreement types:

![A screenshot of the Review your agreement section that shows the cursor hovered over a segment in the Agreement types chart.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Overview_Review-your-agreements_hover-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4NzksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.YljcV3G1HQgE85DGtzUzL4wK5N9NdqREs0rXSOpgzPI&_LANG=enus "A screenshot of the Review your agreement section that shows the cursor hovered over a segment in the Agreement types chart.")

Use this section to locate more specific agreement data. For example:

- View the document count for a segment of a bar or pie chart: Hover the cursor over a chart segment to see the document count and segment name.
- View the detailed report view, filtered for a selected chart segment: Select a segment in the bar or pie chart to go to the report view. The report opens and it is filtered to show the agreements that are associated with the selected segment.
- View a visual summary of all executed and completed agreements in your account: Select the
  View Agreements Dashboard option. The
  Agreements Dashboard opens, and various
  insights and reports are displayed.

See these topics to learn more about these features:

- [Agreements Dashboard](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=bec1702945765624.html&_LANG=enus "This topic answers the question: Where can I find data visualizations and reports for all the agreements in my Docusign account?")

![A snippet of the Overview dashboard that shows Resources to suggest new features and access help documentation.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Overview_Resources-section.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY4NzksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.YljcV3G1HQgE85DGtzUzL4wK5N9NdqREs0rXSOpgzPI&_LANG=enus "A snippet of the Overview dashboard that shows Resources to suggest new features and access help documentation.")

- Share your ideas for new features and up-vote other suggestions: Select the + Suggest
  a feature option to share an idea. The Docusign
  Community Ideas page opens, and a list of featured topics are
  displayed. For example, the [How to Use the Ideas Feature](https://community.docusign.com/community-announcements-5/how-to-use-the-ideas-feature-5450) topic.
- Review help documentation for using eSignature reports: Select the eSignature Report Help Docs option to read about eSignature reports for senders and administrators. For example, the [Using Reports](https://support.docusign.com/s/document-item?bundleId=cnu1642812012214&topicId=wou1578456440763.html) topic.
- Read about IAM dashboards and reports: Select the Agreement dashboard Help Docs option to learn more about using IAM dashboards and reports. For example, the [Agreement Dashboard](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=jyq1749000353004.html).
