# IAM Fields Manager

Read about the Fields manager, including its purpose, use, and role within the Intelligent Agreement Management (IAM) platform.

The *Fields Manager* is the single source of truth for the account's custom and standard
fields. It is the central place for administrators and contract managers to define,
organize, and manage the fields in their account. These custom and standard fields are
reused across the Docusign Intelligent Agreement Management (IAM)
platform.

This feature is accessed from the Agreement Manager homepage. Expand the Settings (gear) menu located on the Completed Documents page and select the Fields option:

## The 'Fields' table

The Fields table lists all the standard (Docusign-provided) and custom (user-defined) fields that are associated with the account:

![A snippet of the Manage Fields page that shows custom and standard fields in the Fields table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Fields-Manager_Manage-Fields_Fields-table.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1NTksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.lL_HWjQWWVzh5Mdz0I50wfXhcXd3lL6aS3mS2OrCuNo&_LANG=enus "A snippet of the Manage Fields page that shows custom and standard fields in the Fields table.")

- The Field column displays the name of the field and includes a field type indicator. For example, currency, date, dropdown, number, and text. Hover the cursor over the indicator to view the field type:![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/FIelds-Manager_Fields-table_Field-column_type-indicator-displayed.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1NTksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.lL_HWjQWWVzh5Mdz0I50wfXhcXd3lL6aS3mS2OrCuNo&_LANG=enus)

  Tip: Use the Sort items (arrows) icon, located in the column heading, to sort the table by the Field column.
- The Category column indicates the classification for the field. For example, General, Payment, Legal and Compliance, Renewal, Termination, Other, and Human Resources.
- The Created By column indicates whether the field is a custom (User) field or a standard (Docusign) field.
- The AI detection column displays the AI status for the field. For example, whether the field is Active or it must be Set Up. You can also Activate the field from this column.

## Search and filter options

Use the search field and table filters to locate the relevant fields:

![A snippet of the Manage Fields page that shows the search field and filter options for the Fields table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Fields-Manager_Fields-table_search-and-filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1NTksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.lL_HWjQWWVzh5Mdz0I50wfXhcXd3lL6aS3mS2OrCuNo&_LANG=enus "A snippet of the Manage Fields page that shows the search field and filter options for the Fields table.")

- Enter a search term in the Search fields... field. As you type the keyword, the table updates and the fields that match the search term are displayed.
- Use the Created By filter to view standard (Docusign) or custom (User) fields.
- Use the Category filter to view fields by classification. For example, General, Payment, Legal and Compliance, Renewal, Termination, Other, and Human Resources.
- Use the AI Status filter to view fields based on their AI detection status, Active or Needs Setup.

## Primary use cases

The *Fields Manager* helps administrators and contract managers to:

- **Standardize agreement data:** Define and manage shared custom and standard fields for use across Agreement Manager and other IAM products.
- **Manage field definitions in one place:** Create and manage custom fields without editing
  each agreement type or template individually.
- **Control where fields are used:** Add or remove fields from one or more agreement types at the same time. The same field can be reused across multiple agreement types.
- **Manage custom field extractions:** Review and refine the AI detection for custom fields, including refining definitions, updating examples, and testing the accuracy.

Tip: See the [Custom Field Extractions](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=condece8d7e-2da8-4e5f-8ba4-3f0175c2a625.html&_LANG=enus "Read about the custom field extractions, including entry points for creating new fields and a review of the AI detection setup wizard.") topic to learn more about creating custom field extractions.

## Fields Manager across the IAM platform

The *Fields Manager* is a shared IAM workspace for the unified fields that are used across several IAM products and workflows.

- **Agreement Manager:** The *Fields Manager* is the central place for administrators and contract
  managers to create and manage custom fields. The workspace also hosts
  the custom extraction wizard that is used to create custom fields. This
  feature is accessed from the Agreement Manager homepage.
- **Agreement Preparation:** The *Fields Manager* provides a unified fields library to the
  agreement preparation surfaces. For example, agreement templates and
  send experiences. Template builders and senders can pick from the same
  set of fields that are used in Agreement Manager. This ensures that the data
  captured during preparation flows into Agreement Manager as structured fields,
  reducing duplicate fields.
- **Agreement Desk:** The *Fields Manager* lets you define shared fields once and reuse them across request forms and approvals. This keeps intake consistent and improves processing speed and accuracy. For example, you can standardize fields like Customer Type or Renewal Date.
- **Web Forms and Integrations:** When external forms and connected systems write into IAM, they
  map directly into the structured *Fields Manager* fields. This
  connection ensures that the data is collected into the same unified
  fields that are used across the IAM platform.
- **Docusign CLM:** The *Fields Manager* provides a unified fields
  library into which CLM maps its field data. This ensures that CLM custom
  attributes are mapped appropriately and the fields are not
  recreated.
