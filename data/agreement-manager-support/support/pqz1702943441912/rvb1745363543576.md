# Update Custom Obligation Types

Learn how to rename your custom obligation types that more accurately reflect your obligations.

If necessary, you can rename your custom obligation types to support specific organizational or departmental use cases.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Expand the Settings (gear) menu from the right-side of the page and select the Obligations option:

   ![A screenshot of the Settings (gear) menu that highlights the Obligation option in the expanded](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Obligation-type-management_Manage-Obligation-Types_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxOTIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.RuHLJIoLlkt_Thfk6weCHZF-fNSECf0GhaWO2k1Fhes&_LANG=enus "A screenshot of the Settings (gear) menu that highlights the Obligation option in the expanded")

   The Manage Obligation Types page opens. The Obligation Types view is displayed. A list of custom and standard obligation types is displayed in the table.
5. Locate the custom obligation type that you want to update.

   Only user-created, custom obligation types can be updated. View the CREATED BY column to verify that the type is user-created.
6. Expand the More actions (three dots) menu and select the Rename option:

   ![A snippet of the Obligation Types table that shows the Rename option in the More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Obligation-type-management_Manage-Obligation-Types_Rename.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxOTIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.RuHLJIoLlkt_Thfk6weCHZF-fNSECf0GhaWO2k1Fhes&_LANG=enus "A snippet of the Obligation Types table that shows the Rename option in the More actions (three dots) menu.")

   The Rename obligation type dialog box opens.

   ![A screenshot of the Rename obligation type dialog box that shows the new name for the obligation type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Obligation-type-management_Rename-obligation-type-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxOTIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.RuHLJIoLlkt_Thfk6weCHZF-fNSECf0GhaWO2k1Fhes&_LANG=enus "A screenshot of the Rename obligation type dialog box that shows the new name for the obligation type.")
7. Enter the new name for the obligation type in the Name field and select the Save option.

   The dialog box closes and the Obligation Type view is displayed. The custom obligation type is listed in the table under the new name.
