# Refine the Detection for Custom Extractions

Learn how to refine the AI detection results for your custom extractions, including fine-tuning the definition and selecting new samples.

## Before you begin

The Refine Results function is available to custom extractions with a configured AI setup. See the [Custom Extractions](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=yrx1747083411531.html&_LANG=enus "Read about custom extractions, including how these data points bring value to our organization.") article to learn more about creating custom extractions.

Note: Administrators automatically have access to configure agreement data. For non-administrator users, this feature requires Organize and manage all agreements access to Agreement Manager. These tasks are primarily completed by administrators and contract managers.

During setup, the 'set definition' and 'example selection' steps are designed to "teach" the system. These steps enable the AI detection to "learn" by example. If you are not achieving the expected results, use the Refine Results option to adjust the AI setup.

Restriction: The Refine with AI option is only available for English (U.S.) and English (global) agreements. This function is used to fine-tune the definition for your custom extractions.

The Refine Results option is available for all custom extractions with a previously configured AI setup.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Go to the applicable component manager. Expand the Settings (gear) dropdown from the top right and select the relevant location, Fields or Agreement Types.

   ![A screenshot of the Settings dropdown that highlights the Fields and Agreement Types options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Types_More-actions-menu_expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3MzksInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.y0hmATrDwtQ_INeM9I-BZ6PtYHAfy_XMCxldoZdGUPE&_LANG=enus "A screenshot of the Settings dropdown that highlights the Fields and Agreement Types options.")

   Depending on your selection, either the Manage Agreement Types page or the Manage FIelds page opens.

**Find the applicable custom extraction**

5. Locate the extraction that you want to update.

   Tip: The Refine Results function is not available for the standard, Docusign-provided extractions. Apply the Created By filter to view User-created extractions.

   - Search for the extraction that you want to update. Enter a keyword in the search field and select a suggested search from the dropdown or press the Enter key.
   - Apply filters to locate the relevant extraction. Expand the filter, choose a filter value,
     and select the Apply option.

   The extractions that matched the search term and applied filters are displayed in the table.
6. Expand the More actions (three dots) menu for the applicable extraction and select the Refine Results option.

   A dialog box opens. You are warned that refining the result will deactivate the custom extraction. After retesting the updated setup, reactivate the extraction.
7. Select the Confirm option.

   The dialog box closes and the custom extraction is deactivated. The Set definition dialog box opens.

**Update the definition**

8. Update the definition for the custom extraction.

   Note: See the [Best Practice: Writing Definitions for AI Detection](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tmk1741317734856.html&_LANG=enus "Read about tips for writing good definitions for the AI detection setup process. These definitions are an important part of the process and help to \"teach\" the AI detection.") article to review the
   guidelines for writing definitions. Follow these guidelines to get
   the most out of the AI detection and to ensure accuracy.

   1. Enter a detailed description that explains the purpose of the extraction. This field is limited to 2,000 characters.

      When the definition exceeds 50 characters, the Refine with AI option becomes available. Use this option to update the definition for consistency and accuracy.
   2. Fine-tune the definition. Select the Refine with AI option. The dialog box refreshes and a suggested definition is displayed.
   3. Confirm the definition. Review and edit the suggested definition as needed.

      Tip: Use the Undo option to revert to the original version of the definition.
   4. Select the Next option.

   Note: Depending on the extraction being updated, you can be prompted to enter additional details. The descriptions help the AI model to identify the data points in your documents.

   In addition to a text definition, you are prompted to provide examples of the extraction. The definition explains what the data means and the examples show how the data displays in your agreements.

**Add examples for context**

Important:

When you provide example field values, include specific phrases or paragraphs so the AI
models can detect the data in a contextual manner. If you provide a single word, the model is unable
to detect the data due to lack of context.

9. Choose your preferred method for providing examples and select the Next option.
   - Select the Paste examples from your documents option to copy and paste your examples.
   - Select the Highlight examples directly in documents option to select your examples from existing documents.

   Depending on your selection, you are prompted to either paste examples of the extraction or select the relevant documents.
10. Provide examples of the extraction.
    - Paste examples of the extraction: For each example, add a paragraph of text that contains the extraction, then type out the value and confirm it. Select the Save option after adding your examples.
    - Highlight examples directly in documents: Search for and select the relevant documents. For each document, highlight a paragraph of text that contains the extraction, then type out the value and confirm it. Select the Next option after adding your examples.

      Your examples are transferred to the Paste examples of the field dialog box. Review your examples and select the Save option.

    A dialog box opens and a message confirms that the AI detection is now available for the extraction.

**Test for accuracy**

Before activating the AI detection, retest the setup to confirm its accuracy. The results are a good indicator of the quality and accuracy that you can expect from the AI detection before activating it.

11. Choose the Test for accuracy option and select the Confirm option. This option is selected by default.

    The Select relevant documents page opens. You are prompted to select documents that are likely to contain the extraction.

    Note: Include at least one document that does not contain the extraction. For example, a document with a similar data point to determine if the AI model detects the difference.
12. Test the AI detection against your selected documents:
    1. Locate the relevant documents. Perform a keyword search and apply filters to locate documents that are likely to contain the extraction.
    2. Select the documents that you want to include in the test.
    3. Select the Start Testing option in the bottom right.

    The Agreement preview opens for the first document. The pane on the left shows the extraction that the AI detection identified in the document.
13. Rate the accuracy of the findings.
    1. Review the identified extraction in the context of the document. Verify that the AI detection surfaced the correct value. Select Yes if the AI detection was correct or No if the extraction was not correctly identified.

       Note: If the extraction was not found in a selected document, a message is displayed. Be sure to include agreements that do not contain the extraction when testing the accuracy. It's important for the AI model to recognize when the extraction is not present.

       The Agreement preview page updates and the next document is displayed.
    2. Repeat this step until you are finished verifying the identified extractions in all the selected documents.
    3. Select the Done option.

    The AI detection test results are displayed. The target score is 80%. Use the Refine definition and examples option to improve the accuracy score.

**Activate the AI detection**

After achieving the target accuracy score, activate the AI detection for the custom extraction.

14. Choose the Activate AI detection option and select Confirm.

    The Activate AI detection dialog box opens. Use this dialog box to choose when the system should look for the custom extraction. The All new agreements option is selected by default. You cannot deselect this option.
15. Choose when to look for the custom extraction and select the Activate option.

    Note: The Find on specific agreements option can be used at any time to process your historical agreements for custom data points. See these articles to learn more about this function:
    - [Find Custom Agreement Types in Specific Documents](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tskb991328c-dc5f-4d45-9617-dada91a5672a.html&_LANG=enus "Learn how to use the Find On Specific Documents function to surface custom agreement types in selected agreements.")
    - [Find Custom Fields in Specific Documents](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk79824e36-d51d-4477-a5ca-7228cde30fb2.html&_LANG=enus "Learn how to use the Find On Specific Documents function to surface custom fields in selected agreements.")

    In this example, activate the AI detection without processing historical agreements.

    The dialog box closes and the extraction is published. You are returned to the previous page,
    the AI detection is activated for the extraction.
16. Return to the Agreement Manager homepage. Select the Close (X)
    icon located in the top right of the page.

    You are returned to the Agreement Manager homepage and the Completed Documents
    view is displayed.

You have successfully refined the AI detection results for a custom extraction.
