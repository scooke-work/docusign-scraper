# Onboarding Callouts for First-Time Users

This topic answers the question: Is there an onboarding experience for first-time users? Read about the walk-through experience for first-time Agreement Manager users.

Agreement Manager functionality is accessed from your Docusign account. Log in to Docusign and select Agreements from the global header.

The Docusign Agreements page opens. The Inbox view is displayed by default.

Select the Completed option from the AGREEMENTS section in the navigation pane to access Agreement Manager:

When new users first access Agreement Manager, a series of onboarding callouts is displayed.

## Onboarding callouts

As a new user, a series of onboarding callouts is displayed when you first access Agreement Manager. These callouts provide a quick tour of Agreement Manager and walk you through a guided flow.

Select Next to review each of the callouts or select Skip All to close the dialog boxes:

![A screenshot of the first onboarding callout that points out that you are viewing documents.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Onboarding_first-time-user_callout-1.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYwNzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.hJIL1rSA9dZSGhVfivtztD3qjanbmuf43nCWOt1W_ao&_LANG=enus)

This callout points out that you are viewing Documents, which are the individual files from your completed agreements.

Tip: Switch to Envelopes to view signatures and agreement history.

Select Next to launch the guided flow. This flow gives you a quick tour of Agreement Manager functionality. Review the table to learn more about each callout.

| **Callout** | **Description** |

At the last callout, select OK. The callout closes. You are currently on the Agreement preview page. From here, you can:

- Select Review All and perform a data review to validate the suggested attribute (field) values. See the [Validate Extracted Field Values (Complete a Data Review)](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=yqq1702945693043.html&_LANG=enus "This topic answers the question: How can I make sure that the suggested attribute (field) values are accurate? Learn how to complete a data review to verify suggested field values.") article to learn more about reviewing suggested values.
- Select Close (X) from the top-left corner to return to the Completed Documents page.
