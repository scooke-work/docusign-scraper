# Add Custom Fields to Agreement Types

Learn how to create and add custom fields to the custom and standard agreement types in your account.

When necessary, administrators and contract managers can add fields to the agreement types in their account. For example, if a use case requires additional data points.

Note: Administrators automatically have access to configure agreement data. For non-administrator users, this feature requires Organize and manage all agreements access to Agreement Manager. These tasks are primarily completed by administrators and contract managers.

Tip: Both custom and standard fields can be added to the agreement types in the account. Likewise,
these fields can be added to custom and standard agreement types.

There are a number of entry points for adding custom fields to agreement types. For example, from the Agreement Types Manager and the Agreement preview page. This topic reviews the steps to add custom fields from the Agreement Types Manager.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Expand the Settings (gear) dropdown from the top right of the page and select the Agreement Types option:

   ![A snippet of the homepage that highlights the Agreement Types option in the Settings (gear) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-types-manager_Settings-menu_Agreement-Types_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.etYGtRkrtjn_Q4xP7XvwWjccZ2ji0OuqnsQB2RGCuSo&_LANG=enus "A snippet of the homepage that highlights the Agreement Types option in the Settings (gear) menu.")

   The Manage Agreement Types page opens. A list of the account's standard and custom agreement types is displayed in the Agreement Types table.
5. Locate the agreement type that you want to update:

   ![A snippet of the Manage Agreement Types page that highlights the search field and filter options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Types_Agreement-Types_Fields_search-filters_generic.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.etYGtRkrtjn_Q4xP7XvwWjccZ2ji0OuqnsQB2RGCuSo&_LANG=enus "A snippet of the Manage Agreement Types page that highlights the search field and filter options.")

   1. Search for the relevant custom agreement type. Enter a keyword in the Search types... field. The table updates to show agreement types that match the entered search term.
   2. Apply filters to locate the relevant custom agreement type. For example, filter the table by Created By, Category, or AI Status. Expand a filter, choose the relevant value, and select the Apply option.

      The table updates and the agreement types that match the applied filters are displayed.
6. Select (click) the relevant agreement type from the results table.

   The Agreement type details view opens for the selected agreement type. The associated custom and standard fields are displayed in the Fields table.
7. Select the + Add Fields option:

   ![A snippet of the agreement type details that highlights the + Add Fields option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Types_Agreement-type-details_Add-Fields-option_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.etYGtRkrtjn_Q4xP7XvwWjccZ2ji0OuqnsQB2RGCuSo&_LANG=enus "A snippet of the agreement type details that highlights the + Add Fields option.")

   The Add a field dialog box opens.![A screenshot of the Add a field dialog box that is used to add a field to an existing agreement type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-type-management_create-type_add-a-field_dist-agreements-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.etYGtRkrtjn_Q4xP7XvwWjccZ2ji0OuqnsQB2RGCuSo&_LANG=enus "A screenshot of the Add a field dialog box that is used to add a field to an existing agreement type.")
8. Enter a name for the custom field in the Field name field.

   As you enter a name, a type-ahead dropdown list opens and fields that match the entered name
   are displayed.

   Tip: Review the type-ahead dropdown list to verify
   that the custom field does not already exist. If the field exists, you are
   unable to create a new one with the same name.
9. Select the + Create [Field name] option to create a custom field:

   ![A snippet of the Add a field dialog box that shows an example of a custom field name being created.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-type-management_add-custom-field_example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.etYGtRkrtjn_Q4xP7XvwWjccZ2ji0OuqnsQB2RGCuSo&_LANG=enus "A snippet of the Add a field dialog box that shows an example of a custom field name being created.")
10. Expand the Field type dropdown and select the type of field that you are adding:

    ![A snippet of the Add a field dialog box that shows the available field types for your custom field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Add-custom-field_Field-type-options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.etYGtRkrtjn_Q4xP7XvwWjccZ2ji0OuqnsQB2RGCuSo&_LANG=enus "A snippet of the Add a field dialog box that shows the available field types for your custom field.")

    Choices are: Text, Number,
    Date, and
    Dropdown.

    Note: If you select
    Dropdown as the field type, enter options
    for the dropdown in the Options text-entry box:![A snippet of the Add a field dialog box that shows the required Options field for adding a dropdown field type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Add-custom-field_Options-for-dropdown-type.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.etYGtRkrtjn_Q4xP7XvwWjccZ2ji0OuqnsQB2RGCuSo&_LANG=enus "A snippet of the Add a field dialog box that shows the required Options field for adding a dropdown field type.")**Important:** Use a semicolon to separate each option.
11. Apply the field to the relevant agreement types. Click in the Agreement types field and select the agreement types to which you want to add the field. The current agreement type is displayed in the Agreement types field by default.

    ![A snippet of the Add a field dialog box that shows the expanded Agreement types field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-type-management_Add-a-field_Agreement-types-Dist-Agreement-ex.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.etYGtRkrtjn_Q4xP7XvwWjccZ2ji0OuqnsQB2RGCuSo&_LANG=enus "A snippet of the Add a field dialog box that shows the expanded Agreement types field.")Choose the relevant agreement types and select the Apply option.
12. Expand the Category dropdown and select a classification for the custom field:

    ![A snippet of the Add a field dialog box that shows the Category options for the new field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Add-custom-field_Category-options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.etYGtRkrtjn_Q4xP7XvwWjccZ2ji0OuqnsQB2RGCuSo&_LANG=enus "A snippet of the Add a field dialog box that shows the Category options for the new field.")Choices are: General, Termination, Renewal, Payment, Legal and Compliance, Human Resources, and Other.
13. Select the Save option.

    The dialog box closes. The  Agreement type details view opens for the selected agreement type. The custom field is displayed in the field list table.

    ![A snippet of a custom agreement type details page that highlights the new custom field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-agreement-types_Custom-field-added-to-custom-type-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.etYGtRkrtjn_Q4xP7XvwWjccZ2ji0OuqnsQB2RGCuSo&_LANG=enus "A snippet of a custom agreement type details page that highlights the new custom field.")

    When you first create a custom field, the AI detection is inactive by default. Complete the AI detection setup process so that data for the field is surfaced in your documents. See the [Custom Field Extractions](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=condece8d7e-2da8-4e5f-8ba4-3f0175c2a625.html&_LANG=enus "Read about the custom field extractions, including entry points for creating new fields and a review of the AI detection setup wizard.") topic to learn more about activating your custom fields for AI detection.
14. Return to the Agreement Types view. Select the Agreement Types option from the breadcrumb navigation in the top left.

    You are returned to the Manage Agreement Types page. The Agreement Types view is displayed.
15. Return to the Agreement Manager homepage. Select the Close (X) icon from the top right.

    The Agreement Manager homepage opens and the Completed Documents view is displayed.

You have successfully added a custom field to an existing agreement type. The next step is to enable the AI detection for the custom extraction. See the [Enable AI Detection for Your Custom Fields](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk02bd31a2-b125-4d53-b632-20d9972163a8.html&_LANG=enus "Learn how to enable the AI detection for your custom fields. Once activated, the field is surfaced in your new agreements automatically.") topic to start the AI detection set-up process.
