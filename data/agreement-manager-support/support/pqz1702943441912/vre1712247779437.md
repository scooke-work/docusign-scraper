# Data Review Onboarding Experience

Read about the first-time onboarding experience for data review, including a walk-through of the review process.

AI-assisted insights help you to find answers to critical business questions quicker, mitigate risk, and accelerate document review. Data review is an important part of using Agreement Manager as it ensures that users can make business decisions based on accurate agreement data.

As a new user, when you first access Docusign Agreement Manager, a series of onboarding callouts is displayed. This guided flow gives you a quick tour of Agreement Manager functionality, including data reviews.

Note: See the [Onboarding Callouts for First-Time Users](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=kpg1722973773604.html&_LANG=enus "This topic answers the question: Is there an onboarding experience for first-time users? Read about the walk-through experience for first-time Agreement Manager users.") topic to learn about the guided Agreement Manager tour.

The last callout in the guided flow takes you to the Details pane on the Agreement preview page. You are prompted to Review new data suggestions:

![A snippet of the Agreement preview page showing an onboarding callout prompting you to review new data suggestions.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-preview-page_snippet-Agreement-details_callout-1.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2MTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.skQ7q01iIqU4UL9mcbYr7HQj9x7PMZAFNxDehZb-Jko&_LANG=enus "A snippet of the Agreement preview page showing an onboarding callout prompting you to review new data suggestions.")

Select OK to continue. The callout closes.

The next callout is displayed once you put the Details pane into *edit* mode. This callout prompts you to view the suggested details in the document.

![A snippet of the Agreement preview page that shows the onboarding callout that opens when you place the page into edit mode.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-preview-page_Review-and-edit-details-pane_callout-2.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2MTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.skQ7q01iIqU4UL9mcbYr7HQj9x7PMZAFNxDehZb-Jko&_LANG=enus "A snippet of the Agreement preview page that shows the onboarding callout that opens when you place the page into edit mode.")

The attribute values that were surfaced with AI are clickable. Select the suggested value to view the value in the context of the agreement:

![A snippet of the Agreement preview page that shows the suggested value in the context of the agreement.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Onboarding_data-review_view-value-in-doc.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2MTQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.skQ7q01iIqU4UL9mcbYr7HQj9x7PMZAFNxDehZb-Jko&_LANG=enus "A snippet of the Agreement preview page that shows the suggested value in the context of the agreement.")

The suggested value is highlighted in the agreement: Review the value in the context of the agreement to determine if it is accurate.

Tip: Use the *Bulk Metadata Editing* feature to overwrite the extracted attribute values with user-provided agreement data. With this feature, you can edit the metadata for up to 1,000 selected documents in a single processing job. See the [Bulk Metadata Editing](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=concb493b18-adf8-4fe5-8b99-dd90cbc28399.html&_LANG=enus "Read about the Bulk Metadata Editing feature, including use cases, typical roles and permissions, and a process overview.") topic to learn about more about this feature.
