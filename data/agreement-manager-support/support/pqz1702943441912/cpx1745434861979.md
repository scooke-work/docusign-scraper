# View Assigned Obligations

Learn how to view your assigned obligations.

Assigned obligation owners are not notified when an obligation is assigned to them. However, you can find assigned obligations from the Obligations dashboard by applying filters.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Insights from the global header.

   The Insights hub opens and the Overview dashboard is displayed.
3. Select Obligations from the navigation pane on the left. It is located in the Dashboards section.

   The Obligations dashboard view opens.
4. Filter the dashboard to locate your assigned obligations:

   ![A screenshot of the Obligation Owner filter that is expanded to show the search field and user names.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Obligation-Management_View-assigned_Select-Obligation-Owner.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxNjcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Dolht3X7mtyo7kL8zu26gSkB-wq60XG-ZvMRCFqPp64&_LANG=enus "A screenshot of the Obligation Owner filter that is expanded to show the search field and user names.")

   1. Expand (select) the Obligation Owner filter.

      The Obligation Owner filter opens.
   2. Enter the relevant name in the Search field.

      Users that match the entered search word are displayed in a type-ahead dropdown list.

      Choose the relevant name in the list and select the Apply option.

   The Obligations dashboard updates and obligations assigned to the selected owner are reflected in the charts.
5. Access the detailed report view for the Obligations by type chart.

   ![A snippet of the Obligations dashboard that highlights the View Report option for the Obligations by type chart.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Obligation-Management_View-assigned_View-obligations-by-type-report.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxNjcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Dolht3X7mtyo7kL8zu26gSkB-wq60XG-ZvMRCFqPp64&_LANG=enus "A snippet of the Obligations dashboard that highlights the View Report option for the Obligations by type chart.")

   - Select the Obligations by type chart heading.
   - Expand the More actions (three dots) menu and select the View Report option.

   The detailed report view opens for the selected chart. Your assigned obligations are represented in the chart and listed in the table.
6. Apply filters and customize the columns in the table as needed.
7. Choose to download or save the report.

   ![A snippet of the report view that highlights the More actions (three dots) menu used to download or save the report.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Report-view_Dropdown-expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxNjcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Dolht3X7mtyo7kL8zu26gSkB-wq60XG-ZvMRCFqPp64&_LANG=enus "A snippet of the report view that highlights the More actions (three dots) menu used to download or save the report.")

   - Expand (select) the More actions (three dots) menu from the top right and select the Download option.

     The report is downloaded to your local Downloads folder.
   - Expand (select) the More actions (three dots) menu from the top right and select the Save As option. The Save Report dialog box opens.

     Enter a Report name and select the Confirm option. The report is saved to the Custom tab on the Reports view.
8. Set up one-time or recurring email report notifications. Select the Manage notifications (bell) icon and complete the Create subscription dialog box.

   ![A screenshot of the Create subscription dialog box that is used to subscribe to report notifications.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Create-Subscription.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxNjcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Dolht3X7mtyo7kL8zu26gSkB-wq60XG-ZvMRCFqPp64&_LANG=enus "A screenshot of the Create subscription dialog box that is used to subscribe to report notifications.")

   See the [Subscribe to Receive Report Notifications](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=wfw1734545937270.html&_LANG=enus "Learn how to set up one-time or recurring email reports for yourself or others to receive notifications about your obligations, renewals, or agreements.") topic to learn more about this feature.
9. Return to the Obligations dashboard. Select the Obligations option from the breadcrumb navigation at the top of the page:

   ![A snippet of the Obligations by type report that highlights the breadcrumb navigation.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Report-dashboard_Review-view_breadcrumb-nav-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxNjcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Dolht3X7mtyo7kL8zu26gSkB-wq60XG-ZvMRCFqPp64&_LANG=enus "A snippet of the Obligations by type report that highlights the breadcrumb navigation.")

   You are returned to the Obligations dashboard.

You have successfully reviewed your assigned obligations.
