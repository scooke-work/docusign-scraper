# Update the Agreement Type for Selected Agreements

Learn how to update the agreement type value for a group of selected agreements.

When documents are ingested or uploaded to Agreement Manager, the agreement type is identified and the associated fields are surfaced. When the type cannot be identified, the agreements can be assigned as *Miscellaneous*. You can update the agreement type for a group of agreements using the *bulk update* feature. For example, update a group of "miscellaneous" agreements to a custom agreement type.

Note: When updating the agreement type, the agreement is processed again to surface content that is relevant to the selected type.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. A list of documents to which the logged-in user has access is displayed in the table.
4. Locate the agreements that you want to update. In this example, filter the agreement list table by Agreement Type.

   ![A screenshot of the Agreement Type filter that shows a selected agreement type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents_Filters_Agreement-Type_Misc-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1MTIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.OwZf_lBEK7q2X3D2wSIpQZtVNYtOaGFq0l1T7Z8URVc&_LANG=enus "A screenshot of the Agreement Type filter that shows a selected agreement type.")

   1. Expand (click) the Filters option and select the Agreement Type filter.

      The Agreement Type filter expands and available types are displayed.
   2. Search for the relevant agreement type. Enter a keyword or search string in the Search field.

      As you enter the search criteria, the list of agreement types updates to match the keyword.
   3. Select the agreement type by which you want to filter your documents.
   4. Select the Apply option.
   5. Repeat this step until the table contains the agreements that you want to update. For example, filter your completed documents by Status or Parties.

   The agreement list table updates, and agreements that match the applied filters are displayed.
5. Select the agreements that you want to update from the list.

   ![A snippet of the agreement list table that shows a group of selected agreements.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Bulk-update-agreement-type_select-agreements.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1MTIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.OwZf_lBEK7q2X3D2wSIpQZtVNYtOaGFq0l1T7Z8URVc&_LANG=enus "A snippet of the agreement list table that shows a group of selected agreements.")

   - Select the individual agreements that you want to update.
   - Select the Choose all checkbox from the header row.

     All the agreements on that page of results are selected.
   - Once you select at least one agreement, the Selected dropdown is displayed. Expand the Selected dropdown and select the Select all that match your filters option.

     All the agreements in the table that match the applied filters are selected. The selection extends beyond the current page of results.
6. Expand the Edit (pencil) dropdown and select the Agreement Type option:

   ![A snippet of the agreement list table that highlights the Agreement Type option in the Edit dropdown menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Edit-option_Agreement-Type_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1MTIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.OwZf_lBEK7q2X3D2wSIpQZtVNYtOaGFq0l1T7Z8URVc&_LANG=enus "A snippet of the agreement list table that highlights the Agreement Type option in the Edit dropdown menu.")The Edit agreement type dialog box opens.
7. Specify the new agreement type for the selected agreements:

   ![A screenshot of the Edit agreement type dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Bulk-update-agreement-type_Edit-agreement-type-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1MTIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.OwZf_lBEK7q2X3D2wSIpQZtVNYtOaGFq0l1T7Z8URVc&_LANG=enus "A screenshot of the Edit agreement type dialog box.")

   1. Enter a keyword in the Select a type field to locate the relevant agreement type.

      As you enter the search criteria, a list of agreement types that matched the keyword are displayed.
   2. Select the relevant Agreement type from the results list.
   3. Select the Save option.

   The Change agreement type dialog box opens.

   ![A screenshot of the Change agreement type dialog box that explains the agreements will be processed for the selected type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Bulk-update-agreement-type_Change-agreement-type-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc1MTIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.OwZf_lBEK7q2X3D2wSIpQZtVNYtOaGFq0l1T7Z8URVc&_LANG=enus "A screenshot of the Change agreement type dialog box that explains the agreements will be processed for the selected type.")

   Important: This feature is available to users with specific agreement-level or Agreement Manager-level permissions:
   - Users with Can Manage or Can Edit access to an agreement can update that agreement's type.
   - Users with Organize and manage all agreements access to Agreement Manager can update the agreement type for any of the agreements in the account.

   The document name is grayed out and is not included in the update if you do not have the appropriate access.
8. Confirm the proposed changes and select the Update agreements option.

   The dialog box closes, and the Completed Documents page is displayed. The agreements are reprocessed to surface content related to the agreement type that you selected.

   Tip: The results of the processing job are available from the job monitoring hub. See the [View and Download Processing Results](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=cvd1742251951627.html&_LANG=enus "Learn how to view and download results for processing jobs.") topic for more information about job monitoring.

You have successfully updated the agreement type for a group of selected agreements.
