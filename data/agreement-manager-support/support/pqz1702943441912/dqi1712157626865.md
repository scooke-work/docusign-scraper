# Grant Users Access to Bulk Upload Documents

This topic answers the question: How do I give users access to bulk upload agreements to
Agreement Manager? Learn how to give users access to upload more than five files to
Agreement Manager.

The Upload function is available to
all Agreement Manager users. With this function, users can upload up to five third-party
documents. However, to upload more than five documents, you need an extra
permission. The *bulk upload* permission is granted by an administrator from
the Admin menu. This permission is assigned manually from the
User profile page or with a permission profile.

Once granted, users can upload up to 1,000 files at a time to
Agreement Manager. Additionally, this permission also gives users access to upload documents
from third-party cloud storage providers.

Important: The *bulk upload* permission is set at the user-level. This
permission is not available at the group-level. Keep in mind that user and group permissions are managed at the account-level as opposed to the organization-level.

1. Log in to your Docusign account using administrator
   credentials.

   The Docusign Home page opens.
2. Select Admin from the global header.

   The Admin menu opens and the Accounts view is displayed.

**Select an account (for organizations with multiple accounts)**

Important: User and group permissions are managed at the *account*-level. Make sure that you are updating records from Accounts administration instead of Organization administration. If your organization has multiple accounts, verify that you are updating the correct account.

3. Expand the Accounts dropdown from the top of the navigation pane and select the relevant account:

   ![A snippet of the Accounts view that highlights the Accounts dropdown in the pane on the left.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Permissions_Admin-menu_Accounts-dropdown_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzMDUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.8Apv5tFph-y2j5aGsYr6IITD0bAeIibNZUFlt4U50xc&_LANG=enus "A snippet of the Accounts view that highlights the Accounts dropdown in the pane on the left.")

   The Accounts view refreshes and the Overview page is displayed for the selected account. Locate the user that you want to update.

**Locate the relevant user**

Bulk permissions are granted at the user-level. This permission is not available at the group-level. Locate
the user that you want to update.

4. Select the Users option from the USERS AND GROUPS section in the navigation pane.

   The Users page opens and the account's users are displayed in the table.
5. Locate the user that you want to update:
   1. Enter the name of the user in the Search Here field.

      A dropdown list of search results is displayed.
   2. Select a name from the type-ahead dropdown or press the Enter key.
   3. Adjust the filters as needed. Filter options include: Status, Permissions, and Groups.

   The Users page refreshes and the users who matched the search criteria are displayed.

**Update Agreement Manager permissions**

6. Expand (select) the Actions option for the relevant user and select the Manage Agreement Manager Access item.

   The User profile page opens and the Agreement Manager Access tab is displayed by default.
7. Assign bulk upload permissions to the selected user. Select the Upload up to 1,000 files at a time toggle so it is in the "on" (blue) position.

   ![A snippet of the User profile page that highlights the bulk upload permission in the Agreement Manager access section.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Navigator-Access_Bulk-upload-toggle_enabled.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzMDUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.8Apv5tFph-y2j5aGsYr6IITD0bAeIibNZUFlt4U50xc&_LANG=enus "A snippet of the User profile page that highlights the bulk upload permission in the Agreement Manager access section.")

   This permission gives users access to *bulk upload* up to 1,000 files at a time. Users also have access to upload agreements to Agreement Manager from third-party cloud storage locations.

   Tip: Turn this toggle off (white position) to revoke the user's
   access to bulk upload agreements to Agreement Manager.
8. Select the Save Changes option.

   A message box opens to confirm that the Agreement Manager access changes were saved.

You have successfully granted bulk upload permission to a
selected user.
