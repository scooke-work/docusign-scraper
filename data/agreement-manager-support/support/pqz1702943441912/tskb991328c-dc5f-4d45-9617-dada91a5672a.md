# Find Custom Agreement Types in Specific Documents

Learn how to use the Find On Specific Documents function to surface custom agreement types in selected agreements.

## Before you begin

Once activated, the custom extraction is surfaced in your new agreements by default. The data points are included in your agreement insights, such as filters, searches, dashboards, and reports.

Administrators and contract managers can also process specific historical document for the new agreement type.

Remember: Administrators automatically have access to configure agreement data. For non-administrator users, agreement type management tasks require Organize and manage all agreements access to Agreement Manager agreements and functionality.

Be sure to complete these topics before attempting this task:

- [Configure New Custom Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=gke1727282238086.html&_LANG=enus "Learn how to create custom agreement types to support organizational use cases.")
- [Define Your Custom Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tske5fe1ad5-2832-4e6c-9177-f33a9ffade15.html&_LANG=enus "Learn how to initiate the AI detection set-up process and complete the first step to define the custom type.")
- [Add Examples of Your Custom Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tskd38465c6-c8c0-4588-bfb3-fdb6916a1926.html&_LANG=enus "Learn how to complete the second step of the AI detection set-up process, Add examples of this agreement type.")
- [Test the Extraction Accuracy for Your Custom Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=ogw1740503179389.html&_LANG=enus "Learn how to test the accuracy of the AI detection for your custom agreement types.")
- [Activate Your Custom Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=cgn1740452596841.html&_LANG=enus "Learn how to activate the AI detection for a custom agreement type and surface it in selected historical agreements.")

The Find On Specific Documents function is used to reprocess
selected documents for custom extractions, such as custom types. If the
custom type is located in the document, the associated attributes (fields)
are surfaced.

Important: The selected documents are reprocessed
*only* for that specific custom agreement type.
If the type is not found in the document, there is no
further processing.

This task is completed from the *Agreement Types Manager*.
Access this feature from the  homepage under the Settings
menu.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Expand the Settings (gear) menu from the
   top right and select the Agreement Types
   option:

   ![A snippet of the Agreement Manager homepage that highlights the Agreement Types option in the Settings (gear) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-types-manager_Settings-menu_Agreement-Types_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.iiZshJRf0s9EvzpItoxT4gg4BNkgo07aVtwtKwY5LJE&_LANG=enus "A snippet of the Agreement Manager homepage that highlights the Agreement Types option in the Settings (gear) menu.")

   The Manage Agreement Types page opens. A list of the account's standard and custom agreement types is displayed in the Agreement Types table.
5. Locate the custom agreement type that you want to find in specific documents.

   ![A snippet of the Agreement Types view that highlights the search field and applied filters to locate active custom types.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Types-Manager_Find-on-specific-fields_locate-active-custom-type.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.iiZshJRf0s9EvzpItoxT4gg4BNkgo07aVtwtKwY5LJE&_LANG=enus "A snippet of the Agreement Types view that highlights the search field and applied filters to locate active custom types.")

   - Search for the relevant custom agreement type. Enter a keyword in the Search types... field. The table updates to show agreement types that match the entered search term.
   - Apply filters to locate the relevant custom agreement type. For example, apply the Created By (User) and AI Status (Active) filters to view eligible agreement types. Expand a filter, choose the relevant value, and select the Apply option.

     Restriction: The Find On Specific Documents option is available only for custom, user-defined agreement types with an activated AI detection. Use the Created By and AI Status filters to view all the active custom agreement types for the account.

     The table updates and the agreement types that match the applied filters are displayed.
6. Expand the More actions (three dots) menu for the relevant agreement type and select the Find On Specific Documents option:

   ![A snippet of the Agreement Types table that shows the Find On Specific Documents option in the More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Types-Manager_More-actions_Find-On-Specific-Documents_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.iiZshJRf0s9EvzpItoxT4gg4BNkgo07aVtwtKwY5LJE&_LANG=enus "A snippet of the Agreement Types table that shows the Find On Specific Documents option in the More actions (three dots) menu.")

   The Select documents to process page opens and a list of agreements is displayed. You are prompted to select at least one document to process for the agreement type.
7. Choose the documents that you want to process for the custom type:

   ![A screenshot of the Select documents to process page that shows selected agreements to process for the new type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Custom-agreement-type_Select-docs-to-process_Start-Processing.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.iiZshJRf0s9EvzpItoxT4gg4BNkgo07aVtwtKwY5LJE&_LANG=enus "A screenshot of the Select documents to process page that shows selected agreements to process for the new type.")

   1. Search for the relevant agreements. Enter a keyword in the search field and select a search suggestion from the dropdown or press the Enter key.

      The table updates and the agreements that matched the search term are displayed.
   2. Refine the results list with filters. Expand a filter, choose the relevant values, and select the Apply option.

      The table updates and the agreements that matched the applied filters are displayed.
   3. Select the documents that you want to process.

      A counter indicates the number of selected documents.

      Tip: Use the Show Selected Agreements option to view your selected documents.
8. Select the Start Processing option in the lower right.

   The Agreement Types view opens. A message box opens and indicates that the selected documents are being checked for the custom type.

   ![A screenshot of the confirmation message that states the documents are being checked for the custom type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Types-Manager_Find-on-specific-fields_confirmation-message_view-progress.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.iiZshJRf0s9EvzpItoxT4gg4BNkgo07aVtwtKwY5LJE&_LANG=enus "A screenshot of the confirmation message that states the documents are being checked for the custom type.")
9. Review the processing status. Select the View Progress link from the message box.

   Note:

   **Navigate directly to the job details.**

   You can access the processing status and
   details for any of your processing jobs from the job
   monitoring hub. Access this feature from the
   Agreement Manager homepage. Expand the Settings (gear) menu and select the
   Uploads and Processing
   Status option.

   The job monitoring
   hub opens and the All jobs
   view is displayed. Go to the
   Processing tab and select
   the View details option for
   the job that you want to review:

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Types-Manager_Find-on-specific-fields_navigate-to-job-details-manually.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.iiZshJRf0s9EvzpItoxT4gg4BNkgo07aVtwtKwY5LJE&_LANG=enus)

   The job monitoring hub opens, and the Processing status and Job details sections are displayed.

   ![A screenshot of the Processing status and Job details for the job that is checking the selected agreements.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Types-Manager_Find-on-specific-docs_Processing-results.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.iiZshJRf0s9EvzpItoxT4gg4BNkgo07aVtwtKwY5LJE&_LANG=enus "A screenshot of the Processing status and Job details for the job that is checking the selected agreements.")
10. Review the processing job results, including any error messages.

    Tip: Select the View File option to view the document from the Agreement preview page. If the custom type is found in any of the documents, they are reprocessed to surface the relevant attributes and values. Once processed, the Details pane reflects the data points that are associated with the custom type.
11. Return to the Agreement Types view. Select the Close (x) icon located in the top right.

    You are returned to the Agreement Types view.
12. Return to the Agreement Manager home page. Select the Close (x) icon located in the top right.

    The Agreement Manager homepage opens and the Completed Documents view is displayed.

    If the custom type was identified in any of the selected documents, the associated attributes were surfaced. When you apply filters that are related to the custom type, the documents are included in the results.
