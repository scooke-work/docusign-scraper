# View the Overview Dashboard for Selected Parties and Groups

Learn how to view a visual summary of all agreements and obligations that are tied to a
specific party or alias group.

The "party hub" provides a unified summary of all agreements and obligations that are
tied to parties and groups. Access the party hub from various places in Agreement Manager, including:

- The Parties column (Completed Documents
  view)
- The Details pane (Agreement preview page)
- The Name column (Parties page)

Note: This topic reviews the steps to access the party hub from
the Completed Documents view. You can also search for and
select the relevant party from the Parties page.

![A snippet of the Parties page that highlights the Search parties field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Parties-page_Search-parties-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vfO-rsXGliffUpJjfdo3EUhhsuKKAKYbz3DlWAKOOJU&_LANG=enus "A snippet of the Parties page that highlights the Search parties field.")

- Go to Home  >  Agreements and select All from the PARTIES section in the navigation pane.
- Enter search criteria in the Search parties
  field.
- Select a party from the search results.

The party hub opens for the selected party or group.

**Access the party hub from the Completed Documents view:**

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign
   Agreements page opens. The Inbox view
   is displayed by default.
3. Select Completed from the
   AGREEMENTS section in the navigation pane.

   The Completed Documents view opens. Filter this view
   for a specific party.
4. Filter the agreement list table for a specific party:

   ![A screenshot of the Parties filter that shows the Search field, Party Role dropdown, and the party list.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Parties-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vfO-rsXGliffUpJjfdo3EUhhsuKKAKYbz3DlWAKOOJU&_LANG=enus "A screenshot of the Parties filter that shows the Search field, Party Role dropdown, and the party list.")

   1. Expand the Filters option and select the Parties filter.
   2. Enter a keyword or search string in the Search field.

      As you enter search criteria, the list of parties updates to match the keyword.
   3. Select the relevant party.

      Important: When using the Parties filter, you must also select a
      Party Role. If you do not know
      which role to select, accept the default value, All.
   4. Apply the Party Role filter to view agreements where the selected party plays a specific role. If you do not know which role to select, accept the default value, All.

      Select a role from the Party Role dropdown. Choose from: Buyer, Seller, or Other.
   5. Select the Apply option.

   The table updates and agreements that are tied to the selected party or group are
   displayed. The names in the Parties column are clickable
   and link directly to the party hub.

   ![A snippet of the Completed Documents view that highlights the Parties column as an entry point for the party hub.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Entry-point_Completed-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vfO-rsXGliffUpJjfdo3EUhhsuKKAKYbz3DlWAKOOJU&_LANG=enus "A snippet of the Completed Documents view that highlights the Parties column as an entry point for the party hub.")
5. Select a name from the Parties column.

   The party hub opens for the selected party or group. The
   Overview tab is displayed by default.

   ![A screenshot of the party hub's Overview dashboard that shows high-level agreement insights for a selected party or group.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-management_Party-hub_Overview_Fontara-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vfO-rsXGliffUpJjfdo3EUhhsuKKAKYbz3DlWAKOOJU&_LANG=enus "A screenshot of the party hub's Overview dashboard that shows high-level agreement insights for a selected party or group.")

   The Overview dashboard displays a visual summary of all
   active agreements that are tied to a specific party or alias group.

   Tip: Use the Favorites (star) icon to "star" your most active, time-sensitive, or high-value parties and groups. Each user can add up to 50 parties and groups to their Starred list.

   Your "favorite" parties and groups are listed under the Starred tab on the Parties page.

**Filter the Overview dashboard:**

6. Filter the dashboard by Agreement Type and
   Time Period:

   ![A snippet of the Overview dashboard that highlights the Agreement Type and Time Period filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Overview-dashboard_Filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vfO-rsXGliffUpJjfdo3EUhhsuKKAKYbz3DlWAKOOJU&_LANG=enus "A snippet of the Overview dashboard that highlights the Agreement Type and Time Period filters.")

   1. Expand the filter.
   2. Select a filter value.
   3. Select the Apply option.

   The selected filters are applied to the entire dashboard.

   Note:

   Use the Party Role dropdown to filter the dashboard to show agreement insights where the selected party plays a specific role in an agreement:![A screenshot of the Party Role filter that shows the role options, Buyer, Seller, and Other.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Overview-dashboard_Party-Role-filter_expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vfO-rsXGliffUpJjfdo3EUhhsuKKAKYbz3DlWAKOOJU&_LANG=enus "A screenshot of the Party Role filter that shows the role options, Buyer, Seller, and Other.")

**View the Party Summary and download a report:**

7. Review the Party Summary section to view agreement insights for the selected party or group, including:

   ![A snippet of the Overview dashboard that shows the Party Summary section for a specific party or alias group.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Overview-dashboard_Party-Summary.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vfO-rsXGliffUpJjfdo3EUhhsuKKAKYbz3DlWAKOOJU&_LANG=enus "A snippet of the Overview dashboard that shows the Party Summary section for a specific party or alias group.")

   - Summary statement about the selected party or group
   - Agreement counts based on the next 90 days
     - All active agreements
     - Expiring soon
     - Up for renewal

   1. Select a tile from the In the Next 90 Days section to view the list of associated agreements. For example, view a list of Active agreements:

      ![A screenshot of the In the Next 90 Days section that shows the cursor hovering over the Active agreement tile.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Overview-dashboard_In-the-Next-90-Days_tile-selected.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vfO-rsXGliffUpJjfdo3EUhhsuKKAKYbz3DlWAKOOJU&_LANG=enus "A screenshot of the In the Next 90 Days section that shows the cursor hovering over the Active agreement tile.")

      The report view opens and a list of relevant agreements are displayed
      in the table.
   2. Use the filters to refine the agreement list as needed.

      Note: Select the All
      Filters option to choose additional filters to apply
      to the report.
   3. Select the Customize columns (toggle) icon and
      choose the columns to display in the table:

      ![A screenshot of the Customize columns dialog box that shows the columns you can choose to display in the agreements table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Overview-dashboard_Customize-columns.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vfO-rsXGliffUpJjfdo3EUhhsuKKAKYbz3DlWAKOOJU&_LANG=enus "A screenshot of the Customize columns dialog box that shows the columns you can choose to display in the agreements table.")
      - Select the toggles to turn the associated field (column) on (blue) or off
        (white).
      - Re-order the columns as needed. Hover the cursor over the Reorder (dots) icon and drag the field (column) to the appropriate location.
      - Select the Save option.

      The dialog box closes and the columns are updated according to your
      configuration.
   4. Expand the More actions (three dots) menu from the top right and select
      Download:

      ![A snippet of the Active agreements report that highlights the Download option in the More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Parties-page_Overview-dashboard-report_Download-option.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vfO-rsXGliffUpJjfdo3EUhhsuKKAKYbz3DlWAKOOJU&_LANG=enus "A snippet of the Active agreements report that highlights the Download option in the More actions (three dots) menu.")

      The associated report is downloaded to your local
      Downloads folder.
   5. Select the party name from the breadcrumb navigation at the top of the
      page.

      You are returned to the Overview
      dashboard.

**View Recent activity:**

8. Review the Recent activity section to view your most recently active agreements over a 12-month period:

   ![A screenshot of the Recent activity section that lists the party's most recently active agreements over a 12-month period.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Overview-dashboard_Recenty-activity-section.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vfO-rsXGliffUpJjfdo3EUhhsuKKAKYbz3DlWAKOOJU&_LANG=enus "A screenshot of the Recent activity section that lists the party's most recently active agreements over a 12-month period.")

   Agreements are included in the chart based on the effective and expiration dates. Documents without effective or expiration dates are not included.

   1. Use the Previous Month and Next Month arrows to scroll through each month in the chart.
   2. Hover your cursor over an item in the chart to view high-level agreement details, such as:

      ![A snippet of the Recent activity section that displays agreement details modal that opens when you hover over an item in the chart.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Overview-dashboard_Recent-activity_agreement-details_hover.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vfO-rsXGliffUpJjfdo3EUhhsuKKAKYbz3DlWAKOOJU&_LANG=enus "A snippet of the Recent activity section that displays agreement details modal that opens when you hover over an item in the chart.")

      - Total contract value
      - Agreement name
      - Effective date
      - Expiration date
      - Renewal types
      - Renewal term

      Use the View Agreement option to view the document from the Agreement preview page.

**View dashboard charts and download a report**

9. Review the dashboard charts to see a visual summary of Agreement types, Renewal
   types, Payment terms, and
   Obligations by due date:

   ![A snippet of the Overview dashboard that shows charts for Agreement types, Renewal types, Payment terms, and Obligations by due date.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Overview-dashboard_Charts.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vfO-rsXGliffUpJjfdo3EUhhsuKKAKYbz3DlWAKOOJU&_LANG=enus "A snippet of the Overview dashboard that shows charts for Agreement types, Renewal types, Payment terms, and Obligations by due date.")

   1. Hover your cursor over a chart segment or bar to view the number of
      representative agreements or obligations.

      Tip: The segments in the
      charts are clickable. Select a segment to view
      the associated agreements. Select the party name from the breadcrumb
      navigation at the top of the page to return to the dashboard.
   2. Expand the More actions (three dots)
      and select the View Report option:

      ![A snippet of the Renewals types chart that highlights the View Report option in the More actions (three dots).](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Overview-dashboard_Actions-menu_View-Report.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vfO-rsXGliffUpJjfdo3EUhhsuKKAKYbz3DlWAKOOJU&_LANG=enus "A snippet of the Renewals types chart that highlights the View Report option in the More actions (three dots).")

      Note: The More actions (three dots) menu also has the option to View as table
      or Download as png:
      - Select View as table to display the
        data in a table instead of a chart.
      - Select Download as png to download an
        image of the chart to your local
        Downloads folder.

      The report view opens and a list of relevant agreements or obligations
      are displayed in the table.
   3. Use the filters to refine the agreement or obligation list as
      needed.

      Note: Select the All
      Filters option to choose additional filters to apply
      to the report.
   4. Use the Customize columns (toggle) icon to choose
      the columns to display in the table:

      - Select the toggles to turn the associated field (column) on (blue) or off
        (white).
      - Re-order the columns as needed. Hover the cursor over the Reorder (dots) icon
        and drag the field (column) to the appropriate
        location.
      - Select the Save option.

      The dialog box closes and the columns are updated according to your
      configuration.
   5. Expand the More actions (three dots) menu from the top right and select the
      Download option:

      ![A snippet of the Renewal types report that highlights the Download option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Parties-page_Overview-dashboard_Download-option-2.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vfO-rsXGliffUpJjfdo3EUhhsuKKAKYbz3DlWAKOOJU&_LANG=enus "A snippet of the Renewal types report that highlights the Download option.")

      The associated report is downloaded to your local
      Downloads folder.
   6. Select the party name from the breadcrumb navigation at the top of the
      page.

      You are returned to the Overview
      dashboard.
10. Select Parties from the breadcrumb navigation at the top
    of the page.

    You are returned to the Parties page.

    Tip: Select Completed
    from the navigation pane on the left to view your agreement list. The
    Completed Documents view opens. The agreements
    to which the user has access are displayed in the table.
