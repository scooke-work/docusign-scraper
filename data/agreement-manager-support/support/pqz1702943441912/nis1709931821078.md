# Subscribe to Renewal Reports

Learn how to set up one-time or recurring email reports for yourself or others to receive notifications about your renewals.

You can subscribe to receive one-time or recurring email reports for any of the charts on the Renewals dashboard.

Tip: Notification settings give you control over the email notifications that you receive. By default, all the notification types related to the completed agreements in your account are selected, or enabled. Deselect a checkbox if you no longer want to receive email notifications for the corresponding notification type.

See the [Customize Notification Settings for Your Completed Agreements](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk155b1ed7-7b45-43cd-be92-2a5279630c0f.html&_LANG=enus "Learn how to customize your email notification settings from the Notification Preferences page.") article to learn more about updating your notification settings. Be sure to periodically check the Notification Preferences page as new notification types will be added in the future.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Insights from the global header.

   The Insights hub opens and the Overview is displayed.
3. Select Renewals from the navigation pane on the left.

   The Renewals dashboard view opens.

4. Filter the Renewals dashboard to the relevant renewal information:

   ![A snippet of the Renewals dashboard that highlights the dashboard filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Renewals-dashboard_Filter-pills.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNzUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Ny3mhhY5SFPmcsfMzzTM9A-0ltJn1vYXxpO7NPjruR0&_LANG=enus "A snippet of the Renewals dashboard that highlights the dashboard filters.")

   1. Expand the filter that you want to apply.
   2. Select or enter the filter value.
   3. Select the Apply option.

   Important: When using the Parties filter, you must also select a Party Role value. If you do not know which role to select, choose the All option. If no role is selected, the search returns agreements where the party is not the buyer or the seller.

   Tip: Use the Filters option to access additional filters:![A screenshot of the Filters dialog box that shows additional filters that you can apply to the agreement list.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_All-Filters-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNzUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Ny3mhhY5SFPmcsfMzzTM9A-0ltJn1vYXxpO7NPjruR0&_LANG=enus "A screenshot of the Filters dialog box that shows additional filters that you can apply to the agreement list.")

   The charts are updated to reflect the filters that you applied.
5. Go to the detailed report view for the dashboard chart to which you want to subscribe:

   ![A snippet of the Renewals dashboard that highlights the View Report option in the More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Renewals-dashboard_Chart_Actions_View-Report.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNzUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Ny3mhhY5SFPmcsfMzzTM9A-0ltJn1vYXxpO7NPjruR0&_LANG=enus "A snippet of the Renewals dashboard that highlights the View Report option in the More actions (three dots) menu.")

   - Select the chart heading.
   - Hover the cursor over the chart, expand the More actions (three dots) menu and select the View Report option.

   The detailed report view opens for the selected chart. The agreements represented in the chart are listed in the table.
6. Filter the table to refine the results list as needed or select Reset to clear existing filters.
7. Customize the information that is displayed in the table, if necessary:

   ![A screenshot of the Show or Hide FIeldsShow or Hide Fields dialog box that shows the columns you can choose to display in the agreements table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Overview-dashboard_Customize-columns.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNzUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Ny3mhhY5SFPmcsfMzzTM9A-0ltJn1vYXxpO7NPjruR0&_LANG=enus "A screenshot of the Show or Hide FIeldsShow or Hide Fields dialog box that shows the columns you can choose to display in the agreements table.")

   Note: If you made changes to the report, expand the More actions (three dots) menu and select the Save option:![A snippet of an agreement dashboard report that shows the Save option in the More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Agreements-Dashboard_Dashboard-report_Save-as.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNzUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Ny3mhhY5SFPmcsfMzzTM9A-0ltJn1vYXxpO7NPjruR0&_LANG=enus "A snippet of an agreement dashboard report that shows the Save option in the More actions (three dots) menu.")

   Once saved, access the report from the Custom tab located under the Reports view. Continue to the next step to subscribe to your custom report.

   The report view updates and the applied filters and customizations are reflected in the chart and agreement list table. Next, subscribe to receive email reports.
8. Select the Manage Notification (bell) icon from the top-right corner.

   The Create Subscription dialog box opens.

   ![A snippet of the Renewals by type report that shows the Manage Notification (bell) icon and the Create subscription dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Renewals-dashboard_Report-view_Subscribe-to-report.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNzUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Ny3mhhY5SFPmcsfMzzTM9A-0ltJn1vYXxpO7NPjruR0&_LANG=enus "A snippet of the Renewals by type report that shows the Manage Notification (bell) icon and the Create subscription dialog box.")
9. Complete the fields in the Create subscription dialog box and select Schedule:
   1. Enter a descriptive name in the Subscription Name field. This field defaults to the name of the report.
   2. Indicate how often and when you want to receive the report notification. Choose from Daily, Weekly, Monthly, and Once.

      Depending on the option you selected, you may be prompted to choose a send day.
   3. If necessary, turn on the Choose start and end date toggle and enter a Start date and End date.
   4. The notification is sent to you by default. Enter additional recipients in the Add a user or group field as needed.

      Note: When you create a subscription, you have the option to add individual users and groups. Once created, the subscription is displayed in the Manage report subscriptions dialog box for any individual users who were added. Currently, the subscription is not visible to users if they were added as part of a group.
   5. Select the Attach to email as CSV option as needed.
   6. Select Schedule.

   The dialog box closes. You have successfully signed up to receive report notifications.

## Next steps

Once created, you can manage the existing subscriptions for a selected report. Go to the report view for the subscription that you want to update and select the Manage Notification (bell) icon:

![A snippet of the detailed report view for a dashboard chart that highlights the editing options for an existing subscription.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Renewals_Renewals_Dashboard_Manage_Notification.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyNzUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.Ny3mhhY5SFPmcsfMzzTM9A-0ltJn1vYXxpO7NPjruR0&_LANG=enus "A snippet of the detailed report view for a dashboard chart that highlights the editing options for an existing subscription.")

Use the Manage Notification option to manage any existing subscriptions for the selected report, such as:

- Edit subscription details, including the subscription name, frequency, and recipient list.
- Delete an existing subscription.

You can also use the Create New option to create a new subscription for the selected report.
