# Filter Your Agreements by Clause Language

Learn how to use clause-related filters to find agreements that contain, or do not contain, specific clause language.

Filter your completed agreements to find agreements that contain (or do not contain) specific clause language. For example, you can confirm that a standard data protection clause appears in all active vendor contracts. You can also identify agreements where that language is missing so you can follow up and reduce risk exposure.

The clause-related filters are located in the Filters option on the Agreement Manager homepage. This task shows you how to apply these filters to confirm the presence of required clauses and identify any gaps.

## Before you begin

- Make sure that you can open Agreement Manager and see the agreements that you want to review. The clause filters only apply to agreements that you already have permission to view.
- If your organization uses AI-extracted clauses, confirm that clause extraction is enabled in your environment. This is required to use the clause-related columns and filters.

  Note: Clause extraction for Agreement Manager is available in Beta. Contact your Docusign account representative to inquire about participating in the Beta.
- If you want to edit or validate clause values, confirm that you have Organize and manage all agreements access to Agreement Manager. Users with Can Manage or Can Edit access to an agreement can also edit of validate clause presence.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.

**Apply clause-related filters**

The clause-related filters indicate the presence or absence of specific clause language in your agreements. Use these filters to confirm required clause language and to identify gaps.

4. Expand the Filters option and locate the filter that you want to apply:

   ![A snippet of the Completed Documents table that highlights a clause filter in the Filters option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Filter-by-clause_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NjcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.hb0AmqAC1uxZCAxlGTWcpXUUKiOCxAvg2DHKUFQf3FA&_LANG=enus "A snippet of the Completed Documents table that highlights a clause filter in the Filters option.")

   1. Enter the name of the clause in the Filter by field:

      Tip: Enter *Clause* in the Filter by field to view all the available clause filters.

      A list of suggested filters is displayed.
   2. Select the relevant clause filter.

   The filter expands and the available filter options are displayed. For clause-related filters, there are two options: Found in agreements and Not found in agreements.
5. Choose a filter value and select the Apply option:

   ![A snippet of the Completed Documents table that shows the filter values for a selected clause filter.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Filters_Clause-filter-options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NjcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.hb0AmqAC1uxZCAxlGTWcpXUUKiOCxAvg2DHKUFQf3FA&_LANG=enus "A snippet of the Completed Documents table that shows the filter values for a selected clause filter.")

   - Select the Found in agreements option to view the agreements that contain the clause language.
   - Select the Not found in agreements option to view the agreements that are missing the clause language.

   The filter closes and the agreements that matched the applied filter are listed in the table. For example, the agreements that contain the selected clause language.

**Verify clause presence in specific agreements**

Apply additional filters to verify the clause presence in a subset of agreements, such as master service agreements (MSAs) or non-disclosure agreements (NDAs).

6. Narrow the agreement list table for a specific subset of agreements. For example, to verify the presence of clause language across all your MSAs.
   1. Expand the Filters option and select the filter that you want to apply. For example, select the Agreement Type filter.

      The filter expands and the available filter values are displayed.
   2. Select the relevant filter value. For example, select the Master Service Agreement value.
   3. Select the Apply option.

   The filter closes and the agreements that matched the applied filter are listed in the table. For example, MSAs that contain Data Breach clause language.

**Identify gaps in your agreements**

Identify gaps where the clause is not present in your agreements. Update the clause filter to show agreements where the clause was not found.

7. Update the clause filter to identify gaps where the clause is not present. For example, locate the MSAs that are missing the Data Breach clause language.

   ![A snippet of the Completed Documents table that shows the filter values for the selected clause filter.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Update-clause-filter_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NjcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.hb0AmqAC1uxZCAxlGTWcpXUUKiOCxAvg2DHKUFQf3FA&_LANG=enus "A snippet of the Completed Documents table that shows the filter values for the selected clause filter.")

   1. Select the applied clause filter from the top of the page.

      The filter expands and available values are displayed.
   2. Select the Not found in agreements value.
   3. Select the Apply option.

   The filter closes and the agreements that matched the applied filter are listed in the table. For example, MSAs that are missing the Data Breach clause language.
8. Review the agreement list and identify the agreements that you want to investigate further.

   Note: Download the agreements that require follow-up:
   - Select the checkboxes associated with the agreements that you want to download.

     The actions bar opens when at least one agreement is selected.
   - Select the Download Documents (arrow) icon from the action bar.

     A zipped folder containing the selected agreements is downloaded to your local Downloads folder.

You have successfully used clause-related filters to confirm the presence of specific clauses, including identifying any gaps.
