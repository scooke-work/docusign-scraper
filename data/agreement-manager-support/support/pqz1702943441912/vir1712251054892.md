# Renewals Dashboard

This topic answers the question: What does the Renewals dashboard show me and how does it work? Read about the Renewals dashboard, including how to use the charts to filter for agreements and available sorting options.

The Renewals dashboard displays renewal data based on the attribute (field) values surfaced in your agreements. The dashboard refreshes every 15 minutes to account for new agreements and agreements with updated renewal-related field values. The refresh is designed to ensure that all agreements are represented in the dashboard.

Important: Data validation is an important part of renewal management. If the renewal-related field values that were surfaced in your agreements are inaccurate, the Renewals dashboard is not accurate. This process relies on the verification of suggested attribute (field) values.

The dashboard charts are populated based on the data points that are surfaced in your agreements.
Apply filters to refine the chart data. The
Renewal Notice Date and
Renewal Type filters are
applied by default.

Access the dashboard from the email notification or by selecting Insights > Renewals.

## Dashboard charts

The dashboard charts show a visual summary of renewal-related insights, including: Upcoming renewals and Renewals by type.

Upcoming renewals

The Upcoming renewals chart shows agreements with Renewal Notice Dates that match the applied filters:

![A screenshot of the Upcoming Renewals chart that is located on the Renewals dashboard.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Renewals-dashboard_Chart_Upcoming-Renewals.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwMjcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.63BIMrYjhkIjFS13cCvy88brmgxMMKWcsiiDUXGmtBI&_LANG=enus "A screenshot of the Upcoming Renewals chart that is located on the Renewals dashboard.")

By default, the chart shows agreements where:

- The Renewal Type is Auto Renewal.
- The Renewal Notice Date value is within the next 12 months.

The agreements are groups by Renewal Notice Date and an aggregated count is displayed for each group.

Note: The dashboard charts can be viewed as regular bar charts or as stacked bar charts. ## Filter options

Filters display at the top of the Renewals dashboard. Your filter selections are applied to the dashboard charts. Apply one or more of the filters to locate the relevant renewal information.

![A snippet of the Renewals dashboard that highlights the dashboard filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Renewals-dashboard_Filter-pills.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwMjcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.63BIMrYjhkIjFS13cCvy88brmgxMMKWcsiiDUXGmtBI&_LANG=enus "A snippet of the Renewals dashboard that highlights the dashboard filters.")

Filters include:

- Parties
- Renewal Notice Date
- Renewal Owner
- Renewal Type
- Status

Important: When using the Parties filter, you must also select a Party Role value. If you do not know which role to select, choose the All option. If no role is selected, the search returns agreements where the party is not the buyer or the seller.

Additional filters are available from the All Filters dialog box:

![A screenshot of the All Filters dialog box that shows additional filters that you can apply to the agreement list.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_All-Filters-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwMjcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.63BIMrYjhkIjFS13cCvy88brmgxMMKWcsiiDUXGmtBI&_LANG=enus "A screenshot of the All Filters dialog box that shows additional filters that you can apply to the agreement list.")

Note: Multiple filters and search criteria can be used together.

- Multiple values within the same filter imply an "or" relationship. For example, select two parties to view agreements for either contracting party.
- Multiple values in separate filters with entered search criteria imply an "and" relationship.

Review the [Download the Upcoming Renewals Report](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=qke1713569186359.html&_LANG=enus "Learn how to locate upcoming renewals from the Renewals dashboard, including viewing and downloading agreements. ## Report limitations

When downloading reports to CSV files from the Insights reporting hub, there is an agreement limit of 25,000 agreements for all IAM reports. This count includes your custom reports.

**Suggested workaround:** Adjust the report filters to run the report in batches until all the relevant agreements are downloaded. For example, run the report in batches according to varying date ranges. Once downloaded, combine the data into one CSV file.

Note: The eSignature reports have a different set of limitations. See the [Using Reports](https://support.docusign.com/s/document-item?bundleId=cnu1642812012214&topicId=wou1578456440763.html) article to learn more about eSignature reports for senders and signers.
