# Update Share Settings for an Agreement Set

Learn how to update the sharing settings for an agreement set, including granting and revoking access to a set.

Contract managers and any users who are assigned the Organize and manage all agreements permission can create and update agreement sets. This includes managing access to agreement sets.

This topic describes the steps to manage the sharing settings for a selected agreement set, including granting and revoking access to a set.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents page is displayed.
4. Expand the Settings (gear) menu from the top right and select the Sets option.

   ![A screenshot of the Settings (gear) menu on the Agreement Manager homepage that highlights the Sets option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-sets_entry-point_Manage-sets-action-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc0MTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.pLYBLkq_y-mXOGJiSoy2h8Gzp6lkreJKUslMxeGDYPI&_LANG=enus "A screenshot of the Settings (gear) menu on the Agreement Manager homepage that highlights the Sets option.")

   The Agreement sets page opens. The All agreement sets view is displayed. The table lists the sets that were shared with or created by the logged-in user.
5. Locate the set that you want to update.

   Tip: Select the sorting arrows in the Name column to sort the table by set name.
6. Expand the More actions (three dots) menu for the set that you want to update and select Share:

   ![A snippet of the All agreement sets view that shows the Share option in the More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Sets-dialog_All-agreement-sets_Share-option.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc0MTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.pLYBLkq_y-mXOGJiSoy2h8Gzp6lkreJKUslMxeGDYPI&_LANG=enus "A snippet of the All agreement sets view that shows the Share option in the More actions (three dots) menu.")

   The Share with others dialog box opens. The People with access section shows the users and groups that currently have access to the agreement set. This includes the assigned access level for the agreements in the set.

   ![A screenshot of the Share with others dialog box that shows the users and groups that currently have access to the selected set.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Sets-dialog_All-agreement-sets_Share-with-others-dialog.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc0MTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.pLYBLkq_y-mXOGJiSoy2h8Gzp6lkreJKUslMxeGDYPI&_LANG=enus "A screenshot of the Share with others dialog box that shows the users and groups that currently have access to the selected set.")

   Update the sharing settings for the set and select Save or Share.

**Share the set with additional users or groups:**

7. Enter the name of the user or group with whom you want to share the set in the Add user or group field.

   A type-ahead dropdown displays names matching the criteria that you entered.
8. Select a user or group from the dropdown list.

   The Add users view is displayed.

   ![A screenshot of the Add users view that shows the notification and access-level options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-sets_share-set_Add-users-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc0MTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.pLYBLkq_y-mXOGJiSoy2h8Gzp6lkreJKUslMxeGDYPI&_LANG=enus "A screenshot of the Add users view that shows the notification and access-level options.")

   Important:

   The Notify users by email option is selected by default. Leave this checkbox selected to notify users that an agreement set was shared with them.

   If you de-select this option, users are not notified of the shared agreement set.
9. Expand the Access dropdown and select the appropriate access level for agreements in the set.

   Choices are: Can Manage, Can Edit, and Can View.
10. Select the Share option.

    The dialog box closes and you are returned to the Agreement sets page. The agreement set has been shared with the selected users or group.

**Update the "Access" level for users and groups with access to the set:**

Begin from the Agreement sets page.

11. Expand the More actions (three dots) menu for the set that you want to update and select Share.

    The Share with others dialog box opens. The People with access section shows the users and groups that currently have access to the agreement set.
12. Update the Access level for users and groups displayed in the People with access list:

    ![A snippet of the Share with others dialog box that highlights the Can View access level in the dropdown.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-set-management_Update-access-level_example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc0MTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.pLYBLkq_y-mXOGJiSoy2h8Gzp6lkreJKUslMxeGDYPI&_LANG=enus "A snippet of the Share with others dialog box that highlights the Can View access level in the dropdown.")

    1. Expand the Access dropdown for the user or group that you want to update.
    2. Choose an access level. Choices are: Can View, Can Edit, and Can Manage.
    3. Select the Save option.

    The dialog box closes and you are returned to the Agreement sets page. The access level for the selected user or group is updated to reflect the new value.

**Revoke user or group access to an agreement set:**

Begin from the Agreement sets page.

13. Expand the More actions (three dots) menu for the set that you want to update and select Share.

    The Share with others dialog box opens. The People with access section shows the users and groups that currently have access to the agreement set.
14. Revoke user or group access to the selected agreement set:

    ![A snippet of the Share with others dialog box that highlights the Remove option in the dropdown.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-set-management_Update-access-level_Remove.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc0MTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.pLYBLkq_y-mXOGJiSoy2h8Gzp6lkreJKUslMxeGDYPI&_LANG=enus "A snippet of the Share with others dialog box that highlights the Remove option in the dropdown.")

    1. Expand the Access dropdown for the user or group that you want to update.
    2. Select the Remove option.
    3. Select the Save option.

       The dialog box closes. The user or group is no longer displayed in the People with access list for the agreement set.
15. Select Completed Documents from the breadcrumb navigation at the top of the page.

    You are returned to the Agreement Manager homepage.

You have successfully updated the *Share* settings for a selected agreement set.
