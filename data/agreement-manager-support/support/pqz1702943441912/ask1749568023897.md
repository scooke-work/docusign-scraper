# Import Documents from Email Attachments

This topic answers the question: How do I add documents from email attachments to Agreement Manager? Learn how to import documents from email attachments to Agreement Manager for processing.

Executed agreements can be imported directly from email attachments into Agreement Manager for
processing. Keep in mind that:

- File names must be fewer than 255 characters
- Only supported file types can be ingested

Attention: The maximum file size for email attachment ingestion is
limited by your email provider. The common file size limit for popular providers,
like Gmail, Outlook, and Yahoo is 25 MB. Some providers, like Outlook, might have
lower limits for certain account types, such as accounts with internet service
providers (ISPs). For these accounts, the file size limit is 20 MB.

Important: **License and permission requirements**

By default, all users can upload or import up to five agreements at a time. However, to ingest email attachments, the email must come from a registered account user. Only registered account users with the appropriate upload permissions can use this feature.

The Upload up to 1,000 files at a time permission is required to ingest more than five email attachments at a time. This toggle is turned on from the user profile page by an administrator.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Expand the Add (+) dropdown and select the Import from Email option:

5. Select the Copy Address option.

   The email address is copied to your clipboard.
6. Select Ok to close the dialog box.
7. Go to your email provider and send an email to the copied address. Paste (Ctrl+v) the email address into the To field.

   Tip: **Agreement Manager email addresses**

   - Add the repository email address to your Contacts list for ongoing use.

     The next time you import agreements to Agreement Manager, the email address is already in your Contacts
     list. You do not have to copy the email address every time
     you want to import email attachments.
   - Add the Agreement ManagerUploader@docusign.net email alias to your Safe Senders list.

     If your ingestion fails, this email alias will send you a notification. Depending on your security policies, the email may be blocked or marked as spam. Add the email address to your Safe Senders list to ensure that you receive failure notifications.
8. Attach the files that you want to import to Agreement Manager and send the email.

   A confirmation email is sent once the files are imported and processed:![A snippet of the confirmation email indicates that the files were imported and processed.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Email-import_email-confirmation.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3MzQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.bWFzSudRo6F-Bw5bgsS7VmKhGu83M_q1TEvSZEn8yiE&_LANG=enus "A snippet of the confirmation email indicates that the files were imported and processed.")

   Tip: Select the Manage Files option to view
   the files that were imported. The Completed view opens and
   the imported files are displayed in the agreement list table.
9. Select the View Job option to view the job status and details of the processing job.

   The job monitoring hub opens, and the status and details for the job are displayed.![A screenshot of the job status and details for a seleted import job.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Email-import_job-status-details.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3MzQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.bWFzSudRo6F-Bw5bgsS7VmKhGu83M_q1TEvSZEn8yiE&_LANG=enus "A screenshot of the job status and details for a seleted import job.")

   Note: If any of your files failed to import, expand the Download and Fix dropdown and select the Download
   CSV option to review the files and try again.![A snippet of the import status that prompts you to download the CSV to view files that failed to import.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Upload-documents_Bulk-upload_Upload-status_Failed-documents.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3MzQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.bWFzSudRo6F-Bw5bgsS7VmKhGu83M_q1TEvSZEn8yiE&_LANG=enus "A snippet of the import status that prompts you to download the CSV to view files that failed to import.")See the [View Upload Results and Exceptions](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=srq1712255787588.html&_LANG=enus "Learn how to view processing results and details for an upload job.") topic to learn more about job
   monitoring.
10. Select the Manage Files option.

    The Completed view opens and the imported files are displayed in the agreement list table.
