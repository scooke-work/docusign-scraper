# Find Custom Fields in Specific Documents

Learn how to use the Find On Specific Documents function to surface custom fields in selected agreements.

## Before you begin

Once a custom field is enabled for AI detection, you can find it in specific documents at any time. This requires you to complete these tasks first:

- [Add Custom Fields to Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=hcn1736879099815.html&_LANG=enus "Learn how to create and add custom fields to the custom and standard agreement types in your account.")
- [Add Custom Fields to Standard Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=vju1740452710836.html&_LANG=enus "Learn how to create and add custom fields to standard agreement types.")
- [Test the Extraction Accuracy for Your Custom Fields](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=wjr1740452628309.html&_LANG=enus "Learn how to test the AI detection accuracy for your custom fields and refining the results to improve accuracy.")

The Find On Specific Documents function is used to
process specific documents for a custom field. If the field is located in the document, the associated value is surfaced.

Important: The selected documents are reprocessed *only* for that specific custom field. If the field is not found in the document, there is no further processing.

This function is found under the Fields option in the Settings (gear) menu.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Expand the Settings (gear) dropdown from the
   top right and select the Fields
   option:

   ![A snippet of the Agreement Manager homepage that highlights the Fields option in the Settings menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Fields-Manager_Settings-menu_Fields-option_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.l-5FDwGrSKi9J5cfHtZcRv3DG8AnmsmPrrD_mySzR-k&_LANG=enus "A snippet of the Agreement Manager homepage that highlights the Fields option in the Settings menu.")

   The Manage Fields page opens. The
   Fields view is displayed. The standard and custom fields that are associated with the account are listed in the table.
5. Locate the field that you want to find in specific documents:

   ![A snippet of the Manage Fields page that shows the search field and filter options for the Fields table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Fields-Manager_Fields-table_search-and-filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.l-5FDwGrSKi9J5cfHtZcRv3DG8AnmsmPrrD_mySzR-k&_LANG=enus "A snippet of the Manage Fields page that shows the search field and filter options for the Fields table.")

   - Search for the relevant custom field.

     Enter a keyword in the Search fields... field. As you type the keyword, the table updates and the fields that match the search term are displayed.
   - Filter the table to locate the relevant custom field. Filter options are: Created
     By, Category, Field
     Type, and AI Status. For example, use
     the Created By filter the table for user-created
     fields.

     Expand the filter, choose a value, and select the
     Apply option.

     Important: The
     Edit Details function is only available for
     user-created or custom fields. You cannot update the details for Docusign-provided fields.

     The table updates
     and the fields that matched the applied filters are
     displayed.
6. Expand the More actions (three dots) menu for the relevant field and select the Find On Specific Documents option:

   ![A snippet of the Fields view that highlights the Find On Specific Documents option in the More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Fields-Manager_Fields-table_More-actions_Find-On-Specific-Documents.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.l-5FDwGrSKi9J5cfHtZcRv3DG8AnmsmPrrD_mySzR-k&_LANG=enus "A snippet of the Fields view that highlights the Find On Specific Documents option in the More actions (three dots) menu.")

   The Select documents to process page opens. Use this page to search for the documents that you want to process for the field.
7. Choose the documents that you want to process for the custom field:

   ![A screenshot of the Select documents to process page that shows applied filters and selected documents.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Fields-Manager_Fields_Select-docs-to-process_selected.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.l-5FDwGrSKi9J5cfHtZcRv3DG8AnmsmPrrD_mySzR-k&_LANG=enus "A screenshot of the Select documents to process page that shows applied filters and selected documents.")

   1. Search for the relevant agreements. Enter a keyword or search string in the search field. The dropdown opens and search suggestions are displayed. Select a search suggestion or press the Enter key.

      The table updates and agreements that matched the search term are displayed.
   2. Filter the agreement list table for relevant agreements. For example, filter by Sets, Status, Parties, Agreement Type, or AI Suggestions.

      Tip: Use the Filters option to apply additional filters to the results table.

      Expand the filter, choose a value, and select the Apply option.

      The table updates and the agreements that matched applied filters are displayed.
   3. Select the relevant documents from the table.
8. Select the Start Processing option from the bottom right.

   You are returned to the Fields view. A message confirms that the selected documents are being processed.

   ![A screenshot of the Fields view that shows the confirmation message that the documents are being checked for the custom field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Fields-Manager_Fields-table_Finding-field-on-docs_confirmation-message.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.l-5FDwGrSKi9J5cfHtZcRv3DG8AnmsmPrrD_mySzR-k&_LANG=enus "A screenshot of the Fields view that shows the confirmation message that the documents are being checked for the custom field.")

   If the custom field is found in any of the documents, the field value is surfaced in the agreement.

   Tip: Select the View Progress option to view the processing status for the job. The job monitoring hub opens, and the Processing status and Job details are displayed. Review the processing results, including any error messages.Select the Close (x) icon in the top right to return to the Agreement Manager homepage.
9. Return to the Agreement Manager homepage. Select the Close (X) icon located in the top right.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
