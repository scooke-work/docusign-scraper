# Create a Hierarchy of Related Agreements

Learn how to create a hierarchy of related agreements for a selected party or group.

Hierarchies show how agreements are legally related. If there are no existing hierarchies for the selected party or group, use the Create Hierarchy option to build one.

This task reviews the steps to create an agreement hierarchy from the party hub pages.

Tip: You can also initiate the Create Hierarchy process from the Related agreements pane on the Agreement preview page: ![A snippet of the Agreement preview page that highlights the Create Hierarchy option on the Related agreements pane.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Preview-page_Create-Hierarchy_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MzUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.MvZVzAbqx6mXekq_J8WSd7yFM2IvH9VxIBjzTNvGMnc&_LANG=enus "A snippet of the Agreement preview page that highlights the Create Hierarchy option on the Related agreements pane.")Select the Create Hierarchy option to initiate the process. The Create Hierarchy page opens in the party hub for the selected party. In this case, the first party that was listed in the Details pane on the Agreement preview page.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens. The Inbox view is displayed by default.
3. Expand the PARTIES section in the navigation pane on the left and select the All option.

   The Parties page opens and a list of individual and merged parties is displayed in the table.
4. Select the relevant party or group from the parties list.

   ![A snippet of the Parties page that highlights the entered keyword and the relevant party in the search results list.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Contract-Hierarchy_Parties_Select-Party_search-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MzUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.MvZVzAbqx6mXekq_J8WSd7yFM2IvH9VxIBjzTNvGMnc&_LANG=enus "A snippet of the Parties page that highlights the entered keyword and the relevant party in the search results list.")

   1. Enter the name of the party or group in the Search parties field and press [Enter].

      Parties and groups matching the keyword are displayed in the table.
   2. Select the party or group from the table.

   The party hub opens for the selected party or group. The Overview tab is displayed by default.
5. Select the Hierarchies tab.

   Hierarchies are created and managed from the IAM party hub under the Hierarchies tab. From this page, you are prompted to create new hierarchies or update existing ones.
6. Create a new hierarchy for the selected party or group. Select the Create Hierarchy option.

   The Create Hierarchy wizard opens and you are prompted to define the hierarchy.
7. Complete the Define step by providing Hierarchy details, such as the title and description.

   ![A screenshot of the Create Hierarchy wizard that shows the Define step for providing hierarchy details.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Contract-Hierarchy_Create-Hierarchy_Define-step-1.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MzUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.MvZVzAbqx6mXekq_J8WSd7yFM2IvH9VxIBjzTNvGMnc&_LANG=enus "A screenshot of the Create Hierarchy wizard that shows the Define step for providing hierarchy details.")

   1. Verify the Party field. This field defaults based on the selected party or group.

      Tip: If the Party field is incorrect, go back to the Parties page and select the correct party or group. Select the Parties option from the breadcrumb navigation at the top of the page.
   2. Enter a title for the hierarchy in the Title field. This field is required.
   3. Enter a description of the hierarchy in the Description field. Use this field to explain how the agreements are related.
   4. Select the Next option.

   The Create Hierarchy wizard advances to the Add step. You are prompted to add related agreements to the hierarchy.
8. Complete the Add step by adding related agreements to the hierarchy.

   ![A screenshot of the Create HIerarchy wizard that shows the Add step for adding related agreements to the hierarchy.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Contract-Hierarchy_Create-Hierarchy_Add-step-2.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MzUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.MvZVzAbqx6mXekq_J8WSd7yFM2IvH9VxIBjzTNvGMnc&_LANG=enus "A screenshot of the Create HIerarchy wizard that shows the Add step for adding related agreements to the hierarchy.")The agreement list is pre-filtered to show agreements for the selected party or group.

   Tip: Remove the pre-applied filter to add documents that are associated with other parties.

   1. Search and filter the agreement list to locate the relevant agreements. Enter a keyword in the search field and apply filters as needed.
   2. Review the list of agreements and select the + Add option for the agreements that you want to add to the hierarchy. If necessary, scroll to the bottom of the page and use the vertical scroll bar to access the + Add option.

      Restriction: You can only add documents to which you have Can Edit or Can Manage access. Also, you cannot add agreements that are already associated with another hierarchy.

      The selected agreements are displayed in the Add related Agreements section for the hierarchy.
   3. Repeat this step until all the relevant documents have been added to the hierarchy.

      Tip: Remove agreements from the hierarchy if they were added by mistake. Locate the agreement that you want to remove from the Add related Agreements section. Expand the associated Actions (vertical ellipsis) menu and select the Remove from Hierarchy option.
   4. Select the Next option from the bottom of the page.

   The Create Hierarchy wizard advances to the Arrange step. You are prompted to provide a structure for the hierarchy. The structure is used to define the relationships between the related agreements.
9. Complete the Arrange step by creating relationships between the agreements.

   ![A screenshot of the Create Hierarchy wizard that shows the Arrange step for creating relationships between the agreements.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Contract-Hierarchy_Create-Hierarchy_Arrange-step-3.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MzUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.MvZVzAbqx6mXekq_J8WSd7yFM2IvH9VxIBjzTNvGMnc&_LANG=enus "A screenshot of the Create Hierarchy wizard that shows the Arrange step for creating relationships between the agreements.")

   1. Update the Primary agreement as needed. The Primary designation defaults to the first agreement that was added to the hierarchy.

      Expand the Arrange actions (vertical ellipsis) menu and select the Make Primary option.

      ![A snippet of the hierarchy details page that highlights the Arrange actions menu options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Contract-Hierarchy_Create-Hierarchy_Arrange-actions-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MzUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.MvZVzAbqx6mXekq_J8WSd7yFM2IvH9VxIBjzTNvGMnc&_LANG=enus "A snippet of the hierarchy details page that highlights the Arrange actions menu options.")
   2. Rearrange the agreements to create structured relationships between them. Consider these options:

      - Hover the cursor over the dots in the File Name column for the agreement that you want to move. When the "grabber" cursor displays, drag the agreement to the appropriate location.
      - Expand the Arrange actions menu and choose to move the agreement up, down, or to a different level.Repeat this step until the hierarchy reflects the appropriate agreement relationships.
   3. Remove agreements that were added by mistake as needed. Expand the Arrange actions menu and select the Remove from Hierarchy option.
   4. Select the Finish option.

   The wizard closes and you are returned to the Hierarchies tab. The new hierarchy is displayed in the table.

   ![A screenshot of the Hierarchies tab in the party hub that highlights the new hierarchy.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Contract-Hierarchy_Hierarchies-tab_Fontara-example_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MzUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.MvZVzAbqx6mXekq_J8WSd7yFM2IvH9VxIBjzTNvGMnc&_LANG=enus "A screenshot of the Hierarchies tab in the party hub that highlights the new hierarchy.")

You have successfully created a new hierarchy for a group of legally related agreements.
