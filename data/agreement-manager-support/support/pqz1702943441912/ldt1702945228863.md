# Use Case: View Active Agreements for a Customer

This topic answers the question: How can I view active agreements for a specific party? Learn how to find all the active agreements for a specific customer.

As a contract viewer, you want to view all active agreements for a specific customer and include this information in the monthly sales report. Locate this information from the party hub.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens. The Inbox view is displayed by default.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. Filter this view for a specific party.
4. Expand the Filters option and select the Parties filter.
5. Search for and select the party or party group from the list. Select the Apply option.

   The agreements that are tied to the selected party or alias group are displayed in the table.
6. Select a name from the Parties column.

   The names in the Parties column are clickable and provide access to the party hub.

   The party hub opens for the selected party or alias group. The Overview tab displays a visual summary of all active agreements that are tied to a specific party or alias group.
7. Review the top-level insights, such as the Party summary and agreement counts for Active agreements, Expiring soon, and Up for renewal:

   ![A snippet of the Overview dashboard that displays the top-level agreement insights for a specific party or alias group.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Overview-dashboard_Highlevel-insights.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYyNzMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.p4TJcbmzpcp9wMPtGxWI4viI04qdq9ZbiDfE5PW3Pv4&_LANG=enus "A snippet of the Overview dashboard that displays the top-level agreement insights for a specific party or alias group.")

   1. Expand the More actions (three dots) menu for the All active agreements section and select View Report.

      The report view opens. All active agreements tied to the selected party or group are displayed in the table.
   2. Use the filters to refine the agreement list as needed.

      Note: Select the All Filters option to choose additional filters to apply to the report.
   3. Select the Show or Hide Fields (gear) icon and choose the columns to display in the table:

      - Select the toggles to turn the associated field on (blue) or off (white).
      - Re-order the columns as needed:
        - Hover the cursor over the dots to the right of a field.

          The cursor updates to an open hand cursor, indicating that the element can be grabbed.
        - Drag the field to the appropriate location.
      - Select Save.

      The dialog box closes and the columns are updated according to your configuration.

      Note: The Show or Hide Fields function only applies to the data that is displayed in the agreement list table. When downloading or exporting the report, all fields are included in the output file. This includes both custom and standard fields.
   4. Select the Download option.

      The associated report is downloaded to your local Downloads folder.
   5. Return to the Agreement Manager homepage. Select the Completed option from the navigation pane on the left.

      You are returned to the Agreement Manager homepage and the Completed Documents view is displayed.

You have successfully located all active agreements for a specific customer. You can share this information for sales reporting.
