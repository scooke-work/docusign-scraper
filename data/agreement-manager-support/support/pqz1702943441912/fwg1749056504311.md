# Complete a Data Review for Extracted Obligations

Learn how to complete a data review to verify the obligations that were surfaced in your agreements.

During processing, key attributes and field values are surfaced in your agreements. These data points are used to extract obligations from your agreements. Currently, these types of obligations are extracted from your agreements:

- Payment
- Renewal
- Termination

Obligations are surfaced in English (U.S.) agreements only. The AI model that processes obligations is run from a U.S. server and can only process English agreements in this region.

Tip: See the *Obligations surfaced with AI* section in the [Obligation Management](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=csm1727128259486.html&_LANG=enus "Read about obligation management, including how to use this functionality to capture, track, and fulfill your agreement obligations. This topic answers the question: How can I find and track my agreement obligations?") topic to learn more about extracted obligations.

Reviews are necessary to validate the data that is surfaced in your agreements, both attribute values and obligations. These reviews are essential for accurate reporting, forecasting, contract management, and other business purposes.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Insights from the global header.

   The Insights hub opens and the Overview is displayed.
3. Select Obligations from the navigation pane on the left.

   The Obligations dashboard view opens. The dashboard provides a visual summary of all the obligations associated with agreements in the account.

   Remember: You can only view obligations for the agreements to which you have access.

   ![A screenshot of the Obligations dashboard that shows available reports and filter options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Obligations-Dashboard.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2NDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9._F8B0Ovy8rjX7-CO42xYiUgBcX2jXVstpHZxr0F8VDU&_LANG=enus "A screenshot of the Obligations dashboard that shows available reports and filter options.")

   Update one of the obligation reports to locate the obligations that you want to review. Both reports provide a list of obligations that match the applied filters.

   The Obligations by type report includes all the surfaced and manually created obligations for your agreements.
4. Access the detailed report view for the Obligations by type chart.
   - Select the chart heading.
   - Expand the More actions (ellipsis) menu and select View Report:![A snippet of the Obligations dashboard that shows the View Report, View as table, and Download as png options in the report actions menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Obligations-Dashboard_Expanded-Actions-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2NDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9._F8B0Ovy8rjX7-CO42xYiUgBcX2jXVstpHZxr0F8VDU&_LANG=enus "A snippet of the Obligations dashboard that shows the View Report, View as table, and Download as png options in the report actions menu.")

   The detailed report view opens for the selected chart. The table displays the obligations that are represented in the chart.

   ![A screenshot of the Obligations by type report that shows obligations grouped by type and listed in the table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Obligations_Obligations-by-type_report-view.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2NDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9._F8B0Ovy8rjX7-CO42xYiUgBcX2jXVstpHZxr0F8VDU&_LANG=enus "A screenshot of the Obligations by type report that shows obligations grouped by type and listed in the table.")

   Tip: ![An image of the spark icon used to denote the use of AI.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/AI-Spark-icon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2NDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9._F8B0Ovy8rjX7-CO42xYiUgBcX2jXVstpHZxr0F8VDU&_LANG=enus "An image of the spark icon used to denote the use of AI.") The spark icon indicates that the agreement contains suggested values. Review, edit, and approve these values to ensure accurate agreement data for reporting, forecasting, and other business purposes.
5. Apply filters to refine the obligation list as needed:

   ![A snippet of the Obligations by type report that shows the available filters, such as due date, parties, and status.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Obligations_Obligations-by-type-report_filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2NDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9._F8B0Ovy8rjX7-CO42xYiUgBcX2jXVstpHZxr0F8VDU&_LANG=enus "A snippet of the Obligations by type report that shows the available filters, such as due date, parties, and status.")

   1. Expand a filter.
   2. Select or enter a filter value.
   3. Select the Apply option.

The obligations that match the applied filters are displayed in the table.

6. Select (click) the obligation that you want to review.

   The Agreement preview page opens for the selected obligation. The obligation details pane is displayed.

   ![A screenshot of the Agreement preview page that shows the obligation details in the pane on the left.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Obligations_Agreement-preview_obligation-detail-pane.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2NDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9._F8B0Ovy8rjX7-CO42xYiUgBcX2jXVstpHZxr0F8VDU&_LANG=enus "A screenshot of the Agreement preview page that shows the obligation details in the pane on the left.")

   Review and update or approve all the attribute values that display the spark icon.
7. Place the pane into *edit* mode:
   - Select the Edit (pencil) icon.
   - Select the Review All option.

   Note: You must have Organize and manage all agreements access to perform data reviews. For all other users, the Edit (pencil) icon and Review All button are grayed out.

   The Edit Obligation view displays and the attribute fields are opened for edits.
8. Accept or edit the suggested values until all the attributes are reviewed and approved.

   ![A snippet of the Edit Obligation pane that highlights a suggested value within the agreement.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Obligation_Data-review_highlighted-in-agreement.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2NDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9._F8B0Ovy8rjX7-CO42xYiUgBcX2jXVstpHZxr0F8VDU&_LANG=enus "A snippet of the Edit Obligation pane that highlights a suggested value within the agreement.")

   1. Compare the suggested attribute values against the data in the agreement. Select (click) an attribute to view ("jump-to") the suggested value within the agreement.

      Tip: The "jump-to" feature is only available for fields with a spark icon.

      The value is highlighted within the agreement and a list of suggested values is displayed.
   2. Select the Accept AI Suggestion (checkmark) icon if the value is *correct*.

      Note: After you accept an AI suggestion, a checkmark icon is displayed next to the value. For example:![A snippet of the Edit Obligation pane that shows a checkmark next an extracted value to indicate the field was reviewed.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Obligation_data-review_accepted-AI-suggestion_checkmark.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2NDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9._F8B0Ovy8rjX7-CO42xYiUgBcX2jXVstpHZxr0F8VDU&_LANG=enus "A snippet of the Edit Obligation pane that shows a checkmark next an extracted value to indicate the field was reviewed.")
   3. Enter the correct value if the suggestion is *incorrect* or *missing*.
   4. Repeat this step until all the suggested values are reviewed and approved.
9. Select the Save option at the bottom of the pane.

   The obligation details are saved. The pane is reverted to read-only mode.
10. Select the Back option located in the top-left corner of
    the page.

    You are returned to the Obligations by type report. The
    spark icon no longer displays next to the obligation that you reviewed.
11. Continue reviewing the extracted obligations as needed.

You have successfully completed a data review for an extracted obligation.
