# Apply Filters to Locate Agreements for Specific Parties

This topic answers the question: How can I see agreements for a specific party? Learn how to use filters to locate agreements for a specific party.

Instead of performing a keyword search, you can apply filters to the agreement list table to find the relevant agreements. In this example, use the Parties filter to locate agreements for a specific party.

Important: When using the Parties filter, select the relevant party
and choose the appropriate role from the Party Role
dropdown. If you do not know the role, accept the default value, All.

Tip: See the [View Agreements for Selected Parties and Groups](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=ghx1734749900921.html&_LANG=enus "Learn how to view all the agreements that are tied to a specific party or party group.") topic to complete this task from the party hub.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Completed Documents view opens. A list of documents to which the logged-in user has access is displayed in the table. The total document count is displayed in the bottom right. This count is updated when you perform a search or apply filters.
4. Expand (click) the Filters option and select the Parties filter:

   ![A snippet of the Completed Documents page that highlights the Parties option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Filters-menu_Parties_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzOTEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.hisayA92QJp_RMIsRE3iLnB_hVm0ajByl9IXKyGZhDI&_LANG=enus "A snippet of the Completed Documents page that highlights the Parties option.")
5. Select the relevant party and role:

   ![A screenshot of the expanded Parties filter that shows a search term and selected parties.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Parties_Tally-Inc-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzOTEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.hisayA92QJp_RMIsRE3iLnB_hVm0ajByl9IXKyGZhDI&_LANG=enus "A screenshot of the expanded Parties filter that shows a search term and selected parties.")
   - Enter a name in the Search field. As you enter the name, the list is updated to display parties matching the keyword.
   - Select the relevant party.
   - The Party Role defaults to All. Update the role as needed.

     Expand the Party Role dropdown and select the appropriate role. Choose from: Buyer, Seller, or Other.

     Important: When using the Parties filter, you must also select a
     Party Role. If you do not know
     which role to select, accept the default value, All.
   - Select Apply.

   The agreements that match the applied filter are displayed in the table.

   Tip: If you routinely search for the same type of agreements or agreement data, save your search criteria for future use. See the [Save Search Criteria for Future Use](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tskc38121a1-bbf8-4da8-8ed7-b364991f7e50.html&_LANG=enus "Learn how to save the search criteria for the queries that you frequently perform.") topic to learn more about the saved search feature.
6. Further refine the agreement list table:

   - Use the sorting arrows to sort the agreement list. For example, sort the table by Total Contract Value, Effective Date, or Expiration Date.
   - Apply additional filters.

   Tip:

   Use the Status filter to filter the agreement list table by agreement status. For example, choose to view only Active agreements that match the current search criteria and filters: ![A screenshot of the Status filter that shows the selected Active options, including Expiring Soon and Renewing Soon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-statuses_Status-filter_Active-selected.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzOTEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.hisayA92QJp_RMIsRE3iLnB_hVm0ajByl9IXKyGZhDI&_LANG=enus "A screenshot of the Status filter that shows the selected Active options, including Expiring Soon and Renewing Soon.") See the [Agreement Statuses](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=con6a3bd080-da17-4d1c-a92a-b9b9340c44fd.html&_LANG=enus "Read about agreement statuses, including how they are calculated, visibility within Agreement Manager, and which types of statuses are being tracked.") topic to learn more about this feature.

   Repeat this process until you locate the relevant agreements.

   Note: After locating the relevant agreements, choose to download selected agreements or export agreement data. See these support articles to learn more about these options:
   - [Download Documents from the Agreement List Table](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=ach1724363496037.html&_LANG=enus "This topic answers the question: How do I download agreements from Agreement Manager?")
   - [Export Data from the Agreement List Table](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=pqf1724363438241.html&_LANG=enus "Learn how to export agreement data to a CSV file for selected agreements.")

You have successfully used filters to locate agreements for specific parties.
