# View Report Data for Sets of Agreements

Learn how to filter an agreement report to display data for a selected set of agreements.

The Agreements Dashboard contains a number of charts and high-level insights that can be viewed as detailed reports. Use the Sets filter to view report data for a selected set of agreements.

Filters can be applied from both the dashboard view and the detailed report view. This topic reviews the steps to filter the Agreements Dashboard to view report data for a selected agreement set.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Insights from the global header.

   The Insights hub opens and the Overview is displayed.
3. Select Agreements from the navigation pane on the
   left.

   The Agreements Dashboard opens.
4. Filter the dashboard to show data for a selected set of agreements:

   ![A snippet of an agreement report that highlights the Sets filter and shows a selected set.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-dashboard_Report_Sets-filter.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5NjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.uIY2Fcu7CheI4_FsgQ47Q-lbY2T8w7iB5LDV7AH_Nas&_LANG=enus "A snippet of an agreement report that highlights the Sets filter and shows a selected set.")

   1. Expand the Sets filter.
   2. Search for and select the relevant agreement set.
   3. Select the Apply option.
5. Go to the detailed report view for an insight or chart. Select (click) the title of the insight or chart that you want to view.

   You can also expand the More actions (three dot) menu and select the View Report option:

   ![A snippet of the Expiring agreements insight that shows the View Report option in the More actions menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Agreements_snippet-ellipsesView-Report.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5NjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.uIY2Fcu7CheI4_FsgQ47Q-lbY2T8w7iB5LDV7AH_Nas&_LANG=enus "A snippet of the Expiring agreements insight that shows the View Report option in the More actions menu.")

   The detailed report view opens and it is pre-filtered to show data from the agreements in the selected set:

   - Agreements in the set are displayed in the agreement list table
   - Agreement data is reflected in the chart and the table
6. Download or save the report:

   ![A snippet of the report view that highlights the More actions (three dots) menu used to download or save a report.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Report-view_Dropdown-expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5NjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.uIY2Fcu7CheI4_FsgQ47Q-lbY2T8w7iB5LDV7AH_Nas&_LANG=enus "A snippet of the report view that highlights the More actions (three dots) menu used to download or save a report.")

   - Expand the More actions (three dots) menu in the top right and select
     the Download option.

     Note:

     When downloading reports to CSV files from the Insights reporting hub, there is an agreement limit of 25,000 agreements for all IAM reports. This count includes your custom reports.

     The report is downloaded to your local
     Downloads folder. The output file contains all
     the available attributes, including standard and non-standard fields.
     Customize the CSV file as needed.
   - Expand the More actions (three dots) menu in the top right and select the Save option. The Save Report dialog box
     opens.

     - Enter a unique name in the Report name field. The field defaults to the name of the selected report.
     - Enter a description about the report's purpose in the Description field.
     - Select the Save option.The report is saved to the Custom tab on the Reports view.

     Enter a Report name and select
     Confirm. The report is saved to the
     Custom tab on the Reports
     view.
7. Set up one-time and recurring email reports for yourself and others. Select the Manage Notification (bell) icon and complete the Create subscription dialog box:

   ![A screenshot of the Create subscription dialog box that is used to subscribe to report notifications.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Create-Subscription.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5NjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.uIY2Fcu7CheI4_FsgQ47Q-lbY2T8w7iB5LDV7AH_Nas&_LANG=enus "A screenshot of the Create subscription dialog box that is used to subscribe to report notifications.")

   Note: When you create a subscription, you have the option to add individual users and groups. Once created, the subscription is displayed in the Manage report subscriptions dialog box for any individual users who were added. Currently, the subscription is not visible to users if they were added as part of a group.

   See the [Subscribe to Receive Report Notifications](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=wfw1734545937270.html&_LANG=enus "Learn how to set up one-time or recurring email reports for yourself or others to receive notifications about your obligations, renewals, or agreements.") topic to learn more about this
   feature.
8. Add the report to Your Favorites list. Select the Favorite (star) icon located in the top right:

   ![A snippet of a detailed report view that highlights the Favorite (star) icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreements-dashboard_Report-details_Favorite-icon-selected_generic.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5NjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.uIY2Fcu7CheI4_FsgQ47Q-lbY2T8w7iB5LDV7AH_Nas&_LANG=enus "A snippet of a detailed report view that highlights the Favorite (star) icon.")Your starred dashboards and reports are displayed in the Your Favorites section on the Overview dashboard.
9. Return to the Agreements Dashboard. Select
   Agreements Dashboard from the breadcrumb navigation
   at the top of the page.

   The Agreements Dashboard view opens.

You have successfully filtered the detailed report view to show data for a selected set of agreements.
