# Assign Obligations to Team Members

Learn how to assign obligations to other users or team members.

Obligation management provides a centralized platform for you to stay updated on deadlines, assign tasks, and track completion status. Contract managers assign obligations to team members to help improve accountability by ensuring that obligations are tracked, monitored, and completed.

Obligation details are updated from the Agreement preview page. There are a number of ways to access an obligation's details:

- Select an obligation from one of the Obligations dashboard reports.
- Select an obligation from the Obligations tab in the party hub.
- Select an obligation from the Obligations pane on the Agreement preview page.

This topic reviews how to update the obligation details by starting from the Completed Documents view.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Apply filters and enter search criteria to locate the relevant agreements.
   1. Enter a keyword or string in the search field and select a suggestion from the dropdown or press the Enter key.

      The agreements containing the keyword or search string that you entered are displayed in the table.
   2. Use the filters to refine the list of search results:

      1. Expand the Filters option and select the filter that you want to apply.
      2. Choose a filter value.
      3. Select the Apply option.

   The agreements that match the entered search criteria and applied filters are displayed in the table.
5. Review the list and select (click) an agreement from the table.

   The Agreement preview page opens. The Details pane is displayed by default.
6. Select the Obligations (flag) icon from the icon tray on the far-left of the page.

   The Obligations pane opens and a list of obligations that were surfaced in the agreement are displayed.
7. Review the list and select (click) the obligation that you want to assign.

   The obligation details are displayed.
8. Place the pane into *edit* mode. Select the Edit (pencil) icon.

   The page refreshes and the Edit Obligation pane is opened for updates.
9. Enter the name or email address of the relevant team member in the Owner field:

   ![A snippet of the Edit Obligations pane that highlights the Owner field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Obligations_Assign-obligation_Owner-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxNjMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.T4o3bgaYpL2EuN7lWzbenRgoUpSzaPgmFiqJYNi3eGg&_LANG=enus "A snippet of the Edit Obligations pane that highlights the Owner field.")

   As you enter the name or email address, a type-ahead dropdown list is displayed.
10. Select the appropriate user from the list.
11. Select the Save option from the bottom of the pane.

    A message states that the obligation details have been saved. The obligation details pane reverts back to *read-only*.

    Note: View your assigned obligations from the Obligations dashboard. See the [View Assigned Obligations](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=cpx1745434861979.html) topic to learn more about viewing your assigned obligations.
12. Select < Back from the top of the pane.

    The Obligations pane opens and a list of agreement obligations are displayed.
13. Repeat this process until all the relevant obligations have been assigned.
14. Return to the Agreement Manager homepage. Select the Close (x) icon from the top left.

    Tip: If you accessed this page from another location, select the back arrow from the top-left corner to return to the previous page. ![A snippet of the Agreement Preview page that highlights the Back arrow in the top-left corner.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Preview-page_Example-with-Back-arrow_hilgihghted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxNjMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.T4o3bgaYpL2EuN7lWzbenRgoUpSzaPgmFiqJYNi3eGg&_LANG=enus "A snippet of the Agreement Preview page that highlights the Back arrow in the top-left corner.")

    For example, select the back arrow to return to the Obligations page in the party hub.

You have successfully assigned an obligation owner from the Agreement preview page.
