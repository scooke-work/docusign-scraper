# Perform a Keyword Search to Locate Agreements

This topic answers the question: How can I locate agreements by performing a keyword search, using type-ahead suggestions, or by selecting from my search history?

Perform a keyword search to locate agreements in Agreement Manager. After reviewing the results, use the feedback icons to rate the quality of the search. You can also download selected agreements or export agreement data.

Tip: If you routinely search for the same type of agreements or agreement data, save your search criteria for future use. See the [Save Search Criteria for Future Use](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tskc38121a1-bbf8-4da8-8ed7-b364991f7e50.html&_LANG=enus "Learn how to save the search criteria for the queries that you frequently perform.") topic to learn more about the saved search feature.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. The documents to which the logged-in user has access are listed in the table. The total document count is displayed the bottom right. This count is updated when you perform a search or apply filters.

   ![A screenshot of the Completed Documents page that shows a list of documents to which the logged-in user has access.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-view_updated-with-document-count.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzNTAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.RwMUimzqQC9WVWfJuO7yTLNWyWuWWxfgB3dJR37wo9I&_LANG=enus "A screenshot of the Completed Documents page that shows a list of documents to which the logged-in user has access.")

   Tip: The search field supports the use of advanced search queries to locate relevant agreements. Enter Boolean operators, proximity searches, term boosting, and wildcards directly in the search field to combine multiple conditions in a single query. See the [Advanced Search Queries](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=conce9d9428-f27e-49ad-9001-3c7a0820ae78.html&_LANG=enus "Read about the advanced search query feature that gives you control over how your results are matched and ranked.") article to learn more about these search queries.
4. Use the search field to locate the relevant agreements:

   ![A snippet of the type-ahead dropdown that shows the search history, suggested questions, and recent documents.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Search_typeahead-dropdown.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzNTAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.RwMUimzqQC9WVWfJuO7yTLNWyWuWWxfgB3dJR37wo9I&_LANG=enus "A snippet of the type-ahead dropdown that shows the search history, suggested questions, and recent documents.")

   1. Click in the search field. A dropdown opens and your search history and recent documents are displayed.

      Tip: Access your Saved Searches from the dropdown to run an existing query. See the [Save Search Criteria for Future Use](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tskc38121a1-bbf8-4da8-8ed7-b364991f7e50.html&_LANG=enus "Learn how to save the search criteria for the queries that you frequently perform.") topic to learn more about this feature.
   2. Enter a keyword or phrase in the search field.

      As you enter a keyword in the search field, suggestions are displayed in the dropdown.
   3. Select a suggestion from the dropdown or press the [Enter] key.

   Important: Keyword searches are different from applying filters. The search reviews the contents of agreements, while the filters display agreements based on surfaced attributes and field values.

   By default, the query attempts to match the search criteria with filter values in your documents. The number of results and the identified filters are displayed. This is not the same as performing a traditional keyword search and the results may vary between the two types of searches.

   Tip: The File Name column lists the attributes (fields) where the keyword or phrase matched the agreement data.
5. Select the Use keyword search instead option:

   ![A snippet of the agreement list table that shows query results and the 'Use keyword search instead' option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Keyword-search_results_Use-keyword-search-instead_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzNTAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.RwMUimzqQC9WVWfJuO7yTLNWyWuWWxfgB3dJR37wo9I&_LANG=enus "A snippet of the agreement list table that shows query results and the 'Use keyword search instead' option.")

   Note: If the query is unable to match the search criteria with any filters, the regular keyword search is displayed.

   The agreements that matched the keyword or phrase are displayed in the agreement list table. The document count is updated based on your search criteria. In this version of the search results, the query returned agreements that contained the entered keyword or phrase. It did not use the search criteria as an applied filter.

   Tip: The keyword search automatically includes synonyms and related concepts in the search results. Select the related terms link to view additional information: ![A snippet of the agreement list table that shows query results and the Show related terms dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Search_Keyword-search_Related-terms-dialog.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzNTAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.RwMUimzqQC9WVWfJuO7yTLNWyWuWWxfgB3dJR37wo9I&_LANG=enus "A snippet of the agreement list table that shows query results and the Show related terms dialog box.") The Related terms toggle is turned on (blue position) by default. Select the toggle to turn this feature off (white position).
6. Review the search results to determine if the query returned the relevant agreements.
7. Use the feedback icons to rate your search experience:

   ![A snippet of the Completed Documents page that highlights the feedback icons and options to provide a reason and comments.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Keyword-search_results_share-feedback-modal_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzNTAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.RwMUimzqQC9WVWfJuO7yTLNWyWuWWxfgB3dJR37wo9I&_LANG=enus "A snippet of the Completed Documents page that highlights the feedback icons and options to provide a reason and comments.")

   1. Choose the appropriate feedback icon. Select either the Rate as helpful (thumbs up) icon or the Rate as unhelpful (thumbs down) icon.

      The Share your reason to help us improve dialog box opens.
   2. Select a reason for choosing the rating. You can select more than one reason. The choices are: Complete, Relevant, Accurate, and Other.
   3. Enter additional details in the Comments text box.
   4. Select the Submit option.

      Tip: Choose the Skip option to submit your rating without providing any additional information. Your rating is submitted for review.

   The dialog box closes and your feedback is submitted to Docusign.
8. If the search did not return the expected agreements, perform another keyword search:
   1. Select the Clear Search (x) icon located in the search field.

      The search criteria is cleared and the agreement list table resets.
   2. Enter new search criteria in the search field.
   3. Select a suggestion from the type-ahead dropdown or press the [Enter] key.

   The agreement list table updates and the agreements that matched the entered keyword or phrase are displayed. Continue repeating this process until you locate the relevant agreements.

   Note: After locating the relevant agreements, choose to download selected agreements or export agreement data. See these support articles to learn more about these options:
   - [Download Documents from the Agreement List Table](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=ach1724363496037.html&_LANG=enus "This topic answers the question: How do I download agreements from Agreement Manager?")
   - [Export Data from the Agreement List Table](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=pqf1724363438241.html&_LANG=enus "Learn how to export agreement data to a CSV file for selected agreements.")

You have successfully performed a keyword search to locate relevant agreements.
