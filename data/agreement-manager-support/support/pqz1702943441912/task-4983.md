# Display Clause Columns in the Agreement List Table

Learn how to display clause-related columns in the agreement list table.

Display clause-related columns in the agreement list table so that you can quickly scan which agreements contain key terms. For example, Termination for Convenience or specific liability caps across a group of contacts.

## Before you begin

- Make sure that you can open Agreement Manager and see the agreements that you want to review. The clause filters only apply to agreements that you already have permission to view.
- If your organization uses AI-extracted clauses, confirm that clause extraction is enabled in your environment. This is required to use the clause-related columns and filters.

  Note: Clause extraction for Agreement Manager is available in Beta. Contact your Docusign account representative to inquire about participating in the Beta.
- If you want to edit or validate clause values, confirm that you have Organize and manage all agreements access to Agreement Manager. Users with Can Manage or Can Edit access to an agreement can also edit of validate clause presence.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Select the Show or Hide Fields (toggle) icon located to the right of the filters:

   ![A snippet of the Completed Documents view that highlights the Show or Hide FIelds (toggle) icon on the right-side of the page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Customize-columns-gear-icon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.uD1c9JHe4rz_5g3lyjUHFnHnqY7Ly3FqJyVZ23iKyJA&_LANG=enus "A snippet of the Completed Documents view that highlights the Show or Hide FIelds (toggle) icon on the right-side of the page.")

   The Show or Hide FIelds dialog box opens. A number of fields are turned on (blue) by default. These fields represent the columns in the agreement list table. Columns display in the order that the fields are represented in the dialog box.

**Show the relevant clause-related columns**

5. Locate the relevant clause-related columns. Enter the name of the clause in the search field.

   Tip: Enter *Clause* in the Search Fields field to view all the clause-related columns.

   The fields (columns) that matched the search criteria are displayed.
6. Turn on the clause fields (columns) that you want to display in the table and select the Save option. Select the toggles to turn on (blue position) the relevant clauses:

   ![A snippet of the Show or Hide Fields dialog box that shows the clause-related columns being displayed in the table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Show-Hide-Fields_clause-columns.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.uD1c9JHe4rz_5g3lyjUHFnHnqY7Ly3FqJyVZ23iKyJA&_LANG=enus "A snippet of the Show or Hide Fields dialog box that shows the clause-related columns being displayed in the table.")

   The dialog box closes and you are returned to the Completed Documents view. The clause-related columns are displayed in the table.

**Re-order the table columns**

The column order in the agreement list table can be updated and saved for future use. For example, display the clauses that you want to track and reorder the columns so they are easy to locate. Use the Save option to save your filters and column customizations for future use.

7. Re-order the clause-related columns so they are easier to locate in the table:

   ![A snippet of the Show or Hide Fields dialog box that highlights a clause column in the process of being dragged to a new location.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Show-Hide-Fields_reorder-columns_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0NzcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.uD1c9JHe4rz_5g3lyjUHFnHnqY7Ly3FqJyVZ23iKyJA&_LANG=enus "A snippet of the Show or Hide Fields dialog box that highlights a clause column in the process of being dragged to a new location.")

   1. Select the Show of Hide Fields (toggle) icon located on the right-side of the table.

      The Show or Hide Fields dialog box opens.
   2. Locate the clause-related fields (columns).
   3. Hove the cursor over the Reorder (dots) icon for the clause that you want to move and drag it to the appropriate location.
   4. Continue reordering the columns as needed.
   5. Select the Save option.

   The dialog box closes and the columns are reordered in the table.

**Save your column layout**

The New saved search feature saves your search terms, applied filters, and your column customizations. You can save the search without entering search criteria or applying filters, basically saving only the table layout.

8. Save your table layout for future use. Select the Save option and complete the New saved search dialog box:
   1. Enter a unique Name for the saved search. For example, "Clause-focused layout".
   2. Provide a Description for the saved search. For example, "The clause columns are displayed first in the table".
   3. Select the Save Search option.

   The dialog box closes and the saved search is displayed. The clause-related columns are displayed prominently in the table.
9. Return to the Agreement Manager homepage. Select the Completed Documents option from the breadcrumb navigation in the top left.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.

   Tip: Since you saved the table layout, you can update the columns and the column order while still having access to the clause-forward layout. See the [Save Search Criteria for Future Use](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tskc38121a1-bbf8-4da8-8ed7-b364991f7e50.html&_LANG=enus "Learn how to save the search criteria for the queries that you frequently perform.") article to learn more about saving searches.

You have successfully displayed clause-related columns in the agreement list table. These columns indicate the presence or absence of key clause language.
