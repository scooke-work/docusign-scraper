# Manually Create an Obligation

Learn how to create an obligation from the Agreement preview page for a selected agreement.

Some obligations are automatically surfaced in your agreements, such as payments, renewal notices, and termination notices. These obligations are displayed in the Obligations pane on the Agreement preview page. Obligations can also be manually created from the Agreement preview page.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Perform a search and apply filters to locate the relevant agreements:
   1. Enter a keyword or string in the search field and select a suggestion from the dropdown or press the Enter key.

      The agreements containing the keyword or search string that you entered are displayed in the table.
   2. Use the filters to refine the list of search results:

      1. Expand the Filters option and select the filter that you want to apply.
      2. Choose a filter value.
      3. Select the Apply option.

      The agreements that match the applied filters are displayed in the table.
5. Select an agreement from the list.

   The Agreement preview page opens for the selected agreement. The Details pane is displayed by default and shows a list of attributes and suggested values. Attributes, or fields, are displayed based on the identified agreement type.

   Before you create a new obligation, review the list of existing obligations.
6. Select the Obligations (flag) icon from the icon tray to the left of the Details pane.

   The Obligations pane opens. A list of obligations that were surfaced with AI or manually created is displayed in the pane.

   The name, owner, due date, and status are displayed for each obligation.
7. Before creating a new obligation, review the list to make sure that you are not adding a duplicate. If necessary, select an obligation to review additional details.
8. Highlight the clause associated with the obligation that you want to create from the preview pane. A dropdown menu of options opens. Select the Create Obligation option:

   ![A snippet of the document preview pane that shows the Create Obligation option for a highlighted clause.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Obligation_Create-obligation-option_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxNjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.UbWqUHm8HaSYiMXK_fujM6_V45Eodv6hg7DSEheyZoI&_LANG=enus "A snippet of the document preview pane that shows the Create Obligation option for a highlighted clause.")

   The Create Obligation pane opens and a list of data fields is displayed.
9. Complete the required data fields in the Create Obligation pane:

   ![A screenshot of the Agreement preview page that highlights the Create Obligation pane including the obligation's data fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Obligations_Create-Obligation-panehighlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxNjAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.UbWqUHm8HaSYiMXK_fujM6_V45Eodv6hg7DSEheyZoI&_LANG=enus "A screenshot of the Agreement preview page that highlights the Create Obligation pane including the obligation's data fields.")

   1. Enter a name for the obligation in the Obligation name field. This field is required.
   2. Select an obligation type from the Type dropdown.
   3. Select an obligation status from the Status dropdown.
   4. Indicate the Notice period for the obligation. The Notice date field is auto-calculated by subtracting the notice period from the expiration date.

      Note: The Notice period and Notice date fields are specific to renewal and termination obligations. All other obligation types have a Due date field that serves the same purpose.
   5. Enter or select a Due date for the obligation.
   6. Select a frequency from the Frequency dropdown. This field is associated with payment obligations.
   7. Enter the name or email address of the Owner.
   8. Select a party from the Obligated party dropdown.
   9. Select a party from the Benefiting party dropdown.
   10. Enter the repercussions for not fulfilling the obligation in the Repercussions  field.
   11. Enter a description for the obligation in the Description field.
10. Select Save.

    The new obligation is saved and displays in the Obligations pane.
11. Repeat this process to create additional obligations as needed.
12. Return to the Agreement Manager homepage. Select the Close (x) icon from the top left.

    The Agreement Manager homepage opens and the Completed Documents view is displayed. Your search criteria and filters are still applied.

You have successfully created an obligation from the Agreement preview page for an agreement.
