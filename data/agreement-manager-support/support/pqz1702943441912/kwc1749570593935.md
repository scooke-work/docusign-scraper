# Report Notifications and Subscriptions

Read about the notification and subscription features and how they can help you proactively manage your agreements, obligations, and renewals.

Use the *Subscriptions* feature to sign up for report notifications. You can subscribe to receive email reports for any of the reports found on the Agreements, Obligations, and Renewals dashboards. You can also subscribe to receive email reports for custom reports.

The *Subscription* feature is accessed from the detailed report view for a dashboard chart or insight:

![A snippet of the detailed report view that highlights the Subscription feature.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Subscriptions_Subscribe-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxMDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.3orTEDscBaALhMhqcT_UbjQtB-4p2qjrT3F0gKK2r-Y&_LANG=enus "A snippet of the detailed report view that highlights the Subscription feature.")

Apply filters and customize the table until the data that you want to track is displayed in the table.

Tip: If you made changes to the report, use the Save option to create a custom report. Expand the dropdown and select Save:![A snippet of an agreement dashboard report that shows the Save option in the dropdown.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Agreements-Dashboard_Dashboard-report_Save-as.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxMDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.3orTEDscBaALhMhqcT_UbjQtB-4p2qjrT3F0gKK2r-Y&_LANG=enus "A snippet of an agreement dashboard report that shows the Save option in the dropdown.")

Once saved, access the report from the Custom tab located under the Reports view.

Complete the Create subscription dialog box to set up one-time or recurring email reports for yourself and others:

![A screenshot of the Create subscription dialog box that shows the subscription configuration fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Create-Subscription.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxMDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.3orTEDscBaALhMhqcT_UbjQtB-4p2qjrT3F0gKK2r-Y&_LANG=enus "A screenshot of the Create subscription dialog box that shows the subscription configuration fields.")

Note: When you create a subscription, you have the option to add individual users and groups. Once created, the subscription is displayed in the Manage report subscriptions dialog box for any individual users who were added. Currently, the subscription is not visible to users if they were added as part of a group.

See the [Subscribe to Receive Report Notifications](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=wfw1734545937270.html&_LANG=enus "Learn how to set up one-time or recurring email reports for yourself or others to receive notifications about your obligations, renewals, or agreements.") topic to learn more about subscribing to receive email reports.

## Notification preferences for completed agreements

Docusign provides email notifications for all account-related updates, including updates related to your agreements in Agreement Manager. Notification settings give you control over the email notifications that you receive.

The Notification Preferences page is accessed from the My Preferences option in the user profile menu. By default, all the notification types in the Completed Agreements section are enabled:![A screenshot of the Notification Preferences page that highlights the selected notification types for completed agreements.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/My-preferences_Notification-Preferences_For-Completed-Agreements_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcxMDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.3orTEDscBaALhMhqcT_UbjQtB-4p2qjrT3F0gKK2r-Y&_LANG=enus "A screenshot of the Notification Preferences page that highlights the selected notification types for completed agreements.")

Deselect a checkbox if you no longer want to receive email notifications for the corresponding notification type.

Tip: Periodically review your Notification Preferences to ensure that you are receiving the relevant email notifications. Additional notification types will be added in the future.

See the [Customize Notification Settings for Your Completed Agreements](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk155b1ed7-7b45-43cd-be92-2a5279630c0f.html&_LANG=enus "Learn how to customize your email notification settings from the Notification Preferences page.") topic for steps to update your notification preferences.
