# Edit Agreement Set Details

Learn how to update the details for an agreement set, including the set name and description.

Contract managers and any users who are assigned the Organize and manage all agreements permission can create and update agreement sets. This topic reviews the steps to update the set details for an agreement set.

Note: An organization can have up to 50 agreement sets per account. If you exceed this limit, an error message is displayed.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents page is displayed.
4. Expand the Settings (gear) menu from the top right and select the Sets option:

   ![A screenshot of the Settings (gear) menu on the Agreement Manager homepage that highlights the Sets option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-sets_entry-point_Manage-sets-action-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczOTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.d8ldvJIQWynR24rbA4LV1fWDapArudEiC0hvs8KBs88&_LANG=enus "A screenshot of the Settings (gear) menu on the Agreement Manager homepage that highlights the Sets option.")

   The Agreement sets page opens. The All agreement sets view is displayed. The table lists the sets that were shared with or created by the logged-in user.
5. Locate the set that you want to update.

   Tip: Select the sorting arrows in the Name column to sort the table by set name.
6. Expand the More actions (three dots) menu for the set that you want to update and select Edit details:

   ![A snippet of the All agreement sets view that highlights the Edit details option in the More actions menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Sets-dialog_All-agreement-sets_Edit-Details.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczOTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.d8ldvJIQWynR24rbA4LV1fWDapArudEiC0hvs8KBs88&_LANG=enus "A snippet of the All agreement sets view that highlights the Edit details option in the More actions menu.")

   The Edit agreement set dialog box opens.

   ![A screenshot of the Edit agreement set dialog box that shows the Name and Description fields for the selected set.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-set-management_Updaed-details_Edit-agreement-set-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczOTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.d8ldvJIQWynR24rbA4LV1fWDapArudEiC0hvs8KBs88&_LANG=enus "A screenshot of the Edit agreement set dialog box that shows the Name and Description fields for the selected set.")
7. Update the Name and Description for the set as needed.
8. Select Save.

   The dialog box closes. The updated set name and description are displayed in the agreement set table.
9. Select Completed Documents from the breadcrumb navigation at the top of the page.

   You are returned to the Agreement Manager homepage.

You have successfully edited the details for a selected agreement set.
