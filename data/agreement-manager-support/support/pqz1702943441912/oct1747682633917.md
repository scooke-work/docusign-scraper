# Administrator: View and Manage Approved Apps

As a Docusign App Center administrator, learn how to access, view, and manage Docusign approved apps.

As a Docusign App Center administrator, you can access, view, and manage Docusign approved apps.

1. From the  [Docusign App Center page](https://apps.docusign.com/app-center), select Extension Apps > Approved Apps.

   Tip: You can alternately select Manage > Approved Apps.

   The List of approved apps section shows apps that are available for your organization's users.

   ![A screenshot of the App Center Approved Apps page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/ac-newest-approved-pg.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTgyMzYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.9eC09A3avE-9WXGr2Kbolp1KbexTm_zQvstardg5qnY&_LANG=enus)
2. For apps that are approved but not installed, select  Install App to install the app for all account users.
3. Use the overflow menu to manage your approved apps.

   | Function | Select to: |
   | --- | --- |
   | View Details | Display the [App Detail page](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=xkg1702568563800.html&_LANG=enus "Use a Docusign app's detail page to view important information such as an app summary, features, permissions, and resources.") for detailed information about the app. |
   | Remove App | Remove from the  **List of approved apps**. Also removes the ability to install the app. Note: Existing users can continue to use the app. To restrict all access to an app, uninstall the app. |
   | Uninstall | Uninstall the app from App Center and remove the app for all users. |

## Select approved apps

You can add apps to an account’s approved apps list from the Select Apps to Approve  section.

1. From the  [Docusign App Center page](https://apps.docusign.com/app-center), select Manage > Approved Apps.

   Note: You can optionally access the Approved Apps page by selecting Extension Apps > Approved Apps.
2. From the Select apps to approve  section, select an app using one of these methods:
   - Select Approve for the app that you want.
   - Search for the app by typing part of the app's name. When the app displays, select Approve.

     ![A screenshot of searching for an app to approve.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/ac-approved-apps-adm-sel-search.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTgyMzYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.9eC09A3avE-9WXGr2Kbolp1KbexTm_zQvstardg5qnY&_LANG=enus)
   - Select one, multiple, or all available apps and choose Add to Approved List.

     ![A screenshot of an App Center approved apps selection example.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/ac-new-sel-apps-to-approve.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTgyMzYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.9eC09A3avE-9WXGr2Kbolp1KbexTm_zQvstardg5qnY&_LANG=enus)

     The apps move to the List of approved apps.
3. In the overflow menu, select Views Details  to display the [app's Detail page](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=xkg1702568563800.html&_LANG=enus "Use a Docusign app's detail page to view important information such as an app summary, features, permissions, and resources.")  for detailed information.

### Install an approved app

Install an approved app to provide access to account users.

1. From the [Docusign App Center page](https://apps.docusign.com/app-center), select Manage > Approved Apps.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/ac-approve-install-tag.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTgyMzYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.9eC09A3avE-9WXGr2Kbolp1KbexTm_zQvstardg5qnY&_LANG=enus)
2. Select Install App.

   The Approved  tag changes to Installed and the app is ready for account user access.

## Remove an approved app

You can remove an app from the approved apps listing.

1. From the [Docusign App Center page](https://apps.docusign.com/app-center), select Manage > Approved Apps.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/ac-approve-install-tag.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTgyMzYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.9eC09A3avE-9WXGr2Kbolp1KbexTm_zQvstardg5qnY&_LANG=enus)
2. Select the overflow menu and select Remove app.
3. At the confirmation dialog box, select Remove App  to move the app to the Select Apps to approve list.

   ![A screenshot of the Remove app from approved list dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/ac-remove-app-db.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTgyMzYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.9eC09A3avE-9WXGr2Kbolp1KbexTm_zQvstardg5qnY&_LANG=enus)

   The app is no longer available for account users.
