# Customize and Save Reports for Future Use

Learn how to customize and save an agreement report for future use.

Create your own reports by applying filters and customizing the agreement table for an existing report. Once saved, the report is listed under the Custom tab on the Reports view.

![A snippet of the Reports view that shows custom reports found on the Custom tab.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Reports-view_Custom-tab-snippet.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5ODQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.k7EMRfqjm6ASvoM4tXaaXTFvlW9_2AZxGHN9pXL-jhA&_LANG=enus "A snippet of the Reports view that shows custom reports found on the Custom tab.")

Tip: Add the report to Your Favorites list. Select the Favorite (star) icon located in the top-right corner: ![A snippet of a detailed report view that highlights the Favorite (star) icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreements-dashboard_Report-details_Favorite-icon-selected_generic.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5ODQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.k7EMRfqjm6ASvoM4tXaaXTFvlW9_2AZxGHN9pXL-jhA&_LANG=enus "A snippet of a detailed report view that highlights the Favorite (star) icon.")Your starred dashboards and reports are displayed in the Your Favorites section on the Overview dashboard.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Insights from the global header.

   The Insights hub opens and the Overview is displayed.

   Note: **Suggested Reports (Beta)**

   The Suggested Reports section provides AI-generated custom report recommendations based on the
   custom fields in your account. Use these suggestions to create custom reports. See the [Create Custom Reports from Suggested Reports (Beta)](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tskfc9f9485-65b6-4a2c-95e3-15bd13ce8562.html&_LANG=enus "Learn how to create a custom report from one of the reports in the Suggested Reports (Beta) section on the Overview dashboard.") to learn more about this feature.
3. Select Agreements from the navigation pane on the left.

   The Agreements Dashboard view opens.
4. Go to the detailed report view for the dashboard chart (report) that you want to customize. Select (click) the title of the chart that you want to customize.

   You can also expand the More actions (three dots) menu and select the View Report option:

   ![A snippet of the Agreements Dashboard that highlights the View Report option in the Actions menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-dashboard_Actions_View-Report_generic.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5ODQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.k7EMRfqjm6ASvoM4tXaaXTFvlW9_2AZxGHN9pXL-jhA&_LANG=enus "A snippet of the Agreements Dashboard that highlights the View Report option in the Actions menu.")

   The detailed report view opens for the selected chart or insight.
5. Refine the report until it reflects the relevant agreement data. Expand a filter, choose a value or date range, and select the Apply option.
6. Use the All Filters option to apply additional filters to the report:

   ![A screenshot of the All Filters dialog box that shows a list of available and applied filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Agreements-Dashboard_Report-View_All-Filters-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5ODQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.k7EMRfqjm6ASvoM4tXaaXTFvlW9_2AZxGHN9pXL-jhA&_LANG=enus "A screenshot of the All Filters dialog box that shows a list of available and applied filters.")

   1. Select the All Filters
      option at the top of the report.

      The All Filters dialog
      box opens.
   2. Expand the filter that you want to apply, such as the Status filter.
   3. Select or enter the filter value.
   4. Select Apply.
   5. Repeat these steps until all necessary filters are applied.
   6. Select Save.

      The dialog box closes. The report is updated to reflect the filters
      applied.
7. Customize the columns in the table:

   Important: The Customize columns function only applies to the data that is displayed in the agreement list table. When downloading or exporting the report, all fields are included in the output file. This includes both custom and standard fields.

   ![A screenshot of the Customize columns dialog box that shows the columns you can choose to display in the agreements table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Overview-dashboard_Customize-columns.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5ODQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.k7EMRfqjm6ASvoM4tXaaXTFvlW9_2AZxGHN9pXL-jhA&_LANG=enus "A screenshot of the Customize columns dialog box that shows the columns you can choose to display in the agreements table.")

   1. Select the Customize columns (toggle) icon.

      The Customize columns dialog box opens. These fields represent the columns in the table.
   2. Select the toggles to turn the associated field on (blue) or off (white).
   3. Re-order the columns as needed. Hover the cursor over the Reorder (dots) icon for the column that you want to move and drag it to the appropriate location.
   4. Select the Save option.

   Tip: If the Status column is not displayed by default, use the Show or Hide Fields (toggle) option to add it to the table. From the Show or Hide Field dialog box, turn on the Status toggle and drag the field to the appropriate location. Be sure to Save your changes.

   Continue to apply filters and customize the table until the report reflects the relevant data.
8. Save the report. Expand the More actions (three dots) menu in the top right and select the Save option:

   ![A snippet of an agreement dashboard report that shows the Save As option in the dropdown.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Report-view_Dropdown-expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5ODQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.k7EMRfqjm6ASvoM4tXaaXTFvlW9_2AZxGHN9pXL-jhA&_LANG=enus "A snippet of an agreement dashboard report that shows the Save As option in the dropdown.")

   The Save Report dialog box opens.

   ![A screenshot of the Save Report dialog box that shows the Report name field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreements-Dashboard_Customize-report_Save-Report-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5ODQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.k7EMRfqjm6ASvoM4tXaaXTFvlW9_2AZxGHN9pXL-jhA&_LANG=enus "A screenshot of the Save Report dialog box that shows the Report name field.")
9. Enter a unique name for the report in the Report name field.

   Note: The name of the selected report is displayed by default. Overwrite this value to reflect your customized report.
10. Enter a description about the report's purpose in the Description field.
11. Select the Save option.

    The dialog box closes and the custom report is saved. Your new report is available from the Custom tab under the Reports view:

    ![A snippet of the Reports view that shows the Custom tab, including custom reports and the View option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreements-Dashboard_Customized-report_Highlighted-in-Custom-section.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5ODQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.k7EMRfqjm6ASvoM4tXaaXTFvlW9_2AZxGHN9pXL-jhA&_LANG=enus "A snippet of the Reports view that shows the Custom tab, including custom reports and the View option.")Custom reports can be updated or deleted at any time from the report details page. See the [View and Update Your Custom Reports](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=yuf1727801071281.html&_LANG=enus "This topic answers the question: How can I review and update my custom reports?") topic to learn about editing options for your custom reports.
