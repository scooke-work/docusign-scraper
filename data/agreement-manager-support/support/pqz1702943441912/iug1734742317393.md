# Monitor and Report on Renewals

Learn how to monitor and report on renewals from the Renewals dashboard, including subscribing to receive notifications. This topic answers the question: How do I report on my renewals?

The Renewals dashboard provides a visual summary of your upcoming agreement renewals through the use of bar and pie charts. After applying filters, you can go to the detailed report view for any of the dashboard charts.

Note: Subscribe to receive daily report notifications about your agreement renewals. This feature is available from the detailed report view. Review the [Subscribe to Receive Report Notifications](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=wfw1734545937270.html&_LANG=enus "Learn how to set up one-time or recurring email reports for yourself or others to receive notifications about your obligations, renewals, or agreements.") article to learn more about this feature.

**Access Renewals dashboard:**

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Insights from the global header.

   The Insights hub opens and the Overview is displayed.
3. Select the Renewals option from the navigation pane on the left.

   The Renewals dashboard opens.

   ![A screenshot of the Renewals dashboard that shows the agreement charts and filter options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Renewals-dashboard.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.2WuSxrIxJSxVJfkB4QaJ37g_NinZILfJCyJWv0kMw6I&_LANG=enus "A screenshot of the Renewals dashboard that shows the agreement charts and filter options.")

**Apply filters to the dashboard:**

4. Filter the Renewals dashboard to locate upcoming renewals:

   ![A snippet of the Renewals dashboard that highlights the dashboard filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Renewals-dashboard_Filter-pills.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.2WuSxrIxJSxVJfkB4QaJ37g_NinZILfJCyJWv0kMw6I&_LANG=enus "A snippet of the Renewals dashboard that highlights the dashboard filters.")

   Filters include: Parties, Renewal Notice Date, Renewal Owner, Renewal Type, and Status.

   1. Expand the filter that you want to apply.
   2. Select or enter the filter value.
   3. Select the Apply option.

   Important: When using the Parties filter, you must also select a Party Role value. If you do not know which role to select, choose the All option. If no role is selected, the search returns agreements where the party is not the buyer or the seller.

   Tip: Use the Filters option to access additional filters:![A screenshot of the Filters dialog box that shows additional filters that you can apply to the agreement list.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_All-Filters-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.2WuSxrIxJSxVJfkB4QaJ37g_NinZILfJCyJWv0kMw6I&_LANG=enus "A screenshot of the Filters dialog box that shows additional filters that you can apply to the agreement list.")

   The charts are updated to reflect the filters that you applied.

**View dashboard charts:**

5. Review the dashboard charts: Upcoming renewals and Renewals by type.

   Hover your cursor over a segment or bar to view the representative agreement count and percentage:![A screenshot of the Renewals by type chart that shows the hover messaging for a selected chart segment.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Renewals_Renewal-types_hover-on-segment.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.2WuSxrIxJSxVJfkB4QaJ37g_NinZILfJCyJWv0kMw6I&_LANG=enus "A screenshot of the Renewals by type chart that shows the hover messaging for a selected chart segment.")

**Access the detailed report view:**

6. Access the detailed report view for a dashboard chart:

   ![A snippet of the Renewals dashboard that shows the View Report option in the More actions (three dots) menu for a dashboard chart.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Renewals-dashboard_Chart_Actions_View-Report.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.2WuSxrIxJSxVJfkB4QaJ37g_NinZILfJCyJWv0kMw6I&_LANG=enus "A snippet of the Renewals dashboard that shows the View Report option in the More actions (three dots) menu for a dashboard chart.")

   - Select the chart title.
   - Hover the cursor over the chart, expand the More actions (three dots) menu and select the View Report option.

   The detailed report view opens for the selected chart. The agreements represented in the chart are listed in the table.

   ![A screenshot of the detailed report view for the Renewals by type dashboard chart.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Renewals-dashboard_detailed-report-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.2WuSxrIxJSxVJfkB4QaJ37g_NinZILfJCyJWv0kMw6I&_LANG=enus "A screenshot of the detailed report view for the Renewals by type dashboard chart.")

**Customize and download the report:**

7. Review the table to verify that the report contains the relevant renewals. If necessary, use the filters to refine the agreement list table.
8. Customize the information that is displayed in the table:

   ![A screenshot of the Show or Hide Fields dialog box that shows the columns you can choose to display in the agreements table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Overview-dashboard_Customize-columns.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.2WuSxrIxJSxVJfkB4QaJ37g_NinZILfJCyJWv0kMw6I&_LANG=enus "A screenshot of the Show or Hide Fields dialog box that shows the columns you can choose to display in the agreements table.")

   1. Select the Show or Hide Fields (toggle) icon. This icon is located on the right-side of the header row in the table.

      The Show or Hide Fields dialog box opens. These fields represent the columns in the table.
   2. Select the toggles to turn the associated field on (blue) or off (white).
   3. Re-order the columns as needed. Hover the cursor over the Reorder (dots) icon for the field that you want to move. Drag it to the appropriate location.
   4. Select the Save option.

   Continue applying filters and customizing the table data as needed.
9. Download the report. Expand the More actions (three dots) menu in the top right and select the Download option:

   ![A snippet of a detailed report view that shows the Download option in the expanded dropdown.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Renewals_Detailed-report_Download.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.2WuSxrIxJSxVJfkB4QaJ37g_NinZILfJCyJWv0kMw6I&_LANG=enus "A snippet of a detailed report view that shows the Download option in the expanded dropdown.")

   The associated report is downloaded to your local Downloads folder.

**Subscribe to the report:**

10. Set up one-time and recurring email reports for yourself and others. Select the Manage Notification (bell) icon and complete the Create subscription dialog box:

    ![A screenshot of the Create subscription dialog box that is used to subscribe to report notifications.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Renewals-dashboard_Report-view_Create-subscriptions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.2WuSxrIxJSxVJfkB4QaJ37g_NinZILfJCyJWv0kMw6I&_LANG=enus "A screenshot of the Create subscription dialog box that is used to subscribe to report notifications.")

    Note: When you create a subscription, you have the option to add individual users and groups. Once created, the subscription is displayed in the Manage report subscriptions dialog box for any individual users who were added. Currently, the subscription is not visible to users if they were added as part of a group.

    Review the [Subscribe to Receive Report Notifications](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=wfw1734545937270.html&_LANG=enus "Learn how to set up one-time or recurring email reports for yourself or others to receive notifications about your obligations, renewals, or agreements.") article to learn more about this feature.

**Add the report to Your Favorites:**

11. Add the report to Your Favorites list. Select the Favorite (star) icon located in the top right:

    ![A snippet of a detailed report view that highlights the Favorite (star) icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Renewals-dashboard_Report-view_Mark-favorite_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.2WuSxrIxJSxVJfkB4QaJ37g_NinZILfJCyJWv0kMw6I&_LANG=enus "A snippet of a detailed report view that highlights the Favorite (star) icon.")Your starred dashboards and reports are displayed in the Your Favorites section on the Overview dashboard.

**Return to the Renewals dashboard:**

12. Return to the Renewals dashboard. Select the Renewals option from the breadcrumb navigation:

    ![A snippet of the detailed report view that highlights the breadcrumb navigation.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Renewals-dashboard_Report-view_Renewals_breadcrumb.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwNDQsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.2WuSxrIxJSxVJfkB4QaJ37g_NinZILfJCyJWv0kMw6I&_LANG=enus "A snippet of the detailed report view that highlights the breadcrumb navigation.")

    You are returned to the Renewals dashboard.

You have successfully reviewed the renewal monitoring and reporting features of the Insights hub.
