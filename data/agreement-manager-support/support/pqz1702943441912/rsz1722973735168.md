# AI Denotation in Agreement Manager

This topic answers the question: In what way does Docusign indicate AI-assisted insights or AI functionality in Agreement Manager?

Methods of denoting AI functionality include:

- Bloom icon
- Spark icon
- AI-Assisted denotation badge
- Dotted lines under AI-suggested attribute (field) values
- Purple highlighting around AI-suggested party names

Important: The dotted lines and purple highlights indicate that the associated values were AI-assisted and must be validated by the user. Once the values are validated against the agreement, the purple dotted lines or highlighting goes away.

The AI-Assisted  denotation badge is clickable. Select this badge to access additional information about how Docusign uses AI functionality:

## "Track obligations" badge

When new obligation-related attributes are available, the Track obligations badge is displayed on the Completed Documents page. For example, when an account is enabled for obligations. This badge indicates that you are not using the latest AI model.

Select the Track obligations badge to get started:

![A screenshot of the expanded Track obligations badge that shows information related to processing agreements for added insights.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reprocessing_New-Insights-pill_expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYwNjMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.2J_iCX50UT8vMw7pGlh975XUFXoydGPck01dJtk-yU4&_LANG=enus "A screenshot of the expanded Track obligations badge that shows information related to processing agreements for added insights.")

Select Start Now to opt in for additional processing. Another dialog box opens:

![A screenshot of the dialog box that contains the opt-in toggle for processing your agreements for new insights.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reprocessing_Turn-on-processing-toggle.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYwNjMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.2J_iCX50UT8vMw7pGlh975XUFXoydGPck01dJtk-yU4&_LANG=enus "A screenshot of the dialog box that contains the opt-in toggle for processing your agreements for new insights.")

Turn on the Process for additional insights toggle (blue position) and select the Save Changes option. The dialog box closes and a message states that your agreements are being processed for additional AI insights.

Note: Once the toggle is activated, it cannot be turned off. Also, you can no longer deactivate processing. Agreements will be processed based on the user's account access. For example, if the user has obligation management as part of their plan, opting in for additional processing includes obligation-related insights.
