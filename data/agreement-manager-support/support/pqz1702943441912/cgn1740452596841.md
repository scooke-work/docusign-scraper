# Activate Your Custom Agreement Types

Learn how to activate the AI detection for a custom agreement type and surface it in selected historical agreements.

## Before you begin

This topic reviews the last step in the AI detection set-up process, *Activate AI detection*. These steps are completed by administrators and contract managers.

Remember: Administrators automatically have access to configure agreement data. For non-administrator users, agreement type management tasks require Organize and manage all agreements access to Agreement Manager agreements and functionality.

Complete these topics before attempting this task:

- [Configure New Custom Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=gke1727282238086.html&_LANG=enus "Learn how to create custom agreement types to support organizational use cases.")
- [Define Your Custom Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tske5fe1ad5-2832-4e6c-9177-f33a9ffade15.html&_LANG=enus "Learn how to initiate the AI detection set-up process and complete the first step to define the custom type.")
- [Add Examples of Your Custom Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tskd38465c6-c8c0-4588-bfb3-fdb6916a1926.html&_LANG=enus "Learn how to complete the second step of the AI detection set-up process, Add examples of this agreement type.")
- [Test the Extraction Accuracy for Your Custom Agreement Types](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=ogw1740503179389.html&_LANG=enus "Learn how to test the accuracy of the AI detection for your custom agreement types.")

In the previous step, you tested the accuracy of the AI detection against a group of selected documents. After verifying the identified agreement types, you selected the Done option and the Results dialog box opened.

![A screenshot of the Results dialog box that shows the results of the accuracy test.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Type_Test-for-accuracy_Results.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MDUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.j0q4HjFlgQY-6eNypwrka_ZPRAvMb7JSFB93HlMarB4&_LANG=enus "A screenshot of the Results dialog box that shows the results of the accuracy test.")

You are prompted to activate the AI detection for the custom agreement type. The
Activate AI detection option is selected by default.

1. Accept the default selection, Activate AI detection, and select the Confirm option.

   The Activate AI detection dialog box opens. Use this dialog box to choose when the system should look for the agreement type. The All new agreements option is selected by default. This option cannot be deselected.

   ![A screenshot of the Activate AI detection dialog box that prompts you to choose when to look for the field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Type_Enable-AI-detection-for-custom-agreement-type--Activate-AI-detection.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MDUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.j0q4HjFlgQY-6eNypwrka_ZPRAvMb7JSFB93HlMarB4&_LANG=enus "A screenshot of the Activate AI detection dialog box that prompts you to choose when to look for the field.")
2. Look for the custom agreement type on your selected historical agreements. Select the Find on specific agreements option.
3. Select the Activate option.

   The AI detection is published and the custom agreement type is active for new agreements.

   The Select documents to process page opens and a list of agreements is displayed. You are prompted to select at least one document to process for the new agreement type.

   Note: The Find on specific agreements option can be used at any time to process specific documents for custom agreement types. See the [Find Custom Agreement Types in Specific Documents](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tskb991328c-dc5f-4d45-9617-dada91a5672a.html&_LANG=enus "Learn how to use the Find On Specific Documents function to surface custom agreement types in selected agreements.") topic to learn more about this function.
4. Locate the documents that you want to process for the new agreement type:

   ![A snippet of the Select documents to process page that shows the search field and filter options to locate documents to process.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Types-Manager_AI-detect-activated_Find-historical-docs.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MDUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.j0q4HjFlgQY-6eNypwrka_ZPRAvMb7JSFB93HlMarB4&_LANG=enus "A snippet of the Select documents to process page that shows the search field and filter options to locate documents to process.")

   1. Search for the relevant agreements. Enter a keyword or search string in the search field. The dropdown opens and search suggestions are displayed. Select a search suggestion or press the Enter key.
   2. Filter the agreement list table to find the relevant agreements. For example, filter by Sets, Status, Parties, Agreement Type, or AI Suggestions.

      Tip: Use the All Filters option to apply additional filters to the results table.

      Expand the filter, choose the relevant filter value, and select the Apply option.

   The results table updates and the documents that match the search criteria and applied filters are displayed.
5. Choose the documents that you want to process.
6. Select the Start Processing option in the bottom right.

   The Agreement Types view opens. A message confirms that the selected documents are being checked for the new type.

   ![A screenshot of the message that states the selected documents are being checked for the new type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-type-management_Enable-AI_check-for-type-in-selected-docs.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2MDUsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.j0q4HjFlgQY-6eNypwrka_ZPRAvMb7JSFB93HlMarB4&_LANG=enus "A screenshot of the message that states the selected documents are being checked for the new type.")

   Tip: Select the View Progress option to view the processing status for the job. The job monitoring hub opens, and the Processing status and Job details are displayed. Review the processing results, including any error messages. Use the Close (x) icon to return to the Agreement Types view.
7. Return to the  homepage. Select the Close (x) icon located in the top right of the Manage Agreement Types page.

   You are returned to the Completed Documents view.

You have successfully activated the AI detection for a custom agreement type, including surfacing it in selected documents.
