# Display the Status Column in the Agreement List Table

Learn how to customize the agreement list table to display agreement status information.

The Status column and filter are associated with the agreement list table and can be found on the:

- Completed page
- Agreements page in the party hub
- Detailed report view for a selected dashboard or report

The column and filter are displayed in the agreement list table by default. However, if your table contains column customizations, you must manually add the Status column to the table.

Use the Customize columns feature to display the Status column in the agreement list table.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.

   ![A snippet of the Completed Documents page that highlights the Customize columns (toggles) icon on the right-side of the page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Customize-columns-gear-icon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY1NjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.GXQPMViGA-3HV22Smz3GzVdoqBI8PzUH_NPxJKuTXWo&_LANG=enus "A snippet of the Completed Documents page that highlights the Customize columns (toggles) icon on the right-side of the page.")
4. Select the Show or Hide Fields (toggle) icon located to the right of the column headings.

   The Show or Hide Fields dialog box opens. A number of fields are turned on (blue) by default. These fields represent the columns in the agreement list table. Columns display in the order that the fields are represented in the dialog box.

   Important: The Show or Hide Fields function only applies to the data that is displayed in the agreement list table. When downloading or exporting the report, all fields are included in the output file. This includes both custom and standard fields.
5. Perform a query to locate and turn on the Status column:

   ![A screenshot of the Show or Hide Fields dialog box that shows the Status toggle.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-statuses_Customize-columns_show-Status.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY1NjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.GXQPMViGA-3HV22Smz3GzVdoqBI8PzUH_NPxJKuTXWo&_LANG=enus "A screenshot of the Show or Hide Fields dialog box that shows the Status toggle.")

   1. Enter Status in the Find columns field.
   2. Select the Status toggle to turn it on (blue position).
   3. Select Save.

   The dialog box closes. The Status column is displayed in the agreement list table.
6. Re-order the columns so you can view the Status column in the agreement list table without scrolling:

   ![A snippet of the Show or Hide Fields dialog box that highlights the re-ordered Status column.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-statuses_Customize-columns_move-Status_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY1NjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.GXQPMViGA-3HV22Smz3GzVdoqBI8PzUH_NPxJKuTXWo&_LANG=enus "A snippet of the Show or Hide Fields dialog box that highlights the re-ordered Status column.")

   1. Locate the Status column and place your cursor over the dotted grid to the right of the name.

      The "grabbing" cursor displays when you hover over an element that can be dragged or moved.
   2. Drag the Status column to the appropriate location in the list.
   3. Select Save.

   The dialog box closes and the Status column is displayed in the new location within the table.

   ![A snippet of the Completed Documents page that highlights the Status column in the agreement list table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-statuses_Status-column-displayed-table_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY1NjgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.GXQPMViGA-3HV22Smz3GzVdoqBI8PzUH_NPxJKuTXWo&_LANG=enus "A snippet of the Completed Documents page that highlights the Status column in the agreement list table.")

You have successfully displayed the Status column in the agreement list table.
