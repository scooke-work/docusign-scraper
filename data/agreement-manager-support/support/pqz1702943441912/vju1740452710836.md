# Add Custom Fields to Standard Agreement Types

Learn how to create and add custom fields to standard agreement types.

Every standard agreement type is associated with a list of fields that capture relevant information for the type. Custom fields capture specific, unique information that isn't covered by standard agreement fields. These fields can be added to standard and custom agreement types to meet specific use cases.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Completed Documents view opens.
4. Expand the Settings (gear) dropdown from the top right and select the Types and Fields option:

   ![A snippet of the Settings dropdown that highlights the Type and Field option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Types_More-actions-menu_expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5ODcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.t71OQx2QK0OCeH0HaYaRDf3VolusWMbeGTWZMLCKnhY&_LANG=enus "A snippet of the Settings dropdown that highlights the Type and Field option.")

   The Manage Agreement Types page opens. The Agreement Types view is displayed.

   ![A screenshot of the Agreement Types view that shows the agreement types that were grouped under the Business Services tab.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Types_Manage-Types_Agreement-Types.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5ODcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.t71OQx2QK0OCeH0HaYaRDf3VolusWMbeGTWZMLCKnhY&_LANG=enus "A screenshot of the Agreement Types view that shows the agreement types that were grouped under the Business Services tab.")
5. Locate the standard agreement type that you want to update and select the View option.

   Tip: Review the CREATED BY column to differentiate between standard and custom agreement types:
   - Standard types are created by Docusign
   - Custom types are created by User

   Tip: Agreement types are categorized and grouped under three tabs: Business Services, Human Resources, and Miscellaneous. Be sure to display the relevant tab when searching for the agreement type that you want to update.

   The Agreement type details view opens for the selected agreement type. The fields that were automatically added to the standard agreement type are displayed in the table.

   ![A screenshot of a selected standard agreement type that shows the fields that were automatically added to the type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-type-management_add-custom-field-to-standard-agreement.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5ODcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.t71OQx2QK0OCeH0HaYaRDf3VolusWMbeGTWZMLCKnhY&_LANG=enus "A screenshot of a selected standard agreement type that shows the fields that were automatically added to the type.")
6. Select the + Add Fields option from the top right.

   The Add a field dialog box opens.![A screenshot of the Add a field dialog box that is used to add fields to existing agreement types.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-type-management_add-custom-fields_Franchise-Agreement-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5ODcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.t71OQx2QK0OCeH0HaYaRDf3VolusWMbeGTWZMLCKnhY&_LANG=enus "A screenshot of the Add a field dialog box that is used to add fields to existing agreement types.")
7. Complete the required fields and select Save:
   1. Enter a name for the custom field in the Field name field.

      As you enter a name, a type-ahead dropdown list opens, and fields that match the entered name are displayed.

      Tip: Review the type-ahead dropdown list to verify that the custom field you are adding does not already exist.
   2. Select the + Create [Field name] option to create a custom field:

      ![A snippet of the Add a field dialog box that shows an example of a custom field name being created.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-type-management_add-custom-field-to-standard-agreement_example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5ODcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.t71OQx2QK0OCeH0HaYaRDf3VolusWMbeGTWZMLCKnhY&_LANG=enus "A snippet of the Add a field dialog box that shows an example of a custom field name being created.")
   3. Expand the Field type dropdown and select the type of field that you are adding:

      ![A snippet of the Add a field dialog box that shows the available field types for your custom field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Add-custom-field_Field-type-options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5ODcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.t71OQx2QK0OCeH0HaYaRDf3VolusWMbeGTWZMLCKnhY&_LANG=enus "A snippet of the Add a field dialog box that shows the available field types for your custom field.")Choices are: Text,
      Number, Date, and
      Dropdown.

      When using the Dropdown field type, the Options field is required.
   4. (Dropdown only) Enter a list of Options for the dropdown in the text-entry box:

      ![A snippet of the Add a field dialog box that shows the Options field for adding a dropdown field type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Add-custom-field_Options-for-dropdown-type.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5ODcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.t71OQx2QK0OCeH0HaYaRDf3VolusWMbeGTWZMLCKnhY&_LANG=enus "A snippet of the Add a field dialog box that shows the Options field for adding a dropdown field type.")

      Important: Separate each option with a semicolon.
   5. Expand the Category dropdown and select a classification for the field:

      ![A snippet of the Add a field dialog box that shows the Category options for the new field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Add-custom-field_Category-options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5ODcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.t71OQx2QK0OCeH0HaYaRDf3VolusWMbeGTWZMLCKnhY&_LANG=enus "A snippet of the Add a field dialog box that shows the Category options for the new field.")

      Choices are: General, Termination, Renewal, Payment, Legal and Compliance, Human Resources, and Other.
   6. The selected agreement type is displayed in the Agreement types field by default. Apply the custom field to additional agreement types as needed:

      ![A snippet of the Add a field dialog box that shows the expanded Agreement types field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-type-management_Add-a-field_Agreement-types-dropdown.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5ODcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.t71OQx2QK0OCeH0HaYaRDf3VolusWMbeGTWZMLCKnhY&_LANG=enus "A snippet of the Add a field dialog box that shows the expanded Agreement types field.")Choose the relevant agreement types and select the Apply option.
   7. Select Save.

   The dialog box closes. The Agreement type details view opens for the selected agreement type. The custom field is displayed in the field list table.

   ![A snippet of the Franchise Agreement details page that highlights the new custom field in the table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-type-management_Add-custom-field-to-standard-agreement_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc5ODcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.t71OQx2QK0OCeH0HaYaRDf3VolusWMbeGTWZMLCKnhY&_LANG=enus "A snippet of the Franchise Agreement details page that highlights the new custom field in the table.")

   When you first create a custom agreement field, the AI detection is inactive by default. Complete the AI detection setup process so that data for the field is surfaced in your documents.

   See the [Enable AI Detection for Custom Fields](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=wjr1740452628309.html) topic for more information.
8. Select the Back  arrow located in the top left.

   You are returned to the Manage Agreement Types page. The Agreement Types view is displayed.
9. Select the Close (X) icon from the top right.

   You are returned to the Completed Documents page.
