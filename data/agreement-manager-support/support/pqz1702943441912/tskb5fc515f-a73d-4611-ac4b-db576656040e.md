# Delete the AI Setup for Custom Extractions

Learn how to delete the AI detection setup for a custom extraction.

## Before you begin

Before you can delete the AI setup, the custom extraction must be activated for AI detection. See the [Custom Extractions](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=yrx1747083411531.html&_LANG=enus "Read about custom extractions, including how these data points bring value to our organization.") article to learn more about creating custom types.

Use the Delete AI Setup option to delete the detection for custom extractions. For example, if the extraction is no longer relevant.

Remember: Once deleted, you can reset the AI detection at any time. See the [Refine the Detection for Custom Extractions](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tskc13b6287-31ac-4687-8b35-b170680f50e0.html&_LANG=enus "Learn how to refine the AI detection results for your custom extractions, including fine-tuning the definition and selecting new samples.") article for steps to complete this task.

This task is completed by users with Organize and manage all agreements access to Agreement Manager agreements and functionality. For example, administrators and contract managers.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Go to the applicable component manager. Expand the Settings (gear) dropdown from the top right and select the relevant location, Fields or Agreement Types.

   ![A screenshot of the Settings dropdown that highlights the Fields and Agreement Types options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Agreement-Types_More-actions-menu_expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3NTMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.9BWL2uf1bS8MMaYr6dQ7UEemduSgfgJJ7S-0_w5BHHg&_LANG=enus "A screenshot of the Settings dropdown that highlights the Fields and Agreement Types options.")

   Depending on your selection, either the Manage Agreement Types page or the Manage FIelds page opens.

**Find the applicable custom extraction**

5. Locate the extraction that you want to update.

   Tip: The Delete AI Setup function is not available for the standard, Docusign-provided extractions. Apply the Created By filter to view User-created extractions.

   - Search for the extraction that you want to update. Enter a keyword in the search field and select a suggested search from the dropdown or press the Enter key.
   - Apply filters to locate the relevant extraction. Expand the filter, choose a filter value, and select the Apply option.

   The extractions that matched the search term and applied filters are displayed in the table.

**Delete the AI setup**

6. Expand the More actions (three dots) menu for the applicable extraction and select the Delete AI Setup option.

   The Delete Detection dialog box opens. You are asked to confirm the deletion of the AI setup.
7. Select the Delete option.

   The dialog box closes and the custom extraction is no longer activated for AI detection.

   Note: The custom extraction is still visible in the component table. The Set Up option is displayed in the AI detection column for the custom extraction.
8. Return to the Agreement Manager homepage. Select the Close (X) icon located in the top right of the page.

   You are returned to the Completed Documents view.

You have successfully deleted the AI setup for a custom extraction.
