# Locate Docusign Apps for Your Workflow Processes

Learn how to locate apps you can use in your workflow processes by using the Docusign Search feature and Works With and Industry filters.

You can locate extension apps you can use in your workflow processes using the Docusign App Center.

You can also find available  [Workflow Builder templates](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=con8f35c78f-56b7-466c-91e6-c0ea7d03c516.html&_LANG=enus "Learn how to select and initiate a Docusign Workflow Builder template from the Docusign App Center page.").

To use the Search feature:

1. Go to the App Center as described in the [Access Docusign Apps](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=bkt1740757069317.html&_LANG=enus "Learn how to browse for and connect to Docusign extension apps from various Docusign feature locations.")
   article.
2. Enter the name in the Search field.

   ![A screenshot of the Docusign App Center search bar.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/ac-new-search.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTgxODcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.h6jw4KRUGgtT_QIQcVnY6OVIRD_bT9zhGN2pFpkAn9A&_LANG=enus "A screenshot of the Docusign App Center search bar.")

   Any apps corresponding to your entry display.

   ![A screenshot showing the results of the Docusign App Center Search feature.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/app-center-search-features.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTgxODcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.h6jw4KRUGgtT_QIQcVnY6OVIRD_bT9zhGN2pFpkAn9A&_LANG=enus "A screenshot showing the results of the Docusign App Center Search feature.")
3. Select the app that you want.

   The app's Detail Listing page displays.
4. Install the app as described in the [Install Apps in the App Center](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=ctg1698170340729.html&_LANG=enus "Learn how to install Docusign extension apps in the Docusign App Center.") article.

## Use filters to refine your apps search

1. Go to the App Center as described in the [Access Docusign Apps](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=bkt1740757069317.html&_LANG=enus "Learn how to browse for and connect to Docusign extension apps from various Docusign feature locations.") article.
2. Select the Works With or Industry
   filter to display a list of Docusign products as shown in this
   Works With filter example.

   ![A screenshot showing the Docusign App Center Works with filter.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/ac-newest-workswith.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTgxODcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.h6jw4KRUGgtT_QIQcVnY6OVIRD_bT9zhGN2pFpkAn9A&_LANG=enus "A screenshot showing the Docusign App Center Works with filter.")
3. Select one or more Docusign products from the list.
   - Select Apply to display apps that work with
     your product.
   - Select Clear to start over.
4. Select from the resulting apps that work with the selected Docusign products.

   ![A screenshot showing the resulting apps for the Works With Agreement Manager Filter entry.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/ac-works-with-result-pg.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTgxODcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.h6jw4KRUGgtT_QIQcVnY6OVIRD_bT9zhGN2pFpkAn9A&_LANG=enus "A screenshot showing the resulting apps for the Works With Agreement Manager Filter entry.")
5. Select the app that you want.

   The app's Detail Listing page displays.
6. Select one of these options:

   - Install the app as described in the [Install Apps in the App Center](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=ctg1698170340729.html&_LANG=enus "Learn how to install Docusign extension apps in the Docusign App Center.")
     article.
   - If the app is already installed, select Use This
     App.

## Use the Approved for you filter to customize your apps search

1. Go to the App Center as described in the [Access Docusign Apps](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=bkt1740757069317.html&_LANG=enus "Learn how to browse for and connect to Docusign extension apps from various Docusign feature locations.") article.
2. Select the Approved for you filter and select one of more of these apps.

   ![A screenshot of the App Center Approved for you filter options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/approved-for-you-filter.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTgxODcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.h6jw4KRUGgtT_QIQcVnY6OVIRD_bT9zhGN2pFpkAn9A&_LANG=enus "A screenshot of the App Center Approved for you filter options.")

   - Approved for you filter: Displays apps your
     Admin has approved for your organization. See the [Use Approved Apps](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=aak1747679824594.html&_LANG=enus "Learn about Docusign approved extension apps and who can install and use.") topic for
     details about using these apps.
     - Administrator approved: All Docusign users can install and use these
       approved apps in eSignature feature and Webforms.
     - Private apps: Custom apps created
       exclusively by and for your organization.
