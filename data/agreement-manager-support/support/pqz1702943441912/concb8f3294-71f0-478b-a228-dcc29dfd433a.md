# Organization-Level Audit Log Events

Read about the audit log event types that occur at the organization level.

The following list shows all audit log event types that occur at the organization level:

Organization Management

- Create Organization
- Update Organization
- Delete Organization
- Update Organization Features (enabled or disabled)

Account Management

- Link Account (Add Organization Account)
- Unlink Account (Delete Organization Account)
- Add Organization Account Invitation
- Resend Organization Account Invitation
- Delete Organization Account Invitation
- Create Subaccount (Request Initiated)
- Create Subaccount (Creation Completed)

Administrator Management

- Add Docusign Administrator
- Update Docusign Administrator
- Remove Docusign Administrator
- Add Docusign Admin Invitation
- Update Docusign Admin Invitation
- Decline Docusign Admin Invitation
- Cancel Docusign Admin Invitation
- Close Docusign Admin Invitation
- Resend Docusign Admin Invitation

Delegated Administrator Permissions

- Add Delegated Administrator Permission
- Update Delegated Administrator Permission
- Remove Delegated Administrator Permission

User Management

- Add User
- Update User
- Update Address (Work Address)
- Send Activation
- Import - Users
- Import - Update Users
- Import - Close External Users
- Export - User List

Group Management

- Add DS Group
- Update DS Group
- Delete DS Group
- Add DS Group User
- Delete DS Group User

Domain Management

- Add Reserved Domain
- Update Reserved Domain
- Delete Reserved Domain
- Verify Reserved Domain

Identity Provider Management

- Add Identity Provider
- Update Identity Provider
- Delete Identity Provider
- Add Identity Provider Certificate Information
- Delete Identity Provider Certificate Information

Permission Management

- Add Administrator Permission
- Update Administrator Permission
- Remove Administrator Permission
- Export - Permission Profiles

Membership Management

- Add Membership
- Update Membership
- Close Membership

Security&Compliance

- View Audit Logs
- Export Monitor Events
- Download Alert History
- Create SCIM Token
- Revoke SCIM Token
- Create SCIM Account
- Remove SCIM Account

Reporting&Analytics

- Create Report
- Export - Account Settings
- Import - Account Settings
- Update Data Feeds Config
- Update Monitor Config

Appliance Management

- Add Appliance Pool
- Update Appliance Pool
- Delete Appliance Pool
- Add Appliance
- Update Appliance
- Delete Appliance
- Link Account to Appliance Pool
- Unlink Account from Appliance Pool

Connect Configuration

- Add Organization Connect Configuration
- Update Organization Connect Configuration
- Delete Organization Connect Configuration
- Add Organization Connect Key
- Delete Organization Connect Key
- Add Organization Connect OAuth
- Update Organization Connect OAuth
- Delete Organization Connect OAuth

- Organization Envelope Publish
- Delete Organization Envelope Publish
