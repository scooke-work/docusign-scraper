# Update the Agreement Type for an Agreement

Learn how to update the agreement type value for an agreement.

When documents are synced or imported to Agreement Manager, the agreement type is identified and the associated fields are surfaced. Although rare, you can update the agreement type. For example, changing a "miscellaneous" agreement to a custom agreement type.

Note: When updating the agreement type, the agreement is processed again to surface content that is relevant to the selected type.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. A list of documents to which the logged-in user has access is displayed in the table.

   Tip: The total document count is displayed the bottom-right corner of the view. This count is updated when you perform a search or apply filters.
4. Locate the agreement that you want to update.

   ![A snippet of the Completed Documents view that highlights the search field and available filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Navigator_Completed-documents_search-field-and-filters-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2MzgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.z1Ara0eKCa6c4pSxYHbxzYqYoe8XtP4RR-Z8FFtY6wA&_LANG=enus "A snippet of the Completed Documents view that highlights the search field and available filters.")

   1. Enter a keyword or search string in the search field and select Enter.
   2. Use the Filters option to refine the list of search results.

      Select the Filters option and locate the filter that you want to apply. Expand a filter, select or enter a value, and select Apply.

      The agreements that match your search criteria and applied filters are displayed in the table.
5. Select (click) the agreement that you want to update.

   The Agreement preview page opens for the selected agreement. The Details pane is displayed by default and shows a list of attributes and suggested values. These attributes and values are displayed based on the identified agreement type.

   A summary displays at the top of the Details pane. The summary includes the identified agreement type and the number of fields and suggestions to review:

   ![A snippet of the Details pane that shows the identified agreement type and the number of attribute fields and suggestions that require review.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Preview-page_Details-pane_summary-statement-snippet.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2MzgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.z1Ara0eKCa6c4pSxYHbxzYqYoe8XtP4RR-Z8FFtY6wA&_LANG=enus "A snippet of the Details pane that shows the identified agreement type and the number of attribute fields and suggestions that require review.")

   Tip: Use the Find details search field to locate specific attributes or values.
6. Place the Details pane into *edit* mode.
   - Select the Edit (pencil) icon.
   - Select the Review All option.

   Note: You must have Organize and manage all agreements access to perform data reviews. For all other users, the Edit (pencil) icon and the Review All option are grayed out.

   The Review & edit details pane opens for updates.

   Important: When you accept or edit the suggested attribute values, the content in the agreement does not change.
7. Update the Agreement type for the document.

   ![A snippet of the Review & edit details pane that highlights a selected value in the Agreement type dropdown.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Preview-page_Details-pane_Update-Agreement-type.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2MzgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.z1Ara0eKCa6c4pSxYHbxzYqYoe8XtP4RR-Z8FFtY6wA&_LANG=enus "A snippet of the Review & edit details pane that highlights a selected value in the Agreement type dropdown.")

   1. Expand the Agreement type dropdown.

      The suggested value is highlighted in the preview pane and a dropdown list of available types displays.
   2. Search for and select the relevant agreement type from the list.

   The Change agreement type dialog box opens.

   ![A screenshot of the Change agreement type dialog box that states the agreement will be processed to surface content specific to the type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Preview-page_Details-pane_Change-agreement-type-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2MzgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.z1Ara0eKCa6c4pSxYHbxzYqYoe8XtP4RR-Z8FFtY6wA&_LANG=enus "A screenshot of the Change agreement type dialog box that states the agreement will be processed to surface content specific to the type.")
8. Review the message and select Change.

   The agreement is processed to surface content that is specific to the selected type. Review
   these topics to learn more about agreement types and AI processing:
   - [IAM Agreement Type and Field Management](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=uie1724446326038.html&_LANG=enus "Read about how agreement type management is being used to unify the way Docusign capitalizes on agreement types and associated fields. This topic answers the question: What is agreement type management and how does it benefit me?")
   - [Processing Agreement Manager Agreements](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=rvw1736364838500.html&_LANG=enus "Read about agreement processing and how to opt in for additional processing to surface new insights within your agreements.")

   The dialog box closes. The agreement type was successfully updated and the document is being processed. The Details pane is reverted to read-only mode.
9. Select the Close (X) icon from the top left.

   You are returned to the Completed Documents view. Your search criteria and filters are still applied.

You have successfully updated the agreement type value for a selected document.
