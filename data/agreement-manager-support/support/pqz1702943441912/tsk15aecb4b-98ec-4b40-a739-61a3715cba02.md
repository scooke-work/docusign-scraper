# View Envelope Details from the Agreement List Table

Learn how to view the envelope details for an agreement that was imported from a completed eSignature envelope.

If an agreement was imported from a completed eSignature envelope, the envelope ID is displayed in the agreements list in the Envelope ID column. Select the hyperlinked envelope ID to view the corresponding envelope details.

Tip: See these topics to learn more about customizing the agreement list table, including displaying the Envelope ID column:

- [Display the Envelope ID in the Agreement List Table](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=sqs1748881628076.html&_LANG=enus "Learn how to configure the agreement list table to display the Envelope ID column.")
- [Customize the Agreement List Table to Show Relevant Data](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=qrh1702945502942.html&_LANG=enus "This topic answers the question: How can I see relevant agreement data in the Agreement Manager table? Learn how to configure the columns in the agreement list table to display relevant data.")

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Perform a search and apply filters to locate the relevant agreements:
   1. Enter a keyword or string in the search field and select a suggestion from the dropdown or press the Enter key.

      The agreements containing the keyword or search string that you entered are displayed in the table.
   2. Use the filters to refine the list of search results. For example, use the Parties filter to view a list of agreements for a specific party.

      1. Expand the Filters option and select the filter that you want to apply.
      2. Choose a filter value.
      3. Select the Apply option.

      The agreements that match the applied filters are displayed in the table.
5. Verify that the Envelope ID column is displayed in the agreement list table.

   If necessary, follow the steps in the [Display the Envelope ID in the Agreement List Table](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=sqs1748881628076.html&_LANG=enus "Learn how to configure the agreement list table to display the Envelope ID column.") topic to display the column.
6. Locate the relevant agreement and select the associated Envelope ID link:

   ![A snippet of the Completed Documents view that highlights the Envelope ID for an agreement.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-view_Customize-columns_Envelope-ID-in-table.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0ODMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.HmhaxUDWsBBfl575NV4O2LQCD_7lq39RFTT4E_Wk_D8&_LANG=enus "A snippet of the Completed Documents view that highlights the Envelope ID for an agreement.")

   Tip: The Envelope ID column is only populated if the agreement was imported from a completed eSignature envelope.

   The envelope details page opens and information about the selected envelope ID is displayed.

   ![A screenshot of the envelope details page for a selected envelope ID.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Envelope-ID_envelope-details-page.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY0ODMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.HmhaxUDWsBBfl575NV4O2LQCD_7lq39RFTT4E_Wk_D8&_LANG=enus "A screenshot of the envelope details page for a selected envelope ID.")
7. Review the envelope details as needed, including the:

   - Envelope status
   - Recipient list and status
   - Message

   Note: See these topics to learn more about viewing and managing envelope details:
   - [Managing Envelopes](https://support.docusign.com/s/document-item?bundleId=oeq1643226594604&topicId=ghu1578456429097.html)
   - [View Envelope Details](https://support.docusign.com/s/document-item?bundleId=oeq1643226594604&topicId=otr1578456349846.html)
8. Select the Agreements tab in the global navigation at the top of the page.

   You are returned to the Agreement Manager homepage. The Completed Documents view is displayed.

You have successfully located the envelope details for an agreement that was imported from a completed eSignature envelope.
