# Install Cloud Storage Apps for Account Administrators

Learn how to install various cloud storage providers from the Docusign App Center. This topic answers the question: How do I install cloud storage providers so users can upload documents to Agreement Manager?

When performing single or bulk uploads, users can choose to upload documents located in various cloud storage locations. Cloud storage apps are installed from the Docusign App Center. This task is completed by an account administrator.

Important: If the apps were previously installed from the Docusign App Center, you must uninstall and reinstall them. Due to versioning issues, backward compatibility is not currently possible.

Note: See the [Docusign Organization Administrators](https://support.docusign.com/s/document-item?bundleId=rrf1583359212854&topicId=god1583359152535.html) pages to learn more about the required permissions.

You can install cloud provider apps from three locations:

- Docusign App Center main page:
  - Select the app tile.
  - Select Install.
- Cloud storage provider's detail page: Select Install from any detail page.
- Select File dropdown:
  - Select the Upload arrow.
  - Expand the Select File dropdown.
  - Select the install option for the relevant app

1. Go to the  [Docusign App Center](https://apps.docusign.com/app-center).

   ![A screenshot of the Docusign App Center main page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Docusign-App-Center_Main-page.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3NTgsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.t23RlAsqTbyEg0DdvVIUetJ_k_WiXcQsxjDy7gt3M1U&_LANG=enus "A screenshot of the Docusign App Center main page.")
2. Select Log In located in the top-right corner of the App Center.
3. Log in to Docusign using account administrator credentials.

   The Docusign App Center main page opens.
4. Locate and select the app for the cloud storage provider that you want to install.

   The app's detail page displays.
5. Review the app's overview, requirements, and resources.
6. Select Install.

   If the app has authentication or account connection requirements, you are prompted to install and authorize the connection.
7. If prompted, select Install and Authorize.

   You are connected to the selected cloud storage app.
8. Review the information on the app's detail page and select Continue.
9. If the app requires authorization, select Connect Account.

   The app is connected to your Docusign account. Next, authorize the app for use in Docusign.
10. Enter your credentials for the app and select Continue.

    The cloud storage provider is now ready to use for document uploads.
