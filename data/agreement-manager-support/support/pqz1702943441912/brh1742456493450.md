# View Obligations for Selected Parties and Groups

Learn how to view all the obligations that are tied to a specific party or alias group.

The "party hub" displays a unified summary of all agreements and obligations that are tied to a party or alias group. Access the party hub from various places within Agreement Manager, including:

- The Parties column (Completed Documents view)
- The Details pane (Agreement preview page)
- The Name column (Parties view)

Note: This topic reviews the steps to access the party hub from
the Completed Documents view. You can also search for and
select the relevant party from the Parties page.

![A snippet of the Parties page that highlights the Search parties field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Parties-page_Search-parties-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.F6f9YkxGNJ6CgpocP2BogKCoHL4MmJyY1GN0VGUBBQI&_LANG=enus "A snippet of the Parties page that highlights the Search parties field.")

- Go to Home  >  Agreements and select All from the PARTIES section in the navigation pane.
- Enter search criteria in the Search parties
  field.
- Select a party from the search results.

The party hub opens for the selected party or group.

**Access the party hub from the Completed Documents view:**

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens. The Inbox view is displayed by default.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. Filter this view for a specific party.
4. Filter the agreement list table for a specific party:

   ![A screenshot of the Parties filter that shows the Search field, Party Role dropdown, and the party list.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Parties-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.F6f9YkxGNJ6CgpocP2BogKCoHL4MmJyY1GN0VGUBBQI&_LANG=enus "A screenshot of the Parties filter that shows the Search field, Party Role dropdown, and the party list.")

   1. Expand the Filters option and select the Parties filter.
   2. Enter a keyword or search string in the Search field.

      As you enter search criteria, the list of parties updates to match the keyword.
   3. Select the relevant party.

      Important: When using the Parties filter, you must also select a
      Party Role. If you do not know
      which role to select, accept the default value, All.
   4. Apply the Party Role filter to view agreements where the selected party plays a specific role. If you do not know which role to select, accept the default value, All.

      Select a role from the Party Role dropdown. Choose from: Buyer, Seller, or Other.
   5. Select the Apply option.

   The table updates and agreements that are tied to the selected party or group are displayed. The names in the Parties column are clickable and link directly to the party hub.

   ![A snippet of the Completed Documents view that highlights the Parties column as an entry point for the party hub.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Entry-point_Completed-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.F6f9YkxGNJ6CgpocP2BogKCoHL4MmJyY1GN0VGUBBQI&_LANG=enus "A snippet of the Completed Documents view that highlights the Parties column as an entry point for the party hub.")
5. Select a name from the Parties column.

   The party hub opens for the selected party or group. The Overview tab is displayed by default.

**View the Obligations list:**

6. Select the Obligations tab:

   ![A snippet of the party hub that highlights the Overview, Agreements, Obligations, and Details tabs.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Overview-dashboard_highlighted-tabs.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.F6f9YkxGNJ6CgpocP2BogKCoHL4MmJyY1GN0VGUBBQI&_LANG=enus "A snippet of the party hub that highlights the Overview, Agreements, Obligations, and Details tabs.")

   The Obligations tab opens. All the obligations tied to the selected party or group are displayed in the table.

   Restriction:

   Obligations are surfaced in English (U.S.) agreements only. The AI model that processes obligations is run from a U.S. server and can only process English agreements in this region.

   If your language and region are not supported, manually create the obligations to include this information in dashboards, reports, and other insights.

   See these support articles to learn more about obligation management:
   - [Obligation Management](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=csm1727128259486.html&_LANG=enus "Read about obligation management, including how to use this functionality to capture, track, and fulfill your agreement obligations. This topic answers the question: How can I find and track my agreement obligations?")
   - [Manually Create an Obligation](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=rrf1727128588540.html&_LANG=enus "Learn how to create an obligation from the Agreement preview page for a selected agreement.")

   ![A screenshot of the party hub Obligations tab that shows a list of obligations that are tied to the selected party or group.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Obligations-tab.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.F6f9YkxGNJ6CgpocP2BogKCoHL4MmJyY1GN0VGUBBQI&_LANG=enus "A screenshot of the party hub Obligations tab that shows a list of obligations that are tied to the selected party or group.")

   Tip: Use the Favorites (star) icon to "star" your most active, time-sensitive, or high-value parties and groups. Each user can add up to 50 parties and groups to their Starred list.

   Your "favorite" parties and groups are listed under the Starred tab on the Parties page.

**Filter and customize the Obligations list:**

7. Apply additional filters to locate relevant obligations.
8. Customize the table data as needed:

   ![A screenshot of the Show or Hide Fields dialog box that shows the columns you can choose to display in the agreements table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Overview-dashboard_Customize-columns.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.F6f9YkxGNJ6CgpocP2BogKCoHL4MmJyY1GN0VGUBBQI&_LANG=enus "A screenshot of the Show or Hide Fields dialog box that shows the columns you can choose to display in the agreements table.")

   1. Select the Show or Hide Fields (toggle) icon located on the right side of the page.

      The Show or Hide Fields dialog box opens. These fields represent the columns in the table.
   2. Select the toggles to turn the associated field (column) on (blue) or off (white).
   3. Re-order the columns as needed. Hover the cursor over the Reorder (dots) icon and drag the column (field) to the appropriate location.
   4. Select the Save option.

   The dialog box closes and the columns are updated according to your configuration.

**View obligation details from the Agreement preview page:**

9. Select an obligation from the table.

   The Agreement preview page opens and the associated obligation is displayed in the Obligations pane.

   ![A snippet of the Agreement preview page that highlights the details of the selected obligation.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Party-hub_Obligations_Agreement-preview.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc4MzAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.F6f9YkxGNJ6CgpocP2BogKCoHL4MmJyY1GN0VGUBBQI&_LANG=enus "A snippet of the Agreement preview page that highlights the details of the selected obligation.")
10. Review the obligation details and place the pane in *Edit* mode to make updates as needed. Be sure to save your changes before leaving the page.

**Return to the party hub:**

11. Select the Back arrow from the top-left corner.

    You are returned to the party hub.
12. Select Parties from the breadcrumb navigation at the top of the page.

    You are returned to the Parties view.

    Tip: Select Completed from the navigation pane on the left to view your agreement list. The Completed Documents view opens. The agreements to which the user has access are displayed in the table.
