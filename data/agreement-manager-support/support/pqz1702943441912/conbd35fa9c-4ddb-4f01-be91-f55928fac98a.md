# IAM Unified Search

Read about the unified search field, a global search feature that provides navigational suggestions across the Intelligent Agreement Management (IAM) platform.

The unified Search field is a global search feature that provides navigational suggestions across the IAM platform. Users can search for information across all the IAM features and products, including jumping to key areas.

 

## Feature availability

Currently, the unified Search feature is available to only a limited number of select accounts. Contact your Docusign account representative to request access to this feature.

## Customer use cases

Here are a few sample use cases for using the unified search feature:

## Locate eSignature envelopes

**Scenario:** As an Executive, I must sign a change request but can't locate the envelope pending my signature:

Locate the envelopes that require your signature. Enter any of these data points into the unified Search field:

- Envelope ID
- Recipient or sender email address
- Party name

As you enter the data point into the field, a list of suggestions displays.

In this example, you want to apply the search criteria to your envelopes. Select the Envelopes option from the Search In section:

![A screenshot of the unified Search field that shows search criteria to locate envelopes requiring action.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Unified-search_locate-envelope_by-party_example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYwODYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.kjVc_AA6feXCEVp4XSHoDrGYpLdt_KXxIAsv89O1HfE&_LANG=enus)

The Inbox opens and the envelopes that matched your search criteria are displayed.

The IAM party hub pages provide an overview of your business relationship with a selected party, including agreements and obligations.

The vendor's Agreements tab opens in the party hub. A list of your agreements with the vendor is displayed:

![A screenshot of the party hub Agreements tab that shows a list of agreements associated with the selected party or group.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Unified-search_locate-party_party-hub-agreements_example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYwODYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.kjVc_AA6feXCEVp4XSHoDrGYpLdt_KXxIAsv89O1HfE&_LANG=enus)

Select the other tabs to gain additional insights:

- Overview tab: Displays a visual summary of all active agreements for specific parties or groups, including a Party summary and Recent activity.
- Hierarchies tab: Displays a list of agreement hierarchies for the selected party or group. Hierarchies show how agreements are related to each other.
- Obligations tab: Displays a list of all the obligations that are tied to the selected party or group.

Note:

The Details tab shows the display name, party type, and party side for the selected party or group. The Organize and manage all agreements permission is required to update these details.

## Advanced search queries

The unified Search field supports full-text searches with a single query string:

![A screenshot of the unified Search field that shows an example of a full-text search using a single query string.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Unified-search_advanced-search_query-string-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYwODYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.kjVc_AA6feXCEVp4XSHoDrGYpLdt_KXxIAsv89O1HfE&_LANG=enus)

These searches support advanced search features, such as:

- Boolean operators (use AND, OR, and NOT operators to combine search terms)
- Boosting (use a caret (^) to prioritize search terms)
- Exact phrase matching (use quotation marks ("") to find an exact phrase)
- Proximity search (use the tilde (~) followed by a number to find agreements based on word proximity to each other)
- Wildcard search (use the asterisk (\*) to find agreements that contain variations of the entered search term)

See the [Advanced Search Queries](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=conce9d9428-f27e-49ad-9001-3c7a0820ae78.html&_LANG=enus "Read about the advanced search query feature that gives you control over how your results are matched and ranked.") article to learn more about using advanced search features with single query strings.
