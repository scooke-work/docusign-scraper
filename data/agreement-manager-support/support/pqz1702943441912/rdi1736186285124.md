# Add Individual Agreements to Existing Agreement Sets

Learn how to add agreements to an existing set from various places within Agreement Manager. This topic answers the question: How do I add additional agreements to an existing set?

Agreements can be added to existing sets from a number of places within Agreement Manager, such as:

- Agreement list table on the Completed Documents page
- Agreement sets pane on the Agreement preview page

This topic reviews both ways to add agreements to existing sets.

**Add agreements to sets from the Completed Documents page:**

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents page is displayed. Perform a search and apply filters to locate agreements.
4. Enter a keyword or search string in the search field and select Enter.

   Agreements containing the keyword or search string that you entered are displayed in the table.
5. Use filters to refine the list of search results:

   ![A snippet of the Completed Documents page that highlights the Filters option and suggested filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Navigator_Completed-documents_search-field-and-filters-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNDYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.3G-f5zeEKnsljFG61xwbaLbeO_kvtoDWRMFRLPdfiFk&_LANG=enus "A snippet of the Completed Documents page that highlights the Filters option and suggested filters.")

   1. Select the Filters option and locate the filter that you want to apply.
   2. Select the relevant filter.
   3. Choose the filter value or enter a date range if applicable.
   4. Select the Apply option.
   5. Repeat this step until the relevant agreements are displayed in the agreement list table.

   The table displays agreements matching the filters that you applied.
6. Select the agreements that you want to add to the set. You can add one or more agreements at a time.

   When one or more agreements are selected, additional actions are displayed in the bar at the top of the table.
7. Add the selected agreements to a set. Expand the Organize dropdown, choose the relevant set, and select the Add option:

   ![A screenshot of the expanded Organize dropdown that shows a list of sets to which the agreements can be added.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents_Organize-expanded_Add-to-sets_Select-set.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNDYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.3G-f5zeEKnsljFG61xwbaLbeO_kvtoDWRMFRLPdfiFk&_LANG=enus "A screenshot of the expanded Organize dropdown that shows a list of sets to which the agreements can be added.")

   The dropdown closes and the selected agreements are added to the set.

   You are returned to the Completed Documents page. The agreements are no longer selected, however, your filters are still applied.

**Add agreements to sets from the Agreement preview page:**

Individual agreements can be added to sets from the Agreement preview page.

8. Select an agreement from the agreement list table on the Completed Documents page.

   The Agreement preview page opens for the selected agreement. The Details pane displays the attributes (fields) and values that were surfaced in the agreement.

   ![A snippet of the Agreement preview page that highlights the Agreement sets icon in the tray on the left-side of the page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Preview-page_Agreement-sets-icon-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNDYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.3G-f5zeEKnsljFG61xwbaLbeO_kvtoDWRMFRLPdfiFk&_LANG=enus "A snippet of the Agreement preview page that highlights the Agreement sets icon in the tray on the left-side of the page.")
9. Select the Agreement sets icon from the tray of icons on the left-side of the page.

   The Agreement sets pane opens and a list of sets to which the agreement has been added is displayed.

   ![A snippet of the Agreement preview page that highlights the Agreement sets pane, including available actions.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Preview-page_Agreement-sets-pane.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTczNDYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.3G-f5zeEKnsljFG61xwbaLbeO_kvtoDWRMFRLPdfiFk&_LANG=enus "A snippet of the Agreement preview page that highlights the Agreement sets pane, including available actions.")

   From this pane, you can:

   - Add the agreement to another set
   - Remove the agreement from a set
   - Create a new set
10. Search for and select the set to which you want to add the agreement:
    1. Enter a keyword or search string in the Search sets field.

       When you click in the field, a list of sets displays. These sets were either shared with you or created by you. The list updates as you enter search criteria.
    2. Select (click) a set from the list of search results.

       The agreement is added to the selected set.

    Note: If the agreement was already added to a set, select Add to Another Set to add it to another set. The Search sets field displays. Perform a query to locate the set to which you want to add the agreement.

    Repeat this step to add the agreement to additional sets.
11. Select the Close (x) icon from the top-left corner.

    You are returned to the Completed Documents page. Your search criteria and filters are still applied.

You have successfully added individual agreements to existing agreement sets.
