# Search for and Merge Parties

This topic answers the question: How do I merge parties under one name? Learn how to search for and merge duplicate parties.

Party management tasks are completed by users with Organize and manage all agreements access to Agreement Manager, such as contract managers. Common tasks include:

- Review and merge duplicate party names.
- Set a preferred name for a merged group of parties.
- Remove parties from merged groups.

These tasks are completed from the Manage parties page. This topic reviews the steps to search for specific parties that you want to merge.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Parties from the navigation pane on the left.

   The Parties page opens.
4. Select the Manage Parties option from the top right:

   ![A snippet of the Parties page that highlights the Manage Parties option in the top right.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-parties_entry-point_Parties-view.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3ODIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vLKDkrYKIo-_9-q3Yz1H88aJre8iCaJ7sM3VcDHaCjA&_LANG=enus "A snippet of the Parties page that highlights the Manage Parties option in the top right.")

   The Manage parties page opens and a list of individual and merged parties is displayed.

**Select parties to merge**

Parties can be merged manually from the Manage parties page.

![A screenshot of the Manage parties view that highlights the search field, selected parties, and the Merge option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Parties_Manage-parties_Selected-parties--and-Merge-highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3ODIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vLKDkrYKIo-_9-q3Yz1H88aJre8iCaJ7sM3VcDHaCjA&_LANG=enus "A screenshot of the Manage parties view that highlights the search field, selected parties, and the Merge option.")

5. Locate and select the parties that you want to merge:
   1. Enter a party name in the Search parties field.

      As you enter the name, a dropdown opens and the parties that match the search term are listed.
   2. Select a party name from the dropdown list or select the View all results option at the bottom of the list.

      The parties list updates and the parties that matched the search term are displayed.
   3. Select the checkboxes for the parties that you want to merge.

   Tip: The Party Side column indicates whether the party is *internal* or *external* to the user's organization. During processing, parties are identified as either the first-party or the third-party:
   - The first-party designation, *Internal*, indicates the user's organization.
   - The third-party designation, *External*, indicates external entities.Review this column to help you determine if parties should be merged under a party alias group.

   When two or more parties are selected, the Merge option becomes active.

**Merge selected parties**

6. Select the Merge option.

   The Review Duplicate Parties page opens. The Select parties to merge and Choose a name sections are displayed.

   ![A screenshot of the Review Duplicate Parties page that shows the suggested duplicate parties and naming options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Manage-Parties_Review-Duplicate-Parties.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc3ODIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.vLKDkrYKIo-_9-q3Yz1H88aJre8iCaJ7sM3VcDHaCjA&_LANG=enus "A screenshot of the Review Duplicate Parties page that shows the suggested duplicate parties and naming options.")
7. Choose the parties that you want to merge from the Select parties to merge section.
   - Deselect the checkboxes for the parties that you do not want to merge. This step is applicable when there are three or more possible duplicate parties.

     Tip: When there are only two suggested duplicate parties, the checkboxes are grayed out.
   - Select the Cancel option from the bottom right if you do not want to merge the selected parties.
8. Select a preferred party name from the Choose a name section. Expand the Party name dropdown and select the preferred name from the list.

   Tip: The merged parties are consolidated under one name. Thi name is used throughout the Intelligent Agreement Management (IAM) platform. For example, this name is displayed in the Parties filter and column on the Completed Documents page in Agreement Manager.
9. Select the Merge Parties option from the bottom right. The button label reflects the number of parties being merged.

   Important:

   This action can take up to an hour before the changes are reflected in the various locations across the IAM platform.

   The Review Duplicate Parties page closes and you are returned to the Manage parties view.
10. Clear the existing search criteria. Select the Clear (x) icon located in the search field.

    The search field is cleared and the party list table is reset.
11. Repeat these steps to merge additional parties as needed.
12. Select the Parties option from the breadcrumb navigation located in the top left.

    You are returned to the Parties page.

    Tip: Select Completed from the navigation pane on the left to view your agreement list. The Completed Documents view opens. The agreements to which the user has access are displayed in the table.
