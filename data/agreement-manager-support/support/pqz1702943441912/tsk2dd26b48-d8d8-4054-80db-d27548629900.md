# Delete Custom Dashboards

Learn how to delete a custom dashboard.

Custom dashboards give you access to the data that's tailored for your priorities. If you are no longer using a custom dashboard, it can be deleted.

1. Log in to your **Docusign** account.

   The Docusign Home page opens.
2. Select Insights from the global header.

   The Insights hub opens and the Overview is displayed.

   Your custom dashboards are displayed in the navigation pane on the left, under the Custom Dashboards grouping.
3. Select the dashboard that you want to delete from the Custom Dashboards grouping in the navigation pane.

   The custom dashboard view opens for the selected dashboard.
4. Expand the More actions (three dots) menu from the top-right corner of the page and select Delete Dashboard:

   ![A snippet of a custom dashboard page that highlights the Delete Dashboard option in the Dashboard Actions menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Delete-custom-dashboard_Delete-Dashboard_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcwOTcsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.ahio8Vrq-Kye0S7EKhkkBpPJvSvXgSyO8avzvsLZW0k&_LANG=enus "A snippet of a custom dashboard page that highlights the Delete Dashboard option in the Dashboard Actions menu.")The Delete dashboard dialog box opens.
5. Review the message and select the Delete option.

   CAUTION: When you delete a dashboard, you are also deleting the user-created reports that were added from the dashboard.

   The dialog box closes. The custom dashboard is no longer displayed in the navigation pane under the Custom Dashboards grouping.

You have successfully deleted a custom dashboard. The dashboard no longer displays in the navigation pane on the left-side of the Insights page.
