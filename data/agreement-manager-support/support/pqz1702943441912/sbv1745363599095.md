# Hide Obligation Types

Learn how to hide and unhide your custom and standard obligation types.

Custom and standard obligation types can be hidden if they are no longer relevant. When an obligation type is hidden, it no longer displays as a filter option in various dropdowns.

Note: If an agreement contains a hidden obligation type, those obligations are still visible. Existing obligation instances with the hidden type are also visible.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed.
4. Expand the Settings (gear) menu from the right-side of the page and select the Obligations option:

   ![A screenshot of the Settings (gear) menu that highlights the Obligation option in the expanded](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Obligation-type-management_Manage-Obligation-Types_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyMTIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9._EwXMQCXvhO3bGzdRR46wE7KVr__lIge1lCRXF2k8N0&_LANG=enus "A screenshot of the Settings (gear) menu that highlights the Obligation option in the expanded")

   The Manage Obligation Types page opens. The Obligation Types view is displayed. A list of custom and standard obligation types is displayed in the table.
5. Locate the custom obligation type that you want to hide.
6. Expand the More actions (three dots) menu and select the Hide option:

   ![A snippet of the Obligation Types table that shows the Hide option in the More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Obligation-type-management_Manage-Obligation-Types_Hide.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyMTIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9._EwXMQCXvhO3bGzdRR46wE7KVr__lIge1lCRXF2k8N0&_LANG=enus "A snippet of the Obligation Types table that shows the Hide option in the More actions (three dots) menu.")

   The table is updated and indicates that the obligation type is *hidden*.

**Unhide an obligation type:**

7. To show a hidden obligation type:

   ![A snippet of the Obligation Types table that shows the Unhide option in the More actions (three dots) menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Obligation-type-management_Manage-Obligation-Types_Unhide.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTcyMTIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9._EwXMQCXvhO3bGzdRR46wE7KVr__lIge1lCRXF2k8N0&_LANG=enus "A snippet of the Obligation Types table that shows the Unhide option in the More actions (three dots) menu.")

   1. Locate the obligation type that you want to show.
   2. Expand the More actions (three dots) menu and select the Unhide option.

   The table is updated and the obligation type is no longer hidden.

**Return to homepage**

8. Repeat these steps to hide or display obligation types as needed.
9. Return to the Agreement Manager homepage. Select the Close (x) icon in the top right.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. The search criteria and filters are still applied.
