# Perform a Search and Apply Filters to Locate Agreements

Learn how to combine search methods and filters to locate relevant agreements. This topic answers the question: How do I perform a keyword search and apply filters to find my completed agreements?

When you perform a keyword search, the search reviews the contents of the agreements to look for the keyword. When you use filters, the agreements are filtered based on surfaced attributes and field values.

Often, it is more effective to perform a keyword search and then apply filters to locate specific agreements.

Tip: If you routinely search for the same type of agreements or agreement data, save your search criteria for future use. See the [Save Search Criteria for Future Use](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tskc38121a1-bbf8-4da8-8ed7-b364991f7e50.html&_LANG=enus "Learn how to save the search criteria for the queries that you frequently perform.") topic to learn more about the saved search feature.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. The documents to which the logged-in user has access are listed in the table. The total document count is displayed the bottom right. This count is updated when you perform a search or apply filters.

   ![A screenshot of the Completed Documents page that shows a list of documents to which the logged-in user has access.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-view_updated-with-document-count.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzNzEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WsbaaKodCvui07hBbd1XdbCoQNII4qZmpfrhTRJFWu0&_LANG=enus "A screenshot of the Completed Documents page that shows a list of documents to which the logged-in user has access.")
4. Use the search field to locate the relevant agreements:

   ![A snippet of the type-ahead dropdown that shows the search history, suggested questions, and recent documents.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Search_typeahead-dropdown.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzNzEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WsbaaKodCvui07hBbd1XdbCoQNII4qZmpfrhTRJFWu0&_LANG=enus "A snippet of the type-ahead dropdown that shows the search history, suggested questions, and recent documents.")

   1. Click in the search field. A dropdown opens and your search history and recent documents are displayed.

      Tip: Access your Saved Searches from the dropdown to run an existing query. See the [Save Search Criteria for Future Use](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tskc38121a1-bbf8-4da8-8ed7-b364991f7e50.html&_LANG=enus "Learn how to save the search criteria for the queries that you frequently perform.") topic to learn more about this feature.
   2. Enter a keyword or phrase in the search field.

      As you enter a keyword in the search field, suggestions are displayed in the dropdown.
   3. Select a suggestion from the dropdown or press the [Enter] key.

   Important: Keyword searches are different from applying filters. The search reviews the contents of agreements, while the filters display agreements based on surfaced attributes and field values.

   The agreements that contain the keyword or phrase are displayed in the table. In this example, agreements containing *force majeure* are displayed in the table:

   ![A snippet of the agreement list table that shows the search results for agreements that have "force majeure".](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Perform-keyword-search_force-majeure-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzNzEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WsbaaKodCvui07hBbd1XdbCoQNII4qZmpfrhTRJFWu0&_LANG=enus "A snippet of the agreement list table that shows the search results for agreements that have \"force majeure\".")

   Tip: The keyword search automatically includes synonyms and related concepts in the
   search results. Select the related terms link to view
   additional information: ![A snippet of the agreement list table that shows query results and the Show related terms dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Search_Keyword-search_Related-terms-dialog.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzNzEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WsbaaKodCvui07hBbd1XdbCoQNII4qZmpfrhTRJFWu0&_LANG=enus "A snippet of the agreement list table that shows query results and the Show related terms dialog box.") The Related terms toggle is turned on (blue
   position) by default. Select the toggle to turn this feature off (white
   position).
5. Use the filters to refine the list of search results:

   ![A snippet of the agreement list table that shows highlights the expanded Filters menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Keyword-search-filters_Filter-dropdown_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzNzEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WsbaaKodCvui07hBbd1XdbCoQNII4qZmpfrhTRJFWu0&_LANG=enus "A snippet of the agreement list table that shows highlights the expanded Filters menu.")

   1. Select the Filters option.
   2. Locate the filter that you want to apply:

      - Search for the filter. Enter a keyword in the Filter by field and select a value from the results list.
      - Select a filter from the list of default filters, including: Sets, Status, Parties, Agreement Type, and AI Suggestions.
      - Use the All Filters option to locate the relevant filter from a list of all available filters.

      For example, use the Agreement Type filter to locate *force majeure* terms within a specific agreement type:

      ![A screenshot of the Agreement Type filter that shows a selected value.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Filter-search-results-by-Type-filter_MSA-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzNzEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WsbaaKodCvui07hBbd1XdbCoQNII4qZmpfrhTRJFWu0&_LANG=enus "A screenshot of the Agreement Type filter that shows a selected value.")
   3. Choose the relevant filter values or enter a date range.
   4. Select the Apply option.

   Tip: A number displays next to filter values to indicate the number of agreements that match the corresponding value.

   The filter closes and master service agreements (MSAs) containing *force majeure* terms are displayed in the table.

   ![A snippet of the agreement list table that shows MSAs that contain force majeure terms.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_Search-with-keyword-and-filter.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzNzEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WsbaaKodCvui07hBbd1XdbCoQNII4qZmpfrhTRJFWu0&_LANG=enus "A snippet of the agreement list table that shows MSAs that contain force majeure terms.")
6. Continue to apply filters to refine the agreement list table as needed.

   Note:

   Use the Status filter to filter the agreement list table by agreement status. For example, choose to view only Active agreements that match the current search criteria and filters: ![A screenshot of the Status filter that shows the selected Active options, including Expiring Soon and Renewing Soon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-statuses_Status-filter_Active-selected.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzNzEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WsbaaKodCvui07hBbd1XdbCoQNII4qZmpfrhTRJFWu0&_LANG=enus "A screenshot of the Status filter that shows the selected Active options, including Expiring Soon and Renewing Soon.") See the [Agreement Statuses](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=con6a3bd080-da17-4d1c-a92a-b9b9340c44fd.html&_LANG=enus "Read about agreement statuses, including how they are calculated, visibility within Agreement Manager, and which types of statuses are being tracked.") topic to learn more about this feature.

   The agreement list table updates and the agreements that matched the entered keyword or phrase are displayed. Repeat this process until you locate the relevant agreements.

   Note: After locating the relevant agreements, choose to download selected agreements or export agreement data. See these support articles to learn more about these options:
   - [Download Documents from the Agreement List Table](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=ach1724363496037.html&_LANG=enus "This topic answers the question: How do I download agreements from Agreement Manager?")
   - [Export Data from the Agreement List Table](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=pqf1724363438241.html&_LANG=enus "Learn how to export agreement data to a CSV file for selected agreements.")
7. Use the feedback icons to rate your search experience:

   ![A screenshot of the feedback mechanism that gives users the chance to rate the quality of the search results.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/In-app-saerch-feedback_generic-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYzNzEsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.WsbaaKodCvui07hBbd1XdbCoQNII4qZmpfrhTRJFWu0&_LANG=enus "A screenshot of the feedback mechanism that gives users the chance to rate the quality of the search results.")

   1. Choose the appropriate feedback icon. Select either the Rate as helpful (thumbs up) icon or the Rate as unhelpful (thumbs down) icon.

      The Share your reason to help us improve dialog box opens.
   2. Select a reason for choosing the rating. You can select more than one reason. The choices are: Complete, Relevant, Accurate, and Other.
   3. Enter additional details in the Comments text box.
   4. Select the Submit option.

      Tip: Choose the Skip option to submit your rating without providing any additional information. Your rating is submitted for review.

   The dialog box closes and your feedback is submitted to Docusign.
8. If necessary, reset the search field and clear the applied filters to perform a new query.
   1. Select the Clear Search (X) icon located in the search field to clear the search criteria.
   2. For applied filters, select the Delete (X) icon in the filter to clear any selected values.

You have successfully located agreements by using a combination of keyword searches and applied filters.
