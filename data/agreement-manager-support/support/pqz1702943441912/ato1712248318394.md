# Enter Values for Empty Attribute Fields

Learn how to locate and enter values for attributes with missing or empty values.

When documents are synced or imported to Agreement Manager, key data points are surfaced. If the system is unable to locate a value within the agreement, the attribute (field) value is empty. In these cases, review the agreement and add a value or verify that the value does not exist within the agreement.

Note: See the [AI-Assisted
Insights for Agreement Manager Agreements](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=uwk1702945852405.html&_LANG=enus "Look up the names and definitions for the attributes and values that are surfaced in Agreement Manager agreements with AI. This topic answers the question: What data is being extracted from my Agreement Manager agreements?") topic for a full list of the attributes
(fields) that are surfaced in your agreements.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. Use the available filters to locate agreements requiring review.
4. Use the Filters option to locate the agreements that you want to review:
   1. Select the Filters option.

      The Filters menu expands.
   2. Select the AI Suggestions filter.

      The AI Suggestions filter opens.

      ![A screenshot of the expanded AI Suggestions dialog box that shows the two filter options, Needs Review and No Suggestions.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents-table_AI-Suggestions-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.j9PfD_nkfCPLN3akSXO7VirdiQWa5Im_SPkW8FRGSZY&_LANG=enus "A screenshot of the expanded AI Suggestions dialog box that shows the two filter options, Needs Review and No Suggestions.")
   3. Choose the Needs Review option and select Apply.

   The filter closes and the agreements that match the applied filter are displayed in the agreement list table.

   Tip: ![An image of the spark icon used to denote the use of AI.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/AI-Spark-icon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.j9PfD_nkfCPLN3akSXO7VirdiQWa5Im_SPkW8FRGSZY&_LANG=enus "An image of the spark icon used to denote the use of AI.") The spark icon indicates that the agreement contains suggested values. Review, edit, and approve these values to ensure accurate agreement data for reporting, forecasting, and other business purposes.
5. Select (click) the agreement that you want to review from the list.

   The Agreement preview page opens for the selected agreement. The Details pane is displayed by default and shows a list of attributes and suggested values. These attributes and values are displayed based on the identified agreement type.

   A summary displays at the top of the Details pane. The summary includes the identified agreement type and the number of fields and suggestions to review:

   ![A snippet of the Details pane that shows the identified agreement type and the number of suggestions that require review.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-Preview-page_Details-pane_summary-statement-snippet.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.j9PfD_nkfCPLN3akSXO7VirdiQWa5Im_SPkW8FRGSZY&_LANG=enus "A snippet of the Details pane that shows the identified agreement type and the number of suggestions that require review.")

   Tip: Use the Find details search field to locate specific attributes or values.

   Validate the suggested values for each attribute. If a field value is missing, review the agreement to locate the value.
6. Place the Details pane into *edit* mode.
   - Select the Edit (pencil) icon.
   - Select the Review All option.

   Note: You must have Organize and manage all agreements access to perform data reviews. For all other users, the Edit (pencil) icon and the Review All option are grayed out.

   The Review & edit details pane opens for updates.

   ![A screenshot of the Agreement preview page that shows the Review & edit details pane with editable attribute value fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Review-and-edit-details-pane_Agreement-details-open-for-edits.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.j9PfD_nkfCPLN3akSXO7VirdiQWa5Im_SPkW8FRGSZY&_LANG=enus "A screenshot of the Agreement preview page that shows the Review & edit details pane with editable attribute value fields.")

   Important: When you accept or edit the suggested attribute values, the content in the agreement does not change.
7. Accept or edit the suggested values until each attribute is validated.
8. If the field value is missing, find the value in the agreement and update the attribute.

   ![A snippet of the Agreement preview page that shows an example search term that was found within the document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Agreement-preview-page_Search-dialog-box.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY2MjIsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.j9PfD_nkfCPLN3akSXO7VirdiQWa5Im_SPkW8FRGSZY&_LANG=enus "A snippet of the Agreement preview page that shows an example search term that was found within the document.")

   1. Select the Search (magnifying glass) icon from the top right. A dialog box opens.
   2. Enter a keyword or search string in the Find in Document field. If the value exists, the keyword or search string is highlighted in the agreement.
   3. Once you locate the value, enter it as the attribute value.

      Tip: If the value does not exist in the agreement, leave the field blank and move to the next attribute.
   4. Continue reviewing and updating attribute values until all the attributes have been verified against the agreement.

      Attributes are grouped based on usage and purpose. A progress bar is displayed at the top of each group and shows the number of suggestions that require review.
9. Select Save.

   The agreement details are saved. The Details pane is reverted to read-only mode.
10. Select the Close (X) icon from the top left.

    You are returned to the Completed Documents view. Your search criteria and filters are still applied and agreements requiring review are displayed in the table.

You have successfully entered a value for an empty attribute field. If the attribute value was not surfaced during processing, locate the value in the agreement and enter it as the field value in the Details pane.
