# Agreements Dashboard

This topic answers the question: Where can I find data visualizations and reports for all the agreements in my Docusign account?

## The Agreements Dashboard

The Agreements Dashboard gives users a visual summary of all executed and completed agreements in your account, including agreement insights and charts or reports.

Note: Dashboards are collections of standard and custom reports. These reports are individual data visualizations, such as bar charts, pie charts, and graphs. Each dashboard can contain up to 20 reports.

![A screenshot of the Agreements Dashboard that shows top-level insights and reports for the agreements in your account.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Agreements_Agreements-Dashboard.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5MDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.iEbOJDzOiaRYqAJt_Jisvw6XYegN8SYGinOtBaL1ljQ&_LANG=enus)

Top-level insights:

- All agreements in the repository
- Expiring agreements over the next 90 days
- Renewing agreements with renewal dates within the next 90 days

## Filter the dashboard by 'Party Role'

The Party Role is surfaced during agreement processing. This extraction identifies the role that a specific party plays within the context of an agreement.

Once surfaced, you can filter the Agreements Dashboard to view insights based on a specific role for the selected party.

The Party Role filter is located under the Parties filter:![A screenshot of the Parties filter that highlights the Party Role dropdown and available roles.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Filter-by-party-role_Parties-filter_Party-Role_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5MDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.iEbOJDzOiaRYqAJt_Jisvw6XYegN8SYGinOtBaL1ljQ&_LANG=enus)

Important: When using the Parties filter, you must also select a Party Role value. If you do not know which role to select, choose the All option. If no role is selected, the search returns agreements where the party is not the buyer or the seller.

The top-level insights and individual charts on the Agreements Dashboard display a summary of the agreements that match any applied filters.

For detailed insights and charts, expand the More actions (three dots) menu and select View Report or click anywhere on the chart:

![A snippet of the Expiring agreements insight that shows the View Report option in the More actions menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Agreements_snippet-ellipsesView-Report.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5MDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.iEbOJDzOiaRYqAJt_Jisvw6XYegN8SYGinOtBaL1ljQ&_LANG=enus)

For details about a specific month, select a segment of the bar, pie, or donut chart to view the agreements that make up that segment:

After selecting the View Report option, the detailed report view opens. The report includes a list of agreements, filters, and a visual representation of the agreements that matched the applied filters.

![A screenshot of the Upcoming expiring agreements report that shows the expiring agreements by month reflected in the chart and listed n the table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Reports_Agreements_Agreements-by-expiration-date_report-view.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY5MDAsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.iEbOJDzOiaRYqAJt_Jisvw6XYegN8SYGinOtBaL1ljQ&_LANG=enus)

The report view displays a list of the agreements that are represented in the bar chart. From this view, you can:

- Apply filters to refine the agreement list and chart. For example, use the Sets filter to display data for the agreements in a selected set.
- Sort the agreement list. Use the sorting arrows in the column headers to sort the table.
- Customize the information that is displayed in the agreement list. Use the Customize columns (toggle) icon to update the columns that are displayed in the table.
- Download the agreement data. Use the Download option in the More actions (three dot) menu to export the report data to an output file.

  Note: When you download or export a report, all attributes and available values are included in the output file. This includes both custom and standard fields.
- Save the report. Use the Save As option in the Report Actions (vertical ellipsis) menu to save the report for future use.
- Set up one-time or recurring email reports. Use the Subscribe option to subscribe to receive notifications about the agreements in the selected report.
- Add the report to Your Favorites list on the Overview dashboard. Use the Favorite (star) icon to "star" your frequently used dashboards and reports.
- View the Details, Obligations, and Agreement sets for an agreement. Select (click) an agreement to view the document in the Agreement preview.

## Report limitations

When downloading reports to CSV files from the Insights reporting hub, there is an agreement limit of 25,000 agreements for all IAM reports. This count includes your custom reports.

**Suggested workaround:** Adjust the report filters to run the report in batches until all the relevant agreements are downloaded. For example, run the report in batches according to varying date ranges. Once downloaded, combine the data into one CSV file.

Note: The eSignature reports have a different set of limitations. See the [Using Reports](https://support.docusign.com/s/document-item?bundleId=cnu1642812012214&topicId=wou1578456440763.html) article to learn more about eSignature reports for senders and signers.
