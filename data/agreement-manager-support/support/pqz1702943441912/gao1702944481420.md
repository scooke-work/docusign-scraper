# Typical Agreement Manager Roles and Use Cases

This topic answers the question: Who uses Agreement Manager and what are example use cases?

All organizations can benefit from the features that Agreement Manager offers. The target audience for this feature includes two primary user roles and one administrator role.

| **Roles** | **Example use cases** |
| --- | --- |
| Administrators | - Manage user and group access - Manage access to Agreement Manager agreements - Manage user access to bulk upload agreements - Manage user access to manage agreement sets, agreement types, obligation types, and custom fields - Enable cloud storage providers from the Docusign App Center |
| Analyzers (Contract Managers) | - Manage documents, including understanding agreement data and reviewing, updating, and verifying AI-assisted insights - Create agreement sets to manage access to groups of agreements - Create agreement types and custom fields to support business use cases - Create obligation types to support organizational and departmental use cases - Analyze agreements for upcoming renewal opportunities and obligations; assess trends within a group of agreements - Set up renewal notice email notifications and subscribe to reports - Analyze contractual relationships - Perform bulk ingestion - Manage parties to ensure party normalization |
| Preparers (Contract Viewers) | - Upload single agreements to Agreement Manager - Verify agreement data, such as renewals and payment terms |

Important: Account administrators have full, secure access to Agreement Manager features by default, excluding access to view agreements in the account. They are responsible for granting Agreement Manager permissions and access to users and groups.

Agreement Manager permissions include:

- View all agreements in this account: authorizes users or groups to view and download all agreements in the account (to which they have access).
- Organize and manage all agreements: authorizes users or groups to view, edit, download, and delete all agreements in the account. This permission level also grants access to manage agreement sets and attributes, like agreement types and fields.
- Upload up to 1,000 files at a time: gives users access to bulk upload up to 1,000 files at a time to Agreement Manager. This toggle is turned on (blue) at the user-level.
