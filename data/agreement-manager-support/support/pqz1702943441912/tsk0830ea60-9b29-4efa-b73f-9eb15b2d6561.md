# Create Custom Fields from the Agreement Preview

Learn how to create custom fields from the Agreement preview page for a selected agreement.

Custom field extractions are used to capture business-specific data points that standard extractions might not track. For example, use custom fields to capture terms, dates, amounts, and other details that are important to your organization.

Note: Administrators automatically have access to configure agreement data. For non-administrator users, this feature requires Organize and manage all agreements access to Agreement Manager. These tasks are primarily completed by administrators and contract managers.

Use the Add a field function to configure new custom fields. This feature has three main entry points:

- Fields Manager
- Agreement Types Manager
- Agreement preview page

This topic reviews the steps to create custom fields from the Agreement preview page. When you add custom fields from this location, the *Define the field* step combines field configuration and definition.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. A list of documents to which the logged-in user has access is displayed in the table.

**Select an agreement**

4. Locate the relevant agreement. For example, the agreement that contains the data points for the new custom field.

   ![A snippet of the Agreement Manager homepage that shows the search field and the Filters option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents_search-field_filters-option-expanded_snippet.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.0rhCh0UBLBKwB_PF1WdjbyqZLsbcsDvYJo3bxf8yvpQ&_LANG=enus "A snippet of the Agreement Manager homepage that shows the search field and the Filters option.")

   1. Search for the relevant agreement. Enter a keyword in the search field. As you enter a value, the dropdown opens and a list of suggested searches is displayed. Select a suggested search from the dropdown or press the Enter key.

      The agreement list table updates and agreements that matched the search term are displayed.
   2. Filter the table to locate the relevant agreement. Expand (click) the Filters option and select the filter that you want to apply. Choose your filter values and select the Apply option.

      The agreement list table updates and agreements that matched the applied filters are displayed.
5. Select (click) an agreement from the table.

   The Agreement preview page opens for the selected agreement.

**Add the custom field**

6. Launch the Add a Field function. There are two ways to access this function from the Agreement preview page.
   - From the Details pane: Select the Add (+) icon and select the Add a Field option.

     ![A snippet of the Agreement preview page that highlights the Add (+) icon in the Details pane.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Fields-Manager_Agreement-preview_Details_Add-icon_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.0rhCh0UBLBKwB_PF1WdjbyqZLsbcsDvYJo3bxf8yvpQ&_LANG=enus "A snippet of the Agreement preview page that highlights the Add (+) icon in the Details pane.")

     Tip: Use this option to add metadata to an agreement without tying it to a specific sentence or clause. For example, you want to create a field to capture data about the agreement, like 'internal owner' or 'risk rating'. This information is not tied to a specific sentence or clause in the document.

     .
   - From the Preview pane: Highlight the text that contains the relevant data points and select Track with AI from the dropdown list.

     ![A snippet of the Agreement preview page that demonstrates the text highlighting option for adding a field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Custom-field-extraction_Agreement-preview_text-highlighting_animated-gif.gif?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.0rhCh0UBLBKwB_PF1WdjbyqZLsbcsDvYJo3bxf8yvpQ&_LANG=enus "A snippet of the Agreement preview page that demonstrates the text highlighting option for adding a field.")

     Tip: Use this option when you want to track content that is directly associated with specific text in the agreement. For example, you want to track a specific clause, sentence, or value in the agreement.

   The Add a Field dialog box opens. When accessing the function from the Agreement preview page, you have the option to initiate the AI detection setup.

   ![A screenshot of the Add a Field dialog box that shows the required fields for creating a field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Fields-Manager_Agreement-preview_Add-a-Field-dialog-box_Full.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.0rhCh0UBLBKwB_PF1WdjbyqZLsbcsDvYJo3bxf8yvpQ&_LANG=enus "A screenshot of the Add a Field dialog box that shows the required fields for creating a field.")

**Define the field details**

7. Enter a name for the custom field in the Field name field:

   ![A snippet of the Add a Field dialog box that highlights the Create Field option for the custom field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Custom-field-extraction_Add-a-Field_Create-field-name_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.0rhCh0UBLBKwB_PF1WdjbyqZLsbcsDvYJo3bxf8yvpQ&_LANG=enus "A snippet of the Add a Field dialog box that highlights the Create Field option for the custom field.")

   As you enter a name, a type-ahead dropdown list opens and fields that match the entered name are displayed.

   Tip: Review the type-ahead dropdown list to verify that the custom field does not already exist. If the field exists, you are unable to create a new one with the same name.

   If the field does not exist, select the + Create [Field name] option.
8. Expand the Field type dropdown and select the type of field that you are adding:

   ![A snippet of the Add a field dialog box that shows the available field types for your custom field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Add-custom-field_Field-type-options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.0rhCh0UBLBKwB_PF1WdjbyqZLsbcsDvYJo3bxf8yvpQ&_LANG=enus "A snippet of the Add a field dialog box that shows the available field types for your custom field.")

   Choices are: Text, Number, Date, and Dropdown.

   Tip: If you are adding a Dropdown field, enter the dropdown options in the text-entry box. Use a semicolon to separate each option.
9. Apply the field to the relevant agreement types. Click in the Agreement types field, choose the agreement types for the field, and select the Apply option:

   ![A snippet of the Add a Field dialog box that highlights the Agreement types dropdown for adding field to multiple types.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Custom-field-extractions_Add-a-field_Agreement-types_expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.0rhCh0UBLBKwB_PF1WdjbyqZLsbcsDvYJo3bxf8yvpQ&_LANG=enus "A snippet of the Add a Field dialog box that highlights the Agreement types dropdown for adding field to multiple types.")

   Note: The current agreement type is displayed in the Agreement types field by default. This value is pulled from the selected agreement.
10. Assign a classification to the field. Expand the Category dropdown and select an option from the list.

    Choices are: General, Termination, Renewal, Payment, Legal and Compliance, Human Resources, and Other.

**Without AI detection (optional)**

11. If AI detection is not necessary for the custom field, you are finished with this process:

    ![A screenshot of the Add a Field dialog box that shows a custom field with no AI detection.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Custom-field_Add-a-field_No-AI-setup_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.0rhCh0UBLBKwB_PF1WdjbyqZLsbcsDvYJo3bxf8yvpQ&_LANG=enus "A screenshot of the Add a Field dialog box that shows a custom field with no AI detection.")

    1. Deselect the Use AI to find the field in future agreements option.

       The remaining fields collapse and are no longer displayed.
    2. Select the Save option.

    The dialog box closes. The new field is displayed in the Details pane for the agreement.

    Note: You can complete the AI detection setup from the data management pages. See the [Enable AI Detection for Your Custom Fields](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk02bd31a2-b125-4d53-b632-20d9972163a8.html&_LANG=enus "Learn how to enable the AI detection for your custom fields. Once activated, the field is surfaced in your new agreements automatically.") topic to learn about activating custom fields for AI detection.

**Initiate the AI detection setup (recommended)**

After defining the field
details, you are prompted to initiate the AI detection setup process. Leave this
option selected if you want to surface the custom field in other agreements. For
example, in newly completed agreements and in selected historical documents.

![A screenshot of the Add a Field dialog box that shows the field details and the AI detection setup option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Custom-field-extraction_Add-a-field_Initiate-AI-detection.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.0rhCh0UBLBKwB_PF1WdjbyqZLsbcsDvYJo3bxf8yvpQ&_LANG=enus "A screenshot of the Add a Field dialog box that shows the field details and the AI detection setup option.")

12. The Use AI to find this field in future agreements option is selected by default. Accept the default selection.
13. Describe the purpose of the field. Enter a detailed definition for the field in the text-entry box. This field is limited to 2,000 characters.

    ![A snippet of the Add a Field dialog box that shows a detailed definition for the new field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Custom-field-extraction_Add-a-field_Enter-detailed-definition.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.0rhCh0UBLBKwB_PF1WdjbyqZLsbcsDvYJo3bxf8yvpQ&_LANG=enus "A snippet of the Add a Field dialog box that shows a detailed definition for the new field.")

    Note: See the [Best Practice: Writing Definitions for AI Detection](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tmk1741317734856.html&_LANG=enus "Read about tips for writing good definitions for the AI detection setup process. These definitions are an important part of the process and help to \"teach\" the AI detection.") support topic to review the guidelines for writing definitions. Follow these guidelines to get the most our of the AI detection and to ensure accuracy.

    When the definition exceeds 50 characters, the Refine with AI option becomes available. Use this option to update the definition for consistency and accuracy.
14. Fine-tune the definition.

    ![A snippet of the Add a Field dialog box that highlights the Refine with AI option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Fields-Manager_Agreement-preview_Add-a-Field_Refine-with-AI_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.0rhCh0UBLBKwB_PF1WdjbyqZLsbcsDvYJo3bxf8yvpQ&_LANG=enus "A snippet of the Add a Field dialog box that highlights the Refine with AI option.")

    1. Select the Refine with AI option.

       The dialog box refreshes and a suggested definition is displayed.
    2. Review and edit the suggested definition as needed.

       ![A snippet of the Add a Field dialog box that shows the updated definition that was refined with AI.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Custom-field-extraction_Add-a-Field_Refine-with-AI_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.0rhCh0UBLBKwB_PF1WdjbyqZLsbcsDvYJo3bxf8yvpQ&_LANG=enus "A snippet of the Add a Field dialog box that shows the updated definition that was refined with AI.")

       Tip: Select the Undo option to revert to the original version of the definition.
15. Select the Save option.

    The dialog box closes and a message confirms that the field is saved.

    The Track this field with AI setup wizard launches:

    ![A screenshot of the Track this field with AI setup wizard that displays when you create custom fields from the Agreement preview page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Custom-field-extractions_Track-this-field-with-AI_setup-wizard.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTc2NTYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.0rhCh0UBLBKwB_PF1WdjbyqZLsbcsDvYJo3bxf8yvpQ&_LANG=enus "A screenshot of the Track this field with AI setup wizard that displays when you create custom fields from the Agreement preview page.")

    There are three steps to enable the AI detection for the custom field: Define the field,
    select examples, and test for accuracy. By creating the field from the
    Agreement preview page, you completed the first step
    when you configured the field.
16. Select the Set Up option.

    The set-up wizard advances to the next step and you are prompted to select examples of the field.

    Complete the [Add Examples of Your Custom Fields](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tskcb56face-b8de-4cf9-85ce-6eac3669dd82.html&_LANG=enus "Learn how to complete the second step of the AI detection set-up process, Select examples.") task to provide examples of the field. This step helps the AI model to find the field in different contexts.

You have successfully created a new custom field from the Agreement preview page. You also completed the first step of the AI detection set-up wizard, *Define the field*.
