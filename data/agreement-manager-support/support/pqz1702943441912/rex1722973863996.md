## Common page elements

This view contains a number of helpful page elements, including:

- **Agreement list table:** Displays all the agreements to which the logged-in user has access.
- **Total document count:** Displays the number of documents in the table. The count is located
  in the bottom-right corner of the page. It is updated when you perform a
  search or apply filters.
- **Search field:** Used to perform a keyword search to locate agreements and data.
- **Filters option:** Used to refine the agreement list table for specific data points.
- **Customize columns (toggle) option:** Used to choose which attributes (fields) to display in the agreement list table.
- **Sort items (arrows) option:** Used to sort the agreement list table by a specific column. The Sort items (arrows) option is not available for all column headers.
- The names in the Parties column link to the party hub pages.
- Each agreement in the table has a More actions (three dots) menu. Use this menu to view, download, and share the corresponding agreement.

  Tip: You can also access the document-level Activity Log from this menu. This log tracks any changes that are made to the document or its attributes.
- The View renewing contracts feature used to access the Renewals dashboard.
- The Settings (gear) menu located in the top right of the page provides access to manage agreement data, such as fields, agreement types, parties, obligations, and sets. The job monitoring hub is also accessed from this menu.

## The Insights bar

The Insights bar is an interactive deck of cards on the Agreement Manager homepage. Select the Show Insights option to view relevant, actionable insights with links to reports or next steps.

Select the Hide Insights option to collapse the bar.

## Agreement Manager Settings (gear) menu

The Settings (gear) menu contains options for managing your agreement data, such as: managing agreement types, fields, obligations, and sets. Here are the options in the menu:

![A screenshot of the Settings (gear) menu that shows the options for managing agreement data.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-documents_Settings-menu_expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYwNzYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.nm_9oheHsh9DSW-XrEGaMnuE44k-MmqeFtAQJf5diL4&_LANG=enus "A screenshot of the Settings (gear) menu that shows the options for managing agreement data.")

- Fields option: Used to track data inside your agreements, including creating and maintaining your custom fields. See the [IAM Fields Manager](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=con7092ce57-207f-4d06-9228-c7bdd2ccff2a.html&_LANG=enus "Read about the Fields manager, including its purpose, use, and role within the Intelligent Agreement Management (IAM) platform.") article to learn about this option.
- Agreement Types option: Used to categorize and structure your agreements, including creating and managing your custom types. See the [IAM Agreement Types Manager](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=con22821364-8601-4fc5-b768-e19f07c43462.html&_LANG=enus "Read about the Agreement Types manager, including its purpose, use, and role within the Intelligent Agreement Management (IAM) platform.") article to learn about this option.
- Parties option: Used to maintain business relationships, including merging parties. See the [Duplicate Party Management and Standardization](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=ssf1712255476566.html&_LANG=enus "This topic answers the question: What does party management in Agreement Manager entail? Read about the party management features designed to help you manage parties that have similar names. For example, party standardization and consolidation at the point of ingestion.") article to learn about this option.
- Obligations option: Used to manage the types of obligations in your account, including adding new obligation types. See the [Obligation Management](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=csm1727128259486.html&_LANG=enus "Read about obligation management, including how to use this functionality to capture, track, and fulfill your agreement obligations. This topic answers the question: How can I find and track my agreement obligations?") article to learn about this option.
- Sets option: Users to create sets of agreements and share them with users and groups, including agreement set management functions. See the [Agreement Set Management](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=zed1730843285476.html&_LANG=enus "Read about agreement set management including how to use sets to control access to agreements. This topic answers the question: What are agreement sets and how can I use them?") article to learn about this option.
- Uploads and Processing Status option: Used to view the status of any job, including uploads, AI processing, and synced agreements. See the [Job Monitoring for Agreement Uploads and Processing](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=ahg1742251814403.html&_LANG=enus "Read about the job monitoring hub that is used to track the progress for agreement uploads and processing jobs.") article to learn about this function.
- View Activity Log option: Used to view aggregate, user-performed events across all the agreements in your account. See the [Agreement Manager Activity Logs](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=pne1747684858805.html&_LANG=enus "Read about the account- and agreement-level activity logs, including the types of events that are captured in the log.") article to learn about this function.

Note: Some of these functions require Organize and manage all agreements access to Agreement Manager agreements and functionality. See the [Agreement Manager Permissions, Profiles, and Roles](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=eut1712157553977.html&_LANG=enus "Read about the available Agreement Manager permissions, including how users and groups are granted access to Agreement Manager agreements and example roles for each permission.") article to learn more about Agreement Manager permissions.

## Add documents menu

The Add (+) menu is available to all users. Use this function to upload single documents to Agreement Manager:

![A screenshot of the Add (+) dropdown menu that shows the Upload, Import from Email, and Import Data options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Completed-view_Add-Documents-dropdown_expanded.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTYwNzYsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.nm_9oheHsh9DSW-XrEGaMnuE44k-MmqeFtAQJf5diL4&_LANG=enus "A screenshot of the Add (+) dropdown menu that shows the Upload, Import from Email, and Import Data options.")

Agreement Manager permissions are not required for uploading single documents, however, larger uploads require an additional permission. By default, all users can upload up to five agreements without special permissions.

See the [Agreement Uploads and Ingestion](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=rrs1712255668350.html&_LANG=enus "This topic answers the question: What are my upload options for Agreement Manager, including limitations and required permissions?") article to learn more about using this function for agreement uploads and imports.
