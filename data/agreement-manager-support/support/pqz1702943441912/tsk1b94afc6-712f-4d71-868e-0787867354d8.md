# Download the Pre-Formatted Spreadsheet File

Learn how to download the pre-formatted spreadsheet file for a group of documents.

## Before you begin

Verify that you have the appropriate permissions before you use the *Bulk Metadata Editing* feature. This feature requires bulk upload access and either Can Manage or Can Edit access to the agreements being updated. See the [Bulk Metadata Editing](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=concb493b18-adf8-4fe5-8b99-dd90cbc28399.html&_LANG=enus "Read about the Bulk Metadata Editing feature, including use cases, typical roles and permissions, and a process overview.") article to learn more about this feature.

Attention: Before downloading the spreadsheet, be sure that your custom fields are enabled in Agreement Manager. The pre-formatted spreadsheet contains all the custom and standard fields in the account. If you plan on adding custom data, the spreadsheet must contain a column for the custom field. You cannot add columns and rows to the pre-formatted spreadsheet.

The *Bulk Metadata Editing* feature is used to update the agreement data for a group of documents all at once. This is accomplished by using a pre-formatted, Docusign-generated spreadsheet file. Use this spreadsheet as a template for editing your agreement data in bulk.

Begin by downloading the pre-formatted spreadsheet file for a group of documents. This task is completed from the agreement list table on the Agreement Manager homepage.

Important: The function used to download the metadata spreadsheet is not the same as the Download files and Export to CSV functions. The metadata spreadsheet is designed for the bulk metadata editing process. You can only download it from the Edit (pencil) dropdown on the Agreement Manager homepage. This option becomes available after selecting at least one document from the agreement list table.

1. Log in to your Docusign account.

   The Docusign Home page opens.
2. Select Agreements from the global header.

   The Docusign Agreements page opens.
3. Select Completed from the AGREEMENTS section in the navigation pane.

   The Agreement Manager homepage opens and the Completed Documents view is displayed. The agreements to which you have access are listed in the table.
4. Filter the agreement list table to locate the documents that you want to update. For example, filter the table by agreement set or upload job.
   - Locate agreements by upload job (recommended for editing recently migrated documents):![A snippet of the Uploaded Agreements table that highlights the Browse Files option for an upload job.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Bulk-metadata-editing_locate-docs_filter-by-upload-job.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3OTMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.NzWGIzHDdPM3JB0zHYVKKvaD_9LE8f4mZMN7yvg4jHQ&_LANG=enus "A snippet of the Uploaded Agreements table that highlights the Browse Files option for an upload job.")

     - Begin from the Agreement Manager homepage. Expand the Settings (gear) menu and select the Uploads and Processing Status option. The job monitoring hub opens to the All jobs page. The Uploaded Agreements view is displayed.
     - Apply filters or scroll to locate the relevant upload job. Filter options include: Created date, Status, and Source. You can also use the Sort items (arrows) icons to sort the table by column.
     - Expand the Actions option for the upload job and select the Export Agreement Data option.

     The View or edit agreement data in a spreadsheet dialog box opens. Proceed to step 7.
   - Locate the agreements by set. Expand the Filters option, select the Sets filter, choose the relevant set, and select the Apply option:![A snippet of the Agreement Manager homepage that shows the Sets filter to locate agreements by set.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Bulk-metadata-editing_locate-docs_filter-by-set.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3OTMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.NzWGIzHDdPM3JB0zHYVKKvaD_9LE8f4mZMN7yvg4jHQ&_LANG=enus "A snippet of the Agreement Manager homepage that shows the Sets filter to locate agreements by set.")

     The agreement list table updates and the documents that matched the applied filters are displayed.

   Tip: You can edit up to 1,000 agreements at a time in a single upload job. Larger data sets are processed in batches.
5. Select the documents that you want to update:
   - Select the Select all checkbox from the header row. This option selects all the agreements on the current page of results.
   - Select the checkboxes for the documents that you want to update.
   - Select all the documents that match the applied filters. After you select at least one document from the table, additional actions are displayed, including the Selected dropdown. This option shows a count of the selected documents:

     Expand the Selected dropdown and select the Select all that match your filters option:

     ![A snippet of the Completed Documents page that shows the Select all that match your filters option in the agreement list table.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Bulk-metadata-editing_Locate-docs_Select-all-that-match-your-filters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3OTMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.NzWGIzHDdPM3JB0zHYVKKvaD_9LE8f4mZMN7yvg4jHQ&_LANG=enus "A snippet of the Completed Documents page that shows the Select all that match your filters option in the agreement list table.")

   Warning: Be sure to select all the documents that you want to update. The pre-formatted spreadsheet contains a row for each of the selected agreements. Due to mapping issues, you cannot add a row to the spreadsheet once it is downloaded. The row must contain a Docusign-generated agreement ID to map the edited metadata to the correct agreement.
6. Expand the Edit (pencil) dropdown and select the Agreement Data option:

   ![A snippet of the Completed Documents page that shows the Edit Agreement Data option in the Edit (pencil) dropdown.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Bulk-metadata-editing_Edit-dropdown_Edit-Agreement-Data_highlighted.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3OTMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.NzWGIzHDdPM3JB0zHYVKKvaD_9LE8f4mZMN7yvg4jHQ&_LANG=enus "A snippet of the Completed Documents page that shows the Edit Agreement Data option in the Edit (pencil) dropdown.")

   The View or edit agreement data in a spreadsheet dialog box opens. The File name field defaults to a system-generated name.
7. Download the pre-formatted spreadsheet for the selected documents:

   ![A screenshot of the View or edit agreement data in a spreadsheet dialog box that shows the Download and Edit option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pqz1702943441912/page/images/Bulk-metadata-editing_View-edit-agreement-data-spreadsheet_Download-and-Edit.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE3OTY3OTMsInNoZWFmIjoicHF6MTcwMjk0MzQ0MTkxMiJ9.NzWGIzHDdPM3JB0zHYVKKvaD_9LE8f4mZMN7yvg4jHQ&_LANG=enus "A screenshot of the View or edit agreement data in a spreadsheet dialog box that shows the Download and Edit option.")

   1. Enter a unique name for the spreadsheet in the File name field or accept the system-generated default value.
   2. Select the Download and Edit option.

   The pre-formatted spreadsheet file is downloaded to your local Downloads folder and the dialog box advances to the next step, *Import data*. You can skip this step for now and come back to it later.
8. After adding the relevant metadata to the spreadsheet, import the same file back into your account. For now, select the Skip for Now option.

   The dialog box closes and you are returned to the Agreement Manager homepage.

You have successfully downloaded the Docusign-generated, pre-formatted spreadsheet file. Use this spreadsheet to update the metadata for the selected documents. Continue to the next step of the process, [Edit the Metadata Spreadsheet](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=tsk8710c3f2-c881-4219-96b0-2b287e8b4e03.html&_LANG=enus "Learn how to edit the metadata spreadsheet file to bulk update the agreement data for the selected documents.").
