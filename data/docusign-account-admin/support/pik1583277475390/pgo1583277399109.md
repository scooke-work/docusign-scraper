# Major Sections of Docusign Account Administration

Read about the major sections of Docusign account administration
settings. Depending on the account plan type, administrators might see only some sections and
not all.

Docusign Account settings include several major sections:

Note: Not all the sections described below are
available to all account types. The sections shown depend on the type of Docusign plan that you have.

[Account Settings](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=nfw1583277366851.html&_LANG=enus "Read how administrators manage account billing, profile, security, and branding.") - Provides information on your plan and billing, account
profile, security settings, regional settings, and branding settings.

[Users and Groups Management](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=dct1583277369801.html&_LANG=enus "Read about how administrators manage individual users, groups of users, and permission profiles for an account through the Users and Groups section.") - Provides information on managing users, groups of
users, and permission profiles.

[Signing and Sending](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=jrx1583277385322.html&_LANG=enus "Learn how account administrators manage the default behavior and options for all documents sent from an account through the Signing and Sending section.") - Provides information on managing the default behavior
and various options for all envelopes sent from your account.

[Integrations](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=cnn1583277397175.html&_LANG=enus "Read about how administrators manage the Docusign Connect configurations and API integration keys for their accounts from the Docusign eSignature account administration web application.") - Provides information on managing the Connect
configurations and API integration keys for your account.

[Logs of Account Changes and Bulk Actions](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=xpg1583277398727.html&_LANG=enus "Read about how account administrators review and download a history of changes to accounts and the results of bulk actions. Audit logs capture events and changes made to accounts such as settings, billing information, and plan type. Bulk actions logs provide details about bulk actions such as envelope transfers and user uploads.") - Provides information on working with the audit log for
the account.

[Automate Common Post-Signature Tasks with Agreement Actions](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=hzy1594673444380.html&_LANG=enus "Read about how account administrators use Agreement Actions to create and manage business rules to determine what happens to completed envelopes. The rules can trigger actions, such as archiving to a cloud storage application, starting a Docusign CLM workflow, and exporting data to a spreadsheet.") - Provides information on automating tasks including archiving to cloud storage, starting a workflow, and exporting data to a spreadsheet.
