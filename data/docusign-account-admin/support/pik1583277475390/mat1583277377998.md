# Disable or Enable Custody Transfer Rules

Learn how to enable or disable custody transfer rules. As an administrator, you can
enable and disable the rules as needed.

Custody transfer rules can be disabled or enabled from the Custody Transfer rule list.

1. From the Accounts view, select Custody
   Transfer.
2. Locate the rule that you want to disable or enable.
3. Select Actions > Disable to disable the rule. If a rule is disabled, select
   Enable to enable the rule.

   The check mark in the Enabled column changes to show if the rule is disabled (no check
   mark) or enabled (check mark).
