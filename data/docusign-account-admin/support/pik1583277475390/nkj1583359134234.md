# Create an Organization

Learn how to create an organization and begin to manage multiple accounts.

1. Log in to a Docusign account that is enabled for Docusign Admin, and select the Admin tab from the
   top navigation bar.
2. On the Overview page in the Accounts view, select Get
   Started.

   ![The screenshot shows the Get Started button to create a new organization.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/The-screenshot-shows-the-Get-Started-button-to-create-a-new-organization.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjU5NTIsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.Fe-zofl7mBeHI5XHkXCqSVrdVswS-nwtIFbRKW4VX4k&_LANG=enus "The screenshot shows the Get Started button to create a new organization.")

   Note: If you do not see
   the Get Started section in your account, Docusign Admin is not enabled on the account. Contact Docusign Support for
   assistance.
3. Enter an organization name and an optional description in the fields, and select
   Next.
4. Link accounts to your organization. The account you used to create the organization is
   automatically linked. Select any additional accounts you and the other Docusign administrators want to manage centrally through your organization.

   The accounts for which you are an account administrator with All Administration
   Capabilities permissions are listed. You can link additional accounts at any time. See
   .
5. Select Create to finish creating your organization.

   Your organization is created.
