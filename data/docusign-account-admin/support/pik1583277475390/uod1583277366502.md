# Topics Related to Configuring and Using Brands

Read topics about configuring and using brands that are helpful to administrators.

Administrators that are interested in learning more about configuring and using brands can
read the following topics:

- [Assign Signing Brands to Groups](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=iby1583277366335.html&_LANG=enus "Read about how administrators assign brands to groups for group members to use when they send envelopes or create templates.") - Use the groups functionality to manage which signing brands are available to your account members when they send envelopes and create templates.
- [Advanced Configuration for Branding](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=cno1583277333030.html&_LANG=enus "Read about the advanced configuration options available to administrators that want to customize the signing experience. Administrators can customize destination URLs, header and footer links, and signing brand languages.") - Specify destination URLs, header and footer links, and customize resource files.
- Create a PowerForm - PowerForms inherit the signing branding specified on the associated template.
