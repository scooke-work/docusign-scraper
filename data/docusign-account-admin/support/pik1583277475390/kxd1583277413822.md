# Remove Users and Groups from Stamps

Learn how administrators remove users and groups from stamps.

As an administrator, you can remove users and groups from stamps. This procedure does not
affect documents that were stamped by the users that you remove.

1. From the Accounts view, select
   Stamps.
2. Select Actions > Users and Groups.
3. Select either the Users or Groups tab
   depending on which type you are removing.
4. Select Remove to remove that user or group.
