# Customize Sending Brand Header and Footer Links

Learn how administrators customize or hide links included in the sending
brand.

Note: These features are temporarily disabled for account plans
purchased on the web after November 20, 2024. For more information, contact Docusign Support.

You can customize or hide the links in the sending brand in the Docusign
applications. You can add up to three footer links.

Note: To customize header and footer links for a signing brand, see [Advanced Configuration for Signing
Brands](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=cno1583277333030.html&_LANG=enus "Read about the advanced configuration options available to administrators that want to customize the signing experience. Administrators can customize destination URLs, header and footer links, and signing brand languages.").

1. From the Accounts view, select
   Brands.
2. Select the Sending tab.

   Your current sending brand configuration displays.
3. Select Header and Footer Links.
4. In the Customize Links page, you can add up to three links to the footer by specifying
   text and URLs for Footer Link 1, 2, and 3.

   ![The screenshot shows the Footer Link 1 section of the Customize Links page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/footerlinksadd.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUwNTIsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.P-Bkge9hCF4qljuzNiJKH32u-pp3UhUHGyv_O6Sv7UM&_LANG=enus "The screenshot shows the Footer Link 1 section of the Customize Links page.")
5. (Optional) To hide a link, select the Hide link toggle next to
   the link.

   If the footer link is blank, with no text or URL, it will not display to your users.
   Using the Hide link option is useful if you have completed the link information, but you
   want to hide it without deleting the information.
6. Select Save.

Your link changes are saved. Users will see the new
links the next time they log in to Docusign.
