# View and Edit Permission Profiles

Learn how account administrators modify the names of permission profiles and their
settings.

As an administrator, you can modify the names of
permission profiles and their settings.

1. From the Accounts view, select Permission
   Profiles.
2. Find the permission profile that you want to edit or view.

   Note: You cannot edit the
   DS Admin, DS Sender, and DS Viewer permission profiles.
3. Select Actions > Edit next to the permission profile.
4. In the Edit Permission Profile dialog box, change the permission
   profile name and settings as needed.

   See the [Permission Profile Options](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=mfo1583277360424.html&_LANG=enus "Read about the administrator and user permissions that administrators use to define the actions available to account users that are assigned to the Permission Profile.") for information about the settings.
5. Select Save.

The changes to the permission profile are saved.
