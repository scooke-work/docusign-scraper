# Delete Stamps

Learn how administrators remove stamps that are not needed.

As an administrator, you can remove stamps that are no longer
needed.

1. From the Accounts view, select
   Stamps.
2. Choose the stamp row, and select Actions > Delete.
3. Select Delete to confirm that you want to delete the
   stamp.
