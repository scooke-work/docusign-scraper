# Turn Off KBA in an Existing Identity Verification Configuration

Learn how administrators can turn off KBA when editing an existing
configuration.

You can turn off KBA in an existing Identity Verification configuration if you want recipients
to identify themselves using methods other than KBA.

1. From the Accounts view, select Identity
   Verification.
2. On the configuration you want to edit, select Actions > Edit.

   The Edit Configuration screen opens.
3. In the Accepted IDs section, uncheck the
   Accept all recommended ID types, issuing countries and
   regions option and select Manage.

   The Manage accepted IDs screen
   opens.
4. Type “United States” in the search field.

   The Manage Accepted IDs screen displays the
   United States entry.
5. Uncheck the KNOWLEDGE-BASED AUTHENTICATION column.
6. Select Save.

   The Edit ID Verification screen
   opens.
7. Select Save.

ID Verification updates your configuration.
