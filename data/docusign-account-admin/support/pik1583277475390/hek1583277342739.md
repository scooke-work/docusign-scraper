# Reset the Default Docusign Sending Brand

Learn how administrators reset a sending brand to the default Docusign sending brand.
This procedure discards all sending brand customizations.

You can revert the sending brand for your account to the default Docusign
sending brand at any time. When you revert to the default brand, any sending brand
customizations you made are discarded.

1. From the Accounts view, select Brands, and
   select the Sending tab.

   Your current sending brand configuration is displayed.
2. Select Edit Your Theme to open the brand customization view.

   ![The screenshot shows the Revert to Default button in the Customize a Sending Brand window.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/sending-brand-reset.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUwNTUsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.qDkROJ1o8skwq7AYa9ZcoaYrOoCElMEuOKANRtgVIbU&_LANG=enus "The screenshot shows the Revert to Default button in the Customize a Sending Brand window.")
3. Select Revert to Default, and select
   Reset to confirm.

The sending brand for your account is reset to the
Docusign default branding.
