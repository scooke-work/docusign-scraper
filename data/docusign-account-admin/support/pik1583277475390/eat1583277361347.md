# Add and Copy Permission Profiles

Learn how account administrators add permission files. To save time, administrators can
create a new permission profile by copying and modifying the settings from an existing profile.

As an administrator, you can add permission profiles or
create new ones from existing profiles.

1. From the Accounts view, select Permission
   Profiles.
2. Select Add Permission Profile.

   Tip: You can select Actions > Copy to create a new permission profile that is a copy of an existing set. The
   copied permission profile contains the same settings as the original permission profile.
   You can modify the name and settings as needed using the following steps.
3. Enter a name for the permission profile.
4. Adjust the settings as needed. See the [Permission
   Profile Options](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=mfo1583277360424.html&_LANG=enus "Read about the administrator and user permissions that administrators use to define the actions available to account users that are assigned to the Permission Profile.") topic for information about each setting.
5. To finish and add the permission profile, select Add.

The permission profile is added to the account.
