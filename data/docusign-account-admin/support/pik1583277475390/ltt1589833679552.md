# Set Signing Brand Languages

Learn how administrators specify a language for a signing brand.

You can select the languages available to your signers and assign a default language for your
signing brand. If a signer is using one of the selected languages, they view the
brand in their preferred language. If they are using another language or have not
selected a signing language, they view the brand in the default language.

1. From the Accounts view, select
   Brands to view the Signing
   tab.
2. Select the menu icon and select Edit to edit a signing
   brand. ![The screenshot shows the Signing Brand menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/Edit-menu-brands.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUwNzMsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.dFoepT-DP_hWKw2iGM3uAO1w4acDp8wXM4Ag_OTbnuo&_LANG=enus "The screenshot shows the Signing Brand menu.")
3. Select Select Brand Languages in the Advanced
   Configuration section.
4. Select the languages that you want to include in this brand.
5. Select a default language.

   ![The screenshot shows the Select Languages window for a brand.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/Select-branding-languages.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUwNzMsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.dFoepT-DP_hWKw2iGM3uAO1w4acDp8wXM4Ag_OTbnuo&_LANG=enus "The screenshot shows the Select Languages window for a brand.")

   Note: If signers aren't using one of the
   selected languages, the default language is applied to the brand.
6. Select Save.

The signing brand languages are updated.
