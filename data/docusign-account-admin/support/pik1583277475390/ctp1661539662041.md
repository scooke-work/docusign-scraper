# Shared Access Permission Access Levels

View the levels of access administrators can grant through the Shared
Access permission.

The following table describes the level of access provided by the Shared
Access permission in combination with each of the Users
and Groups access levels.

| Administrative area | Page | Level of access: View Only | Level of access: Edit Groups Only | Level of access: Edit Users & Groups | Level of access: Edit Users, Groups, & Administrators |
| --- | --- | --- | --- | --- | --- |
| N/A | Overview | View | View | View | View |
| Users and Groups | Users | View | View | Edit[1](https://support.docusign.com/s/document-item?language=en_US&bundleId=pik1583277475390&topicId=ctp1661539662041.html&_LANG=enus&anchor=fntarg_1) | Edit[2](https://support.docusign.com/s/document-item?language=en_US&bundleId=pik1583277475390&topicId=ctp1661539662041.html&_LANG=enus&anchor=fntarg_2) |
| Users and Groups | Permission Profiles | View | View | View | View |
| Users and Groups | Groups | View | Edit | Edit | Edit |
| Users and Groups | Signing Groups | None | None | None | None |
| Auditing | Audit Logs | None | None | None | None |
| Auditing | Bulk Actions | View | View | View | View |
| Auditing | Validations | None | None | None | None |

[1](https://support.docusign.com/s/document-item?language=en_US&bundleId=pik1583277475390&topicId=ctp1661539662041.html&_LANG=enus&anchor=fnsrc_1) Edit access does not include the ability to edit users with "Allow All"
administrative permissions enabled, such as those with the DS
Admin permission profile.

[2](https://support.docusign.com/s/document-item?language=en_US&bundleId=pik1583277475390&topicId=ctp1661539662041.html&_LANG=enus&anchor=fnsrc_2) Edit access does not include access to Shared Access or
Shared Envelopes unless the
Envelope Sharing permission is also
selected, and does not include access to Transfer
Envelopes or Transfer
Templates unless the Transfer
Envelopes permission is also
selected.
