# Set Draft Envelope Retention Periods

Learn how account administrators modify the retention policy for draft envelopes.

You can modify the draft envelope retention policy to specify the number of days that draft
envelopes are retained. After the number of days has passed, the drafts are deleted
from the system without further notification.

1. From the Accounts view, select Document
   Retention.
2. Select the Envelope Retention Policy tab.
3. Select Set the draft envelope retention policy for this
   account.
4. In the Keep for this many days field, enter the number
   of days between 30 and 120 to keep draft envelopes before sending them to the
   purge queue. The default value is 30 days.
5. Select Save.
