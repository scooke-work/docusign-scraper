# Delete Permission Profiles

Learn how account administrators delete permission profiles when they are not
needed.

When a permission profile is no longer needed, you
can remove it.

1. From the Accounts view, select Permission
   Profiles.
2. Find the permission profile that you want to delete.

   Note: You cannot delete
   the DS Admin, DS Sender, and DS Viewer permission profiles.
3. Select Actions > Delete.
4. Confirm that you want to delete the permission profile from the account.

The permission profile is removed from the list.
