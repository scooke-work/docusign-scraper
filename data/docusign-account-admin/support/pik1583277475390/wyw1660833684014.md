# View or Remove Shared Access with a User

Learn how administrators manage shared access to envelopes among users.

As an administrator, you can view and manage the shared access with a specific user. The
Shared with [User Name] page shows the list of users that the current user shares
access to their envelopes.

1. From the Accounts view, select
   Users.
2. Locate the user that you want to manage sharing settings for, and select Actions > Manage Shared Access.
3. Select Shared with [User Name].

   The users that shared access to their envelopes with this user are
   listed.
4. (Optional) Select View next a user name that has the
   shared access for this user.
5. In the list for the shared access for the user you selected, select
   Remove to remove the shared access to their envelopes
   for the user you are managing.
6. Select Remove on the confirmation message.

   The shared access is removed for the user.
