# Turn Off Bank Login Identity Verification for a Specific Country or Region

Learn how account administrators can turn off the bank login option if they want
recipients to identify themselves only using a driver license or a passport for a specific
country or region.

Recipients who need to identify themselves using an identity verification method can choose
between using a bank login, a driver license, a passport, or KBA. You can turn off
the bank login option if you want recipients to identify themselves only using a
driver license or a passport for a specific country or region.

1. From the Accounts view, select Identity
   Verification.
2. Search for the configuration you want to edit.
3. Select Actions >  Edit.
4. In the Accepted IDs section, uncheck the
   Accept all recommended ID types, issuing countries and
   regions option and select Manage.

   The Manage accepted IDs screen opens.
5. Search the countries for which you want to deactivate the Bank Login identity
   verification.
6. Uncheck the ELECTRONIC ID column.
7. Select Save.

   The Edit Configuration screen opens.
8. Select Save.

The configuration is updated.
