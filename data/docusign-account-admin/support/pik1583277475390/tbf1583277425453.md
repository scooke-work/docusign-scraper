# View or Edit a User

Learn how account administrators view or edit user information. Only users with the Active status are available to edit.

User details include contact information, permission profile, groups, and the API Username. View or edit users with the Active status. Users with the Closed or Pending status are view-only.

Note: The API Username is used in the OAuth JSON Web Token grant flow and other API methods. For more information about the OAuth JWT grant flow, visit the  [Docusign Developer Center](https://developers.docusign.com/platform/auth/jwt/).

1. From the Accounts view, select Users.
2. Locate the user that you want to view or edit.

   Note: Only users with the Active status are available to edit.
3. In the Action menu for the selected user, select View or Edit.

   The user information page opens.

   ![The screenshot shows the User Profile information.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/view-user.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUxNDMsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.0cdHTYkbPUhMJJylEElo69ZYtbCkh9e7zNvw2ZD-nPE&_LANG=enus "The screenshot shows the User Profile information.")
4. Review the user details and update the entries as needed.
5. Select Save Changes after making updates.

The changes are saved and the Users page displays.
