# Download Permission Profiles Data

Learn how account administrators download the data associated with permission profiles as
a comma-separated value (CSV) file.

Administrators can download a list of permission profiles
and the data associated with them as a comma-separated value (CSV) file.

1. From the Accounts view, select Permission
   Profiles.
2. Select Download Permission Profile.

The data associated with the permission profile is exported to a CSV file and saved to your
local Downloads folder.
