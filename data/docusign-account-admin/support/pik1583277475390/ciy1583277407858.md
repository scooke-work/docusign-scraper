# Enable Stamps to be Used for Accounts

Learn how administrators enable stamps to be used by users and groups. Users of accounts
that have stamps enabled can assign stamp fields to documents. Users that open a document that
contains a stamp field can upload or create one.

To enable stamps to be used by users and groups of a Docusign account, an
administrator must enable the Enable signature stamp field check box in
the Sending Settings page.

You have the option to either create a stamp or upload a stamp.
