# Bulk Actions Log

Read about how account administrators use the Bulk Actions Log
to track the progress and results of bulk actions, such as an upload of multiple users and
the transfer of envelopes for an account. Administrators can export the log information as a
comma-separated value (CSV) file.

The Bulk Actions Log provides an easy way for administrators to
track different bulk actions, such as uploading multiple users and transferring
envelopes for their accounts.

As an accoun tadministrator, you can view details about each bulk action job and its
progress. The log displays the type of action, its status, the name of the administrator
who initiated the action, and the date of the action.

From the Actions menu, you can view more details about each action, such as error
messages, and export the log information as a comma-separated value (CSV) file.
