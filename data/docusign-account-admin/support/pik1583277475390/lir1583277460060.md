# Add eNote Credentials

Learn how account administrators add eOriginal credentials to their accounts so that
eNotes can be sent.

To use eNotes, account administrators must add their
eOriginal credentials to their accounts.

1. From the Accounts view, under Integrations,
   select eNote Credentials.
2. In the eNote Credentials page, select Add eNote
   Configuration.
3. Enter the eOriginal credentials for your organization.

   **Tip:**
   Docusign recommends that you create an eOriginal Admin user specifically
   for Docusign API calls and use those credentials here.
4. Select Verify and Save to check and store your eOriginal
   credentials.

Your eOriginal credentials are stored as eNote credentials.
