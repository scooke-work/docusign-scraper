# Reset User Passwords

Learn how account administrators reset user passwords either for security reasons or when users forget them. Administrators request a reset, which sends an email to users with instructions and a link for changing the password.

You can initiate a password reset for users on your account. You can reset a user password if a user forgets it or if you want users to create a new password for security reasons. When you initiate a password reset, users receive a link and instructions for changing the password.

1. From the Accounts view, select Users.
2. Locate the user whose password needs to be reset.
3. Select Actions > Reset Password.

The user receives an email with a link and instructions for changing the password.
