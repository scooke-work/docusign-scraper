# Display Language Values

Look up the code for the language the user views in the user interface.

The Language value is the default language for the user. The value can be any of the codes
shown below:

## Language = code

- Chinese Simplified = zh\_CN
- Chinese Traditional = zh\_TW
- Dutch = nl
- English = en
- French = fr
- German = de
- Italian = it
- Japanese = ja
- Korean = ko
- Portuguese = pt
- Portuguese Brazil = pt\_BR
- Russian = ru
- Spanish = es
