# Define Security Settings

Learn how administrators define settings to control security for accounts.

As an administrator, you can control account
security.

1. From the Accounts view, select Security
   Settings.
2. Adjust your security settings as needed. The options are described in additional topics
   for each section of settings.
3. Select Save.
