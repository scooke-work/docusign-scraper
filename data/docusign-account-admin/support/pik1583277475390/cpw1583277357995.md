# Signature Styles

Read about the signature styles that signers can select.

Select the signature styles available for signers to choose from in the Adopt Your Signature
dialog box. If no styles are selected, Arial Unicode is used. The Arial Unicode signature
style cannot be removed from the list.

For language-speciﬁc styles, select the language tab and select the signature styles. The
language-specific styles are presented to a signer who is signing in that language.
