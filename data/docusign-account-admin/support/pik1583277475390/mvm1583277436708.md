# Download User Lists and Permission Profiles

Learn how administrators download either a full list or a selected list of users on an
account to a comma-separated value (CSV) file. Optionally, administrators can include
permissions and values.

As an administrator, you can download either a full list or a
selected list of users on an account to a comma-separated value (CSV) file. Optionally, you
can include permissions and values.

1. From the Accounts view, select Users.
2. (Optional) Use the Search and Filter
   controls to select the set of users to download. The download includes only those users
   that are displayed.
3. On the Users page, select Download Users.

   Note: If you get a message
   that your download contains too many users to process, use the
   Search or Filters controls to reduce the
   results to an acceptable number.
4. (Optional) Select Include user permissions in the
   Download Users dialog box to include the list of user permissions
   and values.
5. Select Download.

The selected data is downloaded in a comma-separated value (CSV)
file to your Downloads folder.
