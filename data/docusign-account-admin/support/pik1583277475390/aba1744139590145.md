# Purchase an IAM Plan

Learn how to purchase an IAM plan, or upgrade from a trial account.

As a new customer or trial user, you can review and purchase an IAM plan.

1. Review the plan options on the [Plans and Pricing](https://ecom.docusign.com/plans-and-pricing/iam) page, and select Buy Now for the plan that you want.
   1. You can switch to a monthly billing cycle by selecting the Monthly toggle.
   2. For a plan not listed, select a category from the Enhanced plans menu, and select Contact Sales.

   The checkout page opens.
2. (Optional) Select the Billing options or User licenses dropdown to make changes to your subscription.

   The totals will update automatically.
3. (Optional) Select the Included features dropdown to review features included in the plan.

   Overage pricing applies if subscription limits are exceeded.
4. Select Overages to open the feature overage pricing  window and review overage prices.
5. Enter your email address and select Continue.

   The Add billing details page opens.
6. Select a payment method and enter your billing details. ![The screenshot shows an example of the Add billing details page with the option to edit your subscription.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/Add-billing-details-checkout-IAM-starter.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ4OTIsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.DmJBMleplMhprGw7KfI6wE1DGEMfNYEgsNbycch8DVE&_LANG=enus "The screenshot shows an example of the Add billing details page with the option to edit your subscription.")
7. Select Purchase.

   The confirmation page opens.

You have purchased an IAM plan, and an order confirmation is sent to your email.

Note: If you are a new customer, activate your account with the link in your email.
