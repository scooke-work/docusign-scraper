# Set Watermarks

Learn how administrators set a watermark on in-process documents to indicate their
incomplete signing status.

Administrators can configure a message and style for a
watermark. When the document signing is incomplete, each page in a printable version of
the document clearly displays a watermark.

1. From the Accounts view, select Signing
   Settings.
2. Select Watermark Configuration to go to the Watermark
   page.
3. Select Apply watermarks to documents that are still in
   progress.
4. Select the appearance options for the watermark: font style, font color, font
   size, transparency percentage, and orientation.
5. Type the watermark message. The maximum length of the watermark message is 50
   characters.
6. Select Preview to see an example of how the watermark
   will display if the document is printed before completely signed. Review the
   information and make any necessary changes.
7. Select Save to save the watermark settings.

The watermark settings are saved.
