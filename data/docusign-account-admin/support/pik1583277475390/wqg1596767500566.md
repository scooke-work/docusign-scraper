# Create an Agreement Rule

Learn how account administrators create agreement rules to automate post-signing tasks with completed envelopes. To create rules, administrators specify the conditions for the rule and the actions to take when the conditions are met.

You can automate actions taken on completed envelopes by creating rules. To create a rule, set the conditions for your rule and the actions that are taken when the conditions are met.

Note: Agreement Action rules do not work if documents in the template were replaced during the envelope preparation process. Instead, create a new template with the correct documents. Configure a new rule to use this new template in the condition.

1. From the Accounts view, select Rules in the AGREEMENT ACTIONS section.
2. Select ADD RULE.
3. Enter a name for your rule.

   Note: The Enable rule option is selected by default. A selected rule becomes active after you save it. You can disable the rule to prevent it from applying to actions.
4. Select the scope of the conditions that affect the rule.

   Condition types:
   - **All:** All conditions are true for the created rule.
   - **Any:** Any aspect of the conditions is true for the created rule.
5. Select your first **If** condition and select who or what the rule applies to.

   - **Document Custom Field:** The rule applies when a value is entered or selected in a specific document custom field. Select the document custom field from the list and enter the value you'd like to check for.
   - **Envelope Custom Field:** The rule applies when a value is entered or selected in a specific envelope custom field. Select the envelope custom field from the list and enter the value you'd like to check for.
   - **Recipient:** The rule applies to the selected recipient. You can enter either a full or partial email address, for example, both joe.sample@docusign.com or @docusign.com.
   - **Sender:** The rule applies to the selected sender. Select the name of the sender.
   - **Template:** The rule applies to the intended template, which you choose by selecting the type or name. When choosing a template name for a rule, ensure that the template name for the actual template does not include spaces or special characters around it. Template names with spaces or special characters can prevent the rule from triggering properly.

   Note: Document custom fields and envelope custom fields display a list of all eligible fields on the account. The account must have fields available to use these conditions.
6. Select the filter criteria for the conditions.

   - **Is:** The rule applies only when the value is present.
   - **Is not:** The rule applies only when the value is not present.
   - **Contains:** The rule applies only when the value is present in some form.
7. If you need another condition, select **Add Another Condition**.

   Note: You can add up to 10 conditions to an agreement rule.
8. Select one of the following actions:

   - Archive to: Archive completed documents to a cloud storage provider.
   - Start Workflow: Begin a workflow in Docusign CLM.DS or Docusign CLM.CM (formerly SpringCM).
   - Export to Spreadsheet: Export the envelope data to a spreadsheet.

   ![The screenshot shows an example of an agreement rule.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/aacreaterule2.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjU4NzAsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.-K1vEm_JL0aCX04LBjB-X6GT26xJSQAEoURKgpnz1Vw&_LANG=enus "The screenshot shows an example of an agreement rule.")
9. (Optional) Add another action.

   Note: You can add up to three actions per rule.
10. Review the rule, and select SAVE.

The rule is saved.
