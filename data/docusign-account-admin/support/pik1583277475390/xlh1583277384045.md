# Delete Envelope Custom Fields

Learn how account administrators delete envelope custom fields when they are no longer
needed.

As an account administrator, you can delete custom fields from
envelopes when they are not needed.

1. From the Accounts view, select Envelope Custom
   Fields.
2. Find the field that you want to delete, and select Actions > Delete.

The field is removed from the account.
