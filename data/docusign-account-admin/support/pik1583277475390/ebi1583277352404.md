# Document Sources

Read about how administrators specify the sources, such as Dropbox or Google Drive, that account members can use when they add a document to an envelope or template.

As an account administrator, you can select the cloud storage providers that can be used when account members add documents to an envelope or template. The sources include Box, Dropbox, Google Drive, and OneDrive.

Account members view the selected document sources in the Upload menu when they add documents to their envelopes and templates.

Important:

This setting only changes the user interface behavior for showing an upload platform in the dropdown list when a user uploads a document in an envelope or a template. Turning off one of the options, such as Google Drive, does not prevent users from uploading a document from that document source by accessing information through the API. A malicious user or attacker accessing the API can bypass this setting and upload documents even if a specific document source is excluded. Be aware of any suspicious files being uploaded if you have turned off any of these settings.
