# View Users Assigned to Permission Profiles

Learn how administrators view the users that are assigned to permission profiles.

As an administrator, you can view the users that are
assigned a specific permission profile.

1. From the Accounts view, select Permission
   Profiles.
2. Find the permission profile that you want to view.
3. Select Actions > View Users for the permission profile.

A list of users assigned with the selected permission profile is shown.
