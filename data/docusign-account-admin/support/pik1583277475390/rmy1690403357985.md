# Deactivate Identity Verification Configurations﻿

Learn how account administrators can deactivate identity verification configurations
without deleting it.

You can restrict an Identity Verification configuration from being used by deactivating it
instead of deleting it.

1. From the Accounts view, select Identity
   Verification.
2. Search the configuration you want to deactivate and select one of the following
   items in the Actions column:

   - If the configuration was created by an administrator on your account,
     select Edit > Deactivate.
   - If the configuration was created by Docusign, select View > Deactivate.
3. Select Deactivate in the confirmation message.

   Note: If backward compatibility is enabled on your account
   and a particular configuration is being used to ensure backward
   compatibility, it cannot be deactivated.

The configuration is now deactivated and cannot be
used.
