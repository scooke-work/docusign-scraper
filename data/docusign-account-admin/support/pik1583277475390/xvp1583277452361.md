# Transfer Shared Envelopes

Learn how Administrators transfer ownership of shared envelopes from one account member
to another.

You can transfer envelopes belonging to other account members if their envelopes are shared
with you.

Note: Shared envelopes is now in maintenance mode and will no longer be
updated. While you can continue to use shared envelopes, shared access also allows users to
share draft envelopes, move the shared envelopes, create custom folders, and view which
users have access to the envelopes. For more information, see [Manage Shared Access to Envelopes](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=imk1656607611659.html&_LANG=enus "Read about how administrators manage settings for shared access to envelopes among users.").

1. From your Docusign account, select the
   Agreements tab in the global header.
2. Select Shared Envelopes and select the user whose envelopes you
   want to transfer.
3. Locate the envelopes that you want to transfer by filtering, searching, or scanning the
   list.
4. To transfer a single envelope:
   1. Select the action menu for the envelope and select Transfer
      Ownership.
   2. Select the new owner for the envelope.

      ![The screenshot shows the Transfer Ownership section.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/transferownership01.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjU1NTEsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.NGlSdqL2ZJpmTs3313g6yfta66Tj1I6Z8xDnGguS5OE&_LANG=enus "The screenshot shows the Transfer Ownership section.")
   3. (Optional). To add the current owner of the envelope as a Receives a Copy
      recipient, select Add Original Owner as Carbon Copy. The
      envelope displays in their Inbox after the transfer is complete, and they are added to
      the envelope routing order as a completed copy received recipient.
   4. Select Transfer.

      The transfer is initiated. The envelope may remain in the original owner's account
      for a brief time until the transfer is complete. Refresh your browser for an updated
      view of the transferred item.
5. To transfer multiple envelopes:
   1. Mark the check box for each envelope that you want to transfer.
   2. Select the overflow menu and select Transfer
      Ownership.

      ![The screenshot shows Transfer Ownership of multiple envelopes.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/transferownership02.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjU1NTEsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.NGlSdqL2ZJpmTs3313g6yfta66Tj1I6Z8xDnGguS5OE&_LANG=enus "The screenshot shows Transfer Ownership of multiple envelopes.")
   3. Select the new owner for the envelopes and select
      Transfer.

      The transfer is initiated. The envelopes may remain in the original owner's
      account for a brief time until the transfer is complete. Refresh your browser for an
      updated view of the transferred items.
