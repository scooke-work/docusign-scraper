# Bulk Update Users

Read about how organization administrators can bulk update existing users in one or more
accounts.

Note: This guide is for Docusign
organization administrators who oversee multiple accounts. For administrators of
individual accounts, see the [User Bulk Actions](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=mxm1583277431469.html&_LANG=enus "Read about how administrators add or update multiple users simultaneously through the Bulk Add and Bulk Update options. To perform bulk actions, administrators upload comma-separated value (CSV) files that contain updates or new user information.") topic.

Docusign organization administrators with the Administrators and Users
Administrators permission profiles can bulk update existing users across one or more
accounts by uploading a comma-separated value (CSV) file. For more information about the
CSV format, see [Build a CSV to Bulk Update Users](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=jxh1593210657819.html&_LANG=enus "Read about how organization administrators can use a CSV file to update multiple users.").

You can update up to 2,000 users on an account and include up to 50 accounts per imported
CSV. The maximum number of updated users per import is 8,000. Only one type of import or
export can be in progress at a time.

Note:
Docusign organization
administrators with a claimed domain can also update email addresses for users on their
domain. For more information, see [Update User Email Addresses](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=sle1593057084011.html&_LANG=enus "Read about how organization administrators can bulk update user email addresses using a CSV file.").
