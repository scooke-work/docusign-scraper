# Connect an Application

Learn how administrators connect applications to Docusign for use with Agreement Rules.

As an account administrator, you can connect applications to Docusign for use with Agreement Rules.

1. From the Accounts view, select Connections in the AGREEMENT ACTIONS section.
2. Select CONNECT NEW APP.
3. Select the application that you want to connect to Docusign.

   ![The screenshot shows the apps that you can connect to.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/The-screenshot-shows-the-apps-you-can-connect-to.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjU5MDEsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.tSdh3M_7KV1_817yV-xehATcVYrQfk0MQwW1OV1jAYA&_LANG=enus "The screenshot shows the apps that you can connect to.")
4. Enter a nickname for your account and select SAVE.

   Note: Consider using a service account and not a personal account to log into your application. A service account allows multiple administrators to manage service account credentials.
5. Log in to the application that you are connecting, and select Grant Access.

   The application is added to your Active Apps list.

   Note: Docusign *does not* store credential information. This process uses standard Open Authentication (OAuth). Docusign only seeks permissions to create files and folders when processing a rule.

The application you selected is displayed in the Active Apps list.
