# Security Settings Permission Access Levels

View the levels of access that administrators can grant through the Security
Settings permission.

The following table describes the level of access provided by the Security
Settings permission.

Note: The Plan and Billing page appears with the title Subscription and Billing for some account types.

| Administrative area | Page | Level of access |
| --- | --- | --- |
| N/A | Overview | View |
| Account | Plan and Billing | None |
| Account | Account Profile | None |
| Account | Security Settings | Edit[1](https://support.docusign.com/s/document-item?language=en_US&bundleId=pik1583277475390&topicId=zun1661539748163.html&_LANG=enus&anchor=fntarg_1) |
| Account | Regional Settings | None |
| Account | Search Settings | None |
| Account | Brands | None |
| Account | Stamps | None |
| Account | Updates | None |
| Account | Value Calculator | None |

[1](https://support.docusign.com/s/document-item?language=en_US&bundleId=pik1583277475390&topicId=zun1661539748163.html&_LANG=enus&anchor=fnsrc_1) This permission excludes Recipient Session Timeout and
Capture voice recording during phone
authentication.
