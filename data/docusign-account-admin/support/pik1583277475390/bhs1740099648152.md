# License Management in Account Administration

Read about license information shown in the Users page.

For accounts on IAM plans, account administrators can view user license counts and assign licenses to users in the License section on the Users page. Administrators can track license assignments without assistance from Docusign Support or account team. Administrators can view license counts in the License section on the Users page in the Accounts view.

Note: All existing customers that purchase an IAM plan will have their users migrated to Full licenses.

The license information includes the following details for each license:

- License Type: the IAM plan license type, such as Full - CX License or Full - Sales License.
- Assigned: the number of licenses assigned to users
- x of x available licenses for your account: the number of licenses assigned to users compared to the total number of licenses purchased under all accounts in the subscription.

Note: Customers may use an unlimited number of free licenses. Free licenses do not count toward your paid licenses.

The following license types are available based on the IAM plan purchased:

| License Type | Plan Variants | Paid or Free |
| --- | --- | --- |
| Full license | - Full - Enterprise license - Full - Professional license - Full - Standard license - Full - Sales license - Full - CX license | Paid |
| Free license | N/A | Free |

The following functionality is included for each license type:

|  | Free / Free User | Paid License User |
| --- | --- | --- |
| Admin | No access to administration features. | Can assign permissions.  Can add users. |
| eSignature | Sign only. | Can send for signature. |
| Agreement Manager | View agreements signed by own user.  View agreements individually shared by other users. | View all agreements in Agreement Manager.  Can edit and manage extraction field.  Can mass upload third-party docs. |
| Workflow Builder | No access to create/run workflows. Note: a workflow [participant](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=gbb1696973048215.html&_LANG=enus) (for example, a customer in an onboarding workflow) doesn't require a Docusign account. | Can create and run workflows. |
| Agreement Desk | Requestor Features.  No playbook access. | Approver Features.  Playbook Access.  Agreement Desk admin. |

Each user can only be assigned one license per subscription. User access to Docusign features is dependent on the account plan, the license type, and the user permission profile.

For more details about the permissions required for using Agreement Manager, see the [Agreement Manager Capabilities by Permissions](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=vhx1743523538507.html&_LANG=enus) topic.
