# Reminders

Read about how account administrators configure reminder emails to signers.

(Default: Off; Values: 1-999 days)

You can turn on reminders to send follow-up emails to signers automatically. When you enable
reminders, you specify when and how often to send notifications. The number of days before
sending the first reminder should be less than the number of days before the request expires.
