# Download a Sample CSV File for the Add Users Bulk Action

Learn how administrators download a sample comma-separated value (CSV) file they can use
to upload users in bulk. The pre-formatted sample CSV file can save administrators time by
eliminating the need to create and format a file. Administrators populate the sample file
with the new user content and upload it to the Docusign
account.

To save time preparing the comma-separated value (CSV)
file needed to upload users in bulk, you can download a sample file. The sample CSV file
is pre-formatted for the bulk add operation and ready to be populated with your new user
information.

1. From the Accounts view, select
   Users.
2. Select Bulk Add.
3. Select Download Template in the Add
   Users dialog box.

The sample file is downloaded and available for you to
populate with your new user information.
