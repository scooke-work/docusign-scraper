# Delete Groups

Learn how administrators delete groups from an account when they are no longer
needed.

As an administrator, you can delete groups from accounts when
the groups are not needed.

1. From the Accounts view, select Groups.
2. Find the group that you want to delete.

   Note: You cannot delete
   the Everyone and Administrators groups.
3. Select Actions > Delete.
4. Confirm that you want to delete the group from the account.

The group is removed from the group list.
