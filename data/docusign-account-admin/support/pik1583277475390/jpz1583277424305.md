# Add Users to Accounts

Learn how administrators add users to accounts and reactivate closed users.

As an administrator, you can add users to accounts and
reactivate closed users.

Note: For accounts on IAM plans, the Users
page includes an additional column titled License that shows the license type assigned to
the user. This is only for customers on IAM plans. Other customers on eSignature plans do
not have this column and do not have licenses.

1. From the Accounts view, select Users.
2. Select Add User.

   ![The screenshot shows the email address field for adding a new user.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/adduser.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUxMzQsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.EyFwNLMl5bej5ADNy-Ytev6fEDliRyobRxxI2vKeXz4&_LANG=enus "The screenshot shows the email address field for adding a new user.")
3. Enter the email address for the new user, and select Next.
4. If the email address is already part of your account:

   - Active or Pending user: If the email address belongs to a
     user whose status is either an active or pending, the Existing
     User dialog box displays. You can select View
     Profile and review and edit the user.
   - Closed user: If the email address belongs to one or more
     closed users only, you have the option to reactivate a closed user or add a new user.
     If you reactivate a closed membership, the user gains access to all associated
     envelopes. If you are recycling an email address to add a new user to your account,
     select Add New User.
5. Enter the profile information for the user, such as user name and default language, and
   select Next.

   After activation, the user can change these fields by using the My
   Preferences option.
   ![The screenshot shows adding the user profile information.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/addluser2.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUxMzQsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.EyFwNLMl5bej5ADNy-Ytev6fEDliRyobRxxI2vKeXz4&_LANG=enus "The screenshot shows adding the user profile information.")
6. (Optional) Enter an access code, and select Next. If you add a
   code, you must provide the code to the user for them to activate their account.
7. In the permission profile and groups step:
   1. Select a permission profile to assign to the new user. See the [Permission Profiles](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=pof1583277362435.html&_LANG=enus "Read about how administrators use permission profiles to manage levels of access for a number of users. Permission profiles are sets of permissions that administrators assign to individual users. By managing the permission profiles, administrators can make changes to multiple users simultaneously.") topic for more information.
   2. Select a license type to assign to the new user. If your account is on an IAM plan,
      you must assign a license or select the Unmanaged Participant license.
   3. (Optional) Select Assign Groups to select groups to assign
      to the new user.

   ![The screenshot shows adding the license, permission, and groups for the new user.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/addluser3.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUxMzQsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.EyFwNLMl5bej5ADNy-Ytev6fEDliRyobRxxI2vKeXz4&_LANG=enus "The screenshot shows adding the license, permission, and groups for the new user.")
8. Select Add User.

   The user is added to the account with a status of Pending.

The user you added receives an activation email. After the user
completes the steps in the activation email, the membership status changes to
Active.
