# Purchase Extensions for an IAM Plan

Learn how to add feature extensions to your IAM account.

As an administrator of an existing IAM plan, you can add feature extensions to your
account.

1. From the Accounts view, select Plan and
   Billing.

   Note: The Plan and Billing page appears with the title Subscription and Billing for some account types.

   The Subscription and Billing page
   opens.
2. In the Add feature extensions section, review available
   features.![The screenshot shows the Add feature extensions section on the Subscription and billing page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/IAM-add-feature-sub-billing.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ5MDIsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.ZPjsrTFFOuFDONE8tD_HLIPlJ4GdJ1xNmP2t_wAOySc&_LANG=enus "The screenshot shows the Add feature extensions section on the Subscription and billing page.")
3. Select Buy Now for the feature that you want to
   purchase.

   The feature purchase window opens.
4. Choose the quantity of the extension that you want to purchase.
5. Select Purchase.

   The confirmation window opens
6. Select Done.

Your extension is purchased and added to your
account.
