# Add Custom Messages to the Banner

Learn how administrators communicate information by adding custom messages to the top
banner in the sending experience.

Administrators can add a custom message that displays in the top banner of the sending
experience. This message can be used to relay announcements or other important
information. Optionally, you can add a longer message that displays when a user
selects the banner message.

**Example Custom Message**
![The screenshot shows an example message that opens after selecting the custom message title.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/The-screenshot-shows-an-example-message-that-opens-after-selecting-the-custom-message-title.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ5NzUsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.P_Tcp6Whv3i6l4n--3D5mIktN3c0NAlFsEsS4fREFRU&_LANG=enus "The screenshot shows an example message that opens after selecting the custom message title.")

1. From the Accounts view, select Account
   Profile.
2. Select the Enable Message toggle.
3. In the Notice field, enter a message title that is fewer
   than 150 characters.

   Note: The default message
   title is "Update from your administrator" and has been translated for all
   supported languages within the eSignature sending experience. If you edit or
   replace this title, it will not be translated for your end users and will
   display as it is written.
4. Optionally, in the Custom Message field, enter a custom
   message that is fewer than 500 characters.

   You can use HTML to format the text and add links in the custom message. HTML
   tags for images and videos are not allowed.

   Note: If you do not add a
   custom message, your users will only view the message title from the
   Notice field. This can be useful when you only
   want to share a brief message. Add longer messages in the Custom
   Message field.

   ![The screenshot shows the custom message fields in the Account Profile page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/custom-message-fields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ5NzUsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.P_Tcp6Whv3i6l4n--3D5mIktN3c0NAlFsEsS4fREFRU&_LANG=enus "The screenshot shows the custom message fields in the Account Profile page.")
5. Select Save.

   The custom message is saved and is displayed for all users on the
   account.
