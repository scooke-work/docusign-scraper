# User List Exports

Read about the type of user information that you can export.

Note: This guide is for Docusign
organization administrators who oversee multiple accounts. For administrators of
individual accounts, see the [User Bulk Actions](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=mxm1583277431469.html&_LANG=enus "Read about how administrators add or update multiple users simultaneously through the Bulk Add and Bulk Update options. To perform bulk actions, administrators upload comma-separated value (CSV) files that contain updates or new user information.") topic.

Docusign organization administrators with the Administrators and Users
Administrators permission profiles can export a list of users across all accounts in the
organization as a comma-separated value (CSV) file. Exports include user details such as
full name, email address, and permission profile.

Export types:

- Users and Memberships: All users and their memberships across
  all accounts in the organization.
- Domain Users: All domain users, their profile details, default
  account, login policy, which login feature they have enabled, and last login
  timestamp.
- External Domain Users: All domain users memberships in
  accounts external to the organization.

Note: Organizations without
a claimed domain will see only the Users and Memberships export option.
