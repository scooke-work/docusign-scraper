# Set Default Signing Brands

Learn how administrators configure a signing brand to use as a default for envelopes and
templates created for an account. Optionally, the configuration can allow senders to modify the
brand.

As an administrator, you can configure a default signing brand
for envelopes and templates that are created for an account. Optionally, you can allow senders
to modify the brand.

1. From the Accounts view, select Brands to
   view the Signing tab.
2. Locate the brand that you want to set as the default.
3. Select the menu icon and select Set as Default.

The selected brand becomes the default signing brand
that is used by envelopes and templates that are created in your account.
