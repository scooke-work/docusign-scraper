# Resend Activation Emails to Pending New Users

Learn how account administrators resend activation emails to users with a status of
Pending.

As an administrator, you can resend activation emails to users
with a status of Pending.

1. From the Accounts view, select Users.
2. Use the filter to locate users with a status of Pending.
3. Select Actions > Resend Invitation.
4. (Optional) For additional security, enter an access code. Provide the code to the user,
   as it is required for account activation.

The user receives an email with activation instructions for the
account.
