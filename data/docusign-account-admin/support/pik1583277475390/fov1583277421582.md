# Assign Groups to Stamps

Learn how administrators assign groups to a stamp.

As an administrator, you can assign groups to stamps.

1. From the Accounts view, select
   Stamps.
2. Select Actions > Users and Groups.
3. Select the Groups tab.
4. Select Assign Groups.
5. Select the groups that you would like to assign access to this stamp.
6. Select Assign.
