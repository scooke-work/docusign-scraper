# Copy Identity Verification Configurations

Learn how account administrators can copy configurations for identity
verification.

You can add a configuration to your Docusign account by copying an existing configuration.

1. From the Accounts view, select Identity
   Verification.
2. Search the configuration you want to copy and select Create a
   copy in the Actions column:
3. Make any changes to the configuration as needed.
4. Select Add to save the new configuration.

The new configuration is created and is now active.
