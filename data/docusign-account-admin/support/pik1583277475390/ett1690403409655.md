# Activate Identity Verification Configurations﻿

Learn how account administrators activate identity verification
configurations.

You can reactivate a previously deactivated Identity
Verification configuration to make it usable again.

1. From the Accounts view, select Identity
   Verification.
2. Search the configuration you want to activate and select
   Activate in the Actions
   column.
3. Select Activate in the confirmation message.

The configuration is now activated and can be used
again.
