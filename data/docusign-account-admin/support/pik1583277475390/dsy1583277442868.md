# Set Default Email Notification Preferences for New Users

Learn how administrators set default notification preferences for new users, including
signers and senders.

As an account administrator, you can set default notification
preferences for new users.

1. From the Accounts view, select Email
   Preferences.
2. Select the User tab.
3. Review the list of email preferences and make changes as needed.
   1. For Signers. This list presents the notifications that your
      account members will receive when they are the recipient of an envelope. This affects
      the Recipient Notifications list in the user's My Preferences
      > Notifications settings.
   2. For Senders. This list presents the notifications that your
      account members will receive when they are the sender of an envelope. This affects the
      Sending Notifications list in the user's My Preferences >
      Notifications settings.
4. Select Save.

Your default notification preferences are saved. Users added to
your account inherit the default email notification preferences. Users can change these
notification preferences at any time using their My Preferences
settings.
