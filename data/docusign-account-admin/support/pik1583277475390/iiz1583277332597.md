# Customize Header and Footer Links

Learn how administrators customize the links that recipients see in the documents. The
customized links replace the default Docusign links.

Note: These features are temporarily disabled for account plans
purchased on the web after November 20, 2024. For more information, contact Docusign Support.

You can customize the links that display when a recipient opens documents to view and sign.
The links can point to web pages or mailto: email addresses. If you do not customize these
links, or if you delete previously saved customizations, the links use the default Docusign links.![The screenshot shows the Close button and the My Document menu with the default links.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/examplehidingsigningelements.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUwNzEsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.N8QjpJlgwSzqXCLU6ZvodOlYYd7gA8idajZUGnLFMNU&_LANG=enus "The screenshot shows the Close button and the My Document menu with the default links.")

1. From the Accounts view, select Brands to
   view the Signing tab.
2. Locate the signing brand that you want to configure, and select the action menu and
   select Edit.
3. In the Advanced Configuration section, select Header and Footer
   Links.
4. (Optional) To hide the following elements in a signing brand, select the
   Hide Button toggle for the link or button:

   - About Docusign link in the Actions
     menu
   - Close button shown on a completed document
   - Help & Support link shown in the
     Actions menu
   ![The screenshot shows the links that you can customize for a Signing brand.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/The-screenshot-shows-the-links-you-can-customize-for-a-Signing-brand.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUwNzEsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.N8QjpJlgwSzqXCLU6ZvodOlYYd7gA8idajZUGnLFMNU&_LANG=enus "The screenshot shows the links that you can customize for a Signing brand.")
5. To customize the Help & Support link, enter your text and link into the
   Help & Support field.
6. In the Customize Links page, you can add up to three links to the footer by specifying
   text and URLs for Footer Link 1, 2, and 3.
7. (Optional) To hide a footer link, select the Hide Link
   toggle.

   If the footer link is blank, with no text or URL, it will not display to your users.
   Using the Hide link option is useful if you have completed the link information, but you
   want to hide it without deleting the information.
8. Select Save.

Your link changes are saved.
