# Change Brands Assigned to Groups

Learn how administrators add or remove brands that are assigned to groups.

As an account administrator, you can add and remove brands
from groups.

1. From the Accounts view, select Groups.
2. Locate the group that you want to assign a brand to by searching or scanning the list,
   and select it.
3. Select the Brands tab to open the list of assigned brands.
4. To remove brands already assigned, select Remove.
5. To assign additional brands to the group, select Assign Brands,
   select the brands to assign, and select Assign.
