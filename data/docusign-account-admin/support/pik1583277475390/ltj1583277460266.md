# Edit or Delete eNote Credentials

Learn how account administrators edit or delete eNote credentials when updates are
necessary or they are no longer needed.

If your eOriginal credentials change, you can edit the eNote credentials. If you no longer
need the eOriginal credentials, you can delete them.

1. From the Accounts view, under Integrations,
   select eNote Credentials.
2. Select the Actions menu for the eNote credentials.
   - To modify your eOriginal credentials, select Edit and make
     the necessary changes. Select Verify and Save.
   - To remove the eOriginal credentials, select Delete, and
     select Confirm.

Based on your selection, the eOriginal credentials are
either updated or deleted from your account.
