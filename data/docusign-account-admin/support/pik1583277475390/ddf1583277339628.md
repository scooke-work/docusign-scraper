# Delete Signing Brands

Learn how administrators delete signing brands from accounts when the brand is not
needed.

As an administrator, you can delete a signing brand from an
account when the brand is no longer needed.

1. From the Accounts view, select Brands to
   view the Signing tab.
2. Locate the brand that you want to delete.
3. Select the menu icon and select Delete.

The selected brand is permanently removed from your
account.
