# Create an Agreement Rule to Create Another Envelope

Learn how to create an agreement rule that creates and sends another
envelope.

## Before you begin

You must create a connection to the eSignature application before
you create the new rule to create another envelope. Next, create the beginning of an
agreement rule and select the condition scope of Any or
All. For more details, see [Connect an Application](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=cnw1598646827061.html&_LANG=enus "Learn how administrators connect applications to Docusign for use with Agreement Rules.") and [Create an Agreement Rule](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=wqg1596767500566.html&_LANG=enus "Learn how account administrators create agreement rules to automate post-signing tasks with completed envelopes. To create rules, administrators specify the conditions for the rule and the actions to take when the conditions are met.").

You can create an agreement rule that
creates another envelope from a template. You can create follow-up envelopes based
on specific parameters and conditions of a completed envelope. You can save the
newly created envelope as a draft for review or choose to send it
automatically.

1. In an agreement rule you created, select Template for
   the condition.

   Note: You must select Template for the condition if you want to use
   Create another envelope for the action type.
2. Select Create another envelope for the action type.
3. Select eSignature for the app.
4. Select a template for the new envelope.
5. Select either Save to drafts to save a draft of the
   template when the conditions are met or Send
   automatically to automatically send the new envelope after the
   conditions are complete. The default setting is Save to
   drafts.

   The Map roles between templates section is completed with the names of the original
   envelope and the new envelope. If the new template includes any required roles,
   they display as text instead of as a dropdown. The red asterisk indicates they
   are required.
6. Select a role from the original envelope to map to its new related role.

   The sender of the envelope is the administrator that created the connection to the
   eSignature application.
   ![The screenshot shows the Create another envelope action.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/Create-another-envelope-action.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjU4ODAsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.35OzZlH11MFZzv-RyTOZxuNyzB7ghY-f0XAyL7UOE_g&_LANG=enus "The screenshot shows the Create another envelope action.")
7. (Optional) Select ADD ANOTHER ROLE.

   Adding another role adds a new line with a dropdown for each template.
8. Select the role from the dropdown to complete the role mapping. Roles that are
   already mapped are disabled.
9. (Optional) Add another type of action.

   You can add only one Create another envelope action in a rule.
10. Select Save.

The new agreement rule is saved.
