# Create a Docusign Support Case from the Help Menu

Learn how to create a support case from your Docusign account. Users
and administrators can submit a case from the Help menu.

You can create support cases directly from your Docusign
account. You can describe a question or an issue in the support form and attach
related files.

You must be a user or administrator in a Docusign
eSignature, IAM, or Docusign Admin account. This feature is not
available to users on free or trial plans or CLM accounts.

1. Log in to your Docusign account, and select the
   Help menu.

   The Help menu opens.![The screenshot shows the Help menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/support01.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ4NDksInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.r_moxb4QuZFalVS-6K3BjntHDo7bdQ-EmszlAOISYY0&_LANG=enus "The screenshot shows the Help menu.")
2. Select Contact Support.

   The Contact Support window opens.![The screenshot shows the Contact Support form.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/support02.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ4NDksInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.r_moxb4QuZFalVS-6K3BjntHDo7bdQ-EmszlAOISYY0&_LANG=enus "The screenshot shows the Contact Support form.")
3. Select the reason for the support case from the following options:

   - Ask a question
   - Report a problem
4. Enter a short subject to identify the issue or question.
5. Enter a detailed description of the issue or question.
6. Select your preferred contact method:

   - Email
   - Phone - Request a callback
7. (Optional) Select Upload file if you need to attach a
   file to the case. You can add up to five files. The total size of the files must
   be less than 25KB.
8. Select Submit.

A message confirms that your case was submitted. A Docusign support representative will contact you.
