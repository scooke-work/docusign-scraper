# Edit Stamps

Learn how administrators edit stamps.

As an administrator, you can edit stamps associated with your
account.

1. From the Accounts view, select
   Stamps.
2. Choose the stamp row and select Actions > Edit.
3. Make any changes to the stamp name, height, and resize setting.
4. Select Save.
