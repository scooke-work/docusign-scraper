# Delete Signing Groups

Learn how account administrators delete signing groups when they are no longer needed.

Deleting a signing group prevents it from being used in the future. It does not affect any in progress envelopes sent to the group. In progress envelopes will still be accessible to group members even if the group is deleted. To remove access to these members, [void](https://support.docusign.com/s/document-item?bundleId=oeq1643226594604&topicId=vks1578456426150.html&_LANG=enus) the envelope.

If a signing group was used in a template, it is replaced with an empty recipient.

1. From the Accounts view, select Signing Groups.
2. Locate the signing group that you want to delete by searching or viewing the list.
3. Select Actions > Delete.
4. Confirm that you want to delete the group from the account.

   The group is removed from the Signing Groups list.
