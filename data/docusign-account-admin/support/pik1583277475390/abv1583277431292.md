# Bulk Actions Log

Read about the Bulk Actions log and how it tracks the actions that an administrator has
taken on an account. Administrators see the results of an action and see details about any
errors that occurred.

This Bulk Actions log page shows a list of bulk actions, such as uploading multiple users and
envelope transfers for your account, and the progress of the actions. You can leave this
page while the processing is in progress. If you choose to remain on the page, you must
refresh the job display to see an updated status. Select the **refresh icon** in the
Actions column to update the job status.

The details dialog box provides information on how many users were successfully added
and a summary of any errors.

![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/uploaduserdetails.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUxOTMsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.uL4ZTbDkRlIyNpgI-CA70cwHIq_MSIwhlFpmu657ld0&_LANG=enus)

For users that were not successfully added or updated, the message provides a short description of the problem and allows you to download the CSV file to correct the issues. The downloaded file includes a "ProcessingResults" column that specifies the issue for any row that failed to process correctly. After correcting the issues, the file can be uploaded and resubmitted for processing.

If there is a problem with the CSV file, processing might fail. If this occurs, download the file, then review and correct the contents. Alternately, you can create and upload a new CSV file.
