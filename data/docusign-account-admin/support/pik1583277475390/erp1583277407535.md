# Specify Post-Signing Destinations

Learn how administrators can configure a destination URL that users see when they finish
signing. A URL can be configured for a variety of situations, such as Finish
Later, Decline, and Session
Timeout.

Note: These features are temporarily
disabled for account plans purchased on the web after November 20, 2024. For more
information, contact Docusign Support.

As an administrator, you can
configure a destination URL that users see when they finish signing. A URL can be configured
for a variety of situations, such as Finish Later,
Decline, and Session Timeout.

1. From the Accounts view, select Brands to
   view the Signing tab.
2. Locate the signing brand that you want to configure, and select the action menu and
   select Edit.
3. In the Advanced Configuration section, select Destination URLs.
4. In the Configure Destinations dialog box, add a URL for each action you want to
   configure.

5. (Optional) Select the Insert Merge Field menu and select a field
   to add to the destination URL.![The screenshot shows the Insert Merge Field dropdown.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/Merge-fields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUwNjksInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.qzxTbt9L2-9usGPv6UNrmeZqnn-IMHWV2yCtFS716Bw&_LANG=enus "The screenshot shows the Insert Merge Field dropdown.")
6. (Optional) To reset any destination to the Docusign default, simply delete your custom entry in the field.
7. To save your changes, select Save.
