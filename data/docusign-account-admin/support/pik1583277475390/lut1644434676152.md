# Set Password Rules

Learn how administrators set rules for account passwords, such as minimum length, type
of characters allowed, and expiration time frames. Depending on the security requirements of
the organization, administrators can either select from existing options or customize the
options to suit their needs.

Updated password requirements take effect when users
create a new password or update an existing password. Users who are doing neither can
still log in with an existing password created with the previous requirements.

1. From the Accounts view, select Security
   Settings.
2. Scroll down to the Password Security section, and select Show
   Password Rules.

   The Password Rules page opens.
3. Select the password strength for the account.
4. (Optional) If you select Custom for the password strength, select the password
   requirements for the length and age of passwords.
5. (Optional) To require that passwords expire, select **Enable password
   expiration** and enter the number of days until new passwords
   expire.
6. Select **Save**.

The password rules for your account are saved.
