# Activate the Identity Verification for FINTRAC Compliance

Learn how account administrators activate the Docusign ID Verification
for FINTRAC compliance configuration.

To use the FINTRAC configuration in templates or envelopes, activate the Docusign ID Verification for FINTRAC compliance configuration. The
FINTRAC configuration is available in all eSignature accounts. To maintain
compliance, the Docusign ID Verification for FINTRAC compliance
configuration cannot be copied or edited.

Note: It is your responsibility to save the data related to a FINTRAC
configuration for a minimum of five years. You can save the data by downloading the
PDF or using the API to collect the data and then store it outside of Docusign.

1. From the Accounts view, select Identity
   Verification.
2. Search the ID Verification for FINTRAC compliance
   configuration and select Actions > Activate.
3. Select Activate in the confirmation message.
4. To test the configuration, create a new template or envelope.
5. In Identity Verification, select Docusign ID Verification for
   FINTRAC compliance.
