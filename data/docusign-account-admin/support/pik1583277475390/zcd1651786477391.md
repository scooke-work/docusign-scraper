# Replace a Brand With the Resource File of Another Brand

Learn how administrators can replace an existing brand using the XML resource file
exported from another brand.

You can replace an existing brand using the XML
resource file exported from another brand. Replacing a brand is a fast way to copy the
same brand across multiple accounts. For example, you might want to replace brands in a
production environment with a brand created in a demo environment. Replacing a brand
maintains any templates and groups associated with the exported brand.

1. From the Accounts view, select
   Brands to view the Signing
   tab.
2. Locate the brand that you want to use as the replacement brand.
3. Select the menu icon and select Export.

   An XML file of the brand is downloaded to your file system.
4. Locate the brand that you want to replace.
5. Select the menu icon and select Replace.
6. (Optional) Before replacing the brand, you can download a backup copy of your
   current brand by selecting the link in the information message.
7. Select Upload File and select the XML file that you
   exported from the brand you want to use as the replacement brand.
8. Select Replace Brand.

The brand is replaced.
