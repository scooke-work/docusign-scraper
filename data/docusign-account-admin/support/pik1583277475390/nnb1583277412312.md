# Generate a Stamp Using Shachihata (Japanese Version Only)

Learn how administrators use Shachihata to generate stamps.

Docusign is integrated with Shachihata and allows you to generate stamps
based on a Japanese last name.

1. Select 印鑑の追加
   (CREATE STAMP).

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/ehanko1.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUwODYsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.bsrT8DOfYKieP1JB8UT-9nukI7SvydqZVcJ0pKo5Q5w&_LANG=enus)
2. Select 取得 (Get).
3. Enter a Japanese name in the field and select 印鑑の取得
   (Get Stamps).![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/ehanko2.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUwODYsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.bsrT8DOfYKieP1JB8UT-9nukI7SvydqZVcJ0pKo5Q5w&_LANG=enus)
4. Select one of the three example stamps generated by Shachihata.
5. Select 名前
   (Name) or 名前と日付
   (Name + Date).
6. Select 次へ
   (NEXT).
7. Review the stamp name and the phonetic information, which is optional, and select
   作成
   (CREATE).

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/ehanko3.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUwODYsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.bsrT8DOfYKieP1JB8UT-9nukI7SvydqZVcJ0pKo5Q5w&_LANG=enus)
8. Verify that your stamp displays in the list of stamps.
