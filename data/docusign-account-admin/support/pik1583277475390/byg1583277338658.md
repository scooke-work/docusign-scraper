# Add a Signing Brand for Recipients

Learn how administrators add a signing brand for senders to select when they send an envelope. A signing brand can include a custom logo, color scheme, brand name, and company information.

As an administrator, you can add a signing brand for senders to select when they send an envelope. A signing brand can include a custom logo, color scheme, brand name, and company information.

1. From the Accounts view, select Brands to view the Signing tab.
2. Select Add Brand, enter a unique brand name, and select Save.

   The Branding page for your new brand opens. The brand shown is the same as the default Docusign sending brand.
3. For the Info section of your new brand, set the following options and values as needed:

   - Brand Name. The brand name is how the brand is identified in Brands administration and to senders when they select a brand to use for the envelopes they send. Enter a distinctive name to help senders select a brand to use.
   - Company Name. (Optional) Enter a company name to use in email communications and on the Electronic Record and Signature Disclosure. To employ the Company Name value entered in a brand, you must enable the Use brand company name option in the Legal Disclosure page and utilize a [custom email resource file](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=yum1583277332870.html&_LANG=enus "Read about how administrators that have XML and HTML skills customize resource files to control text elements in an account. Administrators can customize one or both of the resource files available for Docusign eSignature. One file is used for signing and the other file is used for email messaging."). See the [Electronic Record and Signature Disclosure](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=aqg1583277336549.html&_LANG=enus "Read about how account administrators can enable an optional Electronic Record and Signature Disclosure to obtain legal consent from signers before signers interact with documents. Administrators can customize fields in the form.") topic for more information.
   - Use sender's company name. (Optional) Select this option to use the company listed in the sender's Preferences > Personal Information. If this option is selected, any entry in the Company Name option for the brand is ignored.

     If neither the Company Name field is completed nor the Use sender's company name option is selected, then the account name is used for email communications and the disclosure.
   - Set as Sign Default. (Optional) Select this option to make the brand the default brand used for all documents sent from the account.
4. Under Customize What Recipients See, select Edit Your Theme.
5. To add a custom logo for signing or email notifications, select one of the following:

   - Upload Signing Logo. Displays a custom logo in the signing view when recipients open your documents to view and sign.
   - Upload Email Logo. Customize the logo used in email notifications sent to envelope recipients.

     Select a logo file that conforms to the image requirements guidelines.

     Maximum file size: 300KB

     Dimensions (recommended): 296x76 pixels

     File formats: JPG, GIF, PNG

6. To specify colors:
   1. Select the color box of the Header or Button background color that you want to edit. The color controls open so that you can select a new color.

      ![The screenshot shows the Header Background color picker.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/Header-background-color.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUwMjksInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.rloVqYYzh-RpB7tTw-oh4fq64af-c7E1A_35v2i-4l0&_LANG=enus "The screenshot shows the Header Background color picker.")
   2. Select a color using the color picker, use the eye dropper to match a color value in your custom logo or other image, or enter the hexadecimal value for the color.
   3. For text, you can select either black or white only.
7. Use the Preview views to check what recipients will see for both the signing experience and email notifications: ![The screenshot shows the Preview view of the Customize Signing & Email window.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/Preview-Customize-Signing-Email-logo.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUwMjksInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.rloVqYYzh-RpB7tTw-oh4fq64af-c7E1A_35v2i-4l0&_LANG=enus "The screenshot shows the Preview view of the Customize Signing & Email window.")
8. Select Apply Brand to close the window.
9. To save your changes, select Save.

The brand is added as tile on the Branding page. The signing brand is available for senders to select when they send an envelope. Brands can be applied to templates and envelopes generated by your account.
