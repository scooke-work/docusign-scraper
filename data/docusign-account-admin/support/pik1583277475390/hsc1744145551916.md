# Add Capacity for an IAM Plan

Learn how to add feature capacity to your IAM plan.

As an administrator of an existing IAM plan, you can add feature capacity to your account.

1. From the Accounts view, select Plan and
   Billing.

   Note: The Plan and Billing page appears with the title Subscription and Billing for some account types.

   The Subscription and Billing page
   opens.
2. In the Feature and extension usage section, review
   available features, feature use status, and quantity of feature remaining.![The screenshot shows the Feature and extension usage section on the Subscription and Billing page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/IAM-feature-and-extension-section.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ5MDUsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.A3ldasmvoX7JaH1Jx9gNkjBFzsx0TMOQcFuGVClB64Y&_LANG=enus "The screenshot shows the Feature and extension usage section on the Subscription and Billing page.")
3. Select Add [feature name] or Upgrade
   Plan in the Actions column for the feature that you want to add
   capacity.

   The feature purchase window opens.
4. Choose the quantity of the feature that you want to purchase.
5. Select Purchase.

   The confirmation window opens.
6. Select Done.

Additional feature capacity is purchased and added to your
account.
