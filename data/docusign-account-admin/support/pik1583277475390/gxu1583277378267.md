# Delete Custody Transfer Rules

Learn how account administrators delete custody transfer rules when they are not
needed.

You can delete custody transfer rules when they are not
needed.

1. From the Accounts view, select Custody
   Transfer.
2. Locate the rule that you want to delete.
3. Select Actions > Delete.
4. Select Delete to confirm the change.

The rule is deleted.
