# Edit Custom Fields for Envelopes

Learn how administrators edit custom fields for envelopes.

As an administrator, you can edit custom fields for
envelopes.

1. From the Accounts view, select Envelope Custom
   Fields.
2. Find the field that you want to edit, and select ACTIONS > Edit.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/editenvcustomfield.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjU2MTksInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.an5mnswvlSD5tHroScDqZoftwoCAIB-BgxRPfnQdXps&_LANG=enus)
3. In the Edit dialog box, change the field information as needed, and select
   Save.
4. In the Apply changes to template dialog box, select whether or not you want the change
   to be applied to any templates that use the field.
