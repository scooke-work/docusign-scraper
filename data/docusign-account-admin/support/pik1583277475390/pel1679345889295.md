# Use the Template Report to Find Templates Where a Removed User Is a Recipient or Signer

Learn how to use the template report to find templates that include a removed
user.

You can use the template report to locate the templates that have a removed user as a
recipient or signer. If a template includes a removed user as a recipient or a
signer, identify the templates that require updating and notify the template owner
to make the changes. For more information about reports, see the [Using Reports](https://support.docusign.com/s/document-item?bundleId=cnu1642812012214&topicId=wou1578456440763.html&_LANG=enus) topic.

1. Log in to your Docusign account, and select
   Reports on the global header navigation menu.
2. Navigate to Usage > Template Report.
3. Select a time period using the template creation date.
4. Download the report.

   Note: If you expect more than 50,000 responses, schedule
   the report or send it by email.
5. Open the downloaded file in a spreadsheet tool such as Excel.
6. Filter the Signer List and Other Recipient columns by the removed user to
   identify the templates that require updating.
