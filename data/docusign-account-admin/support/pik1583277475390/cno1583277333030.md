# Advanced Configuration for Signing Brands

Read about the advanced configuration options available to administrators that want to
customize the signing experience. Administrators can customize destination URLs, header and
footer links, and signing brand languages.

In addition to the logo and color customization you can apply to a [signing brand](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=lfr1583277366660.html&_LANG=enus "Branding offers account administrators a way to customize the Docusign experience for both senders and recipients."), you can use additional advanced
configuration options to modify the signing experience.
