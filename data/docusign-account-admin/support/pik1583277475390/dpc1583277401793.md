# Update Your Account Name and Address

Learn how administrators change the name of an account and the address associated with
it.

The account name is a global value for every user on your Docusign
account. The account name displays in notification emails to recipients and in the account
drop down menu. This value can only be changed by an account administrator.

1. From the Accounts view, select Account
   Profile. ![The screenshot shows Account Profile in the Account view.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/Account-Profile-menu-item.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ5NjksInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.rHQM6jTKpDUFANiVkRUJApZkbvmi2QPiA6tl4WatguU&_LANG=enus "The screenshot shows Account Profile in the Account view.")
2. To edit your account name:
   1. Select Edit next to your current account name.

      The Change Account Name widow opens.
   2. In the Change Account Name window, enter the new value, and
      select Change Name.![The screenshot shows the Change Account Name window.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/Edit-Account-Name-window.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ5NjksInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.rHQM6jTKpDUFANiVkRUJApZkbvmi2QPiA6tl4WatguU&_LANG=enus "The screenshot shows the Change Account Name window.")
3. To edit your account address:
   1. Scroll down to the Account Address section at the end of the Account Profile
      page.
   2. Change the values for the address fields as needed.

      All fields with a red asterisk (\*) must be completed.

      ![The screenshot shows the Account Address fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/Edit-Account-Address.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ5NjksInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.rHQM6jTKpDUFANiVkRUJApZkbvmi2QPiA6tl4WatguU&_LANG=enus "The screenshot shows the Account Address fields.")
   3. Select Save.

The changes are saved.
