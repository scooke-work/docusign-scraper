# Rename a Group

Learn how administrators edit the names of groups.

As an account administrator, you can edit the name of
groups.

1. From the Accounts view, select Groups.
2. Find the group that you want to rename.

   Note: You cannot edit the
   Everyone and Administrators groups.
3. Select Actions > Rename Group.
4. In Group Name, change the group name and select Save.
