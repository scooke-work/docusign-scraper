# Add Groups

Learn how administrators create groups to organize users into functional units.

As an account administrator, you can organize users into
groups.

1. From the Accounts view, select Groups.
2. Select Add Group.
3. Enter a name for the group.
4. Select Add.

   The group is created.
5. Select Assign Users.
6. Locate the account users that you want to add to the group by searching or viewing the
   list of all users.
7. Select the checkboxes next to the account users that you want to add to the
   Selected list.
8. Select Assign.

The users are added to the new group.
