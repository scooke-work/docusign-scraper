# Search for Users

Learn how account administrators quickly locate users by name or email address from the Users page. Filters help refine the search by limiting it to users by permission profile, user group, or status, such as active, pending, and closed.

As an account administrator, you can search for users by name or by email address from the Users page. You can use filters to refine your search and limit it to users by permission profile, user group, or status, such as active, pending, and closed.

1. From the Accounts view, select Users.
2. Enter any portion of a name or email address in the Search box.

   The matching users appear in the dropdown list.
3. Select the user that you want to view, or select Enter at the end of the dropdown list to view all matching users.
4. (Optional) Select Filter to restrict your search to users by permission profile, group, and status. Select the filter combination that you want to use, and select Apply.

   You can combine the filters to narrow your results, but you must have at least one option selected for Status. By default, Active and Pending are selected.

The users that meet your search selections are displayed.
