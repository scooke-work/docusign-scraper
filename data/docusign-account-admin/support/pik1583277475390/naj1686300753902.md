# Change the Order of Signature Types Available to Senders for Envelope Creation

Learn how to change the order of signature types in the list available to senders.

## Before you begin

Ensure you can access the account administration settings before
changing the order of signature types.

As an account administrator, your organization requires that you change the order of signature
types available to senders.

1. From the Accounts view, select Signature
   Types.
2. Select Edit.

   The Manage Signature Type List screen opens.
3. Select a signature type and move it to the position of your choice in the
   list.

   Moving a signature type to the top of the list designates it as the
   default choice for signature type selection. Default signature types apply to
   signers unless senders select a distinct signature type.
4. Select Save.

   A confirmation message shows on the Signature Types
   screen.

The list of signature types updates with your changes.
