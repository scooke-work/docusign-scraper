# Bulk Add New Users

Read about how organization administrators can bulk add new users to one or more
accounts.

Note: This guide is for Docusign
organization administrators who oversee multiple accounts. For administrators of
individual accounts, see the [Add and Update Users in
Bulk](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=mxm1583277431469.html&_LANG=enus "Read about how administrators add or update multiple users simultaneously through the Bulk Add and Bulk Update options. To perform bulk actions, administrators upload comma-separated value (CSV) files that contain updates or new user information.") topic.

Docusign organization administrators with the Administrators and Users
Administrators permission profiles can bulk add new users to one or more accounts by
uploading a comma-separated value (CSV) file. The format of the CSV must match the [sample file](https://admin.docusign.com/static-resources/organization-user-import.csv). For more information about the CSV format, see
[Build a CSV to Bulk Add Users](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=wec1593211169952.html&_LANG=enus "Read about how organization administrators can use a CSV file to add multiple users.").

You can add up to 2,000 users to an account and include up to 50 accounts per
imported CSV. The maximum number of users per import is 8,000.

Note: Only one type of import or export can be in
progress at a time.
