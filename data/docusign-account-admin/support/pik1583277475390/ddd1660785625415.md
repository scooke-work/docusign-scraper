# Share Access with Other Users

Learn how administrators manage shared access to envelopes among users.

As an administrator, you can allow users of an account
to share access to envelopes with other users on the account.

1. From the Accounts view, select
   Users.
2. Locate the user that you want to manage sharing settings for, and select Actions > Manage Shared Access.
3. Select Share Access with Others.
4. Select one of the following permissions, and select
   Next:

   - Allow to manage on behalf of [User Name]: Allows
     the selected user to view envelopes, view history, download files, and
     move envelopes into folders.
   - Allow to edit on behalf of [User Name]: In
     addition to the manage permissions, allows the selected user to correct,
     resend, and void the user's envelopes.
   - Allow to send on behalf of [User Name]: In
     addition to the manage and edit permissions, allows the selected user to
     send envelopes, edit drafts, resend, void, and correct in addition to
     managing on the user's behalf.

   Note: Depending on your account plan, you might not have
   access to all permissions.

   Users that do not have sending permissions can only be granted
   access to manage another user’s envelopes. Inactive users or users pending
   activation cannot be granted access and cannot have their envelopes accessed
   by other users.

   The send permission allows others to send and manage envelopes on
   behalf of the user, whereas the manage permission allows others to manage
   envelopes on behalf of the user but not send.
5. Select the start and finish dates to control the duration of the shared
   access.

   The start date can be the current date or any future date. An end date is
   optional and can be any date equal to or after the start date. If an end date is
   specified, the access will expire on that date without any manual action needed.
   If you do not specify an end date, the access continues until someone manually
   removes it. ![The screenshot shows Set Duration for Shared Access.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/Share_Access_with_Other_Users_2.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUyMTEsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.Rlami54MP26pStbOjUXJG_ApgTfQvs5cAJHsEzbYycE&_LANG=enus "The screenshot shows Set Duration for Shared Access.")
6. Select Next.
7. Select the users for the shared access permission.

   To locate users more easily, use the Search option. You can select up
   to 100 users to share access to a user's envelopes.

   If users are not available for selection, those users have shared access. To
   update their permission, remove their current access.

   You can view which users have shared envelopes access to this user. This is
   helpful if you want to create similar access using the shared access
   feature.

   ![The screenshot shows Select Users for Shared Access.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/Share_Access_with_Other_Users_3.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUyMTEsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.Rlami54MP26pStbOjUXJG_ApgTfQvs5cAJHsEzbYycE&_LANG=enus "The screenshot shows Select Users for Shared Access.")
8. Select Save.

The users you selected have shared access to the envelopes of the
current user.
