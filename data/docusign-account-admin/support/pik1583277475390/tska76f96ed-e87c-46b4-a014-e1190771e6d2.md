# View Feature Usage

As an administrator, you can review feature usage information to track feature status, usage, and increase feature quantity.

## Before you begin

1. From the Accounts view, select Plan and Billing.
2. View your reports in the Feature usage section.

   ![The screenshot shows the Feature usage section on the Plan and Billing page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/featureusage.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ5MTcsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.wqaCV5RtMqf5tL4qcNv4GiRGE6en2yfY2L-IDacZ-E0&_LANG=enus "The screenshot shows the Feature usage section on the Plan and Billing page.")
3. Select view your reports for a detailed and filterable summary of your usage.

You can view your feature usage, learn more about each feature, and increase feature quantity.
