# Default Mode for Signature Adoption

Read about the signing modes that administrators can choose as a default for the
Adopt Your Signature dialog box.

This option sets which of the available signing modes is shown first when a signer is
adopting their signature or initials. If you prefer an option, but have not ruled out the
others, you can set the preferred option to display when the Adopt Your Signature dialog box
displays. Recipients can still select any of the other signing modes configured for the
account.

- Start on adopt shows the Select Style mode, with the different signature styles.
- Start on draw shows the Draw mode, where the signer draws their signature or initials.
- Start on upload shows the upload mode, where the signer can upload an existing file with their signature or initials.

Separate settings are available for Docusign web app and mobile web signers.
