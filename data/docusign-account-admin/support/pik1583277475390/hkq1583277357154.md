# Minimum Pixel Size

Read about the importance of administrators specifying a minimum pixel size for
signatures.

To ensure that drawn signatures and initials are visible on completed documents, set a
reasonable minimum pixel size. This minimum is applied to drawn signatures only, and does not
apply to uploaded images.
