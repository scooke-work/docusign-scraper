# Transfer Envelopes and Templates in Bulk with a CSV File

Learn how account administrators use a CSV file to transfer envelopes and templates in
bulk.

As an account administrator, you can transfer envelopes and
templates in bulk by using a comma-separated value (CSV) file.

1. Open the [downloaded CSV file](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=ngj1583277448218.html&_LANG=enus "Learn how account administrators download information about envelopes and templates in CSV files to help them make decisions about actions to take on them.") and remove all data
   except the EnvelopeID or Template ID column with each ID for the items you want to
   transfer.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/transfercsvupload.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjU1MzksInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.F1f_9vb2wI5M5nE2f17STmrUcs7m3ZAA_QskJW_vBOE&_LANG=enus)
2. From the Accounts view, select Custody
   Transfer.
3. Select Transfer and select either
   Envelopes or Templates.
4. Select Transfer Using CSV.
5. In the Transfer Using CSV dialog box:
   1. Select Upload .CSV File and select the prepared CSV
      file.
   2. Select the user that the items are being transferred to and select
      Transfer.
   3. (Optional for envelopes only) Check Add Original Owner as Carbon
      Copy to add the original sender as a “Receives a copy” recipient for the
      envelopes. If you do not select this option, the original sender loses all access to
      the transferred items.

      ![The screenshot shows the Upload CSV window.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/transfercsvuploadfinish.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjU1MzksInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.F1f_9vb2wI5M5nE2f17STmrUcs7m3ZAA_QskJW_vBOE&_LANG=enus "The screenshot shows the Upload CSV window.")
6. Select Confirm to confirm the transfer.

   The transfer job is placed in a queue for processing and listed in the Bulk Actions
   log. This log shows a list of bulk actions for your account and the progress of the
   actions.

   When the transfer is complete, you can select Details
   to review the transfer information and download a copy of the CSV file with the transfer
   information. After the job is completed, the transferred items are moved to the new
   owner's account.
