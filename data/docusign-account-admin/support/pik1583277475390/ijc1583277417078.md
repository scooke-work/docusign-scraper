# What the Signer Sees When Using Stamps

Read about how users interact with stamps.

**Opening the envelope**

When a recipient opens an envelope that contains a stamp, they see the stamp field in the
location where it was placed.

![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/stampsigningfield.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUwODAsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.B_MPWDRlZw2H4WV3DiZs1v7dZUtRwa4ySvU1kY0sU0o&_LANG=enus)

**Clicking on the stamp**

Clicking on the stamp field displays a dialog box where the recipient can either choose an existing stamp or assign a new one.

**Selecting a stamp**

The recipient can select a stamp and select DONE to apply that stamp
to the document.

![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/stampssigningselect.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUwODAsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.B_MPWDRlZw2H4WV3DiZs1v7dZUtRwa4ySvU1kY0sU0o&_LANG=enus)

The stamp is applied to the document.

![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/stampstimedatesigning.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUwODAsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.B_MPWDRlZw2H4WV3DiZs1v7dZUtRwa4ySvU1kY0sU0o&_LANG=enus)

**Stamp options**

The recipient can modify the stamp before completing the envelope through several
options.

- **Rotate** - Rotate the stamp.
- **Custom Date** - Change the date on the stamp (only for **Name + Date**
  format stamps). The date must be in 'YY.MM.DD format.
  ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/stampschangedate.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUwODAsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.B_MPWDRlZw2H4WV3DiZs1v7dZUtRwa4ySvU1kY0sU0o&_LANG=enus)
- **Create New** - Replace the stamp with a new one.
- **Choose From Existing** - Replace the stamp with an existing stamp.
- **Remove** - Remove the selected stamp.

![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/stampssigningoptions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUwODAsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.B_MPWDRlZw2H4WV3DiZs1v7dZUtRwa4ySvU1kY0sU0o&_LANG=enus)

If the recipient selects either the **Create New** or **Add New** option in the
**Existing Stamps** dialog box, a new dialog box displays to generate a new stamp.
