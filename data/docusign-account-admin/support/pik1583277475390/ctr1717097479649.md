# Review Seat Requests From Users

Learn how to review and confirm or deny new seat requests.

When a user requests to join an account through the Account Discovery feature, you
receive an email to review and confirm or deny the request.

1. From the Accounts view, select
   Users.

   Note: For some plans, the Users page is
   titled Seats & Users.

   ![The screenshot shows the Users page with new requests to join the account.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/seatmgmt2.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ5NTQsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.N8Vuny6fjnDPglglmFcBPWMOkFBzFPbEqoaKyC0lxp0&_LANG=enus "The screenshot shows the Users page with new requests to join the account.")
2. In the User Requests section, review any requests awaiting
   approval.
3. Select Confirm or Deny to accept
   or decline seat requests.

   This action processes the request and removes the request from the list.

   Note: Depending on your plan, you can purchase more seats if
   needed.

New users are added to the Users list. Refresh the page to
view the updated list of users. Approved users are assigned a seat on the account and
are assigned the DS Sender permission.
