# Assign Users to Stamps

Learn how administrators assign individual users to a stamp.

As an administrator, you can assign individual users to
stamps.

1. From the Accounts view, select
   Stamps.
2. Select Actions > Users and Groups.
3. Select Assign User.
4. Select the checkboxes next to the users that you would like to assign access to this
   stamp.
5. Select Assign.
