# Email Preferences

Read about how administrators set default email notification preferences for new users
when they are added to an account. Users can change these settings in their My
Preferences. API User settings apply to envelopes sent
using an API integration.

These settings are the default email notification preferences for all new users added to your
account. Users can change these settings in their preferences, as described in the [Notification Preferences](https://support.docusign.com/s/document-item?topicId=dhc1578456326388.html&bundleId=jux1643235969954&_LANG=enus) guide. Changes to these defaults do not affect existing users.

The API User settings apply to envelopes sent using an API integration.
