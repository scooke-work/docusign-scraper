# Reset User Passwords with the AI Agent

Learn how to use the Docusign AI Agent to reset user passwords, resend invitations, and reactivate users.

Note: This feature is currently available only to account administrators on digital plans, either paid or trial, in the US region and the English locale.

1. In the Accounts view, hover over the AI icon and select Try the Docusign AI Agent.

   ![The screenshot shows the Try the Docusign AI Agent option on the Overview page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/ai-agent-pw1.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUxNjUsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.aq35LjQjIpPLaY7e_Xorhap8UZWKiPxL5yJhbR1LmB0&_LANG=enus "The screenshot shows the Try the Docusign AI Agent option on the Overview page.")

   The AI agent chat window opens.
2. Select the Reset a user’s password quick action.

   ![The screenshot shows the Reset a user's password quick action.](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/ai-agent-pw2.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjUxNjUsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.aq35LjQjIpPLaY7e_Xorhap8UZWKiPxL5yJhbR1LmB0&_LANG=enus "The screenshot shows the Reset a user's password quick action.")
3. Follow the prompts from the AI agent and provide the user name and email address.

   You can provide multiple users in the same request.
4. Review the user results found by the AI agent, and confirm if you want to continue with the password reset.

   If the user is pending, closed, not found, or ineligible due to SSO, follow the alternative steps provided by the AI agent. For example, you might need to resend the invitation, reactivate the user, or add a new user.
