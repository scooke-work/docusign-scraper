# Customize your Docusign ID Verification with IC-chip Reading for Japan Configuration

Learn how to copy and edit the read-only Docusign ID Verification with IC-chip Reading for Japan configuration to meet your specific requirements.

## Before you begin

To customize the Docusign ID Verification with IC-chip Reading for Japan configuration with your specific needs, you must have access to the read-only configuration. For more information about the requirements for using this configuration, read the [ID Verification with IC-chip Reading for Japan Configuration](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=con136df38a-18a8-4c57-9d7c-69d9e8e9d9cd.html&_LANG=enus "Read about the Docusign ID Verification with IC-chip Reading for Japan read-only configuration. Learn about its configuration and the parameters that you can edit in your customized configuration.") page.

You are the administrator of a Docusign account hosted in Japan that already has access to the read-only Docusign ID Verification with IC-chip Reading for Japan configuration. As your organization wants to implement its own configuration, you must copy the default configuration in order to customize it to fit your specific requirements. If you want to start setting the configuration from scratch, you may add it from the Add Configuration module.

1. From the Accounts view, select Identity Verification.
2. Locate the read-only Docusign ID Verification with IC-chip Reading for Japan configuration.
3. Select Create a Copy.
4. Customize the available settings for this configuration.

   For details on which settings you can change, read the [ID Verification with IC-chip Reading for Japan Configuration](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=con136df38a-18a8-4c57-9d7c-69d9e8e9d9cd.html&_LANG=enus "Read about the Docusign ID Verification with IC-chip Reading for Japan read-only configuration. Learn about its configuration and the parameters that you can edit in your customized configuration.") page.
5. Select Add.

   ID Verification saves your configuration.
