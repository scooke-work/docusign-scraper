# Signing Brand Company Name

Read about how administrators display a company name that is associated with a brand in
the ERSD.

If your account uses multiple signing brands, you can specify a company name in the brand that
can be used in the ERSD, as in the following example. For more information about brands, see
the [Configure Brands](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=lfr1583277366660.html&_LANG=enus "Branding offers account administrators a way to customize the Docusign experience for both senders and recipients.") topic.

![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pik1583277475390/page/images/Signing_Brand01.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjU1NjUsInNoZWFmIjoicGlrMTU4MzI3NzQ3NTM5MCJ9.L7m3njIIe5l4OHpFCsogsw0gz2vgBSdW7-wp0R_Shts&_LANG=enus)

If you set the controlling option on the Legal Disclosure settings, each time that
the field for CompanyName is used in the ERSD, it will be replaced by the company name
specified in the brand. This substitution only works if you use the default legal disclosure
or if you apply the CompanyName the merge fields in a custom ERSD.
