# User Lists and Permission Profiles Downloads

Read about how administrators download assigned permission profiles and permission
settings for account users as a comma-separated value (CSV) file. Administrators can download
just the permission profile name or a full list of permissions for users.

As a Docusign account administrator with User delegated permissions or
greater, you can download a list of the users on your account and their assigned permission
profiles. You can optionally include a full list of the permission settings and their values
through the Include user permissions option.

The downloaded comma-separated value (CSV) file includes the following information:

- Email ("UserEmail")
- Status ("Status")
- User First Name ("FirstName")
- User Last Name ("LastName")
- Job Title ("UserTitle")
- Company Name ("CompanyName")
- Permission Profile ("PermissionSet")
- Added Date ("AddedDate")
- API User Name ("APIUserName")
- Groups ("Group")
