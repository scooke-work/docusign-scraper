# Set Account Reminders and Expiration Defaults

Learn how account administrators set default reminders and expiration time frames for
envelopes sent from an account.

As an account administrator, you can set reminders and
expiration time frames for envelopes sent from an account.

1. From the Accounts view, select Reminders and
   Expirations.
2. To allow users to change the account defaults on the envelopes they send, select
   Allow senders to override account defaults.
3. To enable and set automatic reminders:
   1. Select Send automatic reminders.
   2. Enter values for how many days to wait until sending the first reminder and how
      many days to wait between subsequent reminders. Use integers from 1-999.

      If you enter 0 for the number of days between reminders, no reminders will be sent
      after the first one.
4. For envelope expiration, enter how many days after sending an envelope expires and how
   many days before expiration to warn signers. Use integers from 1-999.
