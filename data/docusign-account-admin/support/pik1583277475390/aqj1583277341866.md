# Custom Logos for Sending Brands

Read about how administrators create a sending brand with logos and colors.
Administrators can customize the Docusign environment for senders.

When you create a custom sending brand, you can add logos and specify colors. You can add
your logo to the sending experience for your users see when using their Docusign account. The sending brand also defines the email logo in the user account activation email.

If you modify the header color for the sending brand, the color is applied to the Welcome back banner
area on the Home page.
