# Define Signing Settings

Learn how administrators define the signing behavior for an account.

As an administrator, you can define the signing behavior for
an account.

1. From the Accounts view, select Signing
   Settings.
2. Adjust your signing settings as needed.

   See [Signing Experience Settings](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=rgk1583277353500.html&_LANG=enus "Read about how administrators use the In-session Landing Pages and Watermark Configuration settings to specify two key behaviors of the signing experience. These settings define where signers are routed in different situations and they ensure that in-process transactions clearly reflect an incomplete signing status.") for descriptions of the signing
   settings.
3. Select Save.
