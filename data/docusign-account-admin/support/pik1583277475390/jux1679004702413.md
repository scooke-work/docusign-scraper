# Find Templates Where a Removed User Is a Recipient or Signer

Learn how to find templates that include a removed user.

You can use filters on the Templates page to locate the templates that have a
removed user as a recipient or signer. If a template includes a removed user as a
recipient or a signer, identify the templates that require updating and notify the
template owner to make the changes. For more information about templates, see [Working with Templates](https://support.docusign.com/s/document-item?bundleId=xry1643227563338&topicId=dqj1578456412286.html&_LANG=enus).

For information on using a report to find a removed user, see [Use the Template Report to Find Templates Where a Removed User Is a Recipient or Signer](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=pel1679345889295.html&_LANG=enus "Learn how to use the template report to find templates that include a removed user.").

1. Log in to your Docusign account, and select
   Templates on the global header navigation menu.
2. In the top right of the Templates page, select
   Filters.
3. Select Recipient Name & Email in the Select Field
   dropdown.
4. Enter the email address of the removed user in the search field.
5. Select Apply.

The results show the templates in the account that include
the removed user.
