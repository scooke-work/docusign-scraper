# Delete a Custom Status

Learn how to delete a custom status that you created when you no longer need it.

You can delete a custom status that you created when you no longer need it.

1. Select Requests in the left navigation.
2. Select Settings > Statuses.
3. Select the trash can to the right of the status you want to delete.

   ![A screenshot showing how to delete a Agreement Desk custom status.](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapf0ca9167-3825-4b36-a385-e98321f7adb5/page/images/ac-del-status.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI1NjYsInNoZWFmIjoibWFwZjBjYTkxNjctMzgyNS00YjM2LWEzODUtZTk4MzIxZjdhZGI1In0.Js3kX_p7S8g_Ej2tqC3L6NYV0dpNj-IeQosg7C76miM&_LANG=enus "A screenshot showing how to delete a Agreement Desk custom status.")

   The Delete status? dialog box opens prompting you to confirm the
   action.
4. Select Yes, delete.

The custom status is no longer visible in the list of statuses.

Note: If you applied a deleted status to an active request, the
request keeps that deleted status until changed by a person with access. The deleted
status can no longer be applied to other requests.

For example, a request has the
legal review status, and someone deletes
legal review as a status. Any active requests that
have legal review as a selected status continue to show
legal reviewas a status even though that status was
deleted.
