# Configure Notifications

Learn how to use the Notifications page to select the types of Agreement Desknotifications users receive after they submit a request. Only users in your account can
receive notifications. External users do not receive notifications.

Use the Notifications page to select the types of Agreement Desk
notifications users receive after they submit an Agreement Deskrequest.
Only users in your account can receive notifications. External users do not receive
notifications.

There are two types of notifications in Agreement Desk:

Submitter notifications
:   Account Admins control Submitter notifications. These
    notifications can only be sent to users within the account. There are
    two types of submitter notifications:

    - Confirmation email when a request is submitted
    - Update the email with changes to the submitter, owner, due date,
      title, and status

    Both types are turned on by default. The admin can turn them off at any
    time.

    Note:

    When submitting requests through email, only users within your
    account receive confirmation email. External users do not receive
    this notification.

Follower notifications
:   Any account user with access to a request can select
    Follow in the top-right corner to receive
    email notifications. These provide updates and modifications made to
    that request.

    You can configure Follower notifications for each user as
    described in the [Manage Notification Preferences](https://support.docusign.com/s/document-item?bundleId=jux1643235969954&topicId=dhc1578456326388.html&_LANG=enus) topic.

1. From the Docusign Admin page, select
   Admin.
2. Select your Agreement Desk account.
