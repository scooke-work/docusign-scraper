# Create a New Request Type

Learn how to

You can create new Agreement Deskrequest types tailored for your
organization's specific needs. Request types are based on Docusign
Web Forms.

1. Select Requests in the left navigation to access the
   Request Dashboard.
2. Select Settings.
3. On the Request type tab, select  +Create Request
   Type.

   ![A screenshot of a sample Agreement Desk Create Request Type dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapf0ca9167-3825-4b36-a385-e98321f7adb5/page/images/ad-create-req-type.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI1MjYsInNoZWFmIjoibWFwZjBjYTkxNjctMzgyNS00YjM2LWEzODUtZTk4MzIxZjdhZGI1In0.hP59hMYSpxIVt-WaqVEAvDuENIcNVNL5uDyfJnbj1Ck&_LANG=enus "A screenshot of a sample Agreement Desk Create Request Type dialog box.")
4. Enter the request type name and optional description and select
   Next.
5. Complete the required fields for your form.

   For more information about using Web Forms to complete the request see the [The Web Forms Builder](https://support.docusign.com/s/document-item?bundleId=gmi1660583110357&topicId=gpc1660591510274.html&_LANG=enus)  article.

   Here is a sample request type.

   ![A screenshot of a sample of a new Agreement Desk request type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapf0ca9167-3825-4b36-a385-e98321f7adb5/page/images/ad-adm-webform-ex.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI1MjYsInNoZWFmIjoibWFwZjBjYTkxNjctMzgyNS00YjM2LWEzODUtZTk4MzIxZjdhZGI1In0.hP59hMYSpxIVt-WaqVEAvDuENIcNVNL5uDyfJnbj1Ck&_LANG=enus "A screenshot of a sample of a new Agreement Desk request type.")
6. When you finish completing the required fields, select
   Activate to make the request public.

   Users can access the request type when creating and modifying requests.
