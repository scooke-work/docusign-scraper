# Grant Agreement Desk Permissions to Users

Learn how Docusign administrators grant specific Agreement Desk permissions to other
users.

## Before you begin

You must have administrator privileges to grant Agreement Desk permissions.

For information about Docusign permissions, see the topic [Permission
Profile Options](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=mfo1583277360424.html&_LANG=enus)﻿.

.

Docusign account administrators can grant specific Agreement Desk permissions to other
users.

1. Select the Admin tab in the top Docusign
   navigation.

   The Admin page opens.
2. Select Users in the USERS AND
   GROUPS section.
3. Locate the user you want to grant access to and select
   Actions in the far right of the user name.
4. Select Manage Agreement Desk access.

   The Agreement Desk Access window opens.

   ![A screenshot of the Agreement DeskAccess page for a user.](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapf0ca9167-3825-4b36-a385-e98321f7adb5/page/images/ad-adm-grant-user-access.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI1MTksInNoZWFmIjoibWFwZjBjYTkxNjctMzgyNS00YjM2LWEzODUtZTk4MzIxZjdhZGI1In0.J3tDbJfrU7BdmlMua8ywcD1Jn6TLaH7RVzBBaTVChSY&_LANG=enus "A screenshot of the Agreement DeskAccess page for a user.")
5. Select the toggle for either or both of the following options and select Save.
   - Manage all requests in the account
   - Configure Agreement Desk settings
