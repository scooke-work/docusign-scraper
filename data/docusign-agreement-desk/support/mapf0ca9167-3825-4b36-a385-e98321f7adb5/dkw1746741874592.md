# Restore a Deleted Custom Status

Learn how to restore a custom request status that you deleted.

If after deleting a custom request status, you find that you need it, you can restore it.

1. Select Requests in the left navigation..
2. Select Settings  > Statuses.
3. Locate the status that you want to restore from the
   Deleted section. Select the arrow to the right.

statuses.
