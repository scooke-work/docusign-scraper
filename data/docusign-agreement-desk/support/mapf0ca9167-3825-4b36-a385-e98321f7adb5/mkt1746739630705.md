# Create a Custom Request Status

Learn how to create a custom status for Agreement Deskrequests.

If you need a status other than the default options provided by Agreement Desk, you can create one.

1. Select Requests to open the Requests dashboard.
2. Select Settings.
3. Select the Statuses tab.
4. Select Add Status.
5. Enter a name for the status.
6. (Optional) Select Closed for the Status type if the status applies to closed requests only.
7. Select an accent color and select Add.

The status you created is visible beneath its status type in the Statuses page.
