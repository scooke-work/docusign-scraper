# Create a New Request Status

Learn how to create new Agreement Desk request types tailored for your
organization's specific needs. Agreement Desk request types are based on Web
Forms.

Create new Agreement Desk request types tailored for your organization's
specific needs. Request types are based on Docusign Web Forms.

1. Select the Statuses tab of the
   Settings page.
2. Select + Add Status.

   ![A screenshot of a sample Agreement Desk Add new status dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapf0ca9167-3825-4b36-a385-e98321f7adb5/page/images/ad-adm-new-req-stat.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI1NTksInNoZWFmIjoibWFwZjBjYTkxNjctMzgyNS00YjM2LWEzODUtZTk4MzIxZjdhZGI1In0.8qHjeLbp_FOwg66nKORM8Hj2hEqeahnnT-t0y2o9tNQ&_LANG=enus "A screenshot of a sample Agreement Desk Add new status dialog box.")
3. Enter a name for the status.
4. In the Status type menu, select whether this status
   should apply to Open or Closed requests.
5. Select an accent color and select Add.

   Note:

   You can make these modifications:

   - Modify the name and accent color for a status by selecting
     Edit.
   - Remove a status by selecting Delete. Deleted
     statuses remain visible under the heading
     Deleted.
6. Use the handle icon on the left side of each status to drag the status to
   re-order.
