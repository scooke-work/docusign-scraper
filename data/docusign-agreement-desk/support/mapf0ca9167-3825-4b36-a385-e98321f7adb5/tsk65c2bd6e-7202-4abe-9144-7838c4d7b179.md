# Turn on Agreement Desk for Specific Users

Learn how to turn on Agreement Desk for specific users so they can view
requests and manage request settings.

Turn on Agreement Desk for specific users so they can view requests
and manage request settings.
​
​
​

1. From the Admin page, select Users.
2. For the user you want to enable, select Actions >  Manage Agreement Desk Access.

   ![A screenshot of the User Action menu Manage Agreement Desk Access selection.](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapf0ca9167-3825-4b36-a385-e98321f7adb5/page/images/ad-adm-user-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI1MTQsInNoZWFmIjoibWFwZjBjYTkxNjctMzgyNS00YjM2LWEzODUtZTk4MzIxZjdhZGI1In0.7tCWqxayksPVYvOrbikoF1EUkixC8RupuGZMNOrcWKs&_LANG=enus "A screenshot of the User Action menu Manage Agreement Desk Access selection.")

   The Agreement Desk page displays
3. Turn on these permissions:
   - Manage all requests in account: View and manage
     all requests.
   - Configure Agreement Desk
     settings: Create request types, statuses, and access email
     settings.

4. Select Save to apply your changes.
