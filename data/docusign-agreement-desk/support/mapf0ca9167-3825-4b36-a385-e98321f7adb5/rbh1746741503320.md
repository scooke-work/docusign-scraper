# Edit a Custom Status

Learn how to edit a custom status that you created.

You can edit a status that you created.

1. Select Requests in the left navigation.
2. Select Settings > Statuses
3. Select Edit to the right of the status that you want to
   edit.

   The Edit status
   page
   opens.

   ![A screenshot of the Agreement Desk Edit status dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapf0ca9167-3825-4b36-a385-e98321f7adb5/page/images/ad-edit-status.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI1NjEsInNoZWFmIjoibWFwZjBjYTkxNjctMzgyNS00YjM2LWEzODUtZTk4MzIxZjdhZGI1In0.ILvXzt9khMnQ6f0wNj3I7hixSmvIRlOY6AkTWRBhUU0&_LANG=enus "A screenshot of the Agreement Desk Edit status dialog box.")
4. Make changes to the status name or accent color, and select
   Save.

The custom status reflects the edits that you made.
