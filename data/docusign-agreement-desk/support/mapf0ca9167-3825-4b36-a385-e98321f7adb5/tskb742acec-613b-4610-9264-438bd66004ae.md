# Duplicate a Request Type

Learn how to use existing request types to save time when you create new request
types.

As an admin, you
may
need to create many request types. You can quickly duplicate
an existing
request
type to change the specifics to fit the new request type.

When you duplicate a request type:

- You duplicate the original request type's name appended by a number, description, and request form.
- the new request type has a draft status.

1. Select Requests in the left navigation to access the
   Request Dashboard.
2. Select Settings.
3. Select the overflow (three dot menu) next to the request type that you want to
   duplicate and select Duplicate.

4. Change the request type name and description and select
   Next.
5. Optionally change the required fields for the associated form.
6. When you finish completing the required fields, select
   Activate to make the request public.

   - Users can access the request type when creating and modifying
     requests.
   - The status of the new Request form and type will be
     Draft.
