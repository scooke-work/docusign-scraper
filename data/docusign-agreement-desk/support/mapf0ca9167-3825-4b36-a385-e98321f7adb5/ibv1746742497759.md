# Reorder Request Statuses

Learn how to reorder the statuses in the Agreement Desk requests Status menu. The status at the top of your list is automatically set as the default for all newly created requests. This default can be easily changed; simply drag your desired status to the top of the list to make it the new default.

You can change the order that statuses are listed in the Agreement Desk
request Status menu.

The status at the top of your list is automatically set as the default for all newly created requests. You can change the default by dragging your desired status to the top of the list to make it the new default.

1. Select Requests.
2. Select Settings > Statuses.

3. Select and hold the verticle lines on the left of a status and drag it to a new
   location in the list.
