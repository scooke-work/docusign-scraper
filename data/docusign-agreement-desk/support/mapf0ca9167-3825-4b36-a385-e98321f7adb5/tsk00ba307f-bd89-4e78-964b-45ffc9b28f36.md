# Block Sending for Signatures for Outstanding Approvals

Learn about how Agreement Desk administrators can restrict users from
manually sending documents for signature in Agreement Desk. Use this feature if approvals
are still pending or have been rejected.

Agreement Desk administrators can restrict users from manually
sending documents for signature in Agreement Desk.

This feature helps enforce that documents are only sent for signature after
stakeholders have officially approved.

Administrators can enable or disable this feature within Agreement Desk Admin settings.

1. Select Requests in the left navigation to access the
   Request Dashboard.
2. Select Settings > Admin.
3. In the  Approvals for sending a document for Signature
   section, toggle on the  "Send for Signature" in Agreement Desk if
   approvals are pending or rejected slider to enable blocking
   signatures.

   This feature is turned off by default.
