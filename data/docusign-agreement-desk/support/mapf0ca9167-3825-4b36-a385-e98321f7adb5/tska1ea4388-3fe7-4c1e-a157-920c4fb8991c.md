# Specify a Default Request Type

Learn how to specify the request type that is automatically applied to requests
submitted by email to the Agreement Desk inbound email
address.

Specify the request type that is automatically applied to requests submmitted by
email to the Agreement Desk inbound email
address.

Sample Request Types

All Agreement Desk accounts come with these four sample
request types. You can use them as is, modify, or delete them.

- New Customer Intake (Default)
- General Legal Request
- NDA Request
- Vendor Contract Request

1. Select Settings from the Requests
   page.

   The Request types tab displays a list of all of
   your request types.
2. Select the overflow (three dot menu) next to the request type that you want to
   use as the default.

   ![A screenshot of a sample Agreement Desk default request type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/mapf0ca9167-3825-4b36-a385-e98321f7adb5/page/images/ad-adm-def-req.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI1MjksInNoZWFmIjoibWFwZjBjYTkxNjctMzgyNS00YjM2LWEzODUtZTk4MzIxZjdhZGI1In0.Vj6XNR28zSBlP8oDGM1ee9RZLnkDrFFWJ9RlS79dmxU&_LANG=enus "A screenshot of a sample Agreement Desk default request type.")
3. Select Set as default.

   Note:

   The request type must be active to become the default.

   The new default request type moves to the top of the list with
   (Default) next to it.
