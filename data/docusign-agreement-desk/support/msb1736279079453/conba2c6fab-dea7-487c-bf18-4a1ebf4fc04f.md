# Use Custom Request Identifiers

Learn about custom Agreement Desk request identifiers that help you improve
discoverability, simplify customer communication, and improve overall request
tracking.

Request identifiers help you better organize and search for requests in Agreement Desk. This enhances discoverability, simplifies customer
communication, and improves overall request tracking.

## Default request identifiers format

As an Admin, you can customize the request prefix (REQ). Agreement Desk automatically provides the numbers starting at 0001 going
upward for each newly created request.

- Default Identifier: REQ-xxxx
  - Prefix: REQ

    Letters A-Z
  - Number: #### ( This cannot be changed.)

Example: REQ-0603

## Customize your request prefix

Account Admins can optionally configure a meaningful identifier prefix to
replace the default REQ.

To ensure system compatibility and clarity, follow these request identifier
format guidelines.

- Prefix name
  - **Length**: Must be from **0 and 5** letters
    long
  - **Acceptable Characters**: Only the letters **A through
    Z** (uppercase or lowercase)
  - **Unacceptable Characters**: Symbols, special
    characters, or numbers
- Numbering

  Agreement Desk automatically provide the
  sequential numbering suffix, starting at 0001.
