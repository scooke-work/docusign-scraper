# Agreement Desk Request Reports Dashboards

Learn about the enhanced request report and the available dashboards that you can use for
your request data analysis.

The Agreement Desk Request Reports in the Reports tab now have
enhanced functionality. You can apply filters, view detailed reports, and download
data.

Docusign
Agreement Desk Reporting also provides deep insights into your overall
requests.

## Navigation

You access request reports by selecting Insights in the global
navigation bar and select Requests from the left side
navigation menu.

## Available dashboards

You can toggle the default chart display to a table. Accessing the chart's overflow
menu and select View as table.

## Understand request statuses

To effectively analyze your team's efficiency, it helps to understand how the chart
processes different stages of a ticket's lifecycle.

- **Custom Statuses:** Workspace administrators can define tailored stages
  to match your team's workflow. For example, Legal
  Review, Ready for Review,
  On Hold, or Ready for
  Signature.
- **Chart Logic:** This chart only displays data for
  Closed requests. It retroactively calculates the
  exact number of days a completed request spent in each of its previous
  **Open** custom statuses. Active or open requests are excluded from
  this chart.
