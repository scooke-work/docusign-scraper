# Mark Agreement Desk Request Documents as Complete

Learn how to upload completed PDF documents signed on external platforms and manually
mark them complete (signed) within the request. This also enables viewing the completed
document in Agreement Manager.

As an Agreement Desk user, you may frequently collaborate with
external parties who may use third-party eSignature software (such as Adobe,
PandaDoc, SignNow) to finalize agreements.

Previously, only PDF documents signed directly through Docusign automatically flow from Agreement Desk into
Agreement Manager.

Now you can upload completed PDF documents signed on external platforms and
manually mark them complete (signed) within the request. This also enables viewing
the completed document in Agreement Manager.

Note:

Only PDF files can be manually marked as completed.

When you manually mark documents as complete:

- It replaces the most recent entry in the document version history.
  It also provides a link to the manually completed signed document in
  Agreement Manager (This is the current behavior with documents signed in Docusign).
- The activity is tracked in the request’s activity feed for audit
  purposes.

1. Select Requests from the left navigation area to open the Requests
   dashboard.
2. Select and open a request.
3. Select the Documents tab.
4. For the document you want to mark as signed, select Mark Document
   as Complete from the vertical three dot menu.

   ![A screenshot showing the Mark document as complete dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-mark-complete-db.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzOTUsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.OQiKP9ZbpnH6PszR3vvCnkpI2-e3JUprJFdTeJEErfQ&_LANG=enus "A screenshot showing the Mark document as complete dialog box.")
5. Select Yes, mark as complete to set the document as
   completed in Agreement Desk and Agreement Manager.
