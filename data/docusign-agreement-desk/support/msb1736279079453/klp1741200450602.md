# Delete Documents from Agreement Desk Requests

Learn how to delete documents associated with Agreement Desk requests when they are no longer needed.

When documents associated with Agreement Desk requests are no longer needed, you can delete them from a request.

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Open the request that contains the document that you want to delete.
3. Select the Documents tab.

   A list of documents associated with the request displays.
4. Select the overflow menu (the three dot menu) next to the document that you
   want to delete.

   ![A screenshot of the Agreement Desk request document actions menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-req-sel-doc-action.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzODMsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.2q_KwcybF4dOEcQStMyKZGHPQeiUBFIfPWhXqDj_OwI&_LANG=enus "A screenshot of the Agreement Desk request document actions menu.")
5. Select Delete.

   A confirmation dialog box displays asking you to confirm that you want to delete the document.
6. Select Confirm.

The document you deleted is no longer listed on the Documents page.
