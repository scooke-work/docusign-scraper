# Change a Request Status

Learn how to change Agreement Desk requests.

You can change Agreement Desk requests.

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Select Settings.
3. Select the Statuses tab.

   The account statuses display.

   ![A screenshot of a sample Agreement Desk Status page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/adesk-new-status.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0NzcsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.p9xrA94or6Y9svVjjpBb0Q9Y0FTjfD83d_ldeazdpw8&_LANG=enus "A screenshot of a sample Agreement Desk Status page.")
4. Select
   Edit
   to display the Edit Status dialog box.

   ![A screenshot of a sample Agreement Desk Edit Status dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-change-req-status.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0NzcsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.p9xrA94or6Y9svVjjpBb0Q9Y0FTjfD83d_ldeazdpw8&_LANG=enus "A screenshot of a sample Agreement Desk Edit Status dialog box.")
5. Change
   the status characteristics.
   - Status name
   - Accent color

   The status changes are visible in the Statuses
   page.
