# View Comment Timestamps in the Sidebar

Learn how to use the Agreement Desk comments sidebar to view timestamps.
These indicate when a comment was added or updated.

1. Select the request that contains the document you want.
2. Select the Documents tab.
3. Select the document that contains your comments.
4. Select Preview from the vertical three dot menu to
   display the document and its comments.
5. Select the comments icon to open the comments sidebar
6. Under or next to the commenter’s name, find the timestamp for the comment, as
   shown in this example:

   ![A screenshot showing a sample of a timestamp in the Agreement Desk comments sidebar.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/adesk-req-comm-time-stamp.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0MTgsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.MLygYya1PgL8atSUFNHiH9XKVJilcgcw3F6jVUQCa_8&_LANG=enus "A screenshot showing a sample of a timestamp in the Agreement Desk comments sidebar.")
7. Review the timestamp to understand when the comment was created or last
   updated.

   Note:

   Timestamps typically use your local time zone or your Agreement Desk user time zone settings.
