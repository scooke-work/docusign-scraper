# Assign an Owner to a Request

Learn how to assign an owner to a Agreement Desk request. The owner is
responsible for incoming requests and must be aware of changes and updates.

The owner is responsible for the activities related to the Agreement Desk
request. By default the owner becomes a follower of the request and receives
notifications about changes and updates.

1. Select Requests from the left
   navigation.

   The Requests dashboard
   opens.
2. Locate the request that you want to edit.

   You can sort and filter requests to find the one you want. For more information, see the topic
   [Search and Filter Requests](https://support.docusign.com/s/document-item?bundleId=msb1736279079453&topicId=yrc1740515145178.html&_LANG=enus "Learn how to search Agreement Desk requests, filter, and sort requests to locate the ones you need.").
3. Select the request to open it.

   The request's Overview page displays.
4. Select the pencil icon in the Information section to
   dipslay the Edit request information dialog box.

   ![A screenshot of the Edit request information dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ac-req-edit-db.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzNTIsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.nV4xaFmp7VWpGxpFlmI9BDY2OmSxG4ZV0KOYkvn5Yew&_LANG=enus "A screenshot of the Edit request information dialog box.")
5. Start entering a name in the Owner field and select from
   the list of names that are displayed as you type.

   If an owner is already assigned, select the X next to
   the name to delete the owner and make the Owner field
   active.
6. Select Save.

The Request information box reflects the name that you selected as the owner.
