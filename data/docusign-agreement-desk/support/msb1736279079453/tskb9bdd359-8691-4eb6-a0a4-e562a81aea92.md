# Agreement Desk Message Drafting

Learn how to use the Agreement Desk message drafting feature to turn text
into a message form and capture the key points of the text.

## Before you begin

An administrator must enable AI agents for you to use this
feature.

Use the Agreement Desk Message Drafting feature to convert a request
status or a markup summary to an email. The AI agent turns text into a message form
and captures the key points of the text.

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Select a request to open it.
3. Create a summary as described in these topics.
   - [Agreement Desk Request Summarization](https://support.docusign.com/s/document-item?bundleId=msb1736279079453&topicId=tsk26e57e79-af86-46b5-8842-45898d4499ca.html&_LANG=enus "Learn how to use the Agreement Desk request summarization features to summarize activity associated with a request.")
   - [Agreement Desk Redline Summarization](https://support.docusign.com/s/document-item?bundleId=msb1736279079453&topicId=tskebbad5d3-e278-4ae5-b42b-e4a04fa627a1.html&_LANG=enus "Learn how to use the Agreement Desk Redline Summarization feature to summarize revisions and comments in Word documents, providing an easy way to track changes.")
4. When the summary completes, select the Draft a message
   icon at the bottom of the summary.

   ![A screenshot of selecting the AI agent Send Message icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-ai-send-msg-agent.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0NjUsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.8g6bRusoHWigIUCP6nsr7W0sT85fU4MXz2hDlUVhzSc&_LANG=enus "A screenshot of selecting the AI agent Send Message icon.")

   The Send Message dialog box displays, with the content of the summary in
   the email.

   ![A screenshot of a sample AI Send Message dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-ai-send-ex.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0NjUsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.8g6bRusoHWigIUCP6nsr7W0sT85fU4MXz2hDlUVhzSc&_LANG=enus "A screenshot of a sample AI Send Message dialog box.")
5. Optionally change the email message and add email recipients.
6. Select Send to forward the summary information through
   email.
