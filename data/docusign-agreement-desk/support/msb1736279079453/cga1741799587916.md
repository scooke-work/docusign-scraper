# Manage Agreement Desk Request Documents

Read about the actions that you can take to manage documents within Agreement Desk.

Agreements and documents are central to most of agreements. Docusign
Agreement Desk provides you with many tools for viewing, managing,
collaborating, and editing drafts of these documents.

- **Preview**: View documents in Agreement Desk by using the Docusign Document Viewer.
- **Download**: Retrieve documents directly from
  Agreement Desk to review as Word
  or PDF files.
- **Edit in Word**: Edit documents using
  Microsoft Word.
- **Rename**: Rename the document.
- **Delete**: Delete the document.
- **Version history**: Track versions of a
  document as changes are made by both the first and
  third-party stakeholders.
- **Upload new version**: Upload a different version of the
  current document.
- **Send for Signature**: Send documents for signature through Docusign eSignature. The status is shown directly in the request.
- **Convert a PDF file to a Word document**: Convert a PDF file to a Microsoft Word
  document in an Agreement Desk request.
- **Compare Documents**: Quickly view a comparison between two .docx documents from any
  Agreement Desk request.
