# Get Notified about Changes to Requests

Learn how to stay informed of changes to specific Agreement Desk requests by
using the Follow option.

Stay informed of changes to Agreement Desk requests, such as document
uploads or status changes, through the Follow option.

Note:

When submitting requests through email, only users within your account receive
confirmation email. External users do not receive this notification.

For information about managing notifications, see the article [Set Up Notification Preferences](https://support.docusign.com/s/document-item?bundleId=msb1736279079453&topicId=tsk5ec97360-6dd8-45ff-aa59-bea4c0e41d05.html&_LANG=enus "Learn how to set up notifications for followers and submitters about the Agreement Desk activity.").

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Open a request that you want to follow.
3. Select Follow from the upper right of the request
   window.

   ![A screenshot showing the follow an Agreement Desk request feature.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-follow.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0NTUsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.Bqet4qQS_H97KFRjexhzoT3T568PSEo485MrGHNCyU0&_LANG=enus "A screenshot showing the follow an Agreement Desk request feature.")

   The button changes to reflect Following.

   Note: The Follow button works as a
   toggle. When you no longer want to follow a request, select
   Following to turn notifications
   off.
