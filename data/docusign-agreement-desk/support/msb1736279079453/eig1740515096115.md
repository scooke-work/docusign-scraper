# View Messages Associated with Requests

Learn how to view the messages that are sent related to an Agreement Desk request.

View the messages sent within an Agreement Desk request. Messages that are sent
related to a request are logged and accessible through the Activity
Feed for the request.

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Select a request to open it.

   The Request details Overview page displays.
3. Select All activity > Messages only .
