# Download Documents Associated with Requests

Learn how to download documents that are associated with am Agreement Desk request.

You can download documents from an Agreement Desk request to save a copy in your
local environment.

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Select the Documents tab.
3. Select the overflow menu (the three dot menu) next to the document that you
   want to download.

   ![A screenshot of the Agreement Desk request document actions menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-req-sel-doc-action.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzODEsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.RAsOjghe03edRmydFi_tOmU4oi3LfueIJkfhXdUTNkI&_LANG=enus "A screenshot of the Agreement Desk request document actions menu.")
4. Select Download.

   The document is downloaded to your local environment.

A copy of the document you selected for download is stored in the location you specified.
