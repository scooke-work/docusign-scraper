# Edit Requests

Learn how to edit Agreement Desk requests to change the title, status,
due date, and owner.

You can edit an Agreement Desk request to change its title, status,
owner, and due date.

1. On the Agreements page, select
   Requests.
2. Locate the request that you want to edit.

   You can sort and filter requests to find the one you want. For more information, see the topic
   [Search and Filter Requests](https://support.docusign.com/s/document-item?bundleId=msb1736279079453&topicId=yrc1740515145178.html&_LANG=enus "Learn how to search Agreement Desk requests, filter, and sort requests to locate the ones you need.").

   ![A screenshot of the Agreement Desk Requests page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-latest-req-ex.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzMzUsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.HXllSxNtM52Jp_G8rx7OyEQjJcxlvfW4velZTFKWdy8&_LANG=enus "A screenshot of the Agreement Desk Requests page.")
3. Select the request to open it.

   The request's Overview page displays.
4. Select the pencil icon in the Information section to
   display the Edit request information dialog box.

   ![A screenshot of the Edit request information dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ac-req-edit-db.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzMzUsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.HXllSxNtM52Jp_G8rx7OyEQjJcxlvfW4velZTFKWdy8&_LANG=enus "A screenshot of the Edit request information dialog box.")
5. Edit the following information:

   - Title: Change or replace the title text in the
     Title field.

     You can enter a text-only
     title to serve as the request's official designation within Agreement Desk and throughout IAM.
   - Status: Select a different status from the
     Status menu.
   - Request type: Enter the for the request type.
   - Due Date Select a new date from the date picker
     or enter it directly into the Due Date
     field.
   - Submitter: A licensed participant who submits the
     request. Select the name from the list that is presented as you type.
   - Owner: A licensed participant who submits the request. Select the name from the list that is presented as you type.

     Note:
     - Each user can only be assigned one license. User access to Docusign features is dependent on the account plan, the license type, and the user permission profile.
     - Only users with Edit permissions can access
       request information.
     - If you want to replace a current owner, select the
       X next to the name.
6. Select Save.
