# Delete Requests

Learn how to delete a request that you no longer need.

Delete an Agreement Deskrequest when you no longer need it.

1. Select Requests from the left navigation.
2. Select a request to open it.

   The Overview page displays.
3. Select Details.
4. Select Delete Request in the right lower area of the window.

   ![A screenshot of the Agreement Desk Delete Request feature.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ac-delete-req.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzNDAsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.-P7PV1WGad8YugMoINm-nSXVitTkZpRRNYmtCTwpUHo&_LANG=enus "A screenshot of the Agreement Desk Delete Request feature.")

   A confirmation dialog box displays.
5. Select Delete.

The Requests dashboard displays and the request that you deleted is removed from the list.
