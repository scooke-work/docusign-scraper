# View Requests Updates on the Activity Feed

Learn how to view the activity associated with Agreement Desk requests,
such as status changes, document uploads through the Activity
Feed.

View activity associated with requests, such as messages, status changes, and document uploads, through the Activity Feed.

1. Select Requests from the left navigation.
2. Select a request to open it.
3. Select Activity Feed on the
   Overview tab.

   ![A screenshot showing sample Agreement Desk request activity.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ac-req-activity-feed.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzNDQsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.-7mjdwtc7o08XkkvYal4foqq4ReCdvrplQGivBagX68&_LANG=enus "A screenshot showing sample Agreement Desk request activity.")
4. (Optional) View only the messages in the feed by selecting Messages only from the All activity menu.
