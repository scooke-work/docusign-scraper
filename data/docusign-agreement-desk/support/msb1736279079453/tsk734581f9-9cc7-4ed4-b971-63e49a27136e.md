# Convert a PDF to a Word Document in Requests

Learn how to convert a PDF file to a Microsoft Word document in an Agreement Desk request.

You may want to mark up a document such as a contract, but the file in the Agreement Desk request is in PDF format. You can convert the PDF to a
Word document to mark up and edit the document.

1. From the Agreements page, select
   Requests

   The Requests dashboard opens.
2. Locate the request that you want to edit and select the request to open
   it.
3. Select the Documents tab.
4. Select the vertical three-dot menu and select Edit in
   Word.

   ![A screenshot showing the Agreement Desk request document Edit in Word menu option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-new-convert-to-word-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzOTcsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.N7wQiWOyFYoAtN-ti493Fc_QxWh3hZT9FzzvtDR1omM&_LANG=enus "A screenshot showing the Agreement Desk request document Edit in Word menu option.")

   The document downloads to a `.docx` format.

   Note: Some PDF formatting may be lost in the conversion.

   Alternative Method

   When previewing a document, select Edit in Word to
   convert to the `.docx` format and download the document.
5. Use the word version to perform other request tasks.

   - Edit the word version of your document and [add it to requests](https://support.docusign.com/s/document-item?bundleId=msb1736279079453&topicId=xzc1740514992382.html&_LANG=enus "Learn how to add documents to an Agreement Desk request.").
   - Compare the document to other versions and other files, including PDF, as described in the [Compare
     Documents](https://support.docusign.com/s/document-item?bundleId=msb1736279079453&topicId=tsk019a50de-2452-4f9a-a639-91c573a38cfa.html&_LANG=enus "Learn how to compare Agreement Desk request documents to view the latest version of a document and immediately compare it to a different version.")  article.
