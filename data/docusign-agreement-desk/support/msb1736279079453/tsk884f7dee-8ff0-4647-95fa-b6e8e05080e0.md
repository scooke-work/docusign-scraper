# Use Unified Fields in Agreement Desk Searches

Learn how unified fields work in Agreement Desk and how to use them as columns and filters on the Agreement Desk dashboard.

## Before you begin

To use unified fields in search, you need:

- At least one Agreement Desk request type whose intake form includes unified fields.
- Permission to view requests of that type on the Agreement Desk dashboard.

Unified fields that are not used in any request types do not display in your data.
However, they may be available as column or filter options.

What are unified fields?
:   Unified fields are shared fields that you can add to Agreement Desk request
    forms to capture consistent data such as line of business, renewal type, or department
    . Each unified field has a name, a type (for example text, date, number, dropdown),
    and one or more values.

    Agreement Desk uses the unified field definitions from the central field
    library. Values you capture on intake can be
    reused across Agreement Desk,
    Agreement Manager, and other IAM
    features.

What can you do with unified fields in search?
:   On the Agreement Desk dashboard, you can now:

    - Add unified fields as columns, so you can see key request data in the table.
    - Filter requests by unified field values using an All filters panel.
    - Configure and save your own column layout so the dashboard opens with your preferred view.

For detailed information about IAM Unified Search, see the [IAM Unified
Search](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=conbd35fa9c-4ddb-4f01-be91-f55928fac98a.html&_LANG=enus)  article.

1. Select the Agreements tab in the top navigation.
2. Select Requests  to open the Requests
   dashboard.

   The main table of requests displays, with columns, such
   as Request title, Status, Last activity, Due date,
   Submitter, and Owner.
3. Add unified fields as columns.
   1. Select the column settings (gear) icon.
   2. Review the columns in the column settings
      panel:

      - Review the columns that are currently in use,
        shown in your current display order.
      - Scroll through the list of available columns.
        These include all standard unified fields in
        alphabetical order.
   3. Show or hide a unified field in the
      table.

      To show a unified field in the
      table:

      - Turn on the toggle next to that field name.
      - Optionally drag the field within the
        In use list to change its
        position in the table.

      To hide a unified field
      column, turn off the toggle next to
      that field name.

## Filter requests by unified fields

Use the All filters panel to
narrow the request list. You can use unified field values,
even when those fields are not visible as columns.

1. On the Agreement Desk dashboard, select
   All Filters.
2. In the filters panel, review the list of available
   filters.

   These include:

   - Core request properties (for example: status,
     owner, due date)
   - All available unified fields are configured
     for your organization.
3. Select a unified field filter.

   Depending on the field type, you see:

   - A single-select drop down
   - A multi-select list
   - A date or numeric selector
4. Choose one or more values to apply to that
   filter.
5. Repeat Steps 3 and 4 to add additional filters.
6. Apply the filters to update the dashboard table.

   Note:
   - You can filter on a unified field
     even if its column is currently hidden.
   - The filters you apply affect only
     your view and do not change the underlying request
     data.

## Reset columns to the default view

You may have experimented with different column layouts
and want to go back to the default configuration.

1. Open the column settings (gear) icon.
2. Look for the option to reset columns to the default
   view (if enabled for your environment).
3. Confirm the reset.

After reset, you see the default columns (Request title
and identifier, Status, Last activity, Due date, Submitter,
Owner) in their original order.

## Save and reuse filtered views

Some Agreement Desk environments support
saving a configured set of columns and filters as a reusable
view or link.

1. Configure the dashboard:

   - Add or remove unified field columns
     as needed.
   - Apply filters in the All filters
     panel.
2. Look for a Save, Save
   view, or a similar control near the
   filters area.
3. Follow the on-screen prompts to:

   - Name the view
   - Copy a link to the current query for
     reuse or sharing

   Note:

   If you do not see a Save option,
   contact your administrator to confirm whether
   saved searches for Agreement Desk are
   enabled in your account.
