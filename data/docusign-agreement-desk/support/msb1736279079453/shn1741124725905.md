# Request Creation Methods

Read about the ways that you can create a request.

You can create a request from your email or from the Requests dashboard.

- Dashboard request: Create a request from the dashboard by
  selecting a specific request type from the Requests
  dashboard. For more information, see the topic [Create a Request from the Requests Dashboard](https://support.docusign.com/s/document-item?bundleId=msb1736279079453&topicId=tpa1740514877858.html&_LANG=enus "Learn how to create a specific request type from the Agreement Desk Requests dashboard.").
- Email request: Create a request by sending an email to your
  organization's Agreement Desk email address. Use the provided default address or a
  customized address. You can attach documents to the email to include them in the
  task.
  - When you submit by email, the default request type is applied.
  - For more information, see the topic [Create a Request through Email](https://support.docusign.com/s/document-item?bundleId=msb1736279079453&topicId=umt1747683535119.html&_LANG=enus "Learn how to create anAgreement Deskrequest directly from your email.").
