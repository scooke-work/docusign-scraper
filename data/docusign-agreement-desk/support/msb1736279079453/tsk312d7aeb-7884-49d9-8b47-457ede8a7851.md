# Create Custom Request Identifiers

Learn how to createe a custom Agreement Desk request identifier to help you
improve discoverability, simplify customer communication, and improve overall request
tracking.

Create custom request identifiers that help you better organize and search
for requests in Agreement Desk.

Important:

Admins should keep these in mind when creating custom request identifiers:

- Admins can change the prefix at any time.
- Changing the prefix only updates the identifier for newly
  created requests. Existing requests keep their original identifier.
- You can only have 1 custom prefix per account.

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Select Settings > Admin.
3. In the Request identifier's Prefix Value field, select
   Edit.

   Select Reset to default to return the prefix value to
   REQ.
4. Enter a custom identifier for your account and select
   Save.
