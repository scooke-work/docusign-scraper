# Import and Export Request Type and Status Settings

## Before you begin

You must have Docusign administrator (DS Admin) privileges or
Agreement Desk settings manager permissions privileges for your
account.

This feature simplifies administrative tasks by transferring complex
request type and status settings between different Docusign
instances (such as Demo and Production environments.)

With this feature, administrators can easily package up and move all the
complex rules for request types and their workflow steps (statuses).

For example, you created or changed several request types, and you want to
use these across all the accounts you administer. You can export the information to
a .zip file, and import the settings to other accounts.

## Export settings

Export request types and status settings for an
account.

1. Select Agreements  > Requests  > Settings.
2. Select Request types.
3. Select Export from the top menu.

   ![A screenshot of the Docusign Import and Export menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-admin-imp-export.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI1MDAsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.aZTS1jEpflodThn47mGbNZVsR_UgD_6XYDJrG21dW5g&_LANG=enus "A screenshot of the Docusign Import and Export menu.")
4. In the Export dialog box, select all or specific request types to export,
   and select Export.

   ![A screenshot of a sample Agreement Desk Export request types dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-admin-export-db.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI1MDAsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.aZTS1jEpflodThn47mGbNZVsR_UgD_6XYDJrG21dW5g&_LANG=enus "A screenshot of a sample Agreement Desk Export request types dialog box.")

   Depending on the number of request types you selected, this could take
   several seconds.
5. When the export completes, a file download dialog box displays. Select the
   location for the export zip file.
6. Select Settings > Statuses.
7. Repeat the previous steps to export Status information for your
   account.

## Import settings

Import request types and status settings to use for your accounts.

1. Select Agreements  > Requests  > Settings.
2. Select Statuses.
3. Select Import from the top menu.

   ![A screenshot of the Docusign Import and Export menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-admin-imp-export.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI1MDAsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.aZTS1jEpflodThn47mGbNZVsR_UgD_6XYDJrG21dW5g&_LANG=enus "A screenshot of the Docusign Import and Export menu.")
4. In the Import dialog box, drag (or select) a request type zip file to the
   Select file area.

   ![A screenshot of the Docusign Import request types dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-admin-import-db.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI1MDAsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.aZTS1jEpflodThn47mGbNZVsR_UgD_6XYDJrG21dW5g&_LANG=enus "A screenshot of the Docusign Import request types dialog box.")
5. Select Import.

   Depending on the number of request types you selected, this could take
   several seconds.

   When the import successfully completes, the `Import
   completed` message displays.
6. Select Settings > Statuses.
7. Repeat the previous steps to import Status information to your
   account.
