# Add Documents to Agreement Desk Requests

Learn how to add documents to an Agreement Desk request.

You can add documents to an Agreement Desk request by uploading the
documents from your local environment or by creating them from a template.

Note: You can add only one document
at a time from the Requests page.

You can add your own document to a request, including those that don't require
document generation. For example, you may want to include documents such as exhibits
or sheets in your request.

You can also use your documents to create templates. Using your own templates ensures
that your users obtain approved templates for your organization.

1. Select Requests from the left pane navigation to access
   the Requests dashboard.
2. Select the Documents tab.

   ![A screenshot showing the Agreement Desk Documents tab.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-add-docs.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzNzEsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.0Cr3GcnkDFlIeUNdpnTjFc32DtsXEIQpWmvy-Yb941Y&_LANG=enus "A screenshot showing the Agreement Desk Documents tab.")
3. Select Add Document and choose one of these
   methods.
   - Upload document: Upload a new document
   - Upload new version: Upload a new version of the
     current document
   - Use document template
     1. In the
        Select
        document template dialog box, search for or select a
        template from the list and select Add
        Selected.

        ![A screenshot of a sample Agreement Desk Select document template dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-select-template-db.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzNzEsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.0Cr3GcnkDFlIeUNdpnTjFc32DtsXEIQpWmvy-Yb941Y&_LANG=enus "A screenshot of a sample Agreement Desk Select document template dialog box.")
     2. Complete the Add Field Data section and select
        Generate.
     3. On the Generate field page, select  Highlight
        Data or Don't Highlight
        Data.
     4. Select
        **Next**
        to generate a template from your
        document.
