# Replace a Document with a Newer Version

Learn how to replace an existing document associated with a request with a newer version.

You can replace an existing document in an Agreement Desk request by uploading a
newer version. The version history is maintained.

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Select the Documents tab.
3. Select the overflow menu (the three dot menu) next to the document that you
   want to replace.

   ![A screenshot of the Agreement Desk request document actions menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-req-sel-doc-action.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzNzgsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.bW8yQCPnfpSBnwJc8ru0wT9y9CQFvYICPwU9P6kNCaQ&_LANG=enus "A screenshot of the Agreement Desk request document actions menu.")
4. Select Upload new version.
5. Select the new version of the document that you want to upload and select Open.

The new version of the document replaces the old version in the list of documents on the Documents tab. You can also see the changes to the document by selecting Version history from the overflow menu.
