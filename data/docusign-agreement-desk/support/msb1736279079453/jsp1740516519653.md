# Send Documents for Signature from Requests

Learn how to send documents for signature from within a
Agreement Desk request.

When you are ready to have
Agreement Desk request documents signed, you can
create an envelope with the documents and send them for signature from within a
request.

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Select a request to open it.
3. Select the Documents tab.
4. Select Send for signature.

   The Select documents you want
   to
   send for signature dialog box
   opens.![A screenshot of the Agreement Desk request Select documents for signature dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-req-docs-send-for-sig.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzODYsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.yvkwTk_OnCa4HF8j5NgTGU1uMbvdtPUTQxOKwMCPBh4&_LANG=enus "A screenshot of the Agreement Desk request Select documents for signature dialog box.")
5. Select
   the documents you want to send and select Send for
   signature.

   ![A screenshot of the Agreement Desk request Select documents for signature dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-req-docs-send-for-sig.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzODYsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.yvkwTk_OnCa4HF8j5NgTGU1uMbvdtPUTQxOKwMCPBh4&_LANG=enus "A screenshot of the Agreement Desk request Select documents for signature dialog box.")
6. Complete the signing process.

Your envelope is sent to the
recipients
you entered for signature. You can view the signing status by
selecting the Envelopes tab.
