# Manage Request Approvals

Learn how to Agreement Desk approve, reject, or change an approval.

You can approve an Agreement Desk request through the request detail
Approval page.

Additionally, you can suggest changes to the approval request, or approve the request
later.

To learn about managing request approvals in IAM Unified Tasks, see the [Request Approvals in IAM Unified
Tasks](https://support.docusign.com/s/document-item?bundleId=msb1736279079453&topicId=con486db68d-874e-4355-af0b-da1594542281.html&_LANG=enus "Learn how to access and approve Agreement Desk requests through Docusign Unified Tasks.") article.

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Locate the request that you want.
3. Select the Approvals tab.

   Any pending approvals display on the page.

   ![A screenshot of a sample Agreement Desk request approval on the Approvals page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-newest-appr-page.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0NjgsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.mLZ_wEnf_8_CDfWp0RqIx-VAvsSe37ch6lLYVXKvjrE&_LANG=enus "A screenshot of a sample Agreement Desk request approval on the Approvals page.")
4. Select one of these request approval actions:

   - Add approval: Add a new approval request
   - Select the vertical three dot menu next to Add approval
     and select **Reset**: Resend approval request to all participants
   - **Remind**: Send a reminder to all approvers that have not responded
   - Select the vertical three dot menu next to **Remind**:
     - **Edit**: Edit the existing content
     - **Reset**: Resend approval requests to all participants
     - **Delete**: Delete the approval request

## Approve the request through email

If you are designated as an approver for a request, you receive an email notification
with a link to the approval in the request. You can accept or reject the request and
add a comment.

1. In the Approval required for new request email
   notification, select View Approval.

   A new window opens to the Approvals tab of the
   request.
2. Select Decide next to the approval request.
3. Select either Approve or
   Reject.

   If you select Reject, you must enter a reason.
