# Set Up Request Notifications

Learn how to set up notifications for followers and submitters about the Agreement Desk activity.

You can set up notifications for followers and submitters about the Agreement Desk activity.

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Select Settings > Notifications to display the Notification  page.

   ![A screenshot showing the Agreement Desk Notifications Settings page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-not-page.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0NTEsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.7SnrmV-WdBcEJRiggKvcypuFfvTsYM4Tx61feKziIZk&_LANG=enus "A screenshot showing the Agreement Desk Notifications Settings page.")

   Note: The Send a confirmation email to the submitter with a link to add more details option only applies to requests submitted through email.

## Set follower notification preferences

Select the activities that generate email notifications to request
followers.

1. On the Notifications page, select the notification
   preferences link.
2. Select the dropdown menu for each category to display the notifications for
   each category.

   The categories include:
   - Envelopes
   - Requests
   - Completed Agreements

     ![A screenshot showing the followers' Docusign notification preferences link.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-notification-settings.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0NTEsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.7SnrmV-WdBcEJRiggKvcypuFfvTsYM4Tx61feKziIZk&_LANG=enus "A screenshot showing the followers' Docusign notification preferences link.")
3. Select notifications types:

   - Select the Select all Notifications checkbox
     to select all notifications in the category. You can toggle this
     feature on and off.
   - Select individual Notification types.

   For information about email notifications in Docusign
   accounts, see the [Manage Notification Preferences](https://support.docusign.com/s/document-item?bundleId=jux1643235969954&topicId=dhc1578456326388.html&_LANG=enus)﻿ article.
4. Repeat for Requests and
   Completed Agreements notifications.
5. Select Save.

## Set submitter notification preferences

Use the sliders to toggle the submitter notifications.

Note: Only account users receive submitter notifications. Make
sure that the user is added to your account.

1. On the Notifications page, scroll down to the Submitter
   notifications section.

   ![A screenshot of the Agreement Desk Submitter notifications section.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-not-submitters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0NTEsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.7SnrmV-WdBcEJRiggKvcypuFfvTsYM4Tx61feKziIZk&_LANG=enus "A screenshot of the Agreement Desk Submitter notifications section.")
2. Optionally turn on or turn off the submitter notification options.
