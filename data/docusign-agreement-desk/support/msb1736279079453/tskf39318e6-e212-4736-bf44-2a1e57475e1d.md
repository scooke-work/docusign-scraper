# Review Agreement Desk Request Reports

Learn about using report insights for viewing detailed information about Agreement Desk requests.

## Before you begin

You need account admin privileges to view all request insight data. Users can only view information for requests they're authorized to access.

You can view statistics about Agreement Desk request statuses with the Report Insights feature.

You can now:

- Build, filter, and export detailed request reports.
- Perform deep-dive analysis on request status, volume, and custom fields (such as ARR or Region).

For information about managing Docusign Agreement Desk request reports, see the  [Agreement Manager Request Report Dashboards](https://support.docusign.com/s/document-item?bundleId=mapf0ca9167-3825-4b36-a385-e98321f7adb5&topicId=con1fc88284-aff5-46c9-ac08-15db3839fce8.html&_LANG=enus)  article.

3. Select Go to Reports to view reports dashboards.

   The Requests Insights page displays, as shown in this example.

4. Customize the report data.
   - Completed date: Select the time frame for when
     the requests were completed.
   - Request type: Select one or more request type.
   - Customize the report display by selecting View as
     table or

     View as chart
   - Make this request view a favorite by selecting the star icon.
