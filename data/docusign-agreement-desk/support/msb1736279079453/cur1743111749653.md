# Get Started with Agreement Desk

Learn how to get started with Docusign
Agreement Desk by creating and assigning your first request.

This quick introduction to Agreement Desk describes how to create a
request, add a document, and send it for approval and signature. It also describes
how to see your activity flow in the Activity Feed.

1. Select Agreements in the top navigation.
2. Select Requests from the left navigation area.

   If you have no requests, the Requests dashboard displays a Create
   Request option along with a Learn
   More link to tutorial videos.

   ![A screenshot of theAgreement Desk Request page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-new-empty-req-pg.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzMjcsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.tPuwFLWL8BppD2DQFzV_hzUTex6F9BQySEBBX3LWn6I&_LANG=enus "A screenshot of theAgreement Desk Request page.")

   If you have Agreement Desk requests, the
   Requests dashboard displays, showing a list of
   requests with relevant information, such as, title created date and time,
   last activity date and time, status, submitter, and owner.
3. Select Create Request.

   Note: You can also create a request directly from your
   inbox. Send an email to the Agreement Desk email alias for your
   organization. See the [Create a Request through Email](https://support.docusign.com/s/document-item?bundleId=msb1736279079453&topicId=umt1747683535119.html&_LANG=enus "Learn how to create anAgreement Deskrequest directly from your email.") article.

   1. Select a form from the list of request types.
   2. Complete the fields in the form.
   3. Select Submit.

   A confirmation dialog box displays.
4. Select View Request to see details about the request you created.
5. Set the status, assign a due date, and assign any owner.
   1. In the Request Information section to the right,
      select the pencil icon to edit the request.
   2. Select a status from the Status menu.
   3. Select a date from the Due Date field.
   4. Select an Owner field.
   5. Select Save.
6. Edit information in the request intake form.
   1. Select the Details tab.
   2. (Optional) Select Edit to change information in
      the form. After making your changes, select
      Submit.
7. Add a document to your request.
   1. Select the Documents tab.
   2. Select Add Documents.
   3. Choose the method for adding the document. Either upload a document or generate a document.

   The document you added is displayed in the list of documents.
8. Send a message. You may want to send the message to yourself so you can test it out.
   1. Select the Overview tab.
   2. Select Send Message.
   3. Add your email address and a message and select Send.

   In a few seconds, you'll see the message that you sent in the activity feed.
9. Send your request for approval. Select an approver from the Contact
   list.
   1. Select the Approvals tab.
   2. Select Add Approval.
   3. Enter a title, description, and approvers. Select Send
      Approval.

   The approval displays on the Approvals page with
   a status of Waiting for approval. To approve the request,
   select Decide and
   Approve.
10. Send your document for signature.
    1. Select the Envelopes tab.
    2. Select Send for signature.
    3. Select the documents that you want to send, and select Send for signature.
    4. Complete the signing process as normal.
