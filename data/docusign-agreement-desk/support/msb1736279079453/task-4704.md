# Get a Direct link to a Comment

1. Select the request that contains the document you want.
2. Select the Documents tab.
3. Select the document that contains your comments.
4. Select Preview from the vertical three-dot menu to
   display the document and its comments.
5. Select the comments icon to open the comments sidebar
6. At the top of the comments list, select the filter control.
7. Select Get link to this comment  option, as shown in the
   following example.

   ![A screenshot of the Agreement Desk Comments menu Get link to this comment option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/adesk-req-comm-get-link.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0MjMsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.ZQWdF7gSyKkGJY9tunurPsDK0gf7KDK1R0myHHLja0s&_LANG=enus "A screenshot of the Agreement Desk Comments menu Get link to this comment option.")

   A confirmation message displays in the bottom left of the page confirming that Agreement Desk
   copies the link to your clipboard.

   ![A screenshot of the Comments Menu Get Link confirmation message.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-clause-link-confirm.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0MjMsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.ZQWdF7gSyKkGJY9tunurPsDK0gf7KDK1R0myHHLja0s&_LANG=enus "A screenshot of the Comments Menu Get Link confirmation message.")
8. Paste the link into an email, chat, or other message so that collaborators can
   open the document and jump directly to that comment.

   Note:

   People who open a direct comment link must have appropriate
   access to the Agreement Desk request and document to view
   the comment.
