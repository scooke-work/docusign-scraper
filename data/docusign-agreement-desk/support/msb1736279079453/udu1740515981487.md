# Notifications About Changes to Requests

Read about how internal request submitters receive updates about changes to Agreement Desk activities.

If you create an Agreement Desk request and if you are an internal user, you
might automatically receive notifications when updates to your request occur.

Updates include changes to owner, due date, and status, for example. Internal users are
those who have an account within the system. External users do not receive notifications
regarding their submitted requests.

For request creators to be automatically notified, an administrator must turn on Submitter notifications in Agreement Desk settings. If your administrator has Submitter notifications turned off, you can still receive updates by following the request.

To follow a request, you must have View permission or higher. If another user shares a request with you, you are automatically granted View permission.

For instructions about following a request, see the topic [Get Notified about Changes to Requests](https://support.docusign.com/s/document-item?bundleId=msb1736279079453&topicId=ent1747168477311.html&_LANG=enus "Learn how to stay informed of changes to specific Agreement Desk requests by using the Follow option.").
