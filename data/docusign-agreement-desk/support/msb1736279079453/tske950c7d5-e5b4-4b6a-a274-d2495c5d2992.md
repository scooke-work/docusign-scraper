# Forward Messages

Learn how to forward Agreement Desk messages.

You can forward a message to other users. When you forward a message, recipients see a
notification in their email inbox and can reply to the email or directly in Agreement Desk. Responses are captured in the request.

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Open a request from the Requests page.

   The Request Detail page displays.
3. In the request Activity Feed, select Forward.

   ![A screenshot of an example showing forwarding an Agreement Deskrequest.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-forward-msg.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0MzIsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.Dlkt5x71fr-v2XXpPm8LpIfFevX-ta8sbXSb40ygDZQ&_LANG=enus "A screenshot of an example showing forwarding an Agreement Deskrequest.")
4. Select a recipient from the list in the To  field.
5. Enter a message for the recipient.
6. Optionally select Attach  to add documents to the
   message. You can upload it from your computer or select a document associated
   with the request.
7. Select Send.

Your message is forwarded, and is reflected in the Activity
Feed. The recipient receives an email with the message and any
attachments.

If
the Account Admin turns on message participation, you cal add recipients who are
not in your account to the message. See the topic [Actions Permitted By Permission
Level and Role](https://support.docusign.com/s/document-item?topicId=oiu1731179949375.html&_LANG=enus)﻿ for more information.
