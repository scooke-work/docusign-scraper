# Work with Document Comments in the Comments Sidebar

Learn how to use the comments sidebar in Agreement Desk request documents to
review, filter, and act on comments without leaving the document view.

Use the comments sidebar in Agreement Desk request documents to
review, filter, and act on comments without leaving the document view.

Use the comments sidebar to:

- Always access document comments from the sidebar, even when no
  comments exist yet.
- Quickly filter comments to focus on open items or resolved
  threads.
- See timestamps that show when each comment was added or
  updated.
- Reply to, resolve, and reopen comments directly from the sidebar
  without scrolling through the document.
- Share direct links to specific comments so collaborators can jump
  straight to the exact place in the document.

1. Select the request that contains the document you want.
2. Select the Documents tab.
3. Select the document that contains your comments.
4. Select Preview from the vertical three-dot menu to
   display the document and its comments.
5. Select the comments icon to open the comments sidebar as shown in the following
   example:

   ![A screenshot of the Agreement Deskrequest document Comments icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/adesk-req-comm-icon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0MDksInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.j7MSo-JfEJMtaenVIROEePZ1S5zv-X1fbR0-5eS24JM&_LANG=enus "A screenshot of the Agreement Deskrequest document Comments icon.")
6. Review the comments list.

   - If the document has comments, the sidebar lists them.
   - If no one has added comments yet, the sidebar shows the
     message No comments yet as shown in this example

   ![A screenshot of a sample Comments sidebar with no comments.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/adesk-req-comm-none.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0MDksInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.j7MSo-JfEJMtaenVIROEePZ1S5zv-X1fbR0-5eS24JM&_LANG=enus "A screenshot of a sample Comments sidebar with no comments.")

   Tip:

   The comments icon and sidebar stay available while you work in
   the document so that you can open or close the panel at any time.
