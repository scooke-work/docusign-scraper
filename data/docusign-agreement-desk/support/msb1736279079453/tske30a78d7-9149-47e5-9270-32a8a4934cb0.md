# Complete Envelope Activities in Agreement Desk

Learn how to manage and complete envelope activities from a DocusignAgreement Desk request.

## Before you begin

Your permission profile must include Docusign Send (DS Send) or Docusign Admin (DS Admin) privileges to perform these tasks.

You can manage and complete envelope activities from a DocusignAgreement Desk request.

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Select the request that contains the envelope you want.
3. Select the Envelopes tab to display the envelope
   details.
4. Select the envelope that you want to manage and select Details from the
   menu.

   The envelope's recipients and Details tab
   display.
5. At the top of the page, select the overflow menu (the three dot menu) to manage
   the envelope.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-envelope-det-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzOTEsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.37G3ajtqbibPoAnxZHuaSXJA-gEab5ylultBrF0sKK8&_LANG=enus)
6. Perform one of these envelope actions:
   - History: Displays the envelope's history as described in the article
     [Check Docusign envelope
     history](https://support.docusign.com/s/articles/How-do-I-check-the-envelope-history).
   - Form data: Displays a summary of the envelope's information.
   - Export as CSV: Select to download the envelope's information in CSV
     format.
   - Delete: Deletes the envelope.
   - Download: Downloads the envelope's documents. You can select one or more
     documents to download.
   - Print: Prints the envelope's documents.
