# Request Approvals in IAM Unified Tasks

Learn how to access and approve Agreement Desk requests through Docusign Unified Tasks.

You can access and approve Agreement Desk requests through Docusign
Unified Tasks.

## What are IAM Unified Tasks?

Docusign Unified Tasks is a single, centralized hub
within the Docusign system that provides visibility to all tasks
assigned to you that require your action.

With Unified Tasks Management, you can access one central view to manage
tasks across your Docusign applications. This process helps you
handle tasks efficiently and prioritize your work. Effective task management can
also help you get agreements signed more efficiently.

Agreement Desk syncs request-based approvals with the Docusign Unified Tasks ecosystem, providing a one-stop shop for
task management.

In addition to using Agreement Desk, you can view, track, and directly
manage Agreement Desk approvals assigned to you in Unified Task
Management.

For detailed information, see the Find assigned Agreement Desk requests
section in the  [IAM Unified Search](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=conbd35fa9c-4ddb-4f01-be91-f55928fac98a.html&_LANG=enus)

## Identify an Agreement Desk request approval task in Unified Tasks

In the Unified Tasks dashboard, the Agreement Desk request
approvals have these format components:

Task type
:   Look for Agreement Desk request approval

Approval Name
:   **Approval**
    **Name** : The task name clearly identifies the request and the
    specific approval needed in this format:

    `Request Title-Approval Title`

    Example: New NDA Request - Finance Approval

See the  [Perform a Task from the Docusign Unified Task View](https://support.docusign.com/s/document-item?bundleId=ikc1746801059132&topicId=spq1746807934085.html&_LANG=enus) article for
information about managing tasks in Unified Tasks.

Approvals created in Agreement Desk that are assigned to individuals show
in IAM Unified Tasks as well as in Agreement Desk.

Note: Approvals assigned to a group show in Agreement Desk, but do not show in Unified Tasks.

## Completion and task removal

The
Request
Approval tasks are automatically removed from the Unified Tasks list once you
approve or reject the approval in Agreement Desk.

Here are the actions that you can take.

- **Single Assignee** or **Any One Approval**: The task is immediately
  removed from the Unified Tasks list after you select **Approve** or
  **Reject**.
- **Require all to approve** is on: The task remains visible until every
  required user has selected **Approve** or **Reject**.
- **Task Reset:** If an approval is reset in Agreement Desk, the Approve task
  re-displays in Unified Tasks. This requires new action.
- **Deletion:** If the Approval or the Request is deleted in Agreement Desk, the
  task is immediately removed from Unified Tasks for all
  assignees.

## Take Action on Agreement Desk Approvals

There are two ways to access the Approve or Reject details.

From the Unified Task list
:   Click the **Approve** task bar in the main Unified Tasks list to navigate directly to the
    **Approvals tab** of the relevant Request in Agreement Desk. The is the fastest way to the action screen.

Agreement Desk requests page
:   See the topic [Manage Request
    Approvals](https://support.docusign.com/s/document-item?bundleId=msb1736279079453&topicId=cqt1740782949607.html&_LANG=enus "Learn how to Agreement Desk approve, reject, or change an approval.") for information about adding approvals from the
    requests page.
