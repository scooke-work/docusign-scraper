# Assign Followers to a Request

Learn how to assign an interested party as a follower to an Agreement Desk request.

You can add followers to a Docusign
Agreement Desk request.

- Add them as a recipient from when you create a request.
- Add them to an existing request.

Note:

If you are an owner of a request, you are automatically a follower.

When an approval request lands in your inbox, select Reply
in the email message.

Your response is automatically captured and logged as a message in the
request’s activity feed for the entire team to see.

For information about Notifications, see the [Manage Notification Preferences](https://support.docusign.com/s/document-item?bundleId=jux1643235969954&topicId=dhc1578456326388.html&_LANG=enus) article.﻿

1. Select Requests from the left navigation.

   The Requests dashboard opens.
2. To add a follower to a new request:
   1. Follow the instructions in the topic [Create a Request from the
      Dashboard](https://support.docusign.com/s/document-item?bundleId=msb1736279079453&topicId=tpa1740514877858.html&_LANG=enus "Learn how to create a specific request type from the Agreement Desk Requests dashboard.").
   2. Select Add Recipient.
   3. Select Can View in the Access field.
3. To add a follower to an existing request:
   1. Follow the instructions in [Share
      Requests](https://support.docusign.com/s/document-item?bundleId=msb1736279079453&topicId=tskf871e5e5-d006-4ec7-8043-dbdaa38578cb.html&_LANG=enus "Learn how to share requests with other agreement desk users.").
   2. Select Can View in the
      Access field.
