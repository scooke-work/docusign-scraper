# Search and Filter Requests

Learn how to search Agreement Desk requests, filter, and sort requests
to locate the ones you need.

You can search, filter, and sort Agreement Desk requests to locate the
requests you need.

![A screenshot of Agreement Desk request filter options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-req-filter-options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzNDIsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.v_6UjsZfkfcjtwSh8AGZVeAk829nZIS8S-mBn450PQ8&_LANG=enus "A screenshot of Agreement Desk request filter options.")

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. In the search box, enter a text in the search box and select
   Enter to see the resulting matches.

   ![A screenshot of an Agreement Deskfilter request entry for MSA.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-req-filter-db.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzNDIsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.v_6UjsZfkfcjtwSh8AGZVeAk829nZIS8S-mBn450PQ8&_LANG=enus "A screenshot of an Agreement Deskfilter request entry for MSA.")
3. (Optional) Filter the results to narrow your search by selecting from one or more of
   the following options.
   - Status Type: Select Open or
     Closed.
   - Status: Select from specific statuses for open and closed
     requests. The options listed include custom statuses.
   - Request type: Select from the request types listed to narrow
     results to only those types.
   - Owner: Select a name from the list of request owners. The owner
     is assigned by the request submitter.
   - Submitter: Select a name from the list of users who created the
     requests shown in the dashboard.
   - Created at: Select from pre-defined date ranges enter a custom
     date range.
   - Last Activity At: Select from a pre-defined date range or enter a
     custom date range.
   - Due Date: Select from a pre-defined date range or enter a
     custom date range.
4. (Optional) Sort the results by selecting the arrows shown in this example for one or
   more column headings to see the information alphabetically or by date in ascending or
   descending order.

   ![A screenshot of a sample Agreement Desk request column head filter.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-req-sort-arrows.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzNDIsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.v_6UjsZfkfcjtwSh8AGZVeAk829nZIS8S-mBn450PQ8&_LANG=enus "A screenshot of a sample Agreement Desk request column head filter.")

Select a request from the list of results to open it.
