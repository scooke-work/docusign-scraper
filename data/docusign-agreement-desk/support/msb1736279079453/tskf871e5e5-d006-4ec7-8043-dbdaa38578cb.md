# Share Requests

Learn how to share requests with other agreement desk users.

You can share Agreement Desk requests with other Agreement Desk users.

1. Select Requests from the left navigation.

   The Requests dashboard opens.
2. Locate the request that you want to edit.
3. Select a request to open it.
4. Select Share in the upper right corner to acess the
   Share with others dialog box.

   ![A screenshot of the Agreement Desk request Share with Others dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-req-share-db.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzNDksInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.Vm_xqNp4HwP4i3woFGxbhb2fD3lCweIresMgD2Anqus&_LANG=enus "A screenshot of the Agreement Desk request Share with Others dialog box.")
5. Begin typing the name of the user you want to add and select it.
6. In the Access field select an Acess permission.
7. Optionally select Copy Link to obtain a link for the
   request.
8. Select Save.
9. Repeat for other users that you want to share this request.
