# Reply to Messages in Requests

Learn how to reply to Agreement Desk request messages within an email
message thread within requests and optionally attach documents.

You can easily respond to an an Agreement Desk request message within the
message thread of a request and attach documents, if necessary.

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Select a request to open it.

   The Request details Overview page
   displays.
3. Select All Activity > Messages Only to view messages exchanged within the request.
4. View the email message thread for a request.
5. Locate the message that you want to reply to and select Reply or Reply all.

   The Reply Message dialog box displays.
6. Select a name in the To field, enter a message, and optionally attach a document.

   ![A screenshot showing the Agreement Desk request reply to a message feature.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-req-reply-msg.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0NDMsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.fkFh0wdhlxP6aBLxpVFxpsDk91RVbzG8zUmd97c89O8&_LANG=enus "A screenshot showing the Agreement Desk request reply to a message feature.")
7. Select Send.

The Activity Feed reflects your reply to a message in the message thread.
