# Filter Comments in the Sidebar

Learn how to determine which comments display in theAgreement Desk request comments sidebar.

You can determine which comments display in theAgreement Desk request comments sidebar.

1. Select the request that contains the document you want.
2. Select the Documents tab.
3. Select the document that contains your comments.
4. Select Preview from the vertical three-dot menu to
   display the document and its comments.
5. Select the comments icon to open the comments sidebar.
6. Select the type of comments you want to see and select
   Reply.

   Filter by all, open, or resolved comments.
