# Manage Comments in Request Documents

Learn about how to add, edit, delete, resolve, and respond to comments in an Agreement Desk request document.

Using Agreement Desk commenting features, you can discuss specific terms,
request input from request stakeholders.

## Add comments

You can add comments to a request document.

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Select the request that contains the document you want.
3. Select the Documents tab.
4. Select the document that contains your comments.
5. Select Preview from the vertical three-dot menu to
   display the document and its comments.

   comments.

## Edit Comments

You can edit comments in a request document.

1. Select the request that contains the document you want.
2. Select the Documents tab.
3. Select the document that contains your comments.
4. Select Preview from the vertical three-dot menu to
   display the document and its comments.
5. Select the comment that you want to change.
6. Select the three-dot vertical menu and select
   Edit.
7. Make changes and optionally use the @mention feature to notify other
   users about your changes
8. Select Post.

   The changes to your comments display in the document viewer.

## Reply to comments

You can reply to comments and @mention additional people.

1. Select the request that contains the document you want.
2. Select the Documents tab.
3. Select the document that contains your comments.
4. Select Preview from the vertical three-dot menu to
   display the document and its comments.
5. Select the comment that you want to change.
6. Select Reply.
7. Add information and select Reply.

   You can also use the @mention feature to notify other users about your
   input.

   The changes to your comments display in the document viewer.

## Resolve comments

You can resolve any unresolved comments.

1. Select the request that contains the document you want.
2. Select the Documents tab.
3. Select the document that contains your comments.
4. Select Preview from the vertical three-dot menu to
   display the document and its comments.
5. Select the three-dot vertical menu and select the
   Resolve checkmark.

   ![A screenshot of the request document comment resolve feature.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-resolve-comment.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0MjYsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.t3ZexJVYC-9KmYZx3BWo2PGkp9JO2yovTO9nf3QAmHU&_LANG=enus "A screenshot of the request document comment resolve feature.")

   The comments are resolved and the changes from your comments are
   implemented.
6. If the subject matter for the comment is still under review, you can select
   the comment and select Unresolve.

document.

1. Select the request that contains the document you want.
2. Select the Documents tab.
3. Select the document that contains your comments.
4. Select Preview from the vertical three-dot menu to
