# Agreement Desk Request Summarization (Beta)

Learn how to use the Agreement Desk request summarization features to
summarize activity associated with a request.

Use the Agreement Desk Request Summarization feature to summarize
activity associated with a request. The agent presents a high-level overview
of the status along with key information such as owner, blockers, and next
steps.

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Select a request to open it.
3. Select Summarize This Request, located on the right-hand
   side of the Overview tab.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-new-ai-summarize.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0NjAsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.ptU4kETk1QKypbVXqSnWk54zTpIhROOfh9ppRicn-mo&_LANG=enus)

   The AI agent summarizes the activity related to the request. When the
   agent finishes, it presents an overview with key information, such as key
   events, owner, blockers, and next steps.

   ![A screenshot of a sample Agreement Deskrequest summary.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-ai-summary.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0NjAsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.ptU4kETk1QKypbVXqSnWk54zTpIhROOfh9ppRicn-mo&_LANG=enus "A screenshot of a sample Agreement Deskrequest summary.")
4. Optionally select the page icon at the bottom of the summary to copy the
   response to the clipboard.
5. Select the X to close the AI agent window.
