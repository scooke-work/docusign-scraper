# Send Messages

Learn how to send a message about anAgreement Deskrequest.

You can send a message about anAgreement Deskrequest to other users. When you
send a message, recipients see a notification in their email inbox and can reply to
the email
or
directly in Agreement Desk. Responses are
captured in the request.

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Open a request from the Requests page.
3. Select Send Message from the Overview page.

   ![A screenshot showing the Agreement Desk request Send Message feature.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-send-msg-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0MzcsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.px29PCu5YLhcnu146mpM3yemtIjoyBDguz89t9zLPl8&_LANG=enus "A screenshot showing the Agreement Desk request Send Message feature.")
4. Alternately, you can select a specific request and select Send
   Message on the Request Detail page.
5. Select a recipient from the list in the To field.
6. Enter a message for the recipient.
7. Optionally select Attach to add a document to the
   message. You can upload it from your computer or select a document associated
   with the request.

   This example shows a completed request message. ad-send-msg-db

   ![A screenshot of a sample Agreement Desk Send Message dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-send-msg-db.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0MzcsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.px29PCu5YLhcnu146mpM3yemtIjoyBDguz89t9zLPl8&_LANG=enus "A screenshot of a sample Agreement Desk Send Message dialog box.")
8. Select Send.

Your message is sent, and is reflected in the Activity
Feed. The recipient receives an email with the message and any
attachments.

If
the Account Admin turns on message participation, you
can
add recipients who are not in your account to the message. See the [Permission Profile Options](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=mfo1583277360424.html&_LANG=enus)﻿ article﻿ for more information.
