# Reply to and Resolve Comments from the Sidebar

Learn how to use the Agreement Desk comments sidebar to directly respond to
and resolve from the comments sidebar.

You can use the Agreement Desk comments sidebar to directly respond to
and resolve from the comments sidebar.

1. Select the request that contains the document you want.
2. Select the Documents tab.
3. Select the document that contains your comments.
4. Select Preview from the vertical three dot menu to
   display the document and its comments.
5. Select the comments icon to open the comments sidebar.
6. In the comments list, select the comment you want to act on.
7. Reply to a comment:
   1. Select the reply field below the comment.
   2. Enter your response.
   3. If needed, use @mention to notify another user.
   4. Select Send to add your reply to the
      thread.
8. Resolve a comment:
   1. Select Resolve from the comment's overflow
      menu.
   2. Enter your response.
   3. If needed, use @mention to notify another user.
   4. Select Send to add your reply to the
      thread.
9. Reopen a resolved comment:
   1. Change the filter to only show resolved comments.
   2. Open the comment and select Reopen.
   3. Select Save.

      Tip:

      Use replies for ongoing discussion and
      Resolve when an issue or question in
      the comment has been addressed.
