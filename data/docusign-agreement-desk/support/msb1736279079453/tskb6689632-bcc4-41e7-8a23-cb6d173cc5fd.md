# View All Agreement Desk Request Followers

Learn how any user with access to a request can view a list of all other users actively
following that request.

Now, any user with access to a request can view a list of all other users
actively following that request. This ensures that they receive email notifications for updates and changes.

This feature provides:

- At-a-glance visibility: View up to three user avatars directly that
  display near the Follow button.
- Detailed follower support: Click on the avatars or the overflow
  count to display a full list of followers
- Dynamic updates: The follower list updates automatically.

Note: Owners of a request are automatically made a follower of the request. This ensures that owners automatically receive follower notifications.

1. Select Requests from the left navigation area. The
   Requests dashboard displays.
2. Select a request to open it.

   In the menu bar, the followers’ avatars display next to the
   Follow and Share
   options.
