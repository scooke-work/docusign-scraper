# Agreement Desk Redline Summarization (Beta)

Learn how to use the  Agreement Desk Redline Summarization feature to
summarize revisions and comments in Word documents, providing an easy way to track
changes.

Use the Redline Summarization
feature
to summarize revisions and comments in Word documents, providing an easy way to track
changes.

Note: nly tracked changes or comments contained in a Word document display in the summary.

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Select a request to open it.
3. Select Documents.
4. For the document you want to review, select Summarize changes
    from the vertical three dot menu.

   ![A screenshot of selecting Summarize changes from the Agreement Desk request document actions menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ac-ai-markup-sum-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0NjMsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.J7iN4LbS7u4Ii7_mVSV2crV4nRXG0Oa3Ezy4bRycSnk&_LANG=enus "A screenshot of selecting Summarize changes from the Agreement Desk request document actions menu.")

   The AI agent panel opens and displays the changes in the document.

   ![A screenshot of a sample Agreement Desk request document markup summary.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ac-ai-markup-sum-results.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0NjMsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.J7iN4LbS7u4Ii7_mVSV2crV4nRXG0Oa3Ezy4bRycSnk&_LANG=enus "A screenshot of a sample Agreement Desk request document markup summary.")
5. Optionally select one of these options to capture the summary:
   - Copy response copies the summary to the
     clipboard.
   - Draft a message about this summary creates an email that includes the
     summary as described in [Agreement Desk Message Drafting](https://support.docusign.com/s/document-item?bundleId=msb1736279079453&topicId=tskb9bdd359-8691-4eb6-a0a4-e562a81aea92.html&_LANG=enus "Learn how to use the Agreement Desk message drafting feature to turn text into a message form and capture the key points of the text.").
