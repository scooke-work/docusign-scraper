# Create a Request from the Requests Dashboard

Learn how to create a specific request type from the Agreement Desk
Requests dashboard.

Create a specific type of request directly from the Agreement Desk
Requests dashboard.

1. Select the Agreements tab in the top navigation.
2. Select Requests from the left navigation area.

   The Requests dashboard displays.
3. Select Create Request.

   ![A screenshot of the Agreement Desk Requests dashboard Create Request feature.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-req-create-sel.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzNTksInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.gTlspYAQG_Q6hBPyIeC3uRuNvOuFVvga1oub_zws-M4&_LANG=enus "A screenshot of the Agreement Desk Requests dashboard Create Request feature.")

   The What type of request is this?
   page opens with a list of request types to choose
   from.

   ![A screenshot of a sample Agreement Desk Create a request dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-req-create-db.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzNTksInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.gTlspYAQG_Q6hBPyIeC3uRuNvOuFVvga1oub_zws-M4&_LANG=enus "A screenshot of a sample Agreement Desk Create a request dialog box.")
4. Select the type of request that you want to create.
5. Complete the form and select Next.
6. Optionally
   create
   your own title.

   Enter a text-only title to serve as the request's
   official designation within Agreement Desk and throughout IAM.

   A summary of your request displays.
7. Review the summary and edit the information if necessary and select Submit.

A confirmation page displays with the option to view the request or select Done.

## Next steps

Select View request to open the request and work with it. Select Done to return to the Requests dashboard.
