# View Documents Associated with Agreement Desk Requests

Learn how to view the documents associated with Agreement Desk requests.

You can view documents associated with an Agreement Desk request.

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Select a request and select the Documents tab.

   Documents associated with the request are listed by title. Columns
   provide the status, date the document was added, and the name of the user who
   added it.

   ![A screenshot of an documentss associated with an Agreement Desk request.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-add-docs.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzNzYsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.BnxKpXcRYW7ge5_0-S3JNhYvVWA5lFo7UaZolIhgZ8w&_LANG=enus "A screenshot of an documentss associated with an Agreement Desk request.")

   Documents can be one of these statuses:
   - **Draft**: The document has not changed.
   - **Signed**: The document was signed through a Docusign eSignature
     envelope.
3. (Optional) Add a document, generate a document, or send a document for signature. Or take action on a specific document, such as downloading it.
