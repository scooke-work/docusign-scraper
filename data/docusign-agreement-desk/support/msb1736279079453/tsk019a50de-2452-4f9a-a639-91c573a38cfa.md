# Compare Documents in Requests

Learn how to compare Agreement Desk request documents to view the latest
version of a document and immediately compare it to a different version.

Use the Agreement Desk Document Compare feature to quickly view a comparison
between two documents from any Agreement Desk request.

You can compare:

- .docx files
- PDF files

Note: The PDF files are automatically converted to .docx
files during the comparison.

1. Select Requests from the
   Agreements page.

   The Requests dashboard opens.
2. Open the request that you want.
3. Select the Documents tab.
4. For the document you want to compare, select the vertical three-dot menu and
   select Compare.

   The Compare documents dialog box
   displays.
5. In the Compare with  menu, select the document and the
   document version you want to use for the comparison.
6. Select Compare to begin the comparison.

7. When the comparison completes, the right pane shows a preview of the document
   comparisons, including additions and deletions.

   ![A screenshot of sample Agreement Desk Compare Documents results.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-new-comp-results.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0MDAsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.cLpkbyuoDdquODNzVoWLEHTkKuWGqQww67fSjJXgwWg&_LANG=enus "A screenshot of sample Agreement Desk Compare Documents results.")
8. Using the menu in the upper right, perform one of these actions

   ![A screenshot of the Agreement Desk Compare Documents finalization options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-comp-actions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0MDAsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.cLpkbyuoDdquODNzVoWLEHTkKuWGqQww67fSjJXgwWg&_LANG=enus "A screenshot of the Agreement Desk Compare Documents finalization options.")

   - From the vertical three-dot menu, select Save as New Version
     to directly save this comparison document as a new
     version.
   - Select Edit in Word to open the composite document
     and make changes.
   - Exit from the preview without changing or saving the document.
