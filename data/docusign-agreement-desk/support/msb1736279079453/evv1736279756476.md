# View Requests

Learn how to view Agreement Desk requests.

View Agreement Desk requests from the Requests
dashboard.

1. Log on to Docusign and select
   Agreements.
2. Select Requests from the left navigation area.
3. Select any request to open it and view request details.

## Next steps

Search and filter requests and take action on a specific
ticket.

## View Request details

You can view detailed information in an Agreement Desk request.

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Select any request to open it and view request details.

   ![A screenshot of a sample Agreement Desk request detail overview page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-req-det-over-pg.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzMzMsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.HVPx40hlGQ-AB__eAJlvXtkAB0Q-nx_zibPrAHcr5_k&_LANG=enus)

   Each menu tab displays information about the request.
3. View information and perform these operations.

   | On this tab: | You can: |
   | --- | --- |
   | Overview | - Filter the display to show all activity or messages - Send a message about the request - View request activity history - View request summary - Summarize the request with the AI - Reply through email to the request originator - Delete the request |
   | Details | - Edit the request information - Change the request type - View detailed and summary request information |
   | Documents | View information about request documents - Send the document for signature - Add documents:   - Upload other documents   - Upload a new version   - Generate a document by selecting a document type and     creating the request - Manage a document by selecting the three   vertical   dot menu to perform these options:   - Preview   - Download   - Rename   - Delete   - View document's version history   - Upload a new document version |
   | Approvals | View associated approvals |
   | Envelopes | View associated envelopes |
