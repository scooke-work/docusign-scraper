# Use Saved Search for Agreement Desk Requests

Learn how to create customized searches to retrieve an updated list of requests that
match the search criteria.

You can create customized searches to retrieve an updated list of requests that match
the search criteria.

When you run the search, the search criteria (one or more request filter types) are
applied to the requests.

1. Select Requests from the left pane navigation to access
   the Requests dashboard.
2. Select the Show Hide icon to display filter fields to create the search criteria.

   ![A screenshot of the Agreement Desk request search show or hide fields icon..](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-show-fields-icon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0OTIsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.zV6JroYRjL8ja8sa4tsaYy1QkpXg4UGBp4D7cHWitnM&_LANG=enus "A screenshot of the Agreement Desk request search show or hide fields icon..")

   For example, you may select submitted by a specific
   user and unassigned.
3. Select the Save search text next to the Managed Search icon

   ![A screenshot of the Agreement Desk request Save Search feature.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-save-search-select.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0OTIsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.zV6JroYRjL8ja8sa4tsaYy1QkpXg4UGBp4D7cHWitnM&_LANG=enus "A screenshot of the Agreement Desk request Save Search feature.")

   The New Saved Search dialog box displays.
4. Enter a name and description and select Save.

   ![A screenshot of a sample Agreement Desk new saved search.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-ss-db-complete.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0OTIsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.zV6JroYRjL8ja8sa4tsaYy1QkpXg4UGBp4D7cHWitnM&_LANG=enus "A screenshot of a sample Agreement Desk new saved search.")
5. Optionally change your saved search by selecting the saved search icon. You
   can:
   - Select a recent search that is listed in dialog box
   - Select Manage saved searches
6. Select the vertical three dot menu and perform one of these actions:

   - Save New Version: Rename the search, but maintain the description.
   - Edit the name and description and select
     Update.
   - Delete: Select Delete in the confirmation
     message.

     Note: Deleting a search does not
     affect any requests.
7. Access your saved searches by selecting the bookmark icon.

   ![Agreement Desk saved searches icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-save-search-bkmark.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjI0OTIsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.zV6JroYRjL8ja8sa4tsaYy1QkpXg4UGBp4D7cHWitnM&_LANG=enus "Agreement Desk saved searches icon.")
