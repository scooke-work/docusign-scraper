# Manage Envelopes from Agreement Desk

Learn how to manage the contents of an Agreement Desk request's
envelopes.

## Before you begin

Your permission profile must include Docusign Send (DS Sender) or Docusign Admin (DS Admin) privileges to perform these tasks.

When an Agreement Desk request includes an envelope, you can view and manage the
envelope and its documents from Agreement Desk.

1. Select Requests from the left navigation area.

   The Requests dashboard displays.
2. Select the request that contains the envelope you want.
3. Select the Envelopes tab to display the envelope
   details.

   ![A screenshot of an Agreement Deskrequest's envelope page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-envelope-details-pg.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzODgsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.K3ypY0s022LsT_Vq1Yc_sEW2GGcudveRljR1LzCNYXk&_LANG=enus "A screenshot of an Agreement Deskrequest's envelope page.")
4. Select the envelope you want to manage to display the envelope's detail page.
5. Select one of these activities:
   - View recipients by selecting the Recipients tab
     (default display).
   - View envelope details by selecting the Details  tab.
   - Depending on the envelope status, Send or Resend the envelope.
   - Select a thumbnail of associated documents to view the
     document.
   - Select Certificate to download the envelope's
     [Certificate of Completion](https://support.docusign.com/s/document-item?bundleId=oeq1643226594604&topicId=gpa1578456339545.html&_LANG=enus). This certificate
     includes the identifying information about the envelope and complete details
     of the envelope events.
   - Using the three-dot menu next to the envelope to select additional actions.

     ![A screenshot showing the Agreement Desk request envelope action menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/msb1736279079453/page/images/ad-envelope-pg-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjIzODgsInNoZWFmIjoibXNiMTczNjI3OTA3OTQ1MyJ9.K3ypY0s022LsT_Vq1Yc_sEW2GGcudveRljR1LzCNYXk&_LANG=enus "A screenshot showing the Agreement Desk request envelope action menu.")

     - Details : View envelope details, such as
       envelope name and ID, sender, and send history.
     - Void: Void the envelope as described in the
       [Void an Envelope](https://support.docusign.com/s/document-item?bundleId=oeq1643226594604&topicId=zyn1578456424164.html&_LANG=enus)  article.
   - Select the overflow menu (the three dot menu) at the top of the page to
     view more envelope management options as described in the topic [Complete Envelope
     Activities](https://support.docusign.com/s/document-item?bundleId=msb1736279079453&topicId=tske30a78d7-9149-47e5-9270-32a8a4934cb0.html&_LANG=enus "Learn how to manage and complete envelope activities from a DocusignAgreement Desk request.").
