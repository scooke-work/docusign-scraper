# Populate Room Details to Capture Transaction Data and Pre-Fill Forms

Learn how to save time by completing the room detail fields for a transaction. Once
completed, these fields can pre-fill forms, trigger due dates, and send reminders for your
tasks.

## Before you begin

Before you begin, you need a room role with Edit room
details permission.

Every room has a Details tab, which is like a cover sheet for the transaction. Your
administrator has configured fields that display on the Details tab, and these fields
display on every room created in your company. You can, and in some cases, must complete
these fields before a transaction can be closed.

It's a good idea to fill in the room details upon creating a room because some fields
pre-populate forms, trigger due dates, and send reminders for task lists. You can then
continue to complete these fields throughout transaction lifecycle as needed.

1. Open the room.
2. Select the DETAILS tab.
3. Select EDIT.
4. Edit the fields as needed. Some fields may be read-only.
5. Select SAVE.
