# Split a PDF Document Using a Template

Learn how to use your company split templates to divide PDF files in Docusign Rooms.

If your company provides templates for splitting PDF
files, you'll see a list of templates when splitting documents. You can make adjustments
after the split.

1. In Docusign Rooms, right-click the PDF file and select
   Split.
2. Select APPLY TEMPLATE.

   The Apply PDF Split Template window displays.

   ![rooms-apply-pdf-split-template](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsapplypdfsplittemplate.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE3MjAsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.njt0VYC10FtU3S4dUHlKfIPxB8d3pgmNK7gZD-oahrE&_LANG=enus "rooms-apply-pdf-split-template")
3. Choose the template.
4. Select APPLY.

   The settings from the template are applied to the document.
5. Make any adjustments as needed. For example, you can rename the split
   documents, adjust the page ranges, or add another document.
6. Select SAVE.
