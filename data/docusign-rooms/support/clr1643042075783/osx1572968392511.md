# Task documents

If the task requires someone to complete a document, type the document name in the
DOCUMENTS tab.

![rooms-task-document](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomstaskdocument.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE4NjAsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.lcO8q6YFsWApzYJVaOtolwGMd0DyAItqBW8pBezJAso&_LANG=enus "rooms-task-document")

Adding a document name creates a placeholder on the DOCUMENTS tab,
so users know which document to provide.

![rooms-task-placeholder-appraisal](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomstaskplaceholderappraisal.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE4NjAsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.lcO8q6YFsWApzYJVaOtolwGMd0DyAItqBW8pBezJAso&_LANG=enus "rooms-task-placeholder-appraisal")

Users can apply more than one document to a placeholder. Therefore, you can create one task
to upload multiple related documents.
