# A “New Version” Notification Displays When I Apply a Form Template

I see the message "There's a new version of <form name> available. You can update the
template in My Docs" when I try to apply a form template.

Docusign Rooms displays this message when a new version of a form is published,
and therefore, the form and template versions do not match. When you see this
message, review your form template to see if updates are required. You can use
the form template, but you will continue to see the message until you open and
save the template. Saving the template updates the template version to match the
form.

## Open your form template, then save it

1. Log in to Docusign Rooms.
2. Select My Docs in the application header.
3. Select the FORM TEMPLATES tab.
4. Select EDIT next to the template that is causing
   the message.
5. Select SAVE & CLOSE.
