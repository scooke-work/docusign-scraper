# Move Documents Between Folders

Learn how to organize documents in Docusign Rooms by moving them to different
folders.

Adding a document to a room initially places it in the ROOM DOCS folder. You can leave
your documents there or move them to different folders to organize them.

## Move documents to a folder in the same room

Use drag and drop when moving documents between folders *in the same room*.

Drag documents into different folders or even onto task lists. If you drop a new
document into a task placeholder that already has a document, the new document
replaces the original one. The old document that's being replaced moves back to its
previous location, either in a folder or the ROOM DOCS folder.

## Move documents to a different room

If a document belongs to a different transaction, move it to the other room. The document is
placed in the ROOM DOCS folder of the new room.

1. In the room DOCUMENTS tab, select the document(s) you want to
   move.
