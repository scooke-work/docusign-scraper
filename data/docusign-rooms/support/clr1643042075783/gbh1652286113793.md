# View the Status or Results of a Member's Room Access Update

## Before you begin

You need a company role with at least one of the following
permissions enabled:

- Manage members with lower access level
- Manage members with the same access level

1. In Docusign Rooms, go to Settings > MEMBERS.
2. Select the member.
3. Select Bulk Updates.

   ![rooms-bulk-updates-success](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsbulkupdatessuccess.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE1NDYsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.hg_d_oM_LYgJmvvJgxUEERdMaCY9xbX03NcCO3IRKqw&_LANG=enus "rooms-bulk-updates-success")

   This page lists activity within the last 90 days. The
   Status column lists the status of each job:
   scheduled, running, or completed.
4. Select VIEW to read a summary of the results (for
   completed jobs only).
5. Select RESULTS DETAIL REPORT to download a CSV file that
   lists the rooms to which the member was added or removed.
