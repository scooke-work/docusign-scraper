# Add or Remove Someone From a Message

If you created a topic in a room, you can add and remove room members as participants. For
example, you might want to remove someone from the conversation if their participation
is no longer needed.

## Add a topic participant

When you add someone to a topic, they see it as an unread topic in the New
Messages section of their Rooms dashboard. They also have access to
the topic through the room MESSAGES tab and will receive a notification (if they
have notifications enabled for messages).

New topic participants can read all previous messages within a topic as well as send
new messages.

1. In the room MESSAGES tab, select VIEW for a topic to open it.
2. Select ADD PARTICIPANT.
   ![A list of participants within a topic with the ADD PARTICIPANT button highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsmessagesaddparticipanthighlight.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE2NTksInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.LU8_s5dqO76i9PyGcYKsh9NQvZb4FH0i2zd-Lwit8RM&_LANG=enus "A list of participants within a topic with the ADD PARTICIPANT button highlighted.")
3. From the list of room members, select the people to add to the topic.
4. Select ADD.

## Remove a topic participant

Topic creators can remove a participant from a topic. If you remove a participant, the topic
is completely removed from their view. They won't see the topic in the room MESSAGES
tab or receive notifications.

1. In the room MESSAGES tab, select VIEW for a topic to open it.
2. In the Participants list, select the "x" icon next to the person to remove.
   ![A list of room participants with an "x" icon next to a participant that can be removed](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsmessagesremoveparticipant.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE2NTksInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.LU8_s5dqO76i9PyGcYKsh9NQvZb4FH0i2zd-Lwit8RM&_LANG=enus "A list of room participants with an \"x\" icon next to a participant that can be removed")
3. Select REMOVE.
