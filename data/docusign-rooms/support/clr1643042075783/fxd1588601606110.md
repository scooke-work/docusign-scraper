# Add Trusted URLs for Docusign Rooms Redirects

When the Manage integration settings permission has been enabled on a
role, users with that role see the Integration settings
option in company settings.

![rooms-admin-integration-settings](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsadminintegrationsettings.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE0OTIsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.Zp1TIcpG77lgdWZJOMZOqAtYSRoXx0kvzYMV2j00bXk&_LANG=enus "rooms-admin-integration-settings")

Here, you can build a trusted URL list that Rooms will use for redirect after member
activation.

To add URLs:

1. In your Rooms account, go to Settings >  COMPANY >  Integration Settings.
2. Select ADD URL.
3. Type the trusted URL.

   Rooms verifies that the URL is valid.
4. Select SAVE.

The URL is added to the list. Select Add
URL to add additional URLs or select Edit or
Delete to manage existing URLs.
