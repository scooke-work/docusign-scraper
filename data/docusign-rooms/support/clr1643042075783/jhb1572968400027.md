# Copy a Room To Create a New, Similar Room

Learn how to copy a room to pre-populate a new room with the same transaction details
as the original room.

Copying a room creates a new room with the same details as the original room. For example, if an offer falls through and you have to create a
new room for the same property, copy the original room to pre-populate a new room with the
property and seller information.

You can copy a room in any status (Active, In Review, or Closed).

1. In your Rooms account, select the ROOMS tab at the top of the
   page.
2. Right-click the room you want to copy, then select Copy
   Room.
3. Type a name for the new room in the Name field.
4. Select COPY.
