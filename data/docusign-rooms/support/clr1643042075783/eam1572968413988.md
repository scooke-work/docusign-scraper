# Use Your My Docs Folder To Store Private Documents

Read about the My Docs folder in Docusign Rooms and how you can use it to create your
own document library.

My Docs is your own document library. Each member has one and no one else in your company can see or access it.

Create a folder within My Docs by clicking the ADD folder button in the upper right corner. You can also add documents to My Docs by clicking the ADD document button to the right of the ADD folder button. Add documents from your computer or cloud storage. Cloud storage integrations can be linked in your Settings.

There are many ways you can take action on a document, such as right-clicking on a document in the list or using the icons at the top of the My Docs tab. Additionally, you can open and view documents and then use the Document Actions menu and icons.

In the upper left, the Document Finder lets you search My Docs for a document by its name. Next to the Document Finder is a drop down menu that lets you filter the document list.
