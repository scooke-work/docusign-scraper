# Create Task List Templates to Group Common Transaction Activities

## Before you begin

You need the Manage task templates
permission enabled on your company role. This permission lets you create and edit task list
templates for the offices and regions you manage.

Tip: If you already have a template similar to the one you want to create, click
the drop-down menu next to that template and choose Copy.

Note: If you create a task list template and
share it with multiple offices or regions, other members cannot edit that template, even if
they have Manage task templates permission. (Only the template
creator can edit templates they enable for multiple offices or regions.)

1. Select Settings in the Rooms header.
2. Select the COMPANY tab.
3. Select Task List Templates.
4. Select NEW TEMPLATE.
5. In the Name field, type a name for the template.
6. To enable this template for specific offices or regions, select these options in the
   Enabled in field.
7. Select ADD TASK to begin adding tasks to the template.
8. For each task, add the [details](https://support.docusign.com/s/document-item?bundleId=clr1643042075783&topicId=aks1572968392261.html&_LANG=enus), [documents](https://support.docusign.com/s/document-item?bundleId=clr1643042075783&topicId=osx1572968392511.html&_LANG=enus), [people](https://support.docusign.com/s/document-item?bundleId=clr1643042075783&topicId=pli1572968392747.html&_LANG=enus), and [reminders](https://support.docusign.com/s/document-item?bundleId=clr1643042075783&topicId=byb1572968392935.html&_LANG=enus) related to the
   task, then select SAVE.

   When you are finished, tasks display under the TASKS section of the template. Select
   EDIT to make additional updates on a task or select ![rooms-drop-down-arrow](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsdropdownarrow.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE0NDIsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.5Z-KjMxPVE7lFzomDaTphQd4wuzvw1DMiPKkrMlwNss&_LANG=enus "rooms-drop-down-arrow") then Delete to remove a task.

   ![rooms-task-list-template](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomstasklisttemplate.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE0NDIsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.5Z-KjMxPVE7lFzomDaTphQd4wuzvw1DMiPKkrMlwNss&_LANG=enus "rooms-task-list-template")

   Note: You can change the order by
   typing a new number next to a task. In a room, tasks display in this order by default,
   but users can sort them differently.
9. Select SAVE.
