# Remove Yourself From a Message

If you leave a message topic, it will be removed from your message list and you will stop
receiving topic notifications.

Note: You can't leave a topic if you created it.

1. Navigate to the room MESSAGES tab.
2. Select VIEW for a topic to open it.
3. Select ![gear icon](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsgearicon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE2NjIsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9._LCcB6eUIAtcxYPHO8A-tMpGeX89SZQNbpBMJvCeuDs&_LANG=enus "gear icon").
4. Select Leave topic.
5. Select LEAVE.
