# Group Real Estate Forms and Make Them Visible to Offices

Learn how to create form packets, so agents know which forms to add to rooms based on
the type of transaction they are working on. Once form administrators create packets, they can
make them required, ensuring compliance.

## Before you begin

Before you begin, you need a company role with Manage
form packets permission.

Form administrators at your company are responsible for managing association forms for your
agents. With the appropriate permissions, form administrators can create form packets so
agents know which forms to add to rooms based on the type of transaction they are working
on. When creating packets, administrators can make certain forms required, ensuring
compliance.

If you don't create packets, agents will have to choose forms they need from a list of
association forms.

1. Log in to your Rooms account.
2. Select Settings in the Rooms header.
3. Select the FORMS tab.

   The Forms page displays folders for each of your form libraries and packets. Expand a
   folder to view the forms within the library or packet.
4. Select ADD > ADD PACKET.
5. Give the packet a name.

   Name the packet so agents can identify which forms are inside.
6. Select the offices that should see this form packet.

   Click inside the offices' text area to display a drop-down list of offices you can
   manage, then select the offices.
7. Select ADD.

Once you have added the packet, select ADD FORMS within the packet
to add forms from your form libraries.
