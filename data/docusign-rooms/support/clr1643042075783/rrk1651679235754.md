# Supported File Formats in Docusign Rooms

Read about the types of documents you can add to Docusign Rooms.

Docusign Rooms supports the following file types:

|  |  |
| --- | --- |
| DOCUMENT | .doc, .docx, .pdf, .rtf, .txt |
| IMAGE | .bmp, .gif, .jpg, .jpeg, .png, .tif, .tiff |
| PRESENTATION | .ppt, .pptx |
| SPREADSHEET | .csv, .xls, .xlsx |

Note: You can upload a file with password security
enabled, but you can't add it to an envelope. To send a file with password security,
remove the security setting before uploading the file. Password-protected documents also
cannot be viewed in the DOCUMENTS tab of a room.
