# Start a New Message To Connect With Room Members

1. Open the room.
2. Select ![rooms-new-message-icon](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsnewmessageicon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE2NTUsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.7Qs2oa3P1NpFnx6xM8uUDBanWBAsHi5x05PqL84O3hI&_LANG=enus "rooms-new-message-icon").
3. Click inside the Recipients field to select the room members
   you'd like to message.
4. Type the subject of your message in the Topic field.
5. Type your message.
6. Select Send.

The participants receive an email notification (unless they've turned off notifications for
messages). They always, however, see the message in the New Messages
section of the Rooms dashboard.

Tip: Want to add comments to yourself that nobody else can see? Start a new topic
and choose yourself as the only participant.
