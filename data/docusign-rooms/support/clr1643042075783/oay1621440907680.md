# Change the Transaction Side of a Room

If you accidentally chose the wrong transaction side (list, buy, or list and buy) when
you created a room, learn how to easily change it.

## Before you begin

Before you begin, you need the Edit room
side permission enabled on your room role.

If you accidentally chose the wrong transaction side when you created a room, you can
easily change it.

When you change the room side, you also change the transaction side for all the room
owners. If there are room detail fields specific to the current side of the
transaction, those won't be visible after you switch sides.

Note: Room ownership doesn't control whether you can edit the room
side. Any room member with the required permissions can change this
information.

1. Open the room Details tab.
2. Select EDIT.
3. Change the side in the Side field.
