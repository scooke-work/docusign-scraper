# Upload the CSV File

Lastly, upload the updated CSV file to import the changes.

1. Go to Settings >  COMPANY >  Task List Templates.
2. Select BULK UPDATE.
3. In the Upload CSV File section, drag and drop the CSV file to the designated
   area. Alternatively, click UPLOAD to navigate to the
   file.

   The CSV file name displays in the upload area.

   ![rooms-bulk-upload-modal-upload](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsbulkuploadmodalupload.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE0NTcsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.f-r2pg_YQSe_XK5ajalsZKs5JlROSWoY6S--W7yfdi0&_LANG=enus "rooms-bulk-upload-modal-upload")
4. Select SUBMIT.

The update process might take a few minutes, depending on how many
tasks you updated. When the update is complete and you refresh the page, you will see a
notification at the top of the screen in Settings >  Task List Templates. Click the View results link to view a summary of
the updates or download a detailed report.
