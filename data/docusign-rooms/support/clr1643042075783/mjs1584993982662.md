# Enable Two-Step Verification for a Role

When you enable two-step verification for a role, the
updates take effect immediately. Users with that room role are prompted for a security
code the next time they attempt to access the room.

1. Log in to your Docusign Rooms account.
2. Select Settings in the Rooms header.
3. Select the COMPANY tab.
4. Select Roles and Permissions.
5. Select EDIT next to the role for which you'd like to
   enable two-step verification.
6. Enable Require two-step verification to access rooms.
7. Select SAVE.

Tip: When inviting someone to a room, you can
check the role's verification settings by selecting VIEW
PERMISSIONS, then expanding the Role Settings section.
