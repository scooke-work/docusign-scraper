# Task Details, Documents, People, and Reminders

When creating or editing a task, complete the information in the Task
Details, Documents,
People, and Reminders sections.

## Task details

Add the general task information and settings in the Task Details
section.

![rooms-task-details-W2](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomstaskdetailsW2.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE4MzAsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.pYhLXYoJOQ4c2zOZ_0VbTnkmpB66GzVM46U4WOt3lb0&_LANG=enus)

The table below describes the fields and options in this section.

| Task Field | Description |
| --- | --- |
| Task Name | Enter a task name. |
| Description | Optionally add a description. If the task doesn't require a document, use the description to communicate what needs to be done. |
| Due Date | Optionally select a due date. The dates in the drop-down list are specific dates configured in the room's Details tab. For example, if you select the transaction close date from the list, the task looks to the room's close date specified on the Details tab to determine the due date.  If you select a due date, a numerical field displays. This sets the number of days from the due date that the task must be completed. For example, if the due date is closing day on 1/15 and the date offset is -2 days, the task is due on 1/13. |
| Required Task | Select if the task must be completed to close the room. If the task is on a task list, the task must be completed before submitting the task list for review. Required tasks must be reviewed. If you select Required Task, the Requires Review option is selected and disabled. |
| Requires Review | The task requires review. If enabled, you are required to choose a reviewer in the People section. |
| Assignee Can Edit | Select to allow edits to the task. To edit the task, members must be on the task (as a watcher, assignee, or reviewer) and have a room role with permission to edit tasks. |
| Assignee Can Delete | Select to allow members to delete the task. To delete the task, members must be on the task (as a watcher, assignee, or reviewer) and have a room role with permission to delete tasks. |

## Task documents

If the task requires someone to complete a document, click ADD in the
Documents area to add the document name to the task. Type the document name and
click SAVE.

Adding a document creates a placeholder so users know which document should be attached to the
task. If the task is required, the document displays
(Required) under the name.

![rooms-task-documents-W2](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomstaskdocumentsW2.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE4MzAsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.pYhLXYoJOQ4c2zOZ_0VbTnkmpB66GzVM46U4WOt3lb0&_LANG=enus)

Users can apply more than one document to a placeholder. Therefore, you can create one
task to upload multiple related documents.

## Task people

The room members involved with the task are listed in the People section.

![rooms-task-people-two-levels](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomstaskpeopletwolevels.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE4MzAsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.pYhLXYoJOQ4c2zOZ_0VbTnkmpB66GzVM46U4WOt3lb0&_LANG=enus)

Click ADD to add room members to the task.

![rooms-task-add-people](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomstaskaddpeople.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE4MzAsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.pYhLXYoJOQ4c2zOZ_0VbTnkmpB66GzVM46U4WOt3lb0&_LANG=enus)

Under each responsibility, use the add link (Add
assignee/Add reviewer/Add
watcher) to add the members with that responsibility.

Note: If you are editing a task with multiple review levels, you can add and remove
reviewers from each level. Each level requires at least one reviewer. You can't add
or remove review levels (these are set on the template).

See descriptions of the task responsibilities below.

| Responsibility | Description |
| --- | --- |
| Assignee | Responsible for completing the task.  The assignee receives notifications when they are initially added to a task, task reminders, due dates, and action is taken on the task. |
| Reviewer | Responsible for approving or declining the task.  If you designate at least one reviewer, the task is set to Requires Review. Reviewers receive a notification when the assignee completes the task. |
| Watcher | Receives notifications when the task is completed or approved. |

## Task reminders

To add a reminder, click ADD in the Reminders section.

For each reminder, select the calendar or room date you'd like the reminder to be associated
with. For example, if the reminder should be sent near the transaction closing date,
choose the closing date. Use the Date offset days (+/-)  to
set the number of days from the due date that the reminder should be sent (or leave
blank to send the reminder on that day).
