# Combine Documents

Learn how to combine multiple documents in Docusign Rooms into a single
document.

1. In Docusign Rooms, select the documents to combine.
2. Right-click and select Combine.

   ![rooms-combine-docs](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomscombinedocs.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE3MjQsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.qLEzX95aRTCQ5QUyGJ7ArcFV6oeaoP-BA9pl13G-QSE&_LANG=enus "rooms-combine-docs")
3. Type a name for the new document.
4. Select SAVE.
