# Stop Receiving Message Notifications

You receive an email notification each time someone posts a new message. If you don't want
to receive email notifications for room messages, you can opt-out.

## Stop all message notifications

1. In the Docusign Rooms header, select your profile picture or initials.
2. Select Preferences.
3. Select Notifications.
4. Clear the Someone sends a message in a topic where you are a
   participant check box.
5. Select SAVE CHANGES.

## Mute notifications for a single topic

1. Open the room.
2. Select the MESSAGES tab.
3. Select the drop-down arrow next to the topic to mute.
4. Select Mute topic.

If you want to start receiving notifications again, select the drop-down arrow next to the
topic, then select Unmute topic.
