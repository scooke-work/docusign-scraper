# Add Documents to Your Personal My Docs Library

Learn how to upload documents to a private location in Docusign Rooms that's only
visible to you. Adding documents to My Docs stores them in a secure, convenient area so you
can move or copy them to your transactions when you are ready.

You can upload documents to a private location in Docusign Rooms that's only visible to you.
Adding documents to My Docs stores them in a secure, convenient area so you can move
or copy them to your transactions when you are ready.

1. Select My Docs in the Rooms header.
2. Select ![rooms-mydocs-add](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsmydocsadd.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE2OTYsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.KTrq8W2CJsBzJcdE0jRB5s5bvJ6W3E-1yWv2VkQJkXo&_LANG=enus "rooms-mydocs-add").
3. Select Computer or one of the cloud storage
   sources.

   When choosing a cloud storage source, you will be prompted to log in if you
   aren't currently logged into the service.
4. Navigate to and select the documents to add.
