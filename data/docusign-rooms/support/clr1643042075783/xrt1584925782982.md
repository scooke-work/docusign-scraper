# Hide a User Role To Prevent Members From Selecting It

Administrators can enable or disable user roles to control which roles are visible to
the company. Disabling a role hides it and prevents members from using it, such as when
inviting someone to a room.

You can enable and disable user roles to control which roles are visible to your company.
Disable roles to hide them. For example, if you are working on a new role, you can
disable it so it is hidden from users.

An Enabled or Disabled indicator displays for
each role.
