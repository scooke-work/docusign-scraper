# Access a Room Using Two-Step Verification

If your room role requires two-step verification, you are prompted to verify a security code
before you can access the room.

1. Click the link in the room invitation email to sign in.

   You can also use your browser to go to the Rooms login page.
2. If you have a Rooms account, type your email address and password to log
   in.

   If you used a room invitation email link to access Rooms and don't have an
   account, follow the on-screen steps to create a password, activate your account,
   and accept the agreement.
3. Select how to receive your verification code (for example
   Email).

   ![rooms-disable-role](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomstwostepmethod.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE0MDUsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.Lf3_WhkNhS1y6uaR1PHk04iezPLx3umLuydMOZDCTmk&_LANG=enus "rooms-disable-role")

   Likely, you will only have one option here for email since email is the
   default notification option. If you updated your notification preferences to use
   different methods, you'd see those options here.
4. Once you receive the code, enter it in the space provided and select
   CONFIRM.

   ![rooms-disable-role](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomstostepcode.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE0MDUsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.Lf3_WhkNhS1y6uaR1PHk04iezPLx3umLuydMOZDCTmk&_LANG=enus "rooms-disable-role")

   You are taken to the room.
