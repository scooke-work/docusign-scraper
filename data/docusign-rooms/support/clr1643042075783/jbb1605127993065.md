# Review a Task List

You will receive an email notification when a list is ready for review.

Note: Reviewing task lists requires a room role
with permission.

1. On the room TASKS tab, choose ACTIONS >  Review task list.
2. If there are multiple task lists in the room, select the task list you'd like
   to review.
3. Select Approve or Correction
   Required to decline the task list.

   If you decline the task list, you are required to add a comment.
