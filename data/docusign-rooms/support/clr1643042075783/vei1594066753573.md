# Bulk Update Roles on Task List Templates

Learn how to use bulk update tools to make sweeping changes to roles on task list
templates. These tools allow administrators to update all roles at once rather than edit
each template individually.

When you update company roles, you might also need to update task list templates to reflect the
changes. For example, adding or removing manager roles might require updates to task
reviewer roles on your templates.

Use the bulk update feature to make sweeping changes to task roles. This feature lets you
export your task list templates to a CSV file, update task roles and responsibilities
within the file, and then upload it back into Rooms to update your templates.

To bulk update task roles, follow this 3-step process:

- Step 1: Export your task list templates
- Step 2: Update roles in the CSV file
- Step 3: Upload the CSV file
