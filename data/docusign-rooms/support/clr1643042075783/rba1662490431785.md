# Add Completed Envelopes in Rooms

Learn how to add completed envelopes in eSignature to a room in Rooms.

This guide shows how to add completed envelopes in Docusign eSignature to a room in Docusign Rooms.

1. Go to the room where you want to add the completed envelopes, and while in the
   desired room, go to Envelopes > Actions.![](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/01.jpg?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE3NTIsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.D-oTmaxHzOYTJCI_IGSKfIvaNp1QPuCEpUkD23IDzjI&_LANG=enus)
2. While in **Actions,** click **Add Existing Envelopes**.![](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/02.jpg?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE3NTIsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.D-oTmaxHzOYTJCI_IGSKfIvaNp1QPuCEpUkD23IDzjI&_LANG=enus)
3. While on the **Add Existing Envelope** screen, there will be an option to
   select a time frame and the option to search for the name of a specific
   envelope.
4. Choose the desired envelope and click **Add**. The envelope will then be
   added to the desired room.![](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/04.jpg?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE3NTIsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.D-oTmaxHzOYTJCI_IGSKfIvaNp1QPuCEpUkD23IDzjI&_LANG=enus)

   Note: An error will occur when you add an envelope to a
   room multiple times and when you add an envelope to multiple rooms.
   - The following error appears when an envelope is added to a room
     multiple times.![](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/05.jpg?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE3NTIsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.D-oTmaxHzOYTJCI_IGSKfIvaNp1QPuCEpUkD23IDzjI&_LANG=enus)
   - The following error will display when you add an envelope to
     multiple rooms.![](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/06.jpg?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE3NTIsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.D-oTmaxHzOYTJCI_IGSKfIvaNp1QPuCEpUkD23IDzjI&_LANG=enus)
