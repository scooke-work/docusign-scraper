# Docusign Rooms System Requirements

Read about the minimum system requirements for using Docusign Rooms. Meeting these
requirements ensures you have the best experience when using the application.

It is important to meet system requirements to obtain the best experience in Docusign Rooms. If requirements are not met, you may experience performance issues such as
lag time or features that do not work as expected. Docusign Rooms does not develop
or test in unsupported environments.

## Internet Connection

Docusign Rooms was designed to work best with a high-speed Internet connection of at
least a DSL or cable modem connection. You can use Docusign Rooms with slower
connections, but you will naturally experience slower load times.

## Browser Support

Docusign Rooms makes every effort to test and support the most recent versions of
these browsers.

- **Microsoft Edge**: [Latest version](https://www.microsoft.com/en-us/windows/microsoft-edge) released by Microsoft
- **Google Chrome**: [Latest
  version](https://www.google.com/chrome/) released by Google
- **Safari**: [Latest version](https://www.apple.com/safari/) released by Apple
- **Mozilla Firefox**: [Latest version](https://www.mozilla.org/en-US/firefox/releases/) released by Mozilla
- **Adobe Acrobat Reader** or similar software is required to view PDF files.
  Most computers today come with this free software from Adobe installed. Need to
  install it? [Download a free copy from Adobe](https://get.adobe.com/reader/).

## Screen Resolution

- 1024 x 768 recommended
