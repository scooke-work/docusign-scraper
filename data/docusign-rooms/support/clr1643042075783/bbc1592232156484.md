# Create Templates for Room Invitation Email Messages

Learn how to create templates for the email messages people receive when they are
invited to a room. Templates are unique to the invitee's role, allowing administrators to
customize the message and subject line for different audiences.

## Before you begin

Before you begin, you need a company role with
Manage email invite templates permission.

You can create templates for room invitation email messages. Templates are unique to
the invitee's role, which allows you to customize the email message and subject line
for different audiences. For example, you might want to create different email
content for internal and external agents. For mortgage transactions, you can create
a template for borrowers with a custom message.

You can customize the area of the email message highlighted below.

1. Log in to your Rooms account.
2. Go to Settings >  COMPANY >  Email Invite Templates.
3. Select ADD TEMPLATE.

   ![rooms-email-invite-template](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsemailinvitetemplate.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE0NjMsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.j3qKi2juRR3XZvFIW_WgMD77KVMkibLf7eTUr98UPYU&_LANG=enus "rooms-email-invite-template")
4. In the Template Name field, name the template (example:
   Borrower Invite).
5. Click SELECT ROLES to apply the template to invitee
   role(s).

   Rooms will use this template when inviting someone to a room with any of these
   roles. Although you can apply the same template to multiple roles, a role can
   only be applied to one template.

   Note: Rooms doesn't require
   you to select a role when you save the template. You will see No
   roles assigned next to templates without roles. Select
   EDIT to modify the template later to add
   roles.

   ![rooms-email-invite-template-no-role](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsemailinvitetemplatenorole.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE0NjMsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.j3qKi2juRR3XZvFIW_WgMD77KVMkibLf7eTUr98UPYU&_LANG=enus "rooms-email-invite-template-no-role")
6. In the Email Subject field, type a subject for the email
   message.
7. In the Email Message field, type the body of the email
   message.

   Select INSERT FIELD to insert fields within the message body. Once
   added, fields display inside brackets. The values from the room will populate
   these fields when Rooms sends the invite.

   ![rooms-invite-template-fields](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsinvitetemplatefields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE0NjMsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.j3qKi2juRR3XZvFIW_WgMD77KVMkibLf7eTUr98UPYU&_LANG=enus "rooms-invite-template-fields")
8. Select SAVE.
