# Review Tasks and Task Lists

Read about reviewing tasks and task lists in Docusign Rooms for Real Estate and how you
can approve individual tasks and entire task lists.

There are two types of task reviews:

- Task review. If an individual task requires review, a reviewer on the
  task needs to approve it.
- Task list review. Once all tasks on a task list are approved, the
  entire task list is submitted for review. All task lists in the room need to be
  approved before the room can reviewed.

If you are the reviewer on a task or task list, you will receive an email notification
when an item needs your review. You can begin your review from the TASKS dashboard or
room TASKS tab.
