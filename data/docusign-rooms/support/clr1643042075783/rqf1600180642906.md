# Add a Document to a Room To Share It With Others

Learn how to upload documents to Docusign Rooms so people collaborating with you on a
transaction can access your files. Sharing documents in a central location saves time and
prevents you from emailing files back and forth.

## Before you begin

Before you begin, you need a room role with Add
documents to rooms permission.

When you add documents to a room, they are added to the Room Docs
folder. You can copy or move them from there to another folder or task list, if
necessary.

1. Open the room.
2. Select the DOCUMENTS tab.
3. Select ADD.
4. Select Computer or one of the cloud storage
   sources.

   When choosing a cloud storage source, you will be prompted to log in if you aren't currently
   logged into the service.
5. Navigate to and select the documents to add.
