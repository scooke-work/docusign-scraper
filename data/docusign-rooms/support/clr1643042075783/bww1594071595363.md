# Update Task Roles in the CSV File

Next, open the CSV file and update the task role names as needed. One row displays for each
unique role and responsibility combination on a task. For example, if a task has one
assignee and two reviewers, then three rows display for that task.

1. Open the CSV file you just downloaded.
2. In each row, update values in the CompanyRoleName and
   TaskResponsibilityType columns to change the roles
   and responsibilities on your tasks.

   Note: The bulk update feature only updates values in the
   CompanyRoleName,
   TaskResponsibilityType, and
   ReviewLevel columns. The CSV file contains
   additional columns with editable values, but edits you make in those columns
   are not populated back to Rooms upon upload.

   Here are a few examples of updates you can make in this file:
   - To change the task reviewer from the Broker role to the Transaction
     Coordinator role, type Transaction Coordinator in
     the CompanyRoleName column to replace
     Broker.
   - To remove a task role/responsibility, delete the role name in the
     CompanyRoleName column. This removes the
     role/responsibility from the task.
   - To add the Transaction Coordinator role as a watcher to the Earnest
     Money Due task:
     1. Copy an existing row from the Earnest Money Due task and paste
        it into a new row.
     2. In the new row, remove the ID in the
        TaskTemplateRoleId column, so it's
        blank. (Rooms will create a unique ID for this
        role/responsibility upon upload.)
     3. Type Transaction Coordinator in the
        CompanyRoleName column.
     4. Type watcher in the
        TaskResponsibilityType column.
3. Save the file.
