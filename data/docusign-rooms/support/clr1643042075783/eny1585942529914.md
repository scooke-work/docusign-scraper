# Add a Detail Field to Company Rooms

To show a new detail field in rooms, add it from the
field library. The library contains all fields you aren't currently using. To add a
field, first expand the section you'd like to display it in, then select ADD
FIELD at the bottom of the section.

1. Log in to your Rooms account.
2. Go to Settings >  Company >  Room Details.
3. Expand the section where to add the field.
4. Select ADD FIELD.

5. Select the field to add.
