# Disable a Role To Hide It From Users

Disabling a role hides it from users.

Note: You cannot disable the Default Admin role.

1. Log in to your Docusign Rooms account.
2. Select Settings in the Rooms header.
3. Select the COMPANY tab.
4. Select Roles and Permissions.
5. Select ![rooms-drop-down-arrow](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsdropdownarrow.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjEzOTIsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.VaKKy95xB4FC3XPSzHlyip4_3mM0pKEPshJ2CLjaHl4&_LANG=enus "rooms-drop-down-arrow") next to the role to disable.
6. Select Disable.

   ![rooms-disable-role](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsdisablerole.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjEzOTIsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.VaKKy95xB4FC3XPSzHlyip4_3mM0pKEPshJ2CLjaHl4&_LANG=enus "rooms-disable-role")
7. If you *do not* have company members with this role, Rooms allows you to
   disable it. Select Disable to confirm.

   If you *do* have company members with this role, Rooms notifies you that you need to
   assign these members a new role before you can disable it. Select
   Edit members in this role to view the list of members
   so you can change their roles.
