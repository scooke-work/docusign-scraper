# Use In-App Help To Access Resources

Learn how to use in-app help to access walk-thrus, written articles, videos, and
information about new features. Onboarding resources are also provided for getting started
with Docusign Rooms.

Use in-app help to access walk-thrus, written articles, videos,
and information about new features. Onboarding resources are also provided for getting
started with Docusign Rooms.

Note: In-app help and
new feature notifications for Docusign Rooms are enabled or disabled on your
Profile page in
Preferences.

![rooms-help-disable](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomshelpdisable.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE1ODIsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.4qU6O6xqcrjScmvAVm-mLmGV47hnXqMqxWDM2WB_uEY&_LANG=enus "rooms-help-disable")

1. Log in to your Rooms account.
2. Select the question mark icon in the application header.

   ![Question mark icon in the Rooms application header.](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomshelpicon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE1ODIsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.4qU6O6xqcrjScmvAVm-mLmGV47hnXqMqxWDM2WB_uEY&_LANG=enus "Question mark icon in the Rooms application header.")

   The help menu opens. If the help menu doesn't open, make sure you have
   In App Help enabled on your
   Profile page.
3. Search or navigate the help menu to find resources.

   ![A menu that reads "how can we help you?" with options to access walk-thrus, articles, videos, new features, and contact support.](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomswalkmemenu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE1ODIsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.4qU6O6xqcrjScmvAVm-mLmGV47hnXqMqxWDM2WB_uEY&_LANG=enus "A menu that reads \"how can we help you?\" with options to access walk-thrus, articles, videos, new features, and contact support.")

   To search all resources, start typing in the search field.

   In the Help menu, click any of the following sections
   to browse content by type:

   - Walk-thrus guide you through a step-by-step
     process within Rooms.
   - Articles link you to written articles on the
     Docusign Support Center.
   - Videos takes you to a list of online videos.
   - New Features links to you the Rooms release
     notes, which describe recent updates and how to use new features.
   - Contact Support gives you quick access to the
     Docusign support email address and phone number.

   The Getting Started tab links to resources for new
   users, such as walk-thrus and videos. What you see here depends on your user
   role in Rooms. The Getting Started menu tracks your progress and marks the items
   you have completed.
