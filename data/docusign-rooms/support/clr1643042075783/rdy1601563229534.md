# Copy a room template

Copying a room template creates a new template with the same settings and task lists
as the original template. This saves time if you want to create a similar template
without having to re-key information.

1. In Rooms, go to Settings > COMPANY > Room Templates.
2. Select EDIT next to the template you want to copy, then
   select Copy.
3. Edit the template if necessary.

   For example, you might want to add or remove task list templates.
4. Select SAVE.
