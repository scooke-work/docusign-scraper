# Print Documents

Learn how to print multiple documents in Docusign Rooms in one action using the bulk
print action.

In Docusign Rooms, you can print multiple documents in one action using the bulk print action.
The bulk print action allows you to generate required documents quickly and
efficiently.

1. In Docusign Rooms, navigate to the document.

   - If the document is in a room, open the room and click the
     DOCUMENTS tab.
2. For one or multiple documents, select the documents and select

3. Select ![rooms-print-icon](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsprinticon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE3NDAsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.xoXI6VZgaiXTIaoYLCs9UTt0FVCVxqWGpA6qniSgEBk&_LANG=enus "rooms-print-icon").
4. For one document, right-click the document and select **Print**.
