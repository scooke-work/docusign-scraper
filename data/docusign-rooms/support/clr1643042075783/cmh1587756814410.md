# Create Envelopes To Obtain Signatures on Your Transaction Documents

Learn how to create envelopes in Docusign Rooms to send transaction documents for
signature.

After you've added documents and people to a room, you can create envelopes to send documents
for signature. Your eSignature permissions determine what you can do with envelopes
(you must have at least DS Sender permission).

1. Open the room.
2. Select the ENVELOPES tab.
3. Select NEW.
4. Add documents to the envelope.

   - Select ADD ROOM DOCUMENTS to add documents from
     the room.
   - Select USE A TEMPLATE to use an eSignature
     template.
   - Select UPLOAD to upload documents from your
     computer or cloud storage.

   ![Add documents to the envelope.](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsenvelopeadddocument.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE2MzcsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.nsQCjn5SzU4UXIac0UlUOPNSscYkyZp07quItTz8aus&_LANG=enus "Add documents to the envelope.")
5. Drag your documents in the order you'd like the recipient to sign or view them.
6. Add recipients. If you added Docusign form(s) to
   the envelope, fields are provided for you to add recipient information for
   each tagged role (such as buyer, seller, etc.).

   Select ADD FROM CONTACTS or the contact icon to browse
   contacts within room participants, your address book, or the company
   directory.

   ![rooms-envelope-add-recipients](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsenvelopeaddrecipients.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE2MzcsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.nsQCjn5SzU4UXIac0UlUOPNSscYkyZp07quItTz8aus&_LANG=enus "rooms-envelope-add-recipients")

   ![rooms-envelope-add-recipients-RE](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsenvelopeaddrecipientsRE.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE2MzcsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.nsQCjn5SzU4UXIac0UlUOPNSscYkyZp07quItTz8aus&_LANG=enus "rooms-envelope-add-recipients-RE")
7. Once all documents are recipients are added to the envelope, select
   NEXT.
8. Add fields to the document.

   If you selected an eSignature template or form document, fields should already
   be placed for your recipients.
9. Select SEND to send the envelope.
