# Use Room Templates to Auto-Populate New Rooms With Task Lists

Room templates simplify the room creation process for your company. When you create a room template, you select all the task list templates associated with that type of room. This ensures that every time someone creates a room, the correct task lists are associated with it.

If you want to create room templates for specific offices or regions, make sure you select
them so members at those locations can see them.

Note: You need Manage room
templates permission to create and manage room templates. If you don't have this
permission, you won't see the Room Templates option in company
settings. In addition, users need permission to apply a room template to a room.
