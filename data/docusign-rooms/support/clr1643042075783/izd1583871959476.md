# Create a Folder

Learn how to create document folders in Docusign Rooms. Create as many folders as you
find useful, but they can only be one level deep. For example, add folders to separate
documents that need digital and wet signatures.

Create as many folders as you find helpful, but they
can only be one level deep. For example, add folders to separate documents that need
digital or wet signatures.

1. Open the room.
2. Click the DOCUMENTS tab.
3. Click ACTIONS > Add
   folder.
4. In the Folder Name field, type a descriptive name for
   the new folder.
5. Click CREATE.

The new folder displays in the DOCUMENTS
tab. Everyone in the room can see the new folder. Access to documents within the folder
depends on how the individual documents are shared.
