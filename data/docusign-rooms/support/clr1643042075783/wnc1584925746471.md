# Require Two-Step Verification To Access a Room

Learn how administrators can require two-step verification to access a room. This added
layer of security requires people with certain roles to provide a security code after
logging in.

You can require two-step verification to access a room. This added layer of security requires
people with certain roles to provide a security code after logging in with their email
and password. Once they enter the code, they are directed to the room. They can keep
working in the room until their session expires.

Note: Two-step verification is not supported through the Rooms API. If you set up a role for
two-step verification, people with that role will be prompted for a security code when
accessing a room through the user interface, but not through the API.
