# Configure Detail Fields for Company Rooms

As the Docusign Rooms administrator, read about the different ways you can configure
the fields that display in company transaction rooms.

Every room has a Details tab, which contains all the important information associated with the transaction, such as property information, closing details, and contact information for involved parties. Some fields trigger due dates and reminders for task lists.

As the company admin, you are responsible for configuring the Details tab. This includes:

- Deciding which fields to display in rooms
- Establishing which fields are required
- Configuring room statuses, which reflect the stages of a transaction
- Renaming field labels to reflect your company naming structure
