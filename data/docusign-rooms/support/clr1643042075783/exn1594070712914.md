# Export Your Task List Templates

First, export your task list templates to a CSV file.

1. Log in to your Rooms account.
2. Select Settings in the Rooms header.
3. Select the COMPANY tab.
4. Select Task List Templates.
5. Select BULK UPDATE.

   ![rooms-task-bulk-update](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomstaskbulkupdate.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE0NTAsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.X42H6_3C8ULoXljnR4CFPk5jQnxu87ZWQVLR7kYUIZg&_LANG=enus "rooms-task-bulk-update")
6. Under Task list templates:

   - Select the template to export from the drop-down list or select All Task List
     Templates to export all templates.
   - Choose whether to export Only tasks with people or All
     tasks.

     Tip: Since you can only update task roles and responsibilities
     in the export file, you only need to export tasks with people.
     Exporting all tasks might be helpful if you want to see all tasks
     together.

   ![rooms-tasks-bulk-update-modal](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomstasksbulkupdatemodal.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE0NTAsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.X42H6_3C8ULoXljnR4CFPk5jQnxu87ZWQVLR7kYUIZg&_LANG=enus "rooms-tasks-bulk-update-modal")
7. Select EXPORT.

Your browser downloads the CSV file to your download
folder.
