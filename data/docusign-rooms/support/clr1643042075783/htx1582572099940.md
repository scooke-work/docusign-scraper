# Create Templates To Split PDF Documents

If your users regularly split the same PDF documents, learn how to create PDF split
templates. When users apply the template, their document is split for them, but they can
adjust if needed.

## Before you begin

Before you begin, you need a company role with
Manage PDF Split Templates permission.

If you regularly split the same PDF documents, consider creating PDF split templates
for your company. These templates simplify the splitting process and save data entry
time for users. To create a PDF split template, upload the original PDF (that
requires splitting) and specify the split settings. When users apply the template,
their document is split for them, but they can adjust if needed.

1. Log in to your Rooms account.
2. Select Settings in the Rooms header.
3. Select the COMPANY tab.
4. Select PDF Split Templates.
5. Select ADD TEMPLATE.

   The Add PDF Split Template page displays. Here, you will upload the PDF you
   want to use on the template.
6. Name the template.

   It's a good idea to name the template the same as (or similar to) the document you are
   splitting. This helps users find the template they need.
7. Select UPLOAD DOCUMENT to browse and open the PDF
   document.

   The window loads the PDF document and shows each page as a thumbnail. To get a
   closer look, hover your mouse over the thumbnail and it will magnify the view
   for you.

   ![A PDF document with each page displaying as a thumbnail.](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomspdfsplittemplate.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE0NTksInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.M6LRGE8pHYnMyL6J2OQVTe1Mw4ZtzdhyxyjDUEkPPlY&_LANG=enus "A PDF document with each page displaying as a thumbnail.")
8. Select + to add a line for each new document that will
   be created as a result of the split.
9. Type a name and page range for each split document.

   You don't have to designate every page to a new file. Any leftover pages will not be
   included in the new documents.
10. Under Enable for, select the regions or offices that
    will have visibility to this template.

    If you make no selections here, the template will be visible to
    everyone.
11. Select SAVE.
