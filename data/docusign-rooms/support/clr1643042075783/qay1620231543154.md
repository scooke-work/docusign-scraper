# Apply a Form Template

When you apply a form template, it populates the form
with the pre-filled data from the template. If there is existing data on the form, the
information on the template will override that data.

A form template is tied to a
specific form. You cannot apply a form template to multiple forms.

Note: These steps show you how to apply a template when
adding the form to a room. To apply a template to a form that is already in the
room, right-click the form and choose Apply form
template.

1. Open the room.
2. Select the DOCUMENTS tab.
3. Select ADD >  DocuSign Forms.
4. Select the form library from the left navigation pane.

   The right pane refreshes to show forms within the selected
   library.
5. Select the forms to add.

   If a template is associated with a form, a drop-down menu displays next to the
   form name.

   ![rooms-form-template-add-form-select-template](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsformtemplateaddformselecttemplate.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE4MDcsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.wWLQxmkRvzf1eXCcmIyyxpj0rQBA0__B8654gjZuilE&_LANG=enus "rooms-form-template-add-form-select-template")
6. For each form, select the template from the drop-down menu.
7. Select ADD SELECTED.

Note: In the rare scenario where templates have conflicting
data, you are prompted to prioritize your templates and use the data from the higher
priority template.
