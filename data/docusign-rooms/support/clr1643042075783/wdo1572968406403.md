# Resend a Room Invitation

You may need to resend a room invitation, or edit part of the invitation, such as the person's name.

1. Open the room.
2. Select the PEOPLE tab.
3. Do one of the following:

   - In grid view, select ![3 dot icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/3dotpeopleicon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE2MjksInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.8TBg84RCL7DXc1AnCUs80BF1Is0_VMt_DLKfwD14tmQ&_LANG=enus "3 dot icon.") under the person's name, then select Contact
     info.
   - In list view, select Actions in the person's row, then select
     Contact info.
4. Select Resend Invite.
5. Make changes as needed, then select SEND INVITATION.
