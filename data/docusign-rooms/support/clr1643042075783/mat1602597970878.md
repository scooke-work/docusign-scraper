# Submit a Task List for Review

When all tasks in a task list are Complete or In
Review, submit the task list for review. All task lists in a room
need to be approved before the room can be approved.

Note: To submit task lists for review, you need a
room role with permission. If your room role has Auto-submit task
lists permission, task lists are automatically submitted for review
when you submit the room for review.

To submit a task list for review:

1. Click the TASKS tab in the room.
2. Click ACTIONS >  Submit task list.
3. Add comments if needed.
4. Click SUBMIT.
