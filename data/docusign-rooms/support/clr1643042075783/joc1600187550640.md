# Split a PDF Document

Learn how to divide a PDF file into multiple files in Docusign Rooms.

1. In Docusign Rooms, right-click the document and select
   Split.

   The window loads the PDF document and displays each page as a thumbnail. Hover
   your mouse over the thumbnail and it will magnify the view for you.

   ![rooms-split-pdf](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomssplitpdf.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE3MTcsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.eG1s6xCfgD6g39plGCQ8k2nl7NZwtJ8LO0wQXy7t7fE&_LANG=enus "rooms-split-pdf")
2. Select Add a Document to add a line for each new
   document that will be created as a result of the split.

   You don't have to designate every page to a new file. Leftover pages will not
   be included in your new document.
3. Type a name and page range for each new split document.
4. Select SAVE.

The DOCUMENTS tab displays your newly
split PDF as well as your original file.
