# Take Action on All Documents in a Folder

Learn how to select all documents in a Docusign Rooms folder to take action on all
documents at once.

Select all documents in a room folder to take action on all of
them at once.

1. Locate the folder in the room DOCUMENTS tab.
2. Click ACTIONS >  Select all.

   ![rooms-documents-select-all](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsdocumentsselectall.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE3MDYsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.gwViKjRq2DoS4NtPf2fWqOQNs8kxMAZEwHK7vUx7ms0&_LANG=enus "rooms-documents-select-all")

   The document thumbnails display ![rooms-documents-selected](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsdocumentsselected.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE3MDYsInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.gwViKjRq2DoS4NtPf2fWqOQNs8kxMAZEwHK7vUx7ms0&_LANG=enus "rooms-documents-selected") after you select them.
3. Use the toolbar or right-click menu to take action on the documents.

   For example, to add all the selected documents to a new envelope, right-click
   and choose Create envelope.
