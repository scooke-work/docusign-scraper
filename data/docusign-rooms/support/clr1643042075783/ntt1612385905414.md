# Correct an Envelope in Docusign Rooms

Learn how to correct the details of in-process envelopes that you sent or are shared
with you, including the recipient, envelope, and document information.

In Docusign Rooms, you can correct the details of in-process envelopes that you sent
or are shared with you, including the recipient, envelope, and document
information.

Note: Completed (signed), declined, and voided envelopes
can't be corrected. However, you can copy the envelope to create a new one with the
same uploaded files, recipients, and signing fields.

You can correct the following information in an envelope:

- Envelope name, email message to recipients: These can be
  changed at any time within the life cycle of the envelope before
  completion.
- Documents: Add or reorder documents if none of the
  recipients have taken their action (signed, viewed, etc.). If a recipient has
  taken their action, the documents that were part of the original envelope will
  be locked.
- Recipients:
  - Adding, reordering, private messages, editing the recipient's contact
    information and responsibility are all available if none of the
    recipients have taken their action. If a recipient has taken their
    action, that recipient would be locked.
  - You can add a recipient back into the signing order again if that
    recipient needs to take another action on the envelope.
- Signing fields: You can add or adjust any of the
  recipient tags within the tagging page. You can't adjust tags for recipients who
  have already taken their action.

1. On the room ENVELOPES tab, right-click the envelope and
   select Correct.

   ![rooms-envelope-correct](https://docusign-be-prod.zoominsoftware.io/api/bundle/clr1643042075783/page/images/roomsenvelopecorrect.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjE2MzksInNoZWFmIjoiY2xyMTY0MzA0MjA3NTc4MyJ9.VIwfE8oeYt0YM8SwEq7-k0CWORd7ySxSbxvfLpoaBR8&_LANG=enus "rooms-envelope-correct")
2. The Envelope Details page appears with the status
   Correcting at the top of the page. Edit the documents
   and recipient fields as needed.
3. Select SEND.

   To edit the fields on the document, select EDIT SIGNING
   FIELDS. Delete, modify, or add fields as needed, select
   CONTINUE, and then SEND to
   reissue the document to your outstanding recipients whose turn it is to
   sign.

In-process recipients receive a new email notification, which contains a new link to the
envelope.
