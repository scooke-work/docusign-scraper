# Filter and Sort Workspaces

Learn how to filter and sort Workspaces to locate those that you
want to interact with.

Filter and sort Workspaces to locate the ones you want to work with. You
can filter by owner, creation time frame, and status. You can sort the results
alphabetically and by oldest and newest dates.

1. Select
   Workspaces
    from the
   Agreements page.

   A list of Workspaces displays.
2. Select all or some of the following options to narrow your search:

   | Option | Description |
   | --- | --- |
   | Search Workspaces | Start entering text and see matching results as you type. You can search by owner name and part of the workspace name. |
   | Created | View Workspaces that were created in the past 30, 60, or 90 days. |
   | Owned by | Select a user and select Apply to see Workspaces owned by specific people. |
   | All Workspaces | Select Active or Closed and select Apply to narrow the results to a subset of all Workspaces. |

Select Clear to reset the search
criteria.

3. (Optional) Sort the results by name or created date by selecting the column
   headings NAME and CREATED.
