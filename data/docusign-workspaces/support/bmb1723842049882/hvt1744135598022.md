# Add Envelopes to Workspaces

Learn how to add envelopes to a workspace. You can either create a new envelope or create an envelope from a template.

To add envelopes to a workspace, either create one or use a template.

1. Open a workspace that you want to add envelopes to.
2. Select Add > Envelope.
3. Select one of these options.

   - Create an envelope

     Create a new envelope with documents from your computer or cloud storage provider. You can add documents from Box, Dropbox, and Google Drive.
   - Use a template

     Select a template that you created or that is shared with you.
4. Enter the information for the recipient or recipients of your envelope.
5. Select Save to workspace to save the envelope as a draft and return to the workspace, or select Get signatures to add recipients and send the documents for signature immediately.
