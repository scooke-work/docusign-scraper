# Download Message History

Learn how to download a CSV file of message history.

You can download the message history for a given topic in a CSV format. This file contains
information, such as transaction ID, participant ID, export time stamp, and message content and time stamps.

1. Select the Messages icon in the upper right.
2. Select an existing topic from the left panel.
3. Select the download icon in the upper right.

A CSV file is downloaded to your computer.
