# Delete Workspaces

Learn how to delete a workspace when you no longer need
it.

If you no longer need a workspace, you can delete it.

1. Select
   Workspaces
    from the
   Agreements page.
2. Select the check box next to the workspace that you
   want to remove.

   To see all closed and active Workspaces, select
   All Workspaces
    from the
   Status button.
3. Select Delete.

The workspace is no longer visible in the
list.
