# Add a Personalized Message to a Workspace

Learn how to add a custom message visible to all envelope recipients within a Workspace.

## Before you begin

You can add a personalized message to your Workspace to provide instructions, greetings, or context. This ensures that every recipient who accesses a task within that Workspace views the same message.

1. From the Agreements page, select Workspaces from the left navigation.

   The Workspaces page opens.
2. Select an existing Workspace or [create a new Workspace](https://support.docusign.com/s/document-item?bundleId=bmb1723842049882&topicId=sha1723842408338.html&_LANG=enus "Learn how to create a workspace, to manage documents and collaborate with other users.").

   The Workspace Overview page opens.
3. Select Settings from the Actions menu.

   The Workspace Settings window opens.![The screenshot shows the Actions menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/bmb1723842049882/page/images/Per_options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE1NDUsInNoZWFmIjoiYm1iMTcyMzg0MjA0OTg4MiJ9.ztxQ--qQzxMMsufRkTWA5lFocnh10uOhNq3Yo_S98zM&_LANG=enus "The screenshot shows the Actions menu.")
4. Enter your personalized message for this Workspace in the Message to recipients field.![](https://docusign-be-prod.zoominsoftware.io/api/bundle/bmb1723842049882/page/images/WSmessage.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE1NDUsInNoZWFmIjoiYm1iMTcyMzg0MjA0OTg4MiJ9.ztxQ--qQzxMMsufRkTWA5lFocnh10uOhNq3Yo_S98zM&_LANG=enus)
5. Select Save.

Your message is saved. It now appears to all recipients in any envelope added to the Workspace.
