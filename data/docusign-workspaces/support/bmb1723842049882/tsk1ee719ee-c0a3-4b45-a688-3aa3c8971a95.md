# Using a Workspace Template

Learn how to create a new Workspace using a published template.

## Before you begin

Once a Workspace template is published, it becomes available for you and others on your account to use. Using a template automatically populates your new Workspace with the pre-defined items from the template.

1. From the Home or Agreements page, select Start > Workspaces > Use a Workspace Template.

   The Use a workspace template page opens.

   Note: You can also access Workplace templates from the Templates page by selecting Workspace Templates. Locate the template you'd like to use and select Use.
2. Locate the template you want to use and select Use.

   The Name the workspace window opens.
3. Enter a name for the new Workspace and select Save.

The Workspace is created with the structure and items defined in the template.
