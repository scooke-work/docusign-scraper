# Send Messages to Workspaces Users

Learn how to send messages to users of a workspace.

## Before you begin

Ensure that the users you want to message have been added to the workspace. You can add users in the
following ways:

- Share the workspace with a user
- Assign a task to a user

Workspaces offers a messaging feature that enhances collaboration between
workspace users. The messages you send are
consolidated within the workspace for easy reference.

1. Open the workspace.
2. Select the View messages icon in the upper right.
3. Select New Message.

   The Create New Message dialog box displays.
4. (Optional) Enter a topic name of 100 characters or less.
5. Select a user from the Workspace participants menu.
6. Select Create.
7. Enter a message in the message field and select the arrow icon to send.

The message you sent displays in the workspace messages view.
