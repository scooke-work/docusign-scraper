# Request Documents from Another User

Learn how to send an upload request for documents to another user. Users that receive
the request can upload the requested documents to the workspace.

Create an upload request when you want to ask
another user to upload documents to the workspace.

1. Open the workspace that you want to request documents
   for.
2. Select Add > Upload Request.

   The Details page for the Upload
   Request displays.
3. You can either send the request immediately or save it to the workspace to send it later with other tasks.

The upload request you saved or sent displays in your workspace. The saved request is labeled as
Draft and the request that you sent immediately shows
Waiting for upload.
