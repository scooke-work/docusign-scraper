# Add Users to Workspaces

Learn how to add users to Workspaces. You can add users both within
and outside of your organization. However, the level of permissions available differs
depending on whether the user is on the same account as the Workspaces
creator.

You can add users to Workspaces so they can upload documents and complete
agreement steps. You can assign a user one of two permission levels: Manage and
Participate.

Note: Only users within the organization can have Manage level permission
assigned to them. To have Manage level permission, the user you add must be on the
same account as the person who created the workspace. Users
external to your organization are restricted to Participate permission only.

1. Open a workspace to which you want to add a user.
2. Select Share.
3. Type the first name, family name, and email address for the user you want to
   add.
4. (Optional) Add users by selecting Add New Person.
5. Select Done.

The user or users you added are listed as participants in the
workspace. Internal users receive an email invitation.
External users receive an email notification when they are assigned tasks.

## Next steps

You can assign actions and share documents with the user you added.
