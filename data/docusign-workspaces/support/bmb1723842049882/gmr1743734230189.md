# Document Upload Requests

Read about how workspace preparers and
managers can request documents from collaborators in a workspace.

As a preparer or manager of a workspace, you can
request documents from collaborators through an Upload Request. Both internal and
external collaborators can receive a request, which consists of a title, optional
description, and a due date.

When you finish completing the request, you can either send it immediately or save it
to the workspace to send later with other tasks. See the topic
[Batch-Send for Envelopes and Upload Requests](https://support.docusign.com/s/document-item?bundleId=bmb1723842049882&topicId=pyq1743623833366.html&_LANG=enus "Read about how workspace preparers and managers send multiple tasks, such as envelopes and upload requests, at one time by using batch send.") to learn more.

The recipient of the Upload Request receives an email notification with a link
to view the request and other tasks. The recipient can view the due date on the request.
If the due date has passed, the request reflects a status of past due or expired.

You can assign an Upload Request to only one user at a time. To request the
same documents from different users, create separate requests.
