# Send Multiple Envelopes and Upload Requests in a Batch

Learn how to send multiple envelopes and upload requests together in a
batch.

As a workspace preparer or manager, you can send multiple
envelopes and upload requests in a batch rather than sending them immediately. The
recipient receives one notification with a link to the envelopes and requests that
need attention.

1. Prepare each upload request or envelope for signature and instead of sending
   them immediately, select Save to workspace
   .

   The request or prepared envelope is saved to your workspace with a status of
   Draft.
2. When you are ready to send the upload requests, envelopes, or a combination
   from the workspace, select the check box for each.
3. Select Send and select Send again
   in the confirmation dialog box.

   The status of the envelopes changes to Waiting for
   others and the status of the upload requests changes to
   Waiting for upload.
