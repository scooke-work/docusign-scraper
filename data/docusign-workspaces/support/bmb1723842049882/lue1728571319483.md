# Add Documents to Workspaces

Learn how to add documents to a workspace. You can add
documents by uploading them from a cloud storage provider, such as Box, Dropbox, and Google
Drive. You can also add them through templates.

You can add documents by uploading them from local or cloud storage, such as Box, Dropbox, and
Google Drive, or by uploading templates.

1. Open a workspace that you want to add documents
   to.
2. Select Add > Document.

   The Add documents page opens.
3. Add documents from one of the following sources.
   - Use template Add templates that you have created or templates
     that are shared with you.
   - Upload Add one or more documents from your computer or a cloud
     storage provider, such as Box, Dropbox, or Google Drive.

The document you selected displays in the Add documents
page.

4. Continue repeating step 3 to add the documents you want.
5. Select Save and close to return to the workspace, or select Get signatures
   to add recipients and send the documents for signature.

   When you select Get signatures for a document, the
   document becomes part of the envelope and is no longer displayed separately in
   the workspace.
