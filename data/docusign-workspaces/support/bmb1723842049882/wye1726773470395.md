# Rename Workspaces

Learn how to change the name of a workspace.

You can change the name of an existing workspace.

1. Open a workspace.
2. Select the overflow menu (the three vertical dots).
3. Select Rename.
4. Edit the name and select Save.

The name of the workspace reflects the
update that you made.
