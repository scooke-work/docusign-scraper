# Disable Messaging

Learn how administrators disable messaging for all users and workspaces in an account.

## Before you begin

You must have administrator permissions to perform this task.

Disable messaging for users and workspaces in an account by following these steps:

1. Select the Admin tab.
2. Select Workspaces Settings from the left navigation.
3. Select the toggle for Enable Messaging Feature to turn messaging off.
