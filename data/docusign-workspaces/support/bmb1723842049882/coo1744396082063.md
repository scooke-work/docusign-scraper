# Limitations on Docusign Workspaces Usage

See information about limits associated with Workspaces
usage.

The following table lists the limitations that exist for Workspaces
usage. If you exceed these limits, you might encounter performance issues.

Note: eSignature envelopes are subject to the standard Docusign envelope limitations listed on the following web page:
[Limitations When Creating or Signing an Envelope
in Docusign](https://support.docusign.com/s/articles/Limitations-When-Creating-or-Signing-an-Envelope-in-DocuSign).

| Type of usage | Maximum |
| --- | --- |
| Maximum storage size per workspace | 1 GB |
| Maximum number of workspaces per user (per month)\* | 100 |
| Number of envelopes or files per workspace (per month)\* | 1,000 |
| File size | 50 MB |
| Number of Workspaces API calls (per month)\* | 1,000,000 |
| Number of Workspaces API calls (per hour)\* | 1000 (configurable to go higher) |
| \* Limit per customer account | |
