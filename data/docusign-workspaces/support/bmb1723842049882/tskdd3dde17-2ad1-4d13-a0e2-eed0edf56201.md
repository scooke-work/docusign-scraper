# Create a Workspace Template

Learn how to create a Workspace template to standardize the content for your recurring projects.

## Before you begin

You must have the permissions required to create envelope templates.

Creating a Workspace template allows you to pre-define the structure for projects that you use frequently. You can add envelope templates, upload requests, and documents to the Workspace template.

1. From the Templates page, select Workspace Templates from the left navigation.

   The Workspace Templates page opens.

   Note: Alternately, access Workspace Templates from the Home tab by selecting Start > Create Workspace Template .
2. Select Create Template.

   The Create Workspace Template window opens.
3. 4. Select Next.

   The overview page for your Workspace template opens.![The screenshot shows the workspace template overview page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/bmb1723842049882/page/images/Build.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE0OTMsInNoZWFmIjoiYm1iMTcyMzg0MjA0OTg4MiJ9.pYdd2WjykXxLAwf1f3uoE-0aBk5kaBeFR8AqggaWacs&_LANG=enus "The screenshot shows the workspace template overview page.")
5. Add envelope templates to the Workspace.
   1. Select Add > Envelope

      The Add Templates page opens.
   2. Locate the envelope templates you want to add and select Add.

      You have added envelope templates to your Workspace template.

      Note: Only users who have permission to use the envelope template will see it in the Workspace. Envelope template access is not automatically shared when added to a Workspace template
6. Add Upload Requests to the Workspace.
   1. Select Add > Upload Request.

      The Details page opens.
   2. Complete the details on the Details page.![The screenshot shows the Upload Request page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/bmb1723842049882/page/images/Upload_Request.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE0OTMsInNoZWFmIjoiYm1iMTcyMzg0MjA0OTg4MiJ9.pYdd2WjykXxLAwf1f3uoE-0aBk5kaBeFR8AqggaWacs&_LANG=enus "The screenshot shows the Upload Request page.")
   3. Select Save.

      You added an Upload Request to your Workspace template.
7. Add documents to the Workspace.
   1. Select Add > Document.![The screenshot shows options under the Add menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/bmb1723842049882/page/images/UPloadrequestmenu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE0OTMsInNoZWFmIjoiYm1iMTcyMzg0MjA0OTg4MiJ9.pYdd2WjykXxLAwf1f3uoE-0aBk5kaBeFR8AqggaWacs&_LANG=enus "The screenshot shows options under the Add menu.")

      The Add documents page opens.
   2. Select Upload and locate the files you would like to add.
   3. Select Add.

      You have added documents to your Workplace template.

You have created a Workspace template.

## Next steps

Publish your Workspace template to make it available for others to use. Until you publish it, the template remains in a draft state and cannot be seen or used to create a new Workspace.
