# Restrict Access to a Workspace Through Identity Verification

Learn how workspace owners restrict access to workspaces by external participants through identity verification checks.

## Before you begin

You can secure a workspace by requiring that external participants pass an identity verification check before accessing it.

1. Open the workspace that you want to secure.
2. Select Share.
3. Select Add new person.
4. Enter the name and email address of the participant.
5. Select the overflow menu (the three stacked dots), and select Add Identity Verification.
6. Select the type of identity verification you want to use.
7. Select Share.

Docusign prompts the participant to perform and pass an identity verification check before allowing access to the participant portal. After passing the check, the participant is admitted to the participant portal and can access the shared workspace.
