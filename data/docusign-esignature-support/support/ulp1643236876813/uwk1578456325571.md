# Use the Contacts List to Add Recipients

Learn how to use the contacts list to add recipients to an envelope.

Your Docusign eSignature account includes a [contacts list](https://support.docusign.com/s/document-item?bundleId=jux1643235969954&topicId=twh1578456324503.html&_LANG=enus). You can select recipients from your contacts list to add to an envelope. Your [permission settings](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=pof1583277362435.html&_LANG=enus) control the types of contacts you have access to.

Note: If you can create personal contacts, recipients are automatically added to your contacts list when you send an envelope, so you only need to enter their information once.

1. From the Prepare view, select the Contacts icon to select a saved contact to add.

   ![The Add recipients section with the Name field circled.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ulp1643236876813/page/images/add-recipients-contact-list.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQxNjksInNoZWFmIjoidWxwMTY0MzIzNjg3NjgxMyJ9.JYzdHfLwhcMvEETxPaj81k9N-bK03vzrPq2h8WkkGjo&_LANG=enus "The Add recipients section with the Name field circled.")

   Note: When adding recipients, special characters (like `< >`) in a name can prevent successful signature selection. To control how a recipient's name displays and to prevent changes during signing, use the Lock Recipient Name feature. This ensures that the recipient signs with the exact name that you provide. For more information, see [How to Lock Recipient Name option prevents recipients from changing their name when adopting a signature?](https://community.docusign.com/esignature-111/how-to-lock-recipient-name-option-prevents-recipients-from-changing-their-name-when-adopting-a-signature-2061)
2. Depending on your user permissions and account settings, you can search and select from the following categories of contacts:

   - **Address Book:** Your address book can contain personal contacts, contacts shared with you, and account signing groups.
   - **Directory:** Your directory contains all active users on your account. Docusign Administrators always have directory access. For other users, the feature must be enabled for your account by your Docusign administrator.![](https://docusign-be-prod.zoominsoftware.io/api/bundle/ulp1643236876813/page/images/addfromcontacts.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQxNjksInNoZWFmIjoidWxwMTY0MzIzNjg3NjgxMyJ9.JYzdHfLwhcMvEETxPaj81k9N-bK03vzrPq2h8WkkGjo&_LANG=enus)
3. Select Apply Selected.

   The contact's information is added with the default action Needs to Sign. You can adjust the action for the recipient as needed.
4. To add multiple contacts to your list of recipients at once:

   1. Select Add Recipient and Add From Contacts.![The Add Recipient menu with Add From Contacts selected.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ulp1643236876813/page/images/addfromcontactsmenuselection.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQxNjksInNoZWFmIjoidWxwMTY0MzIzNjg3NjgxMyJ9.JYzdHfLwhcMvEETxPaj81k9N-bK03vzrPq2h8WkkGjo&_LANG=enus "The Add Recipient menu with Add From Contacts selected.")
   2. Select one or more contacts and select Apply Selected to add them as Needs to Sign recipients.
   3. Adjust each added recipient's action and details as needed.
