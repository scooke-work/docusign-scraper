# Edit Field Properties

Learn how to edit field properties and adjust settings.

To edit field properties, access the settings by either of the following methods:

- For a single field, select the field to choose it.

  The field properties panel
  slides in over the page guide on the left, showing adjustable settings.

  ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/ulp1643236876813/page/images/signfieldproperties2.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQyMTMsInNoZWFmIjoidWxwMTY0MzIzNjg3NjgxMyJ9.xUEpLkTNmZ0KoWm9l_QpeR1ny83SnUIFPB4irZzvyh8&_LANG=enus)
- To select multiple fields, drag to encompass them or use **Shift+select** or **Cmd+select** to
  choose each one.

  The field properties panel slides in, applying changes to all selected
  fields.
