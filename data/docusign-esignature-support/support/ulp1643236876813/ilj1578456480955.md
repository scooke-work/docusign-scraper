# Set Signing Order and View the Order Diagram

Learn how to set a signing order for your document and view a visual diagram of the
routing.

To set a signing order and view the order diagram:

1. From the Prepare view, add your recipients.
2. Select the Set signing order check box.

   Signing order values display at the left end of the recipient rows. By default, the
   signing order is the order in which you added the recipients.
3. To change the order, enter new values in the routing order boxes.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/ulp1643236876813/page/images/changesigningorder.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQxNzYsInNoZWFmIjoidWxwMTY0MzIzNjg3NjgxMyJ9.unsyOiln-0-tckN3JZpiLKqreE925ciTCzTSdwWLl5w&_LANG=enus)
4. To send the document to multiple recipients at the same time, set the same signing
   order value for each.
5. Select View to open the signing order diagram and see a visual
   representation of the recipient routing for your document.

   In the signing order diagram, each recipient is represented by their initials,
   following their position in the signing order. Select each recipient circle to see more
   details.
