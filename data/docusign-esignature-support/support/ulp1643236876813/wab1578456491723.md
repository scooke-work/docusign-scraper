# Remove a Private Message

Learn how to remove a private message added for individual recipients.

You can remove a private message from a draft envelope or when [correcting](https://support.docusign.com/s/document-item?bundleId=oeq1643226594604&topicId=dxr1578456334187.html&_LANG=enus) an in-process envelope.

1. In the recipient's row, select the private message panel to expand it.
2. In the private message panel, select the trash can icon.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/ulp1643236876813/page/images/discardprivate-message.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQxODUsInNoZWFmIjoidWxwMTY0MzIzNjg3NjgxMyJ9.htqqFxWkyewBqPgDRhelxgQ02XZkeRV1x2rovx0aNn8&_LANG=enus)

The private message is removed.
