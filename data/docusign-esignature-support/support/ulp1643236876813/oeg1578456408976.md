# Preview and Send

Read about previewing a document before sending it for signature.

Once you finish adding documents, recipients, messages, and signing fields, you can preview your work and send your envelope.

The top navigation bar presents options to preview and edit the message, recipients, and documents:

![](https://docusign-be-prod.zoominsoftware.io/api/bundle/ulp1643236876813/page/images/cancelsavepreview.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQyNDYsInNoZWFmIjoidWxwMTY0MzIzNjg3NjgxMyJ9.2DcZKs2qj1BW2sx27zZVLqDaOIeQsWzqGSuVJixWQbI&_LANG=enus)

The top navigation bar presents the option to send the envelope:

![](https://docusign-be-prod.zoominsoftware.io/api/bundle/ulp1643236876813/page/images/backsendnav.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQyNDYsInNoZWFmIjoidWxwMTY0MzIzNjg3NjgxMyJ9.2DcZKs2qj1BW2sx27zZVLqDaOIeQsWzqGSuVJixWQbI&_LANG=enus)

- To save a draft without sending it, select the "x" and Save and Close.

![](https://docusign-be-prod.zoominsoftware.io/api/bundle/ulp1643236876813/page/images/saveandclose.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQyNDYsInNoZWFmIjoidWxwMTY0MzIzNjg3NjgxMyJ9.2DcZKs2qj1BW2sx27zZVLqDaOIeQsWzqGSuVJixWQbI&_LANG=enus)

- To cancel the envelope preparation and return to your Agreements page, select the "x" and Discard.
- To preview your recipients' experience, select Preview.
- To send your envelope, select Send.

After sending your envelope, recipients receive an email to review and sign. You can start the process directly from your account if you're a signer. For more information, see the [Sign Documents In Your Account](https://support.docusign.com/s/document-item?bundleId=oeq1643226594604&topicId=hbw1578456422642.html&_LANG=enus) guide.

## Preview mode

Preview mode offers a simplified view of each recipient's signing experience. You can use this mode to step through each recipient's fields.

- To preview your recipients' experience, select Preview. ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/ulp1643236876813/page/images/preview1.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQyNDYsInNoZWFmIjoidWxwMTY0MzIzNjg3NjgxMyJ9.2DcZKs2qj1BW2sx27zZVLqDaOIeQsWzqGSuVJixWQbI&_LANG=enus)

  1
  :   Select a recipient to preview.

  2
  :   Select the display format.

  3
  :   Close the preview mode.

  4
  :   Required fields have a dark purple border.

  5
  :   Optional fields have a light purple border.
- To check field entries, select the radio buttons and dropdown values or enter text.
