# Add Fields to Documents

Read about how to add fields to documents that instruct signers where to provide their signature, initials, or other information.

Once your envelope includes files, recipients, and messages, you add fields instructing your signers to provide their signature, initials, or other information. You add a unique set of signing fields for each recipient.

You can now perform a single search that scans across all available field libraries simultaneously.

Note: If you cannot add fields to the documents and templates you create, this is most likely because of your user permission profile. Contact your Docusign administrator and ask to adjust your permission settings. These settings are documented in the [Permission Profiles](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=pof1583277362435.html&_LANG=enus) guide.

## Overview of adding fields

The Add Fields view has the following basic areas:

1) Recipients
:   Lists all recipients who will receive your documents to sign. The fields you add are color-coded and apply only to the selected recipient. You can add fields for signers. Use the Edit Recipients option to edit details or add/remove recipients. You can also add sender fields or "pre-fill fields" to input data before sending the agreement.

2) Undo/Redo and Copy/Paste
:   Access common field actions with one-click icons. Use standard keyboard controls or right-click for easy field management.

3) Fields
:   **Standard Fields:** Displays the set of primary, out-of-the-box fields that you can add to your document.
:   **Salesforce Fields:** Specialized fields to merge and map data between Docusign and Salesforce records.
:   **Custom Fields:** User-defined fields created to capture specific data not covered by standard options.
:   **Data Verification:** Fields that function as input fields to ensure recipient data matches a specific required format or pattern before submission.

4) Page Guide
:   Displays a scrolling list of all pages in your document, including all added files. Select any page to make it active. You can delete, rotate, add, remove, or reorder pages from the Page Guide. Use Actions > Edit Documents for more options.

5) Field Properties
:   (Not shown) View and edit properties for selected fields, such as formatting, recipient assignment, and tooltips. This panel slides over the Page Guide when fields are selected.

When recipients review your document, they navigate through the fields based on the Auto-Navigation setting for your account. Your Docusign Administrator sets this option using the Signing Settings in Docusign Settings. For more information, see the [Signing Settings](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=lue1583277359885.html&_LANG=enus) guide.

The default setting navigates through all fields on the document. Alternate options include Page Only where recipients scroll manually, and Navigate required fields, which only guides recipients through the required fields.
