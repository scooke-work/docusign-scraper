# Add a Document to an Envelope

Learn how to add documents to envelopes.

File names are used to generate the default envelope name and to identify documents that are downloaded using the separate PDF option.

To add a document to an envelope:

1. Start a new envelope from Start > Envelopes > Send an Envelope.
2. From the Prepare view, select Upload to add your files.
3. To add files from one of your [authorized cloud storage providers](https://support.docusign.com/s/document-item?bundleId=gbo1643332197980&topicId=qqb1578456263434.html&_LANG=enus), select the provider. Next, select your files and select Add Selected.

   Note: Your account administrator controls the cloud storage providers that you can select from. For more information, see the [Document Sources](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=ebi1583277352404.html&_LANG=enus) guide.
