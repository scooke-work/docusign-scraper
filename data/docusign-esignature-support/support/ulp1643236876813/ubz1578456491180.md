# Add a Message to All Recipients

Learn how to add a group message to all recipients.

The message subject line for all recipients is automatically generated from the names of the files you upload, prefixed with "Complete with Docusign:". For example, uploading a file named "NDA Agreement" creates the subject "Complete with Docusign: NDA Agreement".

If you upload multiple files, recipient names are added to the subject line, limited to 100 characters. You can change the subject line to anything you like.

Note: The option to add a message to all recipients is only available when sending the envelope by email. It is not available when using the SMS (Text) option.

All recipients see the message information. As the sender, you can see the message that you added to a document on the [Details](https://support.docusign.com/s/document-item?bundleId=ulp1643236876813&topicId=ztv1578456348404.html&_LANG=enus) view.

The message fields have the following character limits:

- **Subject:** 100 characters
- **Message:** 10,000 characters

1. From the Prepare view in the Add message section, edit the default subject line as necessary.

   Important: This entry serves as both your subject and envelope title. A clear subject line indicates the nature of your documents. For repeated documents, use a custom subject line. If you keep the default subject, all envelopes will have the same title, making it hard to identify individual ones.
2. Enter your message in the message field.

Your message is added and sent to all recipients.
