# Add Private Messages to Individual Recipients

Learn how to add a private message to individual recipients.

You can add private messages to send to individual recipients. Only the selected recipient sees these messages, along with the general message. The private message displays in the email notification under Notes and is visible when the recipient views and signs the documents.

The private message field is limited to 1,000 characters and does not support HREF links, only plain text.

As the sender, you can see a private message that you added to an envelope on the [Details](https://support.docusign.com/s/document-item?bundleId=oeq1643226594604&topicId=otr1578456349846.html&_LANG=enus) view.

To add a private message for one recipient:

1. In the recipient's row, select Customize and Add private message.

   ![Add recipient card with Customize menu expanded and Add private message option highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ulp1643236876813/page/images/addprivatemessage.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQxODMsInNoZWFmIjoidWxwMTY0MzIzNjg3NjgxMyJ9.FbUGqW3XROlxIfblB6mUvPX88SIs_MoWI9jTTXfnpco&_LANG=enus "Add recipient card with Customize menu expanded and Add private message option highlighted.")

   Note: The option to add a private message is only available when sending the envelope by email. It is not available when using the SMS (Text) option.
2. Enter your message.

   The private message field is limited to 1,000 characters.
3. Collapse the panel.

Your private message is added and sent to individual recipients.
