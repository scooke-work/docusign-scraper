# Field Properties

Learn how to set Field properties to control the look and behavior of the fields you add to documents.

When adding fields to your documents, you can set various properties to:

- Make a field required or optional
- Change the recipient for a field
- Enter data in the field and make it read-only for recipients
- Apply data validation to enforce data entry rules for the recipient
- Set the font, size, or color for a text field
- Hide the recipient's entry in a text field with asterisks for added data security
- Set up conditional field relationships

The properties are similar for all field types, though not all properties apply to all fields. You can adjust the properties simultaneously on a single-field tag basis or multiple fields.
