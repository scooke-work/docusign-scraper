# Visibility of Read-Only Fields

Read about the visibility of read-only fields and the factors that affect what recipients can see.

Several factors affect which recipients can see read-only field contents. However, recipients cannot change these values.

- **Account settings to show values to all recipients**. Account administrators can enable a setting that makes the values you enter visible to all recipients before sending an envelope. For more information, see the [Sending Settings](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=bjh1583277352580.html&_LANG=enus) guide.
- **Show read-only field values to all recipients**. If the setting to show values isn't enabled, set a signing order and assign read-only fields to the first recipient in the order.
- **Show read-only field values for recipients who have finished signing.** If you don't use a signing order and don't enable the setting to hide values, recipients can see their own field values and those of previous signers.
