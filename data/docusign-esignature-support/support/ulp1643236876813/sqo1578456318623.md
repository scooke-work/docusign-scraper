# How to Add Recipients

Learn how to add recipients to act on or receive documents and specify the signing order and other options.

Add the people who will act on or receive a copy of your documents. You can specify a signing order, require additional authentication, and set other options.

Note: When adding recipients, special characters (like `< >`) in a name can prevent successful signature selection. To control how a recipient's name displays and to prevent changes during signing, use the Lock Recipient Name feature. This ensures that the recipient signs with the exact name that you provide. For more information, see [How to Lock Recipient Name option prevents recipients from changing their name when adopting a signature?](https://community.docusign.com/esignature-111/how-to-lock-recipient-name-option-prevents-recipients-from-changing-their-name-when-adopting-a-signature-2061)

1. At the Add recipients section, enter the email address and name of your first recipient.

   ![The Add recipients section where you add recipients.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ulp1643236876813/page/images/add-recipients-how-to-add-recipients.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQxNjcsInNoZWFmIjoidWxwMTY0MzIzNjg3NjgxMyJ9.A6X8CKyqKLY1pGLrA77Gi60XBg0j87sc-kAWZjk9kgA&_LANG=enus "The Add recipients section where you add recipients.")

   Note: To add a fax recipient, see the [Fax Documents for Signature](https://support.docusign.com/s/document-item?bundleId=gav1643676262430&topicId=nry1578456314231.html&_LANG=enus) guide.
2. To set the action for the recipient, select one of the following from the dropdown list:

   Document recipients:

   Needs to Sign
   :   Recipients must complete the required fields assigned to them, such as signature, initial, and date.

   In-Person Signer
   :   Recipients join a Docusign account holder who facilitates a signing session. For more information, see the [In-Person Signing](https://support.docusign.com/s/document-item?bundleId=gav1643676262430&topicId=hak1578456316043.html&_LANG=enus) guide.

   Receives a Copy
   :   Recipients receive a copy with no further action required. For more information, see the guide to [CC a Recipient on an Envelope](https://support.docusign.com/s/document-item?bundleId=gav1643676262430&topicId=sbq1631221886588.html&_LANG=enus).

   Needs to View
   :   Recipients must open and view the document.

   Specify Recipients
   :   Recipients can complete the name and email address of the role recipients in the same position or later in the signing order.

   Update Recipients
   :   Recipients can edit recipient details for any remaining recipients in the signing order.

   Allow to Edit
   :   Recipients can make recipient and document changes (correct) to an in-process envelope. This recipient must have a valid Docusign account.

   For more information about Manager recipients, see the [Manager Recipients](https://support.docusign.com/s/document-item?bundleId=gav1643676262430&topicId=jbn1578456307301.html&_LANG=enus) guide.
3. To add additional recipient rows, select Add Recipient.

   Note: The maximum number of recipients for a single envelope is 99.

   Tip: To add yourself as a recipient on a document, select the Name field and select your name and email:

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/ulp1643236876813/page/images/addme.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQxNjcsInNoZWFmIjoidWxwMTY0MzIzNjg3NjgxMyJ9.A6X8CKyqKLY1pGLrA77Gi60XBg0j87sc-kAWZjk9kgA&_LANG=enus)
4. (Optional) Specify a signing order to control the order in which your recipients receive and sign your documents. For more information, see the [Set a Signing Order](https://support.docusign.com/s/document-item?bundleId=ulp1643236876813&topicId=kcb1578456479440.html&_LANG=enus) guide.
5. (Optional) To require a recipient to give additional information to identify themselves during the signing process, add a verification method:
   1. In the recipient's row, select Customize and Add identity verification.

      ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/ulp1643236876813/page/images/recipientmoremenu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQxNjcsInNoZWFmIjoidWxwMTY0MzIzNjg3NjgxMyJ9.A6X8CKyqKLY1pGLrA77Gi60XBg0j87sc-kAWZjk9kgA&_LANG=enus)
   2. Select a verification method for the recipient and complete the required fields. For more information, see the [Recipient Identity Verification](https://support.docusign.com/s/document-item?bundleId=gav1643676262430&topicId=kzp1578456318101.html&_LANG=enus) guide.

      The summary verification details are shown below the recipient information. ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/ulp1643236876813/page/images/authsummary.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQxNjcsInNoZWFmIjoidWxwMTY0MzIzNjg3NjgxMyJ9.A6X8CKyqKLY1pGLrA77Gi60XBg0j87sc-kAWZjk9kgA&_LANG=enus)

      Note: For enhanced security, you can add an access code and an identity verification method (phone call, SMS, or Knowledge-based).
6. For Standards-Based Signatures accounts (SBS) only, select the signature type. For more information, see the SBS [Sending Types](https://support.docusign.com/s/document-item?bundleId=xcm1643837555908&topicId=ehf1578456571322.html&_LANG=enus) guide and the list of supported [trust service providers](https://support.docusign.com/s/document-item?bundleId=xww1594245808169&topicId=hxl1594246065723.html&_LANG=enus).
