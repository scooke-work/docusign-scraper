# Add Fields to a Document

Learn how to add fields to a document for selected recipients to complete.

To add fields to a document:

1. Start an envelope and add documents and recipients.
2. Select **Next: Add Fields**.
3. In the recipient list at the top, select a person to add fields for.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/ulp1643236876813/page/images/addfieldsfor.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQxOTQsInNoZWFmIjoidWxwMTY0MzIzNjg3NjgxMyJ9.CQ4lZNAk-axrNsbPSgPDvra1yQJhZgglAkyKGfOmiSQ&_LANG=enus)

   Note: The fields you add are color-coded and apply only to the selected recipient. You can add fields for signers, but not for those who just need to view or receive a copy.
4. In the page guide on the right, select a page you want to tag with fields. The selected page becomes the active page and displays in the center panel.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/ulp1643236876813/page/images/newpageguide.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQxOTQsInNoZWFmIjoidWxwMTY0MzIzNjg3NjgxMyJ9.CQ4lZNAk-axrNsbPSgPDvra1yQJhZgglAkyKGfOmiSQ&_LANG=enus)

   Tip: To adjust a field's look, assigned recipient, or other characteristics, select the field. The field properties panel will slide in over the page guide. Set the properties as desired. For more information, see the [Field Properties](https://support.docusign.com/s/document-item?bundleId=ulp1643236876813&topicId=alk1578456282186.html&_LANG=enus) guide.
