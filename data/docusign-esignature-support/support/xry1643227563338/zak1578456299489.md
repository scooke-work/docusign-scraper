# Template Usage Options

Read about the different template advanced options settings.

Through the template advanced options settings, you can control how a template is used. The
selected template usage option applies only when the template is used from the
Templates page. It does not apply if the template is applied to
a document or added to an in-process envelope.

Details about the three template usage options and tips on why to use each type are
presented in the following sections.

## Quick send with advanced edit option

When the template is used, the sender sees the quick send view. In this view, the
sender simply completes the recipient and envelope information. The template used
must contain at least one document, one recipient, and one recipient field.
Optionally, the sender can select the Advanced Edit option
and use the full envelope edit mode. If the template is incomplete, the sender goes
directly to the full envelope edit mode. This is the default setting.

Why use this option: Offers full control to the sender over how they want to
complete the envelope generated from the template.

## Quick send only

When the template is used, the sender simply completes the recipient and envelope
information and sends. The sender cannot make any changes to the recipient list,
documents, or recipient fields. This setting requires that the template have at
least one document, one recipient, and one recipient field.

Note: The quick send only option is not supported for templates
containing any pre-fill fields.

Why use this option: Restricts senders to using the template as is, without
any options to change recipient, document, or field details.

## Advanced edit only

When the template is used, the sender completes the recipient and envelope
information using the full envelope edit mode. This sender can make any allowed
changes, including prefilling recipient fields.

Why use this option: Senders must go through all envelope and document
information. Use this option when senders must complete field entries before sending
or make other changes to recipients or documents.
