# Example Searches

Review some examples of how you can use the template search to locate templates in your
account.

Here are a few examples of how you can use the template search to locate templates in your account.

## Search for one of your templates by template name

Select the My Templates folder, enter "terms" in the search field, and
select the Template Name field filter.

The results include all of your templates with "terms" in the name. If the template is
located in one of your personal or shared folders, the folder name is listed in the
Folders column. If the Folders column is blank and you are not the
template owner, then the template is one that was shared with you directly and is in your
Shared with Me folder.

## Search for a template that someone shared with you

Select the Shared with Me folder or any shared template folder. Select the
Owner Name and Email field and enter any portion of the owner's
name.

## Search for one of your templates by recipient email

Find all templates that you own that have a recipient with a specific email address. Select your
My Templates folder and enter "records@dsxtr.com" in the search
field and select the Recipient Name and Email field to search:

## Search all templates by template name

Select the All Templates folder and enter "conference" in the search
field.

The results list all of the templates you have access to that include "conference" in the
template name. If the template is in a personal or shared folder, the folder name is listed
in the Folders column. If the Folders column is blank and you are not
the template owner, then the template is one that was shared with you directly and is in
your Shared with Me folder.

![Example search of all templates by template name.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/templatesearchbynameexample.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ2NzgsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.9GkjClf4GgG8UdpnVXFSg-QoBhGSq3Vs8KlCJgXu4K8&_LANG=enus "Example search of all templates by template name.")

## Search all templates by template ID

Locate a specific template by the template ID. Select the All
Templates folder, enter the complete template ID in the search field, and
select the Template ID field.

If the ID matches a template you have access to, it is listed in the results. If the
template is located in one of your personal or shared folders, the folder name is listed in
the Folders column. If the Folders column is blank and you are not
the template owner, then the template is one that was shared with you directly and is in
your Shared with Me folder.
