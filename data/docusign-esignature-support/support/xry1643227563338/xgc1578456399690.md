# Sharing Templates by Adding Them to a Shared Template Folder

You can select one or more templates and then select one or more shared folders to add them
to.

1. Navigate to the Templates page.
2. Locate the template you wish to share by searching, filtering, or simply scanning the
   Template list, and mark the check box to select it. You can select multiple templates at
   once.
3. Click SHARE TO FOLDERS.
4. In the Share to Folders dialog, search or scroll to locate the
   shared folder you want to add your selected templates to. To select multiple folders, hold
   down the Ctrl key and click the folders. Each folder name you click remains
   highlighted.
5. Click SHARE.

The templates are added to the selected shared folders
and available to all users with access to the folders.
