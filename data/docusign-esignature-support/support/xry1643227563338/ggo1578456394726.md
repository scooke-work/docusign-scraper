# Add a Password to a Template

Learn how to protect your shared templates from being modified or copied by adding a
password.

Protect your templates from being modified or copied
by adding a password. If the template is shared with other users, they must provide the
password to edit, copy, or share the template with others.

Note: Account administrators don't need to provide
passwords on protected templates. They can edit any template created within the account.

1. Start by creating a new template (see [Create
   Templates](https://support.docusign.com/s/document-item?bundleId=xry1643227563338&topicId=uab1578456394214.html&_LANG=enus "Creating a template is a lot like starting a document. You add files, recipients, messages, and signing fields much like when you create a document.")) or editing an existing template (see [Edit Templates](https://support.docusign.com/s/document-item?bundleId=xry1643227563338&topicId=xns1578456372645.html&_LANG=enus "You can edit any template that you previously defined and saved.")).
2. In the Add Documents and Recipients view, select Advanced
   Options.

   ![The 'Add documents' page with an arrow pointing to the ADVANCED OPTIONS link.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/advancedoptionslink2.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ1ODIsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.7_SFpuK7JmNofb0Lc9mMEVRokK9spS320p56FgyUJuU&_LANG=enus "The 'Add documents' page with an arrow pointing to the ADVANCED OPTIONS link.")
3. In the Advanced Options dialog, under the Template
   Password section, turn on password protection and enter a password.

   Template passwords are limited to 15 characters.
4. Click Save.
5. Finish creating or editing your template, and then save and close it.

   A key icon on the Template Details view indicates a password is
   active on the template.

   If your template is shared with other users, they must provide the password in order
   to edit it. Administrators are not required to enter passwords and can edit any template
   on the account.
