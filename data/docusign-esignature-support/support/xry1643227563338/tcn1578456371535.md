# Template History

Learn how to locate the history for a template. The template history shows the template
details including the template ID, owner, list of documents, and more.

The template history shows the template details including the template ID, owner, list of
documents, and more. The activity history lists the date and user for when the template was
created and each time it was edited.

1. Browse to the Templates page.
2. Locate the template that you want to edit by searching, filtering, or simply scanning
   the Templates list.
3. Click the template title to select it and open the template details view.
4. Click the MORE menu and select History.
