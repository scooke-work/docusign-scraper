# Template Passwords

Read about when to use template passwords.

If the templates that are shared with you have a password applied, you'll need the password
to edit the template. Passwords are set by the template creator. [Learn more...](https://support.docusign.com/s/document-item?bundleId=xry1643227563338&topicId=xwo1578456395432.html&_LANG=enus "Template passwords protect against editing. If the template is shared with other users, for them to edit the shared template, they must provide the password.")

If you are an account administrator, you don't need a password to edit the templates created
by your account users.
