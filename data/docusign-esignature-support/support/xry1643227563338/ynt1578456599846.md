# Specify Template Modification Settings

Follow the procedure to specify template modification settings.

1. From the template prepare view (either on a new template that you create or when
   editing an existing template), select ADVANCED OPTIONS.

   ![The Add Documents page with a document uploaded and an arrow pointed to the ADVANCED OPTIONS button.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/advancedoptionslink2.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ1OTMsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.1OI1xaq-EUJ40Dhq9v3TbDKsJDmqxn5Yo5bYaHg36K4&_LANG=enus "The Add Documents page with a document uploaded and an arrow pointed to the ADVANCED OPTIONS button.")
2. In the Template Modification section, select the check boxes for the specific restrictions you want to apply to the template:

   - Don't allow sender to edit, add, or remove recipients
   - Don't allow senders to edit the subject, email, or private messages
   - Don't allow senders to edit the brand
     ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/templatemodificationsettings.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ1OTMsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.1OI1xaq-EUJ40Dhq9v3TbDKsJDmqxn5Yo5bYaHg36K4&_LANG=enus)
3. Click Save to save your changes to the advanced options.
4. To finish editing your template, click Save and Close.
