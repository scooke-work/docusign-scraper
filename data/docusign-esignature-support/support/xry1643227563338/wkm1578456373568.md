# Add Merge Fields to a Template Message Subject

Learn how to use merge fields to personalize template message subjects with recipient
data and custom envelope fields.

You can add merge
fields to include recipient data and envelope custom fields (ECF) in the message subject. By
customizing the message subject with this data, you avoid creating multiple envelopes with
identical titles. The customized titles help you distinguish one transaction from one another.

1. Create a new template and add your files as usual.
2. Add one or more recipients, completing the Role field. You can
   also enter a name and email address for the recipient but it is not required.

   ![Template prepare view showing the Add Recipients section with two role recipients defined.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/mergerecipient.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ1NzAsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.zPEJuCcmriEpSdxX6XA2QlgjpOMKrhOOtMoXnUbHzMI&_LANG=enus "Template prepare view showing the Add Recipients section with two role recipients defined.")
3. For the
   Subject, select the merge field icon
   at the end of the
   field,
   and select one or more options to insert.

   The dropdown list contains options to insert the name and email address for each
   recipient on the template. Also listed are any ECFs defined on your account and visible to
   select on your template.

   ![Example of selecting an ECF to insert into the message subject.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/mergecodes.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ1NzAsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.zPEJuCcmriEpSdxX6XA2QlgjpOMKrhOOtMoXnUbHzMI&_LANG=enus "Example of selecting an ECF to insert into the message subject.")

   Note: The
   Subject field is limited to 100 characters. To insert a merge field, there
   must be sufficient characters remaining. You can edit the subject as needed to
   insert a merge field.
4. Add fields for your
   recipients
   as usual, and save and close the template.

When you use the template to start a new envelope, the message
subject and envelope title include the recipient information you specified. To populate ECF
merge fields, specify the values in the Add envelope custom fields
section of your envelope. If you don't set ECF values, the merge codes are ignored.
