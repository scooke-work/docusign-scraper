# Recipient Modification Settings

Learn about recipient modification settings.

You can limit the changes that senders can make to individual recipients on your template using the recipient advanced options.

If your template includes any manager recipients, that is, recipients with the action Allow to Edit, Specify Recipients, or Update Recipients, you can limit the recipient changes they can make. For more information about these recipient types, see [Manager Recipients](https://support.docusign.com/s/document-item?bundleId=xry1643227563338&topicId=jbn1578456307301.html&_LANG=enus).

## Limits for senders

For senders who use your template, the settings and their effects when the template is used or applied to a document are as follows:

![The Advanced Settings for a recipient highlighting the Recipient Modification options that don't allow senders to edit or delete the recipient.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/templaterecipientmodificationsettings.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ2MDMsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.JSgdAfXZIBA4KbEQbbWOjwTAGD-Qd1Dv_bG8ook5hLo&_LANG=enus "The Advanced Settings for a recipient highlighting the Recipient Modification options that don't allow senders to edit or delete the recipient.")

- Don't allow senders to edit recipient

  Completed recipient information for the selected recipient cannot be modified. If there is any missing recipient information, such as when only the role is provided, and the name and email address are still needed to send the envelope, the sender can complete all remaining fields.
- Don't allow senders to delete recipient

  The selected recipient cannot be removed. This setting ensures that the recipient is included on all envelopes sent from the template.

## Limits for manager recipients

For manager recipients, the settings and their effects when they are sent an envelope are as follows:

![The Advanced Settings page with Restrict Modification by manager recipients that lock names and emails.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/templaterecipientmodificationmanagerrecipients.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ2MDMsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.JSgdAfXZIBA4KbEQbbWOjwTAGD-Qd1Dv_bG8ook5hLo&_LANG=enus "The Advanced Settings page with Restrict Modification by manager recipients that lock names and emails.")

- Lock Name

  A manager recipient can't modify the recipient's name. If the name is not provided, either on the template or completed by the sender, then the manager recipient can enter it.
- Lock Email

  A manager recipient can't modify the recipient's email. If the email has not been provided, either on the template or completed by the sender, then the manager recipient can enter it.
