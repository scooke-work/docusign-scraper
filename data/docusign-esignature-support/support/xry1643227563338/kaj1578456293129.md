# Search for a Template

Learn how to search for a template.

You can search for templates by:

- template name
- owner name and email
- recipient name and email
- template ID

You can apply a date range filter to narrow your search results to a particular time
period.

1. From the Templates page, select the collection you want to search
   in, such as All Templates or any other template folder in your account.
2. Click in the Search box to enter a search term.

   ![Templates page with the search box and filters highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/selecttemplates.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ2NzMsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.wIy7tT-wmFTtYNxtMABsSMJpzv2Ux14x15RwKB_GrUo&_LANG=enus "Templates page with the search box and filters highlighted.")
3. (Optional) Adjust the Date range. By default, the range is set to "All time", but you
   can select shorter time periods to narrow your search. The Date range operates on the Last
   Change date.
4. Use the Advanced search to choose which template fields to
   search:

   ![Field selection for template search filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/searchtemplatesselectfieldoptions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ2NzMsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.wIy7tT-wmFTtYNxtMABsSMJpzv2Ux14x15RwKB_GrUo&_LANG=enus "Field selection for template search filters.")
5. Click Apply.

   Your template list updates to show all matching results. The
   Folders column shows you where a template is located, such as "My
   Templates" or a folder name.

   Note: The search and filter criteria persist until you change or clear them. So,
   if you navigate to other folders and then return to Templates later, the filter criteria
   are still applied. Likewise, if you navigate to a different page in Docusign and then return to Templates later, the filter criteria are
   still applied.
