# Folders for Shared Templates

Read about how shared templates are organized. Review descriptions of the different
categories to understand what each contains.

Provided you have the necessary user permissions to use templates, your Templates page includes a set of folders for shared templates:

![Templates list with the Shared with Me collection selected.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/sharedtemplates.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ3MzQsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.Khmb1CGskBMLoM4JBsZF93Nkq8QPRontotujbEDtga0&_LANG=enus "Templates list with the Shared with Me collection selected.")

Shared templates are organized into the following categories:

Shared Folders
:   Contains any shared template folders that others created and shared with you. Included
    are templates shared with you directly and with groups to which you belong. When someone
    adds a template to a shared template folder, it is shared automatically with anyone with
    access. Shared Folders also contains any folders that you create to share with others on
    your account. See this article for more information: [Share Templates](https://support.docusign.com/s/document-item?bundleId=xry1643227563338&topicId=lrh1578456401532.html&_LANG=enus "Sharing a template allows other users and user groups on your account to access and use it to send documents.").

Shared with Me
:   A collection of all templates that are shared with you, either directly or through a
    shared folder.

All Templates
:   A collection of all templates you can access. This collection includes your own
    templates and those shared with you. If you are an account administrator, this collection also includes all
    templates on the account that you do not own.
