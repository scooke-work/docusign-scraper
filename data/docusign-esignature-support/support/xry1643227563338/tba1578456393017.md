# Create a Template with Merge Fields for Bulk Send

Learn how to add merge fields to your template message subject to personalize bulk send
envelopes. By including recipient data and envelope custom fields, you can ensure that each
message subject is unique and relevant to the recipient.

If you have a template you want to use for a bulk send, you can customize the message subject
and title for each envelope generated. You can add merge fields to include recipient data
and envelope custom fields (ECF) in the message subject.

By customizing the message subject with this data, you avoid creating multiple envelopes
with identical titles. The customized titles help you distinguish one transaction from one
another.

When the template is used for a bulk send, the bulk send list contains the custom data for
each envelope. The list contains the recipient data for each role specified on the template.
If you include ECF merge codes, your bulk list must include a column with the ECF name and
provide the text string value. For ECFs, bulk send supports text string entries only. List
value ECFs are not supported. For more information, see [Prepare your template for the bulk send](https://support.docusign.com/s/document-item?bundleId=wtn1643071711000&topicId=sed1578456384133.html&_LANG=enus).

Note: For information on bulk send, see [Bulk Send for Multiple Recipients](https://support.docusign.com/s/document-item?bundleId=wtn1643071711000&topicId=kjz1578456386508.html&_LANG=enus).

1. Create a new template and add your documents as usual.
2. In the
   Add
   Recipients section, add the recipient roles.
3. In the Add Message  section, click the merge field
   icon at the end of the message subject field. Select one or more options to
   insert, up to the character limit for the field.

   The dropdown list contains options to insert the Name and
   Email Address for each recipient role on the template. Also
   listed are any envelope custom fields defined on your account and
   visible to select on your template.

4. Add fields for your role recipients as usual, and save and close the template.

When you use the template for a bulk send, the uploaded bulk
list provides the data for the merge
fields. For ECFs,
include a column with the ECF name and enter the value for each envelope in the bulk
list. Each envelope message subject is customized, making it easier to locate
and identify individual envelopes in the bulk send.
