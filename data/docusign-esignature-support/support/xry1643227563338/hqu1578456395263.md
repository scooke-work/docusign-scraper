# Change or Remove a Template Password

Learn how to change or remove a template password.

Template passwords are encrypted. If you forget the password you added to a template, you can easily remove it or enter a new one.

Only the template creator or an account administrator can change or remove a password.

1. Start by editing your template (as described in the [Edit Templates](https://support.docusign.com/s/document-item?bundleId=xry1643227563338&topicId=xns1578456372645.html&_LANG=enus "You can edit any template that you previously defined and saved.") guide).
2. Select ADVANCED OPTIONS > Template Password >  Reset.

   ![Screenshot of the Template Password section of the advanced options for a template.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/resettemplatepassword.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ1ODQsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.gKMIAOcKdATeLi0M_rn1nOQoD8TNPkhV-lYYHMRqJII&_LANG=enus "Screenshot of the Template Password section of the advanced options for a template.")

   The empty password fields display.
3. Do one of the following:
   - To enter a new password, complete the password fields.
   - To remove the password, leave the fields blank.
4. Select Save.
5. Save and close the template.
