# Template Passwords

Template passwords protect against editing. If the template is shared with other users,
for them to edit the shared template, they must provide the password.

Template passwords protect against editing, sharing, copying, or downloading a template. You
can set a password on any template you create.

If a [template is shared with other users](https://support.docusign.com/s/document-item?bundleId=xry1643227563338&topicId=lrh1578456401532.html&_LANG=enus "Sharing a template allows other users and user groups on your account to access and use it to send documents.") who have
the permissions to edit (template create or share), they must first provide the password. In
this way, you can share templates and not worry about them being edited, shared, or copied. If
you want to permit another user to edit your template, you can provide them with the password.

Note: Account administrators don't need to provide
passwords on protected templates. They can edit any template created within the account.

When you download a template, the password is removed from the downloaded copy. You can provide the downloaded template to others without having to provide password details.
