# Common Use Cases

Review some common use cases for uploading and downloading templates.

Templates are downloaded in JSON format when downloaded one at a time from the Template
details view. If you select templates from the Templates page to download, they are delivered
in JSON format in a .ZIP file. You can edit the template using a standard text editor or a
specialized editor. To send the downloaded file to another Docusign user, use your email or
other shared file service. Once the person receives your file, they can upload it to their
account.

Some common use cases for downloading and uploading templates:

- Give a copy of a template to someone in a different Docusign account
- Troubleshoot a template issue with Docusign Customer Support
- Move a copy of a template developed in your Docusign demo account to your production account
- Receive a template built for you by Docusign Professional Services
