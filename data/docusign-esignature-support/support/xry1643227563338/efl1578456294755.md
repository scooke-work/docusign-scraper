# Create, Edit, and Delete Folders

Learn how to create, edit, and delete template folders.

1. Go to the Templates page.
2. To create a folder:
   1. In the left-hand navigation pane, select Show More to reveal additional elements.
   2. Select the + icon next to
      FOLDERS.

      ![Screenshot of the menu control to add a new template folder.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/templatefolderadd.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ3OTYsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.c5mfE_F9a2MhYrI1fYKfp7ehuOOSZOYxNQpgr_NJxTw&_LANG=enus "Screenshot of the menu control to add a new template folder.")
   3. Enter a folder name and select Create.

   Your new folder is added to the list of folders.
3. To rename, move, or delete an existing folder, click the folder's 3-dot Action menu and make your selection.

   ![Screenshot of the Action menu for a folder.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/folder_action_menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ3OTYsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.c5mfE_F9a2MhYrI1fYKfp7ehuOOSZOYxNQpgr_NJxTw&_LANG=enus "Screenshot of the Action menu for a folder.")
4. To create a subfolder, click the folder's 3-dot Action menu and select
   New Subfolder.

   The new subfolder is nested under the selected folder.
