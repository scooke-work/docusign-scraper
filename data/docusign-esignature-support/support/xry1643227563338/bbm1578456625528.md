# Check and Set Matching Behavior on a Template

Template creators and account administrators can set template matching behavior on
templates. Template matching compares uploaded documents with template documents. Matches can
speed up envelope preparation by importing the recipients, messages, and recipient fields from
the template.

Template matching compares uploaded documents with
template documents. Matches can speed up envelope preparation by importing the recipients,
messages, and recipient fields from the template. As a template creator or an account
administrator, you can select which templates to include in matching.

Note: The ability to review and set the template matching behavior on an
individual template is limited to certain account plans. Your plan may not support the task
described in this topic.

1. Navigate to the Templates page.
2. Locate the template that you want to check by searching, filtering, or simply scanning
   the Templates list.
3. The current matching state is shown in the Templates list.

   ![My Templates list with three templates highlighted and showing the matching setting for each one.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/templatelistmatching.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ3NjYsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.AvR7ouuZZW5uuTzDMTK4_YjlnZegqAiyPrkV3huvjKc&_LANG=enus "My Templates list with three templates highlighted and showing the matching setting for each one.")
4. Set matching behavior from the Templates list:
   1. Select one or more templates by clicking the check box next to the template
      name.
   2. Click the more actions menu and select the matching behavior
      from the list.

   Include in Matching
   :   Template remains in the matching pool until you exclude or delete it.

   Exclude from Matching
   :   Template is removed from the matching pool until you include it again.
5. You can check matching from the template details view. The current
   matching behavior is listed below the template title. You can change the current setting
   from the MORE menu.
