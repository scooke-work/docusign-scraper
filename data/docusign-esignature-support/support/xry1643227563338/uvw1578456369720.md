# Edit a Template

Learn how to edit a template.

1. Go to the Templates page.
2. Locate the template that you want to edit by searching, filtering, or simply scanning
   the Template list.
3. Click the template actions menu and select Edit.

   If you do not see the Edit option, check with your Docusign administrator regarding
   your user permissions for templates.
   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/edittemplatelist.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ2MzQsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.NRVMuyqR60hJpMUTOMmLp8x67w7w_JXaBT25Rna5E3E&_LANG=enus)
4. If the template is password protected and you are not the owner, enter the password in the Enter Password dialog and click SUBMIT.

   The selected template opens in the Prepare Template view.
5. Edit the template. You can change any aspect of the template, including the template name, and the files, recipients, and recipient fields.
