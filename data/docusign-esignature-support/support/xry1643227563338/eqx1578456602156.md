# Specify Sender Permission Settings for a Field

1. From the template add fields view (either on a new template that you create or when
   editing an existing template), place the field onto the document.
2. In the properties panel on the right, expand the Sender Permissions
    section:

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/templatefieldlocks.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ2MTMsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.Kblq0XLzKqsPLrXDHDmvG4kBOF0QvJ7NmDixAMwrIyo&_LANG=enus)
3. Select the permission settings that you want to enforce when senders use the template:

   - **Restrict changes** - sender cannot change any field properties or delete the
     field
   - **Mandatory** - sender cannot delete the field
4. Save and close the template.
