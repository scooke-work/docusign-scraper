# Create a Template and Add Pre-fill Fields

Learn how to create a template that contains pre-fill fields. Your senders can use these
templates to pre-fill document fields before sending.

When you create a template for documents you send frequently, you can add
pre-fill fields to save more time. While a manager or administrator usually creates
templates, any user who has permission to create templates can perform this step.

For pre-fill text and checkbox fields, you can add data validation. Data validation
allows you to enforce a validation standard on the field, helping to ensure you get
the right data from the sender. Learn more about [data validation for text fields](https://support.docusign.com/s/document-item?bundleId=gbo1643332197980&topicId=jgd1578456357365.html&_LANG=enus) and [checkboxes](https://support.docusign.com/s/document-item?bundleId=gbo1643332197980&topicId=srj1578456517556.html&_LANG=enus).

When you use the template and send an envelope, as soon as a signer opens the document to view
or sign, or when a carbon copy recipient receives the envelope, data entered in a
pre-fill field becomes a permanent part of the document. The pre-fill data is
visible to all recipients with access to the document that it is placed on. From
that point on, the pre-filled data can no longer be changed, corrected, or
removed.

Note: If you add pre-fill fields to a template the template usage
option for quick send only is not supported. Template usage
is an advanced option setting. Read more about template usage in the [Template Usage Options](https://support.docusign.com/s/document-item?bundleId=xry1643227563338&topicId=zak1578456299489.html&_LANG=enus "Read about the different template advanced options settings.") topic.

You should already know how to [start a new
template](https://support.docusign.com/s/document-item?bundleId=xry1643227563338&topicId=uab1578456394214.html&_LANG=enus "Creating a template is a lot like starting a document. You add files, recipients, messages, and signing fields much like when you create a document.").

1. When you enter the prepare view for your template, select the pencil icon on
   the left to access the Pre-fill Tools.

   ![The Pre-fill Tools tab.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/prefilltoolstemplate072021.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ1NjMsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.vlOly3QQiS7w4OVtK_LKwYLKmGxd2nwn0ffln2ALIRA&_LANG=enus "The Pre-fill Tools tab.")
2. Drag the pre-fill fields that you want to use onto the document. See the [pre-fill field types](https://support.docusign.com/s/document-item?bundleId=gbo1643332197980&topicId=hlt1627421322398.html&_LANG=enus) for details.
3. (Optional) You can require senders to complete a pre-fill field before sending.
   Select the field, then select Mandatory in the Field
   Properties panel.

   ![Selected pre-fill field and property panel highlighting the Mandatory Sender Permissions option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/prefilltoolstemplatemandatory072021.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ1NjMsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.vlOly3QQiS7w4OVtK_LKwYLKmGxd2nwn0ffln2ALIRA&_LANG=enus "Selected pre-fill field and property panel highlighting the Mandatory Sender Permissions option.")
4. (Optional) For pre-fill text and checkbox fields, you can add data validation.
   Select the field, then expand the Validation property and
   select the validation to apply.

   ![Validation property menu expanded to show list of options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/validationoptions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ1NjMsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.vlOly3QQiS7w4OVtK_LKwYLKmGxd2nwn0ffln2ALIRA&_LANG=enus "Validation property menu expanded to show list of options.")
5. (Optional) If you want to set initial values for any pre-fill fields before
   saving the template (such as the name of your department or division), do that
   now. Select the field, then enter the value for it in the Add
   Text property.

   Note: To lock the values so senders can't change them,
   select Restrict Changes in the Sender
   Permissions property.
6. Finish preparing your template with fields for signers.
7. Select SAVE AND CLOSE.

The template is saved, and all administrators on the account can
access and use it. If you want others in your organization to use the template, you must
share it. Those users must have permissions to use templates. For details, see [Permission Profiles](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=pof1583277362435.html&_LANG=enus).
