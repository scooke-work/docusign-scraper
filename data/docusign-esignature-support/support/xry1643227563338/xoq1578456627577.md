# Upload Templates

Follow the procedure to upload temples to your account.

When you upload a template, it creates a new template in your account and you are the owner of the template.

You can upload any individual template downloaded from Docusign
eSignature. You can upload individual templates in either JSON or XML format to any Docusign account. You can also upload .ZIP files containing multiple
templates. When you upload a .ZIP file, it creates separate templates for each file in the
.ZIP file.

Note: There is a 25-MB limit for uploading
templates in any format (JSON, XML, or as a .ZIP file).

1. Go to the Templates page.
2. Select Start > Envelope Templates > Upload Template.

   ![Upload template menu selection.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/uploadtemplate.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ3ODUsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.VHEkOMdVSwdqoW21kAwnpsuQ7qctz9ghAM8-MtigkYU&_LANG=enus "Upload template menu selection.")
3. Browse to the location of the template file you want to upload and open it.

   You can upload an individual JSON or XML template file, or a collection of zipped
   template files. If you upload a zipped file, the system automatically unzips and creates
   individual templates.
4. The new template displays at the top of your Templates list.

   If an uploaded template has the same name as an existing template, the name of the
   uploaded template has a number in parentheses added to the template name.
