# Specify the Template Usage Option

Learn how to specify the template usage option.

Use the step-by-step procedure to specify the
template usage option.

1. From the template prepare view (either on a new template that you create or when
   editing an existing template), select ADVANCED OPTIONS.

   ![The 'Add documents' page with an arrow pointed to the ADVANCED OPTIONS link.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/advancedoptionslink2.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ2MDAsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.tyEu6KKnMwU6JcOEJSxumUo7SuTjN-hargAn8PK0EuI&_LANG=enus "The 'Add documents' page with an arrow pointed to the ADVANCED OPTIONS link.")
2. In the Template Usage section, select the option for the
   specific usage option you want to apply to the template:

   - Quick send with advanced edit option (default)
   - Quick send only
   - Advanced edit only
     ![The list of options for the specific template usage that you want to apply to the template.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/templateusageoptions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ2MDAsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.tyEu6KKnMwU6JcOEJSxumUo7SuTjN-hargAn8PK0EuI&_LANG=enus "The list of options for the specific template usage that you want to apply to the template.")
3. Click Save.
4. To finish editing your template, click Save and Close.
