# Use a Template to Create an Envelope

Learn how to quickly create an envelope using a prepared template.

You can use any saved template you can access to start a new envelope. Everything defined
in the saved template is applied and added to the new envelope. You can use the template as
is, add additional files, modify signing fields, add recipients, and add and modify
messages.

Watch a short video on creating and sending an envelope using a template. Then use the
step-by-step instructions to guide you through the process.

1. Navigate to the Templates page.
2. Select the template collection you want to select from:

   - **My Templates** - contains all of the templates you created or were transferred to your ownership.
   - **Shared with Me** - contains all the templates that are shared with you, either directly or through a shared folder. If you are an account admin, Shared with Me will include all templates on the account that you do not own.
   - **All Templates** - contains all templates to which you have access. This collection includes your own templates and those shared with you.
   - **Favorites** - contains all templates you mark as a favorite. You can have up to 50 favorite templates.
   - **Deleted** - contains all templates you have recently deleted.
3. Locate the template you want to use by searching, filtering, or simply scanning the list.
4. To use the template to start a new document, click USE.

   ![Screenshot of the Favorites template list with the Use button highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/usetemplatelist.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ3MTUsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.mEd1gYuErb2TM3FZsJBo5KLYXsFGQ2ZHwq5ebX3ZIGg&_LANG=enus "Screenshot of the Favorites template list with the Use button highlighted.")
5. A new envelope opens with your template settings loaded. If your envelope is complete
   with at least one file, one recipient, and one field, then follow [Use a Complete Template](https://support.docusign.com/s/document-item?bundleId=xry1643227563338&topicId=opk1578456289004.html&_LANG=enus "Learn how to use complete templates. Complete templates have everything required to generate an envelope. All you do is enter the recipient information.").

   Note: Depending on the usage options specified on the template, you may be
   directed to use the template as is, as described in [Use a Complete Template](https://support.docusign.com/s/document-item?bundleId=xry1643227563338&topicId=opk1578456289004.html&_LANG=enus "Learn how to use complete templates. Complete templates have everything required to generate an envelope. All you do is enter the recipient information."). Or you may have to complete the remaining steps in
   this procedure.
6. For any recipient roles, enter the actual email and name values.

   ![Add recipients section with a role recipient.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/rolerecipient.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ3MTUsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.mEd1gYuErb2TM3FZsJBo5KLYXsFGQ2ZHwq5ebX3ZIGg&_LANG=enus "Add recipients section with a role recipient.")
7. Enter your email message subject and text. If your template includes a message subject
   and text, that information is added to your envelope. You can use the template message as
   is, or modify it.

   TIP: If you send the same envelope over and over, enter a
   custom subject line. If you use the default subject as is, the envelopes will all be
   titled the same, making it hard to identify a particular one.
8. To review the recipient fields, click NEXT.
9. To send your envelope click SEND.
