# Remove a Template From a Shared Template Folder

Anyone with template *share* permission can remove a template from any of their shared template folders.

1. Navigate to the Templates page and select the All Templates
   view.
2. Locate the template you wish to remove from sharing by searching for some portion of
   the template name.

   ![Folder tags indicating which folder a template is in](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/searchalltemplates.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ3MDMsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.uBQDfMjs9ow9CCntfSB14cFU7SbXkuAMulKr3DiXFqc&_LANG=enus "Folder tags indicating which folder a template is in")

   If you know which shared folder the template is in, you can just navigate there
   directly and skip to Step 4.
3. The FOLDERS column shows which folders the template is located in.
   Click the shared folder label to view the folder's contents. Example: From the above
   screenshot, click the "HR" label to open the HR shared template folder.

   ![Templates in a shared template folder](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/sharedtemplatefolder.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ3MDMsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.uBQDfMjs9ow9CCntfSB14cFU7SbXkuAMulKr3DiXFqc&_LANG=enus "Templates in a shared template folder")
4. Select the check box for the templates you want to remove from the shared folder, and
   then click MORE and select Remove from
   Folder.

   ![Remove option to remove a template from a shared folder](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/removetemplatefromshared-folder.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ3MDMsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.uBQDfMjs9ow9CCntfSB14cFU7SbXkuAMulKr3DiXFqc&_LANG=enus "Remove option to remove a template from a shared folder")

   The template is removed and no longer shared with the users who have access to the
   shared folder.
