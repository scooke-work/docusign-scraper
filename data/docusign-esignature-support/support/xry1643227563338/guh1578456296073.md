# Move a Template Into a Folder

Learn how to move a template from the Templates list into a folder.

- From the Templates list:
  - Select one or more templates, and click Move and select the folder to
    move to.
    ![Screenshot of the Templates list that shows two selected templates and the Move button.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/movetemplate.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ3OTksInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.5PkzUv63cJGNb6WrgCxs1LCjg7OrHSqe6xuniXWVN6g&_LANG=enus "Screenshot of the Templates list that shows two selected templates and the Move button.")
  - To move one template, click the template's more actions menu and select
    Move. Select the folder to move the template to, and click
    MOVE.
    ![The Move to Folder window that displays when you select the Move button.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/movetemplateactions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ3OTksInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.5PkzUv63cJGNb6WrgCxs1LCjg7OrHSqe6xuniXWVN6g&_LANG=enus "The Move to Folder window that displays when you select the Move button.")
- From the Template details view:
  - Click MOVE and select a folder to move the template to.
  ![Screenshot of the template details view with the MOVE button highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/movetemplatedetails.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ3OTksInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.5PkzUv63cJGNb6WrgCxs1LCjg7OrHSqe6xuniXWVN6g&_LANG=enus "Screenshot of the template details view with the MOVE button highlighted.")
