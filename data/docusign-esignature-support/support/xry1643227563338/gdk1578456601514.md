# Specify Recipient Modification Settings

1. From the template prepare view (either on a new template that you create or when
   editing an existing template), on a recipient row, select CUSTOMIZE >  Advanced settings.

   ![Template role recipient card with Customize menu expanded and Advanced settings option highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/xry1643227563338/page/images/advancedrecipientoptions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ2MDgsInNoZWFmIjoieHJ5MTY0MzIyNzU2MzMzOCJ9.rk9dHBHkH5czfWnnTt8WQbfNKa0mp5C6LPUl8mgratQ&_LANG=enus "Template role recipient card with Customize menu expanded and Advanced settings option highlighted.")
2. In the Advanced Settings panel, select the check boxes for the
   modification settings you want to apply to the recipient:

   - Don't allow senders to edit recipient
   - Don't allow senders to delete recipient
3. If your envelope includes any manager recipients (a recipient of the type Allow to
   Edit, Specify Recipients, or Update Recipients), in the Advanced Settings panel, you can
   choose to lock their name or email address.
4. Click Done.

   The panel collapses and shows that settings are selected.
