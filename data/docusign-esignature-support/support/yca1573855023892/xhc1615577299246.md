# Sign With Singpass

Learn how to sign a document with Docusign and a Singaporean digital ID from
Singpass.

If you’re eligible for a digital ID from [Singpass](https://www.singpass.gov.sg/),
a sender might require you to use one to sign a document. You must switch between Docusign in your browser and the Singpass mobile app. You should
have:

- An iOS mobile device (version 11.0 or higher)
- The Singpass app installed and set up on your mobile device

1. In the notification that you have a document to sign, select REVIEW
   DOCUMENT.
2. Sign the document by following the steps in the [Docusign eSignature: How To Sign a Document](https://support.docusign.com/s/articles/How-do-I-sign-a-DocuSign-document-Basic-Signing) video,
   then select CONTINUE.
3. You’re prompted to finish signing with your digital ID by logging in to the
   Singpass mobile app. Select CONTINUE.

   ![Screenshot of the Docusign prompt to finish signing with Singpass.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/singpass1finishsigning.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ1MDAsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.dmu9C2l4GeX8iaFe10svDeKcV9URwz1aQcgSxRmspgM&_LANG=enus "Screenshot of the Docusign prompt to finish signing with Singpass.")
4. A reference code and a QR code display. Open the Singpass mobile app on your
   phone, where you enter these codes.

   ![Screenshot of the Singpass mobile app reference code and QR code.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/singpass2qrcode.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ1MDAsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.dmu9C2l4GeX8iaFe10svDeKcV9URwz1aQcgSxRmspgM&_LANG=enus "Screenshot of the Singpass mobile app reference code and QR code.")

   Note: The QR code is only
   valid for a short time (less than two minutes). If you don’t scan the QR
   code with Singpass right away, you might receive an error. You must then
   request a new QR code. If this happens, close the window on your computer
   that contains the QR code. Upon return to the document in the Docusign app, select Continue to
   start the process of connecting to Singpass again. A new QR code for you to
   scan displays.
5. In the Singpass mobile app under SAFEENTRY CHECK-IN,
   select Scan QR code.

   ![Screenshot of selecting the QR code scan location in the Singpass app.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/singpass3singpassapp1.jpg?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ1MDAsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.dmu9C2l4GeX8iaFe10svDeKcV9URwz1aQcgSxRmspgM&_LANG=enus "Screenshot of selecting the QR code scan location in the Singpass app.")
6. Scan the QR code that displays on your computer screen with the app.
7. A reference code displays in the Singpass mobile app. To check that you’re
   signing the right document, make sure that this reference code matches the one
   from the Docusign app, and select
   Sign.

   ![Screenshot of selecting Sign after confirming that the Signpass reference codes match.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/singpass4appsignscreen.jpg?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ1MDAsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.dmu9C2l4GeX8iaFe10svDeKcV9URwz1aQcgSxRmspgM&_LANG=enus "Screenshot of selecting Sign after confirming that the Signpass reference codes match.")
8. Authenticate with Singpass by using your fingerprint or facial recognition, or
   by entering your Singpass passcode.

   ![Screenshot of the waiting screen in the Singpass app.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/singpass5singpassappfinalscreen.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ1MDAsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.dmu9C2l4GeX8iaFe10svDeKcV9URwz1aQcgSxRmspgM&_LANG=enus "Screenshot of the waiting screen in the Singpass app.")

   ![Screenshot of the Docusign confirmation page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/singpass6done.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ1MDAsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.dmu9C2l4GeX8iaFe10svDeKcV9URwz1aQcgSxRmspgM&_LANG=enus "Screenshot of the Docusign confirmation page.")

   Important: The Docusign and Singpass apps are temporarily in waiting mode
   for up to a minute while you complete steps in the opposite app. Then the
   system processes and applies your signature. Don’t close either app until
   you see a notification that signing is complete.

The Docusign app displays a
Finalizing signature message. When the system finishes
processing your signature, you receive a notification that signing is complete in the
Docusign and Singpass apps. You can also log on to Docusign to save a copy of the document to your Docusign account.
