# Use Comments To Ask and Answer Questions About a Document

Learn how to add information to or ask for clarification in the documents you are signing.

## Post a Comment

When you're signing a document and you have questions about it, or disagree with something in it, you can post a comment in the document. You can type in a question and control that of the other recipients sees it.

For more information, see the [Settings to Allow Comments on Documents and Envelope](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=odr1583277453624.html&_LANG=enus) article.

1. Open the document about which you have a question or comment.
2. In the document, select the comment icon ![The esignature Add Comment icon](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnewaddcomm.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MDQsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.nl-Pa3U3U4AL2HgBry_4N3jUDvZD1jJbuVZVrH7OuxY&_LANG=enus "The esignature Add Comment icon").
3. Click in the document where you want to place the comment.

   ![A screenshot of a comment indicator.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnewcommindicator.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MDQsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.nl-Pa3U3U4AL2HgBry_4N3jUDvZD1jJbuVZVrH7OuxY&_LANG=enus "A screenshot of a comment indicator.")
4. In the text box on the right, type your question or comment.

   ![A screenshot of entering a comment.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnewcommquestion.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MDQsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.nl-Pa3U3U4AL2HgBry_4N3jUDvZD1jJbuVZVrH7OuxY&_LANG=enus "A screenshot of entering a comment.")
5. Select who can see the comment. In this example, only one user can view the comment.

   ![A screenshot of a sample Select Who Can See this pop-up prom](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnewcommsee.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MDQsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.nl-Pa3U3U4AL2HgBry_4N3jUDvZD1jJbuVZVrH7OuxY&_LANG=enus "A screenshot of a sample Select Who Can See this pop-up prom")

   You can only notify recipients who:

   - Have their preferences set so they receive notifications
   - Are either currently signing the envelope or have already signed

   Important: You cannot edit or delete a comment after you've posted it. Before posting your comment, carefully enter your text and select the person to view the comment.
6. Select POST.

   After you select POST in the disclaimer, your comment is posted. If you select CANCEL, your comment is not posted.

   After you post a comment, it looks like the following example:

   ![A screenshot of completed comment posting](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnewcommfinal.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MDQsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.nl-Pa3U3U4AL2HgBry_4N3jUDvZD1jJbuVZVrH7OuxY&_LANG=enus "A screenshot of completed comment posting")

Depending on how your notification preferences are set, you receive an email notification when someone replies to your comment. If you do not have an account with Docusign, you receive an email notification each time a comment or reply is made that is visible to you.
