# Print and Sign Documents

Learn how to use the Docusign Print & Sign option to return the document to the
sender by upload or fax.

Complete the entire print and sign task in a single session. If you quit in the middle, you
must start at the beginning the next time you attempt to print and sign.

Print and sign your documents

1. From the document you are signing, open the menu OTHER ACTIONS.
2. Select PRINT & SIGN.

   Note:

   PRINT & SIGN is only available if the sender turns
   on this feature. If the  PRINT & SIGN option is not
   available in the Other Actions menu, select FINISH
   LATER. Contact the sender and ask to resend the envelope with the
   Print & Sign option enabled.

   If you see Print & Sign: Download Document, the
   document's sender doesn't support return by FAX.

If the sender supports documents to be returned by either upload or fax, this Print
& Sign dialog box displays. You can choose how you return the signed documents.

Choose between the Upload or Fax return options:

- Prepare your documents for upload.
- Ensure all required fields are completed.
- Scan the signed documents.
- Upload the scanned documents to the designated platform.

You can sign the documents electronically anytime by selecting
CANCEL. Selecting CANCEL brings you back to
the electronic signing.
