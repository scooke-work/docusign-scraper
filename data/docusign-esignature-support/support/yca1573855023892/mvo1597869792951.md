# Add Drawings to Documents You Sign

Learn how to use the Drawing field to add free-form drawings to specific areas within
the document.

Add
free-form drawings to specific areas within the
document
with the Drawing field. The sender defines these areas and can
include an option for you to add your own image.

Here are some examples for using the Drawing field:

Accident report
:   Draw on the accident information diagram. Optionally upload an image of your
    car and draw on it to indicate the damaged areas.

House survey plan
:   Draw on the survey plan to indicate fence installation.

Medical record form
:   Draw on an image of the human body to indicate injury or pain points.

To draw on images in a document:

1. Open the document as described in the [Docusign eSignature Signing Process](https://support.docusign.com/s/document-item?bundleId=yca1573855023892&topicId=lou1573854977109.html&_LANG=enus "Learn about the Docusign eSignature signing process and eSignature signing terminology.")
   article.
2. Locate the Drawing field.

   ![Screenshot of the FILL IN prompt in a drawing field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signerfirstdraw.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQzOTksInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.22ctq4nxPHfUU1yg0InwR6r76n9vk0vYhcKq6SfI0UY&_LANG=enus "Screenshot of the FILL IN prompt in a drawing field.")
3. Click anywhere in the Drawing field to activate the drawing area and display
   the Draw menu.

   ![Screenshot of the Drawing field menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signerdrawmenu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQzOTksInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.22ctq4nxPHfUU1yg0InwR6r76n9vk0vYhcKq6SfI0UY&_LANG=enus "Screenshot of the Drawing field menu.")

   Use a simple online pen tool to draw lines and shapes in six basic colors anywhere within
   the dimensions of the field.

   Using the Draw menu, you can:

   - Select a color
   - Draw with different thicknesses
   - Clear everything you've drawn with CLEAR ALL
   - Save your changes and drawing before you finish signing
   - Resize
     uploaded
     images
4. Add your drawings to the Drawing field as shown in this auto insurance accident
   report example.

   ![Screenshot of a completed Drawing field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signerdrawfield.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQzOTksInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.22ctq4nxPHfUU1yg0InwR6r76n9vk0vYhcKq6SfI0UY&_LANG=enus "Screenshot of a completed Drawing field.")
5. If the sender included the option to upload an image in an additional Drawing
   field:

   1. Click anywhere in the Drawing field and select
      Upload.
   2. Use the slider to adjust the file size if required, and
      select
      Crop.
      ![Screenshot of upload a resizing image for a drawing you are adding.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signerdrawslider.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQzOTksInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.22ctq4nxPHfUU1yg0InwR6r76n9vk0vYhcKq6SfI0UY&_LANG=enus "Screenshot of upload a resizing image for a drawing you are adding.")

   The following examples show a completed auto accident report.

   ![Screenshot of a document's completed drawing fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signerdrawcompleteupload.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQzOTksInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.22ctq4nxPHfUU1yg0InwR6r76n9vk0vYhcKq6SfI0UY&_LANG=enus "Screenshot of a document's completed drawing fields.")
6. Complete any other signing fields and finish signing as described in the [Complete the Signing Session](https://support.docusign.com/s/document-item?bundleId=yca1573855023892&topicId=abm1573854992359.html&_LANG=enus "Learn how to complete signing and optionally save or print a copy of your signed documents.") article.
