# Select Your Default eSignature Account

If you have multiple Docusign eSignature accounts associated with the same email
address, learn how to designate a default account to receive envelopes and the associated
documents.

If you have multiple Docusign eSignature accounts associated with the same email address, you
can designate a default account to receive envelopes and the associated documents.

You can now change envelope routing to a designated account that you can easily monitor.

1. Log on to your preferred eSignature account.
2. If you have multiple accounts, make sure you are logged on to the one you want
   for your default. To switch to a different account, select your profile image
   and select Switch Account.
3. In the account you want as your default, select your profile image and select
   My Preferences.
4. In General Settings, select Your Default
   eSignature Account.
5. To ensure consistent envelope routing select Set as
   default twice.

Any envelopes you receive now go to default account.

Note: If you are a recipient in an envelope you sent, you receive
that envelope in the sending account. Disregard the default account.
