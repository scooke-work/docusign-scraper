# Search for Text in a Document

Learn how to use the Search feature to quickly locate specific keywords or phrases in a document during a signing session.

## Before you begin

You must use a desktop device set to English to search envelopes with 10 or fewer documents and 100 or fewer pages. This feature is disabled when Universal Signatures are enabled for the envelope. Image-only documents are not searchable.

You can use the Search feature to find exact text matches within a document. This allows you to quickly navigate to relevant sections without scrolling through the entire file.

1. During a signing session, open the Search tool using one of the following methods:
   1. Select the Search icon (magnifying glass) in the right-hand vertical toolbar.
   2. On Windows, select Ctrl+F**.**
   3. On Mac, select Cmd+F.
2. Enter a search term containing two or more characters.

   All exact matches are highlighted in the document.![The screenshot shows the Search tool and highlighted keywords in a document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/Searchexample.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQzNTQsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9._cH1IlD9h9BxbJhRgPz18V2y3x8QE4mUTNFw5WJp2C0&_LANG=enus "The screenshot shows the Search tool and highlighted keywords in a document.")
3. Use the Up and Down arrows in the search panel to move between matches.

You have located the specific text within the document.
