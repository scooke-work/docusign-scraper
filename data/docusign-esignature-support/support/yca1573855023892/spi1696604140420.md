# There was an unexpected fault (I get an error while using My SAT certificate )

Situations that cause the There was an unexpected fault error
message and how to solve them.

The error message There was an unexpected fault displays after I submit
my SAT certificate and private key for signing documents.

## Cause

- The certificate is expired.
- The certificate is revoked.
- The network encountered an issue.
- The uploaded file is not supported.

## Remedy

1. Ensure that your internet connection is working properly.
2. Ensure you submitted a .cer file for your SAT certificate and
   the`.key` file for the private key.
3. Open your certificate and check the following details:

   - The end date is still valid.
   - The certificate is not revoked.

   If the details of your certificate contain inaccurate or expired
   information, contact the department that provided you with your
   certificate to create a new one.
