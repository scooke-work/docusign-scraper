# Additional Signing Actions

Read about all signing options when signing Docusign agreements.

Docusign eSignature provides many other signing options:

- In the Finish menu, select the dropdown arrow to display the
  Other Actions options.

  ![A screenshot of the Signing Other Actions menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/sign-new-other-actions-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0NDQsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.rXKd9ikd0gUlDsVLqcxQkyin7PgxPQ2WM438nthd-Mw&_LANG=enus "A screenshot of the Signing Other Actions menu.")

  You can perform these signing actions on a document you're signing as
  described in these topics:

  - Finish later: Save your changes and complete signing at another time
  - [Change Signing Responsibility](https://support.docusign.com/s/document-item?bundleId=yca1573855023892&topicId=uin1573855010064.html&_LANG=enus "Learn how to assign a document to another person if you're not the right person to sign the document.")
  - [Decline To Sign an Envelope](https://support.docusign.com/s/document-item?bundleId=yca1573855023892&topicId=fqo1573855017663.html&_LANG=enus "Learn how and when to decline to sign an envelope you review.")
  - [Print and Sign Documents](https://support.docusign.com/s/document-item?bundleId=yca1573855023892&topicId=wdr1573855014901.html&_LANG=enus "Learn how to use the Docusign Print & Sign option to return the document to the sender by upload or fax.")
