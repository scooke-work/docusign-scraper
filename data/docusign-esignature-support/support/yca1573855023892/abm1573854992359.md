# Complete the Signing Session

Learn how to complete signing and optionally save or print a copy of your signed
documents.

After you add information to all the required fields and any signatures or initials, you
are ready to complete the signing session.

For short documents, such as a job offer letter, you can easily find and sign all the
required fields. However, for long or complex documents, such as a merger proposal, you may
fail to find all required fields. During a Docusign signing session, the
system makes sure you find and complete all required fields before completing the
process.

1. Review the information that you added to all the information fields and make any
   changes if necessary.

   Note the following:

   - You cannot complete the signing session if you enter information in an optional
     field, but the data is invalid. For example, if you added your zip code, but omitted a
     number. When you attempt to complete signing, the system takes you to the zip code
     field and prompts you to revise your input.
   - Make sure the provided information is correct, as you cannot change it after
     completing the signing session.
2. After reviewing the document, and completing all required fields, the following prompt
   displays.

   Note the following:
   - If you select Finish without completing all required signing
     fields, the system displays the next required field that you must complete.

     ![A screenshot of a prompt to complete the next signing field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnewpromptifnotdone.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MzksInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.8rO1HPAFpZze0h9FcywPyZLGlkBOu4JUjDZgEd-wjjE&_LANG=enus "A screenshot of a prompt to complete the next signing field.")
   - If you attempt to exit the browser, change the URL, or refresh the signing session
     page, the system prompts you to confirm that you want to abandon your signing
     session.

     ![A screenshot of the Leave site confirmation message.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnewleaveconfirm.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MzksInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.8rO1HPAFpZze0h9FcywPyZLGlkBOu4JUjDZgEd-wjjE&_LANG=enus "A screenshot of the Leave site confirmation message.")
3. Complete the required fields and select Finish.

   Your signing information is saved and a confirmation displays.

   - If you already have a Docusign account, a copy is saved to your account.
   - If you don't have a Docusign account, a copy is sent to your email account when
     completed by all signers.

     You can optionally sign up for a free Docusign account as
     described in the [Electronic signing made simple](https://www.docusign.com/esignature/sign-documents-free) article. Sign up and save the document in
     your account.
4. If you don't complete the signing session, it eventually times out, and the following
   notice displays:

   ![A screenshot of signing session timeout notice.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signtimeoutnotice.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MzksInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.8rO1HPAFpZze0h9FcywPyZLGlkBOu4JUjDZgEd-wjjE&_LANG=enus "A screenshot of signing session timeout notice.")
5. Optionally download or print the document by selecting the appropriate icon.

   ![Screenshot of signing download and print icons.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signingprintdownload.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MzksInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.8rO1HPAFpZze0h9FcywPyZLGlkBOu4JUjDZgEd-wjjE&_LANG=enus "Screenshot of signing download and print icons.")

   1. Select the Download icon to save the document as separate PDFs or combined PDFs and
      choose the download location.

      - Separate PDF: Downloads the documents in a .ZIP file and individual PDFs.
      - Combined PDF: Downloads a single PDF that includes all documents.
   2. Select the Print icon and select a printer.

   You have successfully completed electronically signing your documents.
