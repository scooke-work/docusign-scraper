# Use Document Mark Up To Negotiate Changes

Use the document mark up feature to suggest changes to your document by adding
suggested changes within the document.

If the sender has enabled the document mark up feature for the envelope, you can use this
feature to negotiate changes to documents during the Docusign signing
workflow.

Note: Mobile devices do not support document mark
up.

1. When Document mark up is enabled, select OTHER ACTIONS > Mark Up .

   ![A screenshot of the Markup option in the Docusign Sign menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/markup-selection.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MDIsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.yL1fTqBUPMif9McVW6YZaLF6-Pu2_GJIZ-mrGFcQCII&_LANG=enus "A screenshot of the Markup option in the Docusign Sign menu.")

   The Markup instructions display.

   ![A screenshot of the Markup instructions when signing a document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnewmarkinstructions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MDIsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.yL1fTqBUPMif9McVW6YZaLF6-Pu2_GJIZ-mrGFcQCII&_LANG=enus "A screenshot of the Markup instructions when signing a document.")
2. Select CONTINUE.
3. A special mark up field displays. You can drag and place the mark up field anywhere on
   the document.

   ![A screenshot of the document mark up field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/markuptag.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MDIsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.yL1fTqBUPMif9McVW6YZaLF6-Pu2_GJIZ-mrGFcQCII&_LANG=enus "A screenshot of the document mark up field.")
4. (Optional) Enter information into the mark up field.

   ![A screenshot of a sample document mark up field you can edit.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/markuptextentry.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MDIsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.yL1fTqBUPMif9McVW6YZaLF6-Pu2_GJIZ-mrGFcQCII&_LANG=enus "A screenshot of a sample document mark up field you can edit.")

   An Initial tag displays at the margin to the right of the field that you must sign to
   complete the mark up action.

   ![A screenshot of a sample document mark up add initials tag.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/markuptext-entry.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MDIsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.yL1fTqBUPMif9McVW6YZaLF6-Pu2_GJIZ-mrGFcQCII&_LANG=enus "A screenshot of a sample document mark up add initials tag.")
5. If other recipients have already signed the document, they receive an email
   notification to review the suggested changes. When they view the document, they are
   prompted to reply.

   ![A screenshot of a reply to a mark up request in a signed document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnewmarkupreply.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MDIsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.yL1fTqBUPMif9McVW6YZaLF6-Pu2_GJIZ-mrGFcQCII&_LANG=enus "A screenshot of a reply to a mark up request in a signed document.")

   Recipients can accept the change by initialing it, or edit the change and initial their
   version. The review process continues until all recipients have initialed all changes.

   ![A screenshot of a prompt to initial document mark up suggested changes.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/markupreviewinitial.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MDIsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.yL1fTqBUPMif9McVW6YZaLF6-Pu2_GJIZ-mrGFcQCII&_LANG=enus "A screenshot of a prompt to initial document mark up suggested changes.")
6. Once all changes are initialed, the document is complete and a completed email
   notification is sent to all recipients. The notification includes details of the mark up
   changes.
7. The final document includes all completed fields, including the mark up changes and
   corresponding initials.

   ![A screenshot of final document listing mark up changes and initials.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/markupdinal.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MDIsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.yL1fTqBUPMif9McVW6YZaLF6-Pu2_GJIZ-mrGFcQCII&_LANG=enus "A screenshot of final document listing mark up changes and initials.")
