# Get Started on the Docusign Signing Welcome Page

Learn about the Docusign signing welcome screen and signing your
documents

The welcome page includes a message from the sender and a prompt to review and act on the
documents you received.

![A screenshot of the Review and Act on These Documents prompt.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/sign-review-other-options.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQzNDQsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.WrAs043YNIV4pTbcCGYMkP_SGxBxuJm1UA95sl9W_Zc&_LANG=enus "A screenshot of the Review and Act on These Documents prompt.")

1. If available, select the Electronic Record and Signature disclosure checkbox. Selecting
   this checkbox indicate you agree to the disclosure. The system starts the signing
   process.
2. Select CONTINUE to open and review the document.

   Note: We recommend that you review the document by scrolling. Or, you can
   also use the page thumbnails in the right-side navigation before signing.
3. Optionally, you can use the OTHER ACTIONS menu to select a
   different option for the documents.

   See the [Other Signing Actions](https://support.docusign.com/s/document-item?bundleId=yca1573855023892&topicId=qdy1573854987499.html&_LANG=enus "Learn about the available options for signers from the Other Actions menu.") article for information about these
   options.
4. After you have added the required information and signing the documents, the actions move
   to the bottom of the browser. FINISH becomes active.

   Select
   FINISH to complete signing. The success message
   displays.
