# Upload Signed Documents

Learn how to upload documents you must print, sign, and return.

To upload your documents as the return method, follow these instructions. If you've
already downloaded the documents but quit or timed out before returning your
documents, download the documents again and return using these
instructions.

1. In the Print &Sign Download Document dialog box, select
   DOWNLOAD to save the document to your computer.

   ![A screenshot of the Print & Sign Download Document Download dialog box to download a document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/sign-p-s-download-db.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0NTQsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.KoJs9tGoXu5paSBDs_-LpjFLhu995wTKIpYkrdEGnPU&_LANG=enus "A screenshot of the Print & Sign Download Document Download dialog box to download a document.")
2. Print the documents that you saved.
3. Complete and sign the printed pages.
4. Scan the signed documents and save them as a file on your computer.
5. In the Print &Sign Download Document dialog
   box, select RETURN DOCUMENT.

   ![A screenshot of the Print & Sign Document Return Document dialog box to return a document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/sign-p-s-return-db.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0NTQsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.KoJs9tGoXu5paSBDs_-LpjFLhu995wTKIpYkrdEGnPU&_LANG=enus "A screenshot of the Print & Sign Document Return Document dialog box to return a document.")
6. Select UPLOAD A FILE.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/sign-p-s-upload-db.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0NTQsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.KoJs9tGoXu5paSBDs_-LpjFLhu995wTKIpYkrdEGnPU&_LANG=enus)
7. Select the signed documents to upload the document.

   - Drag the signed documents to the dialog box.
   - Select UPLOAD A FILE, and choose the signed
     documents.
8. Select FINISH to send the document.

   ![A screenshot of the Print & Sign Document Finish dialog box to complete uploading a document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/sign-p-s-finish-db.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0NTQsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.KoJs9tGoXu5paSBDs_-LpjFLhu995wTKIpYkrdEGnPU&_LANG=enus "A screenshot of the Print & Sign Document Finish dialog box to complete uploading a document.")

   The final document includes the original document and the
   uploaded, signed files.

   Tip:

   To avoid unnecessary document duplication, only return the
   relevant signature pages.
