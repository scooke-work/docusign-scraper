# System Requirements for Signing

Review these minimum signing system requirements. These include signing online, signing with mobile devices, signing with assistive technologies, and Docusign Notary signing.

Modern computers work well with Docusign eSignature and must adher to these requirements

## Operating systems

- Windows 10 and newer
- macOS 10.13 (High Sierra) and newer
- Android 10 and newer
- iOS 14 and newer

## Browsers

Docusign supports the latest stable release (except where noted) of these browsers:

- Chrome
- Chrome for Android
- Firefox
- Microsoft Edge
- Safari
- Safari for iOS

Note: Prerelease (such as Beta) versions of operating systems and browsers are not supported.

## Recommended screen resolution

- Recommended: 1400x900 or higher
- Minimum: 1366x768

## Security settings

- Enable per-session cookies.
- Enable JavaScript
- Users access the internet through a Proxy Server must enable HTTP 1.1 settings through a proxy connection.
- Users access to the server through the firewall setting `https://docucdn-a.akamaihd.net`. Docusign uses Akamai as a content delivery service to enhance our application's performance.

## PDF reader

- Acrobat Reader or similar software might be required to view PDF files.
- The minimum required PDF version that your PDF reader must be able to open is PDF 1.5.

  For detailed information about PDF versions, see this [PDF versions](https://www.prepressure.com/pdf/basics/version) article.

Note: When uploading PDF/A files, we recommend using PDF version 1.5 to ensure that the files maintain their original appearance. For information about PDF/A files, see the [PDF/A file format](https://www.prepressure.com/pdf/basics/pdfa) page.

## Mobile signing

- The latest Apple iOS version
- The latest Android version

## Docusign Notary remote signing

Docusign Notary remote signing has additional requirements as well as the general signing requirements.

Note: See the  [Docusign Notary Signing Guide](https://support.docusign.com/s/document-item?bundleId=zle1621013049545&topicId=lap1604000866404.html&_LANG=enus)  for detailed information about signing documents with a remote Docusignonline notary.

- Devices: A laptop, desktop, or mobile device with a working camera and microphone

  Note: For the best results, Notaries and signers should use a laptop or desktop device. Signers can optionally use mobile devices.
- Browsers
  - Windows
    - Google Chrome current version
    - Firefox current version
    - Microsoft Edge current version for signers
  - macOS
    - Safari (macOS only) 12 +
    - Google Chrome current version
    - Firefox current version
    - Microsoft Edge current version
  - Mobile device: signers only; notary users are not supported
    - Apple iOS 11 +
    - Android 5.0 +
- Internet connection
  - User internet access broadband service must support a live, real-time audio and video connection. For the best results, we recommend that you hard-wire your device to the internet.
  - We recommend the following minimum internet speeds. Test your internet speed at [Speedtest.net](https://www.speedtest.net/).
    - Upload: 5 Mbps
    - Download: 5 Mbps

Validate that your system meets all requirements at the [Pre-call Test](https://tokbox.com/developer/tools/precall/) page.

## Signing with assistive technologies

With Docusign, you can use different assistive technologies to ensure that all users are able to sign our documents. You can sign documents in Docusign using:

- Screen readers that permit visually impaired users to follow the necessary guidelines to sign an envelope
- Keyboard-only use to make sure that any action can only be executed using this device

For specific information, see [Web Content Accessibility Guidelines (WCAG) 2.2](https://www.w3.org/TR/WCAG22/#new-features-in-wcag-2-2).
