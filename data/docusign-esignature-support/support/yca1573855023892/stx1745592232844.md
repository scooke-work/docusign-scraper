# Supported Languages for Signing

Discover which languages Docusign supports for signing documents in
localized languages.

Docusign offers users the ability to sign documents in localized
languages. Languages are detected through browser settings, and document senders can
also configure language settings accordingly. See the Extensive language support
for signing section in the [Global Business](https://www.docusign.com/products/electronic-signature/learn/global) article to view the available
languages for signing
