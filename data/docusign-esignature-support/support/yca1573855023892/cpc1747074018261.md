# Sign a Document with Data Verification Address Auto-Suggestion Fields

Learn about using address auto-suggestions to enter address fields easily and accurately
in Docusign envelopes and documents.

Your sender may use address auto-suggestion fields to easily and accurately
obtain addresses in Docusign envelopes and documents. With these
auto-suggestion address fields you can:

- Speed up the process of entering addresses
- Minimize errors

1. Begin signing the document as described in the [Sign a Document With Data
   Verification Fields](https://support.docusign.com/s/document-item?bundleId=yca1573855023892&topicId=vyg1698168078020.html&_LANG=enus "Learn how to sign a document that has fields with data that must be verified before successfully signing the document.") article.
2. In the Address Line 1  field, enter the first field of
   the address block.

   ![A screenshot of the typing in Address Line 1 to begin the Auto-suggestion address process.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/dvsmartyautosugline1.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MzQsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.wUmBd2SFiyC5ntaSriIGUC1B7rfYG9vSB-zC1pAN-Ns&_LANG=enus "A screenshot of the typing in Address Line 1 to begin the Auto-suggestion address process.")

   The Auto-Complete feature displays address options for the information that
   you entered.

   ![A screenshot of completed auto-complete address fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/dvsmartyautosugoptions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MzQsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.wUmBd2SFiyC5ntaSriIGUC1B7rfYG9vSB-zC1pAN-Ns&_LANG=enus "A screenshot of completed auto-complete address fields.")
3. Select the correct information from the suggestions.

   The Auto-Complete feature completes the remaining address fields.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/dvsmartyautosugcomplete.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MzQsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.wUmBd2SFiyC5ntaSriIGUC1B7rfYG9vSB-zC1pAN-Ns&_LANG=enus)
   ![A screenshot of completed auto-complete address fields.]()

   Tip: Optionally select Undo
   autofill and enter your own address
   information.

   When you've completed your entry and the verification process completes, you
   received a verification confirmation message.

   `Verification succeeded`

   4.

   5. If you receive an Address Verification failed message, select Check
   Address to display a list of suggested addresses.
4. If you receive an `Address Not Found` message, select
   Continue
   to
   return to the address fields and try again.
5. If you receive an `Address verification failed. We found a better match
   to ensure deliverability` message, select Check Address
   to
   display a list of suggested addresses.
6. Select Confirm to accept the address.
7. Complete any other signing fields and select
   Finish.
