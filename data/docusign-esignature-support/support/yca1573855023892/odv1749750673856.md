# Verify a Signature Request

Learn how Docusign proactively combats fraudulent activity, by
flagging possibly risky envelopes and requesting that the signer verify the signing
request.

To proactively combat fraudulent activity, Docusign
scores the risk associated with envelopes. This risk is based on
the sender account and envelope characteristics.

If
an envelope’s risk score is too high, you may
receive
a request to verify that the sender and contents are legitimate by confirming:

- You recognize the sender
- You were expecting the envelope

1. Begin signing as described in the [Docusign eSignature Signing Process](https://support.docusign.com/s/document-item?bundleId=yca1573855023892&topicId=lou1573854977109.html&_LANG=enus "Learn about the Docusign eSignature signing process and eSignature signing terminology.")
   article.

   If it is determined that this envelope is possibly fraudulent, a message similar to
   this
   example displays:

2. Select one of these actions:
   - Report Abuse if you don't recognize the sender or
     content. See the [Report Abuse or Fraudulent Activity](https://support.docusign.com/s/document-item?bundleId=yca1573855023892&topicId=uec1690221505200.html&_LANG=enus "Learn about the Docusign Report Abuse feature that you can use to notify Docusign about instances of fraudulent activity.") article for more
     information.
   - Continue if you're expecting the document or are
     familiar with the request's content.
