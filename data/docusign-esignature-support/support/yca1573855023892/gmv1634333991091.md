# Sign a Document Received through Email

Learn how to sign a document you receive through email.

Learn how to sign a document you receive through
email.

1. Open the email.

   ![A screenshot of the Review and Act on These Documents prompt.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnewersddb2.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQzNDcsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.OyfTEh9u3UX4gOIso7yYS4EdIKx8utddQOWgQFMRnYg&_LANG=enus "A screenshot of the Review and Act on These Documents prompt.")
2. If available, select the Electronic Record and Signature disclosure checkbox.
   Selecting this checkbox indicate you agree to the disclosure. The system starts
   the signing process.
3. If there is no Electronic Record and Signature disclosure, select the
   CONTINUE to start the signing process..
4. Add information to the required fields and select
   Sign.
5. If this is your first time signing a Docusign document, you are asked to adopt
   a signature as described in the [Adopt Your Signature](https://support.docusign.com/s/document-item?bundleId=yca1573855023892&topicId=dga1573854990297.html&_LANG=enus "Learn how to choose a style for your signature and initial when you first sign a document with Docusign.")
   topic.
6. Select Adopt and Sign to confirm your signature.
7. Select Finish to complete signing your document.

Note the following:

- If you select Finish without completing all required
  signing fields, the system displays the next required field that you must
  complete.

  ![A screenshot of the prompt identifying a required signing field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnewpromptifnotdone.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQzNDcsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.OyfTEh9u3UX4gOIso7yYS4EdIKx8utddQOWgQFMRnYg&_LANG=enus "A screenshot of the prompt identifying a required signing field.")
- If you attempt to exit the browser, change the URL, or refresh the signing session page,
  the system prompts you to confirm that you want to abandon your signing
  session by selecting CANCEL or
  LEAVE.

8. After completing the signing, optionally set up a free Docusign account to more
   easily manage your signed documents.

After all parties have signed the document, you receive a
copy of the document.
