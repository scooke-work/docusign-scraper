# Save or Print Document Supplements

Learn how to save or print a supplement to a document you're signing.

Save a copy of a supplement

A document you print or download can include supplements. Because supplements can be
lengthy, sometimes supplements may not be included when you download the signed
document.

For instructions on how to download or print a document, see the [How do I download or print a document I
receive?](https://support.docusign.com/s/document-item?bundleId=sto1596228661240.ditamap&topicId=kug1599162827686.html&_LANG=enus) article.

If supplements are not included in the document download, you can download a copy or
print from the supplement view. The easiest way to access a document is from the email
notification.

To download or print a supplement from the email notification:

1. Open an email notification that you received about the document.
2. Select Review Document.
3. Select Agree to use electronic records and signatures  > Continue

   If there's no electronic records and signature (ERSD) option, the document opens
   when you select Review Document..
4. From the opened document, find the supplement to download and select the supplement
   thumbnail to open the document.

   ![Screenshot of the thumbnails of supplemental documents.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signsuppliment.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MjAsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.ZtbNhKaVyM-Lp8jD19_koHKLPOTbVMyEMN970dhGGRI&_LANG=enus "Screenshot of the thumbnails of supplemental documents.")
5. After viewing the supplement, save a copy of it by selecting the
   Download icon ![Document download icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnewdownloadicon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MjAsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.ZtbNhKaVyM-Lp8jD19_koHKLPOTbVMyEMN970dhGGRI&_LANG=enus "Document download icon.") .

   The PDF of the supplement opens in a new browser window. In the new window, select
   the Download icon on the top right of the PDF and save to
   your computer.
6. After viewing the supplement, you can print it by selecting the
   Print icon ![Document print icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnewprinticon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MjAsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.ZtbNhKaVyM-Lp8jD19_koHKLPOTbVMyEMN970dhGGRI&_LANG=enus "Document print icon.")

   The PDF of the supplement opens in a new browser window. In the new window, select
   the Print and follow printing instructions for your
   printer.
