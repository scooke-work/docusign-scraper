# View Comments and Replies Through the Notification Center

Learn how to use the Docusign Notification Center to view comments or
replies.

The Notification Center contains notices about pending tasks that
you complete, as well as message replies.

1. Log on to your Docusign account.

   If you have messages, the notification icon ![Docusign account notification icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnotifyicon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MTAsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.o0En8laSBkOKrXZ6s2bIkoENVeAFQmGwO3UQEtnNGhs&_LANG=enus "Docusign account notification icon."), it has a red indicator that you have notification messages.
2. Select the ![Docusign account notification icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnotifyicon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MTAsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.o0En8laSBkOKrXZ6s2bIkoENVeAFQmGwO3UQEtnNGhs&_LANG=enus "Docusign account notification icon.") to view the Notifications menu.

   ![A screenshot Actions Required in the Notification Menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnewactionsreq.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MTAsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.o0En8laSBkOKrXZ6s2bIkoENVeAFQmGwO3UQEtnNGhs&_LANG=enus "A screenshot Actions Required in the Notification Menu.")
3. Select Actions required to display your notifications.
4. In the Notification Center, select the checkbox next to the comment or reply you want
   to view.

   ![Screenshot of sample notifications for required actions.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signactionreq.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MTAsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.o0En8laSBkOKrXZ6s2bIkoENVeAFQmGwO3UQEtnNGhs&_LANG=enus "Screenshot of sample notifications for required actions.")

   Note: You can sort comments by selecting Filters,
   and entering search criteria.

   ![A screenshot of message filters with APPLY or RESET options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signinboxfilters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MTAsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.o0En8laSBkOKrXZ6s2bIkoENVeAFQmGwO3UQEtnNGhs&_LANG=enus "A screenshot of message filters with APPLY or RESET options.")
5. Selecting a comment in the notifications list opens the document and takes you to the
   comment or reply.

   ![A screenshot of a posted comment.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnewcommfinalresponse.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0MTAsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.o0En8laSBkOKrXZ6s2bIkoENVeAFQmGwO3UQEtnNGhs&_LANG=enus "A screenshot of a posted comment.")

   View comments and replies through email

   To view comments and replies in an envelope, select the REVIEW
   DOCUMENTS link in the email notification indicating a comment or reply was
   made.

   - Select the REVIEW DOCUMENTS link in a comment or reply
     notification. The envelope opens and displays the comment or reply.
   - Select the REVIEW DOCUMENTS link in the original email
     notification asking for your signature. The envelope opens and you can scroll to view
     comments.
