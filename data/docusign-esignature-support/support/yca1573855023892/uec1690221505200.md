# Report Abuse or Fraudulent Activity

Learn about the Docusign
Report Abuse feature that you can use to notify Docusign about instances of fraudulent activity.

The Report Abuse feature reports fraudulent emails or abuse
directly to Docusign.

1. Choose Other Actions to access the Report Abuse
   feature.
   - From the Review and complete page within the
     document you're signing, select the vertical ellipsis and select
     Report Abuse

     ![A screenshot of the Docusign Other Action menu Report Abuse menu option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnewselrptabuse.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0NjYsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.EjRA7UTebVcVK3HcOD1LA-wAml8WMTYNK53Hf2QdbgQ&_LANG=enus "A screenshot of the Docusign Other Action menu Report Abuse menu option.")
   - From the Initial page, select Other Actions > Report Abuse > Continue.

     Note: This option is only available if
     your sender implements this feature in this screen.

     ![A screenshot of the Docusign Other Options menu for an envelope you received.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signinitscreenselect.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0NjYsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.EjRA7UTebVcVK3HcOD1LA-wAml8WMTYNK53Hf2QdbgQ&_LANG=enus "A screenshot of the Docusign Other Options menu for an envelope you received.")
2. Select one of the following Report Abuse options:

   ![A screenshot of the Report Abuse options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnewreportabusereason.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0NjYsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.EjRA7UTebVcVK3HcOD1LA-wAml8WMTYNK53Hf2QdbgQ&_LANG=enus "A screenshot of the Report Abuse options.")
   - I know the sender and want them to stop contacting
     me.
   - I don't know the sender and think this is
     spam.
   - I believe this is fraudulent or contains illegal
     content.
3. Add details in the Comment section.
4. Optionally, select a follow-up action and select SEND ABUSE
   REPORT.

   The following example shows a completed fraud report.

   ![A screenshot of a completed Docusign envelope fraud report.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signlbrptabcompletedb.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0NjYsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.EjRA7UTebVcVK3HcOD1LA-wAml8WMTYNK53Hf2QdbgQ&_LANG=enus "A screenshot of a completed Docusign envelope fraud report.")
5. If you select the I believe this is fraudulent or contains illegal
   content option:
   1. Select Start to access the New
      Case page.

      ![A screenshot of the Docusign New Case page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/newcasereport.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0NjYsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.EjRA7UTebVcVK3HcOD1LA-wAml8WMTYNK53Hf2QdbgQ&_LANG=enus "A screenshot of the Docusign New Case page.")
   2. Select Accept and complete the relevant New Case
      Incident report fields.
6. Select Submit.

You receive a confirmation message that your report was sent to Docusign.![A screenshot of the report abuse confirmation message.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/reportconfirmation.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQ0NjYsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.EjRA7UTebVcVK3HcOD1LA-wAml8WMTYNK53Hf2QdbgQ&_LANG=enus "A screenshot of the report abuse confirmation message.")
