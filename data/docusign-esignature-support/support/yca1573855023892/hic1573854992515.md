# Add Information to Documents

Learn how to complete the required fields, optional fields, and comments to add
information to the documents you are signing.

Use these resources to add information to
documents when signing a document.
.

Control
Toolbar

In the toolbar on the right, there are controls for:

- Displaying the thumbnail view of document pages in the display
- Posting a comment or question about the document
- Generating a PDF of all the documents in the envelope to download them
- Generating a PDF to print the document
- Selecting the zoom options

Freeform fields

If the page displays a list of fields
on the left side of the page, the sender has sent you documents where you must place and
complete the fields. This is called freeform signing. For information on free-form signing,
see the [Use Freeform Signing](https://support.docusign.com/s/document-item?bundleId=yca1573855023892&topicId=anf1573854985078.html&_LANG=enus "Learn how to use Docusign Freeform signing using the menu on the left side of the signing page.")
article.

Signing indicators

Selecting or tapping the yellow
START action indicator takes you to the first field that requires
your input.

![A screenshot of the signing START action indicator.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnewstart.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQzNzcsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.K4cMN4xDgUMQgla-fSjonKJqlDGcGHzkso8ChfJSs6o&_LANG=enus "A screenshot of the signing START action indicator.")

The
NEXT action indicator moves through the document, showing where you
must provide information. You can place your cursor over a field to see any tips the sender
has provided.

![A screenshot of the signing NEXT action indicator.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnewnextreq.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQzNzcsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.K4cMN4xDgUMQgla-fSjonKJqlDGcGHzkso8ChfJSs6o&_LANG=enus "A screenshot of the signing NEXT action indicator.")

Senders
can set the order where fields are presented. Therefore, there might be times when the
action indicator jumps to a different part of the page. You can still scroll through the
pages to read them, but the system does not let you finish signing until you complete all
the required fields.

Note: if you want
to finish signing later, select the OTHER ACTIONS menu. You can also
use the Print & Sign option, or decline to sign. The available
options depend on sender settings. See the [Other Signing Actions](https://support.docusign.com/s/document-item?bundleId=yca1573855023892&topicId=qdy1573854987499.html&_LANG=enus "Learn about the available options for signers from the Other Actions menu.") article for
more information about these options.

Fields that are outlined in red are required.
Add entries to these fields to complete the signing process.

Note: Some fields, such as signing date fields,
are automatically completed based on information available to the system. You cannot change
this information.

Select options

- To select an option or check box, select or tap on the appropriate option or check box.
  Note that you can only select one option in a group.
- To select an option from a dropdown list, select the list and select the appropriate
  option.
- To add other information in a boxed field, type the requested information into the field
  and select NEXT or press Tab to
  continue.
- To sign or initial, select the SIGN or
  INITIAL field on the document.

  The first time you select a
  SIGN or INITIAL field, you are prompted
  to adopt your signature and initials. See the [Adopt Your Signature](https://support.docusign.com/s/document-item?bundleId=yca1573855023892&topicId=dga1573854990297.html&_LANG=enus "Learn how to choose a style for your signature and initial when you first sign a document with Docusign.") article for additional information.
- If there is an attachment field (a yellow field with a paperclip icon) on a document,
  the sender has requested that you provide supporting documentation.

  To do so, upload
  supporting documents during the signing process or fax it.
- When you reach an attachment field, you are prompted to select the attachment field to
  add the supporting document. See the [Add Attachments to a Document to Provide Supporting Information](https://support.docusign.com/s/document-item?bundleId=yca1573855023892&topicId=nbl1573854991156.html&_LANG=enus "Learn how to return requested documents to the sender as an attachment.") article for
  information about submitting the documents.

Complete your information

After you have added
information to all the required fields and any signatures and initials the
FINISH prompt displays:
