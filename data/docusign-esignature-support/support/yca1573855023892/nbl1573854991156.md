# Add Attachments to a Document to Provide Supporting Information

Learn how to return requested documents to the sender as an attachment.

If the sender enables the Add Attachment feature for their documents, they can request that
you add additional supporting information to the document. These are the options for adding
attachments:

Add multiple attachments per field
:   If enabled, you can add multiple attachments to a Signer
    Attachment field.

Append attachments to the document with the attachment field
:   If enabled, when you upload an attachment, the document is inserted at the end of the
    envelope. Additionally, with this option, the attachment becomes a permanent part of the
    envelope. You cannot remove the attachment after uploading it. Also, if the envelope is
    ever copied, any signer attachments are included in the new envelope.

These attachments are uploaded to the Docusign system and shared with the sender and any
other signers or viewers on the document.

Attached items are converted to a PDF and must be under
50
MB in size. For more information on supported attachments and file sizes, see the [Supported File Formats](https://support.docusign.com/s/document-item?bundleId=gbo1643332197980&topicId=xln1578456261162.html&_LANG=enus)
article.

Note: You cannot upload zipped or compressed files.

1. Select the Attachment icon.

2. Select UPLOAD A FILE and select a file to attach.

   ![Screenshot of Upload a File.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signingattachment2.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MjQzOTYsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.gQGfTN20u1DmALC_om_ON2ft4VMhCeqmPRabu7goTs0&_LANG=enus "Screenshot of Upload a File.")

   Note: You can attach a
   variety of file formats. For a full list, see the [Supported File Formats](https://support.docusign.com/s/document-item?bundleId=gbo1643332197980.ditamap&topicId=xln1578456261162.html&_LANG=enus) article.
3. Repeat for any additional files you want to attach.
4. When finished adding attachments, click DONE.

After you have uploaded the documents and completed all other tags, you can finish signing
as described in the [Complete the Signing Session](https://support.docusign.com/s/document-item?bundleId=yca1573855023892&topicId=abm1573854992359.html&_LANG=enus "Learn how to complete signing and optionally save or print a copy of your signed documents.") article.
