# How to enable Docusign Monitor for your organization

This topic walks you through the process of how you can enable Docusign Monitor for your organization.

## Step 1. Create a developer account

If you already have a developer account, move on to step 2.

To create a developer account:

1. At the top right of the Developer Center site, select **Create account**.
2. Enter your information, then use your provided email address to activate your new account.

## Step 2. Create an organization

To create an organization and gain access to the Monitor admin tile:

1. From the Docusign [developer environment](https://apps-d.docusign.com), select **Admin** at the top of the navigation bar.
2. In the admin tool, select **Get Started** on the **Docusign Admin is enabled for your company** banner.
3. If you do not already have an organization, choose the accounts to be linked to your new organization, then create the organization.
4. All demo accounts have access to the paid version of Monitor, and prod accounts might be entitled to the free version of Monitor. In Organization Admin, navigate to Features tab and toggle Docusign Monitor status to “Enabled.” If you’re having trouble accessing Monitor, reach out to your support manager or create a case in [Docusign Support Center](https://support.docusign.com/). Please note that the API for SIEM integrations is only available with the paid version of Monitor for prod accounts. Please contact sales if you’re interested in that.
