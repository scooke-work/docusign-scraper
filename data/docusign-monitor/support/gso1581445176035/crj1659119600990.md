# Docusign Monitor eSignature Events

Review the details for Docusign Monitor eSignature events.

Docusign Monitor tracks these eSignature events.

| Account user added | Member invited to an account Note: This event is available in Monitor Free Edition. |
| Account user bulk import | Administrator imported users through a CSV bulk import |
| Account user bulk update | Administrator updated users through a CSV bulk import |

| Account users downloaded | User completed a CSV download of an account's users |
