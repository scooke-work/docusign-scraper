# Monitor Agreement Manager Events

Review the details for the Docusign Monitor Agreement Manager events.

Docusign Monitor tracks these Agreement Manager events.

| Event | Description |
| --- | --- |
| Agreement access updated | User's access to an agreement has been updated |
| Agreement attributes updated | Agreement attributes have been updated |
| Agreement deleted | User deleted an agreement |
| Agreement downloaded | User downloaded an agreement |
| Agreement uploaded | User uploaded an agreement |
| Agreement set access updated | User's access to an agreement set has been updated |
| Agreement set created | Agreement set has been created |
| Agreement set deleted | Agreement set has been deleted |
| Agreement set updated | Agreements were added or removed from the agreement set. |
| Agreement set name updated | Agreement set name has been updated |
| AI extractions approved | User reviewed the extracted value and used the  Accept AI Suggestions option to confirm the value |
