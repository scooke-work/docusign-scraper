# Error Message: API Data Error and Lens Unauthorized Reference ID Errors

Troubleshoot an error that occurred when attempting to get API data or a Lens reference
ID in the Docusign Monitor Splunk dashboard.

This error occurs when attempting to get API data or a Lens reference ID in the Docusign Monitor Splunk dashboard. Lens is the Monitor API endpoint (where the programs
communicate with each other).

You receive this error
message:

```
Error occurred when getting API data or error getting lens data unauthorized
reference id.
```

The user does not have the required permissions to get API data or a Lens
reference ID.

## API Data error and Lens data unauthorized reference ID error

Responsible party: Docusign organization
administrator

In Docusign Administration, assign Administrator
or Security Reports Admin permissions to the
user.
