# Download Docusign Monitor Event Information to a CSV File

Take a snapshot of Monitor events by using the Download CSV feature to store Monitor
event details in a CSV file.

To take a snapshot of a Monitor event information, use
the Download CSV feature to store Monitor event details in a CSV file.

Note: This feature is not available in Monitor Free
Edition.

1. From your Docusign home page, select Admin > Organization > Monitor from the side menu.
2. Select an event by selecting the event's name to display the Event
   Details page.
3. Select Download CSV.
4. Name and download the CSV file that includes the information shown in this
   example:

   ![A screenshot of a sample Monitor events CSV file.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/mon-new-event-hx-csv.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc0MzcsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.hMt9RP1SlzOWhPzo18JD2oR0p6FONWnn4MMBFtio0uE&_LANG=enus "A screenshot of a sample Monitor events CSV file.")

   Note the following:

   - You cannot customize the column names for this release.
   - The CSV download has the following limits:
     - 50,000 total rows
     - 25,000 total unique users
     - Ascending sort order only
