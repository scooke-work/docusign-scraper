# Configure the Alerts Lookup Table

Configure the Monitor blocklist lookup tables to track alerts.

Configure alert severity using the Alerts Lookup
table. Docusign recommends using the [Splunk App
for Lookup File Editing](https://splunkbase.splunk.com/app/1724) to configure lookup tables.

1. Access the docusign\_alert\_lookup table using the Splunk
   lookup editor.
2. Set the severity and severity id using these values only.

   | severity | severity\_id |
   | --- | --- |
   | low | 1 |
   | medium | 2 |
   | high | 3 |
   | critical | 4 |

   Note: By default, all alerts are
   set to medium severity. Set the
   severity and severity\_id in the
   docusign\_alert\_lookup.csv based on your organization’s
   requirements.
