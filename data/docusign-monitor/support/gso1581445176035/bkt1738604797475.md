# Manage Alert Notifications

Learn how to add additional recipients or prevent specific Docusign
user from receiving alert notifications. Use this feature to reduce email traffic from users
who should not receive alert notifications.

Within each alert's settings, you can specify which organization or security report
admins receive Monitor alerts. If you do not specify any email recipients, all
organization administrators and Security Reports administrators receive alert
notifications by default.

You can also prevent specific users from triggering alert notifications by adding them to a safelist. For example, you can
prevent shared accounts from triggering the Simultaneous login alert.

See the [Monitor alerts](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=refd45c3b91-27da-49ee-aba6-f946c7e203da.html&_LANG=enus "Review descriptions and recommended actions for Docusign Monitor eSignature, Admin settings, and login and security alerts.") article for a list of alerts.

These customized settings are only available to organizations with the full version of
Monitor.

1. From the Docusign home page, select Admin > Organization > Monitor from the side menu.
2. Select Settings to display the alerts.
3. Select Edit next to the alert that you want to
   configure.
4. In the  Edit Alert dialog box, update the fields as
   described in the [Configure Alert Threshold
   Parameters](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=bzc1620336954182.html&_LANG=enus "Adjust the alert sensitivity or event thresholds that generate alerts for your deployment.") article.

   ![A screenshot of a sample Edit alert details dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/mon-new-man-al-settings..png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc0MTcsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.FKYxQWhQ_uwxKe8QHyurSl1Oyv8OwocjVRP6Gbr2gFI&_LANG=enus "A screenshot of a sample Edit alert details dialog box.")
5. Optionally select users to customize the alert notification settings:
   - Alert only these recipients to specify who gets
     the alert notifications.
   - Prevent users from triggering alert to prevent
     alert notifications when the users take action that may generate the
     alert.
6. Select Save.
