# Use Filters to Customize Monitor Displays

Review how Docusign Monitor filters let you customize Monitor displays to view the data
you need.

Filters let you change and customize Monitor events and alert data displays.

You can filter an alert tile display on the Monitor dashboard.

![A screenshot of sample display filters.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/mon-newest-filter-page.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc1NDMsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.frLSWDgt8Dja9cYyCPmlrrhFkqp1ATYLYer_Gm_7fz0&_LANG=enus "A screenshot of sample display filters.")

## Filter display information

You can filter the alert display by selecting from the drop-down list for:

- Alerts ID: Select one or more alerts to view the alert
  history. The default is All Alerts.
- Event ID: Select one or more events to view the alert
  history. The default is All Alerts.
- User ID: Select one or more events to view the alert
  history. The default is All Alerts.
- Alert History time range
  - Last 24 hours
  - Last 7 days (default)
  - Last 30 days
  - Last 60 days
  - Last 90 days
  - Last 6 months
  - Year to date
  - Last 365 days

## Alert tile display filters

You can customize the information on Monitor's configurable dashboard tiles that
display and track events.

![A screenshot of a sample Monitor display tile filter menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/mon-newest-tile-display-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc1NDMsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.frLSWDgt8Dja9cYyCPmlrrhFkqp1ATYLYer_Gm_7fz0&_LANG=enus "A screenshot of a sample Monitor display tile filter menu.")

Select the three-dot menu to modify the display:

- Edit: Display and change the event filter information.
- Refresh Data: Updates the display information.
- Remove from Dashboard: Removes the tile from the Monitor dashboard display.

## Filter event information

From the User Investigations > Event  > Event Details display, you can add or remove data from the display by selecting a
filter.

- Search users: Specify one or more users by username or
  email address.
- Use the Date range field to select the data time
  frame:
  - 7 days
  - 30 days
  - 90 days
  - Last 12 months
  - A custom date range
- Accounts field: Specify one or more accounts
- Customize Columns icon ![A screenshot of the Customize columns icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/mon-cust-col-icon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc1NDMsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.frLSWDgt8Dja9cYyCPmlrrhFkqp1ATYLYer_Gm_7fz0&_LANG=enus "A screenshot of the Customize columns icon.")): Add or remove display columns.

  ![A screenshot of a sample Monitor Customize columns dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/mon-cust-cols.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc1NDMsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.frLSWDgt8Dja9cYyCPmlrrhFkqp1ATYLYer_Gm_7fz0&_LANG=enus "A screenshot of a sample Monitor Customize columns dialog box.")

Select Clear all to remove your customizations.

Widget display options: Use these options to change the
display format by selecting the filter and entering the information:

- Show: Filter by which events display in the widget (the
  default is All).

  See the [Monitor Alerts](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=refd45c3b91-27da-49ee-aba6-f946c7e203da.html&_LANG=enus "Review descriptions and recommended actions for Docusign Monitor eSignature, Admin settings, and login and security alerts.") and the [Monitor
  Events](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=crj1659119600990.html&_LANG=enus "Review the details for Docusign Monitor eSignature events.")

  articles for a list and descriptions of alerts and
  events you can include.
- IP Addresses: Include the following IP Address
  information. See the [Configure Approved Docusign IP Addresses](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=mfq1645630427780.html&_LANG=enus "Learn how to create a list of trusted IP addresses with the Docusign Monitor IP Safe List feature.") article for
  information about configuring the IP Safe List.
  - All (default)
  - All IP Addresses in Safe List
  - All IP Addresses outside Safe List
- Grouped by: The information is filtered by the following
  event information:

  Note: This option may be disabled
  depending on the selected chart type.

  - Overall (default)
  - User
  - Account
  - IP Address
  - Event Name
  - Browser
  - Operating System
  - Device Type
  - Integrator key
- Displayed as: Widget's display format
  - Line Chart
  - Bar Chart
  - World Map
  - Table
