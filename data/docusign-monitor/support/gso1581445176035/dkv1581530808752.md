# Monitor Use Cases

Read about examples where Docusign Monitor tracks and alerts you
about suspicious activity.

Monitor tracks and notifies you about these types of use cases.

- An administrator removes a security policy from any user (such as the policy to enforce SSO
  login)

  Potential threat: Unauthorized access and data loss

  Action:
  Contact Docusign the administrator to verify the activity was intentional and
  warranted
- A user logs in from a location from which they haven’t logged on from in the last 90
  days

  Potential threat: Malicious outside hacker

  Action: Notify user, force logoff, change password
- A user tries to access or delete more than 50 envelopes in a day

  Potential threat:
  Unauthorized access or data loss

  Action: Contact the administrator to
  verify the activity was intentional and warranted
- Docusign REST API activity increases suddenly

  Potential threat: Compromised credentials
  leading to an unauthorized flow of data

  Action: Contact the administrator
  to verify the activity was intentional and warranted
