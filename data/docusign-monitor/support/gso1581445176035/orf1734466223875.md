# Docusign Monitor Events

Learn about the security events that Docusign Monitor
tracks.

Docusign

Transmit the event data to your Security information and event management (SIEM) to establish event thresholds as described in the [Integrate a SIEM System article](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=uxz1581533216990.html&_LANG=enus "Integrate your SIEM to receive security alerts directly from Monitor.").

- Dubious attempts to access accounts
- Unusual envelope activity patterns
- Authentication anomalies

Note:

- Monitor does not record login failure events for organizations using SSO.
- Monitor tracks login successes for either SSO and password logins.
