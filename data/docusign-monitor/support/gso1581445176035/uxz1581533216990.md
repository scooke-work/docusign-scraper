# Integrate a SIEM System

Integrate your SIEM to receive security alerts directly from Monitor.

With Monitor, you can directly deliver security
notifications to your security information and event management (SIEM) system. When you
integrate your SIEM (such as Splunk, Tableau, or Power BI) with Monitor, you can continue
to use familiar tools and create custom alerts.

Prerequisites

Depending on your SIEM, make sure you have:

- Integration key
- RSA key
- JWT token and access token as described in the [How to get an access token with JWT Grant authentication](https://developers.docusign.com/platform/auth/jwt/jwt-get-token/)
  page
