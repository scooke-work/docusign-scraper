# Monitor CLM Alerts

Review descriptions and recommended actions for Docusign Monitor CLM alerts.

CLM administrators can use Monitor to track document activity, user activity, and settings changes. See the [Track CLM Document Activities from Docusign Monitor](https://support.docusign.com/s/document-item?bundleId=alb1642965087302&topicId=dcd1687284122199.html&_LANG=enus) article for detailed information.

| Alert | Generated when: | Recommendations |
| --- | --- | --- |
| CLM documents deleted | A user deleted more than n  documents in 1 hour. The `n` is a configurable threshold. The default is 20. | Investigate user activity and contact the user if this behavior is unusual. |
| CLM documents downloaded | A user downloaded more than n  documents in 1 hour. The `n` is a configurable threshold. The default is 20. | Investigate user activity and contact the user if this behavior is unusual. |
| CLM security settings modified | CLM folder security or Idle session timeout settings were changed. | Verify that settings changes were intentional. If this was not authorized, close the user's membership, as described in the [Manage Docusign CLM Support user permissions](https://support.docusign.com/s/document-item?bundleId=pxt1643324456371&topicId=qfh1617310293710.html&_LANG=enus) article. |
