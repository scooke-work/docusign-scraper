# Review User Activity with User Investigations

Learn how to use the Docusign Monitor User Investigations feature for
easy review of recent user activity.

Use the Monitor User Investigations feature for easy review of recent user activity. You can
quickly investigate with:

- User Search
- Filters

By default, the display includes User, Account, IP Address, and Location.

Note:

The User Investigations tab replaces the Events tab.

1. From your Docusign home page, select Admin > Organization > Monitor from the side menu.
2. Select User Investigations to display user and alert
   information.

   Here is a sample User Investigation page.

   ![A screenshot of a sample Monitor User Investigations page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/mon-newest-u-invest.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc0MjAsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.zK68xZhRE6TpNSw_Vmus3_QMJeOSAIVSOPdIBiDk8Ag&_LANG=enus)
3. Customize your investigation with these features.

   | Option | Description |
   | --- | --- |
   | User | Specify one or more users by typing a name or email. Optionally select from the list. |
   | Event | Specify events. The default is all events. |
   | Account | Specify one or more accounts |
   | Date | Select a date range from the list. |
   | Customize columns | Select the Customize columns icon A screenshot of the Customize columns icon. and specify the columns to include in the display. |
   | Download CSV | Select to download the data you specified using filters. |
   | Expand the display | Use the expand arrow to display event details and access information. |

   See the Filter event information section in the [Use Filters to Customize Monitor
   Displays](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=ngm1706891120953.html&_LANG=enus "Review how Docusign Monitor filters let you customize Monitor displays to view the data you need.") article for more information.
