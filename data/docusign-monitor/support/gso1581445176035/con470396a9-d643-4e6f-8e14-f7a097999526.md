# Monitor and Splunk Integration Setup Steps

Identify and implement the required procedures for integrating Splunk with Docusign Monitor.

Note: Make sure you review the Splunk Enterprise 9.3+compatibility and the Docusign app support information described in the [Integrate
Docusign Monitor Security Notifications with Splunk](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=sfj1611351664499.html&_LANG=enus "Use the Docusign Splunk App and Docusign Add-on to directly deliver Docusign Monitor notifications to Splunk.")
article before completing the integration steps.

These are the required procedures for integrating Splunk with Docusign

1. [Install Docusign App and Docusign Add-on](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=rvv1611352188922.html&_LANG=enus "Learn how to install the Splunk Docusign App and Docusign Add-on.")
2. [Configure Monitor Data Search Macros](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=pya1611352337492.html&_LANG=enus "Use search macros to easily search and find Monitor data.")
3. [Configure Monitor Data Inputs](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=hvj1633980712678.html&_LANG=enus "Configure Monitor account, monitor, and user data inputs to display in your Splunk dashboard.")
4. [Create and Configure Monitor Lookup Tables](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=sei1611352399033.html&_LANG=enus "Use lookup tables to track Monitor account, users, alerts, and blocklists.")
5. [Configure and Enable Alerts](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=ubn1611352527444.html&_LANG=enus "As a Splunk App administrator, you can configure custom alerts.")
