# Select Alert Email Recipients

For each alert, specify which users receive Docusign Monitor alert email
notifications.

You can specify which users receive alert email notifications for each alert.

Note:

- For both Monitor Paid and Monitor Free editions, only oraganization administrators or security
  administrators can receive email notifications.

  If you do not specify any email
  recipients, organization administrators and Security Reports administrators
  will receive alert email notifications by default.
- If your organization has the Monitor Paid edition, you can restrict alerts to send only to specific email addresses entered here.

1. From your Docusign home page, select Admin > Organization > Monitor from the side menu.
2. Select Alert List.
3. Select the alert for which you want to customize email notifications.
4. Select Edit.
5. Select Alert Email Recipients.
6. Begin typing into display the first recipient name.
