# Monitor Free Edition Features

Learn about the Docusign Monitor Free Edition features.

Monitor Free Edition demonstrates the powers of Monitor by letting you work with
limited
features.

You need Docusign Organization Admin Basic (The free version
of Organization Management) to use this product.

This article summarizes Free Edition Monitor features.

## Dashboard

The Free Edition Monitor dashboard provides:

- A 30-day summary of Monitor Free Edition core metrics, including the number of:
  - Events
  - Alerts
  - Active users
  - Login events
  - Envelopes deleted
- Up to a 7-day summary in:
  - Alerts: Notify you when specific sets of events occur in your
    organization
  - Events: Specific activities in your organization
  - Envelope deletion activity: User deletes twenty or more envelopes in one hour,
    indicating possible unauthorized access or data loss
  - Log on activity: Successful or failed logon attempts
  - User tracking:
    - Changes in user information, such as email address, password, profiles,
      or recovery information
    - Activity such as account access, or account lockout
- Configurable dashboard widgets that display and track alert activity

Note:

With Monitor Free Edition, you cannot:

- Assign other admins to receive email notifications
- Add additional email recipients
- Add ignored users assigned to the paid version of Monitor to the Monitor Free
  version

## Alerts

Monitor Free Edition provides these alerts:

- Password changed from new location
- Recipient authentication disabled
- Unusual envelope deletion volume

See the [Docusign Monitors](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=refd45c3b91-27da-49ee-aba6-f946c7e203da.html&_LANG=enus "Review descriptions and recommended actions for Docusign Monitor eSignature, Admin settings, and login and security alerts.") article for detailed information about these
and other Monitor eSignature alerts.

## Events

Monitor Free Edition provides these events:

See the [Docusign Monitor Events](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=orf1734466223875.html&_LANG=enus "Learn about the security events that Docusign Monitor tracks.") for detailed information about these
and other Monitor eSignature events.

- Administrator added
- Delegated administrator added
- Envelope created
- Envelope deleted
- Envelope voided
- Login failure
- Login success
- Member added to organization
- Member invited to an account
- Member profile updated in Organization
- Password changed
- Password rule updated
