# Configure Monitor Data Search Macros

Use search macros to easily search and find Monitor data.

Set up a search macro for each Monitor data type.
All the scheduled searches and dashboards within the Docusign App and Docusign Add-on use these macros. Therefore, it is important to define the
proper index and source type in each macro.

1. From the App, select Settings  > Search > Advanced > Macros.

   ![A screenshot of sample Splunk search macros for Monitor.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/monitor_splunk_config_macros.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc0OTUsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.Vut0WZQJ-d7vCTTzNB-qjyd6w5PplqdRGlctRscDzNo&_LANG=enus)

   Note: Replace
   index=docusign with the index name for your data.
   This name is customer specific.
2. Enter these inputs for each macro.

   | Macro | Sourcetype | Example |
   | --- | --- | --- |
   | docusign\_account\_data | docusign:accounts | index=docusign sourcetype=docusign:accounts |
   | docusign\_monitor\_data | docusign:monitor | index=docusign sourcetype=docusign:monitor |
   | docusign\_user\_data | docusign:users | index=docusign sourcetype=docusign:users |
