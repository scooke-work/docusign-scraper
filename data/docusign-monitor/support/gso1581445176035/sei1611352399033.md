# Create and Configure Monitor Lookup Tables

Use lookup tables to track Monitor account, users, alerts, and blocklists.

Splunk lookups match field-value combinations in event
data with field-value combinations in external lookup tables. For Docusign Monitor, you
configure account, users, alerts and blocklist lookup tables.
