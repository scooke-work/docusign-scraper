# Use the Docusign Monitor API

Learn about using the Monitor API to send data directly to the application or website
you choose to integrate with Docusign.

Integrate the Docusign Monitor API to receive a real-time stream of event data for
your Docusign account.

The Docusign Monitor API operates on the same principles as the Docusign’s API stack,
so trust that your data remains secure, reliable and up to date.

For detailed information about the Monitor API, see the [Monitor
API Overview](https://developers.docusign.com/docs/monitor-api/) page.
