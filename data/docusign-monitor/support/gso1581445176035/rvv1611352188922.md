# Install Docusign App and Docusign Add-on

Learn how to install the Splunk Docusign App and Docusign Add-on.

Follow these steps to install the programs.

1. Download the [Docusign Monitor Add-on](https://splunkbase.splunk.com/app/5665) and [Docusign Monitor Dashboards and Alerts App](https://splunkbase.splunk.com/app/5666).
2. Install the Docusign App on the Search Heads.
3. Install the Docusign Add-on.

Note:

- Install on a Splunk Heavy Forwarder and Search Heads for the best results.
- For Splunk Cloud customers, install the Add-on on the IDM (Inputs Data Manager)
  instance if possible, and if the Splunk Cloud is available.

When installation is complete, access the Docusign Add-on for Splunk menu.
