# Docusign Monitor Versions Comparison

Compare the Docusign Monitor Free Edition introductory features to Monitor's extensive
features.

Docusign Monitor Free Edition provides a low-risk opportunity to experience the value and
functionality of Monitor. You can use Monitor Free Edition to determine how you might
use Monitor in your organization.

Important:
Docusign Monitor Free
Edition is a pilot program that is currently ongoing for existing customers. It is not
available for new Docusign Monitor customers.

Compare the features of the Free Edition of Docusign Monitor to Monitor.

Table 1.
Docusign Monitor Free Edition and Monitor Comparison

| Feature | Monitor Free Edition | Monitor |
| --- | --- | --- |
| Alerts | 3 | 24 |
| Events | 12 | 106 |
| Data display time frame | 30 days maximum | Since November 2020 |
| Dashboard widgets | - Login activity: Only world map display - Administrator and user settings: Only line chart display | Select from these display modes:  - Bar chart - Line chart - Table - World map |
| Advanced features | No access | - [Create custom   widgets](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=psb1620335565782.html&_LANG=enus "Configure Docusign Monitor deployment alert sensitivity or event thresholds to customize when and how Monitor generates your organization alerts.") - [Download alert   history](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=OLDorv1594064852688.html&_LANG=enus "View and customize detailed Monitor alert information using the Alert History filters and options.") - [Download event   history](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=fap1613668711861.html&_LANG=enus "Take a snapshot of Monitor events by using the Download CSV feature to store Monitor event details in a CSV file.") |
| API | No access | [Monitor API](https://developers.docusign.com/docs/monitor-api/) |
| SIEM integration | None | [Splunk](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=sfj1611351664499.html&_LANG=enus "Use the Docusign Splunk App and Docusign Add-on to directly deliver Docusign Monitor notifications to Splunk.") |

Note: Both Monitor versions require a claimed domain for your organization.
For more information, see the [Claim a Domain](https://support.docusign.com/s/document-item?bundleId=rrf1583359212854&topicId=gso1583359141256.html&_LANG=enus)
article.
