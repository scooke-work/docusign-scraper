# Monitor Workflow Builder Alerts

Review descriptions and recommended actions for Docusign Monitor Workflow Builder alerts.

Administrators can use Monitor to track these activity alerts. For more information about Workflow Builder , see the [Creating workflows](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=tco1697826811758.html&_LANG=enus) article.

| Alert | Generated when: | Recommendations |
| --- | --- | --- |
| Unusual workflow deletion volume | A user deleted more than n  workflows in one hour. The `n` is a configurable threshold. The default is 5. | Investigate user activity and contact the user if this behavior is unusual. |
| Unusual workflow unpublished volume | A user unpublished more than n  workflows in one hour. The `n` is a configurable threshold. The default is 5. | Investigate user activity and contact the user if this behavior is unusual. |
