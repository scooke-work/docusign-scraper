# Configure Monitor Free Edition Alerts

Learn how to enable or disable the Monitor Free Edition alerts.

Use the Alerts feature to enable or disable
alerts.

Alerts are disabled by default. This prevents unexpected alert activity before a you have configured their monitoring preferences.

1. Select Admin > Organization > Monitor from the side menu.
2. Select Settings to view the Alerts list.

   ![A screenshot showing the alerts in the Docusign Monitor Free Edition Settings tab.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/mon-new-free-settings.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTczOTksInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.mRjssZsNfjF4ttkBSnfGHXXWFS3I86v7zGYUhvoagS4&_LANG=enus "A screenshot showing the alerts in the Docusign Monitor Free Edition Settings tab.")
3. Use the Enabled slider to enable or disable alerts.
