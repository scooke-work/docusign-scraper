# Use Docusign Monitor Free Edition

Learn about how to access and start using Docusign Monitor Free Edition.

Docusign Monitor Free Edition is available for all Docusign organizations in Docusign Organization Admin.
This only applies to organizations that aren't already paying for Monitor.

ISV and Resellers are excluded.

1. Access the Docusign Admin dashboard for your account.
2. Select Monitor.
3. Complete the Monitor Free Edition Onboarding Wizard.
4. Select Get Started to access the Monitor dashboard where you
   can:
   - Use the User Investigations tab to review alert activity
     over the last seven days.
   - Use the Alert History tab to view historical alert
     information
   - Use the IP Safe List  tab to create a list of trusted IP
     addresses

With Monitor Free Edition, you work with limited Monitor features. At any time, you can buy
Monitor to take advantage of all functionality by contacting [Docusign sales](https://www.docusign.com/contact-sales) to upgrade.
