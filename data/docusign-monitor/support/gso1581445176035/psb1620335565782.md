# Adjust Docusign Monitor Alert Sensitivity and Thresholds

Configure Docusign Monitor deployment alert sensitivity or event thresholds to
customize when and how Monitor generates your organization alerts.

You can adjust Monitor alert sensitivity or event thresholds. You can generate alerts based on
your:

- Organization size
- Activity

Here are some examples of adjusting alert sensitivity options.

| Item | Description | Example |
| --- | --- | --- |
| Maximum allowable events | Specify a new threshold for generating an alert. Set the maximum number of documents that is considered to be anomalous during one hour. | Unusual envelope download volume alert |
| Distance Threshold | Specify the minimum separation distance in KM for inclusion. Set the minimum number ( KM) that generates the alert over the last hour. For example, when a user logs in from a location that is more than 100 km from any other location where they logged on. | Simultaneous login alert |

See [eSignature Alerts](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=crj1659119600990.html&_LANG=enus "Review the details for Docusign Monitor eSignature events.") and [CLM Alerts](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=refccb593b2-ea43-40d9-93bd-5655e2162e5c.html&_LANG=enus "Review the details for the Docusign Monitor CLM events.") for detailed
information about alert thresholds you can change.

## Alert severity

The default severity level for all alerts is set to Auto. You can
change an alert's severity level to High,
Moderate, or Low depending on your
business needs as described in the [Change an Alert's Default Risk Status](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=lop1659713225796.html&_LANG=enus "Learn about changing Docusign Monitor alerts' default risk status to refine the number and frequency of the alerts you receive.") article.
