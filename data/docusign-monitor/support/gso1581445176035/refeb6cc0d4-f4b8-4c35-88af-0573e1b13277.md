# Monitor Agreement Manager Alerts

Review descriptions and recommended actions for Docusign Monitor Agreement Manager alerts.

Administrators can use Monitor to track Agreement Manager activities. For information about Agreement Manager see the [Agreement Manager Overview and Onboarding](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=adf1702945446135.html&_LANG=enus) article.

| Alert | Generated when: | Recommendations |
| --- | --- | --- |
| Unusual agreement delete volume | A user deleted more than n completed agreements in one hour. The `n` is a configurable threshold. The default is 20. | Investigate user activity and contact the user if this behavior is unusual. |
| Unusual agreement download volume | A user downloaded more than n  completed agreements in one hour. The `n` is a configurable threshold. The default is 20. | Investigate user activity and contact the user if this behavior is unusual. |
