# Set Up Notifications in Docusign Monitor Settings

View, enable, or disable Docusign Monitor security alerts for your
deployment .

Use this feature to view, turn on, and turn off
event alerts.

Use Monitor Settings to select alerts that provide notifications about
activities that may indicate security risks, such as:

- Substantial amounts of envelope downloads or deletions
- Too many logons
- Attempts from one user to log on to Docusign from two locations

1. From your Docusign home page, select Admin > Organization > Monitor from the side menu.
2. Select Settings.
3. Select the menu arrow in an Alert category to display the associated alerts.

   ![A screenshot of the Monitor Settings page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/mon-new-settings-pg.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc0MTQsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.ElUVZbmjcMByuN47OCuiZQ-69NX8w_56PIlKbbAcv8Q&_LANG=enus "A screenshot of the Monitor Settings page.")
4. Use the Enabled slider to enable or disable events.
5. Optionally select Edit to configure alert thresholds and
   notification recipients as described in the [Configure Alert Threshold Parameters](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=bzc1620336954182.html&_LANG=enus "Adjust the alert sensitivity or event thresholds that generate alerts for your deployment.") article.
