# Use Monitor Alert History

Learn how to use the Docusign Monitor Alert History to review your
organization’s alert
history.
Additionally, you can take investigative action based on the alert
activity..

Use Alert History to quickly review Monitor alerts over a chosen time frame. You can see
who was involved, and decide what to do next without leaving the page. With the Alert
History page, you can:

- **See investigation details at a glance**

  Shows key information, such as alert name, user, account, severity, and tailored
  recommendations. You no longer must open each alert to get basic context.
- **Take action directly from the alert list**

  The overflow (three-dot) menu on each alert row has options that let you take action
  without leaving Monitor.
- **Filter and export the data that you need**

  You can filter alerts by date range, alert type, severity, and account. You can
  download a CSV snapshot of the results for audits, reporting, or offline
  analysis.
- **Stay focused on the riskiest activity**

  Severity and recommendations help you quickly distinguish routine activity from
  potential threats. You can prioritize investigation time where it matters
  most.
- **Improve future alert quality**

  The **Report false positive** option lets you flag alerts that represent
  acceptable behavior to helps us improve Monitor’s alert accuracy over time.

Here's how to access Alert History.

1. From your Docusign home page, select Admin > Organization > Monitor from the side menu.
2. Select Alert History to display alerts.

   Tip: You can refine the alert
   display details by:
   - Selecting the date range
   - Specifying one or more alerts

## Download Alert History to a CSV File

To take a snapshot of Monitor alert information, use the Download Alert History
feature to store the details in a CSV file.

Note:

- When you select this option, Monitor creates a `.zip`  that
  includes a CSV for the specified alerts and a CSV for events.
- Downloading a CSV file is not available in Monitor Free Edition.

1. Select Alert History to display alerts.
2. Select the criteria for the information download.

   - Time frame
   - One, multiple, or all alerts
3. Select Download Alert History.
