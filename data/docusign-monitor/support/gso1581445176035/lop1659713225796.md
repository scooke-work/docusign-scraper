# Change an Alert's Default Risk Status

Learn about changing Docusign Monitor alerts' default risk status to refine the number
and frequency of the alerts you receive.

Based on your experience as an administrator, you can change the alert risk status
assigned to Monitor alerts in the Alert List tab.

The monitor includes these alert severity assignments:

- Auto: System default for when to take action
- High: Take action within a few hours
- Medium: Take action within one day
- Low: Action may or may not be required

To set alert severity assignments:

1. From your Docusign home page, select Admin > Organization > Monitor from the side menu.
2. Select Settings  to display alerts.
3. For the alert you want to change, select Edit.
4. In the Set Default Risk Level dialog box, select a default
   risk level.

   ![A screenshot of a sample Monitor Risk Threshold dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/mon-env-del-thresh-db.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc0MzQsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.OnqRo-FzoE2a1Qi3hfhGdGkf1EABLRQK75Yd2b8o7BE&_LANG=enus "A screenshot of a sample Monitor Risk Threshold dialog box.")
5. Enter a number for the Event Threshold. In this example,
   you're setting the maximum number of envelope deletions before generating the
   alert.
6. Optionally assign the alert's email notifications by entering the responsible
   administrator's email address.
