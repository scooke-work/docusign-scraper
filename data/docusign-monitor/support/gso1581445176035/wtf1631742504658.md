# Error Message: Error Occurred Getting Lens Data in a Demo Environment

Troubleshoot an error that occurred getting Lens data for Docusign Monitor Splunk
dashboard in a demo environment.

This error occurs when obtaining Lens data for Docusign Monitor Splunk dashboard in a demo
environment.

You receive this error message:

```
Error occurred getting Lens data.
```

An organization
administrator modified the `authServer` and
`apiURLs` in the Docusign User data input,
Docusign Account data input, or the Docusign Monitor data input in a demo environment.

## Reset `authServer` or `apiURL` data inputs

Responsible party: Docusign
organization
administrator

Reset `authServer` and the `apiURL` to the
default values.
