# Monitor API Authentication

Learn about how the Monitor API authenticates calls through OAuth 2.0 authentication and
a JWT grant.

The Monitor API authenticates all calls using a variant of the JSON Web Token
(JWT) grant. Customer service integrations can exchange a JWT containing organization
and scope data for an access token that authenticates calls to the API.

See the [Authentication](https://developers.docusign.com/docs/monitor-api/monitor101/auth/) article for information about
authenticating this API.
