# Use the Monitor Free IP Safe List

Learn about how to use the Configure IP
Safe List feature to create a list of trusted IP addresses in Docusign Monitor Free Edition.

As a security administrator, use the Configure IP
Safe List feature to create a list of trusted IP addresses.

Follow the instructions in the [Configure Approved Docusign IP Addresses](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=mfq1645630427780.html&_LANG=enus "Learn how to create a list of trusted IP addresses with the Docusign Monitor IP Safe List feature.")  article
to add IP addresses in the Monitor Safe List to generate an IP
address outside allowed ranges alert. If you do not
configure any IP Address ranges, the alert does not generate.
