# Customize the Monitor Dashboard Display Tables

Customize which parameter columns display in a table and the order where table columns
display in Docusign Monitor dashboard table views.

Customize which parameter columns display in a table and the order where table columns
display in Monitor dashboard table views.

Note the following:

- You can specify sort order in any table-view dashboard event widget.
- Due to implementation limitations, sorting by Event Name, Device, Browser, OS do not
  always yield results in alphabetical order. There may be slight discrepancies in sort
  order of results, but like-values are grouped together.
- Rows are sortable one column at a time.
- Columns
  - All columns are sortable except for User/Account (assigned post-query).
  - Sortable columns have double arrows visible in the header.

1. From your Docusign home page, select Admin > Organization > Monitor from the side menu.
2. Select the table display for a widget.
3. Select the Gear icon on the right of the table header to open the selection
   modal.

   ![A screenshot of Monitor Alert display table modification gear icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/monitorconfiggear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc1NTMsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.Tl2lsmgGO5w1uE7ER9yajHgra7jL7j_MiQS79IL7V80&_LANG=enus "A screenshot of Monitor Alert display table modification gear icon.")
4. Select the columns that you want to display in the alert display.

   ![A dialog box for Docusign Monitor table column configuration.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/monnewseldisplaycolumns.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc1NTMsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.Tl2lsmgGO5w1uE7ER9yajHgra7jL7j_MiQS79IL7V80&_LANG=enus "A dialog box for Docusign Monitor table column configuration.")
5. To reorder the columns, select the dots icon to the left of the column name. Drag
   the column item up or down in the list.
6. Select SAVE.

   If you saved the display as a dashboard widget, the selections persist in the
   widget.

   Note: You can create up to 10 custom dashboard
   widgets.
