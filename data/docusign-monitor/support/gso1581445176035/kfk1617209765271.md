# Use the Splunk App Dashboard with Docusign Monitor

Use the Splunk App dashboard to view Docusign Monitor security metrics.

Splunk App dashboards contain panels that display data visualizations such as charts, tables,
event lists, and maps for your Docusign
eSignature
deployment events.

To access Monitor dashboards, from Splunk Enterprise select Apps > Docusign App for Splunk. The Docusign Monitor dashboard displays.

## Overview dashboards

Overview dashboards provide a synopsis of your threat posture for:

- Monitor overall activity.
- Alert activity.
- User activity.
- Block list activity: IP addresses or IP address ranges for which activity is
  blocked. For example, displays an alert if a user logs in from the specified IP or IP
  range.

## Investigator dashboards

Once you review the overview dashboards, you may see trends or activity that you want to
examine further.

Use the investigator dashboards to refine and capture data using preset filter
values.

For example, you noticed an increase in alerts for unusually high document downloads.
Using the Docusign Alert Investigator filters, you refine your data review by
selecting:

- 24-hour time range
- All accounts
- Unusual document download alert
- All severity levels
- All users

Note: Searching without adding filters may result in
performance issues.

## Additional data review

Use these additional features to view detailed Monitor information.

Docusign Users

Investigate Docusign user events, and drill down to view specific user
event details.

Note: View Monitor user events that are set up through data
inputs. When setting up data inputs in Monitor, select Add
Users. If users are members of more than one account, you must
create a data input for each account. See the [Configure Monitor Data Inputs](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=hvj1633980712678.html&_LANG=enus "Configure Monitor account, monitor, and user data inputs to display in your Splunk dashboard.")
article for details.
