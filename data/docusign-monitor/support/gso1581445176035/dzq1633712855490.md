# Customize the Alert Date Range

Specify a custom date range for an alert in Docusign Monitor's Alert History using the
Date Picker.

If you administer a large deployment, or if you want to examine a specific time period, use
Date Picker to select a custom date range.

1. From your Docusign home page, select Admin > Organization > Monitor from the side menu.
2. Select Alert History to display alerts.
3. Select an alert.
4. Select View Alert Details.

   Tip: You can also
   view alert details when you select the name of an existing event tile.
5. Select the dropdown Date Picker.
6. Select Custom Date Range.
7. Select the Calendar Icon.
8. Select the date range that you want as shown in this example.
