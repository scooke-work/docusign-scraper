# Error Message: Error Occurred Getting Access Token

Troubleshoot an error that occurred getting access token for Docusign Monitor Splunk
dashboard.

This error occurs when getting an access token for Docusign Monitor Splunk
dashboard.

You receive this error message:

```
Error occurred getting access token. Please check configuration values.
```

One of these events occurred:

- The user does not have the correct authorization scope or was not granted consent to the
  scope for the configured user ID.
- You added a manual input to create a callback URL in the settings.

## Authentication scope

Responsible party: Docusign organization
administrator

1. In Docusign Admin, assign Administrator or
   Security Reports Administrator permissions to
   the user.
2. Make sure the scopes that you request in the body of the JWT include
   `signature` and `impersonation` and
    `organization_read`  for Splunk.
