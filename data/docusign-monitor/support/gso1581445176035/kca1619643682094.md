# Monitor Dashboard Features

Read about the security monitoring features available in the Docusign Monitor dashboard. Using these features, you can customize the alerts for events you want to monitor and choose how they display the information.

With the Monitor dashboard, you can view and collect data about your organization's events. You can customize the type of event alerts to track and set up the alert data display. Specifically, you can:

- View events graphical overview and details with the Dashboard
- Review recent user activity with User Investigations
- View alert details and historical information with Alert History
- View and configure trusted IP addresses with the IP Safelist
- Enable or disable alerts with Settings

For example, use the Monitor dashboard to view these types of events:

- Mass envelope deletion: Checks for mass deletions of agreements
- Recipient authorization removal for viewing envelopes: This can indicate questionable activity or user error
- Multiple user logon failures: This can indicate user error or questionable activity

Additionally you can create a filter to view the alert interval for the alert data:

- 24 hours
- 7 days
- 30 days
- 60 days
- 90 days
- Year to date
- 365 days
- Custom time range

Note: For Monitor activity tracking:

- Monitor only tracks user activity if the users have an email address that is in a domain that your organization has claimed.
- If the account's Prevent Unmanaged Signups setting is enabled, Docusign Admin does not permit adding users to accounts if they are not members of a claimed domain. Additionally, Docusign Monitor does not track these user activities. For detailed information, see the [Administrator Checklist Before Enforcing SSO](https://support.docusign.com/s/document-item?bundleId=rrf1583359212854&topicId=azf1636145324908.html&_LANG=enus) article.
- Monitor captures activities such as envelope sends or document downloads that are associated with the account regardless of user status.
