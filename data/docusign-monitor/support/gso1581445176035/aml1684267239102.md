# View Monitor Free Edition Alert History

Learn how to view the Docusign Monitor Free Edition alert activity for up to 7 days from
the dashboard.

Use Alert History to view the alert activity data.

1. Select Admin > Organization > Monitor from the side menu.
2. Select Alert History.
3. Select an activity date range for the Alert History display.

   The data date range display choices are 24 hours or
   7 days.
4. View alert details and recommended actions as described in the [View Alert Details with Monitor Alert History](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=OLDorv1594064852688.html&_LANG=enus "View and customize detailed Monitor alert information using the Alert History filters and options.") article.
