# Configure Alert Threshold Parameters

Adjust the alert sensitivity or event thresholds that generate alerts for your
deployment.

Configure alert parameters to suit your deployment activities.

This example shows how to configure the Unusual Envelope Download
Volume alert. For this alert, you can configure
Event Threshold: Sets a new threshold
for generating an alert. For example, setting this value to 15 indicates
that if a user downloads 15 or more unique envelopes in an hour, it's
considered anomalous. It meets the first criterion for alert generation.

1. From your Docusign home page, select Admin > Organization > Monitor from the side menu.
2. Select Settings.

   A list of alerts displays.

   ![A screenshot of the Docusign Monitor alerts in the Settings page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/mon-newest-settings-pg.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc0MzIsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.YBiHnoz3lv1QyU6e_0RLDIIdB1j9DXwgtqao6f5VBfs&_LANG=enus "A screenshot of the Docusign Monitor alerts in the Settings page.")
3. Select Edit to display the alert's configurable
   fields.
4. Configure the alert settings and select Save.
   - Set Default Risk Level: Select the risk level for when Monitor generates
     an alert.
   - Alert only these recipients: Select recipients to
     receive the alert report
   - Prevent users from triggering alert: Select users that are blocked from
     generating the alert.
