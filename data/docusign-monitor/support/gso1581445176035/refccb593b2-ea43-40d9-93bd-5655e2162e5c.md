# Monitor CLM Events

Review the details for the Docusign Monitor CLM events.

Docusign Monitor tracks these CLM events.

| Event | Description |
| --- | --- |
| Document deleted | User deleted a CLM document |
| Document downloaded | User downloaded a CLM document |
| Document external review canceled | User canceled a CLM document review |
| Document external review complete | CLM document review was completed |
| Document external review sent | User sent a CLM document for review |
| Document generation form updated | User updated a CLM document generation form |
| Document new version | User created a new version of a CLM document |
| Folder security modified | User modified a CLM folder's security |
| Security idle session timeout updated | User updates the CLM account security session timeout |
