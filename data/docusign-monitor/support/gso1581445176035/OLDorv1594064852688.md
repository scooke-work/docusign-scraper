# View Alert Details with Monitor Alert History

View and customize detailed Monitor alert information using the Alert History filters and
options.

Use this feature to view the Alert History and detailed information for each alert.
Additionally, you can take investigative action based on the alert activity.

For example, if a user has multiple login failures, and you suspect suspicious activity,
you can: contact the user or lock that user's access to prevent further actions.

1. From your Docusign home page, select Admin > Organization > Monitor from the side menu.
2. Select Alert History to display alerts.

   Tip: You can refine the alert
   display details by:
   - Selecting the date range
   - Specifying one or more alerts

## Download Alert History to a CSV File

To take a snapshot of Monitor alert information, use the Download Alert History
feature to store the details in a CSV file.

Note:

- When you select this option, Monitor creates a `.zip`  that
  includes a CSV for the specified alerts and a CSV for events.
- Downloading a CSV file is not available in Monitor Free Edition.

1. Select Alert History to display alerts.
2. Select the criteria for the information download.

   - Time frame
   - One, multiple, or all alerts
3. Select Download Alert History.
