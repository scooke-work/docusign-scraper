# Error Message: Could Not Deserialize Key Data Error

Troubleshoot an error that occurred when the system could not deserialize key data for
Docusign Monitor Splunk integration data inputs.

This error occurs when the system cannot deserialize key data for Docusign Monitor
Splunk data inputs.

You receive this error message:

```
Could not deserialize key data.
```

You entered an incomplete RSA key when configuring Monitor data inputs.

## Incomplete RSA key Entry

1. Make sure you copy or enter the entire RSA key, including the
   ---Begin--- and
   –-End--- portions.
2. See these resources:

   - [Apps and keys](https://support.docusign.com/guides/ndse-admin-guide-api-and-keys)
   - [How to get an access token with JWT Grant
     authentication](https://developers.docusign.com/platform/auth/jwt/jwt-get-token/)
