# Create Account and User List Lookup Tables

Configure the Monitor account and user lookup tables to track these activities in
Splunk.

Create and configure accounts and user list lookup tables.

1. From the App, select Settings > Searches, reports and alerts >  Docusign – Generate Account Lookup.

   Running this search generates
   docusign\_account\_list.csv.
2. Enable the Docusign - Generate Account List Lookup
   search so it is automatically refreshed daily.
3. Run the Docusign – Generate Users Lookup search to generate
   the docusign\_users\_list.csv.
4. Enable the Docusign – Generate Users List Lookup search
   so it is automatically refreshed daily.

   Note the following:

   - The searches are dependent on data collected for the docusign\_account
     and docusign\_users sourcetypes. A sourcetype is a field that identifies
     the data structure of an event. The indexer identifies and adds the
     source type field when it indexes the data. If this data has never been
     indexed, Splunk does not create a lookup table.
   - The searches run once overnight by default. You can set a frequency that is
     specific to your organization.
