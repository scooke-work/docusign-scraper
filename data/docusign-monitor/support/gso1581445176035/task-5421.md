# Configure Monitor Free Events through User Investigations

Learn about how to view and examine Docusign Monitor Free Edition events in the User Investigations tab.

Use the User Investigations feature to view and configure Monitor Free Edition events.

1. Select Admin > Organization > Monitor from the side menu.
2. Select User Investigations.
3. Select an event from the list to display the Event
   Details.

   ![A screenshot of Docusign Monitor Free EditionUser Investigations Events detail.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/mon-free-new-u-i-pg.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc0MDQsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.EySkpnLX-GsJ2N7BGVjbE1spU7dSlaRTbhjg5H6YdG4&_LANG=enus "A screenshot of Docusign Monitor Free EditionUser Investigations Events detail.")
4. Optionally use filters refine your display.
   - Use the Search users box to specify one or more
     users.
   - Use the Date field to select the data d frame:
     - 7 days
     - Custom date range
   - Use the Customize Columns icon (![A screenshot of the Customize columns icon.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/mon-cust-col-icon.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc0MDQsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.EySkpnLX-GsJ2N7BGVjbE1spU7dSlaRTbhjg5H6YdG4&_LANG=enus "A screenshot of the Customize columns icon.")) to add or remove display columns.

     ![A screenshot of a sample Monitor Customize columns dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/mon-cust-cols.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc0MDQsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.EySkpnLX-GsJ2N7BGVjbE1spU7dSlaRTbhjg5H6YdG4&_LANG=enus "A screenshot of a sample Monitor Customize columns dialog box.")

     The features you select display in the Filters menu.

   Select Clear all to remove your customizations.
