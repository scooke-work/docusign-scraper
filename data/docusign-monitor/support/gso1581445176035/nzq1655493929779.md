# Monitor Data Streaming

Learn about calling the Monitor streaming endpoint to download data to your SIEM or
database.

Once you have successfully authenticated calls as described in the [Monitor API Authentication](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=qxp1582325276688.html&_LANG=enus "Learn about how the Monitor API authenticates calls through OAuth 2.0 authentication and a JWT grant.") article, you can call the Monitor streaming endpoint to
begin downloading data into a SIEM or database of your choice.

See the [How to get monitoring data](https://developers.docusign.com/docs/monitor-api/how-to/get-monitoring-data/) page for information about
downloading data.
