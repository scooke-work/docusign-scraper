# Access Monitor from Docusign Admin

Learn how to access Docusign from Docusign
Admin.

As a Docusign organization administrator, use Docusign Admin to enable and manage Monitor, as described in the [Introduction to Docusign
Admin](https://support.docusign.com/s/document-item?bundleId=rrf1583359212854&topicId=oum1583359129731.html&_LANG=enus) article.
Make sure that your organization has a license for Monitor. Contact your account team to
confirm.

Only users with Docusign Administrator or Security Reports Administrator permissions can access and manage Monitor. For more information, see the [Docusign Administrator Permissions](https://support.docusign.com/s/document-item?bundleId=rrf1583359212854&topicId=dpo1583359153647.html&_LANG=enus) article.

To enable the paid version of Docusign Monitor from Docusign Admin:

1. From your Docusign home page, select Admin > Organization > Monitor from the side menu.
2. Select Actions > ENABLE.

   The  Monitor Enable Organization Feature dialog box displays. ![A screenshot of the Enable Docusign Monitor feature dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/monnewaccessmonitor.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTczODIsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.3xovWkPDvULTJQooKsspFtWYbLsWNI5PlVrLhXYVlns&_LANG=enus "A screenshot of the Enable Docusign Monitor feature dialog box.")

   Contact your account team for assistance if:

   - You do not see the option to enable or disable Docusign Monitor.
   - The message `Not Available` displays.
3. Select ENABLE.

   A confirmation message displays.

   ![A screenshot of the Docusign Monitor successfully enabled confirmation message.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/monitorenableconfirm.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTczODIsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.3xovWkPDvULTJQooKsspFtWYbLsWNI5PlVrLhXYVlns&_LANG=enus "A screenshot of the Docusign Monitor successfully enabled confirmation message.")

You have activated Monitor. Access the feature in the Docusign Admin dashboard's Monitor section to view and manage
organization alerts.

Note:

If you're using Docusign Monitor Free Edition, it's automatically enabled and you do not need a license. Access the feature in the Docusign Admin dashboard's Monitor section.
