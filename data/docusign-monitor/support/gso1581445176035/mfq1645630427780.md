# Configure Approved Docusign IP Addresses

Learn how to create a list of trusted IP addresses with the Docusign
Monitor IP Safe List feature.

As a security administrator, use the Configure IP Safe
List feature to create a list of trusted IP addresses.

Note the following:

- This feature only applies to the  IP address outside accepted ranges
  alert.
- Enable this alert in the Monitor Settings page as described in the [Set Up Notifications in Docusign Monitor Settings](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=ovg1582324205227.html&_LANG=enus "View, enable, or disable Docusign Monitor security alerts for your deployment .") article.
- Add IP addresses in the Monitor Safe List to generate an IP address
  outside allowed ranges alert. If you do not configure any IP
  Address ranges, the alert does not generate.

Note: Users should access their accounts through controlled IP
addresses. Avoid adding individual IP addresses.

1. From your Docusign home page, select Admin > Organization > Monitor from the side menu.
2. Select IP Safe List to display the IP Address Safe List
   window.
3. Select Add to Safe List. Any configured acceptable IP
   addresses display.

   ![A screenshot of the Monitor IP Safe List dashboard with approved IP addresses.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/mon-new-ip-safe-list-pg.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc0NDAsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.dAhuxppxOiyf_yCQV2QUfV3xCzuukGrMW47NjTZanr8&_LANG=enus "A screenshot of the Monitor IP Safe List dashboard with approved IP addresses.")

   If there are no approved IP addresses, the IP Safe List display is empty,
   with the option to add IP addresses.

   ![A screenshot of the Add to Safe List message.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/monnewaddsafelistdb.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc0NDAsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.dAhuxppxOiyf_yCQV2QUfV3xCzuukGrMW47NjTZanr8&_LANG=enus "A screenshot of the Add to Safe List message.")
4. Select  Add to Safe List to create new acceptable IP
   addresses.
   1. In the Add IP address dialog box, enter an IP
      address.

      ![A screenshot of a dialog box to add approved Monitor IP addresses.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/monnewaddtosafe.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc0NDAsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.dAhuxppxOiyf_yCQV2QUfV3xCzuukGrMW47NjTZanr8&_LANG=enus "A screenshot of a dialog box to add approved Monitor IP addresses.")
   2. Optionally enter the last IP address in the range that you want to
      include.

      Tip: You can enter a Classless
      Inter-Domain Routing (CIDR) address in this field.

      CIDR notation
      is shorthand for the subnet mask. The CIDR address looks like a
      normal IP address, except that it ends with a slash followed by
      a number. The number after the slash represents the number of
      addresses in the range. For example, enter the CIDR address
      128.0.0.0/24.

      ![A screenshot showing entering approved Monitor IP addresses using a CIDR IP address range.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/monnewaddiprange.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc0NDAsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.dAhuxppxOiyf_yCQV2QUfV3xCzuukGrMW47NjTZanr8&_LANG=enus "A screenshot showing entering approved Monitor IP addresses using a CIDR IP address range.")

      Note: When entering the CIDR address, there may be
      a slight delay before the Last IP Address field completes the
      Last IP Address field.
   3. Select Save.

      The IP address or range of addresses displays in the IP Safe
      List.

      Note: Manually refresh the IP Safe List
      page to reflect your changes.
5. Edit an existing IP address or range of addresses.
   1. Select an IP address entry and select ACTIONS > Edit.

      ![A screenshot of a dialog box for Monitor edit IP address options with Edit or Remove options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/monitoripeditoptions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc0NDAsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.dAhuxppxOiyf_yCQV2QUfV3xCzuukGrMW47NjTZanr8&_LANG=enus "A screenshot of a dialog box for Monitor edit IP address options with Edit or Remove options.")
   2. Change the first, last, or CIDR address IP address fields.
   3. Select SAVE.
   4. Optionally delete the entry by selecting ACTIONS > Remove.

      ![A screenshot of the Remove from Monitor Safe List confirmation message with Remove or Cancel options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/monitoripremove.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc0NDAsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.dAhuxppxOiyf_yCQV2QUfV3xCzuukGrMW47NjTZanr8&_LANG=enus "A screenshot of the Remove from Monitor Safe List confirmation message with Remove or Cancel options.")

      The system displays a confirmation message to remove the IP address or IP address
      range.
   5. Select Remove to delete the IP address from the
      IP Address Safe List.

      The command prompts you to confirm removing the IP addresses.

      `Are you sure you want to remove the IP range? This cannot be
      undone`

      Manually refresh the IP Safe List page to display to reflect your
      changes.

## Search for IP addresses

Search the Docusign Monitor IP Safe List to determine whether
an IP address has already been submitted.

When you add IP addresses to your safe
list, they display in the Monitor IP Safe List. All IP addresses are considered
unauthorized unless you add them to the Safe List.

1. From Docusign Admin, select Monitor.
2. Select IP Safe List. Any configured acceptable IP
   addresses display.
3. Select Search IP Addresses to locate existing IP
   Safe List addresses.

   ![A screenshot of the Monitor IP Safe List dashboard with existing approved IP addresses.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/monitoripsafedash.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc0NDAsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.dAhuxppxOiyf_yCQV2QUfV3xCzuukGrMW47NjTZanr8&_LANG=enus "A screenshot of the Monitor IP Safe List dashboard with existing approved IP addresses.")

If the IP address for which you searched does not display in the
IP Address Safe List, optionally add it.

## View Unauthorized IP address activity

As a security administrator, you want to view activity from unknown or unapproved
locations. The Docusign Monitor dashboard includes a tile with
event and envelope activity for IP addresses that are not specified in the
Monitor IP Safe List.

1. From Docusign Admin, select Monitor.
2. Select IP Safe List.

   The unauthorized IP Address activity displays under the IP Safe List dashboard.

   ![A screenshot of a Monitor dashboard with an unauthorized IP Address Activity display.](https://docusign-be-prod.zoominsoftware.io/api/bundle/gso1581445176035/page/images/monnewunauthip.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTc0NDAsInNoZWFmIjoiZ3NvMTU4MTQ0NTE3NjAzNSJ9.dAhuxppxOiyf_yCQV2QUfV3xCzuukGrMW47NjTZanr8&_LANG=enus "A screenshot of a Monitor dashboard with an unauthorized IP Address Activity display.")

   These types of activities over the last 90 days from any unauthorized IP addresses display
   in these widgets in the dashboard:

   - Unauthorized events
   - Unauthorized events by location
   - Unauthorized events by name
3. View the alert history and detailed information as described in the [View Alert Details with Monitor Alert History](https://support.docusign.com/s/document-item?bundleId=gso1581445176035&topicId=OLDorv1594064852688.html&_LANG=enus "View and customize detailed Monitor alert information using the Alert History filters and options.") article.
