# FolderSearchTasks Resource

FolderSearchTasks resource

Developing with the CLM API is only available
for CLM customers with a production account.
To purchase a plan to gain access to the CLM API,
contact [Docusign Sales](https://www.docusign.com/contact-sales).

## Methods Supported

| Method | Description |
| --- | --- |
| [Get](https://developers.docusign.com/docs/clm-api/reference/tasks/foldersearchtasks/get/) | GET  ```  /v2/{accountId}/foldersearchtasks/{taskid} ```  Retrieve a previous folder search |
| [GetSearchResult](https://developers.docusign.com/docs/clm-api/reference/tasks/foldersearchtasks/getsearchresult/) | GET  ```  /v2/{accountId}/foldersearchtasks/{taskid}/result ```  Retrieve a previous folder search result |
| [Post](https://developers.docusign.com/docs/clm-api/reference/tasks/foldersearchtasks/post/) | POST  ```  /v2/{accountId}/foldersearchtasks ```  Search for folders |
