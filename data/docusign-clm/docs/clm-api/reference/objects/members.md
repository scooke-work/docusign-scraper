# Members Resource

Members resource

Developing with the CLM API is only available
for CLM customers with a production account.
To purchase a plan to gain access to the CLM API,
contact [Docusign Sales](https://www.docusign.com/contact-sales).

## Methods Supported

| Method | Description |
| --- | --- |
| [Get](https://developers.docusign.com/docs/clm-api/reference/objects/members/get/) | GET  ```  /v2/{accountId}/members/{id} ```  Get a member |
| [GetCurrent](https://developers.docusign.com/docs/clm-api/reference/objects/members/getcurrent/) | GET  ```  /v2/{accountId}/members/current ```  Get the current member |
| [GetCurrentPermissionSets](https://developers.docusign.com/docs/clm-api/reference/objects/members/getcurrentpermissionsets/) | GET  ```  /v2/{accountId}/members/current/permissionsets ```  Get the permission sets that the current member has assigned |
| [GetCurrentRecentDocuments](https://developers.docusign.com/docs/clm-api/reference/objects/members/getcurrentrecentdocuments/) | GET  ```  /v2/{accountId}/members/current/recentdocuments ```  Get the recently accessed documents of the current member |
| [GetCurrentWatchedDocuments](https://developers.docusign.com/docs/clm-api/reference/objects/members/getcurrentwatcheddocuments/) | GET  ```  /v2/{accountId}/members/current/watcheddocuments ```  Get the watched documents of the current member |
| [GetCurrentWatchedDocumentsProcessTrackingActivities](https://developers.docusign.com/docs/clm-api/reference/objects/members/getcurrentwatcheddocumentsprocesstrackingactivities/) | GET  ```  /v2/{accountId}/members/current/watcheddocumentsprocesstrackingactivities ```  Get the watched documents process tracking activities of the current member |
| [GetCurrentWorkflowActivities](https://developers.docusign.com/docs/clm-api/reference/objects/members/getcurrentworkflowactivities/) | GET  ```  /v2/{accountId}/members/current/workitems ```  Get the workflow inbox of the current member |
| [GetCurrentWorkflowQueues](https://developers.docusign.com/docs/clm-api/reference/objects/members/getcurrentworkflowqueues/) | GET  ```  /v2/{accountId}/members/current/workflowqueues ```  Get the workflow queues of the current member |
| [GetGroups](https://developers.docusign.com/docs/clm-api/reference/objects/members/getgroups/) | GET  ```  /v2/{accountId}/members/{id}/groups ```  Get the groups of a user |
| [GetGroupsCurrent](https://developers.docusign.com/docs/clm-api/reference/objects/members/getgroupscurrent/) | GET  ```  /v2/{accountId}/members/current/groups ```  Get the groups of the current user |
| [GetMembers](https://developers.docusign.com/docs/clm-api/reference/objects/members/getmembers/) | GET  ```  /v2/{accountId}/members ```  Get the members in the current account |
| [GetPermissionSetsForMember](https://developers.docusign.com/docs/clm-api/reference/objects/members/getpermissionsetsformember/) | GET  ```  /v2/{accountId}/members/{id}/permissionsets ```  Get the permission sets that the member has assigned |
| [GetWorkflowActivities](https://developers.docusign.com/docs/clm-api/reference/objects/members/getworkflowactivities/) | GET  ```  /v2/{accountId}/members/{id}/workitems ```  Get the workflow inbox of a member |
| [GetWorkflowQueues](https://developers.docusign.com/docs/clm-api/reference/objects/members/getworkflowqueues/) | GET  ```  /v2/{accountId}/members/{id}/workflowqueues ```  Get the workflow queues of a member |
| [Patch](https://developers.docusign.com/docs/clm-api/reference/objects/members/patch/) | PATCH  ```  /v2/{accountId}/members/{id} ```  Update a member |
| [PatchCurrent](https://developers.docusign.com/docs/clm-api/reference/objects/members/patchcurrent/) | PATCH  ```  /v2/{accountId}/members/current ```  Update the current member |
| [Put](https://developers.docusign.com/docs/clm-api/reference/objects/members/put/) | PUT  ```  /v2/{accountId}/members/{id} ```  Update a member |
| [PutCurrent](https://developers.docusign.com/docs/clm-api/reference/objects/members/putcurrent/) | PUT  ```  /v2/{accountId}/members/current ```  Update the current member |
