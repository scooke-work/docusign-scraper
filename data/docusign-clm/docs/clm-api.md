# CLM API overview

Developing with the CLM API is only available for CLM customers with a production account. To purchase a plan to gain access to the CLM API, contact [Docusign Sales](https://www.docusign.com/contact-sales).

Integrate document generation and management workflow into your Salesforce and custom apps. The Docusign Contract Lifecycle Management (CLM) API enables developers to interact with Docusign CLM documents, data, and services.

This content is for the latest version of the Docusign CLM API. If you are looking for legacy Spring CM API content, see the [CLM.CM API](https://developers.docusign.com/docs/clm-api/clm.cm/) version.

## Features

### Easy transition from SOAP API to CLM API

1-to-1 mapping of endpoints with additional endpoints to use

### Cross-Origin Resource Sharing (CORS)

Access a normally restricted resource via JavaScript, such as uploading and downloading documents, directly to Docusign CLM from white-listed domains

### Access tokens for more security

Along with other authentication flows, an access token must be used to make any API call
