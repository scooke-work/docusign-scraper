# Manage Tasks for Managed Users in Docusign CLM

Learn how to manage tasks for managed users in Docusign CLM.

## Before you begin

Select Enable updated task management
page on the Account Settings page. The updated page has a
more modern layout.

You can claim, unassign, and reassign tasks for users who have your login set as their
profile's Managed By attribute.

1. Log on to Docusign CLM.
2. Select Tasks in the navigation header.

   The task management page opens. Any tasks for managed users are available in the
   Managed Tasks view.
3. Select the Managed Tasks view.

   The product displays the task view.
4. Select the checkbox for one or more tasks, select any of the following actions, and follow any prompts to complete the
   activity:

   Claim
   :   Removes the task from the user and assigns the task to you.

   Reassign
   :   Opens a selection window where you can assign a new owner.

   Unassign
   :   Removes the task assignment from the user.

You have completed the task management activity for a managed
user.
