# Enable Notifications for a Folder or Document in Docusign CLM.CM

Learn how to enable notifications for a folder or document. Users can enable
notifications to receive alerts about folder or document activity.

Users can enable notifications for a folder or
document in Docusign CLM.CM. Enabling notifications allows users to receive alerts about
folder or document activity.

Note: This topic applies to Docusign CLM.CM.
To determine which kind of Docusign CLM account you have, see [Identify your Docusign CLM Account Type](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=xzt1631832579827.html&_LANG=enus).

1. Log on to Docusign CLM.CM.
2. Select Documents from in the navigation header.

   The Documents screen displays, listing all
   documents.
3. Select the checkbox for the folder or document you want to receive notifications
   for.

   Note: Selecting a folder or document
   that already has notifications enabled disables notifications.
4. Select Manage  >  Enable Notification.

   You have successfully enabled notifications for a document or folder in Docusign CLM.CM. A modal displays indicating the change in notifications for the folder or
   document.
