# Activate Upload

Learn how to upload a CSV file into a folder that contains documents to receive
attributes. Administrators perform this upload as part of assigning attribute values to multiple
documents without the need for manual entry.

Upload the CSV file into the folder which contains the documents and which was specified in the "In Folder" filter of your rule. The Bulk Attribute Assignment will begin immediately, and Docusign CLM will send e-mail notifications to the user who last modified the rule if any exceptions or failures arise.

Setting up Bulk Attribute Import requires application of Docusign CLM attribute groups,
smart rules, and attribute import mappings.

Note: For more information about creating attribute import mappings and
managing attribute groups, see the documentation about [Document Attributes in Docusign CLM](https://support.docusign.com/s/document-item?topicId=rxl1576795529933.html&_LANG=enus) and [Smart Rules in Docusign CLM](https://support.docusign.com/s/document-item?topicId=yhx1576610044302.html&_LANG=enus).
