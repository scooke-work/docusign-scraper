# Customize Navigation and Actions Settings in Docusign CLM

Learn how to customize the user interface for navigation and customize the Actions menu
in Docusign CLM.

Administrators and super administrators can customize
the navigation header and Actions menu in Docusign CLM. Navigation and
action items can be hidden or shown, based on selected options.

1. Log in to Docusign CLM as an administrator or super
   administrator.
2. Select Admin in the navigation header.

   An administrative sidebar opens.
3. Select Navigation & Actions under Account Management.

   The Navigation & Actions page opens.
4. Select the Who is this for? drop-down menu, and choose a group
   or account to apply your changes to.

   Note: By default, navigation changes are applied to all users on
   your account.
5. Select the What are you customizing? drop-down menu, and choose
   the navigation options that you want to customize.
6. Select the toggles to customize the visiblity of navigation links and actions.
7. Select Save.

You have successfully customized navigation and actions for a
group in Docusign CLM. The next time users log in, they will see the
changes.
