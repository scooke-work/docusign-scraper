# Split a PDF in Docusign CLM

Learn how to create multiple PDFs using Split in Docusign CLM.

The Split feature enables you to divide one PDF into
multiple PDFs without altering the original.

1. Log on to Docusign CLM.
2. Select Documents from the Navigation Header.

   The Documents list displays.
3. Select a PDF.

   The selected PDF displays in Document Review.
4. Select the Pages menu, and choose
   Split.
5. Enter the desired page numbers into the Pages field. If you need
   to add more pages, select Add More.

   Note: You can enter page numbers as
   individual pages or ranges of pages in the Pages fields. For example, entering "3-5, 7,
   1" in one Pages field will copy pages 3, 4, 5, 7, and 1 in that order into the new
   PDF.
6. Once all pages have been chosen, select Save.

   You have successfully split a PDF in Docusign CLM. Select the
   generated links to view and download the new PDF files.
