# Create a custom CSV data file

Learn how to create a custom CSV data file. Administrators create this file as part of
assigning attribute values to multiple documents without the need for manual entry.

This file will contain the attribute information associated with each filename organized
under columns formatted with ‘group.field’ as the header. All attribute values entered into
the cells of the CSV file MUST contain a valid value for the specified attribute, including
drop-down field values. Enter dates in a "mm/dd/yyyy" format. List the filenames in column
one. Column one must have no header.

For example, a CSV file for a bulk import of resumes contains information in a certain
format. The document names in a folder is listed on the left-hand side. The attribute
group.attribute field is listed in the column headers and the attribute values are completed
in the other CSV value fields.
