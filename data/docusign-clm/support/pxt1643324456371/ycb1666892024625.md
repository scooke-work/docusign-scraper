# Document Classification in Docusign CLM

Learn about the different types of document types in Docusign CLM. Administrators can
enable custom document types, and make changes to the visibility of document
types.

In Docusign CLM, document types are classified into three broad categories: standalone
documents, amendments, and supplements. Each of these categories encompass several types
of agreements. In Docusign CLM, administrators can create alternate names for document
types. This enables administrators to fit document categories to their specific use
cases.
