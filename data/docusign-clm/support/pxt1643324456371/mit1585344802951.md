# Two-Step Verification in CLM.CM

Read about two-step verification in CLM.CM. Administrators and users can manage two-step
verification to add more control over security for the account.

In CLM.CM accounts, administrators and users can manage two-step user verification. Using this
verification can give your organization more control over security for the account.

Administrators can enable and disable two-step verification for the entire account. If an
administrator has not enabled two-step verification at the account level, users can
manage the verification for their profiles.

Note: For a Docusign CLM account, the Docusign administrator can enable and
disable two-factor authentication for the entire organization. For more information,
see [Two-Step Verification](https://support.docusign.com/s/document-item?topicId=txt1578456659215.html&_LANG=enus) and [Identify Your Docusign CLM Account Type](https://support.docusign.com/s/document-item?topicId=xzt1631832579827.html&_LANG=enus).
