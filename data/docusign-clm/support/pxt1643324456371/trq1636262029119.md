# Set a Trash Retention Limit for Your CLM Account

Learn how to set a trash-retention policy to manage your deleted CLM
content.

When you delete CLM content, your trash folder retains the content for 90 days by default. You
can change the limit to retain trash anywhere between 0 and 365 days.

Important: When content has been in the trash for the
specified number of days, the content gets permanently deleted. This content is
removed from all Docusign servers and can never be recovered.

Use this procedure to change the default number of days that trash is retained.

1. Log in to Docusign CLM as an administrator.
2. Select Admin from the navigation header.

   An administrative sidebar opens.
3. Select Account Settings under Account Management.

   The Account Settings page opens.
4. Type a value in the Trash Retention Days field:

   - 90 is the default setting. Files get permanently deleted after 90 days.
   - 0 is the most aggressive setting. All files get permanently deleted at
     the end of each day.
   - 365 is the least aggressive setting. All files get permanently deleted
     after one year.
   - You can enter any number between 0 and 365.
5. Select Save at the bottom of the page.

   A message indicates that your changes were successfully
   saved.

You have completed the process.
