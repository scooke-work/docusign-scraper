# Create a Docusign CLM Document Rule

Learn how to create a document rule in Docusign CLM. Administrators can build document
rules to automate actions on documents when events happen.

1. Select Admin in the navigation header.
2. Select Smart Rules under Automation Tools.
3. Select Rule > New Document Rule.
4. Enter a short description in the Name field.

   An example is "AP Invoice Migration".
5. Open the Event drop-down menu, and select Document
   Entered Folder.
6. Select an option under Execute as to control who executes the
   rule.
7. Add a name filter:
   1. Open the Filter drop-down menu, and select Has
      Name Like.
   2. Enter `*.csv` in the Name field in the
      Filters section.

      Asterisks (\*) function as wildcards throughout CLM.
   3. Select Add Filter to add the name filter.
8. Add a folder filter:
   1. Open the Filter drop-down menu, and select In
      Folder.
   2. Select Select Folder, choose the folder that contains the
      documents, and select the Select button.
9. Add an action:
   1. Open the Action drop-down menu, and select Bulk
      Attribute Import.
   2. Choose a value from the Attribute Import Mapping drop-down
      list.
   3. Select Add Action.
10. Select Save to save the rule.
11. Enable the rule:
    1. Select the check box to the left of the document rule.
    2. Select Rule > Enable.

       The rule's status changes to Enabled.
