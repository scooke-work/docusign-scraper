# Create a Shared Dashboard in Docusign CLM

Learn how to create a shared dashboard in Docusign CLM. Administrators can create shared dashboards to control the visibility of folders, reports, and document-linked tasks.

In Docusign CLM, administrators can create a shared dashboard to assign to users or groups. Administrators can use shared dashboards to control the visibility of folders, reports, and document-linked tasks. For example, the administrator can share the dashboard only with certain users.

Note: Giving users access to a shared dashboard does not give them access to anything that they do not already have access to. Additionally, the users cannot modify the dashboard or add widgets

1. Log in to Docusign CLM as an administrator.
2. Select Dashboard in the navigation header.

   The dashboard page opens.
3. Select Add > Dashboard.

   The Add New Dashboard dialog box opens.
4. Define the dashboard properties:
   1. Type a name in the Name field, and select Personal Dashboard under Visibility.
   2. (Optional) Select the Set this dashboard as my personal default checkbox.
   3. Select Save to create the dashboard.

      The product creates the dashboard.
5. Select Add > Widget to [add each widget](https://support.docusign.com/s/document-item?bundleId=pxt1643324456371&topicId=htp1576795529743.html&_LANG=enus "Learn how to add widgets to your dashboard in Docusign CLM. You can use widgets to organize information for your business purposes.") to set up your dashboard for use.

   After setting up all widgets, you can share the dashboard with your audience.
6. Convert the dashboard to a shared dashboard:
