# Create a Persona View for the Documents Page in CLM

Learn how to create a persona view for the Documents page in CLM. Administrators can use
personas to help simplify a user's experience and actions while working in the
application.

CLM administrators use personas to create a unique display in the interface for menu, toolbar,
and navigation options. Personas can help simplify a user's experience and actions
while working in the application. Use this procedure to create a persona view for
the Documents page in CLM.

1. Select Documents in the navigation header.

   A Documents landing page opens.
2. Select the View menu name, and select one of the
   following options:

   - Customize Toolbar (CLM.CM only)
   - Customize Menu

   You create a menu item that displays for the persona. Or you can customize a toolbar
   in CLM.CM and assign it to appear for a persona.
3. Make your changes, and select Save.

Any user who gets assigned the persona now sees the
customized display in CLM.
