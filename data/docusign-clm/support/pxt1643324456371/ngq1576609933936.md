# Operating System Requirements and Browser Support for Docusign CLM

Look up operating system requirements and browser support for Docusign CLM.

The following sections describe the operating system requirements and supported browsers
for Docusign CLM.

Note:

- While access to Docusign CLM is possible through mobile browsers,
  there is no dedicated Docusign CLM app for Android or iOS. Depending
  on the settings selected by your Docusign CLM Administrator, some
  Document Preview and Docusign CLM features may not function as
  intended in mobile browsers.
- Any inbound or outbound connection with Docusign CLM must use the TLS 1.2 encryption
  protocol. This requirement impacts all methods of communicating with the Docusign CLM
  platform.

## PC Requirements (Docusign CLM and Desktop Applications)

PC requirements are as follows:

- Microsoft Windows 10 or newer
- Microsoft .NET Framework 4.0 for Microsoft Windows
- Visual C++ Runtime Libraries for Microsoft Windows

## Mac Operating System Requirements (Docusign CLM and Desktop Applications)

Mac operating system requirements are as follows:

- MacOS 10.14 (Mojave) or newer

## Supported Browsers

Docusign recommends using the latest version of the following supported browsers:

- Google Chrome
- Mozilla Firefox
- Apple Safari
- Microsoft Edge
