# Permanently Remove All CLM Content from the Trash Folder

Learn how to delete CLM content from the trash folder. Administrators can permanently
remove content to free storage in the CLM account or serve other needs.

Administrators can delete content from the trash folder in Docusign CLM. This is useful when
you want to free storage in your CLM account.

Important: Data that gets deleted from the
trash folder gets removed from all Docusign datacenters and can never be
restored.

Use this procedure to permanently delete the contents of your trash folder.

1. Log in to Docusign CLM as an administrator.
2. Select the Trash folder in the folder list in the
   Documents view.

   The Trash page opens, showing all deleted
   content.
3. Select File > Empty Trash.

   A dialog box opens where you must confirm the deletion.
4. Type DELETE in the blank text field, and select
   OK.

   ![Enter the word DELETE](https://docusign-be-prod.zoominsoftware.io/api/bundle/pxt1643324456371/page/images/Enter-the-word-DELETE.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTg1OTEsInNoZWFmIjoicHh0MTY0MzMyNDQ1NjM3MSJ9.BaMcQGynTwXq19HvTsodLOjSMmPPopHpO8vTlnIMZFE&_LANG=enus "Enter the word DELETE")

   The dialog box closes. A message indicates that the product successfully
   emptied the trash folder.

You have permanently deleted the contents of the trash folder. All
data from the trash folder is removed from all Docusign data centers. This data can not
be restored.
