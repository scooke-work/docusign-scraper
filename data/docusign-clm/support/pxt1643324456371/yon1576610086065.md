# Create a User in a CLM.CM Account

Learn how to create a user in a CLM.CM Account. Administrators can create a user who
can then view or work on documents in your account.

Administrators can create a user when somebody needs access to CLM.CM. The user can view or
work on documents in your account. This person can be a team member, client, vendor, or
other individual.

Assign a permission profile that matches the level of action that the user needs to take.
For example, if a client only needs to view documents, assign a Guest permission profile. If
your internal team member needs to add documents and edit the documents, assign a Full
Subscriber permission profile.

Note: When setting security on documents and folders, you can define
access based on user roles. For more information about delegated administrative permissions,
see the [Limited Administrative Permissions with Delegated Permissions in Docusign CLM](https://support.docusign.com/s/document-item?bundleId=pxt1643324456371&topicId=bra1576610051531.html&_LANG=enus "Read about delegated permissions in Docusign CLM. Administrators can assign permissions to users in Docusign CLM (or full subscribers in CLM.CM). Users with delegated permissions have access to assigned administrative functions.") topic.

This procedure describes how to create a user who is not an API-only user.

1. Log in to CLM.CM.
2. Select Admin in the navigation header.

   The Admin sidebar opens.
3. Select Users under User Management.

   The Users page opens.
4. Select New User.

   The Create New User page opens.
5. Complete the Account Details section:
   1. Enter a name in the Login Name field.

      Login names must be unique.
   2. Select the user's role from the Permission Profile drop-down
      list.
   3. (Optional) Select a permission from the Delegated Permission
      drop-down list.

      A list displays only when you have saved permissions in your account.

   password.

   Note: The password expiration default is configurable in Account
   Settings.
8. Select the Send Email box to send the user an email to activate
   the user account.
9. Complete the Profile Information section:
   1. Enter the user's first name in the First Name field.
   2. Enter the user's last name in the Last Name field.
   3. Enter a valid email address in the Email Address
      field.

      Note: Each new user must have a unique email address.
   4. (Optional) Supply a phone number, fax number, company, department, title, address,
      or country.
10. Select Save .

The product creates the user.
