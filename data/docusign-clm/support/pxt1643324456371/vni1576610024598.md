# Customizing Menus and Toolbars in Docusign CLM

Read about customizing menus and toolbars in Docusign CLM. In Docusign CLM,
administrators can customize menus and toolbars to simplify a user’s experience.

In Docusign CLM, administrators and Super Administrators can customize menu actions for
users. Document Actions options can be enabled or disabled to streamline the user
experience.

Administrators and Super Administrators can also create custom actions. Customizations can
help simplify the user experience by creating shortcuts to common actions. Menu options can be
created to execute actions in workflows, convert documents, and other document management
options.
