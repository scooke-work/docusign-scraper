# Enable Task Completion Screens in Docusign CLM or CLM.CM

Learn how to enable task completion screens in Docusign CLM.

Note: This documentation covers legacy document preview experiences for Docusign CLM. As of July 1, 2025, most customers have been upgraded to Document Review as their default document preview experience in Docusign CLM. For more information on deprecation of document preview experiences, see [Docusign Document Preview Experience Changes FAQ](https://support.docusign.com/s/document-item?bundleId=ssg1729007411846&topicId=twq1729007565221.html&_LANG=enus) For more information on features and enhancements available in Document Review, see [Document Review in Docusign CLM](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=eyk1666363075054.html&_LANG=enus).

Docusign CLM administrators can enable the task completion screens for users. These screens present a more modern layout.

1. Log in to the product as an administrator.
2. Select Admin in the navigation header.

   An administrative sidebar opens.
3. Select Account Settings under Account Management.

   The Account Settings page opens.
4. Select the Enable task completion screens checkbox in the Workflow & Tasks section.
5. Select Save.

You have enabled the task completion screens for your users.
