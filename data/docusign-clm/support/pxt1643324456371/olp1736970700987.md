# Perform the Tasks of Other Task Group Members in Docusign CLM

Learn how to access and perform the tasks of other task group members in Docusign CLM.

In Docusign CLM, an administrator or user can perform a task that is
assigned to another task group member. This process creates more flexibility when
managing tasks and handling business needs.

For example, assume that an approval task is assigned to Jason in the Legal Users
task group. Jason is on vacation. Another user in the group can access the task
directly and complete it.

1. Log in to Docusign CLM.
2. Select Tasks in the navigation header.

   The task management page opens.
3. Select a task view.
4. Select the task name.

   The product accesses the task.
5. View and complete the task as needed.

   A message confirms that you completed the task.

You are finished working with the task of another task group
member.
