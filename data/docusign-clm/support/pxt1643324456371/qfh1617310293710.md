# Manage Support User Permissions in Docusign CLM

Learn how to manage Docusign Support user access
permissions.

As a CLM administrator, you can manage permissions for
the Docusign Support user.

1. Log in to Docusign CLM as an administrator.
2. Select Admin in the navigation header.

   An administrative sidebar opens.
3. Select Users under User Management.
4. Select the user named Docusign Support.
5. Make your changes to User Persona, Group
   Memberships, and Managed By.
6. Select Save.

   You have changed permissions for the Docusign Support
   user.
