# Enable Two-Step Verification for Your User Profile in CLM.CM

Learn how to enable two-step verification for your user profile in CLM.CM.

If an administrator has not enabled two-factor authentication at the account level, you can
enable two-step verification for your user profile. This action does not impact
other users in your account.

1. Log in to your CLM.CM account.
2. Select ![profile icon](https://docusign-be-prod.zoominsoftware.io/api/bundle/pxt1643324456371/page/images/Utilitymenu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTg1NjYsInNoZWFmIjoicHh0MTY0MzMyNDQ1NjM3MSJ9.Xuh3KQVX2_qmR71X2uSj9-vfCuWZz4JVR1AbIidfrVM&_LANG=enus "profile icon") in the top right corner.
3. Select **My Preferences**.
4. Select **2-Step Verification** in the left pane.

   The 2-Step Verification page opens.
5. Select **On** to enable two-step verification, and select
   Save.

The product saves your setting and logs you out.

## Next steps

The next time you log in, [set up two-step verification](https://support.docusign.com/s/document-item?bundleId=pxt1643324456371&topicId=fnt1585347689093.html&_LANG=enus "Learn how to set up two-step verification as a CLM.CM user.") for your
situation.
