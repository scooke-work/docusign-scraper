# List of Delegated Permissions in Docusign CLM

Look up delegated permission values and definitions. Administrators can assign limited
administrative permissions through delegated permissions in CLM.

The following delegated permissions can be assigned to users by CLM administrators.

| Permission | Administrative Area | Description |
| --- | --- | --- |
| Document Generation | Automation Tools | Allows a user to create, edit, and delete in the document generation tool. |
| Object Mapping (not available for CLM.CM) | Automation Tools | Allows configuration and mapping of artificial intelligence (AI) tools. |
| Workflow & Automation | Automation Tools | Allows a user to create, edit, and delete the following components:  - Workflows - Smart rules - Schedules - Forms - Reminder templates   Note: The workflow engine has unrestricted access to the document repository; however,users with this permission still respect existing folder security for the CLM User permission profile (Docusign CLM) or Full Subscriber permission profile (CLM.CM). Any membership in groups also continues to be respected. |
| UI Preferences | Account Management | Allows a user to change account branding, navigation, and actions. |
| Security | Account Management | Allows a user to change account-level security settings. These settings include two-step verification, logon restrictions, SAML SSO, and password strengths. |
| Attributes & Auto Numbers | Documents & Folders | Allows a user to configure document and folder attribute fields. These fields include Attribute Groups, Auto Numbers, and Attribute Import Mappings. |
| Inbound Folder Content | Documents & Folders | Allows a user to manage inbound folder content from fax and email. This content includes inbound email addresses and inbound fax lines. |
| Create, Edit and Delete Any Report | Reporting | Allows a user to create, edit, and delete reports within the account. |
| Integrations | Integrations | Allows a user to administer settings for third-party integration configurations, including Salesforce, electronic signature, and REST API. |
| Users | User Management | Allows a user to create, edit, and delete users and task groups in the address book. |
