# Create a CLM Administrator for your Organization

Learn how to add a CLM administrator to the organization. Administrators can use these
steps to add a user as a CLM administrator.

Organization administrators can access Docusign CLM by default; however, you must add at least
one CLM administrator to the organization. Use this procedure to add a user as a CLM
administrator.

Note: You can also assign administrative rights to an existing user. To do
so, [add the CLM Admin permission profile to
the user](https://support.docusign.com/s/document-item?bundleId=pxt1643324456371&topicId=das1581520893906.html&_LANG=enus "Learn how to add a CLM Admin permission profile to an existing user. Administrators add this profile to give the user administrative rights for Docusign CLM.").

1. Access Docusign Admin.
2. Select Users.

   The Users page opens.
3. Select Add User.
4. Complete the User Profile page, and select
   Next: Accounts at the bottom.

   The Add Account Membership page opens.
5. Select an account from the Account drop-down list.

   A list of applications displays.
6. Select a level of access from the Permission Profile
   drop-down list in the eSignature section.
7. Enable administrative permission for CLM:
   1. Move the toggle to Enabled in the CLM
      section.
   2. Select CLM Admin from the Permission
      Profile drop-down list.
8. Select Next to complete the rest of the process, review
   the user profile, and save the user.

You are finished. The CLM administrator can access the CLM
account.
