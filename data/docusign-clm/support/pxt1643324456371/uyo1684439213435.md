# Set IP Access Restrictions in CLM.CM

Learn how to restrict access to CLM.CM by IP address.

Super administrators can restrict or allow access to
the CLM.CM environment by IP address.

Note: This topic applies to
CLM.CM. To determine which kind of CLM account you have, see [Identify your Docusign CLM Account
Type](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=xzt1631832579827.html&_LANG=enus).

1. Log in to CLM.CM as a super administrator.
2. Select Admin in the navigation header.

   An administrative sidebar opens.
3. Select Security Settings under Account Management.

   The Security Settings page opens.
4. Select your option in the Network section:

   - To set up IP addresses that are allowed to access CLM.CM, select
     Allow.
   - To set up IP addresses that are denied access to CLM.CM, select
     Deny.
5. Enter the IP addresses you want to allow or deny in the IP
   Addresses field.

   Note: Multiple IP addresses must be separated by a comma.
   Regular expressions (RegEx) can be used to allow or restrict a range of
   addresses.
6. Select Save.

You have successfully set up access to CLM.CM based on IP
address.
