# Document Management in Docusign CLM

Read about the tools for document management available to administrators in Docusign CLM. Administrators can manage document attributes, eForms, and uploads.

Administrators in Docusign CLM have several tools and features available to help manage
documents. Administrators can use these tools to manage document uploads and document
attributes in Docusign CLM.

Administrators can also customize the user experience as part of document management.
Menus and actions can be tailored to reduce confusion and increase efficiency for
document workflows.
