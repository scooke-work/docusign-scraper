# First Steps for an Org Administrator in Docusign CLM

Read about the first steps that an organization administrator can take after Docusign CLM account provisioning. Administrators can use this process to set up Docusign CLM.

Docusign Admin supports Docusign CLM accounts and administrative components for your
agreement processes. As an organization administrator, your CLM permission profile was
included in account provisioning.

You can use the following process to set up Docusign CLM.

Note: The credentials that you use for Docusign Admin and eSignature are
also your Docusign CLM credentials.

1. Log on to Docusign eSignature with Docusign org administrator credentials.
2. Open the Docusign Admin Dashboard﻿. For information about how to access the Docusign Admin
   Dashboard, see [Open the Docusign Admin
   Dashboard](https://support.docusign.com/s/document-item?bundleId=rrf1583359212854&topicId=hao1583359136204.html&_LANG=enus).
3. Log on to Docusign CLM and create a CLM administrator for your organization. For information
   about how to create a CLM administrator for your organization, see [Create a CLM Administrator for your Organization](https://support.docusign.com/s/document-item?bundleId=pxt1643324456371&topicId=viv1581375160992.html&_LANG=enus "Learn how to add a CLM administrator to the organization. Administrators can use these steps to add a user as a CLM administrator.").
