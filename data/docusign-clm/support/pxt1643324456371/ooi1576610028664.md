# Change the Actions Menu Name in Docusign CLM

Learn how to change the Actions menu name in Docusign CLM.

Administrators and super administrators can change
the name of the Actions menu in Docusign
CLM.

1. Log in to Docusign CLM as an administrator or super
   administrator.
2. Select Admin in the navigation header.

   An administrative sidebar opens.
3. Select Navigation & Actions under Account Management.

   The Navigation & Actions page opens.
4. Select the Actions Menu tab.
5. Specify a name in the Actions Menu Name field, and select
   Save.

You have successfully changed the name of the
Actions menu in Docusign CLM.
