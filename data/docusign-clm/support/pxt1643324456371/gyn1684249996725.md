# Account Management in Docusign CLM

Read about the various features available to Docusign CLM administrators for managing
their account. Administrators have multiple options for setting up login security, device
management, and user interface.

As an administrator in Docusign CLM, you have a wide array of controls and options for setting
up Docusign CLM. Part of getting your new environment ready includes options for
security, user interface and experience, and user permissions.

These guides are intended to lead you through these options, and to help make decisions
on how these features are used.

Note: Several of these topics are specific to either Docusign CLM or
Docusign CLM.CM, as indicated by their title. To determine which product you have, see
[Identify Your Docusign CLM Account
Type](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=xzt1631832579827.html&_LANG=enus).
