# Disable an eForm in Docusign CLM

Learn how to disable eForms in Docusign CLM that that are obsolete or
unneeded.

Administrators and Super Administrators can disable
eForms that that are obsolete or unneeded.

1. Log on to Docusign CLM as an administrator or Super Administrator.
2. Select Admin from the navigation header.

   The Docusign CLM admin panel displays.
3. Navigate to Automation Tools  >  eForms.
4. Select the checkbox for the eForm you want to disable.
5. In the Access menu, select Disable
   Submission.

   You have successfully disabled the selected eForm.
