# Add the CLM Admin Permission Profile to an Existing User

Learn how to add a CLM Admin permission profile to an existing user. Administrators add
this profile to give the user administrative rights for Docusign CLM.

You must have at least one CLM administrator in the organization. An administrator can add a
CLM Admin permission profile to an existing user. Adding this profile gives the user
administrative rights for Docusign CLM.

Note: To add this profile, Docusign CLM must be turned on in the
organization.

1. Access Docusign Admin.
2. Locate the user, and open the user's profile.

   The default view is the Memberships
   tab.
3. Select Actions > Edit for the account where you want this user to be an
   administrator.

   The Edit Account Membership page opens. If a CLM
   account is provisioned for the organization, a CLM tile displays in the list of
   applications.
4. Enable administrative permission for CLM:
   1. Move the toggle to Enabled in the CLM
      section.
   2. Select CLM Admin from the Permission
      Profile drop-down list.
5. Select Save.

The product saves your changes. You have given the user
administrative rights for Docusign CLM.
