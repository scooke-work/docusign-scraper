# Enable or Disable Edits to Submitted eForms in Docusign CLM

Learn how to enable or disable edits on submitted eForms in Docusign
CLM.

Administrators and super administrators control
whether submitted eForms can be updated in Docusign CLM.

1. Log in to Docusign CLM as an administrator or super
   administrator.
2. Select Admin in the navigation header.

   An administrative sidebar opens.
3. Select eForms under Automation Tools.

   The Forms page opens.
4. Select the check box for the eForm that you want to change.
5. Perform one of the following actions:

   - Select Access > Allow Updates to enable editing of submitted eForms.
   - Select Access > Disable Updates to disable editing of submitted eForms.

You have successfully enabled or disabled updates for submitted
eForms.
