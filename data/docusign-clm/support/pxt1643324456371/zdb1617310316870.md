# Revoke Support User Access in Docusign CLM

Learn how to revoke Docusign Support user access.

If the Docusign Support user is no longer needed, a
Docusign CLM Administrator or Super Administrator can revoke access.

1. Log on to Docusign CLM as an Administrator.
2. Select the Help Icon
   ![Help Icon Button](https://docusign-be-prod.zoominsoftware.io/api/bundle/pxt1643324456371/page/images/Help-Icon-Button.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTg2OTUsInNoZWFmIjoicHh0MTY0MzMyNDQ1NjM3MSJ9.O4BfMfHtJdglBMvWtFA399X0mmDAS3iHxTJLAO3nwlc&_LANG=enus "Help Icon Button"), and choose Grant Support Access.

   The Grant Support Access dialog box
   opens.
3. Select Revoke access.

   You have successfully disabled access for the Docusign Support user.
