# Schedules in Docusign CLM

Read about schedules in Docusign CLM. Administrators can use schedules for recurring
and explicit document management events in Docusign CLM.

Schedules define specific or recurring dates and times used by other Docusign CLM features,
such as rules or reports. There are two kinds of schedules in Docusign CLM:

- Explicit schedules allow for specified dates and times.
- Recurring schedules allow you to specify a day and time relative to the hour,
  week, or month.

Administrators should use schedules to run smart rules on an explicit date and time, or as a
recurring event. The scheduled smart rule can then execute a variety of actions according to
the selected schedule, including:

- Sending an email
- Starting a workflow
- Sending a custom report
- Moving one or more documents to the trash

Note: For more information about smart rules, see the documentation about
[smart rules in Docusign CLM](https://support.docusign.com/s/document-item?bundleId=qaz1642962729191&topicId=yhx1576610044302.html&_LANG=enus).
