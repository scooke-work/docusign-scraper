# Assign a CLM User to a Persona

Learn how to assign a CLM user to a persona. Administrators can use personas to help
simplify a user's experience and actions while working in the application.

## Before you begin

Create a group and name it after the role that you want. For
example, you can name it *Accounts Payable Processor*.

CLM administrators use personas to create a unique display in the interface for menu,
toolbar, and navigation options. Personas can help simplify a user's experience and
actions while working in the application. In a user's profile, you can assign a
single persona.

1. Select Admin in the navigation header.

   The Admin sidebar opens.
2. Select Users under User
   Management.

   The Users page opens.
3. Select the user to assign the persona to.

   The product displays the user's details.
4. Type the name of the group in the User Persona
   field.

You have assigned a CLM user to a persona. You can now create a
persona view for the Documents page in CLM.
