# Set an Account Homepage in Docusign CLM

Learn how to set a customized account homepage for users in Docusign
CLM.

Administrators and super administrators can change
the account homepage for users in Docusign CLM. Users will see the account
home page when they log in. By default, the account homepage is the
Documents page.

1. Log in to Docusign CLM as an administrator or super
   administrator.
2. Select Admin in the navigation header.

   An administrative sidebar opens.
3. Select Navigation & Actions under Account Management.

   The Navigation & Actions page opens.
4. Select the Who is this for? drop-down menu, and select
   Account Default.
5. Select the What are you customizing? drop-down menu, and select
   User Navigation Header.
6. Hover your cursor over a page row in the Pages tab, and select the
   Set Homepage check box.

   The product now labels this item as the homepage.
7. Select Save.

You have successfully set a default account homepage. Users will
see the page when they log in or select the Docusign logo in the navigation
header.
