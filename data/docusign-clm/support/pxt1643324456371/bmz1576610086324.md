# Edit a User in a CLM.CM Account

Learn how to edit a user in a CLM.CM account. For example, administrators can change
the permission profile, password settings, or profile information.

You can edit a user in CLM.CM. For example, you can
change the assigned permission profile, change password settings, or change profile
information.

1. Log in to CLM.CM.
2. Select Admin in the navigation header.

   The Admin sidebar opens.
3. Select Users under User Management.

   The Users page opens.
4. Select the box next to a user, and select Edit.

   The user profile page opens.
5. Make changes as necessary, and select Save.

The product applies the changes to the user.
