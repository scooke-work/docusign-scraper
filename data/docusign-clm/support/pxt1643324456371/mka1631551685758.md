# Choose a Delivery Method for Notifications in CLM.CM

Learn how to configure a notification delivery method to receive activity alerts in
CLM.CM. Enabling notifications allows users to receive alerts by real-time emails or a
single digest.

Users can choose a notification delivery method to
receive alerts about activity in CLM.CM. Notifications can be received as real-time
emails or a single digest.

Note: This topic applies to CLM.CM. To
determine which kind of CLM account you have, see [Identify your Docusign CLM Account
Type](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=xzt1631832579827.html&_LANG=enus).

1. Log in to CLM.CM.
2. Select the Profile icon ![Profile icon for CLM.CM](https://docusign-be-prod.zoominsoftware.io/api/bundle/pxt1643324456371/page/images/Person.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTg3MTIsInNoZWFmIjoicHh0MTY0MzMyNDQ1NjM3MSJ9.LmhFW0qXyYLDYaME9dpGPJ3vaZIYOLWtPCmsiEwfXr8&_LANG=enus "Profile icon for CLM.CM"), and select My Preferences.

   The My Preferences page opens.
3. Select Notification Preferences.
4. Select the check boxes to configure your notification preferences.
5. Select Save.

   You have successfully set up notification preferences for your account
   in CLM.CM.
