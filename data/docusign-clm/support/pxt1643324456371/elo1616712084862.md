# Support Access in Docusign CLM

Read about how to provide temporary access to Docusign Support.

Docusign CLM administrators can provide temporary access to Docusign Support to troubleshoot
issues. The administrator can control what the Docusign Support user can access. The
administrator can also provide elevated administrative permissions if necessary. All the
Support user's actions get recorded in document and folder history.

For more information on enabling Support access in Docusign CLM, see
[How to Grant Access to Support](https://support.docusign.com/s/articles/Docusign-CLM-How-to-Grant-Access-to-Support?language=en_US).

Note:

- Do not attempt to delete the Support user. The user is a standard
  product feature that is available when needed. Docusign does not impose a fee
  for providing this user.
- For CLM.CM accounts, only super administrators can provide access to Docusign Support. To determine your account type, see the documentation about [identifying the CLM account
  type](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=xzt1631832579827.html&_LANG=enus).
