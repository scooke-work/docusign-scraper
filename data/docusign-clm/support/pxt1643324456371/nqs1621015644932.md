# Update Users in Bulk in Docusign CLM

Learn how to update Docusign CLM users in bulk. Administrators can perform a bulk update on users by using a CSV file.

Administrators can upload a CSV with CLM-specific user fields to update CLM users in bulk. Fields that you can update include Persona, ManagedBy, DelegatedPermissions, and MailingLists.

Note: This article applies to Docusign CLM. You cannot use this bulk update feature to add new users.

The product uses the CSV's Email field to locate users to update. If a user email cannot be found, the product skips that row in the CSV.

1. Log in to Docusign CLM.
2. Select Admin in the navigation header.

   The Admin sidebar opens.
3. Select Users under User Management.
4. Select Bulk update.

   A Bulk User Update dialog box displays.
5. Select Export CSV.

   A CSV template file downloads to your computer.
6. Complete user details in the CSV, and upload the updated CSV file.
7. Select Submit in the Bulk User Update dialog box.

The CSV file is uploaded. You receive an email when users get updated. The email contains details of any changes.
