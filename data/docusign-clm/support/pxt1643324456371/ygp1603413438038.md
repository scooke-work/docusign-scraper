# Configure Document Generation Forms to use Writeback in Docusign CLM

Learn how to configure Document Generation Forms for use with Writeback to Salesforce in
Docusign CLM.

To use Writeback, you must configure Document Generation Forms in Docusign CLM. For
more information about Document Generation Forms in Docusign CLM, see [Document Generation Forms in Docusign CLM](https://support.docusign.com/s/document-item?bundleId=uqj1643324072491&topicId=rsb1576795539656.html&_LANG=enus).

Writeback only works for merge fields within the Document Generation form that are
linked to Salesforce fields. To determine if a merge field is linked to Salesforce,
select the field name within the form and open the Properties
tab. Under Field Value, locate the Link to Salesforce
field checkbox. If this box is checked, the field is linked to
Salesforce.

For all of the document fields that you want to review using Writeback, ensure that the
merge tags for the linked merge fields are pasted in your Word template. Merge fields
that are linked to Salesforce contain TrackName in the merge tag.![CLM Writeback merge tag](https://docusign-be-prod.zoominsoftware.io/api/bundle/pxt1643324456371/page/images/CLM-Writeback-merge-tag.jpg?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTg5NzAsInNoZWFmIjoicHh0MTY0MzMyNDQ1NjM3MSJ9._4ah3fWg1KebSFptg8Q6X8GDF2n-843NZtFBh3wm1es&_LANG=enus "CLM Writeback merge tag")

TrackName in the merge tag indicates that this is tracked content in CLM, and
there is content control in your Word Document. You can configure settings for your
tracked content by editing the merge tag properties.

Properties for tracked content include:

- Title="Company Name" (Title gives your content control a title that is
  exposed in the Word UI)
- Hidden="true" (Hidden hides the content control)
- Deletable="true" (Deletable allows the content control to be deleted or
  not)
- Editable="true" (Editable locks editing)

For more information about word termplate merge tags, see [Document Generation Word Template Merge Tags](https://support.docusign.com/s/document-item?bundleId=uqj1643324072491&topicId=uno1576795541214.html&_LANG=enus).

Writeback supports the following Document Generation fields and the
corresponding linked Salesforce objects.

- Currency
- Date
- Dropdown
- Text Box
- Text Field
- Number
