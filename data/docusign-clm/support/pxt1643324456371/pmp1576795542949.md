# Enable User Account Notifications in Docusign CLM.CM

Learn how to enable notifications to receive alerts about user account activity.
Administrators can receive notifications about alerts by email.

Administrators can enable notifications to receive
alerts about user account activity. Administrators receive alerts when users enable, disable,
or change their Permission Profile.

Note: This topic applies to Docusign CLM.CM. To determine which kind of Docusign CLM account you have, see [Identify your Docusign CLM Account Type](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=xzt1631832579827.html&_LANG=enus).

1. Log on to Docusign CLM.CM as a Super Administrator.
2. Select Admin from the navigation header.

   The Docusign CLM.CM admin panel displays.
3. Navigate to Account Management  >  Account Settings  >  Notifications.
4. Select your options for the notifications you want to receive.
5. Once all options have been chosen, select Save.

   You have successfully setup notifications for user activity in Docusign CLM.CM.
