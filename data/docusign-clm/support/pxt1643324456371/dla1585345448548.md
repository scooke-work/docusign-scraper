# Enable Two-Factor Authentication in CLM.CM

Learn how to enable two-factor authentication in CLM.CM.

In CLM.CM, super administrators can enable two-factor
authentication for all users. Two-factor authentication provides an additional level of
login security for users.

Note: This topic applies to CLM.CM. To
determine which kind of CLM account you have, see [Identify your Docusign CLM Account
Type](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=xzt1631832579827.html&_LANG=enus).

1. Log in to CLM.CM as a super administrator.
2. Select Admin in the navigation header.

   An administrative sidebar opens.
3. Select Security Settings under Account Management.

   The Security Settings page opens.
4. Select Yes in the Enable 2-Step
   Verification section.

   Note: Optionally, you can select the Allow
   users to login to SpringCM without 2-Step Verification for 30
   days check box to enable a grace period.
5. Select Save.

You have successfully enabled two-factor authentication for
CLM.CM. Users will need to set up an authenticator as part of login.
