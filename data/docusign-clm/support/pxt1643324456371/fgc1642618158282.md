# Widgets for Docusign CLM Dashboards

Look up the functions of widgets in Docusign CLM dashboards.

You can use widgets to organize information for your business purposes. The following table describes widgets that can be configured on Docusign CLM dashboards:

| Widget | Description |
| --- | --- |
| Folder | Displays the contents of any folder that the user has access to. This widget allows you to search folders and documents. You can also execute document actions. You can configure the widget to show specific folders or all folders in the account. |
| Folder watchlist | Displays document tasks in a specified folder that are assigned to users as part of a workflow. To use this widget, folders must be added to the watchlist through a workflow step. Note: For more information, see the Add Documents to Watched Documents List step description in [Automated Document Management Workflow Steps](https://support.docusign.com/s/document-item?bundleId=yks1643320936212&topicId=hpj1576610096401.html&_LANG=enus). |
| My watchlist | Displays document tasks that are assigned to users as part of a workflow. To use this widget, documents must be added to the watchlist through a workflow step. Only documents that have active tasks get displayed in the widget. After the task gets completed, the document no longer displays in the widget. Note: For more information, see the Add Documents to Watched Documents List step description in [Automated Document Management Workflow Steps](https://support.docusign.com/s/document-item?bundleId=yks1643320936212&topicId=hpj1576610096401.html&_LANG=enus). |
| Recent documents | Displays the 10 most recently accessed, viewed, or modified documents by the current user. You can also perform document actions on any of the listed documents. |
| Report | Displays a selected report. This widget allows some reports to be exported in CSV or XLSX format. Note: For more information about the widget, see [Visualize Document Reports on the Dashboard in Docusign CLM](https://support.docusign.com/s/document-item?bundleId=lye1642705772274&topicId=yfw1603415758563.html&_LANG=enus). |
| Saved Search | (Available in [modernized search](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=jbg1707491822735.html&_LANG=enus)) Displays the results from a saved search. The person who adds the widget selects the saved search to display. From an existing widget, you can perform the following activities:  - Select a document from the results to open the document preview in a new tab - Select Open to access the full page of search results for the saved search   Note: You can edit the widget to change to a different saved search. After saving your changes, you must refresh the browser page. |
| Search | Enables selection of a saved search and displays search results on the dashboard. Note: For more information on search in Docusign CLM, see [Search in Docusign CLM](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=aug1576609932447.html&_LANG=enus). |
| Workflow inbox | Displays workflow tasks that are assigned to the current user. This widget can be configured to show tasks from specific workflows. |
