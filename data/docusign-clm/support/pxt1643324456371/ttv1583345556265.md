# User Management in Docusign CLM

Learn about user management in Docusign CLM.

A user is a person who logs in to access Docusign CLM documents, tasks, reports, and so on. A
user requires a license in your CLM account. Administrators create and manage users and
groups in Docusign Admin.

Note: In Docusign CLM, administrators can perform a bulk update for
multiple users at one time. You can perform this update from the
Users page by uploading a CSV file.

In Docusign Admin, user management includes the following activities:

- Add users
- Edit users (including group membership and permission profile)
- Perform bulk actions (such as adding or updating users in bulk)
- Add or edit groups
