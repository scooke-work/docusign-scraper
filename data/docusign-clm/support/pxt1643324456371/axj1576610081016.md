# Disabling or Enabling Password Logon for SSO Users

Read about how Docusign CLM administrators can enable or disable
password logon when SSO is configured.

If single sign-on (SSO) is set up for your organization, administrators can disable (or
enable) password logon for a select SSO user or for all SSO users. See [Docusign Single Sign-On Overview](https://support.docusign.com/s/document-item?topicId=njp1583359138365.html&_LANG=enus) for more information.

In Docusign CLM.CM, Super Administrators or User Administrators can set the account defaults,
while only Super Administrators can disable or enable a specific SSO user’s ability to log on
by using a password. When a User Administrator creates a new user, the user receives the
account password defaults. For CLM.CM account types, see [Single Sign-On
in Docusign CLM.CM](https://support.docusign.com/s/document-item?topicId=mma1584137313140.html&_LANG=enus).
