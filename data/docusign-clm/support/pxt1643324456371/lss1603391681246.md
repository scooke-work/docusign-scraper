# Define Language Settings for a Docusign CLM Account

Learn how to define language settings for a Docusign CLM account. Administrators use
these steps to customize the product to meet the language needs of users.

On the Account Settings page in
Docusign CLM, an administrator can choose language settings for the account. These
settings help you customize the product to meet the language needs of users. You can
control which languages are enabled in the account. You can also assign the default
language for the account.

1. Log in to Docusign CLM as an administrator.
2. Select Admin in the navigation header.

   An administrative sidebar opens.
3. Select Account Settings under Account Management.

   The Account Settings page opens.
4. Select the Languages Enabled in this Account drop-down,
   and select the checkboxes for the languages that you want to enable.

   Note: If a language gets disabled, users with that
   language selected get assigned the default account language.
5. Select the Default Account Language drop-down, and
   select the default account language for users.
6. Select Save.

The product saves your settings. You finished setting the
language settings for the Docusign CLM user interface.
