# Add a Widget to Your Dashboard in Docusign CLM

Learn how to add widgets to your dashboard in Docusign CLM. You can use widgets to
organize information for your business purposes.

In Docusign CLM, you can use widgets to create
visibility on document information and reports. Widgets help you organize information for your
business purposes.

Note: To learn more about widgets and their functions
in Docusign CLM, see [Widgets for Docusign CLM Dashboards](https://support.docusign.com/s/document-item?bundleId=pxt1643324456371&topicId=fgc1642618158282.html&_LANG=enus "Look up the functions of widgets in Docusign CLM dashboards.").

1. Log in to Docusign CLM.
2. Select Dashboard in the navigation header.

   The dashboard page opens.
3. Select Add > Widget.

   The Add Widget menu opens at the bottom of the
   screen.
4. Select a widget, and drag it onto the dashboard.

   The product might display a dialog box for you to configure widget properties.
   If not, the product adds the widget to the dashboard.
5. Configure the widget properties (if prompted), and select
   Save.

You have added a widget to your dashboard in Docusign CLM.
