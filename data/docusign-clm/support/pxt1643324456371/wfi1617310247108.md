# Enable Support Access in Docusign CLM

Learn how to provide temporary access to a Docusign Support
user.

As a Docusign CLM administrator, you can provide a Docusign Support user temporary access to your account. The first time you grant access to
Support, a user called Docusign Support gets created.

Note: Before enabling support access, contact Docusign Support for assistance. Docusign
Support will let you know if access to your account is necessary.

1. Log in to Docusign CLM as an administrator or super
   administrator.
2. Select the Help Icon
   ![Help Icon Button](https://docusign-be-prod.zoominsoftware.io/api/bundle/pxt1643324456371/page/images/Help-Icon-Button.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTg2ODgsInNoZWFmIjoicHh0MTY0MzMyNDQ1NjM3MSJ9.BMv0d360ON1b5pxnbJfLhvIajeL45IvtsMGrUBgTVq4&_LANG=enus "Help Icon Button"), and choose Grant Support Access.

   The Grant Support Access dialog box
   opens.
3. Choose the amount of time you want to grant access.
4. (Optional) Clear the Use previous Docusign Support’s
   permissions check box if you do not want to reuse previous
   permissions.

   Note: This option is enabled by default. When enabled,
   permissions that were previously configured for the Docusign Support user remain.
5. Review the terms, and select the checkbox for By checking the
   checkbox, you agree to the following terms.
6. Select Grant Access.

   The Support Access Granted dialog box opens, identifying when
   Support access will end.
7. Select Close.

   You have successfully enabled Support Access in Docusign CLM. You
   should inform Docusign Support that you have enabled Grant
   Support Access.
