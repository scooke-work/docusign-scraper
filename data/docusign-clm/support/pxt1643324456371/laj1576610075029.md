# Show or Hide Task Management Columns in Docusign CLM

Learn how to show or hide columns in the task management page of Docusign CLM. Administrators and users can control columns to create a view that
meets their specific needs.

## Before you begin

You must have Enable updated task management
page selected on the Account Settings page.

Administrators and users can show or hide columns on their task management page in Docusign CLM. Customizing the columns helps you create a view that meets
your specific needs.

Note: Administrators cannot control which columns get displayed for all
users. Individual users control how their page's columns are set up.

1. Log in to Docusign CLM.
2. Select Tasks in the navigation header.

   The task management page opens.
3. Select the ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pxt1643324456371/page/images/Gear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTg2MTksInNoZWFmIjoicHh0MTY0MzMyNDQ1NjM3MSJ9.5JPL79vlw6PH2_imMfJcpt-T6bwk2dS5rqqnpN1iWek&_LANG=enus) icon.

   The Customize columns dialog box opens. Currently visible
   columns have a switch that is blue and right-aligned. Currently hidden columns have a
   switch that is white and left-aligned.
4. Show or hide columns as needed:

   - Show a column by selecting the switch to change it to blue and right-aligned.
   - Hide a column by selecting the switch to change it to white and left-aligned.
5. (Optional) Select ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/pxt1643324456371/page/images/Drag-and-Move.jpg?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTg2MTksInNoZWFmIjoicHh0MTY0MzMyNDQ1NjM3MSJ9.5JPL79vlw6PH2_imMfJcpt-T6bwk2dS5rqqnpN1iWek&_LANG=enus) next to a
   column to drag the column and rearrange the column order.
6. Select Apply to save your layout.

The task management page now shows the layout that you
applied.
