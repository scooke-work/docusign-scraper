# Enable External Review Notifications in CLM.CM

Learn how to enable notifications in CLM.CM to receive alerts about external review
activity. Administrators can receive notifications about alerts by email.

Administrators can enable notifications in CLM.CM to
receive alerts about external review activity. Administrators receive alerts when a
reviewer, CC recipient, or sender completes or cancels an external review.

Note: This topic applies to CLM.CM. To determine which kind of CLM
account you have, see [Identify your Docusign CLM Account Type](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=xzt1631832579827.html&_LANG=enus).

1. Log in to CLM.CM as a super administrator.
2. Select Admin in the navigation header.

   An administrative sidebar opens.
3. Select Account Settings under Account Management

   The Account Settings page opens.
4. Navigate to the Notifications section, and select the
   check boxes for your preferences in the External Review
   section.
5. Select Save.

   You have successfully set up notifications for external reviews in
   CLM.CM.
