# Allow Users to Manage Contacts in Docusign CLM

Learn how to give users access to contacts in Docusign CLM. Administrators can use these
steps to allow users to create and edit contacts.

In Docusign CLM, administrators can manage contacts. By default, contact management is
disabled for users. To allow users to create and edit contacts, an administrator
must enable contacts. Follow these steps to give users access to manage
contacts.

1. Log in to Docusign CLM.
2. Select Admin in the navigation header.

   The Admin sidebar opens.
3. Select Navigation & Actions under Account
   Management.

   The Navigation & Actions page opens.
4. Select User Navigation Header in the What are
   you customizing field.
5. Enable the toggle for Contacts.
6. Select Save.

You have enabled contact management for your CLM users. Users can
now see the Contacts tab and create and edit contacts.
