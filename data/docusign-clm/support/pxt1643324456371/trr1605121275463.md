# Agreement Actions in Docusign CLM

Learn about Agreement Actions in Docusign CLM. Administrators can use Agreement Actions
to automate post-signature tasks.

Docusign eSignature administrators can use Agreement Actions to automate post-signature tasks
for a completed envelope. When Agreement Actions is linked to Docusign CLM, you can
create agreement rules and trigger the following actions:

- Archive completed agreement documents to Docusign CLM
- Start a Docusign CLM workflow

Note: To use Agreement Actions with CLM, you must connect CLM to your
Docusign eSignature Admin account. For more information, see [Automate Common Post-Signature Tasks with Agreement
Actions](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=hzy1594673444380.html&_LANG=enus).

Agreement Actions supports the following actions for CLM:

- Archive to
- Start Workflow

Note: For more information about agreement rules, actions, and
conditions, see [Create and Manage
Agreement Rules](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=btt1594673504460.html&_LANG=enus).

Use cases for Agreement Actions in CLM workflows are as follows:

- Setting attributes for a completed envelope or agreement document
- Kicking off post-signature processes downstream

These use cases are typically configured together as an overall workflow solution.
