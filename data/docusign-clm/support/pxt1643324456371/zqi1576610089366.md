# Reset a User's Password in CLM.CM

Learn how to reset a user's password in CLM.CM. Super administrators or user
administrators can reset a password for a user.

CLM.CM super administrators or user administrators
can reset a password for a user.

1. Log on to CLM.CM.

   Note: For UAT accounts, use the [CLM.CM](https://loginuat.springcm.com/) UAT
   logon page.
2. Select Admin in the navigation header.

   The Admin sidebar opens.
3. Select Users under User Management.
4. Select the box next to the user whose password you want to reset.
5. Select Manage > Reset Password.

The product sends a *password reset* email to the
user’s email address. The user cannot log on with the previous password.
