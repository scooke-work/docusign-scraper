# Manage Schedules in Docusign CLM

Learn how to manage schedules in Docusign CLM. Administrators can use
schedules for recurring and explicit document-management events.

Administrators can manage schedules in Docusign CLM. Administrators can use schedules for recurring and explicit
document-management events.

1. Log in to Docusign CLM as an administrator.
2. Select Admin in the navigation header.

   An administrative sidebar opens.
3. Select Schedules under Automation Tools.

   The Schedules page opens.
4. Select the checkbox next to the schedule that you want to manage.
5. Edit or delete the schedule as needed:
   - Select Schedule > Edit to make changes to the schedule.
   - Select Schedule > Delete to eliminate the schedule.

   Note: Before deleting a schedule, check your smart rules to ensure
   that the schedule is not in use. If the schedule is used in a smart rule, change the
   rule to point to a different schedule.

You are finished managing schedules.
