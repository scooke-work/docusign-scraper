# Template Mapping in Docusign CLM Document Generation

Learn how templates are created for document generation in Docusign CLM.

Part of setting up your implementation of Docusign CLM is the creation of document generation
templates.

Document generation templates are created in a two-step process. First, a finalized
document is marked up with comments corresponding to fields and field types for document
generation. Next, the markups are cataloged in a spreadsheet that connects the markups
to field types in Docusign CLM. Each field type can be conditionally formatted for user
input, and conditional inclusion or exclusion in a generated document.

Note: These features apply to all versions of Docusign CLM. For
Docusign CLM Essentials, Docusign Professional Services or your implementing partner
will assist in the review and availability of fields.
