# Public Links for Sharing Documents in Docusign CLM

Learn how public links can be used to share documents and resources. Administrators can
use public links to allow non-CLM users to view and download documents.

In Docusign CLM, public links can be used to share documents and content with non-CLM users.
Public links allow non-CLM users to view documents and download them. Links can be set
to expire on a given date or removed manually by an administrator.
