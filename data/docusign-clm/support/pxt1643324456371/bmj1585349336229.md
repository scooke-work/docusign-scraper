# Disable Two-Factor Authentication in CLM.CM

Learn how to disable two-factor authentication in CLM.CM.

In CLM.CM, super administrators can disable two-factor
authentication for all users.

Note: This topic applies to CLM.CM.
To determine which kind of CLM account you have, see [Identify your Docusign CLM Account
Type](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=xzt1631832579827.html&_LANG=enus).

1. Log in to CLM.CM as a super administrator.
2. Select Admin in the navigation header.

   An administrative sidebar opens.
3. Select Security Settings under Account Management.

   The Security Settings page opens.
4. Select No in the Enable 2-Step
   Verification section.
5. Select Save.

You have successfully disabled two-factor authentication for
CLM.CM.
