# Switch to Docusign Admin from Docusign eSignature

Learn how to switch from the Docusign eSignature application to Docusign Admin.
Administrators can switch applications to manage CLM users and groups for Docusign CLM
accounts.

## Before you begin

You must perform the steps in the topic [Create an Organization from eSignature Admin](https://support.docusign.com/s/document-item?bundleId=pxt1643324456371&topicId=rbi1585149742539.html&_LANG=enus "Learn how to create an organization from eSignature Admin. Creating this organization allows administrators to manage CLM users and groups and manage other CLM settings.").

As an organization administrator, you can set up or manage CLM users and groups for Docusign CLM accounts. To do so, switch from your eSignature application to Docusign Admin.

1. Log in to your Docusign eSignature account.
2. Select the app switcher menu in the global header.

   ![admin-global-header](https://docusign-be-prod.zoominsoftware.io/api/bundle/pxt1643324456371/page/images/adminglobalheader.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTg1MTAsInNoZWFmIjoicHh0MTY0MzMyNDQ1NjM3MSJ9.Ke4nJijfoboyj7wg3ZuKaMlhYYzDwsjXhIQYGJkOvaU&_LANG=enus "admin-global-header")
3. Select Admin from the application list.

   The Docusign Admin application opens. Your organization administrator
   credentials log you into all Docusign applications. You can now manage
   groups and users from Docusign Admin.
