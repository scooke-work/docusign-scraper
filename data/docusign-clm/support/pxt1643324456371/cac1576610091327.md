# Create a Group in a CLM.CM Account

Learn how to create a group in a CLM.CM account. Administrators can specify access
through groups to help simplify the security access process.

In CLM.CM, super administrators can create or modify groups. Specifying access through
groups can help simplify security access. This procedure describes how to create a
group.

Note: You cannot add contacts to groups.
Contacts are not allowed in a group.

1. Log in to CLM.CM.
2. Select Admin in the navigation header.

   The Admin sidebar opens.
3. Select Groups under User
   Management.

   The Groups page opens.
4. Select New Group.

   The Create a New Group page opens.
5. Enter a name in the Group Name field.
6. Select Add Group Members.

   The Add Group Members window opens.
7. Select each member:
   1. (Optional) Select filter options to narrow down the roles.
   2. Type the name of a user or group in the Search field, and
      select an entry.

      Note: Selecting a group adds all users from that group to your
      new group.

      The product selects the member for the group.
8. Select Apply.

   The product adds the selected members to the group.
9. Select Save.

   The product creates the group.
