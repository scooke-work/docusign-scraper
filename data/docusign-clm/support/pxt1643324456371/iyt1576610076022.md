# Unassign a Task in Docusign CLM

Learn how to unassign a task in Docusign CLM.

## Before you begin

- Select Enable updated task management page on the
  Account Settings page. The updated page has a more modern
  layout.
- To perform this activity on tasks for other task group members, users must have the
  proper permissions. For more information, see the documentation about [user permissions that administrators can assign to
  task group members](https://support.docusign.com/s/document-item?bundleId=yks1643320936212&topicId=rij1684257629881.html&_LANG=enus).

You can unassign tasks. This process helps you manage
workloads and manage business processes.

1. Log in to Docusign CLM.
2. Select Tasks in the navigation header.

   The task management page opens.
3. Select a task view.
4. Select the checkbox next to the task that you want to unassign.
5. Select Unassign.

   A prompt opens, requesting you to confirm the action.
6. Select Unassign.

You have unassigned the task.
