# Log in to CLM.CM with Two-Step Verification

Learn how to log in to CLM.CM by using two-step verification.

## Before you begin

You must [set up two-step
verification](https://support.docusign.com/s/document-item?bundleId=pxt1643324456371&topicId=fnt1585347689093.html&_LANG=enus "Learn how to set up two-step verification as a CLM.CM user.").

With two-step verification enabled, you must verify a code through your Google Authenticator
app when you log in to CLM.CM.

1. Specify your credentials on the login page for your CLM.CM account, and select
   Log In.

   The product prompts you to enter a six-digit verification code from your Google
   Authenticator app.
2. Find the code in the Google Authenticator app, enter the code on the
   Two-step verification page in CLM.CM, and select
   Log In.

The product logs you on. Each time you log on, you must get a
verification code from the Google Authenticator app.
