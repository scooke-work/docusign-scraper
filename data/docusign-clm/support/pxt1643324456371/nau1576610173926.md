# Convert an eForm from Public to Private in CLM.CM

Learn how to convert a public eForm into a private eForm in CLM.CM.

In CLM.CM, all eForms are private by default. If you
have public eForms, super administrators can convert those eForms from public to private.

Note: This topic applies to CLM.CM. To determine which kind
of CLM account you have, see [Identify your Docusign CLM Account Type](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=xzt1631832579827.html&_LANG=enus).

1. Log in to CLM.CM as a super administrator.
2. Select Admin in the navigation header.

   An administrative sidebar opens.
3. Select eForms under Automation Tools.
4. Select the checkbox for the eForm that you want to modify.
5. Select Access > Make Private.

   You have converted a public eForm into a private eForm.
