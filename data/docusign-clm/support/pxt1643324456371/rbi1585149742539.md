# Create an Organization from eSignature Admin

Learn how to create an organization from eSignature Admin. Creating this organization
allows administrators to manage CLM users and groups and manage other CLM
settings.

## Before you begin

You must be a Docusign eSignature account administrator with
the *All Administration Capabilities* permissions. The account that you use to
create an organization must be enabled with Docusign Admin.

To create a dedicated CLM
account in Docusign Admin, you must create an organization. Creating this
organization gives you the ability to create and manage CLM users and groups. You
can also manage other CLM settings.

1. Log in to Docusign eSignature.
2. Select **Admin** in the header.
3. Select **Get Started** to launch the organization setup process.

   ![eSignature Admin - Create an Org](https://docusign-be-prod.zoominsoftware.io/api/bundle/pxt1643324456371/page/images/eSignature-Admin--Create-an-Org.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTg1MDYsInNoZWFmIjoicHh0MTY0MzMyNDQ1NjM3MSJ9.5wZ9ZAACACjgtokDQ_OjKaAUfkZZf-yAGTqfjIF0SiI&_LANG=enus "eSignature Admin - Create an Org")

   Note: New Docusign customers might not see
   the Get Started widget during the first log on to eSignature Admin. If not,
   log off and log back in to refresh the server. Existing Docusign customers
   might not see the Getting Started widget. If not, clear your cache and then
   log back in to eSignature.
4. Enter a name in the Organization Name field.
5. (Optional) Enter a description for the organization.
6. Select Next.
7. Select the accounts that you want to add under this organization.

   Note: The account that you
   used to create the organization is automatically linked. Select any
   additional accounts that you want to manage centrally through your
   organization. The list shows all accounts for which you are an account
   administrator with All Administration Capabilities permissions. You can link
   additional accounts at any time.
8. Select Create.

The product creates the organization and links your selected
accounts.
