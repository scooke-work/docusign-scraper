# Define a Group's Security Access to a Folder

Learn how to give a group access to a folder. Certain administrators or users with Set
Access rights can set up the group's security access to the folder.

## Before you begin

To manage security on a folder, you must be authorized as
follows:

- (CLM.CM) Super administrator or user who is granted Set Access rights to the
  folder.
- (Docusign CLM) a CLM administrator

You can give a group access to folders and documents within those folders. You select the
level of security access for the group.

1. Select Documents in the navigation header.
2. Navigate to the folder for which you want to define access.
3. Select the box next to the folder.
4. Select Manage > Security.

   The Folder Security page opens.
5. Type the group name in the text field at the bottom, and select the name from the
   results drop-down list.

   Tip: You can also select the name from the address
   book.

   The product adds the name to the table of user roles and assigned security
   settings.
6. Select a security level from the Security drop-down list for the group.

   Members of this group will have this security level access.
7. Select Save.

You are finished setting up the folder access for the
group.
