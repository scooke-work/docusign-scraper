# User Management in Docusign CLM and CLM.CM

Read about user management in Docusign CLM and CLM.CM. Administrators can manage users
to control how the users interact with the product. The management methods depend on the CLM
account type.

A user is a person who logs in to access CLM documents. A user requires a Docusign CLM
license in your account. Based on their permission profile, a user can potentially perform any
of the following activities:

- View, add, edit, and share documents
- Administer the account

Administrators can manage users to control how the users interact with the product. The way
that administrators manage users depends on the CLM account type:

- For Docusign CLM account types, you manage users from Docusign Admin.
- For CLM.CM account types, you manage users on the Users page in the
  Admin section of the CLM application.

Note: To see what CLM account type you have, see the topic [Identify Your Docusign CLM Account Type](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=xzt1631832579827.html&_LANG=enus).
