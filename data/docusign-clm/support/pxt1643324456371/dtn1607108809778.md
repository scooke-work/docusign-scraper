# Enable the Updated Task Management Page in Docusign CLM

Learn how to enable the updated task management page in Docusign CLM.
Administrators can enable this page, which has a modern layout.

Administrators can enable the updated task management
page in Docusign CLM. This page has a modern layout.

1. Log in to Docusign CLM as an administrator.
2. Select Admin in the navigation header.

   An administrative sidebar opens.
3. Select Account Settings under Account Management.

   The Account Settings page opens.
4. Select Enable updated task management page under the
   Workflow & Tasks section, and select
   Save.

The updated task management page is now enabled. Users see the
updated page when they select Tasks in the navigation header in
Docusign CLM.
