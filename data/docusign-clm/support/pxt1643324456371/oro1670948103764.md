# Custom Clause Libraries in Docusign CLM

Learn about clause libraries for Docusign CLM. Administrators can create clause
libraries for use in document generation.

In Docusign CLM, a clause library contains pre-approved language to use during document
creation. An administrator can create a custom clause library. Docusign CLM users can
then access the library in several ways. For example, you can use clause libraries to
create document generation forms in Docusign CLM.

Note: Users can also access CLM clause libraries in Microsoft Word. For more
information on using your Docusign CLM Clause Library in Microsoft Word, see [Docusign CLM Clause Library Add-In for Microsoft
Office](https://support.docusign.com/s/document-item?bundleId=alb1642965087302&topicId=ceq1670951972419.html&_LANG=enus).
