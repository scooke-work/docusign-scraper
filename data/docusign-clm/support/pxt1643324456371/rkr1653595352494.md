# Remove Public Share Links on a Folder in Docusign CLM

Learn how to disable public share links on a folder that you previously shared in Docusign CLM.

Docusign CLM administrators can
create public share links to allow non-CLM users to view documents and content on your
account. If a public share link is no longer needed, a Docusign CLM
administrator can remove it.

Note: You must be a Docusign CLM administrator or have delegated permissions to create
public share links. For more information about delegated permissions, see [Limited Administrative Permissions with
Delegated Permissions in Docusign CLM](https://support.docusign.com/s/document-item?bundleId=pxt1643324456371&topicId=bra1576610051531.html&_LANG=enus).

1. Log in to Docusign CLM as an administrator.
2. Select Documents in the navigation header.

   The folder list displays.
3. Select the drop-down menu for the folder, and select Send  >  Share Link.

   A window opens that gives you access to any public links.
4. Select View Active Links.

   Another window opens to display any public links that exist.
5. Remove the public share link:
   1. Select the X icon for the public link that you
      want to remove
   2. Select Continue to disable public access to the
      link.

      The product removes the link.

You have removed a public share link for a folder in Docusign CLM.
