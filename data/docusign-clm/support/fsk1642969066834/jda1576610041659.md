# View Document History in Docusign CLM

Learn how to view the history of a document in Docusign CLM. Administrators and users
can see any changes that occurred on a document across revisions.

In Docusign CLM, users can see the history of a
selected document. This feature enables users to see any changes that occurred on a document
across revisions.

1. Log on to Docusign CLM.
2. Select Documents in the navigation header.

   The Documents page opens.
3. Navigate to the folder containing the document that you want to view.
4. Select the checkbox for that document, and select View > History.

   The document history page opens.
5. (Optional) Perform any of the following actions to manipulate the history data:

   - Sort entries by selecting the column header for any field.
   - Filter activity records by selecting the checkboxes for document action types.
6. Select Done to return to the document list.

   You have viewed the history of a selected document.
