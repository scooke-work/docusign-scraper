# Check-Out, Edit, and Check-In a Document in Docusign CLM

Learn how to check-out a document for edits and upload a new version for check-in in
Docusign CLM. Checking-out a document prevents other users from editing it
for the duration of the check-out. Checking-in a document uploads a new version of the document
to Docusign CLM.

Checking-out a document In Docusign
CLM prevents other users from editing or changing the selected document. Once the changes to
the document have been completed, users should check-in the document to unlock it. Checked-in
documents are saved to Docusign CLM as a new version of the document.

Note:
Docusign Edit can also be used to check-out and make
changes to a selected document. For more information, see [Edit Documents With Docusign Edit](https://support.docusign.com/s/document-item?bundleId=alb1642965087302&topicId=vhz1658776840526.html&_LANG=enus).

1. Log on to Docusign CLM.
2. Select Documents from the Navigation Header.

   The Documents screen displays.
3. Select the checkbox for the document that you want to check-out.
4. Select Edit  >  Check Out.

   A notification appears, indicating that you have checked-out the selected
   document. The ![Locked document](https://docusign-be-prod.zoominsoftware.io/api/bundle/fsk1642969066834/page/images/Lock.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTg0NTQsInNoZWFmIjoiZnNrMTY0Mjk2OTA2NjgzNCJ9.Tz1G96k7fONGAcvkZE-w4BgEZ9bWnnNB3TRFejrrk94&_LANG=enus "Locked document") icon appears next to the document name, indicating that it has been
   checked-out.
5. (Optional) Select File  >  Download to download a copy of the selected document to your computer.

   Note: A document can also manually be checked-in by selecting
   Cancel Check Out.
6. Once all changes have been made, select Edit  >  Check In.

   The Check In screen displays.
7. Select your options:

   - To upload a new copy of the selected document, select Upload,
     and choose a document.
   - To select a document that is already in Docusign CLM, choose
     Select, and select the checkbox for the document.
8. (Optional) Add a comment in the Description field for version
   history.
9. Select Save.

   A message displays, indicating that check-in was successful. You have
   successfully checked-out and checked-in a document in Docusign CLM. The
   document will be unlocked for editing.
