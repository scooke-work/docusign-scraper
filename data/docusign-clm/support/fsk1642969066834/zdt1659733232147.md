# Manage Language and Region Settings in Docusign CLM

Learn how to manage language and region settings in Docusign CLM. You can update the
sending and signing language for documents. You can also update the regional settings for
time zone, date, address, and more.

Users can update their language and region settings in
Docusign CLM. You can update the sending and signing language for documents. You can
also update the regional settings for time zone, name, date, address, and
more.

1. Log in to Docusign CLM.
2. Select the user profile icon in the top right corner, and select
   Manage Profile.

   The My Profile page opens.
3. Select Language & Region.
4. Update the sending and signing language for documents:
   1. Select Update in the
      Language section.
   2. Select the languages for sending and signing documents, and select
      Save.
5. Update the region settings:
   1. Select Update in the Regional
      Settings section.
   2. Update settings and format for region, time zone, date and time, name
      display, intials, and address.
   3. Select Save.

You have successfully updated your language and region
settings in Docusign CLM.
