# Merge PDFs in Docusign CLM

Learn how to merge PDF documents in Docusign CLM.

In Docusign CLM, you can merge two
or more PDFs into a single PDF document. PDFs are merged in the order that they appear in
Document Review.

1. Log in to Docusign CLM.
2. Select Documents in the navigation header.

   The Documents page opens.
3. Select the checkboxes for the PDFs that you want to merge.
4. Select File > Merge as PDF.

   The Merge Documents window opens.
5. Set up your merge as needed:
   1. Specify a name for the merged file.
   2. (Optional) Drag and drop documents to arrange them in the order that you want them
      merged.
   3. (Optional) Select Delete source documents after merge completes
      successfully.
   4. (Optional) Select the Select Folder link, and choose a
      different folder location for the merged PDF.

      The target folder gets displayed.
6. Select Merge.

   A message confirms that the documents were submitted for merging.
7. Refresh the product page until you see the merged PDF.

You have successfully merged PDF documents in Docusign CLM.
