# Troubleshooting Document and Folder History in Docusign CLM

Learn how to troubleshoot error messages for document and folder history in Docusign CLM.

You receive the message: "There are no history records matching your current filters. Please
broaden your criteria to see more history."

## No History Records for the Document or Folder

This message is displayed when there are either:

- No history records for the document or folder history page you have
  selected
- Filters have been selected that return no results

To address this, you can try any of the following:

1. Check or uncheck all filter boxes available to display all available
   history records.
2. Confirm that there has been activity in the folder or on the document
   in question.
3. Verify with your Docusign CLM administrator that you have permission to
   view history records for that folder or document.
4. Review Folder History Actions to confirm that the history events are
   ones that are being tracked. For more information about Folder History
   Actions, see [Folder History Actions in Docusign CLM](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=hks1576610042319.html&_LANG=enus "Learn about each of the logged folder history events in Docusign CLM.").
