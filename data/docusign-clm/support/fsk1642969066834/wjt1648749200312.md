# Add an Attribute Field Filter to Advanced Search in Docusign CLM

Learn how to add an attribute field as a search filter to Docusign CLM Advanced
Search.

## Before you begin

This task applies to the legacy search experience. By
default, the [modernized search experience](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=jbg1707491822735.html&_LANG=enus "Read about modernized search in Docusign CLM. Administrators and users can search for documents and content. You can apply filtering, save searches, customize search results, and export results.") is
enabled in account settings. We recommend using modernized search.

Note: Advanced search is available in CLM and CLM+ but not in CLM
Essentials.

You can add attribute fields as search filters in Docusign CLM advanced search.
Attributes you add as a search filter are only displayed in your CLM instance.

1. Navigate to advanced search:
   1. Select Search
      ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/fsk1642969066834/page/images/Search.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTgzNTcsInNoZWFmIjoiZnNrMTY0Mjk2OTA2NjgzNCJ9.DqiERwZS5KwuZt7tEplZKFacx-Dn-SdHAsjMZE0tp9c&_LANG=enus) from the
      global navigation header.
   2. Select Go to Advanced Search.

   The Search Results page displays.
2. Select Add Attribute Field from the
   Filters tab on the side panel.

   The Select Attribute Fields dialog box
   displays.
3. Select an attribute to add as a search filter.
4. Select Apply.

The attribute field is added to the
Filters tab, and you can search using the attribute.
