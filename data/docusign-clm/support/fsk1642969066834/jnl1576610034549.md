# Resume a Document Upload in Docusign CLM

Learn how to resume a document upload in Docusign CLM. For example, you can resume an
upload that was interrupted because of a network issue.

Document uploads might get interrupted. For example,
a browser timeout or network issue may cause an interruption. You can resume exactly where the
upload stopped.

Note: You can resume only when the original upload method
was drag-and-drop or the File > Upload menu action. The browser must support HTML5. If the following conditions are
not met, the upload starts from the beginning (rather than resuming):

- The document name, file size, and modified date must match the partially uploaded
  document.
- The document must be uploaded by the same user and from the same browser as the
  partially uploaded document.
- You must resume the upload within 72 hours.

1. Open the web browser that you previously used to perform the upload.
2. Log on to Docusign CLM.
3. Navigate to the folder where you want to upload the document.
4. Upload the document through drag-and-drop or the File > Upload menu action.

The resumed file uploads into the selected folder.
