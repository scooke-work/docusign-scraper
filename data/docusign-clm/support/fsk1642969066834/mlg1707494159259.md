# Export Search Results from Docusign CLM

Learn how to export search results from Docusign CLM. Exporting search results lets
administrators and users save or share the information as needed.

## Before you begin

The Enable Modernized Search Experience setting is selected by
default in account settings. You cannot perform this task if an administrator disables
the setting.

After performing a search in Docusign CLM, you can export the search results to your computer
as a CSV file. Exporting search results lets you save or share the information as
needed.

Note: The CSV file contains the columns that are configured on the search
results page. To make adjustments, you can customize columns on the search results
page.

1. Log in to Docusign CLM.
2. Select the ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/fsk1642969066834/page/images/Search.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTgzMzQsInNoZWFmIjoiZnNrMTY0Mjk2OTA2NjgzNCJ9.0bXAMSRLA2erA1COmG8DjzDCQiiSap5kIMZblX_ioRU&_LANG=enus) icon
   in the navigation header.

   The product opens a search window. The window also shows any recent search queries,
   recent documents, and saved searches.
3. [Execute a search by entering search criteria
   and optional filters.](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=xto1707493278474.html&_LANG=enus "Learn how to use a modernized search to find documents and content in Docusign CLM. Administrators and users can also apply filters or save a search query for later use.")

   The product displays search results.
4. Export results by performing *one* of the following actions:
   - Export all results by selecting
     ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/fsk1642969066834/page/images/Overflow-Vertical.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTgzMzQsInNoZWFmIjoiZnNrMTY0Mjk2OTA2NjgzNCJ9.0bXAMSRLA2erA1COmG8DjzDCQiiSap5kIMZblX_ioRU&_LANG=enus)
      > Export as CSV.
   - Export specific results by selecting the checkbox for each listed item
     and selecting Export as CSV above the list.

You have exported the search results to your computer as a
CSV file.
