# Export Search Results in Docusign CLM

Learn how to export search results from Docusign CLM. Exporting search results lets you
save or share the information as needed.

## Before you begin

This task applies to the legacy search experience. By
default, the [modernized search experience](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=jbg1707491822735.html&_LANG=enus "Read about modernized search in Docusign CLM. Administrators and users can search for documents and content. You can apply filtering, save searches, customize search results, and export results.") is
enabled in account settings. We recommend using modernized search.

After performing a search in Docusign CLM, you can export the search results to your computer
as a CSV file. Exporting search results lets you save or share the information as
needed.

Note: The downloaded CSV file contains the columns as they are
configured on the Search Results page. To learn about
customizing the columns in search results, see [Customize Columns on the Search Results Page in Docusign CLM](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=jjn1648752647865.html&_LANG=enus "Learn how to customize the columns on the Search Results page in Docusign CLM.").

1. Log on to Docusign CLM.
2. Navigate to the Search Results page:
   1. Select ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/fsk1642969066834/page/images/Search.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTgzNTIsInNoZWFmIjoiZnNrMTY0Mjk2OTA2NjgzNCJ9.IQlqJ_U75RSvTpgC6McxGgU6SBbQglyzYbeCWlFWCWU&_LANG=enus) in
      the navigation header.
   2. Select Go to Advanced Search.

   The Search Results page opens.
3. Perform a search:
   1. Complete fields on the Filters panel to set up
      your search criteria.
   2. Select Search.

      The product displays search results.
4. (Optional) Select Customize Columns
   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/fsk1642969066834/page/images/Gear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTgzNTIsInNoZWFmIjoiZnNrMTY0Mjk2OTA2NjgzNCJ9.IQlqJ_U75RSvTpgC6McxGgU6SBbQglyzYbeCWlFWCWU&_LANG=enus), and[customize the columns for the search
   results](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=jjn1648752647865.html&_LANG=enus "Learn how to customize the columns on the Search Results page in Docusign CLM.").
5. Select Actions > Export Search Results.

The product exports the search results to your computer as a CSV
file.
