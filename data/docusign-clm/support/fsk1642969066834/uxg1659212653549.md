# Profile Management in Docusign CLM

Learn how to manage your profile details in Docusign CLM.

In Docusign CLM, your profile contains your user information. You can change your profile
picture, name and signature style, language and region controls, and apps connected to
your Docusign account.

By updating these options, you can customize your Docusign CLM account.

My Profile
:   Select this option to upload or change your profile picture, display name, email
    address, and contact information in Docusign CLM. For more information about
    configuring profile options, see [Update My Profile in Docusign CLM](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=sgp1659723990769.html&_LANG=enus).

Privacy & Security
:   Select this option to configure logon security, account recovery options, device
    security, and certificates for your Docusign CLM account. For more information
    about configuring privacy and security options, see [Update Your Privacy & Security Settings in
    Docusign CLM](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=zsa1659726257051.html&_LANG=enus).

Connected Apps
:   Select this option to view all applications that are connected to your Docusign CLM account. For more information about connecting apps to Docusign, see [Connect Document Sources and Manage Other
    Connected Applications](https://support.docusign.com/s/document-item?bundleId=jux1643235969954&topicId=qsm1578456669396.html&_LANG=enus).

Signatures
:   Select this option to add or update your name and signature styles for Docusign CLM. For
    more information about signature styles in Docusign CLM, see [To Create or Edit a
    Signature](https://support.docusign.com/s/document-item?bundleId=jux1643235969954&topicId=lrz1578456442739.html&_LANG=enus).

Stamps
:   Select this option to add or update your stamp styles for Docusign CLM. For more
    information about stamps in Docusign CLM, see [Manage Stamps](https://support.docusign.com/s/document-item?bundleId=jux1643235969954&topicId=dys1578456666697.html&_LANG=enus).

Language & Region
:   Select this option to configure your language and region settings for Docusign CLM. For more
    information about language and region settings, see [Update Language & Region
    Settings in Docusign CLM](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=zdt1659733232147.html&_LANG=enus).
