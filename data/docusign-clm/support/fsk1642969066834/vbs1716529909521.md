# Search for Folders in Docusign CLM

Learn how to search for folders in Docusign CLM. You can use these
steps to add a filter and search for folders.

## Before you begin

The Enable Modernized Search Experience setting is selected
by default in account settings. You cannot perform this task if an administrator
disables the setting.

In Docusign CLM modernized search, you can search for folders. Folder
searches can be useful for exploring the contents of folders. For example, you can
find a vendor’s folder and see which documents are associated with the vendor.

Tip: After searching for folders, you can save the search query
for future reuse.

1. Log in to Docusign CLM.
2. Select the ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/fsk1642969066834/page/images/Search.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTgzNDMsInNoZWFmIjoiZnNrMTY0Mjk2OTA2NjgzNCJ9.KAa767Jmy4t2O5yTIrkwjuB0p89cg4GaScmGFYbjgOU&_LANG=enus) icon
   in the navigation header.

   The product opens a search window. The window also shows any recent search queries,
   recent documents, and saved searches.
3. [Execute a search by entering search
   criteria](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=xto1707493278474.html&_LANG=enus "Learn how to use a modernized search to find documents and content in Docusign CLM. Administrators and users can also apply filters or save a search query for later use.").

   The product displays search results.
4. Apply a filter to display folders in the results:
   1. Select Change above the search results.
   2. Select one of the following options:

      - Documents and Folders
      - Folders Only

      The product updates the search results.
5. (Optional) [Refine the search by entering
   search criteria and optional filters](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=ops1707494327789.html&_LANG=enus "Learn how to apply filters to search results in Docusign CLM. Administrators and users can apply filtering to refine and narrow the results.").

   The product displays updated search results.
6. Select a folder name in the Name column to open the folder
   location.

The product opens the folder location in a separate browser window. You have completed a search for folders in Docusign CLM.
