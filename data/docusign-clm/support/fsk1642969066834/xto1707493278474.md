# Search for Documents and Content in Docusign CLM

Learn how to use a modernized search to find documents and content in Docusign CLM.
Administrators and users can also apply filters or save a search query for later
use.

You can search for documents and content in Docusign CLM. The modernized search interface
provides an efficient, flexible, and personalized search experience.

Note: Modernized search is not available for CLM.CM account types.
This search experience is available only for Docusign CLM accounts. On the
Account Settings page, the Enable Modernized
Search Experience setting is selected by default.

You can also apply filters to search for specific content properties or attributes.
For example, you can filter results to show only documents that were created in the
past seven days.

Tip:

- Users and administrators can save a search query for later use. Administrators can share
  the saved search with all users; however, each user who executes the shared
  search might get different results, depending on permissions.
- To search for cascading attribute values, you can select the values from
  drop-down filters. The value must be an exact match to a value in the
  corresponding input file.

## Example

**Search for a Customer’s Contract to Find a Reusable
Clause**Alex is a contract manager at Fontara. He needs to create a new contract with
existing customer Nesis Biotech. Alex needs to find an existing Nesis Biotech contract
and reuse one of its clauses. Alex filters the search to show only active contracts for
the Nesis Biotech party.He finds the target contract and opens it directly from
the search results page. He finds the product warranty clause and copies the content.
Alex reuses the clause in his draft of the new services agreement contract.
