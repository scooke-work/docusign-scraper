# Search in Docusign CLM

Read about search in Docusign CLM. You can execute searches, apply filtering, save
searches, customize search results, and export search results.

In Docusign CLM, administrators and users can search for documents and content.

Note: The default search behavior is the [modernized search experience](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=jbg1707491822735.html&_LANG=enus "Read about modernized search in Docusign CLM. Administrators and users can search for documents and content. You can apply filtering, save searches, customize search results, and export results."). On the Account
Settings page, the Enable Modernized Search Experience
setting is selected by default. If an administrator disables the setting, the product provides
a [legacy search experience](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=tgh1708707979539.html&_LANG=enus "Read about legacy search in Docusign CLM. Administrators and users can search for documents and content. You can apply filtering, save searches, customize search results, and export results."). We recommend using the
modernized search.

When performing searches, you can also perform any of the following activities:

- Apply filtering to refine a search
- Save a search query
- Customize columns for search results
- Export search results
- Download documents from search results
