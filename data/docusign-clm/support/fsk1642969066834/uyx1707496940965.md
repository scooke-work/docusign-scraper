# Download Documents from Search Results in Docusign CLM

Learn how to download documents from search results in Docusign CLM. Administrators and
users can download documents for relevant business needs.

## Before you begin

The Enable Modernized Search Experience setting is selected by
default in account settings. You cannot perform this task if an administrator disables
the setting.

You can download up to 25 documents from the search results page in Docusign CLM. You can
download documents in their native format. You can also download a document as a PDF
when the document is a PDF, DOC, DOCX, PPT, PPTX, XLS, or XLSX file. If you download
multiple documents, they get downloaded in a ZIP file.

1. Log in to Docusign CLM.
2. Select the ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/fsk1642969066834/page/images/Search.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTgzMzksInNoZWFmIjoiZnNrMTY0Mjk2OTA2NjgzNCJ9.hB4gvcVHYa7HCPJbLSBGOW53JugbkB4PCLnv_7k4ZCg&_LANG=enus) icon
   in the navigation header.

   The product opens a search window. The window also shows any recent search queries,
   recent documents, and saved searches.
3. [Execute a search by entering search criteria
   and optional filters](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=xto1707493278474.html&_LANG=enus "Learn how to use a modernized search to find documents and content in Docusign CLM. Administrators and users can also apply filters or save a search query for later use.").

   The product displays search results.
4. Select the checkboxes next to target documents, and perform any of the
   following actions:
   - Select Download > Document to download documents in their native format.
   - Select Download > Document as PDFs for PDF, DOC, DOCX, PPT, PPTX, XLS, or XLSX
     documents.

The product downloads the selected documents. If you download
multiples files, the files get downloaded together in a ZIP file. You are finished
downloading your documents.
