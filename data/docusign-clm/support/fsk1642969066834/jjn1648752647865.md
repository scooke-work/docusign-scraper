# Customize Columns on the Search Results Page in Docusign CLM

Learn how to customize the columns on the Search Results page in Docusign CLM.

## Before you begin

This task applies to the legacy search experience. By
default, the [modernized search experience](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=jbg1707491822735.html&_LANG=enus "Read about modernized search in Docusign CLM. Administrators and users can search for documents and content. You can apply filtering, save searches, customize search results, and export results.") is
enabled in account settings. We recommend using modernized search.

You can customize the columns that are displayed on the Search Results
page in Docusign CLM. You can also add account attributes as columns. Column
customizations display only in your instance of CLM. The customizations persist
until the next time you customize them.

1. Log in to Docusign CLM.
2. Navigate to the Search Results page:
   1. Select Search
      ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/fsk1642969066834/page/images/Search.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTgzNjIsInNoZWFmIjoiZnNrMTY0Mjk2OTA2NjgzNCJ9.54aWC8VMdnUqN60-8OZntG9RkU_o1OFG_iBjnUKLDPw&_LANG=enus) in the
      navigation header.
   2. Select Go to Advanced Search.

   The Search Results page opens.
3. Select Customize Columns
   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/fsk1642969066834/page/images/Gear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTgzNjIsInNoZWFmIjoiZnNrMTY0Mjk2OTA2NjgzNCJ9.54aWC8VMdnUqN60-8OZntG9RkU_o1OFG_iBjnUKLDPw&_LANG=enus).

   The Customize Columns dialog opens.
4. (Optional) Reorder columns by dragging columns and dropping them in the
   organization.
5. (Optional) Remove columns by deselecting the checkbox next to a column
   name.
6. Add columns:
   1. Select Add Columns.
   2. Select an attribute to add.
   3. Select Apply.

      The attribute is added as a column.
7. Select Apply to save your customizations.

The Search Results page gets refreshed. The
page now displays the column customizations that you configured.
