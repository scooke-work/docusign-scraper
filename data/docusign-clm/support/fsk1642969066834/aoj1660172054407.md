# Move and Copy Documents in Docusign CLM

Learn how to move and copy documents within folders in Docusign CLM.

In Docusign CLM, users can move documents between
locations; cut and paste documents between locations; and copy and paste documents
between folders.

Use this procedure to move and copy one or more documents in Docusign CLM

Tip: In addition to the method that this
procedure describes, other methods exist as follows:

- You can move documents by selecting and dragging them between folders on the
  Documents page.
- You can select the checkboxes for documents on the
  Documents page, and select the
  Cut, Copy, or
  Paste options from the Edit menu.

1. Log on to Docusign CLM.
2. Select Documents in the navigation header.

   The Documents page opens.
3. Select the checkbox for each document to manage.
4. Select File > Move or Copy.

   The Move or Copy page opens.
5. Select the Select link and choose a folder destination

   The product shows your selection in the
   Destination field.
6. Perform one of the following actions to move or copy documents:

   - Select Move to move the selected documents to the
     selected folder.

   - Select Copy to copy the selected documents to the
     selected folder.

   You have moved or copied selected documents to a new
   location.
