# Edit a Widget on Your Dashboard in Docusign CLM

Learn how to edit widgets on your dashboard in Docusign CLM.

You can make changes to widgets on a Docusign CLM
dashboard.

1. Locate the widget you want to edit.
2. In the widget, select Edit Widget from the
   Edit Properties
   ![Edit Properties](https://docusign-be-prod.zoominsoftware.io/api/bundle/fsk1642969066834/page/images/Overflow-Horizontal.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTgzMDMsInNoZWFmIjoiZnNrMTY0Mjk2OTA2NjgzNCJ9.87uY0Tv-Aln49ejCFf1f26qoPhg6wwIc0dcHB_ihrec&_LANG=enus "Edit Properties") dropdown.
3. Make any necessary changes, and select Save.

You have edited the widget on your dashboard.
