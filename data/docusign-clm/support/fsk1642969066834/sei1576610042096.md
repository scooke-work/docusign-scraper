# Document History Events in Docusign CLM

Learn about each of the logged document history events in Docusign CLM.

Here is a list of logged Document History Actions that will display when document history
is viewed in Docusign CLM:

|  |  |  |  |
| --- | --- | --- | --- |
| - When a document is uploaded - When a document is moved - When a document is previewed - When a document is downloaded - When a document is viewed - When a document is checked out - When a document is checked in - When a description is added, changed, or removed - When a document is renamed - When attributes are added, changed, or removed - When attribute groups are added, changed, or removed - When a document is copied | - When a new document is created - When a document is shared, emailed, or sent - When a document is deleted - When a document is sent for signature - When a document signature request is signed - When a document signature request expires - When a document signature request is rejected - When a document signature request fails - When a document signature request is delegated - When a document checkout is canceled - When a document download justification occurs - When a document bookmark is added or changed | - When a document is renamed due to duplicate handling - When a document Chatter post is submitted - When a document is sent through a workflow process - When a document is sent out for fax - When a document fax submission fails - When a document fax submission succeeds - When a document reminder is scheduled, changed, deleted - When a document reminder is processed - When a document is added to or removed from the Watch List - When a document is sent out for external review - When external review expires, completes, or is canceled - When an external review request is resent | - Document History Filters - Attribute Changes - Document Activity - Document Reminders - Downloads - E-mail Activity - External Review - Fax Activity - Previews - Share Activity - Signatures - Versions - Workflows |
