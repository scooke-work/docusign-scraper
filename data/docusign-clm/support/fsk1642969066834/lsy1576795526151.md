# Configure a Saved Search Using Advanced Search in Docusign CLM

Learn how to save a pre-configured search. Administrators can use these steps to create
a saved search for users.

## Before you begin

This task applies to the legacy search experience. By default,
the [modernized search experience](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=jbg1707491822735.html&_LANG=enus "Read about modernized search in Docusign CLM. Administrators and users can search for documents and content. You can apply filtering, save searches, customize search results, and export results.") is enabled in
account settings. We recommend using modernized search.

Note: Advanced search is available in CLM and CLM+ but not in CLM
Essentials.

Docusign CLM administrators can configure and save searches. After an administrator saves a
search, users can access the search from the Saved Search tab on the
side panel in advanced search.

1. Log in to Docusign CLM.
2. Navigate to advanced search:
   1. Select Search
      ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/fsk1642969066834/page/images/Search.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTgzNjAsInNoZWFmIjoiZnNrMTY0Mjk2OTA2NjgzNCJ9.0-nK8bWm7u6KqQE1ODOsurJTn3fm8J6yA9jn3YU5WVU&_LANG=enus) in the navigation
      header.
   2. Select Go to Advanced Search.

   The Search Results page opens.
3. Configure search filters in the Filters tab on the side
   panel.
4. Select Search to execute the search.

   The product displays search results.
5. Select Actions > Save Search.

   The Save Search dialog opens.
6. Type a name for the search, and select Save.

The product saves the search. This saved search is now available to all users.
