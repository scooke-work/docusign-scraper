# Save a Search Query for Reuse in Docusign CLM

Learn how to save a search in Docusign CLM. Administrators and users can create saved
searches to save time and help with recurring search tasks.

## Before you begin

The Enable Modernized Search Experience setting is selected by
default in account settings. You cannot perform this task if an administrator disables
the setting.

After performing a search, you can save the search query for later use in Docusign CLM. Saving
searches can be helpful for reusing complex or common search queries. Saving
searches can also save time and help with recurring search tasks.

Note:

- Administrators can share a saved search with all users in the account;
  however, each user who executes the shared search might get different
  results, depending on permissions.
- If you saved searches before modernized search became active for the
  account, you must create those saved searches again.

1. Log in to Docusign CLM.

   in the navigation header.

   The product opens a search window. The window also shows any recent search queries,
   recent documents, and saved searches.
3. [Execute a search by entering search criteria
   and optional filters](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=xto1707493278474.html&_LANG=enus "Learn how to use a modernized search to find documents and content in Docusign CLM. Administrators and users can also apply filters or save a search query for later use.").

   The product displays search results.
4. Select Save on the search results page.

   The Save Search dialog opens.
5. Define the properties for the saved search:
   1. Specify a title for the search.
   2. (Optional for administrators) Select Share this search with
      others to share the saved search with all users in the
      CLM account.
   3. Select Save.

The product saves your search. Saved searches get displayed
in a Saved section when you open a search window. You can select a saved search to
execute it. From the results page for a saved search, you can edit the search; however,
any changes must be saved as a new saved search
