# Download Documents from Docusign CLM

Learn how to download documents from Docusign CLM. Documents can be
downloaded in their original format or converted to PDF.

In Docusign CLM, you can download a copy of a stored document to your local machine. You
can download the document in its original format or download it as a PDF.

Documents can also be downloaded when the document is locked or checked out by another
user. In all cases, the copy of the document in Docusign CLM remains
unaffected in its native file format.

1. Log in to Docusign CLM.
2. Select Documents in the navigation header.

   The Documents view opens.
3. Navigate to the folder that contains one or more documents to download.
4. Select the checkbox next to one or more documents, and perform one of the following
   menu selections:

   - Select File > Download to download files in their original format.
   - Select File > Download as PDF to download documents in PDF format.

   The selected document is downloaded to your computer. You have successfully
   downloaded one or more documents from Docusign CLM.
