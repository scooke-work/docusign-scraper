# Delete Documents and Folders in Docusign CLM

Learn how to delete documents and folders in Docusign CLM.

When you no longer want documents or folders, you can
delete these items in Docusign CLM.

Note: Users can delete only the
CLM files and folders to which they have at least view, edit, and delete
access.

1. Log in to Docusign CLM.
2. Select Documents in the navigation header.

   The Documents view opens.
3. Select the checkbox for the document or folder that you want to delete.
4. Select File > Delete.

   A confirmation message displays.
5. Select OK.

The product moves the item to the trash. Deleted items no longer
appear in the Documents view. Deleted documents no longer appear in
search results, unless you have a filter to include the trash in your search.
