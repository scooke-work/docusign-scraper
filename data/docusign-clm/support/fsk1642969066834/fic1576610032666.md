# Document Check-In and Check-Out in Docusign CLM

Read about check-in and check-out of documents for editing or changes in Docusign CLM.

Users can check-in and check-out documents for editing and changes In Docusign CLM. Checked-out documents are locked by a user, preventing edits,
changes, or revisions by other users. Only the person who has checked-out the document can
edit it. Users that have the correct permissions for the document will still be able to view
and download it.

Once all changes have been made, users can check-in the document to save the revised document
to Docusign CLM. Checked-in documents are saved as a new version of that
document, and preserved in the version history in Document Review.

Note:

- Check-ins, check-outs, and document versions are preserved as part of document version
  history in Document Review. For more information on version history and comparing document
  versions, see [Compare Document Versions With Document
  Review in Docusign CLM](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=shn1666363438998.html&_LANG=enus).
- Docusign Edit can be used to check-out and make changes to a selected
  document. For more information, see [Edit Documents With
  Docusign Edit](https://support.docusign.com/s/document-item?bundleId=alb1642965087302&topicId=vhz1658776840526.html&_LANG=enus).
