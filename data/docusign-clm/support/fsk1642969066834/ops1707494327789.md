# Apply Filtering to Refine a Search in Docusign CLM

Learn how to apply filters to search results in Docusign CLM. Administrators and users can apply filtering to refine and narrow the results.

## Before you begin

The Enable Modernized Search Experience setting is selected by default in account settings. You cannot perform this task if an administrator disables the setting.

When performing a search in Docusign CLM, you can use filters to refine the search results. For example, you can filter your search based on file location, contract type, or object type.

You can add the following filters to refine the search results:

- Document metadata from the system

  Examples include agreement ID, created date, document ID, document file type, document location, modified by, modified date, object ID, object type, and party ID.
- Standard document attributes
- Attribute fields from any custom attribute group
- Filters that control whether search results include documents, folders, or both

Note: In modernized search, administrators can hide groups of attributes from searches. Users will not see hidden attributes as filters in searches or in search results. When creating or modifying an attribute group, the administrator can select the Hide attributes from search checkbox. If you hide a group, the Hidden from Search column displays a checkbox for that group on the Attribute Groups page.

   The product opens a search window. The window also shows any recent searches and saved searches.
3. Perform one of the following actions:
   - Execute a search by entering search criteria.
   - Select a recent search query or saved search from the list.

   The product displays search results.
4. (Optional) Activate filters to control whether the results include documents, folders, or both:
   1. Select Change above the search results.
   2. Select one of the following options:

      - Documents Only (enabled by default)
      - Documents and Folders
      - Folders Only

   The product adds the filter and updates the search results.
5. Select ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/fsk1642969066834/page/images/Filter.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTgzMjUsInNoZWFmIjoiZnNrMTY0Mjk2OTA2NjgzNCJ9.FRj1NRgJfl3SWZH5jFZD1mq8TOK80OrVyOSWADae3Ds&_LANG=enus) All Filters on the search results page.

   The All Filters dialog opens.
6. Add each filter:
   1. (Optional) Type text in the Find by filter name text box to search all attribute fields in your CLM account.

      The product lists matching attributes.
   2. Select an attribute, configure its properties, and select Apply.

      Tip: To search for items in the trash folder, select the In trash filter to enable it. You do not need to configure properties for this filter.
   3. (Optional) Select the ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/fsk1642969066834/page/images/Star.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTgzMjUsInNoZWFmIjoiZnNrMTY0Mjk2OTA2NjgzNCJ9.FRj1NRgJfl3SWZH5jFZD1mq8TOK80OrVyOSWADae3Ds&_LANG=enus) (star) icon next to a filter to save it as a "starred" filter.

      The product darkens the icon and adds the filter to the Starred tab. The first four filters on the Starred tab appear as visible filters on any search page or search results page. You can select the visible filter and configure it as needed.

      Note: You can deselect a ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/fsk1642969066834/page/images/Star-Filled.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTgzMjUsInNoZWFmIjoiZnNrMTY0Mjk2OTA2NjgzNCJ9.FRj1NRgJfl3SWZH5jFZD1mq8TOK80OrVyOSWADae3Ds&_LANG=enus) icon next to a filter to remove the filter from the Starred tab. You can also select the ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/fsk1642969066834/page/images/Drag-and-Move.jpg?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTgzMjUsInNoZWFmIjoiZnNrMTY0Mjk2OTA2NjgzNCJ9.FRj1NRgJfl3SWZH5jFZD1mq8TOK80OrVyOSWADae3Ds&_LANG=enus) icon to drag a filter to a new order in the list.
   4. Select Save on the All Filters dialog.

The product adds the filters and updates the search results. You finished applying filtering to your search results. You can apply more filters if needed.
