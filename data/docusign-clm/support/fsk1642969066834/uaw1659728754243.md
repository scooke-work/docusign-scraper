# Upload a Public Key to Your Docusign CLM Account

Learn how to upload a public key to your Docusign CLM account for SFTP
authentication.

You can upload a public key to your Docusign CLM account. Docusign CLM uses this public
key for SFTP authentication and transactions.

1. Log on to Docusign CLM.
2. Select My Profile from the user profile menu.
3. Select Privacy & Security  >  Certificates  >  Manage.

   The Certificates window displays.
4. Select Upload Public Key, and navigate to where your
   public key is stored.

   You have successfully uploaded a public key to Docusign CLM for SFTP
   authentication and transactions.
