# Reset Your Password in Docusign CLM

Learn how to reset your password in Docusign CLM.

Learn how to reset your password in Docusign CLM.

Note:

- This topic covers password reset instructions for both Docusign CLM (CLM.DS) and
  SpringCM (CLM.CM) account types. To see what CLM account type your have, see [Identify Your Docusign CLM Account Type](https://support.docusign.com/s/document-item?bundleId=fsk1642969066834&topicId=xzt1631832579827.html&_LANG=enus "Learn how to identify your account type in Docusign CLM.").
- To learn more about resetting a user password as a CLM administrator, see [Reset User Password](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=lyr1583277364866.html&_LANG=enus).

1. To reset your password for Docusign CLM (CLM.DS):
   1. Log on to Docusign CLM (CLM.DS).
   2. Select My Profile from the user profile menu.
   3. Select Privacy & Security.
   4. Select Default Security  >  Change Password.

      The password change screen displays.
   5. Enter a new password that follows these rules:

      - New password must be at least 6 characters.
      - New password cannot contain spaces, or the following characters: < > ,
      - New password must be different from your last 12 passwords.
   6. (Optional) Select Add Question to add questions for password
      recovery.
   7. Select Save.

      You have successfully changed your password for Docusign CLM
      (CLM.DS).
2. To reset your password for SpringCM (CLM.CM):
   1. Log on to SpringCM (CLM.CM)
   2. Select My Preferences from the user profile menu.
   3. Choose Reset Password, and then select Reset
      Password.

      You will receive an email to verify the password reset request.
   4. Once you receive the password reset email, select the link titled Reset
      Password.

      A webpage displays for resetting your SpringCM (CLM.CM)
      password.
   5. Enter and confirm a new password. Once a password has been entered, select
      Confirm.

      You have successfully changed your password for SpringCM
      (CLM.CM).
