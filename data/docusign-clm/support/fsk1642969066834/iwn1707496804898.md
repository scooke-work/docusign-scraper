# Customize Columns for Search Results in Docusign CLM

Learn how to customize the displayed columns for search results in Docusign CLM.
Administrators and users can customize columns to present the most relevant information as
needed.

## Before you begin

The Enable Modernized Search Experience setting is selected by
default in account settings. You cannot perform this task if an administrator disables
the setting.

You can control which columns get displayed on the search results page in Docusign CLM. You
can add the following items as visible columns in search results.

- Document metadata from the system

  Examples include agreement ID, created
  date, document ID, document file type, document location, modified by,
  modified date, object ID, object type, and party
- Standard document attributes
- Attribute fields from any custom attribute group

Column customizations persist unless you use a different browser. You can also select
a visible column name to sort the search results based on that column. For example,
you can display a column for the Expiration Date attribute. You can then sort the
results based on document expiration dates.

Note:

- You cannot move the Name column.
- You cannot sort results based on the Created Date column.
- If you customize columns and then launch a saved search, the saved search
  retains its original columns.

   The product displays search results.
4. Select the ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/fsk1642969066834/page/images/Gear.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTgzMzIsInNoZWFmIjoiZnNrMTY0Mjk2OTA2NjgzNCJ9.umky_U4jV2rO2cBoGk21zl7O7CTOHlEx9Cuza-yYsvY&_LANG=enus)
   (Customize Columns) icon on the search results page.

   The Columns dialog opens. The dialog lists columns
   that are currently visible on the search results page. Each listed column has a
   switch that is blue and right-aligned.
5. (Optional) Add columns:
   1. Select the Find by name text box.

      The product displays all available items. Items that are already
      visible have checkmarks next to them in the list.
   2. Scroll through the list, or type text in the text box, then select the
      desired item.

      The product adds the selected item to the Visible
      list.
6. (Optional) Hide or rearrange columns by performing any of the following actions
   in the Columns dialog:
   - Hide a column by selecting the right-aligned blue switch to remove the
     column.
   - (Optional) Rearrange a column’s order by selecting the ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/fsk1642969066834/page/images/Drag-and-Move.jpg?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTgzMzIsInNoZWFmIjoiZnNrMTY0Mjk2OTA2NjgzNCJ9.umky_U4jV2rO2cBoGk21zl7O7CTOHlEx9Cuza-yYsvY&_LANG=enus) icon next to the
     column and dragging the column to a new location.

   The product updates the list of columns. You are now ready to save your
   customizations.
7. Select Save to apply your customizations.

The product updates the columns. Your search results page
now shows your column customizations.
