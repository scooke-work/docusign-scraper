# View Folder History in Docusign CLM

Learn how to view the history of a folder in Docusign CLM. Administrators and users use
these steps to see changes on a folder across revisions and additions.

You can see the history of a folder in Docusign CLM.
Administrators and users can see any changes that occurred on a folder across revisions and
additions.

1. Log on to Docusign CLM.
2. Select Documents in the navigation header.

   The Documents page opens.
3. Navigate to the folder that you want to view.
4. Select the checkbox for the folder, and select View > History.

   The folder history page opens.
5. (Optional) Perform any of the following actions to manipulate the history data:

   - Sort entries by selecting the column header for any field.
   - Filter activity records by selecting the checkboxes for action types.
6. Select Done to return to the document list.

   You have viewed the history of a selected folder.
