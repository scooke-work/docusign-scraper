# Set Up the Docusign App Center ServiceNow Instance, Step 5: Assign Application Access to ServiceNow Tables

Learn how to grant Docusign access to ServiceNow tables.

Exchange data between Docusign and ServiceNow with Docusign IAM for ServiceNow. Use the integration to kick off software and hardware
procurement requests and other downstream workflows.

This topic shows how to grant Docusign access to ServiceNow tables.

1. At the top right, select the globe icon. Make sure that
   the application scope is the same as the scope of the table that you are
   granting access to.
2. Select the All tab from the top menu. Search for and
   select tables.

   The Tables page displays.
3. Select a table that you want Docusign to access.

   The details page for the table displays.
4. In the Application Access tab, for the
   Accessible from field, select All
   application scopes.
5. Click the Edit Herebutton. Check the boxes for
   Can read, Can create, and
   Can update and select
   Update.
