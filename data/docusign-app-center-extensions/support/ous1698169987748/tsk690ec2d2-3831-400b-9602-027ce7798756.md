# Access Docusign Workflow Builder Templates in the App Center

Learn how to select and initiate a Docusign Workflow Builder template from the Docusign App Center page.

You can select and initiate a Docusign Workflow Builder template from the Docusign App Center page.

1. Access workflow templates using one of these methods to display available templates:

   - Featured
     Workflow templates section: Select an available workflow
     template from the list or by selecting See All Templates.

     ![A screenshot showing an sample Workflow Builder template from the Featured Workflow section.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-new-workflow-temp-ex.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4MTAsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.LO2-X7vuoFeQ_7qr2kFCPfkscxcMOr_fnyrzjIu64gc&_LANG=enus "A screenshot showing an sample Workflow Builder template from the Featured Workflow section.")
   - An App Detail page: In the Related workflow templates  section, select a template to display the Workflow template
     page. Review the information to learn about the workflow template.

     ![A screenshot of a sample Docusign workflow detail page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-newest-workflow-det-pg.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4MTAsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.LO2-X7vuoFeQ_7qr2kFCPfkscxcMOr_fnyrzjIu64gc&_LANG=enus "A screenshot of a sample Docusign workflow detail page.")

     Note:
     - If the associated app is installed, create a workflow
       template by selecting Use
       Template.
     - If the associated app is not installed, log on and install
       the app. After installation, you can use the associated
       workflow template.
2. Optionally, you can narrow your search by selecting filters:

   - Industry menu: Select an Industry sector
   - Steps: Select a workflow step.

   This example shows the resulting workflow template based on the selected
   Works with and Industry
   filters.

   ![A screenshot of a workflow template selection using Industry and step type.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-new-wf-filter-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4MTAsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.LO2-X7vuoFeQ_7qr2kFCPfkscxcMOr_fnyrzjIu64gc&_LANG=enus "A screenshot of a workflow template selection using Industry and step type.")
3. Select a workflow template and select Use.

   The Workflow Builder workflow template displays in a new window.

   ![A screenshot of a sample Docusign Workflow Builder Workflow that is ready to use.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-workflow-access-results.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4MTAsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.LO2-X7vuoFeQ_7qr2kFCPfkscxcMOr_fnyrzjIu64gc&_LANG=enus "A screenshot of a sample Docusign Workflow Builder Workflow that is ready to use.")

   See the [Workflow Builder Workflow Templates](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=zbf1698707257742.html&_LANG=enus) article for
   information about using Workflow Builder Workflows.

## View workflow template details

You can view information about a Workflow Builder workflow to
learn about its features and how to use it in your workflow.

1. In the App Center's Featured Workflow templates
   section, select a workflow template.

   This example shows a workflow detail page.
2. Depending on the app, log on to use it.
3. Optionally view additional information.

   - Instructional video
   - Template overview
   - Template requirements
   - Setup guidelines

   This example shows a sample workflow template details page.
