# Configure a Dropbox Store Files Step in an Agreement Workflow

Learn how to configure a Dropbox
store
files step in a Docusign agreement workflow.

Use the Dropbox
store
files step to save documents generated by workflow steps.
Generally, you position your archive step at the end of your workflow to archive
completed documents.

Restriction: Do not use a Dropbox
store
files step if your workflow uses an eSignature step with multiple
participants and your first signer is designated as an embedded signer.
In this scenario, the first signer goes directly to sign from a prior workflow step.
Documents are archived directly after the embedded signer finishes signing and would
not include the actions of any subsequent participants.

Before you begin
:   - You must be an account administrator or have Workflows: Create user permissions as described in the [Workflow Builder Roles and Permission Requirements](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=gbb1696973048215.html&_LANG=enus) article.
    - The extension app must be [installed and authorized](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=ctg1698170340729.html&_LANG=enus) on your account.

1. Create a new workflow as described in the [Create a Basic Workflow](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=uhm1697826856678.html&_LANG=enus) article.
2. Select Add a step >  Store files with Dropbox >  Configure.
3. Select the file to archive and select Apply.
4. Specify where to archive the file:
   1. Select the connection.
   2. Select the site.
   3. Select a drive.
   4. (Optional) Select a folder on the drive by selecting Add
      Folder. All existing folders are listed, as well as an
      option to create a new folder.
5. Specify a name for your archive. Use the text and variable controls to build
   your file name.

   ![A screenshot of the Dropbox Archive naming dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/acarchivewhat.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4ODUsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.5upAkB5FUClk4CztbO0RYu6eQLQ4iA5NRgwGunxxUxM&_LANG=enus "A screenshot of the Dropbox Archive naming dialog box.")
6. Review the completed file path and select Apply.

Your Dropbox
store
files step is configured.
