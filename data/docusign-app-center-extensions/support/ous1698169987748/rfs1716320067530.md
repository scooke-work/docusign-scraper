# Edit Docusign App Connections

Learn how to update Docusign app connection names in the Docusign App Center Manage Connections page.

You can change your Docusign app connection name to something more
specific or meaningful.

Editing a name does not change the name of the account. It only changes how it displays
in workflows.

To learn how to edit Salesforce connections for an agreement workflow, see [Edit the Name of a Salesforce Organization
Connected to Workflow Builder](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=naa1719257879043.html&_LANG=enus).

Note:

Currently, you can only rename app connections. You cannot change:

- The connection type. That is, toggle from Shared to
  Private.
- Connection ownership.

To make these changes, create a new app connection with the characteristics you want
as described in the [Create Docusign App Connections](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=zsb1714159452358.html&_LANG=enus "Learn how to create Docusign apps connections for different user credentials in Docusign agreement workflows. You can create connections that are shared by all account users or used privately by the user who created the app connection.") article.

1. Select the Docusign app from the App Center page.
2. On the app detail page, select Manage Connections to access the
   Connections page.

   ![A screenshot of a sample App Center Connection page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-newest-conn-dash-inst-apps.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4NTYsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.YfjqCLJWpQI1C22VkHuZT3vSJGk46Jci5hcuVbnDppk&_LANG=enus "A screenshot of a sample App Center Connection page.")
3. From the Overflow menu, select Edit connection name.

   The Edit connection name dialog box displays.

   ![A screenshot of the Docusign Edit connection name dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-newest-edit-conname-db.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4NTYsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.YfjqCLJWpQI1C22VkHuZT3vSJGk46Jci5hcuVbnDppk&_LANG=enus "A screenshot of the Docusign Edit connection name dialog box.")
4. Enter a new name and select Save.

   The system displays a confirmation message:

   `Successfully edited connection`
