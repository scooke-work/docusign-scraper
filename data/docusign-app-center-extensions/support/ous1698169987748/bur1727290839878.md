# Troubleshoot Installing the Docusign IAM for ServiceNow App

Learn how to troubleshoot issues with installing Docusign IAM for
ServiceNow.

Exchange data
between Docusign and ServiceNow with Docusign
IAM for ServiceNow. Use the integration to kick off software and hardware
procurement requests and other downstream workflows.

This topic shows how to troubleshoot issues that you might run into while installing and
configuring the app in ServiceNow. If your configuration fails, try these steps:

1. Check the application scope.
2. Verify the OAuth credentials (Client ID and Client Secret).
3. Ensure that the user has the required roles.
4. Confirm that the table required for integration is present in the
   sys\_property.
5. If you do not see a new ServiceNow table after adding it, refresh your browser. Also clear any filters that are applied to the table list.
6. For issues on the Docusign App Center, disable pop-up
   blocking.
