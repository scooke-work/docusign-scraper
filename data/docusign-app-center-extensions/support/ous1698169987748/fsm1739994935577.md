# Configure a Workato Workflow in a Workflow Builder Agreement Flow

Learn how to configure a Workato workflow in a Docusign Workflow Builder agreement flow.

## Before you begin

- Install and authorize the Workato Extension app on your account. See the  [Install the Workato Extension App from
  the Docusign App Center](https://support.docusign.com/s/document-item?bundleId=kyh1739988794638&topicId=dih1739994835924.html&_LANG=enus)  topic for details.
- Enter a valid bearer token when connecting Docusign Workflow Builder with the Workato account. See the [Connect Docusign Workflow Builder with the Workato Account](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=vnk1739994877491.html&_LANG=enus "Learn how to connect Docusign Workflow Builder with the Workato account.") topic for details.
- Enter a valid API token when connecting Docusign Workflow Builder with the Workato account. See the [Connect Docusign Workflow Builder with the Workato Account](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=vnk1739994877491.html&_LANG=enus "Learn how to connect Docusign Workflow Builder with the Workato account.") topic for details.
- Ensure the connection between Docusign Workflow Builder and Workato account is successful. See the [Connect Docusign Workflow Builder with the Workato Account](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=vnk1739994877491.html&_LANG=enus "Learn how to connect Docusign Workflow Builder with the Workato account.") topic for details.
- Review the [Getting started](https://docs.workato.com/getting-started.html) section of the Workato documentation for information on Workato recipes.
- Create your Workato recipe as an API recipe with an API endpoint as the starting point. See the [API Recipes](https://docs.workato.com/api-mgmt/api-recipes/#api-recipes) section of the Workato Developer API documentation for details.
- Must be an account administrator or have **Workflows: Create user permissions** as described in the
  [Workflow Builder Roles and Permission
  Requirements](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=gbb1696973048215.html&_LANG=enus).
- Log in to your Docusign account.

Use this procedure to configure a Workato workflow in a Workflow Builder agreement flow.

Important: In this release, Docusign Workflow Builder only supports synchronous workflows available in the Workato application.

Procedure

1. Create a workflow as described in the [Create a Basic Workflow](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=uhm1697826856678.html&_LANG=enus) topic.
2. Select Add a step > Trigger Workato workflows > Configure.

   The configuration panel for the Trigger Workato workflows step opens.
3. Choose a workflow:
   1. Select the required Workato workflow (for example, API-Create Employee in BambooHR) from the Workflow Name or ID  dropdown list.

      Note: The Workflow Name or ID dropdown list only displays the workflows associated with the API collection selected in step 4b of the [Obtain the Auth Token from Workato](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=tsk628e1bf4-013c-4f6d-b72e-c5c9349f5960.html&_LANG=enus "Learn how to obtain the auth token from the Workato application.") procedure.
   2. Select Next.

   ![A screenshot to select the required Workato workflow.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ds-workato-select-workflow.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4OTgsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.CwxOANdSrRKmLCYf3q35pWjdsSZulA9zL_wWnHnf_GI&_LANG=enus "A screenshot to select the required Workato workflow.")
4. Map the workflow variables to fields:
   1. Select the required fields (for example, First Name and Account) for the selected Workato workflow input.
   2. Select Apply.

   ![A screenshot to add the required input variables.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/add_required_input_varaibles.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4OTgsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.CwxOANdSrRKmLCYf3q35pWjdsSZulA9zL_wWnHnf_GI&_LANG=enus "A screenshot to add the required input variables.")

You have created and configured a Workato workflow step in Docusign Workflow Builder. You are redirected to the workflow overview in Workflow Builder. You can add additional steps to the workflow (for example, **Collect Data with Web Forms**, **Show a Confirmation Screen**, and so on). You can save a draft version of the workflow or publish it based on your requirement.
