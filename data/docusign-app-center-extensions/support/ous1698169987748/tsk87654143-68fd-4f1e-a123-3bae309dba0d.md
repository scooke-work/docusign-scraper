# Use the Learn Feature to Access App Learning Resources

Use the Docusign App Center Learn tab to access a centralized learning resource for apps and integration.

Use the App Center Learn tab to access a centralized learning resource for apps and integration.

This feature is currently under development, providing quick start information for Salesforce. It
will provide links to Docusign Support Center articles,
partner documentation, videos, and other resources.

- The App Center dashboard Learn  tab
- An individual app's detail page

You can access the Learn information from:

1. Access [Docusign App Center](https://apps-d.docusign.com/app-center).
2. From the top navigation menu, select the Learn
   tab.
3. On the Learn page, browse the tiles and select the content
   you want to open.

   If the guide includes platform tabs (such as Salesforce or HubSpot), select
   the tab that matches your platform.
