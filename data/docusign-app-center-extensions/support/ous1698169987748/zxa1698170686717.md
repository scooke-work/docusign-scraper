# Manage Apps in the Docusign App Center

Learn how to view, install, or uninstall Docusign App Center apps.

In Docusign App Center, you can view,
install, and uninstall apps from the Managed Apps page.

![A screenshot of a sample Docusign Manage Apps Page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-newest-conn-dash-inst-apps.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4MTMsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.Z3s6DZlJObHsKMDzwONecKm5_MVpWkcf22jdFEGVqdA&_LANG=enus "A screenshot of a sample Docusign Manage Apps Page.")

The Manage Apps page provides a left-pane menu that provides these
actions:

- Installed Apps: Select this option to display installed
  apps for accounts under your organization.

  The display includes details, such
  as activity status and version number.
- Approved Apps: Select this option to display apps that
  are approved for all users in your account.

  As an administrator, use the
  access toggle switch to permit non-administrator users in the account to
  install the apps.
- View all Admin settings: Select this option to access
  your account's Admin dashboard, as described in the [What is Docusign
  Admin?](https://support.docusign.com/s/document-item?bundleId=rrf1583359212854&topicId=jte1583359129941.html&_LANG=enus)
  article.

  Use the Docusign Admin page to perform tasks such
  as setting user permissions for your apps.

To perform management activities from the Manage Apps page:

1. Access  [Docusign App Center](https://apps-d.docusign.com/app-center).
2. Select Log in.
3. Select Manage Apps to display your apps.
4. Select an app to display its app detail page.

   You can:

   - View a detailed app description.
   - Display app details, as shown in this example.

     ![A screenshot showing sample App Details page information.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-newest-details-works-with.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4MTMsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.Z3s6DZlJObHsKMDzwONecKm5_MVpWkcf22jdFEGVqdA&_LANG=enus "A screenshot showing sample App Details page information.")
5. Optionally, you can uninstall an app by selecting the app and selecting Manage App  > Uninstall App.

   You are prompted to confirm the action.

   ![A screenshot of the Docusign App Center uninstall app dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/prompt-to-uninstall-an-app.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4MTMsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.Z3s6DZlJObHsKMDzwONecKm5_MVpWkcf22jdFEGVqdA&_LANG=enus "A screenshot of the Docusign App Center uninstall app dialog box.")

   Important: If you uninstall an app, or remove the app's authorization, the operation fails. For example, apps you use in a Docusign Workflow Builder workflow step, or Data Verification fields.
6. Enter UNINSTALL and select
   Uninstall to remove the app from Docusign App Center.

   Note: Use all uppercase letters to activate the
   Uninstall button.
