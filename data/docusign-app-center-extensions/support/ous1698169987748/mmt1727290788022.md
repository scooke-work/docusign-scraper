# Set Up the Docusign App Center ServiceNow Instance, Step 8: Install the ServiceNow App From the Docusign App Center

Learn how to install the ServiceNow app for use with Docusign Workflow Builder.

Exchange data between Docusign and ServiceNow with Docusign IAM for ServiceNow. Use the integration to kick off software and hardware procurement requests and other downstream workflows.

This topic shows how to install the ServiceNow app from the Docusign App Center. This is the last step required to use the integration with Docusign Workflow Builder.

1. Follow the steps in the article [Install Apps in Docusign App Center﻿](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=ctg1698170340729.html&_LANG=enus "Learn how to install Docusign extension apps in the Docusign App Center.").
2. In the Connect Account step at the end of the installation process, provide the **Client ID**,**Client Secret**, and **ServiceNow Instance Name** from the previous steps.If the new connection doesn't automatically load in a pop-up window, click the refresh button and it should appear.
3. Select Save.

You completed this task, and all the tasks required to set up your ServiceNow instance for use with Docusign Workflow Builder.
