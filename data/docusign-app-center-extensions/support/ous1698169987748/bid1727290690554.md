# Set Up the Docusign App Center ServiceNow Instance, Step 3: Update the System Property

Learn how to update the system property in ServiceNow.

Exchange data between Docusign and ServiceNow with Docusign IAM for ServiceNow. Use the integration to kick off software and hardware
procurement requests and other downstream workflows.

This topic shows how to update the system property in ServiceNow.

1. Select the All tab from the top menu. Enter
   sys\_properties.list in the search box.

   The System Properties page displays a list of
   search results.
2. Search for and open the record
   x\_dosi\_docusign\_in.table.filter. The record will open
   once you hit the Enterkey, as the desired table will
   **not**appear in the dropdown results.
3. In the Value field, provide a comma-separated list of
   all the tables that the integration can access. When you are done, select
   Update.
