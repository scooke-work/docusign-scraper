# Configure a Stores Files in Microsoft OneDrive in an Agreement Workflow

Learn how to configure a Microsoft OneDrive store files step in a Docusign agreement workflow.

Use the Microsoft OneDrive (OneDrive) file stores step to save documents generated by workflow steps. Generally, you position your archive step at the end of your workflow to archive completed documents.

Note: In addition to file archive, you can upload content to Docusign Agreement Manager for changes and revisions.

- See the [Agreement Uploads and Ingestion﻿](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=rrs1712255668350.html&_LANG=enus) article to learn about using
  Agreement Manager.
- See the [Upload Documents from Cloud Storage Locations﻿](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=lxo1726792845151.html&_LANG=enus) article to access and upload files.

Restriction: Do not use a OneDrive file archive step if your workflow uses an eSignature step with multiple participants and your first signer is designated as an embedded signer. In this scenario, the first signer goes directly to sign from a prior workflow step. Documents are archived directly after the embedded signer finishes signing and would not include the actions of any subsequent participants.

1. Create a new workflow as described in the [Create a Basic Workflow](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=uhm1697826856678.html&_LANG=enus) article.
2. Select Add a step >  Store files with OneDrive >  Configure.
3. Select the file to archive and select Apply.
4. Specify where to archive the file:
   1. Select the connection.
   2. Select the site.
   3. Select a drive.
   4. (Optional) Select a folder on the drive by choosing Add Folder. All existing folders are listed, as well as an option to create a new folder.
5. Specify a name for your archive. Use the text and variable controls to build your file name.

   ![A screenshot of the OneDrive Archive naming dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/acarchivewhat.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4OTAsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.KB4OzraIOjzzi2IIq7iCrTK-u5HuitNlDp100UWNYuU&_LANG=enus "A screenshot of the OneDrive Archive naming dialog box.")
6. Review the completed file path and select Apply.

Your OneDrive file archive step is configured.
