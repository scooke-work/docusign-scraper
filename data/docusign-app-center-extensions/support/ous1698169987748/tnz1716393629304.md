# Access Docusign App Center Connections

Learn how to add a new Docusign app connection for a feature workflow
from the Docusign App Center.

You can add a Docusign app connection to a feature workflow from the
App Center main page.

To learn how to add Salesforce connections to an agreement workflow, see the [Connect Workflow Builder to Multiple Salesforce
Organizations](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=ogk1719256147008.html&_LANG=enus) article.

1. Select Manage Apps from the App Center main menu.
2. Select Installed Apps from the side menu.

   ![A screenshot of a sample Docusign Manage Connections page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-newest-conn-dash-inst-apps.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4NTEsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.Sp9IxaDDooETgbuzTFV9KgwZ3wOKwZWbZigMb0fDdYs&_LANG=enus "A screenshot of a sample Docusign Manage Connections page.")

   Note: You can optionally access the app you want by
   selecting Extension Apps in the App Center side
   menu.
3. Select an app to display the app's detail page.

   ![A screenshot showing the menu to manage connections from the App Detail page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-newest-conn-menu-det-page.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4NTEsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.Sp9IxaDDooETgbuzTFV9KgwZ3wOKwZWbZigMb0fDdYs&_LANG=enus "A screenshot showing the menu to manage connections from the App Detail page.")

   Note: Only Admins can uninstall
   apps.
4. Depending on the selected app, if there are connections for the app, select
   Manage and choose a connection management action from
   the overflow menu.
5. If there are no connections, select Add New
   Connection.

   Follow the instructions in the [Create App
   Connections](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=zsb1714159452358.html&_LANG=enus "Learn how to create Docusign apps connections for different user credentials in Docusign agreement workflows. You can create connections that are shared by all account users or used privately by the user who created the app connection.") article to add your connections.
