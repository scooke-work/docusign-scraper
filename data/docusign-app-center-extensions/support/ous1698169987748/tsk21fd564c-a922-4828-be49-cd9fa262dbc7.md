# Remove or Disable a Docusign App with an Associated Rule

Learn how to make adjustments in Docusign Account Administration when
you use Docusign App Center to disable or remove an app that has
associated rules.

Through Docusign Account Administration, you created and manage agreement action rules to automate feature actions as described in the [Create an Agreement Rule](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=wqg1596767500566.html&_LANG=enus)﻿﻿ article.

In App Center, if you disable or remove an app that has associated rules, the system
displays a important message as shown in this example.

![A screenshot showing a sample message when you attempt to delete or disable an app connection with an associated rule.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-app-con-disable-app-msg.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4MjQsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.1pA0OLHltlFAG_z8IpPTglok9B70tR9E3QsyDiA3Dus&_LANG=enus "A screenshot showing a sample message when you attempt to delete or disable an app connection with an associated rule.")

Make adjustments in Docusign Account Administration to prevent
problems with applications.

1. Select an app in the App Center.
2. From the overflow menu, select Disable or
   Remove to modify the app.

   ![A screenshot showing the App Center Disable or Remove menu choices.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-app-con-disable-app.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4MjQsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.1pA0OLHltlFAG_z8IpPTglok9B70tR9E3QsyDiA3Dus&_LANG=enus "A screenshot showing the App Center Disable or Remove menu choices.")
3. If you receive the message shown in this example, go to the RULES listing page
   in Account Administration as described in the [Agreement Rules Modification](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=cpx1599852519793.html&_LANG=enus) article.

   ![A screenshot showing a sample message when you attempt to delete or disable an app connection with an associated rule.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-app-con-disable-app-msg.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4MjQsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.1pA0OLHltlFAG_z8IpPTglok9B70tR9E3QsyDiA3Dus&_LANG=enus "A screenshot showing a sample message when you attempt to delete or disable an app connection with an associated rule.")
4. For the rule in question, hover over the alert icon to display the message as
   shown in this example.

   ![A screenshot showing an example of the RULES page with a rule alert.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-app-con-disable-agree-rules.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4MjQsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.1pA0OLHltlFAG_z8IpPTglok9B70tR9E3QsyDiA3Dus&_LANG=enus "A screenshot showing an example of the RULES page with a rule alert.")
5. Select the Action Menu and select
   Edit.

   ![A screenshot showing a sample Agreement RULES page ACTION menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-app-con-agree-rule-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4MjQsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.1pA0OLHltlFAG_z8IpPTglok9B70tR9E3QsyDiA3Dus&_LANG=enus "A screenshot showing a sample Agreement RULES page ACTION menu.")
6. On the Edit Rules page, a message displays as shown in this example.

   ![A screenshot of a sample Agreement rules inactive rule warning message.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-app-con-agree-rule-edit-pg.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4MjQsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.1pA0OLHltlFAG_z8IpPTglok9B70tR9E3QsyDiA3Dus&_LANG=enus "A screenshot of a sample Agreement rules inactive rule warning message.")
7. Change the rule's fields and save the rule to eliminate the warning
   message.
