# Access Docusign eSignature Integrations in App Center

Learn about the Docusign eSignature integration apps available in
Docusign App Center.

With Docusign eSignature integration apps, you can easily add Docusign functionality to your favorite software tools without
leaving them.

For information about eSignature integrations see [Streamline process with 1,000+ integrations](https://www.docusign.com/integrations?filter=Integrations).

These eSignature integration apps are currently available:

- eSignature for Google Workspace
- eSignature for HubSpot
- eSignature for Microsoft Dynamics
- eSignature for Microsoft PowerAutomate
- eSignature for Microsoft SharePoint
- eSignature for Microsoft Teams
- eSignature for Salesforce
- eSignature for ServiceNow
- eSignature for Workday
- ID Verification with Clear

As an Admin, when you access an app's detail page, install the application
so that it's available for your users.

1. Access [Docusign App Center](https://apps-d.docusign.com/app-center).
2. From the left navigation pane, select eSignature
   integrations to display the available eSignature apps.

   ![A screenshot of the Docusign App Center eSignature integrations page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-esig-int-dash.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE3ODQsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.CGHQ2nkXoX9VRZhKj_LUhHcHpx0MYhCV6k9CzZLu_uU&_LANG=enus "A screenshot of the Docusign App Center eSignature integrations page.")
3. Select an application to display the detail page.
4. Select Install.

   - If the app has not been installed, you're directed to an external
     site. Here you can access the app as shown in this example.

     ![A screenshot of a sample App Center eSignature integration app that needs installation.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-esig-int-not-inst.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE3ODQsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.CGHQ2nkXoX9VRZhKj_LUhHcHpx0MYhCV6k9CzZLu_uU&_LANG=enus "A screenshot of a sample App Center eSignature integration app that needs installation.")

     Obtain and install the app in App Center as described in the [Install Apps in the Docusign App Center](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=ctg1698170340729.html&_LANG=enus "Learn how to install Docusign extension apps in the Docusign App Center.") article.
   - If your administrator installed the app, a message similar to this
     example displays.

     ![A screenshot of a sample App Center eSignature integration app that has been installed.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-esig-int-already-inst.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE3ODQsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.CGHQ2nkXoX9VRZhKj_LUhHcHpx0MYhCV6k9CzZLu_uU&_LANG=enus "A screenshot of a sample App Center eSignature integration app that has been installed.")

     Use the app as described in the [Non-admin Users: Work with Approved
     Apps](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=xni1748970087448.html&_LANG=enus "As a Docusign account user, learn how to find and manage Docusign apps you’re approved to use.")  article.
