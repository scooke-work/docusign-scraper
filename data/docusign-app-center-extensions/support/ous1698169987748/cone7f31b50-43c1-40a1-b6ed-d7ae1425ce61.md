# Use Docusign FedRAMP-Certified Apps

Learn how to identify FedRAMP-certified apps in the Docusign App Center.

When FedRAMP customers access the Docusign App Center, they must know which
apps are FedRAMP compliant. Using this information, customers can make informed decisions
about app installation and creating connections. Administrators and non-admin users can now
install and use FedRAMP compliant apps in Workflow Builder and Agreement Manager.

Currently, these FedRAMP-certified apps are available in the Docusign App Center:

- Coupa
- Email Verification
- Microsoft Dynamics 365
- SharePoint
- Salesforce
- Smarty
- SSN and FEIN Verification
- Twillio
- Vonage

The FedRAMP-certified badge identifies FedRAMP-certified apps in the Docusign App Center. The badge displays on the app’s tile in the App Center page or the app's
detail page as shown in this example.

## Access FedRAMP apps

You can identify and select FedRAMP apps in the Docusign App Center or in
a Docusign Workflow Builder workflow.

From the App Center page

Select FedRAMP-certified  from the left menu pane to display
FedRAMP-certified apps.

From the Extension Apps page

Select Extension Apps  > Certifications > FedRAMP-certified .

![A screenshot of selecting Docusign Fed-RAMP-certified apps. from the Certifications menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-new-fedramp-sel-certs-filter.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4NDMsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.7SgbVzug3i7RXcB8wVnO_7ALMEAaggdgEg9hjCOqyMU&_LANG=enus "A screenshot of selecting Docusign Fed-RAMP-certified apps. from the Certifications menu.")

For Workflow Builder in-product discovery

When adding a new step
to a workflow, as described in the [Creating Workflows](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=tco1697826811758.html&_LANG=enus) article, select one of these access options.

- Browse Apps option
  1. Select Add a step
  2. Scroll down to the Connected Apps section and select Browse
     Apps to display the Docusign App Center page.

     ![A screenshot showing browsing for Docusign apps from a Workflow Builder workflow step.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-app-fedramp-maestro.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4NDMsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.7SgbVzug3i7RXcB8wVnO_7ALMEAaggdgEg9hjCOqyMU&_LANG=enus "A screenshot showing browsing for Docusign apps from a Workflow Builder workflow step.")
