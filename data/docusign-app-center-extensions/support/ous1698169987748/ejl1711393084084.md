# Create the ServiceNow App Writeback Step

Learn how to create the ServiceNow writeback step to use a ServiceNow instance in
Docusign Workflow Builder workflows.

1. Create a new workflow as described in the [Create a Basic Workflow](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=uhm1697826856678.html&_LANG=enus) article.
2. Select Add a step Writeback to ServiceNow.
3. Enter the Step name.
4. Enter the Connection name.
5. Select the ServiceNow object to write to and select
   Next.
6. Update selected fields and select Save.

   The following example shows a completed ServiceNow workflow step.
