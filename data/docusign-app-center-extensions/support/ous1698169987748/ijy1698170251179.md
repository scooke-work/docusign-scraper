# Welcome to Docusign App Center

Learn about using the Docusign App Center to discover and enable applications to extend your agreement workflows.

[Docusign App Center](https://apps.docusign.com/app-center) is a place to discover and install partner applications that extend Docusign product capabilities.

## How extension apps work in Docusign

Use extension apps to customize and extend agreement processes across multiple Docusign products, including Docusign Workflow Builder, the agreement workflow builder tool.

To learn more about Docusign Workflows, see the [Workflow Builder Overview](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=dnx1696972415150.html&_LANG=enus) article.

For a summary of App Center questions, see the  [Docusign App Center FAQ Overview](https://support.docusign.com/s/document-item?bundleId=gcc1750788193890&topicId=dvb1750792155723.html&_LANG=enus) article.

## App Center permissions

Before you can install the apps, make sure that you have the required permissions for both the Docusign account and the app you want to install.

Install and manage apps
:   You must be a Docusign account administrator with full administrative permissions to install and manage Docusign apps. See the [Permission Profiles](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=pof1583277362435.html&_LANG=enus) article to learn more about these permissions.
:   You install apps at the account level. To change accounts, select Switch Account in the Profile menu.

Banner messages
:   Depending on permissions, banner messages display on app detail pages. See the  [Docusign App Center Banners and Messages and App Labels](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=map1704218548543.html&_LANG=enus "Learn about the messages and availability app labels that you encounter when using Docusign App Center and the actions you should take.") article to learn more about their meaning.

App account restrictions
:   Currently, App Center only supports work and school accounts for extension apps. For example, App Center does not support personal accounts for Google Drive or SharePoint.

## Supported languages and regions

Organizations can globally access Docusign App Center.

Note:

- Along with U.S. English, Docusign App Center and Apps are available in the [Sender-supported languages](https://support.docusign.com/s/document-item?bundleId=ulp1643236876813&topicId=spo1578456459828.html).

  Note: Region-specific apps may not be supported in all sender-supported languages.
- Some apps are specific to certain regions. You may not see all apps if they're not deployed in your region. See the [App Center Regional Availability](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=stg1748631650074.html&_LANG=enus "Learn about the regional availability for Docusign App Center apps and where to find this information in Docusign App Center.") article for information about supported regions and region designations that display on the app's App Detail page.

## Use Apps within Docusign features

Use apps from the Docusign App Center within these Docusignfeatures:

- Workflow Builder

  Use Docusign Workflow Builder to build and deploy customizable, code-free agreement workflows with the appropriate user permissions assigned to you by your Docusign Administrator.

  See the [Workflow Builder Roles and Permissions](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=gbb1696973048215.html&_LANG=enus) article to learn more about Workflow Builder permissions.
- eSignature Data Verification

  Use the Data Verification capabilities with prebuilt, drag-and-drop components powered by trusted apps. Data Verification verifies information and helps decrease document errors. See the  [Welcome to Docusign Data Verification](https://support.docusign.com/s/document-item?bundleId=qki1698158012210&topicId=gxs1698158285710.html&_LANG=enus) article to learn about using Data Verification apps.
- CLM

  Use Google Docs for CLM to open documents in Google Docs for changes and revisions. Once you make changes in Google Docs, you can save the document to CLM. See the [Google Docs for CLM](https://support.docusign.com/s/document-item?bundleId=alb1642965087302&topicId=gcn1714403494382.html&_LANG=enus)  App article for instructions on configuring the Google Docs for CLM app.
- Agreement Manager

  Upload documents for changes and revisions in Docusign Agreement Manager. See the [File Ingestion from Third-Party Cloud Providers](https://support.docusign.com/s/document-item?bundleId=pqz1702943441912&topicId=byx1727376126901.html&_LANG=enus)  article to learn about using Google Drive with Agreement Manager.

## Build extension apps

As a developer, you can create, test, and publish apps to the Docusign App Center through the  [Docusign Developer Center](https://developers.docusign.com/).

Developers can review Docusign API documentation to learn how to build extension apps for the Docusign App Center. See the [Extension apps 101 overview](https://developers.docusign.com/extension-apps/extension-apps-101/) on the Docusign Developer Center.

For more information about building extension apps, access the Docusign University course [Build Extension Apps for Docusign App Center](https://dsucustomers.docusign.com/build-extension-apps-for-the-docusign-app-center).

## App Center community group

See the [App Center community page](https://community.docusign.com/docusign-app-center-82) for more information.
