# Create Docusign App Connections

Learn how to create Docusign apps connections for different user
credentials in Docusign agreement workflows. You can create connections
that are shared by all account users or
used
privately by the user who created
the
app connection.

You can create Docusign
app
connections for use in Docusign agreement workflows. See the [Manage Docusign App
Connections](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=xyw1716318604389.html&_LANG=enus "Learn how to create and manage Docusign app connections. Use connections to access different accounts and environments.") article for information about user permissions and connection
types.

Connection types

Depending on your user credentials, you can create
these
app connection types:

- Shared: Created by an Admin. Any account user can view
  and use the connection.
- Private: Created by an account user or an Admin. Only the
  account user who created the connection can view and use the connection.

Before you begin

Install and configure the app as described in the [Install Apps in the Docusign App Center](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=ctg1698170340729.html&_LANG=enus "Learn how to install Docusign extension apps in the Docusign App Center.") article.

## Create Shared Connections

As a user with Administrator privileges, you can create shared
app
connections. Any account user can view and use in Docusign platform services, such as Workflow Builder and Agreement Manager.

1. From the Docusign App Center Manage page, select an app
   to display the app's detail page as shown in this example.

   ![a screenshot of a detail page with the Manage menu to access app connections.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-new-det-w-man-con-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4NTQsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.bGJc63oVa4YaKWIgtlH1h3euRAzT5nw7p66PMKW1icg&_LANG=enus "a screenshot of a detail page with the Manage menu to access app connections.")
2. Select Manage Connections to display the app's Connections page.

   Note: If you have more than one account, select the
   account for which you want to add a connection.
3. Select New Connection.
4. Salesforce only select a Salesforce account environment.

   Choose from:
   - Salesforce Production
   - Salesforce Sandbox
5. At the Select Account Type page, Select the Shared connection type and select
   Next.
6. At the Name Connection page, select a name for your connection.

   Tip: Change the default name to something you can easily identify.
7. If prompted, select Login to
   connect
   your app.

   You have created a shared app connection to use with platform services.

   See the [Configure Docusign Apps in an Agreement Workflow](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=hno1706034267152.html&_LANG=enus "You can integrate Docusign apps to automate your agreement workflows and boost productivity.")
   article for more information.

## Create Private App Connections

As a user without Administrator privileges, you can only create
private
app connections. Only you can view and use
these
app connections in agreement platform processes.

1. From the Docusign App Center Manage page, select an app
   to display the app's Detail Listing page.
2. Select Log in and enter your credentials.
3. If prompted, select Connect Account to enable access
   to the app account.

   Note: If you have more than one account, select the
   account where you want to add a connection.
4. At the Select Account Type page, select the Private
   connection type and select Next.
5. At the Name Connection page, select a name for your connection.

   Tip: Change the default name to something
   that you can easily identify.
6. If prompted, select Login to connect to your
   app.

   - Select Allow
     to confirm access to the account.
   - Enter configuration or authorization information specific to your
     app and select Next.

   You have created a private app connection to use with platform services.

   See the [Configure Docusign Apps in an Agreement Workflow](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=hno1706034267152.html&_LANG=enus "You can integrate Docusign apps to automate your agreement workflows and boost productivity.")
   article for more information.
