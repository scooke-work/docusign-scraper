# Set Up the Docusign App Center ServiceNow Instance

Learn how to set up the ServiceNow instance for use in the Docusign
App Center.

Exchange data between Docusign and
ServiceNow with Docusign IAM for ServiceNow. Use the integration
to kick off software and hardware procurement requests and other downstream
workflows.

To install the app for use with Docusign Workflow Builder,
follow these steps. If you run into any issues, see [Troubleshoot Installing the Docusign IAM for ServiceNow App](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=bur1727290839878.html&_LANG=enus "Learn how to troubleshoot issues with installing Docusign IAM for ServiceNow.").

1. [Set Up the Docusign App Center ServiceNow Instance, Step 1: Install the Docusign IAM for ServiceNow App](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=dnf1727290653940.html&_LANG=enus "Learn how to install the Docusign IAM app in ServiceNow.").
2. [Set Up the Docusign App Center ServiceNow Instance, Step 2: Generate a New Client Secret for OAuth](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=jgz1727290675860.html&_LANG=enus "Learn how to create a new client secret for Docusign IAM for ServiceNow.").
3. [Set Up the Docusign App Center ServiceNow Instance, Step 3: Update the System Property](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=bid1727290690554.html&_LANG=enus "Learn how to update the system property in ServiceNow.").
4. [Set Up the Docusign App Center ServiceNow Instance, Step 4: Assign the User Role](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=qvz1727290705170.html&_LANG=enus "Learn how to assign the user role for the Docusign IAM for ServiceNow app.").
5. [Set Up the Docusign App Center ServiceNow Instance, Step 5: Assign Application Access to ServiceNow Tables](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=sdo1727290722252.html&_LANG=enus "Learn how to grant Docusign access to ServiceNow tables.").
6. [Set Up the Docusign App Center ServiceNow Instance, Step 6: Create Access Controls](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=bvm1727290749720.html&_LANG=enus "Learn how to create access controls so that Docusign can access ServiceNow tables.").
7. [Set Up the Docusign App Center ServiceNow Instance, Step 7: Create Cross-Scope Privilege Records](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=yjb1727290763658.html&_LANG=enus "Learn how to create cross-scope privilege records so that Docusign can access ServiceNow tables.").
8. [Set Up the Docusign App Center ServiceNow Instance, Step 8: Install the ServiceNow App From the Docusign App Center](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=mmt1727290788022.html&_LANG=enus "Learn how to install the ServiceNow app for use with Docusign Workflow Builder.").

You completed this task. Your ServiceNow instance is ready
for use with Docusign Workflow Builder.
