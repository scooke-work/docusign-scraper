# Delete a Docusign App Connection

Learn how to delete a Docusign app connection. The deleted app
connection is no longer available for Docusign workflows.

You can delete a Docusign app connection in the Docusign
Manage Connections Page.

To learn how to remove Salesforce connections from an agreement workflow, see [Remove a Salesforce Organization From
Workflow Builder](https://support.docusign.com/s/document-item?bundleId=qwr1706590641805&topicId=pwt1719260196069.html&_LANG=enus).

1. Select the Docusign app from the App Center page.
2. On the app detail page, select Manage Connections
   to display the Connections Page.

   ![A screenshot of the Docusign App Connections page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-new-manage-conn-menu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4NjEsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.tvTYPCLOvwLUvy16icIu9Fd2y4Nlt-x9x-mkf5AFSfU&_LANG=enus "A screenshot of the Docusign App Connections page.")
3. Select Delete connection from the overflow menu next to
   display the Delete connection prompt:

   ![A screenshot of the Docusign Delete this Connection dialog box.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/prompt-to-delete-connection.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4NjEsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.tvTYPCLOvwLUvy16icIu9Fd2y4Nlt-x9x-mkf5AFSfU&_LANG=enus "A screenshot of the Docusign Delete this Connection dialog box.")
4. Enter DELETE.

   Note: Use all uppercase letters to activate the
   Delete Connection option.

The connection is no longer available for Docusign workflows.

Attention: Deleting a connection breaks any platform
service workflow that uses this connection.
