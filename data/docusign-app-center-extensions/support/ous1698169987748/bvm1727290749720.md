# Set Up the Docusign App Center ServiceNow Instance, Step 6: Create Access Controls

Learn how to create access controls so that Docusign can access
ServiceNow tables.

Exchange data between Docusign and ServiceNow with Docusign IAM for ServiceNow. Use the integration to kick off software and hardware
procurement requests and other downstream workflows.

This topic shows how to create access controls (read, create, write).

1. Elevate your role so that you can edit the access controls. In the same table to which you granted Docusign access in the
   previous step, select your profile icon > Elevate role.

   The Elevate role dialog box displays.
2. Check the box next to security\_admin and select
   Update.

   ![A screenshot of the Elevate role dialog box. The box next to security_admin is checked.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/servicenowelevaterole.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE5NDIsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.eGVrPE6jiw7ESwFSc99bSE4nXHI5314TI3oWpyQGdY0&_LANG=enus "A screenshot of the Elevate role dialog box. The box next to security_admin is checked.")

   You return to the table record. A New button now
   displays on the right side of the Access Controls tab.
3. Select the New button to create a new access
   control.

   The Access Control New record page
   displays.
4. In the Operation field, set the control type
   (read, create, or
   write.)
5. Under Requires Role, in the Role
   field, add the role
   x\_dosi\_docusign\_in.docusign\_scripted\_rest\_api\_access\_role
   and select Submit.

   You return to the table record.
6. Repeat the process of creating a new access control for each required access
   control (read, create,
   write).If you run into issues granting access permission, it is likely
   because of a mismatch in scope between thetarget scopeand
   target table. Ensure the target scope and target
   table have the same name to avoid this.

You completed this task.
