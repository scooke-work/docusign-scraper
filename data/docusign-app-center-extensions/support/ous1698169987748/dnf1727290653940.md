# Set Up the Docusign App Center ServiceNow Instance, Step 1: Install the Docusign IAM for ServiceNow App

Learn how to install the Docusign IAM app in ServiceNow.

Exchange data between Docusign and ServiceNow with Docusign IAM for ServiceNow. Use the integration to kick off software and hardware
procurement requests and other downstream workflows.

This topic shows how to install the app in ServiceNow.

1. If you don’t already have a ServiceNow instance, create an instance from the
   [ServiceNow developer site](http://developer.servicenow.com/).
2. Install the Docusign IAM App from the ServiceNow Store.
3. In the top menu, select the globe icon.

   A panel containing a list of scopes displays.
4. Change the application scope to Docusign IAM.
