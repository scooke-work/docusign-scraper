# Docusign App Center Connected Apps Workflow

Learn about the new workflow for using Docusign App Center to manage
app connections that were previously managed through Docusign Account
Administration Agreement Actions.

Here is the new workflow for using Docusign App Center to
manage app connections that were previously managed through Docusign
Account Administration Agreement Actions.

1. Install the application in Docusign App Center, as
   described in [Install Apps in the Docusign App
   Center](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=ctg1698170340729.html&_LANG=enus "Learn how to install Docusign extension apps in the Docusign App Center.").﻿

   This example shows the app detail page where you select  Install
   App.

   ![A screenshot of the Install App option on an app detail page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-newest-det-install.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4MjEsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.oXQ7TSLnY7FjjHZGznM8KjtoH560_G4Ra37NdQXP1Go&_LANG=enus "A screenshot of the Install App option on an app detail page.")
2. Create app connections for installed apps in Docusign
   App Center as described in the [Create Docusign App Connections](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=zsb1714159452358.html&_LANG=enus "Learn how to create Docusign apps connections for different user credentials in Docusign agreement workflows. You can create connections that are shared by all account users or used privately by the user who created the app connection.") article.

   1. On the app's detail page, select Manage > New Connection

      These connection types are available:

      - Shared: Created by an Admin. Any account user
        can view and use the connection.
      - Private: Created by an Admin or a Non-admin
        account user. Only the account user who created the connection can
        view and use the connection.

        The Select Account type page displays.

        ![A screenshot of the Connection Select Account Type page.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/ac-newest-add-new-connection.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4MjEsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.oXQ7TSLnY7FjjHZGznM8KjtoH560_G4Ra37NdQXP1Go&_LANG=enus "A screenshot of the Connection Select Account Type page.")
   2. From this page, complete the steps to create the app connection as described in the [Create Docusign App Connections](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=zsb1714159452358.html&_LANG=enus "Learn how to create Docusign apps connections for different user credentials in Docusign agreement workflows. You can create connections that are shared by all account users or used privately by the user who created the app connection.") article.
3. Create Agreement rules for your app connections in Docusign Account Administration as described in the [Create an Agreement Rule](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=wqg1596767500566.html&_LANG=enus)﻿﻿ article.

   ![A screenshot of the Agreement Rule actions.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/app-actions-add-actions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE4MjEsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.oXQ7TSLnY7FjjHZGznM8KjtoH560_G4Ra37NdQXP1Go&_LANG=enus "A screenshot of the Agreement Rule actions.")

You have now created an agreement action for an app using Docusign App
Center.
