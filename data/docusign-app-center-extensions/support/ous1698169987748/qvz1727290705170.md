# Set Up the Docusign App Center ServiceNow Instance, Step 4: Assign the User Role

Learn how to assign the user role for the Docusign IAM for ServiceNow
app.

Exchange data between Docusign and ServiceNow with Docusign IAM for ServiceNow. Use the integration to kick off software and hardware
procurement requests and other downstream workflows.

This topic shows the part of the authorization process where you assign the user
role.

1. Select the All tab in the top menu. Search for and
   select Users.

   The Users page displays a list of
   users.
2. At the top right, select New to create a new user.

   The User New Record page displays.
3. Enter the details for the new user and select
   Submit.

   You return to the list of users.
4. Search for and open the user record that you just created and want to grant access to.
5. In the Roles tab, select
   Edit.

   ![A screenshot of the page for the user record. The Roles tab is highlighted.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/servicenowrolestab.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE5MzcsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.-at-Fd6hTBYvaiyT4kqgPh3q57Xfhq0VBqxb2YA37mw&_LANG=enus "A screenshot of the page for the user record. The Roles tab is highlighted.")

   The Edit Members page displays. A list of available
   roles displays on the left. The user and any current roles assigned to them
   display on the right.
6. In the Collection search box on the left, search for the
   role
   x\_dosi\_docusign\_in.docusign\_scripted\_rest\_api\_access\_role.
   Select it in the list on the left and select the right arrow.

   ![A screenshot of the Edit Members page showing the available roles on the left and assigned roles on the right.](https://docusign-be-prod.zoominsoftware.io/api/bundle/ous1698169987748/page/images/servicenowassignrole.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MTE5MzcsInNoZWFmIjoib3VzMTY5ODE2OTk4Nzc0OCJ9.-at-Fd6hTBYvaiyT4kqgPh3q57Xfhq0VBqxb2YA37mw&_LANG=enus "A screenshot of the Edit Members page showing the available roles on the left and assigned roles on the right.")

   This role authorizes a user's admin**read from**and **writeback**access to the IAM
   Spoke app APIs.
7. When you are finished assigning roles to the user, select
   Save.

You completed this task. The required role is
assigned.
