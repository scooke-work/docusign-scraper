# Use a form to update a registration

This procedure explains how to update an extension app registration using a form. You also have the option of updating a registration by [editing the app manifest file](https://developers.docusign.com/extension-apps/build-an-extension-app/update-registration/use-manifest/). See [Form-based and app manifest method comparison](https://developers.docusign.com/extension-apps/build-an-extension-app/register/#form-based-and-app-manifest-method-comparison) for details about the two methods.

See [Extension app statuses that affect updates](https://developers.docusign.com/extension-apps/build-an-extension-app/update-registration/#extension-app-statuses-that-affect-updates) for details about steps you may need to take before you can update an extension app registration.

To update an extension app registration using a form:

1. Log in to the [Developer Console](https://devconsole.docusign.com).
2. Select the extension app in the list. The **Overview** page is displayed.

   ![](data:image/svg+xml;charset=utf-8,%3Csvg height='1201' width='1446' xmlns='http://www.w3.org/2000/svg' version='1.1'%3E%3C/svg%3E)

   ![Overview page](https://images.ctfassets.net/aj9z008chlq0/4sIIKGejSjBwabPXoqLjN1/73a14c40079883a2a8eeef06edda430b/Overview.png?w=1446&h=1201&q=50&fm=png)
3. See the table below for the next steps to take, depending on the type of update you want to make.

| If you want to update | Select this option from the left nav | See this procedure for details about the settings |
| --- | --- | --- |
| The extension app name and short and long descriptions | App Center Listing | [Update basic information](https://developers.docusign.com/extension-apps/build-an-extension-app/register/use-form/app-center-listing/#update-basic-information) |
| The publisher name, website, and support information | App Center Listing | [Update publisher details and contact information](https://developers.docusign.com/extension-apps/build-an-extension-app/register/use-form/app-center-listing/#update-publisher-details-and-contact-information) |
| Screenshots that appear in the extension app listing | App Center Listing | [Manage screenshots](https://developers.docusign.com/extension-apps/build-an-extension-app/register/use-form/app-center-listing/#manage-screenshots) |
| The icon that appears wherever the extension app is listed in the Developer Console, App Center, and Workflow Builder | App Center Listing | [Update app icon](https://developers.docusign.com/extension-apps/build-an-extension-app/register/use-form/app-center-listing/#update-app-icon) |
| The changelog that appears in the extension app listing for a new [version](https://developers.docusign.com/extension-apps/build-an-extension-app/versioning/) of an extension app | App Center Listing | [Update changelog](https://developers.docusign.com/extension-apps/build-an-extension-app/register/use-form/app-center-listing/#update-changelog) |
| Settings that enable an extension app to obtain authorization from the external API service or identity provider | Integration Details | [Update connection settings](https://developers.docusign.com/extension-apps/build-an-extension-app/register/use-form/integration-details/#update-connection-settings) |
| Settings that the Docusign platform uses to send requests to external API endpoints | Integration Details | [Update extension and action settings](https://developers.docusign.com/extension-apps/build-an-extension-app/register/use-form/integration-details/#update-extension-and-action-settings) |
| Settings that determine which [extensions](https://developers.docusign.com/extension-apps/extension-apps-101/supported-extensions/) the extension app implements | Integration Details | [Add an extension](https://developers.docusign.com/extension-apps/build-an-extension-app/register/use-form/integration-details/#add-an-extension) or [Delete an extension](https://developers.docusign.com/extension-apps/build-an-extension-app/register/use-form/integration-details/#delete-an-extension) |
