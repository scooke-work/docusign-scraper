# Working with elastic templates

An *elastic template* is a special kind of template that you can use in [embedded](https://developers.docusign.com/docs/esign-rest-api/esign101/concepts/embedding/) scenarios to simplify getting consent to your agreements. Elastic templates are a key feature of the Click API. Each Click API how-to guide demonstrating how to use elastic templates is listed below:
