# API Categories

This API is separated into major categories. Each category covers one or more resources and the resources' methods.

| Category | Summary |
| --- | --- |
| [Accounts](https://developers.docusign.com/docs/esign-rest-api/reference/accounts/) | Account management |
| [Billing](https://developers.docusign.com/docs/esign-rest-api/reference/billing/) | Invoices |
| [BulkEnvelopes](https://developers.docusign.com/docs/esign-rest-api/reference/bulkenvelopes/) | Bulk envelopes |
| [CloudStorage](https://developers.docusign.com/docs/esign-rest-api/reference/cloudstorage/) | This section shows you how to manage your cloud storage providers and users. |
| [Connect](https://developers.docusign.com/docs/esign-rest-api/reference/connect/) | Docusign Connect is the Docusign platform notification service. The service uses webhooks to proactively notify your application when an event occurs that your application wants to know about. Docusign recommends that all applications use Connect instead of polling the Docusign Signature Service. This section provides information about Connect configuration and logging.  **Note:** To make API calls to any of the Connect endpoints, you must be an account administrator.  For more information about Connect, see [Docusign Connect](https://developers.docusign.com/platform/webhooks/connect/) and [Custom Connect Configuration](https://developers.docusign.com/platform/webhooks/connect/create-configuration-trigger-event/). |
| [CustomTabs](https://developers.docusign.com/docs/esign-rest-api/reference/customtabs/) | Custom tabs |
| [Diagnostics](https://developers.docusign.com/docs/esign-rest-api/reference/diagnostics/) | Diagnostics and request logging |
| [EmailArchive](https://developers.docusign.com/docs/esign-rest-api/reference/emailarchive/) | This section shows you how to set and manage the BCC email addresses that you want to use to archive the emails that Docusign sends. |
| [Envelopes](https://developers.docusign.com/docs/esign-rest-api/reference/envelopes/) | This section shows you how to create and send envelopes. |
| [Folders](https://developers.docusign.com/docs/esign-rest-api/reference/folders/) | Folders allow you to organize envelopes and templates. |
| [Notary](https://developers.docusign.com/docs/esign-rest-api/reference/notary/) | Docusign eNotary (Legacy) makes the notarization process fully digital for senders, signers, and notaries. It enables a notary public to act as an in-person witness to electronic signing of documents. |
| [Payments](https://developers.docusign.com/docs/esign-rest-api/reference/payments/) | Manage Payments |
| [PowerForms](https://developers.docusign.com/docs/esign-rest-api/reference/powerforms/) | The PowerForms category provides methods that enable you to create and manage fillable forms. |
| [SigningGroups](https://developers.docusign.com/docs/esign-rest-api/reference/signinggroups/) | Send a signing request to a group of potential signers. |
| [Templates](https://developers.docusign.com/docs/esign-rest-api/reference/templates/) | Template creation and management |
| [UserGroups](https://developers.docusign.com/docs/esign-rest-api/reference/usergroups/) | Group users for common permissions and branding |
| [Users](https://developers.docusign.com/docs/esign-rest-api/reference/users/) | User management |
| [Workspaces](https://developers.docusign.com/docs/esign-rest-api/reference/workspaces/) | A Docusign workspace is a collaboration area for sharing files and data. |
