# Custom tabs

You can customize any of the Docusign tab types and save the resulting *custom tab* as a new tab type that is available to your account. Custom tabs can save your users time when tagging documents because they can use your predefined custom tabs in multiple places without having to create and configure each tab manually in each location.

You can add custom tabs to envelopes and templates by setting the `customTabId` property when creating an envelope or template recipient, or when adding a new tab for an existing recipient. The custom tab must be added as the correct tab type. For example, if the custom tab type is `text`, it cannot be used as a number tab.

When the `customTabId` property is set, the new tab inherits all of the specified custom tab’s properties. In this case, all required information that is not included in the custom tab, such as document ID (which identifies the document that the tab is associated with) and page ID (which identifies the page number of the document where the tab appears), must be included when adding the tab. If the custom tab does not have anchor tab placement settings, then you must include fixed positions (`xPosition` and `yPosition`).

You can also set your custom tabs to be read-only by setting the `locked` parameter to true. While `locked` is true, signers cannot change the data of the custom tab.
