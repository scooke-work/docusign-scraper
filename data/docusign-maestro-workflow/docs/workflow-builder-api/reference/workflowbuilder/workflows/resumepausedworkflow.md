# : resumePausedWorkflow

This operation enables new workflow instances to be created

HTTP Request

POST

```
/v1/accounts/{accountId}/workflows/{workflowId}/actions/resume
```

Base URL : Demo

```
https://api-d.docusign.com
```

## Request

Path Parameters

accountId

---

UUID

Use example values

adf8c455-de2f-4119-b183-d952d649e87c

\*required

The unique identifier of the account.

pattern: ^[{]?[0-9a-fA-F]{8}-([0-9a-fA-F]{4}-){3}[0-9a-fA-F]{12}[}]?$

workflowId

---

WorkflowId

Use example values

adf8c455-de2f-4119-b183-d952d649e87c

\*required

A universally unique identifier (UUID) in standard 8-4-4-4-12 format.

pattern: ^[{]?[0-9a-fA-F]{8}-([0-9a-fA-F]{4}-){3}[0-9a-fA-F]{12}[}]?$
