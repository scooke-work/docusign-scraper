# Workflow Utility Steps

This section contains detailed configuration information for functional steps in the agreement process.

[Configure a Set Up Invite Step](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=kfx1698167344720.html&_LANG=enus "Learn how to use a Set Up Invite step in your agreement workflow to invite participants to join the workflow.")
:   Set up and manage steps to invite participants to join the workflow. Inviting participants is required for them to receive the necessary information and access to participate.

[Configure a Show a Confirmation Screen Step](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=rpc1698682081448.html&_LANG=enus "Learn how to add a Show a Confirmation Screen step to your workflow. This step generates a message screen that appears to a specific workflow participant to convey information about their progress.")
:   Design and configure steps to confirm actions or decisions made by participants. Confirmations provide clear acknowledgment and record of their input.

[Configure a Send an Email Step](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=ker1716504515479.html&_LANG=enus "Learn how to add an email notification step to your agreement workflow. This step generates an ad hoc email notification to convey information to any workflow participant or other named individual.")
:   Set up automated email notifications to keep participants informed at various stages of the workflow. Notifications enhance communication and transparency.

[Configure a Branching Rule](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=mnv1698169910928.html&_LANG=enus "You can create conditional rules in your agreement workflow. These rules validate a condition and branch the workflow based on whether the condition is met or not.")
:   Implement branching rules to direct the workflow based on specific conditions or participant responses. Branching provides dynamic and adaptable workflow paths.

[Configure a Multi Branch Rule](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=xtg1749585007265.html&_LANG=enus "Learn how to use the Add a Multi Branch Rule step in Docusign Workflow Builder to create workflows with multiple paths based on specific conditions. This feature allows for dynamic processes that adapt based on data collected from web forms.")
:   Add flexibility to your workflow with a multi branch rule that can contain up to 15 different paths.

[Configure an Add Parallel Branches Step](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=tsk9f5912a8-48d1-4433-9408-f0c6c9470c21.html&_LANG=enus "Use the Add Parallel Branches step to split a single workflow path into multiple branches that run at the same time.")
:   Run simultaneous tasks with up to 15 parallel branch paths.

[Connect to Any Service Using Web Requests](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=conbf387387-0435-452a-a7bb-fd1f719980eb.html&_LANG=enus "Docusign Workflow Builder allows you to trigger external APIs using the Make a Web Request utility step.")
:   Send HTTP requests (GET, POST, or PUT) to almost any external or internal service.
