# Store Envelope Files to Third-Party Storage from an Event

Learn how to configure a workflow to automatically save documents from an
Envelope or Recipient event to a third-party storage provider or System of Record.

## Before you begin

- Your workflow must use the From an Event start method.
- Configure the start method to use one of the Envelope or
  Recipient events.
- The extension app (for example, Google Drive, Box, SharePoint, OneDrive, or Salesforce) for your
  chosen storage provider or System of Record (SOR) is installed on your account. See [this help topic](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=bgb1697668738892.html&_LANG=enus "Learn how administrators locate, install, and authorize third-party extension applications from the Docusign App Center to use inside Workflow Builder.") for more information
  about installing extension apps.

In a workflow that starts from an envelope event, you can use a Store
Files... step to capture documents from the envelope event and securely
archive them to your external cloud storage SOR provider. Using
this feature ensures that your documents are automatically saved with the exact
configuration you need: individual files, a combined PDF, or a zip file.

Note:

The steps for this task show how to use this feature to store files to **Google
Drive**. You can use any of the extension apps that offer a
Store Files... step to archive documents to the storage
or SOR provider of your choice. The details for other apps may differ from what
is presented for the Google Drive app.

1. Edit your workflow and select Add a step >  Store files in Google Drive >  Configure.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/add_store_files_step.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU4MDEsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.T9QcPg7Kw_datKiyl2sY36aFx3_ieNC7RU_T9tO3eIU&_LANG=enus)

   The configuration panel for your Google store files step opens.
2. Select the Envelope Info from the list of Variables from Workflow Start as the source of the files you want to store:
   1. In the What are you storing?  field, select the
      dropdown and choose Variables from Workflow
      Start.
   2. Select Envelope Info.

      ![Screenshot showing initial configuration of the Store files step.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/envelope_info_variable.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU4MDEsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.T9QcPg7Kw_datKiyl2sY36aFx3_ieNC7RU_T9tO3eIU&_LANG=enus "Screenshot showing initial configuration of the Store files step.")
3. Choose how to configure the files you are storing by selecting one of the following options:
   - Combine all documents into a single PDF.
   - Store all documents individually.
   - Combine all documents into a ZIP file.

   (Optional) Check the Include Certificate of Completion box to store this file with your documents. ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/configure_store_files.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU4MDEsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.T9QcPg7Kw_datKiyl2sY36aFx3_ieNC7RU_T9tO3eIU&_LANG=enus)
4. Select the location where the files will be stored, and click
   Next. Depending on which extension app and
   corresponding Store files... step you are using (for
   example, Google Drive, Salesforce, Box, and so on), the specifics for selecting
   the storage location will vary. For example,

   - For Google Drive, select your connection, drive, and folder.
   - For Salesforce, select your connection and a Record id variable.
5. Create a custom file name, and then select Apply:
   1. Use the Add Variable and Add Text buttons to build the file name using envelope data and manual entries.
   2. (Optional) Check the Add numbered suffix to files with duplicate titles box to ensure unique file names.

You have successfully configured the Store Files in Google Drive step
to save documents automatically from the specified envelope event to your third-party
storage location. This task used Google Drive as an example. You can store files to any
supported third-party storage location with the appropriate Store
files... extension app.
