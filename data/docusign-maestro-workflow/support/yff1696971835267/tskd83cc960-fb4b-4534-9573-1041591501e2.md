# Configure a Prepare eSignature Template Step

Learn how to incorporate an eSignature template into your process, prefill recipient fields using upstream variables, and route completed documents to a secure repository.

## Before you begin

- Required: Access to a valid eSignature envelope template to use for your workflow step. For more information about template requirements, see the [Template Requirements](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=xnw1709852160602.html&_LANG=enus "Important information for process builders to consider before starting to develop workflows that include a Prepare eSignature Template step. When you prepare a template to use in a Send Documents for Signature step, there are several requirements and limitations that must be respected to configure the step successfully.") help topic.
- Recommended: If you are planning to map data from a web forms step, review the [Mapping Data From Web Forms](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=cond11d1369-0918-4721-b25c-35149675b539.html&_LANG=enus "Learn the structural requirements and formatting rules for mapping advanced data types, such as checkboxes, radio groups, and dropdowns, from a web form to an agreement template.") help topic.

With the Prepare eSignature Template step, process builders can design a workflow to prepare a document to send for signature. This step uses an envelope template from your account. You can prefill recipient fields by mapping them to workflow variables.

A typical scenario for a Prepare eSignature Template step would be to start with a Collect Data with Web Forms step to gather participant information, such as name, email address, phone number and other data to be used in the document. Next, the Prepare eSignature Template step uses variables from the web form to prefill recipient fields in the document. A Send Documents for Signature step follows using the generated document.

1. Create a workflow as described in [Create a Basic Workflow](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=uhm1697826856678.html&_LANG=enus "Learn the basics of how to create an agreement workflow. Process creators and account administrators can create workflows to automate the agreement process.").
2. Add the steps that you want to precede your Prepare eSignature Template step, such as a Collect Data with Web Forms step.
3. Select Add a step >  Prepare eSignature Template.
4. Select the template to use for the step and select Next.

   The selected template displays with the Mapping fields panel.
5. (Optional) Map workflow variables to template fields.

   Assigning workflow variables will prefill data in recipient fields. You can map any variable generated by steps occurring earlier in the workflow.

   Tip: Use the Map for Me option to generate recommended field mappings. The AI analyzes the **context of the document**—specifically the text surrounding a field and its physical placement—to suggest the best matches. Matches are based on this visual context rather than data labels or other field properties defined in your template. Review and apply the recommendations, or undo the action to use manual mapping.

   ![Assign workflow variables to template fields to prefill data for recipients.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/map_fields_env_template.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU3MjcsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.YccPORBlmvU2yBTrQ0CIQQjuKtQvbyNl2y_nsibyG2k&_LANG=enus "Assign workflow variables to template fields to prefill data for recipients.")
6. Select Next > Apply.

Your Prepare eSignature Template step is configured. You can use the prepared template in a Send Documents for Signature step.
