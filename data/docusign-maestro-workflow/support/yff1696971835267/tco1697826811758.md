# Creating Workflows

Learn how process builders plan, prepare, and structure end-to-end agreement automation using the Workflow Builder design canvas.

This section presents information on how process builders can create agreement workflows. Start
by reviewing [Create a Basic Workflow](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=uhm1697826856678.html&_LANG=enus "Learn the basics of how to create an agreement workflow. Process creators and account administrators can create workflows to automate the agreement process."). Once you have an idea of the kind of
workflow you want to create, gather the following information to prepare:

- How will your workflow be initiated? Manually by someone on your account? Or through an
  external system that will pass data into the process? Reference [Workflow Start Methods](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=ztb1727892686033.html&_LANG=enus "Review the different automated and manual trigger methods available to initiate processes in Workflow Builder, including links, events, API calls, and CLM integrations."), [Enhance CLM Workflows with Workflow Builder Automation](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=hmz1727408977389.html&_LANG=enus "Learn how process builders trigger Workflow Builder processes from Docusign CLM to access advanced delivery options and signature features.")
- Who are the participants who will take part in your agreement process? Reference
  [Define Workflow Participants](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=ypt1698264964168.html&_LANG=enus "Learn how to define, configure, and manage participant roles within your workflow steps to control how users interact with your agreement automation.")
- Do you want to gather data from participants to use in the agreement process? What
  stand-alone web form will you use? Reference [Configure a Collect Data with Web Forms Step](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=gua1698120920620.html&_LANG=enus "Learn how to configure a Collect Data with Web Forms step in an agreement workflow. You use a web forms step to gather information from participants. You can use any existing stand-alone web form on your account that you have access to for your step.")
- Do you need to bring participants into the workflow after it starts by sending an invitation?
  Reference [Configure a Set Up Invite Step](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=kfx1698167344720.html&_LANG=enus "Learn how to use a Set Up Invite step in your agreement workflow to invite participants to join the workflow.")
- Will you have a participant run an identity verification check? Reference [Configure a Verify Someone's Identity Step](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=qht1698126463108.html&_LANG=enus "Learn how to configure a Verify Someone's Identity step in an agreement workflow. You use an identity verification step to verify a participant's identity using Docusign's ID Verification process.")
- What documents do you want to use in the agreement process? Reference [Configure a Prepare Document Template Step](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=cra1749229692878.html&_LANG=enus "Learn how to incorporate a document template into your process to generate customized agreements automatically using workflow variables.") and [Configure a Prepare eSignature Template Step](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=tskd83cc960-fb4b-4534-9573-1041591501e2.html&_LANG=enus "Learn how to incorporate an eSignature template into your process, prefill recipient fields using upstream variables, and route completed documents to a secure repository.")
- Which participants will receive and sign your documents? Reference [Configure a Send Documents for Signature Step](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=uww1698167288211.html&_LANG=enus "Learn how to add and configure a workflow step to send a document for signature.")
- Do you want to use a branching rule in your workflow to adjust for different scenarios? Reference [Configure a Branching Rule](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=mnv1698169910928.html&_LANG=enus "You can create conditional rules in your agreement workflow. These rules validate a condition and branch the workflow based on whether the condition is met or not.")
