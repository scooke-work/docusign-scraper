# Agreement Process Steps

This section provides detailed configuration procedures and requirements for individual workflow steps, helping process builders design automated, end-to-end agreement lifecycles.

This section contains detailed configuration information for key steps in the agreement
process.

[Configure a Collect Data with Web Forms Step](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=gua1698120920620.html&_LANG=enus "Learn how to configure a Collect Data with Web Forms step in an agreement workflow. You use a web forms step to gather information from participants. You can use any existing stand-alone web form on your account that you have access to for your step.")
:   Learn how to design and configure web forms to efficiently gather necessary
    information from participants.

[Configure a Prepare eSignature Template Step](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=tskd83cc960-fb4b-4534-9573-1041591501e2.html&_LANG=enus "Learn how to incorporate an eSignature template into your process, prefill recipient fields using upstream variables, and route completed documents to a secure repository.")
:   Use an envelope template to create a document to send for signature and map data from other workflow steps to template fields.

[Configure a Prepare Document Template Step](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=cra1749229692878.html&_LANG=enus "Learn how to incorporate a document template into your process to generate customized agreements automatically using workflow variables.")
:   Use a document template to generate a document to send for signature.

[Configure a Verify Someone's Identity Step](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=qht1698126463108.html&_LANG=enus "Learn how to configure a Verify Someone's Identity step in an agreement workflow. You use an identity verification step to verify a participant's identity using Docusign's ID Verification process.")
:   Understand the various methods available to verify participant identities,
    ensuring the security and validity of your agreements.

[Configure a Collect Data From an ID Data Step](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=huu1708131883761.html&_LANG=enus "Learn how to configure a Collect ID Data step in an agreement workflow. You use this step to extract information from a participant's government-issued ID to autofill data in later workflow steps.")
:   Gather verified identity data for use in other workflow steps.

[Configure a Send Documents for Signature Step](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=uww1698167288211.html&_LANG=enus "Learn how to add and configure a workflow step to send a document for signature.")
:   Explore the steps to use eSignature templates to obtain signatures and other
    information from workflow participants.

[Configure a Create Request Step](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=task-7234.html&_LANG=enus "Learn how to add a Create Request step to a workflow to generate automatically or interactively a new Agreement Desk request.")
:   Generate a new Agreement Desk request automatically or interactively.

[Configure a Store Files in Google Drive Step](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=wqt1698167316926.html&_LANG=enus "Learn how to configure a Google Drive storage step to automatically archive documents generated in earlier workflow steps.")
:   Save a copy of a workflow document to a Google Drive folder. Google is one of
    several apps that Workflow Builder supports for archiving files.
