# Configure a DS Sender Profile for Workflow Builder Preparer Permissions

Learn how to configure a custom Docusign DS Sender permission profile to grant users Workflow Builder preparer access.

Docusign uses permission profiles to control which actions users are allowed to perform. The default DS Sender profile allows users to view and send envelopes, use templates, and other actions. To enable Workflow Builder functionality for a sender, you need to create a custom permission profile based on DS Sender and add the **Workflows: Prepare** permission. You also need to assign your users the permission profile. The following steps show an example of how to enable Workflow Builder functionality for a new Docusign sender profile.

1. Log in as a Docusign Administrator. You must have administrative access to your Docusign account to manage permission profiles and user settings.
2. Select the Admin tab, and select Permission Profiles in the left navigation menu.

3. Locate the DS Sender profile, and selectActions > Copy.

   ![The screenshot shows the Actions menu where you can copy the DS Sender permission profile.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/ds-sender2.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU2NDAsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.jCdvlU96R2HnQHuVKP_70SBxXirQcbYoDFPNAg_XUPc&_LANG=enus "The screenshot shows the Actions menu where you can copy the DS Sender permission profile.")

   The Copy Permission Profile dialog box opens.
4. Enter a descriptive name for the new permission profile, for example, "DS Sender with Workflow Access" or "Workflow Preparer Profile."
5. Select the User Permissions tab.
6. Scroll down to the  Workflows setting, and select Prepare.

   ![The screenshot shows the Workflows setting in the User Permissions options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/copy_permission_profile.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU2NDAsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.jCdvlU96R2HnQHuVKP_70SBxXirQcbYoDFPNAg_XUPc&_LANG=enus "The screenshot shows the Workflows setting in the User Permissions options.")
7. Select Add to create this permission profile.
8. Select Users in the left navigation menu.
9. Find the user that you want to enable Workflow Builder for.
10. Select Actions > Edit from the dropdown menu.
11. In the Permission Profile dropdown, select the new permission profile that you created for Workflow Builder DS Sender or the descriptive name you applied.
12. Select Save.

The new permission profile using Workflow Builder is applied to the user. Assign the new permission profile to any other users that need preparer access to Workflow Builder.
