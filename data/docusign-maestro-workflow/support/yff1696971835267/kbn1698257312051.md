# Unpublish or Delete a Workflow

As a
process
builder, you can unpublish any of your paused workflows. You can edit and
republish unpublished workflows, or you can permanently delete them.

As a
process
builder, you can unpublish any of your paused workflows. Pausing a
workflow allows any running instances to complete. Once all instances are completed
or canceled, you can unpublish the workflow.

Unpublishing
a workflow helps you manage the quota of active workflows you can have on your
account. If you need to free up capacity, you can pause and unpublish any workflows
you don't need.
Unpublished
workflows do not count towards your account's quota.

Important: If you republish a previously unpublished
workflow, a new workflow URL will be generated for it. You will need to update any
distributed or embedded URLs to the new workflow URL.

You can also permanently delete your unpublished workflows. You cannot recover a deleted
workflow.

1. Log in to your Docusign account and select Agreements >  Workflows.

   The Workflows list shows all workflows that you
   have access to.
2. Locate the workflow you want to unpublish. Filter by Status:
   Published and Created by: Me.
3. Select the workflow more actions menu and choose
   Pause New Instances.

   Your workflow status changes to "Paused". Any running instances will
   continue but no new instances can be started. Once all instances are complete or
   canceled, you can unpublish the workflow.
4. To unpublish the paused workflow, first confirm that there are no running
   instances.
   1. Select View Instances for the workflow.
   2. For any in progress instance, select
      Cancel.![Screenshot of a running instance with the action menu expanded showing the Cancel option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/cancelrunninginstance.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU4ODUsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.U7e0NccLk0h9LyzGrDBvUT5g5ZFh8dW0nXAAaxn9nG4&_LANG=enus "Screenshot of a running instance with the action menu expanded showing the Cancel option.")
   3. Return to the Workflows list.
5. Select the workflow's more actions menu and choose
   Unpublish.
6. In the confirmation window, select Unpublish.

   Your workflow status changes to "Unpublished".
7. (Optional)
   To delete an unpublished workflow, select the workflow more actions menu >  Delete and confirm your action in the confirmation window.

Your workflow is unpublished. If you chose to delete it, it is removed from your account and cannot be recovered.
