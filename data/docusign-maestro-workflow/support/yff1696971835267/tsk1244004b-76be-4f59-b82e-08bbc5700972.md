# Access Rule Logs for Agreement Rules

Learn how to access execution logs for agreement rules used to trigger a Workflow Builder process.

## Before you begin

Ensure you have access to the Agreement Rules list and the necessary permissions to view logs.

When you create a workflow from an agreement rule, the Workflow Builder Activity Logs contain information for the workflow activities. Since the workflow starts from a completed envelope, the Workflow Builder logs won't have details about the envelope.

You can find activity logs pertaining to a workflow-powered rule in the Agreement Rules list. Workflow Builder rule logs are stored separately from other rule logs. Logs for these rules are available through the Action menu for the rule. (Other rules use the Logs tab to record and access activity information.)

1. In Admin > Agreement Actions  > Rules, locate the rule with the "Powered by Workflow Builder" badge.
2. From the Rules tab, select **Action** > **View Instance Logs**.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/view_instance_logs.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU3MTMsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.KH-vWo3Qiv7Q7Sa1sWoJSaR_gNUW9YTvPJ8BN_JOIME&_LANG=enus)
3. Review the list of envelopes using the rule, which provides status and the event date and time of the rule execution.
4. Select **View Details** for complete information on the envelope, including action type, status, and other details.
