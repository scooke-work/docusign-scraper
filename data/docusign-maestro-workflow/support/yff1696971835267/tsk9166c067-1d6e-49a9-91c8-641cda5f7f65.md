# Create a Workflow from an Agreement Rule

Learn how to generate a workflow from an Agreement Rule, using the rule's event as the automation trigger and its archive action as the initial workflow step.

## Before you begin

- You must have administrator privileges to access Agreement Rules.
- Create a rule with an Archive to action. This rule must have the Powered by Workflow Builder tag to be used to create a workflow.![](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/powered_by_Maestro.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU3MDgsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.VJ_VPfzbpByfK9KUhagy7RIhkA0g-fw2z9nf6VTQyi4&_LANG=enus)

This topic explains how to create a workflow from an Agreement Rule.

Important: Only Archive to rules created **after** this feature was released are workflow-powered and available to use to create a workflow.

When you create a workflow from a rule, Workflow Builder uses the rule's event as the workflow trigger and its Archive to action as the initial workflow step.
After the workflow is created, you can open it in Workflow Builder to add more steps.

Activity logs for Workflow Builder-powered agreement rules are available from the Agreement Rules list. See the topic [Access Rule Logs for Agreement Rules](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=tsk1244004b-76be-4f59-b82e-08bbc5700972.html&_LANG=enus "Learn how to access execution logs for agreement rules used to trigger a Workflow Builder process.") for details.

For more information on Agreement Rules, see the help topic [Create and Manage Agreement Rules](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=btt1594673504460.html&_LANG=enus).

1. In Admin > Agreement Actions  > Rules, locate the rule with the "Powered by Workflow Builder" badge.

   At this time, only new rules created with an Archive to action are available for this process.
2. Select Actions > Add Automation Steps.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/add_automation_step.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU3MDgsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.VJ_VPfzbpByfK9KUhagy7RIhkA0g-fw2z9nf6VTQyi4&_LANG=enus)

   A new workflow is created with a From an Event start method. The trigger event is configured with the details from the agreement rule. The trigger event is set to Envelope Completed and the conditions specified in the rule are applied. A file archive step matching the selected rule's action is added to the workflow. ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/workflow_created.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU3MDgsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.VJ_VPfzbpByfK9KUhagy7RIhkA0g-fw2z9nf6VTQyi4&_LANG=enus)
3. Add more steps to the workflow as needed to continue your business process after the archive step.

Your workflow is created from the workflow-powered agreement rule. Note that changes to the rule you used to create the workflow are not reflected in the workflow. The workflow uses a system-generated copy of the rule that cannot be modified. To make changes, edit your workflow details directly.
