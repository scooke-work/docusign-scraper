# View Account Activity Log

Learn how administrators can access an activity log that provides a comprehensive view of all Workflow Builder activity for the account.

## Before you begin

You must be an account administrator or a workflow administrator to view the account-level activity log.

Account and workflow administrators can access an account-level activity log. This log provides a
comprehensive view of all Workflow Builder activity. The log details all actions taken on workflow
definitions and all workflow instances across the account.

1. Log in to your Docusign account and go to Agreements >  Workflows.
2. From the Workflows list, select the more actions menu and choose View Activity Log.

   ![A screenshot of the Docusign Workflows list page, showing a table of active workflows and the open activity log menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/View_activity_log.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU5MjYsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.YGwV3TLFw6e2szV9ZmfhAmwHbfyd2RjF2FZu7YgaJZ4&_LANG=enus "A screenshot of the Docusign Workflows list page, showing a table of active workflows and the open activity log menu.")
3. Review the Activity Log details:

   - **Filter by date**. Use the Date selection to filter the log by a date range.
   - **Order by column**. Select any column to order the log in ascending or descending order.
   - **Page through results**. Use the pagination controls to page through the listings.
   - **Download data**. Use the Download CSV option to generate a CSV file with your data for further processing and analysis.
