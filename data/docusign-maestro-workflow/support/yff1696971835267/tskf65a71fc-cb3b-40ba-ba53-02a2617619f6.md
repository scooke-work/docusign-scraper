# Duplicate a Workflow

Create an identical copy of an existing workflow to accelerate development and reuse standardized logic.

Duplicate an existing workflow to **create a complete, identical copy** and accelerate development. By reusing existing logic and configurations, you can build new processes without starting from scratch. This feature is restricted to workflow owners and administrators.

Important:

- The duplication is based on the **most recently saved version** (including unpublished drafts).
- Templates and web forms are **shared** between the original and the duplicate. They are not copied.
- The new workflow is saved automatically as an **Unpublished Draft**.

1. Browse to the **Workflows** list page or open a workflow in the builder canvas.
2. Access the **Duplicate** command through one of these methods:
   - **From the Workflows List:** Locate the workflow, select the **Menu (...)**, and click **Duplicate**.
   - **From the Builder Canvas:** Open the workflow, click the **Menu (...)** in the top-right corner of the canvas, and select **Duplicate**.![Workflow canvas with the actions menu expanded and the Duplicate option selected.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/duplicate_wf.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU4NjgsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.WFSc-s3J5ouwQdFZCnNm3I8kL2jlLJpmBITfvquETYY&_LANG=enus "Workflow canvas with the actions menu expanded and the Duplicate option selected.")

   The workflow is duplicated and open for editing.
3. (Optional) Rename the duplicate workflow.

   "COPY" is appended to the beginning of the workflow name, and the status is "Draft (Unpublished)". Click the Menu (...) and select Rename.
4. Select Save Draft to save your duplicate workflow.

The workflow is duplicated and added to your Workflows list.
