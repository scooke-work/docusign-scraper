# Cancel a Running Instance

You can cancel any in progress instance of a workflow.

You can cancel any in progress instance of a workflow. Once an instance is complete or failed,
you can no longer cancel it.

You cannot restart an instance once you cancel it. Participants can still complete any
in-progress steps, but further steps in the workflow will not trigger.

1. From the Workflows list, select View Instances for a workflow to see the Instance
   list.
2. For any in progress instance, select Cancel.

run, but no additional workflow steps will execute.
