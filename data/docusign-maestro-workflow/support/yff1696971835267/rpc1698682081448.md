# Configure a Show a Confirmation Screen Step

Learn how to add a Show a Confirmation Screen step to your
workflow. This step generates a message screen that appears to a
specific workflow participant to convey information about their progress.

Use the Show a Confirmation
Screen step to show an onscreen message to a specific workflow
participant. A confirmation step generates a message screen for specific workflow participants. By supporting rich text formatting and workflow variables, these screens deliver visually engaging, dynamic content that clearly guides users through their next steps. Use one of the preconfigured messages or customize the message with your own content.

Confirmation steps are usually added to the end of the
participant's actions to inform them they are finished. They are
also used with branching rules to handle a failed state when the rule conditions are not met.
For information on branching rules, see [this
topic](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=mnv1698169910928.html&_LANG=enus "You can create conditional rules in your agreement workflow. These rules validate a condition and branch the workflow based on whether the condition is met or not.").

1. Create a new workflow as described in the topic [Create a Basic Workflow](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=uhm1697826856678.html&_LANG=enus "Learn the basics of how to create an agreement workflow. Process creators and account administrators can create workflows to automate the agreement process.").
2. After a step where you want a confirmation message, select Add a step >  Show a Confirmation Screen >  Configure.

   The configuration panel for your step opens with the default message type
   selected. ![Screenshot of the configuration panel for a confirmation step.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/confirmationconfigurator.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU4MjQsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.HduR0JX1qkihiqF7bAW477u77mxM3ArZ36AR_scVsnA&_LANG=enus "Screenshot of the configuration panel for a confirmation step.")
3. Select the Participant who will see the confirmation
   message.
4. Select the Message type you want to use. You can use one
   of the pre-built messages or write your own custom message.

   Use the rich text formatting and add workflow variables to create engaging, personalized content for your participant. Type "/" to select a variable. ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/message_with_variable.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU4MjQsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.HduR0JX1qkihiqF7bAW477u77mxM3ArZ36AR_scVsnA&_LANG=enus)
5. Select Apply to save your configuration.

Your Show a Confirmation Screen step is
configured. When a participant reaches this step in your workflow, your message will
display on their screen.
