# Monitor a Running Workflow Instance

Learn how to check the progress of the agreement instances you start from a workflow.

The workflow details views show you all agreement instances for the workflow. You can see all
in progress, completed, failed, and expired processes. You can see status details on each
step in the workflow and get information such as the completion time or reason for
failure.

Before you begin
:   - You must have Workflows: Prepare user permissions or greater.
    - Your account must have at least one workflow with one or more
      agreement instances.

1. Log in to your Docusign account and select Agreements >  Workflows.

   The Workflows list shows all of the workflows that you can
   access.
2. Locate the workflow that you want to use. Select the more actions
   menu and choose View Instances.

   The list of workflow instances displays.
3. Locate the instance you want to check. The PROGRESS
   column shows the state that the process is in.

   Tip: Use the Status
   filter to locate instances with errors you need to address. Or use the
   Search box to search for a specific instance name
   or ID.

   ![Screenshot of the instance list for a workflow with the Status filter expanded.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/maestroinstancefilters.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU5MTIsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.sDrmGta568SswF3KFVo0PSTpjCWbNdSJP4HnmJlZANY&_LANG=enus "Screenshot of the instance list for a workflow with the Status filter expanded.")
4. For more information about the process, select the action
   menu and choose View details.

   The workflow canvas opens.
5. Select any workflow step and click View Details.

   - Details include step start and completion times and step participant
     role names.
   - For an in progress instance, the current step is outlined in black and labeled Active step.
   - Failed steps have a red outline and you can see what action to take.

   Note: If a failed step involves an extension app, check
   with your account administrator to see if the app is still installed and
   authorized for use.
