# Import a Workflow

Learn how to import a workflow into your account.

## Before you begin

- You must have Workflows: Create permissions to import
  workflows.
- If the workflow includes web form or eSignature steps, you'll need the
  permissions to create those assets as well.
- If the workflow includes any third-party extension app steps, those apps must be
  installed and connected before you import the workflow. Extension apps are
  managed by your account administrator.
- Have the ZIP file with the exported workflow available to import.

You can import an exported workflow into your account. You can also clone a workflow
by exporting it and then importing it to the same account.

1. Log in to your Docusign account and go to Agreements >  Workflows.
2. Select Create Workflow.

   ![Screenshot of the Create a workflow view and the Import Workflow option.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/importworkflowbutton.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU4OTQsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.1W5SlsnZCXLlnqQh2uzAyCWjHEvQCnB8ibIzp703lh0&_LANG=enus "Screenshot of the Create a workflow view and the Import Workflow option.")

   The Create a workflow view opens.
3. Select the Import Workflow tile.
4. In your file picker, open the zipped file containing the workflow export.

   The import process begins. The import may take a few minutes. Do not
   navigate away from the process until it finishes. When the import process is
   complete, your new workflow opens in the editing view.
5. Review each step in the workflow. Confirm or update step configurations as
   needed.
6. Publish the workflow or save the draft.

The workflow is imported into your account.
