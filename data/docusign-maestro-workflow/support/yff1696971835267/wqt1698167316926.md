# Configure a Store Files in Google Drive Step

Learn how to configure a Google Drive storage step to automatically archive documents generated in earlier workflow steps.

## Before you begin

- You must have [installed and authorized](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=ctg1698170340729.html&_LANG=enus) the Google Drive extension app on your account.
- You must be an account administrator or have Workflows: Create user permissions as described in the [Workflow Builder Roles and Permission Requirements](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=gbb1696973048215.html&_LANG=enus "Read about the different roles involved with creating and using workflows with Docusign Workflow Builder. Each role requires specific user permissions.") article.
- You must have a supported file type (Google Sheets, CSV, or MS Excel) saved on your configured Google Drive.

You can use the Store Files in Google Drive step to save documents generated by Send Documents for Signature or Verify Someone's Identity workflow steps. Generally, you will position your Google Drive archive step as follows depending on the use case:

- **Archive completed documents.** Add the Store Files... step at the end of your workflow to help ensure you are archiving a completed version of the document. Whatever state a document is in when the archive step generates is what is saved to the specified file location.
- **Archive an ID Evidence file**. If you want to access and export this file generated by a Verify Someone's Identity step, add the Store Files... step after the identity step.

Important:

Google Drive is one of several third-party storage apps supported for storing and archiving workflow files. For information on other file storing apps, see the [Configure Docusign Apps in Agreement Workflows](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=hno1706034267152.html&_LANG=enus) article.

1. Create a new workflow as described in [Create a Basic Workflow](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=uhm1697826856678.html&_LANG=enus "Learn the basics of how to create an agreement workflow. Process creators and account administrators can create workflows to automate the agreement process.").
2. Select Add a step >  Store Files with Google Drive >  Configure.

   The configuration panel for your Google store files step opens.
3. Select the file to archive and select Next.

   Documents and proof files from any earlier Send Documents for Signature (Envelope) or Verify Someone's Identity (ID Evidence) steps are available to archive.
4. Specify where to archive the file:
   1. The default connection is the Google account configured by your account administrator.
   2. Select the drive. All available drives connected to the Google account display.
   3. (Optional) Select a folder on the selected drive. Listed are all existing folders and an option to create a new folder. For a new folder, you can create a folder name with text and any available workflow variables.

      ![Create a new folder for Google Drive using text.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/createnewgooglefolder.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU3OTgsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.HklpnyJcvJ3KqbRjO1x3Wl12MagzFXLBBHeWCX5zeQ8&_LANG=enus "Create a new folder for Google Drive using text.")
   4. Once you define the full path for the stored file, select Next.
5. Specify a file name for your archive. Use the text and variable controls to build your file name.

   ![](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/filenamegooglearchive.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU3OTgsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.HklpnyJcvJ3KqbRjO1x3Wl12MagzFXLBBHeWCX5zeQ8&_LANG=enus)
6. Review the completed file path and select Apply.

Your Store Files in Google Drive step is configured.
