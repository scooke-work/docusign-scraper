# Installing Extension Apps for Workflow Builder

Learn how administrators locate, install, and authorize third-party extension applications from the Docusign App Center to use inside Workflow Builder.

In addition to Docusign apps, administrators can get extensions applications from the Docusign App Center. An extension app is a type of application that enables you to integrate
third-party application functionality into the Workflow Builder platform. You can include
workflow steps to trigger actions by an installed extension (such as Google Drive or
SharePoint), automating more of your agreement process.

Note: The HTTP app is listed on the App Center but does not need to be installed. Unlike third-party extensions that you must manually install from the Docusign App Center, the HTTP Web Request module is a **built-in** feature. It is automatically available for all accounts on IAM plans with Maestro add-ons.

You can install a variety of file storage and data write-back partner apps. More partner app
extensions will be added in future releases. As an administrator, you access the
[Docusign App Center](https://apps.docusign.com/app-center) and log in with your
Production or Demo Docusign account. You then install
and authorize extension apps for your account, enabling workflow creators to add
steps into a workflow that use the installed partner applications.

For more information for administrators on how to view, install, and manage partner extension
apps, see the [Docusign App Center
Guide](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=ijy1698170251179.html&_LANG=enus).

Note: Developers can review our extensive API documentation on building
extension apps for the Docusign App Center. See [Extension apps 101 overview](https://developers.docusign.com/extension-apps/extension-apps-101/) on the Docusign Developer
Center.

1. Log in to your Docusign account with account administrator credentials.
2. Navigate to the App Center in one of the following ways:
   - Open a new browser window, navigate to the appropriate environment for
     your account:
     - PROD <https://apps.docusign.com/app-center>
     - DEMO <https://apps-d.docusign.com/app-center>
   - Open the workflow designer view (create or edit a workflow) and select
     the App center link.
3. Review the available partner extension apps and install the apps you want to
   add to your account.

   For more information for administrators on how to view, install, and manage partner
   extension apps, see the [Docusign App
