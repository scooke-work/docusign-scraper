# Export a Workflow

Learn about exporting a published workflow from your account.

## Before you begin

- For some workflows, you may encounter this error when you attempt to export:
  ![Screenshot of an error message when exporting an older published workflow.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/exportworkflowrepublisherror.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU4OTIsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.JVZ93dEjc0HZoEILMwz2vxEp8wnnccHVcRKcXRG6SCE&_LANG=enus "Screenshot of an error message when exporting an older published workflow.")

  If this happens, you must republish the workflow. This may require you to
  edit the workflow to conform to current requirements.
- You must have Workflows: Create permissions to export
  workflows.

You can export any published workflow available in your account.

1. Log in to your Docusign account and go to Agreements >  Workflows.
2. Locate the published workflow that you want to export. Use the
   Status filter to get a list of your published
   workflows.
3. In the more actions menu for the workflow, select
   Export.

   ![Screenshot of the menu control to export a workflow.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/exportworkflowmenu.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU4OTIsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.JVZ93dEjc0HZoEILMwz2vxEp8wnnccHVcRKcXRG6SCE&_LANG=enus "Screenshot of the menu control to export a workflow.")
4. In the Export this workflow? confirmation screen, select
   Export.

A ZIP file is generated and exported to your local downloads
folder. The ZIP file name is the workflow name. The file contains the workflow and all
of the configuration details.

## Next steps

Use the exported ZIP file to [import the workflow](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=zoi1742412999176.html&_LANG=enus "Learn how to import a workflow into your account.") to another account. You can
also import it to the same account to create a complete copy of the workflow.
