# I Cannot Find the Identity Verification Configuration I Want for a Verify Someone's Identity Step

Troubleshoot why a specific identity verification configuration does not appear in your drop-down menu when configuring a
Verify Someone's Identity step in Workflow Builder.

I can't see the identity verification configuration I want in the Identity
Verification Configuration drop-down list of the Verify
Someone’s Identity step. Although this configuration exists in the
Identity Verification settings, I can’t find a way to use it in my workflow.

The identity verification configuration you want to add to your Verify
Someone’s Identity step has the following characteristics:

- It is an ID Verification for EU Qualified
  configuration or a custom configuration based on ID Verification
  for EU Qualified
- It includes the Manually review the ID document
  option
- It includes the Allow SMS authentication after one successful
  Photo ID or Passport verification option

Any identity verification configuration that includes either of those
characteristics is not supported in Workflow Builder and cannot, therefore, appear in the
Identity Verification Configuration drop-down list of
the Verify Someone’s Identity step.

You can remove these non-compliant
options from your identity verification configuration or choose a configuration
that does not include them.

For more information on modifying configurations,
see the topic [Edit Identity Verification Configurations](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=uwh1583277457043.html&_LANG=enus)
