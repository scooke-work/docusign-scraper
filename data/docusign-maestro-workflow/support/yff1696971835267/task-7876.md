# Configure an Export Data to SharePoint Step

Learn how to configure a workflow step to export data in a tabular format to SharePoint.

You can export variables, including eSignature and web form data, to tabular format files in SharePoint to analyze and extract meaningful insights. Use the **Export Data to SharePoint** step to select and export these variables.

The following file formats are supported for data export:

- CSV
- MS Excel

Important:

The **Export Data to SharePoint** step only supports export of **string-type** workflow variables to SharePoint.

## Before you begin

- You must have [installed and authorized](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=ctg1698170340729.html&_LANG=enus) the SharePoint extension app on your account.
- You must be an account administrator or have Workflows: Create user permissions as described in the [Workflow Builder Roles and Permission Requirements](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=gbb1696973048215.html&_LANG=enus "Read about the different roles involved with creating and using workflows with Docusign Workflow Builder. Each role requires specific user permissions.") article.
- You must have a supported file type (CSV or MS Excel) saved on your configured SharePoint account.

1. Create a workflow as described in the [Create a Basic Workflow](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=uhm1697826856678.html&_LANG=enus "Learn the basics of how to create an agreement workflow. Process creators and account administrators can create workflows to automate the agreement process.") topic.
2. Add the required step to provide data to export to SharePoint (for example, **Collect Data with Web Forms** step or **Send Documents for Signature** step).
3. Select Add a step > Export Data to SharePoint > Configure.

   The configuration panel for the Export Data to SharePoint step opens.
4. Select where to store the data:
   1. Select the existing connection from the Select connection dropdown list where the destination tabular sheet (CSV or MS Excel) exists.

      Note: If the required SharePoint connection is not available in the dropdown list, create the connection using the Add New Connection button.
   2. Select Choose File.

      ![Screenshot to select the required connection and file to export data.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/maestro-sharepoint-choosefile.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU4MDgsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.tUq8w7zGtsUYvNsCu89DIbMYFd43SHM8_VVaGXuB_RE&_LANG=enus "Screenshot to select the required connection and file to export data.")

      The Select files from SharePoint dialog box opens.

      Note: All available sites and drives connected to the SharePoint account display.
   3. Select the required SharePoint site and select Next.
   4. Select the required drive and select Next.

      Note: The drive is also referred to as **Document Library** in SharePoint.
   5. Go to the file to use for the export and select the checkbox.
   6. Select Next.
5. Select an export method to export data variables.

   ![Screenshot to select the required export method and variables to export data.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/maestro-sharepoint-addvariables.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU4MDgsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.tUq8w7zGtsUYvNsCu89DIbMYFd43SHM8_VVaGXuB_RE&_LANG=enus "Screenshot to select the required export method and variables to export data.")

   - [**Bulk Export**]: Use this method to export all envelope metadata to the tabular format:
     1. Select Bulk export.
     2. Select Done.
     3. Select a data source for the export.
     4. Choose the Envelope Info variable to use from the dropdown list.

        Note: Any envelope generated by earlier workflow steps is available here to select.
   - [**Specific Export**]: Use this method to export specific workflow variables to the tabular format:
     1. Select Specific export.
     2. Select Done.
     3. Select the Add variable button.
     4. Select the Search for variable or step dropdown list to select the required data variables.
     5. Repeat steps **c** and **d** to add more variables.

   Important: To ensure that data is exported correctly, the column
   headers in the destination file must exactly match the names of your selected data fields. If a
   match is not found, the system will create a new column for that data.
   Columns in the destination file that are not configured for export will be
   left blank.
6. Select Apply.

You have now configured an Export Data to SharePoint step in a workflow.
