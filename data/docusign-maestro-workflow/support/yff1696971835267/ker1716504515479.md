# Configure a Send an Email Step

Learn how to add an email notification step to your agreement workflow. This step
generates an ad hoc email notification to convey information to any workflow participant or
other named individual.

As part of a workflow, you can automatically generate an ad hoc email notification.
The email notification step is useful to convey information to one or more
individuals at a particular point in the workflow.

For the notification, you can hardcode recipients, use starting variables, or any other email
addresses gathered in prior workflow steps. You can include up to 10 recipients in
an email notification step.

These automated emails are customized with your account's default signing brand to maintain a professional and trusted appearance. If no default brand is defined, standard Docusign branding is used.

This step generates an email message to the specified recipients. The email comes from the
sender "Docusign Workflows" at "workflows@docusign.net". All step
recipients receive the same email message. If you want to generate unique messages,
add a separate step for each message you want to send.

Some example use cases for this step:

- Notify a customer that their account application was approved.
- Notify an internal stakeholder about the status of a workflow instance.

1. Create a new workflow as described in the topic [Create a Basic Workflow](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=uhm1697826856678.html&_LANG=enus "Learn the basics of how to create an agreement workflow. Process creators and account administrators can create workflows to automate the agreement process.").
2. At the point in the workflow where you want to send the email, select Add a step >  Send an Email >  Configure.

   The configuration panel for your step opens for you to add the recipients
   and email message.
3. Add up to 10 email recipients. You can select any workflow email variable, or
   you can add hardcoded email addresses.
4. Compose the message. Add a subject, headline, and body text.

   For the body text, use the rich text formatting and add workflow variables to create engaging, personalized content for your recipients. Type "/" to select a workflow variable.

   ![Screenshot of the configuration panel for the Send an Email step.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/sendemailconfigurator.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU4MjcsInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.1lDm-QyWGxZVs-hrFUvw2MRk3ZyxZ8aELEj1ve1Ilxs&_LANG=enus "Screenshot of the configuration panel for the Send an Email step.")
5. (Optional) Select the Preview tab to see how your message will appear.
6. Select Apply.

Your Configure a Send Email Notification step is configured.
