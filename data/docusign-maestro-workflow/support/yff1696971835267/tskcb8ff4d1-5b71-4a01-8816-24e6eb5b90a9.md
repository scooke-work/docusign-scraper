# View Workflow and Instance Activity Logs

Learn how process builders can view activity for their workflows, covering both actions on workflow definitions and all instances for these workflows.
Preparers can access the instance activity log to view activity for the instances they initiate.

## Before you begin

You must be either a process builder or preparer as defined in the topic [Workflow Builder Roles and Permission Requirements](https://support.docusign.com/s/document-item?bundleId=yff1696971835267&topicId=gbb1696973048215.html&_LANG=enus "Read about the different roles involved with creating and using workflows with Docusign Workflow Builder. Each role requires specific user permissions."). Account and workflow administrators have access to all Workflow Builder activity log data.

Process builders and preparers can access activity logs specific to the workflows and instances they own.

- **Process builders**: can access a workflow activity log for the workflows they create. This log provides a detailed view of all activity for a given workflow. These details include all actions on the workflow definition and details of all instances for the workflow.
- **Preparers**: can access the instance activity log. This log provides a detailed record of all workflow steps for all instances initiated by the preparer.

1. Log in to your Docusign account and navigate to Agreements >  Workflows.
2. For **process builders**, from the Workflows list, select the more actions menu and choose View Activity Log.

   ![A screenshot of the Docusign Workflows list page, showing a table of active workflows and the open activity log menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/View_activity_log.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU5MjksInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.aYlctzmefs2U7MXz09ZuiGJ_AtR5hyv9YufrU5qVAPI&_LANG=enus "A screenshot of the Docusign Workflows list page, showing a table of active workflows and the open activity log menu.")
3. For **preparers**, from the Workflows list:
   1. Locate the workflow you ran and select the more actions menu.
   2. Choose View Instances.
   3. On the Instances page, select the more actions menu for the specific instance and choose View Activity Log.

   ![A screenshot of a workflow's Instances page, showing an individual instance with its status and an open more actions menu.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yff1696971835267/page/images/instance_activity_log.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDU5MjksInNoZWFmIjoieWZmMTY5Njk3MTgzNTI2NyJ9.aYlctzmefs2U7MXz09ZuiGJ_AtR5hyv9YufrU5qVAPI&_LANG=enus "A screenshot of a workflow's Instances page, showing an individual instance with its status and an open more actions menu.")
4. For either log, review the details:

   - **Filter by date**. Use the Date selection to narrow the log to a specific time range.
   - **Order by column**. Select any column to order the log in ascending or descending order.
   - **Page through results**. Use the pagination controls to navigate through long lists.
   - **Download data**. Use the Download CSV option to generate a file for further processing and analysis.
