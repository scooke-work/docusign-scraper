# Install Docusign Apps for Data Verification

Learn how to install Data Verification apps from the Docusign App Center.

Docusign App Center is a place to discover and install partner applications that extend Docusign product capabilities. The App Center provides apps that you can use for Data Verification in your Docusign envelopes or templates.

For information about Docusign App Center, see the [Welcome to Docusign App Center](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=ijy1698170251179.html&_LANG=enus) article.

Note: See the [Create a Private Docusign App](https://support.docusign.com/s/document-item?bundleId=qki1698158012210&topicId=bpc1740167704919.html&_LANG=enus "Learn how to create a private Custom private app for data verification , eSignature, or other Docusign features. Private apps provides custom data verification fields for your organization.") article for information about creating and installing a Custom Data Verification app for your organization.

1. Determine your assigned permissions as described in the  [Permission Profiles](https://support.docusign.com/s/document-item?bundleId=pik1583277475390.ditamap&topicId=pof1583277362435.html&_LANG=enus) article.

   Your permissions determine whether you can:

   - As a Docusign administrator, install and authorize app
   - As an account user (non-administrator), install apps approved for your use.
2. Install Docusign App Center apps as described in the [Install Apps in the Docusign App Center](https://support.docusign.com/s/document-item?bundleId=ous1698169987748&topicId=ctg1698170340729.html&_LANG=enus) article.
