# Select Data Verification Fields for Envelopes and Templates

Learn how to select and include Docusign Data Verification fields from third-party applications for documents in Docusign envelopes or templates.

## Before you begin

Make sure the Data Verification apps you want to use have been installed by someone with administrator privileges as described in the [Install Docusign Apps for Data Verification](https://support.docusign.com/s/document-item?bundleId=qki1698158012210&topicId=pyp1698158616536.html&_LANG=enus "Learn how to install Data Verification apps from the Docusign App Center.") article.

Data Verification fields are a type of eSignature agreement preparation field that can connect with third-party applications. Use these fields to verify information such as email or phone number in real-time. Use these fields to verify data in envelopes or templates. For detailed information about setting up fields for an envelope or template, see the [Document Custom Fields](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=crp1583277382187.html&_LANG=enus) article or the [Create a template](https://support.docusign.com/s/document-item?bundleId=xry1643227563338&topicId=lsg1578456368117.html&_LANG=enus) article.

Note: Docusign offers another option to verify signer input: Data Validation for text fields. With Data Validation for text fields, your signer is prompted to provide correctly formatted data. See the [Data Validation for Text Fields](https://support.docusign.com/s/document-item?bundleId=gbo1643332197980&topicId=jgd1578456357365.html&_LANG=enus) article to learn more about this feature.

1. Start an envelope with the Send an Envelope option or use a template.
2. Upload your files and add recipients as usual.
3. On the Add Fields page, select a data verification field.
   1. Basic Data Verification fields display in the Standard fields section.

   2. Premium Data Verification fields display in the Custom fields section.

      ![A screenshot of selecting a Premium Docusign Data Verification field.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qki1698158012210/page/images/dvnewpremiumselect.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDcyMzIsInNoZWFmIjoicWtpMTY5ODE1ODAxMjIxMCJ9.VNsFoEe12hiPkO0UQI27t7HR0v2G4dJNQiQgZq28GR8&_LANG=enus "A screenshot of selecting a Premium Docusign Data Verification field.")

   For details about Basic and Premium field types, see the [Docusign Data Verification Field Types](https://support.docusign.com/s/document-item?bundleId=qki1698158012210&topicId=omw1740164520691.html&_LANG=enus "Learn about the different types of Docusign Data Verification Fields that you can use in your eSignature envelope preparation.") article.
4. Drag the appropriate fields to the document.

   Tip: You can select a field group where users can specify information types, such as the bank account type information.

   The following example shows selected email address and Social Security number information Data Verification fields.

   ![A screenshot of a sample field group for email and social security number information.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qki1698158012210/page/images/dvfieldgroup.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDcyMzIsInNoZWFmIjoicWtpMTY5ODE1ODAxMjIxMCJ9.VNsFoEe12hiPkO0UQI27t7HR0v2G4dJNQiQgZq28GR8&_LANG=enus "A screenshot of a sample field group for email and social security number information.")

   Note: If you attempt to delete an individual required field from a group, the following warning displays and you cannot change the field group.

   `Individual fields cannot be deleted from a verification group. Select all fields in the group to delete them.`
5. Review the verification type and the apps that work with the selected verification fields.
6. Select Must verify to sign to prevent recipients from completing the envelope unless this verification successfully passes.

   Note:

   The Must verify to sign is an optional setting. Be selective when selecting this setting to avoid false negatives. Even if you choose not to turn on the Must verify to sign setting, the data entered into your document's data verification fields are verified in real-time during signing.
7. After you've added all fields, select Send.

   For detailed information about sending envelopes, see the [Sending Documents for Signature](https://support.docusign.com/s/document-item?bundleId=ulp1643236876813&topicId=lak1578456412477.html&_LANG=enus) article.
8. Optionally track data verification data in the envelope history and Certificate of Completion (CoC) as described in the [Review Data Verification Events](https://support.docusign.com/s/document-item?bundleId=qki1698158012210&topicId=cxe1701287400366.html&_LANG=enus "Learn how to use your Docusign account's envelope history and Certificate of Completion to determine whether the data verification passed or failed.") article.
