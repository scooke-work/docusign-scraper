# Example: Enable Email Data Verification Fields

Learn how to select and include email Data Verification Fields for documents in Docusign envelopes or templates.

This example shows how to use email Data Verification Fields to verify email addresses for
documents in Docusign envelopes or templates.

1. Start an envelope with the Send an Envelope option or use
   a template.
2. Upload your files and add recipients as usual.
3. In the field explorer, select the Custom Fields
   icon.
4. Select the Email Address field, and drag the
   Email Address fields to the document.
5. Select the Email Address to display the customization
   pane on the right side of the page.
6. Select an email address verification app.

   ![A screenshot of email Data Verification field configuration.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qki1698158012210/page/images/dvnewemailconfigoptions.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDcyMzYsInNoZWFmIjoicWtpMTY5ODE1ODAxMjIxMCJ9.x0Olo4Yjxk2i2COCsa4h8cY_4HtiXdMfE4Bc9JDwvbI&_LANG=enus "A screenshot of email Data Verification field configuration.")
7. Select Must verify to sign to prevent recipients from
   completing the envelope unless this verification successfully passes.

   Note:

   The Must verify to sign is an optional setting. Be
   selective when selecting this setting to avoid false negatives. If you
   choose not to turn on the Must verify to sign
   setting, the data entered into the verification fields are verified in
   real-time during signing.
8. After you've added any additional fields, select
   Send.

   For detailed information about sending envelopes, see the [Sending Documents for
   Signature](https://support.docusign.com/s/document-item?bundleId=ulp1643236876813&topicId=lak1578456412477.html&_LANG=enus) article.
