# Example: Enable Phone Number Data Verification Fields

Learn how to select and include Phone Number Data Verification Fields for documents in
Docusign envelopes or templates.

This example shows how to select and include phone number Data Verification Fields with Vonage
for documents in Docusign envelopes or templates.

1. Start an envelope with the Send an Envelope option or use
   a template.
2. Upload your files and add recipients as usual.
3. In the field explorer, select Phone Number to view the
   Phone Number fields.

   ![A screenshot of the Data Verification phone number fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qki1698158012210/page/images/dvphonefields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDcyMzksInNoZWFmIjoicWtpMTY5ODE1ODAxMjIxMCJ9.Xq9M6wve9PskxnUYCkjWFYDsN_ryCn-c4DyXUpb2iHw&_LANG=enus "A screenshot of the Data Verification phone number fields.")

   Important: Signers will not be allowed to
   continue unless they complete all the Phone Number fields. Signers must
   enter both the Country Code and Phone Number for data verification.
4. Drag the Phone Number fields to the document.
5. Select the Phone Number.

   The Phone Number fields display on the right side of
   the page. Make sure Vonage displays in the App field.

   ![A screenshot of the Data Verification phone number field configuration.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qki1698158012210/page/images/dvvonagecustom.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDcyMzksInNoZWFmIjoicWtpMTY5ODE1ODAxMjIxMCJ9.Xq9M6wve9PskxnUYCkjWFYDsN_ryCn-c4DyXUpb2iHw&_LANG=enus "A screenshot of the Data Verification phone number field configuration.")
6. Select Must verify to sign to prevent recipients from
   completing the envelope unless this verification successfully passes.

   Note:

   The Must verify to sign is an optional setting. Be
   selective when selecting this setting to avoid false negatives. If you
   choose not to turn on the Must verify to sign
   setting, the data entered into the verification fields added to your
   document are still verified in real-time during signing.
7. After you've added any additional fields, select
   Send.

   For detailed information about sending envelopes, see the [Sending Documents for
   Signature](https://support.docusign.com/s/document-item?bundleId=ulp1643236876813&topicId=lak1578456412477.html&_LANG=enus) article.
