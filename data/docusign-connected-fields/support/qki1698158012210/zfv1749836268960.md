# Auto-install for Docusign Data Verification Apps

When you select Data Verification Fields, the auto-install feature installs the Docusign Data Verification app in the
background,
without requiring further user action..

The auto-install feature streamlines deploying Docusign Data
Verification packaged extension apps.

When you [Select Data Verification Fields for Envelopes and Templates](https://support.docusign.com/s/document-item?bundleId=qki1698158012210&topicId=yvx1699550441650.html&_LANG=enus "Learn how to select and include Docusign Data Verification fields from third-party applications for documents in Docusign envelopes or templates.") and select the field, the app auto-installs
in the background, without requiring further user action.

Packaged extension apps are included in plans at no extra cost and do not require
connections with third-party accounts to be created.

Packaged Data Verification apps are designated as Approved Apps for installation.

See the [Non-admin Users: Work with Approved
Apps](https://support.docusign.com/s/document-item?bundleId=xni1748970087448&topicId=xni1748970087448.html&_LANG=enus) article for information about using Approved Apps.

These apps are currently
auto-installed
for entitled accounts.

- [Smarty (postal address verification)](https://apps.docusign.com/app-center/app/04bfc1ae-1ba0-42d0-8c02-264417a7b234)
- [Twilio](https://apps.docusign.com/app-center/app/6ff9ae39-ad45-4d04-b0c2-a6e2214f5925) (email verification)
- [SSN and FEIN Verification](https://apps.docusign.com/app-center/app/b1adf7ad-38fd-44ba-85f5-38ea87f4af23) (by LexisNexis)

  Note: U.S. customers only. This app requires the Premium Data
  Verification add-on SKU or the IAM for CX plan.
- [Vonage](https://apps.docusign.com/app-center/app/d16f398f-8b9a-4f94-b37c-af6f9c910c04) (phone verification)
