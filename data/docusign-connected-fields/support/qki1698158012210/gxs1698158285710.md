# Welcome to Docusign Data Verification

Learn about the Docusign Data Verification feature that provides real-time
database verification for increases accuracy, trust, and decreased document
errors.

## Data Verification overview

Docusign Data Verification enables real-time database verification that
increases accuracy and trust and helps decrease document errors. If you capture
personal and business-critical data through agreements, you can easily adopt and
integrate Data Verification with prebuilt components powered by trusted apps.

You can use the prebuilt apps from the Docusign App Center or create your
own private apps. See the [Extension Apps 101](https://developers.docusign.com/extension-apps/extension-apps-101/) article for detailed information
about building extension apps.

With these features, your organization can create accurate signer information and
deliver a streamlined user experience, all while upholding compliance standards.

With Data Verification, you can:

- Verify data against trusted databases in real-time to reduce Not-In-Good-Order
  (NIGO) documents caused by incomplete fields or errors.
- Maintain compliance with a comprehensive audit log that tracks envelope history
  and provides evidence of verification.
- Eliminate manual verification processes that cost time and money downstream with
  packaged applications that work immediately.
- Confirm signer-provided data instantaneously, decreasing having to rework after signing and
  boost confidence in information accuracy.

The Data Verification Fields are available when creating an envelope or template as described
in the [Select Data Verification Fields for Envelopes and Templates](https://support.docusign.com/s/document-item?bundleId=qki1698158012210&topicId=yvx1699550441650.html&_LANG=enus "Learn how to select and include Docusign Data Verification fields from third-party applications for documents in Docusign envelopes or templates.") article.

For information on Data Verification extension apps, see the [Docusign Data Verification Extensions](https://support.docusign.com/s/document-item?bundleId=qki1698158012210&topicId=cfj1715705896400.html&_LANG=enus "Learn about the different Docusign data verification extensions that you can integrate in your Docusign documents.") article.

## Required permissions

To install Docusign apps, you need Docusign
administrator permissions.

The Docusign Admin for a Docusign eSignature account
must install the extension applications for Data Verification. Learn about assigning
permissions in the [Permission Profiles](https://support.docusign.com/s/document-item?bundleId=pik1583277475390&topicId=pof1583277362435.html&_LANG=enus) article.

Tip: To use DocusignData Verification Fields,
you or your Docusign administrator need to install and configure
the associated app before you can use them. See [Install Docusign Apps for Data Verification](https://support.docusign.com/s/document-item?bundleId=qki1698158012210&topicId=pyp1698158616536.html&_LANG=enus "Learn how to install Data Verification apps from the Docusign App Center.")
to learn how.

## Supported regions

Basic Data Verification apps are globally available.

Premium Data Verification apps are only available in the U.S.

Additionally, some apps are specific to certain regions. You may not see
all apps if it’s not deployed in your region.

  - Connected Fields API

    The Connected Fields API lets developers discover custom,
    extension, or Docusign data models for installed extension apps.
    Data models are mapped to and rendered as Data Verification
    Fields.

    You can add Data Verification Fields to envelopes
    through the  [eSignature REST API](https://developers.docusign.com/docs/esign-rest-api/) that uses the
    output of the Connected Fields API discovery endpoint.

    See
    the following references for detailed information:

    - [Connect Fields API](https://developers.docusign.com/docs/connected-fields-api/concepts/) article
    - [Docusign IAM APIs: How to
      Integrate Data Verification using the Connected Fields

  - Connected Fields extensions

    Connected fields extensions perform custom
    data verification for any type of value. The connected fields
    extensions are validated against your system of record, and display
    autofill values.

    For information about using Connected fields
    extensions, see the [Connected fields extension
    overview](https://developers.docusign.com/extension-apps/extension-apps-101/supported-extensions/connected-fields/)
    article.
