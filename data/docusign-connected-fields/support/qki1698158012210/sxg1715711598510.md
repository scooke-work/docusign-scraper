# Example: Enable Postal Address Data Verification Fields

Learn how to set up Postal Address Data Verification Fields for documents in Docusign envelopes or templates.

This example shows how to select and include the postal address Data Verification Fields for
documents in Docusign envelopes or templates.

1. Start an envelope with the Send an Envelope option or use
   a template.
2. Upload your files and add recipients as usual.
3. In the field explorer, select the Custom Fields
   icon.
4. Select Verification to display available Verification
   fields from installed third-party verification applications.
5. Select Postal Address from the menu to display the
   address fields.

   ![A screenshot of Docusign Data Verification Postal Address field options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qki1698158012210/page/images/dvnewpostalsel.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDcyNDQsInNoZWFmIjoicWtpMTY5ODE1ODAxMjIxMCJ9.SQDA4DxDyWa4tbxe_9FUAt7wrHCtOFJ6kXpbntEf54o&_LANG=enus "A screenshot of Docusign Data Verification Postal Address field options.")

   Note: Fields with asterisks (\*) are required.
6. Drag the Postal Address field group to the
   document.
7. Select a field to display that field's configuration info in the App
   field.

   This example shows the Street configuration fields.

   ![A screenshot of the DocusignData Verification Postal Address street field configuration.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qki1698158012210/page/images/dvsetupbankfields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDcyNDQsInNoZWFmIjoicWtpMTY5ODE1ODAxMjIxMCJ9.SQDA4DxDyWa4tbxe_9FUAt7wrHCtOFJ6kXpbntEf54o&_LANG=enus "A screenshot of the DocusignData Verification Postal Address street field configuration.")
8. Select Must verify to sign to prevent recipients from
   completing the envelope unless this verification successfully passes.

   Note:

   The Must verify to sign is an optional
   setting.
   It's important to be selective when using this option. If false negative
   or positives occur, signers may be improperly blocked from completing
   envelopes.

   If
   you choose not to turn on the Must verify to sign
   setting, the data entered into the verification fields are still
   verified during signing.
9. After you've added any additional fields, select
   Send.

   For detailed information about sending envelopes, see the [Sending Documents for
   Signature](https://support.docusign.com/s/document-item?bundleId=ulp1643236876813&topicId=lak1578456412477.html&_LANG=enus) article.
