# Docusign Data Verification Extensions

Learn about the different Docusign data verification
extensions
that you can integrate in your Docusign documents.

You can integrate these Docusign data verification
extensions
in your Docusign documents.

- Bank account information includes:
  - Routing number
  - Account number
  - Account type (Checking or Savings)
- Bank account owner information includes:
  - Routing number
  - Account number
  - Account type (Checking or Savings)
  - First name
  - Last name
- Business entity information includes:
  - Federal Employer Identification Number (FEIN)
  - Company name
- Email Address verification: Verifies an email address
- Phone number verification: Verifies a phone number, including
  the country code
- Physical Address includes:
  - Address line 1
  - Address line 2
  - City
  - County or region
  - Postal code
  - State
- Social security number information includes:
  - Social Security number
  - First name
  - Last name
  - Date of birth (YYYY, M, D)

## Custom extensions

You can build your own extension app to verify data against your internal databases,
or the web service of your choice. When you do so, the resulting app provides custom
data verification
extensions
for your organization.

See the [Create a Private Docusign App](https://support.docusign.com/s/document-item?bundleId=qki1698158012210&topicId=bpc1740167704919.html&_LANG=enus "Learn how to create a private Custom private app for data verification , eSignature, or other Docusign features. Private apps provides custom data verification fields for your organization.") article for details.

## Extension contracts

Extension apps enable you to build custom third-party functionality into Docusign and publish your solutions in the Docusign App Center.

Each extension app has an extension contract. The contract is a set of actions that
your extension app must implement.

See the following information to learn about the Docusign Data
Verification extension contracts:

- [Business entity](https://developers.docusign.com/extension-apps/extension-app-reference/extension-contracts/business-fein-verification/)
- [Email Address verification](https://developers.docusign.com/extension-apps/extension-app-reference/extension-contracts/email-address-verification/)
