# Use a Certificate of Completion to Review Data Verification Events

Learn about using a signer's Certificate of Completion (CoC) to determine whether the
signer successfully completed Data Verification fields.

Use an envelope's Certificate of Completion (CoC) to determine whether the signer successfully
completed the Data Verification fields.

For Data Verification, the CoC shows:

- If the data verification was complete, and if the verification succeeded or
  failed
- The user who entered the data
- The Data Verification fields or fields mapped to extension apps
- The name and version of the extension app that provides the verification

For detailed information, see the [Certificate of
Completion](https://support.docusign.com/s/document-item?bundleId=oeq1643226594604&topicId=gpa1578456339545.html&_LANG=enus) article.

1. From the Manage page, locate the envelope by searching, filtering, or simply
   scanning the list.

   Note: To view the Certificate of Completion while signing a document, see
   the [Sign a Document With Data Verification Fields](https://support.docusign.com/s/document-item?bundleId=qki1698158012210&topicId=vyg1698168078020.html&_LANG=enus "Learn how to sign a document that has fields with data that must be verified before successfully signing the document.") article.
2. Select the envelope to show the Details view.
3. To download and view the Certificate of Completion PDF, select the Downloaded
   icon and select Certificate of Completion.
