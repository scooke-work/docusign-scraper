# Use Envelope History to Review Data Verification Events

Learn about whether a signer successfully completed a data verification field using your
account's eSignature envelope history.

Using your account's envelope history, as described in the [Envelope History](https://support.docusign.com/s/document-item?bundleId=oeq1643226594604&topicId=dpr1578456338693.html&_LANG=enus) article, you can view whether a signer successfully completed Data Verification fields.

The history logs the final result for every verification attempt by each signer.

1. In your eSignature account, select Manage to view a list
   of envelopes.

   Note: To view the Envelope History while signing a
   document, see the [Sign a Document With Data Verification Fields](https://support.docusign.com/s/document-item?bundleId=qki1698158012210&topicId=vyg1698168078020.html&_LANG=enus "Learn how to sign a document that has fields with data that must be verified before successfully signing the document.") article.
2. Select a completed envelope that contains verified data.
3. Select View Envelope History.

   The envelope history shows which Data Verification fields were successfully and
   unsuccessfully verified, as shown in this example.
