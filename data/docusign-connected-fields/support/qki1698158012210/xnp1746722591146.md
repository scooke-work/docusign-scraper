# Use Address Auto-Suggestion Fields

Learn how to use the Smarty extension app physical address auto-suggestions fields to
enter address fields easily and accurately in Docusign envelopes and
documents.

The [Smarty extension app](https://apps.docusign.com/app-center/app/04bfc1ae-1ba0-42d0-8c02-264417a7b234) provides accurate type-ahead
address suggestions. Using Smarty, Docusign envelope recipients
can easily and accurately enter address fields in Docusign
envelopes and documents. With Smarty, signers can:

- Enter at least one address line to start an address prediction. This can
  decrease 80% of the keystrokes typically required to enter an address.
- View a list of suggested addresses for faster agreement completion.
- Receive support in 242 countries as described in the [Supported Countries](https://www.smarty.com/docs/cloud/international-address-autocomplete-api#supported-country-codes) article.

Before you begin

Make sure that the Smarty extension app is available to use. Find and select the
Smarty app in the [Docusign App Center](https://apps.docusign.com/app-center) to display the
Smarty listing page.

If the Smarty app:

- Is not installed for your account. See the [Install Docusign Apps for Data Verification](https://support.docusign.com/s/document-item?bundleId=qki1698158012210&topicId=pyp1698158616536.html&_LANG=enus "Learn how to install Data Verification apps from the Docusign App Center.")
  article to learn about installing the app.
- Is already installed, select Use this App. The system
  displays your Docusign account Agreements page. You can start
  an envelope.

Tip: Including the Country field value narrows down
the auto-complete results. If you're using envelope templates:

- Include the Country field
- Consider placing the Country field as the first (at the top) address
  field.

Note: If the Country field is empty, the signer’s detected browser
location determines the country.

1. Start an envelope with the Send an Envelope option or use
   a template.
2. Upload your files and add recipients as usual.
3. In the field explorer, select Address from the Standard
   Fields to display the Smarty address fields.

   ![A screenshot showing selecting the address fields for an envelope.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qki1698158012210/page/images/dvsmartyseladdress.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDcyNDYsInNoZWFmIjoicWtpMTY5ODE1ODAxMjIxMCJ9.6i1WE8oBQW_ZgW5Iz3f4dzVw0D0nRFsUaPbE7VnYXdQ&_LANG=enus "A screenshot showing selecting the address fields for an envelope.")
4. Drag the Address field group to the document.

   This example shows the Smarty app's address fields.

   ![A screenshot of the Smarty app auto-suggestion address fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/qki1698158012210/page/images/dvsmartyavailablefields.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDcyNDYsInNoZWFmIjoicWtpMTY5ODE1ODAxMjIxMCJ9.6i1WE8oBQW_ZgW5Iz3f4dzVw0D0nRFsUaPbE7VnYXdQ&_LANG=enus "A screenshot of the Smarty app auto-suggestion address fields.")
5. Optionally add configuration information and send the envelope as described in
   the [Example: Enable Postal Address Data Verification Fields](https://support.docusign.com/s/document-item?bundleId=qki1698158012210&topicId=sxg1715711598510.html&_LANG=enus "Learn how to set up Postal Address Data Verification Fields for documents in Docusign envelopes or templates.") article.

   When signers receive the envelope, they enter information in
   Address Line 1 to activate the address
   suggestions.
