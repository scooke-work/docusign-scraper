# Sign a Document With Data Verification Fields

Learn how to sign a document that has fields with data that must be verified before
successfully signing the document.

With Data Verification Fields in a Docusign document, the verified data is
authenticated against data sources in real-time. For example, when completing an
employment application, you must verify your email address and phone number.

When you complete the signing field entries, the information you enter is
authenticated by a third-party application. Upon verification, you can complete the
signing.

Note: Your organization may create custom Data Verification Fields specific
to your company. See the [Data Verification
Field Types](https://support.docusign.com/s/document-item?bundleId=qki1698158012210&topicId=omw1740164520691.html&_LANG=enus) article for details.

1. Open the document as described in the [Get Started on the Docusign Signing Welcome Page](https://support.docusign.com/s/document-item?bundleId=yca1573855023892&topicId=uwx1573854978359.html&_LANG=enus) article.
2. Fields that require verification are indicated by a tool tip when you hover
   over them, as shown in the following example:

   ![An example of a tool tip for a data field that must be verified when signing a document.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/dvreqfieldtip.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDcyNzYsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.UMwrSDPPuyxf0ftwomJeENNvBcFYPGCb3HwSN9jQQcc&_LANG=enus "An example of a tool tip for a data field that must be verified when signing a document.")
3. Enter the information and go to the next field.

   Note: If you're entering a phone number, you
   are required to include both the country code and phone number.

   While validating, the system displays the following message:

   `Verifying data`
4. If you correctly entered the information, you received a verification
   confirmation message.

   `Verification succeeded`
5. If you incorrectly entered the information, you receive a failed verification
   message:

   `Verification failed`

   Review and reenter your information.
6. Continue entering information in any subsequent fields.

   ![A screenshot of a sample document with Data Verification Fields.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signing-example.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDcyNzYsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.UMwrSDPPuyxf0ftwomJeENNvBcFYPGCb3HwSN9jQQcc&_LANG=enus "A screenshot of a sample document with Data Verification Fields.")
7. Select Finish to complete the signing process.
8. To review the audit trail details about the agreements you signed, select the
   vertical three-dot menu and select View History to view
   the envelope history or View Certificate to view the
   Certificate of Completion.

   ![A screenshot of the Other Actions menu audit trail options.](https://docusign-be-prod.zoominsoftware.io/api/bundle/yca1573855023892/page/images/signnewviewcert.png?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJjdXN0b21lciI6ImRvY3VzaWduX3Byb2R1Y3Rpb24iLCJleHAiOjE3ODE4MDcyNzYsInNoZWFmIjoieWNhMTU3Mzg1NTAyMzg5MiJ9.UMwrSDPPuyxf0ftwomJeENNvBcFYPGCb3HwSN9jQQcc&_LANG=enus "A screenshot of the Other Actions menu audit trail options.")

   See these articles for detailed audit information:

   - [Envelope history](https://support.docusign.com/s/document-item?bundleId=qki1698158012210&topicId=uro1698158848949.html): Includes a list of sender
     and recipient activities to date
   - [Certificate of Completion](https://support.docusign.com/s/document-item?bundleId=qki1698158012210&topicId=qjs1701290377826.html): Includes the
     identifying information about the envelope and complete details of the
     envelope events
